<div id="export-data">
  ## Exporter les données
</div>

Si vous souhaitez exporter vos données d’Auth0, plusieurs options s’offrent à vous. Veuillez noter que ces outils n’exportent pas les hachages de mots de passe de vos utilisateurs stockés dans la base de données hébergée par Auth0. Vous pouvez tout de même demander ces renseignements en ouvrant un [ticket de soutien](https://support.auth0.com/). Si vous prévoyez ne plus utiliser notre service, consultez [Exporter les données](/fr-CA/docs/troubleshoot/customer-support/manage-subscriptions/export-data) pour en savoir plus.

<div id="use-the-importexport-extension">
  ### Utiliser l’extension Import/Export
</div>

Vous pouvez utiliser notre [extension Import/Export](/fr-CA/docs/manage-users/user-migration/user-import-export-extension) pour exporter presque toutes les données de votre tenant Auth0.

<div id="use-the-management-api">
  ### Utiliser la Management API
</div>

Si vous souhaitez exporter par programmation certains ensembles de données, utilisez le point de terminaison Bulk User Export de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>.

<div id="transfer-data">
  ## Transfert de données
</div>

Auth0 ne transfère pas de données d’un tenant Auth0 à un autre. Cela s’applique autant aux clients de Public Cloud que de Private Cloud.

Toutes les données de votre tenant Auth0 demeurent en tout temps sous votre contrôle et sont accessibles par l’entremise de la Management API. Les seules informations qui ne sont pas accessibles par l’entremise de l’API, pour des raisons de sécurité, sont les hachages de mot de passe de vos utilisateurs de base de données hébergée par Auth0 ainsi que les clés privées.
