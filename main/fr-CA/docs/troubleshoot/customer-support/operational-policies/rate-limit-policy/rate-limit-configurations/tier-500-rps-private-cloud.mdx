title: Performance de Private Cloud 500 RPS (5x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total pour l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une forte utilisation combinée de l’Authentication API et de la Management API, ou certains flux d’authentification comme ROPG vers l’Authentication API, peuvent entraîner une baisse des performances à un RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit pour le type d’abonnement Private Cloud Performance 500 RPS (5x).

Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud. Par conséquent, nous recommandons de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
