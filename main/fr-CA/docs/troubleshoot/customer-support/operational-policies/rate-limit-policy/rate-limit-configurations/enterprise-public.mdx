<div id="authentication-api">
  ## Authentication API
</div>

Consultez le tableau suivant pour connaître les limites de débit globales de l’Authentication API, c’est-à-dire la politique de limite de débit qui protège presque tous les points de terminaison de l’Authentication API.

Limites de débit pour l’Authentication API et les points de terminaison d’API du type d’abonnement Enterprise :

| Tenant                                   | [Limite de requêtes en rafale](/fr-CA/docs/troubleshoot/customer-support/operational-policies/rate-limit-policy#rate-limit-algorithm) | [Limite de requêtes soutenue](/fr-CA/docs/troubleshoot/customer-support/operational-policies/rate-limit-policy#rate-limit-algorithm) |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Production                               | 100/seconde                                                                                                                     | 100/seconde                                                                                                                    |
| Production (2x Public Performance Burst) | 200/seconde pendant 48 h par mois                                                                                               | 100/seconde                                                                                                                    |
| Production (3x Public Performance Burst) | 300/seconde pendant 48 h par mois                                                                                               | 100/seconde                                                                                                                    |
| Production (4x Public Performance Burst) | 400/seconde pendant 48 h par mois                                                                                               | 100/seconde                                                                                                                    |
| Hors production                          | 100/seconde                                                                                                                     | 100/seconde                                                                                                                    |

Ces limites s’appliquent pendant un maximum de 48 heures par mois. Après 48 heures, elles reviennent aux limites du produit. Pour en savoir plus, consultez [Public Performance Burst](/fr-CA/docs/troubleshoot/customer-support/operational-policies/rate-limit-policy#public-performance).

<div id="scim-api">
  ## API SCIM
</div>

Limites de débit pour les points de terminaison de l’API SCIM entrante dans les abonnements au cloud public qui incluent des connexions d’entreprise :

| Type de limite                                           | Chemin du point de terminaison          | Opération     | Limite                   |
| -------------------------------------------------------- | -------------------------------------- | ------------- | ------------------------ |
| Point de terminaison d’une connexion SCIM unique         | `/scim/v2/connections/{connection-id}` | Toute requête | 25 requêtes par seconde  |
| Limite globale du tenant pour toutes les connexions SCIM | `/scim/v2/connections/*`               | Toute requête | 100 requêtes par seconde |
