title: Private Cloud Basic 100 RPS (1x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total pour l’Authentication API et l’API MyAccount dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une forte utilisation combinée des Authentication API et Management API, ou certains flux d’authentification comme ROPG vers l’Authentication API, peuvent entraîner une baisse des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit associées au type d’abonnement Private Cloud Basic 100 RPS (1x). Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud.

Nous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
