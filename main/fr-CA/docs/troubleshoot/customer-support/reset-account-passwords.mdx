---
description: Décrit comment réinitialiser les mots de passe des comptes Auth0.
title: Réinitialiser les mots de passe des comptes
---

Si vous devez modifier votre mot de passe ou si vous avez oublié celui de votre compte Auth0, vous pouvez, dans la plupart des cas, définir un nouveau mot de passe à partir du <Tooltip tip="Auth0 Dashboard : produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. La réinitialisation d’un mot de passe entraîne l’expiration des sessions Auth0.

1. Sur l’écran de connexion au compte Auth0, entrez l’adresse courriel associée à votre compte, puis cliquez sur **Continue**. À l’écran suivant, choisissez **Forgot password?**
2. Confirmez votre adresse courriel et cliquez sur **Continue** pour envoyer votre demande. Vous recevrez ensuite un courriel contenant la marche à suivre pour réinitialiser votre mot de passe.
3. Si votre demande a bien été reçue, le message suivant s’affichera : « Please check the email address [your email] for instructions to reset your password. »

Il existe certains cas particuliers de réinitialisation de mot de passe :

* Si vous avez activé la <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=MFA">MFA</Tooltip> et que vous devez réinitialiser la MFA pour votre compte administrateur, communiquez avec [Auth0 Support](https://support.auth0.com).
* Si vous utilisez un compte social ou d’entreprise pour vous connecter à Auth0, réinitialisez votre mot de passe auprès du <Tooltip tip="Fournisseur d’identité (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=identity+provider">fournisseur d’identité</Tooltip> approprié.
* Si vous êtes un administrateur qui tente de réinitialiser le mot de passe d’un utilisateur, deux méthodes s’offrent à vous :

  * Utiliser le flux interactif de réinitialisation de mot de passe avec Universal Login ou l’Authentication API, dans lequel l’utilisateur reçoit un courriel contenant un lien qui ouvre la page de réinitialisation de mot de passe Auth0 pour saisir un nouveau mot de passe.
  * Définir directement le nouveau mot de passe au moyen de la Management API ou du Dashboard.
