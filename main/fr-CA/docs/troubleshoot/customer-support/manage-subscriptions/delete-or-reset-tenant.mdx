Vous pouvez supprimer votre tenant Auth0 ou le réinitialiser à sa configuration par défaut dans l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Vous pouvez supprimer les données de votre compte Auth0 avec l’aide d’Auth0 Support.

<div id="delete-tenant">
  ## Supprimer le tenant
</div>

<Warning>
  La suppression d’un tenant est une opération irréversible et destructive qui ne peut pas être annulée. Les tenants supprimés ne peuvent pas être restaurés, le nom du tenant ne peut pas être réutilisé lors de la création d’un nouveau tenant, et les métriques associées au tenant sont perdues.
</Warning>

Avant de supprimer un tenant, envisagez les options suivantes :

* [Changer le propriétaire du tenant](/fr-CA/docs/get-started/manage-dashboard-access).
* [Réinitialiser la configuration du tenant](#reset-tenant-configuration).

<div id="remove-custom-domain">
  ### Supprimer le domaine personnalisé
</div>

Si vous avez configuré un <Tooltip tip="Domaine personnalisé : domaine tiers avec un nom spécialisé ou personnalisé." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=custom+domain">domaine personnalisé</Tooltip> pour un tenant que vous souhaitez supprimer, vous devez d’abord retirer ce domaine personnalisé. Sinon, vous ne pourrez pas configurer ce domaine personnalisé pour un autre tenant.

<div id="determine-deletion-process">
  ### Déterminer le processus de suppression
</div>

Le processus de suppression d’un tenant dépend du [forfait d’abonnement](https://www.auth0.com/pricing) auquel il est associé et du [type de tenant](/fr-CA/docs/get-started/auth0-overview/create-tenants/child-tenants) dont il s’agit (principal ou enfant). Il se peut que vous puissiez supprimer le tenant dans l’[Auth0 Dashboard](#delete-tenant-in-auth0-dashboard), ou que vous deviez [communiquer avec Auth0 Support ou l’Auth0 Community](#contact-auth0-support-or-community).

|                                           | plan Free                         | Forfait en libre-service       | plan Enterprise               |
| ----------------------------------------- | ---------------------------------- | ------------------------------ | ------------------------------ |
| **Tenant principal**                      | Auth0 Dashboard                    | Auth0 Dashboard                | Communiquez avec Auth0 Support |
| **Tenant principal avec tenants enfants** | Communiquez avec l’Auth0 Community | Communiquez avec Auth0 Support | Communiquez avec Auth0 Support |
| **Tenant enfant**                         | Auth0 Dashboard                    | Auth0 Dashboard                | Auth0 Dashboard                |

<div id="downgrade-tenant">
  ### Rétrograder le tenant
</div>

Avant de pouvoir supprimer un tenant dans le Dashboard, vous devez d’abord le faire passer au plan Free. Pour en savoir plus, consultez [Rétrograder ou annuler des abonnements Auth0](/fr-CA/docs/troubleshoot/customer-support/manage-subscriptions/downgrade-or-cancel-subscriptions).

<div id="delete-tenant-in-dashboard">
  ### Supprimer un tenant dans le Dashboard
</div>

Vous pouvez supprimer un tenant dans le Dashboard si vous disposez du [niveau d’accès approprié](/fr-CA/docs/get-started/manage-dashboard-access/feature-access-by-role).

1. Accédez à [Dashboard &gt; Settings](https://manage.auth0.com/#/tenant/advanced), puis passez à la vue **Advanced**.
2. Repérez la **Danger Zone** au bas de la page, puis sélectionnez **Delete**.
3. Lisez l’invite de confirmation et confirmez que vous souhaitez supprimer définitivement votre tenant.
4. Sélectionnez **Yes, Delete Tenant**.

<div id="contact-auth0-support-or-community">
  ### Contacter Auth0 Support ou Auth0 Community
</div>

Si votre tenant relève d’un plan Trial, Essential, Professional ou Enterprise et que vous ne pouvez pas supprimer votre tenant dans le Auth0 Dashboard, [ouvrez un ticket auprès d’Auth0 Support](/fr-CA/docs/troubleshoot/customer-support/open-and-manage-support-tickets) pour obtenir de l’assistance.

Si votre tenant relève du plan Free et que vous ne pouvez pas le supprimer dans le Auth0 Dashboard, [publiez un message dans Auth0 Community](https://community.auth0.com/) et demandez de l’assistance.

<div id="reset-tenant-configuration">
  ## Réinitialiser la configuration du tenant
</div>

Vous pouvez rétablir la configuration par défaut de votre tenant à l’aide de l’[Auth0 Deploy CLI](https://github.com/auth0/auth0-deploy-cli).

Le Deploy CLI est un outil qui vous aide à gérer la configuration de votre tenant Auth0. Il s’intègre à vos flux de travail de développement en tant que CLI autonome ou module Node.js. Pour en savoir plus sur l’outil Deploy CLI et sur la façon de démarrer, consultez [Deploy CLI Tool](/fr-CA/docs/deploy-monitor/deploy-cli-tool).

<div id="delete-auth0-account-data">
  ## Supprimer les données du compte Auth0
</div>

Si vous souhaitez supprimer les données de votre compte Auth0 (y compris votre adresse courriel, vos informations de facturation, etc.), suivez les étapes pour supprimer votre tenant, puis [publiez un message dans Auth0 Community](https://community.auth0.com/) pour demander de l’aide.

Pour en savoir plus sur les responsabilités d’Auth0 à titre de responsable du traitement, consultez la [politique de confidentialité d’Okta](https://www.okta.com/privacy-policy/).
