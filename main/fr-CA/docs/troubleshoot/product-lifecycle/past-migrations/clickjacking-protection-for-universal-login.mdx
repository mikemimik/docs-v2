description: Décrit comment Auth0 ajoute un moyen d’empêcher l’intégration des pages Universal Login dans une iframe afin de les protéger contre les attaques par détournement de clic.
title: Modification de la protection contre le détournement de clic pour Universal Login
---

Le détournement de clic est une attaque qui consiste à tromper un utilisateur pour l’amener à cliquer sur un élément de page Web invisible ou déguisé en un autre élément. Pour ce faire, du contenu est chargé dans une iframe et des éléments sont superposés par-dessus. Dans le contexte des pages <Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur l’Authorization Server d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Universal+Login">Universal Login</Tooltip>, un attaquant pourrait inciter l’utilisateur à cliquer sur un bouton **Connexion** ou **Réinitialiser le mot de passe**.

On peut éviter cela en définissant les en-têtes HTTP suivants :

`X-Frame-Options: deny
Content-Security-Policy: frame-ancestors 'none'`

Même si cette attaque potentielle ne présente pas de risque majeur, il demeure recommandé, d’un point de vue sécurité, d’ajouter ces en-têtes. De plus, leur absence est détectée par les analyseurs de sécurité; les rapports de tests d’intrusion pourraient donc la signaler.

<div id="actions">
  ## Actions
</div>

Si vous affichez la page de connexion dans un iframe, l’ajout de ces en-têtes pourrait constituer un changement incompatible. Auth0 vous permet donc de choisir d’activer ces en-têtes plutôt que de les ajouter pour tous les clients, et nous vous recommandons fortement de le faire.

L’action suivante n’est pas nécessaire si vous utilisez la nouvelle expérience Universal Login, puisque ces en-têtes sont alors toujours définis.

Pour activer ce changement :

1. Allez à [Tenant Settings &gt; Advanced Settings](https://manage.auth0.com/#/tenant/advanced).
2. Faites défiler jusqu’à **Migrations**, puis désactivez le paramètre **Disable clickjacking protection for Classic Universal Login**.
