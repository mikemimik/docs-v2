title: Ajouter l’authentification multifacteur pour l’accès à Auth0 Dashboard
---

Chaque utilisateur d’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> devrait s’inscrire lui-même à l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA). Vous pouvez vous inscrire à la plupart des facteurs dans [Votre profil](https://manage.auth0.com/#/profile). La biométrie de l’appareil, toutefois, nécessite une inscription progressive.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 recommande les facteurs WebAuthn comme méthodes d’authentification les plus sécuritaires et les plus conviviales. Pour en savoir plus, consultez [Authentification FIDO avec WebAuthn](/fr-CA/docs/secure/multi-factor-authentication/fido-authentication-with-webauthn).
</Callout>

<Warning>
  Les Admins doivent activer au moins un facteur pour utiliser le MFA. Auth0 recommande fortement de configurer plusieurs facteurs afin que vous puissiez tout de même accéder à votre compte si vous perdez votre appareil principal.

  La configuration idéale consiste à utiliser trois facteurs :

  1. WebAuthn, Guardian ou OTP comme méthode principale.
  2. Un ou plusieurs numéros SMS comme solution de secours (offerts uniquement pour les tenants associés à un plan d’abonnement).
  3. Un code de récupération.

  Si vous ne pouvez pas fournir votre jeton MFA et que vous ne disposez pas de méthodes de secours appropriées, votre compte pourrait être irrécupérable.
</Warning>

<div id="add-mfa">
  ## Ajouter la MFA
</div>

Pour vous inscrire vous-même à la MFA, chaque utilisateur du Dashboard doit suivre les étapes suivantes :

1. Cliquez sur votre nom d’utilisateur dans le coin supérieur droit du Dashboard, puis sur [Votre profil](https://manage.auth0.com/#/profile).
2. Repérez la méthode prise en charge de votre choix et cliquez sur **+ ADD** dans cette ligne.

   <Frame>![Dashboard - Profil - Multifacteur - Authentification](/docs/images/cdy7uua7fh8z/2NyLNvP92RruwZ46dLPm5x/8ece3df2b550af8dd78368ca36d8a661/MFA.png)</Frame>
3. Suivez les instructions à l’écran pour terminer l’inscription.

<div id="device-biometrics">
  ## Biométrie de l’appareil
</div>

WebAuthn with biométrie de l’appareil est la seule méthode que vous ne pouvez pas ajouter dans la page Paramètres du compte. Auth0 inscrit plutôt progressivement tous vos appareils compatibles avec WebAuthn. Auth0 vous invite à inscrire ces appareils après l’inscription à toute autre méthode MFA. Cette invite revient chaque fois que vous vous connectez à Auth0 Dashboard.

<Frame>![undefined](/docs/images/cdy7uua7fh8z/1cBjDSZrIeHAS2Fyzbt5HK/7045d8e2c5b0b0588d5eb88f4a5620b0/deviceenrollment.png)</Frame>

Dans le cadre de l’inscription, Auth0 vous invite à nommer vos appareils. Vous pouvez ainsi les gérer facilement dans la page Paramètres du compte.

Les navigateurs avec Javascript désactivé ou sans prise en charge de WebAuthn platform authenticator ne permettent pas d’inscrire ni d’authentifier des utilisateurs avec la biométrie de l’appareil. Les versions les plus récentes des navigateurs et systèmes d’exploitation courants prennent en charge WebAuthn with Security Keys. Pour en savoir plus, consultez la [section sur la compatibilité des navigateurs sur webauthn.me](https://webauthn.me/browser-support).

<div id="recovery-codes">
  ## Codes de récupération
</div>

Immédiatement après avoir activé l’authentification à deux facteurs, Auth0 vous invite à copier un code de récupération. Si vous perdez l’accès à tous vos facteurs configurés, vous pouvez utiliser ce code de récupération pour vous connecter à votre compte. Auth0 recommande de copier et d’imprimer les codes de récupération, ou de les conserver dans un endroit sûr, comme un gestionnaire de mots de passe.

Si vous perdez vos codes de récupération ou souhaitez simplement en générer de nouveaux, vous pouvez le faire à partir de [Votre profil](https://manage.auth0.com/#/profile).

<div id="log-in-to-the-dashboard-with-mfa-enabled">
  ## Connectez-vous au Dashboard avec MFA activée
</div>

Se connecter avec MFA activée diffère seulement légèrement d’une connexion normale. Lorsque vous entrez les informations d’identification du compte Admin, une deuxième invite s’affiche, selon le type de facteurs MFA que vous avez activés.

Si un utilisateur perd l’accès à un facteur principal, il peut cliquer sur **Sélectionner une autre méthode** et essayer n’importe lequel des autres facteurs, y compris les codes de récupération. C’est pourquoi il est si important d’activer plusieurs méthodes pour éviter d’être verrouillé hors de votre compte.

Après avoir ajouté avec succès votre deuxième facteur d’authentification et ouvert une session à partir d’un nouvel appareil compatible avec WebAuthn, vous verrez une invite vous invitant à « Ouvrir une session plus rapidement sur cet appareil ». Cela vous permettra d’utiliser cet appareil pour l’authentification multifacteur la prochaine fois.
