title: Supprimer ou modifier l’authentification multifacteur du Dashboard
---

Les utilisateurs du Dashboard peuvent supprimer ou modifier des facteurs d’<Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA) qui ne sont plus disponibles ou ne sont plus souhaités. Pour en savoir plus sur la MFA pour les utilisateurs du Dashboard, consultez [Authentification multifacteur pour les utilisateurs du Dashboard](/fr-CA/docs/get-started/manage-dashboard-access/add-change-remove-mfa).

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 recommande les facteurs WebAuthn comme méthodes d’authentification les plus sûres et les plus faciles à utiliser. Pour en savoir plus, consultez [Authentification FIDO avec WebAuthn](/fr-CA/docs/secure/multi-factor-authentication/fido-authentication-with-webauthn).
</Callout>

<Warning>
  Les administrateurs doivent activer au moins un facteur pour utiliser la MFA. Auth0 recommande fortement de configurer plusieurs facteurs afin que vous puissiez quand même accéder à votre compte si vous perdez votre appareil principal.

  La configuration idéale consiste à utiliser trois facteurs :

  1. WebAuthn, Guardian ou OTP comme méthode principale.
  2. Un ou plusieurs numéros de téléphone pour les SMS comme solution de secours (offert uniquement pour les tenants associés à un forfait d’abonnement).
  3. Un code de récupération.

  Si vous ne pouvez pas fournir votre jeton MFA et que vous n’avez pas de méthodes de secours adéquates, il se peut que votre compte soit irrécupérable.
</Warning>

<div id="remove-or-change-an-mfa-factor-from-the-dashboard">
  ## Supprimer ou remplacer un facteur MFA dans le Dashboard
</div>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Vous ne pouvez pas ajouter un facteur Device Biometrics (comme la Touch Bar d’un MacBook, Windows Hello, Touch ID ou Face ID sur iOS, ou encore la reconnaissance faciale ou par empreinte sur Android) dans le Dashboard. Pour savoir comment ajouter un facteur d’authentification Device Biometrics, consultez [Ajouter l’authentification multifacteur pour les utilisateurs du Dashboard.](/fr-CA/docs/get-started/manage-dashboard-access/add-change-remove-mfa/add-mfa)
</Callout>

Les utilisateurs du Dashboard qui peuvent se connecter avec leurs facteurs MFA actuels peuvent suivre ces étapes :

1. Dans le coin supérieur droit du Dashboard, cliquez sur votre nom d’utilisateur, puis sur [Votre profil](https://manage.auth0.com/#/profile).

   <Frame>![Dashboard - Profil - Authentification multifacteur](/docs/images/cdy7uua7fh8z/2NyLNvP92RruwZ46dLPm5x/8ece3df2b550af8dd78368ca36d8a661/MFA.png)</Frame>
2. Repérez le nouveau facteur d’authentification que vous souhaitez utiliser, puis cliquez sur **+ ADD** dans cette ligne. Suivez les instructions à l’écran pour terminer l’inscription.
3. Toujours dans [Votre profil](https://manage.auth0.com/#/profile), repérez le facteur d’authentification que vous ne voulez plus utiliser, puis cliquez sur **REMOVE**.
4. Cliquez sur **Oui** pour confirmer la suppression.
5. Auth0 vous demande de vous authentifier à l’aide de vos facteurs actuels. Une fois l’authentification réussie, Auth0 supprime le facteur.

<div id="remove-or-change-a-lost-mfa-factor">
  ## Supprimer ou remplacer un facteur MFA perdu
</div>

Les utilisateurs du Dashboard qui **ne peuvent pas** se connecter avec leurs facteurs MFA actuels peuvent suivre ces étapes :

1. Essayez de vous connecter au Dashboard. Auth0 vous invite à vous authentifier avec vos facteurs actuels.
2. Lorsque Auth0 vous demande l’appareil ou les identifiants que vous avez perdus, cliquez sur **Essayer une autre méthode**.
3. Dans la zone Other Methods, cliquez sur une autre méthode pour vous authentifier.
4. Connectez-vous pour accéder au Dashboard.
5. Dans le coin supérieur droit du Dashboard, cliquez sur votre nom d’utilisateur, puis sur [Votre profil](https://manage.auth0.com/#/profile).

   <Frame>![Dashboard - Profil - Authentification multifacteur](/docs/images/cdy7uua7fh8z/2NyLNvP92RruwZ46dLPm5x/8ece3df2b550af8dd78368ca36d8a661/MFA.png)</Frame>
6. Repérez la méthode d’authentification que vous ne pouvez plus utiliser et cliquez sur **REMOVE**.
7. Cliquez sur **Yes** pour confirmer la suppression.
8. Auth0 vous invite de nouveau à vous authentifier à l’aide de vos facteurs actuels. Répétez les étapes 2 à 4 pour vérifier votre identité. Auth0 supprime le facteur perdu.

<div id="get-help-from-auth0-support">
  ## Obtenir de l’aide auprès du soutien Auth0
</div>

Si vous êtes bloqué et que vous n’avez accès à aucun de vos facteurs MFA activés, rien ne garantit que vous pourrez retrouver l’accès à votre compte. Un autre administrateur doit soumettre un ticket de soutien Auth0 en votre nom. Dans certains cas, Auth0 peut vérifier la demande et procéder à une réinitialisation de la MFA. Toutefois, il se peut que nous ne soyons pas en mesure de confirmer que le compte vous appartient. C’est pourquoi il est si important d’activer plusieurs facteurs différents.

Le soutien Auth0 ne réinitialise pas les comptes des **utilisateurs finaux**. Vous êtes responsable des comptes qui accèdent à vos applications et à vos API. Pour en savoir plus sur les comptes d’utilisateurs finaux, consultez [Manage Users](/fr-CA/docs/manage-users).
