Consultez ce guide de référence pour voir des exemples en langage naturel qui vous permettent d’interagir avec votre tenant Auth0 à l’aide d’Auth0 MCP Server et de n’importe quel [client MCP](https://modelcontextprotocol.io/clients).

<div id="application-management">
  ## Gestion des applications
</div>

| Outil                       | Description                                                                    | Exemples en langage naturel                                                                                                                                                                                                                                                                                                                   |
| --------------------------- | ------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `auth0_list_applications`   | Répertorier toutes les applications du tenant Auth0 ou les rechercher par nom. | &quot;Montre-moi toutes mes applications Auth0.&quot;<br /><br />&quot;Trouve les applications dont le nom contient &#39;api&#39;.&quot;<br /><br />&quot;Quelles applications ai-je dans mon tenant Auth0?&quot;                                                                                                                             |
| `auth0_get_application `    | Obtenir les détails d&#39;une application Auth0 précise.                       | &quot;Montre-moi les détails de l&#39;application appelée &#39;Customer Portal&#39;.&quot;<br /><br />&quot;Obtiens des renseignements sur mon application avec le Client ID {clientID}.&quot;<br /><br />&quot;Quelles sont les URL de rappel de mon &#39;Mobile App&#39;?&quot;                                                             |
| `auth0_create_application ` | Créer une nouvelle application Auth0.                                          | &quot;Crée une nouvelle application monopage appelée &#39;Analytics Dashboard&#39;.&quot;<br /><br />&quot;Configure une nouvelle application native mobile appelée &#39;iOS Client&#39;.&quot;<br /><br />&quot;Crée une application machine à machine pour notre service d&#39;arrière-plan.&quot;                                          |
| `auth0_update_application`  | Mettre à jour une application Auth0 existante.                                 | &quot;Mets à jour les URL de rappel de mon &#39;Web App&#39; pour inclure `https://staging.example.com/callback`.&quot;<br /><br />&quot;Modifie l&#39;URL de déconnexion de &#39;Customer Portal&#39;.&quot;<br /><br />&quot;Ajoute des métadonnées d&#39;environnement de développement à mon application &#39;Admin Dashboard&#39;.&quot; |

<div id="combining-operations">
  ## Combiner des opérations
</div>

Vous pouvez aussi demander à [Claude](https://claude.ai/) d’effectuer plusieurs opérations connexes à la suite :

<div id="set-up-a-new-api-with-client-access">
  #### Configurer une nouvelle API avec accès client
</div>

```bash wrap lines
Create a new API called "Inventory API" with read and write scopes, then update my "Admin Dashboard" application to have access to it.
```

<div id="create-and-deploy-a-security-action">
  #### Créer et déployer une Action de sécurité
</div>

```bash wrap lines
Créez une Action qui exige le MFA pour les utilisateurs qui se connectent depuis l'extérieur de notre réseau d'entreprise, puis déployez-la dans le flux de connexion.
```

<div id="analyze-security-events">
  #### Analyser les événements de sécurité
</div>

```bash wrap lines
Montre-moi toutes les tentatives de connexion échouées de la semaine dernière et identifie les tendances ou toute activité suspecte.
```

<div id="updating-branding-across-forms">
  #### Mettre à jour l’identité visuelle dans tous les formulaires
</div>

```bash wrap lines
Update all our Auth0 forms to use our new logo at <https://example.com/new-logo.png> and change the primary color to #0066CC.
```

<div id="best-practices">
  ## Meilleures pratiques
</div>

1. Soyez précis dans vos requêtes : incluez des noms, des IDs ou d&#39;autres renseignements permettant d&#39;identifier les ressources précises auxquelles vous faites référence.
2. Utilisez un langage naturel : vous n&#39;avez pas besoin de mémoriser les noms des outils ni les paramètres; décrivez simplement ce que vous voulez accomplir.
3. Vérifiez avant de confirmer : lorsque Claude propose des modifications, examinez-les attentivement avant de confirmer.
4. Commencez simplement : commencez par des requêtes simples pour vous familiariser avec la façon dont Claude interprète vos requêtes.

<div id="troubleshooting">
  ## Dépannage
</div>

Si Claude ne parvient pas à effectuer une opération, tenez compte des points suivants :

1. Authentification : assurez-vous d’être connecté à l’aide de l’Auth0 CLI (`auth0 login`).
2. Permissions : vérifiez que votre compte dispose des permissions nécessaires dans Auth0.
3. Précision : essayez d’être plus précis dans votre demande.
