---
description: Décrit comment créer un tenant supplémentaire dans l’Auth0 Dashboard.
title: Créer plusieurs tenants
---

Vous pouvez configurer plusieurs tenants pour créer [différents environnements](/fr-CA/docs/get-started/auth0-overview/create-tenants/set-up-multiple-environments) dans le <Tooltip tip="Auth0 Dashboard : produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> afin de prendre en charge des configurations complexes. Par exemple, vous pourriez avoir deux domaines distincts (un interne et un accessible au public), ou vouloir que les utilisateurs se connectent différemment selon l’application. Pour y parvenir, créez plus d’un Auth0 tenant afin d’avoir des ensembles distincts d’applications, de connexions et d’utilisateurs pour les applications et les groupes d’utilisateurs que vous devez prendre en charge.

1. Accédez à l’[Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez le nom de votre tenant, puis sélectionnez **Create Tenant**.

   <Frame>![Menu déroulant du tenant dans le Dashboard pour créer un tenant](/docs/images/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/5efad1137eb8b04031b30137c2fc5ac5/2025-02-26_14-51-39.png)</Frame>
2. Entrez le **Tenant Domain** de votre choix, sélectionnez une **Region**, puis sélectionnez **Create**.
