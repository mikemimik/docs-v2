Vous pouvez supprimer une application à l’aide du <Tooltip tip="Auth0 Dashboard : le produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou de la <Tooltip tip="Auth0 Dashboard : le produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>. Une fois l’opération confirmée, elle ne peut pas être annulée.

<div id="use-the-dashboard">
  ## Utiliser le Dashboard
</div>

1. Allez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis sélectionnez le nom de l’application à consulter.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler jusqu’au bas de la page **Settings**, repérez la section **Danger Zone**, sélectionnez **Delete**, puis confirmez.

<div id="use-the-management-api">
  ## Utiliser la Management API
</div>

Effectuez une requête `DELETE` vers le point de terminaison [`/Clients/delete_clients_by_id`](https://auth0.com/docs/api/management/v2#!/Clients/delete_clients_by_id). Assurez-vous de remplacer les valeurs de remplacement `{yourClientId}` et `MGMT_API_ACCESS_TOKEN` par votre <Tooltip tip="Client ID : valeur d’identification attribuée à votre ressource enregistrée dans Auth0." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=client+ID">ID client</Tooltip> et votre <Tooltip tip="Access Token : information d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=access+token">jeton d’accès</Tooltip> de la Management API, respectivement.

  ```bash cURL
  curl --request DELETE \
     --url 'https://{yourDomain}/api/v2/clients/%7ByourClientId%7D' \
     --header 'authorization: Bearer {yourMgmtApiToken}'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/clients/%7ByourClientId%7D");
  var request = new RestRequest(Method.DELETE);
  request.AddHeader("authorization", "Bearer {yourMgmtApiToken}");
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
  	"fmt"
  	"net/http"
  	"io/ioutil"
  )

  func main() {

  	url := "https://{yourDomain}/api/v2/clients/%7ByourClientId%7D"

  	req, _ := http.NewRequest("DELETE", url, nil)

  	req.Header.Add("authorization", "Bearer {yourMgmtApiToken}")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

</AuthCodeGroup>

| Valeur                  | Description                                                                                                              |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| `{yourClientId}`        | ID de l’application à supprimer.                                                                                         |
| `MGMT_API_ACCESS_TOKEN` | [Jetons d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec le scope `delete:clients`. |
