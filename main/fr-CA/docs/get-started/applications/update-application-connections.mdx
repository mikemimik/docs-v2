title: Mettre à jour les connexions d’une application
---

Vous pouvez modifier les connexions activées de votre application dans le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Accédez à [Dashboard &gt; Applications](https://manage.auth0.com/#/applications) et cliquez sur le nom de l’application pour l’ouvrir.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Cliquez sur l’onglet **Connexions**, puis activez ou désactivez les connexions appropriées pour l’application.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Les [applications tierces](/fr-CA/docs/get-started/applications/third-party-applications) peuvent authentifier les utilisateurs au moyen de connexions rendues disponibles au niveau du domaine ou de connexions explicitement associées à l’application. Pour en savoir plus, consultez [Rendre les connexions disponibles au niveau du domaine](/fr-CA/docs/authenticate/identity-providers/promote-connections-to-domain-level).
</Callout>
