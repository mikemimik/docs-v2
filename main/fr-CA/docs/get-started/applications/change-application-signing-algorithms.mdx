description: Décrit comment modifier l’algorithme de signature d’une application dans l’Auth0 Dashboard.
title: Modifier l’algorithme de signature d’une application
---

Vous pouvez modifier l’<Tooltip tip="Algorithme de signature : algorithme utilisé pour signer numériquement des jetons afin de garantir que le jeton n’a pas été altéré." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=signing+algorithm">algorithme de signature</Tooltip> de votre application dans le Dashboard.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis sélectionnez le nom de l’application à afficher.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler la page jusqu’au bas de l’onglet **Settings**, puis sélectionnez **Show Advanced Settings**.
3. Accédez à l’onglet **OAuth**.

   <Frame>![Onglet OAuth des paramètres de l’application dans Dashboard Applications](/docs/images/cdy7uua7fh8z/5bbq6VrI9tUMht0PwMimre/08574482efef944b16ecfbb445be50ab/Oauth_Tab_-_English.png)</Frame>
4. Repérez **JsonWebToken Signature Algorithm**, puis sélectionnez l’algorithme de signature approprié pour l’application.
5. Cliquez sur **Save Changes**.
