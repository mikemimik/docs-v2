title: Authentification Passwordless avec Universal Login
---

[Universal Login](/fr-CA/docs/authenticate/login/auth0-universal-login) est la mise en œuvre par Auth0 du flux de connexion. Chaque fois qu’un utilisateur doit prouver son identité, vos applications le redirigent vers <Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le serveur d’autorisation d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Universal+Login">Universal Login</Tooltip>, et Auth0 effectue les étapes nécessaires pour garantir l’identité de l’utilisateur. C’est la méthode privilégiée pour mettre en œuvre l’authentification <Tooltip tip="Passwordless : forme d’authentification qui ne repose pas sur un mot de passe comme premier facteur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Passwordless">Passwordless</Tooltip>.

Pour mettre en œuvre l’authentification Passwordless dans Universal Login, vous devez personnaliser le HTML de la page de connexion.

1. Accédez à [Dashboard &gt; Branding &gt; Universal Login](https://manage.auth0.com/#/login_settings), puis cliquez sur **Advanced Options**.

2. Cliquez sur l’onglet **Login**, puis activez la bascule **Customize Login Page**.

<Frame>![Connexion personnalisée](/docs/images/cdy7uua7fh8z/5iwLcz2O100hLuxlKN4rSs/e4f35f8c44516b98b3584d56bcaac12d/manage.auth0.com_dashboard_us_pavo_login_page__2_.png)</Frame>

3. Dans le menu déroulant du modèle par défaut, choisissez le modèle **Custom Login Form**. Le modèle HTML sera mis à jour avec du code utilisant CSS et le SDK Auth0.js.

4. Utilisez HTML et CSS pour personnaliser le modèle selon vos besoins, puis cliquez sur **Save Changes**. Pour en savoir plus sur l’utilisation du SDK Auth0.js avec le modèle **Custom Login Form**, consultez [SDK Auth0.js](/fr-CA/docs/libraries/auth0js).

5. Sélectionnez l’onglet **Preview** pour passer en revue vos modifications. Assurez-vous de sélectionner la bonne application pour laquelle vous souhaitez afficher un aperçu de la page de connexion.

Pour intégrer Universal Login à votre application, consultez nos [Quickstarts](/fr-CA/docs/quickstarts), où vous trouverez des exemples complets pour tous les types d’application et les plateformes populaires.
