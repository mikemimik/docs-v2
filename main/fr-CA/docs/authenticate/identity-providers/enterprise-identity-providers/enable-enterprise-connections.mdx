description: Décrit comment activer les connexions d’entreprise pour les applications à l’aide de l’Auth0 Dashboard.
title: Activer les connexions d’entreprise
---

Vous pouvez activer des connexions d’entreprise pour les applications à l’aide de l’<Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Pour activer votre connexion d’entreprise, vous devez d’abord l’avoir configurée.

1. Accédez à [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), puis sélectionnez le type de connexion à afficher.

   <Frame>![Dashboard - Connexions - Enterprise](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. Sélectionnez le nom de la connexion à afficher.

   <Frame>![Dashboard - Connexions - Enterprise - AD/LDAP - Liste](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/LDAP_page_-_EN.png)</Frame>
3. Sélectionnez la vue **Applications**, puis activez ou désactivez la connexion pour l’application ou les applications appropriées.

   <Frame>![Dashboard - Connexions - Enterprise - Applications](/docs/images/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/38fbae514304297bbbf72beeedb8892f/SSL_-_EN.png)</Frame>

Lorsque vous configurez votre connexion d’entreprise <Tooltip tip="Identity Provider (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=IdP">IdP</Tooltip>, gardez à l’esprit que le paramètre `connection` transmis au point de terminaison `/authorize,` `/whr,` ou `/wsfed` d’Auth0 redirigera vers l’IdP en amont, **sauf** s’il s’agit d’une connexion de base de données ou Active Directory.
