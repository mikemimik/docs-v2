Vous pouvez tester des connexions d’entreprise pour les applications à l’aide du Auth0 Dashboard.

Pour effectuer le test correctement, vous devez avoir déjà [configuré votre connexion d’entreprise](/fr-CA/docs/authenticate/identity-providers/enterprise-identity-providers/enable-enterprise-connections).

1. Accédez à [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), puis sélectionnez le type de connexion à afficher.

   <Frame>![Dashboard - Connexions - Enterprise](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. Sélectionnez la flèche **Try** à côté de la connexion que vous souhaitez tester.

   <Frame>![Dashboard - Connexions - Enterprise - AD/LDAP - Liste](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/oidc-connection-try.png)</Frame>
3. Connectez-vous et donnez votre consentement pour autoriser l’accès à votre application. Si tout est configuré correctement, la page **It Works!** s’affichera.

   <Frame>![Dashboard - Connexions - Database - Try - Réussite](/docs/images/cdy7uua7fh8z/5DWJWLf8zvZJH0rSlhK0cr/f33377edf6f82f17e11e08ec30d8d75a/connection-social-try-success.png)</Frame>
