---
description: Décrit comment voir les connexions activées dans l’Auth0 Dashboard.
title: Voir les connexions
---

Vous pouvez voir toutes les connexions disponibles et configurées dans l’<Tooltip tip="Auth0 Dashboard : le produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Les connexions configurées peuvent être utilisées pour vous connecter à vos applications.

1. Accédez à [Auth0 Dashboard &gt; Authentication](https://manage.auth0.com/#/connections).
2. Sélectionnez un type de connexion :

* [base de données](https://manage.auth0.com/#/connections/database)
* [Social](https://manage.auth0.com/#/connections/social)
* [d’entreprise](https://manage.auth0.com/#/connections/enterprise)
* [Passwordless](https://manage.auth0.com/#/connections/passwordless)
