---
description: Utilisez Auth0 Dashboard pour trouver l’ID d’une connexion Auth0 à utiliser pour configurer des services Auth0, par exemple dans Actions et la gestion des utilisateurs.
title: Repérer l’ID ou le nom de la connexion
---

Certaines tâches de gestion des utilisateurs nécessitent un ID de connexion, c’est-à-dire un identifiant unique pour une source d’utilisateurs. Par exemple, si vous importez ou exportez des utilisateurs, ou si vous vous connectez à des <Tooltip tip="Identity Provider (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=identity+providers">fournisseurs d’identité</Tooltip>, vous devez fournir un ID de connexion.

1. Accédez à [Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez **Authentication**, puis sélectionnez le type de connexion pour lequel vous avez besoin de l’ID.
2. Sélectionnez le nom de la connexion pour laquelle vous avez besoin de l’ID.
3. Dans la vue **Settings**, repérez le champ **Name** ; il contient l’ID de connexion.
