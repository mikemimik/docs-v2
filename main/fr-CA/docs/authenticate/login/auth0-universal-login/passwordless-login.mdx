<Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le Authorization Server d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Universal+Login">
  Universal Login
</Tooltip>

est l’implémentation par Auth0 du flux d’authentification, une fonctionnalité clé d’un

<Tooltip tip="Authorization Server : serveur centralisé qui contribue à définir les limites de l’accès d’un utilisateur. Par exemple, votre serveur d’autorisation peut contrôler les données, les tâches et les fonctionnalités offertes à un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Authorization+Server">
  Authorization Server
</Tooltip>

. Chaque fois qu’un utilisateur doit prouver son identité, vos applications redirigent vers Universal Login, et Auth0 garantit l’identité de l’utilisateur. Vous pouvez choisir d’activer l’authentification

<Tooltip tip="Passwordless : forme d’authentification qui ne repose pas sur un mot de passe comme premier facteur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Passwordless">
  Passwordless
</Tooltip>

lorsque vous configurez Universal Login dans le

<Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">
  Auth0 Dashboard
</Tooltip>

.

Déterminez d’abord si vous voulez que les utilisateurs utilisent SMS, le courriel ou WebAuthn avec la biométrie de l’appareil comme méthode d’authentification Passwordless. Il n’est actuellement pas possible de prendre en charge les trois méthodes d’authentification dans un seul flux de connexion.

<div id="passwordless-with-email-or-sms">
  ## Passwordless par courriel ou SMS
</div>

Vous pouvez configurer Auth0 pour permettre aux utilisateurs de s’authentifier à l’aide d’un Magic Link ou d’un code à usage unique (OTP) envoyé par courriel, ou d’un OTP envoyé à leur numéro de téléphone. Pour en savoir plus, consultez [Configurer le courriel ou le SMS pour l’authentification Passwordless](/fr-CA/docs/authenticate/login/auth0-universal-login/passwordless-login/email-or-sms).

<div id="passwordless-with-webauthn-with-device-biometrics">
  ## Passwordless avec WebAuthn et la biométrie de l’appareil
</div>

Lorsqu’ils utilisent WebAuthn avec biométrie de l’appareil, les utilisateurs peuvent s’authentifier avec leur mot de passe ou avec n’importe lequel de leurs appareils compatibles avec WebAuthn avec biométrie de l’appareil.

Chaque fois qu’ils se connectent à partir d’un nouvel appareil, Auth0 leur proposera d’utiliser la biométrie au lieu d’un mot de passe. Pour en savoir plus, consultez [Passwordless avec WebAuthn avec biométrie de l’appareil](/fr-CA/docs/authenticate/login/auth0-universal-login/passwordless-login/webauthn-device-biometrics).
