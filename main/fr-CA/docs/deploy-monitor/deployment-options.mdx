title: Options de déploiement
---

Auth0 offre des options de déploiement en cloud public et en cloud privé, ainsi que des outils pour vous aider dans votre déploiement en entreprise.

La plateforme d’identité Auth0 peut être déployée des façons suivantes :

| Option                                                                                         | Description                                                                                                                                                                                 |
| ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Public Cloud**                                                                               | Un service infonuagique multilocataire exécuté dans le cloud d’Auth0, avec des fonctionnalités standard d’authentification, de personnalisation, de gestion des utilisateurs et de sécurité |
| [**Private Cloud on AWS**](/fr-CA/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | Un service infonuagique dédié et géré exécuté sur AWS, offrant l’isolation, des performances supérieures, des instances de développement, des modules complémentaires Geo-HA et plus encore |
| [**Private Cloud on Azure**](/fr-CA/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Un service infonuagique dédié exécuté sur Azure, offrant l’isolation, des performances supérieures, des instances de développement, des modules complémentaires Geo-HA et plus encore       |

Les offres **Private Cloud** sont des services gérés que vous pouvez utiliser si vous :

* ne pouvez pas utiliser un service de cloud public multilocataire dans votre organisation
* avez besoin d’un nombre garanti de requêtes par seconde (RPS)

Si vous avez des exigences particulières en matière de soutien ou si vous souhaitez obtenir plus d’information sur les services professionnels que nous offrons, veuillez [contacter Auth0 Sales](https://auth0.com/contact-us).
