---
description: Recommandations pour améliorer votre application Auth0 avant son déploiement en production
title: 'Vérifications de préparation à la production : correctifs non critiques'
---

Les vérifications suivantes permettent de déterminer si vous avez suivi toutes les recommandations (facultatives) pour réussir votre déploiement en Production.

Pour en savoir plus sur les résultats de vos vérifications, consultez [Exécuter les vérifications de production](/fr-CA/docs/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Vérification                                                                                                              | Description                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configurer la diffusion des journaux                                                                                      | Assurez-vous de diffuser les entrées de journal du tenant vers un emplacement de stockage ou un fournisseur externe. Nous vous recommandons de [configurer un service de diffusion de journaux](/fr-CA/docs/customize/log-streams) afin de conserver les journaux pendant plus de 30 jours et de faciliter l&#39;analyse et les alertes.                                                                                           |
| [Custom Domain](/fr-CA/docs/customize/custom-domains) doit être configuré                                                       | Utilisez des [domaines personnalisés avec Universal Login](https://manage.auth0.com/#/tenant/custom_domains) pour offrir à vos utilisateurs finaux une expérience aussi fluide que sécurisée.                                                                                                                                                                                                                                |
| [Custom Error Page](/fr-CA/docs/customize/login-pages/custom-error-pages) doit être configurée                                  | [Configurez une page d&#39;erreur personnalisée](https://manage.auth0.com/#/account) avec les renseignements propres à votre application et l&#39;image de marque de votre entreprise.                                                                                                                                                                                                                                       |
| Activer Universal Login                                                                                                   | Assurez-vous d&#39;utiliser la version la plus récente de [Universal Login](/fr-CA/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login). Nous vous recommandons de faire passer votre page de connexion de l&#39;expérience Classic à Universal Login afin de profiter d&#39;une meilleure accessibilité, de meilleures performances, de la personnalisation sans code et de nouvelles fonctionnalités. |
| [Email Templates](/fr-CA/docs/customize/email/manage-email-flow) doivent être configurés                                        | [Configurez des modèles de courriel personnalisés](https://manage.auth0.com/#/emails) avec les renseignements propres à votre application et l&#39;image de marque de votre entreprise.                                                                                                                                                                                                                                      |
| [Multi-factor Auth](/fr-CA/docs/secure/multi-factor-authentication) doit être configuré                                         | Nous vous recommandons d&#39;envisager l&#39;authentification multifacteur comme pratique sécuritaire pour authentifier vos utilisateurs.                                                                                                                                                                                                                                                                                    |
| [MFA for Tenant Administrators](/fr-CA/docs/get-started/manage-dashboard-access) est activé                                     | [Activez l&#39;authentification multifacteur](https://manage.auth0.com/#/account/admins) pour les administrateurs du tenant.                                                                                                                                                                                                                                                                                                 |
| [Redirect Logout URL](/fr-CA/docs/authenticate/login/logout#set-the-allowed-logout-urls-at-the-account-level) doit être définie | Vérifiez les [Allowed Redirect Logout URLs](https://manage.auth0.com/#/account/advanced) de votre application.                                                                                                                                                                                                                                                                                                               |
| Les applications devraient utiliser RS256 au lieu de HS256 pour l&#39;algorithme de signature des JSON Web Token (JWT)    | Définissez l&#39;[algorithme de signature](/fr-CA/docs/get-started/apis#signing-algorithms) des JSON Web Token (JWT) sur RS256 plutôt que HS256 pour chaque application.                                                                                                                                                                                                                                                           |
| Définir [Application Login URI](/fr-CA/docs/get-started/applications/application-settings#application-uris)                     | Indiquez un [Application Login URI](/fr-CA/docs/get-started/applications/application-settings#application-uris) pour toutes les applications interactives.                                                                                                                                                                                                                                                                         |
| Définir [Tenant Allowed Logout URL](/fr-CA/docs/get-started/applications/application-settings#application-uris)                 | Créez au moins une route de déconnexion accessible au public et [mettez à jour les paramètres de votre tenant](/fr-CA/docs/get-started/applications/application-settings#application-uris) pour qu&#39;ils pointent vers celle-ci.                                                                                                                                                                                                 |
| Définir [Tenant Login URI](/fr-CA/docs/get-started/tenant-settings#login-and-logout)                                            | Publiez une route de connexion par défaut et [mettez à jour les paramètres de votre tenant](/fr-CA/docs/get-started/tenant-settings#login-and-logout) pour qu&#39;ils pointent vers celle-ci.                                                                                                                                                                                                                                      |
