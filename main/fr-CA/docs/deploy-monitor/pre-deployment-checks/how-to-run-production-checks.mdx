title: Exécuter les vérifications de préparation à la production
---

Consultez les résultats des vérifications de production pour votre tenant avant de déployer votre environnement de production.

1. Accédez au tableau de bord de gestion Auth0 et ouvrez une session dans votre compte.
2. Dans la liste déroulante, sélectionnez le tenant pour lequel vous souhaitez exécuter les vérifications de production
3. Dans cette même liste déroulante, cliquez sur **Run Readiness Check** pour ouvrir l’interface

Facultatif :

1. Sélectionnez une ou plusieurs [Applications](/fr-CA/docs/get-started/applications) associées à ce tenant pour lesquelles vous souhaitez exécuter des vérifications en cliquant sur **Filters.**
2. Cliquez sur **Apply**.

Lorsque le test est terminé, l’écran s’actualise automatiquement pour afficher les résultats des vérifications.

<div id="production-check-results">
  ## Résultats des vérifications de Production
</div>

Il existe trois catégories de correctifs pour les résultats de vérification :

* [Correctifs critiques](/fr-CA/docs/deploy-monitor/pre-deployment-checks/production-check-required-fixes)
* [Correctifs non critiques](/fr-CA/docs/deploy-monitor/pre-deployment-checks/production-check-recommended-fixes)
* [Bonnes pratiques](/fr-CA/docs/deploy-monitor/pre-deployment-checks/production-checks-best-practices)

Voici les résultats possibles de votre vérification :

* Réussite
* Échec
* Bonnes pratiques

Sous chaque ensemble de résultats, Auth0 vous indique le nombre de vérifications réussies par votre application, ainsi que le nombre de vérifications à examiner.

Si votre application a **échoué** à une ou plusieurs vérifications, vous verrez :

* Le nom de la vérification
* Des renseignements sur ce que la vérification cherche à détecter
* Un lien **Configure** vers la section appropriée, où vous pouvez apporter les correctifs requis pour que votre Application réussisse la vérification
* Un hyperlien vers la documentation correspondante

Toutes les vérifications que votre application a **réussies** sont regroupées dans l’objet passed de l’ensemble de résultats. Vous pouvez consulter le nom de la vérification et les renseignements qui s’y rapportent, examiner la documentation associée et utiliser l’hyperlien pour accéder à la zone de configuration correspondante afin d’apporter des modifications (au besoin).
