title: 'Vérifications de préparation à la production : Meilleures pratiques'
---

Les vérifications suivantes ne peuvent pas être automatisées; nous vous recommandons donc de vérifier manuellement ces points avant un déploiement en Production.

| Vérification                                                                                             | Description                                                                                                                                                                                                                                                                                                                      |
| -------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Externaliser les [paramètres de configuration](/fr-CA/docs/authenticate/database-connections/custom-db)        | [Externalisez, au lieu de coder en dur, tous les paramètres de configuration](https://manage.auth0.com/#/connections/database), comme les informations d’authentification, les chaînes de connexion, les clés API, etc., lorsque vous développez des Rules, des Hooks ou des connexions de base de données personnalisées.       |
| Vérifier les délais d’expiration du [Single Sign-On (SSO)](/fr-CA/docs/glossary?term=Single+Sign-on+%28SSO%29) | Vérifiez les [valeurs par défaut de délai d’expiration du cookie SSO](https://manage.auth0.com/#/account/advanced) et assurez-vous qu’elles correspondent à vos besoins.                                                                                                                                                         |
| Tenants et administrateurs                                                                               | Passez en revue tous les tenants et leurs administrateurs afin de vous assurer qu’ils sont exacts. Mettez hors service les tenants qui ne sont plus utilisés. Assurez-vous que le rôle d’administrateur de tenant est limité aux utilisateurs qui en ont besoin.                                                                 |
| Vérifier les ID client dans le code de l’application                                                     | Assurez-vous que les ID client dans le code de votre application correspondent à la configuration de leurs Applications Auth0.                                                                                                                                                                                                   |
| Ajouter les IP publiques d’Auth0 à la liste d’autorisation                                               | Ajoutez les IP d’Auth0 à la liste d’autorisation si vous vous connectez à des services internes ou à des services derrière un pare-feu lorsque vous utilisez des Rules, des Hooks ou des bases de données personnalisées. Vous trouverez la liste des adresses IP dans l’infobulle lorsque vous configurez l’un de ces éléments. |
| Vérifier la protection contre les attaques                                                               | Nous vous recommandons d’[examiner la fonctionnalité de protection d’Auth0 et sa configuration](/fr-CA/docs/secure/attack-protection).                                                                                                                                                                                                 |
