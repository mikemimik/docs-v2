description: Répertorie les filtres de journal pour les erreurs, les avertissements et les événements réussis.
title: Filtrer les événements de journal
---

Vous pouvez filtrer les journaux d’erreur, d’avertissement et de réussite dans le [Dashboard](https://manage.auth0.com/#/logs) en cliquant sur la flèche vers le bas **Filter**.
