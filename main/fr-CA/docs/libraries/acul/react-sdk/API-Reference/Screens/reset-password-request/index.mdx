title: "ResetPasswordRequest"
description: "Décrit tous les hooks et toutes les méthodes disponibles pour personnaliser l’écran `reset-password-request` d’Universal Login."
---

L’écran `reset-password-request` s’affiche au début du flux de réinitialisation du mot de passe. Il invite l’utilisateur à saisir son nom d’utilisateur ou son adresse courriel pour recevoir un lien de réinitialisation du mot de passe.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge **l’importation partielle** et **l’importation racine** pour chaque écran.

* L’utilisation de l’importation partielle vous permet d’inclure uniquement le code dont vous avez besoin pour votre cas d’utilisation précis.
* L’utilisation de l’importation racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez qu’une version unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useResetPasswordRequest } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useResetPasswordRequest,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useLoginIdentifiers,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  backToLogin,
  resetPassword,
} from '@auth0/auth0-acul-react/reset-password-request';

function ResetPasswordRequestScreen() {
  const { resetPassword } = useResetPasswordRequest();
  return (
    <button onClick={() => resetPassword({ username: 'user@example.com' })}>
      Send Reset Link
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `reset-password-request`. Importez-les depuis `@auth0/auth0-acul-react/reset-password-request`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit la configuration de l’image de marque, comme le logo, les couleurs et les paramètres du thème affichés sur l’écran `reset-password-request`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-request';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit les configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `reset-password-request`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-request';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si la réinitialisation du mot de passe est limitée à une Organization. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-request';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Aucun contexte d’Organization</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-request';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnResetPasswordRequest">ScreenMembersOnResetPasswordRequest</a></span>}>
  Ce hook contient des détails propres à l’écran `reset-password-request`, notamment sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/reset-password-request';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/reset-password-request';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembersOnResetPasswordRequest">TransactionMembersOnResetPasswordRequest</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `reset-password-request`, comme les connections actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/reset-password-request';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme une adresse courriel ou un nom d’utilisateur prérempli à partir de paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/reset-password-request';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des détails sur l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/reset-password-request';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useResetPasswordRequest" type={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordRequestMembers">ResetPasswordRequestMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte accessibles sur l’écran `reset-password-request`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="backToLogin" type="Promise<void>">
  Cette méthode redirige l’utilisateur vers l’écran de connexion et annule le processus de réinitialisation du mot de passe.

  ```jsx Example
  import { useResetPasswordRequest } from '@auth0/auth0-acul-react/reset-password-request';

  function BackToLoginButton() {
    const { backToLogin } = useResetPasswordRequest();
    return (
      <button onClick={() => backToLogin()}>
        Back to Login
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="resetPassword" type="Promise<void>">
  Cette méthode soumet le nom d’utilisateur ou l’adresse courriel afin de lancer le processus de réinitialisation du mot de passe.

  ```jsx Example
  import { useResetPasswordRequest } from '@auth0/auth0-acul-react/reset-password-request';

  function ResetPasswordForm() {
    const { resetPassword } = useResetPasswordRequest();
    return (
      <button onClick={() => resetPassword({ username: 'user@example.com' })}>
        Send Reset Link
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ResetPasswordRequestOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordRequestOptions).
    </ParamField>

    <ParamField body="username" type="string" required>
      Le nom d’utilisateur ou l’adresse courriel du compte pour lequel le mot de passe doit être réinitialisé.
    </ParamField>

    <ParamField body="captcha?" type="string">
      Le jeton de réponse CAPTCHA, requis lorsque CAPTCHA est activé pour le tenant.
    </ParamField>
  </Expandable>
</ParamField>

  ## Hooks utilitaires généraux

<ParamField body={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useLoginIdentifiers">useLoginIdentifiers</a>} type="Hooks">
  Ce hook renvoie les types d’identifiants de connexion (courriel, nom d’utilisateur, téléphone) configurés pour la connexion actuelle.

<ParamField body={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Ce hook récupère les options du thème actuel, avec la configuration du contexte d’image de marque mise à plat.
</ParamField>

<ParamField body={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Ce hook renvoie une fonction permettant de changer la langue d’affichage sur l’écran ACUL actuel.
</ParamField>

<ParamField body={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Ce hook récupère le contexte et l’état de l’écran actuel.
</ParamField>

<ParamField body={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Ce hook lit et gère les erreurs serveur, client et développeur affichées à l’écran.
</ParamField>
