title: "BruteForceProtectionUnblock"
description: "Décrit tous les hooks et toutes les méthodes permettant de personnaliser l’écran `brute-force-protection-unblock` de Universal Login."
---

L’écran `brute-force-protection-unblock` s’affiche lorsqu’Auth0 bloque le compte d’un utilisateur après un trop grand nombre de tentatives de connexion infructueuses, puis l’invite à vérifier son identité pour rétablir l’accès.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation depuis la racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’importation depuis la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez une build unifiée capable de gérer tous les écrans possibles.

```jsx Import Example
// importation racine
import { useBruteForceProtectionUnblock } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useBruteForceProtectionUnblock,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  unblockAccount,
} from '@auth0/auth0-acul-react/brute-force-protection-unblock';

function UnblockForm() {
  const { unblockAccount } = useBruteForceProtectionUnblock();
  return (
    <button onClick={() => unblockAccount()}>
      Unblock Account
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte d’Auth0 dans l’écran `brute-force-protection-unblock`. Importez-les depuis `@auth0/auth0-acul-react/brute-force-protection-unblock`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les paramètres d’image de marque, comme le logo, les couleurs et les réglages du thème affichés dans l’écran `brute-force-protection-unblock`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `brute-force-protection-unblock`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si le flux de déblocage est limité à une Organization. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnBruteForceProtectionUnblock">ScreenMembersOnBruteForceProtectionUnblock</a></span>}>
  Ce hook contient des détails propres à l’écran `brute-force-protection-unblock`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `brute-force-protection-unblock`, comme les connections actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme un nom d’utilisateur prérempli à partir des paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit les détails de l’utilisateur actif, y compris `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useBruteForceProtectionUnblock" type={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BruteForceProtectionUnblockMembers">BruteForceProtectionUnblockMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte disponibles sur l’écran `brute-force-protection-unblock`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="unblockAccount" type="Promise<void>">
  Cette méthode envoie la requête pour débloquer le compte de l’utilisateur après le déclenchement de la protection contre les attaques par force brute.

  ```jsx Example
  import { useBruteForceProtectionUnblock } from '@auth0/auth0-acul-react/brute-force-protection-unblock';

  function UnblockForm() {
    const { unblockAccount } = useBruteForceProtectionUnblock();
    return (
      <button onClick={() => unblockAccount()}>
        Unblock Account
      </button>
    );
  }
  ```
</ParamField>
