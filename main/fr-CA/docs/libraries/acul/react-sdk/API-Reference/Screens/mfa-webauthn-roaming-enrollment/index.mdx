L’écran `mfa-webauthn-roaming-enrollment` demande à l’utilisateur d’enregistrer une clé de sécurité itinérante, comme une YubiKey ou un autre authentificateur matériel FIDO2, en tant que facteur MFA.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge les **importation partielle** et les **importation racine** pour chaque écran.

* L’utilisation d’une importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’utilisation d’une importation racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous voulez qu’une version unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useMfaWebAuthnRoamingEnrollment } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useMfaWebAuthnRoamingEnrollment,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  enroll,
  showError,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment";

function MfaWebAuthnRoamingEnrollmentScreen() {
  const { enroll } = useMfaWebAuthnRoamingEnrollment();
  return (
    <button onClick={() => enroll()}>
      Enroll Security Key
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `mfa-webauthn-roaming-enrollment`. Importez-les depuis `@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit la configuration d’image de marque, notamment le logo, les couleurs et les paramètres de thème affichés sur l’écran `mfa-webauthn-roaming-enrollment`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `mfa-webauthn-roaming-enrollment`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si le flux MFA est limité à une Organization. Renvoie `null` en l’absence de contexte d’Organization.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt en cours dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaWebAuthnRoamingEnrollment">ScreenMembersOnMfaWebAuthnRoamingEnrollment</a></span>}>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="enroll" type="Promise<void>">
  Cette méthode lance le processus d’inscription de la clé de sécurité itinérante.

  ```jsx Example
  import { useMfaWebAuthnRoamingEnrollment } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';

  function EnrollButton() {
    const { enroll } = useMfaWebAuthnRoamingEnrollment();
    return (
      <button onClick={() => enroll()}>
        Enroll Security Key
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="showError" type="Promise<void>">
  Cette méthode signale une erreur WebAuthn survenue lors de l’inscription de la clé de sécurité.

  ```jsx Example
  import { useMfaWebAuthnRoamingEnrollment } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';

  function HandleWebAuthnError({ error }) {
    const { showError } = useMfaWebAuthnRoamingEnrollment();
    return (
      <button onClick={() => showError({ error })}>
        Report Error
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ShowErrorOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ShowErrorOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      L’objet d’erreur de l’API WebAuthn (`navigator.credentials.create()`) à signaler.

      <ParamField body="name" type="string" required>
        Le nom de l’erreur (p. ex., `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        Le message d’erreur décrivant le problème.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode redirige vers l’écran de sélection de la méthode MFA afin que l’utilisateur puisse choisir un autre facteur d’authentification.

  ```jsx Example
  import { useMfaWebAuthnRoamingEnrollment } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaWebAuthnRoamingEnrollment();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>
