<div id="import">
  ## Importation
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’importation racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez qu’une compilation unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useLogout } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useLogout,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  confirmLogout,
} from '@auth0/auth0-acul-react/logout';

function LogoutScreen() {
  const { confirmLogout } = useLogout();
  return (
    <button onClick={() => confirmLogout({ action: 'accept' })}>
      Sign Out
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `logout`. Importez-les depuis `@auth0/auth0-acul-react/logout`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations de branding, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `logout`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/logout';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `logout`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/logout';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si le logout est limité à l’Organization. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/logout';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/logout';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Ce hook contient des détails propres à l’écran `logout`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/logout';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/logout';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `logout`, comme les connections actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/logout';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme des paramètres provenant de la query string de l’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/logout';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des détails sur l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/logout';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useLogout" type={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/LogoutMembers">LogoutMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte disponibles dans l’écran `logout`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="confirmLogout" type="Promise<void>">
  Cette méthode confirme ou annule la demande de déconnexion pour la session en cours.

  ```jsx Example
  import { useLogout } from '@auth0/auth0-acul-react/logout';

  function LogoutScreen() {
    const { confirmLogout } = useLogout();
    return (
      <>
        <button onClick={() => confirmLogout({ action: 'accept' })}>
          Se déconnecter
        </button>
        <button onClick={() => confirmLogout({ action: 'deny' })}>
          Rester connecté
        </button>
      </>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ConfirmLogoutOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ConfirmLogoutOptions).
    </ParamField>

    <ParamField body="action" type="&#x22;accept&#x22; | &#x22;deny&#x22;" required>
      Indique s’il faut confirmer (`"accept"`) ou annuler (`"deny"`) la déconnexion.
    </ParamField>
  </Expandable>
</ParamField>
