L’écran `mfa-recovery-code-enrollment` affiche un code de récupération que l’utilisateur peut enregistrer pour accéder à son compte s’il perd l’accès à sa méthode MFA principale.

<div id="import">
  ## Importation
</div>

Chaque écran dispose de son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation depuis la racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’importation depuis la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez une compilation unifiée capable de gérer tous les écrans possibles.

```jsx Import Example
// importation racine
import { useMfaRecoveryCodeEnrollment } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useMfaRecoveryCodeEnrollment,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
} from "@auth0/auth0-acul-react/mfa-recovery-code-enrollment";

function MfaRecoveryCodeEnrollmentScreen() {
  const { continueMethod } = useMfaRecoveryCodeEnrollment();
  return (
    <button onClick={() => continueMethod({ isCodeCopied: true })}>
      I've Saved My Recovery Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks associés à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `mfa-recovery-code-enrollment`. Importez-les depuis `@auth0/auth0-acul-react/mfa-recovery-code-enrollment`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit des configurations d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `mfa-recovery-code-enrollment`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `mfa-recovery-code-enrollment`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si le flux MFA est limité à une Organization. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaRecoveryCodeEnrollment">ScreenMembersOnMfaRecoveryCodeEnrollment</a></span>}>
  Ce hook contient des détails propres à l’écran `mfa-recovery-code-enrollment`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function TenantInfo() {
  ```

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  ```

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>


</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode confirme que l’utilisateur a enregistré son code de récupération et poursuit le flux d’inscription.

  ```jsx Example
  import { useMfaRecoveryCodeEnrollment } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';

  function ContinueButton() {
    const { continueMethod } = useMfaRecoveryCodeEnrollment();
    return (
      <button onClick={() => continueMethod({ isCodeCopied: true })}>
        I've Saved My Recovery Code
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaRecoveryCodeEnrollmentContinueOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaRecoveryCodeEnrollmentContinueOptions).
    </ParamField>

    <ParamField body="isCodeCopied" type="boolean" required>
      Indique si l’utilisateur a copié ou enregistré son code de récupération.
    </ParamField>
  </Expandable>
</ParamField>
