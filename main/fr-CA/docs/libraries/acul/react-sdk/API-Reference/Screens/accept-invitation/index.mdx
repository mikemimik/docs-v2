---

L’écran `accept-invitation` s’affiche lorsqu’un utilisateur a été invité à rejoindre une Organization. Il permet à l’utilisateur d’accepter l’invitation et de poursuivre le flux d’authentification.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation à la racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’usage.
* L’importation à la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous voulez qu’une version unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useAcceptInvitation } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useAcceptInvitation,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  acceptInvitation,
} from '@auth0/auth0-acul-react/accept-invitation';

function AcceptInvitationScreen() {
  const { acceptInvitation } = useAcceptInvitation();
  return (
    <button onClick={() => acceptInvitation()}>Accept Invitation</button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks à portée d’écran qui fournissent un accès en lecture seule aux données de contexte d’Auth0 sur l’écran `accept-invitation`. Importez-les depuis `@auth0/auth0-acul-react/accept-invitation`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `accept-invitation`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/accept-invitation';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `accept-invitation`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/accept-invitation';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization que l’utilisateur a été invité à rejoindre. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/accept-invitation';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/accept-invitation';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnAcceptInvitation">ScreenMembersOnAcceptInvitation</a></span>}>
  Ce hook contient des détails propres à l’écran `accept-invitation`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/accept-invitation';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/accept-invitation';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `accept-invitation`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/accept-invitation';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook traite les données non fiables transmises à l’écran, comme les paramètres préremplis provenant de la chaîne de requête de l’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/accept-invitation';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des renseignements sur l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/accept-invitation';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useAcceptInvitation" type={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/AcceptInvitationMembers">AcceptInvitationMembers</a>}>
  Ce hook renvoie toutes les méthodes et tout le contexte disponibles dans l’écran `accept-invitation`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="acceptInvitation" type="Promise<void>">
  Cette méthode envoie l’acceptation par l’utilisateur de l’invitation à l’Organization, puis poursuit le flux d’authentification.

  ```jsx Example
  import { useAcceptInvitation } from '@auth0/auth0-acul-react/accept-invitation';

  function AcceptButton() {
    const { acceptInvitation } = useAcceptInvitation();
    return (
      <button onClick={() => acceptInvitation()}>
        Accept Invitation
      </button>
    );
  }
  ```
</ParamField>
