description: "Décrit tous les hooks et toutes les méthodes disponibles pour personnaliser l’écran `redeem-ticket` dans Universal Login."
---

L’écran `redeem-ticket` s’affiche lorsqu’un utilisateur doit valider un ticket pour poursuivre le flux d’authentification, par exemple pour accepter une invitation ou effectuer la vérification de l’adresse courriel.

<div id="import">
  ## Importation
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation depuis la racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’importation depuis la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous voulez qu’un build unifié prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useRedeemTicket } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useRedeemTicket,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
} from '@auth0/auth0-acul-react/redeem-ticket';

function RedeemTicketScreen() {
  const { continueMethod } = useRedeemTicket();
  return (
    <button onClick={() => continueMethod()}>Continue</button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks associés à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `redeem-ticket`. Importez-les depuis `@auth0/auth0-acul-react/redeem-ticket`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations de branding, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `redeem-ticket`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/redeem-ticket';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `redeem-ticket`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/redeem-ticket';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si l’échange du ticket est associé à une Organization. Il renvoie `null` lorsqu’aucun contexte Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/redeem-ticket';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/redeem-ticket';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Ce hook contient des détails propres à l’écran `redeem-ticket`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/redeem-ticket';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/redeem-ticket';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `redeem-ticket`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/redeem-ticket';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme les paramètres préremplis provenant des chaînes de requête de l’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/redeem-ticket';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des renseignements sur l’utilisateur actif, notamment le `username`, l’`email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/redeem-ticket';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useRedeemTicket" type={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/RedeemTicketMembers">RedeemTicketMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte disponibles dans l’écran `redeem-ticket`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode exécute l’action par défaut à l’écran `redeem-ticket` et poursuit le flux d’authentification une fois le ticket utilisé.

  ```jsx Example
  import { useRedeemTicket } from '@auth0/auth0-acul-react/redeem-ticket';

  function ContinueButton() {
    const { continueMethod } = useRedeemTicket();
    return (
      <button onClick={() => continueMethod()}>
        Continue
      </button>
    );
  }
  ```
</ParamField>
