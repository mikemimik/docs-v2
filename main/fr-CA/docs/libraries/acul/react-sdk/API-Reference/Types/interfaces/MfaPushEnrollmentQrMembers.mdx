    <ParamField body="options" type={<span><a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/type-aliases/MfaPollingOptions">MfaPollingOptions</a></span>}>
      <ParamField body="MfaPollingOptions" type="object">
        Options de configuration pour gérer un processus de polling des notifications push MFA.

        Fournissez ces options au démarrage d’une session de polling pour contrôler la fréquence
        à laquelle le point de terminaison du challenge est interrogé et la façon dont la réussite ou l’échec sont gérés.

        <Expandable title="propriétés">
          <ParamField body="intervalMs?" type="number">
            Intervalle facultatif, en **millisecondes**, entre deux requêtes de polling consécutives.

            * S’il est omis, l’intervalle interne par défaut du SDK est utilisé (habituellement 5000 ms).
            * Choisissez une valeur qui offre un bon équilibre entre la réactivité et les limites de débit.
          </ParamField>

          <ParamField body="onCompleted()?" type="void">
            Fonction de rappel facultative exécutée une fois que le MFA push challenge est
            approuvé avec succès et que le polling est terminé.

            * Appelée exactement **une** fois, après quoi le polling s’arrête automatiquement.
            * Utile pour faire progresser le flux de connexion (par ex., appeler `continue()`).
          </ParamField>

          <ParamField body="onError()?" type="void">
            Fonction de rappel facultative invoquée si une erreur se produit pendant le polling.

            Reçoit un objet [ULError](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error) contenant :

            * `status` — le code d’état HTTP de la requête ayant échoué

            * `responseText` — le corps brut de la réponse, s’il est disponible

            * Appelée une fois par événement d’erreur; le polling peut réessayer selon le type d’erreur.

            * À utiliser pour afficher des messages d’erreur ou déclencher une logique personnalisée de nouvelle tentative ou d’abandon.
