Interface qui définit les méthodes et les propriétés disponibles pour l’écran reset-password-mfa-email-challenge

```ts Example
export interface ResetPasswordMfaEmailChallengeMembers extends BaseMembers {
  screen: ScreenMembersOnResetPasswordMfaEmailChallenge;

  /**
   * Poursuit le défi par courriel en utilisant le code fourni.
   * @param payload Les options contenant le code et l'indicateur rememberDevice.
   */
  continue(payload: ContinueOptions): Promise<void>;

  /**
   * Renvoie le code de courriel.
   * @param payload Options personnalisées facultatives à inclure avec la requête.
   */
  resendCode(payload?: ResendCodeOptions): Promise<void>;

  /**
   * Permet à l'utilisateur d'essayer une autre méthode d'AMF.
   * @param payload Options personnalisées facultatives à inclure avec la requête.
   */
  tryAnotherMethod(payload?: TryAnotherMethodOptions): Promise<void>;

  /**
   * Récupère la fonctionnalité de renvoi avec la gestion du délai d'attente pour cet écran
   * @param options Options de configuration pour la fonctionnalité de renvoi
   */
  resendManager(options?: StartResendOptions): ResendControl;
}
```
