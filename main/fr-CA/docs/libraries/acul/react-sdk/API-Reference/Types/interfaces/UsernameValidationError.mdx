---
title: "UsernameValidationError"
---

Représente une erreur de validation de nom d’utilisateur renvoyée lors de la validation.

Chaque erreur comporte un `code` exploitable par machine et un `message` en langage clair.

```ts Example
export interface UsernameValidationError {
  code: string;
  message: string;
  field: string;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="code" type="string" required>
  Un identifiant unique de la règle de validation qui a échoué.

  Exemple : `"username-too-short"`, `"username-invalid-characters"`
</ParamField>

<ParamField body="message" type="string" required>
  Une description de l’erreur en langage clair.

  Exemple : `"Le nom d’utilisateur doit contenir au moins 3 caractères."`
</ParamField>

<ParamField body="field" type="string" required>
  Le champ associé à l’erreur de validation, généralement &quot;username&quot;, &quot;email&quot; ou &quot;phone&quot;.
</ParamField>
