---
title: "PasswordValidationResult"
---

Le type de retour de [`usePasswordValidation`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation). Contient le résultat global de validité ainsi qu’une liste détaillée des règles de complexité du mot de passe qui ont été évaluées.

```typescript
export interface PasswordValidationResult {
  isValid: boolean;
  results: PasswordComplexityRule[];
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="isValid" type="boolean" required>
  Résultat global de toutes les vérifications de la politique de mot de passe. `true` uniquement si chaque règle de [`PasswordComplexityRule`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) a un `status` égal à `"valid"`.
</ParamField>

<ParamField body="results" type={<span><a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>} required>
  Tableau des règles de mot de passe évaluées. Chaque élément est une [`PasswordComplexityRule`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) dont le champ [`status`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule#status) reflète le résultat de validation calculé pour cette règle.
</ParamField>
