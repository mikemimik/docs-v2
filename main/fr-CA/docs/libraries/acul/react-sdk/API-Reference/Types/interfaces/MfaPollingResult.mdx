Objet résultat renvoyé par [useMfaPolling](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling).

<div id="properties">
  ## Propriétés
</div>

<ParamField body="isRunning" type="boolean">
  Indique si le processus de polling du push MFA est actuellement actif.

  * `true` — Le polling est en cours et en attente de la fin du processus.
  * `false` — Le polling s’est arrêté, soit parce que le processus est terminé,
    soit à la suite d’une annulation manuelle ou du démontage du composant.
</ParamField>

<ParamField body="startPolling()" type="void">
  Démarre ou reprend le processus de polling.

  * Si le polling est déjà actif, cet appel n’a aucun effet.
  * S’il a déjà été arrêté, cet appel redémarre la boucle de polling.
</ParamField>

<ParamField body="stopPolling()" type="void">
  Arrête immédiatement le processus de polling.

  * Annule tous les minuteurs planifiés ou toutes les requêtes en cours.
  * Peut être appelé plusieurs fois sans risque ; les appels suivants n’ont aucun effet.
</ParamField>
