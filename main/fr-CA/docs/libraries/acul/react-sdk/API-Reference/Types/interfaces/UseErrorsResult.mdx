---
title: "UseErrorsResult"
---

Objet de résultat renvoyé par [`useErrors`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Fournit un accès à l’état d’erreur actuel ainsi qu’à des méthodes permettant de supprimer une erreur précise ou toutes les erreurs sur les écrans d’Auth0 Universal Login.

```ts Example
export interface UseErrorsResult {
  errors: ErrorsResult;
  hasError: boolean;
  dismiss: (id: string) => void;
  dismissAll: () => void;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="errors" type={<a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorsResult">ErrorsResult</a>} required>
  La liste des erreurs actuelles, y compris les messages et les identifiants.
</ParamField>

<ParamField body="hasError" type="boolean" required>
  `true` lorsqu’une ou plusieurs erreurs sont présentes; utile pour afficher conditionnellement l’UI d’erreur.
</ParamField>

<ParamField body="dismiss" type="(id: string) => void" required>
  Supprime une erreur précise à l’aide de son `id`.

  <Expandable title="Paramètres">
    <ParamField body="id" type="string">
      L’identifiant de l’erreur à supprimer.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="dismissAll" type="() => void" required>
  Efface toutes les erreurs actives d’un seul coup.
</ParamField>
