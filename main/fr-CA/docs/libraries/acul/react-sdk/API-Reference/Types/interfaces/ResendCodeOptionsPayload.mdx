ResendCodeOptions
description Options pour la méthode `resendCode` à l’écran Login Email Verification.
Cette opération est utilisée lorsque l’utilisateur demande qu’un nouveau code de vérification soit envoyé à son adresse courriel,
par exemple s’il n’a pas reçu le code initial ou si celui-ci a expiré.

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
