<div id="methods">
  ## Méthodes
</div>

* [useAuth0Themes](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes) : Hook React permettant d’obtenir les options du thème actuel avec une configuration aplatie à partir du contexte d’image de marque.
* [useChangeLanguage](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage) : Hook React qui renvoie une fonction permettant de changer la langue d’affichage sur l’écran ACUL actuel.
* [useCurrentScreen](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen) : Hook React qui récupère le contexte d’écran et l’état actuels.
* [useErrors](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) : Hook React pour lire et gérer les erreurs dans ACUL (Advanced Customization of Universal Login). Inclut toutes les erreurs de validation et les erreurs côté serveur. Il regroupe les erreurs en trois types : - `server` — erreurs renvoyées par Auth0 ou votre propre backend. - `client` — erreurs provenant de la validation côté client (par exemple, une saisie de formulaire invalide). - `developer` — erreurs causées par une intégration incorrecte ou une mauvaise utilisation du SDK.
* [useLoginIdentifiers](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useLoginIdentifiers) : Renvoie une liste des types d’identifiant actifs (comme `'email'`, `'phone'` ou `'username'`) actuellement utilisés dans le flux d’authentification ou le processus de connexion.
* [useMfaPolling](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling) : Hook React pour gérer l’interrogation périodique du push MFA (par exemple, en attendant l’approbation d’une notification push) sur un écran Auth0 Advanced Customization of Universal Login (ACUL).
* [useResend](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useResend) : Ce hook React gère les actions de « renvoi » (par exemple, le renvoi d’un code de vérification) sur les écrans ACUL.
* [useSignupIdentifiers](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useSignupIdentifiers) : Renvoie une liste des identifiants activés (courriel, phone ou nom d’utilisateur), chacun avec son état `required`, selon les identifiants d’inscription de l’écran actuel.
* [usePasswordValidation](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation) : Ce hook React valide un mot de passe selon la politique de mot de passe Auth0 actuelle et renvoie un résultat structuré indiquant si le mot de passe respecte chaque règle.
* [useUsernameValidation](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useUsernameValidation) : Hook React permettant de valider un nom d’utilisateur selon la politique de nom d’utilisateur Auth0 actuelle.
