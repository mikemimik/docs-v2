---
title: "Membres de l’utilisateur"
---

```ts Example
export interface UserMembers {
  id: string | null;
  email: string | null;
  username: string | null;
  phoneNumber: string | null;
  picture: string | null;
  enrolledFactors: Array<string> | null;
  enrolledEmails: Array<EnrolledEmail> | null;
  enrolledPhoneNumbers: Array<EnrolledPhoneNumber> | null;
  enrolledDevices: Array<EnrolledDevice> | null;
  organizations: Organizations[] | null; // OPT-IN 
  userMetadata: { [key: string]: string } | null; // OPT-IN
  appMetadata: { [key: string]: string } | null; // OPT-IN
}
```

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * OPT-IN : Vous devez configurer explicitement ces propriétés pour qu’elles soient disponibles à l’écran. Pour en savoir plus, consultez [Configurer ACUL](/fr-CA/docs/customize/login-pages/advanced-customizations/configure).

  * Lorsque le serveur Auth0 ne connaît pas l’utilisateur actuel, les propriétés `user` ne sont pas disponibles dans les écrans suivants :

  | Écrans                            | Écrans                             | Écrans                                                   | Écrans                                              |
  | --------------------------------- | ---------------------------------- | -------------------------------------------------------- | --------------------------------------------------- |
  | Écrans de connexion               | Écrans OTP MFA                     | Écrans Accepter l’invitation                             | Écrans InterstitialCaptcha                          |
  | Écrans d’inscription              | Écrans Téléphone MFA               | Écrans de code d’appareil                                | Écrans de réinitialisation du mot de passe          |
  | Écrans Passkey                    | Écrans Push MFA                    | Écrans de vérification du courriel                       | Écrans des indicatifs de pays MFA                   |
  | Écrans de défi                    | Écrans de code de récupération MFA | Écrans d’abandon de la déconnexion                       | Écrans de détection des capacités du navigateur MFA |
  | Écrans d’identifiant de courriel  | Écrans WebAuthn MFA                | Écrans de fin de déconnexion                             | Écrans d’inscription MFA                            |
  | Écrans d’identifiant de téléphone | Écrans Accepter l’invitation       | Écrans de protection contre les attaques par force brute |                                                     |
</Callout>
