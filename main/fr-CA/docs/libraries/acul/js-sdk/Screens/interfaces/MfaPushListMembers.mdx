---
title: "MfaPushListMembers"
---

Interface qui définit les méthodes et les propriétés disponibles pour l’écran mfa-push-list

```ts Example
export interface MfaPushListMembers extends BaseMembers {
  /**
   * Sélectionne un appareil enregistré dans la liste pour initier un push MFA.
   * @param payload Les options contenant l'index de l'appareil.
   */
  selectMfaPushDevice(payload: SelectMfaPushDeviceOptions): Promise<void>;
  /**
   * Retourne à l'écran précédent.
   * @param payload Options personnalisées facultatives à inclure avec la requête.
   */
  goBack(payload?: CustomOptions): Promise<void>;
}
```
