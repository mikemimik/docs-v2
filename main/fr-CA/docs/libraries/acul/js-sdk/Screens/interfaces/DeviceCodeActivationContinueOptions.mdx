payload pour poursuivre l’activation du code d’appareil.

Cette interface étend `CustomOptions`, ce qui signifie que toutes les propriétés des options personnalisées
sont fusionnées au niveau supérieur de l’objet. Cela évite l’imbrication et offre
une structure plus simple pour les utilisateurs de cette interface.

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```
