<ParamField body="continue" type="Promise<void>">
  Permet à l’utilisateur de poursuivre le flux d’authentification après une inscription WebAuthn réussie.
  Cette méthode envoie une requête POST au point de terminaison `/u/mfa-webauthn-enrollment-success` avec `action: "default"`.

  Une promesse résolue lorsque l’action de continuation est soumise avec succès.
  En cas de réussite, le navigateur est généralement redirigé vers l’étape suivante du flux.

  #### Exceptions

  Lève une erreur si la soumission du formulaire échoue (p. ex., problème de réseau, état invalide).

  ```typescript Example
  // En supposant que 'sdk' est une instance de MfaWebAuthnEnrollmentSuccess
  try {
    await sdk.continue();
    // L’utilisateur est redirigé vers l’étape suivante.
  } catch (error) {
    console.error("Failed to continue after WebAuthn enrollment:", error);
    // Gérez l’erreur, notamment en inspectant sdk.transaction.errors si la page est réaffichée avec une erreur.
  }
  ```
