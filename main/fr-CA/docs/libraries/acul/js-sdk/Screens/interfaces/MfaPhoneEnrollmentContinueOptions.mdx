---
title: "MfaPhoneEnrollmentContinueOptions"
---

Options pour poursuivre l’inscription du téléphone à la MFA.

Étend `CustomOptions` pour permettre de transmettre des options personnalisées avec phone et type.

```ts Example
export interface ContinueOptions extends CustomOptions {
  phone: string;
  type: 'sms' | 'voice';
}
```
