---
title: "UsernameValidationResult"
---

Le résultat de la validation d’un nom d’utilisateur selon une ou plusieurs règles.

```ts Example
export interface UsernameValidationResult {
  /**
   * Indique si le nom d'utilisateur a satisfait à toutes les règles de validation.
   */
  isValid: boolean;

  /**
   * Un tableau d'objets {@link UsernameValidationError} représentant
   * les règles de validation individuelles qui ont échoué, le cas échéant.
   *
   * Ce tableau est vide si {@link UsernameValidationResult.isValid} est `true`.
   */
  errors: UsernameValidationError[];
}
```
