<div id="properties">
  ## Propriétés

<ParamField body="rememberDevice?" type="boolean">
  Lorsqu’il est défini à `true`, cet appareil est enregistré comme appareil de confiance afin que l’utilisateur n’ait pas à effectuer une authentification MFA push lors des connexions ultérieures à partir du même appareil.
</ParamField>
