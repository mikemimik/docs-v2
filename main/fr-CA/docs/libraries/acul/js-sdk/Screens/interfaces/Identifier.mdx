---
title: "Identifiant"
---

Décrit un champ d’identifiant configuré (p. ex. courriel, numéro de téléphone ou nom d’utilisateur)
destiné à être utilisé dans un formulaire d’inscription ou un flux d’authentification.

```ts Example
export interface Identifier {
  /**
   * Le type d'identifiant, tel que `'email'`, `'phone'`, ou `'username'`.
   *
   * @see IdentifierType
   */
  type: IdentifierType;

  /**
   * Indique si cet identifiant est requis pour l'inscription.
   */
  required: boolean;
}
```

<div id="example">
  ## Exemple
</div>

```ts
const emailIdentifier: Identifier = {
  type: 'email',
  required: true
};
```
