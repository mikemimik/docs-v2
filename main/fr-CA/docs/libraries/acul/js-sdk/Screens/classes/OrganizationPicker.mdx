---
title: "OrganizationPicker"
description: "Décrit toutes les propriétés et méthodes offertes pour personnaliser l’écran `organization-picker` de Universal Login."
---

La classe OrganizationPicker fournit la fonctionnalité de l’écran `organization-picker`. Cet écran permet aux utilisateurs de choisir une organisation dans une liste d’organisations disponibles.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran OrganizationPicker :

```typescript Example
import OrganizationPicker from '@auth0/auth0-acul-js/organization-picker';
const organizationPickerManager = new OrganizationPicker();
await organizationPickerManager.selectOrganization({ organization: 'org_XXXXXXXXXXXXXXX' });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page de connexion Universal Login.

  ```typescript Example
  import OrganizationPicker from '@auth0/auth0-acul-js/organization-picker';
  const organizationPickerManager = new OrganizationPicker();
  organizationPickerManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue correspondant à la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette valeur est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="selectOrganization" type="Promise<void>">
  Cette méthode envoie l’ID de l’organisation sélectionnée.

  ```typescript Example
  import OrganizationPicker from '@auth0/auth0-acul-js/organization-picker';
  const organizationPickerManager = new OrganizationPicker();
  await organizationPickerManager.selectOrganization({ organization: 'org_XXXXXXXXXXXXXXX' });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [SelectOrganizationOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/SelectOrganizationOptions).
    </ParamField>

    <ParamField body="organization" type="string" required>
      L’ID de l’organisation à sélectionner.
    </ParamField>

    <ParamField body="state?" type="string">
      Une valeur `state` facultative à inclure dans la requête.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="skipOrganizationSelection" type="Promise<void>">
  Cette méthode envoie l’action permettant d’ignorer la sélection de l’organisation.

  ```typescript Example
  import OrganizationPicker from '@auth0/auth0-acul-js/organization-picker';
  const organizationPickerManager = new OrganizationPicker();
  await organizationPickerManager.skipOrganizationSelection();
  ```
</ParamField>
