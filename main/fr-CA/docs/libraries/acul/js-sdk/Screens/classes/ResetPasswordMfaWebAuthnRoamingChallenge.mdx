La classe ResetPasswordMfaWebAuthnRoamingChallenge implémente les fonctionnalités de l’écran `reset-password-mfa-webauthn-roaming-challenge`. Cet écran invite l’utilisateur à vérifier son identité à l’aide d’une clé de sécurité itinérante (comme une clé USB FIDO2) dans le cadre de l’étape MFA d’un flux de réinitialisation du mot de passe.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran ResetPasswordMfaWebAuthnRoamingChallenge :

```typescript Example
import ResetPasswordMfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/reset-password-mfa-webauthn-roaming-challenge';
const resetPasswordMfaWebAuthnRoamingChallengeManager = new ResetPasswordMfaWebAuthnRoamingChallenge();
try {
  await resetPasswordMfaWebAuthnRoamingChallengeManager.useSecurityKey({
    rememberDevice: true,
  });
} catch (error) {
  if (error instanceof DOMException) {
    await resetPasswordMfaWebAuthnRoamingChallengeManager.showError({
      error: { name: error.name, message: error.message },
    });
  }
}
```

<ParamField body="useSecurityKey" type="Promise<void>">
  Cette méthode lance la vérification par clé de sécurité WebAuthn et soumet les informations d’identification obtenues pour terminer l’étape MFA dans le flux de réinitialisation du mot de passe. Si le navigateur génère une `DOMException` (par exemple, si l’utilisateur annule l’opération), interceptez l’erreur et appelez `showError` pour en aviser Auth0.

  ```typescript Example
  import ResetPasswordMfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/reset-password-mfa-webauthn-roaming-challenge';
  const resetPasswordMfaWebAuthnRoamingChallengeManager = new ResetPasswordMfaWebAuthnRoamingChallenge();
  try {
    await resetPasswordMfaWebAuthnRoamingChallengeManager.useSecurityKey({
      rememberDevice: true,
    });
  } catch (error) {
    if (error instanceof DOMException) {
      await resetPasswordMfaWebAuthnRoamingChallengeManager.showError({
        error: { name: error.name, message: error.message },
      });
    }
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ResetPasswordMfaWebAuthnRoamingChallengeUseSecurityKeyOptions](/fr-CA/docs/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaWebAuthnRoamingChallengeUseSecurityKeyOptions).
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Lorsque `true`, l’appareil est marqué comme fiable afin que l’utilisateur n’ait pas à refaire la vérification MFA sur cet appareil.
    </ParamField>
  </Expandable>
</ParamField>
