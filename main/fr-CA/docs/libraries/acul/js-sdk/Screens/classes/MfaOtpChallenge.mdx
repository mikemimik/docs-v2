title: "MfaOtpChallenge"
description: "Décrit toutes les propriétés et méthodes offertes pour personnaliser l’écran `mfa-otp-challenge` de Universal Login."
---

La classe MfaOtpChallenge implémente les fonctionnalités de l’écran `mfa-otp-challenge`. Cet écran s’affiche lorsqu’un utilisateur doit saisir un code OTP dans le cadre de la MFA.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran MFA OTP Challenge :

```typescript Example
import MfaOtpChallenge from '@auth0/auth0-acul-js/mfa-otp-challenge';

const mfaOtpChallenge = new MfaOtpChallenge();
await mfaOtpChallenge.continue({
  code: '123456',
  rememberDevice: true
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page de connexion Universal Login.

  ```typescript Example
  import MfaOtpChallenge from '@auth0/auth0-acul-js/mfa-otp-challenge';
  const mfaOtpChallenge = new MfaOtpChallenge();
  mfaOtpChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue correspondant à la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque défini à `'session'`, la langue sélectionnée est conservée pendant toute la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Cette méthode poursuit la vérification OTP à l’aide du code fourni.

  ```typescript Example
  import MfaOtpChallenge from '@auth0/auth0-acul-js/mfa-otp-challenge';

  const mfaOtpChallenge = new MfaOtpChallenge();
  await mfaOtpChallenge.continue({
    code: '123456',
    rememberDevice: true
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ContinueOTPOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ContinueOTPOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code OTP saisi par l’utilisateur.
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Indique s’il faut mémoriser ce navigateur.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a pas.
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode permet à l’utilisateur d’essayer une autre méthode MFA.

  ```typescript Example
  import MfaOtpChallenge from '@auth0/auth0-acul-js/mfa-otp-challenge';

  const mfaOtpChallenge = new MfaOtpChallenge();
  await mfaOtpChallenge.tryAnotherMethod();
  ```
</ParamField>
