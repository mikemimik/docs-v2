La classe ResetPasswordMfaSmsChallenge implémente la fonctionnalité de l’écran `reset-password-mfa-sms-challenge`. Cet écran invite l’utilisateur à saisir le code de vérification envoyé à son téléphone par SMS dans le cadre de l’étape MFA du flux de réinitialisation du mot de passe.

  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran ResetPasswordMfaSmsChallenge :

```typescript Example
import ResetPasswordMfaSmsChallenge from '@auth0/auth0-acul-js/reset-password-mfa-sms-challenge';
const resetPasswordMfaSmsChallengeManager = new ResetPasswordMfaSmsChallenge();
await resetPasswordMfaSmsChallengeManager.continueMfaSmsChallenge({
  code: '123456',
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import ResetPasswordMfaSmsChallenge from '@auth0/auth0-acul-js/reset-password-mfa-sms-challenge';
  const resetPasswordMfaSmsChallengeManager = new ResetPasswordMfaSmsChallenge();
  resetPasswordMfaSmsChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue correspondant à la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette valeur est définie à `'session'`, la langue sélectionnée est conservée pendant toute la session.
    </ParamField>
  </Expandable>
</ParamField>

  ```typescript Example
  import ResetPasswordMfaSmsChallenge from '@auth0/auth0-acul-js/reset-password-mfa-sms-challenge';
  const resetPasswordMfaSmsChallengeManager = new ResetPasswordMfaSmsChallenge();
  await resetPasswordMfaSmsChallengeManager.continueMfaSmsChallenge({
    code: '123456',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ResetPasswordMfaSmsChallengeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaSmsChallengeOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code de vérification SMS saisi par l’utilisateur.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="resendCode" type="Promise<void>">

<ParamField body="resendManager" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Cette méthode fournit une fonctionnalité de renvoi avec gestion du délai d’attente. Elle retourne un objet [`ResendControl`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ResendControl) doté d’une méthode `startResend()` qui déclenche le renvoi et impose une période d’attente avant que l’utilisateur puisse renvoyer le code.

  ```typescript Example
  import ResetPasswordMfaSmsChallenge from '@auth0/auth0-acul-js/reset-password-mfa-sms-challenge';
  const resetPasswordMfaSmsChallengeManager = new ResetPasswordMfaSmsChallenge();
  const { startResend } = resetPasswordMfaSmsChallengeManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // Appeler startResend lorsque l’utilisateur clique sur le bouton de renvoi
  startResend();
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [StartResendOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La durée de la période d’attente, en secondes, avant que l’utilisateur puisse renvoyer le code.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Fonction de rappel invoquée chaque fois que l’état du renvoi change; elle retourne le nombre de secondes restantes et indique si la fonctionnalité de renvoi est actuellement désactivée.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Fonction de rappel invoquée lorsque la période d’attente prend fin et que le renvoi redevient possible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
