  ## Envoyer un message Telegram
</div>

Envoyez un message dans une conversation Telegram

<Frame>![](/docs/images/cdy7uua7fh8z/kgSIPsWN5djsT1yQqg5Z8/fea0821d0e919c5101d01b84b968dd53/sendtelegrammessage.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                        | Description                                                                                                             |
| -------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| ID de conversation (obligatoire) | L’identifiant de la conversation Telegram. À obtenir auprès du [BotFather de Telegram](https://core.telegram.org/bots). |
| Messages (obligatoire)           | Le contenu du message Telegram.                                                                                         |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez la réponse attendue et les codes d’état dans la [documentation de Telegram](https://core.telegram.org/method/messages.sendMessage).
