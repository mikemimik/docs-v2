Cette liste d’actions Twilio vous permet d’envoyer des SMS et de passer des appels avec Twilio comme fournisseur.

<div id="configure-your-twilio-vault-connection">
  ## Configurez votre connexion Vault Twilio
</div>

Découvrez comment configurer une connexion Vault pour vos actions de flux Twilio dans [Twilio Vault Connection](/docs/fr-CA/customize/forms/vaults/twilio).

<div id="send-sms">
  ## Envoyer un SMS
</div>

Envoie un SMS.

<Frame>![](/docs/images/cdy7uua7fh8z/4lPqmcYGCJ2oRHM3GSta6w/10b0141d163e4be601c2786895486440/sendsmstwilio.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre             | Description                                                                                   |
| --------------------- | --------------------------------------------------------------------------------------------- |
| De (obligatoire)      | Le nom de l’expéditeur du SMS.                                                                |
| À (obligatoire)       | Le numéro de téléphone de destination au [format E.164](https://en.wikipedia.org/wiki/E.164). |
| Message (obligatoire) | Le contenu du SMS.                                                                            |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez tous les codes d’état attendus dans [la documentation de Twilio](https://www.twilio.com/docs/messaging/api/message-resource#message-status-values).

<div id="make-call">
  ## Passer un appel
</div>

Passe un appel.

<Frame>![](/docs/images/cdy7uua7fh8z/4JaAbJIVuVUJY40lhQuVUN/1a1bd6143c8033c12e18fa49252bd36b/makecalltwilio.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                   | Description                                                                                                                                                       |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| De (obligatoire)            | Votre numéro de téléphone Twilio. L’indicatif du pays doit être inclus. Exemple : +18005550175, où 1 est l’indicatif des États-Unis.                              |
| À (obligatoire)             | Le numéro de téléphone de destination au [format E.164](https://en.wikipedia.org/wiki/E.164).                                                                     |
| Payload TwiML (obligatoire) | Les instructions TwiML pour votre message vocal.<br />Pour en savoir plus, consultez [la documentation TwiML de Twilio](https://www.twilio.com/docs/voice/twiml). |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez tous les codes d’état prévus dans la [documentation de Twilio](https://www.twilio.com/docs/voice/twiml).
