<div id="get-your-sendgrid-keys">
  ## Récupérez vos clés SendGrid
</div>

Vous les trouverez sur la page Paramètres du [SendGrid dashboard](https://app.sendgrid.com/settings/api_keys), dans la section Clés API.

<div id="configure-your-sendgrid-vault-connection">
  ## Configurez votre SendGrid Vault Connection
</div>

Pour configurer une SendGrid Vault Connection pour vos Flow Actions SendGrid, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre | Description                                      |
| --------- | ------------------------------------------------ |
| API Key   | Clé API pour autoriser l’accès à l’API SendGrid. |
