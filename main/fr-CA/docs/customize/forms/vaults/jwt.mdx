<div id="configure-your-jwt-vault-connection">
  ## Configurez votre connexion Vault JWT
</div>

Pour configurer une connexion Vault JWT afin de permettre à vos Actions JWT dans Flows de signer, décoder et vérifier des JSON Web Tokens, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > JWT](/docs/images/customize/forms/vaults/jwt-vault-connection.png)</Frame>

<div id="input-settings">
  ## Paramètres d’entrée
</div>

| Paramètre | Description                                                    |
| --------- | -------------------------------------------------------------- |
| Secret    | Secret à utiliser avec l’algorithme HS256 pour encoder le JWT. |
