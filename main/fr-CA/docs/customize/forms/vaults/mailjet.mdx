<div id="get-your-mailjet-keys">
  ## Obtenez vos clés MailJet
</div>

Trouvez-les sur la page [Compte du Dashboard MailJet](https://app.mailjet.com/account/api_keys), dans la section des clés API.

<div id="configure-your-mailjet-vault-connection">
  ## Configurez votre MailJet Vault Connection
</div>

Pour configurer une MailJet Vault Connection pour vos Flow Actions MailJet, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre   | Description                                          |
| ----------- | ---------------------------------------------------- |
| Clé API     | Clé API pour l’authentification à l’API MailJet.     |
| Clé secrète | Clé secrète pour l’authentification à l’API MailJet. |
