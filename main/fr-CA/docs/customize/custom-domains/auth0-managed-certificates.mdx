title: Configurer des domaines personnalisés avec des certificats gérés par Auth0
---

Si vous voulez qu’Auth0 gère les certificats de votre <Tooltip tip="Domaine personnalisé : domaine tiers avec un nom spécialisé ou personnalisé." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=custom+domain">domaine personnalisé</Tooltip>, vous n’avez qu’à ajouter un enregistrement CNAME pour le domaine. Auth0 valide l’enregistrement, puis génère le certificat sur ses serveurs. Ces certificats sont renouvelés automatiquement tous les trois mois. La configuration est simple et vous n’aurez pas à gérer les certificats vous-même.

Pour configurer votre domaine personnalisé à l’aide de certificats gérés par Auth0, vous devez fournir votre nom de domaine à Auth0 et vérifier que vous en êtes propriétaire. Une fois la vérification effectuée, vous devrez configurer les fonctionnalités Auth0 afin qu’elles utilisent votre domaine personnalisé.

<div id="provide-your-domain-name-to-auth0">
  ## Indiquez votre nom de domaine à Auth0
</div>

1. Allez à [Dashboard &gt; Settings &gt; Custom Domains](https://manage.auth0.com/#/tenant/custom_domains) ou à [Dashboard &gt; Branding &gt; Custom Domains](https://manage.auth0.com/#/custom_domains). Entrez votre domaine personnalisé dans le champ prévu à cet effet, puis sélectionnez **certificats gérés par Auth0**.
2. Cliquez sur **Add Domain**.Vous ne pouvez ajouter qu’un seul domaine par tenant, même si le bouton **Add Domain** s’affiche toujours après l’ajout d’un domaine.

<div id="verify-ownership">
  ## Vérifier la propriété
</div>

Avant de pouvoir utiliser le domaine avec Auth0, vous devrez vérifier que vous en êtes bien propriétaire.

1. Accédez à [Dashboard &gt; Branding &gt; Custom Domains](https://manage.auth0.com/#/branding/custom_domains) et ajoutez l’enregistrement de vérification CNAME indiqué dans le Dashboard à l’enregistrement DNS de votre domaine.

   <Frame>![Auth0 Branding Custom Domains Auth0-Managed Certificate Verify Domain](/docs/images/cdy7uua7fh8z/3dz8ii70nm0a5amS0PNJ5K/c7b58043042293f5502ec012901f5dac/Custom_Domain_-_Pending_Verification_-_English.png)</Frame>
2. Cliquez sur **Verify** pour continuer.
   Selon vos paramètres DNS, Auth0 pourrait prendre quelques minutes avant de pouvoir vérifier votre enregistrement CNAME. Si Auth0 réussit à vérifier votre nom de domaine, une fenêtre de confirmation s’affichera. Cela signifie que le processus de vérification est terminé. Dans un délai de 1 à 2 minutes, votre domaine personnalisé devrait être prêt à être utilisé.

<div id="add-cname-verification-record-to-dns-record">
  ## Ajouter l’enregistrement de vérification CNAME à l’enregistrement DNS
</div>

Une fois ajouté, l’enregistrement CNAME doit rester présent en tout temps afin d’éviter des problèmes lors du renouvellement du certificat.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 recommande de désactiver l’aplatissement CNAME, sauf si cela est strictement nécessaire, car l’aplatissement CNAME pour les certificats gérés par Auth0 est une configuration non prise en charge et peut faire cesser de fonctionner le domaine personnalisé sans préavis.
</Callout>

Si vous devez activer l’aplatissement CNAME pour tous les sous-domaines gérés par Cloudfare, tout en configurant un sous-domaine précis comme domaine personnalisé Auth0, envisagez de [déléguer le sous-domaine](https://developers.cloudflare.com/dns/manage-dns-records/how-to/subdomains-outside-cloudflare/) à un autre fournisseur DNS pour Auth0.
Vous pourrez ainsi utiliser l’aplatissement CNAME pour tous les sous-domaines, sauf celui utilisé pour Auth0.

Les étapes suivantes peuvent varier selon votre fournisseur de services de domaine.

1. Ouvrez une session dans votre service de gestion de domaine.
2. Créez un nouvel enregistrement.

   | Paramètre                 | Valeur                                                                                               |
   | ------------------------- | ---------------------------------------------------------------------------------------------------- |
   | **Type d’enregistrement** | **CNAME**                                                                                            |
   | **Nom**                   | Entrez le nom de votre domaine personnalisé (par exemple **login.northwind.com**).                   |
   | **Durée de vie (TTL)**    | Utilisez la valeur par défaut.                                                                       |
   | **Valeur**                | Collez la valeur **CNAME** fournie par l’Auth0 Dashboard pour l’enregistrement DNS de votre domaine. |
3. Lorsque vous avez terminé, enregistrez l’enregistrement.

Si Auth0 a pu vérifier votre nom de domaine, une fenêtre de confirmation s’affichera. Cela signifie que le processus de vérification est terminé. Dans un délai de 1 à 2 minutes, votre domaine personnalisé devrait être prêt à être utilisé.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Si votre fournisseur DNS active un proxy sur l’enregistrement CNAME par défaut, le domaine personnalisé restera indéfiniment en attente. Vous devrez peut-être vérifier les paramètres de votre fournisseur DNS et désactiver le proxy.
</Callout>

Si vous ne parvenez pas à terminer le processus de vérification, attendez au moins 4 heures avant de reprendre ces étapes. Pour éviter toute interruption de votre service, ne recréez pas le domaine personnalisé au moment de refaire la vérification.

  ## Étapes supplémentaires pour certaines fonctionnalités Auth0
</div>
