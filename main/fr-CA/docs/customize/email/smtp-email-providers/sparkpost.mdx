<div id="prerequisites">
  ## Prérequis
</div>

* Vous devez avoir un compte [SparkPost](https://www.sparkpost.com/).

* Vous devez configurer votre fournisseur pour autoriser les connexions entrantes à partir d’[adresses IP Auth0](/fr-CA/docs/secure/security-guidance/data-security/allowlist) précises.

<div id="configure-sparkpost">
  ## Configurer SparkPost
</div>

1. Connectez-vous à votre compte [SparkPost](https://www.sparkpost.com/).
2. Accédez à la section [Sending Domains](https://app.sparkpost.com/account/sending-domains) de SparkPost, puis ajoutez votre domaine personnalisé. SparkPost permet d’envoyer des courriels uniquement à partir de domaines vérifiés.
3. Accédez à la section [Account API Keys](https://app.sparkpost.com/account/credentials) de SparkPost, puis cliquez sur **New API key**. Enregistrez la valeur de cette clé et assurez-vous qu’elle dispose de l’accès `Transmissions: Read/Write`. Copiez cette valeur.
4. Accédez à [Dashboard &gt; Branding &gt; fournisseur de courriel](https://manage.auth0.com/#/branding/email_provider) dans Auth0.
5. Activez la bascule **Use my own fournisseur de courriel**.
6. Cliquez sur le logo **SparkPost**.
7. Indiquez une adresse courriel **From**, entrez la **API Key** SparkPost que vous avez copiée précédemment, puis sélectionnez votre **Region**. Si vous passez à la région EU dans un compte déjà configuré pour la région US, vous devez remplacer la **API Key** dans Auth0 par une clé API SparkPost EU.
8. Cliquez sur **Save**.

<div id="send-a-test-email">
  ## Envoyer un courriel de test
</div>

Envoyez un courriel de test à l’aide du bouton **Send Test Email**. Si la configuration est correcte, vous recevrez un courriel de confirmation. Si vous ne recevez aucun courriel après quelques minutes, consultez vos [journaux Auth0](https://manage.auth0.com/#/logs) pour vérifier les codes d’erreur.

Si vous n’avez pas enregistré de <Tooltip tip="Domaine personnalisé : domaine tiers avec un nom spécialisé ou personnalisé." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=custom+domain">domaine personnalisé</Tooltip> auprès de SparkPost, vous pouvez envoyer un maximum de cinq courriels de test depuis le domaine `sparkpostbox.com`. Pour en savoir plus, consultez la page de SparkPost [Transmissions: The Sandbox Domain](https://developers.sparkpost.com/api/transmissions.html#header-the-sandbox-domain).

Une fois la configuration terminée, la page SparkPost [Message Events](https://app.sparkpost.com/reports/message-events) affiche des renseignements sur la livraison de tous les courriels envoyés à vos utilisateurs.
