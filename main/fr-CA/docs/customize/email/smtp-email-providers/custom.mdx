---
title: Configuration de Custom Email Provider avec Actions
description: Écrivez une Action pour envoyer des messages par l’entremise de votre fournisseur de courriel et mettre en œuvre une logique personnalisée pour votre cas d’utilisation.
---

Pour personnaliser davantage la configuration de votre fournisseur de courriel, vous pouvez configurer votre tenant avec une <Tooltip tip="Action : une fonction Node.js qui s’exécute à des points définis pendant le runtime Auth0." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Action">Action</Tooltip>.

Au lieu d’utiliser notre fournisseur de courriel ou nos intégrations de serveur SMTP, vous écrivez le code de votre Action pour envoyer des messages par l’entremise du fournisseur de courriel ou du serveur SMTP de votre choix.

Vous avez ainsi davantage de contrôle sur le processus d’envoi des courriels et pouvez mettre en œuvre une logique personnalisée pour votre cas d’utilisation, par exemple pour réessayer après un échec, modifier les destinataires ou créer une logique propre à l’Organization.
