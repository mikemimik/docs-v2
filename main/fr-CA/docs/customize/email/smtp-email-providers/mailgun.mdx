<div id="prerequisites">
  ## Prérequis
</div>

* Vous devez avoir un compte [Mailgun](https://mailgun.com).

* Vous devez configurer votre fournisseur pour autoriser les connexions entrantes à partir de certaines [adresses IP d’Auth0](/fr-CA/docs/secure/security-guidance/data-security/allowlist).

<div id="configure-mailgun">
  ## Configurer Mailgun
</div>

1. Connectez-vous à votre compte [Mailgun](https://mailgun.com).
2. Dans Mailgun, accédez à **Sending &gt; Domains**, puis ajoutez votre domaine personnalisé. Mailgun permet d’envoyer des courriels uniquement à partir de domaines vérifiés.
3. Dans Mailgun, accédez à **Settings &gt; API Keys**. Votre **API Key** a été créée lors de l’inscription à votre compte; copiez-la à partir de **Private API Key**.
4. Dans Auth0, accédez à [Dashboard &gt; Branding &gt; fournisseur de courriel](https://manage.auth0.com/#/branding/email_provider).
5. Activez la bascule **Use my own fournisseur de courriel**.
6. Cliquez sur le logo **Mailgun**.
7. Indiquez une adresse courriel **From**, saisissez le **Domain** que vous avez déjà ajouté à Mailgun, entrez l’**API Key** Mailgun que vous avez copiée précédemment, puis sélectionnez votre **Region**.
8. Cliquez sur **Save**.

<div id="send-a-test-email">
  ## Envoyer un courriel de test
</div>

Envoyez un courriel de test à l’aide du bouton **Send Test Email**. Si la configuration est correcte, vous recevrez un courriel de confirmation. Si vous ne recevez pas de courriel après quelques minutes, consultez vos [logs Auth0](https://manage.auth0.com/#/logs) pour y repérer les codes d’erreur.

Une fois la configuration effectuée, le [Dashboard de Mailgun](https://app.mailgun.com/app/dashboard) affiche des renseignements sur la livraison de tous les courriels envoyés à vos utilisateurs.
