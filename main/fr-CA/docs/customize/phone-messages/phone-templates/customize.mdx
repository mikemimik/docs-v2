<Warning>
  Pour personnaliser les modèles de messages téléphoniques, vous devez [configurer un fournisseur de téléphonie externe](../configure-phone-messaging-providers).
</Warning>

<div id="customize-phone-templates">
  ## Personnaliser les modèles de messages téléphoniques
</div>

Pour personnaliser un modèle de message téléphonique :

1. Accédez à [**Dashboard &gt; Branding &gt; Phone Templates**](https://manage.auth0.com/#/phone).

2. Dans la liste déroulante **Template**, sélectionnez le modèle de message téléphonique que vous souhaitez mettre à jour.

3. Activez le curseur **Status** pour activer le template s’il ne l’est pas déjà.

4. Mettez à jour le numéro de téléphone **From** et/ou le **modèle de message texte**.

5. Sélectionnez **Save** pour enregistrer vos modifications, **Try** pour les tester ou **Reset** pour les annuler.

<div id="from-phone-number">
  ### Numéro de téléphone d’envoi
</div>

<Warning>
  Si vous utilisez les [Twilio Messaging Services](https://www.twilio.com/docs/messaging/services) comme source SMS, vous ne pouvez pas définir de numéro de téléphone **From** personnalisé. Twilio choisit plutôt un numéro dans le pool d’expéditeurs de votre Messaging Service, en fonction de votre configuration Twilio.
</Warning>

Le champ **From** définit le numéro de téléphone à partir duquel les messages texte sont envoyés. Par défaut, le numéro de téléphone **From** est celui que vous avez défini lors de la configuration de votre fournisseur de services téléphoniques.

Pour définir un numéro de téléphone **From** personnalisé pour un modèle de message téléphonique, saisissez le numéro de téléphone dans ce champ.

<div id="text-message-template">
  ### Modèle de message texte
</div>

Le champ **Modèle de message texte** définit le contenu du message texte. Chaque modèle est fourni avec un corps de message par défaut que vous pouvez modifier ou supprimer complètement pour rédiger le vôtre.

Ce champ prend en charge la syntaxe Liquid pour générer du contenu dynamique. Pour en savoir plus, consultez [Syntaxe Liquid prise en charge pour les modèles de SMS](./supported-liquid-syntax).

<div id="test-updated-templates">
  ## Tester les modèles mis à jour
</div>

Pour tester un modèle de message téléphonique, cliquez sur **Try**. Auth0 envoie le message téléphonique à une application par défaut portant le nom de votre tenant. Pour tester des modèles pour différentes applications, créez un utilisateur test pour parcourir les flux de messages téléphoniques.

Vous pouvez déclencher manuellement des e-mails de vérification pour des applications et des utilisateurs précis à l’aide de l’endpoint [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) de la Management API.
