Les courriels envoyés dans le cadre des workflows Auth0 ont chacun un email template correspondant qui détermine l’adresse d’expéditeur, l’objet, le corps du message, et plus encore.

Après avoir [configuré un phone provider externe](../phone-messages/configure-phone-messaging-providers), vous pouvez modifier les modèles de téléphone pour personnaliser vos messages téléphoniques.

Les modèles de messages texte prennent en charge [Liquid](https://liquidjs.com/index.html), un langage de gabarits open source qui étend les fonctionnalités du HTML. Liquid permet de :

* remplir dynamiquement des variables avec des données, par exemple en remplaçant `{{ user.name }}` par le nom de l’utilisateur
* implémenter une logique, comme des instructions `if`/`else`
* modifier des données à l’aide de filtres pour manipuler des chaînes de caractères et des tableaux, appliquer des opérateurs mathématiques, et plus encore

Vous pouvez activer, désactiver et personnaliser les modèles de téléphone sur la [page des modèles de téléphone du Dashboard](https://manage.auth0.com/#/phone).

<div id="available-phone-templates">
  ## Modèles de téléphone disponibles
</div>

Voici un résumé des modèles de téléphone disponibles :

| Nom du modèle de téléphone                 | Description                                                                                         |
| ------------------------------------------ | --------------------------------------------------------------------------------------------------- |
| Compte bloqué                              | Avertit le client que son compte a été bloqué à la suite d’une activité suspecte.                   |
| Changement de mot de passe                 | Envoie au client son code de changement de mot de passe.                                            |
| Mot de passe à usage unique - Activation   | Envoie au client le code de vérification pour l’activation du mot de passe à usage unique (OTP).    |
| Mot de passe à usage unique - Vérification | Envoie au client le code de vérification pour la vérification du mot de passe à usage unique (OTP). |
