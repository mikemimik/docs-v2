L’objet `event` du déclencheur event-stream d’Actions fournit des renseignements contextuels sur l’exécution du déclencheur.

<div id="eventmessage">
  ## `event.message`
</div>

<ResponseField name="event.message" type="dictionary">
  Le message CloudEvent qui contient toutes les propriétés de l’événement.

  <Expandable title="propriétés du message" defaultOpen>
    <ResponseField name="id" type="string">
      Identifie l’événement.
    </ResponseField>

    <ResponseField name="type" type="string">
      Décrit le type d’événement lié à l’événement d’origine.
    </ResponseField>

    <ResponseField name="data" type="dictionary" post={["optional"]}>
      La charge utile de l’événement.
    </ResponseField>

    <ResponseField name="source" type="string">
      Identifie le contexte dans lequel l’événement s’est produit.
    </ResponseField>

    <ResponseField name="specversion" type="string">
      La version de la spécification CloudEvents utilisée par l’événement.
    </ResponseField>

    <ResponseField name="time" type="string" post={["optional"]}>
      Horodatage indiquant quand l’événement s’est produit. Doit être conforme à la RFC 3339.
    </ResponseField>
  </Expandable>
</ResponseField>
