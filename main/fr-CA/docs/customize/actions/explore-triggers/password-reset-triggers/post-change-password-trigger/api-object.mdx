L’objet API du déclencheur post-change-password d’Actions comprend :

<div id="apicache">
  ## `api.cache`
</div>

Enregistrez et récupérez des données qui persistent d’une exécution à l’autre.

<div id="apicachedeletekey">
  ### `api.cache.delete(key)`
</div>

Supprime l’enregistrement décrivant une valeur mise en cache associée à la clé fournie, s’il existe.

Retourne un objet `CacheWriteResult` avec `type: "success"` si une valeur a été supprimée du cache. Si l’opération échoue, retourne `type: "error"`. En cas d’erreur, l’objet retourné comporte une propriété
`code` indiquant la nature de l’échec.

| Paramètre | Description                                                |
| --------- | ---------------------------------------------------------- |
| `key`     | *String*. La clé de l’enregistrement stocké dans le cache. |

<div id="apicachegetkey">
  ### `api.cache.get(key)`
</div>

Récupère un enregistrement décrivant une valeur mise en cache associée à la
clé `key` fournie, s’il existe. Si un enregistrement est trouvé, la valeur mise en cache se
trouve dans la propriété `value` de l’objet renvoyé.

Renvoie un enregistrement du cache si un élément est trouvé dans le cache pour la
clé `key` fournie. Les enregistrements du cache sont des objets dotés d’une
propriété `value` contenant la valeur mise en cache, ainsi que d’une
propriété `expires_at` indiquant la date d’expiration maximale de
l’enregistrement, en millisecondes depuis l’époque Unix.

**Important :**
Ce cache est conçu pour des données éphémères de courte durée. Il est possible que des éléments ne soient pas
disponibles dans des transactions ultérieures, même s’ils sont encore dans leur
durée de vie.

| Paramètre | Description                                                |
| --------- | ---------------------------------------------------------- |
| `key`     | *String*. La clé de l’enregistrement stocké dans le cache. |

<div id="apicachesetkey-value-options">
  ### `api.cache.set(key, value, [options])`
</div>

Stocke ou met à jour une valeur de type chaîne dans le cache à la clé indiquée.

Les valeurs stockées dans ce cache sont limitées au déclencheur dans lequel elles sont définies.
Elles sont assujetties aux
[Actions Cache Limits](/fr-CA/docs/customize/actions/limitations).

Les valeurs stockées de cette façon ont une durée de vie pouvant aller jusqu&#39;aux valeurs
`ttl` ou `expires_at` spécifiées. Si aucune durée de vie n&#39;est
spécifiée, une durée de vie par défaut de 15 minutes sera utilisée. Les durées de vie ne peuvent
pas dépasser la durée maximale indiquée dans
[Actions Cache Limits](/fr-CA/docs/customize/actions/limitations).

| Paramètre            | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `key`                | *String*. La clé de l&#39;enregistrement stocké dans le cache.                                                                                                                                                                                                                                                                                                                                                                                                  |
| `value`              | *String*. La valeur de l&#39;enregistrement à stocker.                                                                                                                                                                                                                                                                                                                                                                                                          |
| `options`            | *Objet optionnel*. Options permettant d&#39;ajuster le comportement du cache.                                                                                                                                                                                                                                                                                                                                                                                   |
| `options.expires_at` | *Nombre optionnel*. L&#39;heure d&#39;expiration absolue en millisecondes depuis l&#39;époque Unix. Bien que les enregistrements mis en cache puissent être évincés plus tôt, ils ne seront jamais conservés au-delà de la valeur `expires_at` fournie.<br />*Remarque :* Cette valeur ne doit pas être fournie si une valeur a également été fournie pour `ttl`. Si les deux options sont fournies, la date d&#39;expiration la plus rapprochée sera utilisée. |
| `options.ttl`        | *Nombre optionnel*. La durée de vie de cette entrée de cache, en millisecondes. Bien que les valeurs mises en cache puissent être évincées plus tôt, elles ne seront jamais conservées au-delà de la valeur `ttl` fournie.<br />*Remarque :* Cette valeur ne doit pas être fournie si une valeur a également été fournie pour `expires_at`. Si les deux options sont fournies, la date d&#39;expiration la plus rapprochée sera utilisée.                       |
