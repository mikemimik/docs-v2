L’intégration native d’Auth0 à Vercel vous permet de connecter votre plateforme d’identité à votre projet Vercel avec un minimum de configuration. Configurez et personnalisez votre fenêtre de connexion sans avoir à la créer de zéro. Ensuite, étendez et faites évoluer vos flux d’authentication et d’Authorization grâce à des fonctionnalités avancées, notamment [Organizations B2B](/fr-CA/docs/manage-users/organizations), [Fine-Grained Authorization (FGA)](https://docs.fga.dev/), [Auth0 for AI Agents](https://auth0.com/ai/docs), et plus encore.

L’intégration native d’Auth0 à Vercel permet :

* le Provisioning automatique d’une application Auth0 connectée à votre projet Vercel;
* le préchargement de vos credentials Auth0 dans le Vercel Dashboard pour une configuration simple et rapide;
* la prise en charge prête à l’emploi de vos applications Next.js grâce aux [Auth0 SDKs](/fr-CA/docs/libraries).

L’intégration native d’Auth0 à Vercel crée un nouvel environnement de tenant Auth0 pour votre projet Vercel. Si vous souhaitez utiliser un compte Auth0 existant, intégrez Vercel manuellement lors de la création d’une application dans l’Auth0 Dashboard.

<div id="prerequisites">
  ## Prérequis
</div>

Avant de commencer, assurez-vous d’avoir :

* un compte Vercel avec un projet actif. Au besoin, [créez un compte Vercel](https://vercel.com/signup)
* une application Next.js qui utilise la dernière version du [Auth0 Next.js SDK](/fr-CA/docs/quickstart/webapp/nextjs)

Après avoir installé l’intégration, [activez Iframe embedding](/fr-CA/docs/customize/login-pages/manage-core-configuration#iframe-embedding) pour permettre à Universal Login ou à Classic Login de se charger dans un iframe, comme l’exige Vercel.

<div id="install-integration">
  ## Installer l’intégration
</div>

Installez l’intégration native de Vercel pour Auth0 depuis le Marketplace de Vercel :

1. Accédez à **Integrations &gt; Browse Marketplace** et recherchez Auth0. Sélectionnez l’**intégration Auth0** sous **Native Integrations**, puis sélectionnez **Install**.

2. Auth0 renseigne automatiquement l’URL localhost et l’URL de rappel de votre application. Sélectionnez votre **plan d’installation**, puis sélectionnez **Continue**.

3. Entrez un nom pour votre application. Sélectionnez **Create**. Une fois le nouveau tenant Auth0 et l’application créés avec succès, sélectionnez **Continue**.

4. Pour vous connecter à un projet Vercel existant, sélectionnez **Connect Project**, puis choisissez votre projet et vos environnements. Entrez un préfixe personnalisé à ajouter à vos variables d’environnement, au besoin. Sélectionnez **Connect**. Une fois votre projet Vercel connecté avec succès, vous devriez être redirigé vers la section [Premiers pas](#getting-started) de l’intégration.

<div id="getting-started">
  ## Premiers pas
</div>

La section Premiers pas du Vercel Dashboard affiche un Quickstart pour mettre votre application en service rapidement et en toute sécurité. Vous pouvez aussi gérer les paramètres de votre application dans le menu de navigation de gauche.

<div id="quickstart">
  ### Quickstart
</div>

Pour commencer à utiliser l’intégration native Vercel d’Auth0, suivez les instructions dans le Vercel Dashboard pour créer et déployer votre application.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Vous pouvez uniquement configurer les variables d’environnement d’Auth0 dans un environnement de production.
</Callout>

<div id="manage-application-settings">
  ## Gérer les paramètres de l’application
</div>

Vous pouvez gérer les paramètres de votre application dans le [Vercel Dashboard](#vercel-dashboard) et l’[Auth0 Dashboard](#auth0-dashboard).

<div id="vercel-dashboard">
  ### Vercel Dashboard
</div>

Pour gérer les paramètres de votre application dans le Vercel Dashboard, accédez à **Settings** dans le menu de navigation de gauche.

Vous pouvez mettre à jour les paramètres d’application suivants :

* **Mettre à jour le nom** : mettez à jour le nom de l’application.
* **Mettre à jour la Configuration** :
  * **Parameters** : mettez à jour l’URL localhost et le point de terminaison de rappel de votre application.
  * **Forfait actuel de l’installation** : accédez aux paramètres de l’intégration pour mettre à jour le plan au niveau de l’installation de votre intégration. Pour en savoir plus, consultez [Settings](#settings).
* **Sécuriser cette ressource**
  * **Faire la rotation des secrets** : faites la rotation des secrets de votre application. Pour en savoir plus, consultez [Rotating environment variables](https://vercel.com/docs/projects/environment-variables/managing-environment-variables#rotating-environment-variables).
  * **Environnements autorisés** : choisissez si l’intégration est disponible dans **Tous les environnements** ou **l’environnement Production seulement**.
* **Supprimer l’intégration** : lorsque vous supprimez l’intégration, vous passez automatiquement au forfait Free de Vercel. Pour en savoir plus, consultez [Supprimer l’intégration](#delete-integration).

<div id="auth0-dashboard">
  ### Auth0 Dashboard
</div>

Gérez des paramètres d’application supplémentaires dans l’[Auth0 Dashboard](https://manage.auth0.com/).

Lorsque vous installez l’intégration native, Auth0 crée une nouvelle application dans un tenant dédié, préremplie avec vos URI Vercel, les URI de rappel autorisés et les URL de déconnexion.

Vous pouvez modifier cette application, par exemple en personnalisant votre expérience Universal Login, comme vous le feriez avec n’importe quelle autre application dans votre Auth0 Dashboard.

<div id="manage-integration">
  ## Gérer l’intégration
</div>

Pour gérer l’intégration dans votre Vercel Dashboard, sélectionnez l’onglet **Integration** de votre projet, faites défiler la page jusqu’à l’intégration Auth0, puis sélectionnez **Manage**.

<div id="invoices">
  ### Factures
</div>

Sur la page **Factures**, consultez la liste des factures associées à votre intégration Auth0. Pour en savoir plus sur les factures, consultez [Billing &amp; Invoices](https://vercel.com/docs/pricing/understanding-my-invoice).

<div id="settings">
  ### Paramètres
</div>

Sur la page **Paramètres**, ajoutez des modes de paiement, modifiez le forfait de l’installation et supprimez l’intégration.

<div id="payment-methods">
  #### Modes de paiement
</div>

Pour ajouter un mode de paiement, sélectionnez **Ajouter**. Pour en savoir plus sur les modes de paiement, consultez [les tarifs de Vercel](https://vercel.com/docs/accounts/plans).

<div id="current-installation-level-plan">
  #### Forfait actuel de l’installation
</div>

Le forfait de l’installation s’applique à tous les produits de l’installation.

Pour modifier le forfait de l’installation :

1. Sous **Paramètres**, faites défiler jusqu’à **Forfait actuel de l’installation** et sélectionnez **Changer de forfait**.
2. Sélectionnez le forfait de l’installation souhaité.
3. Sélectionnez **Continuer**.
4. Vérifiez votre sélection, puis sélectionnez **Mettre à jour**.
5. Une fois l’installation mise à jour, sélectionnez **Terminé**.

<div id="delete-integration">
  #### Supprimer l’intégration
</div>

Lorsque vous supprimez l’intégration, vous passez automatiquement au forfait Free de Vercel.

Pour supprimer l’intégration :

1. Sous **Paramètres**, faites défiler jusqu’à **Supprimer &lt;YOUR&#95;PRODUCT&gt;** et sélectionnez le bouton **Supprimer &lt;YOUR&#95;PRODUCT&gt;**.
2. Saisissez le slug de l’intégration et votre équipe, puis sélectionnez **Je reconnais que cette action supprimera mon compte connecté à Auth0**.
3. Sélectionnez **Supprimer &lt;YOUR&#95;PRODUCT&gt;**.
