title: Exporter des données utilisateur vers Oracle Eloqua
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans Eloqua à l’aide du Contact Upload Wizard.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par aller à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Dans la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** au format de fichier requis. Eloqua accepte l’importation de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | Adresse courriel |
| `created_at`      | Date de création |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, vous pouvez obtenir plus d’informations en consultant le [Oracle Eloqua Help Center: Uploading Contacts](https://docs.oracle.com/en/cloud/saas/marketing/eloqua-user/Help/Contacts/Tasks/UploadingContacts.htm).

Pour importer votre fichier CSV dans Eloqua, suivez ces étapes :

1. Connectez-vous à Eloqua et accédez à **Audience &gt; Contacts &gt; Upload** pour ouvrir le **Contact Upload Wizard**.
2. Configurez l’importation à l’aide des champs fournis dans l’onglet **Pick Data Source**.
3. Cliquez sur le bouton **Cloud**, puis sélectionnez le fichier à téléverser.
4. Après avoir vérifié le contenu du fichier dans l’onglet **Review**, cliquez sur **Next Step**.
5. Dans l’onglet **Map Fields**, entrez les paramètres de mappage des champs pour faire correspondre les données de votre fichier CSV aux champs de contact. Cliquez sur **Next Step** pour continuer.
6. Terminez la dernière étape de l’assistant, puis cliquez sur **Finish**.

Et voilà ! Vous avez importé vos utilisateurs Auth0 dans Eloqua avec succès.
