title: Exporter les données utilisateur vers Salesforce
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans Salesforce à l’aide de l’Assistant d’importation de données. Pour consulter la configuration requise, visitez l’article Salesforce [Assistant d’importation de données](https://help.salesforce.com/articleView?id=data_import_wizard.htm).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Dans la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** sur le format de fichier requis. Salesforce accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

Dans le haut, à la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name      |
| ------------- | ---------------- |
| `email`       | Adresse courriel |
| `created_at`  | Date de création |
| `given_name`  | Prénom           |
| `family_name` | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importez un fichier de données utilisateur
</div>

Selon le type de compte, des étapes supplémentaires peuvent être nécessaires pour réussir l’importation. Prenez un moment pour consulter l’article de Salesforce [Prepare Your Data for Import](https://help.salesforce.com/articleView?id=import_prepare.htm). Pour en savoir plus, consultez [Salesforce: Import Data with the Data Import Wizard](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm).

Avant de commencer, assurez-vous que votre compte Salesforce dispose des [permissions utilisateur](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm) requises pour importer des enregistrements.

Pour importer votre fichier CSV dans Salesforce, suivez les étapes suivantes :

1. Connectez-vous à [Salesforce](https://login.salesforce.com/).
2. Accédez à **Setup** et ouvrez le **Assistant d’importation de données**.
3. Cliquez sur **Launch Wizard**.
4. Configurez l’importation et sélectionnez le fichier CSV à téléverser à l’aide des champs fournis.
5. Cliquez sur **Next**.
6. Dans la page **Edit Field Mapping**, faites correspondre les champs de données de votre fichier CSV aux champs de données de Salesforce. Cliquez sur **Next** pour continuer.
7. Après avoir vérifié les renseignements sur la page **Review**, cliquez sur **Start Import**.
8. Consultez l’onglet **Recent Import Jobs** sur la page d’accueil du **Assistant d’importation de données** pour suivre l’état de votre importation.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Salesforce.
