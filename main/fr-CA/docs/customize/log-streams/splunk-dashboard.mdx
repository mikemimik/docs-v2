Vous pouvez utiliser **Auth0 App for Splunk** pour visualiser les données de votre tenant Auth0 dans un tableau de bord Splunk. Le tableau de bord vous permet de surveiller l’état du trafic de connexion d’un tenant. L’application vous permet d’utiliser les agrégations recommandées par Auth0 ou de vous en servir comme point de départ pour créer vos propres visualisations.

<div id="install-the-auth0-app-for-splunk">
  ## Installer l’Auth0 App for Splunk
</div>

1. Connectez-vous à [Splunk](https://www.splunk.com/).
2. Accédez à la page de l’application [Auth0 App for Splunk](https://splunkbase.splunk.com/app/5193).
3. Sélectionnez le bouton vert **Install**.
4. Entrez votre username et votre password, acceptez les conditions générales, puis sélectionnez **Login and Install**.
5. Dans le widget **Complete** qui s’affiche, sélectionnez le bouton **Open the App**.
   Le tableau de bord Auth0 s’affiche avec les filtres par défaut appliqués.

<div id="dashboard-filters">
  ## Filtres du tableau de bord
</div>

Les filtres suivants vous permettent d’examiner plus précisément les détails de votre trafic. Vous pouvez saisir `*` pour rechercher dans toutes les valeurs de ce champ.

| Filtre            | Description                                                                                                                                                                                                                        |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Période           | Un champ de sélection temporelle Splunk qui vous permet de choisir la période d’affichage des événements.                                                                                                                          |
| Index             | L’`index` Splunk dans lequel vous voulez effectuer la recherche. Lorsque vous créez votre `Http Event Collector Token`, si vous avez indiqué un index particulier, vous pouvez sélectionner cette valeur dans la liste déroulante. |
| Source HTTP       | Le nom de `source` dans Splunk.                                                                                                                                                                                                    |
| IP                | L’adresse IP dont vous voulez examiner le trafic. Cela correspond à `data.ip` dans l’événement de journal.                                                                                                                         |
| Client            | Le client dont vous voulez examiner le trafic. Cela correspond à `data.client_name` dans l’événement de journal.                                                                                                                   |
| Pays              | Le pays dont vous voulez examiner le trafic. Il s’agit d’un champ obtenu à l’aide de la commande de recherche `iplocation data.ip` dans la requête. Un champ `Country` est ainsi ajouté à chaque événement de journal.             |
| Nom d’utilisateur | L’adresse courriel dont vous voulez examiner le trafic de connexion. Cela correspond à `data.user_name` dans l’événement de journal.                                                                                               |

<div id="customize-dashboard">
  ## Personnaliser le tableau de bord
</div>

Vous pouvez personnaliser votre tableau de bord de sécurité Splunk Auth0 pour y ajouter des widgets de données personnalisées.

1. Accédez à votre tableau de bord de sécurité Auth0, puis sélectionnez **edit** dans le coin supérieur droit.
2. Dans le panneau Edit Dashboard, sélectionnez **add panel**, puis choisissez un type de contenu, par exemple un graphique linéaire, un événement ou un graphique en aires.
3. Saisissez la période, le titre du contenu et la chaîne de recherche pour la visualisation des données.
4. Sélectionnez **Add to dashboard** pour ajouter votre nouveau widget de données.
