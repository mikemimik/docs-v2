title: Mettre à jour les applications dans le tableau de bord SSO
---

Utilisez l’extension du tableau de bord SSO pour gérer l’accès <Tooltip tip="Authentification unique (SSO) : service qui, lorsqu’un utilisateur ouvre une session dans une application, l’authentifie automatiquement dans d’autres applications." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=SSO">SSO</Tooltip> de vos utilisateurs à plusieurs applications d’entreprise.

Pour en savoir plus, consultez [l’extension du tableau de bord SSO](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension) ou [consultez cette extension sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).

<div id="prerequisites">
  ## Prérequis
</div>

* [Créer l’application du tableau de bord SSO](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application) dans Auth0.
* [Installer l’extension du tableau de bord SSO](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).
* [Ajouter des applications au tableau de bord SSO](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard).

<div id="update-application">
  ## Mettre à jour l’application
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) et sélectionnez la vue **Extensions installées**.
2. Sélectionnez **Tableau de bord SSO**.
   Si vous êtes administrateur, vous pouvez aussi vous connecter au Tableau de bord SSO à l’aide de `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
3. Sélectionnez **Paramètres** dans le menu déroulant sous le nom de votre tenant.

* Pour préciser si les utilisateurs peuvent voir l’application lorsqu’elle est activée, sélectionnez **Publier** ou **Dépublier**.
* Pour mettre à jour les paramètres d’une application, sélectionnez son icône d’engrenage.
* Pour supprimer une application, sélectionnez son **X**. Une fenêtre contextuelle de confirmation s’affichera.
