<Card title="Avant de commencer">
  1. [Créez une application SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application) dans Auth0.
  2. Copiez la valeur du **Client ID** de votre application SSO Dashboard.

Utilisez l’extension <Tooltip tip="Single Sign-On (SSO) : service qui, après qu’un utilisateur s’est connecté à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Single+Sign-On">Single Sign-On</Tooltip> (SSO) Dashboard pour gérer l’authentification SSO de vos utilisateurs dans plusieurs applications d’entreprise. Pour en savoir plus, [consultez cette extension sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).

<div id="install-extension">
  ## Installer l’extension
</div>

Vous pouvez installer l’extension SSO Dashboard dans le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez **SSO Dashboard** dans la liste des extensions offertes. La fenêtre **Install Extension** s’ouvrira :

   <Frame>![Dashboard - Extensions - SSO Dashboard - Install Extension](/docs/images/cdy7uua7fh8z/75heEyuJAN6FGSszXx06Hm/1cf10550c770beb56731285709442afb/SSO_Extension_-_EN.png)</Frame>
2. Définissez les variables de configuration suivantes :

   | Variable              | Obligatoire | Description                                                                                         |
   | --------------------- | ----------- | --------------------------------------------------------------------------------------------------- |
   | `EXTENSION_CLIENT_ID` | Obligatoire | **Client ID** de l’application Auth0 que vous avez créée pour l’extension.                          |
   | `TITLE`               | Facultatif  | Titre personnalisé qui apparaîtra en haut de la page SSO Dashboard.                                 |
   | `CUSTOM_CSS`          | Facultatif  | Chemin vers un fichier CSS personnalisé pour personnaliser l’apparence de votre page SSO Dashboard. |
   | `FAVICON_PATH`        | Facultatif  | Chemin vers un favicon personnalisé.                                                                |
   | `AUTH0_CUSTOM_DOMAIN` | Facultatif  | [Domaine personnalisé](/fr-CA/docs/customize/custom-domains) configuré avec Auth0.                        |

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     La définition de la variable `AUTH0_CUSTOM_DOMAIN` n’a pas d’incidence sur l’URL de l’extension ; elle modifie uniquement l’URL du point de terminaison d’autorisation utilisé par l’extension. Lorsqu’un domaine personnalisé est utilisé, les utilisateurs qui se connectent à l’extension seront redirigés vers `https://{yourAuth0CustomDomain}/login` au lieu de l’URL par défaut `https://{yourTenant}.{yourRegion}.auth0.com/login`.
   </Callout>
3. Sélectionnez **Install**.

Si vous accédez à [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), vous remarquerez qu’une application nommée `auth0-sso-dashboard` a été créée automatiquement. Cette application est utilisée par l’extension pour autoriser l’accès à la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>, et vous ne devez pas la modifier.

<div id="use-extension">
  ## Utiliser l’extension
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez la vue **Installed Extensions**.
2. Sélectionnez **SSO Dashboard**. La première fois que vous sélectionnerez l’extension installée, vous devrez lui accorder les permissions requises. Une fois que vous aurez accepté, vous serez redirigé vers votre **SSO Dashboard** personnalisé, qui affichera le **Title** que vous avez fourni en haut de la page. Si vous avez fourni un fichier CSS personnalisé, ce style sera également appliqué.
3. Pour vous connecter au **SSO Dashboard** :

   1. Pour les **Admins,** utilisez :
      `https://{yourTenant}.YOUR_REGION_KEY.webtask.io/auth0-sso-dashboard/admins/login`
   2. Pour les **Users,** utilisez :
      `https://{yourTenant}.YOUR_REGION_KEY.webtask.io/auth0-sso-dashboard/login`
