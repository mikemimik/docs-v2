---

Utilisez l’extension Single Sign-On (<Tooltip tip="Single Sign-On (SSO) : service qui, lorsqu’un utilisateur se connecte à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=SSO">SSO</Tooltip>) Dashboard pour gérer la connexion SSO de vos utilisateurs dans plusieurs applications d’entreprise.

<div id="prerequisites">
  ## Prérequis
</div>

* [Créer l’application SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application).
* [Installer l’extension SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).

<div id="add-applications">
  ## Ajouter des applications
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez votre nouvelle extension SSO Dashboard.
   Si vous êtes administrateur, vous pouvez aussi vous connecter au SSO Dashboard à l’aide de `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
2. Dans le coin supérieur droit, sélectionnez **Paramètres** dans le menu déroulant sous le nom de votre tenant.
3. Cliquez sur **Créer une application** pour ajouter une nouvelle application.

   Le formulaire **Nouvelle application** s’affiche.
4. Remplissez les champs suivants pour la nouvelle application :

   | Champ                 | Description                                                                                                                                                                                                                                                                                                                                |
   | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   | **Type**              | Une liste déroulante dans laquelle vous sélectionnez SAML, OpenID-Connect ou WS-Federation selon le type d’application.                                                                                                                                                                                                                    |
   | **Application**       | Une liste déroulante dans laquelle vous sélectionnez l’application que vous souhaitez ajouter.                                                                                                                                                                                                                                             |
   | **Nom**               | Le nom est rempli automatiquement en fonction de l’application que vous avez sélectionnée. Vous pouvez le modifier ou utiliser la valeur par défaut.                                                                                                                                                                                       |
   | **Logo**              | L’URL du logo que vous souhaitez utiliser comme icône pour l’application.                                                                                                                                                                                                                                                                  |
   | **Callback**          | L’une des **URL de rappel autorisées** dans vos [Paramètres de l&#39;application](https://manage.auth0.com/#/applications) pour l’application.                                                                                                                                                                                             |
   | **Connexion**         | (*Facultatif*) Le type de connexion. Vous pouvez ajouter ou modifier vos types de connexion disponibles dans la [section Connexions du tableau de bord de gestion Auth0](https://manage.auth0.com/#/connections/database). Si aucune connexion n’est définie et que l’utilisateur n’est pas connecté, il verra la page de connexion Auth0. |
   | **URL personnalisée** | (*Facultatif*) Cochez cette case pour utiliser une URL personnalisée plutôt que l’URL Auth0. Si vous cochez la case, un champ s’affiche dans lequel vous pouvez saisir votre URL personnalisée.                                                                                                                                            |
   | **Activée**           | Case à cocher pour que cette application soit visible (publiée) pour vos utilisateurs.                                                                                                                                                                                                                                                     |
5. Sélectionnez **Créer**. Votre nouvelle application apparaîtra alors sur la page **Applications** du SSO Dashboard avec toutes les autres applications qui ont été créées.
6. Cliquez sur une application pour tester la connexion.
