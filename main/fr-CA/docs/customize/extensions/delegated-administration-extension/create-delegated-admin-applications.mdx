description: Découvrez comment créer une application à utiliser avec la Delegated Admin Extension afin d’exposer de façon sélective la section des utilisateurs du Dashboard.
title: Créer des applications Delegated Admin
---

Utilisez la [Delegated Admin Extension](/docs/fr-CA/customize/extensions/delegated-administration-extension) pour exposer la section **Users** du <Tooltip tip="Auth0 Dashboard: produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> à un groupe restreint d’utilisateurs, sans leur donner accès au reste du Dashboard. Avant d’ajouter la Delegated Admin Extension, vous devez créer l’application Delegated Admin dans Auth0.

* Accédez à [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis sélectionnez **Create Application**.
* Entrez un nom descriptif pour votre Application (par exemple, **Users Dashboard**), sélectionnez le type d’application **Single-Page Web Application**, puis cliquez sur **Create**.
* Dans la vue **Settings**, définissez les **Allowed Callback URLs** et les **Allowed Logout URLs** selon votre emplacement, puis cliquez sur **Save Changes**.

**URL de rappel par région :**

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Remplacez `{yourTenant}` par le nom de votre tenant. Par exemple, si le nom de votre tenant est `tenant_name_example`, alors pour un tenant exécuté dans un environnement US-3, l’URL de rappel correspondante est : `https://tenant_name_example.us.webtask.run/auth0-delegated-admin/login`
</Callout>

| Emplacement | Region | Lien de connexion                                                 |
| ----------- | ------ | ----------------------------------------------------------------- |
| Australie   | AU     | `https://{yourTenant}.au.webtask.io/auth0-delegated-admin/login`  |
| Europe      | EU     | `https://{yourTenant}.eu.webtask.io/auth0-delegated-admin/login`  |
| Europe      | EU-2   | `https://{yourTenant}.eu.webtask.run/auth0-delegated-admin/login` |
| Japon       | JP-1   | `https://{yourTenant}.jp.webtask.run/auth0-delegated-admin/login` |
| Royaume-Uni | UK     | `https://{yourTenant}.uk.webtask.run/auth0-delegated-admin/login` |
| États-Unis  | US-1   | `https://{yourTenant}.us.webtask.io/auth0-delegated-admin/login`  |
| États-Unis  | US-3   | `https://{yourTenant}.us.webtask.run/auth0-delegated-admin/login` |

Ensuite, [installez la Delegated Admin Extension](/docs/fr-CA/customize/extensions/delegated-administration-extension/install-delegated-admin-extension).
