---
description: Découvrez comment utiliser le Memberships Query Hook avec la Delegated Administration Extension.
title: 'Delegated Administration: Memberships Query Hook'
---

Lors de la création d’un nouvel utilisateur, l’interface utilisateur affiche une liste déroulante qui vous permet de sélectionner la ou les Memberships à attribuer à l’utilisateur. Ces Memberships sont ensuite définies à l’aide du **Memberships Query Hook**.

<div id="hook-contract">
  ## Contrat du hook
</div>

* **ctx** : Objet de contexte.
* **callback(error, &#123; createMemberships: true/false, memberships: &#125;)** : Callback à laquelle vous pouvez renvoyer une erreur et un objet contenant la configuration des memberships.

<div id="sample-use">
  ## Exemple d’utilisation
</div>

Les utilisateurs du service informatique devraient pouvoir créer des utilisateurs dans les autres services. Les utilisateurs des autres services devraient seulement pouvoir créer des utilisateurs pour leur propre service.

```javascript lines
function(ctx, callback) {
  var currentDepartment = ctx.payload.user.app_metadata && ctx.payload.user.app_metadata.department;
  if (!currentDepartment || !currentDepartment.length) {
    return callback(null, [ ]);
  }

  if (currentDepartment === 'IT') {
    return callback(null, [ 'IT', 'HR', 'Finance', 'Marketing' ]);
  }

  return callback(null, [ ctx.payload.user.app_metadata.department ]);
}
```

<div id="notes">
  ## Remarques
</div>

Comme cette requête ne peut être utilisée que dans l’UI, vous devrez attribuer les memberships à l’aide du **Write Hook** si vous devez faire respecter des règles concernant l’affectation d’utilisateurs à des départements précis.

S’il n’y a qu’un seul groupe d’appartenance possible, le champ Memberships ne s’affichera pas dans l’UI.

Vous pouvez permettre à l’utilisateur final de saisir n’importe quelle valeur dans le champ **memberships** en définissant **createMemberships** sur true :

```javascript lines
function(ctx, callback) {
  var currentDepartment = ctx.payload.user.app_metadata.department;
  if (!currentDepartment || !currentDepartment.length) {
    return callback(null, [ ]);
  }

  return callback(null, {
    createMemberships: ctx.payload.user.app_metadata.department === 'IT' ? true : false,
    memberships: [ ctx.payload.user.app_metadata.department ]
  });
}
```
