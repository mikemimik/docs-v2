---
title: "Unlink"
description: "Point de terminaison obsolète permettant de dissocier des comptes d’utilisateurs; utilisez plutôt la Management API."
---

<div id="endpoint">
  ## Point de terminaison
</div>

`POST /login/unlink`

<Note>
  Ce point de terminaison est **obsolète**. Utilisez plutôt [DELETE /api/v2/users/{id}/identities/{provider}/{user_id}](/fr-CA/api/management/v2/users/delete-user-identity-by-user-id).
</Note>

À partir de l&#39;`access_token` et du `user_id` d&#39;un utilisateur connecté, ce point de terminaison dissocie le compte de l&#39;utilisateur de son fournisseur d&#39;identité.

<div id="learn-more">
  ### En savoir plus
</div>

* [Dissocier des comptes d’utilisateurs](https://auth0.com/docs/manage-users/user-accounts/user-account-linking/unlink-user-accounts)

<div id="response-messages">
  ## Messages de réponse
</div>

| Statut | Description                             |
| ------ | --------------------------------------- |
| 200    | Compte utilisateur dissocié avec succès |
