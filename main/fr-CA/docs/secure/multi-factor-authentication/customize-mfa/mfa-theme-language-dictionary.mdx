<div id="languagedictionary">
  ## languageDictionary
</div>

Le `languageDictionary` permet de remplacer le texte dans plusieurs sections du processus de <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification qui ajoute un facteur au username et au Password, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=MFA">MFA</Tooltip> de l’expérience Classic Login. Pour ce faire, utilisez le format suivant :

```yaml lines
languageDictionary:
  optionsCategory:
    option: value
```

Voici un exemple de modification de l’attribut `headerText` dans `smsEnrollmentConfirm` :

```js lines
return new Auth0MFAWidget({

...

  languageDictionary:{ // Utiliser l'option Language Dictionary
    smsEnrollmentConfirm:{  // Indiquer la catégorie/section à modifier
      headerText: "Some custom text - In order to confirm enrollment we need to confirm your phone. Please enter the received code." // Fournir la nouvelle valeur pour l'attribut spécifique à modifier
    }
  }

...

})
```

Vous trouverez ci-dessous la liste des catégories et des options offertes.
