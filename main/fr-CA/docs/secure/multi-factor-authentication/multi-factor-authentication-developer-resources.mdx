Grâce aux SDK Auth0, vous pouvez personnaliser l’expérience d’<Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA) de vos utilisateurs, et même créer des applications qui reposent sur nos fonctionnalités multifacteur.

<div id="mfa-api">
  ## MFA API
</div>

Les points de terminaison de l’[MFA API](/fr-CA/docs/secure/multi-factor-authentication/multi-factor-authentication-developer-resources/mfa-api) vous permettent d’imposer la MFA lorsque les utilisateurs interagissent avec les [points de terminaison de token](https://auth0.com/docs/api/authentication#get-token), ainsi que d’inscrire et de gérer les facteurs MFA.

<div id="create-custom-mfa-pages">
  ## Créez des pages MFA personnalisées
</div>

Utilisez les bibliothèques client suivantes pour personnaliser l’apparence de la page MFA afin qu’elle s’harmonise avec votre organisation.

* [Bibliothèque client Auth0 MFA](https://github.com/auth0/auth0-guardian.js)
* [Créer un widget MFA personnalisé](https://github.com/auth0/auth0-guardian.js/tree/master/example)

<div id="create-custom-enrollments">
  ## Créer des inscriptions personnalisées
</div>

Vous pouvez [personnaliser le processus d’inscription](/fr-CA/docs/secure/multi-factor-authentication/multi-factor-authentication-developer-resources/create-custom-enrollment-tickets) de vos utilisateurs en créant des tickets d’inscription personnalisés.

<div id="build-custom-mobile-applications">
  ## Créez des applications mobiles personnalisées
</div>

Créez des applications personnalisées comme Guardian ou ajoutez une fonctionnalité multifacteur à vos applications grâce à nos SDK.

* [Guardian pour Android](/fr-CA/docs/secure/multi-factor-authentication/auth0-guardian/guardian-for-android-sdk)
* [Guardian pour iOS](/fr-CA/docs/secure/multi-factor-authentication/auth0-guardian/guardian-for-ios-sdk)
