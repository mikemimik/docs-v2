Chaque propriété de la clé est définie dans la spécification JWK [RFC 7517 Section 4](https://tools.ietf.org/html/rfc7517#section-4) ou, pour les propriétés propres à un algorithme, dans la [RFC 7518](https://tools.ietf.org/html/rfc7518).

| Nom de la propriété | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `alg`               | L’algorithme cryptographique précis utilisé avec la clé. Il s’agit d’un paramètre facultatif. Par défaut, Auth0 inclut dans le JSON Web Key Set (JWKS) l’algorithme de signature défini au niveau du tenant, qui est ensuite publié. Pour permettre aux clés d’être utilisées avec plusieurs algorithmes plutôt qu’avec un seul, par ex. RS256, désactivez **Include Signing Algorithms in JSON Web Key Set** dans les paramètres de tenant avancés du Dashboard. Cela supprime le paramètre `alg` et oblige les consommateurs du JWKS à interpréter les algorithmes de signature au besoin. |
| `kty`               | La famille d’algorithmes cryptographiques utilisée avec la clé.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `use`               | L’usage prévu de la clé ; `sig` représente la signature.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `x5c`               | La chaîne de certificats x.509. La première valeur du tableau est le certificat à utiliser pour vérifier le token ; les autres certificats peuvent servir à vérifier ce premier certificat.                                                                                                                                                                                                                                                                                                                                                                                                  |
| `n`                 | Le module de la [clé publique RSA](https://tools.ietf.org/html/rfc7518#page-30).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `e`                 | L’exposant de la [clé publique RSA](https://tools.ietf.org/html/rfc7518#page-30).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `kid`               | L’identifiant unique de la clé.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `x5t`               | L’empreinte du certificat x.509 (empreinte SHA-1).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 prend uniquement en charge HMAC et RSA ; il ne prend pas en charge, pour le moment, le chiffrement à courbe elliptique (ECDSA).
</Callout>
