title: Jetons d’accès à la Management API
---

Pour appeler l’[Auth0 Management API](https://auth0.com/docs/api/management/v2), vous devez fournir un <Tooltip tip="Jeton d’accès : information d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=access+token">jeton d’accès</Tooltip> qui contient l’<Tooltip tip="Jeton d’accès : information d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=audience">audience</Tooltip> et les scopes appropriés.

<div id="audience">
  ## Audience
</div>

La valeur d’audience pour la <Tooltip tip="Management API : produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip> est `https://{yourDomain}/api/v2/`.

<div id="scopes">
  ## Scopes
</div>

Chaque endpoint de la Management API exige un ensemble précis de scopes. Consultez le [Management API Explorer](https://auth0.com/docs/api/management/v2/) pour en savoir plus.

<div id="lifetime">
  ## Durée de vie
</div>

La durée de vie par défaut du jeton de la Management API est de 86 400 secondes (24 heures).

Vous pouvez [configurer la durée de vie du jeton](/fr-CA/docs/secure/tokens/access-tokens/update-access-token-lifetime) pour chaque API séparément.

<div id="security">
  ## Sécurité
</div>

* **Jeton d’accès**: Une fois délivré, un jeton d’accès ne peut pas être révoqué. Auth0 recommande de limiter la durée de vie de vos jetons d’accès à la valeur la plus courte possible pour votre cas d’utilisation.
* **<Tooltip tip="Jeton d’actualisation: Jeton utilisé pour obtenir un nouveau jeton d’accès sans obliger les utilisateurs à se reconnecter." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Refresh+token">Jeton d’actualisation</Tooltip>**: Vous pouvez [révoquer un jeton d’actualisation](/fr-CA/docs/secure/tokens/refresh-tokens/revoke-refresh-tokens) à l’aide du Dashboard, de l’Authentication API ou de la Management API.
* **<Tooltip tip="Secret du client: Secret utilisé par un client (application) pour s’authentifier auprès du serveur d’autorisation; il ne doit être connu que du client et du serveur d’autorisation et doit être suffisamment aléatoire pour ne pas pouvoir être deviné." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Client+secret">Secret du client</Tooltip>**: Vous pouvez [faire la rotation du secret du client d’une application](/fr-CA/docs/get-started/applications/rotate-client-secret) à l’aide du Dashboard ou de la Management API.

<div id="quota">
  ## Quota
</div>

Les quotas de jetons sont déterminés par le niveau d’abonnement. Vous pouvez [consulter votre quota actuel](https://support.auth0.com/reports/quota) dans l’Auth0 Support Center. Si vous avez des questions sur la tarification ou les quotas, veuillez [communiquer avec notre équipe des ventes](https://auth0.com/contact-us).

* Les jetons émis pour les [audiences internes d’Auth0](/fr-CA/docs/secure/tokens/access-tokens/get-access-tokens#control-access-token-audience) ne sont pas comptabilisés dans votre quota.
* Les jetons émis pour des audiences personnalisées sont comptabilisés dans votre quota et sont assujettis aux [limites de débit de l’Auth0 Management API](/fr-CA/docs/troubleshoot/customer-support/operational-policies/rate-limit-policy/management-api-endpoint-rate-limits).
