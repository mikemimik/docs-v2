---
description: Décrit comment vous pouvez exporter les données des utilisateurs afin de respecter les exigences du GDPR en matière de portabilité des données.
title: 'GDPR : Portabilité des données'
---

Conformément à l’article 20 du GDPR, les utilisateurs ont le droit de recevoir les données personnelles les concernant dans un format structuré, couramment utilisé et lisible par machine.

Vous pouvez exporter les données des utilisateurs stockées dans le user store d’Auth0, soit manuellement, soit par programmation. Les données brutes d’Auth0 peuvent être exportées au format JSON (qui est lisible par machine).

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le contenu de ces documents ne constitue pas un avis juridique et ne doit pas non plus être considéré comme un remplacement à une assistance juridique. Il vous incombe en dernier ressort de comprendre le GDPR et de vous y conformer, bien qu’Auth0 vous aide à respecter les exigences du GDPR lorsque cela est possible.
</Callout>

<div id="export-data-manually">
  ## Exporter les données manuellement
</div>

1. Accédez à [Auth0 Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users).
2. Recherchez l’utilisateur, puis cliquez sur son nom pour afficher ses détails.
3. Cliquez sur l’onglet **Raw JSON**. Vous pouvez y voir le profil utilisateur complet au format JSON.
4. Cliquez sur **Copy JSON**. Le profil est copié dans votre presse-papiers.
5. Collez le contenu du presse-papiers dans un éditeur et enregistrez-le.

<div id="export-data-using-the-api">
  ## Exporter des données avec l’API
</div>

Vous pouvez exporter le profil complet d’un utilisateur avec notre <Tooltip tip="Management API : Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. La réponse sera au format JSON. Vous pouvez soit rechercher un utilisateur à l’aide de son identifiant, soit exporter une liste de vos utilisateurs.
