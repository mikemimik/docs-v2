Si vous utilisez des flux de connexion et d’inscription <Tooltip tip="Sans mot de passe : forme d’authentification qui ne repose pas sur un mot de passe comme premier facteur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=passwordless">sans mot de passe</Tooltip>, vous pouvez activer <Tooltip tip="Sans mot de passe : forme d’authentification qui ne repose pas sur un mot de passe comme premier facteur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Bot+Detection">Bot Detection</Tooltip> pour afficher une étape CAPTCHA lorsqu’une requête est jugée à risque élevé par Auth0.

<Warning>
  Si vous utilisez le [SDK Lock](/fr-CA/docs/libraries/lock) avec l’expérience Classic Login, vous devez passer à la version 11.35.0 ou ultérieure pour utiliser Bot Detection avec les flux sans mot de passe.

  Si vous utilisez le [SDK Auth0.js](/fr-CA/docs/libraries/auth0js), vous devez passer à la version 9.20.0 ou ultérieure pour utiliser Bot Detection avec les flux sans mot de passe.
</Warning>

<div id="configure-bot-detection-for-passwordless-flows">
  ## Configurer Bot Detection pour les flux sans mot de passe
</div>

Vous pouvez utiliser le <Tooltip tip="Auth0 Dashboard : l’interface principale d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> pour configurer le fonctionnement de la fonctionnalité CAPTCHA.

1. Dans l’[Auth0 Dashboard](https://manage.auth0.com/dashboard), accédez à [Security &gt; Attack Protection &gt; Bot Detection](https://manage.auth0.com/dashboard/#/security/attack-protection/bot-detection).
2. Repérez la section **Response**.
3. Pour le paramètre **Enforce CAPTCHA for passwordless flows**, sélectionnez à quel moment exiger un CAPTCHA :

   * **Never** : les utilisateurs n’ont pas à remplir de CAPTCHA pour se connecter.
   * **When Risky** : les utilisateurs doivent remplir un CAPTCHA si la connexion présente un risque élevé.
   * **Always** : les utilisateurs doivent toujours remplir un CAPTCHA pour se connecter.
