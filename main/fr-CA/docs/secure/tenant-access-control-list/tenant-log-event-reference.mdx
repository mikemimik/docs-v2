title: Référence de l’événement de journal du tenant pour les règles de Tenant Access Control List
sidebarTitle: Événements de journal du tenant
description: Consultez la description de tous les champs de l’événement de journal du tenant Tenant ACL (`acls_summary`).
---

L’[événement de journal](/fr-CA/docs/deploy-monitor/logs) `acls_summary` comprend les champs suivants :

<ResponseField name="acls_summary" type="object">
  <Expandable>
    <ResponseField name="acl_id" type="string">
      Identifiant interne de la règle.

      Exemple : `acl_vENikGo5fcJd7FYMLvjKKP`
    </ResponseField>

    <ResponseField name="description" type="string">
      Description de la règle.

      Exemple : `Allow traffic that originates from the United States`
    </ResponseField>

    <ResponseField name="priority" type="number">
      Ordre d’exécution de la règle.

      Exemple : `1`
    </ResponseField>

    <ResponseField name="action" type="object">
      Action exécutée par la règle.

      Exemple : `{ "allow": true }`
    </ResponseField>

    <ResponseField name="match" type="object">
      <Expandable>
        <ResponseField name="success" type="number">
          Nombre total de requêtes auxquelles la règle a correspondu avec succès.

          Exemple : `1`
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="total_request_count" type="object">
      <Expandable>
        <ResponseField name="successes" type="number">
          Nombre total de requêtes que la règle a évaluées avec succès.

          Exemple : `3`
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="start_time" type="string">
      Horodatage du début de la période de comptabilisation des journaux.

      Exemple : `2025-03-19T16:11:07.429Z`
    </ResponseField>

    <ResponseField name="end_time" type="string">
      Horodatage de la fin de la période de comptabilisation des journaux.

      Exemple : `2025-03-19T16:21:07.429Z`
    </ResponseField>
  </Expandable>
</ResponseField>
