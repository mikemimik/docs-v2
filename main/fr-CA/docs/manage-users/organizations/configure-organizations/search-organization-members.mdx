---
description: Comment utiliser User Search pour trouver des membres d’une Organization
title: Rechercher des membres d’une Organization
---

En plus de [lister les membres d’une organization](/docs/fr-CA/manage-users/organizations/configure-organizations/retrieve-members), vous pouvez utiliser [User Search](/docs/fr-CA/manage-users/user-search) pour trouver des utilisateurs en les filtrant selon leur appartenance à une organization à l’aide du paramètre `organization_id` dans une recherche.

Par exemple, voici un filtre pour les utilisateurs dont le domaine de courriel est `acme.com`, mais qui ne sont pas membres de l’organization Acme :

`email.domain:"acme.com" AND NOT organization_id:"ACME_ORG_ID"`

La requête suivante trouve tous les utilisateurs qui font partie de l’organization Acme et qui se connectent avec une connexion de base de données précise.

`organization_id:"ACME_ORG_ID" AND identities.connection:"my-db-connection-name"`

L’exemple ci-dessous trouve les utilisateurs dont l’adresse courriel commence par jane et qui sont membres de l’organization Acme.

`organization_id:"ACME_ORG_ID" AND email:jane*`

Consultez [User Search](/docs/fr-CA/manage-users/user-search) pour plus de détails.
