---
title: Créer des domaines d’organisation
description: Invitez les utilisateurs à choisir la bonne organisation pendant les flux d’inscription et de connexion à l’aide de domaines d’organisation vérifiés.
---

Les domaines d’organisation fonctionnent avec [Organization Domain Discovery](./define-organization-behavior) pour aider les utilisateurs à trouver la bonne organisation pendant l’inscription et les [flux de connexion](/fr-CA/docs/manage-users/organizations/login-flows-for-organizations).

Une fois configurés, les domaines d’organisation vérifiés et Organization Domain Discovery aident Auth0 à détecter automatiquement l’organisation d’un utilisateur ou à réduire le nombre d’organisations proposées lorsque les utilisateurs saisissent leur adresse courriel. Les domaines d’organisation et Organization Domain Discovery ne restreignent pas les personnes qui peuvent s’inscrire ou ouvrir une session.

<div id="create-an-organization-domain-with-the-auth0-dashboard">
  ## Créer un domaine d’organisation avec l’Auth0 Dashboard
</div>

Pour créer un domaine d’organisation dans l’Auth0 Dashboard :

1. Accédez à [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations).
2. Sélectionnez l’onglet **Domains**.
3. Choisissez **Add Domain**.
   Dans le champ **Domain**, entrez le domaine de courriel à associer pour la découverte de l’organisation. Le statut demeure en attente jusqu’à ce que la propriété soit confirmée.
4. Sélectionnez **Add Domain**.
5. Au besoin, configurez [Organization Domain Discovery](./define-organization-behavior).

<div id="create-an-organization-domain-with-the-management-api">
  ## Créer un domaine d’organisation avec la Management API
</div>

Pour créer un domaine d’organisation avec la Management API, appelez le [point de terminaison Créer un domaine de découverte d’organisation](/fr-CA/docs/api/management/v2/organizations/post-discovery-domains) (`POST /organizations/{id}/discovery-domains`). Au besoin, activez le domaine pour Organization Domain Discovery à l’aide du paramètre `use_for_organization_discovery`.

Pour activer Organization Domain Discovery, appelez le [point de terminaison Patch clients par ID](/fr-CA/docs/api/management/v2/clients/patch-clients-by-id#body-organization-discovery-methods-one-of-0) (`PATCH /client/{id}`) et définissez le paramètre `organization_discovery_methods`.
