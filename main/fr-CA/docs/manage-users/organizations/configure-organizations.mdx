Vous pouvez configurer des organisations à l’aide de l’<Tooltip tip="Auth0 Dashboard: principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou de la <Tooltip tip="Auth0 Dashboard: principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. Pour en savoir plus sur les étapes de configuration de votre organisation, consultez [Create Your First Organization](/docs/fr-CA/manage-users/organizations/create-first-organization).

<Card title="La disponibilité varie selon le plan Auth0">
  Votre plan Auth0 ou votre entente personnalisée détermine si cette fonctionnalité est offerte. Pour en savoir plus, consultez [Pricing](https://auth0.com/pricing).
</Card>
