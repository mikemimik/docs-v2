description: Découvrez le cycle de vie de la session.
title: Cycle de vie de la session
---

Le cycle de vie d’une session Auth0 commence lorsqu’un utilisateur se connecte, reste actif pendant les interactions de l’utilisateur, puis se termine lorsque la session expire ou qu’elle est explicitement fermée.

Auth0 considère qu’une session est active lorsqu’un utilisateur interagit avec le serveur d’autorisation.

Par exemple :

* Appels au endpoint [/authorize](/fr-CA/docs/api/authentication/authorization-code-flow/authorize-application)
* Déclencher une [authentification silencieuse](/fr-CA/docs/authenticate/login/configure-silent-authentication)
* Se connecter à une nouvelle application à l’aide de l’[authentification unique](/fr-CA/docs/authenticate/single-sign-on)

Une session peut être persistante ou non persistante; cela vous permet de configurer le comportement d’un cookie de session dans un navigateur.

* **Les sessions persistantes** stockent des cookies avec une date d’expiration ultérieure.
* **Les sessions non persistantes** stockent des cookies avec `Expires=0`. Cela indique au navigateur de supprimer le cookie à la fermeture du navigateur.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le comportement de persistance des cookies de session dépend de l’implémentation du navigateur. Auth0 ne peut pas garantir que tous les navigateurs supprimeront les cookies non persistants comme prévu.
  Pour en savoir plus, consultez [Cookies](https://auth0.com/docs/manage-users/cookies).
</Callout>

<div id="session-lifetime">
  ## Durée de vie de la session
</div>

Les sessions persistantes et non persistantes peuvent être configurées avec leurs propres durées de vie d’inactivité et absolues (maximales) afin de déterminer l’expiration d’une session.

<div id="idle-timeout">
  ### Délai d’inactivité
</div>

Le délai d’inactivité définit la durée maximale d’inactivité autorisée. Chaque fois qu’un utilisateur interagit avec Auth0 par l’entremise de l’authentification unique (SSO), de l’authentification silencieuse ou de /authorize, le délai d’inactivité est réinitialisé. S’il n’y a aucune interaction avec Auth0 pendant la période configurée, la session expire.

<div id="absolute-timeout-maximum">
  ### Délai d’expiration absolu (maximum)
</div>

Le délai d’expiration absolu définit la durée maximale pendant laquelle une session peut rester valide, peu importe l’activité de l’utilisateur. Une fois ce délai atteint, la session expire et l’utilisateur devra s’authentifier de nouveau.

Découvrez comment [configurer la durée de vie de la session](/fr-CA/docs/manage-users/sessions/configure-session-lifetime).

<div id="session-lifetime-limits">
  ## Limites de durée de vie de session
</div>

La durée de vie d’une session est soumise aux limites suivantes :

| Type de session | Type de délai d’expiration      | Description                                                                                      | Maximum (plans en libre-service) | Maximum (plans Enterprise) |
| --------------- | ------------------------------- | ------------------------------------------------------------------------------------------------ | -------------------------------- | -------------------------- |
| Non persistante | Délai d’expiration d’inactivité | Période après laquelle une session expire en l’absence d’interaction avec le serveur d’autorisation | 3 jours                          | 100 jours                  |
|                 | Exiger une connexion après      | Durée de vie maximale de la session, peu importe l’activité                                      | 30 jours                         | 365 jours                  |
| Persistante     | Délai d’expiration d’inactivité | Période après laquelle une session expire en l’absence d’interaction avec le serveur d’autorisation | 3 jours                          | 100 jours                  |
|                 | Exiger une connexion après      | Durée de vie maximale de la session, peu importe l’activité                                      | 30 jours                         | 365 jours                  |
