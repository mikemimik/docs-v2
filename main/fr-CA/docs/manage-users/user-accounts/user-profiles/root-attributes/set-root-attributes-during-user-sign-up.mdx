Vous pouvez définir les attributs racine d’un utilisateur pendant l’inscription à l’aide de la <Tooltip tip="Management API : Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip> d’Auth0. Cela vous permet de réduire le nombre d’appels à l’API nécessaires pour définir les attributs racine lors de la création d’utilisateurs.

Effectuez un appel `POST` au [point de terminaison Create a User](https://auth0.com/docs/api/management/v2#!/Users/post_users). Assurez-vous de remplacer les valeurs fictives `MGMT_API_ACCESS_TOKEN`, `CONNECTION_NAME`, `EMAIL_VALUE`, `PASSWORD_VALUE`, `GIVEN_NAME_VALUE`, `FAMILY_NAME_VALUE`, `NAME_VALUE`, `NICKNAME_VALUE` et `PICTURE` par votre <Tooltip tip="Access Token : Identifiant d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisé pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Access+Token">jeton d’accès</Tooltip> de la Management API, le nom de connexion initial, l’adresse courriel, le mot de passe, le prénom, le nom de famille, le nom, le pseudonyme et l’URL de l’image, respectivement.

</AuthCodeGroup>

| Valeur                  | Description                                                                                                                                         |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec la [portée](/fr-CA/docs/glossary?term=scope) `create:users`. |
| `CONNECTION_NAME`       | Nom de la connexion par laquelle les informations initiales de l’utilisateur ont été reçues.                                                        |
| `EMAIL_VALUE`           | Adresse courriel de l’utilisateur à créer.                                                                                                          |
| `PASSWORD_VALUE`        | Mot de passe de l’utilisateur à créer.                                                                                                              |
| `GIVEN_NAME_VALUE`      | Prénom de l’utilisateur à créer.                                                                                                                    |
| `FAMILY_NAME_VALUE`     | Nom de famille de l’utilisateur à créer.                                                                                                            |
| `NAME_VALUE`            | Nom complet de l’utilisateur à créer.                                                                                                               |
| `NICKNAME_VALUE`        | Pseudonyme de l’utilisateur à créer.                                                                                                                |
| `PICTURE_VALUE`         | URL de l’image de l’utilisateur à créer.                                                                                                            |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
