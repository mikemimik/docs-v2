Vous pouvez mettre à jour les attributs racine d’un profil utilisateur existant à l’aide de la <Tooltip tip="Management API: Un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip> d’Auth0.

Le [Normalized User Profile](/fr-CA/docs/manage-users/user-accounts/user-profiles/user-profile-structure) d’Auth0 comprend des attributs racine que vous pouvez mettre à jour. Les attributs racine précis que vous pouvez mettre à jour dépendent du type de [connection](/fr-CA/docs/authenticate/identity-providers) que vous utilisez. Pour en savoir plus sur la connection que vous utilisez, consultez [Attributs racine du profil utilisateur](/fr-CA/docs/manage-users/user-accounts/user-profiles/root-attributes).

Effectuez un appel `PATCH` au [endpoint Update a User](https://auth0.com/docs/api/management/v2/users/patch-users-by-id). Assurez-vous de remplacer les valeurs d’exemple `USER_ID`, `MGMT_API_ACCESS_TOKEN`, `GIVEN_NAME_VALUE`, `FAMILY_NAME_VALUE`, `NAME_VALUE`, `NICKNAME_VALUE` et `PICTURE` par votre ID utilisateur, votre <Tooltip tip="Access Token: Identifiant d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisé pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Access+Token">jeton d’accès</Tooltip> de la Management API, le prénom, le nom de famille, le nom, le pseudonyme et l’URL de l’image, respectivement.

</AuthCodeGroup>

| Valeur                  | Description                                                                                                                                         |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `USER_ID`               | L’identifiant de l’utilisateur à mettre à jour.                                                                                                     |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec la [portée](/fr-CA/docs/glossary?term=scope) `update:users`. |
| `GIVEN_NAME_VALUE`      | Prénom de l’utilisateur à mettre à jour.                                                                                                            |
| `FAMILY_NAME_VALUE`     | Nom de famille de l’utilisateur à mettre à jour.                                                                                                    |
| `NAME_VALUE`            | Nom complet de l’utilisateur à mettre à jour.                                                                                                       |
| `NICKNAME_VALUE`        | Pseudonyme de l’utilisateur à mettre à jour.                                                                                                        |
| `PICTURE_VALUE`         | URL de la photo de l’utilisateur à mettre à jour.                                                                                                   |

<div id="removing-attributes">
  ## Suppression des attributs
</div>

Attribuer la valeur `null` à n’importe quel champ supprime l’attribut de l’utilisateur.
