description: Décrit comment dissocier les appareils attribués à un utilisateur dans Auth0 Dashboard.
title: Dissocier des appareils des utilisateurs
---

Vous pouvez dissocier les appareils attribués à un utilisateur dans le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Cette opération révoque le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=refresh+token">jeton d’actualisation</Tooltip> de l’appareil sélectionné.

1. Accédez à [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users), puis cliquez sur le nom de l’utilisateur.
2. Cliquez sur l’affichage **Devices**, puis sur l’icône de corbeille à côté de l’appareil que vous voulez dissocier, puis confirmez.
