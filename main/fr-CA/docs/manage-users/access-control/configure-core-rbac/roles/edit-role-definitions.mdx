Vous pouvez modifier la définition d’un [rôle](/fr-CA/docs/manage-users/access-control/rbac) à l’aide du <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou de la <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>. Les rôles sont utilisés avec l’ensemble de fonctionnalités Authorization Core de l’API.

<div id="prerequisite">
  ## Prérequis
</div>

Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API à l’aide du Dashboard ou de la Management API. La fonctionnalité Authorization Core est différente de l’Authorization Extension. Pour les comparer, consultez [Authorization Core vs. Authorization Extension](/fr-CA/docs/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Allez à [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles) et cliquez sur le nom du rôle pour l’afficher.
2. Modifiez le nom et la description du rôle, puis cliquez sur **Save**.

<div id="management-api">
  ## Management API
</div>

Effectuez un appel `PATCH` à l’[endpoint Update Role](https://auth0.com/docs/api/management/v2#!/Roles/patch_roles_by_id). Assurez-vous de remplacer les valeurs d’espace réservé `ROLE_ID`, `MGMT_API_ACCESS_TOKEN`, `ROLE_NAME` et `ROLE_DESC` par l’ID de votre rôle, votre <Tooltip tip="Jeton d’accès : information d’identification d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=access+token">jeton d’accès</Tooltip> de la Management API, le nom du rôle et la description du rôle, respectivement.

</AuthCodeGroup>

| Valeur                  | Description                                                                                                            |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `ROLE_ID`               | L’identifiant du rôle dont vous voulez modifier la définition.                                                         |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec la portée `update:roles`. |
| `ROLE_NAME`             | Nom du rôle.                                                                                                           |
| `ROLE_DESC`             | Description du rôle conviviale pour l’utilisateur.                                                                     |
