<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Les ensembles de fonctionnalités [Authorization Core](/fr-CA/docs/manage-users/access-control/rbac) et [Authorization Extension](/fr-CA/docs/customize/extensions/authorization-extension) sont entièrement distincts. Pour gérer les groupes, les rôles ou les permissions, vous devez utiliser la fonctionnalité dans laquelle ils ont été créés à l’origine.
</Callout>

Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API au moyen du Dashboard ou de l’<Tooltip tip="Management API: A product to allow customers to perform administrative tasks." cta="View Glossary" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>. Les fonctionnalités d’Authorization Core sont différentes de celles d’Authorization Extension. Pour les comparer, consultez [Authorization Core vs. Authorization Extension](/fr-CA/docs/manage-users/access-control/authorization-core-vs-authorization-extension).

Nous proposons différentes fonctions pour vous aider à gérer vos rôles, accessibles soit dans l’<Tooltip tip="Auth0 Dashboard: Auth0's main product to configure your services." cta="View Glossary" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, soit dans l’Auth0 Management API :

* [Créer des rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/create-roles)
* [Modifier les définitions de rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/edit-role-definitions)
* [Ajouter des permissions à des rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/add-permissions-to-roles)
* [Retirer des permissions de rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/remove-permissions-from-roles)
* [Afficher les permissions d’un rôle](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Afficher les utilisateurs associés à un rôle](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/view-users-assigned-to-roles)
* [Supprimer des rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/delete-roles)
