title: Comptes utilisateur
---

La base de données infonuagique hébergée d’Auth0 stocke différents renseignements sur vos utilisateurs, auxquels vous avez accès. Ces renseignements sont accessibles dans un profil utilisateur, et vos utilisateurs sont regroupés par tenant. Les renseignements utilisateur eux-mêmes peuvent provenir de diverses sources, notamment des <Tooltip tip="Identity Provider (IdP) : Service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=identity+providers">fournisseurs d’identité</Tooltip>, de vos propres bases de données et des connexions d’entreprise (Active Directory, <Tooltip tip="Identity Provider (IdP) : Service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=SAML">SAML</Tooltip>).
