title: Premiers pas
description: Découvrez les notions de base et commencez à créer votre solution d’authentification.
---

Auth0 est une plateforme d’identité qui vous permet de gérer l’accès à vos applications.

Si vous débutez en gestion des identités et des accès (IAM), découvrez quelques notions de base et planifiez la solution la mieux adaptée à votre technologie et à vos besoins. Si vous connaissez bien l’IAM, vous pouvez passer directement à la création.

***

<div id="start-building">
  ## Commencez à créer
</div>

Pour vous lancer rapidement, choisissez votre type d’application et suivez un tutoriel Quickstart étape par étape.

<Columns cols={2}>
  <Card title="Application native/mobile" href="/docs/fr-CA/quickstart/native">
    Application mobile ou de bureau qui fonctionne nativement sur un appareil, comme iOS ou Android.
  </Card>

  <Card title="Application monopage" href="/docs/fr-CA/quickstart/spa">
    Application web Javascript qui s’exécute dans le navigateur, comme AngularJS + Node.js ou React.
  </Card>

  <Card title="Application web traditionnelle" href="/docs/fr-CA/quickstart/webapp">
    Application web traditionnelle qui s’exécute sur le serveur, comme Express.js ou ASP.NET.
  </Card>

  <Card title="Backend/API" href="/docs/fr-CA/quickstart/backend">
    Une API ou un service protégé par Auth0, comme une API Express.js ou une API ASP.NET.
  </Card>

  <Card title="Agent IA" href="https://auth0.com/ai/docs">
    Un agent IA qui appelle des API externes au nom de l’utilisateur et intègre des contrôles avec intervention humaine, comme LangChain ou LlamaIndex.
  </Card>
</Columns>
