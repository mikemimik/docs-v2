title: Tableau Server を SAMLサービスプロバイダーとして設定する
---

Tableau Server には、次の <Tooltip tip="Security Assertion Markup Language (SAML): パスワードなしで 2 者間の認証情報を交換できる標準化されたプロトコルです。" cta="用語集を見る" href="/ja/docs/glossary?term=SAML">SAML</Tooltip> 設定を使用します。

1. [Auth0 Dashboard &gt; アプリケーション &gt; アプリケーション](https://manage.auth0.com/#/applications) に移動し、新しいアプリケーションを作成するか、更新するアプリケーション名をクリックします。

2. **Addons** タブに移動し、**SAML2 Web App** トグルを有効にします。

3. **設定** タブで、**アプリケーションのコールバックURL** を次のように設定します: `http://{yourTableauServer}/wg/saml/SSO/index.html`。

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Settings Tab](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>

4. 次のコードを **設定** テキストボックスに貼り付け、**Debug** をクリックします。

   ```json lines
   {
        "audience":  "{yourTableauAudience}",
        "recipient": "http://{yourTableauServer}/wg/saml/SSO/index.html",
        "mappings": {
       "email": "username"
        },
        "createUpnClaim":       false,
        "passthroughClaimsWithNoMapping": false,
        "mapUnknownClaimsAsIs": false,
        "mapIdentities":        false,
        "signatureAlgorithm":   "rsa-sha1",
        "digestAlgorithm":      "sha1",
        "destination":          "http://{yourTableauServer}/wg/saml/SSO/index.html",
        "lifetimeInSeconds":    3600,
        "signResponse":         false,
        "nameIdentifierFormat": "urn:oasis:names:tc:SAML:2.0:attrname-format:basic",
        "nameIdentifierProbes": [
          "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/name"
        ]
       }
   ```

5. ページの一番下までスクロールし、**Enable** をクリックします。

6. **使用状況** タブで **IDプロバイダーのメタデータ** を見つけ、**Download** をクリックしてメタデータファイルをダウンロードします。これは、Auth0 をアイデンティティプロバイダーとして設定する際に必要です。

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Usage Tab](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>
