title: SAML2 Web App アドオンを有効にする
---

アプリケーションで使用する SAML2 Web App アドオンを設定できます。

1. [Auth0 Dashboard &gt; アプリケーション &gt; アプリケーション](https://manage.auth0.com/#/applications) に移動します。
2. SAML2 Web App を使用するアプリケーションを見つけて、そのアプリケーション名をクリックし、設定画面を開きます。
3. **Addons** タブに移動します。
4. **SAML2 Web App** ボックスで、スライダーをクリックしてアドオンを有効にします。

   <Frame>![Auth0 Dashboard アプリケーション アプリケーション Addons タブ SAML2 Web App Settings タブ](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>
5. **Settings** タブに移動し、SAML レスポンスを受け取る **Application Callback URL** と **Settings** を入力します。**Settings** タブには、有効にする設定を決める際に使えるサンプルオブジェクトがあります。
6. **Debug** をクリックして、指定したパラメーターが有効かどうかをテストします。
7. 完了したら **Save** をクリックします。

  ## 制限事項
</div>
