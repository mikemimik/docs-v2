title: パスワードレス認証向けにメールまたは SMS を設定する
---

Auth0 では、<Tooltip tip="Universal Login: アプリケーションは、ユーザーの本人確認を行うために、Auth0 の認可サーバーでホストされている Universal Login にリダイレクトされます。" cta="用語集を見る" href="/ja/docs/glossary?term=Universal+Login">Universal Login</Tooltip> を設定することで、ユーザーはメールで受け取るマジックリンクまたはワンタイムパスワード (OTP) 、あるいは SMS で受け取る OTP を使って認証できます。

SMS またはメールで <Tooltip tip="パスワードレス: 第1認証要素としてパスワードに依存しない認証の形式です。" cta="用語集を見る" href="/ja/docs/glossary?term=Passwordless">パスワードレス</Tooltip>認証を有効にするには、アプリケーション用の [パスワードレス接続](/ja/docs/authenticate/passwordless) を定義する必要があります。

接続を作成したら、それを使用するように Universal Login を設定する必要があります。方法は 3 つあります。

<div id="universal-login">
  ## Universal Login
</div>

この方法では、パスワードレス接続を標準でサポートする Universal Login エクスペリエンスを使用します。詳しくは、[Universal Login でのパスワードレス](/ja/docs/authenticate/passwordless/passwordless-with-universal-login)をご覧ください。

<div id="universal-login-lock-passwordless">
  ## Universal Login + Lock (パスワードレス)
</div>

この方法では、**Lock (パスワードレス)** テンプレートを使用したカスタムログインページを使って、[Lock for Web SDK](/ja/docs/libraries/lock) でユーザーを認証します。

1. [Auth0 Dashboard &gt; Branding &gt; Universal Login](https://manage.auth0.com/#/login_settings) に移動し、**Login** タブをクリックします。
2. **Custom Login Page** のトグルを有効にし、**Lock (パスワードレス)** テンプレートを選択します。すると HTML テンプレートが更新され、パスワードレス向けのカスタマイズオプションを備えた Lock widget を使用するコードが表示されます。
3. テンプレートをカスタマイズし、**変更を保存** をクリックします。

HTML と CSS を使用してログインフォームをカスタマイズでき、変更内容は Auth0 Dashboard 内でプレビューできます。**Lock (パスワードレス)** テンプレートのカスタマイズ方法について詳しくは、[Lock for Web SDK](/ja/docs/libraries/lock) のパスワードレスのセクションを参照してください。

<div id="universal-login-custom-ui-auth0js">
  ## Universal Login + カスタム UI + Auth0.js
</div>

この方法では、**Custom Login Form** テンプレートを使用したカスタムログインページを使い、[Auth0.js SDK](/ja/docs/libraries/auth0js) を使用してユーザーを認証します。詳しくは、[Universal Login を使用したパスワードレス認証](/ja/docs/authenticate/passwordless/implement-login/universal-login) をご覧ください。
