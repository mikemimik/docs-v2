description: Auth0 Dashboard を使用して、AD/LDAP エンタープライズ接続の接続レベルで資格情報のキャッシュを無効にする方法について説明します。
title: AD/LDAP 接続の資格情報キャッシュを無効にする
---

[AD/LDAP 接続](/ja/docs/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector)では、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 のメイン製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用して、接続レベルで資格情報のキャッシュを無効にできます。

1. [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise) に移動し、**Active Directory/LDAP** の接続タイプを選択します。
2. 編集する接続名を選択します。
3. **Disable Cache** トグルを有効にします。
