description: Auth0 Dashboard を使用して、アプリケーションでエンタープライズ接続を有効にする方法について説明します。
title: エンタープライズ接続を有効にする
---

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用すると、アプリケーションのエンタープライズ接続を有効にできます。エンタープライズ接続を有効にするには、事前に設定が完了している必要があります。

1. [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise) に移動し、表示する接続タイプを選択します。

   <Frame>![Auth0 Dashboard - 接続 - Enterprise](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. 表示する接続の名前を選択します。

   <Frame>![Auth0 Dashboard - 接続 - Enterprise - AD/LDAP - 一覧](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/LDAP_page_-_EN.png)</Frame>
3. **アプリケーション**ビューを選択し、該当するアプリケーションに対して接続を有効または無効にします。

   <Frame>![Auth0 Dashboard - 接続 - Enterprise - アプリケーション](/docs/images/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/38fbae514304297bbbf72beeedb8892f/SSL_-_EN.png)</Frame>

エンタープライズ <Tooltip tip="IDプロバイダー（IdP）: デジタルアイデンティティを保存および管理するサービス。" cta="用語集を見る" href="/ja/docs/glossary?term=IdP">IdP</Tooltip> 接続を設定する際は、Auth0 の `/authorize,` `/whr,` または `/wsfed` エンドポイントに渡される `connection` パラメータにより、データベースまたは Active Directory 経由で接続されている場合を**除き**、アップストリームの IdP にリダイレクトされることに注意してください。
