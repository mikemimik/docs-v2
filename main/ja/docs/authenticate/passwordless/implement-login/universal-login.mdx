title: Universal Login を使用したパスワードレス認証
---

[Universal Login](/docs/ja/authenticate/login/auth0-universal-login) は、Auth0 が提供するログインフローの実装です。ユーザーが本人確認を行う必要があるたびに、アプリケーションは <Tooltip tip="Universal Login: ユーザーの本人確認を行うために、アプリケーションは Auth0 の Authorization Server でホストされている Universal Login にリダイレクトします。" cta="用語集を見る" href="/docs/ja/glossary?term=Universal+Login">Universal Login</Tooltip> にリダイレクトされ、Auth0 がユーザーの本人確認に必要な処理を行います。これは、<Tooltip tip="パスワードレス: 第1認証要素としてパスワードに依存しない認証の形式です。" cta="用語集を見る" href="/docs/ja/glossary?term=Passwordless">パスワードレス</Tooltip>認証を実装する際に推奨される方法です。

Universal Login でパスワードレス認証を実装するには、ログインページの HTML をカスタマイズする必要があります。

1. [Auth0 Dashboard &gt; Branding &gt; Universal Login](https://manage.auth0.com/#/login_settings) に移動し、**Advanced Options** をクリックします。

2. **Login** タブをクリックし、**Customize Login Page** トグルを有効にします。

<Frame>![Custom Login](/docs/images/cdy7uua7fh8z/5iwLcz2O100hLuxlKN4rSs/e4f35f8c44516b98b3584d56bcaac12d/manage.auth0.com_dashboard_us_pavo_login_page__2_.png)</Frame>

3. Default Template ドロップダウンメニューから、**Custom Login Form** テンプレートを選択します。すると、HTML テンプレートが CSS と auth0.js SDK を使用したコードに更新されます。

4. HTML と CSS を使ってテンプレートを要件に合わせてカスタマイズし、**Save Changes** をクリックします。**Custom Login Form** テンプレートで auth0.js SDK を使用する方法の詳細については、[Auth0.js SDK](/docs/ja/libraries/auth0js) を参照してください。

5. **Preview** タブを選択して変更内容を確認します。ログインページをプレビューする対象の正しいアプリケーションを選択してください。

アプリケーションに Universal Login を統合する方法については、[Quickstarts](/docs/ja/quickstarts) を参照してください。すべてのアプリケーションタイプと主要なプラットフォーム向けの完全な例が掲載されています。
