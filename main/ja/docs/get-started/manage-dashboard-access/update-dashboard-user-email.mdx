description: Auth0 Dashboard のテナントメンバーのメールアドレスを更新する方法を学びます。
title: Dashboard ユーザーのメールアドレスを更新する
---

テナント管理者は、新しいメールアドレスで新しいユーザーを追加し、古いメールアドレスに紐づくユーザーを削除することで、既存の<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 のメイン製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>メンバーに関連付けられたメールアドレスを更新できます。

1. [Auth0 Dashboard &gt; 設定 &gt; Tenant Members](https://manage.auth0.com/#/tenants/admins) に移動します。
2. **Add Member** をクリックします。
3. 新しいメールアドレスを入力し、アクセス権限を選択します。
4. **Invite** をクリックして、ユーザーに招待メールを送信します。
5. ユーザーが招待を承諾したら、古いメールアドレスに紐づくユーザーを削除します。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 Dashboard または API を通じて、エンドユーザーが自分自身のメールアドレスを更新することはできません。アカウントに関連付けられたメールアドレスを更新する必要がある場合は、管理者にお問い合わせください。
</Callout>
