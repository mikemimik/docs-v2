---
description: Auth0 テナントの Auth0 Dashboard へのアクセス権を持つテナントメンバーを削除する方法を説明します。
title: テナントメンバーを削除
---

テナント管理者であれば、Auth0 Dashboard へのアクセス権を取り消すことができます。

1. [Auth0 Dashboard &gt; 設定 &gt; テナントメンバー](https://manage.auth0.com/#/tenant/admins) に移動します。
2. ユーザー名またはメールアドレスの右側にある **...** をクリックし、**削除** をクリックします。
3. **保存** をクリックします。
