description: Auth0 テナントの Auth0 Dashboard ユーザーのロールを編集する方法を説明します。
title: テナントメンバーの編集
---

テナント管理者であれば、ユーザーに割り当てられたロールを変更できます。変更は、ユーザーが Auth0 Dashboard を再読み込みすると反映されます。

1 人のユーザーに複数のロールを割り当てることができ、その場合は権限が統合されます。あらかじめテナントメンバーを追加しておく必要があります。

1. [Auth0 Dashboard &gt; 設定 &gt; テナントメンバー](https://manage.auth0.com/#/tenant/admins) に移動します。
2. ユーザー名またはメールアドレスの右側にある **...** をクリックし、**ロールを編集** をクリックします。
   このオプションが表示されない場合は、そのユーザーがまだ招待を承諾していないことを意味します。
3. ロールを追加で選択するか、不要なロールを削除します。
4. **保存** をクリックします。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  ユーザーが Auth0 Dashboard を再読み込みすると、権限が表示されます。再読み込みしない場合、特定の領域へのアクセス権がないことを示すエラーが表示されることがあります。そのようなエラーが発生した場合は、いったんログアウトしてから再度ログインする必要があります。
</Callout>

**パスワードリセット**: あなたまたは Auth0 Dashboard ユーザーのいずれかがログイン時に問題を抱えている場合は、詳しくは [Reset Auth0 Account Password](/ja/docs/troubleshoot/customer-support/reset-account-passwords) を参照してください。

**メールアドレスが無効**: テナントメンバーのメールアドレスを確認し、必要に応じて更新する方法については、[Update Dashboard User Email Addresses](/ja/docs/get-started/manage-dashboard-access/update-dashboard-user-email) を参照してください。
