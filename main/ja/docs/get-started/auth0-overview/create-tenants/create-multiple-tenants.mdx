---
description: Auth0 Dashboard を使用して追加のテナントを作成する方法を説明します。
title: 複数のテナントを作成する
---

複雑な構成に対応するために、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> では、[異なる環境](/ja/docs/get-started/auth0-overview/create-tenants/set-up-multiple-environments)を作成できるよう、複数のテナントを設定できます。たとえば、2 つの別々のドメイン (1 つは社内向け、もう 1 つは一般公開向け) を使用したり、アプリケーションごとにユーザーのログイン方法を変えたい場合があります。これを実現するには、複数の Auth0 テナントを作成します。そうすることで、サポートする必要があるアプリケーションやユーザーグループごとに、アプリケーション、コネクション、ユーザーのセットを分けて管理できます。

1. [Auth0 Dashboard](https://manage.auth0.com/#/) に移動し、テナント名を選択して、**Create Tenant** を選択します。

   <Frame>![Dashboard Tenant Drop-Down Menu Create Tenant](/docs/images/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/5efad1137eb8b04031b30137c2fc5ac5/2025-02-26_14-51-39.png)</Frame>
2. 希望する **Tenant Domain** を入力し、**Region** を選択して、**Create** を選択します。
