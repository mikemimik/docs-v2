title: 複数のテナントを1つのサブスクリプションにリンクする
validatedOn: 2026-03-04
---

Auth0では、Enterpriseサブスクリプションをご利用のお客様向けに、1つのAuth0サブスクリプションに複数のテナントをリンクする機能を提供しています (これらのリンクされたテナントは、子テナントと呼ばれることもあります) 。

この機能は、次のような場合に役立ちます。

* 開発用テナントと本番用テナントを分けながら、開発用テナントでも本番用テナントと同じ機能を利用できるようにする
* 1つのEnterpriseサブスクリプションで複数の本番用テナントを保有する

<div id="linking-tenants">
  ## テナントのリンク
</div>

Enterprise サブスクリプションでは、新しいテナントの作成時に **Create Under** ドロップダウンから **Custom Agreement** オプションを選択すると、既存の Auth0 サブスクリプション配下にテナントを自動的にリンクできます。

<Frame>![](/docs/images/cdy7uua7fh8z/7ggqdv5yw011z0TsdlaHOO/7707d826dd585e074bd0d8e25b30d9d6/publiccloudtenants.png)</Frame>

現在 Enterprise サブスクリプションに含まれていない既存のテナントをリンクする必要がある場合は、アカウント担当チームに連絡するか、[Auth0 Support](https://support.auth0.com) にチケットを作成してください。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  セルフサービス型のサブスクリプション (Enterprise 以外) をご利用の場合でも、テナントをリンクできます。詳しくは、[Tenant Management](/ja/docs/get-started/auth0-teams/tenant-management) をご覧ください。
</Callout>

<div id="usage-consolidation">
  ## 使用状況の集約
</div>

リンクされたすべてのテナントの使用状況は、Enterprise サブスクリプションの上限に算入されます。リンクされたすべてのテナントを合算した使用状況は、使用状況レポートおよびクォータ レポートで確認できます。
