description: Auth0 DashboardのAdminがログインセッションを確認し、取り消す方法を説明します。
title: Auth0 Dashboard ログインセッション管理
---

Login Sessions を使用すると、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> の管理者は、複数のデバイスやブラウザーにまたがる各 Dashboard アプリケーションのアクティブなセッションを追跡できます。Teams Dashboard、Support Center、Auth0 Dashboard へのアクセスによって作成されたユーザーセッションを確認し、取り消すことができます。

Auth0 Dashboard の[プロファイルページ](https://manage.auth0.com/#/profile/sessions)では、アクティブなセッションの一覧を確認できます。

<Frame>![Auth0 Dashboard > プロファイル](/docs/images/cdy7uua7fh8z/3hBNRo2Bc4w6oKpF0mMt4o/3796b322593eae38eeba0a3af3698eca/English_user_profle_login_sessions.png)</Frame>

ログアウトするかセッションを取り消すには、対象のセッションを見つけて **取り消す** を選択します。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  セッションを取り消しても、Extensions に属するセッションは自動的には削除されません。ただし、Extensions に属するセッションは有効期間が短く、ログインセッションが取り消された場合や期限切れになった場合でも更新されません。
</Callout>
