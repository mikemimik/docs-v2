title: データベース接続をテストする
---

<Tooltip tip="サービスの設定を行う Auth0 のメイン製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用すると、アプリケーションのデータベース接続をテストできます。設定済みのデータベース接続は、アプリケーションへのログインに使用できます。

適切にテストするには、事前に[データベース接続を設定](/ja/docs/get-started/applications/set-up-database-connections)し、そのデータベース接続用の[ユーザーを作成](/ja/docs/manage-users/user-accounts/create-users)しておく必要があります。

1. [Auth0 Dashboard &gt; Authentication &gt; Database](https://manage.auth0.com/#/connections/database) に移動し、テストする接続の横にある **試す** の矢印を選択します。

   <Frame>![Dashboard - Authentication - Database - データベース接続一覧](/docs/images/cdy7uua7fh8z/1Hf6ctA9xhsLOQO9IyG0VU/9061b8436368e003b165efb890d86473/DB_Connections_-_EN.png)</Frame>
2. テストユーザーのユーザー名とパスワードを入力します。正しく設定できていれば、**It Works!** ページが表示されます。

   <Frame>![Dashboard - Connections - Database - 試す - 成功](/docs/images/cdy7uua7fh8z/5DWJWLf8zvZJH0rSlhK0cr/f33377edf6f82f17e11e08ec30d8d75a/connection-social-try-success.png)</Frame>
