<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Guide は現在 早期アクセス 段階にあり、米国の Public Cloud リージョンのユーザーに順次展開されています。Guide が不完全または不正確な回答を返す場合があるため、必ず提供されるリンクを使って内容を[確認し、検証](#how-does-guide-avoid-ai-hallucinations%3F)してください。Auth0 のリリースの詳細については、[Product Release Stages](/docs/ja/troubleshoot/product-lifecycle/product-release-stages) を参照してください。
</Callout>

Guide は、Auth0 に関する質問に答える AI 搭載のチャットボットです。情報源として、[Documentation](https://auth0.com/docs)、[Blog](https://auth0.com/blog)、および [Community](http://community.auth0.com) を利用します。Enterprise subscription のお客様向けには、Guide は [Security Center](/docs/ja/secure/security-center) の Auth0 テナント データも活用し、テナントのセキュリティに関するベストプラクティスに沿って、監視、トラブルシューティング、実装を支援します。

Guide は、「パスキーを設定するにはどうすればよいですか？」のような質問に対して、要約された回答と、テナントおよびユーザーのセキュリティに関連するドキュメントや[インシデントプレイブック](/docs/ja/secure/attack-protection/playbooks)へのリンクを提示します。

<div id="how-guide-works">
  ## Guide の仕組み
</div>

Guide はメッセージを解析して質問内容を判断し、関連情報を Auth0 のソースから検索したうえで、見つかった内容を要約し、参照した資料へのリンクとともに回答を生成します。

続けて質問したり、提供されたリンクを使って[回答を検証](#how-does-guide-avoid-ai-hallucinations%3F)したり、回答を評価して Guide にフィードバックを送ったりできます。

Enterprise subscription をご利用のお客様向け: Security Center で[セキュリティ監視アラートを設定](/docs/ja/secure/security-center/security-alerts)すると、Guide がインシデントのトラブルシューティングを支援し、関連する[インシデントプレイブック](/docs/ja/secure/attack-protection/playbooks)に基づくセキュリティガイダンスを提案します。

たとえば、次のスクリーンショットのように、大規模なボットトラフィックの急増によって <Tooltip tip="ボット検出: Auth0 がログインプロセス中に CAPTCHA を有効化して、不審なボットトラフィックをブロックする攻撃対策の一種。" cta="用語集を見る" href="/docs/ja/glossary?term=Bot+Detection">ボット検出</Tooltip> アラートが発報されると、Guide は影響を受ける認証フローで CAPTCHA を有効にするなど、[Bot Detection Attack playbook](/docs/ja/secure/attack-protection/playbooks/bot-detection-playbook) の対処手順を推奨し、テナントログ内の関連イベントを確認するよう促します。

<div id="use-guide">
  ## Guide を使う
</div>

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>の右上にある **Ask Guide** ボタンを選択すると、Guide にアクセスできます。

**Ask Guide** を選択すると、Guide がポップアップで表示され、Auth0 の機能について質問できます。Auth0 Dashboard 内で現在表示している場所に応じて、関連する質問が提案されることもあります。たとえば、**Branding &gt; <Tooltip tip="Universal Login: アプリケーションは、ユーザーのアイデンティティを確認するために、Auth0 の認可サーバーでホストされている Universal Login にリダイレクトされます。" cta="用語集を見る" href="/docs/ja/glossary?term=Universal+Login">Universal Login</Tooltip>** を開いている場合、Guide には Universal Login に関する次のような質問が表示されます。

* Universal Login とは何ですか？
* Universal Login のメリットは何ですか？

<Frame>![Auth0 Dashboard の右上にある Ask Guide ボタンを選択して開きます。](/docs/images/cdy7uua7fh8z/e3lTjWgGC16Ob6zeUfU1V/ea8eef92439697bced3c147c9ca87d87/image3.png)</Frame>

<div id="questions-guide-can-answer">
  ### Guide が回答できる質問
</div>

Guide は、次のような Auth0 の製品やプラットフォームに関する質問に回答できます。

* Universal Login とは何ですか？
* <Tooltip tip="Single Sign-On (SSO): ユーザーが1つのアプリケーションにログインすると、その後ほかのアプリケーションにも自動的にログインできるようになるサービス。" cta="用語集を見る" href="/docs/ja/glossary?term=single+sign-on">シングルサインオン</Tooltip>連携を有効にするにはどうすればよいですか？
* パスキーを設定するにはどうすればよいですか？

<Frame>![Guide は、Auth0 の製品やプラットフォームに関する質問に回答できます。](/docs/images/cdy7uua7fh8z/nGVAVWxnoqiXaFXTQQyHX/101b6fdfde5f8488d6d2401e5dee5a42/auth0-guide-sso-screenshot.png)</Frame>

エンタープライズサブスクリプションをご利用のお客様向けに、Guide は Security Center のメトリクスやデータに関する Auth0 テナント関連の質問にも回答できます。たとえば、次のようなものです。

* この失敗した <Tooltip tip="多要素認証 (MFA): SMS で送信されるコードなど、ユーザー名とパスワードに加えて認証要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/docs/ja/glossary?term=MFA">MFA</Tooltip> チャレンジをトラブルシューティングするにはどうすればよいですか？
* ボット保護のアラートしきい値がトリガーされた原因は何ですか？
* 「クレデンシャルスタッフィング」アラートへの対応手順は何ですか？

<div id="questions-guide-cannot-answer">
  ### Guide では回答できない質問
</div>

Guide は、Auth0 の製品やプラットフォームに関係のない質問や、次のようなテナントや設定に固有の質問には回答できません。

* 有効になっている接続はどれですか？
* 先月、私のアプリケーションにログインしたユーザーは何人ですか？
* Vim と Emacs では、どちらが優れていますか？

<Warning>
  個人情報やその他の機密データを Guide に送信しないでください。
</Warning>

<div id="disable-guide">
  ## Guide を無効にする
</div>

[Auth0 Support](https://support.auth0.com/) に連絡すると、テナント内で Guide を無効にできます。

<div id="data-security-in-guide">
  ## Guide におけるデータセキュリティ
</div>

お客様のメッセージは、Okta のエンタープライズレベルのセキュリティおよびコンプライアンス基準によって保護されています。

Guide が行うこと:

* 提供されたメッセージを使用して、公開されている Auth0 のソースから関連情報を検索します。

Guide が行わないこと:

* お客様のアカウントやtenantデータにアクセスすること。
* 提供されたメッセージを LLM の学習目的で使用すること。
* お客様のメッセージを第三者と共有すること。

<div id="how-does-guide-avoid-ai-hallucinations">
  ## GuideはどのようにAIのハルシネーションを防いでいますか？
</div>

Guideは、Auth0が作成したコンテンツをもとに回答を生成し、すべての回答に参照元ドキュメントへのリンクを含めています。参照されたリンクをたどることで、元のソースデータを確認し、さらに詳しい情報を得ることができます。
