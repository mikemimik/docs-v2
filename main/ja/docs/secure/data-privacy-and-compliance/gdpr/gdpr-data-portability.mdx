title: 'GDPR: データポータビリティ'
---

GDPR 第20条に基づき、ユーザーには、自身に関する個人データを、構造化され、一般的に使用され、機械で読み取り可能な形式で受け取る権利があります。

Auth0 のユーザーストアに保存されているユーザーデータは、手動またはプログラムでエクスポートできます。Auth0 の生データは、JSON 形式 (機械で読み取り可能) でエクスポートできます。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  これらの文書の内容は、法的助言を目的としたものではなく、法的支援に代わるものでもありません。GDPR の内容を理解し、遵守する最終的な責任はお客様にありますが、Auth0 は可能な限り GDPR 要件への対応を支援します。
</Callout>

<div id="export-data-manually">
  ## データを手動でエクスポートする
</div>

1. [Auth0 Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) に移動します。
2. ユーザーを検索し、名前をクリックして詳細を開きます。
3. **Raw JSON** タブをクリックします。ここで、JSON 形式の完全なユーザープロファイルを確認できます。
4. **Copy JSON** をクリックします。プロファイルがクリップボードにコピーされます。
5. クリップボードの内容をエディタに貼り付けて保存します。

<div id="export-data-using-the-api">
  ## API を使ってデータをエクスポートする
</div>

<Tooltip tip="Management API: お客様が管理タスクを実行するための製品。" cta="用語集を見る" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使うと、ユーザーの完全なプロファイルをエクスポートできます。レスポンスは JSON 形式です。ユーザー ID を使って特定のユーザーを検索することも、ユーザーの一覧をエクスポートすることもできます。
