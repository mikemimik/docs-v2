お使いのテナントで[リカバリーコードが有効](/docs/ja/secure/multi-factor-authentication/configure-recovery-codes-for-mfa)になっている場合、Auth0 は、ユーザーが <Tooltip tip="多要素認証（MFA）: SMS によるコードなど、ユーザー名とパスワードに加えて認証要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/docs/ja/glossary?term=multi-factor+authentication">多要素認証</Tooltip> (MFA) に登録すると、自動的にリカバリーコードを生成します。ユーザーはこのリカバリーコードを保存しておく必要があります。このコードは、MFA に使用しているデバイスやアカウントにアクセスできなくなった場合に、後から使用できます。

リカバリーコードが無効になっている場合、ユーザーの最初の MFA 認証要素を関連付けても MFA API はリカバリーコードを返さず、ユーザーはリカバリーコードを使って認証することもできません。

MFA API を使用すると、ユーザーがリカバリーコードを使って認証できるようにできます。

1. ユーザーにリカバリーコードの入力を求めます。ユーザーが認証するには、その値をアプリケーションに入力する必要があります。

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Auth0 は、DUO およびレガシー `google-authenticator` 認証要素のリカバリーコードは生成しません。
   </Callout>
2. リカバリーコードで認証します。リカバリーコードを使って認証し、新しいリカバリーコードを生成するには、OAuth Token エンドポイントを呼び出します。次のパラメータを指定する必要があります。

   | パラメータ           | 値                                                     |
   | --------------- | ----------------------------------------------------- |
   | `grant_type`    | `http://auth0.com/oauth/grant-type/mfa-recovery-code` |
   | `recovery_code` | ユーザーから提供されたリカバリーコード。                                  |

<AuthCodeGroup>
  ```bash cURL
  curl --request POST \
  --url 'https://{yourDomain}/oauth/token' \
  --header 'content-type: application/x-www-form-urlencoded' \
  --data grant_type=http://auth0.com/oauth/grant-type/mfa-recovery-code \
  --data 'client_id={yourClientId}' \
  --data 'client_secret={yourClientSecret}' \
  --data 'mfa_token={mfaToken}' \
  --data 'recovery_code={recoveryCode}'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/oauth/token");
  var request = new RestRequest(Method.POST);
  request.AddHeader("content-type", "application/x-www-form-urlencoded");
  request.AddParameter("application/x-www-form-urlencoded", "grant_type=http%3A%2F%2Fauth0.com%2Foauth%2Fgrant-type%2Fmfa-recovery-code&client_id={yourClientId}&client_secret=%7ByourClientSecret%7D&mfa_token=%7BmfaToken%7D&recovery_code=%7BrecoveryCode%7D", ParameterType.RequestBody);
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
     "fmt"
     "strings"
     "net/http"
     "io/ioutil"
  )

  func main() {

     url := "https://{yourDomain}/oauth/token"

     payload := strings.NewReader("grant_type=http%3A%2F%2Fauth0.com%2Foauth%2Fgrant-type%2Fmfa-recovery-code&client_id={yourClientId}&client_secret=%7ByourClientSecret%7D&mfa_token=%7BmfaToken%7D&recovery_code=%7BrecoveryCode%7D")

     req, _ := http.NewRequest("POST", url, payload)

     req.Header.Add("content-type", "application/x-www-form-urlencoded")

     res, _ := http.DefaultClient.Do(req)

     defer res.Body.Close()
     body, _ := ioutil.ReadAll(res.Body)

     fmt.Println(res)
     fmt.Println(string(body))

  }
  ```


3. ユーザーにリカバリーコードを保存するよう促します。呼び出しが成功すると、認証トークンと新しいリカバリーコードが返されます。

   ```json lines
   {
       "access_token": "O3...H4",
       "id_token": "eyJh...w",
       "scope": "openid profile",
       "expires_in": 86400,
       "recovery_code": "K6LGLV3RSH3VERMKET8L7QKU",
       "token_type": "Bearer"
   }
   ```

4. 新しいリカバリーコードが生成されたことをユーザーに通知し、保存するよう求めます。
