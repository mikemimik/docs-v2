* [許可リスト用の Auth0 IP アドレス](/ja/docs/secure/security-guidance/data-security/allowlist) - ご利用の環境がファイアウォールの内側にある場合、カスタムデータベース接続、Actions、フック、ルールを正しく機能させるために、特定の Auth0 IP アドレスを許可リストに追加する必要があることがあります。
* [ユーザー属性を拒否リストに追加する](/ja/docs/secure/security-guidance/data-security/denylist) - プライバシー上の理由から Auth0 データベースに保存すべきでないユーザーフィールドがある場合は、ユーザー属性を拒否リストに追加できます。
* [ユーザーデータの保存](/ja/docs/secure/security-guidance/data-security/user-data-storage) - ユーザーデータの保存に Auth0 データストアを使用する場合と、外部データベースを使用する場合の違いについて学びます。
* [トークンの保存](/ja/docs/secure/security-guidance/data-security/token-storage) - トークンベースの認証で使用されるトークンを、どのように、どこへ保存するかを学びます。
