title: Datadog で Auth0 ダッシュボードテンプレートを使用する
---

Auth0 には、Datadog で Auth0 テナントのデータを可視化するために使える定義済みテンプレートが用意されています。これらの可視化を表示することで、テナントのログイントラフィックの健全性を監視できます。テンプレートを使うと、Auth0 が推奨する集計を利用したり、独自のカスタム可視化を作成する際の出発点として活用したりできます。

| テンプレート        | 用途                                               | ダウンロード可能なファイル                                                                                                 |
| ------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------------------------------- |
| セキュリティダッシュボード | ログイン、アカウント作成、異常検知のメトリクスを含むテナントのログイントラフィックを監視します。 | [Auth0SecDashboard.json](https://cdn.auth0.com/website/docs/logs/streams/datadog/Auth0SecurityDashboard.json) |

また、次の機能を使って可視化を整理することもできます。

* Datadog の時間選択機能を使用して、可視化の対象期間を選択します。
* フィルターを使用して、IP アドレス、国、ユーザー、クライアントアプリケーションなどの設定可能なパラメーターでデータ分析の範囲を絞り込みます。
* **Edit Widgets** を使用して、タイルや可視化を並べ替えたり、必要に応じてクエリを確認してパラメーターを追加できるコピーを作成したりできます。

<div id="prerequisites">
  ## 前提条件
</div>

* [Datadog へのログストリーミングを設定する](https://marketplace.auth0.com/integrations/datadog-log-streaming).
* Auth0 から必要なダッシュボードテンプレートのJSONファイルをダウンロードします。

<div id="upload-dashboard-templates-in-datadog">
  ## Datadog でダッシュボードテンプレートをアップロードする
</div>

1. [Datadog](https://www.datadoghq.com/)にログインします。
2. 左側のナビゲーションバーで **Dashboards** に移動します。
3. ダッシュボードの種類として **Screenboard** を選択します。
4. 右上の空白のキャンバスで歯車アイコンを選択し、**Import Dashboard JSON** をクリックします。
5. 使用するダッシュボードの JSON ファイル (例: `Auth0SecDashboard.json`) をアップロードします。

<div id="security-dashboard-visualizations">
  ## セキュリティダッシュボード の可視化
</div>

Auth0 から セキュリティダッシュボード をインポートすると、以下の可視化を使用してテナントの健全性を確認したり、Datadog で独自の可視化やアラートを作成する際の出発点にしたりできます。

| 可視化                                      | 目的                                                                                                    |
| ---------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| Login Success                            | テナントで成功したログインイベント数を表示します。                                                                             |
| Login Failures                           | テナントで失敗したログインイベント数を表示します。ログイン失敗の急増は、異常なトラフィックの兆候である可能性があります。                                          |
| Login Failures by Geolocation            | テナントで失敗したログインイベント数を地理的位置別に表示します。特定の地域で予期しないトラフィックの急増が見られる場合、異常なトラフィックの兆候である可能性があります。                  |
| Number of Unique IPs sending traffic     | テナントにトラフィックを送信しているユニーク IP アドレス数を表示します。この数は多くの場合、予測可能なパターンに従うため、大きな変動は異常なトラフィックの兆候である可能性があります。         |
| Top 5 IPs with failures                  | ログインエラー数が最も多い上位 5 件の IP アドレスを表示します。                                                                   |
| Account Creation Events in top countries | 上位の国々で作成されたアカウント数を表示します。通常のパターンや地域分布に変化が見られる場合、異常なトラフィックの兆候である可能性があります。                               |
| Account Creation Events by Platform      | Windows、OSX、Android、iOS など、プラットフォーム別に作成されたアカウント数を表示します。                                               |
| IPs with most accounts created           | 最も多くのアカウントが作成された IP アドレスを表示します。                                                                       |
| Anomaly Detection Events                 | Brute Force Protection や Breached Password Detection によりブロックされた IP アドレス数を含む、テナントで発生した異常検知イベント数を表示します。 |
| Log Stream                               | テナントログの生イベントを表示します。                                                                                   |

<div id="customize-dashboard">
  ## ダッシュボードをカスタマイズする
</div>

Datadog Auth0 セキュリティダッシュボードは、カスタムデータウィジェットを追加するようにカスタマイズできます。

1. Auth0 セキュリティダッシュボードに移動し、サイドナビゲーションの **Dashboards** を開きます。
2. 一覧から、作成したダッシュボードの名前を選択します。
3. 可視化を選択し、ページのメインセクションにドラッグします。
4. データグラフの設定を行います。
