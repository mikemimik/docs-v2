---
description: ログストリームで使用できるログフィルターと、各フィルターに含まれるログを一覧表示します。
title: ログストリームフィルター
---

ログストリームをフィルタリングして、選択したログとログカテゴリのみを配信できます。ログストリームのフィルターを設定するには、次の手順に従います。

1. [Dashboard &gt; Monitoring &gt; Streams](https://manage.auth0.com/#/log-streams) に移動します。
2. ログストリームを選択し、**Settings** タブを開きます。
3. **Filter By Event Category** メニューでフィルターの有効／無効を切り替えます。
4. **Save** をクリックします。
