この記事では、Auth0のユーザーデータをCSVファイルにエクスポートし、その後Sailthruのダッシュボードにインポートする方法を学びます。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Auth0 Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。Extension のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Sailthru では CSV 形式のファイルをインポートできるため、`Tab Separated Value file (*.csv)` オプションを選択してください。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性について **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ### ユーザーデータファイルをインポートする
</div>

開始する前に、[Sailthru Documentation: Sailthru と Sailthru Lists へのユーザーの追加](https://getstarted.meetmarigold.com/engagebysailthru/Content/audience/managing-users/add-users-lists-st.html?TocPath=Audience%7CManaging%20Users%7C_____3#ListFileUpload)を参照すると、詳細を確認できます。

CSV ファイルを Sailthru にインポートするには、次の手順に従います。

1. Sailthru にログインし、[Lists](https://my.sailthru.com/lists) に移動します。
2. リストを選択し、**Upload List** をクリックして CSV ファイルを選択します。
3. **Action** で **Add To List** を選択します。
4. **Replace Vars** オプションを設定し、必要に応じて通知を受け取るメールアドレスを入力します。
5. 設定内容を確認し、**Submit** をクリックしてファイルをアップロードします。

これで完了です。Auth0 ユーザーは Sailthru に正常にインポートされました。
