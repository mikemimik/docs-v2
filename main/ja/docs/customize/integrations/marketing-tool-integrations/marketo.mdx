この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートし、Marketo REST API の [Bulk Leads endpoint](https://developer.adobe.com/marketo-apis/api/mapi/#tag/Bulk-Import-Leads) を使って Marketo にインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Auth0 Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。Extension ページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Marketo は CSV 形式でのファイルのインポートに対応しているため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性に対して **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ### ユーザーデータファイルをインポートする
</div>

開始する前に、詳細については [Marketo Documentation: Bulk Lead Import](https://experienceleague.adobe.com/en/docs/marketo-developer/marketo/rest/bulk-import/bulk-lead-import) を参照してください。

ユーザーデータファイルを Marketo にインポートするには、[Bulk Leads endpoint](https://developer.adobe.com/marketo-apis/api/mapi/#operation/importLeadUsingPOST) に対して POST リクエストを送信します。リクエストの content-type ヘッダーを `multipart/form-data` に設定し、エクスポートした CSV ファイルを含む `file` パラメータと、`csv` に設定した format パラメータを含めます。例:

<AuthCodeGroup>
  ```bash cURL lines
  curl --request POST \
    --url https://marketo_rest_api_base_url/bulk/v1/leads.json \
    --header 'authorization: Bearer {MARKETO_ACCESS_TOKEN}' \
    --form file=@auth0_users.csv \
    --form format=csv
  ```

  ```csharp C# lines
  var client = new RestClient("https://marketo_rest_api_base_url/bulk/v1/leads.json");
  var request = new RestRequest(Method.POST);
  request.AddHeader("authorization", "Bearer {MARKETO_ACCESS_TOKEN}");
  request.AddHeader("content-type", "multipart/form-data; boundary=---011000010111000001101001");
  request.AddParameter("multipart/form-data; boundary=---011000010111000001101001", "-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"file\"; filename=\"auth0_users.csv\"\r\nContent-Type: text/csv\r\n\r\n\r\n-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"format\"\r\nContent-Type: text/plan\r\n\r\ncsv\r\n-----011000010111000001101001--\r\n", ParameterType.RequestBody);
  IRestResponse response = client.Execute(request);
  ```
