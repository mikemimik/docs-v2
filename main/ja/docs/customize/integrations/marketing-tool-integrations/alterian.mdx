この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートし、その後 Campaign Manager の Data Import ツールを使って Alterian にインポートする方法を学びます。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Auth0 Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Alterian は CSV 形式でのファイルのインポートに対応しているため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性に対して **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

始める前に、[Alterian Campaign Manager: Data Import](https://cm.help.alterian.com/CM404/Default.htm#Customer_Analytics/Import_Export/Data_Import.htm) にアクセスすると、さらに詳しい情報を確認できます。

CSV ファイルを Alterian にインポートするには、次の手順に従ってください。

1. Alterian Campaign Manager を開き、**Documents** をクリックします。
2. ファイルのインポート先のディレクトリを選択します。
3. **Upload Files** をクリックし、CSV ファイルを選択します。
4. **Upload** をクリックします。
5. アップロードが完了したら、**Data Import** ツールを開きます。
6. **New Imports data** ウィンドウで、各フィールドを使用してインポート設定を行います。
7. 設定内容を確認したら、**Run Processes** をクリックします。

以上で完了です。Auth0 ユーザーが Alterian に正常にインポートされました。
