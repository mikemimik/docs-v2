Auth0 は、CMS のインストール環境を Auth0 アカウントと統合するための Content Management System プラグイン/Extension を提供しています。これらのプラグインにより、すべてのインスタンスとプラットフォームで、エンタープライズ、ソーシャルログイン、ユーザー名/パスワードによるログイン向けの <Tooltip tip="Single Sign-On (SSO): あるアプリケーションにユーザーがログインすると、その後ほかのアプリケーションにも自動的にログインされるサービス。" cta="用語集を見る" href="/ja/docs/glossary?term=Single+Sign-on">シングルサインオン</Tooltip> (SSO) を利用できるようになります。

ログイン機能は、新しい Login Widget (Auth0 提供) によって実装され、以下が可能になります。

* Enterprise Directory (LDAP、AD、Google Workspace、Office365、<Tooltip tip="Security Assertion Markup Language (SAML): パスワードなしで 2 者間の認証情報のやり取りを可能にする標準化プロトコル。" cta="用語集を見る" href="/ja/docs/glossary?term=SAML">SAML</Tooltip> Provider) との SSO
* 複数の WordPress アカウント間でユーザー名/パスワードを共有する SSO
* 30 を超える [Social Providers](/ja/docs/authenticate/identity-providers) との SSO
* ユーザー管理ダッシュボード
* オプションの二要素認証
* WordPress と他のアプリケーション間の SSO
* レポートと分析
