<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  電話メッセージプロバイダーは現在、早期アクセスで提供されています。Auth0 のリリースの詳細については、[Product Release Stages](/ja/docs/troubleshoot/product-lifecycle/product-release-stages)を参照してください。
</Callout>

Auth0 では、SMS と音声通信の管理、監視、トラブルシューティングを行えるように、独自の電話メッセージプロバイダーを設定できます。テナントごとに、すべての SMS と音声通信に対して設定できる電話プロバイダーは 1 つのみです。

Auth0 は現在、次の電話メッセージプロバイダーをサポートしています。

* [Twilio](/ja/docs/customize/phone-messages/configure-phone-messaging-providers/configure-twilio-as-a-phone-messaging-provider)
* [カスタム電話プロバイダー](/ja/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider)

Actions を使用してカスタム電話プロバイダーを設定することもできます。詳しくは、[カスタム電話プロバイダーを設定する](/ja/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider)および [Actions トリガー: custom-phone-provider Event Object](/ja/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider/actions-triggers-custom-phone-provider-event-object)をご覧ください。
