Auth0ワークフローの一部として送信される各メールには、それぞれ対応するメールテンプレートがあり、送信元アドレス、件名、本文などを決定します。

[外部の電話プロバイダーを設定](../phone-messages/configure-phone-messaging-providers)した後、電話テンプレートを編集して電話メッセージをカスタマイズできます。

テキストメッセージテンプレートは、HTML の機能を拡張するオープンソースのテンプレート言語である [Liquid](https://liquidjs.com/index.html) をサポートしています。Liquid を使用すると、次のことができます。

* `{{ user.name }}` をユーザー名に置き換えるなど、データを使って変数を動的に埋め込む
* `if`/`else` 文のようなロジックを実装する
* 文字列や配列の操作、数学演算子などに使うフィルターでデータを加工する

[Dashboard の電話テンプレートページ](https://manage.auth0.com/#/phone)では、電話テンプレートの有効化、無効化、カスタマイズを行えます。

<div id="available-phone-templates">
  ## 利用可能な電話テンプレート
</div>

以下は、利用可能な電話テンプレートの概要です。

| 電話テンプレート名                      | 説明                                     |
| ------------------------------ | -------------------------------------- |
| Blocked Account                | 不審なアクティビティによりアカウントがブロックされたことを顧客に通知します。 |
| Change Password                | パスワード変更用のコードを顧客に送信します。                 |
| One Time Password - Enrollment | ワンタイムパスワード (OTP) 登録用の確認コードを顧客に送信します。   |
| One Time Password - Verify     | ワンタイムパスワード (OTP) 認証用の確認コードを顧客に送信します。   |
