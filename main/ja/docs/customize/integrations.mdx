---
description: Auth0 とサードパーティアプリケーションおよびサービスとの連携について説明します。
title: Auth0 連携
---

信頼できる Auth0 の審査済み連携を活用することで、実装にかかる時間を短縮できます。Auth0 プラットフォームは高い拡張性を備えており、カスタムコードでアイデンティティフローを調整したり、サードパーティアプリケーションや各種ツールと連携したりすることで、固有のニーズに対応できます。

[Auth0 Marketplace](https://marketplace.auth0.com/) にアクセスして、当社製品と連携するサードパーティのアイデンティティソリューションを見つけて有効にしてください。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 Manage Dashboard と Management API を使用して Auth0 にサードパーティの Marketplace 連携をインストールすると、その連携の利用規約およびプライバシーポリシー、ならびに Auth0 Marketplace の[利用規約](https://auth0.com/legal)に同意したものとみなされます。
</Callout>
