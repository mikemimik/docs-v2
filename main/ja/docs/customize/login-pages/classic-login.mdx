title: Classic Login をカスタマイズ
---

<Warning>
  現在、Auth0 が注力して開発しているのは Universal Login であり、Classic Login はすでに更新対象ではありません。ログインページをカスタマイズする場合は、Auth0 は [Advanced Customizations for Universal Login](/ja/docs/customize/login-pages/advanced-customizations) を推奨しています。
</Warning>

Classic Login は、JavaScript によるカスタマイズに対応した Auth0 ホスト型のログインエクスペリエンスです。Classic Login の導入は、認証プロセスをアプリに直接埋め込むよりも複雑さが低く、クロスオリジン認証に伴うリスクの軽減にも役立ちます。

Classic Login は、[Universal Login](/ja/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) の代替手段として利用できます。2 つのログインエクスペリエンスの機能比較については、[Universal Login vs. Classic Login](/ja/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login) を参照してください。
