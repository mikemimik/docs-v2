send-phone-message Actions トリガーのAPIオブジェクトには、以下が含まれます。

<div id="apicache">
  ## `api.cache`
</div>

実行間で保持されるデータを保存・取得します。

<div id="apicachedeletekey">
  ### `api.cache.delete(key)`
</div>

指定されたキーのキャッシュ値を表すレコードが存在する場合は、それを削除します。

キャッシュから値が削除された場合は、`type: "success"` を持つ `CacheWriteResult` オブジェクトを返します。操作に失敗した場合は、`type: "error"` を返します。エラー時には、返されるオブジェクトに
失敗の内容を示す `code` プロパティが含まれます。

| パラメータ | 説明                             |
| ----- | ------------------------------ |
| `key` | *String*。キャッシュに保存されているレコードのキー。 |

<div id="apicachegetkey">
  ### `api.cache.get(key)`
</div>

指定された
`key` に対応するキャッシュ済みの値が存在する場合、その内容を表すレコードを取得します。レコードが見つかった場合、キャッシュされた値は
返されるオブジェクトの `value` プロパティで確認できます。

指定された
`key` に対応する項目がキャッシュ内で見つかった場合は、キャッシュレコードを返します。キャッシュレコードはオブジェクトで、
キャッシュされた値を保持する `value` プロパティと、
Unix epoch からの経過ミリ秒数で
レコードの最大有効期限を示す `expires_at` プロパティを持ちます。

**重要:**
このキャッシュは、短期間だけ保持される一時的なデータ向けに設計されています。項目は、指定された
有効期間内であっても、後続のトランザクションでは利用できない場合があります。

| パラメーター | 説明                             |
| ------ | ------------------------------ |
| `key`  | *String*。キャッシュに保存されているレコードのキー。 |

<div id="apicachesetkey-value-options">
  ### `api.cache.set(key, value, [options])`
</div>

指定したキーで、キャッシュに文字列値を保存または更新します。

このキャッシュに保存された値は、設定されたトリガーごとにスコープが設定されます。
また、[Actions Cache Limits](/ja/docs/customize/actions/limitations)の制限が適用されます。

この方法で保存された値の有効期間は、指定された
`ttl` または `expires_at` の値までです。有効期間が
指定されていない場合は、デフォルトで 15 分の有効期間が使用されます。有効期間は、
[Actions Cache Limits](/ja/docs/customize/actions/limitations)に記載されている最大時間を
超えることはできません。

| Parameter            | Description                                                                                                                                                                                         |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `key`                | *文字列*。キャッシュに保存されるレコードのキー。                                                                                                                                                                           |
| `value`              | *文字列*。保存するレコードの値。                                                                                                                                                                                   |
| `options`            | *省略可能なオブジェクト*。キャッシュの動作を調整するためのオプション。                                                                                                                                                                |
| `options.expires_at` | *省略可能な数値*。Unix エポックからの経過時間をミリ秒で表した絶対有効期限。キャッシュされたレコードはこれより前に削除される場合がありますが、指定された `expires_at` を超えて保持されることはありません。<br />*注:* `ttl` の値も指定する場合は、この値を指定しないでください。両方のオプションが指定された場合は、2 つのうち早いほうの有効期限が使用されます。 |
| `options.ttl`        | *省略可能な数値*。このキャッシュエントリの存続時間をミリ秒で表した値。キャッシュされた値はこれより前に削除される場合がありますが、指定された `ttl` を超えて保持されることはありません。<br />*注:* `expires_at` の値も指定する場合は、この値を指定しないでください。両方のオプションが指定された場合は、2 つのうち早いほうの有効期限が使用されます。         |
