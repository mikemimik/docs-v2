Signup トリガーと Login トリガーの詳細については、以下のセクションを参照してください。

| Trigger                                                                                                                             | Description                                                         |
| ----------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| [Login Trigger](/ja/docs/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)                                   | Login トリガーは、Auth0 テナント内のいずれかのアプリケーションにユーザーがログインしたときに実行されます。         |
| [Pre-user Registration Trigger](/ja/docs/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger)   | Pre-user Registration トリガーは、ユーザーがデータベースまたはパスワードレス接続に追加される前に実行されます。  |
| [Post-user Registration Trigger](/ja/docs/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | Post-user Registration トリガーは、ユーザーがデータベースまたはパスワードレス接続に追加された後に実行されます。 |
