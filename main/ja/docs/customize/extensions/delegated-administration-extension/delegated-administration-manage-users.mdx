Delegated Administration extension によって公開されるアプリケーションでは、**Users** と **Logs** の 2 つのビューを利用できます。**Users** ビューでは、Auth0 アカウントに関連付けられたユーザーを表示および変更できます。

デフォルトではすべてのユーザーが表示されますが、Filter Hook を設定することで表示する一覧を絞り込めます。

Filter Hook の詳細については、[Delegation Administration: Filter Hook](/ja/docs/customize/extensions/delegated-administration-extension/delegated-administration-hooks/delegated-administration-filter-hook) を参照してください。

<div id="user-actions-in-delegated-administration-dashboard">
  ## Delegated Administration ダッシュボードでのユーザー操作
</div>

Delegated Administration extension が公開する Dashboard へのアクセス権を持つユーザーに表示されるオプションを制限するには、Access Hook を設定します。

Access Hooks について詳しくは、[Delegated Administration: Access Hook](/ja/docs/customize/extensions/delegated-administration-extension/delegated-administration-hooks/delegated-administration-access-hook)を参照してください。

**プロファイルを変更** オプションは、Custom Fields を設定している場合にのみ利用できます。

extension では、新たに **パスワードをリセット** オプションも利用できます。このオプションを使うと、ユーザーにメールが送信され、新しいパスワードを選択できるようになります。これを行うには、ユーザーを選択し、**Actions** &gt; **パスワードをリセット** を選択します。

これにより、パスワードを変更するためのリンクを含むメールがユーザーに送信されます。

プロファイルに `Delegated Admin - Administrator` ロールが示されている場合、**Logs** ビューでユーザーによる認証の一覧を確認できます (このビューは `Delegated Admin - Administrator` ロールを持つユーザーにのみ表示されます) 。このビューの内容は、[Auth0 Dashboard &gt; Monitoring &gt; Logs](https://manage.auth0.com/#/logs) に表示されるデータの一部です。また、Logs ダッシュボードには、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> で実行された管理操作のデータも表示されます。

<div id="create-users">
  ## ユーザーの作成
</div>

**Users**ビューで **+ ユーザー作成** ボタンを選択すると、新しいユーザーを作成できます。メールアドレスとパスワードを指定する必要があります。ロールによっては、新しいユーザーの所属する **部署** を設定できない場合があります。

たとえば、`Delegated Admin - Administrator` ロールを持つユーザーは **部署** フィールドを表示でき、その値を自由に選択できます。

<Frame>![Extensions - Administration Dashboard - Create admin](/docs/images/cdy7uua7fh8z/4vrbU6nRuIkF30ZPM4yHnG/73fd2060fe9618c9f465025cb56deb02/create-user-admin.png)</Frame>

一方、`Delegated Admin - User` ロールを持ち、財務部門に所属する Kelly には **部署** フィールドは表示されません。彼女が作成したユーザーは、自動的に財務部門に割り当てられます。

<Frame>![Extensions - Delegated Administration Dashboard - Create user](/docs/images/cdy7uua7fh8z/2CSdiPbyiyPsURHH9X5VVZ/8416222627933bd35c044aa15c22dfae/create-user-kelly.png)</Frame>

<Card title="Delegated Administration のセッションタイムアウト">
  デフォルトでは、token の有効期限は 10 時間です。ただし、セキュリティ上の理由から、Delegated Administration の使用時に Auth0 は token を Cookie や `sessionStorage` に保存しません。ページを再読み込みするたびに、新しいセッションを開始する必要があります。
</Card>
