[Delegated Admin Extension](/docs/ja/customize/extensions/delegated-administration-extension) をインストールすると、選択したユーザーグループに <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> の **Users** セクションのみを公開し、Auth0 Dashboard の他の部分へのアクセスは許可しないようにできます。

<div id="prerequisite">
  ## 前提条件
</div>

Auth0 で [Delegated Admin Dashboard アプリケーションを作成する](/docs/ja/customize/extensions/delegated-administration-extension/create-delegated-admin-applications)。

<div id="install-extension">
  ## 拡張機能をインストール
</div>

1. [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) に移動し、**Delegated Admin** で絞り込みます。
2. 提供されている拡張機能の一覧から **Delegated Administration Dashboard** を選択します。**Install Extension** ウィンドウが開きます。

   <Frame>![Dashboard - Extensions - Delegated Admin - Install](/docs/images/cdy7uua7fh8z/51vm5Sttau4SgnuLbUrKOK/c988e2a52c4b0fc1d091287f14edccda/dashboard-extensions-install_delegated-admin.png)</Frame>
3. 次の設定変数を設定し、**Install** を選択します。

   | Variable                        | Description                                                                                                             |
   | ------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
   | **EXTENSION&#95;CLIENT&#95;ID** | この拡張機能で使用するアプリケーションの **Client ID**。                                                                                     |
   | **TITLE**                       | Delegated Administration Dashboard ページの上部に表示されるカスタムタイトル。                                                                |
   | **CUSTOM&#95;CSS**              | (*任意*) Delegated Administration Dashboard ページの見た目をカスタマイズするために使用できるカスタム CSS へのリンク。                                       |
   | **FAVICON&#95;PATH**            | (*任意*) カスタム favicon へのパス。                                                                                               |
   | **AUTH0&#95;CUSTOM&#95;DOMAIN** | (*任意*) カスタムドメインを設定している場合は、ここに入力します (例: `login.example.com`) 。これにより、認可エンドポイントは `https://login.example.com/login` に変わります。 |
   | **FEDERATED&#95;LOGOUT**        | (*任意*) ユーザーがログアウトするときに、接続からもサインアウトするかどうかを示します。                                                                          |

   `AUTH0_CUSTOM_DOMAIN` 変数を設定しても拡張機能の URL には影響せず、認可エンドポイントのみが変更されます。カスタムドメインを使用している場合、拡張機能にログインするユーザーは、デフォルトの `https://tenant-name.us.auth0.com/login` ではなく `https://AUTH0_CUSTOM_DOMAIN/login` に移動します。
   [Auth0 Dashboard &gt; アプリケーション &gt; アプリケーション](https://manage.auth0.com/#/applications) に移動すると、この拡張機能によって追加のアプリケーションが作成されていることを確認できます。このアプリケーションには Management API へのアクセス権が付与されているため、変更しないでください。

<div id="add-callback-urls-to-application">
  ## アプリケーションにコールバック URL を追加
</div>

1. [Auth0 Dashboard &gt; アプリケーション &gt; アプリケーション](https://manage.auth0.com/#/applications) に移動し、この拡張機能で使用するアプリケーションの名前を選択します。ここで選択するのは、先ほど Delegated Administration Dashboard の設定変数の構成時に入力した Client ID を持つアプリケーションです。
2. **Allowed Callback URLs** を見つけて、お使いのリージョンとテナントの拡張機能ランタイムに対応する適切なコールバック URL を追加します。

   | Location  | Name | Allowed Callback URL for Node.js 12                                |
   | --------- | ---- | ------------------------------------------------------------------ |
   | USA       | US-1 | `https://{yourTenant}.us12.webtask.io/auth0-delegated-admin/login` |
   | USA       | US-3 | `https://{yourTenant}.us.webtask.run/auth0-delegated-admin/login`  |
   | Australia | AU   | `https://{yourTenant}.au12.webtask.io/auth0-delegated-admin/login` |
   | Europe    | EU   | `https://{yourTenant}.eu12.webtask.io/auth0-delegated-admin/login` |
   | Japan     | JP-1 | `https://{yourTenant}.jp.webtask.run/auth0-delegated-admin/login`  |

   | Location  | Name | Allowed Callback URL for Node.js 8                               |
   | --------- | ---- | ---------------------------------------------------------------- |
   | USA       | US-1 | `https://YOUR_TENANT.us8.webtask.io/auth0-delegated-admin/login` |
   | Europe    | EU   | `https://YOUR_TENANT.eu8.webtask.io/auth0-delegated-admin/login` |
   | Australia | AU   | `https://YOUR_TENANT.au8.webtask.io/auth0-delegated-admin/login` |
3. **変更を保存** を選択します。

<div id="use-extension">
  ## 拡張機能を使用する
</div>

1. [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) に移動し、**Installed Extensions** ビューを選択します。
2. **Delegated Administration Dashboard** を選択します。新しいタブが開き、ログインプロンプトが表示されます。

   データベース接続の設定時に Signup を無効にしたため、ログイン画面には Sign Up オプションは表示されません。

   有効な資格情報を入力すると、カスタム **Delegated Administration Dashboard** ページに移動します。このページには、ページ上部に指定した **Title** が表示され、カスタム CSS ファイルを指定している場合は、そのスタイルが適用されます。
