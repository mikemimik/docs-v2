<div id="get-your-sendgrid-keys">
  ## SendGrid のキーを取得する
</div>

[SendGrid ダッシュボードの](https://app.sendgrid.com/settings/api_keys)設定ページにある API キーの項目で確認できます。

<div id="configure-your-sendgrid-vault-connection">
  ## SendGrid Vault 接続を設定する
</div>

SendGrid Flow Actions 用の SendGrid Vault 接続を設定するには、必要なパラメーターを設定します。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ   | 説明                             |
| ------- | ------------------------------ |
| API Key | SendGrid API の認証に使用する API Key。 |
