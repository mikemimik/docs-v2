<div id="configure-your-http-vault-connection-options">
  ## HTTP Vault 接続オプションを設定する
</div>

HTTP Vault 接続ではさまざまな認証オプションを設定でき、HTTP Request フローの Actions で利用できます。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > HTTP](/docs/images/customize/forms/vaults/http-vault-connection-options.png)</Frame>

<div id="configure-your-http-vault-connection-for-bearer-token">
  ## Bearer Token 用の HTTP Vault 接続を設定する
</div>

**Bearer Token** 認証を使用する HTTP Vault 接続を設定するには、必要なパラメーターを設定します。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > HTTP](/docs/images/customize/forms/vaults/http-vault-connection-bearer-token.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| 項目           | 説明                    |
| ------------ | --------------------- |
| Bearer Token | 外部サービスの認可に使用する固定トークン。 |

<div id="configure-your-http-vault-connection-for-oauth-client-credentials">
  ## OAuth Client Credentials 向けに HTTP Vault 接続を設定する
</div>

**Client Credentials** 認証を使用して HTTP Vault 接続を設定するには、必要なパラメーターを指定します。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > HTTP ](/docs/images/customize/forms/vaults/http-vault-connection-oauth-client-credentials.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter           | Description                                         |
| ------------------- | --------------------------------------------------- |
| Tenant Endpoint     | アクセストークンを発行する認可サーバーの Tenant Endpoint。               |
| Client ID           | カスタム API へのアクセスを認可された M2M アプリケーションの Client ID。      |
| Client Secret       | カスタム API へのアクセスを認可された M2M アプリケーションの Client Secret。  |
| Audience (Optional) | カスタム API の audience。通常は Auth0 で定義したカスタム API で使用します。 |
| Resource (Optional) | カスタム API の Resource。audience とほぼ同じ意味を持つ標準パラメーターです。  |
| Scope (Optional)    | スペース区切りの文字列形式で指定するスコープのリスト。                         |

<div id="configure-your-http-vault-connection-for-api-key">
  ## API Key 用の HTTP Vault 接続を設定する
</div>

**API Key** 認証を使用して HTTP Vault 接続を設定するには、必要なパラメーターを指定します。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > HTTP ](/docs/images/customize/forms/vaults/http-vault-connection-api-key.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter | Description                                                              |
| --------- | ------------------------------------------------------------------------ |
| Name      | Header またはクエリパラメータとして追加する API Key の名前。                                   |
| Value     | Header またはクエリパラメータとして追加する API Key の値。                                    |
| Add To    | デフォルトでは Header に追加されます。API Key を Header として追加するか、クエリパラメータとして追加するかを指定します。 |

<div id="configure-your-http-vault-connection-for-basic-authentication">
  ## Basic Authentication の HTTP Vault 接続を設定する
</div>

**Basic Auth** 認証を使用する HTTP Vault 接続を設定するには、必要なパラメーターを設定します。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > HTTP ](/docs/images/customize/forms/vaults/http-vault-connection-basic-authentication.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter | Description                                                  |
| --------- | ------------------------------------------------------------ |
| ユーザー名     | 連結してBase64エンコードし、Basic Authorization ヘッダーパラメーターとして追加するユーザー名。 |
| パスワード     | 連結してBase64エンコードし、Basic Authorization ヘッダーパラメーターとして追加するパスワード。 |

<Warning>
  Basic Auth が設定された HTTP Vault 接続は、現在 Flows HTTP Request Action に組み込まれているオプションを将来的に置き換える予定です。
</Warning>
