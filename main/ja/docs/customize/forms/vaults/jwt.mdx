<div id="configure-your-jwt-vault-connection">
  ## JWT Vault 接続を設定する
</div>

Flows の JWT Actions で JSON Web トークンに署名し、デコードして検証できるように JWT Vault 接続を設定するには、必要なパラメーターを指定する必要があります。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > JWT](/docs/images/customize/forms/vaults/jwt-vault-connection.png)</Frame>

<div id="input-settings">
  ## 入力設定
</div>

| パラメータ  | 説明                                     |
| ------ | -------------------------------------- |
| Secret | JWT のエンコードに使用する HS256 アルゴリズム用の Secret。 |
