<div id="get-your-telegram-token">
  ## Telegramのトークンを取得する
</div>

トークンの生成方法については、[Telegram の Bot API](https://core.telegram.org/#bot-api)をご覧ください。

<div id="configure-your-telegram-vault-connection">
  ## Telegram Vault 接続を設定する
</div>

Telegram のフロー Actions 用の Telegram Vault 接続を設定するには、必要なパラメーターを指定します。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > SendGrid](/docs/images/customize/forms/vaults/telegram-vault-connection.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター | 説明                           |
| ------ | ---------------------------- |
| トークン   | Telegram API の認証用に生成されたトークン。 |
