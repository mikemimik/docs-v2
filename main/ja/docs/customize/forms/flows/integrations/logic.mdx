title: ロジック

この一覧にあるロジックアクションを使用すると、Form の条件付きロジックの追加、共有変数の保存、値のマッピング、カスタムエラーメッセージの表示を行えます。

<div id="ifthen-condition">
  ## If/then 条件
</div>

入力やプロパティに応じてフローの動作をカスタマイズできる、True / False 分岐を持つ条件付き Action を追加します。

<Frame>![](/docs/images/cdy7uua7fh8z/4ehTUSu8W1LibUd1uj3x5s/00d76b15c68746ed74e44127804c9557/if-then-condition-action.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ     | 説明                                                                                           |
| --------- | -------------------------------------------------------------------------------------------- |
| Condition | 結果に応じて `True` / `False` の分岐をトリガーする、カスタマイズ可能な条件付きアクションです。`AND` / `OR` 演算子を使って、複数の条件を入れ子にできます。 |

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ    | 型    | 説明                                    |
| -------- | ---- | ------------------------------------- |
| `result` | ブール値 | 条件の判定結果に応じて、`true` または `false` を返します。 |

<div id="output-object-example">
  ### 出力オブジェクトの例
</div>

```json lines
{
  "result": true
}
```

<div id="store-shared-variable">
  ## 共有変数を保存
</div>

Form内および関連するフロー全体で利用できるプライベート state に変数を保存します。

これらの共有変数は `{{vars.*}}`**.** で参照できます。詳しくは、[変数](/ja/docs/customize/forms/variables)をご覧ください。

<Frame>![](/docs/images/cdy7uua7fh8z/4MO9wvChhnCJOg9OAdWuEf/98182a2bd8b6cf4c4baebefb96bce121/stored-shared-variable.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ    | 説明                                                                    |
| -------- | --------------------------------------------------------------------- |
| データフィールド | Form内やそのフロー全体で使用するために、プライベート state に保存する変数です。変数を追加するには、キーと値のペアを使用します。 |

<div id="show-error-message">
  ## エラーメッセージを表示
</div>

ユーザーにカスタムのエラーメッセージを表示します。

<Warning>
  このフローアクションを実行すると、フローは終了します。これ以降に挿入されたアクションは実行されません。
</Warning>

<Frame>![](/docs/images/cdy7uua7fh8z/1Kd4SyaBtAC9S8nfWFeGfh/6e42b16c0ab69dc6d026fb3601133fbe/show-error-message-flow-action.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ    | 説明                   |
| -------- | -------------------- |
| エラーメッセージ | ユーザーに表示するエラーメッセージです。 |

<div id="input-value-mapping">
  ## 入力値のマッピング
</div>

入力変数の値に基づいて、出力先をマッピングします。

<Frame>![](/docs/images/cdy7uua7fh8z/4VtAr4vCSXtKlSaxtiXQhV/bcb07221f688483aa2010d80394ae630/input-value-mapping.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ     | 説明                                            |
| --------- | --------------------------------------------- |
| 入力値 (必須)  | 入力変数です。                                       |
| マッピングテーブル | 入力変数の値を出力値にマッピングします。                          |
| デフォルトの出力値 | マッピングテーブルに出力値が設定されていない場合に使用するデフォルトの出力値を設定します。 |

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ    | 型      | 説明                 |
| -------- | ------ | ------------------ |
| `result` | String | 入力にマッピングされた値を返します。 |

<div id="output-object-example">
  ### 出力オブジェクトの例
</div>

```json lines
{
	"result": "Please enter a valid email address."
}
```
