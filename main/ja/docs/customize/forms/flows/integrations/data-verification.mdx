title: データ検証

このデータ検証用Actionsの一覧では、メールアドレスの検証とワンタイムパスワードの生成・検証を行えます。

<div id="generate-one-time-password">
  ## ワンタイムパスワードを生成
</div>

メールやSMS、その他のメッセージアプリで送信できる、有効期限5分の数字コードを生成します。

<Frame>![](/docs/images/cdy7uua7fh8z/1h69FWzZwk9beKKjB4Cq38/16aa515e61cc6e9ebdfd0849201e3141/generate-one-time-password.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ        | 説明                                        |
| ------------ | ----------------------------------------- |
| 参照値 (必須)     | 生成したコードを電話番号、メールアドレス、または任意の値に紐付けることができます。 |
| コードの長さ (必須)  | 生成されるコードの桁数です。                            |

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ  | 型      | 説明       |
| ------ | ------ | -------- |
| `code` | String | 生成されたコード |

<div id="output-object-example">
  ### 出力オブジェクトの例
</div>

```json lines
{
  "code": "███",
}
```

<div id="verify-one-time-password">
  ## ワンタイムパスワードを検証する
</div>

指定されたワンタイムパスワードが有効かどうかを検証します。

<Frame>![](/docs/images/cdy7uua7fh8z/3bkT8QDN4dcl0YHTrSOIGj/9698a07f55353a11ca98164377cd4217/verify-otp.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ       | 説明                                                     |
| ------------ | ------------------------------------------------------ |
| 参照値 (必須)     | Generate one-time password action で設定したものと同じ参照値を使用します。 |
| コードの長さ (必須)  | 生成されるコードの桁数です。                                         |

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ    | 型       | 説明                                                                                                                                                                                               |
| -------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `valid`  | Boolean | 指定されたコードが有効かどうかを示します。値は `true` または `false` です。                                                                                                                                                   |
| `result` | String  | 指定されたコードに応じて、異なる結果コードを返します：<ul> <li> `MISSING:` 指定されたコードは存在しません。 </li> <li> `EXPIRED:` 指定されたコードの有効期限が切れています。 </li> <li> `VALID:` 指定されたコードは有効です。 </li> <li> `UNEQUAL:` 指定されたコードは無効です。 </li> </ul> |

<div id="output-object-example">
  ### 出力オブジェクトの例
</div>

```json lines
{
  "valid": false,
  "result": "UNEQUAL"
}
```

```json lines
{
  "valid": true
}
```

<div id="verify-email-address">
  ## メールアドレスを検証
</div>

指定したメールアドレスに対して、選択した複数の検証ルールを実行します。

<Frame>![](/docs/images/cdy7uua7fh8z/6qkqny1hhObFalHwdEG342/1a2e18195b29102c3f68744f42277917/verify-email.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter                        | Description                                                           |
| -------------------------------- | --------------------------------------------------------------------- |
| Email (required)                 | 検証するメールアドレス。                                                          |
| Require MX record                | MXレコードを必須にします。ドメインにMXレコードがない場合、そのドメイン宛てのメールをどこに送ればよいかメールサーバーが判断できません。 |
| Block free email providers       | @gmail.com、@hotmail.com などの無料メールプロバイダーをブロックします。                       |
| Block disposable email providers | @mailnator.com、@maildrop.cc などの使い捨てメールプロバイダーをブロックします。                 |
| Block email account aliases      | `jane.doe+alias@gmail.com` のようなエイリアス付きメールアドレスをブロックします。                |
| Allowlist domains                | リストで承認されたドメインからのメールのみを許可します。                                          |
| Blocklist domains                | 特定のメールドメインをブロックします。                                                   |

<div id="output-object">
  ### 出力オブジェクト
</div>

| Property | Type    | Description                                                                                                                                                                                                                                                                                                                                                               |
| -------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `valid`  | Boolean | メールが検証ルールを満たしているかどうかに応じて、`true` または `false` を返します。                                                                                                                                                                                                                                                                                                                        |
| `cause`  | String  | `valid` プロパティが `<br/>false<br/>` の場合、メールが最初に満たさなかったルールを示す値を返します: <ul> <li> `MISSING_MX_RECORD:` メールのドメインに MX レコードがありません。 </li> <li> `FREE_EMAIL:` メールのドメインは無料のメールプロバイダーです。</li> <li> `DISPOSABLE_EMAIL:` メールが使い捨てメールアドレスです。</li> <li> `BLOCKLISTED:` メールのドメインが Blocklist domains 設定に含まれています。</li> <li> `NOT_ALLOWED:` メールのドメインが Allowlist domains 設定に含まれていません。</li> </ul> |

<div id="output-object-example">
  ### 出力オブジェクトの例
</div>

```json lines
{
  "valid": false,
  "cause": "FREE_EMAIL"
}
```

```json lines
{
  "valid": true,
  "cause": null
}
```
