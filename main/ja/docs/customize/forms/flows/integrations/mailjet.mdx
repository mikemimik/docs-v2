<div id="send-email">
  ## メールを送信
</div>

Mailjetをメールプロバイダーとして使用して、メールを送信します。

<Frame>![](/docs/images/cdy7uua7fh8z/5z1CqSZTrLs1oMdHKvHOiM/7e1fa3b5d7ea5a5b2ec51dd58fa1d2ed/mailjet.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター           | 説明                                                                                     |
| ---------------- | -------------------------------------------------------------------------------------- |
| 送信者名             | 送信者の名前                                                                                 |
| 送信者メールアドレス (必須)  | 送信者のメールアドレス                                                                            |
| 受信者名             | 受信者の名前                                                                                 |
| 受信者メールアドレス (必須)  | 受信者のメールアドレス                                                                            |
| CC名              | CCの名前                                                                                  |
| CCメールアドレス        | CCのメールアドレス                                                                             |
| 件名               | メールの件名                                                                                 |
| メールタイプ           | HTML コンテンツのメールを送信するか、保存済みテンプレートを使用します (任意)                                             |
| メッセージ (必須)       | HTML 形式のメールメッセージ、またはテンプレート (任意) 。                                                      |
| テンプレート ID (任意)   | テンプレートを使用する場合は、その ID を [ Mailjet のダッシュボード ](https://app.mailjet.com/template) で確認できます。 |
| カスタム変数 (任意)      | テンプレートに挿入するカスタム変数を追加します。                                                               |
| メールアクティビティを追跡    | メールアクティビティを追跡するには、チェックボックスをオンにします。                                                     |

<div id="output-object">
  ### 出力オブジェクト
</div>

想定されるレスポンスとステータスコードについては、[Mailjetのドキュメント](https://dev.mailjet.com/email/reference/send-emails/)を参照してください。
