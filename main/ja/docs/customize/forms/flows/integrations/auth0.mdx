<div id="custom-api-request">
  ## カスタム API リクエスト
</div>

Auth0 API のエンドポイントにカスタム API リクエストを送信します。

<Frame>![Auth0 Dashboard > カスタマイズ > Flows > 連携 > Auth0](/docs/images/cdy7uua7fh8z/66uV9iuhwfnXUdE1quLOyx/2ae3ff26335acbcd172cbb40618a5a90/Screenshot_2025-01-24_at_18.50.26.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター     | 説明                                                                                                                                                                                                                                                                                                                                                                                                           |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| パス         | APIエンドポイントの絶対パス名。<br />( 例: `/api/v2/organizations` )<br />**注:** 一部のエンドポイントでは、たとえば [ Get a user ](https://auth0.com/docs/api/management/v2/users/get-users-by-id) のように、`-`、`.`、`_`、`~` などの英数字以外の [ 文字 ](https://datatracker.ietf.org/doc/html/rfc3986#section-2.3) を使用する場合、URIエンコードされたパラメーターが必要です。<br />その場合は、encodeURIComponent 関数を使用できます。<br />`{{ functions.encodeURIComponent(context.user.user_id) }}` |
| メソッド       | 使用可能なHTTPメソッドは次のとおりです。 <ul> <li>`GET`</li> <li>`POST`</li> <li>`PUT`</li> <li>`PATCH`</li> <li>`DELETE`</li> </ul>                                                                                                                                                                                                                                                                                           |
| ボディ        | リクエストのボディ。                                                                                                                                                                                                                                                                                                                                                                                                   |
| ヘッダー       | リクエストのヘッダー。                                                                                                                                                                                                                                                                                                                                                                                                  |
| URL Params | URLパラメーター。これらのパラメーターは自動的にURLエンコードされます。                                                                                                                                                                                                                                                                                                                                                                       |

<div id="output-object">
  ### 出力オブジェクト
</div>

想定されるレスポンスとステータスコードについては、[Auth0 Management API](https://auth0.com/docs/api/management/v2/introduction) を参照してください。

<div id="send-email">
  ## メールを送信
</div>

[設定済みのメールプロバイダー](/ja/docs/customize/email)を使用して、カスタムメールを送信します。

<Warning>
  このActionを使用する前に、[メールプロバイダー](/ja/docs/customize/email)を有効にしておく必要があります。
</Warning>

<Frame>![](/docs/images/forms/flows/integrations/auth0-notifications-send-email.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター        | 説明                                                                                           |
| ------------- | -------------------------------------------------------------------------------------------- |
| カスタム送信者メールを使用 | デフォルトでは、カスタマイズしたメールプロバイダーの設定で構成した送信者メールを使用します。別の送信者メールアドレスまたは送信者名を使用する場合は、このオプションを有効にしてください。 |
| 受信者メール        | 受信者のメールアドレスです。                                                                               |
| カスタム変数        | メールテンプレートに挿入するカスタム変数を追加します。設定後は、`{{custom_vars.your_variable_key}}` を使用して参照できます。             |
| 件名            | メールの件名です。カスタム変数と Liquid 構文に対応しています。                                                          |
| メッセージ         | メールの本文は HTML 形式で、Liquid 構文を使用でき、カスタム変数にも対応しています。                                             |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  件名とメッセージでは、Liquid 構文を[共通変数](https://auth0.com/docs/customize/email/email-templates/supported-liquid-syntax)や `{{custom_vars.your_variable_key}}` 構文のカスタム変数と組み合わせて使用できます。

  カスタム変数を使用すると、フォーム / Flows のコンテキストと件名およびメッセージの内容の間で情報を受け渡せます。
</Callout>

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ   | 型       | 説明                                             |
| ------- | ------- | ---------------------------------------------- |
| success | Boolean | 配信に成功したかどうかにかかわらず、メール送信が正常にキューに追加されたかどうかを示します。 |

### 出力オブジェクトの例

```json
{
  "success": true
}
```

<div id="send-sms">
  ## SMSを送信
</div>

[設定済みの電話プロバイダー](https://auth0.com/docs/customize/phone-messages/configure-phone-messaging-providers)を使用して、カスタムSMSを送信します。

<Warning>
  このActionを使用する前に、テキストメッセージを配信できるように [Phone Provider](https://auth0.com/docs/customize/phone-messages/configure-phone-messaging-providers) を有効にしておく必要があります。
</Warning>

<Frame>![](/docs/images/forms/flows/integrations/auth0-notifications-send-sms.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter        | Description                                                                     |
| ---------------- | ------------------------------------------------------------------------------- |
| From             | デフォルトでは、カスタマイズした電話プロバイダーの設定で構成した送信元の電話番号を使用します。別の送信元電話番号を使う場合は、カスタム値を入力してください。  |
| To               | 受信者の電話番号です。                                                                     |
| カスタム変数 | 電話テンプレートに挿入するカスタム変数を追加します。設定後は、`{{custom_vars.your_variable_key}}` を使用して参照できます。 |
| Message          | 電話メッセージでは Liquid 構文を使用でき、カスタム変数もサポートされています。                                     |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Message では、[共通変数](https://auth0.com/docs/customize/email/email-templates/supported-liquid-syntax)と Liquid 構文を組み合わせて使用でき、`{{custom_vars.your_variable_key}}` 構文を使ったカスタム変数にも対応しています。

  カスタム変数を使うと、フォーム / Flows のコンテキストと Message の内容の間で情報を受け渡せます。
</Callout>

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ   | 型       | 説明                                             |
| ------- | ------- | ---------------------------------------------- |
| success | Boolean | 配信の成否にかかわらず、電話メッセージが送信待ちキューに正常に追加されたかどうかを示します。 |

<div id="output-object-example">
  ### Output objectの例
</div>

```json
{
  "success": true
}
```

<div id="make-call">
  ## 通話をかける
</div>

[設定済みの電話プロバイダー](https://auth0.com/docs/customize/phone-messages/configure-phone-messaging-providers)を使用して、カスタム音声メッセージを送信します。

<Warning>
  この Action を使用する前に、音声メッセージ配信用の [Phone Provider](https://auth0.com/docs/customize/phone-messages/configure-phone-messaging-providers) を有効にしておく必要があります。
</Warning>

<Frame>![](/docs/images/forms/flows/integrations/auth0-notifications-make-call.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter | Description                                                                      |
| --------- | -------------------------------------------------------------------------------- |
| From      | デフォルトでは、カスタマイズした電話プロバイダーの設定で構成した送信元の電話番号を使用します。別の送信元電話番号を使用する場合は、カスタム値を入力してください。 |
| To        | 受信者の電話番号です。                                                                      |
| カスタム変数    | 電話テンプレートに挿入するカスタム変数を追加します。設定後は、`{{custom_vars.your_variable_key}}` を使って参照できます。   |
| Message   | 電話メッセージでは Liquid 構文を使用でき、カスタム変数もサポートされています。                                      |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Message では、Liquid 構文を[共通変数](https://auth0.com/docs/customize/email/email-templates/supported-liquid-syntax)と組み合わせて使用できるほか、`{{custom_vars.your_variable_key}}` 構文を使ってカスタム変数を利用することもできます。

  カスタム変数を使用すると、フォーム / Flows のコンテキストと Message の内容の間で情報を受け渡せます。
</Callout>

<div id="output-object">
  ### 出力オブジェクト
</div>

| Property | Type    | Description                                             |
| -------- | ------- | ------------------------------------------------------- |
| success  | Boolean | 電話メッセージが送信待ちキューに正常に追加されたかどうかを示します。実際に配信に失敗したかどうかは問いません。 |

### 出力オブジェクトの例

```json
{
  "success": true
}
```
