[Auth0 Universal Login](/ja/docs/authenticate/login/auth0-universal-login) は、<Tooltip tip="Authorization Server: ユーザーのアクセス範囲の定義に寄与する集中管理型サーバー。たとえば、認可サーバーはユーザーが利用できるデータ、タスク、機能を制御できます。" cta="用語集を見る" href="/ja/docs/glossary?term=authorization+server">認可サーバー</Tooltip>の中核機能であるログインフローを提供します。ユーザーがアプリケーションにアクセスするために本人確認を行う必要がある場合は、<Tooltip tip="Authorization Server: ユーザーのアクセス範囲の定義に寄与する集中管理型サーバー。たとえば、認可サーバーはユーザーが利用できるデータ、タスク、機能を制御できます。" cta="用語集を見る" href="/ja/docs/glossary?term=Universal+Login">Universal Login</Tooltip> にリダイレクトして、認証プロセスを Auth0 に任せることができます。

Universal Login は、Auth0 の主要なホスト型ログインソリューションです。[Universal Login](/ja/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) には使いやすいカスタマイズツールが用意されており、エンドユーザーによりシンプルで迅速な体験を提供します。一方 Auth0 は、カスタマイズに JavaScript コントロールを使用するホスト型ログインエクスペリエンスである [Classic Login](/ja/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/classic-experience) もサポートしています。

ログインエクスペリエンスの比較については、[Universal Login vs. Classic Login](/ja/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login) を参照してください。
