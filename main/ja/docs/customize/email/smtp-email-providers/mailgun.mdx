<div id="prerequisites">
  ## 前提条件
</div>

* [Mailgun](https://mailgun.com) のアカウントが必要です。

* 特定の [Auth0 IP アドレス](/ja/docs/secure/security-guidance/data-security/allowlist) からの接続を許可するよう、プロバイダーを設定する必要があります。

<div id="configure-mailgun">
  ## Mailgun を設定する
</div>

1. [Mailgun](https://mailgun.com) にログインします。
2. Mailgun の **Sending &gt; Domains** に移動し、カスタムドメインを追加します。Mailgun では、検証済みドメインからのみメールを送信できます。
3. Mailgun の **Settings &gt; API Keys** に移動します。**API Key** はアカウント登録時に作成されているため、**Private API Key** からコピーしてください。
4. Auth0 の [Auth0 Dashboard &gt; ブランディング &gt; メールプロバイダー](https://manage.auth0.com/#/branding/email_provider) に移動します。
5. **Use my own email provider** トグルを有効化します。
6. **Mailgun** のロゴをクリックします。
7. **From** のメールアドレスを指定し、先ほど Mailgun に追加した **Domain** と、先ほどコピーした Mailgun の **API Key** を入力して、**Region** を選択します。
8. **Save** をクリックします。

<div id="send-a-test-email">
  ## テストメールを送信する
</div>

**Send Test Email** ボタンを使用して、テストメールを送信します。設定が正しければ、確認メールが届きます。数分経ってもメールが届かない場合は、エラーコードを [Auth0のログ](https://manage.auth0.com/#/logs) で確認してください。

設定後は、[Mailgun Dashboard](https://app.mailgun.com/app/dashboard) で、ユーザーに送信されたすべてのメールの配信状況を確認できます。
