---
title: "Auth0 向け Agent Skills"
description: "Claude Code、Cursor、GitHub Copilot などの AI コーディングアシスタントを使って、Auth0認証をアプリケーションに統合できます"
---

Agent Skills は、AI コーディングアシスタントが Auth0認証を正しく実装できるようにするための構造化された指示です。一度インストールすれば、あとはアシスタントにログインの追加、ルートの保護、API の保護、MFA の設定、別のプロバイダーからの移行、ログインページのカスタマイズを依頼するだけで、残りはすべて任せられます。

Claude Code、Cursor、GitHub Copilot のほか、[Agent Skills](https://agentskills.io) 形式に対応した [40 以上のエージェント](https://agentskills.io/clients) で利用できます。

<div id="prerequisites">
  ## 前提条件
</div>

* [Auth0アカウント](https://auth0.com/signup) (無料プランあり)
* [Auth0 Dashboard](https://manage.auth0.com) にある Auth0 の **ドメイン** と **client ID**
* AIコーディングアシスタント (Claude Code、Cursor、または GitHub Copilot)

<div id="install">
  ## インストール
</div>

<Tabs>
  <Tab title="Claude Code">
    Auth0 は公式の Claude Code プラグインのマーケットプレイス (`claude-plugins-official`) で提供されています。まずマーケットプレイスを最新の状態にしてから、インストールします。

    ```bash
    /plugin marketplace update claude-plugins-official
    /plugin install auth0@claude-plugins-official
    ```

    または、セッションで `/plugin` と入力し、**Discover** タブを開いて「Auth0」を検索します。

    セッションを開始せずに、ターミナルからインストールすることもできます。

    ```bash
    claude plugin marketplace update claude-plugins-official
    claude plugin install auth0@claude-plugins-official
    ```

    <Note>
      「not found」エラーが表示される場合は、まずマーケットプレイスを追加してから、再度インストールしてください。

      ```bash
      /plugin marketplace add anthropics/claude-plugins-official
      /plugin install auth0@claude-plugins-official
      ```
    </Note>
  </Tab>

  <Tab title="Cursor">
    Auth0 は [Cursor marketplace](https://cursor.com/marketplace/auth0) で提供されています。掲載ページを開き、**Add** をクリックしてインストールします。

    **Cursor 設定 &gt; ルール &gt; Add Rule &gt; Remote Rule (GitHub)** からインストールし、[Auth0 Agent Skills リポジトリの URL](https://github.com/auth0/agent-skills) を入力することもできます。
  </Tab>

  <Tab title="GitHub Copilot">
    GitHub Copilot を対象に、[Skills CLI](https://github.com/vercel-labs/skills) を使ってインストールします。

    ```bash
    npx skills add auth0/agent-skills --agent github-copilot
    ```
  </Tab>

  <Tab title="Other Agents">
    [Skills CLI](https://github.com/vercel-labs/skills) は、Claude Code、Cursor、Copilot、Codex、および [40 種類以上の他のエージェント](https://agentskills.io/clients) で利用できます。

    ```bash
    npx skills add auth0/agent-skills --all
    ```

    `--agent` で特定のエージェントを指定します。

    ```bash
    npx skills add auth0/agent-skills --agent claude-code cursor
    ```
  </Tab>
</Tabs>

<div id="try-it-now">
  ## 今すぐ試す
</div>

インストール後、次のいずれかを AI コーディングアシスタントに貼り付けてください:

```
Add Auth0 login to my app
```

```
Secure my Express API with Auth0 JWT validation
```

```
Migrate from Firebase Auth to Auth0
```

<Tip>
  プロンプトを入力する際は、コンテキストも含めてください。使用しているフレームワーク、ルーターの種類、ログイン、API 保護、またはその両方が必要かどうかを明記します。プロンプトが具体的であるほど、出力の精度も高くなります。
</Tip>

<div id="how-it-works">
  ## 仕組み
</div>

<Steps>
  <Step title="必要なことを説明する">
    React アプリにログインを追加する、Spring Boot API を保護する、Cognito から移行する、MFA を追加する、ログインページをカスタマイズするなど、やりたいことをアシスタントに伝えます。どの スキル や SDK を使えばよいかを知っている必要はありません。
  </Step>

  <Step title="適切な スキル が有効化される">
    アシスタントがプロジェクトファイル (`package.json`、`requirements.txt`、`build.gradle` など) を読み取り、使用中のフレームワークを特定し、25 種類の中から対応する スキル を有効化します。
  </Step>

  <Step title="動作するコードが手に入る">
    その スキル は、適切な SDK のインストール、認証ルートの作成、環境変数の設定、ルート保護の追加、Auth0 のベストプラクティスに沿った実装まで、実装全体を通してアシスタントを導きます。
  </Step>
</Steps>

<Info>
  スキル を手動で選ぶ必要はありません。フレームワーク検出によって SDK スキル は自動的に処理され、MFA や Migration のような機能 スキル は、必要なことを説明するだけで直接呼び出せます。
</Info>

<div id="get-started">
  ### はじめに
</div>

| Skill                 | できること                                                                                      | 試してみる                                         |
| --------------------- | ------------------------------------------------------------------------------------------ | --------------------------------------------- |
| **Quickstart Router** | フレームワークを検出し、適切なSDKを使ってAuth0をセットアップします                                                      | `Add Auth0 authentication to my app`          |
| **Migration**         | Firebase、Cognito、Supabase、Clerk、または独自の認証から、ユーザーとコードを移行します                                  | `Help me migrate from Firebase Auth to Auth0` |
| **CLI**               | Auth0 CLIのコマンドリファレンス — アプリケーション、API、ユーザー、ロール、組織、Actions、ログ、カスタムドメイン、Universal Loginを管理できます | `Create an Auth0 application using the CLI`   |
