---
title: "認証要素 を削除"
description: "関連付けられた MFA 認証要素を ID を指定して削除します。"
---

<div id="endpoint">
  ## エンドポイント
</div>

`DELETE /mfa/authenticators/{AUTHENTICATOR_ID}`

IDを指定して、関連付けられている認証要素を削除します。認証要素のIDは、認証要素の一覧を取得することで確認できます。

このエンドポイントにアクセスするには、次のクレームを含むアクセストークンを Authorization ヘッダーに設定する必要があります。

* `scope`: `remove:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="path-parameters">
  ## パスパラメータ
</div>

<ParamField path="AUTHENTICATOR_ID" type="string" required>
  削除対象の認証要素のID。
</ParamField>

<div id="headers">
  ## ヘッダー
</div>

<ParamField header="Authorization" type="string" required>
  ログイン時に取得したアクセストークンです。
</ParamField>

<div id="response-messages">
  ## レスポンスメッセージ
</div>

| Status | Description                |
| ------ | -------------------------- |
| 204    | 認証要素 は正常に削除されました。 |
| 401    | 未認証 - 無効な アクセストークン です。 |
| 404    | 認証要素 が見つかりません。    |
