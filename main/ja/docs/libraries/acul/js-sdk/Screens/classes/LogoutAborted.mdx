LogoutAborted クラスは、`logout-aborted` 画面の機能を提供します。この画面は、ユーザーがログアウト処理を中止した場合に表示されます。

<div id="constructors">
  ## コンストラクター
</div>

LogoutAborted画面マネージャーのインスタンスを作成します:

```typescript Example
import LogoutAborted from '@auth0/auth0-acul-js/logout-aborted';
const logoutAbortedManager = new LogoutAborted();
logoutAbortedManager.changeLanguage({
  language: 'fr',
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import LogoutAborted from '@auth0/auth0-acul-js/logout-aborted';
  const logoutAbortedManager = new LogoutAborted();
  logoutAbortedManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      変更先の言語の locale コード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語がセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションのエラーの array を取得します。エラーが存在しない場合は、空の array を返します。
</ParamField>
