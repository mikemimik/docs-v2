ResetPasswordEmail クラスは、`reset-password-email` 画面の機能を実装します。この画面では、パスワードリセットメールが送信されたことをユーザーに通知し、必要に応じて再送できるようにします。

<div id="constructors">
  ## コンストラクター
</div>

ResetPasswordEmail画面マネージャーのインスタンスを作成します:

```typescript Example
import ResetPasswordEmail from '@auth0/auth0-acul-js/reset-password-email';
const resetPasswordEmailManager = new ResetPasswordEmail();
resetPasswordEmailManager.resendEmail();
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import ResetPasswordEmail from '@auth0/auth0-acul-js/reset-password-email';
  const resetPasswordEmailManager = new ResetPasswordEmail();
  resetPasswordEmailManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションのエラー配列を取得します。エラーが存在しない場合は、空の配列を返します。
</ParamField>

<ParamField body="resendEmail" type="Promise<void>">
  このメソッドは、ユーザーの登録済みメールアドレスにパスワードリセットメールを再送します。

  ```typescript Example
  import ResetPasswordEmail from '@auth0/auth0-acul-js/reset-password-email';
  const resetPasswordEmailManager = new ResetPasswordEmail();
  resetPasswordEmailManager.resendEmail();
  ```
</ParamField>
