MfaPhoneEnrollment クラスは、`mfa-phone-enrollment` 画面の機能を提供します。この画面では、ユーザーは電話番号を使って MFA に登録できます。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

MFA電話登録画面マネージャーのインスタンスを作成します。

```typescript Example
import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

const mfaPhoneEnrollment = new MfaPhoneEnrollment();
await mfaPhoneEnrollment.continueEnrollment({ phone: '+1234567890', type: 'sms' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';
  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  mfaPhoneEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語がセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueEnrollment" type="Promise<void>">
  このメソッドは、指定した電話番号と種類 (SMS または音声) を使用して登録手順を続行します。

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.continueEnrollment({ phone: '+1234567890', type: 'sms' });
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [MfaPhoneEnrollmentContinueOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/MfaPhoneEnrollmentContinueOptions).
    </ParamField>

    <ParamField body="phone" type="string" required>
      ユーザーの電話番号。
    </ParamField>

    <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;" required>
      確認コードの配信方法。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストから transaction のエラー配列を取得します。エラーが存在しない場合は空の配列を返します。
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  このメソッドは、国番号を選択する画面に移動します。

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.pickCountryCode();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドを使うと、ユーザーは別のMFA方式を試すことができます。

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.tryAnotherMethod();
  ```
</ParamField>
