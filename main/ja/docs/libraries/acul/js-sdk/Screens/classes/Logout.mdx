Logout クラスは、`logout` 画面の機能を実装します。この画面では、ユーザーはログアウトアクションを承認するか拒否するかを選択できます。

<div id="constructors">
  ## コンストラクター
</div>

Logout画面マネージャーのインスタンスを作成します。

```typescript Example
import Logout from '@auth0/auth0-acul-js/logout';
const logoutManager = new Logout();
logoutManager.confirmLogout({
  action: 'accept',
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import Logout from '@auth0/auth0-acul-js/logout';
  const logoutManager = new Logout();
  logoutManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="Parameters">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コードです (たとえば、`'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語は session の間保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="confirmLogout" type="Promise<void>">
  このメソッドは、ログアウトアクションを確認します (承認または拒否) 。

  ```typescript Example
  import Logout from '@auth0/auth0-acul-js/logout';
  const logoutManager = new Logout();
  logoutManager.confirmLogout({
    action: 'accept',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="Parameters">
    <ParamField body="options">
      [ConfirmLogoutOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/ConfirmLogoutOptions)。
    </ParamField>

    <ParamField body="action" type="&#x22;accept&#x22; | &#x22;deny&#x22;" required>
      認証トランザクションの状態です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。存在しない場合は空の配列を返します。トランザクションコンテキスト内のエラーオブジェクトの配列です。
</ParamField>
