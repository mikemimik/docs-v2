SignupId クラスは、`signup-id` 画面の機能を提供します。この画面では、ユーザーの識別子を収集します。

  <img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/5lzGYj0k4VcxvPCBYH37SO/ebd291cc0dbd673a70f93530f724bf65/Signup_ID.png" alt="Signup ID" />
</Frame>

<div id="constructors">
  ## コンストラクター
</div>

SignupId画面マネージャーのインスタンスを作成します:

```typescript Example
import SignupId from '@auth0/auth0-acul-js/signup-id';
const signupIdManager = new SignupId();
signupIdManager.signup({
  email: 'test@example.com',
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import SignupId from '@auth0/auth0-acul-js/signup-id';
  const signupIdManager = new SignupId();
  signupIdManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedSignup" type="Promise<void>">
  このメソッドは、Google や Facebook など、さまざまなソーシャル識別子を使ったサインアップを処理します。

  ```typescript Example
  import SignupId from '@auth0/auth0-acul-js/signup-id';
  const signupIdManager = new SignupId();
  signupIdManager.federatedSignup({
    connection: 'google-oauth2',
  });
  ```

  **メソッドパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [FederatedSignupOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/FederatedSignupOptions)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      使用するソーシャル接続名です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。存在しない場合は空の配列を返します。返されるのは、トランザクションコンテキスト内のエラーオブジェクトの配列です。
</ParamField>

<ParamField body="getSignupIdentifiers" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Identifier">Identifier</a>[] | null</span>}>
  このメソッドは、`signup-id` form で有効になっている 識別子 の一覧を返し、トランザクション設定に基づいてそれぞれが必須か任意かを示します。返されるのは、識別子 オブジェクトの配列です (例: email、phone、username) 。

  ```typescript Example
  import SignupId from '@auth0/auth0-acul-js/signup-id';
  const signupIdManager = new SignupId();
  const identifiers = signupIdManager.getSignupIdentifiers();
  // [{ type: 'email', required: true }, { type: 'username', required: false }]
  ```
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  このメソッドは、ユーザーを国番号の選択リストにリダイレクトし、電話番号の国番号プレフィックスを変更できるようにします。

  ```typescript Example
  import SignupId from '@auth0/auth0-acul-js/signup-id';
  const signupIdManager = new SignupId();
  signupIdManager.pickCountryCode();
  ```
</ParamField>

<ParamField body="signup" type="Promise<void>">
  このメソッドは、`signup-id` に関連する設定を処理します。さまざまな 識別子 を使って新規ユーザーをサインアップできます。

  ```typescript Example
  import SignupId from '@auth0/auth0-acul-js/signup-id';
  const signupIdManager = new SignupId();
  signupIdManager.signup({
    email: 'test@example.com',
  });
  ```

  **メソッドパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [SignupOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/SignupOptions)。
    </ParamField>

    <ParamField body="email?" type="string">
      ユーザーのメールアドレスです。
    </ParamField>

    <ParamField body="username?" type="string">
      ユーザーの 識別子 です。
    </ParamField>

    <ParamField body="phone?" type="string">
      ユーザーの電話番号です。
    </ParamField>

    <ParamField body="captcha?" type="string">
      captcha コード、または CAPTCHA provider からのレスポンスです。Auth0 テナントで [ボット検出](/ja/docs/secure/attack-protection/bot-detection) が有効になっている場合、このプロパティは必須です。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="validateUsername" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/UsernameValidationResult">UsernameValidationResult</a></span>}>
  このメソッドは、指定されたユーザー名を、トランザクション コンテキストで定義されている現在のユーザー名ポリシーに対して検証します。ユーザー名が有効かどうかと、その理由を示すオブジェクトを返します。

  ```typescript Example
  import SignupId from '@auth0/auth0-acul-js/signup-id';
  const signupIdManager = new SignupId();
  const result = signupIdManager.validateUsername('myusername');
  // result => { valid: true, errors: [] }
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="username" type="string" required>
      検証するユーザー名の文字列です。
    </ParamField>
  </Expandable>
</ParamField>
