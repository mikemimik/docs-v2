PasskeyEnrollment クラスは、`passkey-enrollment` 画面の機能を提供します。この画面では、ユーザーは認証方法としてパスキーを登録できます。

  ## コンストラクター

PasskeyEnrollment画面マネージャーのインスタンスを作成します。

<div id="methods">
  ## メソッド
</div>

<ParamField body="abortPasskeyEnrollment" type="Promise<void>">
  このメソッドは、パスキー登録プロセスを中止します。

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  await passkeyEnrollmentManager.abortPasskeyEnrollment();
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [AbortEnrollmentOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/AbortEnrollmentOptions).
    </ParamField>

    <ParamField body="doNotShowAgain?" type="boolean">
      `true` に設定すると、以降のセッションでパスキー登録のプロンプトを表示しません。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  passkeyEnrollmentManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      希望する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中維持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continuePasskeyEnrollment" type="Promise<void>">
  このメソッドは、パスキー登録プロセスを続行します。

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  await passkeyEnrollmentManager.continuePasskeyEnrollment();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。存在しない場合は空の配列を返します。
</ParamField>
