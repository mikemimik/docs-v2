EmailVerificationResult クラスは、`email-verification-result` 画面の機能を実装します。この画面には、メールアドレスの確認を試行した結果が表示されます。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

EmailVerificationResult画面マネージャーのインスタンスを作成します:

```typescript Example
import EmailVerificationResult from '@auth0/auth0-acul-js/email-verification-result';
const emailVerificationResultManager = new EmailVerificationResult();
emailVerificationResultManager.changeLanguage({ language: 'fr' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import EmailVerificationResult from '@auth0/auth0-acul-js/email-verification-result';
  const emailVerificationResultManager = new EmailVerificationResult();
  emailVerificationResultManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      目的の言語の locale コードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッションの間保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクション エラーの配列を取得します。エラーがない場合は、空の配列を返します。
</ParamField>
