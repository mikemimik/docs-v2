ResetPasswordMfaRecoveryCodeChallenge クラスは、`reset-password-mfa-recovery-code-challenge` 画面の機能を実装します。この画面では、パスワードリセットフローにおける MFA ステップの一部として、ユーザーにリカバリーコードの入力を求めます。

<div id="constructors">
  ## コンストラクター
</div>

ResetPasswordMfaRecoveryCodeChallenge 画面マネージャーのインスタンスを作成します:

```typescript Example
import ResetPasswordMfaRecoveryCodeChallenge from '@auth0/auth0-acul-js/reset-password-mfa-recovery-code-challenge';
const resetPasswordMfaRecoveryCodeChallengeManager = new ResetPasswordMfaRecoveryCodeChallenge();
await resetPasswordMfaRecoveryCodeChallengeManager.continue('RECOVERY_CODE');
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import ResetPasswordMfaRecoveryCodeChallenge from '@auth0/auth0-acul-js/reset-password-mfa-recovery-code-challenge';
  const resetPasswordMfaRecoveryCodeChallengeManager = new ResetPasswordMfaRecoveryCodeChallenge();
  resetPasswordMfaRecoveryCodeChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッションの間保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  このメソッドは、パスワードリセットフロー中の MFA チャレンジを続行するために、リカバリーコードを送信します。

  ```typescript Example
  import ResetPasswordMfaRecoveryCodeChallenge from '@auth0/auth0-acul-js/reset-password-mfa-recovery-code-challenge';
  const resetPasswordMfaRecoveryCodeChallengeManager = new ResetPasswordMfaRecoveryCodeChallenge();
  await resetPasswordMfaRecoveryCodeChallengeManager.continue('RECOVERY_CODE');
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="code" type="string" required>
      ユーザーが入力したリカバリーコード。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーがない場合は空の配列を返します。
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドでは、チャレンジを完了するために別の MFA 方式を選択できます。

  ```typescript Example
  import ResetPasswordMfaRecoveryCodeChallenge from '@auth0/auth0-acul-js/reset-password-mfa-recovery-code-challenge';
  const resetPasswordMfaRecoveryCodeChallengeManager = new ResetPasswordMfaRecoveryCodeChallenge();
  await resetPasswordMfaRecoveryCodeChallengeManager.tryAnotherMethod();
  ```
</ParamField>
