MfaSmsChallenge クラスは、`mfa-sms-challenge` 画面の機能を提供します。この画面は、ユーザーが本人確認のために SMS のコードを入力する必要がある場合に表示されます。

<div id="constructors">
  ## コンストラクター
</div>

MFA SMS チャレンジ 画面マネージャーのインスタンスを作成します:

```typescript Example
import MfaSmsChallenge from '@auth0/auth0-acul-js/mfa-sms-challenge';

const mfaSmsChallenge = new MfaSmsChallenge();
await mfaSmsChallenge.continueMfaSmsChallenge({
  code: '123456',
  rememberDevice: true,
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaSmsChallenge from '@auth0/auth0-acul-js/mfa-sms-challenge';

  const mfaSmsChallenge = new MfaSmsChallenge();
  mfaSmsChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコード (たとえば、`'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueMfaSmsChallenge" type="Promise<void>">
  このメソッドは、指定されたコードを使用して MFA SMS チャレンジを送信します。

  ```typescript Example
  import MfaSmsChallenge from '@auth0/auth0-acul-js/mfa-sms-challenge';

  const mfaSmsChallenge = new MfaSmsChallenge();
  await mfaSmsChallenge.continueMfaSmsChallenge({
    code: '123456',
    rememberDevice: true,
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaSmsChallengeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/MfaSmsChallengeOptions)。
    </ParamField>

    <ParamField body="code" type="string" required>
      ユーザーが入力した SMS コード。
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      `true` に設定すると、このデバイスが信頼済みとして登録され、同じデバイスから今後ログインする際には MFA を求められなくなります。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getACall" type="Promise<void>">
  このメソッドは、確認方法を音声通話に切り替えます。

  ```typescript Example
  import MfaSmsChallenge from '@auth0/auth0-acul-js/mfa-sms-challenge';

  const mfaSmsChallenge = new MfaSmsChallenge();
  await mfaSmsChallenge.getACall();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーが存在しない場合は、空の配列を返します。
</ParamField>

<ParamField body="pickSms" type="Promise<void>">
  このメソッドは、ユーザーが別の SMS 番号を選択できる画面に移動します。

  ```typescript Example
  import MfaSmsChallenge from '@auth0/auth0-acul-js/mfa-sms-challenge';

  const mfaSmsChallenge = new MfaSmsChallenge();
  await mfaSmsChallenge.pickSms();
  ```
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  このメソッドは、SMS の確認コードをユーザーに再送します。

  ```typescript Example
  import MfaSmsChallenge from '@auth0/auth0-acul-js/mfa-sms-challenge';

  const mfaSmsChallenge = new MfaSmsChallenge();
  await mfaSmsChallenge.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  このメソッドは、設定可能なタイムアウトとステータス管理を備えた再送機能を提供します。

  ```typescript Example
  import MfaSmsChallenge from '@auth0/auth0-acul-js/mfa-sms-challenge';

  const mfaSmsChallenge = new MfaSmsChallenge();
  const { startResend } = mfaSmsChallenge.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    },
  });

  startResend();
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [StartResendOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions)。
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      再送のステータスが変化するたびに呼び出されるコールバックです。残り秒数と、現在再送が無効かどうかを受け取ります。
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      再送のタイムアウトが過ぎて、再送オプションが利用可能になったときに呼び出されるコールバックです。
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      再送オプションが利用可能になるまでの秒数です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドを使うと、ユーザーは別のMFA方式を試すことができます。

  ```typescript Example
  import MfaSmsChallenge from '@auth0/auth0-acul-js/mfa-sms-challenge';

  const mfaSmsChallenge = new MfaSmsChallenge();
  await mfaSmsChallenge.tryAnotherMethod();
  ```
</ParamField>
