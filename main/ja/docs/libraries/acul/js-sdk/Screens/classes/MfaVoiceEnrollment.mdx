MfaVoiceEnrollment クラスは、`mfa-voice-enrollment` 画面の機能を実装します。この画面は、登録を続行するための確認コードを音声通話で受け取るために、ユーザーが電話番号を入力する必要がある場合に表示されます。

<div id="constructors">
  ## コンストラクタ
</div>

MFA音声登録画面マネージャーのインスタンスを作成します:

```typescript Example
import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

const mfaVoiceEnrollment = new MfaVoiceEnrollment();
await mfaVoiceEnrollment.continue({ phone: '+1234567890' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

  const mfaVoiceEnrollment = new MfaVoiceEnrollment();
  mfaVoiceEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語がセッション中維持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  このメソッドは、指定した電話番号で音声登録プロセスを続行します。

  ```typescript Example
  import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

  const mfaVoiceEnrollment = new MfaVoiceEnrollment();
  await mfaVoiceEnrollment.continue({ phone: '+1234567890' });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaVoiceEnrollmentContinueOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/MfaVoiceEnrollmentContinueOptions)。
    </ParamField>

    <ParamField body="phone" type="string" required>
      音声通話を受けるユーザーの電話番号です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーがない場合は空の配列を返します。
</ParamField>

<ParamField body="selectPhoneCountryCode" type="Promise<void>">
  このメソッドは、ユーザーが電話番号の国番号を選択する画面に移動します。

  ```typescript Example
  import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

  const mfaVoiceEnrollment = new MfaVoiceEnrollment();
  await mfaVoiceEnrollment.selectPhoneCountryCode();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドは、ユーザーが別の MFA 方法を試せるようにします。

  ```typescript Example
  import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

  const mfaVoiceEnrollment = new MfaVoiceEnrollment();
  await mfaVoiceEnrollment.tryAnotherMethod();
  ```
</ParamField>
