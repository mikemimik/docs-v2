ResetPasswordMfaVoiceChallenge クラスは、`reset-password-mfa-voice-challenge` 画面の機能を実装します。この画面では、パスワードリセットフロー中の MFA ステップの一環として、音声通話で送信された確認コードを入力するようユーザーに求めます。

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import ResetPasswordMfaVoiceChallenge from '@auth0/auth0-acul-js/reset-password-mfa-voice-challenge';
  const resetPasswordMfaVoiceChallengeManager = new ResetPasswordMfaVoiceChallenge();
  resetPasswordMfaVoiceChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語は session の間保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  このメソッドは、パスワードリセット フロー中の MFA チャレンジ を続行するために、音声通話で届いた確認コードを送信します。

  ```typescript Example
  import ResetPasswordMfaVoiceChallenge from '@auth0/auth0-acul-js/reset-password-mfa-voice-challenge';
  const resetPasswordMfaVoiceChallengeManager = new ResetPasswordMfaVoiceChallenge();
  await resetPasswordMfaVoiceChallengeManager.continue({ code: '123456' });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [ResetPasswordMfaVoiceChallengeContinueOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaVoiceChallengeContinueOptions)。
    </ParamField>

    <ParamField body="code" type="string" required>
      音声通話でユーザーに送信された確認コード。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストから transaction errors の array を取得し、存在しない場合は空の array を返します。
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  このメソッドは、確認コードを音声通話でユーザーに再送します。

  ```typescript Example
  import ResetPasswordMfaVoiceChallenge from '@auth0/auth0-acul-js/reset-password-mfa-voice-challenge';
  const resetPasswordMfaVoiceChallengeManager = new ResetPasswordMfaVoiceChallenge();
  await resetPasswordMfaVoiceChallengeManager.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  このメソッドは、timeout management を備えた resend functionality を提供します。再送をトリガーする `startResend()` メソッドを持つ [`ResendControl`](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResendControl) オブジェクトを返し、ユーザーが再度再送できるようになるまでクールダウン時間を適用します。

  ```typescript Example
  import ResetPasswordMfaVoiceChallenge from '@auth0/auth0-acul-js/reset-password-mfa-voice-challenge';
  const resetPasswordMfaVoiceChallengeManager = new ResetPasswordMfaVoiceChallenge();
  const { startResend } = resetPasswordMfaVoiceChallengeManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // ユーザーが再送ボタンをクリックしたときに startResend を呼び出します
  startResend();
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [StartResendOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions)。
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      ユーザーが再度コードを再送できるようになるまでのクールダウン時間 (秒) 。
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      resend status が変化するたびに呼び出されるコールバックです。残り秒数と、resend functionality が現在無効かどうかを返します。
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      クールダウンが終了し、再送が再び可能になったときに呼び出されるコールバックです。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  このメソッドは、ユーザーを音声通話ではなくSMS認証に切り替えます。

  ```typescript Example
  import ResetPasswordMfaVoiceChallenge from '@auth0/auth0-acul-js/reset-password-mfa-voice-challenge';
  const resetPasswordMfaVoiceChallengeManager = new ResetPasswordMfaVoiceChallenge();
  await resetPasswordMfaVoiceChallengeManager.switchToSms();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドを使用すると、ユーザーはチャレンジを完了するために別のMFA方式を選択できます。

  ```typescript Example
  import ResetPasswordMfaVoiceChallenge from '@auth0/auth0-acul-js/reset-password-mfa-voice-challenge';
  const resetPasswordMfaVoiceChallengeManager = new ResetPasswordMfaVoiceChallenge();
  await resetPasswordMfaVoiceChallengeManager.tryAnotherMethod();
  ```
</ParamField>
