LogoutComplete クラスは、`logout-complete` 画面の機能を提供します。この画面は、ユーザーがログアウト処理を完了すると表示されます。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

LogoutComplete 画面マネージャーのインスタンスを生成します:

```typescript Example
import LogoutComplete from '@auth0/auth0-acul-js/logout-complete';
const logoutCompleteManager = new LogoutComplete();
logoutCompleteManager.changeLanguage({
  language: 'fr',
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import LogoutComplete from '@auth0/auth0-acul-js/logout-complete';
  const logoutCompleteManager = new LogoutComplete();
  logoutCompleteManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーが存在しない場合は、空の配列を返します。トランザクションのコンテキスト内のエラーオブジェクトの配列です。
</ParamField>
