---
title: "ContinueWithOrganizationNameOptions"
---

選択したorganization名での続行に関するオプション。

```ts Example
export interface ContinueWithOrganizationNameOptions {
  /**
   * organization名。
   */
  organizationName: string;
  /**
   * 追加のカスタムオプション。
   */
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## インデックスアクセス可能
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

追加のカスタム オプションを任意に指定できます。

<div id="properties">
  ## プロパティ
</div>

<ParamField body="organizationName" type="string">
  organization名。
</ParamField>
