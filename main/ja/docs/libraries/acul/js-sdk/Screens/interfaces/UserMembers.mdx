---
title: "UserMembers"
---

```ts Example
export interface UserMembers {
  id: string | null;
  email: string | null;
  username: string | null;
  phoneNumber: string | null;
  picture: string | null;
  enrolledFactors: Array<string> | null;
  enrolledEmails: Array<EnrolledEmail> | null;
  enrolledPhoneNumbers: Array<EnrolledPhoneNumber> | null;
  enrolledDevices: Array<EnrolledDevice> | null;
  organizations: Organizations[] | null; // OPT-IN（オプトイン）
  userMetadata: { [key: string]: string } | null; // OPT-IN（オプトイン）
  appMetadata: { [key: string]: string } | null; // OPT-IN（オプトイン）
}
```

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * OPT-IN: これらのプロパティを画面で利用できるようにするには、明示的な設定が必要です。詳しくは、[ACUL を設定する](/ja/docs/customize/login-pages/advanced-customizations/configure)をご覧ください。

  * Auth0 サーバーが現在のユーザーを認識していない場合、以下の画面では `user` プロパティを利用できません。

  | 画面        | 画面              | 画面           | 画面                     |
  | --------- | --------------- | ------------ | ---------------------- |
  | ログイン画面    | MFA OTP 画面      | 招待承諾画面       | InterstitialCaptcha 画面 |
  | Signup 画面 | MFA Phone 画面    | デバイスコード画面    | パスワード再設定画面             |
  | パスキー画面    | MFA プッシュ画面      | メール確認画面      | MFA 国番号画面              |
  | チャレンジ画面   | MFA リカバリーコード画面  | ログアウト中断画面    | MFA ブラウザー機能検出画面        |
  | メール識別子画面  | MFA WebAuthn 画面 | ログアウト完了画面    | MFA 登録画面               |
  | 電話番号識別子画面 | 招待承諾画面          | ブルートフォース対策画面 |                        |
</Callout>
