---
title: "識別子"
---

サインアップフォームまたは認証フローで使用する、設定された識別子フィールド (例: メールアドレス、電話番号、またはユーザー名) について説明します。

```ts Example
export interface Identifier {
  /**
   * 識別子のタイプ（`'email'`、`'phone'`、`'username'` など）。
   *
   * @see IdentifierType
   */
  type: IdentifierType;

  /**
   * この識別子がサインアップに必須かどうか。
   */
  required: boolean;
}
```

<div id="example">
  ## 例
</div>

```ts
const emailIdentifier: Identifier = {
  type: 'email',
  required: true
};
```
