<ParamField body="continue" type="Promise<void>">
  ユーザーが新しいリカバリーコードを保存したことを確認し、認証フローを続行します。
  このメソッドは、表示されたリカバリーコードを安全に保管したことをユーザーが示した後に呼び出してください。
  次のステップに進むための確認を Auth0 サーバーに送信します。

  確認の送信が正常に完了すると解決される Promise です。

  #### 送出

  送信に失敗した場合 (例: ネットワークの問題、無効な state) にエラーを送出します。

  ```typescript Example
  import MfaRecoveryCodeChallengeNewCode from '@auth0/auth0-acul-js/mfa-recovery-code-challenge-new-code';

  const screenManager = new MfaRecoveryCodeChallengeNewCode();

  // ユーザーがコードを保存したことを確認するチェックボックスをオンにしていると仮定します
  try {
    await screenManager.continue();
    // 成功すると次の画面にリダイレクトされます
  } catch (error) {
    console.error("Failed to confirm recovery code saved:", error);
    // エラーを処理します。例: screenManager.transaction.errors のメッセージを表示します
  }
  ```
