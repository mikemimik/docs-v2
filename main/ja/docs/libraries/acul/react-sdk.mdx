Auth0 ACUL React SDK を使えば、最小限の労力でカスタムの Universal Login エクスペリエンスを構築できます。ログイン、サインアップ、パスワードレス、パスキーの登録など、どのような場合でも、この SDK なら最新の認証フローにスムーズに対応できます。

<div id="why-choose-the-auth0-acul-react-sdk">
  ### Auth0 ACUL React SDK を選ぶ理由
</div>

* **迅速な連携:** `login-id` や `signup-password` などの構築済みの画面をすぐに導入できます。
* **モダンな機能:** パスワードレス、パスキー、ソーシャルログインをサポートします。
* **柔軟なモジュール:** 必要なものだけをインポートできます。
* **統一された状態管理:** 単一のトランザクションオブジェクトを使って、すべての認証フローを処理できます。

<div id="architecture">
  ### アーキテクチャ
</div>

<div id="how-it-works">
  #### どのような仕組みですか？
</div>

この SDK は、モジュール性と拡張性を考慮して設計されています。構成は次のとおりです。

* **画面:** `login-id` や `signup-password` など、各認証フローはそれぞれ独立したモジュールになっています。
* **送信ハンドラー:** フォームの送信処理と画面遷移を管理します。
* **動的フォーム:** 画面の設定に応じて、フィールドやバリデーションを調整します。
* **ヘルパープロパティ:** ブランディングやトランザクションなどの共通データに、一貫した方法でアクセスできます。
