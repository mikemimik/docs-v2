`mfa-phone-challenge` 画面では、SMS または音声通話で電話に送信されたコードを入力して本人確認を行います。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有のフックとメソッドがあります。SDK は、各画面で**partialインポート**と**rootインポート**をサポートしています。

* partialインポート を使用すると、特定のユースケースに必要なコードだけを含めることができます。
* rootインポート を使用すると、単一のバンドルからすべての画面を読み込めるため、想定されるあらゆる画面に対応する統一ビルドが必要な場合に便利です。

```jsx Import Example
// rootインポート
import { useMfaPhoneChallenge } from '@auth0/auth0-acul-react';

// partialインポート
import {
  useMfaPhoneChallenge,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティフック
  useChangeLanguage,
  // メソッド
  continueMethod,
  pickPhone,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-phone-challenge";

function MfaPhoneChallengeScreen() {
  const { continueMethod } = useMfaPhoneChallenge();
  return (
    <button onClick={() => continueMethod({ type: 'sms' })}>
      Send Code via SMS
    </button>
  );
}
```

<div id="context-hooks">
  ## コンテキストフック
</div>

`mfa-phone-challenge` 画面で、Auth0 のコンテキストデータへの読み取り専用アクセスを提供する画面スコープのフックです。`@auth0/auth0-acul-react/mfa-phone-challenge` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`mfa-phone-challenge` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-phone-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`mfa-phone-challenge` 画面の `id`、`name`、`logoUrl` など、クライアント関連の設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-phone-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、MFAフローが組織スコープの場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-phone-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フローにおける現在の prompt に関するデータが含まれます。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-phone-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaPhoneChallenge">ScreenMembersOnMfaPhoneChallenge</a></span>}>
  このフックには、設定やコンテキストを含む、`mfa-phone-challenge` 画面固有の詳細が含まれます。

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-phone-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  このフックには、`id` や関連するメタデータなど、テナントに関連するデータが含まれます。

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-phone-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  このフックは、現在の MFAフロー の状態など、`mfa-phone-challenge` 画面に関するトランザクション固有のデータを提供します。

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-phone-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  このフックは、URLパラメーターからの事前入力値など、画面に渡される信頼されていないデータを扱います。

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-phone-challenge';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```

</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  このフックは、`username`、`email`、利用可能な認証方法など、現在のユーザーの詳細を提供します。

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-phone-challenge';

  }
  ```

<ParamField body="useMfaPhoneChallenge" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPhoneChallengeMembers">MfaPhoneChallengeMembers</a>}>

</ParamField>

<div id="methods">
  ## メソッド
</div>

<ParamField body="continueMethod" type="Promise<void>">
  このメソッドは、指定した送信方法でユーザーの電話番号に確認コードを送り、電話チャレンジを開始します。

  ```jsx Example
  import { useMfaPhoneChallenge } from '@auth0/auth0-acul-react/mfa-phone-challenge';

  function PhoneChallengeForm() {
    const { continueMethod } = useMfaPhoneChallenge();
    return (
      <button onClick={() => continueMethod({ type: 'sms' })}>
        Send Code via SMS
      </button>
    );
  }
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [MfaPhoneChallengeContinueOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPhoneChallengeContinueOptions).
    </ParamField>

    <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;" required>
      実行するチャレンジの種類です。

      * `sms`: テキストメッセージでコードを送信します。
      * `voice`: 音声通話でコードを送信します。

      #### デフォルト

      ```ts
      'sms'
      ```
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickPhone" type="Promise<void>">
  このメソッドは電話番号の選択画面に移動し、ユーザーがチャレンジに使用する電話番号を選択できるようにします。

  ```jsx Example
  import { useMfaPhoneChallenge } from '@auth0/auth0-acul-react/mfa-phone-challenge';

  function PickPhoneButton() {
    const { pickPhone } = useMfaPhoneChallenge();
    return (
      <button onClick={() => pickPhone()}>
        Choose Phone Number
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドはMFA方式の選択画面に移動し、ユーザーが別の認証要素を選択できるようにします。

  ```jsx Example
  import { useMfaPhoneChallenge } from '@auth0/auth0-acul-react/mfa-phone-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaPhoneChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>

<div id="commonutility-hooks">
  ## 共通/ユーティリティ フック
</div>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  このフックは、ブランディングのコンテキストからフラット化された設定を含む現在のテーマオプションを取得します。

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  このフックは、現在の ACUL 画面で表示言語を変更するための関数を返します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  このフックは、現在の画面のコンテキストと状態を取得します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  このフックは、画面上のサーバーエラー、クライアントエラー、開発者向けエラーを取得して管理します。
