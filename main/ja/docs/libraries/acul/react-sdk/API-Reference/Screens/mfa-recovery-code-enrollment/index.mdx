`mfa-recovery-code-enrollment` 画面には、ユーザーが保存できるリカバリーコードが表示されます。これは、メインの MFA方式 を利用できなくなった場合に、アカウントにアクセスするために使用できます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有のフックとメソッドがあります。SDK では、各画面で **partial import** と **root import** の両方をサポートしています。

* partial import を使用すると、特定のユースケースに必要なコードだけを取り込めます。
* root import を使用すると、単一のバンドルからすべての画面を読み込めるため、想定されるあらゆる画面を 1 つの統一されたビルドで処理したい場合に便利です。

```jsx Import Example
// ルートインポート
import { useMfaRecoveryCodeEnrollment } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useMfaRecoveryCodeEnrollment,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティフック
  useChangeLanguage,
  // メソッド
  continueMethod,
} from "@auth0/auth0-acul-react/mfa-recovery-code-enrollment";

function MfaRecoveryCodeEnrollmentScreen() {
  const { continueMethod } = useMfaRecoveryCodeEnrollment();
  return (
    <button onClick={() => continueMethod({ isCodeCopied: true })}>
      I've Saved My Recovery Code
    </button>
  );
}
```

<div id="context-hooks">
  ## コンテキストHooks
</div>

`mfa-recovery-code-enrollment` 画面で Auth0 のコンテキストデータへの読み取り専用アクセスを提供する、画面スコープのフックです。`@auth0/auth0-acul-react/mfa-recovery-code-enrollment` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`mfa-recovery-code-enrollment` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`mfa-recovery-code-enrollment` 画面の `id`、`name`、`logoUrl` など、クライアント関連の設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、MFA フローが組織スコープの場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フロー内の現在の prompt に関するデータが含まれます。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaRecoveryCodeEnrollment">ScreenMembersOnMfaRecoveryCodeEnrollment</a></span>}>
  このフックには、設定やコンテキストを含む、`mfa-recovery-code-enrollment` 画面に固有の詳細が含まれます。

  ```jsx Example

  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>

<div id="methods">
  ## メソッド
</div>

<ParamField body="continueMethod" type="Promise<void>">
  このメソッドは、ユーザーがリカバリーコードを保存したことを確認し、登録フローを続行します。

  ```jsx Example
  import { useMfaRecoveryCodeEnrollment } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';

  function ContinueButton() {
    const { continueMethod } = useMfaRecoveryCodeEnrollment();
    return (
      <button onClick={() => continueMethod({ isCodeCopied: true })}>
        I've Saved My Recovery Code
      </button>
    );
  }
  ```

  **メソッド パラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaRecoveryCodeEnrollmentContinueOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaRecoveryCodeEnrollmentContinueOptions)。
    </ParamField>

    <ParamField body="isCodeCopied" type="boolean" required>
      ユーザーがリカバリーコードをコピーまたは保存したかどうかを示します。
    </ParamField>
  </Expandable>
</ParamField>
