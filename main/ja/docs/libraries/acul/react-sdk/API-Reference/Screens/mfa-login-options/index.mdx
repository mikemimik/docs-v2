`mfa-login-options` 画面には、ログインフロー中にユーザーが選択できる MFA 認証要素が表示されます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有のフックとメソッドのセットがあります。SDK では、各画面で **partial import** と **root import** をサポートしています。

* partial import を使用すると、特定のユースケースに必要なコードだけを含めることができます。
* root import を使用すると、単一のバンドルからすべての画面を読み込めるため、想定されるすべての画面に対応する単一のビルドにしたい場合に便利です。

```jsx Import Example
// ルートインポート
import { useMfaLoginOptions } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useMfaLoginOptions,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティフック
  useChangeLanguage,
  // メソッド
  enroll,
  returnToPrevious,
} from "@auth0/auth0-acul-react/mfa-login-options";

function MfaLoginOptionsScreen() {
  const { enroll } = useMfaLoginOptions();
  return (
    <button onClick={() => enroll({ action: 'totp' })}>
      Use Authenticator App
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

`mfa-login-options` 画面で、Auth0 のコンテキストデータへの読み取り専用アクセスを提供する画面スコープのフックです。`@auth0/auth0-acul-react/mfa-login-options` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`mfa-login-options` 画面に表示されるロゴ、色、テーマ設定などのブランディング構成を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-login-options';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`mfa-login-options` 画面の `id`、`name`、`logoUrl` など、クライアント関連の構成を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-login-options';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、MFA フローが組織にスコープ設定されている場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-login-options';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フロー内の現在のプロンプトに関するデータが含まれています。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-login-options';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaLoginOptions">ScreenMembersOnMfaLoginOptions</a></span>}>
  このフックには、設定やコンテキストなど、`mfa-login-options` 画面に固有の詳細が含まれています。

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-login-options';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  このフックには、`id` や関連するメタデータなど、テナントに関連するデータが含まれています。

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-login-options';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  このフックは、現在の MFA フローの状態など、`mfa-login-options` 画面の transaction 固有のデータを提供します。

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-login-options';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  このフックは、URLパラメーターから事前入力された値など、画面に渡される信頼できないデータを扱います。

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-login-options';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  このフックは、`username`、`email`、利用可能な認証方法など、現在アクティブなユーザーの詳細を提供します。

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-login-options';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaLoginOptions" type={<a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaLoginOptionsMembers">MfaLoginOptionsMembers</a>}>
  このフックは、`mfa-login-options` 画面で利用可能なすべての認証方法とコンテキストを返します。
</ParamField>

<div id="methods">
  ## メソッド
</div>

<ParamField body="enroll" type="Promise<void>">
  このメソッドは、指定した MFA 認証要素を選択し、その認証要素のチャレンジ画面にログインフローを進めます。

  ```jsx Example
  import { useMfaLoginOptions } from '@auth0/auth0-acul-react/mfa-login-options';

  function LoginOptions() {
    const { enroll } = useMfaLoginOptions();
    return (
      <button onClick={() => enroll({ action: 'totp' })}>
        Use Authenticator App
      </button>
    );
  }
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LoginEnrollOptions](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/LoginEnrollOptions)。
    </ParamField>

    <ParamField body="action" type="string" required>
      登録する認証要素を示す action です。&quot;push-notification&quot;、&quot;otp&quot;、&quot;sms&quot;、&quot;phone&quot;、&quot;voice&quot;、&quot;email&quot;、&quot;recovery-code&quot;、&quot;webauthn-roaming&quot;、&quot;webauthn-platform&quot;、または &quot;duo&quot; のいずれかを指定します。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="returnToPrevious" type="Promise<void>">
  このメソッドは、認証フローの前の画面に戻ります。

  ```jsx Example
  import { useMfaLoginOptions } from '@auth0/auth0-acul-react/mfa-login-options';

  function BackButton() {
    const { returnToPrevious } = useMfaLoginOptions();
    return (
      <button onClick={() => returnToPrevious()}>
        Back
      </button>
    );
  }
  ```
</ParamField>
