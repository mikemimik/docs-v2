`mfa-email-list` 画面には、MFA の確認に使用できるメールアドレスの一覧が表示され、ユーザーはその中から 1 つを選択できます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有の Hooks とメソッドがあります。SDK は、各画面で **部分インポート** と **ルートインポート** をサポートしています。

* 部分インポートを使うと、特定のユースケースに必要なコードだけを含めることができます。
* ルートインポートを使うと、単一のバンドルからすべての画面を読み込めるため、想定されるあらゆる画面に対応する統一ビルドが必要な場合に便利です。

```jsx Import Example
// ルートインポート
import { useMfaEmailList } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useMfaEmailList,
  // コンテキストHooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通Hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティHooks
  useChangeLanguage,
  // メソッド
  goBack,
  selectMfaEmail,
} from "@auth0/auth0-acul-react/mfa-email-list";

function MfaEmailListScreen() {
  const { selectMfaEmail } = useMfaEmailList();
  return (
    <button onClick={() => selectMfaEmail({ index: 0 })}>
      Select Email
    </button>
  );
}
```

<div id="context-hooks">
  ## コンテキストHooks
</div>

`mfa-email-list` 画面で、Auth0 のコンテキストデータへの読み取り専用アクセスを提供する、画面にスコープが設定されたHooksです。`@auth0/auth0-acul-react/mfa-email-list` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このHooksは、`mfa-email-list` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-email-list';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このHooksは、`mfa-email-list` 画面の `id`、`name`、`logoUrl` など、クライアントに関する設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-email-list';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このHooksは、MFA フローが組織にスコープが設定されている場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-email-list';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このHooksには、認証フローにおける現在の prompt に関するデータが含まれています。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-email-list';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaEmailList">ScreenMembersOnMfaEmailList</a></span>}>
  このHooksには、設定やコンテキストを含む、`mfa-email-list` 画面固有の詳細が含まれています。

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-email-list';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  このHooksには、`id` や関連するメタデータなど、テナントに関するデータが含まれています。

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-email-list';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  このHooksは、現在の MFA フローの状態など、`mfa-email-list` 画面におけるトランザクション固有のデータを提供します。

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-email-list';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  このHooksは、URLパラメーターから事前入力された値など、画面に渡される信頼できないデータを扱います。

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-email-list';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  このフックは、`username`、`email`、利用可能な認証方法など、現在のユーザーの詳細を提供します。

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-email-list';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaEmailList" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaEmailListMembers">MfaEmailListMembers</a>}>
  このフックは、`mfa-email-list` 画面で利用できるすべてのメソッドとコンテキストを返します。
</ParamField>

<div id="methods">
  ## メソッド
</div>

<ParamField body="goBack" type="Promise<void>">
  このメソッドは、MFAフローの前の画面に戻ります。

  ```jsx Example
  import { useMfaEmailList } from '@auth0/auth0-acul-react/mfa-email-list';

  function BackButton() {
    const { goBack } = useMfaEmailList();
    return (
      <button onClick={() => goBack()}>
        Back
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="selectMfaEmail" type="Promise<void>">
  このメソッドは、MFA challenge に使用するメールアドレスを一覧から選択します。

  ```jsx Example
  import { useMfaEmailList } from '@auth0/auth0-acul-react/mfa-email-list';

  function EmailSelector() {
    const { selectMfaEmail } = useMfaEmailList();
    return (
      <button onClick={() => selectMfaEmail({ index: 0 })}>
        Use This Email
      </button>
    );
  }
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [SelectMfaEmailOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/SelectMfaEmailOptions)。
    </ParamField>

    <ParamField body="index" type="number" required>
      選択するメールアドレスのインデックス。
    </ParamField>
  </Expandable>
</ParamField>
