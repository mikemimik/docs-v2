`mfa-webauthn-roaming-enrollment` 画面では、YubiKey やその他の FIDO2 ハードウェア認証器などのローミングセキュリティキーを、MFA の認証要素として登録するようユーザーに求めます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有の Hooks とメソッドのセットがあります。SDK は、各画面で **partial import** と **root import** をサポートしています。

* partial import を使用すると、特定のユースケースに必要なコードだけを含めることができます。
* root import を使用すると、単一のバンドルからすべての画面を読み込めます。これは、想定されるすべての画面に対応する統一ビルドを使いたい場合に便利です。

```jsx Import Example
// ルートインポート
import { useMfaWebAuthnRoamingEnrollment } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useMfaWebAuthnRoamingEnrollment,
  // コンテキスト Hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通 Hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティ Hooks
  useChangeLanguage,
  // メソッド
  enroll,
  showError,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment";

function MfaWebAuthnRoamingEnrollmentScreen() {
  const { enroll } = useMfaWebAuthnRoamingEnrollment();
  return (
    <button onClick={() => enroll()}>
      Enroll Security Key
    </button>
  );
}
```

<div id="context-hooks">
  ## コンテキスト Hooks
</div>

`mfa-webauthn-roaming-enrollment` 画面で Auth0 のコンテキストデータへの読み取り専用アクセスを提供する、画面スコープの Hooks です。`@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`mfa-webauthn-roaming-enrollment` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`mfa-webauthn-roaming-enrollment` 画面の `id`、`name`、`logoUrl` など、クライアントに関連する設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、MFA フローが Organization スコープの場合に、ユーザーの組織に関する情報を提供します。Organization コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フロー内の現在のプロンプトに関するデータが含まれます。

  ```jsx Example

  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```

<div id="methods">
  ## メソッド
</div>

<ParamField body="enroll" type="Promise<void>">
  このメソッドは、ローミングセキュリティキーの登録プロセスを開始します。

  ```jsx Example
  import { useMfaWebAuthnRoamingEnrollment } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';

  function EnrollButton() {
    const { enroll } = useMfaWebAuthnRoamingEnrollment();
    return (
      <button onClick={() => enroll()}>
        Enroll Security Key
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="showError" type="Promise<void>">
  このメソッドは、セキュリティキーの登録中に発生したWebAuthnエラーを報告します。

  ```jsx Example
  import { useMfaWebAuthnRoamingEnrollment } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';

  function HandleWebAuthnError({ error }) {
    const { showError } = useMfaWebAuthnRoamingEnrollment();
    return (
      <button onClick={() => showError({ error })}>
        Report Error
      </button>
    );
  }
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [ShowErrorOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ShowErrorOptions)。
    </ParamField>

    <ParamField body="error" type={<span><a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      報告対象となる、WebAuthn API (`navigator.credentials.create()`) から返されたerror object。

      <ParamField body="name" type="string" required>
        エラー名 (例: `'NotAllowedError'`) 。
      </ParamField>

      <ParamField body="message" type="string" required>
        発生した問題を説明するエラーメッセージ。
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドはMFA方式の選択画面に移動し、ユーザーが別の認証要素を選べるようにします。

  ```jsx Example
  import { useMfaWebAuthnRoamingEnrollment } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-enrollment';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaWebAuthnRoamingEnrollment();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>
