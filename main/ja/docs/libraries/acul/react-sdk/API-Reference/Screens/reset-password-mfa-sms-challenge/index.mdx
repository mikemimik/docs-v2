`reset-password-mfa-sms-challenge` 画面は、SMS による多要素認証が必要な場合に、パスワードリセットフロー中に表示されます。この画面では、リセット処理を続けるために、ユーザーは電話に送信された確認コードを入力する必要があります。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有の Hooks とメソッドのセットがあります。SDK では、各画面ごとに **部分インポート** と **ルートインポート** をサポートしています。

* 部分インポートを使用すると、特定のユースケースに必要なコードだけを含められます。
* ルートインポートを使用すると、単一のバンドルからすべての画面を読み込めるため、想定されるすべての画面に対応する統一ビルドが必要な場合に便利です。

```jsx Import Example
// ルートインポート
import { useResetPasswordMfaSmsChallenge } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useResetPasswordMfaSmsChallenge,
  // コンテキストHooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通Hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useResend,
  // ユーティリティHooks
  useChangeLanguage,
  // メソッド
  continueMfaSmsChallenge,
  getACall,
  resendCode,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-sms-challenge';

function ResetPasswordMfaSmsChallengeScreen() {
  const { continueMfaSmsChallenge } = useResetPasswordMfaSmsChallenge();
  return (
    <button onClick={() => continueMfaSmsChallenge({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">

`reset-password-mfa-sms-challenge` 画面で Auth0 のコンテキストデータへの読み取り専用アクセスを提供する、画面スコープの Hooks です。`@auth0/auth0-acul-react/reset-password-mfa-sms-challenge` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`reset-password-mfa-sms-challenge` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example

  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`reset-password-mfa-sms-challenge` 画面で使用される `id`、`name`、`logoUrl` などのクライアント関連の設定を提供します。


<ParamField body="useOrganization" type={<span>() =&gt; <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、パスワードリセットが組織にスコープが設定されている場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example

  function OrgSelector() {
    const organization = useOrganization();

      return <p>組織コンテキストがありません</p>;
    }
  }
  ```
</ParamField>

<div id="methods">
  ## メソッド
</div>

<ParamField body="continueMfaSmsChallenge" type="Promise<void>">
  このメソッドは、パスワードリセットフローのMFAステップを続行するために、SMSで受け取った確認コードを送信します。

  ```jsx Example
  import { useResetPasswordMfaSmsChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-sms-challenge';

  function VerifyCodeForm() {
    const { continueMfaSmsChallenge } = useResetPasswordMfaSmsChallenge();
    return (
      <button onClick={() => continueMfaSmsChallenge({ code: '123456' })}>
        Verify Code
      </button>
    );
  }
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [ResetPasswordMfaSmsChallengeOptions](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordMfaSmsChallengeOptions)。
    </ParamField>

    <ParamField body="code" type="string" required>
      ユーザーが入力したSMSの確認コード。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getACall" type="Promise<void>">
  このメソッドは、SMSではなく音声通話で確認コードを受け取れるようリクエストします。

  ```jsx Example
  import { useResetPasswordMfaSmsChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-sms-challenge';

  function GetACallButton() {
    const { getACall } = useResetPasswordMfaSmsChallenge();
    return (
      <button onClick={() => getACall()}>
        Call Me Instead
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  このメソッドは、SMSの確認コードをユーザーの電話番号に再送します。

  ```jsx Example
  import { useResetPasswordMfaSmsChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-sms-challenge';

  function ResendCodeButton() {
    const { resendCode } = useResetPasswordMfaSmsChallenge();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドは、ユーザーが別のMFA方式を選択できる画面に移動します。

  ```jsx Example
  import { useResetPasswordMfaSmsChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-sms-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useResetPasswordMfaSmsChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>
