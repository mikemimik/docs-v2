---
title: "MfaOtpContinueOptions"
---

MFA OTP の登録コードで続行するためのオプション。

```ts Example
export interface ContinueOptions {
  /**
   * ユーザーが入力したOTPコード。
   */
  code: string;

  /**
   * その他のカスタムオプション。
   */
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## インデックス可能
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

追加のカスタムオプション。

<div id="properties">
  ## プロパティ
</div>

<ParamField body="code" type="string">
  ユーザーが入力するOTPコード。
</ParamField>
