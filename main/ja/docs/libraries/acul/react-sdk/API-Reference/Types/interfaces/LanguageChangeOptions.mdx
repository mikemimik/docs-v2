---
title: "LanguageChangeOptions"
---

[`useChangeLanguage`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage) が返す関数で使用できるオプションです。

```typescript
interface LanguageChangeOptions {
  language: string;
  persist?: "session";
  [key: string]: string | number | boolean | undefined;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="language" type="string" required>
  切り替え先のロケールコード。Auth0 テナントで有効になっているロケールのいずれかと一致する必要があります。例: `"en"`, `"fr"`, `"es"`。
</ParamField>

<ParamField body="persist" type="&#x22;session&#x22;">
  言語設定を保持する範囲。現在サポート対象なのは `"session"` のみです。デフォルトは `"session"` です。
</ParamField>

<div id="remarks">
  ## 注記
</div>

* `language` と `persist` に加えて、追加のカスタムフィールドを渡すこともできます。カスタムフィールドには `"ulp-"` プレフィックスを付ける必要があり、`string`、`number`、`boolean`、または `undefined` 型の値を受け付けます。
