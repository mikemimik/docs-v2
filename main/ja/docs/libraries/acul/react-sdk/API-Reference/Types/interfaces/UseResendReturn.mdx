---
title: "UseResendReturn"
---

[useResend](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/useResend) の戻り値の型です。

```ts Example
export interface UseResendReturn {
  remaining: number;
  disabled: boolean;
  startResend: () => void;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="remaining" type="number" required>
  次に再送できるようになるまでの残り秒数です。UI のカウントダウンタイマーの制御に使用します。
</ParamField>

<ParamField body="disabled" type="boolean" required>
  再送処理が進行中であるか、クールダウン期間がまだ終了していないために、現在再送アクションを利用できない場合は `true` です。
</ParamField>

<ParamField body="startResend" type="() => void" required>
  現在アクションが許可されている場合は、すぐに再送を試行します。
</ParamField>
