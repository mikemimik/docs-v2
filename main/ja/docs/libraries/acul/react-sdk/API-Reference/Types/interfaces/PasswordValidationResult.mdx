---
title: "PasswordValidationResult"
---

[`usePasswordValidation`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation) の戻り値の型です。パスワード全体のバリデーション結果と、評価されたパスワード複雑性ルールの詳細な一覧が含まれます。

```typescript
export interface PasswordValidationResult {
  isValid: boolean;
  results: PasswordComplexityRule[];
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="isValid" type="boolean" required>
  すべてのパスワードポリシーチェックの総合結果。[`PasswordComplexityRule`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) 内のすべてのルールで `status` が `"valid"` と等しい場合にのみ `true` になります。
</ParamField>

<ParamField body="results" type={<span><a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>} required>
  評価済みのパスワードルールの配列。各項目は [`PasswordComplexityRule`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) で、その [`status`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule#status) フィールドに、そのルールの計算済みのバリデーション結果が反映されます。
</ParamField>
