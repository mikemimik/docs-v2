---
title: "MfaPushEnrollmentQrWithRememberOptions"
---

オプションのデバイス信頼設定を指定して、MFAプッシュ登録を続行するためのオプションです。

`CustomOptions` を拡張し、デバイスを記憶する設定とあわせてカスタムオプションを渡せるようにします。

```ts Example
export interface MfaPushEnrollmentQrWithRememberOptions extends CustomOptions {
  rememberDevice?: boolean;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## インデックス指定可能
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## プロパティ
</div>

<ParamField body="rememberDevice?" type="boolean">
  `true` に設定すると、このデバイスが信頼済みとして登録され、以後同じデバイスからログインする際に MFA プッシュ認証を求められなくなります。
</ParamField>
