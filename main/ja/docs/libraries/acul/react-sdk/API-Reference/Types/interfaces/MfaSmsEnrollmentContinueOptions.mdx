---
title: "MfaSmsEnrollmentContinueOptions"
---

MFA の SMS 登録を続行するためのオプション。

`CustomOptions` を拡張し、phone や captcha とあわせてカスタムオプションを渡せるようにします。

```ts Example
export interface MfaSmsEnrollmentContinueOptions extends CustomOptions {
  phone: string;
  captcha?: string;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## インデックス指定可能
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## プロパティ
</div>

<ParamField body="phone" type="string" required>
  SMS登録に使用するユーザーの電話番号。
</ParamField>

<ParamField body="captcha?" type="string">
  検証に使用するオプションのCAPTCHAトークン。
</ParamField>
