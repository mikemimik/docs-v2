国番号を選択するオプション

```ts Example
export interface SelectCountryCodeOptions {
  /** 国コード（例：'US'、'GB'） */
  country_code: string;
  /** 電話番号のプレフィックス（例：'+1'、'+44'） */
  phone_prefix: string;
  /** その他のカスタムオプション */
  [key: string]: string | number | boolean | undefined;
}
```
