---
title: "MfaPollingResult"
---

[useMfaPolling](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling) が返す結果オブジェクト。

<div id="properties">
  ## プロパティ
</div>

<ParamField body="isRunning" type="boolean">
  MFAプッシュのポーリング処理が現在実行中かどうかを示します。

  * `true` — ポーリングは実行中で、完了を待っています。
  * `false` — ポーリングは停止しており、完了、
    手動でのキャンセル、またはコンポーネントのアンマウントが原因です。
</ParamField>

<ParamField body="startPolling()" type="void">
  ポーリング処理を開始または再開します。

  * ポーリングがすでに実行中の場合、この呼び出しによる影響はありません。
  * 以前に停止していた場合は、これを呼び出すとポーリングループが再開されます。
</ParamField>

<ParamField body="stopPolling()" type="void">
  ポーリング処理を直ちに停止します。

  * スケジュールされているタイマーや進行中のリクエストをすべてキャンセルします。
  * 複数回呼び出しても安全で、それ以降の呼び出しによる影響はありません。
</ParamField>
