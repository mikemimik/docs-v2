続行オプション

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * ユーザーがWebAuthnセキュリティキーに割り当てたい新しいニックネーム。
   * この値はサーバーに送信され、更新されます。
   * サーバーはこのニックネームを検証します（例：文字数、使用可能な文字）。
   * @type {string}
   * @example "Primary YubiKey"
   * @example "Home Office Key"
   */
  nickname: string;
}
```

  ## プロパティ

<ParamField body="nickname" type="string">
  WebAuthnセキュリティキーの新しいニックネームです。
  これは、ユーザーが入力フィールドに入力する値です。
  サーバーで定義された文字数および使用可能文字の制約に従う必要があります。
</ParamField>
