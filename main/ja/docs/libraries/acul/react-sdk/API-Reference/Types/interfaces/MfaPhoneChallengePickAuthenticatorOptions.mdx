PickAuthenticatorOptions
&#39;pick-authenticator&#39; アクションのオプションを定義します。
これにより、利用可能な場合はユーザーが別のMFA方式を選択できます。

```ts Example
export interface PickAuthenticatorOptions extends CustomOptions {}
```
