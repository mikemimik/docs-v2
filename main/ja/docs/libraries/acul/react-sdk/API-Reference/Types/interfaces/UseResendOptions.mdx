---
title: "UseResendOptions"
---

[useResend](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useResend) のオプション設定。

```ts Example
export interface UseResendOptions {
  timeoutSeconds?: number;
  onTimeout?: () => void;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="timeoutSeconds" type="number" default="10">
  再送信が再び可能になるまでのカウントダウン時間 (秒) 。
  デフォルトは `10` です。
</ParamField>

<ParamField body="onTimeout" type="() => void">
  カウントダウンが終了し、再送信操作が再び可能になったときに呼び出されるコールバック。
</ParamField>
