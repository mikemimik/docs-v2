title: "MfaVoiceChallengeMembers"
---

MFA Voice Challenge画面で使用できるすべてのメンバーと操作を定義するインターフェース。

````ts Example
export interface MfaVoiceChallengeMembers extends BaseMembers {
  /**
   * 画面固有のプロパティとデータ。
   */
  screen: ScreenMembersOnMfaVoiceChallenge;

  untrustedData: UntrustedDataMembersOnMfaVoiceChallenge;

  /**
   * 音声確認コードを送信してMFAチャレンジを検証します。
   *
   * @param payload - 確認コードとオプションパラメーターを含むオブジェクト
   * @returns コードの検証が成功すると解決されるPromise
   *
   * @example
   * ```typescript
   * const mfaVoiceChallenge = new MfaVoiceChallenge();
   * mfaVoiceChallenge.continue({
   *   code: '123456',
   *   rememberDevice: true
   * });
   * ```
   */
  continue(payload: MfaVoiceChallengeContinueOptions): Promise<void>;

  /**
   * 別の電話番号を選択する画面に移動します。
   *
   * @param payload - オプションのカスタムパラメーター
   * @returns 画面遷移が完了すると解決されるPromise
   *
   * @example
   * ```typescript
   * const mfaVoiceChallenge = new MfaVoiceChallenge();
   * mfaVoiceChallenge.pickPhone();
   * ```
   */
  pickPhone(payload?: CustomOptions): Promise<void>;

  /**
   * 音声通話の代わりにSMS認証方式に切り替えます。
   *
   * @param payload - オプションのカスタムパラメーター
   * @returns 切り替えが完了すると解決されるPromise
   *
   * @example
   * ```typescript
   * const mfaVoiceChallenge = new MfaVoiceChallenge();
   * mfaVoiceChallenge.switchToSms();
   * ```
   */
  switchToSms(payload?: CustomOptions): Promise<void>;

  /**
   * 確認コードを含む新しい音声通話をリクエストします。
   *
   * @param payload - オプションのカスタムパラメーター
   * @returns 新しいコードが送信されると解決されるPromise
   *
   * @example
   * ```typescript
   * const mfaVoiceChallenge = new MfaVoiceChallenge();
   * mfaVoiceChallenge.resendCode();
   * ```
   */
  resendCode(payload?: CustomOptions): Promise<void>;

  /**
   * 別のMFA方式を選択する画面に移動します。
   *
   * @param payload - オプションのカスタムパラメーター
   * @returns 画面遷移が完了すると解決されるPromise
   *
   * @example
   * ```typescript
   * const mfaVoiceChallenge = new MfaVoiceChallenge();
   * mfaVoiceChallenge.tryAnotherMethod();
   * ```
   */
  tryAnotherMethod(payload?: CustomOptions): Promise<void>;

  /**
   * この画面の再送機能とタイムアウト管理を取得します
   * @param options 再送機能の設定オプション
   */
  resendManager(options?: StartResendOptions): ResendControl;
}
````

<ParamField body="continue" type="Promise<void>">
  MFAチャレンジを検証するため、音声による確認コードを送信します。

  コードの検証が正常に完了すると解決されるPromise

  ```typescript Example
  const mfaVoiceChallenge = new MfaVoiceChallenge();
  mfaVoiceChallenge.continue({
    code: '123456',
    rememberDevice: true
  });
  ```

  <Expandable title="パラメーター">

      **プロパティ**

      <ParamField body="code" type="string">
