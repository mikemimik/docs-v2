---
title: "UseErrorOptions"
---

[`useError`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/useError) フックのオプションオブジェクトです。Auth0 Universal Login の画面内でのエラー表示の動作を制御します。

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="includeDevErrors" type="boolean">
  `true` の場合、エラー出力に開発時専用の詳細なエラー情報が含まれます。開発中のデバッグには便利ですが、本番環境では無効にする必要があります。
</ParamField>
