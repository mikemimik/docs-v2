---
title: "ErrorItem"
---

ACUL のエラーストア内の単一のエラーエントリを表します。[`useErrors`](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) によって、[`UseErrorsResult`](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorsResult) オブジェクトの一部として返されます。基本の `Error` インターフェースを拡張したもので、`id`、`type`、および省略可能な `label` を含みます。

```typescript
interface ErrorItem {
  id: string;
  code: string;
  message: string;
  type?: ErrorType;
  field?: string;
  label?: string;
  rules?: PasswordComplexityRule[];
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="id" type="string" required>
  このエラーインスタンスの一意の識別子です。`dismiss(id)` で特定のエラーを指定するために使用されます。
</ParamField>

<ParamField body="code" type="string" required>
  エラーの種類を識別する機械可読なエラーコードです。例: `"invalid_password"`、`"username_exists"`。
</ParamField>

<ParamField body="message" type="string" required>
  エラーの人が読んで理解できる説明です。UI に表示するメッセージとして適しています。
</ParamField>

<ParamField body="type" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>}>
  エラーを `"auth0"`、`"validation"`、または `"configuration"` に分類します。`errors.byType()` での絞り込みに使用されます。
</ParamField>

<ParamField body="field" type="string">
  該当する場合、このエラーに関連付けられたフォームフィールドです。例: `"username"`、`"email"`。`errors.byField()` での絞り込みに使用されます。
</ParamField>

<ParamField body="label" type="string">
  エラーの任意の表示ラベルです。エラーメッセージと併せて、わかりやすいフィールド名を表示する必要がある場合に使用されます。
</ParamField>

<ParamField body="rules" type={<span><a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>}>
  エラーがパスワードのバリデーションに関連している場合に、このエラーに関連付けられたパスワード複雑性ルールです。
</ParamField>
