---
title: "ErrorsResult"
---

[`useErrors`](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) フックによって [`UseErrorsResult`](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorsResult) の一部として返される、配列のような object です。`ReadonlyArray<ErrorItem>` を拡張しているため、直接反復処理したり、展開したり、インデックスでアクセスしたりできます。型またはフィールドでエラーの一覧を照会するための 2 つのメソッドを提供します。

```typescript
interface ErrorsResult extends ReadonlyArray<ErrorItem> {
  byType(type: ErrorType, opts?: { field?: string }): readonly ErrorItem[];
  byField(field: string, opts?: { type?: ErrorType }): readonly ErrorItem[];
}
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="byType" type={<span>readonly <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem">ErrorItem</a>[]</span>}>
  指定した [`ErrorType`](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType) に一致するすべてのエラーを返します。必要に応じて、特定のフィールドに絞り込むこともできます。

  <Expandable title="パラメータ">
    <ParamField body="type" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>} required>
      絞り込みに使用するエラータイプ: `"auth0"`、`"validation"`、または `"configuration"`。
    </ParamField>

    <ParamField body="opts" type="{ field?: string }">
      オプションのフィルター設定です。

      <Expandable title="プロパティ">
        <ParamField body="field" type="string">
          指定した場合、結果をこのフィールド名に関連付けられたエラーにさらに絞り込みます。
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="byField" type={<span>readonly <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem">ErrorItem</a>[]</span>}>
  指定したフィールド名に関連付けられたすべてのエラーを返します。必要に応じて、特定の [`ErrorType`](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType) に絞り込むこともできます。

  <Expandable title="パラメータ">
    <ParamField body="field" type="string" required>
      絞り込みに使用するフィールド名です。例: `"username"`、`"email"`。
    </ParamField>

    <ParamField body="opts" type={<span>&#123; type?: <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a> &#125;</span>}>
      オプションのフィルター設定です。

      <Expandable title="プロパティ">
        <ParamField body="type" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>}>
          指定した場合、結果をこのタイプのエラーにさらに絞り込みます。
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>
