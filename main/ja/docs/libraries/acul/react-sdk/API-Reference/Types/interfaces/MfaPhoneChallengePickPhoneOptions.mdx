PickPhoneOptions
&#39;pick-phone&#39; Action のオプションを定義します。
インターフェースには電話番号が 1 つしか表示されない場合でも、この Action は
サーバーで電話番号選択画面へのリダイレクトが必要なシナリオで使用されることがあります。

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```

<div id="indexable">
  ## インデックス可能
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`
