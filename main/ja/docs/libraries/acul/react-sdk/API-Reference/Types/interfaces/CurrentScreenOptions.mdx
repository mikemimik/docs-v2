---
title: "CurrentScreenOptions"
---

Auth0 Universal Loginフローにおける現在の画面コンテキストを表すoptions objectです。実行時に利用可能なクライアント、組織、テナント、トランザクション、画面のメタデータにアクセスできます。

```ts Example
export interface CurrentScreenOptions {
  client: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  organization: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  prompt: {
    name: string;
  } | null;
  screen: {
    name: string;
  } | null;
  tenant: {
    enabledLocales: string[];
  } | null;
  transaction: {
    errors: TransactionError[] | null;
    state: string;
    locale: string;
  } | null;
  untrustedData: {
    authorizationParams: Record<string, string> | null;
  } | null;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="client" type="{ id: string; metadata: Record<string, string> | null } | null">
  現在の認可リクエストに関するクライアントの詳細です。クライアント識別子と、関連するメタデータが含まれます。
</ParamField>

<ParamField body="organization" type="{ id: string; metadata: Record<string, string> | null } | null">
  現在のリクエストの組織コンテキストです。組織識別子とメタデータが含まれます。
</ParamField>

<ParamField body="prompt" type="{ name: string } | null">
  現在のprompt設定です。どのpromptステップがレンダリングされているかを示します。
</ParamField>

<ParamField body="screen" type="{ name: string } | null">
  現在の画面設定です。prompt内のどの画面がアクティブかを示します。
</ParamField>

<ParamField body="tenant" type="{ enabledLocales: string[] } | null">
  テナントレベルの設定です。このUniversal Loginデプロイで有効になっているロケールの一覧が含まれます。
</ParamField>

<ParamField body="transaction" type="{ errors: TransactionError[] | null; state: string; locale: string } | null">
  現在の認証フローのトランザクション stateです。バリデーションエラー、state識別子、アクティブなロケールが含まれます。
</ParamField>

<ParamField body="untrustedData" type="{ authorizationParams: Record<string, string> | null } | null">
  フローに渡される信頼できないデータです。たとえば、セキュリティ上の判断の根拠にすべきでない認可パラメータなどが含まれます。
</ParamField>
