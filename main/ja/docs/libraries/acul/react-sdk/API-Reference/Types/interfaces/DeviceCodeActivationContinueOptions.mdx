---
title: "DeviceCodeActivationContinueOptions"
---

デバイスコードのアクティベーションを続行するためのペイロード。

このインターフェースは `CustomOptions` を拡張しているため、すべてのカスタムオプションのプロパティが
オブジェクトのトップレベルにマージされます。これによりネストを避けられ、
このインターフェースを利用する側にとってよりシンプルな構造になります。

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```

  ## インデックス可能
</div>
