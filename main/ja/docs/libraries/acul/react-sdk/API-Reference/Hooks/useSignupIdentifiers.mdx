<ParamField body="useSignupIdentifiers()" type={<span>[] | <a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier">Identifier[]</a></span>}>
  有効になっている識別子 (メールアドレス、電話番号、またはユーザー名) の一覧を返します。各識別子には `required` ステータスが含まれ、
  現在の画面のサインアップ識別子に基づいて決まります。

  ## 戻り値

  [] | [`Identifier`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier)[]

  各要素に `type` (識別子の種類) と、サインアップ時に必須かどうかを示す `required` フラグを含む、[Identifier](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier) オブジェクトの配列。

  ## サポート対象の画面

  * `signup`
  * `signup-id`

  ```tsx Example
  import { useSignupIdentifiers } from '@auth0/auth0-acul-react/signup';

  const identifiers = useSignupIdentifiers();
  const emailIdentifier = identifiers.find(({ type }) => type === 'email');
  const phoneIdentifier = identifiers.find(({ type }) => type === 'phone');
  const usernameIdentifier = identifiers.find(({ type }) => type === 'username');

  const emailRequired = emailIdentifier?.required ?? false;
  const phoneRequired = phoneIdentifier?.required ?? false;
  const usernameRequired = usernameIdentifier?.required ?? false;

  // 出力例:
  // [
  //   { type: 'email', required: true },
  //   { type: 'phone', required: false },
  //   { type: 'username', required: true },
  // ]
  ```
