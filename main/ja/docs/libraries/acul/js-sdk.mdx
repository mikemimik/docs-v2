Auth0 ACUL JS SDK を使うと、わずかな手間でカスタムの Universal Login エクスペリエンスを構築できます。ログイン、サインアップ、パスワードレス、パスキーの登録など、どのユースケースでも、この SDK なら最新の認証フローにスムーズに対応できます。

<div id="why-choose-the-auth0-acul-js-sdk">
  ### Auth0 ACUL JS SDK を選ぶ理由
</div>

* **スピーディーな連携:** `login-id` や `signup-password` などのあらかじめ用意された画面をすぐに導入できます。
* **モダンな機能:** パスワードレス、パスキー、ソーシャルログインに対応。
* **柔軟なモジュール:** 必要なものだけをインポートできます。
* **一元的な状態管理:** 1 つの トランザクション オブジェクトで、すべての認証フローを処理できます。

<div id="architecture">
  ### アーキテクチャ
</div>

<div id="how-it-works">
  #### どのような仕組みですか？
</div>

SDK は、モジュール性と拡張性を重視して構築されています。各要素は次のように連携します。

* **画面:** `login-id` や `signup-password` など、各認証フローは独立したモジュールです。
* **送信ハンドラー:** フォームの送信処理と画面遷移を管理します。
* **動的フォーム:** 画面の設定に応じて、フィールドやバリデーションを調整します。
* **ヘルパープロパティ:** ブランディングやトランザクションなどの共通データに、統一された方法でアクセスできます。
