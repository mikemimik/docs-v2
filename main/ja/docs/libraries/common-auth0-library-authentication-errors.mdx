title: 一般的なAuth0ライブラリの認証エラー
---

サインアップやログインの処理中に、ユーザーの操作や入力データが原因でエラーが発生することがあります。ここでは、認証にAuth0ライブラリを使用する場合に発生する可能性がある、代表的なエラーの一覧を示します。

<div id="sign-up">
  ## 新規登録
</div>

新規登録に失敗した場合、よくあるエラーは次のとおりです。

| **Error**                                       | **Description**                                                               |
| ----------------------------------------------- | ----------------------------------------------------------------------------- |
| **invalid&#95;password**                        | 使用したパスワードが、その接続のパスワードポリシーに準拠していない場合                                           |
| **invalid&#95;signup**                          | 新規登録しようとしているユーザーが無効です                                                         |
| **password&#95;dictionary&#95;error**           | 選択したパスワードは一般的すぎます                                                             |
| **password&#95;no&#95;user&#95;info&#95;error** | 選択したパスワードがユーザー情報に基づいています                                                      |
| **password&#95;strength&#95;error**             | 選択した[パスワードが脆弱すぎます](/docs/ja/authenticate/database-connections/password-strength) |
| **unauthorized**                                | このアプリケーションでは新規登録できない場合。特定のルールに違反している可能性があります                                  |
| **user&#95;exists**                             | 新規登録しようとしているユーザーはすでに登録されています                                                  |
| **username&#95;exists**                         | 新規登録に使用しようとしているユーザー名はすでに使用されています                                              |

<div id="log-in">
  ## ログイン
</div>

ログインに失敗した場合によく見られるエラーは、次のとおりです。

| **エラー**                               | **説明**                                                                                                                                                                                                   |
| ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **access&#95;denied**                 | Web ベースの認証を使用している場合、OAuth 2.0 の仕様に従ってリソースサーバーがアクセスを拒否します                                                                                                                                                 |
| **invalid&#95;user&#95;password**     | 認証に使用されたユーザー名またはパスワードが無効です                                                                                                                                                                               |
| **mfa&#95;invalid&#95;code**          | ユーザーが入力した<Tooltip tip="多要素認証 (MFA): SMS 経由のコードなど、ユーザー名とパスワードに加えて認証要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/docs/ja/glossary?term=multi-factor+authentication+%28MFA%29">多要素認証 (MFA)</Tooltip>コードが無効または期限切れです |
| **mfa&#95;registration&#95;required** | 管理者が [多要素認証](/docs/ja/secure/multi-factor-authentication) を必須にしていますが、ユーザーはまだ登録を完了していません                                                                                                                     |
| **mfa&#95;required**                  | ユーザーが認証されるには、[多要素認証](/docs/ja/secure/multi-factor-authentication) コードの入力が必要です                                                                                                                               |
| **password&#95;leaked**               | パスワードが漏えいしているため、別のパスワードを使用する必要があります                                                                                                                                                                      |
| **PasswordHistoryError**              | サインアップまたは更新時に指定されたパスワードは、すでに使用されています ([パスワード履歴](/docs/ja/authenticate/database-connections/password-options#password-history) 機能が有効な場合に報告されます)                                                              |
| **PasswordStrengthError**             | 指定されたパスワードが、この接続の[強度要件](/docs/ja/authenticate/database-connections/password-strength)を満たしていません                                                                                                              |
| **too&#95;many&#95;attempts**         | サインインの試行回数が多すぎるため、アカウントがブロックされています                                                                                                                                                                       |
| **unauthorized**                      | サインインしようとしているユーザーはブロックされています                                                                                                                                                                             |
