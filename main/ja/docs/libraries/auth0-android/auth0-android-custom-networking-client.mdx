このライブラリでは、一般的な構成に合わせてネットワーク クライアントの動作をカスタマイズしたり、独自のネットワーク クライアント実装を定義して利用したりできます。

Auth0 クラスは `NetworkingClient` で設定でき、リクエストの実行時にはその設定が使用されます。デフォルトのクライアントでは、カスタムのタイムアウト値、すべてのリクエストで送信するヘッダー、さらにリクエスト/レスポンス情報をログに記録するかどうか (本番環境以外でのデバッグ用途に限る) を設定できます。より高度な構成が必要な場合は、`NetworkingClient` の独自実装を指定できます。

<div id="timeout-configuration">
  ### タイムアウト設定
</div>

export const codeExample1 = `val netClient = DefaultClient(
    connectTimeout = 30,
    readTimeout = 30
)

val account = Auth0("{yourClientId}", "{yourDomain}")
account.networkingClient = netClient`;
