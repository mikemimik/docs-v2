Lock では、ログ機能のオン / オフを簡単に切り替えられるほか、ログ関連の各種設定も調整できます。

<div id="loglevel">
  ## logLevel
</div>

デフォルトは `.off` で、Syslog のログレベルに対応しています。

```swift lines
.withOptions {
  $0.logLevel = .all
}
```

<div id="loghttprequest">
  ## logHttpRequest
</div>

このオプションでは、Auth0.swift API のリクエストをログに記録するかどうかを選択できます。デフォルトでは `false` です。

```swift lines
.withOptions {
  $0.logHttpRequest = true
}
```

<div id="loggeroutput">
  ## loggerOutput
</div>

ロガーの出力ハンドラーを指定できます。デフォルトでは、`print` 文を使用します。

```swift lines
.withOptions {
  $0.loggerOutput = CleanroomLockLogger()
}
```

上記のコードでは、`loggerOutput` は [CleanroomLogger](https://github.com/emaloney/CleanroomLogger) を使用するように設定されています。通常、これは `loggerOutput` プロトコルを実装することで実現できます。もちろん、お好みのロガーライブラリを使用することもできます。以下は、CleanroomLogger を使ってロガー出力を処理する使用例です。

```swift lines
class CleanroomLockLogger: LoggerOutput {
  func message(_ message: String, level: LoggerLevel, filename: String, line: Int) {
    let channel: LogChannel?
    switch level {
    case .debug:
        channel = Log.debug
    case .error:
        channel = Log.error
    case .info:
        channel = Log.info
    case .verbose:
        channel = Log.verbose
    case .warn:
        channel = Log.warning
    default:
        channel = nil
    }
    channel?.message(message, filePath: filename, fileLine: line)
  }
}
```
