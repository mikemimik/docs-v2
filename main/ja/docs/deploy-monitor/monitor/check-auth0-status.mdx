Auth0のパブリッククラウド版のサービス可用性を確認するには、[Auth0 Public Cloud Status](https://status.auth0.com)ページにアクセスします。

ドロップダウンメニューから**Region**を選択すると、Core Services (User Authentication や Machine-to-Machine Authentication など) および Supporting Services (<Tooltip tip="多要素認証（MFA）: SMSで送信されるコードなど、ユーザー名とパスワードに加えて別の要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/ja/docs/glossary?term=MFA">MFA</Tooltip>、<Tooltip tip="Management API: 顧客が管理タスクを実行できるようにする製品。" cta="用語集を見る" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip>、テナントログ、CDN など) のステータスを確認できます。

利用可能なリージョンの詳細については、[テナントを作成](/ja/docs/get-started/auth0-overview/create-tenants)の[Region, locality, and sub-locality](/ja/docs/get-started/auth0-overview/create-tenants#region-locality-and-sub-locality)表を参照してください。

<div id="subscribe-to-public-cloud-status-updates">
  ## Public Cloud のステータス更新を購読する
</div>

更新を受け取るには、**Region** を選択してから **Subscribe** を選択します。ステータスを確認する方法は 2 つあります。

1. 最新のステータス更新を受け取るには、X で [@auth0status](https://x.com/auth0status) をフォローします。
2. Auth0 の Atom フィードまたは RSS フィードを購読すると、テナントに影響するステータス更新を受け取れます。`{YOUR_TENANT_DOMAIN}` はテナントの **Domain** に置き換えてください。

テナントの **Domain** を確認するには、[Auth0 Dashboard &gt; Applications](https://manage.auth0.com/dashboard/#/applications) に移動し、アプリケーションを選択してから **Settings** に進みます。テナントの **Domain** は **Basic Information** に表示されています。

<div id="check-auth0-status-for-private-cloud">
  ## Private Cloud の Auth0 ステータスを確認する
</div>

[Auth0 Public Cloud Status](https://status.auth0.com) にアクセスし、**Private Cloud Login** を選択します。

認証されると、Auth0 Converged Platform 上で利用可能な任意の非開発環境 (たとえば `travel0-prod-1`) のステータスを確認できます。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Private Cloud 環境のステータスにアクセスするには、対応する Public Cloud サポートテナントのメンバーである必要があります。詳細については、[Open and Manage Support Tickets](/ja/docs/troubleshoot/customer-support/open-and-manage-support-tickets) および [Manage Dashboard Access](/ja/docs/get-started/manage-dashboard-access) を参照してください。
</Callout>

Auth0 Public Cloud Status ページに戻るには、右上のナビゲーションで **Auth0 Public Cloud Status** を選択します。

<div id="subscribe-to-private-cloud-status-updates">
  ## Private Cloud のステータス更新を購読する
</div>

Environment を選択し、次に **Subscribe** を選択します。

環境に影響するステータス更新を受け取るには、「Subscribe」ボックスに表示される生成済み URL をコピーして、Auth0 の Atom または RSS フィードを購読してください。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  生成される ATOM / RSS フィード URL は Private Cloud 環境ごとに固有で、[サポートに問い合わせる](https://support.auth0.com/)ことでローテーションできるトークンが含まれています。
</Callout>
