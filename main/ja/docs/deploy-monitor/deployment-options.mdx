title: デプロイオプション
---

Auth0 は、パブリッククラウドとプライベートクラウドの両方のデプロイオプションに加え、エンタープライズ向けデプロイを支援するツールも提供しています。

Auth0 のアイデンティティプラットフォームは、次の方法でデプロイできます。

| オプション                                                                                          | 説明                                                                     |
| ---------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| **Public Cloud**                                                                               | 標準的な認証、パーソナライズ、ユーザー管理、セキュリティ機能を備えた、Auth0 のクラウド上で稼働するマルチテナント型のクラウドサービス  |
| [**Private Cloud on AWS**](/docs/ja/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | AWS 上で稼働する専用のマネージドクラウドサービスで、分離性、高パフォーマンス、開発用インスタンス、Geo-HA add-on などを提供 |
| [**Private Cloud on Azure**](/docs/ja/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Azure 上で稼働する専用のクラウドサービスで、分離性、高パフォーマンス、開発用インスタンス、Geo-HA add-on などを提供    |

**Private Cloud** パッケージは、次のような場合に利用できるマネージドサービスです。

* 組織でマルチテナント型のパブリッククラウドサービスを利用できない
* 1 秒あたりのリクエスト数 (RPS) の保証が必要である
