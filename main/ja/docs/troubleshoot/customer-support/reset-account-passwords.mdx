description: Auth0 アカウントのパスワードをリセットする方法を説明します。
title: アカウントのパスワードをリセットする
---

パスワードを変更したい場合や Auth0 アカウントのパスワードを忘れた場合は、ほとんどのケースで <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要プロダクト。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> から新しいパスワードを設定できます。パスワードリセットを行うと、Auth0 のセッションは失効します。

1. Auth0 アカウントのログイン画面で、アカウントに関連付けられているメールアドレスを入力し、**Continue** をクリックしてから、次の画面で **Forgot password?** を選択します。
2. メールアドレスを確認し、**Continue** をクリックして送信します。パスワードのリセット手順が記載されたメールが届きます。
3. リクエストが正常に受け付けられると、「パスワードをリセットする手順については、メールアドレス [your email] を確認してください。」と表示されます。

パスワードリセットには、いくつか特別なケースがあります。

* <Tooltip tip="Multi-factor authentication (MFA): SMS で送信されるコードなど、ユーザー名とパスワードに加えて認証要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/ja/docs/glossary?term=MFA">MFA</Tooltip> を有効にしていて、管理者アカウントの MFA リセットが必要な場合は、[Auth0 Support](https://support.auth0.com) にお問い合わせください。
* Auth0 へのログインにソーシャルアカウントまたはエンタープライズアカウントを使用している場合は、該当する <Tooltip tip="Identity Provider (IdP): デジタルアイデンティティを保存および管理するサービス。" cta="用語集を見る" href="/ja/docs/glossary?term=identity+provider">アイデンティティプロバイダー</Tooltip> 側でパスワードをリセットしてください。
* ユーザーのパスワードをリセットしようとしている管理者の場合は、次の 2 つの方法があります。

  * Universal Login または Authentication API を使用した対話型のパスワードリセットフローを利用すると、ユーザーにメールが送信され、リンクを開くと Auth0 のパスワードリセットページで新しいパスワードを入力できます。
  * Management API または Auth0 Dashboard を使用して、新しいパスワードを直接設定します。
