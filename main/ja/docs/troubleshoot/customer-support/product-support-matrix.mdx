Auth0 は、[利用規約](https://auth0.com/terms)および[料金プランとサポートプラン](https://auth0.com/pricing/)に基づいてサポートを提供しています。以下は、Auth0 が提供するサポートの種類です。

| Level  | Description                                                             |
| ------ | ----------------------------------------------------------------------- |
| サポート対象 | Auth0 はこれらの項目を正式にテストしており、 (該当する場合) 新機能の提供やバグ修正を行うことがあり、カスタマーサポートも提供します。 |
| 継続サポート | Auth0 はこれらの項目について、 (該当する場合) バグ修正を行うことがあり、カスタマーサポートも提供します。               |
| バグ修正のみ   | Auth0 はこれらの項目について、 (該当する場合) バグ修正を行うことがありますが、カスタマーサポートは提供しません。           |
| コミュニティ | Auth0 はこれらの項目についてカスタマーサポートを提供せず、すべての支援はコミュニティから提供されます。                  |
| パートナー  | Auth0 はこれらの項目についてカスタマーサポートを提供せず、すべての支援はそれらを保守しているパートナーから提供されます。         |

<div id="auth0-support-coverage">
  ## Auth0 Support の対応範囲
</div>

Auth0 が提供する公式サポートは、明示的に記載されている言語、プラットフォーム、バージョン、テクノロジーに限定されます。記載のないもの、またはコミュニティサポートとして記載されているものは、Auth0 Support の対象外です。サポート対象外の項目について質問がある場合は、[Auth0 の公開フォーラム](https://community.auth0.com/)に質問を投稿できます。

コミュニティサポート対象の項目については、特定のライブラリや SDK の GitHub リポジトリに issue を登録するなどして、それらの項目を担当する開発者に支援を依頼してください。

Auth0 Support は、記載されているテクノロジーの最新バージョンにのみ対応しています (特に指定がない限り) 。古いバージョンを使用している場合は、サポートを受ける前にアップグレードする必要があります。
