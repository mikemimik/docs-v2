---

パスワードリセットが成功した後にユーザーがリダイレクトされる URI は、以下の条件の組み合わせによって決まります。

1. テナントで Universal Login と クラシックログイン のどちらを使用しているか
2. Management API を使って password change ticket を直接生成しているか
3. Authentication API、ログインページ、またはその他のソースからパスワード変更メールを送信しているか

次のフローチャートは、設定に応じてテナントのリダイレクト URI を設定するためのガイドです。
