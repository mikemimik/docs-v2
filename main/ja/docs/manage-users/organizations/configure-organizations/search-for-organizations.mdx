---
description: Auth0 Dashboard で Organizations を検索する方法を説明します。
title: Organizations を検索する
---

[Auth0 Dashboard](https://manage.auth0.com/#/organizations) の Organizations セクションでは、[組織](/docs/ja/manage-users/organizations/organizations-overview) を名前または表示名で検索できます。

<Frame>![Auth0 Dashboard - Organizations - 検索バー ](/docs/images/cdy7uua7fh8z/5CeeKLjPGxjTqy682lbnyj/3a41b87a369e65ca624b1822c99dbc57/2025-01-27_17-27-28.png)</Frame>

検索バーの横にあるドロップダウンメニューでは、名前または表示名で検索できます。検索バーに 3 文字以下を入力した場合は、完全一致する結果のみが返されます。4 文字以上を入力した場合は、入力した文字列を含むあいまい一致の結果が返されます。たとえば、`Big Holdings, Co.` という名前の organization があり、検索ターム `hold` または `ings` を入力すると、Big Holdings, Co. organization が返されます。

検索結果は最大 50 件です。目的の結果が見つからない場合は、より具体的な検索タームを試してください。

<Warning>
  organization の検索には結果整合性が採用されています。最近作成または変更された organization は、検索結果にすぐに表示されない場合がありますが、最終的にはテナント内のエンティティが結果に正確に反映されます。
</Warning>
