[組織](/ja/docs/manage-users/organizations/organizations-overview)からメンバーを削除するには、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>または<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip>のいずれかを使用できます。organization からユーザーを削除しても、削除されるのはそのユーザーの該当する organization におけるメンバーシップのみであり、接続や Auth0 テナント からユーザー自体が削除されるわけではありません。削除処理にはおよそ 5 ～ 10 秒かかります。

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Auth0 Dashboard からメンバーを削除するには:

1. [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) に移動し、メンバーシップを設定する organization を選択します。
2. **Members** ビューを選択し、削除するメンバーを見つけて、**More Options** (**...**) メニューを開きます。
3. **Remove member** を選択し、確認します。

<div id="management-api">
  ## Management API
</div>

Management API 経由でメンバーを削除するには:
`Delete Organization Members` エンドポイントに `DELETE` リクエストを送信します。`ORG_ID`、`MGMT_API_ACCESS_TOKEN`、`USER_ID` のプレースホルダー値は、それぞれ organization ID、Management API の <Tooltip tip="Access Token: API へのアクセスに使用される認可資格情報で、opaque 文字列または JWT の形式を取ります。" cta="用語集を見る" href="/ja/docs/glossary?term=Access+Token">Access Token</Tooltip>、ユーザー ID に必ず置き換えてください。1 回のリクエストで最大 10 人のメンバーを削除できます。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Auth0 Domain を確認する**

  Auth0 Domain は、テナント名、リージョンのサブドメイン (テナントが US リージョンにあり、2020 年 6 月より前に作成された場合を除く) 、および `.auth0.com` を組み合わせたものです。たとえば、テナント名が `travel0` の場合、Auth0 Domain 名は `travel0.us.auth0.com` になります。 (テナントが US にあり、2020 年 6 月より前に作成されている場合、Domain 名は `https://travel0.auth0.com` になります。)

  カスタムドメインを使用している場合は、ここにそのカスタムドメイン名を指定します。
</Callout>

| 値                       | 説明                                                                                                                                             |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | メンバーシップを削除する対象の organization の ID。                                                                                                             |
| `MGMT_API_ACCESS_TOKEN` | scope `delete:organization_members` を持つ [Access Token for the Management API](/ja/docs/secure/tokens/access-tokens/management-api-access-tokens)。 |
| `USER_ID`               | 指定した organization から削除するユーザーの ID。                                                                                                              |

<div id="response-status-codes">
  ### レスポンスステータスコード
</div>

考えられるレスポンスステータスコードは次のとおりです。

| ステータスコード | エラーコード                 | メッセージ                                                                                    | 原因                                                    |
| -------- | ---------------------- | ---------------------------------------------------------------------------------------- | ----------------------------------------------------- |
| `204`    |                        | ユーザーは organization から正常に削除されました。                                                         |                                                       |
| `400`    | `invalid_body`         | 無効なリクエスト本文です。メッセージは原因によって異なります。                                                          | リクエストのペイロードが有効ではありません。                                |
| `400`    | `invalid_query_string` | 無効なリクエストのクエリ文字列です。メッセージは原因によって異なります。                                                     | クエリ文字列が有効ではありません。                                     |
| `401`    |                        | 無効なトークンです。                                                                               |                                                       |
| `401`    |                        | JSON Web トークンのバリデーションで無効な署名を受信しました。                                                      |                                                       |
| `401`    |                        | クライアントはグローバルではありません。                                                                     |                                                       |
| `403`    | `insufficient_scope`   | スコープが不足しています。次のいずれかが必要です: `delete:organization_members`。                                 | 指定されたベアラートークンのスコープでは許可されていないフィールドの読み取りまたは書き込みを試行しました。 |
| `429`    |                        | リクエストが多すぎます。X-RateLimit-Limit、X-RateLimit-Remaining、および X-RateLimit-Reset ヘッダーを確認してください。 |                                                       |
