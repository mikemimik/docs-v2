[組織](/ja/docs/manage-users/organizations/organizations-overview)をプログラムで扱う場合、組織のメンバー一覧を取得する必要があることがあります。

組織に関連付けられたメンバーは、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>または<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip>のいずれかで確認できます。

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

1. [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) に移動し、メンバーシップを確認したい 組織 を選択します。
2. **Members** ビューを選択します。

<div id="management-api">
  ## Management API
</div>

<Warning>
  Auth0 Dashboard または Management API では、最大 1000 件の organization のメンバーを表示できますが、実際にはそれ以上存在する場合があります。
</Warning>

`Get Organization Members` エンドポイントに `GET` リクエストを送信します。`ORG_ID` と `MGMT_API_ACCESS_TOKEN` のプレースホルダー値は、それぞれお使いの 組織ID と Management API の <Tooltip tip="Access Token: API へのアクセスに使用される、opaque 文字列または JWT の形式の Authorization 用認証情報。" cta="用語集を見る" href="/ja/docs/glossary?term=Access+Token">アクセストークン</Tooltip> に必ず置き換えてください。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Auth0 ドメインを確認する**

  Auth0 ドメインは、テナント名、リージョンのサブドメイン (テナントが US リージョンにあり、2020 年 6 月より前に作成された場合を除く) 、および `.auth0.com` を組み合わせたものです。たとえば、テナント名が `travel0` の場合、Auth0 ドメイン名は `travel0.us.auth0.com` になります。 (テナントが US にあり、2020 年 6 月より前に作成された場合、ドメイン名は `https://travel0.auth0.com` になります。)

  カスタムドメインを使用している場合は、そのカスタムドメイン名を指定してください。
</Callout>

| 値                       | 説明                                                                                                                                          |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | メンバーを取得する対象の組織のID。                                                                                                                |
| `MGMT_API_ACCESS_TOKEN` | scope `read:organization_members` を持つ[Access Token for the Management API](/ja/docs/secure/tokens/access-tokens/management-api-access-tokens)。 |

<div id="response-status-codes">
  ### レスポンスのステータスコード
</div>

想定されるレスポンスのステータスコードは次のとおりです。

| Status code | Error code           | Message                                                                                  | Cause                                                    |
| ----------- | -------------------- | ---------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| `200`       |                      | メンバーの取得に成功しました。                                                                          |                                                          |
| `400`       | `invalid_paging`     | 要求したページが、許可されている最大 1000 件を超えています。                                                        | API は最大 1000 件までしか返せないように制限されています。                       |
| `400`       | `invalid_body`       | 無効なリクエスト本文です。メッセージは原因によって異なります。                                                          | リクエストのペイロードが無効です。                                        |
| `401`       |                      | 無効なトークンです。                                                                               |                                                          |
| `401`       |                      | JSON Web トークンのバリデーションで無効な署名を受信しました。                                                      |                                                          |
| `401`       |                      | クライアントはグローバルではありません。                                                                     |                                                          |
| `403`       | `insufficient_scope` | スコープが不足しています。必要なもの: `read:organization_members`。                                         | 提供された Bearer トークンのスコープでは許可されていないフィールドの読み取りまたは書き込みを試みました。 |
| `429`       |                      | リクエストが多すぎます。X-RateLimit-Limit、X-RateLimit-Remaining、および X-RateLimit-Reset ヘッダーを確認してください。 |                                                          |
