<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Auth0 Dashboard で 組織 のメンバーからロールを削除するには、次の手順に従います。

1. [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) に移動し、メンバーシップを設定する 組織 を選択します。
2. **Members** ビューを選択し、ロールを削除するメンバーの名前をクリックします。
3. 削除するロールを見つけてごみ箱アイコンをクリックし、確認します。

<div id="management-api">
  ## Management API
</div>

Management API を使用して 組織 のメンバーからロールを削除するには:
`Delete Organization Member Roles` エンドポイントに `DELETE` リクエストを送信します。`ORG_ID`、`MGMT_API_ACCESS_TOKEN`、`USER_ID`、`ROLE_ID` の placeholder values は、それぞれ組織ID、Management API の <Tooltip tip="アクセストークン: API へのアクセスに使用される、opaque な文字列または JWT 形式の Authorization クレデンシャル。" cta="用語集を見る" href="/ja/docs/glossary?term=Access+Token">Access Token</Tooltip>、user ID、role ID に必ず置き換えてください。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Auth0ドメインを確認する**

  Auth0ドメインは、テナント名、リージョンのサブドメイン (テナントがUSリージョンにあり、2020年6月以前に作成されている場合を除く) 、および `.auth0.com` を組み合わせたものです。たとえば、テナント名が `travel0` の場合、Auth0ドメイン名は `travel0.us.auth0.com` になります。 (テナントがUSにあり、2020年6月以前に作成されている場合、ドメイン名は `https://travel0.auth0.com` になります。)

  カスタムドメインを使用している場合は、ここにはそのカスタムドメイン名を指定します。
</Callout>

| Value                   | Description                                                                                                                              |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | メンバーからロールを削除する対象の 組織 の ID。                                                                                                     |
| `MGMT_API_ACCESS_TOKEN` | スコープ `delete:organization_member_roles` を持つ [Management API 用のアクセストークン](/ja/docs/secure/tokens/access-tokens/management-api-access-tokens)。 |
| `USER_ID`               | 指定したロールを削除する対象ユーザーの ID。                                                                                                                  |
| `ROLE_ID`               | 指定した 組織 について、指定したユーザーから削除するロールの ID。                                                                                            |

<div id="response-status-codes">
  ### レスポンスのステータスコード
</div>

考えられるレスポンスのステータスコードは次のとおりです。

| ステータスコード | エラーコード                 | メッセージ                                                                                | 原因                                                             |
| -------- | ---------------------- | ------------------------------------------------------------------------------------ | -------------------------------------------------------------- |
| `204`    |                        | 組織 のメンバーからロールが正常に削除されました。                                                  |                                                                |
| `400`    | `invalid_body`         | 無効なリクエスト本文です。メッセージは原因によって異なります。                                                      | リクエスト payload が有効ではありません。                                      |
| `400`    | `invalid_query_string` | 無効なリクエスト query string です。メッセージは原因によって異なります。                                          | query string が有効ではありません。                                       |
| `401`    |                        | 無効な token です。                                                                        |                                                                |
| `401`    |                        | JSON Web トークンのバリデーションで無効な署名を受信しました。                                                  |                                                                |
| `401`    |                        | クライアントはグローバルではありません。                                                                 |                                                                |
| `403`    | `insufficient_scope`   | scope が不足しています。必要な scope は次のいずれかです: `delete:organization_member_roles`。              | 提供された bearer token の scopes では許可されていないフィールドの読み取りまたは書き込みを試みました。 |
| `429`    |                        | リクエストが多すぎます。X-RateLimit-Limit、X-RateLimit-Remaining、X-RateLimit-Reset ヘッダーを確認してください。 |                                                                |
