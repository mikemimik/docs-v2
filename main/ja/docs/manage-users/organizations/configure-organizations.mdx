<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Management+API">Management API</Tooltip> を使用して 組織 を設定できます。organization の設定手順について詳しくは、[最初の organization を作成する](/docs/ja/manage-users/organizations/create-first-organization) を参照してください。

<Card title="利用可否は Auth0 の plan によって異なります">
  この機能を利用できるかどうかは、Auth0 の plan またはカスタム契約によって異なります。詳しくは、[Pricing](https://auth0.com/pricing) を参照してください。
</Card>
