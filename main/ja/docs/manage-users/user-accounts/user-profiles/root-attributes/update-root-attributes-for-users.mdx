既存のユーザープロファイルのルート属性は、Auth0 の <Tooltip tip="Management API: お客様が管理タスクを実行するための製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用して更新できます。

Auth0 の [Normalized User Profile](/ja/docs/manage-users/user-accounts/user-profiles/user-profile-structure) には、更新可能なルート属性があります。更新できるルート属性は、使用している [connection](/ja/docs/authenticate/identity-providers) の種類によって異なります。使用している connection に応じた詳細については、[User Profile Root Attributes](/ja/docs/manage-users/user-accounts/user-profiles/root-attributes) を参照してください。

[Update a User endpoint](https://auth0.com/docs/api/management/v2/users/patch-users-by-id) に `PATCH` リクエストを送信します。`USER_ID`、`MGMT_API_ACCESS_TOKEN`、`GIVEN_NAME_VALUE`、`FAMILY_NAME_VALUE`、`NAME_VALUE`、`NICKNAME_VALUE`、および `PICTURE` の各プレースホルダー値は、それぞれユーザー ID、Management API <Tooltip tip="Access Token: API へのアクセスに使用される、JWT または不透明な文字列形式の Authorization 認証情報です。" cta="用語集を見る" href="/ja/docs/glossary?term=Access+Token">アクセストークン</Tooltip>、名、姓、名前、ニックネーム、画像 URL に置き換えてください。

  ```bash cURL
  curl --request PATCH \
    --url 'https://{yourDomain}/api/v2/users/USER_ID' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
    --header 'cache-control: no-cache' \
    --header 'content-type: application/json' \
    --data '{ "given_name": GIVEN_NAME_VALUE, "family_name": FAMILY_NAME_VALUE,"name": NAME_VALUE, "nickname": NICKNAME_VALUE,"picture": PICTURE_VALUE }'
  ```

  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }

</AuthCodeGroup>

| 値                       | 説明                                                                                                                                    |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| `USER_ID`               | 更新するユーザーの ID。                                                                                                                         |
| `MGMT_API_ACCESS_TOKEN` | `update:users` の [scope](/ja/docs/glossary?term=scope) を持つ [Management API 用のアクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
| `GIVEN_NAME_VALUE`      | 更新するユーザーの名。                                                                                                                           |
| `FAMILY_NAME_VALUE`     | 更新するユーザーの姓。                                                                                                                           |
| `NAME_VALUE`            | 更新するユーザーの氏名。                                                                                                                          |
| `NICKNAME_VALUE`        | 更新するユーザーのニックネーム。                                                                                                                      |
| `PICTURE_VALUE`         | 更新するユーザーの画像 URL。                                                                                                                      |

<div id="removing-attributes">
  ## 属性の削除
</div>

属性の値を `null` に設定すると、その属性はユーザーから削除されます。
