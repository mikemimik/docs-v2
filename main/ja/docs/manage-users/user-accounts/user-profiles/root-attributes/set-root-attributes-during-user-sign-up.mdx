Auth0&#39;s <Tooltip tip="Management API: 顧客が管理タスクを実行できるようにする製品。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用すると、サインアップ時にユーザーのルート属性を設定できます。これにより、ユーザー作成時にルート属性を設定するために必要な API 呼び出しの回数を最小限に抑えられます。

[Create a User エンドポイント](https://auth0.com/docs/api/management/v2#!/Users/post_users) に `POST` リクエストを送信します。`MGMT_API_ACCESS_TOKEN`、`CONNECTION_NAME`、`EMAIL_VALUE`、`PASSWORD_VALUE`、`GIVEN_NAME_VALUE`、`FAMILY_NAME_VALUE`、`NAME_VALUE`、`NICKNAME_VALUE`、および `PICTURE` のプレースホルダー値は、それぞれ Management API の <Tooltip tip="Access Token: API へのアクセスに使用される、不透明な文字列または JWT の形式の認可資格情報。" cta="用語集を表示" href="/ja/docs/glossary?term=Access+Token">Access Token</Tooltip>、初期接続名、メールアドレス、パスワード、名、姓、名前、ニックネーム、画像 URL に必ず置き換えてください。

<AuthCodeGroup>
  ```bash cURL
  curl --request POST \
    --url 'https://{yourDomain}/api/v2/users' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
    --header 'cache-control: no-cache' \
    --header 'content-type: application/json' \
    --data '{ "connection": CONNECTION_NAME, "email": EMAIL_VALUE, "password": PASSWORD_VALUE, "given_name": GIVEN_NAME_VALUE, "family_name": FAMILY_NAME_VALUE,"name": NAME_VALUE, "nickname": NICKNAME_VALUE,"picture": PICTURE_VALUE }'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/users");
  var request = new RestRequest(Method.POST);
  request.AddHeader("content-type", "application/json");
  request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
  request.AddHeader("cache-control", "no-cache");
  request.AddParameter("application/json", "{ "connection": CONNECTION_NAME, "email": EMAIL_VALUE, "password": PASSWORD_VALUE, "given_name": GIVEN_NAME_VALUE, "family_name": FAMILY_NAME_VALUE,"name": NAME_VALUE, "nickname": NICKNAME_VALUE,"picture": PICTURE_VALUE }", ParameterType.RequestBody);
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
  	"fmt"
  	"strings"
  	"net/http"
  	"io/ioutil"
  )

  func main() {

  	url := "https://{yourDomain}/api/v2/users"

  	payload := strings.NewReader("{ "connection": CONNECTION_NAME, "email": EMAIL_VALUE, "password": PASSWORD_VALUE, "given_name": GIVEN_NAME_VALUE, "family_name": FAMILY_NAME_VALUE,"name": NAME_VALUE, "nickname": NICKNAME_VALUE,"picture": PICTURE_VALUE }")

  	req, _ := http.NewRequest("POST", url, payload)

  	req.Header.Add("content-type", "application/json")
  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")
  	req.Header.Add("cache-control", "no-cache")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

| 値                       | 説明                                                                                                                                  |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | `create:users` の [スコープ](/ja/docs/glossary?term=scope) を持つ [Management API のAccess Token](https://auth0.com/docs/api/management/v2/tokens)。 |
| `CONNECTION_NAME`       | 初期ユーザー情報を受け取った接続の名前。                                                                                                                |
| `EMAIL_VALUE`           | 作成するユーザーのメールアドレス。                                                                                                                   |
| `PASSWORD_VALUE`        | 作成するユーザーのパスワード。                                                                                                                     |
| `GIVEN_NAME_VALUE`      | 作成するユーザーの名。                                                                                                                         |
| `FAMILY_NAME_VALUE`     | 作成するユーザーの姓。                                                                                                                         |
| `NAME_VALUE`            | 作成するユーザーのフルネーム。                                                                                                                     |
| `NICKNAME_VALUE`        | 作成するユーザーのニックネーム。                                                                                                                    |
| `PICTURE_VALUE`         | 作成するユーザーのプロフィール画像の URL。                                                                                                             |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  ユーザーのサインアップに Lock または [パブリック signup エンドポイント](https://auth0.com/docs/api/authentication#signup) を使用している場合は、同じ方法でルート属性を設定できます。
</Callout>
