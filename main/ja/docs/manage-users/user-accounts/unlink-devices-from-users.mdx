description: Auth0 Dashboard を使用して、ユーザーに割り当てられたデバイスのリンクを解除する方法を説明します。
title: ユーザーに割り当てられたデバイスのリンクを解除する
---

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用すると、ユーザーに割り当てられたデバイスのリンクを解除できます。これにより、選択したデバイスの <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を表示" href="/ja/docs/glossary?term=refresh+token">リフレッシュトークン</Tooltip> が取り消されます。

1. [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) に移動し、対象のユーザー名をクリックします。
2. **Devices** ビューをクリックし、リンクを解除するデバイスの横にあるごみ箱アイコンをクリックして確認します。
