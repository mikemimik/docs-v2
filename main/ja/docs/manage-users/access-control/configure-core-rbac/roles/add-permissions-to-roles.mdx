[roles](/ja/docs/manage-users/access-control/rbac) に権限を追加するには、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用できます。ロールとその権限は、API Authorization Core の機能セットで使用できます。

<div id="prerequisites">
  ## 前提条件
</div>

* ロールベースのアクセス制御 (RBAC) を正しく機能させるには、Auth0 Dashboard または Management API を使用して、API で有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。違いについては、[Authorization Core と Authorization Extension の比較](/ja/docs/manage-users/access-control/authorization-core-vs-authorization-extension)をご覧ください。
* ロールは事前定義された値から選択します。ロールの一覧が空の場合は、[ロールを作成](/ja/docs/manage-users/access-control/configure-core-rbac/roles/create-roles)する必要があります。
* Auth0 Dashboard で [API を設定](/ja/docs/get-started/auth0-overview/set-up-apis)します。
* 権限は事前定義された値から選択します。権限の一覧が空の場合は、API に[権限を追加](/ja/docs/get-started/apis/add-api-permissions)する必要があります。

<div id="dashboard">
  ## Auth0 Dashboard
</div>

1. [Auth0 Dashboard &gt; ユーザー管理 &gt; Roles](https://manage.auth0.com/#/roles) に移動し、詳細を表示するロール名をクリックします。
2. **Permissions** タブをクリックし、**Add Permissions** をクリックします。
3. 権限を割り当てる API を選択し、ロールに追加する権限を選択して、**Add Permissions** をクリックします。

<div id="management-api">
  ## Management API
</div>

[Add Role Permissions エンドポイント](https://auth0.com/docs/api/management/v2#!/Roles/post_role_permission_assignment) に `POST` リクエストを送信します。`ROLE_ID`、`MGMT_API_ACCESS_TOKEN`、`API_IDENTIFIER`、および `PERMISSION_NAME` のプレースホルダー値は、それぞれロール ID、Management API の <Tooltip tip="アクセストークン: API へのアクセスに使用される、不透明な文字列または JWT の形式の Authorization 資格情報。" cta="用語集を見る" href="/ja/docs/glossary?term=access+token">アクセストークン</Tooltip>、API 識別子 (<Tooltip tip="アクセストークン: API へのアクセスに使用される、不透明な文字列または JWT の形式の Authorization 資格情報。" cta="用語集を見る" href="/ja/docs/glossary?term=audience">audience</Tooltip>) 、および permission 名に必ず置き換えてください。

| Value                   | 説明                                                                                                   |
| ----------------------- | ---------------------------------------------------------------------------------------------------- |
| `ROLE_ID`               | 権限を追加する対象のロールの ID。                                                                                   |
| `MGMT_API_ACCESS_TOKEN` | スコープ `update:roles` を持つ [Management API のアクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
| `API_IDENTIFIER`        | 指定したロールに追加する permission に関連付けられた API の識別子です。これはオーディエンスとも呼ばれます。API ID ではありません。                        |
| `PERMISSION_NAME`       | 指定したロールに追加する permission の名前。                                                                         |
