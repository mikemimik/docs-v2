---
description: Auth0 Dashboard または Management API を使用して、ロールに割り当てられたユーザーを表示する方法について説明します。
title: ロールに割り当てられたユーザーを表示
---

[ロール](/ja/docs/manage-users/access-control/rbac)に割り当てられているユーザーは、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>または<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip>で確認できます。ロールは、API Authorization Core 機能セットで使用されます。

<div id="prerequisites">
  ## 前提条件
</div>

ロールベースのアクセス制御 (RBAC) を正しく機能させるには、Auth0 Dashboard または Management API のいずれかを使用して、API でこれを有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。違いについては、[Authorization Core と Authorization Extension の比較](/ja/docs/manage-users/access-control/authorization-core-vs-authorization-extension)をご覧ください。

<div id="dashboard">
  ## Auth0 Dashboard
</div>

1. [Auth0 Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/users) に移動し、表示するロール名をクリックします。
2. **Users** タブをクリックします。

各ユーザーについて、次の情報が表示されます。

| 列        | 説明                   |
| -------- | -------------------- |
| プロフィール画像 | ユーザープロファイルのプロフィール画像。 |
| 名前       | ユーザープロファイルの名前。       |
| メールアドレス  | ユーザープロファイルのメールアドレス。  |

<div id="management-api">
  ## Management API
</div>

ロールに割り当てられているユーザーの一覧を取得するには、[ロールのユーザーを取得](https://auth0.com/docs/api/management/v2#!/Roles/get_role_user)を参照してください。
