---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  [Authorization Core](/ja/docs/manage-users/access-control/rbac) 機能セットと [Authorization Extension](/ja/docs/customize/extensions/authorization-extension) は、まったく別の機能です。グループ、ロール、または権限を管理するには、それらを最初に作成した機能を使用する必要があります。
</Callout>

ロールベースのアクセス制御 (RBAC) を正しく機能させるには、Auth0 Dashboard または <Tooltip tip="Management API: 顧客が管理タスクを実行できるようにする製品。" cta="用語集を見る" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用して、API でこれを有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。比較については、[Authorization Core と Authorization Extension の比較](/ja/docs/manage-users/access-control/authorization-core-vs-authorization-extension) を参照してください。

ロールを管理するためのさまざまな機能が用意されており、これらは <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または Auth0 Management API から利用できます。
