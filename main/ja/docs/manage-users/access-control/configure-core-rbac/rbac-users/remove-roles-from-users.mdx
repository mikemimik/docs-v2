ユーザーに割り当てられている[ロール](/ja/docs/manage-users/access-control/rbac)は、<Tooltip tip="Auth0 Dashboard: サービスを構成するための Auth0 の主要製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Auth0 Dashboard: サービスを構成するための Auth0 の主要製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用して削除できます。割り当てられたロールは、API Authorization Core 機能セットで使用されます。

<div id="prerequisites">
  ## 前提条件
</div>

ロールベースのアクセス制御 (RBAC) を正しく機能させるには、Auth0 Dashboard または Management API を使用して、API の設定でこれを有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。違いについては、[Authorization Core と Authorization Extension の比較](/ja/docs/manage-users/access-control/authorization-core-vs-authorization-extension)をご覧ください。

<div id="dashboard">
  ## Auth0 Dashboard
</div>

1. [Auth0 Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) に移動し、表示するユーザー名をクリックします。
2. **Roles**ビューをクリックし、削除するロールの横にあるごみ箱アイコンをクリックします。

<div id="management-api">
  ## Management API
</div>

[Delete User Roles endpoint](https://auth0.com/docs/api/management/v2#!/Users/delete_user_roles) に `DELETE` リクエストを送信します。`USER_ID`、`MGMT_API_ACCESS_TOKEN`、`ROLE_ID` の各プレースホルダー値は、それぞれ実際のユーザー ID、Management API の <Tooltip tip="Access Token: API へのアクセスに使用される、オペーク文字列または JWT の形式の Authorization 資格情報。" cta="用語集を見る" href="/ja/docs/glossary?term=access+token">access token</Tooltip>、ロール ID に必ず置き換えてください。

<AuthCodeGroup>
  ```bash cURL
  curl --request DELETE \
    --url 'https://{yourDomain}/api/v2/users/USER_ID/roles' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
    --header 'cache-control: no-cache' \
    --header 'content-type: application/json' \
    --data '{ "roles": [ "ROLE_ID", "ROLE_ID" ] }'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/users/USER_ID/roles");
  var request = new RestRequest(Method.DELETE);
  request.AddHeader("content-type", "application/json");
  request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
  request.AddHeader("cache-control", "no-cache");
  request.AddParameter("application/json", "{ "roles": [ "ROLE_ID", "ROLE_ID" ] }", ParameterType.RequestBody);
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
  	"fmt"

  	req, _ := http.NewRequest("DELETE", url, payload)

  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")
  	req.Header.Add("cache-control", "no-cache")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

</AuthCodeGroup>

| 値                       | 説明                                                                                                  |
| ----------------------- | --------------------------------------------------------------------------------------------------- |
| `USER_ID`               | 更新するユーザーのID。                                                                                        |
| `MGMT_API_ACCESS_TOKEN` | スコープ `update:users` を持つ[Management API のアクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
| `ROLE_ID`               | 指定したユーザーから削除するロールのID。                                                                               |
