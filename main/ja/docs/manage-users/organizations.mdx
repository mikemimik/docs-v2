<Card title="利用可否は Auth0 のプランによって異なります">
  この機能を利用できるかどうかは、個別のログイン実装と、Auth0 のプランまたはカスタム契約の両方によって決まります。詳しくは、[Pricing](https://auth0.com/pricing) をご覧ください。
</Card>

Organizations 機能は Auth0 プラットフォームの大幅なアップデートであり、企業間取引 (B2B) のお客様がパートナー企業や顧客企業をより適切に管理し、エンドユーザーによるアプリケーションや API へのアクセス方法を柔軟にカスタマイズできるようにします。Auth0 のお客様は Organizations を使用して、次のことを行えます。

* Auth0 内で顧客企業やパートナー企業を表現し、それらのメンバーシップを管理する。
* 企業ごとにブランド化されたフェデレーションログインフローを設定する。
* 顧客企業やパートナー企業が B2B API を利用できるよう、machine-to-machine アクセスを提供する。
* Organizations API を使用して自社製品に管理機能を組み込み、それらの企業が自社の組織を管理できるようにする。
