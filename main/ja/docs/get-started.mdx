title: はじめに
description: 基本を学び、認証ソリューションの構築を始めましょう。
---

Auth0は、アプリケーションへのアクセスを管理するためのアイデンティティプラットフォームです。

identity and access management (IAM) を初めて扱う方は、まず基本を学び、ご利用の技術やニーズに最も適したソリューションを計画しましょう。IAMに慣れている方は、すぐに構築を始められます。

***

<div id="start-building">
  ## 開発を始める
</div>

すばやく利用を開始するには、アプリケーションの種類を選択して、手順に沿った Quickstart チュートリアルをご利用ください。

<Columns cols={2}>
  <Card title="Native/モバイルアプリ" href="/ja/docs/quickstart/native">
    iOS や Android のように、デバイス上でネイティブに動作するモバイルアプリまたはデスクトップアプリです。
  </Card>

  <Card title="Single Page App" href="/ja/docs/quickstart/spa">
    AngularJS + Node.js や React のように、ブラウザー上で動作する JavaScript の web app です。
  </Card>

  <Card title="Regular Web App" href="/ja/docs/quickstart/webapp">
    Express.js や ASP.NET のように、サーバー上で動作する従来型の web app です。
  </Card>

  <Card title="Backend/API" href="/ja/docs/quickstart/backend">
    Express.js API や ASP.NET API のような、Auth0 で保護された API またはサービスです。
  </Card>

  <Card title="AI Agent" href="https://auth0.com/ai/docs">
    LangChain や LlamaIndex のように、ユーザーに代わって外部 API を呼び出し、人による確認を含む制御を利用する AI agent です。
  </Card>
</Columns>
