Auth0 es una plataforma de identidad para gestionar el acceso a tus aplicaciones.

Si eres nuevo en la gestión de identidades y accesos (IAM), aprende algunos conceptos básicos y planifica la solución que mejor se adapte a tu tecnología y a tus necesidades. Si ya estás familiarizado con IAM, puedes comenzar de inmediato a crear.

***

<div id="start-building">
  ## Comienza a crear
</div>

Para empezar rápidamente, elige tu tipo de aplicación y sigue un tutorial Quickstart paso a paso.

<Columns cols={2}>
  <Card title="Aplicación nativa/móvil" href="/es/docs/quickstart/native">
    Aplicación móvil o de escritorio que se ejecuta de forma nativa en un dispositivo, como iOS o Android.
  </Card>

  <Card title="Aplicación de una sola página" href="/es/docs/quickstart/spa">
    Aplicación web de JavaScript que se ejecuta en el navegador, como AngularJS + Node.js o React.
  </Card>

  <Card title="Aplicación web tradicional" href="/es/docs/quickstart/webapp">
    Aplicación web tradicional que se ejecuta en el servidor, como Express.js o ASP.NET.
  </Card>

  <Card title="Backend/API" href="/es/docs/quickstart/backend">
    Una API o un servicio protegido por Auth0, como una API de Express.js o de ASP.NET.
  </Card>

  <Card title="Agente de IA" href="https://auth0.com/ai/docs">
    Un agente de IA para invocar APIs externas en nombre del usuario y usar controles con supervisión humana, como LangChain o LlamaIndex.
