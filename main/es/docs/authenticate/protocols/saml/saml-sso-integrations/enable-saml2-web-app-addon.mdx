---
description: Aprenda a habilitar el add-on SAML2 Web App para usar Auth0 como proveedor de identidad para el inicio de sesión único (SSO) de SAML.
title: Habilitar el add-on SAML2 Web App
---

Puede configurar un add-on SAML2 Web Application para usarlo con una aplicación.

1. Vaya a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications).
2. Identifique la aplicación con la que desea usar SAML2 Web Application y haga clic en el nombre de la aplicación para ir a su configuración.
3. Vaya a la pestaña **Addons**.
4. En el cuadro **SAML2 Web App**, haga clic en el control deslizante para habilitar el add-on.

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Settings Tab](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>
5. Vaya a la pestaña **Settings** e indique la **URL de devolución de llamada de la aplicación** que recibirá la respuesta SAML, así como los valores de **Settings**. En la pestaña **Settings**, hay un objeto de ejemplo que puede usar para determinar qué configuraciones desea habilitar.
6. Haga clic en **Debug** para comprobar si los parámetros proporcionados son válidos.
7. Haga clic en **Save** cuando termine.

<div id="provide-application-details">
  ## Proporciona los detalles de tu aplicación
</div>

Aunque los pasos concretos para proporcionar los detalles de tu aplicación a tu SAML2 Web App pueden variar, puedes obtener la información de configuración que necesitas en la pestaña **Usage** después de configurar el addon.
