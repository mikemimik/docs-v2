---
description: Aprende a desaprovisionar usuarios de tu integración SAML.
title: Desaprovisionar usuarios en integraciones SAML
---

Si necesitas quitarles a los usuarios el acceso a la aplicación, como mínimo tendrás que desaprovisionarlos en el <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=identity+provider">proveedor de identidad</Tooltip>. Según el proveedor de identidad, los pasos necesarios para desaprovisionar la cuenta de un usuario varían. Consulta con tu proveedor para obtener más instrucciones.

Una vez que la cuenta de un usuario se elimine o deshabilite en el proveedor de identidad, el usuario no podrá iniciar sesión.

También puede que quieras eliminar las cuentas de usuario de Auth0 de quienes hayan sido desaprovisionados si Auth0 es el proveedor de servicios o si tu aplicación se integra con Auth0. Independientemente de si Auth0 es el proveedor de identidad o el proveedor de servicios, puedes eliminar usuarios desde el [Dashboard](https://manage.auth0.com/#/users) o mediante la <Tooltip tip="Management API: un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.
