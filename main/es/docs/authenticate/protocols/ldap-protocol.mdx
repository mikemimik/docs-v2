title: Protocolo Ligero de Acceso a Directorios
---

El Protocolo Ligero de Acceso a Directorios (LDAP) es un protocolo de aplicación que se utiliza para acceder a servicios distribuidos de información de directorios y mantenerlos a través de una red de Protocolo de Internet (IP). La función de LDAP es permitir el acceso a un directorio existente, como Active Directory (AD). Auth0 utiliza un [conector AD/LDAP](/es/docs/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) que instalas en tu red para integrarse con AD/LDAP.
