---
title: Autenticación passwordless con Embedded Login
description: Describe cómo implementar la autenticación passwordless con Universal Login.
---

Para integrar la interfaz de usuario de login en tu aplicación, puedes usar nuestra [API de Passwordless integrada](/es/docs/authenticate/passwordless/implement-login/embedded-login/relevant-api-endpoints) o nuestros SDKs.

Según el tipo de aplicación que quieras crear, deberás implementarla de forma diferente:

* Para las aplicaciones de una sola página (SPA) (por ejemplo, Angular o React), consulta [Autenticación passwordless integrada para aplicaciones de una sola página (SPA)](/es/docs/authenticate/passwordless/implement-login/embedded-login/spa).
* Para las aplicaciones nativas (por ejemplo, iOS, Android o aplicaciones de escritorio), consulta [Login passwordless integrado en aplicaciones nativas](/es/docs/authenticate/passwordless/implement-login/embedded-login/native).
* Para las aplicaciones web clásicas (por ejemplo, NodeJS, Java, Rails o .NET), consulta [Login passwordless integrado en aplicaciones web clásicas](/es/docs/authenticate/passwordless/implement-login/embedded-login/webapps).

El Embedded Login en aplicaciones web tradicionales y SPA requiere configurar la autenticación de origen cruzado. Para obtener más información, consulta [Autenticación de origen cruzado](/es/docs/authenticate/login/cross-origin-authentication).
