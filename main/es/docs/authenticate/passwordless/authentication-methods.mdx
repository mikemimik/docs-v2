title: Métodos de autenticación sin contraseña en Auth0
sidebarTitle: Métodos de autenticación
description: Describe los métodos de autenticación compatibles con las conexiones sin contraseña de Auth0, incluidos el correo electrónico, el enlace mágico y los SMS.
---

Con las conexiones <Tooltip tip="Sin contraseña: forma de autenticación que no depende de una contraseña como primer factor." cta="Ver glosario" href="/es/docs/glossary?term=Passwordless">Sin contraseña</Tooltip>, los usuarios pueden iniciar sesión sin contraseña. En su lugar, pueden usar otros métodos de autenticación. Las conexiones Passwordless de Auth0 admiten los siguientes métodos de autenticación:

| Factor                                                                                   | Descripción                                                                                                                                                                                                                                                                                   |
| ---------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Correo electrónico](/es/docs/authenticate/passwordless/authentication-methods/email-otp)   | Se le pide al usuario que introduzca su dirección de correo electrónico. Auth0 le envía a esa dirección un código de un solo uso, que el usuario introduce en la aplicación.                                                                                                                  |
| [Enlace mágico](/es/docs/authenticate/passwordless/authentication-methods/email-magic-link) | Se le pide al usuario que introduzca su dirección de correo electrónico. Auth0 le envía a esa dirección un correo con un enlace. El usuario hace clic en el enlace y accede directamente a la aplicación.                                                                                     |
| [SMS](/es/docs/authenticate/passwordless/authentication-methods/sms-otp)                    | Se le pide al usuario que introduzca su número de teléfono. Auth0 le envía a ese número un código de un solo uso. De forma predeterminada, Auth0 usa Twilio para enviar el código, pero si tiene una puerta de enlace SMS personalizada, puede modificar la conexión para usarla en su lugar. |
