---
title: Universal Login vs. Classic Login
description: Compara las funcionalidades disponibles en las experiencias Universal Login y Classic Login
---

[Auth0 Universal Login](/es/docs/authenticate/login/auth0-universal-login) proporciona la función esencial de un <Tooltip tip="Servidor de autorización: Servidor centralizado que ayuda a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funcionalidades disponibles para un usuario." cta="Ver glosario" href="/es/docs/glossary?term=authorization+server">servidor de autorización</Tooltip>: el flujo de inicio de sesión. Cuando un usuario necesita demostrar su identidad para obtener acceso a su aplicación, puede redirigirlo a <Tooltip tip="Servidor de autorización: Servidor centralizado que ayuda a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funcionalidades disponibles para un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Universal+Login">Universal Login</Tooltip> para que Auth0 se encargue del proceso de autenticación.

Universal Login es la principal solución de inicio de sesión alojado de Auth0. [Universal Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) ofrece herramientas de personalización fáciles de usar y una experiencia más simple y rápida para los usuarios finales. Como alternativa, Auth0 también admite [Classic Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/classic-experience), una experiencia de inicio de sesión alojado que utiliza controles de JavaScript para su personalización.

<Warning>
  En este momento, el desarrollo activo de Auth0 se centra en Universal Login, y Classic Login ya no recibe actualizaciones. A menos que su caso de uso específico requiera la experiencia Classic, se recomienda implementar Universal Login.
</Warning>

Para ayudarle a elegir la mejor solución para sus necesidades, la siguiente tabla compara las funcionalidades de ambas experiencias de inicio de sesión.

| **Solución sin código**                                                             | Sí, mediante el [editor sin código](/es/docs/customize/login-pages/universal-login/customize-themes)                                                                                           | No                                                                                                                                                                               |
| **Autenticación passwordless**                                                      | Sí                                                                                                                                                                                          | Sí, con [SMS o correo electrónico](/es/docs/authenticate/login/auth0-universal-login/passwordless-login)                                                                            |
| **Enlace mágico por correo electrónico**                                            | No                                                                                                                                                                                          | [Sí](/es/docs/authenticate/passwordless/authentication-methods/email-magic-link)                                                                                                    |

| **Cumplimiento de las Pautas de Accesibilidad para el Contenido Web (WCAG)**        | Sí                                                                                                                                                                                          | No                                                                                                                                                                               |
| **Compatibilidad con Organizations**                                                | [Sí](/es/docs/manage-users/organizations)                                                                                                                                                      | No                                                                                                                                                                               |

| **Localización completa**                                                           | Sí, en [estos](/es/docs/customize/internationalization-and-localization/universal-login-internationalization) idiomas                                                                          | Solo para la página de inicio de sesión                                                                                                                                          |

| **MFA por correo electrónico**                                                      | [Sí](/es/docs/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-email-notifications-for-mfa)                                                                    | No                                                                                                                                                                               |
| **MFA por voz**                                                                     | [Sí](/es/docs/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-sms-voice-notifications-mfa)                                                                    | No                                                                                                                                                                               |
| **MFA de Duo**                                                                      | Sí, pero Duo debe ser el único factor de MFA habilitado                                                                                                                                     | Sí                                                                                                                                                                               |
