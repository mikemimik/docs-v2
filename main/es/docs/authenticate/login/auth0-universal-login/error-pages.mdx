title: Página de error predeterminada de Auth0
---

Si tu URL de devolución de llamada es válida, cuando se produce un error de autorización, el <Tooltip tip="Servidor de autorización: servidor centralizado que ayuda a definir los límites del acceso de un usuario. Por ejemplo, tu servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Authorization+Server">servidor de autorización</Tooltip> devuelve los parámetros de error y state correspondientes a tu URL de devolución de llamada. Si tu URL de devolución de llamada no es válida, tu aplicación mostrará la página de error genérica predeterminada de Auth0. También puedes personalizar las páginas de error. Para obtener más información, consulta [Custom Error Pages](/es/docs/customize/login-pages/custom-error-pages).

Para encontrar el nombre predeterminado de la página de error genérica, consulta [Use Version Control to Manage Your Universal Login Pages](/es/docs/customize/login-pages/classic-login/version-control).

<div id="display">
  ## Visualización
</div>

La página de error genérica muestra la siguiente información si ha configurado los campos correspondientes en su [Configuración del tenant](https://manage.auth0.com/#/tenant/).

| Campo                         | Descripción                                                         |
| ----------------------------- | ------------------------------------------------------------------- |
| Nombre comercial              | Nombre de su empresa                                                |
| URL del logotipo              | URL del logotipo de su empresa                                      |
| Correo electrónico de soporte | Dirección de correo electrónico del equipo de soporte de su empresa |
| URL de soporte                | URL de la página de soporte de su empresa                           |

<div id="parameters">
  ## Parámetros
</div>

El servidor de autorización devuelve parámetros añadidos a la URL como una cadena de consulta.

| Parámetro           | Descripción                                                                  |
| ------------------- | ---------------------------------------------------------------------------- |
| `client_id`         | Identificador de la aplicación de Auth0.                                     |
| `connection`        | Conexión utilizada en el momento del error.                                  |
| `lang`              | Idioma establecido para su uso en el momento del error.                      |
| `error`             | Código del error.                                                            |
| `error_description` | Descripción del error.                                                       |
| `tracking`          | Identificador que Auth0 utiliza para localizar errores en los logs internos. |

Los parámetros que se muestran varían según el tipo de error y son específicos de la solicitud. Por ejemplo, si la solicitud no contenía un `client_id` y produjo un error, la página de error no devolverá ningún <Tooltip tip="ID de cliente: Valor de identificación asignado por Auth0 a tu recurso registrado." cta="Ver glosario" href="/es/docs/glossary?term=client+ID">ID de cliente</Tooltip>.
