title: Supervisar el Conector AD/LDAP con System Center Operations Manager
---

El Conector AD/LDAP de Auth0 puede ejecutarse como un servicio en equipos con Windows.

Puede supervisar el estado del servicio con System Center, como haría con cualquier otro servicio.

1. Abra el **Add Monitoring Wizard** y seleccione **Monitoring Type: Windows Service**:

   <Frame>![Pantalla 1 del asistente para supervisar el Conector AD/LDAP](/docs/images/cdy7uua7fh8z/6234mP3Hx1g3Iwf7y9mKH9/7a2ac217f7d22070c20a769939461f75/ss-2014-12-11T22-48-51.png)</Frame>
2. Introduzca un nombre y una descripción:

   <Frame>![Pantalla 2 del asistente para supervisar el Conector AD/LDAP](/docs/images/cdy7uua7fh8z/2KZkQoBuubAgAEWBPnkJx0/abdcdf5e1e0fad20829b255401db523f/ss-2014-12-11T22-49-57.png)</Frame>
3. Seleccione el servidor en el que está instalado el Conector AD/LDAP y, a continuación, elija **Auth0 ADLDAP**:

   <Frame>![Pantalla 3 del asistente para supervisar el Conector AD/LDAP](/docs/images/cdy7uua7fh8z/DOQHiJYYiJKZrWzHYUIiB/21eae1ad8435a0754c6e221d5e33bab5/ss-2014-12-11T22-50-37.png)</Frame>
4. Seleccione los límites de **CPU** y **Memory limits**. Por ejemplo, un 10 % de CPU y 200 MB de RAM son buenos límites para activar alertas.

También le recomendamos configurar una transacción sintética para supervisar la autenticación de extremo a extremo.

Para obtener más información, consulte [Supervisar Auth0 con System Center Operations Manager](/es/docs/deploy-monitor/monitor/monitor-using-scom).
