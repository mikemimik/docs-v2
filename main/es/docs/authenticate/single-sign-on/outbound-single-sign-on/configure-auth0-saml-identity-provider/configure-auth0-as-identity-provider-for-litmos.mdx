Utilice la siguiente configuración de <Tooltip tip="Security Assertion Markup Language (SAML): protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin contraseña." cta="Ver glosario" href="/es/docs/glossary?term=SAML">SAML</Tooltip> para Litmos.

1. Vaya a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y cree una nueva aplicación o haga clic en el nombre de una aplicación para actualizarla.

2. Vaya a la pestaña **Addons** y active el interruptor **SAML2 Web App**.

3. En la pestaña **Settings**, establezca la **URL de devolución de llamada de la aplicación** en: `https://{yourDomain}.litmos.com/integration/samllogin`.

   <Frame>![Dashboard Applications Applications pestaña Addons SAML2 Web App pestaña Settings](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>

4. Pegue el siguiente código en el cuadro de texto **Settings** y haga clic en **Debug**.

   <AuthCodeBlock children={codeExample} language="json" />

5. Desplácese hasta la parte inferior de la página y haga clic en **Enable**.

6. En la pestaña **Usage**, localice **Identity Provider Metadata** y haga clic en **Download** para descargar el archivo de metadatos. Lo necesitará cuando configure Auth0 como proveedor de identidad.

   <Frame>![Dashboard Applications Applications pestaña Addons SAML2 Web App pestaña Usage](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>
