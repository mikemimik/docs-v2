---
description: Aprende a configurar Auth0 como proveedor de identidad con el addon SAML2 Web App para Workpath.
title: Configurar Workpath como proveedor de servicios SAML
---

Usa la siguiente configuración de <Tooltip tip="Security Assertion Markup Language (SAML): Protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin una contraseña." cta="Ver glosario" href="/es/docs/glossary?term=SAML">SAML</Tooltip> para Workpath.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y crea una aplicación nueva o haz clic en el nombre de una aplicación para actualizarla.

2. Ve a la pestaña **Addons** y activa el interruptor de **SAML2 Web App**.

3. En la pestaña **Settings**, establece la **URL de devolución de llamada de la aplicación** en: `https://api.workpath.com/v1/saml/assert/{yourWorkpathSubdomain}`.

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Settings Tab](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>

4. Pega el siguiente código en el cuadro de texto **Settings** y haz clic en **Debug**.

   ```json lines
   {
         "audience": "https://api.workpath.com/v1/saml/metadata/{yourWorkpathSubdomain}",
         "mappings": {
       "email": "email",
       "given_name": "first_name",
       "family_name": "last_name"
         }
       }
   ```

5. Desplázate hasta el final de la página y haz clic en **Enable**.

6. En la pestaña **Usage**, busca **Identity Provider Metadata** y haz clic en **Download** para descargar el archivo de metadatos. Lo necesitarás cuando configures Auth0 como proveedor de identidad.

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Usage Tab](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>
