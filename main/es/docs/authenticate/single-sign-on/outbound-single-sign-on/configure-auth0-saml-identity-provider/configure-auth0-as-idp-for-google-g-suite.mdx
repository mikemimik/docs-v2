---
description: Aprende a configurar Auth0 como proveedor de identidad con SAML2 Web App para Google Workspace.
title: Configurar Google Workspace como proveedor de servicios SAML
---

Usa la siguiente configuración de <Tooltip tip="Security Assertion Markup Language (SAML): Protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin contraseña." cta="Ver glosario" href="/es/docs/glossary?term=SAML">SAML</Tooltip> para Google Workspace.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y crea una aplicación nueva o haz clic en el nombre de una aplicación para actualizarla.

2. Ve a la pestaña **Addons** y habilita el interruptor de **SAML2 Web App**.

3. En la pestaña **Settings**, establece la **URL de devolución de llamada de la aplicación** en: `https://www.google.com/a/{YOUR-GOOGLE-DOMAIN}/acs`.

   <Frame>![Dashboard Applications Applications, pestaña Addons, pestaña Settings de SAML2 Web App](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>

4. Pega el siguiente código en el cuadro de texto de **Settings** y haz clic en **Debug**.

   ```json lines
   {
         "audience": "https://www.google.com/a/{yourGoogleDomain}/acs",
         "mappings": {
       "nickname": "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/name",
         },
         "createUpnClaim": false,
         "passthroughClaimsWithNoMapping": false,
         "mapUnknownClaimsAsIs": false,
         "mapIdentities": false,
         "signatureAlgorithm": "rsa-sha256",
         "digestAlgorithm": "sha256",
         "nameIdentifierFormat": "urn:oasis:names:tc:SAML:2.0:nameid-format:email",
         "nameIdentifierProbes": [
       "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/emailaddress"
         ],
       }
   ```

5. Desplázate hasta la parte inferior de la página y haz clic en **Enable**.

6. En la pestaña **Usage**, busca **Identity Provider Metadata** y haz clic en **Download** para descargar el archivo de metadata. Lo necesitarás cuando configures Auth0 como proveedor de identidad (IdP).

   <Frame>![Dashboard Applications Applications, pestaña Addons, pestaña Usage de SAML2 Web App](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>
