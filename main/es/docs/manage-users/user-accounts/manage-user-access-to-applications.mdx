title: Gestionar el acceso de los usuarios a las aplicaciones
---

Todos los usuarios asociados con un solo tenant de Auth0 se comparten entre las aplicaciones del tenant (y, por lo tanto, tienen acceso a ellas).

* Para mantener a los usuarios separados y restringir su acceso, te recomendamos que [crees un tenant adicional](/docs/es/get-started/auth0-overview/create-tenants/create-multiple-tenants) con las aplicaciones a las que quieres que tengan acceso distintos grupos de usuarios.

* Para restringir el acceso de los usuarios a determinadas aplicaciones, puedes usar [reglas](/docs/es/customize/rules).

  * Dentro de una regla, las variables `context.clientName` y `context.clientID` están disponibles para comprobar qué aplicación está usando el usuario para iniciar sesión.
  * Consulta [esta regla como ejemplo](https://github.com/auth0/rules/blob/aeaf93bc058408e260192d0941a688963449d6be/src/rules/simple-user-whitelist-for-app.js).

* Para restringir el acceso de los usuarios a las aplicaciones, configura una nueva conexión y da acceso solo a una aplicación específica.

  * Ve a [Dashboard &gt; Authentication &gt; Social](https://manage.auth0.com/#/connections/social) y haz clic en una conexión.
  * En la pestaña **Applications**, habilita o deshabilita las aplicaciones.

* Para deshabilitar el acceso de los usuarios a tus aplicaciones, puedes [bloquear y desbloquear usuarios](/docs/es/manage-users/user-accounts/block-and-unblock-users) en el Dashboard.
