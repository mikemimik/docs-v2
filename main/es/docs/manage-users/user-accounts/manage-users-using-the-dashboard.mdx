title: Administrar usuarios con el Dashboard
---

Puedes usar el Dashboard para administrar tus usuarios. Para empezar, ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users). Aquí es donde puedes crear, ver, modificar o eliminar usuarios.

<Card title="La disponibilidad varía según el plan de Auth0">
  Tanto tu implementación específica de login como tu plan de Auth0 o acuerdo personalizado afectan la disponibilidad de esta función. Para obtener más información, consulta [Pricing](https://auth0.com/pricing).
</Card>

De forma predeterminada, los atributos del perfil de usuario proporcionados por <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=identity+providers">proveedores de identidad</Tooltip> distintos de Auth0 (como Google, Facebook o X) no se pueden editar directamente, porque se actualizan desde el proveedor de identidad cada vez que el usuario inicia sesión. Para obtener más información sobre los atributos de los proveedores de identidad, consulta [Configurar la conexión del proveedor de identidad para las actualizaciones del perfil de usuario](/es/docs/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0).

Para poder editar los atributos raíz `name`, `nickname`, `given_name`, `family_name` o `picture` en el perfil de usuario normalizado, debes [configurar la sincronización de tu conexión con Auth0](/es/docs/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0) para que los atributos del usuario se actualicen desde el proveedor de identidad solo cuando se cree el perfil de usuario. Estos atributos raíz estarán disponibles para editarse individualmente o mediante importaciones masivas.

Para aprender a administrar el acceso al <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> de los miembros de tu equipo, consulta [Administrar el acceso al Dashboard](/es/docs/get-started/manage-dashboard-access).
