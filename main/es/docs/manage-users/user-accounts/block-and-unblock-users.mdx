---
title: Cómo bloquear y desbloquear usuarios como administrador del tenant
sidebarTitle: Bloquear y desbloquear usuarios
description: Controla si un usuario puede iniciar sesión en tu aplicación agregando o quitando bloqueos del administrador del tenant.
---

Los administradores del tenant pueden bloquear usuarios para impedir que accedan a las aplicaciones (y desbloquearlos para permitirles volver a acceder).

Si un usuario bloqueado intenta iniciar sesión en una aplicación, se lo redirige a la URL de devolución de llamada con los siguientes parámetros de error:

```
error=unauthorized&error_description=user%20is%20blocked
```

Este evento también queda registrado en tus [registros del tenant](/docs/es/deploy-monitor/logs/log-event-type-codes) como un evento de **inicio de sesión fallido** (`f`) con la descripción `user is blocked`.

<div id="add-tenant-administrator-blocks">
  ## Agregar bloqueos del administrador del tenant
</div>

Para agregar un bloqueo del administrador del tenant con el Auth0 Dashboard:

1. Ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users).
2. Selecciona el nombre del usuario que quieres bloquear.
3. En la pestaña **Details**, junto a **Block user**, selecciona **Block**.

Esta opción también está disponible en el menú **...** a la derecha del nombre del usuario.

Para agregar un bloqueo del administrador del tenant con la Management API, usa el [endpoint Update a User](/docs/es/api/management/v2/users/patch-users-by-id) (`PATCH /users/{id}`) para establecer el [atributo `blocked` del usuario](/docs/es/api/management/v2/users/patch-users-by-id#body-blocked) en `true`.

<div id="remove-tenant-administrator-blocks">
  ## Eliminar los bloqueos del administrador del tenant
</div>

Cuando un administrador del tenant bloquea a un usuario, el bloqueo no caduca automáticamente. Un administrador del tenant debe desbloquear al usuario para restablecer su acceso a las aplicaciones.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Los bloqueos por [protección contra ataques de fuerza bruta](/docs/es/secure/attack-protection/brute-force-protection) se gestionan por separado de los bloqueos del administrador del tenant. Obtén más información sobre [cómo eliminar los bloqueos por protección contra ataques de fuerza bruta](/docs/es/secure/attack-protection/brute-force-protection#removing-brute-force-protection-blocks).
</Callout>

Para eliminar un bloqueo del administrador del tenant con el Auth0 Dashboard:

1. Ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users).
2. Selecciona el nombre del usuario que tiene el bloqueo del administrador del tenant.
3. En la pestaña **Details**, junto a **Block user**, selecciona **Unblock**.

Esta opción también está disponible en el menú **...** a la derecha del nombre del usuario.

Para eliminar un bloqueo del administrador del tenant con la Management API, usa el [endpoint Update a User](/docs/es/api/management/v2/users/patch-users-by-id) (`PATCH /users/{id}`) para establecer el [atributo `blocked` del usuario](/docs/es/api/management/v2/users/patch-users-by-id#body-blocked) en `false`.

Si usas la Management API para restablecer la contraseña de un usuario, también se eliminan los bloqueos del administrador del tenant.
