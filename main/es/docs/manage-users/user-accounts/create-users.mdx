---
description: Describe cómo crear usuarios en el Auth0 Dashboard.
title: Crear usuarios
---

Puedes crear un usuario desde el Auth0 Dashboard o mediante la [Auth0 Management API](https://auth0.com/docs/api/management/v2/users/post-users).

1. Ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) y selecciona **Create User**.
2. Selecciona la **conexión** y, a continuación, introduce el **Email** y la **Password** de tu usuario.

   | Atributo | Descripción                                                                                                                                                                                                                                                                                                                     |
   | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Conexión | La conexión de base de datos que se utilizará para autenticar al usuario. La lista desplegable muestra todas las conexiones de base de datos configuradas en tu tenant. La conexión que uses debe estar asociada a una aplicación; de lo contrario, recibirás un mensaje de error indicando que la conexión está deshabilitada. |
   | Email    | La dirección de correo electrónico del usuario. El formato de la dirección de correo electrónico debe ajustarse a la especificación proporcionada en [RFC 5321 on IETF](https://datatracker.ietf.org/doc/html/rfc5321#section-4.1.2)                                                                                            |
   | Password | La contraseña del usuario. No hay un límite máximo para la longitud de la contraseña.                                                                                                                                                                                                                                           |
3. Selecciona **Create**.
   El usuario se crea y se te redirigirá al perfil del usuario recién creado.

Si habilitas [múltiples dominios personalizados](/es/docs/customize/custom-domains/multiple-custom-domains), tienes la opción de seleccionar **Domain** en la sección **Conexión** para elegir el dominio de notificación de tu tenant.

Si habilitas [Flexible Identifiers](/es/docs/authenticate/database-connections/flexible-identifiers-and-attributes), además de **email**, puedes seleccionar el **username** o el **phone** de tu usuario, o una combinación de dos o más.
