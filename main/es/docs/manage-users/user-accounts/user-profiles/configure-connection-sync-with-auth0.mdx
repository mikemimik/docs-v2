Puedes actualizar las preferencias de la conexión de un <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=identity+provider">proveedor de identidad ascendente</Tooltip> externo para controlar cuándo se permiten las actualizaciones de los atributos raíz del perfil de usuario mediante el <Tooltip tip="Auth0 Dashboard: producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>.

De forma predeterminada, los atributos del perfil de usuario proporcionados por proveedores de identidad distintos de Auth0 (como Google, Facebook o X) no se pueden editar directamente porque se actualizan desde el proveedor de identidad cada vez que el usuario inicia sesión.

Para poder editar los atributos raíz name, nickname, given&#95;name, family&#95;name o picture del perfil de usuario normalizado, debes configurar la sincronización de tu conexión con Auth0 para que los atributos del usuario se actualicen desde el proveedor de identidad solo al crear el perfil de usuario.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Sincronizar usuarios al iniciar sesión** no se aplica a las conexiones passwordless y no se puede habilitar desde la interfaz de usuario del tenant ni mediante la Management API.
</Callout>

Puedes editar los [atributos raíz individualmente](/es/docs/manage-users/user-accounts/user-profiles/root-attributes/update-root-attributes-for-users) o mediante una [importación masiva](/es/docs/manage-users/user-accounts/user-profiles/root-attributes/set-root-attributes-during-user-import) usando la Management API.

<div id="use-the-dashboard">
  ## Usa el Dashboard
</div>

1. Ve a [Auth0 Dashboard &gt; Autenticación](https://manage.auth0.com/#/connections) y selecciona el tipo de conexión: Database, Social, empresarial o Passwordless.
2. Selecciona el nombre de una conexión para ver su configuración.
3. Busca la sección **Avanzado**, ajusta el interruptor **Sincronizar los atributos del perfil de usuario en cada inicio de sesión** según la configuración deseada y selecciona **Guardar**.

<div id="use-the-management-api">
  ## Usa la Management API
</div>

Antes de completar este paso, primero debes [recuperar los valores existentes del objeto `options` de la conexión](/es/docs/authenticate/identity-providers/retrieve-connection-options) para evitar sobrescribir los valores actuales. Si no lo haces, los parámetros que falten en el objeto original se perderán después de la actualización.

Haz una llamada `PATCH` al [endpoint para actualizar una conexión](https://auth0.com/docs/api/management/v2#!/Connections/patch_connections_by_id). Asegúrate de incluir los valores originales de las opciones en la llamada para evitar sobrescribir los valores actuales. Además, asegúrate de reemplazar los valores de marcador de posición `CONNECTION_ID`, `MGMT_API_ACCESS_TOKEN` y `ATTRIBUTE_UPDATE_VALUE` por tu ID de conexión, el <Tooltip tip="Token de acceso: Credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=access+token">token de acceso</Tooltip> de la Management API y el valor de actualización del atributo, respectivamente.

| Valor                    | Descripción                                                                                                                                                                                                                   |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `CONNECTION_ID`          | ID de la conexión para la que deseas permitir actualizaciones de los atributos raíz.                                                                                                                                          |
| `MGMT_API_ACCESS_TOKEN`  | [token de acceso de la Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `update:connections`.                                                                                                     |
| `ATTRIBUTE_UPDATE_VALUE` | Indica cuándo deseas permitir actualizaciones de los atributos raíz del perfil de usuario. Los valores válidos son `on_first_login` y `on_each_login`. El valor predeterminado para las conexiones nuevas es `on_each_login`. |
