Puede actualizar los atributos raíz de un perfil de usuario existente mediante la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> de Auth0.

El [Perfil de usuario normalizado](/docs/es/manage-users/user-accounts/user-profiles/user-profile-structure) de Auth0 incluye atributos raíz que puede actualizar. Los atributos raíz concretos que puede actualizar dependen del tipo de [conexión](/docs/es/authenticate/identity-providers) que esté utilizando. Para obtener información específica sobre la conexión que está utilizando, consulte [Atributos raíz del perfil de usuario](/docs/es/manage-users/user-accounts/user-profiles/root-attributes).

Haga una solicitud `PATCH` al [endpoint Update a User](https://auth0.com/docs/api/management/v2/users/patch-users-by-id). Asegúrese de reemplazar los valores de marcador de posición `USER_ID`, `MGMT_API_ACCESS_TOKEN`, `GIVEN_NAME_VALUE`, `FAMILY_NAME_VALUE`, `NAME_VALUE`, `NICKNAME_VALUE` y `PICTURE` por su ID de usuario, el <Tooltip tip="Token de acceso: Credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">Token de acceso</Tooltip> de la Management API, el nombre, el apellido, el nombre completo, el apodo y la URL de la imagen, respectivamente.

| Valor                   | Descripción                                                                                                                                       |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| `USER_ID`               | El ID del usuario que se actualizará.                                                                                                             |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso de la Management API](https://auth0.com/docs/api/management/v2/tokens) con el [scope](/docs/es/glossary?term=scope) `update:users`. |
| `GIVEN_NAME_VALUE`      | Nombre de pila del usuario que se actualizará.                                                                                                    |
| `FAMILY_NAME_VALUE`     | Apellido del usuario que se actualizará.                                                                                                          |
| `NAME_VALUE`            | Nombre completo del usuario que se actualizará.                                                                                                   |
| `NICKNAME_VALUE`        | Apodo del usuario que se actualizará.                                                                                                             |
| `PICTURE_VALUE`         | URL de la imagen del usuario que se actualizará.                                                                                                  |

<div id="removing-attributes">
  ## Eliminar atributos
</div>

Asignar cualquier valor a `null` eliminará el atributo del usuario.
