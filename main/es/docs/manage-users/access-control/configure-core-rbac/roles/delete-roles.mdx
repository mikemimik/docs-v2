Puede eliminar un [rol](/docs/es/manage-users/access-control/rbac) usando el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Los roles se usan con el conjunto de funciones de API Authorization Core.

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debe habilitarlo para su API mediante el Auth0 Dashboard o la Management API. La funcionalidad de Authorization Core es diferente de la de Authorization Extension. Para ver una comparación, lea [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Ve a [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles) y haz clic en el nombre del rol para verlo.
2. Haz clic en **Eliminar este rol** y confirma la acción.

<div id="management-api">
  ## Management API
</div>

Haz una solicitud `DELETE` al [endpoint Delete Role](https://auth0.com/docs/api/management/v2#!/Roles/delete_roles_by_id). Asegúrate de reemplazar los valores de marcador de posición `ROLE_ID` y `MGMT_API_ACCESS_TOKEN` por el ID de tu rol y el <Tooltip tip="Token de acceso: credencial de autorización, en forma de cadena opaca o JWT, usada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> de la Management API, respectivamente.

  ```bash cURL
  curl --request DELETE \
    --url 'https://{yourDomain}/api/v2/roles/ROLE_ID' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/roles/ROLE_ID");

  IRestResponse response = client.Execute(request);
  ```

| Valor                   | Descripción                                                                                                            |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `ROLE_ID`               | El ID del rol que quieres eliminar.                                                                                    |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `delete:roles`. |
