Auth0 permite importar usuarios desde aplicaciones externas mediante conexiones de base de datos personalizadas, la Auth0 <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> o la User Import/Export Extension.

<div id="automatic-migrations">
  ## Migraciones automáticas
</div>

Auth0 admite la migración automática de usuarios desde una [conexión de base de datos personalizada](/docs/es/authenticate/database-connections/custom-db) a Auth0. Esta práctica a veces se conoce como **migración gradual** o **migración diferida**. Al activar esta funcionalidad, sus usuarios:

* Se transfieren a Auth0 la primera vez que inician sesión después de configurar la integración.
* No tienen que restablecer su contraseña como resultado de la migración.

<Card title="La disponibilidad varía según el plan de Auth0">
  Tanto su implementación específica de login como su plan de Auth0 o acuerdo personalizado determinan si esta funcionalidad está disponible. Para obtener más información, consulte [Pricing](https://auth0.com/pricing).
</Card>

Cuando un usuario se autentica mediante una conexión de base de datos personalizada marcada para importarse a Auth0, tiene lugar el siguiente proceso:

* Auth0 autentica a los usuarios migrados con la base de datos de Auth0.
* Si el usuario no ha sido migrado, Auth0 ejecuta su script de Login personalizado y, si el inicio de sesión se realiza correctamente, agrega al usuario a la base de datos de Auth0.
* En los inicios de sesión posteriores, las credenciales del usuario se recuperan de Auth0, **NO** de su base de datos personalizada.
* Los usuarios nuevos se agregan automáticamente a la base de datos de Auth0.

<div id="bulk-user-imports-with-the-management-api">
  ## Importación masiva de usuarios con la Management API
</div>

Si ya tiene una base de datos de usuarios, puede usar el endpoint de la Management API [`/post_users_imports`](https://auth0.com/docs/api/management/v2#!/Jobs/post_users_imports) para rellenar una conexión de base de datos con esta información.

<div id="migrate-users-with-the-user-importexport-feature">
  ## Migra usuarios con la función User Import/Export
</div>

La función User Import/Export te permite:

* Importar masivamente en Auth0 los usuarios existentes de tu base de datos.
* Buscar y exportar algunos (o todos) los usuarios de base de datos de Auth0.

Para usar la importación masiva, debes ser Dashboard Editor - Users o un Admin. Para usar la exportación masiva, debes ser Dashboard Viewer - Users, Editor - Users o un Admin.

Puedes importar y exportar datos de usuarios con la [función User Import/Export](/docs/es/manage-users/user-migration/bulk-user-import-export) disponible en el Dashboard.
