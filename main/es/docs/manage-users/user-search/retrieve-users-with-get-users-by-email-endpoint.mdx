El [`GET /api/v2/users-by-email` endpoint](https://auth0.com/docs/api/management/v2#!/Users_By_Email/get_users_by_email) te permite buscar usuarios mediante sus direcciones de correo electrónico. La búsqueda requiere una coincidencia exacta con la dirección de correo electrónico proporcionada y distingue entre mayúsculas y minúsculas.

Este endpoint tiene **consistencia inmediata**, por lo que recomendamos usarlo para:

* Búsquedas de usuarios realizadas durante el proceso de autenticación.
* Búsquedas de usuarios realizadas como parte del proceso de vinculación de cuentas.

La solicitud debe incluir un [token de acceso de la Management API](/es/docs/secure/tokens/access-tokens/management-api-access-tokens).

  package main

  import (

  func main() {

  	body, _ := ioutil.ReadAll(res.Body)
