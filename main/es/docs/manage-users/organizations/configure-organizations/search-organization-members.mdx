---
description: Cómo usar User Search para encontrar miembros de una organización
title: Buscar miembros de una organización
---

Además de [listar los miembros de una organización](/es/docs/manage-users/organizations/configure-organizations/retrieve-members), puedes usar [User Search](/es/docs/manage-users/user-search) para buscar usuarios filtrados por pertenencia a una organización mediante el parámetro `organization_id` en una búsqueda.

Por ejemplo, a continuación se muestra un filtro para usuarios que tienen un dominio de correo electrónico `acme.com`, pero no son miembros de la organización Acme:

`email.domain:"acme.com" AND NOT organization_id:"ACME_ORG_ID"`

La siguiente consulta encuentra todos los usuarios que pertenecen a la organización Acme e inician sesión con una conexión de base de datos específica.

`organization_id:"ACME_ORG_ID" AND identities.connection:"my-db-connection-name"`

El ejemplo siguiente encuentra usuarios cuyas direcciones de correo electrónico empiezan por jane y son miembros de la organización Acme.

`organization_id:"ACME_ORG_ID" AND email:jane*`

Consulta [User Search](/es/docs/manage-users/user-search) para obtener más información.
