---
title: Eliminar dominios de la organización
description: Elimine la asociación entre una organización y un dominio mediante el Auth0 Dashboard o la Management API.
---

Cuando elimina un dominio de la organización, la asociación entre el dominio y la organización se elimina de forma permanente. Esta operación no se puede deshacer.

<div id="delete-an-organization-domain-with-the-auth0-dashboard">
  ## Eliminar un dominio de la organización con el Auth0 Dashboard
</div>

Para eliminar un dominio de la organización en el Auth0 Dashboard:

1. Ve a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) y elige tu organización.
2. Selecciona la pestaña **Domains**.
3. Junto al dominio que quieres quitar, abre el menú de acciones (**...**) y selecciona **Delete**.
4. En la ventana que se abre, confirma la eliminación.

<div id="delete-an-organization-domain-with-the-management-api">
  ## Eliminar un dominio de la organización con la Management API
</div>

Para eliminar un dominio de la organización con la Management API, llama al [endpoint para eliminar un dominio de descubrimiento de una organización](/docs/es/api/management/v2/organizations/delete-discovery-domains-by-discovery-domain-id) (`DELETE /organizations/{id}/discovery-domains/{discovery_domain_id}`).
