Al eliminar una [organización](/docs/es/manage-users/organizations/organizations-overview), se eliminará la asociación de membresía de todos los usuarios con la organización eliminada, aunque los usuarios en sí no se eliminarán de tu tenant.

<Warning>
  Una vez confirmada, esta operación no se puede deshacer.
</Warning>

Puedes eliminar organizaciones mediante el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Para eliminar una organización desde el Auth0 Dashboard:

1. Ve a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) y selecciona la organización que quieres eliminar.
2. Desplázate hasta la sección **Danger Zone**, selecciona **Delete** y confirma.

<div id="management-api">
  ## Management API
</div>

Haz una llamada `DELETE` al endpoint `Delete Organization`. Asegúrate de reemplazar los valores de marcador de posición `ORG_ID` y `MGMT_API_ACCESS_TOKEN` por el ID de tu organización y tu <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">token de acceso</Tooltip> para la Management API, respectivamente.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Encuentra tu dominio de Auth0**

  Si tu dominio de Auth0 es el nombre de tu tenant, tu subdominio regional (a menos que tu tenant esté en la región de EE. UU. y se haya creado antes de junio de 2020), más `.auth0.com`. Por ejemplo, si el nombre de tu tenant fuera `travel0`, el nombre de tu dominio de Auth0 sería `travel0.us.auth0.com`. (Si tu tenant estuviera en EE. UU. y se hubiera creado antes de junio de 2020, entonces el nombre de tu dominio sería `https://travel0.auth0.com`).

  Si usas dominios personalizados, este debe ser el nombre de tu dominio personalizado.
</Callout>

| Valor                   | Descripción                                                                                                                                   |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | ID de la organización que quieres eliminar.                                                                                                   |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens) con el scope `delete:organizations`. |

<div id="response-status-codes">
  #### Códigos de estado de la respuesta
</div>

Los posibles códigos de estado de la respuesta son los siguientes:

| Código de estado | Código de error      | Mensaje                                                                                                    | Causa                                                                                                    |
| ---------------- | -------------------- | ---------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| `204`            |                      | La organización se eliminó.                                                                                |                                                                                                          |
| `400`            | `invalid_uri`        | URI de la solicitud no válida. El mensaje variará según la causa.                                          | La ruta no es válida.                                                                                    |
| `401`            |                      | Token no válido.                                                                                           |                                                                                                          |
| `401`            |                      | Se recibió una firma no válida para la validación del JSON Web Token.                                      |                                                                                                          |
| `403`            | `insufficient_scope` | scope insuficiente; se esperaba alguno de los siguientes: `delete:organizations`.                          | Se intentó leer o escribir un campo que no está permitido con los scopes del bearer token proporcionado. |
| `404`            |                      | La organización no existe.                                                                                 |                                                                                                          |
| `429`            |                      | Demasiadas solicitudes. Consulta los headers X-RateLimit-Limit, X-RateLimit-Remaining y X-RateLimit-Reset. |                                                                                                          |
