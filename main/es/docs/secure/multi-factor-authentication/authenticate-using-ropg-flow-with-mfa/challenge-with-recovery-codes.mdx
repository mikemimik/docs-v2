Cuando los [códigos de recuperación están habilitados](/docs/es/secure/multi-factor-authentication/configure-recovery-codes-for-mfa) para tu tenant, Auth0 los genera automáticamente cuando un usuario se registra en <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuarios que utiliza un factor adicional al nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> (MFA). El usuario debe guardar el código de recuperación. Más adelante, este código puede usarse si el usuario pierde acceso al dispositivo o la cuenta que usa para MFA.

Cuando los códigos de recuperación están deshabilitados, la API de MFA no devolverá un código de recuperación cuando asocies el primer factor de MFA de un usuario, y los usuarios no podrán autenticarse con un código de recuperación.

Puedes permitir que los usuarios se autentiquen con un código de recuperación mediante la API de MFA.

1. Solicita al usuario su código de recuperación. Ese valor debe ingresarse en la aplicación para que el usuario se autentique.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Auth0 no genera códigos de recuperación para DUO ni para el factor heredado `google-authenticator`.
   </Callout>
2. Autentica al usuario con un código de recuperación. Llama al endpoint OAuth Token con el código de recuperación para autenticar al usuario y generar un nuevo código de recuperación. Debes especificar los siguientes parámetros:

   | Parámetro       | Valor                                                   |
   | --------------- | ------------------------------------------------------- |
   | `grant_type`    | `http://auth0.com/oauth/grant-type/mfa-recovery-code`   |
   | `recovery_code` | El código de recuperación proporcionado por el usuario. |

<AuthCodeGroup>
  ```bash cURL
  curl --request POST \
  --url 'https://{yourDomain}/oauth/token' \
  --header 'content-type: application/x-www-form-urlencoded' \
  --data grant_type=http://auth0.com/oauth/grant-type/mfa-recovery-code \
  --data 'client_id={yourClientId}' \
  --data 'client_secret={yourClientSecret}' \
  --data 'mfa_token={mfaToken}' \
  --data 'recovery_code={recoveryCode}'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/oauth/token");
  var request = new RestRequest(Method.POST);
  request.AddHeader("content-type", "application/x-www-form-urlencoded");
  request.AddParameter("application/x-www-form-urlencoded", "grant_type=http%3A%2F%2Fauth0.com%2Foauth%2Fgrant-type%2Fmfa-recovery-code&client_id={yourClientId}&client_secret=%7ByourClientSecret%7D&mfa_token=%7BmfaToken%7D&recovery_code=%7BrecoveryCode%7D", ParameterType.RequestBody);
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
     "fmt"
     "strings"
     "net/http"
     "io/ioutil"
  )

  func main() {

     url := "https://{yourDomain}/oauth/token"

     payload := strings.NewReader("grant_type=http%3A%2F%2Fauth0.com%2Foauth%2Fgrant-type%2Fmfa-recovery-code&client_id={yourClientId}&client_secret=%7ByourClientSecret%7D&mfa_token=%7BmfaToken%7D&recovery_code=%7BrecoveryCode%7D")

     req, _ := http.NewRequest("POST", url, payload)

     req.Header.Add("content-type", "application/x-www-form-urlencoded")

     res, _ := http.DefaultClient.Do(req)

     defer res.Body.Close()
     body, _ := ioutil.ReadAll(res.Body)

     fmt.Println(res)
     fmt.Println(string(body))

  }
  ```

  ```python Python
  import http.client

  conn = http.client.HTTPSConnection("")
  ```

3. Indique al usuario que guarde el código de recuperación. Si la llamada se realiza correctamente, obtendrá los tokens de autenticación y un nuevo código de recuperación:

   ```json lines
   {
       "access_token": "O3...H4",
       "id_token": "eyJh...w",
       "scope": "openid profile",
       "expires_in": 86400,
       "recovery_code": "K6LGLV3RSH3VERMKET8L7QKU",
       "token_type": "Bearer"
   }
   ```

4. Notifique al usuario que se ha generado un nuevo código de recuperación y pídale que lo guarde.
