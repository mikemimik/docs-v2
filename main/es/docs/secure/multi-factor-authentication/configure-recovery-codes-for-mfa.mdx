---
description: Cómo configurar códigos de recuperación para la autenticación multifactor.
title: Configurar códigos de recuperación para MFA
---

Un código de recuperación es un código único generado por Auth0 que permite a un usuario recuperar el acceso a su cuenta. Así, si un usuario no puede acceder al dispositivo o a la cuenta que utilizó para la inscripción a la <Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuario que utiliza un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/es/docs/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> (MFA), puede usar un código de recuperación para autenticarse.

<div id="how-it-works">
  ## Cómo funciona
</div>

Al usar <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Universal+Login">Universal Login</Tooltip> con los códigos de recuperación habilitados:

1. Un usuario inicia la inscripción en MFA.
2. Auth0 genera un código de recuperación.
3. Durante la inscripción en MFA, se le muestra al usuario el prompt del código de recuperación.
4. El usuario guarda el código de recuperación y completa el proceso de inscripción.
5. Después, el usuario puede completar la MFA con el código de recuperación que guardó si pierde acceso al dispositivo o a la cuenta que inscribió en MFA.

<Frame>![Ejemplo del prompt de código de recuperación durante la inscripción en la autenticación multifactor](/docs/images/cdy7uua7fh8z/6LkXKthRal1r4vIPi3137l/87d45c4202087244b20259dd30642435/2025-02-26_14-31-30.png)</Frame>

Al usar Universal Login con los códigos de recuperación deshabilitados:

* Los usuarios no verán el prompt del código de recuperación durante la inscripción en MFA.
* Los usuarios no pueden autenticarse con un código de recuperación.

<div id="enable-recovery-codes">
  ## Habilitar códigos de recuperación
</div>

Los códigos de recuperación están desactivados de forma predeterminada. Puedes activarlos en [Dashboard &gt; Seguridad &gt; Autenticación multifactor](https://manage.auth0.com/#/security/mfa).
