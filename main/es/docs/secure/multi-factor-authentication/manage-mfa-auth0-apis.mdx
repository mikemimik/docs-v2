Auth0 ofrece dos API diferentes para ayudarte a ti y a tus usuarios a gestionar los factores de autenticación de <Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuarios que utiliza un factor adicional al username y la password, como un code por SMS." cta="Ver glosario" href="/es/docs/glossary?term=MFA">MFA</Tooltip>.

<div id="auth0-authentication-api">
  ## Auth0 Authentication API
</div>

Usa los [endpoints de Multi-factor Authentication de la Authentication API](https://auth0.com/docs/api/authentication#multi-factor-authentication) para que tus usuarios gestionen sus propios factores de autenticación MFA.

Este método se basa en la autenticación mediante el flujo <Tooltip tip="Resource Owner: Entidad (como un usuario o una aplicación) capaz de conceder acceso a un recurso protegido." cta="Ver glosario" href="/es/docs/glossary?term=Resource+Owner">Resource Owner</Tooltip> Password Grant (ROPG). Para obtener más información, consulta [gestionar factores de autenticación con la Authentication API](/es/docs/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

<div id="auth0-management-api">
  ## Auth0 Management API
</div>

Usa los [endpoints de Users de la Auth0 Management API](https://auth0.com/docs/api/management/v2) para gestionar los factores de autenticación MFA de tus usuarios.

Este método requiere autenticarse mediante una aplicación confidencial. Para obtener más información, lee [Gestionar métodos de autenticación con Management API](/es/docs/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authentication-methods-with-management-api).
