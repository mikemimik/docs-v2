description: Describe la API de MFA de Auth0, que te permite inscribir y gestionar MFA para los usuarios.
title: API de MFA de Auth0
---

Auth0 proporciona un flujo integrado de inscripción y autenticación de <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación del usuario que utiliza un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/es/docs/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> (MFA) mediante [Universal Login](/es/docs/authenticate/login/auth0-universal-login). Usa la [API de MFA](https://auth0.com/docs/api/authentication#multi-factor-authentication) en los siguientes casos:

* [Autenticar usuarios con el Resource Owner Password Grant](/es/docs/secure/multi-factor-authentication/authenticate-using-ropg-flow-with-mfa).
* Crear una interfaz para que [los usuarios gestionen sus propios factores de autenticación](/es/docs/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

Para usar la API de MFA, debes habilitar el tipo de concesión MFA para tu aplicación. Para habilitar la concesión MFA en el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, ve a [Dashboard &gt; Applications &gt; Advanced Settings &gt; Grant Types](https://manage.auth0.com/#/applications) y selecciona **MFA**. Consulta [Habilitar la autenticación multifactor](/es/docs/secure/multi-factor-authentication/enable-mfa) para obtener más información.

<div id="limitations">
  ## Limitaciones
</div>

La API de MFA está diseñada para funcionar con factores de SMS, Voz, notificaciones push a través de la aplicación Auth0 Guardian, correo electrónico y OTP. Actualmente no admite el registro con Duo ni WebAuthn.
