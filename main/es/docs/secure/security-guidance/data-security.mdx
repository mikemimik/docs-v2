title: Seguridad de los datos
---

* [Direcciones IP de Auth0 para listas de permitidos](/es/docs/secure/security-guidance/data-security/allowlist) - Si tu entorno está detrás de un firewall, las conexiones de base de datos personalizadas, Actions, Hooks y Rules pueden requerir que agregues direcciones IP específicas de Auth0 a la lista de permitidos para garantizar su correcto funcionamiento.
* [Agregar atributos de usuario a la lista de denegados](/es/docs/secure/security-guidance/data-security/denylist) - Si hay campos de usuario que no deberían almacenarse en las bases de datos de Auth0 por motivos de privacidad, puedes agregar atributos de usuario a la lista de denegados.
* [Almacenamiento de datos de usuario](/es/docs/secure/security-guidance/data-security/user-data-storage) - Conoce las diferencias entre usar el almacén de datos de Auth0 y una base de datos externa para almacenar datos de usuario.
* [Almacenamiento de tokens](/es/docs/secure/security-guidance/data-security/token-storage) - Aprende cómo y dónde almacenar los tokens utilizados en la autenticación basada en tokens.
