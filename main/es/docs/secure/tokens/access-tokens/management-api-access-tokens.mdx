---
description: Aprende cómo funcionan los tokens de acceso de la Auth0 Management API y cómo utilizarlos.
title: Tokens de acceso de la Management API
---

Para llamar a la [Auth0 Management API](https://auth0.com/docs/api/management/v2), debes proporcionar un <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> que contenga el <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=audience">audience</Tooltip> y los scopes adecuados.

<div id="audience">
  ## Audience
</div>

El valor de audience para la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> es `https://{yourDomain}/api/v2/`.

<div id="scopes">
  ## Scopes
</div>

Cada endpoint de la Management API requiere un conjunto específico de scopes. Consulta el [Management API Explorer](https://auth0.com/docs/api/management/v2/) para más información.

<div id="lifetime">
  ## Tiempo de vida
</div>

El tiempo de vida predeterminado del token para la Management API es de 86,400 segundos (24 horas).

Puedes [configurar el tiempo de vida del token](/docs/es/secure/tokens/access-tokens/update-access-token-lifetime) para cada API de forma individual.

<div id="security">
  ## Seguridad
</div>

* **Token de acceso**: Una vez emitido, un token de acceso no se puede revocar. Auth0 recomienda limitar la vida útil de tus tokens de acceso al menor tiempo posible que permita tu caso de uso.
* **<Tooltip tip="Token de actualización: token utilizado para obtener un nuevo token de acceso sin obligar a los usuarios a volver a iniciar sesión." cta="Ver glosario" href="/docs/es/glossary?term=Refresh+token">token de actualización</Tooltip>**: Puedes [revocar un token de actualización](/docs/es/secure/tokens/refresh-tokens/revoke-refresh-tokens) desde el Dashboard, la Authentication API o la Management API.
* **<Tooltip tip="Secreto de cliente: secreto que usa un cliente (aplicación) para autenticarse ante el servidor de autorización; solo deben conocerlo el cliente y el servidor de autorización, y debe ser lo suficientemente aleatorio como para que no se pueda adivinar." cta="Ver glosario" href="/docs/es/glossary?term=Client+secret">secreto de cliente</Tooltip>**: Puedes [rotar el secreto de cliente de una aplicación](/docs/es/get-started/applications/rotate-client-secret) desde el Dashboard o la Management API.

<div id="quota">
  ## Cuota
</div>

Las cuotas de tokens se determinan según el nivel de suscripción. Puedes [consultar tu cuota actual](https://support.auth0.com/reports/quota) en el Support Center de Auth0. Si tienes preguntas sobre precios o cuotas, [contacta a nuestro equipo de ventas](https://auth0.com/contact-us).

* Los tokens emitidos para [audiencias internas de Auth0](/docs/es/secure/tokens/access-tokens/get-access-tokens#control-access-token-audience) no cuentan para tu cuota.
* Los tokens emitidos para audiencias personalizadas cuentan para tu cuota y están sujetos a los [límites de frecuencia de Auth0 Management API](/docs/es/troubleshoot/customer-support/operational-policies/rate-limit-policy/management-api-endpoint-rate-limits).
