Puedes desactivar la <Tooltip tip="Token de actualización: token que se utiliza para obtener un nuevo token de acceso sin que los usuarios tengan que volver a iniciar sesión." cta="Ver glosario" href="/es/docs/glossary?term=refresh+token">rotación del Refresh Token</Tooltip> para cada aplicación mediante el Dashboard o la <Tooltip tip="Token de actualización: token que se utiliza para obtener un nuevo token de acceso sin que los usuarios tengan que volver a iniciar sesión." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>.

<div id="disable-with-the-dashboard">
  ## Desactivar desde el Dashboard
</div>

1. Desde el [Auth0 Dashboard](https://manage.auth0.com/#), ve a **Applications &gt; Applications** y selecciona la aplicación que deseas configurar.
2. En la pestaña **Settings**, localiza la sección de rotación del token de actualización y desactiva el interruptor **Allow Refresh Token Rotation**.
3. Selecciona **Save Changes** en la parte inferior de la pantalla.

<div id="disable-with-the-management-api">
  ## Deshabilitar mediante la Management API
</div>

1. Deshabilita la rotación del Refresh Token para cada aplicación mediante la Management API:

   {/* codeblockOld.header.login.logInButton codeblockOld.header.login.configureSnippet */}

   <AuthCodeBlock children={codeExample} language="javascript" />

2. Configura los ajustes del token de actualización sin rotación de la siguiente manera:

   ```http lines
   PATCH /api/v2/clients/{client_id}
   {
      "refresh_token": {
        "rotation_type": "non-rotating",
        "expiration_type": "non-expiring"
      }
   }
   ```
