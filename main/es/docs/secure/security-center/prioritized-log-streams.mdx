title: Flujos de logs priorizados
---

**Flujos de logs priorizados** es una función del Dashboard que resalta los eventos de seguridad. En caso de ataque o de un aumento de la actividad de los usuarios, los Flujos de logs priorizados y los eventos predefinidos seguirán activos para que la automatización y los procesos de seguridad posteriores no se vean afectados.

Puedes crear un nuevo flujo de logs priorizado con un servicio compatible o configurar un webhook personalizado para enviar datos a un servicio que ya uses, y todos los logs pueden exportarse a un servicio de análisis de eventos. Los flujos de logs priorizados no cuentan para el límite de flujos de logs que se muestra en el plan de precios.

<div id="configure-a-prioritized-log-stream">
  #### Configurar un flujo de logs priorizado
</div>

1. Sigue las instrucciones de la [página de Log Streams](/es/docs/customize/log-streams) para configurar un nuevo flujo de logs.
2. Mientras configuras el nuevo flujo, ve a la pestaña **Settings** en tu Auth0 Dashboard y selecciona la casilla **Prioritized Log** para activar los eventos de seguridad predefinidos que se detallan a continuación. Selecciona **Save**.
3. Ve a la página de tu **Stream** y navega hasta la etiqueta **Prioritized**.

<div id="prioritized-logs-event-types">
  #### Tipos de eventos de registros priorizados
</div>

Auth0 clasifica los siguientes códigos de evento como eventos relacionados con la seguridad:

| Código de evento  | Evento                                                       |
| ----------------- | ------------------------------------------------------------ |
| `limit_mu`        | Dirección IP bloqueada                                       |
| `limit_sul`       | Cuenta bloqueada (limitación por IP)                         |
| `limit_wc`        | Cuenta bloqueada (protección contra ataques de fuerza bruta) |
| `pwd_leak`        | Contraseña comprometida en Login                             |
| `ublkdu`          | Bloqueo de inicio de sesión del usuario levantado            |
| `signup_pwd_leak` | Contraseña comprometida en Signup                            |
| `reset_pwd_leak`  | Contraseña comprometida al restablecer                       |
| `gd_send_sms`     | SMS de MFA enviado                                           |
