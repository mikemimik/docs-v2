title: Retención de datos de logs
---

El período de retención de logs en Auth0 depende de tu nivel de suscripción.

| Plan             | Retención de logs |
| ---------------- | ----------------- |
| Starter          | 1 día             |
| B2C Essentials   | 5 días            |
| B2C Professional | 10 días           |
| B2B Essentials   | 5 días            |
| B2B Professional | 10 días           |
| Enterprise       | 30 días           |

El [equipo de soporte de Auth0](https://support.auth0.com) puede ayudarte a integrarte con servicios externos de logging si necesitas conservar los logs más allá del período de retención. Este enfoque también es útil para consolidar los logs de toda tu organización.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 no proporciona logs en tiempo real para tu tenant. Aunque hacemos todo lo posible por indexar los eventos a medida que llegan, puede que notes algunos retrasos.
</Callout>

Además, puedes usar el flujo de logs de Auth0 para exportar datos de logs. Para obtener más información, ve a [Log Streaming](https://marketplace.auth0.com/features/log-streaming) en Auth0 Marketplace.
