description: Describe las opciones de implementación de Auth0 en nube pública y nube privada.
title: Opciones de implementación
---

Auth0 ofrece opciones de implementación tanto en nube pública como en nube privada, así como herramientas para ayudarte con tu implementación empresarial.

La plataforma de identidad de Auth0 se puede implementar de las siguientes maneras:

| Opción                                                                                         | Descripción                                                                                                                                                                |
| ---------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Public Cloud**                                                                               | Un servicio en la nube multi-tenant que se ejecuta en la nube de Auth0 con funciones estándar de autenticación, personalización, gestión de usuarios y seguridad           |
| [**Private Cloud on AWS**](/es/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | Un servicio en la nube dedicado y administrado que se ejecuta en AWS y proporciona aislamiento, mayor rendimiento, instancias de desarrollo, add-ons de Geo-HA y mucho más |
| [**Private Cloud on Azure**](/es/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Un servicio en la nube dedicado que se ejecuta en Azure y proporciona aislamiento, mayor rendimiento, instancias de desarrollo, add-ons de Geo-HA y mucho más              |

Los paquetes de **Private Cloud** son servicios administrados que puedes usar si:

* No puedes usar un servicio en la nube pública multi-tenant en tu organización
* Necesitas un nivel garantizado de peticiones por segundo (RPS)

Si tienes requisitos de soporte específicos o necesitas más información sobre los Servicios Profesionales que ofrecemos, [contacta con Ventas de Auth0](https://auth0.com/contact-us).
