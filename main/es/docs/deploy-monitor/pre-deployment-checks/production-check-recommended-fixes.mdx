---
description: Recomendaciones sobre cómo mejorar tu aplicación de Auth0 antes del despliegue en producción
title: 'Comprobaciones de preparación para producción: correcciones no críticas'
---

Las siguientes comprobaciones verifican si has completado todas las recomendaciones (opcionales) para lograr un despliegue correcto en producción.

Para obtener más información sobre el resultado de estas comprobaciones, consulta [Run Production Checks](/es/docs/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Comprobación                                                                                                                        | Descripción                                                                                                                                                                                                                                                                                                                                                      |
| ----------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configurar Log Streaming                                                                                                            | Asegúrate de transmitir las entradas de logs del tenant a una ubicación de almacenamiento externa o a un proveedor externo. Te recomendamos [configurar un servicio de transmisión de logs](/es/docs/customize/log-streams) para conservar los logs durante más de 30 días y facilitar mejores análisis y alertas.                                                  |
| Se debe configurar un [dominio personalizado](/es/docs/customize/custom-domains)                                                       | Usa [dominios personalizados con Universal Login](https://manage.auth0.com/#/tenant/custom_domains) para ofrecer a tus usuarios finales una experiencia más fluida y segura.                                                                                                                                                                                     |
| Se debe configurar una [página de error personalizada](/es/docs/customize/login-pages/custom-error-pages)                              | [Configura una página de error personalizada](https://manage.auth0.com/#/account) con los datos específicos de tu aplicación y la identidad visual de tu empresa.                                                                                                                                                                                                |
| Habilitar Universal Login                                                                                                           | Asegúrate de usar la versión más reciente de [Universal Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login). Te recomendamos actualizar tu página de inicio de sesión de la experiencia Classic a Universal Login para beneficiarte de una mejor accesibilidad y rendimiento, personalización sin código y nuevas funciones. |
| Se deben configurar las [plantillas de correo electrónico](/es/docs/customize/email/manage-email-flow)                                 | [Configura plantillas de correo electrónico personalizadas](https://manage.auth0.com/#/emails) con los datos específicos de tu aplicación y la identidad visual de tu empresa.                                                                                                                                                                                   |
| Se debe configurar la [autenticación multifactor](/es/docs/secure/multi-factor-authentication)                                         | Se recomienda considerar la autenticación multifactor como una práctica segura para autenticar a tus usuarios.                                                                                                                                                                                                                                                   |
| La [autenticación multifactor para administradores del tenant](/es/docs/get-started/manage-dashboard-access) está habilitada           | [Habilita la autenticación multifactor](https://manage.auth0.com/#/account/admins) para los administradores del tenant.                                                                                                                                                                                                                                          |
| Se debe definir la [URL de redirección de logout](/es/docs/authenticate/login/logout#set-the-allowed-logout-urls-at-the-account-level) | Revisa las [URL de redirección de logout permitidas](https://manage.auth0.com/#/account/advanced) de tu aplicación.                                                                                                                                                                                                                                              |
| Las aplicaciones deben usar RS256 en lugar de HS256 para el algoritmo de firma de JSON Web Token (JWT)                              | Configura el [algoritmo de firma](/es/docs/get-started/apis#signing-algorithms) de JSON Web Token (JWT) en RS256 en lugar de HS256 para cada aplicación.                                                                                                                                                                                                            |
| Configurar el [URI de inicio de sesión de la aplicación](/es/docs/get-started/applications/application-settings#application-uris)      | Indica un [URI de inicio de sesión de la aplicación](/es/docs/get-started/applications/application-settings#application-uris) para todas las aplicaciones interactivas.                                                                                                                                                                                             |
| Configurar la [URL de logout permitida del tenant](/es/docs/get-started/applications/application-settings#application-uris)            | Crea al menos una ruta de logout accesible públicamente y [actualiza la configuración del tenant](/es/docs/get-started/applications/application-settings#application-uris) para que apunte a ella.                                                                                                                                                                  |
| Configurar el [URI de inicio de sesión del tenant](/es/docs/get-started/tenant-settings#login-and-logout)                              | Publica una ruta de inicio de sesión predeterminada y [actualiza la configuración de tu tenant](/es/docs/get-started/tenant-settings#login-and-logout) para que apunte a ella.                                                                                                                                                                                      |
