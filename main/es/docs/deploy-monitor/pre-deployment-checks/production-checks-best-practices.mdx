title: 'Verificaciones de preparación para producción: prácticas recomendadas'
---

Las siguientes verificaciones no se pueden automatizar, por lo que recomendamos revisar manualmente estas áreas antes del despliegue en Producción.

| Verificación                                                                                                    | Descripción                                                                                                                                                                                                                                                                                                        |
| --------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Externalizar [parámetros de configuración](/es/docs/authenticate/database-connections/custom-db)                   | [Externalice, en lugar de codificar de forma fija, todos los parámetros de configuración](https://manage.auth0.com/#/connections/database), como credenciales, cadenas de conexión, claves de API, etc., al desarrollar Rules, Hooks o conexiones de base de datos personalizadas.                                 |
| Revisar los valores de tiempo de espera de [Single Sign-on (SSO)](/es/docs/glossary?term=Single+Sign-on+%28SSO%29) | Revise los valores predeterminados de [tiempo de espera de la cookie de SSO](https://manage.auth0.com/#/account/advanced) y asegúrese de que se ajusten a sus requisitos.                                                                                                                                          |
| Tenants y administradores                                                                                       | Revise todos los tenants y los administradores del tenant para asegurarse de que sean correctos. Desactive los tenants que ya no estén en uso. Asegúrese de que los administradores del tenant se limiten a los usuarios necesarios.                                                                               |
| Verificar los ID de cliente en el código de la aplicación                                                       | Asegúrese de que los ID de cliente en el código de su aplicación coincidan con la configuración correspondiente de la aplicación en Auth0.                                                                                                                                                                         |
| Agregar las IP públicas de Auth0 a la lista de permitidos                                                       | Incluya las IP de Auth0 en la lista de permitidos si se está conectando a servicios internos o a servicios detrás de un firewall al usar Rules, Hooks o bases de datos personalizadas. Puede obtener una lista de direcciones IP en la información sobre herramientas al configurar cualquiera de estos elementos. |
| Revisar la protección contra ataques                                                                            | Se recomienda [revisar la funcionalidad de Protección de Auth0 y su configuración](/es/docs/secure/attack-protection).                                                                                                                                                                                                |
