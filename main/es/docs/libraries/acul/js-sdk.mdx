El Auth0 ACUL JS SDK te permite crear experiencias personalizadas de Universal Login con el mínimo esfuerzo. Tanto si trabajas con el inicio de sesión, el registro, passwordless o el registro de claves de acceso, este SDK está diseñado para gestionar fácilmente los flujos de autenticación modernos.

<div id="why-choose-the-auth0-acul-js-sdk">
  ### ¿Por qué elegir Auth0 ACUL JS SDK?
</div>

* **Integración rápida:** Integra pantallas prediseñadas como `login-id` o `signup-password`.
* **Funciones modernas:** Compatible con passwordless, claves de acceso e inicios de sesión con redes sociales.
* **Módulos flexibles:** Importa solo lo que necesites.
* **Gestión unificada del estado:** Usa un único objeto de transacción para gestionar todos los flujos de autenticación.

<div id="architecture">
  ### Arquitectura
</div>

<div id="how-it-works">
  #### ¿Cómo funciona?
</div>

El SDK está diseñado teniendo en cuenta la modularidad y la escalabilidad. Así es como encaja todo:

* **Pantallas:** Cada flujo de autenticación, como `login-id` y `signup-password`, es un módulo independiente.
* **Controladores de envío:** Gestionan el envío de formularios y las transiciones.
* **Formularios dinámicos:** Adaptan los campos y las validaciones según la configuración de la pantalla.
* **Propiedades auxiliares:** Permiten acceder de forma coherente a datos comunes, como la identidad visual o la transacción.
