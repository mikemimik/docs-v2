La pantalla `reset-password-email` se muestra después de que un usuario solicite un restablecimiento de contraseña. Informa al usuario de que se le ha enviado un correo para restablecerla y le ofrece la opción de reenviarlo si es necesario.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz te permite cargar todas las pantallas desde un único bundle, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useResetPasswordEmail } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useResetPasswordEmail,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Métodos
  resendEmail,
} from '@auth0/auth0-acul-react/reset-password-email';

function ResetPasswordEmailScreen() {
  const { resendEmail } = useResetPasswordEmail();
  return (
    <button onClick={() => resendEmail()}>
      Resend Email
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexto
</div>

Hooks específicos de la pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `reset-password-email`. Impórtalos desde `@auth0/auth0-acul-react/reset-password-email`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona la configuración de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `reset-password-email`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-email';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `reset-password-email`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-email';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el restablecimiento de contraseña tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-email';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Sin contexto de organización</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-email';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnResetPasswordEmail">ScreenMembersOnResetPasswordEmail</a></span>}>
  Este hook contiene detalles específicos de la pantalla `reset-password-email`, incluida su configuración y su contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/reset-password-email';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/reset-password-email';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `reset-password-email`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/reset-password-email';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook contiene datos no confiables que se pasan a la pantalla, como un correo electrónico o nombre de usuario precargados desde los parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/reset-password-email';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/reset-password-email';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useResetPasswordEmail" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordEmailMembers">ResetPasswordEmailMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `reset-password-email`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="resendEmail" type="Promise<void>">
  Este método vuelve a enviar al usuario el correo electrónico de restablecimiento de contraseña.

  ```jsx Example
  import { useResetPasswordEmail } from '@auth0/auth0-acul-react/reset-password-email';

  function ResendButton() {
    const { resendEmail } = useResetPasswordEmail();
    return (
      <button onClick={() => resendEmail()}>
        Reenviar correo electrónico
      </button>
    );
  }
  ```
</ParamField>
