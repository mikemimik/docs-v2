La pantalla `mfa-sms-list` muestra una lista de números de teléfono registrados y permite al usuario seleccionar uno para recibir un código de verificación por SMS.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo cual resulta útil cuando quieres una única compilación que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useMfaSmsList } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useMfaSmsList,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks de utilidad
  useChangeLanguage,
  // Métodos
  backAction,
  selectPhoneNumber,
} from "@auth0/auth0-acul-react/mfa-sms-list";

function MfaSmsListScreen() {
  const { selectPhoneNumber } = useMfaSmsList();
  return (
    <button onClick={() => selectPhoneNumber({ index: 0 })}>
      Use This Phone Number
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexto
</div>

Hooks de ámbito de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-sms-list`. Impórtalos desde `@auth0/auth0-acul-react/mfa-sms-list`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona la configuración de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `mfa-sms-list`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-sms-list';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-sms-list`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-sms-list';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el flujo de MFA tiene ámbito de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-sms-list';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-sms-list';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-sms-list`, incluida su configuración y su contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-sms-list';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-sms-list';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-sms-list`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-sms-list';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja los datos no confiables que se pasan a la pantalla, como valores rellenados previamente a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-sms-list';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook ofrece detalles del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-sms-list';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaSmsList" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaSmsListMembers">MfaSmsListMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `mfa-sms-list`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="backAction" type="Promise<void>">
  Este método vuelve a la pantalla anterior del flujo de MFA.

  ```jsx Example
  import { useMfaSmsList } from '@auth0/auth0-acul-react/mfa-sms-list';

  function BackButton() {
    const { backAction } = useMfaSmsList();
    return (
      <button onClick={() => backAction()}>
        Go Back
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="selectPhoneNumber" type="Promise<void>">
  Este método selecciona un número de teléfono de la lista de números registrados para usarlo en la verificación por SMS.

  ```jsx Example
  import { useMfaSmsList } from '@auth0/auth0-acul-react/mfa-sms-list';

  function PhoneNumberSelector() {
    const { selectPhoneNumber } = useMfaSmsList();
    return (
      <button onClick={() => selectPhoneNumber({ index: 0 })}>
        Use This Phone Number
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaSmsListOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaSmsListOptions).
    </ParamField>

    <ParamField body="index" type="number" required>
      El índice del número de teléfono que se va a seleccionar.
    </ParamField>
  </Expandable>
</ParamField>
