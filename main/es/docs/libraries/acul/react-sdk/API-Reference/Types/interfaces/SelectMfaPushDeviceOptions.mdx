---
title: "SelectMfaPushDeviceOptions"
---

Opciones para seleccionar un dispositivo de MFA push.

```ts Example
export interface SelectMfaPushDeviceOptions {
  /**
   * El índice del dispositivo a seleccionar de la lista de dispositivos registrados.
   * El índice es base 0.
   */
  deviceIndex: number;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propiedades
</div>

<ParamField body="deviceIndex" type="number">
  El índice del dispositivo que se debe seleccionar de la lista de dispositivos registrados.
  El índice comienza en 0.
</ParamField>
