ResendCodeOptions
description Opciones para el método `resendCode` en la pantalla Login Email Verification.
Esta operación se utiliza cuando el usuario solicita que se envíe un nuevo código de verificación a su correo electrónico,
por ejemplo, si no recibió el código inicial o si este ha caducado.

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
