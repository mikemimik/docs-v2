El valor que devuelve [`usePasskeyAutofill`](/es/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasskeyAutofill).

```typescript
interface UsePasskeyAutofillResult {
  inputRef: RefObject<HTMLInputElement>;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  Una referencia de React para adjuntarla al campo de nombre de usuario. Al asociarla, el SDK configura automáticamente el atributo `autocomplete` del campo como `"username webauthn"`, lo que habilita la interfaz de usuario condicional del navegador para el autocompletado con claves de acceso. Asociar esta referencia es opcional si `autocomplete="username webauthn"` ya está declarado en tu código.
</ParamField>
