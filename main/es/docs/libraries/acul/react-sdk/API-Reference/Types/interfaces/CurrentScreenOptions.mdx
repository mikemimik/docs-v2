---
title: "CurrentScreenOptions"
---

Objeto de opciones que describe el contexto de la pantalla actual en un flujo de Auth0 Universal Login. Proporciona acceso al cliente, la organización, el tenant, la transacción y los metadatos de la pantalla disponibles en runtime.

```ts Example
export interface CurrentScreenOptions {
  client: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  organization: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  prompt: {
    name: string;
  } | null;
  screen: {
    name: string;
  } | null;
  tenant: {
    enabledLocales: string[];
  } | null;
  transaction: {
    errors: TransactionError[] | null;
    state: string;
    locale: string;
  } | null;
  untrustedData: {
    authorizationParams: Record<string, string> | null;
  } | null;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="client" type="{ id: string; metadata: Record<string, string> | null } | null">
  Detalles del cliente de la solicitud de autorización actual, incluidos el identificador del cliente y cualquier metadato asociado.
</ParamField>

<ParamField body="organization" type="{ id: string; metadata: Record<string, string> | null } | null">
  Contexto de la organización para la solicitud actual, incluidos el identificador de la organización y los metadatos.
</ParamField>

<ParamField body="prompt" type="{ name: string } | null">
  La configuración actual del prompt, que identifica qué paso del prompt se está renderizando.
</ParamField>

<ParamField body="screen" type="{ name: string } | null">
  La configuración actual de la pantalla, que identifica qué pantalla del prompt está activa.
</ParamField>

<ParamField body="tenant" type="{ enabledLocales: string[] } | null">
  Configuración a nivel de tenant, incluida la lista de configuraciones regionales habilitadas para esta implementación de Universal Login.
</ParamField>

<ParamField body="transaction" type="{ errors: TransactionError[] | null; state: string; locale: string } | null">
  Estado de la transacción del flujo de authentication actual, incluidos los errores de validación, el identificador de estado y la configuración regional activa.
</ParamField>

<ParamField body="untrustedData" type="{ authorizationParams: Record<string, string> | null } | null">
  Datos no confiables pasados al flujo, como parámetros de autorización en los que no se debe basar ninguna decisión de seguridad.
</ParamField>
