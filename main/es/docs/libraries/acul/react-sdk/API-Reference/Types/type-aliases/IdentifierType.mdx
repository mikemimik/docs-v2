---
title: "IdentifierType"
---

Un tipo de unión de cadenas que representa los métodos válidos para identificar a un usuario durante la autenticación. Se usa para limitar los valores de identificador a uno de tres valores específicos.

```typescript
type IdentifierType = "phone" | "email" | "username";
```

<div id="values">
  ## Valores
</div>

<ParamField body="&#x22;phone&#x22;" type="IdentifierType">
  Un identificador de teléfono.
</ParamField>

<ParamField body="&#x22;email&#x22;" type="IdentifierType">
  Un identificador de correo electrónico.
</ParamField>

<ParamField body="&#x22;username&#x22;" type="IdentifierType">
  Un identificador de nombre de usuario.
</ParamField>
