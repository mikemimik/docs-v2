title: "MfaOtpContinueOptions"

Opciones para continuar con el código OTP de inscripción en MFA.

```ts Example
export interface ContinueOptions {
  /**
   * El código OTP ingresado por el usuario.
   */
  code: string;

  /**
   * Cualquier opción personalizada adicional.
   */
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

Cualquier otra opción personalizada.

  ## Propiedades
</div>
