title: "MfaPollingResult"
---

Objeto de resultado que devuelve [useMfaPolling](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling).

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isRunning" type="boolean">
  Indica si el proceso de sondeo de MFA push está activo en este momento.

  * `true` — El sondeo está en curso y a la espera de que finalice.
  * `false` — El sondeo se ha detenido, ya sea por finalización,
    cancelación manual o desmontaje del componente.
</ParamField>

<ParamField body="startPolling()" type="void">
  Inicia o reanuda el proceso de sondeo.

  * Si el sondeo ya está activo, esta llamada no surte efecto.
  * Si se detuvo previamente, esta llamada reinicia el bucle de sondeo.
</ParamField>

<ParamField body="stopPolling()" type="void">
  Detiene el proceso de sondeo de inmediato.

  * Cancela cualquier temporizador programado o solicitud en curso.
  * Se puede llamar varias veces sin problema; las llamadas posteriores no surten efecto.
</ParamField>
