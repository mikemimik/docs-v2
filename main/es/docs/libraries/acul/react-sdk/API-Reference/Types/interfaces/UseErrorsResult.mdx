---
title: "UseErrorsResult"
---

Objeto de resultado devuelto por [`useErrors`](/es/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Proporciona acceso al estado actual de los errores y métodos para descartar errores individuales o todos los errores en las pantallas de Auth0 Universal Login.

```ts Example
export interface UseErrorsResult {
  errors: ErrorsResult;
  hasError: boolean;
  dismiss: (id: string) => void;
  dismissAll: () => void;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="errors" type={<a href="/es/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorsResult">ErrorsResult</a>} required>
  La colección de errores actuales, incluidos sus mensajes e identificadores.
</ParamField>

<ParamField body="hasError" type="boolean" required>
  `true` cuando hay uno o más errores; útil para mostrar condicionalmente la interfaz de error.
</ParamField>

<ParamField body="dismiss" type="(id: string) => void" required>
  Elimina un error específico por su `id`.

  <Expandable title="Parámetros">
    <ParamField body="id" type="string">
      El identificador del error que se va a eliminar.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="dismissAll" type="() => void" required>
  Elimina todos los errores activos de una sola vez.
</ParamField>
