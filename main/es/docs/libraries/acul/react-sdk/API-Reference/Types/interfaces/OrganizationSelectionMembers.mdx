Interfaz que define los métodos y las propiedades disponibles de la pantalla de selección de organización.

```ts Example
export interface OrganizationSelectionMembers extends BaseMembers {
  /**
   * Las propiedades de la pantalla.
   */
  screen: ScreenMembersOnOrganizationSelection;
  /**
   * Continúa con el nombre de organización seleccionado.
   * @param payload Las opciones que contienen el nombre de la organización.
   */
  continueWithOrganizationName(payload: ContinueWithOrganizationNameOptions): Promise<void>;
}
```
