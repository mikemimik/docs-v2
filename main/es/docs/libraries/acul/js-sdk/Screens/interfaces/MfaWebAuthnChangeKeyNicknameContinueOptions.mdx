ContinueOptions

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * El nuevo alias que el usuario desea asignar a su clave de seguridad WebAuthn.
   * Este valor se enviará al servidor para su actualización.
   * El servidor validará este alias (p. ej., longitud, caracteres permitidos).
   * @type {string}
   * @example "Primary YubiKey"
   * @example "Home Office Key"
   */
  nickname: string;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="nickname" type="string">
  El nuevo alias de la clave de seguridad WebAuthn.
  Este es el valor que el usuario introduce en el campo de entrada.
  Debe cumplir las restricciones de longitud y caracteres definidas por el servidor.
</ParamField>
