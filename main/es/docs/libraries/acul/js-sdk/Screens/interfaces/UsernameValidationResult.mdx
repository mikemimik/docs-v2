---
title: "UsernameValidationResult"
---

El resultado de validar un nombre de usuario según una o más reglas.

```ts Example
export interface UsernameValidationResult {
  /**
   * Indica si el nombre de usuario superó todas las reglas de validación.
   */
  isValid: boolean;

  /**
   * Un arreglo de objetos {@link UsernameValidationError} que representan
   * las reglas individuales que fallaron, si las hay.
   *
   * Este arreglo está vacío si {@link UsernameValidationResult.isValid} es `true`.
   */
  errors: UsernameValidationError[];
}
```
