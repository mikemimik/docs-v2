ScreenMembersOnResetPasswordMfaWebAuthnRoamingChallenge

```ts Example
export interface ScreenMembersOnResetPasswordMfaWebAuthnRoamingChallenge extends ScreenMembers {
  /**
   * Un acceso directo a los datos de `passkey.public_key` desde `screen.data`.
   * Este objeto contiene el desafío y otras opciones necesarias para la
   * llamada a la API de WebAuthn `navigator.credentials.get()`.
   * Es `null` si `screen.data.passkey.public_key` no está disponible.
   * @type {PasskeyRead['public_key'] | null}
   * @public
   */
  publicKey: PasskeyRead['public_key'] | null;

  /**
   * Un acceso directo al indicador `show_remember_device` desde `screen.data`.
   * Indica si la UI debe presentar al usuario una opción para recordar este dispositivo/navegador
   * y potencialmente omitir el MFA en futuros inicios de sesión. El valor predeterminado es `false` si no está definido en `screen.data`.
   * @type {boolean}
   * @public
   */
  showRememberDevice: boolean;
}
```
