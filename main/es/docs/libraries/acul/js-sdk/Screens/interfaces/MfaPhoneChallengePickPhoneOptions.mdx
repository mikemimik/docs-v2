PickPhoneOptions
Define las opciones para la Action &#39;pick-phone&#39;.
Aunque la interfaz solo muestra un único número de teléfono, esta Action puede usarse
en casos en los que el servidor necesita redirigir a una pantalla de selección de teléfono.

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```
