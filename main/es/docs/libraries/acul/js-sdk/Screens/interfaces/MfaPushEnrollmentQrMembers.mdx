    <ParamField body="options" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/type-aliases/MfaPollingOptions">MfaPollingOptions</a></span>}>
      <ParamField body="MfaPollingOptions" type="object">
        Opciones de configuración para gestionar un proceso de polling de notificaciones push de MFA.

        Proporcione estas opciones al iniciar una sesión de polling para controlar la frecuencia
        con la que se consulta el endpoint del challenge y cómo se gestionan los casos de éxito o error.

        <Expandable title="Propiedades">
          <ParamField body="intervalMs?" type="number">
            Intervalo opcional, en **milisegundos**, entre solicitudes de polling consecutivas.

            * Si se omite, se usa el intervalo predeterminado interno del SDK (normalmente 5000 ms).
            * Elija un valor que equilibre la capacidad de respuesta con las consideraciones sobre el límite de frecuencia.
          </ParamField>

          <ParamField body="onCompleted()?" type="() => void">
            Devolución de llamada opcional que se ejecuta cuando el challenge push de MFA se aprueba
            correctamente y el polling finaliza.

            * Se llama exactamente **una vez**, después de lo cual el polling se detiene automáticamente.
            * Útil para avanzar en el flujo de inicio de sesión (por ejemplo, llamar a `continue()`).
          </ParamField>

          <ParamField body="onError()?" type="(error) => void">
            Devolución de llamada opcional que se invoca si se produce un error durante el polling.

            Recibe un objeto [ULError](/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error) que contiene:

            * `status` — el código de estado HTTP de la solicitud fallida

            * `responseText` — el cuerpo de la respuesta sin procesar, si está disponible

            * Se llama una vez por cada evento de error; el polling puede reintentarse según el tipo de error.

            * Úselo para mostrar mensajes de error o activar lógica personalizada de reintento o cancelación.
