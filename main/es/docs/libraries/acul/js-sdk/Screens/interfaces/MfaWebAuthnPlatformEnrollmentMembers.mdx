<ParamField body="submitPasskeyCredential" type="Promise<void>">
  Inicia el proceso de creación de la credencial de plataforma de WebAuthn usando las opciones de clave pública
  disponibles en `this.screen.publicKey` y envía la credencial resultante al servidor.
  Este método llama internamente a `createPasskeyCredentials` (que encapsula `navigator.credentials.create()`).

  Una promesa que se resuelve cuando la credencial se crea y se envía correctamente.

  #### Throws

  Lanza un error si `this.screen.publicKey` no está disponible, si `createPasskeyCredentials` falla
  (por ejemplo, por cancelación del usuario o problemas de hardware) o si falla el envío al servidor.

  ```typescript Example
  // Suponiendo que 'sdk' es una instancia de MfaWebAuthnPlatformEnrollment
  try {
    if (!sdk.screen.publicKey) { // Comprueba si las opciones están disponibles
      throw new Error("Public key creation options are not available on the screen context.");
    }
    await sdk.submitPasskeyCredential(); // No es necesario pasar publicKey explícitamente
    // En caso de éxito, Auth0 gestiona la redirección.
  } catch (error) {
    console.error('Passkey enrollment failed:', error);
    // Maneja el error, potencialmente llamando a reportBrowserError si es un error de la API de WebAuthn
    if (error.name && error.message) { // Comprueba si parece un error de WebAuthn
      await sdk.reportBrowserError({ error: { name: error.name, message: error.message } });
    }
  }
  ```
