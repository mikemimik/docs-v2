---
title: "Identificador"
---

Describe un campo identificador configurado (p. ej., correo electrónico, teléfono o nombre de usuario)
para usarlo en un formulario de registro o en un flujo de autenticación.

```ts Example
export interface Identifier {
  /**
   * El tipo de identificador, como `'email'`, `'phone'`, o `'username'`.
   *
   * @see IdentifierType
   */
  type: IdentifierType;

  /**
   * Si este identificador es obligatorio para el registro.
   */
  required: boolean;
}
```

<div id="example">
  ## Ejemplo
</div>

```ts
const emailIdentifier: Identifier = {
  type: 'email',
  required: true
};
```
