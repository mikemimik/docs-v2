Opciones para realizar operaciones de inicio de sesión

```ts Example
export interface LoginOptions {
  /** El nombre de usuario/email con el que iniciar sesión */
  username: string;
  /** La contraseña para la autenticación */
  password: string;
  /** Valor de captcha opcional si es necesario */
  captcha?: string;
  /** Cualquier opción adicional personalizada */
  [key: string]: string | number | boolean | undefined;
}
```
