---
title: "MfaPhoneEnrollmentContinueOptions"
---

Opciones para continuar con el registro del teléfono para MFA.

Extiende `CustomOptions` para permitir pasar opciones personalizadas junto con el teléfono y el tipo.

```ts Example
export interface ContinueOptions extends CustomOptions {
  phone: string;
  type: 'sms' | 'voice';
}
```
