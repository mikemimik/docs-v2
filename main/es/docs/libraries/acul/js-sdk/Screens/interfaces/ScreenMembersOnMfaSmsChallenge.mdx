Interfaz para los datos específicos de la pantalla `mfa-sms-challenge`

```ts Example
export interface ScreenMembersOnMfaSmsChallenge extends ScreenMembers {
  data: {
    /**
     * El número de teléfono al que se envió el SMS
     */
    phoneNumber?: string;
    /**
     * Si se debe mostrar la opción de recordar dispositivo
     */
    showRememberDevice?: boolean;
    /**
     * Si se debe mostrar el enlace para cambiar a verificación por llamada de voz
     */
    showLinkVoice?: boolean;
  } | null;
}
```
