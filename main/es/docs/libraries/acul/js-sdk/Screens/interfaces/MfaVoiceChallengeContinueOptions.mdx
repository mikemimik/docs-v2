title: "MfaVoiceChallengeContinueOptions"
---

Opciones para enviar el código de verificación por voz.

```ts Example
export interface MfaVoiceChallengeContinueOptions {
  /**
   * El código de verificación recibido mediante llamada de voz.
   */
  code: string;

  /**
   * Indicador opcional para recordar el dispositivo durante 30 días, omitiendo futuros desafíos de MFA.
   */
  rememberDevice?: boolean;

  /**
   * Opciones custom adicionales para incluir en la solicitud.
   */
  [key: string]: string | number | boolean | undefined;
}
```
