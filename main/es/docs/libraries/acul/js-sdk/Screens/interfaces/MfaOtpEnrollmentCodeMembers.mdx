title: "MfaOtpEnrollmentCodeMembers"

Interfaz que define los métodos y propiedades disponibles para la pantalla `mfa-otp-enrollment-code`.

```ts Example
export interface MfaOtpEnrollmentCodeMembers extends BaseMembers {
  /**
   * Las propiedades de pantalla para la pantalla mfa-otp-enrollment-code.
   */
  screen: ScreenMembersOnMfaOtpEnrollmentCode;

  /**
   * Continúa el proceso de registro de MFA OTP enviando el código OTP.
   * @param payload Las opciones que contienen el código OTP.
   */
  continue(payload: ContinueOptions): Promise<void>;

  /**
   * Permite al usuario probar otro método de MFA.
   * @param payload Opciones Custom opcionales para incluir en la solicitud.
   */
  tryAnotherMethod(payload?: TryAnotherMethodOptions): Promise<void>;
}
```
