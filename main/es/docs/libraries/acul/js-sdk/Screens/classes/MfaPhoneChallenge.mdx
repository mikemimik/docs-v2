La clase MfaPhoneChallenge implementa la funcionalidad de la pantalla `mfa-phone-challenge`. Esta pantalla se muestra cuando el usuario necesita recibir un código para verificar su identidad.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla MFA Phone Challenge:

```typescript Example
import MfaPhoneChallenge from '@auth0/auth0-acul-js/mfa-phone-challenge';

const mfaPhoneChallenge = new MfaPhoneChallenge();
await mfaPhoneChallenge.continue({ type: 'sms' });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaPhoneChallenge from '@auth0/auth0-acul-js/mfa-phone-challenge';

  const mfaPhoneChallenge = new MfaPhoneChallenge();
  mfaPhoneChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método envía la elección del usuario de recibir el código de MFA por SMS o por llamada de voz.

  ```typescript Example
  import MfaPhoneChallenge from '@auth0/auth0-acul-js/mfa-phone-challenge';

  const mfaPhoneChallenge = new MfaPhoneChallenge();
  await mfaPhoneChallenge.continue({ type: 'sms' });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaPhoneChallengeContinueOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/MfaPhoneChallengeContinueOptions).
    </ParamField>

    <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;" required>
      El método de entrega del código de MFA.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera del contexto el arreglo de errores de la transacción, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="pickPhone" type="Promise<void>">
  Este método inicia el proceso para que el usuario seleccione otro número de teléfono.

  ```typescript Example
  import MfaPhoneChallenge from '@auth0/auth0-acul-js/mfa-phone-challenge';

  const mfaPhoneChallenge = new MfaPhoneChallenge();
  await mfaPhoneChallenge.pickPhone();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario probar otro método de MFA.

  ```typescript Example
  import MfaPhoneChallenge from '@auth0/auth0-acul-js/mfa-phone-challenge';

  const mfaPhoneChallenge = new MfaPhoneChallenge();
  await mfaPhoneChallenge.tryAnotherMethod();
  ```
</ParamField>
