La clase ResetPasswordMfaEmailChallenge implementa la funcionalidad de la pantalla `reset-password-mfa-email-challenge`. Esta pantalla solicita al usuario que introduzca el código de verificación enviado a su dirección de correo electrónico como parte del paso de MFA durante un flujo de restablecimiento de contraseña.

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  resetPasswordMfaEmailChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método envía el código de verificación por correo electrónico para continuar con el desafío MFA durante el flujo de restablecimiento de contraseña.

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  await resetPasswordMfaEmailChallengeManager.continue({
    code: '123456',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ContinueOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código de verificación enviado a la dirección de correo electrónico del usuario.
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Indica si se debe recordar el dispositivo para futuros desafíos MFA.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción del contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Este método reenvía el código de verificación a la dirección de correo electrónico del usuario.

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  await resetPasswordMfaEmailChallengeManager.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Este método proporciona la funcionalidad de reenvío con gestión del tiempo de espera para la pantalla `reset-password-mfa-email-challenge`.

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  const { startResend } = resetPasswordMfaEmailChallengeManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // Llama a startResend cuando el usuario hace clic en el botón de reenvío
  startResend();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [StartResendOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La cantidad de segundos que se debe esperar antes de permitir que el usuario reenvíe el código.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Devolución de llamada que se invoca cada vez que cambia el estado del reenvío; devuelve los segundos restantes e indica si la funcionalidad de reenvío está deshabilitada en ese momento.
    </ParamField>

    <ParamField body="onTimeout?" type="void">
      Devolución de llamada que se activa cuando expira el tiempo de espera y el botón de reenvío vuelve a estar disponible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
