La clase MfaWebAuthnChangeKeyNickname implementa la funcionalidad de la pantalla `mfa-webauthn-change-key-nickname`. Esta pantalla se muestra cuando el usuario necesita cambiar el alias de su clave de seguridad de WebAuthn.

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla MFA WebAuthn Change Key Nickname:

```typescript Example
import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
await mfaWebAuthnChangeKeyNickname.continueWithNewNickname({ nickname: 'My YubiKey' });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

  const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
  mfaWebAuthnChangeKeyNickname.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueWithNewNickname" type="Promise<void>">
  Este método envía el nuevo alias de la clave de seguridad de WebAuthn del usuario.

  ```typescript Example
  import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

  const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
  await mfaWebAuthnChangeKeyNickname.continueWithNewNickname({ nickname: 'My YubiKey' });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaWebAuthnChangeKeyNicknameContinueOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/MfaWebAuthnChangeKeyNicknameContinueOptions).
    </ParamField>

    <ParamField body="nickname" type="string" required>
      El nuevo alias de la clave de seguridad de WebAuthn.
    </ParamField>
  </Expandable>
</ParamField>
