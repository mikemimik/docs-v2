La clase LoginPasswordlessEmailCode implementa la funcionalidad de la pantalla `login-passwordless-email-code`. Esta pantalla recopila el correo electrónico y el código del usuario.

<ParamField body="resendManager" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Este método proporciona la funcionalidad de reenvío con gestión del tiempo de espera. Devuelve un objeto [`ResendControl`](/es/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl) con un método `startResend()` que activa el reenvío y aplica un período de espera antes de que el usuario pueda volver a reenviar.

  ```typescript Example
  import LoginPasswordlessEmailCode from '@auth0/auth0-acul-js/login-passwordless-email-code';

  const loginPasswordlessEmailCodeManager = new LoginPasswordlessEmailCode();
  const { startResend } = loginPasswordlessEmailCodeManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Reenvío disponible en ${remainingSeconds}s, deshabilitado: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('El reenvío ya está disponible');
    }
  });

  // Llama a startResend cuando el usuario haga clic en el botón de reenvío
  startResend();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [StartResendOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La duración del período de espera en segundos antes de que el usuario pueda reenviar el código de nuevo.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Devolución de llamada que se invoca cada vez que cambia el estado del reenvío; devuelve los segundos restantes y si la funcionalidad de reenvío está deshabilitada en ese momento.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Devolución de llamada que se invoca cuando el período de espera expira y el reenvío vuelve a estar disponible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  Este método alterna entre una conexión de base de datos (contraseña) y una conexión passwordless (Email/SMS OTP).

  ```typescript Example
  import LoginPasswordlessEmailCode from '@auth0/auth0-acul-js/login-passwordless-email-code';
  const loginPasswordlessEmailCodeManager = new LoginPasswordlessEmailCode();
  loginPasswordlessEmailCodeManager.switchConnection({
    connection: 'email',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SwitchConnectionOptionsEmailCode](/es/docs/libraries/acul/js-sdk/Screens/interfaces/SwitchConnectionOptionsEmailCode).
    </ParamField>

    <ParamField body="connection" type="string" required>
      El nombre de la conexión a la que quieres cambiar. Usa `'email'` o `'sms'` para passwordless, o el nombre de una conexión de base de datos para la autenticación basada en contraseña.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
