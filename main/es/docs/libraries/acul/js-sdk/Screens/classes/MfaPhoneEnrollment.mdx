La clase MfaPhoneEnrollment implementa la funcionalidad de la pantalla `mfa-phone-enrollment`. Esta pantalla permite a los usuarios registrar un número de teléfono para usarlo con MFA.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de inscripción del teléfono de MFA:

```typescript Example
import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

const mfaPhoneEnrollment = new MfaPhoneEnrollment();
await mfaPhoneEnrollment.continueEnrollment({ phone: '+1234567890', type: 'sms' });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';
  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  mfaPhoneEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueEnrollment" type="Promise<void>">
  Este método continúa el proceso de inscripción con el número de teléfono y el tipo proporcionados (SMS o voz).

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.continueEnrollment({ phone: '+1234567890', type: 'sms' });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaPhoneEnrollmentContinueOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/MfaPhoneEnrollmentContinueOptions).
    </ParamField>

    <ParamField body="phone" type="string" required>
      El número de teléfono del usuario.
    </ParamField>

    <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;" required>
      El método de entrega del código de verificación.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  Este método navega a la pantalla de selección del código de país.

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.pickCountryCode();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario probar otro método de MFA.

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.tryAnotherMethod();
  ```
</ParamField>
