title: "PasskeyEnrollmentLocal"
description: "Describe todas las propiedades y los métodos disponibles para personalizar la pantalla `passkey-enrollment-local` de Universal Login."
---

La clase PasskeyEnrollmentLocal implementa la funcionalidad de la pantalla `passkey-enrollment-local`. Esta pantalla permite a los usuarios registrar una clave de acceso almacenada localmente en su dispositivo como método de autenticación.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla PasskeyEnrollmentLocal:

```typescript Example
import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
await passkeyEnrollmentLocalManager.continuePasskeyEnrollment();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="abortPasskeyEnrollment" type="Promise<void>">
  Este método cancela el proceso de inscripción de clave de acceso.

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  await passkeyEnrollmentLocalManager.abortPasskeyEnrollment({ doNotShowAgain: true });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [AbortEnrollmentOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/AbortEnrollmentOptions).
    </ParamField>

    <ParamField body="doNotShowAgain?" type="boolean">
      Cuando se establece en `true`, oculta el prompt de inscripción de clave de acceso en sesiones futuras.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  passkeyEnrollmentLocalManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continuePasskeyEnrollment" type="Promise<void>">
  Este método continúa el proceso de inscripción de clave de acceso.

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  await passkeyEnrollmentLocalManager.continuePasskeyEnrollment();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el context, o un arreglo vacío si no existe ninguno.
</ParamField>
