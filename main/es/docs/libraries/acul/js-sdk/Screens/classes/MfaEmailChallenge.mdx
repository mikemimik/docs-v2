La clase MfaEmailChallenge implementa la funcionalidad de la pantalla `mfa-email-challenge`. Esta pantalla se muestra cuando un usuario necesita verificar su correo electrónico durante la autenticación MFA.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de desafío de MFA por correo electrónico:

```typescript Example
import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';

const mfaEmailChallenge = new MfaEmailChallenge();
await mfaEmailChallenge.continue({
  code: '123456',
  rememberDevice: true
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';
  const mfaEmailChallenge = new MfaEmailChallenge();
  mfaEmailChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método continúa con el desafío por correo electrónico usando el código proporcionado.

  ```typescript Example
  import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';

  const mfaEmailChallenge = new MfaEmailChallenge();
  ```

    <ParamField body="code" type="string" required>

    <ParamField body="rememberDevice?" type="boolean">

<ParamField body="resendCode" type="Promise<void>">
  Este método reenvía el código por email.

  ```typescript Example
  import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';
  const mfaEmailChallenge = new MfaEmailChallenge();
  await mfaEmailChallenge.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Este método proporciona la funcionalidad de reenvío con gestión del tiempo de espera para esta pantalla.

  ```typescript Example
  import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';

  const mfaEmailChallenge = new MfaEmailChallenge();
  const { startResend } = mfaEmailChallenge.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Reenvío disponible en ${remainingSeconds}s, deshabilitado: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('El reenvío ya está disponible');
    }
  });

  // Llama a startResend cuando el usuario haga clic en el botón de reenvío
  startResend();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [StartResendOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La duración, en segundos, del período de espera antes de que el usuario pueda reenviar el código de nuevo.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Devolución de llamada que se invoca cada vez que cambia el estado del reenvío; devuelve los segundos restantes e indica si la funcionalidad de reenvío está deshabilitada en ese momento.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Devolución de llamada que se invoca cuando expira el período de espera y el reenvío vuelve a estar disponible.
    </ParamField>
  </Expandable>
</ParamField>
