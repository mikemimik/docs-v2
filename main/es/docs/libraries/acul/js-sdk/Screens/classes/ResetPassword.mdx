title: "ResetPassword"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `reset-password` de Universal Login."
---

La clase ResetPassword implementa la funcionalidad de la pantalla `reset-password`. Esta pantalla permite que el usuario establezca una nueva contraseña después de verificar su identidad.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla ResetPassword:

```typescript Example
import ResetPassword from '@auth0/auth0-acul-js/reset-password';
const resetPasswordManager = new ResetPassword();
resetPasswordManager.resetPassword({
  'password-reset': 'Test@123!',
  're-enter-password': 'Test@123!',
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import ResetPassword from '@auth0/auth0-acul-js/reset-password';
  const resetPasswordManager = new ResetPassword();
  resetPasswordManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de idioma o locale deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no hay ninguno.
</ParamField>

<ParamField body="resetPassword" type="Promise<void>">
  Este método envía la nueva contraseña del usuario para completar el flujo de restablecimiento de contraseña.

  ```typescript Example
  import ResetPassword from '@auth0/auth0-acul-js/reset-password';
  const resetPasswordManager = new ResetPassword();
  resetPasswordManager.resetPassword({
    'password-reset': 'Test@123!',
    're-enter-password': 'Test@123!',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordOptions).
    </ParamField>

    <ParamField body="password-reset" type="string" required>
      La nueva contraseña ingresada por el usuario.
    </ParamField>

    <ParamField body="re-enter-password" type="string" required>
      Confirmación de la nueva contraseña. Debe coincidir con `password-reset`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="validatePassword" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/PasswordValidationResult">PasswordValidationResult</a></span>}>
  Este método valida una contraseña según la política de contraseñas del tenant y devuelve un resultado de validación.

  ```typescript Example
  import ResetPassword from '@auth0/auth0-acul-js/reset-password';
  const resetPasswordManager = new ResetPassword();
  resetPasswordManager.validatePassword('MyP@ssw0rd!');
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="password" type="string" required>
      La contraseña que se debe validar.
    </ParamField>
  </Expandable>
</ParamField>
