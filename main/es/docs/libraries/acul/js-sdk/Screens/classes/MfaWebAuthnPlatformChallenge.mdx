---

La clase MfaWebAuthnPlatformChallenge implementa la funcionalidad de la pantalla `mfa-webauthn-platform-challenge`. Esta pantalla se muestra cuando el usuario necesita verificar su identidad mediante un autenticador de plataforma, como Touch ID o Face ID.

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla de desafío de la plataforma WebAuthn para MFA:

```typescript Example
import MfaWebAuthnPlatformChallenge from '@auth0/auth0-acul-js/mfa-webauthn-platform-challenge';

const mfaWebAuthnPlatformChallenge = new MfaWebAuthnPlatformChallenge();
await mfaWebAuthnPlatformChallenge.verify();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaWebAuthnPlatformChallenge from '@auth0/auth0-acul-js/mfa-webauthn-platform-challenge';

  const mfaWebAuthnPlatformChallenge = new MfaWebAuthnPlatformChallenge();
  mfaWebAuthnPlatformChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de transacción del contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="reportBrowserError" type="Promise<void>">
  Este método informa un error del navegador detectado durante la operación `navigator.credentials.get()` de WebAuthn.

  ```typescript Example
  import MfaWebAuthnPlatformChallenge from '@auth0/auth0-acul-js/mfa-webauthn-platform-challenge';

  const mfaWebAuthnPlatformChallenge = new MfaWebAuthnPlatformChallenge();
  await mfaWebAuthnPlatformChallenge.reportBrowserError({
    error: { name: 'NotAllowedError', message: 'The operation was not allowed.' },
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaWebAuthnPlatformChallengeReportErrorOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/MfaWebAuthnPlatformChallengeReportErrorOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      El objeto de error de la API de WebAuthn (`navigator.credentials.get()`) que se va a informar.

      <ParamField body="name" type="string" required>
        El nombre del error (por ejemplo, `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        El mensaje de error que describe lo que salió mal.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario probar otro método de MFA.

  ```typescript Example
  import MfaWebAuthnPlatformChallenge from '@auth0/auth0-acul-js/mfa-webauthn-platform-challenge';

  const mfaWebAuthnPlatformChallenge = new MfaWebAuthnPlatformChallenge();
  await mfaWebAuthnPlatformChallenge.tryAnotherMethod();
  ```
</ParamField>

<ParamField body="verify" type="Promise<void>">
  Este método inicia el desafío del autenticador de plataforma de WebAuthn.

  ```typescript Example
  import MfaWebAuthnPlatformChallenge from '@auth0/auth0-acul-js/mfa-webauthn-platform-challenge';

  const mfaWebAuthnPlatformChallenge = new MfaWebAuthnPlatformChallenge();
  await mfaWebAuthnPlatformChallenge.verify();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [CustomOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Cuando es `true`, recuerda el navegador para futuros desafíos MFA. Solo se aplica si `screen.showRememberDevice` es `true`.
    </ParamField>
  </Expandable>
</ParamField>
