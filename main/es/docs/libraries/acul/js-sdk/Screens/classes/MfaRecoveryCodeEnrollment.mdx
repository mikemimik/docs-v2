---

La clase MfaRecoveryCodeEnrollment implementa la funcionalidad de la pantalla `mfa-recovery-code-enrollment`. Esta pantalla se muestra cuando el usuario necesita inscribir un código de recuperación para MFA.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de inscripción del código de recuperación de MFA:

```typescript Example
import MfaRecoveryCodeEnrollment from '@auth0/auth0-acul-js/mfa-recovery-code-enrollment';

const mfaRecoveryCodeEnrollment = new MfaRecoveryCodeEnrollment();
await mfaRecoveryCodeEnrollment.continue({ isCodeCopied: true });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaRecoveryCodeEnrollment from '@auth0/auth0-acul-js/mfa-recovery-code-enrollment';

  const mfaRecoveryCodeEnrollment = new MfaRecoveryCodeEnrollment();
  mfaRecoveryCodeEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método indica que el usuario ha guardado el código de recuperación y continúa el flujo de inscripción.

  ```typescript Example
  import MfaRecoveryCodeEnrollment from '@auth0/auth0-acul-js/mfa-recovery-code-enrollment';

  const mfaRecoveryCodeEnrollment = new MfaRecoveryCodeEnrollment();
  await mfaRecoveryCodeEnrollment.continue({ isCodeCopied: true });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaRecoveryCodeEnrollmentContinueOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/MfaRecoveryCodeEnrollmentContinueOptions).
    </ParamField>

    <ParamField body="isCodeCopied?" type="boolean">
      Indica si el usuario ha copiado el código de recuperación.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción del contexto, o un arreglo vacío si no existe ninguno.
</ParamField>
