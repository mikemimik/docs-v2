**Lock v2 para iOS** te permite especificar campos adicionales que el usuario debe completar antes de crear una cuenta nueva. Estos campos se mostrarán después de los campos básicos (correo electrónico, nombre de usuario, contraseña).

<div id="adding-custom-fields">
  ## Agregar campos personalizados
</div>

Al registrarse, la información obligatoria predeterminada es el correo electrónico y la contraseña del usuario. Puede ampliar los datos que desea recopilar según sea necesario. Los campos adicionales que se recopilen aquí durante el registro se almacenarán en `user_metadata`. Puede obtener más información en [Metadata](/es/docs/manage-users/user-accounts/metadata).

```swift lines
.withOptions {
  $0.customSignupFields = [
    CustomTextField(name: "first_name", placeholder: "First Name", icon: LazyImage(name: "ic_person", bundle: Lock.bundle)),
    CustomTextField(name: "last_name", placeholder: "Last Name", icon: LazyImage(name: "ic_person", bundle: Lock.bundle))
  ]
}
```

Debes especificar el icono que se usará con tu campo de texto personalizado.

¡Eso es todo! Si has habilitado el registro de usuarios en el Dashboard de la aplicación, después de que completen los campos básicos (correo electrónico/nombre de usuario, contraseña) y hagan clic en Enviar, se les pedirá que completen los campos restantes. El usuario debe completar todos los campos personalizados antes de poder finalizar el registro.

Cuando se solicite el registro o el inicio de sesión de un usuario, los campos adicionales se adjuntarán al atributo `user_metadata` en el cuerpo de la solicitud. Puedes acceder a ellos consultando el perfil del usuario en cualquier momento, incluso desde la sección Users del Dashboard.
