**Auth0.Android** incluye la clase `WebAuthProvider` para gestionar el flujo de autorización mediante el navegador. Pero ¿qué pasa si quieres usar tu propia implementación o una versión Native de un `AuthProvider`?

<div id="the-authprovider-class">
  ## La clase AuthProvider
</div>

Crea una clase que implemente la interfaz `AuthProvider` y sobrescriba sus métodos.

También puedes usar cualquiera de las implementaciones **Native** que Auth0 ya proporciona. Los proveedores de [autenticación social nativa](/es/docs/libraries/lock-android/lock-android-native-social-authentication) disponibles actualmente son Google y Facebook.
