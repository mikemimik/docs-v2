Authentication API proporciona métodos para autenticar y registrar usuarios de bases de datos.

<div id="signing-up-with-a-database-connection">
  ## Registro con una conexión de base de datos
</div>

Para registrarse, debes llamar al método `createUser` y pasar el `email`, la `password` y el nombre de la `conexión` del usuario para iniciar el proceso de registro. También puedes especificar metadatos adicionales del usuario para almacenarlos.

```swift lines
Auth0
    .authentication()
    .createUser(
        email: "support@auth0.com",
        password: "secret-password",
        connection: "Username-Password-Authentication",
        userMetadata: ["first_name": "First",
                       "last_name": "Last"])
    .start { result in
        switch result {
        case .success(let user):
            print("User: \(user)")
        case .failure(let error):
            print("Failed with \(error)")
        }
    }
```

<div id="login-with-a-database-connection">
  ## Iniciar sesión con una conexión de base de datos
</div>

Para iniciar sesión con una conexión de base de datos, debes llamar a `login` con el nombre de usuario/correo electrónico del usuario, la contraseña y el nombre de la conexión (como `Username-Password-Authentication`) con la que quieres autenticarte. La respuesta será un objeto Credentials.

```swift lines
Auth0
    .authentication()
    .login(
        usernameOrEmail: "support@auth0.com",
        password: "secret-password",
        realm: "Username-Password-Authentication",
        scope: "openid")
     .start { result in
         switch result {
         case .success(let credentials):
            print("Credentials: \(credentials)")
         case .failure(let error):
            print("Failed with \(error)")
         }
     }
```
