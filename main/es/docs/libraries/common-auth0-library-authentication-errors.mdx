title: Errores comunes de autenticación en las bibliotecas de Auth0
---

Las acciones de tus usuarios o los datos que introducen durante los procesos de registro o de inicio de sesión pueden provocar errores. Aquí tienes una lista de los errores más comunes que puedes encontrar si utilizas cualquiera de las bibliotecas de Auth0 para la autenticación.

<div id="sign-up">
  ## Registrarse
</div>

En caso de que falle el registro, los errores más comunes son:

| **Error**                                       | **Descripción**                                                                                       |
| ----------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| **invalid&#95;password**                        | Si la contraseña utilizada no cumple con la política de contraseñas de la conexión                    |
| **invalid&#95;signup**                          | El usuario que intentas registrar no es válido                                                        |
| **password&#95;dictionary&#95;error**           | La contraseña elegida es demasiado común                                                              |
| **password&#95;no&#95;user&#95;info&#95;error** | La contraseña elegida se basa en información del usuario                                              |
| **password&#95;strength&#95;error**             | La [contraseña elegida es demasiado débil](/es/docs/authenticate/database-connections/password-strength) |
| **unauthorized**                                | Si no puedes registrarte en esta aplicación. Puede deberse a la infracción de una rule específica     |
| **user&#95;exists**                             | El usuario que intentas registrar ya se ha registrado                                                 |
| **username&#95;exists**                         | El nombre de usuario con el que intentas registrarte ya está en uso                                   |

<div id="log-in">
  ## Iniciar sesión
</div>

En caso de un inicio de sesión fallido, los errores más comunes son:

| **Error**                             | **Descripción**                                                                                                                                                                                                                                                                                                                                                                               |
| ------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **access&#95;denied**                 | Al usar autenticación basada en web, el servidor de recursos deniega el acceso según las especificaciones de OAuth2                                                                                                                                                                                                                                                                           |
| **invalid&#95;user&#95;password**     | El nombre de usuario y/o la contraseña usados para la autenticación no son válidos                                                                                                                                                                                                                                                                                                            |
| **mfa&#95;invalid&#95;code**          | El <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuarios que utiliza un factor adicional al nombre de usuario y la contraseña, como un código enviado por SMS." cta="Ver glosario" href="/es/docs/glossary?term=multi-factor+authentication+%28MFA%29">código de autenticación multifactor (MFA)</Tooltip> proporcionado por el usuario no es válido o ha caducado |
| **mfa&#95;registration&#95;required** | El administrador ha exigido la [autenticación multifactor](/es/docs/secure/multi-factor-authentication), pero el usuario no se ha registrado                                                                                                                                                                                                                                                     |
| **mfa&#95;required**                  | El usuario debe proporcionar el código de [autenticación multifactor](/es/docs/secure/multi-factor-authentication) para autenticarse                                                                                                                                                                                                                                                             |
| **password&#95;leaked**               | Si la contraseña se ha filtrado y es necesario usar otra                                                                                                                                                                                                                                                                                                                                      |
| **PasswordHistoryError**              | La contraseña proporcionada para registrarse o actualizarla ya se ha utilizado (se muestra cuando la función de [historial de contraseñas](/es/docs/authenticate/database-connections/password-options#password-history) está habilitada)                                                                                                                                                        |
| **PasswordStrengthError**             | La contraseña proporcionada no cumple con los [requisitos de seguridad](/es/docs/authenticate/database-connections/password-strength) de la conexión                                                                                                                                                                                                                                             |
| **too&#95;many&#95;attempts**         | La cuenta está bloqueada por haber superado el número máximo de intentos de inicio de sesión                                                                                                                                                                                                                                                                                                  |
| **unauthorized**                      | El usuario con el que intentas iniciar sesión está bloqueado                                                                                                                                                                                                                                                                                                                                  |
