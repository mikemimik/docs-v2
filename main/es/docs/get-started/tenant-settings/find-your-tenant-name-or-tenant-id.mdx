---
description: Cómo encontrar el nombre de tu tenant de Auth0
title: Encuentra el nombre o el ID de tu tenant
---

Cada tenant de Auth0 tiene un nombre. Usas este nombre con frecuencia en tu código para identificar el tenant. Cuando veas la variable `{yourTenantId}`, `{yourTenantName}`, `{yourTenant}` o alguna similar en la documentación de Auth0, reemplázala por el nombre del tenant.

1. Ve al [Auth0 Dashboard](https://manage.auth0.com/#).
2. En la esquina superior izquierda, busca el nombre del tenant junto al logotipo de Auth0.
