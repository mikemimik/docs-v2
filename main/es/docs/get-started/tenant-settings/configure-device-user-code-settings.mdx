title: Configurar los ajustes del código de usuario para dispositivos
---

Puedes configurar los ajustes del código de usuario generado por tu aplicación durante el [flujo de autorización de dispositivos](/es/docs/get-started/authentication-and-authorization-flow/device-authorization-flow) mediante el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Ve a [Dashboard &gt; Settings](https://manage.auth0.com/#/tenant) y haz clic en la pestaña **Advanced**.
2. Desplázate hasta la sección **Device Flow User Code Format**, busca **User Code Character Set** y **User Code Mask**, introduce la configuración deseada y haz clic en **Save**.

| Setting                          | Description                                                                                                                                                                                             |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Device Flow User Code Format** | Conjunto de caracteres que se utiliza para generar aleatoriamente un código de usuario.                                                                                                                 |
| **User Code Mask**               | Máscara que se utiliza para definir la longitud y el formato de un código de usuario generado aleatoriamente. Su propósito es mejorar la legibilidad del código de usuario y facilitar su introducción. |
