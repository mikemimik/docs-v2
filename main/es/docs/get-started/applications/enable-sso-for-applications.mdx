---
description: Describe cómo crear una integración de inicio de sesión único (SSO) para una aplicación. Solo para usar con tenants heredados.
title: Habilitar integraciones de inicio de sesión único para aplicaciones
---

De forma predeterminada, el <Tooltip tip="Inicio de sesión único (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, lo inicia automáticamente en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=single+sign-on">inicio de sesión único</Tooltip> (SSO) sin fricción está habilitado para todos los nuevos tenants de Auth0; sin embargo, los **tenants heredados** pueden [elegir si habilitar esta función a nivel de tenant](/docs/es/get-started/tenant-settings/enable-sso-for-legacy-tenants). Si no ha habilitado el SSO a nivel de tenant, puede habilitarlo por aplicación.

Puede habilitar el SSO para su aplicación mediante el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="prerequisite">
  ## Requisito previo
</div>

Antes de habilitar SSO para una aplicación, primero debes crear y configurar una conexión para cada <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=identity+provider">proveedor de identidad</Tooltip> que quieras usar. En el caso de los proveedores de identidad sociales, asegúrate de que la conexión no use claves de desarrollador si utilizas la [experiencia de Universal Login](/docs/es/authenticate/login/auth0-universal-login).

<div id="enable-sso">
  ## Habilitar SSO
</div>

1. Ve a [Auth0 Dashboard &gt; Applications &gt; SSO Integrations](https://manage.auth0.com/#/externalapps) y selecciona el nombre de la integración que quieres ver.
2. Desplázate hasta la parte inferior de la página **Settings**, busca **Use Auth0 instead of the IdP to do inicio de sesión único**, activa el interruptor y selecciona **Save Changes**.
