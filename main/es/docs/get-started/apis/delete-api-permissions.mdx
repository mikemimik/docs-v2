Puedes eliminar permisos de una API desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o con la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>.

<div id="use-the-dashboard">
  ## Use el Dashboard
</div>

1. Vaya a [Dashboard &gt; Applications &gt; APIs](https://manage.auth0.com/#/apis) y haga clic en el nombre de la API que desea ver.

   <Frame>![Lista de APIs en Applications del Dashboard](/docs/images/cdy7uua7fh8z/3rhmhghYZDSi6YWHRA5yMQ/c71340259481b0b6787d5f3887cfda0f/dashboard-apis-list.png)</Frame>
2. Vaya a la pestaña **Permissions**, haga clic en el icono de la papelera junto al permiso que desea eliminar y confirme.

   <Frame>![Pantalla Define Permissions de Agregar permisos de API para una API en Applications del Dashboard](/docs/images/cdy7uua7fh8z/32Pb185OFs2mC6z2fmunEw/2708c52c2869d016066cc456dd00b6a2/dashboard-applications-apis-permissions.png)</Frame>

<div id="use-the-management-api">
  ## Usa la Management API
</div>

Realiza una llamada PATCH al [endpoint Update Resource Server](https://auth0.com/docs/api/management/v2#!/Resource_Servers/patch_resource_servers_by_id) que incluya todos los permisos que quieras conservar y excluya todos los permisos que quieras eliminar. Sustituye los valores de marcador de posición `API_ID,` `MGMT_API_ACCESS_TOKEN`, `PERMISSION_NAME` y `PERMISSION_DESC` por el ID de tu API, el <Tooltip tip="Access Token: credencial de Authorization, en forma de una cadena opaca o JWT, que se utiliza para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=Access+Token">Access Token</Tooltip> de la Management API, los nombres de los permisos y las descripciones de los permisos, respectivamente.

</AuthCodeGroup>

| Valor                   | Descripción                                                                                                                    |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `API_ID`                | ID de la API para la que quieres eliminar permisos.                                                                            |
| `MGMT_API_ACCESS_TOKEN` | [Access Token for the Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `update:resource_servers`. |
| `PERMISSION_NAME`       | Nombre o nombres de los permisos que quieres conservar para la API especificada.                                               |
| `PERMISSION_DESC`       | Descripción o descripciones fáciles de entender de los permisos que quieres conservar para la API especificada.                |
