description: Describe cómo crear un tenant adicional con Auth0 Dashboard.
title: Crear varios tenants
---

Puedes configurar varios tenants para crear [distintos entornos](/es/docs/get-started/auth0-overview/create-tenants/set-up-multiple-environments) en el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> y así permitir configuraciones complejas. Por ejemplo, podrías tener dos dominios independientes (uno interno y otro público), o puede que quieras que los usuarios inicien sesión de forma diferente según la aplicación. Para lograrlo, crea más de un tenant de Auth0, lo que te permitirá tener conjuntos independientes de aplicaciones, conexiones y usuarios para las aplicaciones y los grupos de usuarios que necesites admitir.

1. Ve al [Auth0 Dashboard](https://manage.auth0.com/#/), selecciona el nombre de tu tenant y luego **Create Tenant**.

   <Frame>![Menú desplegable del tenant en Dashboard: Create Tenant](/docs/images/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/5efad1137eb8b04031b30137c2fc5ac5/2025-02-26_14-51-39.png)</Frame>
2. Introduce el **Tenant Domain** que quieras, selecciona una **Region** y luego **Create**.
