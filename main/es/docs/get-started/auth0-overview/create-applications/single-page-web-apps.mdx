Para integrar Auth0 con una aplicación web de una sola página, primero debes registrar tu aplicación en Auth0 desde el Dashboard.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y haz clic en **Create Application**.
2. Introduce un nombre descriptivo para tu aplicación, selecciona **aplicación web de página única** y haz clic en **Create**.

   Una vez creada la nueva aplicación, puedes ver la página **configuración de la aplicación**, que incluye las siguientes pestañas:

   | Pestaña de configuración | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                  |
   | ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Quick Start**          | Muestra toda la documentación disponible para tu tipo de aplicación.                                                                                                                                                                                                                                                                                                                                                         |
   | **Settings**             | Muestra toda la configuración disponible para tu aplicación. De forma predeterminada, la mayoría de los ajustes se crean automáticamente. Para obtener más información, consulta [configuración de la aplicación](/es/docs/get-started/applications/application-settings).                                                                                                                                                                |
   | **Credentials**          | Muestra el método de autenticación de la aplicación y las credenciales configuradas. Para obtener más información, consulta [Application Credentials](/es/docs/secure/application-credentials)                                                                                                                                                                                                                                  |
   | **Add-ons**              | Te permite habilitar complementos asociados a una aplicación. Normalmente, se usan para integraciones de SAML o WS-Fed, que no suelen ser necesarias para las SPA. Para obtener más información, consulta [Enable SAML2 Web App Addon](/es/docs/authenticate/protocols/saml/saml-sso-integrations/enable-saml2-web-app-addon) y [Configure WS-Fed Applications](/es/docs/get-started/applications/update-application-connections). |
   | **Connections**          | Te permite habilitar conexiones para tu aplicación. Las conexiones son fuentes de usuarios; se pueden habilitar y compartir entre varias aplicaciones. Para obtener más información, consulta [Connections](/es/docs/authenticate/identity-providers).                                                                                                                                                                          |
   | **Organizations**        | Te permite autenticar a los usuarios en las organizaciones a las que pertenecen. Para obtener más información, consulta [Organizations](/es/docs/manage-users/organizations).                                                                                                                                                                                                                                                   |
   | **Login Experience**     | Te permite configurar la experiencia de inicio de sesión de los usuarios de tu organización. Para obtener más información, consulta [Understand how Auth0 Organizations Work](/es/docs/manage-users/organizations/organizations-overview#login-experience).                                                                                                                                                                     |

   Para obtener más información, consulta [configuración de la aplicación](/es/docs/get-started/applications/application-settings).

<div id="next-steps">
  ## Próximos pasos
</div>

Una vez que hayas registrado tu aplicación, revisa y completa estos pasos de configuración:

* Configura al menos una conexión (como una conexión de base de datos o un inicio de sesión con redes sociales) y habilítala para tu aplicación. Sin una conexión habilitada, los usuarios no podrán autenticarse.
* Modifica el código de tu aplicación para usar la aplicación que registraste en Auth0. Revisa nuestros [Quickstarts de Auth0](/es/docs/quickstarts), donde encontrarás instrucciones detalladas sobre cómo implementar el inicio y cierre de sesión, gestionar las sesiones de tus usuarios, obtener y mostrar información del perfil de usuario, y mucho más.
* Usa las [API de Auth0](/es/docs/api).

  * La [Authentication API](https://auth0.com/docs/api/authentication) se encarga de todas las funciones principales relacionadas con la identidad (por ejemplo, iniciar sesión, cerrar sesión y obtener el perfil del usuario). La mayoría de los usuarios usan esta API a través de nuestros Quickstarts, el [SDK SPA de Auth0](/es/docs/libraries/auth0-spa-js) o el [widget Lock](/es/docs/libraries/lock). Para aplicaciones heredadas, consulta la [biblioteca Auth0.js](/es/docs/libraries/auth0js). Sin embargo, si estás creando manualmente toda la interfaz de usuario de autenticación, tendrás que interactuar directamente con esta API.
  * La [Management API](https://auth0.com/docs/api/management/v2) te permite automatizar varias tareas a las que también puedes acceder desde el Dashboard de Auth0 (por ejemplo, crear usuarios o configurar tipos de concesión de aplicación).
