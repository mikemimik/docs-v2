---
description: Aprende a editar los roles de usuario en Auth0 Dashboard para tu tenant de Auth0.
title: Editar miembros del tenant
---

Si eres administrador del tenant, puedes cambiar los roles asignados a un usuario. Los cambios surtirán efecto cuando los usuarios actualicen su Dashboard.

Puedes asignar más de un rol a un usuario y los permisos se combinarán. Primero debes haber agregado al miembro del tenant.

1. Ve a [Dashboard &gt; Settings &gt; Tenant Members](https://manage.auth0.com/#/tenant/admins).
2. Haz clic en **...** a la derecha del nombre o correo electrónico de un usuario y luego en **Edit Roles**.
   Si no ves esa opción, significa que el usuario todavía no ha aceptado su invitación.
3. Selecciona roles adicionales o elimina roles.
4. Haz clic en **Save**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Los permisos del usuario se mostrarán cuando actualice su Dashboard. Si el usuario no lo actualiza, puede encontrarse con errores que indiquen que no tiene acceso a un área específica. Si ocurre uno de esos errores, el usuario tendrá que cerrar sesión y volver a iniciar sesión.
</Callout>

**Restablecimiento de contraseña**: Para obtener más información, consulta [Reset Auth0 Account Password](/es/docs/troubleshoot/customer-support/reset-account-passwords) si tú o alguno de los usuarios de tu Dashboard tiene problemas para iniciar sesión.

**Dirección de correo electrónico no válida**: Para obtener más información, consulta [Update Dashboard User Email Addresses](/es/docs/get-started/manage-dashboard-access/update-dashboard-user-email) para ver las direcciones de correo electrónico de los miembros de tu tenant y actualizarlas si es necesario.
