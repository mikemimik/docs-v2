El servidor MCP de Auth0 conecta a los agentes de IA con tu tenant de Auth0, lo que les permite realizar operaciones complejas de varios pasos en Auth0, como crear aplicaciones, administrar usuarios o implementar Actions. Aprovecha el [Model Context Protocol (MCP)](https://modelcontextprotocol.io/introduction), un protocolo estandarizado que permite a los agentes de IA comunicarse con herramientas y API externas para ejecutar tareas en tu nombre.

Diseñado para desarrolladores que usan herramientas impulsadas por IA como Cursor, Windsurf o Claude Desktop, el servidor MCP de Auth0 transforma la forma en que los desarrolladores gestionan su infraestructura de identidad, y hace que trabajar con los servicios de Auth0 sea más rápido y sencillo.

<div id="authentication-and-credential-security">
  ## Autenticación y seguridad de las credenciales
</div>

El servidor MCP de Auth0 prioriza una seguridad sólida al gestionar la autenticación, validar las solicitudes y garantizar una comunicación segura con la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> de Auth0. Esto incluye:

* [Flujo de autorización de dispositivos de OAuth 2.0](/docs/es/get-started/authentication-and-authorization-flow/device-authorization-flow): Autentica las solicitudes con Auth0 de forma segura.
* [Almacenamiento seguro de credenciales](/docs/es/get-started/applications/rotate-credentials): Las credenciales se almacenan de forma segura en el llavero de tu sistema y nunca se exponen en texto sin formato.
* [Permisos mínimos de API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens): Solo se solicitan los permisos esenciales, en estricto cumplimiento del principio de mínimo privilegio.

<div id="supported-tools-and-integrations">
  ## Herramientas e integraciones compatibles
</div>

El servidor MCP de Auth0 actualmente se integra con los siguientes [clientes MCP](https://modelcontextprotocol.io/clients), entre ellos:

* [Claude Desktop](https://claude.ai/download)
* [Cursor](https://www.cursor.com/downloads)
* [Windsurf](https://www.windsurf.com/editor)

Se añadirán integraciones adicionales a medida que vayan surgiendo nuevos clientes relevantes. Para consultar la lista completa de herramientas compatibles y obtener información detallada sobre sus funciones, consulta nuestra [Referencia de herramientas del servidor MCP de Auth0](/docs/es/get-started/auth0-mcp-server/auth0-mcp-tools-reference).
