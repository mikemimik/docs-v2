<div id="endpoint">
  ## Endpoint
</div>

`POST /oauth/ro`

Con las credenciales del usuario, este endpoint autentica al usuario con el proveedor y devuelve un objeto JSON con el token de acceso y un ID Token.

<Note>
  Este endpoint forma parte del flujo de autenticación heredado y ha sido reemplazado por [Password Grant](#resource-owner-password). Para obtener más información sobre el flujo de autenticación más reciente, consulta [Introducing OIDC Conformant Authentication](/api-auth/intro).
</Note>

<div id="remarks">
  ### Observaciones
</div>

* Este endpoint solo funciona con Database Connections, conexiones Passwordless, Active Directory/LDAP, Windows Azure AD y ADFS.
* El `scope` `profile` solicita acceso a los Claims predeterminados del perfil del usuario final, incluidos `name`, `family_name`, `given_name` y otros.
* El `scope` `email` solicita acceso a los Claims `email` y `email_verified`.
