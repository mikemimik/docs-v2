---
title: "Obtener los metadatos de WS-Federation"
description: "Devuelve los metadatos de WS-Federation del tenant de Auth0."
---

<div id="endpoint">
  ## Endpoint
</div>

`GET /wsfed/FederationMetadata/2007-06/FederationMetadata.xml`

Este endpoint devuelve los metadatos de WS-Federation.

  ### Más información
