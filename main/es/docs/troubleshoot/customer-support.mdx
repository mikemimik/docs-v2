<div id="other-support-resources">
  ## Otros recursos de soporte
</div>

* La [página de Auth0 Status](https://status.auth0.com) contiene información sobre el estado actual del entorno de producción. Consulta la página de estado antes de enviar una solicitud de soporte.

  Si la página de estado muestra una notificación sobre una interrupción del servicio, nuestro equipo ya estará trabajando para restablecerlo y mantendrá actualizada la página de estado. Publicamos un análisis de la causa raíz en la página de estado una vez que hayamos investigado el problema. Puedes suscribirte para recibir notificaciones de cualquier cambio.

* El [Auth0 Support Center oficial](https://support.auth0.com/) ofrece información de resolución de problemas y soluciones para muchos problemas habituales.

* La [comunidad pública de preguntas y respuestas](https://community.auth0.com/) de Auth0 ofrece soporte para todos. Puedes buscar entre las preguntas existentes y publicar una nueva si la tuya aún no se ha planteado. No se garantizan tiempos de respuesta para las preguntas publicadas.
