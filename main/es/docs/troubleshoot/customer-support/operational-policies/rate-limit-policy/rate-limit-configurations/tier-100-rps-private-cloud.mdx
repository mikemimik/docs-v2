title: Private Cloud Basic 100 RPS (1x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  En general, se admite un rendimiento sostenido total de hasta 500 RPS entre Authentication API y MyAccount API en un entorno; sin embargo, determinadas cargas de trabajo específicas, un uso combinado intensivo de Authentication API y Management API, o determinados flujos de autenticación, como ROPG en Authentication API, pueden provocar una degradación del rendimiento con valores de RPS más bajos.
</Callout>

Consulta a continuación los límites de frecuencia del tipo de suscripción Private Cloud Basic 100 RPS (1x). Estos límites se aplican a cada tenant que crees en el entorno de Private Cloud.

Por lo tanto, recomendamos desplegar un tenant por entorno de Private Cloud para mitigar riesgos.
