title: Rendimiento de Private Cloud 10.000 RPS (100x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele ser compatible hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, el uso combinado intensivo de Authentication API y Management API, o determinados flujos de autenticación como ROPG con Authentication API, pueden provocar una degradación del rendimiento incluso con menos RPS.
</Callout>

A continuación, consulta los límites de frecuencia del tipo de suscripción Private Cloud Performance 10.000 RPS (100x).

Por lo tanto, recomendamos implementar un tenant por entorno de Private Cloud para mitigar riesgos.
