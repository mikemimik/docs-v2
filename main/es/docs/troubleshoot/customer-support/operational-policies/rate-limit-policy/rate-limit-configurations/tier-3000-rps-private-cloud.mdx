title: Rendimiento de Private Cloud 3000 RPS (30x) y 3000 RPS (30x) Burst
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele estar admitido hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, un uso combinado intensivo de Authentication API y Management API, o determinados Auth Flows como ROPG en Authentication API, pueden provocar una degradación del rendimiento incluso con un RPS inferior.
</Callout>

Consulte a continuación los límites de frecuencia de los tipos de suscripción Private Cloud Performance 3000 RPS (30x) y 3000 RPS (30x) Burst. Estos límites se aplican a cada tenant que cree en el entorno de Private Cloud.

Por lo tanto, recomendamos implementar un tenant por cada entorno de Private Cloud para mitigar riesgos.
