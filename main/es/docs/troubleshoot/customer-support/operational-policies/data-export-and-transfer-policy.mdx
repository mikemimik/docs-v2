<div id="export-data">
  ## Exportar datos
</div>

Si quieres exportar tus datos de Auth0, puedes hacerlo de varias maneras. Ten en cuenta que estas herramientas no exportan los hashes de contraseña de los usuarios de tu base de datos alojada en Auth0. Aun así, puedes solicitar esta información abriendo un [ticket de soporte](https://support.auth0.com/). Si has decidido dejar de usar nuestro servicio, consulta [Exportar datos](/docs/es/troubleshoot/customer-support/manage-subscriptions/export-data) para obtener más información.

<div id="use-the-importexport-extension">
  ### Usa la extensión Import/Export
</div>

Puedes usar nuestra [extensión Import/Export](/docs/es/manage-users/user-migration/user-import-export-extension) para exportar casi todos los datos de tu tenant de Auth0.

<div id="use-the-management-api">
  ### Usa la Management API
</div>

Si quieres exportar ciertos conjuntos de datos de forma programática, usa el endpoint Bulk User Export de la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<div id="transfer-data">
  ## Transferencia de datos
</div>

Auth0 no transferirá datos de un tenant de Auth0 a otro. Esto se aplica tanto a los clientes de Public Cloud como de Private Cloud.

Todos los datos de tu tenant de Auth0 están siempre bajo tu control y disponibles a través de la Management API en cualquier momento. La única información que no está disponible a través de la API son los hash de contraseña de los usuarios de tu base de datos alojada en Auth0 y las claves privadas, por motivos de seguridad.
