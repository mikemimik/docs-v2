Para obtener más información sobre los triggers de registro e inicio de sesión, consulta las secciones a continuación.

| Trigger                                                                                                                                  | Descripción                                                                                                                                     |
| ---------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| [Trigger de inicio de sesión](/es/docs/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)                          | Los triggers de inicio de sesión se ejecutan cuando un usuario inicia sesión en cualquier aplicación de un tenant de Auth0.                     |
| [Trigger de pre-registro de usuario](/es/docs/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger)   | El trigger de pre-registro de usuario se ejecuta antes de que se agregue un usuario a una conexión de base de datos o conexión Passwordless.    |
| [Trigger de post-registro de usuario](/es/docs/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | El trigger de post-registro de usuario se ejecuta después de que se agregue un usuario a una conexión de base de datos o conexión Passwordless. |
