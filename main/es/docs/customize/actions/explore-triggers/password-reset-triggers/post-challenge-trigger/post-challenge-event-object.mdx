            * `passkey` Se utilizó una passkey para autenticar como primer factor a un usuario de una conexión de base de datos.
            * `sms` Se utilizó una conexión de SMS sin contraseña para autenticar al usuario como primer factor.
            * `email` Se utilizó una conexión de correo electrónico sin contraseña para autenticar al usuario como primer factor o verificar el correo electrónico para el restablecimiento de contraseña.
            * `phone_number` Se utilizó un número de teléfono para el restablecimiento de contraseña.
            * `mock` Se utiliza para pruebas internas.
