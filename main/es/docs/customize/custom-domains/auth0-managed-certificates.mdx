description: Aprende a configurar dominios personalizados en los que Auth0 administra los certificados TLS (SSL).
title: Configurar dominios personalizados con certificados administrados por Auth0
---

Si quieres que Auth0 administre los certificados de tu <Tooltip tip="Dominio personalizado: Dominio de terceros con un nombre especializado o personalizado." cta="Ver glosario" href="/es/docs/glossary?term=custom+domain">dominio personalizado</Tooltip>, solo tienes que agregar un registro CNAME al dominio. Auth0 valida el registro y luego genera el certificado en sus servidores. Estos certificados se renuevan automáticamente cada tres meses. Puedes configurarlo fácilmente y no tendrás que encargarte del mantenimiento de los certificados.

Para configurar tu dominio personalizado con certificados administrados por Auth0, debes proporcionar tu nombre de dominio a Auth0 y verificar que eres el propietario de ese dominio. Una vez verificado, tendrás que configurar las funciones de Auth0 para empezar a usar tu dominio personalizado.

<div id="provide-your-domain-name-to-auth0">
  ## Proporciona tu nombre de dominio a Auth0
</div>

1. Ve a [Dashboard &gt; Settings &gt; Custom Domains](https://manage.auth0.com/#/tenant/custom_domains) o [Dashboard &gt; Branding &gt; Custom Domains](https://manage.auth0.com/#/custom_domains). Introduce tu dominio personalizado en el campo correspondiente y selecciona **certificados administrados por Auth0.**
2. Haz clic en **Add Domain**. Solo puedes añadir un dominio por tenant, aunque el botón **Add Domain** siga apareciendo después de añadir un dominio.

<div id="verify-ownership">
  ## Verificar la titularidad
</div>

Antes de poder usar el dominio con Auth0, tendrás que verificar que eres su propietario.

1. Ve a [Dashboard &gt; Branding &gt; Custom Domains](https://manage.auth0.com/#/branding/custom_domains) y añade a los registros DNS de tu dominio el registro CNAME de verificación que aparece en el Dashboard.

   <Frame>![Auth0 Branding Custom Domains Auth0-Managed Certificate Verify Domain](/docs/images/cdy7uua7fh8z/3dz8ii70nm0a5amS0PNJ5K/c7b58043042293f5502ec012901f5dac/Custom_Domain_-_Pending_Verification_-_English.png)</Frame>
2. Haz clic en **Verify** para continuar.
   Auth0 puede tardar unos minutos en verificar tu registro CNAME, según la configuración de tu DNS. Si Auth0 puede verificar tu nombre de dominio, verás una ventana de confirmación. Esto significa que el proceso de verificación se ha completado. En un plazo de 1 a 2 minutos, tu dominio personalizado debería estar listo para usarse.

<div id="add-cname-verification-record-to-dns-record">
  ## Agrega un registro CNAME de verificación al DNS
</div>

Una vez agregado, el registro CNAME debe estar presente en todo momento para evitar problemas durante la renovación del certificado.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 recomienda desactivar el aplanamiento de CNAME, a menos que sea estrictamente necesario, porque el aplanamiento de CNAME para certificados administrados por Auth0 es una configuración no compatible y puede hacer que el dominio personalizado deje de funcionar sin previo aviso.
</Callout>

Si necesitas habilitar el aplanamiento de CNAME para todos los subdominios administrados por Cloudflare y, además, configurar un subdominio específico como dominio personalizado de Auth0, considera [delegar el subdominio](https://developers.cloudflare.com/dns/manage-dns-records/how-to/subdomains-outside-cloudflare/) de Auth0 a otro proveedor de DNS.
Esto te permitirá usar el aplanamiento de CNAME para todos los subdominios, excepto el que se usa para Auth0.

Los siguientes pasos pueden variar según tu proveedor de dominios.

1. Inicia sesión en tu servicio de administración de dominios.
2. Crea un registro nuevo.

   | Parámetro                | Valor                                                                                               |
   | ------------------------ | --------------------------------------------------------------------------------------------------- |
   | **Tipo de registro**     | **CNAME**                                                                                           |
   | **Nombre**               | Ingresa el nombre de tu dominio personalizado (por ejemplo, **login.northwind.com**).               |
   | **Tiempo de vida (TTL)** | Usa el valor predeterminado.                                                                        |
   | **Valor**                | Pega el valor de **CNAME** proporcionado por el Auth0 Dashboard para el registro DNS de tu dominio. |
3. Cuando termines, guarda el registro.

Si Auth0 pudo verificar tu nombre de dominio, verás una ventana de confirmación. Esto significa que el proceso de verificación se completó. En un plazo de 1 a 2 minutos, tu dominio personalizado debería estar listo para usarse.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Si tu proveedor de DNS habilita un proxy en el registro CNAME de forma predeterminada, el dominio personalizado quedará en estado pendiente indefinidamente. Es posible que debas revisar la configuración de tu proveedor de DNS y desactivar el proxy.
</Callout>

Si no puedes completar el proceso de verificación, espera al menos 4 horas antes de repetir estos pasos. Para evitar interrupciones en el servicio, no vuelvas a crear el dominio personalizado al verificarlo de nuevo.
