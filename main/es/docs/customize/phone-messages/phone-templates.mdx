Cada uno de los correos electrónicos enviados como parte de los workflows de Auth0 tiene una plantilla de correo electrónico correspondiente que determina la dirección del remitente, el asunto, el cuerpo del mensaje y mucho más.

Después de [configurar un proveedor externo de telefonía](../phone-messages/configure-phone-messaging-providers), puedes modificar las plantillas telefónicas para personalizar tus mensajes telefónicos.

Las plantillas de mensajes de texto admiten [Liquid](https://liquidjs.com/index.html), un lenguaje de plantillas de código abierto que amplía la funcionalidad de HTML. Liquid te permite:

* Rellenar variables dinámicamente con datos, por ejemplo, reemplazar `{{ user.name }}` por el nombre del usuario
* Implementar lógica, como sentencias `if`/`else`
* Modificar datos con filtros para manipular cadenas y arreglos, usar operadores matemáticos y mucho más

Puedes habilitar, deshabilitar y personalizar las plantillas telefónicas en la [página de plantillas telefónicas del Dashboard](https://manage.auth0.com/#/phone).

  ## Plantillas telefónicas disponibles
</div>

A continuación, se incluye un resumen de las plantillas telefónicas disponibles:

| Nombre de la plantilla telefónica        | Descripción                                                                                   |
| ---------------------------------------- | --------------------------------------------------------------------------------------------- |
| Cuenta bloqueada                         | Informa al cliente de que su cuenta ha sido bloqueada como resultado de actividad sospechosa. |
| Cambiar la contraseña                    | Envía al cliente su código para cambiar la contraseña.                                        |
| Contraseña de un solo uso - Registro     | Envía al cliente el código de verificación para registrar la contraseña de un solo uso (OTP). |
| Contraseña de un solo uso - Verificación | Envía al cliente el código de verificación para verificar la contraseña de un solo uso (OTP). |
