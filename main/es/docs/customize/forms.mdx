Formularios es un editor visual que te permite crear formularios personalizables para ampliar tus flujos de identidad con pasos adicionales y lógica de negocio.

<Frame>![Dashboard > Actions > Forms](/docs/images/cdy7uua7fh8z/3QLVE6XsMRwhL17SUSMw8H/c679b99a8930d463a30352e1d11ee755/2024-05-09_14-20-42.png)</Frame>

Algunas de las capacidades clave de Formularios incluyen:

* Una experiencia controlada y segura dentro del dominio de tu tenant. No necesitas [redirigir](/es/docs/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger/redirect-with-actions) a los usuarios a sitios externos.
* Componentes prediseñados con validaciones de frontend y backend.
* Lógica de negocio personalizada con integraciones con terceros listas para usar.
* Una experiencia de marca coherente con <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Universal+Login">Universal Login</Tooltip>.

Usar Formularios te permite crear casos de uso como:

* **Perfilado progresivo:** Define un conjunto de condiciones para pedir a los usuarios que completen información adicional a medida que interactúan con tu producto. Por ejemplo, cuando un usuario inicie sesión por tercera vez, puedes pedirle que complete su perfil.
* **Aceptación de políticas personalizadas:** Si necesitas actualizar tus políticas, puedes definir lógica para comprobar si los usuarios han aceptado las políticas más recientes y, en función de eso, mostrarles un formulario personalizado que requiera su aceptación.
* **Pasos personalizados de registro o inicio de sesión:** Amplía y personaliza tus flujos con pasos adicionales y lógica personalizada. Por ejemplo, puedes verificar direcciones de correo electrónico o números de teléfono con OTP y vinculación de cuentas, o verificar datos de pago con Stripe, entre otros.
