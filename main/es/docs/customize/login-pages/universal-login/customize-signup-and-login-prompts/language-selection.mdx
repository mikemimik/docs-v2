La selección flexible de idioma es una función opcional que permite a los usuarios elegir su idioma preferido en los prompts de <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Universal+Login">Universal Login</Tooltip> asociados con tu aplicación, como la pantalla de inicio de sesión. Puedes configurar esta función de forma individual para distintos prompts de Universal Login.

Después de implementar esta función para un prompt específico, se agrega un menú de selección de idioma a la pantalla asociada. Los usuarios pueden elegir un idioma preferido desde este menú para renderizar automáticamente el prompt en ese idioma.

<Frame>![Dos prompts de Universal Login que muestran el menú de selección de idioma. ](/docs/images/cdy7uua7fh8z/4Z5pt3aH3vpRRYV0RDDnba/dc7a5abc8ee2cb462b2be4b43490393c/Language_Selector_Prompts.png)</Frame>

<div id="before-you-begin">
  ## Antes de comenzar
</div>

Antes de poder implementar la selección flexible de idioma, asegúrese de cumplir los siguientes requisitos:

* Use [Universal Login](/es/docs/authenticate/login/auth0-universal-login).

  * Esta función no está disponible para páginas de inicio de sesión personalizadas.
* Configure un [dominio personalizado](/es/docs/customize/custom-domains).
* Habilite los idiomas que planea agregar a sus prompts en su tenant de Auth0.

  * En el Auth0 Dashboard, vaya a [Settings &gt; General &gt; Languages](https://manage.auth0.com/#/tenant/general) y habilite uno o más idiomas.

    * Para obtener más información, consulte [Universal Login Internationalization](/es/docs/customize/internationalization-and-localization/universal-login-internationalization).

<div id="implement-flexible-language-selection">
  ## Implementa la selección flexible de idioma
</div>

La configuración de la selección flexible de idioma incluye dos pasos principales:

1. Preparar tu plantilla de página para facilitar la selección de idioma.
2. Configurar los prompts individuales para mostrar la selección de idioma a los usuarios.

<div id="prepare-your-universal-login-page-template">
  ### Prepara la plantilla de tu página de Universal Login
</div>

Para empezar, agrega el siguiente script a tu [plantilla de página de Universal Login](/es/docs/customize/login-pages/universal-login/customize-templates) para facilitar la selección del idioma.

```html lines
<script>
  function updateSelectedLanguage() {
    const selectElement = document.getElementById('language');
    const selectedValue = selectElement.value;
    const options = selectElement.options;
    for (let i = 0; i < options.length; i++) {
      if (options[i].value === selectedValue) {
        options[i].selected = true;
      } else {
        options[i].selected = false;
      }
    }
    document.getElementById('changeLanguage').click();
  }
</script>
```

Este script permite que los prompts de Universal Login rendericen un menú dinámico de selección de idioma. Los usuarios pueden acceder a este menú para aplicar su idioma preferido al prompt configurado.

Para agregar este script a tu plantilla de página, usa el endpoint [Set Template for Universal Login](https://auth0.com/docs/api/management/v2/branding/put-universal-login).

<Warning>
  Ten en cuenta que esta llamada **sobrescribirá cualquier configuración existente** que hayas realizado en tu plantilla de página de Universal Login. Para evitar interrupciones, asegúrate de incluir en esta llamada todas las personalizaciones necesarias de la plantilla de página.
</Warning>

<div id="configure-individual-universal-login-prompts">
  ### Configura prompts individuales de Universal Login
</div>

A continuación, agrega la selección de idioma a uno o varios prompts de Universal Login configurando partials personalizados para prompts. Los Partials son fragmentos de código personalizado que se insertan en un punto de entrada dentro de una pantalla de prompt, como la pantalla de inicio de sesión. Para obtener más información, consulta [Personaliza los prompts de Signup y Login](/es/docs/customize/internationalization-and-localization/universal-login-internationalization).

Para agregar la selección de idioma a un prompt específico, usa la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip> de Auth0 para configurar partials personalizados para prompts con los siguientes parámetros:

| Parámetro  | Descripción                                                                                                                                                                                                                                                                                                                                       | Valor de ejemplo  |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------- |
| `language` | Obligatorio.<br /><br />Uno o varios idiomas para incluir en el menú de selección de idioma que se muestra a los usuarios.                                                                                                                                                                                                                        | `es`,`fr`,`en`    |
| `persist`  | Obligatorio.<br /><br />Determina si el idioma debe mantenerse durante la sesión.                                                                                                                                                                                                                                                                 | `session`         |
| `action`   | Opcional.<br /><br />Usa este parámetro cuando quieras que los usuarios puedan actualizar su idioma sin avanzar el prompt.<br /><br />Si se proporciona este valor, el formulario **no** se envía automáticamente cuando un usuario cambia su idioma. De lo contrario, el prompt avanza automáticamente cuando se selecciona un idioma diferente. | `change-language` |

Consulta las secciones a continuación para ver ejemplos de código para distintos tipos de prompts de Universal Login.

<Warning>
  En los ejemplos de código a continuación, asegúrate de reemplazar los marcadores de posición con los valores adecuados:

  * Reemplaza `{yourDomain}` por `yourdomain.auth0.com`.
  * Reemplaza `{mgmtApiToken}` por tu access token.
</Warning>
