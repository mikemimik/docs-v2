title: Casos de uso de ACUL

ACUL admite diversos stacks tecnológicos para personalizar por completo la experiencia de inicio de sesión de tus usuarios finales y adaptarse a numerosos casos de uso empresariales mediante una personalización avanzada.
Usa los siguientes escenarios para personalizar los [temas](/es/docs/customize/login-pages/universal-login/customize-themes), las [plantillas](/es/docs/customize/login-pages/universal-login/customize-templates) y los [elementos de texto](/es/docs/customize/login-pages/universal-login/customize-text-elements) de tu Universal Login. Para obtener más información sobre el inicio de sesión alojado de Auth0, consulta [Auth0 Universal Login](/es/docs/authenticate/login/auth0-universal-login).
