<Warning>
  En este momento, los esfuerzos de desarrollo activo de Auth0 se centran en Universal Login, y Classic Login ya no recibe actualizaciones. Para personalizar tus páginas de inicio de sesión, Auth0 recomienda [Advanced Customizations for Universal Login](/es/docs/customize/login-pages/advanced-customizations).
</Warning>

Classic Login es una experiencia de inicio de sesión alojada por Auth0 que se personaliza con JavaScript. Implementar Classic Login es menos complejo que integrar directamente el proceso de autenticación en tu aplicación, y puede ayudar a evitar los riesgos de la autenticación entre orígenes.

Classic Login es una alternativa a [Universal Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience). Para comparar las funcionalidades de ambas experiencias de inicio de sesión, consulta [Universal Login vs. Classic Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login).
