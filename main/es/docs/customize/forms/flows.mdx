Flows te permite crear visualmente la lógica del lado del servidor que puede ejecutarse dentro de tus Forms para procesar la información recopilada, integrar cualquier servicio con una API o activar automatizaciones.

<Frame>![Dashboard > Forms > Flows](/docs/images/cdy7uua7fh8z/6u6Huh5CbH3GEQJv3AH7cx/e49ea613b2c46a43c83c89ffecddbafe/Flows.png)</Frame>

Puedes usar acciones de Flows con tu formulario para:

* Crear validaciones personalizadas del lado del servidor para los pasos de tu formulario.
* Integrarte con tu propia API mediante solicitudes HTTP personalizadas.
* Integrarte con terceros usando nuestras integraciones listas para usar.
* Ejecutar distinta lógica de negocio según la entrada o las propiedades del usuario.
* Validar datos usando <Tooltip tip="JSON Web Token (JWT): Formato estándar de ID Token (y, a menudo, de Access Token) que se usa para representar claims de forma segura entre dos partes." cta="Ver glosario" href="/es/docs/glossary?term=JSON+Web+Tokens">JSON Web Tokens</Tooltip>.
