Vault te permite almacenar de forma segura secretos o configuraciones comunes (por ejemplo: claves de API, <Tooltip tip="Access Token: Credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=access+tokens">tokens de acceso</Tooltip>, <Tooltip tip="Access Token: Credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=refresh+tokens">tokens de actualización</Tooltip>) para autenticar solicitudes con servicios externos o integraciones como conexiones de Vault.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * Hay conexiones que requieren que el usuario proporcione credenciales (por ejemplo, con una application machine-to-machine (M2M) de Auth0, generas un token de acceso con un tiempo de expiración y un token de actualización). La conexión almacena ambos tokens y genera un nuevo token de acceso antes de que caduque.
  * Hay servicios externos que requieren autorizar scopes para generar credenciales de API. Si cambias los scopes, actualiza la conexión a esos servicios para evitar problemas de conexión.
  * Cuando se agrega una conexión, está lista para usarse en tus flujos. Cualquier actualización de una conexión existente se propagará automáticamente.
</Callout>

<div id="create-and-modify-vault-connections">
  ## Crear y modificar conexiones de Vault
</div>

Crea y modifica conexiones de Vault en la página de Vault o en el editor de flujo cuando una Action requiere credenciales.

<div id="create-vault-connections-from-the-vault-page">
  ### Crear conexiones de Vault desde la página de Vault
</div>

Agrega una conexión de Vault desde la página de Vault siguiendo estos pasos:

1. Abre el editor de formularios seleccionando **Auth0 Dashboard &gt; Actions &gt; Forms.**
2. En la esquina superior izquierda, selecciona el menú del tenant y luego **Vault** para abrir la página de Vault.

   <Frame>![Dashboard > Forms > Vault](/docs/images/cdy7uua7fh8z/1sPNPKO6nuEDCuiYGRWSur/d85dc616170eaf2d7d5f321750863ccf/vault-connection.png)</Frame>
3. Selecciona **Añadir conexión de Vault**.
4. Selecciona la Action que quieras y **Continuar**.
5. Agrega un **Connection name** de Vault y selecciona **Continuar**.
6. Ingresa las credenciales.
   Si no tienes las credenciales específicas, puedes seleccionar **crear una conexión vacía** y agregarlas más tarde. Las conexiones vacías tienen un icono naranja que indica que faltan credenciales.
7. Selecciona **Añadir conexión de Vault**.

<div id="create-vault-connections-from-the-flow-editor">
  ### Crear conexiones de Vault desde el editor de flujos
</div>

Añade una conexión de Vault desde una Action del flujo, por ejemplo, Twilio, siguiendo estos pasos:

1. Abre el editor de flujos seleccionando **Auth0 Dashboard &gt; Actions &gt; Forms &gt; Flows**
2. Selecciona un flujo y añade una Action **Send SMS with Twilio**.
3. En la configuración **conexión de Vault** de la Action, selecciona el icono **+**.

   <Frame>![Dashboard > Forms > Vault ](/docs/images/cdy7uua7fh8z/1AsaznhRKxm5LZxPlcbfUw/10e5a8af700aab909f5b03fcd266da7e/twilio-vault-connection.png)</Frame>
4. Añade un **Connection name** de Vault y selecciona **Continuar**.
5. Introduce las credenciales.
   Si no tienes las credenciales específicas, puedes seleccionar **Crear una conexión vacía** y añadir las credenciales más tarde. Las conexiones vacías tienen un icono naranja que indica que faltan credenciales.

   <Frame>![Dashboard > Forms > Vault > Configuración de la conexión de Twilio](/docs/images/cdy7uua7fh8z/yobXanutPRCO3FWek8cOk/bc29e39fb8593cff35e2738234bfe6fe/twilio-vault-settings.png)</Frame>
6. Selecciona **Añadir conexión de Vault**.
