<div id="get-your-sendgrid-keys">
  ## Obtén tus claves de SendGrid
</div>

Encuéntralas en la página Settings del [Dashboard de SendGrid](https://app.sendgrid.com/settings/api_keys), en la sección API keys.

<div id="configure-your-sendgrid-vault-connection">
  ## Configura tu Conexión de Vault de SendGrid
</div>

Para configurar una Conexión de Vault de SendGrid para tus Flow Actions de SendGrid, debes indicar los parámetros obligatorios.

<Frame>![Dashboard > Actions > Formularios > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Ajustes de entrada
</div>

| Parámetro | Descripción                                         |
| --------- | --------------------------------------------------- |
| API Key   | API Key para la autorización de la API de SendGrid. |
