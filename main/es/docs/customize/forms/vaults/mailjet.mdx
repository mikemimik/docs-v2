<div id="get-your-mailjet-keys">
  ## Obtén tus claves de MailJet
</div>

Las encontrarás en la página Account del [Dashboard de MailJet](https://app.mailjet.com/account/api_keys), en API keys.

<div id="configure-your-mailjet-vault-connection">
  ## Configura tu conexión de MailJet en Vault
</div>

Para configurar una conexión de MailJet en Vault para tus Flow Actions de MailJet, debes establecer los parámetros necesarios.

<Frame>![Dashboard > Actions > formulario > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro     | Descripción                                               |
| ------------- | --------------------------------------------------------- |
| API Key       | API Key para la autenticación en la API de MailJet.       |
| Clave secreta | Clave secreta para la autenticación en la API de MailJet. |
