<div id="configure-your-jwt-vault-connection">
  ## Configura tu Conexión de Vault JWT
</div>

Para configurar una Conexión de Vault JWT para que tus Actions JWT de Flows firmen, decodifiquen y verifiquen JSON Web Tokens, debes establecer los parámetros necesarios.

<Frame>![Dashboard > Actions > formulario > Vault > JWT](/docs/images/customize/forms/vaults/jwt-vault-connection.png)</Frame>

<div id="input-settings">
  ## Configuración de entrada
</div>

| Parámetro | Descripción                                                       |
| --------- | ----------------------------------------------------------------- |
| Secret    | Secret que se usará con el algoritmo HS256 para codificar el JWT. |
