<div id="get-your-twilio-credentials">
  ## Obtén tus credenciales de Twilio
</div>

Puedes encontrarlas en el [Dashboard de la consola de Twilio](https://console.twilio.com/).

<div id="configure-your-twilio-vault-connection">
  ## Configura tu Twilio Vault Connection
</div>

Para configurar una Twilio Vault Connection para tus Twilio Flow Actions, debes establecer los parámetros necesarios.

<Frame>![Dashboard > Actions > Forms > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro   | Descripción                                           |
| ----------- | ----------------------------------------------------- |
| Account SID | Account SID para la autorización de la API de Twilio. |
| Auth token  | Auth token para la autorización de la API de Twilio.  |
