<div id="configure-your-whatsapp-vault-connection">
  ## Configura tu conexión de Vault de WhatsApp
</div>

Para configurar una conexión de Vault de WhatsApp para tus WhatsApp Flow Actions, debes establecer los parámetros requeridos.

<Frame>![Dashboard > Actions > formularios > Vault > WhatsApp](/docs/images/customize/forms/vaults/whatsapp-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                       |
| --------- | ------------------------------------------------- |
| Token     | Token generado para autorizar la API de WhatsApp. |
