<div id="send-message">
  ## Enviar mensaje
</div>

Envía un mensaje.

<Frame>![](/docs/images/cdy7uua7fh8z/5zZPQw5KKYwBqxbR9jF1wZ/edba085456e3536f5167ff4c21bea77c/whatsapp.png)</Frame>

<div id="input-settings">
  ### configuración de entrada
</div>

| Parámetro                                         | Descripción                                                                                                                                                                                                                                                        |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ID del teléfono del remitente (obligatorio)       | El ID del teléfono del remitente se encuentra en la [configuración de WhatsAPP de Facebook Business Manager](https://developers.facebook.com/apps). Ten en cuenta que no es un número de teléfono.                                                                 |
| Número de teléfono del destinatario (obligatorio) | El número de teléfono del destinatario en [formato E.164](https://en.wikipedia.org/wiki/E.164).                                                                                                                                                                    |
| Tipo de mensaje (obligatorio)                     | Selecciona el tipo de mensaje en el menú desplegable. Puede ser uno de los siguientes: <ul> <li>Audio</li> <li>Contactos</li> <li>Documento</li> <li>Imagen</li> <li>Interactivo</li> <li>Ubicación</li> <li>Pegatina</li> <li>Plantilla</li> <li>Texto</li> </ul> |
| Objeto del mensaje (obligatorio)                  | El contenido del objeto del mensaje variará según el tipo de mensaje seleccionado. Para obtener más información, consulta [Mensajes](https://developers.facebook.com/docs/whatsapp/cloud-api/reference/messages).                                                  |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes encontrar todos los códigos de estado previstos en la [documentación de Facebook](https://developers.facebook.com/docs/whatsapp/cloud-api/support/error-codes).
