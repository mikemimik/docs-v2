En la aplicación expuesta por la extensión Administración delegada, hay dos vistas disponibles: **Users** y **Logs**. En la vista de **User**s, puedes ver y modificar los usuarios asociados a tu cuenta de Auth0.

De forma predeterminada, se muestran todos los usuarios, pero puedes filtrar la lista configurando un Filter Hook.

Para obtener más información sobre el Filter Hook, consulta [Administración delegada: Filter Hook](/es/docs/customize/extensions/delegated-administration-extension/delegated-administration-hooks/delegated-administration-filter-hook).

<div id="user-actions-in-delegated-administration-dashboard">
  ## Acciones de usuario en el Dashboard de administración delegada
</div>

Para limitar la cantidad de opciones de una persona con acceso al Dashboard expuesto por la extensión Administración delegada, configure un Access Hook.

Obtenga más información sobre los Access Hooks en [Administración delegada: Access Hook](/es/docs/customize/extensions/delegated-administration-extension/delegated-administration-hooks/delegated-administration-access-hook).

La opción **Change Profile** está disponible solo si ha configurado campos personalizados.

Tenga en cuenta la nueva opción **Reset Password** disponible a través de la extensión. Esta opción enviará un correo electrónico al usuario para que pueda elegir una nueva contraseña. Para hacerlo, seleccione un usuario y luego seleccione **Actions** &gt; **Reset Password**.

Esto enviará un correo electrónico al usuario con un enlace para cambiar la contraseña.

Si su perfil indica que tiene el rol `Delegated Admin - Administrator`, la vista **Logs** le permite ver una lista de autenticaciones realizadas por sus usuarios (esta vista solo es visible para los usuarios con el rol `Delegated Admin - Administrator`). El contenido de esta vista es un subconjunto de los datos que se muestran en [Auth0 Dashboard &gt; monitoreo &gt; Logs](https://manage.auth0.com/#/logs). El panel de Logs también muestra datos sobre las acciones administrativas realizadas en el <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="create-users">
  ## Crear usuarios
</div>

Puedes crear un usuario nuevo seleccionando el botón **+ Crear usuario** en la vista **Users**. Debes especificar el correo electrónico y la contraseña. Según tu rol, es posible que no puedas establecer el **Departmen**t al que pertenece el nuevo usuario.

Por ejemplo, los usuarios con el rol `Delegated Admin - Administrator` pueden ver el campo **Department** y seleccionar cualquiera de sus valores.

<Frame>![Extensions - Administration Dashboard - Create admin](/docs/images/cdy7uua7fh8z/4vrbU6nRuIkF30ZPM4yHnG/73fd2060fe9618c9f465025cb56deb02/create-user-admin.png)</Frame>

Por otro lado, Kelly, que tiene el rol `Delegated Admin - User` y pertenece al departamento de Finanzas, no puede ver el campo **Department**. El usuario que cree se asignará automáticamente al departamento de Finanzas.

<Frame>![Extensions - Delegated Administration Dashboard - Create user](/docs/images/cdy7uua7fh8z/2CSdiPbyiyPsURHH9X5VVZ/8416222627933bd35c044aa15c22dfae/create-user-kelly.png)</Frame>

<Card title="Tiempo de espera de la sesión de Administración delegada">
  De forma predeterminada, el tiempo de expiración del token es de 10 horas. Sin embargo, por razones de seguridad, al usar Administración delegada, Auth0 no guarda ningún token en las cookies ni en `sessionStorage`. Debes iniciar una nueva sesión cada vez que se recargue la página.
</Card>
