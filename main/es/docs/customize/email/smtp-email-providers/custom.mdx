---
title: Configuración de Custom Email Provider con Actions
description: Escribe una Action para enviar mensajes a tu proveedor de correo electrónico e implementar lógica personalizada para tu caso de uso.
---

Si quieres más opciones de personalización en la configuración de tu proveedor de correo electrónico, puedes configurar tu tenant con una <Tooltip tip="Action: Una función de Node.js que se ejecuta en puntos definidos durante el runtime de Auth0." cta="Ver glosario" href="/es/docs/glossary?term=Action">Action</Tooltip>.

En lugar de usar nuestro proveedor de correo electrónico o las integraciones con servidores SMTP, puedes escribir el código de tu Action para enviar mensajes a tu proveedor de correo electrónico o al servidor SMTP que elijas.

Esto te da más control sobre el proceso de entrega de correos electrónicos y te permite implementar lógica personalizada para tu caso de uso, como reintentar envíos fallidos, cambiar destinatarios o crear lógica específica de la organización.
