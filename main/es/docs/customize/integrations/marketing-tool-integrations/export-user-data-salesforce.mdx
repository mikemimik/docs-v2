title: Exportar datos de usuario a Salesforce
---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego importarlos en Salesforce mediante el Data Import Wizard. Para consultar los requisitos del sistema, visita el artículo de Salesforce [Data Import Wizard](https://help.salesforce.com/articleView?id=data_import_wizard.htm).

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Comienza por ir a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, configura **Export Format** con el formato de archivo requerido. Salesforce acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, proporciona un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de agregar los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando termine, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Es posible que algunos tipos de cuenta requieran pasos adicionales para que la importación se realice correctamente. Tómese un momento para revisar el artículo de Salesforce [Prepare Your Data for Import](https://help.salesforce.com/articleView?id=import_prepare.htm). Para obtener instrucciones adicionales, visite [Salesforce: Import Data with the Data Import Wizard](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm).

Antes de comenzar, asegúrese de que su cuenta de Salesforce tenga los [permisos de usuario](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm) necesarios para importar registros.

Para importar su archivo CSV a Salesforce, siga estos pasos:

1. Inicie sesión en [Salesforce](https://login.salesforce.com/).
2. Vaya a **Setup** y abra el **Data Import Wizard**.
3. Haga clic en **Launch Wizard**.
4. Configure la importación y seleccione el archivo CSV que desea cargar en los campos correspondientes.
5. Haga clic en **Next**.
6. En la página **Edit Field Mapping**, asigne los campos de datos de su archivo CSV a los campos de datos de Salesforce. Haga clic en **Next** para continuar.
7. Después de verificar la información en la página **Review**, haga clic en **Start Import**.
8. Consulte la pestaña **Recent Import Jobs** en la página principal del **Data Import Wizard** para ver actualizaciones sobre el estado de la importación.

¡Eso es todo! Importó correctamente sus usuarios de Auth0 a Salesforce.
