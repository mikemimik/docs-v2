title: Exportar datos de usuario a Salesforce Marketing Cloud
---

En este artículo, aprenderás a exportar datos de usuario de Auth0 a un archivo CSV y, luego, a importarlos en Salesforce Marketing Cloud con Email Studio.

Para obtener más información, consulta el artículo de Salesforce [Get Started Checklist for Email Studio](https://help.salesforce.com/s/articleView?id=mktg.mc_es_get_started_checklist_for_email_studio.htm\&type=5).

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para empezar, ve a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, establece **Export Format** en el formato de archivo requerido. Salesforce Marketing Cloud acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, especifica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de agregar los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación se complete, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importa un archivo de datos de usuario
</div>

Antes de empezar, asegúrate de que tu cuenta de Salesforce tenga los permisos de usuario necesarios para importar registros. Además, te recomendamos revisar el [artículo de Salesforce sobre la importación de suscriptores](https://help.salesforce.com/s/articleView?id=mktg.mc_es_import_subscribers_from_file.htm\&type=5) junto con [What permissions do I need to import records](https://help.salesforce.com/s/articleView?id=xcloud.faq_import_general_permissions.htm\&type=5).

Para importar tu archivo CSV a Salesforce Marketing Cloud, sigue estos pasos:

1. Inicia sesión en Salesforce Marketing Cloud y abre **Email Studio**.
2. Ve a **Subscribers &gt; Lists**.
3. Elige la lista en la que quieres importar y selecciona la acción **Import**.
4. Después de la introducción del asistente de importación, selecciona tu archivo CSV como **Upload Source** y CSV como **Data Format**. Completa la configuración restante y haz clic en **Next**.
5. En el cuadro de diálogo **Map Attributes**, asigna los campos de datos de tu archivo CSV a los campos de datos correctos. Cuando termines los mapeos, haz clic en **Next**.
6. Después de verificar los mapeos en el cuadro de diálogo **Confirmed Mappings**, haz clic en **Begin** para iniciar la importación.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a Salesforce Marketing Cloud.
