En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego importarlos en el Dashboard de Sailthru.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para empezar, ve a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, establece **Export Format** en el formato de archivo requerido. Sailthru acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de añadir los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación termine, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ### Importar un archivo de datos de usuario
</div>

Antes de comenzar, puedes obtener más información en [Documentación de Sailthru: Agregar usuarios a Sailthru y a las listas de Sailthru](https://getstarted.meetmarigold.com/engagebysailthru/Content/audience/managing-users/add-users-lists-st.html?TocPath=Audience%7CManaging%20Users%7C_____3#ListFileUpload).

Para importar tu archivo CSV a Sailthru, sigue estos pasos:

1. Inicia sesión en Sailthru y ve a [Lists](https://my.sailthru.com/lists).
2. Elige una lista, haz clic en **Upload List** y selecciona tu archivo CSV.
3. Selecciona **Add To List** en **Action**.
4. Configura la opción **Replace Vars** e introduce un correo electrónico para recibir notificaciones, si lo deseas.
5. Revisa la configuración y haz clic en **Submit** para cargar el archivo.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a Sailthru.
