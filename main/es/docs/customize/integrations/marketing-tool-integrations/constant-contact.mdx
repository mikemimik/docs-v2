title: Exportar datos de usuario a Constant Contact
---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego importarlos en el panel de control de Constant Contact.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para comenzar, ve a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, configura **Export Format** con el formato de archivo requerido. Constant Contact acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, especifica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name                     |
| ------------- | ------------------------------- |
| `email`       | Dirección de correo electrónico |
| `created_at`  | Fecha de creación               |
| `given_name`  | Nombre                          |
| `family_name` | Apellido                        |

Después de agregar los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación se haya completado, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Para importar tu archivo CSV en Constant Contact, sigue estos pasos:

1. [Inicia sesión en Constant Contact](https://login.constantcontact.com) y ve a **Contactos**.
2. En la página **Contactos**, haz clic en **Agregar contactos &gt; Subir desde archivo**.
3. Sube tu archivo CSV en la página **Subir desde archivo** y luego haz clic en **Continuar**.
4. Revisa las columnas y relaciónalas con los campos de Constant Contact. Cuando termines, haz clic en **Continuar**.
5. En la página **Seleccionar listas**, elige la lista o las listas a las que quieres agregar tus contactos.
6. Haz clic en **Subir**.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a Constant Contact.
