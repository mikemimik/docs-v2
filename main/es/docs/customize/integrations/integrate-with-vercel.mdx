La integración nativa de Auth0 con Vercel te permite conectar tu plataforma de identidad a tu proyecto de Vercel con una configuración mínima. Configura y personaliza tu experiencia de inicio de sesión sin tener que crearla desde cero. Después, amplía y escala tus flujos de autenticación y autorización con funciones avanzadas, como [B2B Organizations](/docs/es/manage-users/organizations), [Fine-Grained Authorization (FGA)](https://docs.fga.dev/), [Auth0 for AI Agents](https://auth0.com/ai/docs) y más.

La integración nativa de Auth0 con Vercel permite:

* El aprovisionamiento automático de una aplicación de Auth0 conectada a tu proyecto de Vercel.
* La carga previa de tus credenciales de Auth0 en el Dashboard de Vercel para una configuración rápida y sencilla.
* Compatibilidad inmediata con tus aplicaciones de Next.js mediante los [SDK de Auth0](/docs/es/libraries).

La integración nativa de Auth0 con Vercel crea un nuevo tenant de Auth0 para tu proyecto de Vercel. Si quieres usar una cuenta de Auth0 existente, intégrala con Vercel manualmente al crear una aplicación en el Auth0 Dashboard.

<div id="prerequisites">
  ## Requisitos previos
</div>

Antes de comenzar, asegúrate de tener:

* Una cuenta de Vercel con un proyecto activo. Si es necesario, [crea una cuenta de Vercel](https://vercel.com/signup)
* Una aplicación de Next.js que use la última versión del [SDK de Auth0 para Next.js](/docs/es/quickstart/webapp/nextjs)

Después de instalar la integración, [habilita iframe embedding](/docs/es/customize/login-pages/manage-core-configuration#iframe-embedding) para permitir que Universal Login o Classic Login se carguen en un iframe, tal como lo requiere Vercel.

<div id="install-integration">
  ## Instalar la integración
</div>

Instala la integración nativa de Vercel de Auth0 desde el Marketplace de Vercel:

1. Ve a **Integrations &gt; Browse Marketplace** y busca Auth0. Selecciona la **integración de Auth0** en **Native Integrations** y, a continuación, selecciona **Install**.

2. Auth0 rellena automáticamente la URL de localhost y la URL de callback de tu aplicación. Selecciona tu **Installation Plan** y, a continuación, selecciona **Continue**.

3. Introduce un nombre para tu aplicación. Selecciona **Create**. Una vez que se hayan creado correctamente el nuevo tenant de Auth0 y la aplicación, selecciona **Continue**.

4. Para conectarte a un proyecto de Vercel existente, selecciona **Connect Project** y, a continuación, selecciona tu proyecto y tus entornos. Si es necesario, introduce un prefijo personalizado para añadirlo a tus variables de entorno. Selecciona **Connect**. Una vez que hayas conectado correctamente un proyecto de Vercel, deberías acceder a la sección [Primeros pasos](#getting-started) de la integración.

<div id="getting-started">
  ## Primeros pasos
</div>

La sección Primeros pasos del Dashboard de Vercel muestra un Quickstart para poner tu aplicación en funcionamiento de forma rápida y segura. También puedes gestionar la configuración de tu aplicación en el menú de navegación de la izquierda.

<div id="quickstart">
  ### Quickstart
</div>

Para comenzar a usar la integración nativa de Vercel de Auth0, sigue las instrucciones del Dashboard de Vercel para crear y desplegar tu aplicación.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Solo puedes configurar las variables de entorno de Auth0 en un entorno de producción.
</Callout>

<div id="manage-application-settings">
  ## Gestionar la configuración de la aplicación
</div>

Puedes gestionar la configuración de tu aplicación en el [Dashboard de Vercel](#vercel-dashboard) y el [Auth0 Dashboard](#auth0-dashboard).

<div id="vercel-dashboard">
  ### Dashboard de Vercel
</div>

Para administrar la configuración de tu aplicación en el Dashboard de Vercel, ve a **Settings** en el menú de navegación de la izquierda.

Puedes actualizar la siguiente configuración de la aplicación:

* **Actualizar nombre**: Actualiza el nombre de la aplicación.
* **Actualizar configuración**:
  * **Parámetros**: Actualiza la URL de localhost y el endpoint de callback de tu aplicación.
  * **Plan actual del nivel de instalación**: Ve a la configuración de la integración para actualizar el plan del nivel de instalación de tu integración. Para obtener más información, consulta [Settings](#settings).
* **Proteger este recurso**
  * **Rotar secretos**: Rota los secretos de tu aplicación. Para obtener más información, consulta [Rotating environment variables](https://vercel.com/docs/projects/environment-variables/managing-environment-variables#rotating-environment-variables).
  * **Entornos permitidos**: Selecciona si la integración está disponible para **Todos los entornos** o **Solo el entorno de producción**.
* **Eliminar integración**: Cuando eliminas la integración, pasas automáticamente a un plan Free de Vercel. Para obtener más información, consulta [Delete integration](#delete-integration).

<div id="auth0-dashboard">
  ### Auth0 Dashboard
</div>

Administra la configuración adicional de la aplicación en el [Auth0 Dashboard](https://manage.auth0.com/).

Cuando instalas la integración nativa, Auth0 crea una nueva aplicación en un tenant dedicado, preconfigurada con tus URI de Vercel, callbacks permitidos y URLs de logout.

Puedes modificar esta aplicación, por ejemplo, para personalizar tu experiencia de Universal Login, igual que harías con cualquier otra aplicación en tu Dashboard de Auth0.

<div id="manage-integration">
  ## Gestionar la integración
</div>

Para gestionar la integración desde el Dashboard de Vercel, selecciona la pestaña **Integration** de tu proyecto, ve hasta la integración de Auth0 y selecciona **Manage**.

<div id="invoices">
  ### Facturas
</div>

En la página de **Facturas**, puedes ver una lista de las facturas relacionadas con tu integración de Auth0. Para obtener más información sobre las facturas, consulta [Billing &amp; Invoices](https://vercel.com/docs/pricing/understanding-my-invoice).

<div id="settings">
  ### Configuración
</div>

En la página de **Configuración**, puede agregar métodos de pago, cambiar el plan del nivel de instalación y eliminar la integración.

<div id="payment-methods">
  #### Métodos de pago
</div>

Para agregar un método de pago, selecciona **Agregar**. Para obtener más información sobre los métodos de pago, consulta [los precios de Vercel](https://vercel.com/docs/accounts/plans).

<div id="current-installation-level-plan">
  #### Plan actual de la instalación
</div>

El plan de la instalación se aplica a todos los productos de la instalación.

Para cambiar el plan de la instalación:

1. En **Settings**, desplázate hasta **plan actual del nivel de instalación** y selecciona **Change Plan**.
2. Selecciona el plan de la instalación que desees.
3. Selecciona **Continue**.
4. Revisa tu selección y, luego, selecciona **Update**.
5. Una vez actualizada la instalación, selecciona **Done**.

<div id="delete-integration">
  #### Eliminar integración
</div>

Al eliminar la integración, tu plan de Vercel se cambia automáticamente al plan Free.

Para eliminar la integración:

1. En **Settings**, desplázate hasta **Delete &lt;YOUR&#95;PRODUCT&gt;** y selecciona el botón **Remove &lt;YOUR&#95;PRODUCT&gt;**.
2. Introduce el slug de la integración y tu equipo; luego, selecciona **Reconozco que esto eliminará mi cuenta conectada a Auth0**.
3. Selecciona **Remove &lt;YOUR&#95;PRODUCT&gt;**.
