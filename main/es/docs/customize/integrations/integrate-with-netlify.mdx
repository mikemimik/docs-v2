<div id="overview">
  ## Descripción general
</div>

Conecta una instancia nueva o existente de Auth0 a tu proyecto de Netlify para mejorar la seguridad, la escalabilidad y la personalización de tu aplicación con la integración de Auth0. Esta integración te permite implementar directamente en tu aplicación un enfoque optimizado para la identidad de clientes.

Usa la nueva integración para crear y personalizar el flujo de identidad de tu proyecto de Netlify. Para poner tu aplicación en marcha de forma rápida y segura, esta integración:

* Configura un entorno de producción en Netlify que se integra con tu tenant de Auth0 en solo unos clics.
* Crea automáticamente una aplicación en tu tenant de Auth0 que se conecta con tu proyecto de Netlify.
* Carga previamente tus credenciales de Auth0 en Netlify, lo que te permite omitir algunos de los pasos manuales de este proceso.

<div id="prerequisites">
  ## Requisitos previos
</div>

Para comenzar a integrar Auth0 en tu proyecto de Netlify, hay algunos requisitos que debes tener listos antes de empezar:

* Una cuenta de Netlify con un proyecto que quieras integrar. Visita [Create your Netlify account](https://www.netlify.com/) para crear una cuenta.
* Una aplicación web que requiera autenticación, como una aplicación de React o Vue, o un sitio estático con funciones serverless.

<div id="add-a-netlify-integration">
  ## Agrega una integración de Netlify
</div>

Después de crear tu cuenta de Netlify e iniciar sesión en tu panel, puedes comenzar el proceso para agregar una integración. Para agregar la integración de Auth0 a un proyecto existente de Netlify:

1. En el panel de Netlify, ve a **Extensions &gt; Directory.**
2. Busca [**Auth0**](https://app.netlify.com/extensions/auth0) y selecciona **Install.**
3. Selecciona **Install** para continuar con la instalación en tu portal de integración de Auth0.

<div id="connect-your-site">
  ## Conecta tu sitio
</div>

Para conectar tu sitio a Auth0:

1. Ve a la **Configuración del sitio** del sitio que quieres conectar a Auth0.
2. En la barra lateral, selecciona **Auth0** en **Acceso y seguridad**.
3. Selecciona **Vincular un tenant de Auth0** y sigue las instrucciones para conectarte a tu cuenta de Auth0. Si aún no tienes una cuenta de Auth0, puedes registrarte en https://auth0.com/signup.

   <Frame>![](/docs/images/cdy7uua7fh8z/3wZZNNVMadwZjeLi9vyQtH/01821cd184326a004592f5e0e20a6f72/2025-03-03_12-12-09.png)</Frame>
4. Selecciona tu tenant en la lista desplegable y luego selecciona **Accept** para autorizar tu cuenta.

   <Frame>![](/docs/images/cdy7uua7fh8z/7mrUvgQh70cgtri09aOHso/a76600ebf472dcddb3977a8b9e726e01/netlify-link.png)</Frame>
5. Después de vincular tu tenant, selecciona **Add tenant**.
6. Completa la información requerida para tu tenant y tu aplicación, activa la casilla **Redeploy site** y selecciona **Create site tenant**.

   <Frame>![](/docs/images/cdy7uua7fh8z/hPxAU8mwd9035AXOPMqxp/f74d0864a6cc5a4aed963bfc77237d7c/auth0_tenant.png)</Frame>
7. Ahora deberías ver el tenant del sitio vinculado correctamente en la configuración de tu sitio.

   <Frame>![](/docs/images/cdy7uua7fh8z/1y8khKk4yjGvySB2QRItD3/0f9a5606373c69761ff6fefa4fe180af/netlify-success.png)</Frame>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Ahora deberías ver tu aplicación cliente y tu API en Auth0 Dashboard, en **Applications**.
</Callout>

<div id="set-up-the-integration">
  ## Configura la integración
</div>

Al integrar Auth0 con tu proyecto de Netlify, tendrás que proporcionar a tu aplicación ajustes de autenticación como el **<Tooltip tip="Client ID: Valor de identificación asignado por Auth0 a tu recurso registrado." cta="Ver glosario" href="/docs/es/glossary?term=Client+ID">Client ID</Tooltip>** y el **dominio de Auth0**.

Si tu aplicación lee estos ajustes desde variables de entorno, la extensión las inyecta automáticamente.

1. Ve al panel de Netlify.
2. Ve a **Configuración del sitio &gt; Variables de entorno.**
3. Haz clic en **Agregar una variable** e ingresa:

   * `AUTH0_CLIENT_ID` (tu Client ID de Auth0)
   * `AUTH0_DOMAIN` (tu dominio de Auth0)

     <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
       Estas variables, junto con toda la demás información de la aplicación, se pueden encontrar en tu Auth0 Dashboard, en **Applications &gt; [Tu aplicación] &gt; Configuración**.
     </Callout>

Ahora ya deberías tener configuradas las credenciales de Auth0 correctas para el despliegue.

<div id="populate-a-configuration-file">
  ### Rellenar un archivo de configuración
</div>

Si tu aplicación lee la configuración de Auth0 desde un archivo, puedes automatizar este proceso durante la fase de compilación. Esto garantiza que tu aplicación siempre tenga la configuración de autorización adecuada sin tener que modificar archivos manualmente.

Para hacerlo:

1. Ve al panel de Netlify.
2. Ve a **Configuración del sitio &gt; Build &amp; deploy &gt; Build settings**.
3. Modifica el **Build command** para incluir un script de shell antes de iniciar la compilación:

   ```bash wrap lines
   echo "{\"clientId\": \"${AUTH0_CLIENT_ID}\", \"domain\": \"${AUTH0_DOMAIN}\"}" > auth_config.json && npm run build
   ```

Este script crea un archivo JSON llamado `auth_config.json` y lo rellena con las variables de entorno `AUTH0_CLIENT_ID` y `AUTH0_DOMAIN` definidas anteriormente, lo que garantiza que la aplicación tenga la configuración correcta de Auth0 en tiempo de ejecución.

<div id="manage-integration">
  ## Gestionar la integración
</div>

Después de agregar la integración de Auth0 a tu proyecto de Netlify, el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> crea una nueva aplicación para gestionar y personalizar tu flujo de trabajo de CIAM.

1. Ve al Auth0 Dashboard.
2. Selecciona **Applications &gt; Applications &gt; [Tu aplicación de Netlify]**
3. Confirma el URI de la aplicación de Netlify en **Settings &gt; Application URIs**.

Ahora puedes personalizar y modificar esta aplicación igual que cualquier otra aplicación de tu Auth0 Dashboard.
