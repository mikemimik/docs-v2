El objeto global `configuration` está disponible en Rules para almacenar valores de uso frecuente, como URL. La información confidencial, como credenciales o claves de API, debe almacenarse mediante el objeto `configuration` y mantenerse fuera del código de tus Rules.

<div id="configure-values">
  ## Configurar valores
</div>

Puedes establecer valores de configuración en la [configuración de Rules](https://manage.auth0.com/#/rules/) del Dashboard.

Para editar o cambiar el valor de una clave de configuración, elimina la configuración existente y sustitúyela por el valor actualizado. Debes haber creado al menos una Rule para ver el área de configuración; de lo contrario, se mostrará la demostración de Rules.

<div id="use-the-configuration-object">
  ## Usa el objeto de configuración
</div>

Puedes acceder a cualquier valor de configuración que establezcas mediante su clave a través del objeto `configuration` en tu código de Rules.

```javascript lines
var MY_API_KEY = configuration.MY_API_KEY;
```

El siguiente ejemplo es una Rule para enviar un mensaje de Slack cuando un nuevo usuario se registra. El [Slack Webhook](https://api.slack.com/incoming-webhooks) es un valor de `configuration` definido con la clave `SLACK_HOOK_URL`.

```javascript lines
function (user, context, callback) {
  // omitir si el usuario ya se registró o está usando un token de actualización
  if (context.stats.loginsCount > 1 || context.protocol === 'oauth2-refresh-token') {
    return callback(null, user, context);
  }

  // obtén la URL del hook de Slack en: https://slack.com/services/10525858050
  const SLACK_HOOK = configuration.SLACK_HOOK_URL;

  const slack = require('slack-notify')(SLACK_HOOK);
  const message = 'New User: ' + (user.name || user.email) + ' (' + user.email + ')';
  const channel = '#some_channel';

  slack.success({
    text: message,
    channel: channel
  });

  // no esperes a que finalice la llamada a la API de Slack, regresa de inmediato (la solicitud continuará en el sandbox)`
  callback(null, user, context);
}
```
