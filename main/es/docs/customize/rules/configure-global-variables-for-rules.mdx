<Warning>
  La fecha de fin de vida útil (EOL) de Rules y Hooks será el **18 de noviembre de 2026** y ya no están disponibles para los nuevos tenants creados a partir del **16 de octubre de 2023**. Los tenants existentes con Hooks activos conservarán el acceso a Hooks hasta el fin de vida útil.

  Recomendamos encarecidamente usar Actions para ampliar Auth0. Con Actions, tienes acceso a información de tipos más completa, documentación integrada y paquetes públicos de `npm`, y puedes conectar integraciones externas que mejoran tu experiencia general de extensibilidad. Para obtener más información sobre lo que ofrece Actions, consulta [Comprende cómo funcionan Auth0 Actions](/es/docs/customize/actions/actions-overview).

  Para ayudarte con la migración, ofrecemos guías que te ayudarán a [migrar de Rules a Actions](/es/docs/customize/actions/migrate/migrate-from-rules-to-actions) y [migrar de Hooks a Actions](/es/docs/customize/actions/migrate/migrate-from-hooks-to-actions). También contamos con una página específica, [Pasarse a Actions](https://auth0.com/extensibility/movetoactions), que destaca comparaciones de funcionalidades, [una demostración de Actions](https://www.youtube.com/watch?v=UesFSY1klrI) y otros recursos para acompañarte en el proceso de migración.

  Para obtener más información sobre la obsolescencia de Rules y Hooks, consulta nuestra entrada del blog: [Preparándose para el fin de vida útil de Rules y Hooks](https://auth0.com/blog/preparing-for-rules-and-hooks-end-of-life/).
</Warning>

Puedes configurar variables globales para Rules con el Auth0 Dashboard. Para obtener más información, consulta [Auth0 Rules](/es/docs/customize/rules).

1. Ve a [Auth0 Dashboard &gt; Auth Pipeline &gt; Rules](https://manage.auth0.com/#/rules) y localiza la sección **Settings**.

   <Frame>![Dashboard - Auth Pipeline - Rules ](/docs/images/cdy7uua7fh8z/4OiSXzc5fYgPagHdOGbfvj/a589bdf811df66658fe21c509aed610c/Dashboard_-_Auth_Pipeline_-_Rules.png)</Frame>
2. Introduce un par clave/valor para la variable y selecciona **+ Add**.

   El valor introducido ya está disponible para todas las Rules mediante el objeto global `configuration` y se puede consultar con el valor de la columna **Code Snippet**.
