description: Décrit comment mettre à jour les connexions activées d’une application à l’aide d’Auth0 Dashboard.
title: Mettre à jour les connexions de l’application
---

Vous pouvez modifier les connexions activées de votre application à l’aide du <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Accédez à [Dashboard &gt; Applications](https://manage.auth0.com/#/applications) et cliquez sur le nom de l’application à afficher.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Cliquez sur l’onglet **Connexions**, puis activez ou désactivez les connexions appropriées pour l’application.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Les [applications tierces](/docs/fr-CA/get-started/applications/third-party-applications) peuvent authentifier les utilisateurs au moyen de connexions promues au niveau du domaine ou de connexions explicitement mappées à l’application. Pour en savoir plus, consultez [Promouvoir les connexions au niveau du domaine](/docs/fr-CA/authenticate/identity-providers/promote-connections-to-domain-level).
</Callout>
