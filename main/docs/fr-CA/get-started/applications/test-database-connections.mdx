---
description: Explique comment tester les connexions de base de données pour les applications dans l’Auth0 Dashboard.
title: Tester les connexions de base de données
---

Vous pouvez tester les connexions de base de données pour les applications dans l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Les connexions de base de données configurées peuvent servir à vous connecter à votre application.

Pour effectuer le test correctement, vous devez déjà avoir [configuré votre connexion de base de données](/docs/fr-CA/get-started/applications/set-up-database-connections) et [créé un utilisateur](/docs/fr-CA/manage-users/user-accounts/create-users) pour cette connexion de base de données.

1. Accédez à [Auth0 Dashboard &gt; Authentification &gt; Base de données](https://manage.auth0.com/#/connections/database), puis sélectionnez la flèche **Try** à côté de la connexion que vous voulez tester.

   <Frame>![Dashboard - Authentification - Base de données - Liste des connexions de base de données](/docs/images/cdy7uua7fh8z/1Hf6ctA9xhsLOQO9IyG0VU/9061b8436368e003b165efb890d86473/DB_Connections_-_EN.png)</Frame>
2. Entrez le nom d’utilisateur et le mot de passe de votre utilisateur test. Si tout est configuré correctement, la page **It Works!** s’affichera :

   <Frame>![Dashboard - Connexions - Base de données - Try - Succès](/docs/images/cdy7uua7fh8z/5DWJWLf8zvZJH0rSlhK0cr/f33377edf6f82f17e11e08ec30d8d75a/connection-social-try-success.png)</Frame>
