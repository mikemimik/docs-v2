title: Activer la prise en charge d’Android App Links
---

[Android App Links](https://developer.android.com/training/app-links) permettent à une application de se définir comme l’application par défaut pour un type de lien donné. Par exemple, si un utilisateur clique sur une URL dans un courriel, le lien s’ouvre dans l’application désignée. Ce guide vous montre comment activer la prise en charge d’Android App Links pour votre application enregistrée dans Auth0 à l’aide de l’Auth0 Dashboard.

1. Accédez à [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis sélectionnez le nom de l’application à afficher.

   <Frame>![Liste des applications dans le Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler jusqu’au bas de la page **Settings**, puis sélectionnez **Show Advanced Settings**.
3. Sélectionnez **Device Settings**, entrez le [App Package Name](https://developer.android.com/studio/build/application-id) ainsi que les empreintes SHA256 du certificat de signature de votre application Android, puis sélectionnez **Save Changes**.

   <Frame>![Onglet Application Settings dans Applications du Dashboard, Advanced Settings, onglet Device Settings](/docs/images/cdy7uua7fh8z/YKKPtWtswHn2FZGFSVWNw/086b32e41b3fd446ac8aa678be33648d/Device_Settings_-_EN.png)</Frame>

Vous pouvez utiliser la commande suivante pour générer l’empreinte à l’aide de l’outil Java `keytool` dans votre terminal :
`keytool -list -v -keystore my-release-key.keystore`

Pour en savoir plus sur les certificats de signature, consultez la documentation pour les développeurs Android [Sign Your App](https://developer.android.com/studio/publish/app-signing.html).

<div id="test-link">
  ## Tester le lien
</div>

1. Accédez à l’URL suivante dans votre navigateur : `https://{yourDomain}/.well-known/assetlinks.json`
2. Si le lien fonctionne correctement, le JSON suivant s’affichera (mis en forme pour en faciliter la lecture) :

   ```json lines
   [{
     "relation": ["delegate_permission/common.handle_all_urls"],
     "target": {
       "namespace": "android_app",
       "package_name": "com.mycompany.app1",
       "sha256_cert_fingerprints":
       ["14:6D:E9:83:C5:73:06:50:D8:EE:B9:95:2F:34:FC:64:16:A0:83:42:E6:1D:BE:A8:8A:04:96:B2:3F:CF:44:E5"]
     }
   }]
   ```

Pour en savoir plus sur la vérification de votre lien d’application, consultez [Verify Android App Links](https://developer.android.com/training/app-links/verify-applinks) à l’adresse https://developer.android.com.
