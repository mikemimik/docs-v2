description: Découvrez comment activer la prise en charge des liens universels pour votre application Auth0 dans Xcode d’Apple à l’aide de l’Auth0 Dashboard.
title: Activer la prise en charge des liens universels dans Xcode d’Apple
---

Les liens universels établissent une relation vérifiée entre les domaines et les applications; les paramètres de votre application Auth0 et ceux de votre application iOS doivent donc correspondre. Pour ce faire, vous devez fournir à Auth0 votre Apple Team ID et votre Bundle Identifier. Vous pouvez activer la prise en charge des liens universels pour votre application enregistrée dans Auth0 à l’aide du <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="obtain-apple-team-id-and-bundle-identifier">
  ## Obtenir l’Apple Team ID et le Bundle Identifier
</div>

1. Pour trouver votre **Apple Team ID**, rendez-vous sur la [page Résumé de votre compte développeur Apple](https://developer.apple.com/membercenter/index.action#accountSummary).
2. Pour trouver le **Bundle Identifier** de votre application iOS, rendez-vous sur la page des [réglages du projet dans Xcode](https://developer.apple.com/library/content/documentation/IDEs/Conceptual/AppDistributionGuide/ConfiguringYourApp/ConfiguringYourApp.html).

<div id="provide-apple-team-id-and-bundle-identifier-to-auth0">
  ## Indiquez l’Apple Team ID et le Bundle Identifier à Auth0
</div>

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/clients) et sélectionnez le nom de l’application que vous voulez afficher.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler jusqu’au bas de la page **Settings**, puis sélectionnez **Show Advanced Settings**.
3. Sélectionnez **Device Settings**, saisissez les valeurs **Team ID** et **Bundle Identifier** de votre application iOS, puis sélectionnez **Save Changes**.

   <Frame>![Onglet Settings de l’application dans Dashboard Applications, paramètres avancés, onglet Device Settings](/docs/images/cdy7uua7fh8z/YKKPtWtswHn2FZGFSVWNw/086b32e41b3fd446ac8aa678be33648d/Device_Settings_-_EN.png)</Frame>

<div id="test-link">
  ## Tester le lien
</div>

Vérifiez si le fichier `apple-app-site-association` des liens universels est accessible pour votre application en ouvrant l’URL suivante dans votre navigateur : `{yourDomain}/apple-app-site-association`

Si le lien fonctionne, le JSON suivant s’affichera (mis en forme pour en faciliter la lecture) :

```json lines
{
  "applinks": {
    "apps": [],
    "details": [{
      "appID": "86WQXF56BC.com.auth0.Passwordless-Email",
      "paths": ["/ios/com.auth0.Passwordless-Email/*"]
    }]
  }
}
```
