description: Découvrez comment activer la prise en charge des liens universels pour votre application Auth0 dans Apple Xcode à l’aide de l’Auth0 Dashboard.
title: Activer la prise en charge des liens universels dans Apple Xcode
---

Les liens universels établissent une relation vérifiée entre les domaines et les applications. Les paramètres de votre application Auth0 et ceux de votre application iOS doivent donc être synchronisés. Pour ce faire, vous devez fournir à Auth0 votre Apple Team ID et votre Bundle Identifier. Vous pouvez activer la prise en charge des liens universels pour votre application enregistrée dans Auth0 au moyen de l’<Tooltip tip="Auth0 Dashboard : l’interface principale d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="obtain-apple-team-id-and-bundle-identifier">
  ## Obtenir l’Apple Team ID et le Bundle Identifier
</div>

1. Pour trouver votre **Apple Team ID**, accédez à la [page Summary de votre compte développeur Apple](https://developer.apple.com/membercenter/index.action#accountSummary).
2. Pour trouver le **Bundle Identifier** de votre application iOS, accédez à la page des [Settings de son projet Xcode](https://developer.apple.com/library/content/documentation/IDEs/Conceptual/AppDistributionGuide/ConfiguringYourApp/ConfiguringYourApp.html).

<div id="provide-apple-team-id-and-bundle-identifier-to-auth0">
  ## Fournissez l’Apple Team ID et le Bundle Identifier à Auth0
</div>

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/clients) et sélectionnez le nom de l’application à consulter.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler jusqu’au bas de la page **Settings**, puis sélectionnez **Show Advanced Settings**.
3. Sélectionnez **Device Settings**, entrez les valeurs **Team ID** et **App bundler identifier** de votre application iOS, puis sélectionnez **Save Changes**.

   <Frame>![Onglet Device Settings dans les paramètres avancés des paramètres de l’application dans Applications du Dashboard](/docs/images/cdy7uua7fh8z/YKKPtWtswHn2FZGFSVWNw/086b32e41b3fd446ac8aa678be33648d/Device_Settings_-_EN.png)</Frame>

<div id="test-link">
  ## Tester le lien
</div>

Vérifiez si le fichier `apple-app-site-association` des liens universels est accessible pour votre application en ouvrant l’URL suivante dans votre navigateur : `{yourDomain}/apple-app-site-association`

Si le lien fonctionne correctement, vous verrez le JSON suivant (mis en forme pour en faciliter la lecture) :

```json lines
{
  "applinks": {
    "apps": [],
    "details": [{
      "appID": "86WQXF56BC.com.auth0.Passwordless-Email",
      "paths": ["/ios/com.auth0.Passwordless-Email/*"]
    }]
  }
}
```
