---
description: Décrit comment modifier l’algorithme de signature d’une application à l’aide de l’Auth0 Dashboard.
title: Modifier les algorithmes de signature d’une application
---

Vous pouvez modifier l’<Tooltip tip="Algorithme de signature : algorithme utilisé pour signer numériquement les jetons afin de garantir que le jeton n’a pas été altéré." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=signing+algorithm">algorithme de signature</Tooltip> de votre application à l’aide du Dashboard.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis sélectionnez le nom de l’application à consulter.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler jusqu’au bas de l’onglet **Settings**, puis sélectionnez **Show Advanced Settings**.
3. Accédez à l’onglet **OAuth**.

   <Frame>![Onglet OAuth des paramètres de l’application dans les applications du Dashboard](/docs/images/cdy7uua7fh8z/5bbq6VrI9tUMht0PwMimre/08574482efef944b16ecfbb445be50ab/Oauth_Tab_-_English.png)</Frame>
4. Repérez **JsonWebToken Signature Algorithm**, puis sélectionnez l’algorithme de signature approprié pour l’application.
5. Cliquez sur **Save Changes**.
