Vous pouvez modifier les types d’autorisation de votre application dans le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou au moyen de la <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>.

<div id="use-the-dashboard">
  ## Utiliser le Dashboard
</div>

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis sélectionnez le nom de l’application à afficher.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler jusqu’au bas de la page, puis sélectionnez **Afficher les paramètres avancés**.
3. Sélectionnez **Types d’octroi**, puis activez ou désactivez les types d’octroi appropriés pour l’application. Lorsque vous avez terminé, sélectionnez **Enregistrer les modifications**. Le type d’octroi de code d’appareil est offert uniquement pour les applications natives.

   <Frame>![Onglet Types d’octroi dans les paramètres avancés des paramètres de l’application du Dashboard Applications](/docs/images/cdy7uua7fh8z/4ZMq4VfeGsKsKfZVFwSqe8/a0057749551915ac8d24b151bc27e875/Grant_Types_-_English.png)</Frame>

<div id="use-the-management-api">
  ## Utiliser la Management API
</div>

Effectuez un appel `PATCH` à l’endpoint [`/Clients/patch_clients_by_id`](https://auth0.com/docs/api/management/v2#!/Clients/patch_clients_by_id). Assurez-vous de remplacer les valeurs d’espace réservé `{yourClientId}`, `{yourManagementApiAccessToken}` et `{grantType}` par votre <Tooltip tip="ID client : valeur d’identification attribuée par Auth0 à votre ressource enregistrée." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=client+ID">ID client</Tooltip>, le <Tooltip tip="ID client : valeur d’identification attribuée par Auth0 à votre ressource enregistrée." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=access+token">jeton d’accès</Tooltip> de la Management API et le type d’octroi souhaité, respectivement.

</AuthCodeGroup>

| Valeur                  | Description                                                                                                                    |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `{yourClientId}`        | L’ID de l’application à mettre à jour.                                                                                         |
| `MGMT_API_ACCESS_TOKEN` | Les [jetons d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) ayant la portée `update:clients`. |
| `GRANT_TYPES`           | Les types d’octroi que vous souhaitez activer pour l’application indiquée.                                               |

<div id="troubleshoot">
  ## Dépannage
</div>

Si vous tentez d’utiliser un flux avec une application qui n’a pas les `grant_types` appropriés pour ce flux (ou si le champ est vide), l’erreur suivante s’affichera :
``Grant type `grant_type` not allowed for the client.``
