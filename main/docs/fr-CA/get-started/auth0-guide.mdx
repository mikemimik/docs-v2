<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Guide est actuellement en Early Access et est en cours de déploiement auprès des utilisateurs de la région Public Cloud des États-Unis. Comme Guide peut parfois fournir des réponses incomplètes ou inexactes, assurez-vous toujours de [les examiner et de les valider](#how-does-guide-avoid-ai-hallucinations%3F) à l’aide des liens fournis. Pour en savoir plus sur les lancements d’Auth0, consultez [Product Release Stages](/docs/fr-CA/troubleshoot/product-lifecycle/product-release-stages).
</Callout>

Guide est un chatbot alimenté par l’IA qui répond à vos questions sur Auth0. Il s’appuie sur l’information provenant de notre [documentation](https://auth0.com/docs), de notre [blogue](https://auth0.com/blog) et de la [Community](http://community.auth0.com). Pour les clients ayant un abonnement Enterprise, Guide exploite également les données du tenant Auth0 dans le [Security Center](/docs/fr-CA/secure/security-center) afin de vous aider à surveiller, à résoudre les problèmes et à appliquer les bonnes pratiques de sécurité du tenant.

Guide répond à vos questions, par exemple « Comment configurer les clés d’accès? », en fournissant une réponse résumée et des liens vers la documentation pertinente ainsi que vers les [guides d’intervention en cas d’incident](/docs/fr-CA/secure/attack-protection/playbooks) pour la sécurité du tenant et des utilisateurs.

<div id="how-guide-works">
  ## Comment fonctionne Guide
</div>

Guide traite votre message pour déterminer ce que vous cherchez, consulte les sources Auth0 pour trouver des renseignements pertinents, puis génère une réponse résumée fondée sur ce qu’il a trouvé, accompagnée de liens vers les documents cités.

Vous pouvez poser des questions complémentaires, [valider ses réponses](#how-does-guide-avoid-ai-hallucinations%3F) à l’aide des liens fournis et donner votre avis sur Guide en évaluant ses réponses.

Pour les clients ayant un abonnement Enterprise : si vous [configurez des alertes de surveillance de la sécurité](/docs/fr-CA/secure/security-center/security-alerts) dans Security Center, Guide vous aide à résoudre les incidents et vous propose des conseils de sécurité tirés d’un [guide d’intervention en cas d’incident](/docs/fr-CA/secure/attack-protection/playbooks) pertinent.

Par exemple, comme l’illustre la capture d’écran suivante, lorsqu’une forte hausse de l’activité des bots déclenche votre alerte <Tooltip tip="Bot Detection : forme de protection contre les attaques dans laquelle Auth0 bloque le trafic de bots suspect en activant un CAPTCHA pendant le processus de connexion." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Bot+Detection">Bot Detection</Tooltip>, Guide recommande des mesures correctives tirées du [guide d’intervention sur l’attaque Bot Detection](/docs/fr-CA/secure/attack-protection/playbooks/bot-detection-playbook), comme l’activation du CAPTCHA pour les flux d’authentification touchés, et vous invite à examiner les événements pertinents dans les journaux du tenant.

<div id="use-guide">
  ## Utiliser Guide
</div>

Vous pouvez accéder à Guide en sélectionnant le bouton **Ask Guide** dans le coin supérieur droit du <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

Lorsque vous sélectionnez **Ask Guide**, Guide s’ouvre et vous invite à poser des questions sur les fonctionnalités d’Auth0. Il peut suggérer des questions pertinentes en fonction de l’endroit où vous vous trouvez dans l’Auth0 Dashboard. Par exemple, si vous vous trouvez dans **Branding &gt; <Tooltip tip="Universal Login : votre application vous redirige vers Universal Login, hébergé sur l’Authorization Server d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Universal+Login">Universal Login</Tooltip>**, Guide vous proposera des questions sur Universal Login, comme :

* Qu’est-ce qu’Universal Login ?
* Quels sont les avantages d’Universal Login ?

<Frame>![Sélectionnez le bouton Ask Guide dans le coin supérieur droit de l’Auth0 Dashboard pour l’ouvrir.](/docs/images/cdy7uua7fh8z/e3lTjWgGC16Ob6zeUfU1V/ea8eef92439697bced3c147c9ca87d87/image3.png)</Frame>

<div id="questions-guide-can-answer">
  ### Questions auxquelles Guide peut répondre
</div>

Guide peut répondre aux questions sur le produit et la plateforme Auth0, par exemple :

* Qu’est-ce que Universal Login ?
* Comment activer les intégrations d’<Tooltip tip="Single Sign-On (SSO) : service qui, après qu’un utilisateur se connecte à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=single+sign-on">authentification unique</Tooltip> ?
* Comment configurer les clés d’accès ?

<Frame>![Guide peut répondre aux questions sur le produit et la plateforme Auth0.](/docs/images/cdy7uua7fh8z/nGVAVWxnoqiXaFXTQQyHX/101b6fdfde5f8488d6d2401e5dee5a42/auth0-guide-sso-screenshot.png)</Frame>

Pour les clients ayant un abonnement Enterprise, Guide peut répondre aux questions liées au tenant Auth0 concernant les métriques et les données du Security Center, par exemple :

* Comment résoudre cette <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=MFA">vérification MFA</Tooltip> échouée ?
* Qu’est-ce qui a déclenché le seuil de mon alerte de protection contre les bots ?
* Quelles sont les étapes de remédiation pour une alerte de « credential stuffing » ?

<div id="questions-guide-cannot-answer">
  ### Questions auxquelles Guide ne peut pas répondre
</div>

Guide ne peut pas répondre aux questions qui ne portent pas sur le produit ou la plateforme Auth0, ou qui sont propres à votre tenant ou à votre configuration, par exemple :

* Quelles connexions ai-je activées ?
* Combien d’utilisateurs se sont connectés à mon application le mois dernier ?
* Lequel est le meilleur : Vim ou Emacs ?

<Warning>
  Ne transmettez pas à Guide de données personnelles ni d’autres données sensibles.
</Warning>

<div id="disable-guide">
  ## Désactiver Guide
</div>

Vous pouvez désactiver Guide dans votre tenant en communiquant avec [Auth0 Support](https://support.auth0.com/).

<div id="data-security-in-guide">
  ## Sécurité des données dans Guide
</div>

Vos messages sont protégés par les normes de sécurité et de conformité de niveau entreprise d’Okta.

Guide :

* utilise les messages fournis pour chercher des renseignements pertinents dans des sources publiques d’Auth0.

Guide ne fait pas ce qui suit :

* n’accède pas aux données de votre compte ou de votre tenant.
* n’utilise pas les messages fournis à des fins d’entraînement des LLM.
* ne partage pas vos messages avec des tiers.

<div id="how-does-guide-avoid-ai-hallucinations">
  ## Comment Guide évite-t-il les hallucinations de l’IA ?
</div>

Guide génère ses réponses à partir de contenu rédigé par Auth0 et inclut, dans chacune d’elles, des liens vers les documents cités. Vous pouvez consulter n’importe lequel de ces liens pour vérifier les renseignements à la source et obtenir de l’information supplémentaire.
