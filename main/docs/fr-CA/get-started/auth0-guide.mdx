<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Guide est actuellement en Early Access et en cours de déploiement auprès des utilisateurs de la région américaine du Public Cloud. Comme Guide peut parfois fournir des réponses incomplètes ou inexactes, [consultez-les et validez-les](#how-does-guide-avoid-ai-hallucinations%3F) toujours à l’aide des liens fournis. Pour en savoir plus sur les versions d’Auth0, consultez les [Product Release Stages](/docs/fr-CA/troubleshoot/product-lifecycle/product-release-stages).
</Callout>

Guide est un agent conversationnel alimenté par l’IA qui répond à vos questions sur Auth0. Il s’appuie sur l’information provenant de notre [documentation](https://auth0.com/docs), de notre [blogue](https://auth0.com/blog) et de la [Community](http://community.auth0.com). Pour les clients ayant un abonnement Enterprise, Guide exploite aussi les données du tenant Auth0 dans [Security Center](/docs/fr-CA/secure/security-center) pour vous aider à surveiller, à résoudre les problèmes et à appliquer les pratiques exemplaires de sécurité du tenant.

Guide répond à vos questions, par exemple « Comment configurer les clés d’accès? », avec une réponse résumée et des liens vers la documentation pertinente ainsi que les [guides d’intervention en cas d’incident](/docs/fr-CA/secure/attack-protection/playbooks) liés à la sécurité des tenants et des utilisateurs.

<div id="how-guide-works">
  ## Comment Guide fonctionne
</div>

Guide traite votre message pour comprendre ce que vous recherchez, consulte les sources d’Auth0 pour y trouver l’information pertinente, puis génère une réponse synthétique basée sur ses résultats, avec des liens vers les sources citées.

Vous pouvez poser des questions de suivi, [valider ses réponses](#how-does-guide-avoid-ai-hallucinations%3F) à l’aide des liens fournis et donner votre avis sur Guide en évaluant ses réponses.

Pour les clients ayant un abonnement Enterprise : si vous [configurez des alertes de surveillance de la sécurité](/docs/fr-CA/secure/security-center/security-alerts) dans Security Center, Guide vous aide à résoudre les incidents et vous propose des recommandations de sécurité tirées d’un [guide d’intervention en cas d’incident](/docs/fr-CA/secure/attack-protection/playbooks) pertinent.

Par exemple, comme l’illustre la capture d’écran suivante, lorsqu’une importante hausse de bots déclenche votre alerte <Tooltip tip="Bot Detection : forme de protection contre les attaques dans laquelle Auth0 bloque le trafic de bot suspect en activant un CAPTCHA pendant le processus de connexion." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Bot+Detection">Bot Detection</Tooltip>, Guide recommande des mesures correctives tirées du [guide Bot Detection Attack](/docs/fr-CA/secure/attack-protection/playbooks/bot-detection-playbook), comme l’activation du CAPTCHA pour les flux d’authentification touchés, et vous invite à examiner les événements pertinents dans les journaux du tenant.

<div id="use-guide">
  ## Utiliser Guide
</div>

Vous pouvez accéder à Guide en sélectionnant le bouton **Ask Guide** dans le coin supérieur droit du <Tooltip tip="Auth0 Dashboard : le principal produit Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

Lorsque vous sélectionnez **Ask Guide**, Guide s’ouvre et vous invite à poser des questions sur les fonctionnalités d’Auth0. Il peut suggérer des questions pertinentes en fonction de l’endroit où vous vous trouvez dans l’Auth0 Dashboard. Par exemple, si vous vous trouvez sous **Branding &gt; <Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur l'Authorization Server d'Auth0, pour vérifier l'identité d'un utilisateur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Universal+Login">Universal Login</Tooltip>**, Guide vous proposera des questions sur Universal Login, comme :

* Qu’est-ce qu’Universal Login ?
* Quels sont les avantages d’Universal Login ?

<Frame>![Sélectionnez le bouton Ask Guide dans le coin supérieur droit de l’Auth0 Dashboard pour l’ouvrir.](/docs/images/cdy7uua7fh8z/e3lTjWgGC16Ob6zeUfU1V/ea8eef92439697bced3c147c9ca87d87/image3.png)</Frame>

<div id="questions-guide-can-answer">
  ### Questions auxquelles Guide peut répondre
</div>

Guide peut répondre aux questions sur le produit et la plateforme Auth0, comme :

* Qu’est-ce que Universal Login ?
* Comment activer les intégrations d’<Tooltip tip="Single Sign-On (SSO) : service qui, après la connexion d’un utilisateur à une application, le connecte automatiquement aux autres applications." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=single+sign-on">authentification unique</Tooltip> ?
* Comment configurer les clés d’accès ?

<Frame>![Guide peut répondre aux questions sur le produit et la plateforme Auth0.](/docs/images/cdy7uua7fh8z/nGVAVWxnoqiXaFXTQQyHX/101b6fdfde5f8488d6d2401e5dee5a42/auth0-guide-sso-screenshot.png)</Frame>

Pour les clients ayant un abonnement Enterprise, Guide peut répondre aux questions sur le tenant Auth0 liées aux métriques et aux données du Security Center, comme :

* Comment résoudre cette demande de vérification <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=MFA">MFA</Tooltip> ayant échoué ?
* Qu’est-ce qui a déclenché le seuil de mon alerte de protection contre les bots ?
* Quelles sont les étapes de remédiation pour une alerte de « credential stuffing » ?

<div id="questions-guide-cannot-answer">
  ### Questions auxquelles Guide ne peut pas répondre
</div>

Guide ne peut pas répondre aux questions qui ne portent pas sur le produit ou la plateforme Auth0, ou qui sont propres à votre tenant ou à votre configuration, par exemple :

* Quelles connexions ai-je activées ?
* Combien d’utilisateurs se sont connectés à mon application le mois dernier ?
* Lequel est préférable : Vim ou Emacs ?

<Warning>
  Ne soumettez pas de données personnelles ni d’autres données sensibles à Guide.
</Warning>

<div id="disable-guide">
  ## Désactiver Guide
</div>

Vous pouvez désactiver Guide dans votre tenant en contactant [Auth0 Support](https://support.auth0.com/).

<div id="data-security-in-guide">
  ## Sécurité des données dans Guide
</div>

Vos messages sont protégés par les normes de sécurité et de conformité d’Okta de calibre entreprise.

Guide fait ce qui suit :

* Utilise les messages fournis pour rechercher des renseignements pertinents dans les sources publiques d’Auth0.

Guide ne fait pas ce qui suit :

* N’accède pas aux données de votre compte ou de votre tenant.
* N’utilise pas les messages fournis pour entraîner des LLM.
* Ne partage pas vos messages avec des tiers.

<div id="how-does-guide-avoid-ai-hallucinations">
  ## Comment Guide évite-t-il les hallucinations de l’IA ?
</div>

Guide génère ses réponses à partir de contenu rédigé par Auth0 et inclut, dans chacune d’elles, des liens vers les documents cités. Vous pouvez consulter n’importe lequel de ces liens pour vérifier les renseignements à la source et obtenir de l’information supplémentaire.
