Vous pouvez supprimer des permissions d’une API à l’aide du <Tooltip tip="Auth0 Dashboard : le produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou de la <Tooltip tip="Auth0 Dashboard : le produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>.

<div id="use-the-dashboard">
  ## Utiliser le Dashboard
</div>

1. Accédez à [Dashboard &gt; Applications &gt; APIs](https://manage.auth0.com/#/apis), puis cliquez sur le nom de l’API à afficher.

   <Frame>![Liste des API dans Dashboard Applications APIs](/docs/images/cdy7uua7fh8z/3rhmhghYZDSi6YWHRA5yMQ/c71340259481b0b6787d5f3887cfda0f/dashboard-apis-list.png)</Frame>
2. Accédez à l’onglet **Permissions**, cliquez sur l’icône de corbeille à côté de la permission que vous souhaitez supprimer, puis confirmez.

   <Frame>![Dashboard Add API Permissions API Define Permissions Screen](/docs/images/cdy7uua7fh8z/32Pb185OFs2mC6z2fmunEw/2708c52c2869d016066cc456dd00b6a2/dashboard-applications-apis-permissions.png)</Frame>

<div id="use-the-management-api">
  ## Utiliser la Management API
</div>

Effectuez une requête PATCH vers le [point de terminaison Update Resource Server](https://auth0.com/docs/api/management/v2#!/Resource_Servers/patch_resource_servers_by_id) en incluant toutes les permissions que vous souhaitez conserver et en excluant toutes celles que vous souhaitez supprimer. Remplacez les valeurs de l’espace réservé `API_ID,` `MGMT_API_ACCESS_TOKEN`, `PERMISSION_NAME` et `PERMISSION_DESC` par l’ID de votre API, le <Tooltip tip="Jeton d’accès : information d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Access+Token">jeton d’accès</Tooltip> de la Management API, le ou les noms de permission et la ou les descriptions de permission, respectivement.

  ```bash cURL
  curl --request PATCH \
    --url 'https://{yourDomain}/api/v2/resource-servers/API_ID' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
    --header 'cache-control: no-cache' \
    --header 'content-type: application/json' \
    --data '{ "scopes": [ { "value": "PERMISSION_NAME", "description": "PERMISSION_DESC" }, { "value": "PERMISSION_NAME", "description": "PERMISSION_DESC" } ] }'
  ```

</AuthCodeGroup>

| Valeur                  | Description                                                                                                                      |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `API_ID`                | ID de l’API pour laquelle vous souhaitez supprimer des permissions.                                                              |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec le scope `update:resource_servers`. |
| `PERMISSION_NAME`       | Nom(s) de la ou des permissions que vous souhaitez conserver pour l’API indiquée.                                                |
| `PERMISSION_DESC`       | Description(s) conviviale(s) de la ou des permissions que vous souhaitez conserver pour l’API indiquée.                          |
