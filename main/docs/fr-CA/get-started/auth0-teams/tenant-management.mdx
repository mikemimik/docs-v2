Accédez aux informations de votre Team et modifiez-les depuis votre tenant. Si aucun tenant n&#39;est encore associé à votre compte, consultez [Create Tenants.](/docs/fr-CA/get-started/auth0-overview/create-tenants)

<div id="access-the-teams-dashboard">
  ## Accéder au Teams Dashboard
</div>

Si vous êtes un Team Owner, vous pouvez utiliser le Teams Dashboard pour :

* Choisir qui peut créer de nouveaux tenants : les Team Owners ou tous les administrateurs de tenant.
* Voir tous les tenants créés dans le cadre de votre abonnement Auth0 ou de votre entente personnalisée.
* Voir les Team Owners et les inviter.

Pour accéder au Teams Dashboard à partir de n’importe lequel de vos tenants :

1. Sélectionnez le nom de votre tenant dans le menu déroulant des tenants du [Auth0 Dashboard](https://manage.auth0.com/dashboard).
2. Repérez le nom de votre Team au bas du menu et sélectionnez **Accéder à la Team**.

<Frame>![Accéder au tableau de bord d’Auth0 Team](/docs/images/cdy7uua7fh8z/3207T6ELEBc56dLZT84hjT/13779947b38dd52ab4694b3869271961/Tenant_Management_-_Teams.png)</Frame>

Pour accéder au Teams Dashboard à l’aide d’une URL :

1. Accédez à `https://accounts.auth0.com/`
2. Si votre compte d’utilisateur n’est membre que d’un seul Team, entrez vos informations d’authentification pour accéder à votre Teams Dashboard. Si votre compte d’utilisateur est membre d’un ou de plusieurs Teams, sélectionnez le Team dans la liste des Teams auxquels votre compte d’utilisateur est associé pour vous connecter.

Si l’option d’accès à Teams n’est pas visible, essayez ce qui suit :

| Cause possible                                | Action                                               |
| --------------------------------------------- | ---------------------------------------------------- |
| Le tenant actuel ne fait pas partie de Teams. | Essayez de passer à un autre tenant.                 |
| Vous n’êtes pas Team Owner.                   | Demandez à un Team Owner de vous inviter à la Team. |

<div id="view-tenants">
  ## Afficher les tenants
</div>

Dans le Teams Dashboard, vous pouvez afficher tous les tenants compris dans votre entente personnalisée ou votre abonnement Auth0, qu’ils se trouvent dans un environnement Public ou Private Cloud. Pour chaque tenant, vous pouvez consulter :

* Nom du tenant
* Region
* Date de création
* Environnement (Development, Staging ou Production)
* administrateur de tenant

1. Sélectionnez **Tenants** dans le menu de gauche du Teams Dashboard.

   Pour les environnements Private Cloud, sélectionnez le menu déroulant **Overview** pour afficher l’option de menu **Tenants**.
2. Si vous êtes administrateur d’un tenant, sélectionnez le nom du tenant pour y accéder. Si vous n’êtes pas administrateur de tenant, vous ne pourrez pas sélectionner ce tenant.
3. Pour consulter les membres du tenant, sélectionnez **View Members**.

<div id="view-private-cloud-environments">
  ## Afficher les environnements Private Cloud
</div>

La liste des environnements Private Cloud est accessible à partir du Teams Dashboard. Vous pouvez y consulter les renseignements suivants :

* Nom de l’environnement
* Type de fournisseur infonuagique (AWS ou Azure)
* Région de déploiement
* Région de basculement
* Version actuelle de l’environnement

1. Dans le menu déroulant **Overview**, sélectionnez l’option de menu **Private Cloud**.

<div id="link-tenants">
  ## Lier des tenants
</div>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Cette fonctionnalité est actuellement offerte pour les types de Team Subscription suivants :

  * B2C - Essentials
  * B2B - Essentials
  * B2C - Professional
  * B2B - Professional
</Callout>

La liaison de tenants vous permet de partager avec le tenant lié les fonctionnalités et les quotas offerts dans votre Team Subscription. Une fois le tenant lié, ses abonnements seront rattachés à votre Team et vous n’aurez plus à gérer son abonnement individuellement.

Vous pouvez lier des tenants à votre Team si les conditions suivantes sont remplies :

* Le tenant ne fait pas déjà partie d’une Team.
* Vous êtes administrateur du tenant que vous souhaitez lier.
* Vous avez le rôle de Team Owner dans la Team.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  La liaison d’un tenant annule l’abonnement associé à ce tenant.
</Callout>

Si ces conditions sont remplies, le tenant s’affichera comme option de liaison possible à votre Team lorsque vous sélectionnerez **Link Existing Tenant** dans la page Tenants.

<div id="create-new-tenant">
  ## Créer un nouveau tenant
</div>

<Warning>
  Cette fonctionnalité est actuellement offerte pour les types de Team Subscription suivants :

  * B2C - Essentials
  * B2B - Essentials
  * B2C - Professional
  * B2B - Professional
  * Enterprise Private Cloud

  En utilisant cette fonctionnalité, vous acceptez les conditions applicables de l’essai gratuit énoncées dans le [Master Subscription Agreement](https://www.okta.com/legal/) d’Okta.
</Warning>

Les Team Owners peuvent créer un tenant à partir du Teams Dashboard, ce qui lie automatiquement le tenant créé à la Team, y compris l’association des détails de la Team Subscription (fonctionnalités offertes et limites de quota) au tenant nouvellement créé.

Pour ce faire, cliquez sur le bouton **Create New Tenant** dans la page Tenants, ce qui ouvrira le formulaire Create Tenant.

<div id="manage-tenants">
  ## Gérer les tenants
</div>

Vous pouvez contrôler qui peut créer de nouveaux tenants dans le cadre de votre entente personnalisée ou de votre abonnement Auth0, sous **Team Settings**. Lors de la création d’un nouveau tenant, les administrateurs peuvent :

* Lier le tenant à une entente personnalisée. Le nouveau tenant est alors couvert par votre entente personnalisée ou votre abonnement. Il bénéficie des mêmes fonctionnalités, compte dans les limites et est visible pour les Team Owners.
  ou
* Créer un tenant sous leur compte personnel. Le nouveau tenant ne fait pas partie de l’entente personnalisée ou de l’abonnement de votre équipe Teams. L’administrateur qui crée le tenant peut y accéder avec les mêmes identifiants de connexion.

Pour permettre aux administrateurs de lier un tenant à une entente personnalisée, le Team Owner doit :

1. Sélectionner **Settings** dans la partie gauche du Teams Dashboard.
2. Sélectionner l’option **Allow Tenant Admins to Create Tenants under the Teams Account**. Si cette option n’est pas cochée, les administrateurs de tenant peuvent uniquement créer des comptes personnels.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     L’option **Tenant Creation** ne s’applique pas aux Team Owners. Les propriétaires ont les permissions requises pour créer de nouveaux tenants dans l’instance Teams dans le cadre de l’entente personnalisée ou de l’abonnement.
   </Callout>
3. Sélectionner **Save**.

   <Warning>
     Si un administrateur de tenant sans permission tente de créer des tenants, il sera invité à communiquer avec le Team Owner.
   </Warning>
