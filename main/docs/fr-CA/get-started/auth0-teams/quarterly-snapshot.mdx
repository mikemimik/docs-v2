title: À propos de Quarterly Snapshot
---

Quarterly Snapshot présente les points saillants de l’utilisation et de la valeur au niveau du compte, y compris des renseignements agrégés sur votre utilisation par rapport au quota, votre environnement d’identité et la détection des menaces.

La fonctionnalité est activée automatiquement lorsqu’Auth0 Teams est provisionné. Elle est offerte exclusivement à certains clients d’entreprise gérés par leur équipe de compte.

<div id="things-to-know">
  ## Points à savoir
</div>

* Le Quarterly Snapshot est accessible uniquement dans [Auth0 Teams](/docs/fr-CA/get-started/auth0-teams). Auth0 Teams doit être provisionné pour le compte avant le début du trimestre civil.
* Le Quarterly Snapshot est une fonctionnalité réservée à un groupe restreint de clients d’entreprise gérés par le DTAM.
* Les Team Owners et les Report Viewers peuvent consulter et télécharger le Quarterly Snapshot dans la section **Reports** du tableau de bord Auth0 Teams.
* Les Team Owners peuvent gérer la liste des Report Viewers dans le tableau de bord Auth0 Teams.
* Les objets de métriques dans les Quarterly Snapshots peuvent varier selon les droits du compte, les fonctionnalités activées et l’utilisation qui en est faite.
* Toutes les métriques sont agrégées au niveau du compte. Pour en savoir plus, consultez le tableau **Métriques** ci-dessous.

<div id="metrics">
  ## Métriques
</div>

Voici les métriques que vous pourriez voir dans votre Quarterly Snapshot.

| Mesure                                | Ce qu’elle mesure                                                                                                                                                                                                                                                                                                                                                            | Commentaires                                                                                                                                            |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Utilisateurs actifs mensuels          | Mesure le nombre mensuel d’utilisateurs actifs uniques du compte par rapport à votre quota.                                                                                                                                                                                                                                                                                  | Pour des renseignements détaillés au niveau d’un tenant, veuillez consulter le rapport [Support Center Quota](https://support.auth0.com/reports/quota). |
| Jetons Machine to Machine             | Affiche le nombre de jetons d’accès émis par Auth0 pour l’[octroi des identifiants du client](/docs/fr-CA/get-started/authentication-and-authorization-flow/client-credentials-flow). Les jetons émis pour l’Auth0 Management API ou d’autres API intégrées d’Auth0 ne sont pas comptabilisés.                                                                                     | Pour des renseignements détaillés au niveau d’un tenant, veuillez consulter le rapport [Support Center Quota](https://support.auth0.com/reports/quota). |
| Enterprise Connections actives        | Cette mesure affiche le nombre d’enterprise connections utilisées par un compte.                                                                                                                                                                                                                                                                                             | Pour plus de détails sur votre quota, communiquez avec votre gestionnaire de compte.                                                                    |
| Nouvelles inscriptions d’utilisateurs | Affiche combien de nouveaux utilisateurs (inscriptions d’utilisateurs) ont été ajoutés à votre ou vos tenants pendant les mois de ce trimestre.                                                                                                                                                                                                                              | Consultez la [page Activity](/docs/fr-CA/get-started/auth0-overview/dashboard/activity) pour en savoir plus sur les nouveaux utilisateurs.                    |
| Utilisateurs inactifs                 | Cette mesure représente la moyenne mensuelle d’utilisateurs qui ne se sont pas authentifiés au cours du dernier mois.                                                                                                                                                                                                                                                        |                                                                                                                                                         |
| Breached Password                     | Nombre de tentatives bloquées en raison d’informations d’identification compromises par la fonctionnalité de [détection des mots de passe compromis](/docs/fr-CA/secure/attack-protection/breached-password-detection).                                                                                                                                                            |                                                                                                                                                         |
| Brute-force Protection                | Requêtes bloquées par la fonctionnalité [Brute-force Protection](/docs/fr-CA/secure/attack-protection/brute-force-protection), si elle est activée. Cette mesure résume toutes les requêtes bloquées spécifiquement sur les tenants de production.                                                                                                                                 |                                                                                                                                                         |
| Suspicious IP Throttling              | Requêtes bloquées par la fonctionnalité [Suspicious IP Throttling](/docs/fr-CA/secure/attack-protection/suspicious-ip-throttling), si elle est activée. Cette mesure additionne toutes les requêtes bloquées spécifiquement sur les tenants de production.                                                                                                                         |                                                                                                                                                         |
| Bot Detection                         | Requêtes qui n’ont pas réussi l’épreuve CAPTCHA. Cette mesure additionne toutes les requêtes de robots bloquées par la fonctionnalité [Bot Detection](/docs/fr-CA/secure/attack-protection/bot-detection) après l’échec d’une épreuve CAPTCHA.                                                                                                                                     |                                                                                                                                                         |
| Adaptive MFA                          | Requêtes pour lesquelles une demande de vérification [MFA](/docs/fr-CA/secure/multi-factor-authentication/adaptive-mfa) a été émise en raison d’un faible score de confiance. Cette mesure additionne toutes les tentatives de connexion pour lesquelles une demande de vérification d’authentification multifacteur a été émise et où le score de confiance calculé était faible. |                                                                                                                                                         |
