<div id="application-architecture-scenarios">
  ## Scénarios d’architecture d’application
</div>

Ces scénarios décrivent les différents types d’architectures technologiques que votre application peut utiliser et la façon dont Auth0 peut vous aider. Leur objectif est de vous guider tout au long du processus de mise en œuvre, du début à la fin.

| Lire...                                                                                                                                | Pour en savoir plus sur...                                                                                                                                                                                                  |
| -------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Organisations multiples](/docs/fr-CA/get-started/architecture-scenarios/multiple-organization-architecture)                                 | Comment configurer une architecture d’application qui utilise soit des organisations à fournisseur d’identité unique, soit des organisations à fournisseurs d’identité multiples.                                           |
| [Authentification unique pour les applications Web traditionnelles](/docs/fr-CA/get-started/architecture-scenarios/sso-for-regular-web-apps) | Comment configurer et mettre en œuvre une application Web qui utilise l’authentification unique pour les employés qui utilisent Active Directory et les sous-traitants qui utilisent un répertoire d’utilisateurs distinct. |
| [Application serveur + API](/docs/fr-CA/get-started/architecture-scenarios/server-application-api)                                           | Comment configurer une API de feuille de temps à l’aide d’une tâche cron pour enregistrer des entrées de feuille de temps.                                                                                                  |
| [Application monopage + API](/docs/fr-CA/get-started/architecture-scenarios/spa-api)                                                         | Comment configurer une API de feuille de temps à l’aide d’une application monopage pour enregistrer des entrées de feuille de temps.                                                                                        |
| [Application mobile + API](/docs/fr-CA/get-started/architecture-scenarios/mobile-api)                                                        | Comment configurer une API de feuille de temps à l’aide d’une application mobile pour enregistrer des entrées de feuille de temps dans une base de données centralisée.                                                     |
