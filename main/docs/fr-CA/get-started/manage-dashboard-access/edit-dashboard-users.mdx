---
description: Découvrez comment modifier les rôles des utilisateurs de l’Auth0 Dashboard pour votre tenant Auth0.
title: Modifier les membres du tenant
---

Si vous êtes un administrateur de tenant, vous pouvez modifier le ou les rôles attribués à un utilisateur. Les modifications prennent effet une fois que l’utilisateur a actualisé son Dashboard.

Vous pouvez attribuer plus d’un rôle à un utilisateur, et les permissions seront combinées. Vous devez d’abord avoir ajouté le membre du tenant.

1. Accédez à [Dashboard &gt; Settings &gt; Tenant Members](https://manage.auth0.com/#/tenant/admins).
2. Cliquez sur **...** à droite du nom ou de l’adresse courriel d’un utilisateur, puis sur **Edit Roles**.
   Si vous ne voyez pas cette option, cela signifie que l’utilisateur n’a pas encore accepté son invitation.
3. Sélectionnez des rôles supplémentaires ou supprimez des rôles.
4. Cliquez sur **Save**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Les permissions de l’utilisateur seront visibles lorsqu’il actualisera son Dashboard. S’il ne l’actualise pas, il pourrait voir des erreurs indiquant qu’il n’a pas accès à une zone précise. Si l’une de ces erreurs se produit, l’utilisateur devra se déconnecter, puis se reconnecter.
</Callout>

**Réinitialisation du mot de passe** : Pour en savoir plus, consultez [Reset Auth0 Account Password](/docs/fr-CA/troubleshoot/customer-support/reset-account-passwords) si vous ou l’un de vos utilisateurs du Dashboard avez des difficultés à vous connecter.

**Adresse courriel invalide** : Pour en savoir plus, consultez [Update Dashboard User Email Addresses](/docs/fr-CA/get-started/manage-dashboard-access/update-dashboard-user-email) pour afficher les adresses courriel de vos membres du tenant et les mettre à jour au besoin.
