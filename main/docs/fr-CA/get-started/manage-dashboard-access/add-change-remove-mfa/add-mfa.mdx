Chaque utilisateur d’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> devrait s’inscrire lui-même à l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA). Vous pouvez vous inscrire à la plupart des facteurs dans [Votre profil](https://manage.auth0.com/#/profile). La biométrie de l’appareil, toutefois, nécessite l’inscription progressive.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 recommande les facteurs WebAuthn comme méthodes d’authentification les plus sûres et les plus conviviales. Pour en savoir plus, consultez [FIDO Authentication with WebAuthn](/docs/fr-CA/secure/multi-factor-authentication/fido-authentication-with-webauthn).
</Callout>

<Warning>
  Les Admins doivent activer au moins un facteur pour utiliser l’authentification multifacteur (MFA). Auth0 recommande fortement de configurer plusieurs facteurs afin que vous puissiez quand même accéder à votre compte si vous perdez votre appareil principal.

  Une configuration idéale consiste à utiliser trois facteurs :

  1. WebAuthn, Guardian ou OTP comme méthode principale.
  2. Un ou plusieurs numéros SMS comme solution de secours (offert uniquement sur les tenants rattachés à un abonnement).
  3. Un code de récupération.

  Si vous ne pouvez pas fournir votre jeton MFA et que vous n’avez pas de méthodes de secours adéquates, il se peut que votre compte soit impossible à récupérer.
</Warning>

<div id="add-mfa">
  ## Ajouter la MFA
</div>

Pour vous inscrire vous-même à la MFA, chaque utilisateur du Dashboard doit suivre les étapes suivantes :

1. Cliquez sur votre nom d’utilisateur dans le coin supérieur droit du Dashboard, puis sur [Votre profil](https://manage.auth0.com/#/profile).
2. Repérez la méthode prise en charge que vous souhaitez utiliser, puis cliquez sur **+ ADD** dans cette ligne.

   <Frame>![Dashboard - Profil - Authentification multifacteur](/docs/images/cdy7uua7fh8z/2NyLNvP92RruwZ46dLPm5x/8ece3df2b550af8dd78368ca36d8a661/MFA.png)</Frame>
3. Suivez les instructions à l’écran pour terminer l’enrôlement.

<div id="device-biometrics">
  ## Biométrie de l’appareil
</div>

WebAuthn avec biométrie de l’appareil est la seule méthode que vous ne pouvez pas ajouter dans la page Paramètres du compte. À la place, Auth0 enrôle progressivement tous vos appareils compatibles avec WebAuthn. Auth0 vous invite à enrôler ces appareils après avoir enrôlé toute autre méthode MFA. Ces invites reviennent chaque fois que vous vous connectez à Auth0 Dashboard.

<Frame>![undefined](/docs/images/cdy7uua7fh8z/1cBjDSZrIeHAS2Fyzbt5HK/7045d8e2c5b0b0588d5eb88f4a5620b0/deviceenrollment.png)</Frame>

Dans le cadre de l’enrôlement, Auth0 vous invite à nommer vos appareils. Vous pouvez ainsi les gérer facilement depuis la page Paramètres du compte.

Les navigateurs dans lesquels Javascript est désactivé ou qui ne prennent pas en charge l’authentificateur de plateforme WebAuthn ne peuvent ni enrôler ni authentifier avec la biométrie de l’appareil. Les versions les plus récentes des navigateurs et systèmes d’exploitation populaires prennent en charge WebAuthn with Security Keys. Pour en savoir plus, consultez la [section sur la compatibilité des navigateurs sur webauthn.me](https://webauthn.me/browser-support).

<div id="recovery-codes">
  ## Codes de récupération
</div>

Immédiatement après avoir activé avec succès l’authentification à deux facteurs, Auth0 vous invite à copier un code de récupération. Si vous perdez l’accès à tous vos facteurs déjà configurés, vous pouvez utiliser ce code de récupération pour vous connecter à votre compte. Auth0 recommande de copier et d’imprimer vos codes de récupération, ou de les conserver dans un endroit sûr, comme un gestionnaire de mots de passe.

Si vous perdez vos codes de récupération ou souhaitez simplement en générer de nouveaux, vous pouvez le faire à partir de [Votre profil](https://manage.auth0.com/#/profile).

<div id="log-in-to-the-dashboard-with-mfa-enabled">
  ## Connectez-vous au Dashboard avec MFA activée
</div>

Se connecter avec MFA activée diffère à peine d’une connexion normale. Lorsque vous entrez les identifiants de votre compte Admin, un deuxième écran s’affiche, selon le type de facteurs MFA que vous avez activés.

Si un utilisateur perd l’accès à un facteur principal, il peut cliquer sur **Select Another Method** et essayer l’une des autres méthodes, y compris les codes de récupération. C’est pourquoi il est si important d’en configurer plusieurs pour éviter de perdre l’accès à votre compte.

Après avoir ajouté avec succès votre deuxième facteur d’authentification et vous être connecté à partir d’un nouvel appareil compatible avec WebAuthn, une invite vous propose de « vous connecter plus rapidement sur cet appareil ». Cela vous permet d’utiliser cet appareil pour l’authentification multifacteur lors de votre prochaine connexion.
