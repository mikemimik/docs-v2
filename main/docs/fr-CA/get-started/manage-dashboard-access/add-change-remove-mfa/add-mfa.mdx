title: Ajouter l’authentification multifacteur pour l’accès à Auth0 Dashboard
---

Chaque utilisateur du <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> devrait s’inscrire lui-même à l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA). Vous pouvez vous inscrire à la plupart des facteurs dans [Your Profile](https://manage.auth0.com/#/profile). La biométrie de l’appareil, cependant, nécessite une inscription progressive.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 recommande les facteurs WebAuthn comme méthodes d’authentification les plus sécuritaires et les plus conviviales. Pour en savoir plus, consultez [FIDO Authentication with WebAuthn](/docs/fr-CA/secure/multi-factor-authentication/fido-authentication-with-webauthn).
</Callout>

<Warning>
  Les administrateurs doivent activer au moins un facteur pour utiliser la MFA. Auth0 recommande fortement de configurer plusieurs facteurs afin que vous puissiez tout de même accéder à votre compte si vous perdez votre appareil principal.

  La configuration idéale consiste à utiliser trois facteurs :

  1. WebAuthn, Guardian ou OTP comme méthode principale.
  2. Un ou plusieurs numéros SMS comme solution de secours (offert uniquement pour les tenants associés à un plan d’abonnement).
  3. Un code de récupération.

  Si vous ne pouvez pas fournir votre jeton MFA et que vous n’avez pas de méthode de secours appropriée, votre compte pourrait être irrécupérable.
</Warning>

<div id="add-mfa">
  ## Ajouter l’authentification multifacteur
</div>

Pour vous inscrire vous-même à l’authentification multifacteur, chaque utilisateur du Dashboard doit suivre les étapes suivantes :

1. Cliquez sur votre nom d’utilisateur dans le coin supérieur droit du Dashboard, puis sur [Your Profile](https://manage.auth0.com/#/profile).
2. Repérez la méthode prise en charge souhaitée et cliquez sur **+ ADD** sur cette ligne.

   <Frame>![Dashboard - Profil - Authentification multifacteur](/docs/images/cdy7uua7fh8z/2NyLNvP92RruwZ46dLPm5x/8ece3df2b550af8dd78368ca36d8a661/MFA.png)</Frame>
3. Suivez les instructions à l’écran pour terminer l’inscription.

<div id="device-biometrics">
  ## Biométrie de l’appareil
</div>

WebAuthn avec biométrie de l’appareil est la seule méthode que vous ne pouvez pas ajouter sur la page Paramètres du compte. À la place, Auth0 inscrit progressivement tous vos appareils compatibles avec WebAuthn. Auth0 vous demande d’inscrire ces appareils après l’inscription de toute autre méthode MFA. Ces invites s’affichent chaque fois que vous vous connectez à Auth0 Dashboard.

<Frame>![undefined](/docs/images/cdy7uua7fh8z/1cBjDSZrIeHAS2Fyzbt5HK/7045d8e2c5b0b0588d5eb88f4a5620b0/deviceenrollment.png)</Frame>

Dans le cadre de l’inscription, Auth0 vous demande de nommer vos appareils. Cela facilite leur gestion à partir de la page Paramètres du compte.

Les navigateurs dans lesquels Javascript est désactivé, ou qui ne prennent pas en charge un authentificateur de plateforme WebAuthn, ne permettent ni l’inscription ni l’authentification avec la biométrie de l’appareil. Les versions les plus récentes des navigateurs et des systèmes d’exploitation populaires prennent en charge WebAuthn avec des clés de sécurité. Pour en savoir plus, consultez la [section sur la compatibilité des navigateurs sur webauthn.me](https://webauthn.me/browser-support).

<div id="recovery-codes">
  ## Codes de récupération
</div>

Immédiatement après avoir activé l’authentification à deux facteurs, Auth0 vous invite à copier un code de récupération. Si vous perdez l’accès à tous les facteurs que vous avez configurés, vous pouvez utiliser ce code de récupération pour vous connecter à votre compte. Auth0 recommande de copier et d’imprimer vos codes de récupération, ou de les conserver dans un endroit sûr, comme un gestionnaire de mots de passe.

Si vous perdez vos codes de récupération ou souhaitez simplement en générer de nouveaux, vous pouvez le faire à partir de [Your Profile](https://manage.auth0.com/#/profile).

<div id="log-in-to-the-dashboard-with-mfa-enabled">
  ## Se connecter au Dashboard avec la MFA activée
</div>

Se connecter avec la MFA activée diffère seulement légèrement d’une connexion normale. Lorsque vous entrez les informations d’identification du compte Admin, un deuxième écran s’affiche, selon le type de facteurs MFA que vous avez activés.

Si un utilisateur perd l’accès à un facteur principal, il peut cliquer sur **Sélectionner une autre méthode** et essayer n’importe lequel des autres facteurs, y compris les codes de récupération. C’est pourquoi il est si important d’activer plusieurs méthodes afin d’éviter de perdre l’accès à votre compte.

Après avoir ajouté avec succès votre deuxième facteur d’authentification et vous être connecté à partir d’un nouvel appareil qui prend en charge WebAuthn, un écran vous invite à « Se connecter plus rapidement sur cet appareil ». Cela vous permet d’utiliser cet appareil pour la MFA la prochaine fois.
