Le serveur Auth0 Model Context Protocol (MCP) connecte les agents IA à votre tenant Auth0, ce qui leur permet d’y effectuer des opérations complexes en plusieurs étapes, comme créer des applications, gérer des utilisateurs ou déployer des Actions. Il s’appuie sur le [Model Context Protocol (MCP)](https://modelcontextprotocol.io/introduction), un protocole standardisé qui permet aux agents IA de communiquer avec des outils externes et des API afin d’exécuter des tâches en votre nom.

Conçu pour les développeurs qui utilisent des outils d’IA comme Cursor, Windsurf ou Claude Desktop, le serveur Auth0 MCP transforme la façon dont ils gèrent leur infrastructure d’identité et permet d’utiliser les services Auth0 plus rapidement et plus facilement.

<div id="authentication-and-credential-security">
  ## Sécurité de l’authentification et des identifiants
</div>

Auth0 MCP Server met l’accent sur une sécurité renforcée en gérant l’authentification, en validant les requêtes et en assurant une communication sécurisée avec la <Tooltip tip="Management API : Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip> d’Auth0. Cela comprend :

* [OAuth 2.0 Device Authorization Flow](/docs/fr-CA/get-started/authentication-and-authorization-flow/device-authorization-flow) : authentifie de façon sécurisée les requêtes auprès d’Auth0.
* [Stockage sécurisé des identifiants](/docs/fr-CA/get-started/applications/rotate-credentials) : les identifiants sont stockés de façon sécurisée dans le trousseau de votre système et ne sont jamais exposés en texte brut.
* [Permissions d’API minimales](/docs/fr-CA/secure/tokens/access-tokens/management-api-access-tokens) : seules les permissions essentielles sont demandées, dans le strict respect du principe du moindre privilège.

<div id="supported-tools-and-integrations">
  ## Outils et intégrations pris en charge
</div>

Auth0 MCP Server s’intègre actuellement aux [clients MCP](https://modelcontextprotocol.io/clients) suivants, notamment :

* [Claude Desktop](https://claude.ai/download)
* [Cursor](https://www.cursor.com/downloads)
* [Windsurf](https://www.windsurf.com/editor)

D’autres intégrations seront ajoutées à mesure que de nouveaux clients d’envergure feront leur apparition. Pour consulter la liste complète des outils pris en charge et le détail des fonctionnalités, veuillez consulter notre [Référence des outils du serveur Auth0 Model Context Protocol (MCP)](/docs/fr-CA/get-started/auth0-mcp-server/auth0-mcp-tools-reference).
