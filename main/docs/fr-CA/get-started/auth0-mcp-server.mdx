Le serveur Auth0 Model Context Protocol (MCP) connecte les agents d’IA à votre tenant Auth0, ce qui leur permet d’exécuter dans Auth0 des opérations complexes en plusieurs étapes, comme créer des applications, gérer des utilisateurs ou déployer des Actions. Il s’appuie sur le [Model Context Protocol (MCP)](https://modelcontextprotocol.io/introduction), un protocole normalisé qui permet aux agents d’IA de communiquer avec des outils externes et des API afin d’exécuter des tâches en votre nom.

Conçu pour les développeurs qui utilisent des outils propulsés par l’IA comme Cursor, Windsurf ou Claude Desktop, le Auth0 MCP Server transforme la façon dont les développeurs gèrent leur infrastructure d’identité, en rendant l’utilisation des services Auth0 plus simple et plus rapide.

<div id="authentication-and-credential-security">
  ## Sécurité de l’authentification et des identifiants
</div>

Auth0 MCP Server met l’accent sur une sécurité renforcée en gérant l’authentification, en validant les requêtes et en assurant une communication sécurisée avec l’<Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip> d’Auth0. Cela comprend :

* [OAuth 2.0 Device Authorization Flow](/docs/fr-CA/get-started/authentication-and-authorization-flow/device-authorization-flow) : permet d’authentifier les requêtes auprès d’Auth0 en toute sécurité.
* [Stockage sécurisé des identifiants](/docs/fr-CA/get-started/applications/rotate-credentials) : les identifiants sont stockés de façon sécurisée dans le trousseau de clés de votre système et ne sont jamais exposés en texte clair.
* [Permissions minimales de l’API](/docs/fr-CA/secure/tokens/access-tokens/management-api-access-tokens) : seules les permissions essentielles sont demandées, dans le respect strict du principe du moindre privilège.

<div id="supported-tools-and-integrations">
  ## Outils et intégrations pris en charge
</div>

Auth0 MCP Server s’intègre actuellement aux [clients MCP](https://modelcontextprotocol.io/clients) suivants, y compris :

* [Claude Desktop](https://claude.ai/download)
* [Cursor](https://www.cursor.com/downloads)
* [Windsurf](https://www.windsurf.com/editor)

D’autres intégrations seront ajoutées à mesure que de nouveaux clients importants feront leur apparition. Pour consulter la liste complète des outils pris en charge et le détail de leurs fonctionnalités, veuillez vous reporter à notre [Référence des outils d’Auth0 Model Context Protocol (MCP) Server](/docs/fr-CA/get-started/auth0-mcp-server/auth0-mcp-tools-reference).
