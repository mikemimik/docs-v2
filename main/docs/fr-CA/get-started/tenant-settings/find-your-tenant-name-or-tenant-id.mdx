title: Trouver le nom de votre tenant ou son ID
---

Chaque tenant Auth0 a un nom. Vous utilisez souvent ce nom dans votre code pour identifier le tenant. Lorsque vous voyez les variables `{yourTenantId}`, `{yourTenantName}`, `{yourTenant}` ou une variable semblable dans la documentation Auth0, remplacez-les par le nom du tenant.

1. Accédez à l’[Auth0 Dashboard](https://manage.auth0.com/#).
2. Dans le coin supérieur gauche, repérez le nom du tenant à côté du logo Auth0.
