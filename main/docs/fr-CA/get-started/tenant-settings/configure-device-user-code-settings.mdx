title: Configurer les paramètres du code utilisateur pour les appareils
---

Vous pouvez configurer les paramètres du code utilisateur généré par votre application pendant le [flux d’autorisation de l’appareil](/docs/fr-CA/get-started/authentication-and-authorization-flow/device-authorization-flow) dans le <Tooltip tip="Auth0 Dashboard : interface principale d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Accédez à [Dashboard &gt; Settings](https://manage.auth0.com/#/tenant) et cliquez sur l’onglet **Advanced**.
2. Faites défiler jusqu’à la section **Device Flow User Code Format**, repérez **User Code Character Set** et **User Code Mask**, saisissez les paramètres souhaités, puis cliquez sur **Save**.

| Paramètre                        | Description                                                                                                                                                                         |
| -------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Device Flow User Code Format** | Jeu de caractères utilisé pour générer aléatoirement un code utilisateur.                                                                                                           |
| **User Code Mask**               | Masque utilisé pour définir la longueur et le format d’un code utilisateur généré aléatoirement. Il vise à améliorer la lisibilité du code utilisateur et à en faciliter la saisie. |
