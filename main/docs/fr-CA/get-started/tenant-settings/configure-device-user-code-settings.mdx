title: Configurer les paramètres du Device User Code
---

Vous pouvez configurer les paramètres du user code généré par votre application dans le cadre du [flux d’autorisation de l’appareil](/docs/fr-CA/get-started/authentication-and-authorization-flow/device-authorization-flow) à l’aide de l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Accédez à [Dashboard &gt; Settings](https://manage.auth0.com/#/tenant), puis cliquez sur l’onglet **Advanced**.
2. Faites défiler jusqu’à la section **Device Flow User Code Format**, repérez **User Code Character Set** et **User Code Mask**, entrez les paramètres souhaités, puis cliquez sur **Save**.

| Paramètre                        | Description                                                                                                                                                 |
| -------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Device Flow User Code Format** | Jeu de caractères utilisé pour générer aléatoirement un user code.                                                                                          |
| **User Code Mask**               | Masque utilisé pour définir la longueur et le format d’un user code généré aléatoirement. Il vise à en améliorer la lisibilité et à en faciliter la saisie. |
