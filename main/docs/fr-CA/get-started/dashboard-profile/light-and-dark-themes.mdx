---
description: Comment passer du thème clair au thème sombre dans votre Dashboard
title: Thèmes clair et sombre
---

Les clients d’Auth0 peuvent utiliser le mode clair ou le mode sombre dans le Dashboard de leur tenant. Vous pouvez aussi configurer votre Dashboard pour qu’il utilise les mêmes réglages que votre système local.

Il y a deux façons de modifier le thème de votre Dashboard :

1. Sélectionnez le menu déroulant de profil en haut à droite de votre Dashboard.
2. Utilisez le menu pour passer d’un thème à l’autre.

<Frame>![Où trouver la bascule des modes sombre et clair](/docs/images/cdy7uua7fh8z/1BImxFvHXoULQVTkqCsQL0/a37f6cfdfce935fd20731aa68f8067de/2024-09-10_09-08-27.png)</Frame>

Vous pouvez aussi cliquer sur **Votre profil** dans le même menu déroulant et faire défiler la page jusqu’en bas, où vous pouvez choisir parmi les trois thèmes.

<Frame>![Où modifier votre thème dans la section Profil](/docs/images/cdy7uua7fh8z/231W9kPYTeEABOsV3rOZxF/1c451d4942d86432a0e701f497c32f24/Light_mode_profile_section.png)</Frame>
