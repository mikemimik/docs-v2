---
description: Comment basculer entre les thèmes clair et sombre dans votre Dashboard
title: Thèmes clair et sombre
---

Les clients d’Auth0 peuvent utiliser le mode clair ou le mode sombre dans le Dashboard de leur tenant. Vous pouvez aussi configurer votre Dashboard pour qu’il utilise les mêmes paramètres que votre système local.

Il existe deux façons de changer le thème de votre Dashboard :

1. Sélectionnez le menu déroulant de profil en haut à droite de votre Dashboard.
2. Utilisez le menu pour passer d’un thème à l’autre.

<Frame>![Où trouver la bascule entre le mode sombre et le mode clair](/docs/images/cdy7uua7fh8z/1BImxFvHXoULQVTkqCsQL0/a37f6cfdfce935fd20731aa68f8067de/2024-09-10_09-08-27.png)</Frame>

Vous pouvez également cliquer sur **Your Profile** dans ce même menu déroulant et faire défiler jusqu’au bas de la page, où vous pourrez choisir parmi les trois thèmes.

<Frame>![Où modifier votre thème dans la section de profil](/docs/images/cdy7uua7fh8z/231W9kPYTeEABOsV3rOZxF/1c451d4942d86432a0e701f497c32f24/Light_mode_profile_section.png)</Frame>
