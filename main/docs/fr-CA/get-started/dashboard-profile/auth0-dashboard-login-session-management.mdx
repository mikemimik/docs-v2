---
description: Décrit comment les administrateurs d’Auth0 Dashboard peuvent voir et révoquer les sessions de connexion.
title: Gestion des sessions de connexion à Auth0 Dashboard
---

Avec les sessions de connexion, les administrateurs d’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> peuvent suivre les sessions actives provenant de différentes applications du Dashboard sur plusieurs appareils et navigateurs. Vous pouvez examiner et révoquer les sessions créées par des utilisateurs qui accèdent à Teams Dashboard, Support Center et Auth0 Dashboard.

Vous pouvez voir la liste de toutes les sessions actives sur la [page de profil](https://manage.auth0.com/#/profile/sessions) de votre Auth0 Dashboard.

<Frame>![Auth0 Dashboard > Profile](/docs/images/cdy7uua7fh8z/3hBNRo2Bc4w6oKpF0mMt4o/3796b322593eae38eeba0a3af3698eca/English_user_profle_login_sessions.png)</Frame>

Pour vous déconnecter ou révoquer une session, repérez la session voulue et sélectionnez **Revoke**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  La révocation d’une session ne supprime pas automatiquement les sessions associées aux extensions. Toutefois, ces sessions sont de courte durée et ne sont pas renouvelées lorsqu’une session de connexion est révoquée ou expire.
</Callout>
