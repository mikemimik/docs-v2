---
description: Décrit comment créer un tenant supplémentaire à l’aide de l’Auth0 Dashboard.
title: Créer plusieurs tenants
---

Vous pouvez configurer plusieurs tenants pour créer [différents environnements](/docs/fr-CA/get-started/auth0-overview/create-tenants/set-up-multiple-environments) dans le <Tooltip tip="Auth0 Dashboard: produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, afin de prendre en charge des configurations complexes. Par exemple, vous pourriez avoir deux domaines distincts (un interne et un accessible au public), ou vouloir que les utilisateurs se connectent différemment selon les applications. Pour y parvenir, créez plus d’un Auth0 tenant afin de disposer d’ensembles distincts d’applications, de connexions et d’utilisateurs pour les applications et groupes d’utilisateurs que vous devez gérer.

1. Accédez à l’[Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez le nom de votre tenant, puis sélectionnez **Create Tenant**.

   <Frame>![Menu déroulant du tenant dans le Dashboard : Create Tenant](/docs/images/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/5efad1137eb8b04031b30137c2fc5ac5/2025-02-26_14-51-39.png)</Frame>
2. Saisissez le **Tenant Domain** souhaité, sélectionnez une **Region**, puis sélectionnez **Create**.
