---
description: Décrit comment créer un tenant supplémentaire à l’aide de l’Auth0 Dashboard.
title: Créer plusieurs tenants
---

Vous pouvez configurer plusieurs tenants pour créer [différents environnements](/docs/fr-CA/get-started/auth0-overview/create-tenants/set-up-multiple-environments) dans le <Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> afin de prendre en charge des configurations complexes. Par exemple, vous pourriez avoir deux domaines distincts (un interne et un public), ou vouloir que les utilisateurs se connectent différemment selon l’application. Pour ce faire, créez plus d’un tenant Auth0 afin d’avoir des ensembles distincts d’applications, de connections et d’utilisateurs pour les applications et les groupes d’utilisateurs que vous devez prendre en charge.

1. Accédez à l’[Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez le nom de votre tenant, puis sélectionnez **Create Tenant**.

   <Frame>![Menu déroulant du tenant dans le Dashboard pour créer un tenant](/docs/images/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/5efad1137eb8b04031b30137c2fc5ac5/2025-02-26_14-51-39.png)</Frame>
2. Saisissez le **Tenant Domain** souhaité, sélectionnez une **Region**, puis sélectionnez **Create**.
