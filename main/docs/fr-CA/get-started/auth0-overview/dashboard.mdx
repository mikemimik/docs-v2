L’[Auth0 Dashboard](https://manage.auth0.com/#) est l’endroit où vous gérez tous les aspects de votre abonnement Auth0 et de votre configuration.

<Frame>![Page Activity de l’Auth0 Dashboard](/docs/images/cdy7uua7fh8z/6k0cRLNEPko6vY6Kw7nIQD/4590d1b9d3e79f8d9c9970d733b08d02/2025-01-28_14-34-41.png)</Frame>

Il comporte plusieurs sections auxquelles vous pouvez accéder à l’aide du menu latéral de gauche.

<div id="configure-implementation">
  ## Configurer l’implémentation
</div>

Le tableau suivant présente un bref aperçu des différentes sections du Dashboard et de ce que vous pouvez faire dans chacune.

| Section             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Applications**    | Gérez vos applications, vos API et vos intégrations d’authentification unique (SSO).<br />**Applications** : Pour chacune de vos apps dans lesquelles vous souhaitez authentifier des utilisateurs avec Auth0, enregistrez une application.<br />**APIs** : Pour chacune de vos API que vous souhaitez sécuriser avec Auth0, enregistrez une API. Créez de nouvelles API et gérez celles qui existent déjà.<br />**SSO Integrations** : Affichez et activez des services externes pour le SSO. Créez de nouvelles intégrations SSO, puis configurez, passez en revue et gérez leurs paramètres.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Authentication**  | Gérez les fournisseurs d’identité par lesquels vous permettez aux utilisateurs de s’authentifier dans vos apps.<br />**Database** : Stockez et gérez de façon sécurisée les identifiants nom d’utilisateur/mot de passe, soit dans une banque de données Auth0, soit dans votre propre base de données. Connectez-vous à des bases de données existantes à l’aide de scripts JavaScript basés sur des modèles qui s’exécutent sur le serveur d’Auth0 lors de chaque authentification. Migrez graduellement vers Auth0 une base de données existante d’anciens identifiants à mesure que les utilisateurs s’authentifient (aucune réinitialisation de mot de passe requise).<br />**Social** : Configurez des fournisseurs d’identité sociaux (comme Facebook, X et GitHub) par lesquels vos utilisateurs peuvent ouvrir une session.<br />**Enterprise** : Configurez des fournisseurs d’identité d’entreprise (comme Active Directory, SAML et Office 365) par lesquels vos utilisateurs peuvent ouvrir une session à l’aide de leurs identifiants d’entreprise.<br />**Passwordless** : Permettez à vos utilisateurs de s’inscrire et d’ouvrir une session au moyen de codes d’accès à usage unique (envoyés par courriel ou SMS) ou de liens en un clic, plutôt qu’avec un mot de passe. |
| **Organizations**   | Gérez les organizations avec lesquelles vous faites affaire et personnalisez l’expérience de leurs utilisateurs lorsqu’ils accèdent à vos applications.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **User Management** | Gérez les identités et les permissions de vos utilisateurs.<br />**Users** : Affichez et créez des profils d’utilisateur, effectuez des réinitialisations de mot de passe, bloquez et supprimez des utilisateurs, et plus encore.<br />**Roles** : Créez et gérez des rôles pour vos apps. | **Marketplace**     | Explorez des intégrations qui aident votre entreprise à en faire plus avec Auth0.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

<div id="manage-account-settings">
  ## Gérer les paramètres du compte
</div>

En haut à gauche, vous pouvez voir le nom et l’icône de votre tenant, ainsi qu’une petite flèche. Cette flèche affiche un menu déroulant que vous pouvez utiliser pour configurer différents aspects de votre compte :

* [**Paramètres**](/docs/fr-CA/get-started/tenant-settings) : Configurez différents aspects de votre tenant.
* [**Inviter un membre**](/docs/fr-CA/get-started/manage-dashboard-access) : Ajoutez un utilisateur supplémentaire comme administrateur ou dans un autre rôle à la configuration de votre tenant.
* [**Utilisation du quota**](https://support.auth0.com/reports/quota) : Consultez les renseignements sur l’utilisation du quota pour votre abonnement et vos tenants.
* [**Créer un tenant**](/docs/fr-CA/get-started/auth0-overview/create-tenants) : Utilisez cette option pour créer un nouveau tenant.
* **Changer de tenant** : Si vous avez [plusieurs tenants](/docs/fr-CA/get-started/auth0-overview/create-tenants/create-multiple-tenants), utilisez cette option pour passer de l’un à l’autre. Si vous créez une application pour un tenant, elle n’apparaîtra pas dans la liste d’un autre tenant.

En haut à droite, vous voyez plusieurs options sur lesquelles vous pouvez cliquer :

* **Recherche** : Recherchez des applications, des éléments du Marketplace, des utilisateurs et d’autres contenus liés à votre tenant.
* **Discutez de vos besoins :** Communiquez avec un expert Auth0 pour vous aider à atteindre vos objectifs avec Auth0.
* **Docs :** Un lien vers le site de documentation que vous consultez en ce moment.
* **Cloche de notifications :** Vous informe des nouvelles communications d’Auth0 ou des paramètres de votre tenant.
* **Profil :** Un lien vers votre profil local associé à votre tenant, ainsi que des options pour [changer votre thème](/docs/fr-CA/get-started/dashboard-profile/light-and-dark-themes) et vous déconnecter.
