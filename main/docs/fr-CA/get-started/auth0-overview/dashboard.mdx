L’[Auth0 Dashboard](https://manage.auth0.com/#) vous permet de gérer tous les aspects de votre abonnement Auth0 et de votre configuration.

<Frame>![page Activity de l’Auth0 Dashboard](/docs/images/cdy7uua7fh8z/6k0cRLNEPko6vY6Kw7nIQD/4590d1b9d3e79f8d9c9970d733b08d02/2025-01-28_14-34-41.png)</Frame>

Il comprend plusieurs sections auxquelles vous pouvez accéder à partir du menu latéral de gauche.

<div id="configure-implementation">
  ## Configurer l’implémentation
</div>

Le tableau suivant présente un bref aperçu des différentes sections du Dashboard et de ce que vous pouvez faire dans chacune.

| Section             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Applications**    | Gérez vos applications, vos API et vos intégrations d’authentification unique (SSO).<br />**Applications** : Pour chacune de vos applications dans lesquelles vous voulez authentifier des utilisateurs avec Auth0, enregistrez une application.<br />**APIs** : Pour chacune de vos API que vous voulez sécuriser avec Auth0, enregistrez une API. Créez de nouvelles API et gérez celles qui existent déjà.<br />**SSO Integrations** : Affichez et activez des services externes pour le SSO. Créez de nouvelles intégrations SSO et configurez, examinez et gérez les paramètres d’intégration.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **Authentication**  | Gérez les fournisseurs d’identité par lesquels vous permettez aux utilisateurs de s’authentifier dans vos applications.<br />**Database** : Stockez et gérez de façon sécurisée les informations d’identification par identifiant et mot de passe, soit dans une base de données Auth0, soit dans votre propre base de données. Connectez-vous à des bases de données existantes à l’aide de scripts JavaScript fondés sur des modèles qui s’exécutent sur le serveur d’Auth0 lors de chaque authentification. Migrez graduellement une base de données existante d’anciennes informations d’identification vers Auth0 à mesure que les utilisateurs s’authentifient (aucune réinitialisation du mot de passe requise).<br />**Social** : Configurez des fournisseurs d’identité sociaux (comme Facebook, X et GitHub) par lesquels vos utilisateurs peuvent se connecter.<br />**Enterprise** : Configurez des fournisseurs d’identité d’entreprise (comme Active Directory, SAML et Office 365) par lesquels vos utilisateurs peuvent se connecter à l’aide de leurs informations d’identification d’entreprise.<br />**Passwordless** : Permettez à vos utilisateurs de s’inscrire et de se connecter à l’aide de codes d’accès à usage unique (envoyés par courriel ou par SMS) ou de liens en un clic, plutôt qu’avec des mots de passe. |
| **Organizations**   | Gérez les organisations avec lesquelles vous faites affaire et personnalisez l’expérience de leurs utilisateurs lorsqu’ils accèdent à vos applications.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **User Management** | Gérez les identités et les permissions de vos utilisateurs.<br />**Users** : Affichez et créez des profils d’utilisateur, effectuez des réinitialisations de mot de passe, bloquez et supprimez des utilisateurs, et plus encore.<br />**Roles** : Créez et gérez des rôles pour vos applications. Les rôles regroupent des permissions et peuvent être attribués aux utilisateurs.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **Branding**        | **Universal Login** : Créez et personnalisez une page de connexion vers laquelle vous pouvez diriger les utilisateurs pour qu’ils s’authentifient.<br />**Custom Domains** : Créez un domaine personnalisé afin d’offrir une expérience cohérente à vos utilisateurs.<br />**Email Templates** : Utilisez des modèles pour créer des flux de travail par courriel de bienvenue, de réinitialisation du mot de passe et de vérification de compte.<br />**Email Provider** : Désignez et configurez les renseignements de votre fournisseur de courriel personnalisé.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **Security**        | Configurez des couches de sécurité supplémentaires en activant des protections pour défendre vos utilisateurs contre différents types d’attaques et d’anomalies d’accès.<br />**Attack Protection** : Gérez les paramètres liés aux attaques de robots, à la limitation par IP, aux attaques par force brute et aux mots de passe compromis.<br />**Multi-factor Auth** : Exigez des facteurs supplémentaires pendant le processus de connexion afin d’empêcher les accès non autorisés.<br />**Monitoring** : Surveillez les événements liés au renseignement sur les menaces à l’aide de l’une de nos intégrations de visualisation de données et d’alerte.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **Actions**         | Configurez des flux comme la connexion, le machine-to-machine, l’inscription d’utilisateurs et les réinitialisations de mot de passe. Créez et gérez des actions personnalisées utilisées dans les flux.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Auth Pipeline**   | **Rules** : Configurez des extraits JavaScript personnalisés qui sont exécutés dans Auth0 dans le cadre de chaque transaction d’authentification utilisateur. Vous pouvez appeler des API externes, filtrer les utilisateurs qui peuvent se connecter à votre application, utiliser une liste d’autorisation, configurer un accès géolocalisé, et ainsi de suite.<br />**Hooks** : Personnalisez le comportement d’Auth0 lorsque vous utilisez des Database Connections en configurant du code Node.js qui s’exécute à des points d’extensibilité (comparables à des webhooks fournis avec un serveur).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Monitoring**      | **Logs** : 	Affichez les données de journal des actions effectuées dans le Dashboard par les administrateurs et des connexions des utilisateurs.<br />**Streams** : Créez et gérez la diffusion en continu des événements de journal vers des services externes d’analyse de données.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Marketplace**     | Explorez des intégrations qui aident votre entreprise à faire davantage avec Auth0.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **Extensions**      | Étendez la plateforme Auth0 avec des modules complémentaires officiels et tiers.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **Settings**        | Configurez vos tenants, gérez votre abonnement Auth0 et vos options de paiement, et contrôlez vos administrateurs de tenant ainsi que les autres rôles utilisateur. <div id="manage-account-settings">
  ## Gérer les paramètres du compte
</div>

En haut à gauche, vous pouvez voir le nom et l’icône de votre tenant, ainsi qu’une petite flèche. Cette flèche ouvre un menu déroulant que vous pouvez utiliser pour configurer différents aspects de votre compte :

* [**Settings**](/docs/fr-CA/get-started/tenant-settings) : Configurez plusieurs aspects de votre tenant.
* [**Inviter un membre**](/docs/fr-CA/get-started/manage-dashboard-access) : Ajoutez un utilisateur supplémentaire comme administrateur ou avec un autre rôle à la configuration de votre tenant.
* [**Utilisation du quota**](https://support.auth0.com/reports/quota) : Consultez des renseignements sur l’utilisation du quota de votre abonnement et de vos tenants.
* [**Créer un tenant**](/docs/fr-CA/get-started/auth0-overview/create-tenants) : Utilisez cette option pour créer un nouveau tenant.
* **Changer de tenant** : Si vous avez [plusieurs tenants](/docs/fr-CA/get-started/auth0-overview/create-tenants/create-multiple-tenants), utilisez cette option pour passer de l’un à l’autre. Si vous créez une application pour un tenant, elle n’apparaîtra pas dans la liste d’un autre tenant.

En haut à droite, vous voyez plusieurs options sur lesquelles vous pouvez cliquer :

* **Recherche** : Recherchez des applications, des entrées du Marketplace, des utilisateurs et d’autres contenus liés à votre tenant.
* **Parlez-nous de vos besoins :** Communiquez avec un expert Auth0 pour vous aider à atteindre vos objectifs avec Auth0.
* **Docs :** Un lien vers le site de documentation que vous lisez en ce moment même.
* **Cloche de notifications :** Vous informe des nouvelles communications d’Auth0 ou des paramètres de votre tenant.
* **Profil :** Un lien vers votre profil local pour ce tenant, ainsi que des options pour [changer votre thème](/docs/fr-CA/get-started/dashboard-profile/light-and-dark-themes) et vous déconnecter.
