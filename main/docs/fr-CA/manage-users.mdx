title: Gérer les utilisateurs
---

La base de données infonuagique hébergée par Auth0 stocke les données utilisateur auxquelles vous pouvez accéder au moyen des profils d’utilisateur. Les renseignements sur les utilisateurs peuvent provenir de diverses sources, notamment de fournisseurs d’identité, de vos propres bases de données et de connexions d’entreprise (comme Active Directory ou SAML).
