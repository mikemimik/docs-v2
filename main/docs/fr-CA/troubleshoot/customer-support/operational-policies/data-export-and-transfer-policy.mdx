<div id="export-data">
  ## Exporter les données
</div>

Si vous souhaitez exporter vos données d’Auth0, plusieurs options s’offrent à vous. Veuillez noter que ces outils n’exportent pas les hachages de mot de passe des utilisateurs de votre base de données hébergée par Auth0. Vous pouvez tout de même demander ces renseignements en ouvrant une [demande de soutien](https://support.auth0.com/). Si vous prévoyez cesser d’utiliser notre service, consultez [Exporter les données](/docs/fr-CA/troubleshoot/customer-support/manage-subscriptions/export-data) pour en savoir plus.

<div id="use-the-importexport-extension">
  ### Utiliser l’extension Import/Export
</div>

Vous pouvez utiliser notre [extension Import/Export](/docs/fr-CA/manage-users/user-migration/user-import-export-extension) pour exporter la quasi-totalité des données de votre tenant Auth0.

<div id="use-the-management-api">
  ### Utiliser l’API Management
</div>

Si vous souhaitez exporter certains jeux de données par programmation, utilisez le point de terminaison Bulk User Export de l’<Tooltip tip="Management API : Un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>.

<div id="transfer-data">
  ## Transfert de données
</div>

Auth0 ne transfère pas de données d’un tenant Auth0 à un autre. Cela s’applique autant aux clients de Public Cloud qu’à ceux de Private Cloud.

Toutes les données de votre tenant Auth0 demeurent en tout temps sous votre contrôle et sont accessibles à tout moment par l’intermédiaire de la Management API. Les seules informations qui ne sont pas accessibles par l’API, pour des raisons de sécurité, sont les hachages de mots de passe de vos utilisateurs de base de données hébergée par Auth0 ainsi que les clés privées.
