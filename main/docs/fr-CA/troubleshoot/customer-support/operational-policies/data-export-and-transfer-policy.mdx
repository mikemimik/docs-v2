<div id="export-data">
  ## Exporter des données
</div>

Si vous souhaitez exporter vos données d’Auth0, plusieurs options s’offrent à vous. Veuillez noter que ces outils n’exportent pas les hachages de mot de passe de vos utilisateurs de base de données hébergés par Auth0. Vous pouvez tout de même demander ces renseignements en ouvrant un [ticket de soutien](https://support.auth0.com/). Si vous souhaitez cesser d’utiliser notre service, consultez [Exporter des données](/docs/fr-CA/troubleshoot/customer-support/manage-subscriptions/export-data) pour plus de détails.

<div id="use-the-importexport-extension">
  ### Utiliser l’extension Import/Export
</div>

Vous pouvez utiliser notre [extension Import/Export](/docs/fr-CA/manage-users/user-migration/user-import-export-extension) pour exporter presque toutes les données de votre tenant Auth0.

<div id="use-the-management-api">
  ### Utiliser la Management API
</div>

Si vous souhaitez exporter certains jeux de données de façon programmatique, utilisez le point de terminaison Bulk User Export de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>.

<div id="transfer-data">
  ## Transfert de données
</div>

Auth0 ne transférera pas de données d’un Auth0 tenant à un autre. Cela s’applique autant aux clients de Public Cloud qu’à ceux de Private Cloud.

Toutes les données de votre Auth0 tenant demeurent en tout temps sous votre contrôle et sont accessibles par l’entremise de la Management API. Les seules informations qui ne sont pas accessibles par l’API, pour des raisons de sécurité, sont les hachages de mots de passe de vos utilisateurs de base de données hébergée par Auth0 ainsi que les clés privées.
