<div id="authentication-api">
  ## Authentication API
</div>

Consultez le tableau suivant pour connaître les limites de débit globales de l’Authentication API, ou la politique de limite de débit qui protège presque tous les point de terminaison de l’Authentication API.

Limites de débit pour l’Authentication API et les point de terminaison d’API du type d’abonnement Enterprise :

| Tenant                                   | [Limite de requêtes en rafale](/docs/fr-CA/troubleshoot/customer-support/operational-policies/rate-limit-policy#rate-limit-algorithm) | [Limite de requêtes soutenue](/docs/fr-CA/troubleshoot/customer-support/operational-policies/rate-limit-policy#rate-limit-algorithm) |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Production                               | 100/seconde                                                                                                                     | 100/seconde                                                                                                                    |
| Production (2x Public Performance Burst) | 200/seconde pendant 48 h/mois                                                                                                   | 100/seconde                                                                                                                    |
| Production (3x Public Performance Burst) | 300/seconde pendant 48 h/mois                                                                                                   | 100/seconde                                                                                                                    |
| Production (4x Public Performance Burst) | 400/seconde pendant 48 h/mois                                                                                                   | 100/seconde                                                                                                                    |
| Non-production                           | 100/seconde                                                                                                                     | 100/seconde                                                                                                                    |

Ces limites s’appliquent pendant un maximum de 48 heures par mois. Après 48 heures, elles reviennent aux limites du produit. Pour en savoir plus, consultez [Public Performance Burst](/docs/fr-CA/troubleshoot/customer-support/operational-policies/rate-limit-policy#public-performance).

  ## API SCIM
</div>

Limites de débit des point de terminaison de l’API SCIM entrante dans les abonnements Public Cloud qui incluent des connexions d’entreprise :

| Type de limite                                           | Chemin du point de terminaison                   | Opération     | Limite                   |
| -------------------------------------------------------- | -------------------------------------- | ------------- | ------------------------ |
| point de terminaison d’une seule connexion SCIM                      | `/scim/v2/connections/{connection-id}` | Toute requête | 25 requêtes par seconde  |
| Limite globale du tenant pour toutes les connexions SCIM | `/scim/v2/connections/*`               | Toute requête | 100 requêtes par seconde |
