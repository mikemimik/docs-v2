<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total pour l’Authentication API et l’API MyAccount dans un environnement est généralement pris en charge jusqu’à 500 RPS ; toutefois, certaines charges de travail particulières, une forte utilisation combinée de l’Authentication API et de la Management API, ou certains flux d’authentification comme ROPG vers l’Authentication API peuvent entraîner une dégradation des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les politiques relatives aux limites de débit pour le type d’abonnement Private Cloud Tier 20 (Development).
