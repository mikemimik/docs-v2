title: Performance de Private Cloud 1500 RPS (15x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total sur l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation combinée intensive des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme le ROPG sur l’Authentication API, peuvent entraîner une baisse des performances à des valeurs de RPS plus faibles.
</Callout>

Consultez ci-dessous les limites de débit du type d’abonnement Private Cloud Performance 1500 RPS (15x). Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud.

Nous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
