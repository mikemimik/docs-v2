title: Performance Private Cloud 1500 RPS (15x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total vers l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation combinée intensive des Authentication API et Management API, ou certains Auth Flows comme ROPG vers l’Authentication API peuvent entraîner une baisse des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit pour le type de subscription Private Cloud Performance 1500 RPS (15x). Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud.

Nous vous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
