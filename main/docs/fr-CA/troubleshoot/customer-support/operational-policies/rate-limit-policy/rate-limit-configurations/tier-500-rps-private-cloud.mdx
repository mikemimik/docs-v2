description: Limites de débit du Tier 500 RPS de Private Cloud
title: Performance Private Cloud 500 RPS (5x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Dans un environnement, le débit total soutenu combiné de l’Authentication API et de la MyAccount API est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation combinée intensive des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme ROPG avec l’Authentication API peuvent entraîner une baisse du rendement à des niveaux de RPS inférieurs.
</Callout>

Consultez ci-dessous les limites de débit pour le type d’abonnement Private Cloud Performance 500 RPS (5x).

Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud. Par conséquent, nous recommandons de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
