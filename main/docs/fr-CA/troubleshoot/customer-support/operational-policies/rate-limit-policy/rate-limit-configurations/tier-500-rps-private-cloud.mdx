title: Performance Private Cloud 500 RPS (5x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total combiné de l’Authentication API et de la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une forte utilisation combinée des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme ROPG vers l’Authentication API peuvent entraîner une dégradation des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit du type d’abonnement Performance Private Cloud 500 RPS (5x).

Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud. Par conséquent, nous recommandons de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
