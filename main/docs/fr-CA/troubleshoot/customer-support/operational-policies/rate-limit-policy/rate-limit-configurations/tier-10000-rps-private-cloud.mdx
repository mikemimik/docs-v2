title: Performance de Private Cloud 10 000 RPS (100x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total vers l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une forte utilisation combinée des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme le ROPG avec l’Authentication API peuvent entraîner une baisse des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit associées au type d’abonnement Private Cloud Performance 10 000 RPS (100x).

Nous recommandons donc de déployer un tenant par environnement de Private Cloud afin d’atténuer les risques.
