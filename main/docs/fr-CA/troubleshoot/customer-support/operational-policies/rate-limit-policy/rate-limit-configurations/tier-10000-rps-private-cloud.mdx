title: Performance de Private Cloud 10 000 RPS (100x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit total soutenu combiné de l’Authentication API et de la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation intensive combinée de l’Authentication API et de la Management API, ou certains flux d’authentification comme ROPG vers l’Authentication API peuvent entraîner une dégradation des performances à des niveaux de RPS inférieurs.
</Callout>

Consultez ci-dessous les limites de débit du type d’abonnement Private Cloud Performance 10 000 RPS (100x).

Nous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
