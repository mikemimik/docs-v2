title: Performance de Private Cloud 6000 RPS (60x) et 6000 RPS (60x) en rafale
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total pour l’Authentication API et la MyAccount API réunies dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation combinée intensive des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme ROPG avec l’Authentication API peuvent entraîner une dégradation des performances à des valeurs de RPS plus faibles.
</Callout>

Consultez ci-dessous les limites de débit des types d’abonnement Private Cloud Performance 6000 RPS (60x) et 6000 RPS (60x) Burst.

Nous recommandons donc de déployer un tenant par environnement de cloud privé pour atténuer les risques.
