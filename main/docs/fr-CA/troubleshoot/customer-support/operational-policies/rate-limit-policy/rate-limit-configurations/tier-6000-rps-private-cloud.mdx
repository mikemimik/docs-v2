title: Private Cloud Performance 6000 RPS (60x) et 6000 RPS (60x) Burst
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total vers l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS ; toutefois, certaines charges de travail particulières, une forte utilisation combinée des Authentication API et Management API, ou certains flux d’authentification comme ROPG vers l’Authentication API, peuvent entraîner une baisse de performance à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit des types d’abonnement Private Cloud Performance 6000 RPS (60x) et 6000 RPS (60x) Burst.

Nous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
