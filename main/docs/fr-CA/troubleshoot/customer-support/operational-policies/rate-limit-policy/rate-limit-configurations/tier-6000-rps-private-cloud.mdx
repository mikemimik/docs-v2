title: Performance de Private Cloud 6000 RPS (60x) et 6000 RPS (60x) Burst
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total combiné de l’Authentication API et de la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS ; toutefois, certaines charges de travail particulières, une utilisation combinée élevée des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme le ROPG vers l’Authentication API peuvent entraîner une baisse des performances à des RPS inférieurs.
</Callout>

Consultez ci-dessous les limites de débit pour les types d’abonnement Private Cloud Performance 6000 RPS (60x) et 6000 RPS (60x) Burst.

Nous recommandons donc de déployer un tenant par environnement de Private Cloud afin d’atténuer les risques.
