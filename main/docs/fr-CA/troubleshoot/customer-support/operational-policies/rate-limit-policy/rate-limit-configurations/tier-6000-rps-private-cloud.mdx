title: Performance de Private Cloud à 6000 RPS (60x) et rafale à 6000 RPS (60x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total vers l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation combinée intensive des Authentication API et des Management API, ou certains Auth Flows comme le ROPG vers l’Authentication API, peuvent entraîner une dégradation des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les rate limits des types de subscription Private Cloud Performance 6000 RPS (60x) et 6000 RPS (60x) Burst.

Nous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
