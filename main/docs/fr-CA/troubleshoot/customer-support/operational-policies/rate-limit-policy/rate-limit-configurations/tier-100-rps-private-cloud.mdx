description: Limites de débit du niveau 100 RPS de Private Cloud
title: Private Cloud Basic 100 RPS (1x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total vers l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une forte utilisation combinée des Authentication API et Management API, ou certains flux d’authentification comme ROPG vers l’Authentication API, peuvent entraîner une dégradation des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit du type d’abonnement Private Cloud Basic 100 RPS (1x). Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud.

Nous vous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
