title: Private Cloud Basic 100 RPS (1x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total vers l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation intensive combinée des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme le ROPG vers l’Authentication API peuvent entraîner une baisse de performance à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit pour le type d’abonnement Private Cloud Basic 100 RPS (1x). Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement infonuagique privé.

Nous recommandons donc de déployer un tenant par environnement infonuagique privé afin d’atténuer les risques.
