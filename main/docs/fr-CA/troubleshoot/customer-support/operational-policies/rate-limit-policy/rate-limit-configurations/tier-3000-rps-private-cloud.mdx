description: Limites de débit du Tier 3000 RPS Private Cloud
title: Performance Private Cloud 3000 RPS (30x) et 3000 RPS (30x) Burst
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total pour l’Authentication API et l’API MyAccount dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une forte utilisation combinée des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme ROPG vers l’Authentication API, peuvent entraîner une baisse des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit des types d’abonnement Private Cloud Performance 3000 RPS (30x) et 3000 RPS (30x) Burst. Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud.

Nous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
