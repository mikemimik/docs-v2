description: Limites de débit du Private Cloud de niveau 3000 RPS
title: Performance de Private Cloud 3000 RPS (30x) et Burst 3000 RPS (30x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total de l’Authentication API et de la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation combinée intensive des API d’authentification et de gestion d’Auth0, ou certains flux d’authentification comme ROPG vers l’Authentication API peuvent entraîner une baisse des performances à des valeurs de RPS inférieures.
</Callout>

Consultez ci-dessous les limites de débit associées aux types d’abonnement Private Cloud Performance 3000 RPS (30x) et Burst 3000 RPS (30x). Ces limites s’appliquent à chaque tenant que vous créez dans l’environnement Private Cloud.

Nous recommandons donc de déployer un tenant par environnement Private Cloud afin d’atténuer les risques.
