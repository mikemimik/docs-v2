Vous pouvez supprimer votre tenant Auth0 ou le réinitialiser à la configuration par défaut dans le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Vous pouvez supprimer les données de votre compte Auth0 avec le soutien d’Auth0 Support.

<div id="delete-tenant">
  ## Supprimer un tenant
</div>

<Warning>
  La suppression d’un tenant est une action irréversible et destructive qui ne peut pas être annulée. Les tenants supprimés ne peuvent pas être restaurés, le nom du tenant ne peut pas être réutilisé lors de la création d’un nouveau tenant et les métriques associées au tenant sont perdues.
</Warning>

Avant de supprimer un tenant, tenez compte des options suivantes :

* [Modifier le propriétaire du tenant](/docs/fr-CA/get-started/manage-dashboard-access).
* [Réinitialiser la configuration du tenant](#reset-tenant-configuration).

<div id="remove-custom-domain">
  ### Retirer le domaine personnalisé
</div>

Si vous avez configuré un <Tooltip tip="Domaine personnalisé : domaine tiers utilisant un nom spécialisé ou personnalisé." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=custom+domain">domaine personnalisé</Tooltip> pour un tenant que vous souhaitez supprimer, vous devez d’abord retirer le domaine personnalisé. Autrement, vous ne pourrez pas configurer ce domaine personnalisé pour un autre tenant.

<div id="determine-deletion-process">
  ### Déterminer le processus de suppression
</div>

Le processus de suppression d’un tenant dépend du [plan d’abonnement](https://www.auth0.com/pricing) auquel il est associé et du [type de tenant](/docs/fr-CA/get-started/auth0-overview/create-tenants/child-tenants) dont il s’agit (principal ou enfant). Vous pourrez peut-être supprimer le tenant dans l’[Auth0 Dashboard](#delete-tenant-in-auth0-dashboard), ou vous devrez peut-être [communiquer avec Auth0 Support ou la Auth0 Community](#contact-auth0-support-or-community).

|                                               | Plan Free                           | Plan en libre-service          | Plan Enterprise                |
| --------------------------------------------- | ----------------------------------- | ------------------------------ | ------------------------------ |
| **Tenant principal**                          | Auth0 Dashboard                     | Auth0 Dashboard                | Communiquer avec Auth0 Support |
| **Tenant principal avec des tenants enfants** | Communiquer avec la Auth0 Community | Communiquer avec Auth0 Support | Communiquer avec Auth0 Support |
| **Tenant enfant**                             | Auth0 Dashboard                     | Auth0 Dashboard                | Auth0 Dashboard                |

<div id="downgrade-tenant">
  ### Rétrograder un tenant
</div>

Avant de pouvoir supprimer un tenant dans le Dashboard, vous devez d’abord le rétrograder au forfait Free. Pour en savoir plus, consultez [Rétrograder ou annuler des abonnements Auth0](/docs/fr-CA/troubleshoot/customer-support/manage-subscriptions/downgrade-or-cancel-subscriptions).

<div id="delete-tenant-in-dashboard">
  ### Supprimer un tenant dans le Dashboard
</div>

Vous pouvez supprimer un tenant dans le Dashboard si vous avez le [niveau d’accès requis](/docs/fr-CA/get-started/manage-dashboard-access/feature-access-by-role).

1. Accédez à [Dashboard &gt; Settings](https://manage.auth0.com/#/tenant/advanced), puis passez à la vue **Advanced**.
2. Repérez **Danger Zone** au bas de la page, puis sélectionnez **Delete**.
3. Passez en revue le message de confirmation et confirmez que vous souhaitez supprimer définitivement votre tenant.
4. Sélectionnez **Yes, Delete Tenant**.

<div id="contact-auth0-support-or-community">
  ### Communiquez avec Auth0 Support ou la communauté Auth0
</div>

Si votre tenant est associé à un plan Trial, Essential, Professional ou Enterprise et que vous ne pouvez pas supprimer votre tenant dans le Auth0 Dashboard, [ouvrez un billet auprès d’Auth0 Support](/docs/fr-CA/troubleshoot/customer-support/open-and-manage-support-tickets) pour obtenir de l’aide.

Si votre tenant est associé à un plan Free et que vous ne pouvez pas le supprimer dans le Auth0 Dashboard, [publiez dans Auth0 Community](https://community.auth0.com/) et demandez de l’aide.

<div id="reset-tenant-configuration">
  ## Réinitialiser la configuration du tenant
</div>

Vous pouvez réinitialiser votre tenant pour revenir à sa configuration par défaut à l’aide de l’[Auth0 Deploy CLI](https://github.com/auth0/auth0-deploy-cli).

Le Deploy CLI est un outil qui vous aide à gérer la configuration de votre tenant Auth0. Il s’intègre à vos processus de développement en tant que CLI autonome ou comme module Node.js. Pour en savoir plus sur l’outil Deploy CLI et sur la façon de commencer, consultez [Deploy CLI Tool](/docs/fr-CA/deploy-monitor/deploy-cli-tool).

<div id="delete-auth0-account-data">
  ## Supprimer les données de votre compte Auth0
</div>

Si vous souhaitez supprimer les données de votre compte Auth0 (y compris votre adresse courriel, vos renseignements de facturation, etc.), suivez les étapes pour supprimer votre tenant, puis [publiez un message sur Auth0 Community](https://community.auth0.com/) pour demander de l’aide.

Pour en savoir plus sur les responsabilités d’Auth0 à titre de responsable du traitement, consultez la [Politique de confidentialité d’Okta](https://www.okta.com/privacy-policy/).
