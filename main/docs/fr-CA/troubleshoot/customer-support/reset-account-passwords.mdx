---
description: Décrit comment réinitialiser les mots de passe des comptes Auth0.
title: Réinitialiser les mots de passe des comptes
---

Si vous devez changer votre mot de passe ou si vous avez oublié celui de votre compte Auth0, dans la plupart des cas, vous pouvez définir un nouveau mot de passe dans le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Les réinitialisations de mot de passe entraînent l’expiration des sessions Auth0.

1. À l’écran de connexion du compte Auth0, saisissez l’adresse courriel associée à votre compte et cliquez sur **Continue,** puis choisissez **Mot de passe oublié ?** à l’écran suivant.
2. Confirmez votre adresse courriel et cliquez sur **Continue** pour soumettre votre demande. Vous recevrez un courriel contenant la marche à suivre pour réinitialiser votre mot de passe.
3. Si votre demande a bien été reçue, vous verrez « Veuillez vérifier l’adresse courriel [your email] pour obtenir les instructions de réinitialisation de votre mot de passe. »

Il existe certaines situations particulières de réinitialisation de mot de passe :

* Si vous avez activé <Tooltip tip="Multi-factor authentication (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=MFA">MFA</Tooltip> et que vous avez besoin d’une réinitialisation MFA pour votre compte administrateur, communiquez avec [Auth0 Support](https://support.auth0.com).
* Si vous utilisez un compte social ou d’entreprise pour vous connecter à Auth0, réinitialisez votre mot de passe auprès du <Tooltip tip="fournisseur d’identité (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=identity+provider">fournisseur d’identité</Tooltip> approprié.
* Si vous êtes un administrateur qui tente de réinitialiser le mot de passe d’un utilisateur, il existe deux méthodes :

  * Utiliser le flux interactif de réinitialisation de mot de passe avec Universal Login ou l’Authentication API, où l’utilisateur reçoit un courriel contenant un lien qui ouvre la page de réinitialisation de mot de passe Auth0 pour saisir un nouveau mot de passe.
  * Définir directement le nouveau mot de passe à l’aide de la Management API ou du Dashboard.
