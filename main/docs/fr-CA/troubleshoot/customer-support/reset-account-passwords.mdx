---
description: Décrit comment réinitialiser les mots de passe des comptes Auth0.
title: Réinitialiser les mots de passe des comptes
---

Si vous devez changer votre mot de passe ou si vous avez oublié celui de votre compte Auth0, vous pouvez, dans la plupart des cas, définir un nouveau mot de passe à partir du <Tooltip tip="Auth0 Dashboard : produit principal d'Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Les réinitialisations de mot de passe entraînent l’expiration des sessions Auth0.

1. Sur l’écran de connexion au compte Auth0, saisissez l’adresse courriel associée à votre compte, puis cliquez sur **Continue** et choisissez **Mot de passe oublié?** à l’écran suivant.
2. Confirmez votre adresse courriel et cliquez sur **Continue** pour envoyer la demande. Vous recevrez un courriel contenant des instructions supplémentaires pour réinitialiser votre mot de passe.
3. Si votre demande a bien été reçue, vous verrez « Veuillez vérifier l’adresse courriel [your email] pour obtenir des instructions afin de réinitialiser votre mot de passe. »

Il existe certaines situations particulières de réinitialisation de mot de passe :

* Si vous avez activé la <Tooltip tip="Multi-factor authentication (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du username et du mot de passe, comme un code par SMS." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=MFA">MFA</Tooltip> et que vous avez besoin d’une réinitialisation MFA pour votre compte administrateur, communiquez avec [Auth0 Support](https://support.auth0.com).
* Si vous utilisez un compte social ou d’entreprise pour vous connecter à Auth0, réinitialisez votre mot de passe auprès du <Tooltip tip="Identity Provider (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=identity+provider">fournisseur d’identité</Tooltip> approprié.
* Si vous êtes un administrateur qui tente de réinitialiser le mot de passe d’un utilisateur, deux méthodes s’offrent à vous :

  * Utilisez le flux interactif de réinitialisation du mot de passe avec Universal Login ou l’Authentication API, dans lequel l’utilisateur reçoit un courriel contenant un lien qui ouvre la page de réinitialisation du mot de passe Auth0 pour saisir un nouveau mot de passe.
  * Définissez directement le nouveau mot de passe à l’aide de la Management API ou du Dashboard.
