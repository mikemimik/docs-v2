Vous pouvez utiliser le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> pour mettre à jour vos informations de facturation. Cette fonctionnalité est offerte uniquement aux administrateurs de tenant pour les abonnements basés sur un tenant. Pour les abonnements basés sur le compte, vous devez être propriétaire d’une équipe pour y avoir accès.

<div id="payment-and-billing">
  ## Paiement et facturation
</div>

Pour mettre à jour vos information de facturation :

1. Accédez à [Auth0 Dashboard &gt; Settings](https://manage.auth0.com/#/dashboard/settings).

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Si vous utilisez une Team, vous trouverez ce paramètre sous **Teams &gt; Settings &gt; Subscriptions**.
   </Callout>

2. Sous **Tenant Settings**, sélectionnez **Payment &amp; Billing**.

3. Cliquez sur **Add Payment Details** pour saisir vos renseignements de paiement.

4. Cliquez sur **Update** pour enregistrer vos modifications.
