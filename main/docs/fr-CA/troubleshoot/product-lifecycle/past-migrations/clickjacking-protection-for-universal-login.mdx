description: Décrit comment Auth0 ajoute un moyen d'empêcher l'intégration des pages Universal Login dans un iframe afin de les protéger contre les attaques de détournement de clics.
title: Protection contre le détournement de clics pour Universal Login
---

Le détournement de clics est une attaque qui amène un utilisateur à cliquer sur un élément d’une page Web qui est invisible ou déguisé en un autre élément. Cette attaque consiste à charger du contenu dans un iframe et à superposer des éléments par-dessus. Dans le contexte des pages <Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le Authorization Server d'Auth0, pour vérifier l'identité d'un utilisateur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Universal+Login">Universal Login</Tooltip>, un attaquant pourrait inciter l’utilisateur à cliquer sur un bouton **Connexion** ou **Réinitialiser le mot de passe**.

On peut éviter cela en définissant les en-têtes HTTP suivants :

`X-Frame-Options: deny
Content-Security-Policy: frame-ancestors 'none'`

Même si cette attaque potentielle ne présente pas de risque majeur, il s’agit d’une bonne pratique de sécurité d’ajouter ces en-têtes. Les analyseurs de sécurité les détectent également, de sorte que les rapports de tests d’intrusion peuvent mentionner leur absence.

<div id="actions">
  ## Actions
</div>

Si vous affichez la page de connexion dans un iframe, l’ajout de ces en-têtes pourrait entraîner une rupture de compatibilité. Plutôt que d’ajouter ces en-têtes pour tous les clients, Auth0 vous permet donc de choisir de les activer, ce que nous vous recommandons fortement.

L’action suivante n’est pas requise si vous utilisez la nouvelle expérience Universal Login, car ces en-têtes sont toujours définis dans ce cas.

Pour activer ce changement :

1. Accédez à [Paramètres du locataire &gt; Paramètres avancés](https://manage.auth0.com/#/tenant/advanced).
2. Faites défiler jusqu’à **Migrations**, puis désactivez le paramètre **Désactiver la protection contre le détournement de clics pour Classic Universal Login**.
