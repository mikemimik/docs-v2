title: Exécuter les vérifications de préparation à la production
---

Passez en revue les résultats des vérifications de production de votre tenant avant de déployer votre environnement de production.

1. Accédez au Dashboard de gestion Auth0 et connectez-vous à votre compte.
2. Dans la liste déroulante, sélectionnez le tenant pour lequel vous voulez exécuter les Production Checks
3. Dans cette même liste déroulante, cliquez sur **Run Readiness Check** pour ouvrir l’interface

Facultatif :

1. Sélectionnez une ou plusieurs [Applications](/docs/fr-CA/get-started/applications) associées à ce tenant pour lesquelles vous voulez exécuter les vérifications en cliquant sur **Filters.**
2. Cliquez sur **Apply**.

Lorsque le test est terminé, l’écran s’actualisera automatiquement pour afficher les résultats des vérifications.

<div id="production-check-results">
  ## Résultats de la vérification de Production
</div>

Il existe trois catégories de correctifs liés aux résultats de vérification :

* [Correctifs critiques](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-check-required-fixes)
* [Correctifs non critiques](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-check-recommended-fixes)
* [Bonnes pratiques](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-checks-best-practices)

Voici les résultats possibles pour votre vérification :

* Réussi
* Failed
* Bonnes pratiques

Sous chaque catégorie de résultats, Auth0 vous indique le nombre de vérifications réussies par votre application, ainsi que le nombre de vérifications à examiner.

Si votre application a **Failed** à une ou plusieurs vérifications, vous verrez :

* Le nom de la vérification
* Des renseignements sur ce que la vérification recherche
* Un lien **Configurer** vers la section appropriée, où vous pouvez apporter les correctifs requis afin que votre Application réussisse la vérification
* Un lien hypertexte vers la documentation correspondante

Toutes les vérifications que votre application a **réussies** sont regroupées dans l’objet des résultats réussis. Vous pouvez consulter le nom de la vérification et les renseignements s’y rapportant, examiner la documentation associée et utiliser le lien hypertexte pour accéder à la section de configuration correspondante, où vous pouvez apporter des modifications (au besoin).
