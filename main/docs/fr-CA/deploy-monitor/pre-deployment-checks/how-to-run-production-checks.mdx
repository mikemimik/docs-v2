title: Exécuter les vérifications de préparation à la production
---

Consultez les résultats des vérifications de production de votre tenant avant de déployer votre environnement de production.

1. Accédez au Dashboard de gestion Auth0 et connectez-vous à votre compte.
2. Dans la liste déroulante, sélectionnez le tenant pour lequel vous souhaitez exécuter les Production Checks.
3. Dans cette même liste déroulante, cliquez sur **Run Readiness Check** pour ouvrir l’interface.

Facultatif :

1. Sélectionnez une ou plusieurs [Applications](/docs/fr-CA/get-started/applications) associées à ce tenant pour lesquelles vous souhaitez exécuter les vérifications en cliquant sur **Filters.**
2. Cliquez sur **Apply**.

Une fois le test terminé, votre écran s’actualisera automatiquement pour afficher les résultats des vérifications.

<div id="production-check-results">
  ## Résultats des vérifications de production
</div>

Il existe trois catégories de correctifs liés aux résultats de vérification :

* [Correctifs critiques](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-check-required-fixes)
* [Correctifs non critiques](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-check-recommended-fixes)
* [Meilleures pratiques](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-checks-best-practices)

Voici les résultats possibles de votre vérification :

* Réussi
* Échec
* Meilleures pratiques

Pour chaque ensemble de résultats, Auth0 indique le nombre de vérifications réussies par votre application, ainsi que le nombre de vérifications qui doivent être examinées.

Si votre application est en **échec** pour une ou plusieurs vérifications, vous verrez :

* Le nom de la vérification
* Des renseignements sur ce que la vérification cherche à détecter
* Un lien **Configurer** vers la section appropriée, où vous pouvez apporter les correctifs requis afin que votre application réussisse la vérification
* Un lien hypertexte vers la documentation correspondante

Toutes les vérifications que votre application a **réussies** sont regroupées dans l’objet « passed » de l’ensemble de résultats. Vous pouvez consulter le nom de la vérification et les renseignements s’y rapportant, examiner la documentation associée et utiliser le lien hypertexte pour accéder à la section de configuration correspondante, où vous pouvez apporter des modifications (au besoin).
