title: Exécuter la vérification de préparation à la production
---

Consultez les résultats des vérifications de la production pour votre tenant avant de déployer votre environnement de production.

1. Accédez au tableau de bord de gestion Auth0 et connectez-vous à votre compte.
2. Dans la liste déroulante, sélectionnez le tenant pour lequel vous voulez exécuter les Production Checks
3. Dans la même liste déroulante, cliquez sur **Run Readiness Check** pour ouvrir l’interface

Facultatif :

1. Cliquez sur **Filters.** pour sélectionner une ou plusieurs [Applications](/docs/fr-CA/get-started/applications) associées à ce tenant pour lesquelles vous souhaitez exécuter des vérifications.
2. Cliquez sur **Apply**.

Une fois le test terminé, votre écran s’actualisera automatiquement pour afficher les résultats de vos vérifications.

<div id="production-check-results">
  ## Résultats de la vérification de la production
</div>

Il existe trois catégories de correctifs pour les résultats de la vérification :

* [Correctifs critiques](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-check-required-fixes)
* [Correctifs non critiques](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-check-recommended-fixes)
* [Pratiques exemplaires](/docs/fr-CA/deploy-monitor/pre-deployment-checks/production-checks-best-practices)

Voici les résultats possibles de votre vérification :

* Réussi
* Failed
* Pratiques exemplaires

Pour chaque ensemble de résultats, Auth0 vous indique le nombre de vérifications réussies par votre application, ainsi que le nombre de vérifications qui doivent être examinées.

Si votre application a obtenu le résultat **Failed** pour une ou plusieurs vérifications, vous verrez :

* Le nom de la vérification
* Des renseignements sur ce que la vérification recherche
* Un lien **Configurer** vers la section appropriée, où vous pouvez apporter les correctifs requis pour que votre Application réussisse la vérification
* Un hyperlien vers la documentation correspondante

Toutes les vérifications que votre application a **réussies** sont regroupées dans l’objet passed de l’ensemble de résultats. Vous pouvez consulter le nom de la vérification et les renseignements à son sujet, passer en revue la documentation associée et utiliser l’hyperlien pour accéder à la zone de configuration correspondante, où vous pouvez apporter des modifications (au besoin).
