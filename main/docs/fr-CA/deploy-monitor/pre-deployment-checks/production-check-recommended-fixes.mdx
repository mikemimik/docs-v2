---
description: Recommandations sur la façon d’améliorer votre application Auth0 avant un déploiement en production
title: 'Vérifications de préparation à la production : correctifs non critiques'
---

Les vérifications suivantes permettent de déterminer si vous avez appliqué toutes les recommandations (facultatives) pour mener à bien un déploiement en production.

Pour en savoir plus sur les résultats de vos vérifications, consultez [Exécuter les vérifications de production](/docs/fr-CA/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Vérification                                                                                                                                    | Description                                                                                                                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configurer la diffusion des journaux                                                                                                            | Assurez-vous de diffuser les entrées du journal du tenant vers un emplacement de stockage externe ou un fournisseur externe. Nous vous recommandons de [configurer un service de diffusion des journaux](/docs/fr-CA/customize/log-streams) pour conserver les journaux pendant plus de 30 jours et faciliter l’analyse et les alertes.                                                                             |
| Le [domaine personnalisé](/docs/fr-CA/customize/custom-domains) doit être configuré                                                                   | Utilisez les [domaines personnalisés avec Universal Login](https://manage.auth0.com/#/tenant/custom_domains) pour offrir l’expérience la plus fluide et la plus sécuritaire à vos utilisateurs finaux.                                                                                                                                                                                                        |
| La [page d’erreur personnalisée](/docs/fr-CA/customize/login-pages/custom-error-pages) doit être configurée                                           | [Configurez une page d’erreur personnalisée](https://manage.auth0.com/#/account) avec les renseignements propres à votre application et l’image de marque de votre entreprise.                                                                                                                                                                                                                                |
| Activer Universal Login                                                                                                                         | Assurez-vous d’utiliser la plus récente version de [Universal Login](/docs/fr-CA/authenticate/login/auth0-universal-login/universal-login-vs-classic-login). Nous vous recommandons de faire passer votre page de connexion de l’expérience Classic à Universal Login afin de profiter d’une meilleure accessibilité, de meilleures performances, d’une personnalisation sans code et de nouvelles fonctionnalités. |
| Les [modèles de courriel](/docs/fr-CA/customize/email/manage-email-flow) doivent être configurés                                                      | [Configurez des modèles de courriel personnalisés](https://manage.auth0.com/#/emails) avec les renseignements propres à votre application et l’image de marque de votre entreprise.                                                                                                                                                                                                                           |
| [L’authentification multifacteur](/docs/fr-CA/secure/multi-factor-authentication) doit être configurée                                                | Nous vous recommandons d’envisager l’authentification multifacteur comme pratique sécuritaire pour authentifier vos utilisateurs.                                                                                                                                                                                                                                                                             |
| [L’authentification multifacteur pour les administrateurs de tenant](/docs/fr-CA/get-started/manage-dashboard-access) est activée                     | [Activez l’authentification multifacteur](https://manage.auth0.com/#/account/admins) pour les administrateurs de tenant.                                                                                                                                                                                                                                                                                      |
| L’[URL de redirection après la déconnexion](/docs/fr-CA/authenticate/login/logout#set-the-allowed-logout-urls-at-the-account-level) doit être définie | Passez en revue les [URL de redirection autorisées après la déconnexion](https://manage.auth0.com/#/account/advanced) pour votre application.                                                                                                                                                                                                                                                                 |
| Les applications devraient utiliser RS256 au lieu de HS256 pour l’algorithme de signature JSON Web Token (JWT)                                  | Définissez l’[algorithme de signature](/docs/fr-CA/get-started/apis#signing-algorithms) du JSON Web Token (JWT) à RS256 plutôt qu’à HS256 pour chaque application.                                                                                                                                                                                                                                                  |
| Définir l’[URI de connexion de l’application](/docs/fr-CA/get-started/applications/application-settings#application-uris)                             | Indiquez une [URI de connexion de l’application](/docs/fr-CA/get-started/applications/application-settings#application-uris) pour toutes les applications interactives.                                                                                                                                                                                                                                             |
| Définir l’[URL de déconnexion autorisée du tenant](/docs/fr-CA/get-started/applications/application-settings#application-uris)                        | Créez au moins une route de déconnexion accessible publiquement et [mettez à jour les paramètres du tenant](/docs/fr-CA/get-started/applications/application-settings#application-uris) pour qu’ils pointent vers celle-ci.                                                                                                                                                                                         |
| Définir l’[URI de connexion du tenant](/docs/fr-CA/get-started/tenant-settings#login-and-logout)                                                      | Publiez une route de connexion par défaut et [mettez à jour les paramètres de votre tenant](/docs/fr-CA/get-started/tenant-settings#login-and-logout) pour qu’ils pointent vers celle-ci.                                                                                                                                                                                                                           |
