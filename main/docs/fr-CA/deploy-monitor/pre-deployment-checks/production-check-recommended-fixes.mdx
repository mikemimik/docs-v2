description: Recommandations sur la façon d’améliorer votre application Auth0 avant le déploiement en production
title: 'Vérifications de préparation à la production : correctifs non critiques'
---

Les vérifications suivantes permettent de déterminer si vous avez suivi toutes les recommandations (facultatives) pour réussir votre déploiement en Production.

Pour en savoir plus sur le résultat de vos vérifications, consultez [Run Production Checks](/docs/fr-CA/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Vérification                                                                                                                             | Description                                                                                                                                                                                                                                                                                                                                                                                                      |
| ---------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configurer la diffusion des journaux                                                                                                     | Assurez-vous de diffuser les entrées du journal du tenant vers un emplacement de stockage ou un fournisseur externe. Nous vous recommandons de [configurer un service de diffusion des journaux](/docs/fr-CA/customize/log-streams) afin de conserver les journaux au-delà de 30 jours et de faciliter l’analyse et les alertes.                                                                                       |
| [Custom Domain](/docs/fr-CA/customize/custom-domains) devrait être configuré                                                                   | Utilisez des [custom domains avec Universal Login](https://manage.auth0.com/#/tenant/custom_domains) pour offrir l’expérience la plus fluide et la plus sécurisée possible à vos utilisateurs finaux.                                                                                                                                                                                                            |
| [Page d’erreur personnalisée](/docs/fr-CA/customize/login-pages/custom-error-pages) devrait être configurée                                    | [Configurez une page d’erreur personnalisée](https://manage.auth0.com/#/account) avec les détails propres à votre application et l’image de marque de votre entreprise.                                                                                                                                                                                                                                          |
| Activer Universal Login                                                                                                                  | Assurez-vous d’utiliser la version la plus récente de [Universal Login](/docs/fr-CA/authenticate/login/auth0-universal-login/universal-login-vs-classic-login). Nous vous recommandons de faire passer votre page de connexion de l’expérience Classic à Universal Login afin de profiter d’une meilleure accessibilité, de meilleures performances, d’une personnalisation sans code et de nouvelles fonctionnalités. |
| [Email Templates](/docs/fr-CA/customize/email/manage-email-flow) devraient être configurés                                                     | [Configurez des modèles de courriel personnalisés](https://manage.auth0.com/#/emails) avec les détails propres à votre application et l’image de marque de votre entreprise.                                                                                                                                                                                                                                     |
| [Multi-factor Auth](/docs/fr-CA/secure/multi-factor-authentication) devrait être configurée                                                    | Nous vous recommandons d’envisager l’authentification multifacteur comme pratique sécuritaire pour authentifier vos utilisateurs.                                                                                                                                                                                                                                                                                |
| [AMF pour les administrateurs de tenant](/docs/fr-CA/get-started/manage-dashboard-access) est activée                                          | [Activez l’authentification multifacteur](https://manage.auth0.com/#/account/admins) pour les administrateurs de tenant.                                                                                                                                                                                                                                                                                         |
| [URL de redirection après logout](/docs/fr-CA/authenticate/login/logout#set-the-allowed-logout-urls-at-the-account-level) devrait être définie | Passez en revue les [URL de redirection après logout autorisées](https://manage.auth0.com/#/account/advanced) pour votre application.                                                                                                                                                                                                                                                                            |
| Les applications devraient utiliser RS256 au lieu de HS256 pour l’algorithme de signature des JSON Web Token (JWT)                       | Définissez l’[algorithme de signature](/docs/fr-CA/get-started/apis#signing-algorithms) du JSON Web Token (JWT) sur RS256 plutôt que HS256 pour chaque application.                                                                                                                                                                                                                                                    |
| Définir [Application Login URI](/docs/fr-CA/get-started/applications/application-settings#application-uris)                                    | Indiquez un [Application Login URI](/docs/fr-CA/get-started/applications/application-settings#application-uris) pour toutes les applications interactives.                                                                                                                                                                                                                                                             |
| Définir [Tenant Allowed Logout URL](/docs/fr-CA/get-started/applications/application-settings#application-uris)                                | Créez au moins une route de logout accessible au public et [mettez à jour les paramètres du tenant](/docs/fr-CA/get-started/applications/application-settings#application-uris) pour qu’ils pointent vers celle-ci.                                                                                                                                                                                                    |
| Définir [Tenant Login URI](/docs/fr-CA/get-started/tenant-settings#login-and-logout)                                                           | Publiez une route de connexion par défaut et [mettez à jour les paramètres de votre tenant](/docs/fr-CA/get-started/tenant-settings#login-and-logout) pour qu’ils pointent vers celle-ci.                                                                                                                                                                                                                              |
