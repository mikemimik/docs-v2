description: Vérifications pour s’assurer que vos Applications respectent les meilleures pratiques d’Auth0
title: 'Vérifications de préparation à la production : meilleures pratiques'
---

Les vérifications suivantes ne peuvent pas être automatisées; nous vous recommandons donc de vérifier manuellement ces éléments avant le déploiement en Production.

| Vérification                                                                                                        | Description                                                                                                                                                                                                                                                                                                 |
| ------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Externaliser les [paramètres de configuration](/docs/fr-CA/authenticate/database-connections/custom-db)                   | [Externalisez tous les paramètres de configuration au lieu de les coder en dur](https://manage.auth0.com/#/connections/database), comme les identifiants, les chaînes de connexion, les clés API, etc., lorsque vous développez des Rules, des Hooks ou des connexions de base de données personnalisées.   |
| Vérifier les délais d’expiration de l’[authentification unique (SSO)](/docs/fr-CA/glossary?term=Single+Sign-on+%28SSO%29) | Vérifiez les [délais d’expiration par défaut du cookie SSO](https://manage.auth0.com/#/account/advanced) et assurez-vous qu’ils correspondent à vos besoins.                                                                                                                                                |
| Tenants et administrateurs                                                                                          | Vérifiez tous les tenants et les administrateurs de tenant pour vous assurer qu’ils sont adéquats. Retirez du service les tenants qui ne sont plus utilisés. Assurez-vous que le rôle d’administrateur de tenant est limité aux utilisateurs qui en ont besoin.                                             |
| Vérifier les ID client dans le code de l’application                                                                | Assurez-vous que les ID client dans le code de votre application correspondent à la configuration de l’Application Auth0 correspondante.                                                                                                                                                                    |
| Ajouter les IP publiques d’Auth0 à l’AllowList                                                                      | Ajoutez les IP d’Auth0 à l’AllowList si vous vous connectez à des services internes ou à des services derrière un firewall lorsque vous utilisez des Rules, des Hooks ou Custom Databases. Vous pouvez obtenir une liste des adresses IP dans l’infobulle lors de la configuration de l’un de ces éléments. |
| Vérifier la protection contre les attaques                                                                          | Nous vous recommandons de [vérifier les capacités et la configuration de protection d’Auth0](/docs/fr-CA/secure/attack-protection).                                                                                                                                                                               |
