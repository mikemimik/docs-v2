---
description: Vérifications pour s'assurer que vos Applications respectent les bonnes pratiques d'Auth0
title: 'Vérifications de préparation à la production : Bonnes pratiques'
---

Les vérifications suivantes ne peuvent pas être automatisées; nous vous recommandons donc de les effectuer manuellement avant le déploiement en Production.

| Vérification                                                                                                                          | Description                                                                                                                                                                                                                                                                                                                                               |
| ------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Externaliser les [paramètres de configuration](/docs/fr-CA/authenticate/database-connections/custom-db)                                     | [Externalisez tous les paramètres de configuration au lieu de les coder en dur](https://manage.auth0.com/#/connections/database), comme les identifiants, les chaînes de connexion, les clés API, etc., lorsque vous développez des Rules, des Hooks ou des connexions de base de données personnalisées.                                                 |
| Vérifier les valeurs de délai d&#39;expiration de l&#39;[authentification unique (SSO)](/docs/fr-CA/glossary?term=Single+Sign-on+%28SSO%29) | Vérifiez les valeurs par défaut du délai d&#39;expiration du [cookie SSO](https://manage.auth0.com/#/account/advanced) et assurez-vous qu&#39;elles correspondent à vos besoins.                                                                                                                                                                          |
| Tenants et administrateurs                                                                                                            | Passez en revue tous les tenants et les administrateurs de tenant pour vous assurer qu&#39;ils sont corrects. Mettez hors service les tenants qui ne sont plus utilisés. Assurez-vous que le rôle d&#39;administrateur de tenant est limité aux utilisateurs qui en ont besoin.                                                                           |
| Vérifier les ID client dans le code de l&#39;application                                                                              | Assurez-vous que les ID client dans le code de votre application correspondent à la configuration de leurs Applications Auth0.                                                                                                                                                                                                                            |
| Ajouter les IP publiques d&#39;Auth0 à la liste d&#39;autorisation                                                                    | Ajoutez les IP d&#39;Auth0 à la liste d&#39;autorisation si vous vous connectez à des services internes ou à des services protégés par un pare-feu lorsque vous utilisez des Rules, des Hooks ou des bases de données personnalisées. Vous pouvez obtenir la liste des adresses IP dans l&#39;infobulle au moment de configurer l&#39;un de ces éléments. |
| Vérifier la protection contre les attaques                                                                                            | Nous vous recommandons de [vérifier les capacités et la configuration de protection d&#39;Auth0](/docs/fr-CA/secure/attack-protection).                                                                                                                                                                                                                         |
