---
description: Vérifications pour s'assurer que vos Applications se conforment aux meilleures pratiques d'Auth0
title: 'Vérifications de préparation à la production : Meilleures pratiques'
---

Les vérifications suivantes ne peuvent pas être automatisées; nous vous recommandons donc de les effectuer manuellement avant le déploiement en Production.

| Vérification                                                                                                        | Description                                                                                                                                                                                                                                                                                                                                                          |
| ------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Externaliser les [paramètres de configuration](/docs/fr-CA/authenticate/database-connections/custom-db)                   | [Externalisez, plutôt que de coder en dur, tous les paramètres de configuration](https://manage.auth0.com/#/connections/database), comme les identifiants, les chaînes de connexion, les clés API, etc., lors du développement de Rules, de Hooks ou de connexions de base de données personnalisées.                                                                |
| Passer en revue les délais d&#39;expiration du [Single Sign-on (SSO)](/docs/fr-CA/glossary?term=Single+Sign-on+%28SSO%29) | Passez en revue les délais d&#39;expiration par défaut du [cookie SSO](https://manage.auth0.com/#/account/advanced) et assurez-vous qu&#39;ils correspondent à vos besoins.                                                                                                                                                                                          |
| Tenants et administrateurs                                                                                          | Passez en revue tous les tenants et les administrateurs de tenant pour vous assurer qu&#39;ils sont corrects. Mettez hors service les tenants qui ne sont plus utilisés. Assurez-vous que seuls les utilisateurs qui en ont besoin sont administrateurs de tenant.                                                                                                   |
| Vérifier les ID client dans le code de l&#39;application                                                            | Assurez-vous que les ID client dans le code de votre application correspondent aux configurations de leurs Applications Auth0.                                                                                                                                                                                                                                       |
| Ajouter les IP publiques d&#39;Auth0 à la liste d&#39;autorisation                                                  | Ajoutez les IP d&#39;Auth0 à la liste d&#39;autorisation si vous vous connectez à des services internes ou à des services derrière un pare-feu lorsque vous utilisez des Rules, des Hooks ou des bases de données personnalisées. Vous pouvez obtenir la liste des adresses IP dans l&#39;infobulle au moment de configurer l&#39;un ou l&#39;autre de ces éléments. |
| Passer en revue la protection contre les attaques                                                                   | Il est recommandé de [passer en revue les fonctionnalités et la configuration de la protection d&#39;Auth0](/docs/fr-CA/secure/attack-protection).                                                                                                                                                                                                                         |
