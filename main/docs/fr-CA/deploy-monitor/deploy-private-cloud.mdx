Les offres Private Cloud sont des services gérés que vous pouvez utiliser si vous :

* ne pouvez pas utiliser un service Public Cloud multilocataire dans votre organisation
* avez des exigences de déploiement géographiques pour lesquelles l’option Public Cloud n’est pas viable
* avez besoin d’un nombre garanti de requêtes par seconde (RPS) ou d’une capacité supérieure à celle offerte par Public Cloud

Le Private Cloud d’Auth0 peut être déployé de l’une des façons suivantes :

| Option                                                                                         | Description                                                                                                                                                  |
| ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [**Private Cloud on AWS**](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | Un service infonuagique dédié et géré sur AWS offrant l’isolation, un rendement supérieur, des instances de développement, des add-ons Geo-HA et plus encore |
| [**Private Cloud on Azure**](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Un service infonuagique dédié sur Azure offrant l’isolation, un rendement supérieur, des instances de développement, des add-ons et plus encore              |

Nous offrons aussi des [fonctionnalités add-on](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-add-on-features) qui peuvent compléter votre déploiement de Private Cloud d’Auth0.

Si vous avez des exigences particulières en matière de soutien ou souhaitez obtenir plus d’information sur les services professionnels que nous offrons, veuillez [contacter Auth0 Sales](https://auth0.com/contact-us).
