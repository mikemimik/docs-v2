Les offres de nuage privé sont des services gérés que vous pouvez utiliser si vous :

* Ne pouvez pas utiliser un service Public Cloud multitenant dans votre organisation
* Avez des exigences de déploiement géographique qui rendent l’option Public Cloud non viable
* Avez besoin d’un nombre garanti de requêtes par seconde (RPS) ou d’une capacité supérieure à celle offerte par le Public Cloud

Le nuage privé d’Auth0 peut être déployé selon les options suivantes :

| Option                                                                                         | Description                                                                                                                                                                   |
| ---------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [**Private Cloud on AWS**](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | Un service infonuagique dédié et géré sur AWS, offrant l’isolation, un rendement supérieur, des instances de développement, des modules complémentaires Geo-HA et plus encore |
| [**Private Cloud on Azure**](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Un service infonuagique dédié sur Azure, offrant l’isolation, un rendement supérieur, des instances de développement, des modules complémentaires et plus encore              |

Nous proposons aussi des [fonctionnalités complémentaires](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-add-on-features) qui peuvent bonifier votre déploiement de nuage privé d’Auth0.

Si vous avez des exigences particulières en matière de soutien ou si vous souhaitez obtenir plus d’information sur les services professionnels que nous offrons, veuillez [communiquer avec Auth0 Sales](https://auth0.com/contact-us).
