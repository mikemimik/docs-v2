title: Consulter les logs
---

Utilisez le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> pour consulter tous les logs générés dans vos tenants, y compris l’authentification des utilisateurs et les actions administratives comme l’ajout et la mise à jour d’applications, de connexions et de Rules. Les actions administratives s’affichent dans les logs comme des logs **API Operation**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Par défaut, les logs peuvent ne pas s’afficher dans l’ordre exact où ils se sont produits. Si vous transmettez les logs du tenant en continu et avez besoin d’un tri chronologique, il est recommandé d’effectuer le tri des logs dans le traitement de votre flux.
</Callout>

1. Accédez à [Dashboard &gt; Monitoring &gt; Logs](https://manage.auth0.com/#/logs).

   <Frame>![Dashboard Monitoring Logs](/docs/images/cdy7uua7fh8z/2l9LfKEjrprVmHfWt80VX0/4e48bc6ad3a8b591014e23fdb7353405/2025-01-28_15-07-43.png)</Frame>
2. La liste affiche les logs les plus récents. Pour effectuer une recherche, placez le curseur dans le champ de recherche, saisissez une requête et appuyez sur **Entrée**. Par exemple, pour rechercher tous les logs avec des connexions contenant « Pass », saisissez `connection:*pass*`. Pour en savoir plus sur la syntaxe de recherche et voir des exemples, consultez [Syntaxe de requête de recherche des logs](/docs/fr-CA/deploy-monitor/logs/log-search-query-syntax).
3. Vous pouvez filtrer votre recherche par type de log. Choisissez un filtre dans la liste déroulante et cliquez sur **Appliquer**. Par exemple, vous pouvez choisir **Connexion échouée** pour filtrer les logs de tentatives de connexion utilisateur échouées. Pour en savoir plus sur les filtres d’événements de log et voir des exemples, consultez [Filtres d’événements de log](/docs/fr-CA/deploy-monitor/logs/log-event-filters).

   <Frame>![Dashboard Monitoring Logs Filters](/docs/images/cdy7uua7fh8z/3bfjw3WURyQRiNhj6f8VlJ/5c71550ff0a1d51f81b0c65a4d2cc22a/dashboard-monitoring-logs-filters.png)</Frame>
4. Vous pouvez choisir une plage de dates pour votre recherche. Cliquez sur la liste déroulante à côté du champ de date, choisissez une date précise ou une plage de dates, puis cliquez sur **Appliquer**.
