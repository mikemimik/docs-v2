title: Afficher les journaux
---

Utilisez le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> pour afficher tous les journaux générés dans vos tenants, y compris l’authentification des utilisateurs et les actions administratives comme l’ajout et la mise à jour d’applications, de connexions et de règles. Les actions administratives s’affichent dans les journaux sous forme de journaux **API Operation**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Par défaut, les journaux peuvent ne pas s’afficher exactement dans l’ordre où ils se sont produits. Si vous utilisez la diffusion des journaux du tenant et avez besoin d’un tri chronologique, nous vous recommandons d’implémenter le tri des journaux dans le traitement de votre stream.
</Callout>

1. Accédez à [Dashboard &gt; Monitoring &gt; Logs](https://manage.auth0.com/#/logs).

   <Frame>![Dashboard Monitoring Logs](/docs/images/cdy7uua7fh8z/2l9LfKEjrprVmHfWt80VX0/4e48bc6ad3a8b591014e23fdb7353405/2025-01-28_15-07-43.png)</Frame>
2. La liste affiche les journaux les plus récents. Pour effectuer une recherche, placez le curseur dans le champ de recherche, saisissez une requête, puis appuyez sur **Entrée**. Par exemple, pour rechercher tous les journaux avec des connexions contenant &quot;Pass&quot;, saisissez `connection:*pass*`. Pour en savoir plus sur la syntaxe des requêtes de recherche et voir des exemples, consultez [Syntaxe des requêtes de recherche de journaux](/docs/fr-CA/deploy-monitor/logs/log-search-query-syntax).
3. Vous pouvez aussi filtrer votre recherche par type de journal. Choisissez un filtre dans la liste déroulante, puis cliquez sur **Appliquer**. Par exemple, vous pouvez choisir **Failed Login** pour filtrer les journaux d’échec de connexion utilisateur. Pour en savoir plus sur les filtres d’événements de journal et voir des exemples, consultez [Filtres d’événements de journal](/docs/fr-CA/deploy-monitor/logs/log-event-filters).

   <Frame>![Dashboard Monitoring Logs Filters](/docs/images/cdy7uua7fh8z/3bfjw3WURyQRiNhj6f8VlJ/5c71550ff0a1d51f81b0c65a4d2cc22a/dashboard-monitoring-logs-filters.png)</Frame>
4. Vous pouvez choisir une plage de dates pour votre recherche. Cliquez sur la liste déroulante à côté du champ de date, choisissez une date précise ou une plage de dates, puis cliquez sur **Appliquer**.
