---
description: Décrit pendant combien de temps les données de journal sont conservées selon votre forfait Auth0.
title: Conservation des données de journal
---

La durée de conservation de vos journaux Auth0 dépend de votre niveau d’abonnement.

| Forfait          | Conservation des journaux |
| ---------------- | ------------------------- |
| Starter          | 1 jour                    |
| B2C Essentials   | 5 jours                   |
| B2C Professional | 10 jours                  |
| B2B Essentials   | 5 jours                   |
| B2B Professional | 10 jours                  |
| Enterprise       | 30 jours                  |

[Auth0 Support](https://support.auth0.com) peut vous aider à intégrer des services externes de journalisation si vous devez conserver vos journaux au-delà de cette durée. Cette approche est aussi utile pour agréger les journaux de toute votre organisation.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 ne fournit pas de journaux en temps réel pour votre tenant. Même si nous faisons de notre mieux pour indexer les événements à mesure qu’ils arrivent, il peut y avoir un certain délai.
</Callout>

De plus, vous pouvez utiliser Log Streaming d’Auth0 pour exporter des données de journal. Pour en savoir plus, accédez à [Log Streaming](https://marketplace.auth0.com/features/log-streaming) dans Auth0 Marketplace.
