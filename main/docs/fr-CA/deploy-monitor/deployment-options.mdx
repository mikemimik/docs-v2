title: Options de déploiement
---

Auth0 offre des options de déploiement en nuage public et en nuage privé, ainsi que des outils pour vous aider dans votre déploiement d’entreprise.

La plateforme d’identité Auth0 peut être déployée des façons suivantes :

| Option                                                                                         | Description                                                                                                                                                                            |
| ---------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Public Cloud**                                                                               | Un service infonuagique mutualisé exécuté dans le nuage d’Auth0, avec des fonctionnalités standard d’authentification, de personnalisation, de gestion des utilisateurs et de sécurité |
| [**Private Cloud on AWS**](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | Un service infonuagique dédié et géré exécuté sur AWS, offrant l’isolation, un meilleur rendement, des instances de développement, des add-ons Geo-HA et plus encore                   |
| [**Private Cloud on Azure**](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Un service infonuagique dédié exécuté sur Azure, offrant l’isolation, un meilleur rendement, des instances de développement, des add-ons Geo-HA et plus encore                         |

Les packages **Private Cloud** sont des services gérés que vous pouvez utiliser si vous :

* ne pouvez pas utiliser un service de nuage public mutualisé dans votre organisation
* avez besoin d’un niveau garanti de requêtes par seconde (RPS)

Si vous avez des exigences particulières en matière de soutien ou souhaitez obtenir plus d’information sur les services professionnels que nous offrons, veuillez [communiquer avec Auth0 Sales](https://auth0.com/contact-us).
