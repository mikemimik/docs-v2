title: Options de déploiement
---

Auth0 offre des options de déploiement Public Cloud et Private Cloud, ainsi que des outils pour vous aider dans votre déploiement en entreprise.

La plateforme d’identité Auth0 peut être déployée de la façon suivante :

| Option                                                                                         | Description                                                                                                                                                                         |
| ---------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Public Cloud**                                                                               | Un service infonuagique multi-tenant hébergé dans le nuage d’Auth0, avec des fonctions standard d’authentification, de personnalisation, de gestion des utilisateurs et de sécurité |
| [**Private Cloud on AWS**](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | Un service infonuagique dédié et géré, exécuté sur AWS, offrant l’isolation, de meilleures performances, des instances de développement, des add-ons Geo-HA et plus encore          |
| [**Private Cloud on Azure**](/docs/fr-CA/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Un service infonuagique dédié, exécuté sur Azure, offrant l’isolation, de meilleures performances, des instances de développement, des add-ons Geo-HA et plus encore                |

Les offres **Private Cloud** sont des services gérés que vous pouvez utiliser si vous :

* ne pouvez pas utiliser un service infonuagique public multi-tenant dans votre organisation
* avez besoin d’un niveau garanti de requêtes par seconde (RPS)

Si vous avez des exigences particulières en matière de soutien ou souhaitez obtenir plus d’information sur les services professionnels que nous offrons, veuillez [communiquer avec Auth0 Sales](https://auth0.com/contact-us).
