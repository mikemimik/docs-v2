description: Découvrez les options de déploiement d’Auth0 dans le cloud public et le cloud privé, ainsi que les outils et les listes de vérification de déploiement.
title: Déployer et surveiller
---

Vous pouvez déployer Auth0 dans le cloud public, dans un cloud privé sur AWS ou dans un cloud privé sur Azure. Les outils de déploiement et les listes de vérification facilitent le processus. Les outils de surveillance et les journaux d’événements vous permettent de suivre l’état du système et les événements.
