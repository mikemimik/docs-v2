Le [point de terminaison `GET /api/v2/users-by-email`](https://auth0.com/docs/api/management/v2#!/Users_By_Email/get_users_by_email) vous permet de rechercher des utilisateurs à l’aide de leur adresse courriel. La recherche exige une correspondance exacte avec l’adresse courriel fournie et tient compte de la casse.

Ce point de terminaison offre une **cohérence immédiate** et, à ce titre, nous vous recommandons de l’utiliser pour :

* Les recherches d’utilisateurs effectuées pendant le processus d’authentification.
* Les recherches d’utilisateurs effectuées dans le cadre du processus de liaison de comptes.

La requête doit inclure un [Management API access token](/docs/fr-CA/secure/tokens/access-tokens/management-api-access-tokens).

  ```bash cURL
  curl --request GET \
    --url 'https://{yourDomain}/api/v2/users-by-email?email=%7BuserEmailAddress%7D' \
    --header 'authorization: Bearer {yourMgmtApiAccessToken}'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/users-by-email?email=%7BuserEmailAddress%7D");

  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
  	"fmt"
  	"net/http"
  	"io/ioutil"
  )

  func main() {

  	url := "https://{yourDomain}/api/v2/users-by-email?email=%7BuserEmailAddress%7D"

  	req, _ := http.NewRequest("GET", url, nil)

  	req.Header.Add("authorization", "Bearer {yourMgmtApiAccessToken}")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```
