Pour paginer les résultats de la recherche d’utilisateurs, utilisez les paramètres `page`, `per_page` et `include_totals` dans la requête.

| Paramètre        | Description                                                                                                                                                                       |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `page`           | Le numéro de la page, en commençant à zéro. Si cette valeur n’est pas définie, nous renvoyons au maximum 50 enregistrements, quel que soit le nombre d’enregistrements existants. |
| `per_page`       | Le nombre d’utilisateurs par page.                                                                                                                                                |
| `include_totals` | Définissez `true` pour inclure un résumé de la requête dans le résultat.                                                                                                          |

<div id="limitation">
  ## Limite
</div>

Auth0 limite à 1000 le nombre total d’utilisateurs que vous pouvez récupérer (par exemple, 100 utilisateurs par page sur 10 pages). Lorsque le paramètre `page` n’est pas défini, nous retournons un maximum de 50 enregistrements, peu importe le nombre d’enregistrements existants.
