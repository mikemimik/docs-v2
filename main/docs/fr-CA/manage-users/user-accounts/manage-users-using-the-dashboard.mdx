description: Décrit comment gérer les utilisateurs dans l’Auth0 Dashboard.
title: Gérer les utilisateurs à l’aide du Dashboard
---

Vous pouvez utiliser le Dashboard pour gérer vos utilisateurs. Pour commencer, accédez à [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users). C’est là que vous pouvez créer, afficher, modifier ou supprimer des utilisateurs.

<Card title="La disponibilité varie selon le plan Auth0">
  La disponibilité de cette fonctionnalité dépend à la fois de votre implémentation de connexion et de votre plan Auth0 ou de votre entente personnalisée. Pour en savoir plus, consultez [Pricing](https://auth0.com/pricing).
</Card>

Par défaut, les attributs du profil utilisateur fournis par des <Tooltip tip="Fournisseur d’identité (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=identity+providers">fournisseurs d’identité</Tooltip> autres qu’Auth0 (comme Google, Facebook ou X) ne peuvent pas être modifiés directement, car ils sont mis à jour à partir du fournisseur d’identité chaque fois que l’utilisateur se connecte. Pour en savoir plus sur les attributs provenant des fournisseurs d’identité, consultez [Configurer la connexion du fournisseur d’identité pour les mises à jour du profil utilisateur](/docs/fr-CA/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0).

Pour pouvoir modifier les attributs root `name`, `nickname`, `given_name`, `family_name` ou `picture` du profil utilisateur normalisé, vous devez [configurer la synchronisation de votre connexion avec Auth0](/docs/fr-CA/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0) afin que les attributs utilisateur soient mis à jour à partir du fournisseur d’identité uniquement lors de la création du profil utilisateur. Ces attributs root pourront ensuite être modifiés individuellement ou au moyen d’importations en bloc

Pour savoir comment gérer l’accès à l’<Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> pour les membres de votre équipe, consultez [Manage Dashboard Access](/docs/fr-CA/get-started/manage-dashboard-access).
