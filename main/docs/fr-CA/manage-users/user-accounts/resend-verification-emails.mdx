---
description: "Décrit comment renvoyer des courriels de vérification aux utilisateurs à l’aide du Dashboard Auth0 et de l’Auth0 Management API."
title: Renvoyer les courriels de vérification
---

Si vous devez renvoyer un courriel de vérification à un utilisateur, vous pouvez le faire à l’aide du Dashboard Auth0 ou de l’Auth0 <Tooltip tip="Management API : produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>.

<Tabs>
  <Tab title="Utiliser Auth0 Dashboard">
    Vous pouvez utiliser le Dashboard Auth0 pour renvoyer un courriel de vérification à un utilisateur.

    1. Dans le Dashboard, accédez à [**User Management &gt; Users**](https://manage.auth0.com/#/users).
    2. Sélectionnez l’utilisateur.
    3. Sélectionnez le menu déroulant **Actions**, puis **Send Verification Email**.

    Si vous activez [Multiple Custom Domains](/docs/fr-CA/customize/custom-domains/multiple-custom-domains), vous pouvez utiliser l’option **Domain** pour sélectionner le domaine de notification de votre tenant.
  </Tab>

  <Tab title="Utiliser Auth0 Management API">
    Vous pouvez utiliser l’Auth0 Management API pour renvoyer un courriel de vérification à un utilisateur.

    1. Obtenez un [Management API access token](/docs/fr-CA/secure/tokens/access-tokens/management-api-access-tokens) avec la portée `update:users`.
    2. Effectuez une requête au point de terminaison [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) de l’Auth0 Management API avec le `user_id` de l’utilisateur.

    #### Exemple de requête

    ```json lines
    {
       "user_id":"auth0|62463410c0242d00699461c8"
    }
    ```

    #### Exemple de réponse

    ```json lines
    {
       "type":"verification_email",
       "status":"pending",
       "created_at":"2022-03-31T23:08:05.612Z",
       "id":"job_PyfpRHlmpkSoDmCv"
    }
    ```

    Si vous activez [Multiple Custom Domains](/docs/fr-CA/customize/custom-domains/multiple-custom-domains), vous devez inclure l’en-tête HTTP `auth0-custom-domain`. Pour en savoir plus, consultez [Multiple Custom Domains](/docs/fr-CA/customize/custom-domains/multiple-custom-domains#customize-email-handling-using-the-management-api).

    #### Vérifier le statut de la tâche

    Vous pouvez vérifier le statut de la tâche créée avec le point de terminaison [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) de l’Auth0 Management API.

    1. Obtenez l’`id` de la tâche (retourné dans la réponse).
    2. Effectuez une requête au point de terminaison [Get a job](https://auth0.com/docs/api/management/v2#!/Jobs/get_jobs_by_id) de l’Auth0 Management API avec l’`id` de la tâche.

    ##### Exemple de réponse

    ```json lines
    {
       "type":"verification_email",
       "status":"completed",
       "created_at":"2022-03-31T23:08:05.612Z",
       "id":"job_PyfpRHlmpkSoDmCv"
    }
    ```
  </Tab>
</Tabs>
