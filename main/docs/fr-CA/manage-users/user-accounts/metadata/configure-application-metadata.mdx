description: Découvrez comment configurer, mettre à jour et supprimer les métadonnées de l’application (client_metadata et clientMetadata) dans les Advanced Settings d’Applications de l’Auth0 Dashboard.
title: Configurer les métadonnées de l’application
---

Les métadonnées de l’application sont facultatives et se composent de clés et de valeurs personnalisables (255 caractères max. chacune) que vous pouvez définir pour chaque application. Les métadonnées sont exposées dans l’objet `Client` sous la forme `client_metadata`, et dans Rules sous la forme `context.clientMetadata`. Vous pouvez y stocker, par exemple, l’URL de la page d’accueil de l’application (un champ qu’Auth0 ne fournit pas par défaut dans les paramètres de l’application).

Les métadonnées du client sont stockées dans les propriétés de l’application (client). Pour en savoir plus sur les types de données, les noms de champs et les limites de stockage, consultez [noms de champs et types de données des métadonnées](/docs/fr-CA/manage-users/user-accounts/metadata/metadata-fields-data).

<Card title="Où stocker les secrets client">
  L’endroit où stocker le secret dépend de sa portée :

  * S’agit-il d’un seul secret par application ? Dans ce cas, `client_metadata` serait un bon endroit.
  * S’agit-il du même secret pour tout le système (c.-à-d. pour toutes les applications ou plusieurs d’entre elles) ? Dans ce cas, les valeurs de configuration de la règle pourraient être un meilleur choix.
  * S’agit-il d’un secret différent pour chaque utilisateur ? Dans ce cas, il pourrait être préférable de le stocker dans `app_metadata`.
</Card>

<Warning>
  Les claims dans l’ID token ne sont pas chiffrés. Selon le flow que vous utilisez, l’utilisateur pourrait donc obtenir le token et en inspecter le contenu. Auth0 **ne** recommande **pas** d’y stocker un secret.
</Warning>

Les applications existantes n’auront aucune valeur pour cette propriété.

Vous pouvez accéder aux métadonnées de l’application dans [Actions](/docs/fr-CA/customize/actions) :

```js lines
exports.onExecutePostLogin = async (event, api) => {
  if (event.client.metadata.SKIP_VERIFICATION === "yes"){
    return;
  }
  // ... continuer cette Action
}
```

... ou dans [Rules](/docs/fr-CA/customize/rules) :

```javascript lines
function applicationMetadataExample (user, context, callback){
  context.clientMetadata = context.clientMetadata || {};
  if (context.clientMetadata.SKIP_VERIFICATION === "yes"){
    return callback();
  }
  // ... continuer cette règle
}
```

Vous pouvez consulter les métadonnées de l’application et y ajouter des paires clé-valeur à l’aide du Dashboard ou de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. Pour savoir comment gérer les métadonnées du client à l’aide de la Management API, consultez [gérer les métadonnées à l’aide de la Management API](/docs/fr-CA/manage-users/user-accounts/metadata/manage-metadata-api).

<div id="add-application-metadata-keyvalue-pairs">
  ## Ajouter des paires clé-valeur aux métadonnées de l’application
</div>

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) et sélectionnez l’application.
2. Faites défiler la page vers le bas et cliquez sur **Advanced Settings**.
3. Dans l’onglet **Application Metadata**, saisissez le nom de la clé et sa valeur, puis cliquez sur **Add**.

   <Frame>![Dashboard Applications Applications onglet Settings Advanced Settings onglet Application Metadata](/docs/images/cdy7uua7fh8z/7GWHyQloNihda3fuYiVEWQ/b3cced7c53690e6f3cbf7b2e23ff8646/App_Metadata_-_English.png)</Frame>
4. Cliquez sur **Save Changes**.

<div id="update-application-metadata-value">
  ## Mettre à jour la valeur des métadonnées de l&#39;application
</div>

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) et sélectionnez l&#39;application.
2. Faites défiler la page et cliquez sur **Advanced Settings**.
3. Dans l&#39;onglet **Application Metadata**, saisissez le nom de la clé que vous voulez modifier, puis saisissez une nouvelle valeur et cliquez sur **Add**.
4. Cliquez sur **Save Changes**

<div id="delete-application-metadata">
  ## Supprimer les métadonnées de l’application
</div>

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) et sélectionnez l’application.
2. Faites défiler la page vers le bas et cliquez sur **Advanced Settings**.
3. Dans l’onglet **Application Metadata**, repérez la paire clé-valeur que vous souhaitez supprimer et cliquez sur l’icône de corbeille.
4. Confirmez la suppression.
5. Cliquez sur **Save Changes**.
