---
description: Décrit comment créer des utilisateurs dans l’Auth0 Dashboard.
title: Créer des utilisateurs
---

Vous pouvez créer un utilisateur à l’aide de l’Auth0 Dashboard ou de l’[Auth0 Management API](https://auth0.com/docs/api/management/v2/users/post-users).

1. Accédez à [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) et sélectionnez **Create User**.
2. Sélectionnez la **Connection**, puis saisissez le **Courriel** et le **Mot de passe** de l’utilisateur.

   | Attribut   | Description                                                                                                                                                                                                                                                                                                                                 |
   | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Connection | La connexion à la base de données à utiliser pour authentifier l’utilisateur. La liste déroulante affiche toutes les connexions à la base de données configurées dans votre tenant. La connexion que vous utilisez doit être associée à une application, sinon vous recevrez un message d’erreur indiquant que la connexion est désactivée. |
   | Email      | L’adresse courriel de l’utilisateur. Le format de l’adresse courriel doit être conforme à la spécification fournie dans la [RFC 5321 de l’IETF](https://datatracker.ietf.org/doc/html/rfc5321#section-4.1.2).                                                                                                                               |
   | Password   | Le mot de passe de l’utilisateur. Il n’existe aucune longueur maximale pour le mot de passe.                                                                                                                                                                                                                                                |
3. Sélectionnez **Create**.
   L’utilisateur sera créé et vous serez redirigé vers le profil du nouvel utilisateur.

Si vous activez [Multiple Custom Domains](/docs/fr-CA/customize/custom-domains/multiple-custom-domains), vous pouvez sélectionner **Domain** dans la section **Connection** afin de choisir le domaine de notification de votre tenant.

Si vous activez [Flexible Identifiers](/docs/fr-CA/authenticate/database-connections/flexible-identifiers-and-attributes), en plus de l’**adresse courriel**, vous pouvez sélectionner le **nom d’utilisateur**, le **phone** de l’utilisateur, ou une combinaison d’au moins deux de ces éléments.
