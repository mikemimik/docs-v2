Ce guide vous montrera comment définir des attributs racine pour un utilisateur lors de l’importation à l’aide de l’<Tooltip tip="Management API : un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip> d’Auth0. Vous pouvez ainsi réduire le nombre d’appels d’API nécessaires pour définir des attributs racine lors de l’importation d’utilisateurs. Pour voir quels attributs vous pouvez importer, consultez la [Structure du profil utilisateur normalisé](/docs/fr-CA/manage-users/user-accounts/user-profiles/user-profile-structure).

Effectuez un appel `POST` à l’[endpoint Create Job to Import Users](https://auth0.com/docs/api/management/v2#!/Jobs/post_users_imports). Assurez-vous de remplacer les valeurs d’espace réservé `MGMT_API_ACCESS_TOKEN`, `CONNECTION_ID` et `JSON_USER_FILE_PATH` par le <Tooltip tip="Jeton d’accès : justificatif d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisé pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Access+Token">jeton d’accès</Tooltip> de votre Management API, l’ID de la connexion et le nom du fichier d’utilisateurs, respectivement.

  ```bash cURL
  curl --request POST \
    --url 'https://{yourDomain}/api/v2/jobs/usersimports' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
    --header 'content-type: multipart/form-data ' \
    --data '{ "connection_id": "CONNECTION_ID", "users": "JSON_USER_FILE_PATH" }'
  ```

  ```go Go
  	req, _ := http.NewRequest("POST", url, payload)

  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

| Valeur                  | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec la [portée](/docs/fr-CA/glossary?term=scope) `create:users`.                                                                                                                                                                                                                                                                                                                                                         |
| `CONNECTION_ID`         | ID de la connexion dans laquelle les utilisateurs seront ajoutés. Vous pouvez obtenir cette information à l’aide du [point de terminaison Get All Connections](https://auth0.com/docs/api/management/v2#!/Connections/get_connections).                                                                                                                                                                                                                                                                    |
| `JSON_USER_FILE_PATH`   | Nom du fichier contenant les utilisateurs à importer. Le fichier doit être au format JSON et inclure les attributs racine des utilisateurs. Pour obtenir la liste des attributs disponibles, consultez [User Profile Attributes](/docs/fr-CA/manage-users/user-accounts/user-profiles/user-profile-structure#attributes). Pour voir un exemple du format de fichier, consultez [Bulk User Import Database Schema and Examples](/docs/fr-CA/manage-users/user-migration/bulk-user-import-database-schema-and-examples). |
