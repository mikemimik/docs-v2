description: Décrit comment dissocier les appareils attribués à un utilisateur à l’aide de l’Auth0 Dashboard.
title: Dissocier des appareils d’utilisateurs
---

Vous pouvez dissocier les appareils attribués à un utilisateur à l’aide de l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Cette action révoque le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=refresh+token">refresh token</Tooltip> de l’appareil sélectionné.

1. Accédez à [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users), puis cliquez sur le nom de l’utilisateur.
2. Cliquez sur la vue **Devices**, puis sur l’icône de corbeille à côté de l’appareil que vous voulez dissocier, puis confirmez.
