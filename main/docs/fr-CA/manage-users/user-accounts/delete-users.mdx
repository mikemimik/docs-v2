---
description: Décrit comment supprimer des utilisateurs.
title: Supprimer des utilisateurs
---

1. Accédez à [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users).
2. Cliquez sur le nom de l’utilisateur que vous souhaitez supprimer.
3. Dans l’onglet **Details**, faites défiler la page jusqu’en bas, à côté de **Delete user**, puis cliquez sur **Delete**.<br />

   Un popup vous avertira que cette action est irréversible et vous demandera de confirmer que vous souhaitez supprimer l’utilisateur. Cliquez sur le bouton **Yes, Delete It** pour confirmer. L’utilisateur sera supprimé définitivement.
