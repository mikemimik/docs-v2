title: Comptes d’utilisateur
---

La base de données infonuagique hébergée d’Auth0 stocke diverses informations sur vos utilisateurs, auxquelles vous pouvez accéder. Ces informations sont accessibles par l’intermédiaire d’un profil utilisateur, et vos utilisateurs sont regroupés par tenant. Les renseignements sur les utilisateurs eux-mêmes peuvent provenir de diverses sources, notamment des <Tooltip tip="Identity Provider (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=identity+providers">fournisseurs d’identité</Tooltip>, de vos propres bases de données et des connexions d’entreprise (Active Directory, <Tooltip tip="Identity Provider (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=SAML">SAML</Tooltip>).
