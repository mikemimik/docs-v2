Pour autoriser l’accès machine-to-machine pour une application, vous devez activer son utilisation d’une organization pour une API précise. Pour ce faire, vous devez associer le client grant de l’application à l’API correspondante avec l’organization. Une fois l’association établie, l’application peut utiliser l’organization lorsqu’elle demande des tokens pour l’API et les scopes définis dans le client grant. Vous devrez définir cette association pour chaque API à laquelle votre application doit accéder pour l’organization.

Vous pouvez autoriser l’accès M2M pour une application à l’aide de l’[Auth0 Dashboard](https://manage.auth0.com/) ou de la [Management API](https://auth0.com/docs/api/management/v2).

<Tabs>
  <Tab title="Auth0 Dashboard">
    Pour associer le client grant d’une application à une organization au moyen de l’Auth0 Dashboard :

    1. Accédez à **Organizations** et choisissez l’organization à associer.
    2. Sélectionnez l’onglet **Machine-to-Machine Access**.
    3. Cliquez sur **Add Access.**
    4. Cliquez sur l’application que vous voulez associer à l’organization.
    5. Sélectionnez une API.
    6. Cliquez sur **Save.**
