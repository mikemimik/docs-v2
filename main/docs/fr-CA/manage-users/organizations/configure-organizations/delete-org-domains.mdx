---
title: Supprimer des domaines d’organisation
description: Supprimez l’association entre une organisation et un domaine à l’aide de l’Auth0 Dashboard ou de la Management API.
---

Lorsque vous supprimez un domaine d’organisation, l’association entre le domaine et l’organisation est supprimée définitivement. Cette opération est irréversible.

<div id="delete-an-organization-domain-with-the-auth0-dashboard">
  ## Supprimer un domaine d’organisation à partir de l’Auth0 Dashboard
</div>

Pour supprimer un domaine d’organisation dans l’Auth0 Dashboard :

1. Accédez à [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) et choisissez votre organisation.
2. Sélectionnez l’onglet **Domains**.
3. À côté du domaine que vous voulez supprimer, ouvrez le menu des actions (**...**) et sélectionnez **Delete**.
4. Dans la fenêtre qui s’ouvre, confirmez la suppression.

<div id="delete-an-organization-domain-with-the-management-api">
  ## Supprimer un domaine d’organisation avec la Management API
</div>

Pour supprimer un domaine d’organisation avec la Management API, appelez le [point de terminaison Delete an organization discovery domain](/docs/fr-CA/api/management/v2/organizations/delete-discovery-domains-by-discovery-domain-id) (`DELETE /organizations/{id}/discovery-domains/{discovery_domain_id}`).
