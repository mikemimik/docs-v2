Pour ajouter un membre à votre [organisation](/docs/fr-CA/manage-users/organizations/organizations-overview), vous devez avoir déjà [créé l’utilisateur](/docs/fr-CA/manage-users/user-accounts/create-users) dans votre tenant. Si vous ne trouvez pas l’utilisateur recherché, vous pouvez plutôt [l’inviter](/docs/fr-CA/manage-users/organizations/configure-organizations/invite-members).

Pour gérer directement les membres, vous pouvez utiliser soit le <Tooltip tip="Auth0 Dashboard : l’interface principale d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, soit <Tooltip tip="Auth0 Dashboard : l’interface principale d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>.

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Pour attribuer des membres dans l’Auth0 Dashboard :

1. Accédez à [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations), puis sélectionnez l’organisation pour laquelle vous souhaitez configurer l’appartenance.
2. Sélectionnez la vue **Members**, sélectionnez **Add members**, puis sélectionnez **Add Users**.
3. Saisissez le ou les noms du ou des utilisateurs que vous souhaitez attribuer comme membres de l’organisation, puis sélectionnez **Add user(s) to organization**.

<div id="management-api">
  ## Management API
</div>

Pour ajouter des membres au moyen de la Management API :

Effectuez une requête `POST` vers le point de terminaison `Create Organization Members`. Assurez-vous de remplacer les valeurs d’espace réservé `ORG_ID`, `MGMT_API_ACCESS_TOKEN` et `USER_ID` par l’ID de votre organisation, le <Tooltip tip="Jeton d’accès : justificatif d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisé pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Access+Token">jeton d’accès</Tooltip> de la Management API et les ID utilisateur, respectivement.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Trouvez votre domaine Auth0**

  Votre domaine Auth0 est composé du nom de votre tenant, de votre sous-domaine régional (sauf si votre tenant se trouve dans la région US et a été créé avant juin 2020), puis de `.auth0.com`. Par exemple, si le nom de votre tenant était `travel0`, votre domaine Auth0 serait `travel0.us.auth0.com`. (Si votre tenant se trouvait aux États-Unis et avait été créé avant juin 2020, votre nom de domaine serait alors `https://travel0.auth0.com`.)

  Si vous utilisez des domaines personnalisés, il doit s’agir de votre nom de domaine personnalisé.
</Callout>

| Valeur                  | Description                                                                                                                                                |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | ID de l’organisation à laquelle vous souhaitez ajouter des membres.                                                                                        |
| `MGMT_API_ACCESS_TOKEN` | [jeton d’accès pour la Management API](/docs/fr-CA/secure/tokens/access-tokens/management-api-access-tokens) avec le scope `create:organization_members`.        |
| `USER_ID`               | ID du ou des utilisateurs que vous souhaitez ajouter à l’organisation indiquée. Vous pouvez envoyer un maximum de 10 membres par organisation par requête. |

<div id="response-status-codes">
  ### Codes d’état de la réponse
</div>

Les codes d’état de réponse possibles sont les suivants :

| Code d’état | Code d’erreur          | Message                                                                                                | Cause                                                                                               |
| ----------- | ---------------------- | ------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------- |
| `204`       |                        | Membres ajoutés à l’organisation avec succès.                                                          |                                                                                                     |
| `400`       | `invalid_body`         | Corps de la requête invalide. Le message varie selon la cause.                                         | La charge utile de la requête n’est pas valide.                                                     |
| `400`       | `invalid_query_string` | Chaîne de requête invalide. Le message varie selon la cause.                                           | La chaîne de requête n’est pas valide.                                                              |
| `401`       |                        | Jeton invalide.                                                                                        |                                                                                                     |
| `401`       |                        | Signature invalide reçue lors de la validation du JSON Web Token.                                      |                                                                                                     |
| `401`       |                        | Le client n’est pas global.                                                                            |                                                                                                     |
| `403`       | `insufficient_scope`   | Portée insuffisante ; une des valeurs suivantes est attendue : `create:organization_members`.          | Tentative de lire ou de modifier un champ non autorisé avec les portées du jeton du porteur fourni. |
| `429`       |                        | Trop de requêtes. Vérifiez les en-têtes X-RateLimit-Limit, X-RateLimit-Remaining et X-RateLimit-Reset. |                                                                                                     |
