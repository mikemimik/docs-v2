---

À partir de la section Organizations de votre [Auth0 Dashboard](https://manage.auth0.com/#/organizations), vous pouvez rechercher des [organisations](/docs/fr-CA/manage-users/organizations/organizations-overview) par nom ou par nom d’affichage.

<Frame>![Auth0 Dashboard - Organizations - Barre de recherche ](/docs/images/cdy7uua7fh8z/5CeeKLjPGxjTqy682lbnyj/3a41b87a369e65ca624b1822c99dbc57/2025-01-27_17-27-28.png)</Frame>

Le menu déroulant à côté de la barre de recherche vous permet de chercher par nom ou par nom d’affichage. Si vous saisissez trois caractères ou moins dans la barre de recherche, seules les correspondances exactes seront renvoyées. Si vous saisissez quatre caractères ou plus, des correspondances approximatives contenant la chaîne saisie seront renvoyées. Par exemple, si vous avez une organisation nommée &quot;Big Holdings, Co.&quot; et que vous entrez les termes de recherche `hold` ou `ings`, l’organisation Big Holdings, Co. sera renvoyée.

Les recherches renvoient un maximum de 50 résultats. Si vous ne trouvez pas le résultat souhaité, essayez d’utiliser un terme de recherche plus précis.

<Warning>
  La recherche d’organisations repose sur une cohérence éventuelle. Les organisations récemment créées ou modifiées peuvent ne pas s’afficher immédiatement dans les résultats de recherche; toutefois, ces résultats finiront par refléter fidèlement les entités de votre tenant.
</Warning>
