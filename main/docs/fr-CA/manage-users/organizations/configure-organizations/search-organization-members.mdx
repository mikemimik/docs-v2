---
description: Comment utiliser User Search pour trouver les membres d’une organisation
title: Rechercher des membres d’une organisation
---

En plus de [lister les membres de l’organisation](/docs/fr-CA/manage-users/organizations/configure-organizations/retrieve-members), vous pouvez utiliser [User Search](/docs/fr-CA/manage-users/user-search) pour trouver des utilisateurs selon leur appartenance à une organisation à l’aide du paramètre `organization_id` dans une recherche.

Par exemple, voici un filtre pour les utilisateurs dont le domaine de l’adresse courriel est `acme.com`, mais qui ne sont pas membres de l’organisation Acme :

`email.domain:"acme.com" AND NOT organization_id:"ACME_ORG_ID"`

L’exemple suivant trouve tous les utilisateurs qui font partie de l’organisation Acme et qui se connectent avec une connexion de base de données précise.

`organization_id:"ACME_ORG_ID" AND identities.connection:"my-db-connection-name"`

L’exemple ci-dessous trouve les utilisateurs dont l’adresse courriel commence par jane et qui sont membres de l’organisation Acme.

`organization_id:"ACME_ORG_ID" AND email:jane*`

Consultez [User Search](/docs/fr-CA/manage-users/user-search) pour en savoir plus.
