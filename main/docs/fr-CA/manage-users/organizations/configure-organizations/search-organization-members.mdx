---
description: Comment utiliser User Search pour trouver les membres d’une Organization
title: Rechercher des membres d’une Organization
---

En plus de [lister les membres de l’Organization](/docs/fr-CA/manage-users/organizations/configure-organizations/retrieve-members), vous pouvez utiliser [User Search](/docs/fr-CA/manage-users/user-search) pour trouver des utilisateurs selon leur appartenance à une Organization à l’aide du paramètre `organization_id` dans une recherche.

Par exemple, ci-dessous se trouve un filtre pour les utilisateurs qui ont un domaine de courriel `acme.com`, mais qui ne sont pas membres de l’Organization Acme :

`email.domain:"acme.com" AND NOT organization_id:"ACME_ORG_ID"`

Ce qui suit permet de trouver tous les utilisateurs qui font partie de l’Organization Acme et qui se connectent avec une database connection précise.

`organization_id:"ACME_ORG_ID" AND identities.connection:"my-db-connection-name"`

L’exemple ci-dessous permet de trouver les utilisateurs dont l’adresse courriel commence par jane et qui sont membres de l’Organization Acme.

`organization_id:"ACME_ORG_ID" AND email:jane*`

Consultez [User Search](/docs/fr-CA/manage-users/user-search) pour plus de détails.
