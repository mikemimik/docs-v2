Lors de la suppression d’une [organisation](/docs/fr-CA/manage-users/organizations/organizations-overview), l’association entre tous les utilisateurs membres et l’organisation supprimée sera supprimée, mais les utilisateurs eux-mêmes ne seront pas supprimés de votre locataire.

<Warning>
  Une fois confirmée, cette opération ne peut pas être annulée.
</Warning>

Vous pouvez supprimer des organisations depuis le <Tooltip tip="Auth0 Dashboard : le produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou la <Tooltip tip="Auth0 Dashboard : le produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>.

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Pour supprimer une organisation depuis l’Auth0 Dashboard :

1. Accédez à [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations), puis sélectionnez l’organisation que vous souhaitez supprimer.
2. Faites défiler la page jusqu’à la section **Danger Zone**, sélectionnez **Delete**, puis confirmez.

<div id="management-api">
  ## Management API
</div>

Envoyez une requête `DELETE` au point de terminaison `Delete Organization`. Assurez-vous de remplacer les valeurs de l’espace réservé `ORG_ID` et `MGMT_API_ACCESS_TOKEN` par l’ID de votre organisation et votre <Tooltip tip="Jeton d’accès : donnée d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Access+Token">jeton d’accès</Tooltip> pour la Management API, respectivement.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Repérez votre domaine Auth0**

  Votre domaine Auth0 correspond au nom de votre locataire, suivi de votre sous-domaine régional (sauf si votre locataire se trouve dans la région des États-Unis et a été créé avant juin 2020), puis de `.auth0.com`. Par exemple, si le nom de votre locataire était `travel0`, votre domaine Auth0 serait `travel0.us.auth0.com`. (Si votre locataire se trouvait aux États-Unis et avait été créé avant juin 2020, votre nom de domaine serait alors `https://travel0.auth0.com`.)

  Si vous utilisez des domaines personnalisés, il doit s’agir du nom de votre domaine personnalisé.
</Callout>

| Valeur                  | Description                                                                                                                                   |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | ID de l’organisation que vous voulez supprimer.                                                                                               |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](/docs/fr-CA/secure/tokens/access-tokens/management-api-access-tokens) avec la portée `delete:organizations`. |

<div id="response-status-codes">
  #### Codes d’état de réponse
</div>

Les codes d’état de réponse possibles sont les suivants :

| Code d’état | Code d’erreur        | Message                                                                                                | Cause                                                                                                      |
| ----------- | -------------------- | ------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------- |
| `204`       |                      | L’organisation a été supprimée.                                                                        |                                                                                                            |
| `400`       | `invalid_uri`        | URI de requête non valide. Le message varie selon la cause.                                            | Le chemin n’est pas valide.                                                                                |
| `401`       |                      | Jeton non valide.                                                                                      |                                                                                                            |
| `401`       |                      | Signature non valide reçue lors de la validation du JSON Web Token.                                    |                                                                                                            |
| `403`       | `insufficient_scope` | Portée insuffisante ; une des portées suivantes est attendue : `delete:organizations`.                 | Tentative de lecture ou de modification d’un champ non autorisée avec les portées du jeton porteur fourni. |
| `404`       |                      | L’organisation n’existe pas.                                                                           |                                                                                                            |
| `429`       |                      | Trop de requêtes. Vérifiez les en-têtes X-RateLimit-Limit, X-RateLimit-Remaining et X-RateLimit-Reset. |                                                                                                            |
