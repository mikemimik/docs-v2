<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Les ensembles de fonctionnalités [Authorization Core](/docs/fr-CA/manage-users/access-control/rbac) et [Authorization Extension](/docs/fr-CA/customize/extensions/authorization-extension) sont entièrement distincts. Pour gérer les groupes, les rôles ou les permissions, vous devez utiliser la fonctionnalité dans laquelle ils ont été créés au départ.
</Callout>

Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API à l’aide du Dashboard ou de l’<Tooltip tip="Management API : Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. La fonctionnalité Authorization Core est différente de l’Authorization Extension. Pour comparer les deux, consultez [Authorization Core vs. Authorization Extension](/docs/fr-CA/manage-users/access-control/authorization-core-vs-authorization-extension).

Nous proposons diverses fonctions pour vous aider à gérer vos rôles, accessibles soit dans le <Tooltip tip="Auth0 Dashboard : Le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, soit dans l’Auth0 Management API :

* [Créer des rôles](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/create-roles)
* [Modifier les définitions de rôles](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/edit-role-definitions)
* [Ajouter des permissions aux rôles](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/add-permissions-to-roles)
* [Supprimer des permissions des rôles](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/remove-permissions-from-roles)
* [Afficher les permissions d’un rôle](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Afficher les utilisateurs attribués à un rôle](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/view-users-assigned-to-roles)
* [Supprimer des rôles](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/delete-roles)
