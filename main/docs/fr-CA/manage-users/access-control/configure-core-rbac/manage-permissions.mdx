* [Supprimer des permissions d’API](/docs/fr-CA/get-started/apis/delete-api-permissions)
* [Ajouter des permissions aux rôles](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/add-permissions-to-roles)
* [Assigner des permissions aux utilisateurs](/docs/fr-CA/manage-users/access-control/configure-core-rbac/rbac-users/assign-permissions-to-users)
* [Afficher les permissions d’un rôle](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Afficher les permissions d’un utilisateur](/docs/fr-CA/manage-users/access-control/configure-core-rbac/rbac-users/view-user-permissions)
* [Supprimer les permissions d’un rôle](/docs/fr-CA/manage-users/access-control/configure-core-rbac/roles/remove-permissions-from-roles)
* [Supprimer les permissions d’un utilisateur](/docs/fr-CA/manage-users/access-control/configure-core-rbac/rbac-users/remove-permissions-from-users)
