Vous pouvez créer des [rôles](/docs/fr-CA/manage-users/access-control/rbac) dans le Dashboard d’Auth0 ou à l’aide de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. Ces rôles peuvent être utilisés avec l’ensemble de fonctionnalités Authorization Core pour les API.

<div id="prerequisites">
  ## Prérequis
</div>

* Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API dans le Dashboard ou au moyen de la Management API. La fonctionnalité Authorization Core est différente de l’Authorization Extension. Pour en savoir plus, consultez [Authorization Core vs. Authorization Extension](/docs/fr-CA/manage-users/access-control/authorization-core-vs-authorization-extension).
* [Configurer une API](/docs/fr-CA/get-started/auth0-overview/set-up-apis) dans le <Tooltip tip="Auth0 Dashboard : le produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.
* Les permissions sont sélectionnées à partir de valeurs prédéfinies. Si votre liste de permissions est vide, vous devez [ajouter des permissions](/docs/fr-CA/get-started/apis/add-api-permissions) à votre API.

<div id="dashboard">
  ## Dashboard
</div>

1. Allez à [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles) et cliquez sur **Create Role**.
2. Donnez un nom au rôle et ajoutez une description, puis cliquez sur **Create**.

<div id="management-api">
  ## Management API
</div>

Effectuez un appel `POST` au [point de terminaison Create Role](https://auth0.com/docs/api/management/v2#!/Roles/post_roles). Assurez-vous de remplacer les valeurs fictives `MGMT_API_ACCESS_TOKEN`, `ROLE_NAME` et `ROLE_DESC` par, respectivement, votre <Tooltip tip="Jeton d’accès : information d’identification d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=access+token">jeton d’accès</Tooltip> pour la Management API, le nom du rôle et sa description.

| **Valeur**              | **Description**                                                                                                       |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec la portée `create:roles`. |
| `ROLE_NAME`             | Nom du rôle que vous souhaitez créer.                                                                                 |
| `ROLE_DESC`             | Description claire du rôle.                                                                                           |
