Vous pouvez créer des [rôles](/docs/fr-CA/manage-users/access-control/rbac) à l’aide de l’Auth0 Dashboard ou de la <Tooltip tip="Management API : un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. Les rôles peuvent être utilisés avec le jeu de fonctionnalités Authorization Core de l’API.

<div id="prerequisites">
  ## Prérequis
</div>

* Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API dans le Dashboard ou à l’aide de la Management API. La fonctionnalité Authorization Core est distincte de l’Authorization Extension. Pour comparer les deux, consultez [Authorization Core vs. Authorization Extension](/docs/fr-CA/manage-users/access-control/authorization-core-vs-authorization-extension).
* [Configurer une API](/docs/fr-CA/get-started/auth0-overview/set-up-apis) dans le <Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.
* Les permissions sont sélectionnées parmi des valeurs prédéfinies. Si votre liste de permissions est vide, vous devez [ajouter des permissions](/docs/fr-CA/get-started/apis/add-api-permissions) à votre API.

<div id="dashboard">
  ## Dashboard
</div>

1. Accédez à [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles), puis cliquez sur **Create Role**.
2. Donnez un nom au rôle et ajoutez une description, puis cliquez sur **Create**.

<div id="management-api">
  ## Management API
</div>

Effectuez un appel `POST` au [point de terminaison Create Role](https://auth0.com/docs/api/management/v2#!/Roles/post_roles). Assurez-vous de remplacer les valeurs d’espace réservé `MGMT_API_ACCESS_TOKEN`, `ROLE_NAME` et `ROLE_DESC` par votre <Tooltip tip="Jeton d’accès : information d’authentification, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=access+token">jeton d’accès</Tooltip> de la Management API, le nom du rôle et la description du rôle, respectivement.

| **Valeur**              | **Description**                                                                                                        |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec la portée `create:roles`. |
| `ROLE_NAME`             | Nom du rôle que vous souhaitez créer.                                                                                  |
| `ROLE_DESC`             | Description du rôle destinée aux utilisateurs.                                                                         |
