Vous pouvez supprimer un [rôle](/docs/fr-CA/manage-users/access-control/rbac) à l&#39;aide du <Tooltip tip="Auth0 Dashboard : le principal produit d'Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou de la <Tooltip tip="Auth0 Dashboard : le principal produit d'Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. Les rôles sont utilisés avec l&#39;ensemble de fonctionnalités Authorization Core pour les API.

Pour que le contrôle d&#39;accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l&#39;activer pour votre API à l&#39;aide du Dashboard ou de la Management API. La fonctionnalité Authorization Core diffère de l&#39;Authorization Extension. Pour les comparer, consultez [Authorization Core vs. Authorization Extension](/docs/fr-CA/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Allez à [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles), puis cliquez sur le nom du rôle à consulter.
2. Cliquez sur **Remove this rôle**, puis confirmez.

<div id="management-api">
  ## Management API
</div>

Effectuez un appel `DELETE` au [point de terminaison Delete Role](https://auth0.com/docs/api/management/v2#!/Roles/delete_roles_by_id). Assurez-vous de remplacer les valeurs d’espace réservé `ROLE_ID` et `MGMT_API_ACCESS_TOKEN` par l’ID de votre rôle et le <Tooltip tip="Jeton d’accès : information d’identification d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=access+token">jeton d’accès</Tooltip> de la Management API, respectivement.

<AuthCodeGroup>
  ```bash cURL
  curl --request DELETE \
    --url 'https://{yourDomain}/api/v2/roles/ROLE_ID' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/roles/ROLE_ID");
  var request = new RestRequest(Method.DELETE);
  request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main
  ```

</AuthCodeGroup>

| Valeur                  | Description                                                                                                            |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `ROLE_ID`               | L’ID du rôle que vous souhaitez supprimer.                                                                             |
| `MGMT_API_ACCESS_TOKEN` | [Jeton d’accès pour la Management API](https://auth0.com/docs/api/management/v2/tokens) avec la portée `delete:roles`. |
