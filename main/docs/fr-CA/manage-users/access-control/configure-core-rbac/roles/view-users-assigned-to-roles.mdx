description: Décrit comment afficher les utilisateurs assignés à un rôle à l’aide de l’Auth0 Dashboard ou de la Management API.
title: Afficher les utilisateurs assignés à des rôles
---

Vous pouvez afficher les utilisateurs assignés à un [rôle](/docs/fr-CA/manage-users/access-control/rbac) à l’aide de l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou de la <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. Les rôles sont utilisés avec l’ensemble de fonctionnalités Authorization Core pour les API.

<div id="prerequisites">
  ## Prérequis
</div>

Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API dans le Dashboard ou au moyen de la Management API. La fonctionnalité Authorization Core est distincte de l’Authorization Extension. Pour une comparaison, consultez [Authorization Core vs. Authorization Extension](/docs/fr-CA/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Allez à [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/users), puis cliquez sur le nom du rôle que vous voulez consulter.
2. Cliquez sur l’onglet **Users**.

Les renseignements suivants s’affichent pour chaque utilisateur :

| Colonne          | Description                                                    |
| ---------------- | -------------------------------------------------------------- |
| Image            | Image de l’utilisateur tirée du profil utilisateur.            |
| Name             | Nom de l’utilisateur tiré du profil utilisateur.               |
| adresse courriel | Adresse courriel de l’utilisateur tirée du profil utilisateur. |

<div id="management-api">
  ## Management API
</div>

Pour récupérer la liste des utilisateurs assignés à un rôle, consultez [Obtenir les utilisateurs d’un rôle](https://auth0.com/docs/api/management/v2#!/Roles/get_role_user).
