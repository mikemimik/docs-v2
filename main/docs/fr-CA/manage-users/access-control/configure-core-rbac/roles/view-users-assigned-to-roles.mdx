description: Décrit comment voir les utilisateurs attribués à un rôle dans l’Auth0 Dashboard ou à l’aide de la Management API.
title: Voir les utilisateurs attribués à des rôles
---

Vous pouvez voir les utilisateurs attribués à un [rôle](/docs/fr-CA/manage-users/access-control/rbac) dans le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou à l’aide de la <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>. Les rôles sont utilisés avec l’ensemble de fonctionnalités Authorization Core de l’API.

<div id="prerequisites">
  ## Prérequis
</div>

Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API à l’aide du Dashboard ou de la Management API. La fonctionnalité Authorization Core diffère de l’Authorization Extension. Pour les comparer, consultez [Authorization Core vs. Authorization Extension](/docs/fr-CA/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Accédez à [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/users), puis cliquez sur le nom du rôle à afficher.
2. Cliquez sur l’onglet **Users**.

Les renseignements suivants s’affichent pour chaque utilisateur :

| Colonne       | Description                                                    |
| ------------- | -------------------------------------------------------------- |
| Picture       | Photo de l’utilisateur tirée du profil utilisateur.            |
| Name          | Nom de l’utilisateur tiré du profil utilisateur.               |
| Email Address | Adresse courriel de l’utilisateur tirée du profil utilisateur. |

<div id="management-api">
  ## Management API
</div>

Pour récupérer la liste des utilisateurs attribués à un rôle, consultez [Récupérer les utilisateurs d’un rôle](https://auth0.com/docs/api/management/v2#!/Roles/get_role_user).
