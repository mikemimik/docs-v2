title: Rechercher des utilisateurs
---

La recherche d’utilisateurs vous permet de récupérer les détails du profil d’un utilisateur au moyen de l’[Auth0 Management API](https://auth0.com/docs/api/management/v2). Les résultats de recherche peuvent être [affichés](/docs/fr-CA/manage-users/user-search/view-search-results-by-page), [triés](/docs/fr-CA/manage-users/user-search/sort-search-results) et [exportés](/docs/fr-CA/manage-users/user-migration/bulk-user-exports).

La plupart des champs du profil d’un utilisateur ne sont pas renvoyés dans un [ID Token](/docs/fr-CA/secure/tokens/id-tokens), ni inclus dans la réponse de le point de terminaison [`/userinfo`](https://auth0.com/docs/api/authentication#get-user-info) de l’Authentication API.

Lorsque vous recherchez des utilisateurs dans Auth0, vous pouvez utiliser plusieurs points de terminaison pour effectuer une recherche par ID, courriel ou selon d’autres critères :

<div id="search-result-terminology">
  ## Terminologie des résultats de recherche
</div>

Nous utilisons les termes suivants pour décrire les résultats de recherche d’utilisateurs :

* **Cohérence éventuelle** : Les résultats de recherche peuvent ne pas refléter une opération d’écriture tout juste terminée. Toutefois, si vous répétez votre requête après un court délai, la réponse renverra des données à jour.
* **Cohérence immédiate** : Les résultats de recherche refléteront les résultats de toutes les opérations d’écriture réussies, y compris celles qui ont eu lieu peu avant votre requête.
