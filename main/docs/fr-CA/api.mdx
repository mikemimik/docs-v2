<div class="api_card_wrapper">
  <Card title="Authentication API" href="/docs/fr-CA/api/authentication" arrow />
</div>

<Columns cols={2}>
  <Column>
    L’Authentication API expose des fonctionnalités d’identité pour Auth0 et les protocoles d’identité pris en charge (y compris OpenID Connect, OAuth et SAML).

    En règle générale, vous devriez utiliser cette API par l’intermédiaire d’un des Auth0 SDKs, comme [Auth0.js](/docs/fr-CA/libraries/auth0js), ou d’une bibliothèque comme [Lock](/docs/fr-CA/libraries/lock). Toutefois, si vous créez votre interface d’authentication manuellement, vous devrez appeler directement l’Authentication API.
  </Column>

  <Column>
    Voici quelques exemples de tâches :

    * Obtenir des [jetons](/docs/fr-CA/secure/tokens) pendant l’authentication
    * Demander le profil d’un utilisateur à l’aide d’un [jeton d’accès](/docs/fr-CA/secure/tokens/access-tokens)
    * Échanger des [jetons d’actualisation](/docs/fr-CA/secure/tokens/refresh-tokens) contre de nouveaux jetons d’accès
    * Demander une vérification pour l’[authentification multifacteur (MFA)](/docs/fr-CA/secure/multi-factor-authentication)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="Management API" href="/docs/fr-CA/api/management/v2" arrow />
</div>

<Columns cols={2}>
  <Column>
    La Management API vous permet de gérer votre compte Auth0 par programmation afin d’automatiser la configuration de votre environnement. La plupart des tâches que vous pouvez effectuer dans le Dashboard d’Auth0 peuvent aussi être réalisées par programmation à l’aide de cette API.
  </Column>

  <Column>
    Voici quelques exemples de tâches :

    * Enregistrer vos applications et vos API auprès d’Auth0
    * Configurer les [connexions](/docs/fr-CA/connections) que vos utilisateurs peuvent utiliser pour s’authentifier
    * [Gérer les utilisateurs](/docs/fr-CA/manage-users)
    * [Lier des comptes d’utilisateur](/docs/fr-CA/manage-users/user-accounts/user-account-linking/link-user-accounts)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="My Account API" href="/docs/fr-CA/api/myaccount" arrow />
</div>

<Columns cols={2}>
  <Column>
    L’API Auth0 My Account fournit un ensemble dédié de points de terminaison permettant aux utilisateurs de gérer les renseignements de leur propre compte. Vous pouvez utiliser ces API pour créer des expériences en libre-service dans vos applications ou ajouter progressivement des détails à un compte d’utilisateur.
  </Column>

  <Column>
    Voici quelques exemples de tâches :

    * Permettre à l’utilisateur final de gérer ses propres facteurs
    * Mettre à jour les détails d’une méthode d’authentification
    * Savoir quelles méthodes d’authentification sont offertes
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="My Organization API" href="/docs/fr-CA/api/myorganization" arrow />
</div>

<Columns cols={2}>
  <Column>
    L’API Auth0 My Organization fournit une interface sécurisée, limitée à l’Organization, qui permet à vos clients d’affaires de gérer leurs propres Organizations au sein de votre tenant Auth0. Cette API sert de fondement technique à l’administration déléguée intégrée et aux intégrations API-first. Pour en savoir plus, consultez [My Organzation API and Embeddable UI Components](/docs/fr-CA/manage-users/my-organization-api).
  </Column>

  <Column>
    Voici quelques exemples de tâches :

    * Détails de l’Auth0 Organization (nom, image de marque, nom d’affichage)
    * Configuration, propriété et relations propres à l’Organization
    * Identity Providers (IdPs) et provisionnement SCIM
    * Configuration des domaines et de Home Realm Discovery (HRD)
  </Column>
</Columns>

<div id="pagination">
  ## Pagination
</div>

Pour les tenants Public Cloud, les points de terminaison de la Management API renvoient un maximum de 50 résultats par réponse. S’il y a plus de résultats qu’une réponse ne peut en contenir, incluez les paramètres `page` et `per_page` dans votre requête pour en récupérer davantage.

Consultez [Migrer vers les requêtes paginées du point de terminaison Management API v2](/docs/fr-CA/troubleshoot/product-lifecycle/past-migrations/migrate-to-paginated-queries) pour savoir comment mettre à jour vos applications.
