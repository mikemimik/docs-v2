---
title: Référence des événements du journal du tenant pour les règles de la Tenant Access Control List
sidebarTitle: Événements du journal du tenant
description: Consultez la description de tous les champs de l’événement du journal du tenant Tenant ACL (`acls_summary`).
---

L’[événement du journal](/docs/fr-CA/deploy-monitor/logs) `acls_summary` comprend les champs suivants :

<ResponseField name="acls_summary" type="object">
  <Expandable>
    <ResponseField name="acl_id" type="string">
      Identifiant interne de la règle.

      Exemple : `acl_vENikGo5fcJd7FYMLvjKKP`
    </ResponseField>

    <ResponseField name="description" type="string">
      Description de la règle.

      Exemple : `Allow traffic that originates from the United States`
    </ResponseField>

    <ResponseField name="priority" type="number">
      Ordre d’exécution de la règle.

      Exemple : `1`
    </ResponseField>

    <ResponseField name="action" type="object">
      Action exécutée par la règle.

      Exemple : `{ "allow": true }`
    </ResponseField>

    <ResponseField name="match" type="object">
      <Expandable>
        <ResponseField name="success" type="number">
          Nombre total de requêtes auxquelles la règle correspondait avec succès.

          Exemple : `1`
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="total_request_count" type="object">
      <Expandable>
        <ResponseField name="successes" type="number">
          Nombre total de requêtes évaluées avec succès par la règle.

          Exemple : `3`
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="start_time" type="string">
      Horodatage du début de la période de comptage du journal.

      Exemple : `2025-03-19T16:11:07.429Z`
    </ResponseField>

    <ResponseField name="end_time" type="string">
      Horodatage de la fin de la période de comptage du journal.

      Exemple : `2025-03-19T16:21:07.429Z`
    </ResponseField>
  </Expandable>
</ResponseField>
