Auth0 offre deux API distinctes pour vous aider, ainsi que vos utilisateurs, à gérer les facteurs d’authentification multifacteur <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur supplémentaire, en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=MFA">MFA</Tooltip>.

<div id="auth0-authentication-api">
  ## Auth0 Authentication API
</div>

Utilisez les [points de terminaison d’authentification multifacteur de l’Authentication API](https://auth0.com/docs/api/authentication#multi-factor-authentication) pour permettre à vos utilisateurs de gérer leurs propres facteurs d’authentification multifacteur.

Cette méthode s’appuie sur le flux d’authentification Password Grant (ROPG) du <Tooltip tip="Resource Owner : entité (comme un utilisateur ou une application) capable d’accorder l’accès à une ressource protégée." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Resource+Owner">Resource Owner</Tooltip>. Pour en savoir plus, consultez [Gérer les facteurs d’authentification avec l’Authentication API](/docs/fr-CA/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

<div id="auth0-management-api">
  ## Auth0 Management API
</div>

Utilisez les [points de terminaison Users de la Management API](https://auth0.com/docs/api/management/v2) pour gérer les facteurs d’authentification multifacteur de vos utilisateurs.

Cette méthode exige une authentification au moyen d’une application confidentielle. Pour en savoir plus, consultez [Gérer les méthodes d’authentification avec la Management API](/docs/fr-CA/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authentication-methods-with-management-api).
