Auth0 propose deux API distinctes pour vous aider, vous et vos utilisateurs, à gérer les facteurs d’authentification <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur reposant sur un facteur supplémentaire au username et au Password, comme un code envoyé par SMS." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=MFA">MFA</Tooltip>.

<div id="auth0-authentication-api">
  ## API d&#39;authentification Auth0
</div>

Utilisez les [points de terminaison d’authentification multifacteur de l’API d’authentification](https://auth0.com/docs/api/authentication#multi-factor-authentication) pour permettre à vos utilisateurs de gérer eux-mêmes leurs facteurs d’authentification MFA.

Cette méthode s’appuie sur le flux d’authentification <Tooltip tip="Resource Owner : entité (comme un utilisateur ou une application) capable d'accorder l'accès à une ressource protégée." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Resource+Owner">Resource Owner</Tooltip> Password Grant (ROPG). Pour en savoir plus, consultez [Gérer les facteurs d’authentification avec l’API d’authentification](/docs/fr-CA/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

<div id="auth0-management-api">
  ## Auth0 API de gestion
</div>

Utilisez les [points de terminaison Users de l’API de gestion](https://auth0.com/docs/api/management/v2) pour gérer les facteurs d’authentification MFA de vos utilisateurs.

Cette méthode nécessite une authentification à l’aide d’une application confidentielle. Pour en savoir plus, consultez [Manage Authentication Methods with API de gestion](/docs/fr-CA/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authentication-methods-with-management-api).
