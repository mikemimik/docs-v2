title: Auth0 MFA API
---

Auth0 offre un flux intégré d’inscription et d’authentification pour l’<Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA) avec [Universal Login](/docs/fr-CA/authenticate/login/auth0-universal-login). Utilisez la [MFA API](https://auth0.com/docs/api/authentication#multi-factor-authentication) dans les scénarios suivants si vous voulez :

* [Authentifier les utilisateurs avec le Resource Owner Password Grant](/docs/fr-CA/secure/multi-factor-authentication/authenticate-using-ropg-flow-with-mfa).
* Créer une interface permettant aux [utilisateurs de gérer leurs propres facteurs d’authentification](/docs/fr-CA/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

Pour utiliser la MFA API, vous devez activer le type de grant MFA pour votre application. Pour activer le grant MFA dans le <Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, accédez à [Dashboard &gt; Applications &gt; Advanced Settings &gt; Grant Types](https://manage.auth0.com/#/applications) et sélectionnez **MFA**. Consultez [Enable Multi-Factor Authentication](/docs/fr-CA/secure/multi-factor-authentication/enable-mfa) pour en savoir plus.

<div id="limitations">
  ## Limitations
</div>

La MFA API est conçue pour fonctionner avec les facteurs SMS, d’appel vocal, push au moyen de l’application Auth0 Guardian, de courriel et OTP. Elle ne prend pas actuellement en charge l’enrôlement avec Duo ou WebAuthn.
