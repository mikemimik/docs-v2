---
description: Répertorie les codes d’erreur Guardian et leur description.
title: Référence des codes d’erreur Guardian
---

Utilisez les codes d’erreur pour afficher des messages informatifs et distinguer les erreurs récupérables des erreurs irrécupérables.

| Code d’erreur                         | Description                                                                                                                                                    |
| ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `invalid_token`                       | Jeton de requête ou de transaction invalide                                                                                                                    |
| `insufficient_scope`                  | Vous n’avez pas les autorisations nécessaires pour effectuer l’opération demandée                                                                              |
| `invalid_bearer_format`               | Le jeton bearer indiqué dans l’en-tête d’authentification n’était pas valide                                                                                   |
| `enrollment_conflict`                 | Il existe déjà une inscription pour cet utilisateur. Vous ne pouvez pas vous inscrire deux fois.                                                               |
| `tenant_not_found`                    | Le tenant associé est introuvable. Cela ne devrait normalement pas se produire, sauf si vous avez supprimé le tenant                                           |
| `login_transaction_not_found`         | La transaction d’authentification MFA n’est pas active ou a déjà expiré                                                                                        |
| `error_sending_push_notification`     | Échec de l’envoi de la notification push                                                                                                                       |
| `push_notification_wrong_credentials` | L’envoi de la notification push a échoué en raison d’identifiants incorrects                                                                                   |
| `invalid_otp`                         | Le code OTP fourni n’était pas valide                                                                                                                          |
| `invalid_recovery_code`               | Le code de récupération fourni n’était pas valide                                                                                                              |
| `invalid_body`                        | La validation du Body a échoué. Requête incorrecte.                                                                                                            |
| `invalid_query_string`                | La validation de la chaîne de requête a échoué. Requête incorrecte.                                                                                            |
| `enrollment_transaction_not_found`    | La transaction d’inscription MFA n’est pas active ou a expiré                                                                                                  |
| `invalid_phone_number`                | Le numéro de téléphone fourni est invalide                                                                                                                     |
| `error_sending_sms`                   | Erreur d’envoi du SMS                                                                                                                                          |
| `feature_disabled`                    | La fonctionnalité demandée n’est actuellement pas disponible globalement (communiquez avec le fournisseur)                                                     |
| `feature_disabled_by_admin`           | La fonctionnalité demandée est actuellement désactivée par votre administrateur                                                                                |
| `pn_endpoint_disabled`                | Nous n’avons pas pu livrer la notification push après de nombreuses tentatives. Essayez de supprimer votre compte de l’appareil, puis de l’ajouter de nouveau. |
| `too_many_sms`                        | Vous avez dépassé le nombre de SMS attribués à votre utilisateur                                                                                               |
| `too_many_pn`                         | Vous avez dépassé le nombre de notifications push attribuées à votre utilisateur                                                                               |
| `too_many_sms_per_tenant`             | Vous avez dépassé le nombre de SMS attribués à votre tenant                                                                                                    |
| `too_many_pn_per_tenant`              | Vous avez dépassé le nombre de notifications push attribuées à votre tenant                                                                                    |
| `field_required`                      | Un champ est requis pour effectuer l’opération (cette erreur comprend aussi un attribut field avec un code pour le champ : `otpCode`, `recoveryCode`)          |
| `method_not_found`                    | Vous avez demandé une méthode qui n’est actuellement pas prise en charge (cela ne devrait pas se produire)                                                     |
| `no_method_available`                 | Il n’existe actuellement aucune méthode d’inscription (elles sont toutes désactivées)                                                                          |
| `enrollment_method_disabled`          | La méthode d’inscription indiquée est désactivée; cette erreur comprend aussi un champ .method                                                                 |
| `auth_method_disabled`                | La méthode d’authentification indiquée est désactivée; cette erreur comprend aussi un champ .method                                                            |
| `invalid_otp_format`                  | Erreur de validation du format OTP                                                                                                                             |
| `invalid_recovery_code_format`        | Erreur de validation du format du code de récupération                                                                                                         |
| `transaction_expired`                 | La transaction a déjà expiré                                                                                                                                   |
| `already_enrolled`                    | Vous êtes déjà inscrit, vous ne pouvez pas vous inscrire de nouveau                                                                                            |
| `not_enrolled`                        | Vous n’êtes pas inscrit. Vous devez d’abord vous inscrire                                                                                                      |
| `invalid_enrollment`                  | L’inscription fournie à la méthode transaction#requestAuth n’est pas valide ou est null/undefined                                                              |
