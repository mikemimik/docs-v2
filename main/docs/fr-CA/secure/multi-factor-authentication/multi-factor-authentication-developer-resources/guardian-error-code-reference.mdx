---
description: Répertorie les codes d’erreur Guardian et leur description.
title: Référence des codes d’erreur Guardian
---

Utilisez les codes d’erreur pour afficher des messages informatifs et distinguer les erreurs récupérables des erreurs irrécupérables.

| Code d’erreur                         | Description                                                                                                                                                |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `invalid_token`                       | Jeton de requête ou de transaction invalide                                                                                                                |
| `insufficient_scope`                  | Vous n’avez pas suffisamment de grants pour effectuer l’opération demandée                                                                                 |
| `invalid_bearer_format`               | Le bearer placé dans l’en-tête d’authentification n’était pas valide                                                                                       |
| `enrollment_conflict`                 | Il existe déjà une autre inscription pour le même utilisateur. Vous ne pouvez pas vous inscrire deux fois.                                                 |
| `tenant_not_found`                    | Le tenant associé est introuvable. Cela ne devrait normalement pas se produire, sauf si vous supprimez le tenant                                           |
| `login_transaction_not_found`         | La transaction d’authentification MFA n’est pas active ou a déjà expiré                                                                                    |
| `error_sending_push_notification`     | L’envoi de la notification push a échoué                                                                                                                   |
| `push_notification_wrong_credentials` | L’envoi de la notification push a échoué en raison de credentials incorrects                                                                               |
| `invalid_otp`                         | Le code OTP fourni n’était pas valide                                                                                                                      |
| `invalid_recovery_code`               | Le recovery code fourni n’était pas valide                                                                                                                 |
| `invalid_body`                        | La validation du Body a échoué. Requête incorrecte.                                                                                                        |
| `invalid_query_string`                | La validation de la chaîne de requête a échoué. Requête incorrecte.                                                                                        |
| `enrollment_transaction_not_found`    | La transaction d’inscription MFA n’est pas active ou a expiré                                                                                              |
| `invalid_phone_number`                | Le numéro de téléphone fourni est invalide                                                                                                                 |
| `error_sending_sms`                   | Erreur d’envoi de SMS                                                                                                                                      |
| `feature_disabled`                    | La fonctionnalité demandée n’est actuellement pas disponible de façon globale (contactez le Provider)                                                      |
| `feature_disabled_by_admin`           | La fonctionnalité demandée est actuellement désactivée par votre Admin                                                                                     |
| `pn_endpoint_disabled`                | Nous n’avons pas pu livrer la notification push après de nombreuses tentatives. Essayez de supprimer votre compte de l’appareil et de l’ajouter à nouveau. |
| `too_many_sms`                        | Vous avez dépassé le nombre de SMS attribué à votre utilisateur                                                                                            |
| `too_many_pn`                         | Vous avez dépassé le nombre de notifications push attribué à votre utilisateur                                                                             |
| `too_many_sms_per_tenant`             | Vous avez dépassé le nombre de SMS attribué à votre tenant                                                                                                 |
| `too_many_pn_per_tenant`              | Vous avez dépassé le nombre de notifications push attribué à votre tenant                                                                                  |
| `field_required`                      | Un champ est requis pour effectuer l’opération (ces erreurs ont un attribut field avec un code pour le champ : `otpCode`, `recoveryCode`)                  |
| `method_not_found`                    | Vous avez demandé une méthode qui n’est actuellement pas prise en charge (cela ne devrait pas arriver)                                                     |
| `no_method_available`                 | Il n’existe actuellement aucune méthode d’inscription (elles sont toutes désactivées)                                                                      |
| `enrollment_method_disabled`          | La méthode d’inscription spécifiée est désactivée; cette erreur a aussi un champ .method                                                                   |
| `auth_method_disabled`                | La méthode d’authentification spécifiée est désactivée; cette erreur a aussi un champ .method                                                              |
| `invalid_otp_format`                  | Erreur de validation du format OTP                                                                                                                         |
| `invalid_recovery_code_format`        | Erreur de validation du format du recovery code                                                                                                            |
| `transaction_expired`                 | La transaction a déjà expiré                                                                                                                               |
| `already_enrolled`                    | Vous êtes déjà inscrit, impossible de vous inscrire de nouveau                                                                                             |
| `not_enrolled`                        | Vous n’êtes pas inscrit. Vous devez d’abord vous inscrire                                                                                                  |
| `invalid_enrollment`                  | L’inscription fournie à la méthode transaction#requestAuth n’est pas valide ou est null/undefined                                                          |
