title: Mettre à jour la durée de vie du ID token
---

Vous pouvez modifier la durée de vie du <Tooltip tip="ID Token : informations d’identification destinées au client lui-même, plutôt que pour accéder à une ressource." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=ID+token">ID token</Tooltip> à l’aide du <Tooltip tip="ID Token : informations d’identification destinées au client lui-même, plutôt que pour accéder à une ressource." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Par défaut, un ID token est valide pendant 36000 secondes (10 heures). En cas de préoccupations liées à la sécurité, vous pouvez réduire le délai avant son expiration, tout en gardant à l’esprit que l’un de ses rôles est d’améliorer l’expérience utilisateur en mettant en cache les renseignements sur l’utilisateur.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) et cliquez sur le nom de l’application à consulter.
2. Faites défiler jusqu’à **ID Token** et repérez le champ **ID Token Expiration**. Saisissez la durée de vie du ID token appropriée (en secondes) pour l’application.
3. Cliquez sur **Save Changes**.
