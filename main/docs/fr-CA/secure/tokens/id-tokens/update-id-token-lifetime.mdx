title: Mettre à jour la durée de vie de l’ID token
---

Vous pouvez modifier la durée de vie du <Tooltip tip="ID Token : Information d’identification destinée au client lui-même, plutôt qu’à l’accès à une ressource." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=ID+token">ID token</Tooltip> dans l’<Tooltip tip="ID Token : Information d’identification destinée au client lui-même, plutôt qu’à l’accès à une ressource." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Par défaut, un ID token est valide pendant 36000 secondes (10 heures). Si vous avez des préoccupations en matière de sécurité, vous pouvez raccourcir cette durée avant l’expiration du token, en gardant à l’esprit que l’un des objectifs du token est d’améliorer l’expérience utilisateur grâce à la mise en cache des renseignements sur l’utilisateur.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis cliquez sur le nom de l’application à afficher.
2. Faites défiler jusqu’à **ID Token** et repérez le champ **ID Token Expiration**. Entrez la durée de vie appropriée de l’ID token (en secondes) pour l’application.
3. Cliquez sur **Save Changes**.
