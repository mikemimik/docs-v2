description: Découvrez comment fonctionnent les jetons d’accès de l’Auth0 Management API et comment les utiliser.
title: Jetons d’accès de la Management API
---

Pour envoyer une requête à l’[Auth0 Management API](https://auth0.com/docs/api/management/v2), vous devez fournir un <Tooltip tip="Jeton d’accès : information d’identification d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=access+token">jeton d’accès</Tooltip> qui contient l’<Tooltip tip="Jeton d’accès : information d’identification d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=audience">audience</Tooltip> et les scopes appropriés.

<div id="audience">
  ## Audience
</div>

La valeur d’audience de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip> est `https://{yourDomain}/api/v2/`.

<div id="scopes">
  ## Scopes
</div>

Chaque point de terminaison de la Management API nécessite un ensemble précis de scopes. Consultez le [Management API Explorer](https://auth0.com/docs/api/management/v2/) pour plus d’informations.

<div id="lifetime">
  ## Durée de vie
</div>

La durée de vie par défaut du jeton pour la Management API est de 86 400 secondes (24 heures).

Vous pouvez [configurer la durée de vie du jeton](/docs/fr-CA/secure/tokens/access-tokens/update-access-token-lifetime) pour chaque API individuellement.

<div id="security">
  ## Sécurité
</div>

* **Jeton d’accès** : Une fois émis, un jeton d’accès ne peut pas être révoqué. Auth0 recommande de limiter la durée de vie de vos jetons d’accès à la valeur la plus courte possible pour votre cas d’utilisation.
* **<Tooltip tip="Refresh Token : Jeton utilisé pour obtenir un nouveau jeton d’accès sans obliger les utilisateurs à se connecter de nouveau." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Refresh+token">Jeton d’actualisation</Tooltip>** : Vous pouvez [révoquer un jeton d’actualisation](/docs/fr-CA/secure/tokens/refresh-tokens/revoke-refresh-tokens) à l’aide du Dashboard, de l’Authentication API ou de la Management API.
* **<Tooltip tip="Client Secret : Secret utilisé par un client (application) pour s’authentifier auprès du serveur d’autorisation; il ne doit être connu que du client et du serveur d’autorisation et doit être suffisamment aléatoire pour ne pas pouvoir être deviné." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Client+secret">Secret client</Tooltip>** : Vous pouvez [effectuer la rotation du secret client d’une application](/docs/fr-CA/get-started/applications/rotate-client-secret) à l’aide du Dashboard ou de la Management API.

<div id="quota">
  ## Quota
</div>

Les quotas de jetons sont déterminés selon le niveau d’abonnement. Vous pouvez [consulter votre quota actuel](https://support.auth0.com/reports/quota) dans l’Auth0 Support Center. Si vous avez des questions sur les tarifs ou les quotas, veuillez [communiquer avec notre équipe des ventes](https://auth0.com/contact-us).

* Les jetons émis pour des [audiences internes d’Auth0](/docs/fr-CA/secure/tokens/access-tokens/get-access-tokens#control-access-token-audience) ne sont pas comptabilisés dans votre quota.
* Les jetons émis pour des audiences personnalisées sont comptabilisés dans votre quota et sont assujettis aux [limites de taux de l’Auth0 Management API](/docs/fr-CA/troubleshoot/customer-support/operational-policies/rate-limit-policy/management-api-endpoint-rate-limits).
