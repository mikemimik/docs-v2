Vous pouvez modifier la durée de vie du <Tooltip tip="Jeton d’accès : information d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=access+token">jeton d’accès</Tooltip> dans <Tooltip tip="Jeton d’accès : information d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="configure-access-token-lifetime">
  ## Configurer la durée de vie du jeton d’accès
</div>

1. Accédez à [Dashboard &gt; Applications &gt; APIs](https://manage.auth0.com/#/apis), puis sélectionnez le nom de l’API à consulter.

   <Frame>![Liste des API dans Dashboard Applications](/docs/images/cdy7uua7fh8z/3rhmhghYZDSi6YWHRA5yMQ/c71340259481b0b6787d5f3887cfda0f/dashboard-apis-list.png)</Frame>
2. Repérez le champ **Durée de vie maximale du jeton d’accès** sous **Access Token Expiration**.

   <Frame>![Dashboard - API - Paramètres du jeton - Expiration](/docs/images/cdy7uua7fh8z/2FFfqqmntXL9CnGrSxea9M/810e5542553168ffcd6081a10bc2db2f/image2.png)</Frame>
3. Entrez la durée de vie souhaitée (en secondes) pour les jetons d’accès émis pour cette API.

   * La valeur par défaut est de `86,400` secondes (24 heures).
   * La valeur maximale est de `2,592,000 seconds` (30 jours).
4. Sélectionnez **Save Changes**.

<div id="token-expiration-for-browser-flows">
  ## Expiration du jeton pour les flux du navigateur
</div>

Le champ **Durée de vie du jeton d’accès pour le flux implicite / hybride** fait référence aux jetons d’accès émis pour l’API au moyen des flux implicites et hybrides et ne couvre pas tous les flux lancés à partir du navigateur.

Par exemple, le flux PKCE (utilisé dans le SDK `auth0-js-spa`) peut être lancé à partir du navigateur, mais il fait référence à la valeur **Expiration du jeton**, et non à la valeur **Durée de vie du jeton d’accès pour le flux implicite / hybride**.

<div id="restricted-lifetime-for-mfa-access-tokens">
  ## Durée de vie restreinte des jetons d’accès MFA
</div>

La durée de vie des jetons d’accès dont l’<Tooltip tip="Audience : identifiant unique du destinataire visé par un jeton émis. Nommée aud dans un jeton, sa valeur contient l’ID d’une application (Client ID) pour un ID Token ou d’une API (API Identifier) pour un Access Token." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=audience">audience</Tooltip> est `{yourAuth0Domain}/mfa` est limitée à `600` secondes (10 minutes) pour des raisons de sécurité et ne peut pas être modifiée.
