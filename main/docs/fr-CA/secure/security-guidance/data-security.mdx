* [Adresses IP Auth0 pour les listes d’autorisation](/docs/fr-CA/secure/security-guidance/data-security/allowlist) - Si votre environnement est protégé par un pare-feu, les connexions de base de données personnalisées, Actions, Hooks et Rules peuvent nécessiter l’ajout de certaines adresses IP d’Auth0 à l’AllowList pour assurer leur bon fonctionnement.
* [Ajouter des attributs utilisateur à la Deny List](/docs/fr-CA/secure/security-guidance/data-security/denylist) - Si certains champs utilisateur ne doivent pas être stockés dans les bases de données Auth0 pour des raisons de confidentialité, vous pouvez ajouter des attributs utilisateur à la Deny List.
* [Stockage des données utilisateur](/docs/fr-CA/secure/security-guidance/data-security/user-data-storage) - Découvrez les différences entre l’utilisation du magasin de données Auth0 et d’une base de données externe pour stocker les données utilisateur.
* [Stockage des jetons](/docs/fr-CA/secure/security-guidance/data-security/token-storage) - Découvrez comment et où stocker les jetons utilisés dans une authentification basée sur des jetons.
