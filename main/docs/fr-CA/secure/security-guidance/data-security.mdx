title: Sécurité des données

* [Adresses IP Auth0 pour les listes d’autorisation](/docs/fr-CA/secure/security-guidance/data-security/allowlist) - Si votre environnement est protégé par un pare-feu, les connexions de base de données personnalisées, Actions, Hooks et Rules peuvent exiger que vous ajoutiez certaines adresses IP d’Auth0 à l’AllowList afin d’assurer le bon fonctionnement.
* [Ajouter des attributs utilisateur à la liste de refus](/docs/fr-CA/secure/security-guidance/data-security/denylist) - Si certains champs utilisateur ne doivent pas être stockés dans les bases de données Auth0 pour des raisons de confidentialité, vous pouvez ajouter des attributs utilisateur à la liste de refus.
* [Stockage des données utilisateur](/docs/fr-CA/secure/security-guidance/data-security/user-data-storage) - Découvrez les différences entre le stockage de données d’Auth0 et une base de données externe pour stocker les données utilisateur.
* [Stockage des jetons](/docs/fr-CA/secure/security-guidance/data-security/token-storage) - Découvrez comment et où stocker les jetons utilisés dans l’authentification par jeton.
