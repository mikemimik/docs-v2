---
title: "Dissocier"
description: "Point de terminaison obsolète permettant de dissocier des comptes d’utilisateur; utilisez plutôt Management API."
---

<div id="endpoint">
  ## Point de terminaison
</div>

`POST /login/unlink`

<Note>
  Ce point de terminaison est **obsolète**. Utilisez plutôt [DELETE /api/v2/users/{id}/identities/{provider}/{user_id}](/api/management/v2/users/delete-user-identity-by-user-id).
</Note>

À l’aide du `access_token` et du `user_id` d’un utilisateur connecté, ce point de terminaison dissocie le compte de l’utilisateur du fournisseur d’identité.

<div id="learn-more">
  ### En savoir plus
</div>

* [Dissocier des comptes d’utilisateur](https://auth0.com/docs/manage-users/user-accounts/user-account-linking/unlink-user-accounts)

<div id="response-messages">
  ## Messages de réponse
</div>

| Statut | Description                               |
| ------ | ----------------------------------------- |
| 200    | Compte d’utilisateur dissocié avec succès |
