<div id="endpoint">
  ## Point de terminaison
</div>

`GET /mfa/authenticators`

Renvoie une liste des authentificateurs associés à votre application.

Pour accéder à ce point de terminaison, vous devez inclure un jeton d’accès dans l’en-tête Authorization, avec les claims suivantes :

* `scope`: `read:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="headers">
  ## En-têtes
</div>

<ParamField header="Authorization" type="string" required>
  Bearer ACCESS&#95;TOKEN
</ParamField>

<div id="response-schema">
  ## Schéma de réponse
</div>

<ResponseSchema>
  La réponse est un tableau d’objets authentificateurs.

  <ResponseField name="id" type="string" required>
    L’ID de l’authentificateur.
  </ResponseField>

  <ResponseField name="authenticator_type" type="string" required>
    Le type d’authentificateur.
  </ResponseField>

  <ResponseField name="active" type="boolean" required>
    Indique si l’authentificateur est actif.
  </ResponseField>

  <ResponseField name="oob_channels" type="string">
    Les canaux OOB pris en charge par cet authentificateur.
  </ResponseField>

  <ResponseField name="name" type="string">
    Le nom de l’authentificateur.
  </ResponseField>
</ResponseSchema>

<div id="response-messages">
  ## Messages de réponse
</div>

| Statut | Description                    |
| ------ | ------------------------------ |
| 200    | Une liste d’authentificateurs. |
