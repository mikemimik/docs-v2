---
title: "Supprimer un authentificateur"
description: "Supprimer un authentificateur MFA associé à l’aide de son ID."
---

<div id="endpoint">
  ## Point de terminaison
</div>

`DELETE /mfa/authenticators/{AUTHENTICATOR_ID}`

Supprime un authentificateur associé à l’aide de son ID. Vous pouvez obtenir les ID des authentificateurs en listant les authentificateurs.

Pour accéder à ce point de terminaison, vous devez définir un jeton d’accès dans l’en-tête Authorization, avec les claims suivantes :

* `scope`: `remove:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="path-parameters">
  ## Paramètres de chemin
</div>

<ParamField path="AUTHENTICATOR_ID" type="string" required>
  L’ID de l’authentificateur à supprimer.
</ParamField>

<div id="headers">
  ## En-têtes
</div>

<ParamField header="Authorization" type="string" required>
  Le jeton d’accès obtenu lors de la connexion.
</ParamField>

<div id="response-messages">
  ## Messages de réponse
</div>

| Statut | Description                              |
| ------ | ---------------------------------------- |
| 204    | Authentificateur supprimé avec succès.   |
| 401    | Non autorisé - jeton d’accès non valide. |
| 404    | Authentificateur introuvable.            |
