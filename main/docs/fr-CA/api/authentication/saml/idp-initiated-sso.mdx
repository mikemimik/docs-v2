---
title: "Flux d’authentification unique (SSO) initié par l’IdP"
description: "Acceptez une SAMLResponse dans un flux initié par l’IdP afin d’authentifier les utilisateurs au moyen de l’authentification unique SAML."
---

<div id="endpoint">
  ## Point de terminaison
</div>

`POST /login/callback`

Ce point de terminaison accepte un SAMLResponse IdP-Initiated Sign On provenant d’un fournisseur d’identité SAML. La connexion correspondant au fournisseur d’identité est spécifiée dans la chaîne de requête. L’utilisateur sera redirigé vers l’application spécifiée dans la section IdP-Initiated Sign On du fournisseur SAML.

<div id="learn-more">
  ### En savoir plus
</div>

* [SAML](https://auth0.com/docs/authenticate/protocols/saml/saml-configuration)

<div id="body-parameters">
  ## Paramètres du corps
</div>

<ParamField body="connection" type="string" required>
  Le nom d’une connexion de fournisseur d’identité configurée pour votre application.
</ParamField>

<ParamField body="SAMLResponse" type="string" required>
  Une SAML Response d’IdP-Initiated Sign On.
</ParamField>
