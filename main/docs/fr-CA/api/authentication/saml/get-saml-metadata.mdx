---
title: "Obtenir les métadonnées"
description: "Renvoie les métadonnées SAML 2.0 d’un tenant Auth0."
---

<div id="endpoint">
  ## Point de terminaison
</div>

`GET /samlp/metadata/{clientId}`

Ce point de terminaison retourne les métadonnées SAML 2.0.

<div id="learn-more">
  ### En savoir plus
</div>

* [SAML](https://auth0.com/docs/authenticate/protocols/saml/saml-configuration)

<div id="path-parameters">
  ## Paramètres de chemin
</div>

<ParamField path="clientId" type="string" required>
  Le client&#95;id de votre application.
</ParamField>

<div id="response-messages">
  ## Messages de réponse
</div>

| Statut | Description                                |
| ------ | ------------------------------------------ |
| 200    | Récupération réussie des métadonnées SAML. |
