Avec les connexions <Tooltip tip="Sans mot de passe : forme d’authentification qui ne repose pas sur un mot de passe comme premier facteur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Passwordless">sans mot de passe</Tooltip>, les utilisateurs peuvent se connecter sans mot de passe. Ils peuvent plutôt utiliser différentes autres méthodes d’authentification. Les connexions sans mot de passe d’Auth0 prennent en charge les méthodes d’authentification suivantes :

| Facteur                                                                               | Description                                                                                                                                                                                                                                                                        |
| ------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Email](/docs/fr-CA/authenticate/passwordless/authentication-methods/email-otp)             | L’utilisateur est invité à saisir son adresse courriel, à laquelle Auth0 envoie un code à usage unique. L’utilisateur saisit ensuite le code dans votre application.                                                                                                               |
| [Magic Link](/docs/fr-CA/authenticate/passwordless/authentication-methods/email-magic-link) | L’utilisateur est invité à saisir son adresse courriel, à laquelle Auth0 envoie un courriel contenant un lien. L’utilisateur clique sur le lien et se connecte directement à votre application.                                                                                    |
| [SMS](/docs/fr-CA/authenticate/passwordless/authentication-methods/sms-otp)                 | L’utilisateur est invité à saisir son numéro de téléphone, auquel Auth0 envoie un code à usage unique. Par défaut, Auth0 utilise Twilio pour envoyer le code, mais si vous avez une passerelle SMS personnalisée, vous pouvez modifier votre connexion pour l’utiliser à la place. |
