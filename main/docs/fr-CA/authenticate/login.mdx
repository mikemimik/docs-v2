Auth0 offre deux façons de mettre en œuvre l’authentification des connexions pour vos applications :

* **<Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le serveur d’autorisation d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Universal+Login">Universal Login</Tooltip>** où les utilisateurs se connectent à votre application par l’intermédiaire d’une page hébergée par Auth0.

* **Embedded Login** où les utilisateurs se connectent à votre application par l’intermédiaire d’une page que vous hébergez.

Dans la grande majorité des cas d’utilisation, nous recommandons Universal Login, mais vous pouvez aussi utiliser Client-Initiated Backchannel Authentication pour authentifier les utilisateurs au moyen d’un flux d’authentification découplé.
