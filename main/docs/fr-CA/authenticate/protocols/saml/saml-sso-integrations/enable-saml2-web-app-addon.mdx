---
description: Découvrez comment activer l’add-on SAML2 Web App pour l’utiliser avec Auth0 comme fournisseur d’identité pour l’authentification unique SAML (SSO).
title: Activer l’add-on SAML2 Web App
---

Vous pouvez configurer un add-on SAML2 Web Application pour l’utiliser avec une application.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications).
2. Repérez l’application avec laquelle vous souhaitez utiliser SAML2 Web Application, puis cliquez sur le nom de l’application pour accéder à ses paramètres de configuration.
3. Accédez à l’onglet **Addons**.
4. Dans la section **SAML2 Web App**, cliquez sur la bascule pour activer l’add-on.

   <Frame>![Dashboard Applications Applications onglet Addons onglet Settings SAML2 Web App](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>
5. Accédez à l’onglet **Settings** et indiquez votre **Application Callback URL**, qui reçoit la réponse SAML, ainsi que les **Settings**. Dans l’onglet **Settings**, vous trouverez un exemple d’objet que vous pouvez utiliser pour déterminer les paramètres à activer.
6. Cliquez sur **Debug** pour vérifier si les paramètres fournis sont valides.
7. Cliquez sur **Save** lorsque vous avez terminé.

<div id="provide-application-details">
  ## Fournissez les détails de l’application
</div>

Bien que les étapes précises pour transmettre les détails de votre application à votre SAML2 Web App puissent varier, vous pouvez obtenir les renseignements de configuration nécessaires dans l’onglet **Usage** après avoir configuré l’addon.

<Frame>![Dashboard Applications Applications onglet Addons SAML2 Web App onglet Usage](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>

  ## Limitation
</div>

Le SAML2 Web App addon ne prend pas en charge les requêtes <Tooltip tip="Security Assertion Markup Language (SAML) : protocole normalisé permettant à deux parties d’échanger des informations d’authentification sans mot de passe." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=SAML">SAML</Tooltip> passives (p. ex. lorsque `isPassive=true`).
