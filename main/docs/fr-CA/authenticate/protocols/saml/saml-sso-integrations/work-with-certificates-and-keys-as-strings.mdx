---
description: Décrit comment utiliser des certificats ou des clés dans des Rules ou des requêtes à la Management API lorsqu’une représentation du fichier sous forme de chaîne de caractères est requise.
title: Utiliser des certificats et des clés sous forme de chaînes de caractères
---

Vous pouvez utiliser la commande suivante dans un shell UNIX pour concaténer un certificat au format PEM avec `\n` (saut de ligne échappé) :

```bash wrap lines
awk 'NF {sub(/\r/, ""); printf "%s\\n",$0;}' certificate.pem | pbcopy
```

Cela copie la valeur dans votre presse-papiers afin que vous puissiez la coller au besoin.

<div id="more-information">
  ### Plus d’information
</div>

Lorsque vous travaillez avec des certificats ou des clés dans Rules ou dans des requêtes v2 de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>, vous aurez très probablement besoin d’une représentation du fichier sous forme de chaîne de caractères.

Si vous ouvrez un fichier de certificat (`cer`, `pem`) dans un éditeur de texte, vous verrez quelque chose comme ceci :

```text lines
-----BEGIN CERTIFICATE-----
MIICzDCCAbQCCQDH8GvxPIeH+DANBgkqhkiG9w0BAQsFADAoMQswCQYDVQQGEwJh
cjEZMBcGA1UEAwwQaHR0cHM6Ly9uaWNvLmNvbTAeFw0xOTA0MDgxODA3NTVaFw0y
//
// autres lignes d'informations encodées en base64
//
nSWyabd+LiBGtLTMB+ZLbOIi3EioWPGw/nHOI8jzPrqhiCLuZCSQmiqrLQYNsc1W
-----END CERTIFICATE-----
```

Les lignes entre l’en-tête `-----BEGIN CERTIFICATE-----` et le pied de page `-----END CERTIFICATE-----` contiennent des données binaires encodées en base64. Les clés publiques et les clés privées (fichiers .key) auront un aspect semblable; seuls l’en-tête et le pied de page seront différents.

Pour obtenir la représentation sous forme de chaîne d’un fichier de certificat ou de clé, vous devrez tout concaténer sur une seule ligne, en remplaçant les sauts de ligne du fichier par la séquence `\n` (saut de ligne échappé). Ainsi, à partir de l’exemple ci-dessus, vous obtiendriez quelque chose comme ceci :

```text lines
"-----BEGIN CERTIFICATE-----
\nMIICzDCCAbQCCQDH8GvxPIeH+DANBgkqhkiG9w0BAQsFADAoMQswCQYDVQQGEwJh\n
[..all the other lines..]
-----END CERTIFICATE-----\n"
```
