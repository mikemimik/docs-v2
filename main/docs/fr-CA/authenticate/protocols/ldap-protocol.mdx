title: Lightweight Directory Access Protocol
---

Le Lightweight Directory Access Protocol (LDAP) est un protocole d’application utilisé pour accéder à des services d’annuaire distribués et les maintenir sur un réseau IP (Internet Protocol). LDAP sert à permettre l’accès à un annuaire existant, comme Active Directory (AD). Auth0 utilise un [Active Directory/LDAP Connector](/docs/fr-CA/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) que vous installez sur votre réseau pour l’intégrer à AD/LDAP.
