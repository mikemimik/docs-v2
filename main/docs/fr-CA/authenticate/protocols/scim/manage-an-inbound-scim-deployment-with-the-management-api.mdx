Pour en savoir plus sur les opérations de gestion SCIM prises en charge et sur la façon de les utiliser, consultez le [Auth0 Management API Explorer](https://auth0.com/docs/api/management/v2) et développez l’onglet **Connections**. La <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=management+API">Management API</Tooltip> permet d’activer, de mettre à jour et de supprimer la configuration SCIM pour une connexion sélectionnée, ainsi que de créer et de supprimer des jetons SCIM.

<div id="register-a-client-app-for-management-api-communication">
  ### Enregistrer une application cliente pour communiquer avec la Management API
</div>

Pour utiliser l’Auth0 Management API, enregistrez une application cliente de type machine-to-machine qui dispose des permissions nécessaires pour gérer votre configuration SCIM.

1. Dans l’[Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez **Applications**.
2. Sélectionnez **Create Application**.
3. Entrez un nom, sélectionnez **Machine to Machine App**, puis sélectionnez **Create**.
4. Dans votre nouvelle application, sélectionnez l’onglet **APIs**, puis définissez **Auth0 Management API** sur **Authorized**.
5. Sous **Permissions**, sélectionnez `read:scim_config`, `create:scim_config`, `update:scim_config`, `delete:scim_config`, `create:scim_token`, `read:scim_token` et `delete:scim_token.`
6. Sélectionnez **Update**.
7. Suivez les instructions dans [Management API Access Tokens](/docs/fr-CA/secure/tokens/access-tokens/management-api-access-tokens) pour apprendre à obtenir des jetons et à appeler les points de terminaison de l’[Auth0 Management API v2](https://auth0.com/docs/api/management/v2/connections/post-scim-configuration).
