title: Configurer Amazon Web Services comme fournisseur de services SAML
---

Configurez l&#39;addon SAML2 Web App pour Amazon Web Services (AWS) pour une application.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis créez une nouvelle application ou cliquez sur le nom d&#39;une application à mettre à jour.

2. Accédez à l&#39;onglet **Addons** et activez la bascule **SAML2 Web App**.

3. Dans l&#39;onglet **Settings**, définissez **Application Callback URL** sur : `https://signin.aws.amazon.com/saml`.

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Settings Tab](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>

4. Collez le code suivant dans la zone de texte **Settings**, puis cliquez sur **Debug**. Assurez-vous de remplacer les espaces réservés `AWS_SSO_ISSUER_URL` et `AWS_SSO_ACS_URL` par les valeurs que vous avez copiées dans AWS.

   ```json lines
   {
     "audience": "AWS_SSO_ISSUER_URL",
     "destination": "AWS_SSO_ACS_URL",
     "mappings": {
       "email": "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/emailaddress",
       "name": "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/name"
     },
     "createUpnClaim": false,
     "passthroughClaimsWithNoMapping": false,
     "mapUnknownClaimsAsIs": false,
     "mapIdentities": false,
     "nameIdentifierFormat": "urn:oasis:names:tc:SAML:1.1:nameid-format:emailAddress",
     "nameIdentifierProbes": [
       "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/emailaddress"
     ]
   }
   ```

5. Faites défiler jusqu&#39;au bas de la page, puis cliquez sur **Enable**.

6. Dans l&#39;onglet **Usage**, repérez **Identity Provider Metadata**, puis cliquez sur **Download** pour télécharger le fichier de métadonnées. Vous en aurez besoin lorsque vous configurerez Auth0 comme fournisseur d&#39;identité.

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Usage Tab](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>
