---
title: Fournisseurs d’identité
description: Une introduction aux sources d’utilisateurs pour les applications, y compris les fournisseurs d’identité, les bases de données et les méthodes d’authentification sans mot de passe.
---

Une connexion correspond à la relation entre Auth0 et une source d’utilisateurs, qui peut inclure des fournisseurs d’identité externes (comme Google ou LinkedIn), des bases de données ou des méthodes d’authentification sans mot de passe. Auth0 se situe entre votre application et ses sources d’utilisateurs, ce qui ajoute un niveau d’abstraction et isole ainsi votre application de tout changement et des particularités propres à la mise en œuvre de chaque source.

Par défaut, Auth0 synchronise automatiquement les données du profil utilisateur à chaque connexion de l’utilisateur, ce qui garantit que les modifications apportées dans la source de connexion sont automatiquement répercutées dans Auth0. Vous pouvez désactiver cette synchronisation afin de pouvoir mettre à jour les attributs du profil à partir de votre application.

Vous pouvez configurer autant de connexions que vous le souhaitez pour vos applications.
