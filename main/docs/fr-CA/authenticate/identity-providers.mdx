---
title: Fournisseurs d’identité
description: Une introduction aux sources d’utilisateurs pour les applications, y compris les fournisseurs d’identité, les bases de données et les méthodes d’authentification sans mot de passe.
---

Une connexion correspond à la relation entre Auth0 et une source d’utilisateurs, qui peut être un fournisseur d’identité externe (comme Google ou LinkedIn), une base de données ou une méthode d’authentification sans mot de passe. Auth0 se place entre votre application et ses sources d’utilisateurs, ce qui ajoute un niveau d’abstraction et isole ainsi votre application des changements et des particularités de la mise en œuvre propres à chaque source.

Par défaut, Auth0 synchronise automatiquement les données de profil utilisateur à chaque connexion, ce qui garantit que les modifications apportées dans la source de connexion sont automatiquement répercutées dans Auth0. Vous pouvez désactiver la synchronisation afin de permettre la mise à jour des attributs du profil à partir de votre application.

Vous pouvez configurer autant de connexions que vous le souhaitez pour vos applications.
