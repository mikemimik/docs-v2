<Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur l’Authorization Server d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Universal+Login">
  Universal Login
</Tooltip>

est la mise en œuvre par Auth0 du flux d’authentification, qui constitue la fonctionnalité clé d’un

<Tooltip tip="Authorization Server : serveur centralisé qui contribue à définir les limites de l’accès d’un utilisateur. Par exemple, votre serveur d’autorisation peut contrôler les données, les tâches et les fonctionnalités accessibles à un utilisateur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Authorization+Server">
  Authorization Server
</Tooltip>

. Chaque fois qu’un utilisateur doit prouver son identité, vos applications redirigent vers Universal Login et Auth0 garantit l’identité de l’utilisateur. Vous pouvez choisir d’activer l’authentification

<Tooltip tip="Passwordless : forme d’authentification qui ne repose pas sur un mot de passe comme premier facteur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Passwordless">
  Passwordless
</Tooltip>

lorsque vous configurez Universal Login dans

<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">
  Auth0 Dashboard
</Tooltip>

.

Déterminez d’abord si vous voulez que les utilisateurs utilisent le SMS, le courriel ou WebAuthn pour la biométrie de l’appareil comme méthode d’authentification Passwordless. Il n’est actuellement pas possible de prendre en charge ces trois méthodes d’authentification dans un seul flux de connexion.

<div id="passwordless-with-email-or-sms">
  ## Authentification Passwordless par courriel ou SMS
</div>

Vous pouvez configurer Auth0 pour permettre aux utilisateurs de s’authentifier à l’aide d’un lien magique ou d’un mot de passe à usage unique (OTP) envoyé par courriel, ou d’un OTP envoyé à leur numéro de téléphone. Pour en savoir plus, consultez [Configurer le courriel ou les SMS pour l’authentification Passwordless](/docs/fr-CA/authenticate/login/auth0-universal-login/passwordless-login/email-or-sms).

<div id="passwordless-with-webauthn-with-device-biometrics">
  ## Passwordless avec WebAuthn with Device Biometrics
</div>

Lorsqu’ils utilisent WebAuthn with Device Biometrics, les utilisateurs peuvent s’authentifier avec leur mot de passe ou au moyen de n’importe lequel de leurs appareils compatibles avec WebAuthn with Device Biometrics.

Chaque fois qu’ils se connectent depuis un nouvel appareil, Auth0 leur proposera d’utiliser la biométrie au lieu d’un mot de passe. Pour en savoir plus, consultez [Passwordless avec WebAuthn with Device Biometrics](/docs/fr-CA/authenticate/login/auth0-universal-login/passwordless-login/webauthn-device-biometrics).
