title: Connecter votre application à Active Directory avec LDAP
---

Auth0 s’intègre à Active Directory (AD) à l’aide du Lightweight Directory Access Protocol (LDAP) par l’intermédiaire d’un **connecteur AD/LDAP** que vous installez sur votre réseau.

Le **connecteur AD/LDAP** (1) sert de passerelle entre votre **Active Directory/LDAP** (2) et le **service Auth0** (3). Cette passerelle est nécessaire, car AD/LDAP est généralement limité à votre réseau interne, tandis qu’Auth0 est un service infonuagique qui s’exécute dans un contexte entièrement différent.

<Frame>![Schéma d’aperçu du connecteur AD/LDAP](/docs/images/cdy7uua7fh8z/2HT4cRvUDzA2OdEPlfgLDV/be32ffe82562dc8f07fdd2097f14d881/ldap-connect.png)</Frame>

Pour la [haute disponibilité et la répartition de charge](/docs/fr-CA/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector/ad-ldap-high-availability), vous pouvez installer plusieurs instances du connecteur. Toutes les connexions sont sortantes, du connecteur vers le serveur Auth0; il n’est donc généralement pas nécessaire de modifier votre pare-feu.

<div id="prerequisites">
  ## Prérequis
</div>

Avant de commencer :

* [Enregistrez votre Application dans Auth0](/docs/fr-CA/get-started/auth0-overview/create-applications).

  * Sélectionnez le **Application Type** approprié.
  * Ajoutez `{https://yourApp/callback}` comme **Allowed Callback URL**.
  * Assurez-vous que les [**Grant Types**](/docs/fr-CA/get-started/applications/update-grant-types) de votre Application incluent les flux appropriés.

<div id="steps">
  ## Étapes
</div>

Pour connecter votre application à Active Directory/LDAP, vous devez :

1. [Créer une connexion d’entreprise dans Auth0](#create-an-enterprise-connection-in-auth0) et télécharger le programme d’installation
2. [Installer le connecteur sur votre réseau](#install-the-connector-on-your-network)
3. [Activer la connexion d’entreprise pour votre Application Auth0](#enable-the-enterprise-connection-for-your-auth0-application)
4. [Tester la connexion](#test-the-connection)

<div id="create-an-enterprise-connection-in-auth0">
  ## Créer une connexion d’entreprise dans Auth0
</div>

1. Accédez à [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), repérez **Active Directory / LDAP** et sélectionnez son `+`.

   <Frame>![Dashboard - Connections - Enterprise](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. Entrez les détails de votre connexion, puis sélectionnez **Create** :

| Champ                                                     | Description                                                                                                                                                                                                                                                                                                        |
| --------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Connection name**                                       | Identifiant logique de votre connexion; il doit être unique dans votre tenant. Une fois défini, ce nom ne peut plus être modifié.                                                                                                                                                                                  |
| **Nom d’affichage** (facultatif)                          | Texte utilisé pour personnaliser le bouton de connexion dans Universal Login. Lorsqu’il est défini, le bouton de connexion de Universal Login affiche : &quot;Poursuivre avec &#123;Nom d’affichage&#125;&quot;.                                                                                                   |
| **URL du logo** (facultatif)                              | URL de l’image utilisée pour personnaliser le bouton de connexion dans Universal Login. Lorsqu’elle est définie, le bouton de connexion de Universal Login affiche l’image sous la forme d’un carré de 20 px sur 20 px.                                                                                            |
| **Domaines IdP** (facultatif)                             | Liste de domaines de courriel valides, séparés par des virgules, qui seront autorisés à se connecter au moyen de cette connexion. Requis uniquement si vous utilisez le widget de connexion Lock.                                                                                                                  |
| **Désactiver le cache**                                   | Lorsqu’elle est activée, désactive la mise en cache.                                                                                                                                                                                                                                                               |
| **Utiliser l’authentification par certificat SSL client** | Lorsqu’elle est activée, utilise l’authentification par certificat SSL client.                                                                                                                                                                                                                                     |
| **Utiliser Windows Integrated Auth (Kerberos)**           | Lorsqu’elle est activée, vous devrez entrer une plage d’adresses IP. Lorsque les utilisateurs se connectent à partir de ces adresses IP, Kerberos est utilisé; sinon, un nom d’utilisateur et un mot de passe AD/LDAP sont demandés. En général, la plage d’adresses IP saisie correspond à des adresses intranet. |
| **Sync user profile attributes at each login**            | Lorsqu’elle est activée, Auth0 synchronise automatiquement les données du profil utilisateur à chaque connexion, ce qui garantit que les modifications apportées dans la source de connexion sont automatiquement mises à jour dans Auth0.                                                                         |

<Frame>![Saisir les détails de la connexion AD / LDAP](/docs/images/cdy7uua7fh8z/2IR80sIwNNyPlDMxIjWv8e/b6748243f90ce0e7d771a39cef24cb0f/LDAP_Connection__1_.png)</Frame>
3. Téléchargez le programme d’installation fourni et notez l’**URL du ticket de Provisioning** fournie.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Nous proposons différentes versions du connecteur à installer sur des plateformes Windows ou Linux.
</Callout>
