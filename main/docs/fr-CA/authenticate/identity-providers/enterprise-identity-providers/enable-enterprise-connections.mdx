description: Décrit comment activer les connexions d’entreprise pour les applications dans l’Auth0 Dashboard.
title: Activer les connexions d’entreprise
---

Vous pouvez activer les connexions d’entreprise pour les applications dans le <Tooltip tip="Auth0 Dashboard : l’interface principale d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Pour activer une connexion d’entreprise, vous devez d’abord l’avoir configurée.

1. Accédez à [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), puis sélectionnez le type de connexion à afficher.

   <Frame>![Dashboard - Connections - Enterprise](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. Sélectionnez le nom de la connexion à afficher.

   <Frame>![Dashboard - Connections - Enterprise - AD/LDAP - List](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/LDAP_page_-_EN.png)</Frame>
3. Sélectionnez la vue **Applications**, puis activez ou désactivez la connexion pour l’application ou les applications appropriées.

   <Frame>![Dashboard - Connections - Enterprise - Applications](/docs/images/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/38fbae514304297bbbf72beeedb8892f/SSL_-_EN.png)</Frame>

Lorsque vous configurez votre connexion d’entreprise <Tooltip tip="Identity Provider (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=IdP">IdP</Tooltip>, n’oubliez pas que le paramètre `connection` transmis au endpoint `/authorize,` `/whr,` ou `/wsfed` d’Auth0 redirigera vers l’IdP en amont, **à moins** qu’il soit connecté à une base de données ou à Active Directory.
