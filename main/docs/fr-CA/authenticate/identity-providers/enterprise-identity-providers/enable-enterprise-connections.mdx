description: Décrit comment activer les connexions d'entreprise pour les applications dans le Auth0 Dashboard.
title: Activer les connexions d'entreprise
---

Vous pouvez activer les connexions d&#39;entreprise pour les applications dans le <Tooltip tip="Auth0 Dashboard : principal produit d'Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Pour activer votre connexion d&#39;entreprise, vous devez d&#39;abord l&#39;avoir configurée.

1. Accédez à [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), puis sélectionnez le type de connexion à afficher.

   <Frame>![Dashboard - Connexions - Enterprise](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. Sélectionnez le nom de la connexion à afficher.

   <Frame>![Dashboard - Connexions - Enterprise - AD/LDAP - Liste](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/LDAP_page_-_EN.png)</Frame>
3. Sélectionnez la vue **Applications**, puis activez ou désactivez la connexion pour la ou les applications appropriées.

   <Frame>![Dashboard - Connexions - Enterprise - Applications](/docs/images/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/38fbae514304297bbbf72beeedb8892f/SSL_-_EN.png)</Frame>

Lorsque vous configurez votre connexion d&#39;entreprise à un <Tooltip tip="Identity Provider (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=IdP">IdP</Tooltip>, gardez à l&#39;esprit que le paramètre `connection` transmis au point de terminaison `/authorize,` `/whr,` ou `/wsfed` d&#39;Auth0 redirigera vers l&#39;IdP en amont, **sauf** si la connexion utilise une base de données ou Active Directory.
