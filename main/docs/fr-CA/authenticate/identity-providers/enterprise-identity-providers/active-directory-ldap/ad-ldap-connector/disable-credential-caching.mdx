title: Désactiver la mise en cache des informations d’identification d’une connexion AD/LDAP
---

Vous pouvez désactiver la mise en cache des informations d’identification au niveau de la connexion pour une [connexion AD/LDAP](/docs/fr-CA/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) à l’aide de l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Accédez à [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), puis sélectionnez le type de connexion **Active Directory/LDAP**.
2. Sélectionnez le nom de la connexion que vous voulez modifier.
3. Activez la bascule **Disable Cache**.
