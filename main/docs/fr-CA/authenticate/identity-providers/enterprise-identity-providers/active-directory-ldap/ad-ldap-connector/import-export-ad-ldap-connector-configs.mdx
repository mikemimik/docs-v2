---
description: Décrit comment importer ou exporter une configuration du AD/LDAP Connector à partir de la Connector Admin Console.
title: Importation et exportation des configurations du AD/LDAP Connector
---

Vous pouvez exporter la configuration du AD/LDAP Connector ou importer une configuration précédemment exportée. C’est utile pour les déploiements comptant plus d’un nœud du AD/LDAP Connector déployé pour assurer une haute disponibilité. Vous pouvez configurer et tester la configuration sur un nœud, puis exporter la configuration fonctionnelle et l’importer sur tous les nœuds suivants.

1. Dans un navigateur, accédez à `http://localhost:8357` pour lancer la **Connector Admin Console** sur le serveur du AD/LDAP Connector.
2. Cliquez sur l’onglet **Import / Export**.
3. Exportez un fichier .zip contenant le fichier `config.json`, le dossier `certs` et le fichier `lib\\profileMapper.js` du dossier `%Program Files(x86)%\Auth0\AD LDAP Connector\`.
4. Pour importer une configuration, téléversez un fichier .zip de sauvegarde dans l’onglet **Import / Export**.
