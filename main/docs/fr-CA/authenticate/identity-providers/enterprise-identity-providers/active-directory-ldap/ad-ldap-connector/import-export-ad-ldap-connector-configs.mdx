---
description: Décrit comment importer ou exporter une configuration d’AD/LDAP Connector à partir de la Connector Admin Console.
title: Importation et exportation des configurations d’AD/LDAP Connector
---

Vous pouvez exporter la configuration d’AD/LDAP Connector ou importer une configuration exportée précédemment. Cela est utile pour les déploiements comprenant plus d’un nœud d’AD/LDAP Connector afin d’assurer la haute disponibilité. Vous pouvez configurer et tester la configuration sur un nœud, puis exporter la configuration fonctionnelle et l’importer sur tous les autres nœuds.

1. Dans un navigateur, accédez à `http://localhost:8357` pour lancer la **Connector Admin Console** sur le serveur AD/LDAP Connector.
2. Cliquez sur l’onglet **Import / Export**.
3. Exportez un fichier .zip contenant le fichier `config.json`, le dossier `certs` et le fichier `lib\\profileMapper.js` du dossier `%Program Files(x86)%\Auth0\AD LDAP Connector\`.
4. Pour importer une configuration, téléversez un fichier .zip de sauvegarde dans l’onglet **Import / Export**.
