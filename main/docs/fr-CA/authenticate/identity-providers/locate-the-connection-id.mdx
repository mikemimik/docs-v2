---
description: Utilisez Auth0 Dashboard pour trouver l’ID de connexion d’une connexion Auth0 afin de configurer des services Auth0, par exemple dans Actions et la gestion des utilisateurs.
title: Trouver l’ID ou le nom de la connexion
---

Certaines tâches de gestion des utilisateurs nécessitent un ID de connexion, c’est-à-dire l’identifiant unique d’une source d’utilisateurs. Par exemple, si vous importez ou exportez des utilisateurs, ou si vous vous connectez à des <Tooltip tip="Identity Provider (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=identity+providers">fournisseurs d’identité</Tooltip>, vous devez fournir un ID de connexion.

1. Accédez à [Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez **Authentication**, puis sélectionnez le type de connexion dont vous avez besoin.
2. Sélectionnez le nom de la connexion dont vous avez besoin.
3. Dans la vue **Settings**, repérez le champ **Name** : il contient l’ID de connexion.
