---
description: Utilisez Auth0 Dashboard pour trouver l’ID d’une connexion Auth0 afin de configurer des services Auth0, par exemple dans Actions et la gestion des utilisateurs.
title: Trouver l’ID ou le nom de la connexion
---

Certaines tâches de gestion des utilisateurs nécessitent un ID de connexion, qui est un identificateur unique d’une source d’utilisateurs. Par exemple, si vous importez ou exportez des utilisateurs, ou si vous vous connectez à des <Tooltip tip="Fournisseur d’identité (IdP) : Service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=identity+providers">fournisseurs d’identité</Tooltip>, vous devez fournir un ID de connexion.

1. Accédez à [Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez **Authentification**, puis le type de connexion dont vous avez besoin.
2. Sélectionnez le nom de la connexion dont vous avez besoin.
3. Dans la vue **Settings**, repérez le champ **Name** ; il contient l’ID de connexion.
