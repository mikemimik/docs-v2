---
description: Décrit comment consulter les connexions activées dans l’Auth0 Dashboard.
title: Voir les connexions
---

Vous pouvez consulter toutes les connexions disponibles et configurées dans le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Les connexions configurées peuvent être utilisées pour se connecter à vos applications.

1. Accédez à [Auth0 Dashboard &gt; Authentication](https://manage.auth0.com/#/connections).
2. Sélectionnez un type de connexion :

* [Database](https://manage.auth0.com/#/connections/database)
* [Social](https://manage.auth0.com/#/connections/social)
* [Enterprise](https://manage.auth0.com/#/connections/enterprise)
* [Passwordless](https://manage.auth0.com/#/connections/passwordless)
