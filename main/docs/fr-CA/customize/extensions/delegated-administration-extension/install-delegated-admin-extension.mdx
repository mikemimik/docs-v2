title: Installer la Delegated Admin Extension
---

Vous pouvez installer la [Delegated Admin Extension](/docs/fr-CA/customize/extensions/delegated-administration-extension), qui vous permet d’exposer la section **Users** du <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> à un groupe choisi d’utilisateurs sans leur donner accès au reste du Dashboard.

<div id="prerequisite">
  ## Prérequis
</div>

[Créez une application Delegated Admin Dashboard](/docs/fr-CA/customize/extensions/delegated-administration-extension/create-delegated-admin-applications) dans Auth0.

<div id="install-extension">
  ## Installer l’extension
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis filtrez sur **Delegated Admin**.
2. Sélectionnez **Delegated Administration Dashboard** dans la liste des extensions offertes. La fenêtre **Install Extension** s’ouvrira.

   <Frame>![Dashboard - Extensions - Delegated Admin - Install](/docs/images/cdy7uua7fh8z/51vm5Sttau4SgnuLbUrKOK/c988e2a52c4b0fc1d091287f14edccda/dashboard-extensions-install_delegated-admin.png)</Frame>
3. Définissez les variables de configuration suivantes, puis sélectionnez **Install** :

   | Variable                        | Description                                                                                                                                                                          |
   | ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   | **EXTENSION&#95;CLIENT&#95;ID** | **Client ID** de l’application avec laquelle vous prévoyez utiliser cette extension.                                                                                                 |
   | **TITLE**                       | Titre personnalisé qui s’affichera en haut de la page du Delegated Administration Dashboard.                                                                                         |
   | **CUSTOM&#95;CSS**              | (*Facultatif*) Lien vers une feuille de style CSS personnalisée que vous pouvez utiliser pour modifier l’apparence de votre page Delegated Administration Dashboard.                 |
   | **FAVICON&#95;PATH**            | (*Facultatif*) Chemin vers un favicon personnalisé.                                                                                                                                  |
   | **AUTH0&#95;CUSTOM&#95;DOMAIN** | (*Facultatif*) Si vous avez configuré un custom domain, entrez-le ici (p. ex., `login.example.com`). Cela remplacera l’authorization endpoint par `https://login.example.com/login`. |
   | **FEDERATED&#95;LOGOUT**        | (*Facultatif*) Indique s’il faut fermer la session de la connexion lorsque les utilisateurs se déconnectent.                                                                        |

   Définir la variable `AUTH0_CUSTOM_DOMAIN` n’a aucune incidence sur l’URL de l’extension; cela modifie seulement l’authorization endpoint. Lorsqu’un custom domain est utilisé, les utilisateurs qui se connectent à l’extension seront redirigés vers `https://AUTH0_CUSTOM_DOMAIN/login` au lieu de l’adresse par défaut `https://tenant-name.us.auth0.com/login`.
   Si vous accédez à [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), vous verrez que l’extension a créé une application supplémentaire. Cette application est autorisée à accéder à la Management API; vous ne devriez donc pas la modifier.

<div id="add-callback-urls-to-application">
  ## Ajouter des URL de rappel à l’application
</div>

1. Accédez à [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), puis sélectionnez le nom de l’application avec laquelle vous prévoyez utiliser cette extension. Il doit s’agir de l’application dont le Client ID correspond à celui que vous avez saisi plus tôt lors de la configuration des variables pour le Delegated Administration Dashboard.
2. Repérez **Allowed Callback URLs**, puis ajoutez l’URL de rappel appropriée pour votre région et le runtime d’extensibilité de votre tenant.

   | Emplacement | Nom  | Allowed Callback URL pour Node.js 12                               |
   | ----------- | ---- | ------------------------------------------------------------------ |
   | États-Unis  | US-1 | `https://{yourTenant}.us12.webtask.io/auth0-delegated-admin/login` |
   | États-Unis  | US-3 | `https://{yourTenant}.us.webtask.run/auth0-delegated-admin/login`  |
   | Australie   | AU   | `https://{yourTenant}.au12.webtask.io/auth0-delegated-admin/login` |
   | Europe      | EU   | `https://{yourTenant}.eu12.webtask.io/auth0-delegated-admin/login` |
   | Japon       | JP-1 | `https://{yourTenant}.jp.webtask.run/auth0-delegated-admin/login`  |

   | Emplacement | Nom  | Allowed Callback URL pour Node.js 8                              |
   | ----------- | ---- | ---------------------------------------------------------------- |
   | États-Unis  | US-1 | `https://YOUR_TENANT.us8.webtask.io/auth0-delegated-admin/login` |
   | Europe      | EU   | `https://YOUR_TENANT.eu8.webtask.io/auth0-delegated-admin/login` |
   | Australie   | AU   | `https://YOUR_TENANT.au8.webtask.io/auth0-delegated-admin/login` |
3. Sélectionnez **Save Changes**.

<div id="use-extension">
  ## Utiliser l’extension
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez la vue **Installed Extensions**.
2. Sélectionnez **Delegated Administration Dashboard**. Un nouvel onglet s’ouvrira pour afficher l’écran de connexion.

   Comme nous avons désactivé les inscriptions pour la connexion de base de données lors de sa configuration, l’écran de connexion n’affichera pas l’option Sign Up.

   Une fois que vous aurez fourni des identifiants valides, vous serez redirigé vers votre page **Delegated Administration Dashboard** personnalisée, qui affichera le **Title** que vous avez fourni en haut de la page. Si vous avez fourni un fichier CSS personnalisé, cette mise en forme sera appliquée.
