---
description: Découvrez comment utiliser le Memberships Query Hook avec la Delegated Administration Extension.
title: 'Delegated Administration : Memberships Query Hook'
---

Lors de la création d’un nouvel utilisateur, l’interface utilisateur affiche une liste déroulante dans laquelle vous pouvez choisir la ou les appartenances à attribuer à l’utilisateur. Ces appartenances sont ensuite définies à l’aide du **Memberships Query Hook**.

<div id="hook-contract">
  ## Contrat du hook
</div>

* **ctx**: Objet de contexte.
* **callback(error, &#123; createMemberships: true/false, memberships: &#125;)**: Fonction de rappel à laquelle vous pouvez renvoyer une erreur et un objet contenant la configuration des appartenances.

<div id="sample-use">
  ## Exemple d’utilisation
</div>

Les utilisateurs du département des TI devraient pouvoir créer des utilisateurs dans d’autres départements. Les utilisateurs des autres départements ne devraient pouvoir créer des utilisateurs que dans leur propre département.

```javascript lines
function(ctx, callback) {
  var currentDepartment = ctx.payload.user.app_metadata && ctx.payload.user.app_metadata.department;
  if (!currentDepartment || !currentDepartment.length) {
    return callback(null, [ ]);
  }

  if (currentDepartment === 'IT') {
    return callback(null, [ 'IT', 'HR', 'Finance', 'Marketing' ]);
  }

  return callback(null, [ ctx.payload.user.app_metadata.department ]);
}
```

<div id="notes">
  ## Remarques
</div>

Comme cette requête ne peut être utilisée que dans l’UI, vous devrez attribuer les appartenances à l’aide du **Write Hook** si vous devez appliquer des règles concernant l’affectation des utilisateurs à des départements précis.

S’il n’existe qu’un seul groupe d’appartenance possible, le champ Memberships ne s’affichera pas dans l’UI.

Vous pouvez permettre à l’utilisateur final de saisir n’importe quelle valeur dans le champ **memberships** en définissant **createMemberships** à true :

```javascript lines
function(ctx, callback) {
  var currentDepartment = ctx.payload.user.app_metadata.department;
  if (!currentDepartment || !currentDepartment.length) {
    return callback(null, [ ]);
  }

  return callback(null, {
    createMemberships: ctx.payload.user.app_metadata.department === 'IT' ? true : false,
    memberships: [ ctx.payload.user.app_metadata.department ]
  });
}
```
