---

Auth0 AD/LDAP Connector Health Monitor expose le point de terminaison d’API de votre choix pour vous permettre de surveiller vos connecteurs AD/LDAP.

<div id="configure-the-extension">
  ## Configurer l’extension
</div>

Pour installer et configurer cette extension :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez **Auth0 AD/LDAP Connector Health Monitor**. La fenêtre **Install Extension** s’ouvre.
2. Définissez les paramètres de configuration suivants, puis sélectionnez **Install** :

* **Auth0&#95;Domain** : Le domaine de votre application Auth0, que vous trouverez dans les [Auth0 Application Settings](https://manage.auth0.com/#/applications/\{yourClientId}/settings).
* **Auth0&#95;Global&#95;Client&#95;ID** : Le <Tooltip tip="Client ID : valeur d’identification attribuée par Auth0 à votre ressource enregistrée." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Client+ID">Client ID</Tooltip> global de votre application Auth0, que vous trouverez dans la section **Global Application Information** des [Tenant Advanced Settings](https://manage.auth0.com/#/tenant/advanced).
* **Auth0&#95;Global&#95;Client&#95;Secret** : Le <Tooltip tip="Client Secret : secret utilisé par un client (application) pour s’authentifier auprès de l’Authorization Server; il ne doit être connu que du client et de l’Authorization Server et doit être suffisamment aléatoire pour ne pas pouvoir être deviné." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Client+Secret">Client Secret</Tooltip> global de votre application Auth0, que vous trouverez dans la section **Global Application Information** des [Tenant Advanced Settings](https://manage.auth0.com/#/tenant/advanced).

<div id="use-the-extension">
  ## Utiliser l’extension
</div>

Pour obtenir l’URL à utiliser avec votre service de monitoring :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis cliquez sur **Installed Extensions**.

2. Sélectionnez **Auth0 AD/LDAP Connector Health Monitor**. Une nouvelle fenêtre s’ouvre avec un extrait JSON semblable à celui-ci :

   ```json lines
   {
     "message": "Use this url: 'https://sandbox.it.auth0.com/api/run/auth0user/auth0-ldap-conector-health-monitor?connection={MY-LDAP-CONNECTOR}' for monitoring your AD/LDAP connector."
   }
   ```

3. Saisissez l’ID de votre connexion LDAP. Pour obtenir cet ID, accédez à [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), cliquez sur la connexion, puis copiez la valeur du champ **Identifier**.

Une fois cela fait, vous pouvez utiliser l’URL fournie avec votre service de monitoring pour surveiller vos connecteurs AD/LDAP.
