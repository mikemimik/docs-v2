---

L’extension Auth0 AD/LDAP Connector Health Monitor expose un point de terminaison d’API de votre choix pour vous permettre de surveiller vos connecteurs AD/LDAP.

<div id="configure-the-extension">
  ## Configurer l’extension
</div>

Pour installer et configurer cette extension :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez **Auth0 AD/LDAP Connector Health Monitor**. La fenêtre **Installer l’extension** s’ouvre.
2. Définissez les paramètres de configuration suivants, puis sélectionnez **Installer** :

* **Auth0&#95;Domain** : Le domaine de votre application Auth0; vous le trouverez dans les [paramètres de votre application Auth0](https://manage.auth0.com/#/applications/\{yourClientId}/settings).
* **Auth0&#95;Global&#95;Client&#95;ID** : Le <Tooltip tip="Client ID : Valeur d’identification attribuée à votre ressource enregistrée par Auth0." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Client+ID">Client ID</Tooltip> global de votre application Auth0; vous le trouverez dans la section **Informations globales sur l’application** des [paramètres avancés du tenant](https://manage.auth0.com/#/tenant/advanced).
* **Auth0&#95;Global&#95;Client&#95;Secret** : Le <Tooltip tip="Client Secret : Secret utilisé par un client (application) pour s’authentifier auprès de l’Authorization Server; il ne doit être connu que du client et de l’Authorization Server et doit être suffisamment aléatoire pour ne pas pouvoir être deviné." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Client+Secret">Client Secret</Tooltip> global de votre application Auth0; vous le trouverez dans la section **Informations globales sur l’application** des [paramètres avancés du tenant](https://manage.auth0.com/#/tenant/advanced).

<div id="use-the-extension">
  ## Utiliser l’extension
</div>

Pour accéder à l’URL à utiliser avec votre service de surveillance :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis cliquez sur **Installed Extensions**.

2. Sélectionnez **Auth0 AD/LDAP Connector Health Monitor**. Une nouvelle fenêtre contenant un extrait JSON semblable à celui-ci s’ouvre :

   ```json lines
   {
     "message": "Use this url: 'https://sandbox.it.auth0.com/api/run/auth0user/auth0-ldap-conector-health-monitor?connection={MY-LDAP-CONNECTOR}' for monitoring your AD/LDAP connector."
   }
   ```

3. Saisissez l’ID de votre connexion LDAP. Pour obtenir l’ID, accédez à  [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), cliquez sur la connexion et copiez la valeur du champ **Identifier**.

Une fois l’opération terminée, vous pouvez utiliser l’URL fournie avec votre service de surveillance pour surveiller vos connecteurs AD/LDAP.
