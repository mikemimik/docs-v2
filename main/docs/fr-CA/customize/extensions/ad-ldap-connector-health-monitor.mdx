---

Auth0 AD/LDAP Connector Health Monitor vous permet de définir le point de terminaison d’API de votre choix pour surveiller vos connecteurs AD/LDAP.

<div id="configure-the-extension">
  ## Configurer l’extension
</div>

Pour installer et configurer cette extension :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez **Auth0 AD/LDAP Connector Health Monitor**. La fenêtre **Install Extension** s’ouvre.
2. Définissez les paramètres de configuration suivants, puis sélectionnez **Install** :

* **Auth0&#95;Domain** : Le domaine de votre application Auth0, que vous trouverez dans vos [Auth0 Application Settings](https://manage.auth0.com/#/applications/\{yourClientId}/settings).
* **Auth0&#95;Global&#95;Client&#95;ID** : Le <Tooltip tip="Client ID: Valeur d’identification attribuée à votre ressource enregistrée par Auth0." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Client+ID">Client ID</Tooltip> global de votre application Auth0, que vous trouverez dans la section **Global Application Information** de vos [Tenant Advanced Settings](https://manage.auth0.com/#/tenant/advanced).
* **Auth0&#95;Global&#95;Client&#95;Secret** : Le <Tooltip tip="Client Secret: Secret utilisé par un client (application) pour s’authentifier auprès de l’Authorization Server; il doit être connu uniquement du client et de l’Authorization Server et être suffisamment aléatoire pour ne pas pouvoir être deviné." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Client+Secret">Client Secret</Tooltip> global de votre application Auth0, que vous trouverez dans la section **Global Application Information** de vos [Tenant Advanced Settings](https://manage.auth0.com/#/tenant/advanced).

<div id="use-the-extension">
  ## Utiliser l’extension
</div>

Pour accéder à l’URL à utiliser avec votre service de surveillance :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis cliquez sur **Installed Extensions**.

2. Sélectionnez **Auth0 AD/LDAP Connector Health Monitor**. Une nouvelle fenêtre contenant un extrait JSON semblable à celui-ci s’ouvre :

   ```json lines
   {
     "message": "Use this url: 'https://sandbox.it.auth0.com/api/run/auth0user/auth0-ldap-conector-health-monitor?connection={MY-LDAP-CONNECTOR}' for monitoring your AD/LDAP connector."
   }
   ```

3. Saisissez l’ID de votre connexion LDAP. Pour obtenir l’ID, accédez à [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), cliquez sur la connexion, puis copiez la valeur du champ **Identifier**.

Une fois cette étape terminée, l’URL fournie peut être utilisée avec votre service de surveillance pour surveiller vos connecteurs AD/LDAP.
