Le <Tooltip tip="Authentification unique (SSO) : service qui, après qu’un utilisateur s’est connecté à une application, connecte automatiquement cet utilisateur à d’autres applications." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Single+Sign-on">Dashboard d’authentification unique</Tooltip> (SSO) est une application web conçue pour résoudre un problème bien connu. Les organisations de toutes tailles utilisent diverses applications pour gérer différentes fonctions opérationnelles, comme la comptabilité, les RH, le développement, le soutien, etc. Il peut être fastidieux de devoir retenir les noms d’utilisateur, les mots de passe et les URL de connexion de toutes vos applications. Grâce à cette extension, vous pouvez simplifier l’expérience d’authentification en activant la connexion SSO pour vos utilisateurs sur plusieurs applications d’entreprise. Elle vous permet de créer une liste de tous les services infonuagiques auxquels un utilisateur peut se connecter avec le SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Un utilisateur ne devrait pas pouvoir accéder, à partir du tableau de bord ou autrement, au fournisseur de services sans disposer des permissions appropriées (groupes/rôles). Idéalement, ces utilisateurs ne verraient dans le tableau de bord aucun fournisseur de services auquel ils n’ont pas accès.
</Callout>

Le SSO Dashboard prend en charge deux types d’utilisateurs :

* **Users** qui peuvent se connecter au tableau de bord pour sélectionner une application à laquelle se connecter avec le SSO.
* **Admins** qui peuvent configurer les applications visibles par les utilisateurs.

Pour configurer cette extension, procédez comme suit :

1. [Créer une Application Dashboard d’authentification unique](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Installer l’extension Dashboard d’authentification unique](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Ajouter des Applications au Dashboard d’authentification unique](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Mettre à jour les Applications dans le Dashboard d’authentification unique](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

Vous pouvez aussi [consulter cette extension Dashboard d’authentification unique sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
