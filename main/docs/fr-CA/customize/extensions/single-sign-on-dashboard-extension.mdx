Le <Tooltip tip="authentification unique (SSO) : service qui, après qu’un utilisateur se connecte à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Single+Sign-on">authentification unique</Tooltip> (SSO) Dashboard est une application web conçue pour résoudre un problème bien connu. Les organisations de toutes tailles utilisent une variété d’applications pour répondre à différents besoins d’entreprise, comme la comptabilité, les RH, le développement, le soutien, etc. Il peut être fastidieux de se souvenir des noms d’utilisateur, des mots de passe et des URL de connexion de toutes vos applications. Grâce à cette extension, vous pouvez simplifier l’expérience d’authentification en activant la connexion SSO pour vos utilisateurs sur plusieurs applications d’entreprise. Elle vous permet de créer une liste de tous les services infonuagiques auxquels un utilisateur peut s’authentifier à l’aide du SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Un utilisateur ne devrait pas pouvoir accéder, à partir du tableau de bord ou autrement, au fournisseur de services sans avoir les permissions appropriées (groupes/rôles). Idéalement, ces utilisateurs ne devraient voir dans le tableau de bord aucun fournisseur de services auquel l’accès ne leur a pas été accordé.
</Callout>

Le SSO Dashboard prend en charge deux types d’utilisateurs :

* **Users** qui peuvent se connecter au tableau de bord pour sélectionner une application à laquelle se connecter avec le SSO.
* **Administrateurs** qui peuvent configurer les applications visibles pour les utilisateurs.

Pour configurer cette extension, procédez comme suit :

1. [Créer une application SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Installer l’extension SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Ajouter des Applications au SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Mettre à jour les Applications dans le SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

Vous pouvez aussi [consulter cette extension SSO Dashboard sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
