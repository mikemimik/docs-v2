Le <Tooltip tip="Authentification unique (SSO) : service qui, après qu’un utilisateur se connecte à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Single+Sign-on">Single Sign-on</Tooltip> (SSO) Dashboard est une application Web conçue pour résoudre un problème bien connu. Les organisations de toutes tailles utilisent une variété d’applications pour prendre en charge différentes fonctions d’entreprise, comme la comptabilité, les RH, le développement, le soutien, etc. Il peut être fastidieux de se souvenir des noms d’utilisateur, des mots de passe et des URL de connexion pour toutes vos applications. Grâce à cette extension, vous pouvez simplifier l’expérience d’authentification en activant la connexion SSO pour vos utilisateurs sur plusieurs applications d’entreprise. Elle vous permet de créer une liste de tous les services infonuagiques auxquels un utilisateur peut s’authentifier au moyen de connexions SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Un utilisateur ne devrait pas pouvoir accéder, à partir du SSO Dashboard ou autrement, au fournisseur de services sans disposer des permissions appropriées (groupes/rôles). Idéalement, ces utilisateurs ne devraient voir dans le SSO Dashboard aucun fournisseur de services auquel ils n’ont pas accès.
</Callout>

Le SSO Dashboard prend en charge deux types d’utilisateurs :

* **Utilisateurs** qui peuvent se connecter au SSO Dashboard pour sélectionner une application à laquelle se connecter avec l’authentification SSO.
* **Administrateurs** qui peuvent configurer les applications visibles par les utilisateurs.

Pour configurer cette extension, procédez comme suit :

1. [Créer une application SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Installer l’extension SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Ajouter des applications au SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Mettre à jour les applications dans le SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

Vous pouvez aussi [consulter cette extension SSO Dashboard sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
