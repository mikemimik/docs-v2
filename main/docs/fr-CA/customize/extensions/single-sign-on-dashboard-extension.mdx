Le <Tooltip tip="Authentification unique (SSO) : service qui, après qu’un utilisateur se connecte à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Single+Sign-on">authentification unique</Tooltip> (SSO) Dashboard est une application Web conçue pour résoudre un problème bien connu. Les organisations de toutes tailles utilisent diverses applications pour gérer différentes fonctions d’affaires, comme la comptabilité, les RH, le développement, le soutien, etc. Il peut être fastidieux de retenir les noms d’utilisateur, les mots de passe et les URL de connexion de toutes vos applications. Grâce à cette extension, vous pouvez simplifier l’expérience d’authentification en activant la connexion SSO pour vos utilisateurs dans plusieurs applications d’entreprise. Elle vous permet de créer une liste de tous les services infonuagiques auxquels un utilisateur peut s’authentifier à l’aide du SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Un utilisateur ne devrait pas pouvoir accéder, à partir du tableau de bord ou autrement, au fournisseur de services sans disposer des permissions appropriées (groupes/rôles). Idéalement, ces utilisateurs ne devraient voir dans le tableau de bord aucun fournisseur de services auquel ils n’ont pas accès.
</Callout>

Le SSO Dashboard prend en charge deux types d’utilisateurs :

* **Users** qui peuvent se connecter au tableau de bord pour sélectionner une application à laquelle se connecter avec le SSO.
* **Admins** qui peuvent configurer les applications visibles pour les utilisateurs.

Pour configurer cette extension, procédez comme suit :

1. [Créer une application SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Installer l’extension SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Ajouter des Applications au SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Mettre à jour les Applications dans le SSO Dashboard](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

Vous pouvez aussi [consulter cette authentification unique Dashboard Extension sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
