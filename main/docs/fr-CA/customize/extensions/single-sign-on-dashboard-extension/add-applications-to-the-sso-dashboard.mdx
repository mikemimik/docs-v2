title: Ajouter des applications au Single Sign-On Dashboard
---

Utilisez l’extension Dashboard Single Sign-On (<Tooltip tip="Single Sign-On (SSO) : service qui, après qu’un utilisateur ouvre une session dans une application, l’authentifie automatiquement dans d’autres applications." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=SSO">SSO</Tooltip>) pour gérer la connexion SSO de vos utilisateurs sur plusieurs applications d’entreprise.

<div id="prerequisites">
  ## Prérequis
</div>

* [Créez l’application du tableau de bord SSO](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application).
* [Installez l’extension du tableau de bord SSO](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).

<div id="add-applications">
  ## Ajouter des applications
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez votre nouvelle extension tableau de bord SSO.
   Si vous êtes administrateur, vous pouvez aussi vous connecter au tableau de bord SSO à l’aide de `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
2. Dans le coin supérieur droit, sélectionnez **Settings** dans le menu déroulant sous le nom de votre tenant.
3. Cliquez sur **Create App** pour ajouter une nouvelle application.

   Le formulaire **New Application** s’affiche.
4. Remplissez les champs suivants pour la nouvelle application :

   | Champ           | Description                                                                                                                                                                                                                                                                                                                           |
   | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Type**        | Une liste déroulante dans laquelle vous sélectionnez SAML, OpenID-Connect ou WS-Federation selon le type d’application.                                                                                                                                                                                                               |
   | **Application** | Une liste déroulante dans laquelle vous sélectionnez l’application que vous souhaitez ajouter.                                                                                                                                                                                                                                        |
   | **Name**        | Le nom est rempli automatiquement en fonction de l’application sélectionnée. Vous pouvez le modifier ou utiliser la valeur par défaut.                                                                                                                                                                                                |
   | **Logo**        | L’URL du logo que vous souhaitez utiliser comme icône pour l’application.                                                                                                                                                                                                                                                             |
   | **Callback**    | L’une des **Allowed Callback URLs** de vos [Application Settings](https://manage.auth0.com/#/applications) pour l’application.                                                                                                                                                                                                        |
   | **Connection**  | (*Facultatif*) Le type de connexion. Vous pouvez ajouter ou modifier vos types de connexion disponibles dans la [section Connections du dashboard de gestion Auth0](https://manage.auth0.com/#/connections/database). Si aucune connexion n’est définie et que l’utilisateur n’est pas connecté, il verra la page de connexion Auth0. |
   | **Custom URL**  | (*Facultatif*) Case à cocher pour utiliser une URL personnalisée plutôt que l’URL Auth0. Si vous cochez la case, un champ s’affiche dans lequel vous pouvez saisir votre URL personnalisée.                                                                                                                                           |
   | **Enabled**     | Case à cocher pour rendre cette application visible (publiée) à vos utilisateurs.                                                                                                                                                                                                                                                     |
5. Sélectionnez **Create**. Votre nouvelle application apparaîtra alors sur la page **Applications** du tableau de bord SSO avec toutes les autres applications qui ont été créées.
6. Cliquez sur une application pour tester la connexion.
