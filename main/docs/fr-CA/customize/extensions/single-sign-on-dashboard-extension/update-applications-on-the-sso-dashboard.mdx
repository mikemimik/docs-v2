title: Mettre à jour les applications dans le Tableau de bord SSO
---

Utilisez l’extension Tableau de bord SSO pour gérer l’authentification <Tooltip tip="Authentification unique (SSO) : service qui, après qu’un utilisateur s’est connecté à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=SSO">SSO</Tooltip> de vos utilisateurs dans plusieurs applications d’entreprise.

Pour en savoir plus, consultez [l’extension Tableau de bord d’authentification unique](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension) ou [voyez cette extension sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).

<div id="prerequisites">
  ## Prérequis
</div>

* [Créer l’application du tableau de bord SSO](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application) dans Auth0.
* [Installer l’extension du tableau de bord SSO](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).
* [Ajouter des applications au tableau de bord SSO](/docs/fr-CA/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard).

<div id="update-application">
  ## Mettre à jour l’application
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) et sélectionnez la vue **Installed Extensions**.
2. Sélectionnez **Tableau de bord SSO**.
   Si vous êtes administrateur, vous pouvez aussi ouvrir une session dans le Tableau de bord SSO à l’aide de `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
3. Sélectionnez **Settings** dans le menu déroulant sous le nom de votre tenant.

* Pour définir si les utilisateurs peuvent voir l’application lorsqu’elle est activée, sélectionnez **Publish** ou **Unpublish**.
* Pour mettre à jour les paramètres d’une application, sélectionnez son icône en forme d’engrenage.
* Pour supprimer une application, sélectionnez son **X**. Une fenêtre contextuelle de confirmation confirmera la suppression.
