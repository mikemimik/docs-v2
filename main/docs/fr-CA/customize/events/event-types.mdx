Vous pouvez créer des flux d’événements qui surveillent différents types d’événements, comme la création d’un nouveau profil utilisateur. Les sections ci-dessous présentent des renseignements précis sur les types d’événements actuellement pris en charge.

<div id="event-object">
  ## Event object
</div>

Les Events sont publiés au moyen d’un schéma conforme à la [spécification CloudEvent](https://cloudevents.io/) :

| Paramètre   | Type          | Obligatoire | Description                                                                                  |
| ----------- | ------------- | ----------- | -------------------------------------------------------------------------------------------- |
| id          | string        | Oui         | Identifie l’événement.                                                                       |
| source      | URI-reference | Oui         | Identifie le tenant qui a généré l’événement, y compris son environnement.                   |
| specversion | string        | Oui         | La version de la spécification CloudEvents qu’utilise l’événement.                           |
| type        | string        | Oui         | Le type d’événement.                                                                         |
| time        | timestamp     | Oui         | Horodatage du moment où l’événement s’est produit.                                           |
| dataschema  | URI           | Oui         | Identifie le schéma auquel les données sont conformes.                                       |
| data        | object        | Oui         | Renseignements supplémentaires liés à l’événement. Ce champ varie selon le type d’événement. |

```json lines expandable
{
  "id": "evt_<unique-id>",
  "source": "urn:auth0:<your-tenant>:users",
  "spec_version": "1.0",
  "type": "com.auth0.user.<event-type>",
  "time": "2025-01-29T14:36:25.794Z",
  "dataschema": "https://<your-tenant>.auth0.com/api/v2/schemas/user.<event-type>",
  "data": {
    "object": {
      "user_id": "<user-id>",
      "email": "<email>",
      "email_verified": <true/false>,
      "family_name": "<family-name>",
      "given_name": "<given-name>",
      "name": "<full-name>",
      "nickname": "<nickname>",
      "phone_number": "<phone-number>",
      "phone_verified": <true/false>,
      "user_metadata": {
        "hobby": "<user-hobby>"
      },
      "app_metadata": {
        "external_user_id": "<external-id>"
      },
      "identities": [
        {
          "connection": "<connection-name>",
          "user_id": "<identity-user-id>",
          "provider": "<identity-provider>",
          "isSocial": <true/false>
        }
      ],
      "created_at": "<user-created-time>",
      "updated_at": "<user-updated-time>",
      "picture": "<profile-picture-url>"
    }
  }
}
```

Pour en savoir plus sur les Types d’événements, consultez le [catalogue des événements](https://auth0.com/docs/events/).
