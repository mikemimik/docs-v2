La sélection flexible de la langue est une fonctionnalité optionnelle qui permet aux utilisateurs de sélectionner leur langue de préférence dans les prompts <Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le serveur d’autorisation d’Auth0, afin de vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Universal+Login">Universal Login</Tooltip> associés à votre application, comme l’écran de connexion. Vous pouvez configurer cette fonctionnalité individuellement pour différents prompts Universal Login.

Une fois cette fonctionnalité mise en œuvre pour un prompt donné, un menu de sélection de la langue est ajouté à l’écran associé. Les utilisateurs peuvent y choisir leur langue de préférence pour que le prompt s’affiche automatiquement dans cette langue.

<div id="before-you-begin">
  ## Avant de commencer
</div>

Avant de pouvoir mettre en place la sélection flexible de la langue, assurez-vous de respecter les exigences suivantes :

* Utilisez [Universal Login](/docs/fr-CA/authenticate/login/auth0-universal-login).

  * Cette fonctionnalité n’est pas offerte pour les pages de connexion personnalisées.
* Configurez un [domaine personnalisé](/docs/fr-CA/customize/custom-domains).
* Activez dans votre tenant Auth0 les langues que vous prévoyez ajouter à vos prompts.

  * Dans l’Auth0 Dashboard, accédez à [Settings &gt; General &gt; Languages](https://manage.auth0.com/#/tenant/general) et activez une ou plusieurs langues.

    * Pour en savoir plus, consultez [Universal Login Internationalization](/docs/fr-CA/customize/internationalization-and-localization/universal-login-internationalization).

<div id="implement-flexible-language-selection">
  ## Configurer la sélection flexible de la langue
</div>

La configuration de la sélection flexible de la langue comprend deux étapes principales :

1. Préparer votre gabarit de page pour permettre la sélection de la langue.
2. Configurer chaque prompt pour afficher l’option de sélection de la langue aux utilisateurs.

<div id="prepare-your-universal-login-page-template">
  ### Préparez le gabarit de votre page Universal Login
</div>

Pour commencer, ajoutez le script suivant à votre [gabarit de page Universal Login](/docs/fr-CA/customize/login-pages/universal-login/customize-templates) pour faciliter la sélection de la langue.

```html lines
<script>
  function updateSelectedLanguage() {
    const selectElement = document.getElementById('language');
    const selectedValue = selectElement.value;
    const options = selectElement.options;
    for (let i = 0; i < options.length; i++) {
      if (options[i].value === selectedValue) {
        options[i].selected = true;
      } else {
        options[i].selected = false;
      }
    }
    document.getElementById('changeLanguage').click();
  }
</script>
```

Ce script permet aux prompts Universal Login d’afficher un menu dynamique de sélection de la langue. Les utilisateurs peuvent accéder à ce menu pour appliquer leur langue préférée à un prompt configuré.

Pour ajouter ce script à votre gabarit de page, utilisez l’endpoint [Set Template for Universal Login](https://auth0.com/docs/api/management/v2/branding/put-universal-login).

<Warning>
  Sachez que cet appel **écrasera toute configuration existante** que vous avez apportée à votre gabarit de page Universal Login. Pour éviter toute interruption, assurez-vous d’inclure dans cet appel toutes les personnalisations nécessaires du gabarit de page.
</Warning>

<div id="configure-individual-universal-login-prompts">
  ### Configurer des prompts Universal Login particuliers
</div>

Ensuite, ajoutez la sélection de la langue à un ou plusieurs prompts Universal Login en configurant des partials de prompt personnalisés. Les partials correspondent à du code personnalisé inséré dans un point d’entrée d’un écran de prompt, comme l’écran de connexion. Pour en savoir plus, consultez [Personnaliser les prompts d&#39;inscription et de connexion](/docs/fr-CA/customize/internationalization-and-localization/universal-login-internationalization).

Pour ajouter la sélection de la langue à un prompt précis, utilisez l’Auth0 <Tooltip tip="Management API : un produit permettant aux clients d'effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip> pour configurer des partials de prompt personnalisés avec les paramètres suivants :

| Paramètre  | Description                                                                                                                                                                                                                                                                                                                                                                     | Valeur d’exemple  |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------- |
| `language` | Obligatoire.<br /><br />Une ou plusieurs langues à inclure dans le menu de sélection de la langue affiché aux utilisateurs.                                                                                                                                                                                                                                                     | `es`,`fr`,`en`    |
| `persist`  | Obligatoire.<br /><br />Détermine si la langue doit être conservée pendant la session.                                                                                                                                                                                                                                                                                          | `session`         |
| `action`   | Facultatif.<br /><br />Utilisez ce paramètre si vous voulez permettre aux utilisateurs de changer de langue sans passer au prompt suivant.<br /><br />Si une valeur est fournie, le formulaire n’est **pas** soumis automatiquement lorsqu’un utilisateur change de langue. Sinon, le prompt passe automatiquement à l’étape suivante lorsqu’une autre langue est sélectionnée. | `change-language` |

Consultez les sections ci-dessous pour voir des exemples de code pour différents types de prompts Universal Login.

<Warning>
  Dans les exemples de code ci-dessous, assurez-vous de remplacer les espaces réservés par les valeurs appropriées :

  * Remplacez `{yourDomain}` par `yourdomain.auth0.com`.
  * Remplacez `{mgmtApiToken}` par votre access token.
</Warning>
