---
description: Liste les filtres de logs disponibles pour les Log Streams ainsi que les logs inclus dans chaque filtre.
title: Filtres de Log Stream
---

Vous pouvez filtrer vos Log Streams afin que seuls certains logs et certaines catégories de logs soient transmis. Pour configurer le filtre d’un Log Stream :

1. Accédez à [Dashboard &gt; Monitoring &gt; Streams](https://manage.auth0.com/#/log-streams).
2. Sélectionnez un Log Stream, puis accédez à l’onglet **Settings**.
3. Activez ou désactivez les filtres dans le menu **Filter By Event Category**.
4. Cliquez sur **Save**.
