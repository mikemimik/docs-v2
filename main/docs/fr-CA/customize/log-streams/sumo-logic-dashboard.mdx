title: Utiliser Auth0 App for Sumo Logic
---

Vous pouvez utiliser **Auth0 App for Sumo Logic** pour visualiser les données de votre tenant Auth0 dans un tableau de bord Sumo Logic. Le tableau de bord vous permet de surveiller l’état du trafic de connexion d’un tenant. L’application vous permet d’utiliser les agrégations recommandées par Auth0 ou de vous en servir comme point de départ pour créer vos propres visualisations personnalisées.

<div id="prerequisite">
  ## Préalable
</div>

[Configurer la diffusion des journaux vers Sumo Logic](https://marketplace.auth0.com/integrations/sumo-logic-log-streaming).

<div id="install-the-auth0-app-for-sumo-logic">
  ## Installer l’Auth0 App for Sumo Logic
</div>

1. Connectez-vous à [Sumo Logic](https://www.sumologic.com).
2. Dans le **App Catalog**, recherchez et sélectionnez l’application Auth0.
3. Pour installer l’application, remplissez les champs suivants :

   | Champ                    | Description                                                                                                                                                                                                                                                   |
   | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Nom de l’application** | Conservez le nom existant ou entrez un nom de votre choix.                                                                                                                                                                                                    |
   | **Source de données**    | Choisissez **Source Category** et sélectionnez une catégorie dans la liste, ou choisissez **Enter a Custom Data Filter** et entrez une catégorie source personnalisée commençant par un trait de soulignement. Par exemple : ***sourceCategory=MyCategory***. |
   | **Advanced**             | Sélectionnez **Location in Library** (par défaut, **Personal** folder) ou cliquez sur **New Folder** pour ajouter un nouveau dossier.                                                                                                                         |
4. Sélectionnez **Add to Library**.

Une fois l’application installée, elle apparaît dans votre dossier **Personal** ou dans le dossier que vous avez sélectionné. Vous pouvez ensuite la partager avec votre organisation.

Les panneaux se rempliront automatiquement au fil du temps. Ils se remplissent lentement avec les données correspondant à la requête de plage horaire et reçues depuis la création du panneau.

<Frame>![Tableau de bord Sumo Logic avec les données de diffusion des journaux Auth0](/docs/images/cdy7uua7fh8z/2BynHH8t2c8axVoRK0h1Et/715854ee4e6774c64973ef1ecd7e8390/sumo-logic-dashboard.png)</Frame>

Le tableau de bord est divisé en catégories ou panneaux suivants :

* **Mesures de connexion** : Consultez le nombre d’actions, comme les connexions réussies et les échecs de connexion.
* **Mesures de création de compte** : Consultez la répartition des créations de compte par pays, plateforme, IP et autres données démographiques.
* **Mesures de sécurité** : Consultez les événements de détection d’anomalies au fil du temps pour détecter les attaques.

<div id="dashboard-filters">
  ## Filtres du tableau de bord
</div>

Chaque tableau de bord comprend un ensemble de filtres que vous pouvez appliquer à l’ensemble du tableau de bord. Cliquez sur l’icône en forme d’entonnoir dans la barre de menu supérieure du tableau de bord pour afficher une liste déroulante des filtres appliqués à l’ensemble du tableau de bord.

Chaque panneau comprend également un ensemble de filtres qui s’appliquent uniquement aux résultats de ce panneau. Cliquez sur l’icône en forme d’entonnoir dans la barre de menu supérieure du panneau pour afficher une liste des filtres propres à ce panneau.

<div id="customize-dashboard">
  ## Personnaliser le tableau de bord
</div>

Vous pouvez personnaliser votre tableau de bord de sécurité Auth0 dans Sumo Logic pour y ajouter des widgets de données personnalisées.

1. Accédez à votre tableau de bord de sécurité Auth0 et sélectionnez **add panel** dans le coin supérieur droit.
2. Sélectionnez un type de panneau en cliquant sur l’icône. Consultez [l’aide de Sumo Logic](https://help.sumologic.com/Visualizations-and-Alerts/Dashboard_\(New\)/Panels) pour en savoir plus.
3. Saisissez une requête de journaux, une requête de métriques ou plusieurs requêtes à partir de la liste déroulante.
4. Pour modifier le type de graphique, consultez [l’aide de Sumo Logic](https://help.sumologic.com/Visualizations-and-Alerts/Dashboard_\(New\)/Panels/01Modify_a_Chart) pour connaître les options offertes.
5. Cliquez sur **Add to Dashboard** dans le coin supérieur droit pour ajouter le panneau à votre tableau de bord.
