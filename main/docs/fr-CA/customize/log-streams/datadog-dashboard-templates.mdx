title: Utiliser les modèles de tableau de bord Auth0 avec Datadog
---

Auth0 fournit des modèles prédéfinis que vous pouvez utiliser pour visualiser les données de votre tenant Auth0 dans Datadog. Affichez des visualisations pour surveiller l’état du trafic de connexion d’un tenant. Les modèles vous permettent d’utiliser les agrégations recommandées par Auth0 ou de vous en servir comme point de départ pour créer vos propres visualisations personnalisées.

| Modèle             | Objectif                                                                                                                                | Fichier téléchargeable                                                                                        |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Security Dashboard | Surveillez le trafic de connexion du tenant, y compris les métriques de connexion, de création de compte et de détection des anomalies. | [Auth0SecDashboard.json](https://cdn.auth0.com/website/docs/logs/streams/datadog/Auth0SecurityDashboard.json) |

Vous pouvez également organiser votre visualisation à l’aide des fonctionnalités suivantes :

* Utilisez le sélecteur de période de Datadog pour choisir la plage de temps de votre visualisation.
* Utilisez des filtres pour affiner l’analyse des données à l’aide de paramètres configurables comme l’adresse IP, le pays, l’utilisateur et l’application cliente.
* Utilisez **Edit Widgets** pour réorganiser les tuiles et les visualisations, ainsi que pour créer des copies dans lesquelles vous pouvez examiner et ajouter des paramètres aux requêtes, au besoin.

<div id="prerequisites">
  ## Prérequis
</div>

* [Configurer le streaming des logs vers Datadog](https://marketplace.auth0.com/integrations/datadog-log-streaming).
* Téléchargez depuis Auth0 le fichier JSON du modèle de tableau de bord souhaité.

<div id="upload-dashboard-templates-in-datadog">
  ## Téléverser des modèles de tableau de bord dans Datadog
</div>

1. Connectez-vous à [Datadog](https://www.datadoghq.com/).
2. Accédez à **Dashboards** dans la barre de navigation de gauche.
3. Sélectionnez **Screenboard** comme type de tableau de bord.
4. Dans le canevas vide, en haut à droite, sélectionnez l’icône d’engrenage et cliquez sur **Import Dashboard JSON**.
5. Téléversez le fichier JSON du tableau de bord que vous souhaitez utiliser (par exemple, `Auth0SecDashboard.json`).

<div id="security-dashboard-visualizations">
  ## Visualisations du Security Dashboard
</div>

Une fois le Security Dashboard d’Auth0 importé, vous pouvez utiliser les visualisations suivantes pour évaluer la santé de votre tenant ou comme point de départ pour créer vos propres visualisations et alertes dans Datadog.

| Visualisation                                             | Objectif                                                                                                                                                                                      |
| --------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Connexions réussies                                       | Affiche le nombre d’événements de connexion réussie pour le tenant.                                                                                                                           |
| Échecs de connexion                                       | Affiche le nombre d’événements de connexion échouée pour le tenant. Des pics d’échecs de connexion peuvent indiquer un trafic anormal.                                                        |
| Échecs de connexion par géolocalisation                   | Affiche le nombre d’événements de connexion échouée pour le tenant par géolocalisation. Des pics de trafic inhabituels dans certaines zones géographiques peuvent indiquer un trafic anormal. |
| Nombre d’adresses IP uniques générant du trafic           | Affiche le nombre d’adresses IP qui génèrent du trafic vers le tenant. Ce nombre suit souvent des tendances prévisibles, et de fortes variations peuvent indiquer un trafic anormal.          |
| Top 5 des adresses IP avec le plus d’échecs               | Affiche les 5 adresses IP qui ont enregistré le plus grand nombre d’erreurs de connexion.                                                                                                     |
| Événements de création de compte dans les principaux pays | Affiche le nombre de comptes créés dans les principaux pays. Des écarts par rapport aux tendances et aux zones géographiques habituelles peuvent indiquer un trafic anormal.                  |
| Événements de création de compte par plateforme           | Affiche le nombre de comptes créés par plateforme, comme Windows, OSX, Android ou iOS.                                                                                                        |
| Adresses IP ayant créé le plus de comptes                 | Affiche les adresses IP à partir desquelles le plus grand nombre de comptes a été créé.                                                                                                       |
| Événements de détection d’anomalies                       | Affiche le nombre d’événements de détection d’anomalies pour le tenant, y compris le nombre d’adresses IP bloquées en raison de Brute Force Protection et de Breached Password Detection.     |
| Log Stream                                                | Affiche les événements bruts du journal du tenant.                                                                                                                                            |

<div id="customize-dashboard">
  ## Personnaliser le tableau de bord
</div>

Vous pouvez personnaliser votre tableau de bord de sécurité Datadog Auth0 pour y ajouter des widgets de données personnalisées.

1. Accédez à votre tableau de bord de sécurité Auth0, puis cliquez sur l’élément **Dashboards** dans la barre de navigation latérale.
2. Sélectionnez dans la liste le nom du tableau de bord que vous avez créé.
3. Sélectionnez une visualisation et faites-la glisser dans la section principale de la page.
4. Configurez les préférences de votre graphique de données.
