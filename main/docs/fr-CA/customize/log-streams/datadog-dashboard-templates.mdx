title: Utiliser les modèles de tableau de bord Auth0 avec Datadog
---

Auth0 propose des modèles prédéfinis que vous pouvez utiliser pour visualiser les données de votre tenant Auth0 dans Datadog. Affichez des visualisations pour surveiller la santé du trafic de login d’un tenant. Les modèles vous permettent d’utiliser les agrégations recommandées par Auth0 ou de vous en servir comme point de départ pour créer vos propres visualisations personnalisées.

| Modèle             | Objectif                                                                                                                      | Fichier téléchargeable                                                                                        |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| tableau de bord de sécurité | Surveiller le trafic de login du tenant, y compris les métriques de login, de création de compte et de détection d’anomalies. | [Auth0SecDashboard.json](https://cdn.auth0.com/website/docs/logs/streams/datadog/Auth0SecurityDashboard.json) |

Vous pouvez aussi organiser votre visualisation à l’aide des fonctionnalités suivantes :

* Utilisez le sélecteur de période de Datadog pour choisir l’intervalle de votre visualisation.
* Utilisez des filtres pour affiner l’analyse des données à l’aide de paramètres configurables comme l’adresse IP, le pays, l’utilisateur et l’application cliente.
* Utilisez **Edit Widgets** pour réorganiser les tuiles et les visualisations, ainsi que pour créer des copies dans lesquelles vous pouvez examiner et ajouter des paramètres aux requêtes au besoin.

<div id="prerequisites">
  ## Prérequis
</div>

* [Configurer la diffusion des journaux vers Datadog](https://marketplace.auth0.com/integrations/datadog-log-streaming).
* Télécharger le fichier JSON du modèle de tableau de bord souhaité à partir d’Auth0.

<div id="upload-dashboard-templates-in-datadog">
  ## Téléverser des modèles de tableau de bord dans Datadog
</div>

1. Connectez-vous à [Datadog](https://www.datadoghq.com/).
2. Accédez à **tableau de bord** dans la barre de navigation de gauche.
3. Sélectionnez **Screenboard** comme type de tableau de bord.
4. Dans la zone de travail vide, en haut à droite, sélectionnez l’icône d’engrenage, puis cliquez sur **Import Dashboard JSON**.
5. Téléversez le fichier JSON du tableau de bord que vous souhaitez utiliser (par exemple, `Auth0SecDashboard.json`).

<div id="security-dashboard-visualizations">
  ## Visualisations du tableau de bord de sécurité
</div>

Une fois que vous avez importé le tableau de bord de sécurité depuis Auth0, vous pouvez utiliser les visualisations suivantes pour évaluer l’état de votre tenant ou comme point de départ pour créer vos propres visualisations et alertes dans Datadog.

| Visualisation                                             | Objectif                                                                                                                                                                                                                          |
| --------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Connexions réussies                                       | Affiche le nombre d’événements de connexion réussie pour le tenant.                                                                                                                                                               |
| Échecs de connexion                                       | Affiche le nombre d’événements de connexion échouée pour le tenant. Des pics d’échecs de connexion peuvent indiquer un trafic anormal.                                                                                            |
| Échecs de connexion par géolocalisation                   | Affiche le nombre d’événements de connexion échouée pour le tenant par géolocalisation. Des pics de trafic inhabituels dans certaines régions peuvent indiquer un trafic anormal.                                                 |
| Nombre d’IP uniques générant du trafic                    | Affiche le nombre d’adresses IP qui génèrent du trafic vers le tenant. Ce nombre suit souvent des tendances prévisibles, et de fortes variations peuvent indiquer un trafic anormal.                                              |
| Top 5 des IP avec le plus d’échecs                        | Affiche les 5 adresses IP ayant enregistré le plus grand nombre d’erreurs de connexion.                                                                                                                                           |
| Événements de création de compte dans les principaux pays | Affiche le nombre de comptes créés dans les principaux pays. Des variations dans les tendances habituelles et la répartition géographique peuvent indiquer un trafic anormal.                                                     |
| Événements de création de compte par plateforme           | Affiche le nombre de comptes créés par plateforme, comme Windows, OSX, Android ou iOS.                                                                                                                                            |
| IP avec le plus de comptes créés                          | Affiche les adresses IP à partir desquelles le plus grand nombre de comptes a été créé.                                                                                                                                           |
| Événements de détection d’anomalies                       | Affiche le nombre d’événements de détection d’anomalies pour le tenant, y compris le nombre d’adresses IP bloquées en raison de la protection contre les attaques par force brute et de la détection des mots de passe compromis. |
| Flux de journaux                                          | Affiche les événements bruts du journal du tenant.                                                                                                                                                                                |

<div id="customize-dashboard">
  ## Personnaliser le tableau de bord
</div>

Vous pouvez personnaliser votre tableau de bord de sécurité Datadog Auth0 pour y ajouter des widgets de données personnalisées.

1. Accédez à votre tableau de bord de sécurité Auth0, puis sélectionnez l’élément **Tableaux de bord** dans le menu latéral.
2. Dans la liste, sélectionnez le nom du tableau de bord que vous avez créé.
3. Sélectionnez une visualisation et faites-la glisser dans la section principale de la page.
4. Configurez les préférences de votre graphique de données.
