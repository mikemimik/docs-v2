title: Vérifier l’état de santé du Log Stream
---

Vous pouvez résoudre les problèmes potentiels liés à votre stream en consultant l’onglet **Health**.

1. Accédez à [Dashboard &gt; Monitoring &gt; Streams](https://manage.auth0.com/#/log-streams).
2. Repérez dans la liste le log stream que vous souhaitez vérifier.

   Votre log stream peut avoir l’un des statuts suivants :

   * **Active**: Votre stream est activé, et nous tenterons de livrer les prochains événements de journal.
   * **Paused**: Vous avez demandé l’arrêt des tentatives de livraison pour ce stream. Vous pouvez cliquer sur **Resume Stream** pour remettre le statut à **Active**.
   * **Disabled**: Nous avons désactivé votre stream en raison d’erreurs consécutives. Vous pouvez cliquer sur **Restart Stream** pour remettre le statut à **Active** et relancer la livraison pour ce stream.
3. Sélectionnez le menu Plus d’options (...) à la fin du stream, puis cliquez sur **Health** dans le menu déroulant.
