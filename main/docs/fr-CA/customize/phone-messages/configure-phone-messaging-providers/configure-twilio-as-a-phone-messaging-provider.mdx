<div id="prerequisites">
  ## Prérequis
</div>

Vous devez avoir un compte [Twilio](https://twilio.com/) avec une option d’envoi valide par SMS et/ou par téléphone.

<div id="configure-twilio">
  ## Configurer Twilio
</div>

1. Connectez-vous à votre compte [Twilio](https://www.twilio.com/login).
2. Accédez à votre [Twilio Console](https://console.twilio.com/). Vous aurez besoin de votre Twilio SID et de votre Twilio Auth Token lorsque vous configurerez Twilio dans votre Auth0 Dashboard.
3. Dans le [Dashboard &gt; Branding &gt; Phone Provider](https://manage.auth0.com/#/branding/phone/provider) d’Auth0, sélectionnez **Twilio** comme **fournisseur téléphonique**.
4. Sélectionnez une **mode d’envoi**. Vous pouvez choisir **Text**, **Voice** ou les deux comme méthodes de livraison pour vos clients.
5. Dans les **Twilio Provider Settings**, sélectionnez une **SMS Source** :

   1. Si vous sélectionnez **Use From**, entrez un numéro de téléphone par défaut à partir duquel vos clients recevront les messages téléphoniques.
   2. Si vous sélectionnez [Messaging Service](https://www.twilio.com/docs/messaging/services), entrez le SID de votre service de messagerie Twilio. Vous trouverez cette valeur dans votre Twilio Console.
   3. Entrez votre Twilio SID et votre Twilio Auth Token à partir de la Twilio Console. Cliquez sur **Save**.

<div id="test-phone">
  ## Tester le téléphone
</div>

Cliquez sur le bouton **Try** pour envoyer un message téléphonique de test. Si vous avez configuré Twilio avec succès comme fournisseur téléphonique externe, Auth0 enverra un message de confirmation téléphonique en utilisant le mode d’envoi sélectionné. Si vous ne recevez pas de message téléphonique après quelques minutes, consultez vos [logs Auth0](https://manage.auth0.com/#/logs) pour vérifier s’il y a eu des échecs.

La page Twilio [Messaging Activity](https://console.twilio.com/us1/develop/sms/overview) fournit des informations sur la livraison de tous les SMS envoyés à vos utilisateurs.
