title: Configure Twilio as a Phone Messaging Provider
---

<div id="prerequisites">
  ## Prérequis
</div>

Vous devez avoir un compte [Twilio](https://twilio.com/) avec une option valide d’envoi par SMS et/ou par téléphone.

<div id="configure-twilio">
  ## Configurer Twilio
</div>

1. Connectez-vous à votre compte [Twilio](https://www.twilio.com/login).
2. Accédez à votre [Twilio Console](https://console.twilio.com/). Vous aurez besoin de votre SID Twilio et de votre Auth token Twilio lorsque vous configurerez Twilio dans votre Auth0 Dashboard.
3. Accédez à [Dashboard &gt; Branding &gt; Phone Provider](https://manage.auth0.com/#/branding/phone/provider) dans Auth0. Sélectionnez **Twilio** comme **Phone Provider**.
4. Sélectionnez une **Delivery Method**. Vous pouvez choisir **Text**, **Voice** ou **Both** comme méthodes de livraison pour vos clients.
5. Dans les **Twilio Provider Settings**, sélectionnez une **SMS Source** :

   1. Si vous sélectionnez **Use From**, entrez un numéro de téléphone par défaut à partir duquel les clients recevront les messages téléphoniques.
   2. Si vous sélectionnez [Messaging Service](https://www.twilio.com/docs/messaging/services), entrez le SID de votre service de messagerie Twilio. Vous trouverez cette valeur dans votre Twilio Console.
   3. Entrez votre SID Twilio et votre Auth token Twilio à partir de la Twilio Console. Cliquez sur **Save**.

<div id="test-phone">
  ## Tester le téléphone
</div>

Cliquez sur le bouton **Try** pour envoyer un message de test. Si vous avez configuré Twilio avec succès comme fournisseur téléphonique externe, Auth0 enverra un message de confirmation selon le mode de livraison sélectionné. Si vous ne recevez pas de message après quelques minutes, consultez vos [logs Auth0](https://manage.auth0.com/#/logs) pour voir s’il y a des erreurs.

La page [Messaging Activity](https://console.twilio.com/us1/develop/sms/overview) de Twilio affiche des renseignements sur la livraison de tous les SMS envoyés à vos utilisateurs.
