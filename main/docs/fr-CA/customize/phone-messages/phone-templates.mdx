Les courriels envoyés dans le cadre des flux de travail Auth0 ont chacun un modèle de courriel correspondant qui détermine l’adresse d’expéditeur, l’objet, le corps du message, et plus encore.

Après avoir [configuré un fournisseur de téléphonie externe](../phone-messages/configure-phone-messaging-providers), vous pouvez modifier les modèles de messages téléphoniques pour personnaliser vos messages téléphoniques.

Les modèles de messages texte prennent en charge [Liquid](https://liquidjs.com/index.html), un langage de gabarits libre qui étend les fonctionnalités du HTML. Liquid vous permet de :

* remplir dynamiquement des variables avec des données, par exemple en remplaçant `{{ user.name }}` par le nom de l’utilisateur
* implémenter une logique, comme des instructions `if`/`else`
* modifier des données à l’aide de filtres pour manipuler des chaînes de caractères et des tableaux, d’opérateurs mathématiques, et plus encore

Vous pouvez activer, désactiver et personnaliser les modèles de messages téléphoniques sur la [page des modèles de messages téléphoniques du Dashboard](https://manage.auth0.com/#/phone).

<div id="available-phone-templates">
  ## Modèles de messages téléphoniques disponibles
</div>

Voici un résumé des modèles de messages téléphoniques disponibles :

| Nom du modèle de messages téléphoniques    | Description                                                                                            |
| ------------------------------------------ | ------------------------------------------------------------------------------------------------------ |
| Compte bloqué                              | Informe le client que son compte a été bloqué à la suite d’une activité suspecte.                      |
| Changer le mot de passe                    | Envoie au client son code de changement de mot de passe.                                               |
| Mot de passe à usage unique - Inscription  | Envoie au client le code de vérification servant à l’inscription au mot de passe à usage unique (OTP). |
| Mot de passe à usage unique - Vérification | Envoie au client le code de vérification servant à vérifier le mot de passe à usage unique (OTP).      |
