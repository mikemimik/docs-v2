Chaque courriel envoyé dans le cadre des flux de travail Auth0 a un modèle de courriel correspondant qui détermine l’adresse d’expéditeur, l’objet, le corps du message et plus encore.

Après avoir [configuré un fournisseur de téléphonie externe](../phone-messages/configure-phone-messaging-providers), vous pouvez modifier les Phone Templates pour personnaliser vos messages téléphoniques.

Les modèles de messages texte prennent en charge [Liquid](https://liquidjs.com/index.html), un langage de gabarits open source qui étend les fonctionnalités du HTML. Liquid vous permet :

* de remplir dynamiquement des variables avec des données, par exemple en remplaçant `{{ user.name }}` par le nom de l’utilisateur;
* d’implémenter de la logique, comme des instructions `if`/`else`;
* de modifier des données à l’aide de filtres pour manipuler des chaînes de caractères et des tableaux, utiliser des opérateurs mathématiques, et plus encore

Vous pouvez activer, désactiver et personnaliser les Phone Templates sur la [page Phone Templates du Dashboard](https://manage.auth0.com/#/phone).

  ## Phone Templates disponibles
</div>

Voici un résumé des Phone Templates disponibles :

| Nom du Phone Template                      | Description                                                                                       |
| ------------------------------------------ | ------------------------------------------------------------------------------------------------- |
| Compte bloqué                              | Informe le client que son compte a été bloqué à la suite d’une activité suspecte.                 |
| Changement de mot de passe                 | Envoie au client son code de changement de mot de passe.                                          |
| Mot de passe à usage unique - Inscription  | Envoie au client le code de vérification pour l’inscription au mot de passe à usage unique (OTP). |
| Mot de passe à usage unique - Vérification | Envoie au client le code de vérification du mot de passe à usage unique (OTP).                    |
