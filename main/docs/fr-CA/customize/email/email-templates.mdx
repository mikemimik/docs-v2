---
title: Modèles de courriel
description: Activez et personnalisez les courriels envoyés par les workflows Auth0 à l’aide de modèles de courriel.
---

Chaque courriel envoyé dans le cadre des workflows Auth0 est associé à un modèle de courriel qui détermine l’adresse de l’expéditeur, l’objet, le corps du message, et plus encore.

Après avoir [configuré un fournisseur de courriel SMTP externe](/docs/fr-CA/customize/email/smtp-email-providers), vous pouvez modifier les modèles de courriel pour personnaliser vos courriels.

Les champs de contenu des modèles de courriel prennent en charge [Liquid](https://liquidjs.com/index.html), un langage de modèles open source qui étend les fonctionnalités du HTML. Liquid vous permet de :

* Insérer dynamiquement des données dans des variables, par exemple en remplaçant `{{ user.name }}` par le nom de l’utilisateur
* Implémenter de la logique, comme des instructions `if`/`else`
* Modifier des données à l’aide de filtres pour manipuler des chaînes de caractères et des tableaux, utiliser des opérateurs mathématiques, et plus encore

Vous pouvez activer, désactiver et personnaliser les modèles de courriel sur la [page des modèles de courriel du Dashboard](https://manage.auth0.com/#/templates).

<div id="email-template-limits">
  ## Limites des modèles de courriel
</div>

* La personnalisation des modèles de courriel n’est pas offerte lorsque vous utilisez le fournisseur de courriel intégré. Pour utiliser les modèles de courriel, [configurez un fournisseur de courriel SMTP externe](/docs/fr-CA/customize/email/smtp-email-providers).

* Chaque type de courriel a un seul modèle. Vous pouvez utiliser Liquid pour implémenter des conditions et d’autres règles logiques pour des cas d’utilisation comme les courriels multilingues.

* Les modèles de courriel ne prennent pas en charge les courriels en texte brut. Tous les courriels doivent être en HTML.

* Dans le modèle de courriel d’inscription MFA, la variable `connection.name` n’est pas disponible.

* Dans le modèle de courriel Passwordless OTP, seule la propriété `email` de l’objet `user` est disponible.

* Contrairement aux autres modèles de courriel, le modèle de courriel Passwordless OTP se configure dans les [paramètres de courriel des connexions Passwordless](https://manage.auth0.com/#/connections/passwordless) plutôt que sur la page Modèle de courriel.

* Pour des raisons de sécurité, le champ **From Address** dans les modèles de courriel ne peut pas contenir `@auth0.com`. Si vous entrez une adresse courriel `@auth0.com` dans les adresses d’expédition, votre tenant n’enverra que les courriels par défaut et ignorera toute personnalisation. Les utilisateurs de Private Cloud peuvent configurer une liste de refus de domaines semblable.
