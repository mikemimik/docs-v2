---
title: Configuration de Custom Email Provider avec Actions
description: Écrivez une Action pour acheminer des messages vers votre fournisseur de courriel et mettre en œuvre une logique personnalisée pour votre cas d’utilisation.
---

Pour personnaliser davantage la configuration de votre fournisseur de courriel, vous pouvez configurer votre tenant avec une <Tooltip tip="Action : une fonction Node.js qui s’exécute à des points définis pendant le runtime d’Auth0." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Action">Action</Tooltip>.

Au lieu d’utiliser notre fournisseur de courriel ou nos intégrations de serveur SMTP, vous écrivez le code de votre Action pour acheminer les messages vers le fournisseur de courriel ou le serveur SMTP de votre choix.

Cela vous donne plus de contrôle sur le processus d’envoi des courriels et vous permet de mettre en œuvre une logique personnalisée pour votre cas d’utilisation, par exemple en réessayant en cas d’échec, en modifiant les destinataires ou en créant une logique propre à l’Organization.
