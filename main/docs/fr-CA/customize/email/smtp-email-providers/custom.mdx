---
title: Configuration du Custom Email Provider avec Actions
description: Écrivez une Action pour envoyer des messages par l’intermédiaire de votre fournisseur de courriel et mettre en place une logique personnalisée pour votre cas d’utilisation.
---

Pour personnaliser davantage la configuration de votre fournisseur de courriel, vous pouvez configurer votre tenant avec une <Tooltip tip="Action : une fonction Node.js qui s’exécute à des points définis dans le runtime Auth0." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Action">Action</Tooltip>.

Au lieu d’utiliser notre fournisseur de courriel ou nos intégrations de serveurs SMTP, vous écrivez le code de votre Action pour envoyer les messages au fournisseur de courriel ou au serveur SMTP de votre choix.

Vous avez ainsi davantage de contrôle sur le processus d’envoi des courriels et pouvez mettre en place une logique personnalisée pour votre cas d’utilisation, par exemple réessayer en cas d’échec, modifier les destinataires ou créer une logique propre à une Organization.
