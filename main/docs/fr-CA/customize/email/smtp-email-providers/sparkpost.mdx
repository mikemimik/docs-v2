<div id="prerequisites">
  ## Prérequis
</div>

* Vous devez disposer d’un compte [SparkPost](https://www.sparkpost.com/).

* Vous devez configurer votre fournisseur afin d’autoriser les connexions entrantes en provenance de certaines [adresses IP d’Auth0](/docs/fr-CA/secure/security-guidance/data-security/allowlist).

<div id="configure-sparkpost">
  ## Configurer SparkPost
</div>

1. Connectez-vous à votre compte [SparkPost](https://www.sparkpost.com/).
2. Accédez à [Sending Domains](https://app.sparkpost.com/account/sending-domains) dans SparkPost, puis ajoutez votre domaine personnalisé. SparkPost permet d’envoyer des courriels uniquement à partir de domaines vérifiés.
3. Accédez à [Account API Keys](https://app.sparkpost.com/account/credentials) dans SparkPost, puis cliquez sur **New API key**. Enregistrez la valeur de cette clé et assurez-vous qu’elle dispose de l’accès `Transmissions: Read/Write`. Copiez cette valeur.
4. Accédez à [Dashboard &gt; Branding &gt; Email Provider](https://manage.auth0.com/#/branding/email_provider) dans Auth0.
5. Activez la bascule **Use my own email provider**.
6. Cliquez sur le logo **SparkPost**.
7. Indiquez une adresse courriel **From**, saisissez l’**API Key** SparkPost que vous avez copiée précédemment, puis sélectionnez votre **Region**. Si vous passez à la région UE dans un compte déjà configuré pour la région É.-U., vous devez remplacer l’**API Key** dans Auth0 par une API Key SparkPost pour l’UE.
8. Cliquez sur **Save**.

<div id="send-a-test-email">
  ## Envoyer un courriel de test
</div>

Envoyez un courriel de test à l’aide du bouton **Envoyer un courriel de test**. Si la configuration est correcte, vous recevrez un courriel de confirmation. Si vous ne recevez aucun courriel après quelques minutes, consultez vos [journaux Auth0](https://manage.auth0.com/#/logs) pour repérer les codes d’erreur.

Si vous n’avez pas enregistré de <Tooltip tip="Domaine personnalisé : domaine tiers doté d’un nom spécialisé ou personnalisé." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=custom+domain">domaine personnalisé</Tooltip> avec SparkPost, vous pouvez envoyer au maximum cinq courriels de test à partir du domaine `sparkpostbox.com`. Pour en savoir plus, consultez la documentation SparkPost : [Transmissions: The Sandbox Domain](https://developers.sparkpost.com/api/transmissions.html#header-the-sandbox-domain).

Une fois la configuration terminée, la page SparkPost [Message Events](https://app.sparkpost.com/reports/message-events) affiche des informations sur la livraison de tous les courriels envoyés à vos utilisateurs.
