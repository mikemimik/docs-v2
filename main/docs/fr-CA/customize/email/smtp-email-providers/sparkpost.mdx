<div id="prerequisites">
  ## Prérequis
</div>

* Vous devez avoir un compte [SparkPost](https://www.sparkpost.com/).

* Vous devez configurer votre fournisseur pour autoriser les connexions entrantes à partir d’[adresses IP d’Auth0](/docs/fr-CA/secure/security-guidance/data-security/allowlist).

<div id="configure-sparkpost">
  ## Configurer SparkPost
</div>

1. Ouvrez une session dans votre compte [SparkPost](https://www.sparkpost.com/).
2. Accédez à la section [Sending Domains](https://app.sparkpost.com/account/sending-domains) de SparkPost et ajoutez votre domaine personnalisé. SparkPost permet d’envoyer des courriels uniquement à partir de domaines vérifiés.
3. Accédez à la section [Account API Keys](https://app.sparkpost.com/account/credentials) de SparkPost, puis cliquez sur **New API key**. Enregistrez la valeur de cette clé et assurez-vous qu’elle dispose de l’accès `Transmissions: Read/Write`. Copiez cette valeur.
4. Accédez à [Dashboard &gt; Branding &gt; Email Provider](https://manage.auth0.com/#/branding/email_provider) dans Auth0.
5. Activez la bascule **Use my own email provider**.
6. Cliquez sur le logo **SparkPost**.
7. Indiquez une adresse courriel **From**, saisissez la **API Key** SparkPost que vous avez copiée précédemment, puis sélectionnez votre **Region**. Si vous passez à la région EU dans un compte déjà configuré pour la région US, vous devez remplacer la **API Key** dans Auth0 par une clé API SparkPost EU.
8. Cliquez sur **Save**.

<div id="send-a-test-email">
  ## Envoyer un courriel de test
</div>

Envoyez un courriel de test à l’aide du bouton **Send Test Email**. Si la configuration est correcte, vous recevrez un courriel de confirmation. Si vous ne recevez aucun courriel après quelques minutes, consultez vos [journaux Auth0](https://manage.auth0.com/#/logs) pour obtenir les codes d’erreur.

Si vous n’avez pas enregistré de <Tooltip tip="Domaine personnalisé : domaine tiers ayant un nom spécialisé ou personnalisé." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=custom+domain">domaine personnalisé</Tooltip> auprès de SparkPost, vous pouvez envoyer au maximum cinq courriels de test à partir du domaine `sparkpostbox.com`. Pour en savoir plus, consultez la documentation de SparkPost : [Transmissions: The Sandbox Domain](https://developers.sparkpost.com/api/transmissions.html#header-the-sandbox-domain).

Une fois la configuration effectuée, la page [Message Events](https://app.sparkpost.com/reports/message-events) de SparkPost affiche des informations sur la livraison de tous les courriels envoyés à vos utilisateurs.
