title: Configurer les domaines personnalisés avec des certificats gérés par Auth0
---

Si vous voulez qu’Auth0 gère les certificats de votre <Tooltip tip="Domaine personnalisé : domaine tiers portant un nom spécialisé ou personnalisé." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=custom+domain">domaine personnalisé</Tooltip>, il vous suffit d’ajouter un enregistrement CNAME au domaine. Auth0 valide cet enregistrement, puis génère le certificat sur ses serveurs. Ces certificats sont renouvelés automatiquement tous les trois mois. Cette configuration est simple à mettre en place, et vous n’aurez pas à gérer les certificats vous-même.

Pour configurer votre domaine personnalisé à l’aide de certificats gérés par Auth0, vous devez fournir votre nom de domaine à Auth0 et vérifier que vous en êtes propriétaire. Une fois cette vérification effectuée, vous devrez configurer les fonctionnalités d’Auth0 pour commencer à utiliser votre domaine personnalisé.

<div id="provide-your-domain-name-to-auth0">
  ## Fournissez votre nom de domaine à Auth0
</div>

1. Accédez à [Dashboard &gt; Settings &gt; Custom Domains](https://manage.auth0.com/#/tenant/custom_domains) ou à [Dashboard &gt; Branding &gt; Custom Domains](https://manage.auth0.com/#/custom_domains). Entrez votre domaine personnalisé dans le champ prévu à cet effet, puis sélectionnez **certificats gérés par Auth0**.
2. Cliquez sur **Add Domain**.Vous ne pouvez ajouter qu’un seul domaine par tenant, même si le bouton **Add Domain** s’affiche toujours après l’ajout d’un domaine.

<div id="verify-ownership">
  ## Vérifier la propriété du domaine
</div>

Avant de pouvoir utiliser ce domaine avec Auth0, vous devrez vérifier que vous en êtes propriétaire.

1. Accédez à [Dashboard &gt; Branding &gt; Custom Domains](https://manage.auth0.com/#/branding/custom_domains) et ajoutez à l’enregistrement DNS de votre domaine l’enregistrement de vérification CNAME indiqué dans le Dashboard.

   <Frame>![Auth0 Branding Custom Domains Auth0-Managed Certificate Verify Domain](/docs/images/cdy7uua7fh8z/3dz8ii70nm0a5amS0PNJ5K/c7b58043042293f5502ec012901f5dac/Custom_Domain_-_Pending_Verification_-_English.png)</Frame>
2. Cliquez sur **Verify** pour continuer.
   Selon votre configuration DNS, quelques minutes peuvent être nécessaires avant qu’Auth0 puisse vérifier votre enregistrement CNAME. Si Auth0 parvient à vérifier votre nom de domaine, une fenêtre de confirmation s’affichera. Cela signifie que la vérification est terminée. Dans 1 ou 2 minutes, votre domaine personnalisé devrait être prêt à être utilisé.

<div id="add-cname-verification-record-to-dns-record">
  ## Ajouter l’enregistrement de vérification CNAME à l’enregistrement DNS
</div>

Une fois ajouté, l’enregistrement CNAME doit rester présent en tout temps afin d’éviter des problèmes lors du renouvellement du certificat.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 recommande de désactiver l’aplatissement CNAME, sauf si cela est strictement nécessaire, car l’aplatissement CNAME pour les certificats gérés par Auth0 est une configuration non prise en charge et peut entraîner l’interruption du domaine personnalisé sans préavis.
</Callout>

Si vous devez activer l’aplatissement CNAME pour tous les sous-domaines gérés par Cloudfare et également configurer un sous-domaine précis comme domaine personnalisé Auth0, envisagez de [déléguer le sous-domaine](https://developers.cloudflare.com/dns/manage-dns-records/how-to/subdomains-outside-cloudflare/) pour Auth0 à un autre fournisseur DNS.
Cela vous permettra d’utiliser l’aplatissement CNAME pour tous les sous-domaines, sauf celui utilisé pour Auth0.

Les étapes suivantes peuvent varier selon votre fournisseur de domaine.

1. Connectez-vous à votre service de gestion de domaine.
2. Créez un nouvel enregistrement.

   | Paramètre                 | Valeur                                                                                                 |
   | ------------------------- | ------------------------------------------------------------------------------------------------------ |
   | **Type d’enregistrement** | **CNAME**                                                                                              |
   | **Nom**                   | Saisissez votre nom de domaine personnalisé (par exemple **login.northwind.com**).                     |
   | **Durée de vie (TTL)**    | Utilisez la valeur par défaut.                                                                         |
   | **Valeur**                | Collez la valeur **CNAME** fournie dans le Auth0 Dashboard pour l’enregistrement DNS de votre domaine. |
3. Lorsque vous avez terminé, enregistrez votre enregistrement.

Si Auth0 a pu vérifier votre nom de domaine, une fenêtre de confirmation s’affichera. Cela signifie que le processus de vérification est terminé. Dans un délai de 1 à 2 minutes, votre domaine personnalisé devrait être prêt à être utilisé.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Si votre fournisseur DNS active un proxy sur l’enregistrement CNAME par défaut, le domaine personnalisé restera indéfiniment à l’état En attente. Vous devrez peut-être vérifier les paramètres de votre fournisseur DNS et désactiver le proxy.
</Callout>

Si vous ne parvenez pas à terminer le processus de vérification, attendez au moins 4 heures avant de répéter ces étapes. Pour éviter toute interruption de service, ne recréez pas le domaine personnalisé lors de la nouvelle vérification.

<div id="additional-steps-for-specific-auth0-features">
  ## Étapes supplémentaires pour certaines fonctionnalités d’Auth0
</div>

Selon les fonctionnalités d’Auth0 que vous utilisez, vous devez effectuer des étapes de configuration supplémentaires. Pour en savoir plus, consultez [Configurer les fonctionnalités pour utiliser des domaines personnalisés](/docs/fr-CA/customize/custom-domains/configure-features-to-use-custom-domains).
