Pour en savoir plus sur les déclencheurs d’inscription et de connexion, consultez les sections ci-dessous.

| Déclencheur                                                                                                                                          | Description                                                                                                                                 |
| ---------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| [Déclencheur de connexion](/docs/fr-CA/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)                                         | Les déclencheurs de connexion s’exécutent lorsqu’un utilisateur se connecte à n’importe quelle application d’un tenant Auth0.               |
| [Déclencheur de préinscription de l’utilisateur](/docs/fr-CA/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger)   | Le déclencheur de préinscription de l’utilisateur s’exécute avant qu’un utilisateur soit ajouté à une connexion Database ou Passwordless.   |
| [Déclencheur de postinscription de l’utilisateur](/docs/fr-CA/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | Le déclencheur de postinscription de l’utilisateur s’exécute après qu’un utilisateur a été ajouté à une connexion Database ou Passwordless. |
