L’objet API du déclencheur post-change-password d’Actions comprend :

<div id="apicache">
  ## `api.cache`
</div>

Permet d’enregistrer et de récupérer des données conservées d’une exécution à l’autre.

<div id="apicachedeletekey">
  ### `api.cache.delete(key)`
</div>

Supprime l’enregistrement de la valeur mise en cache associée à la clé fournie, s’il existe.

Renvoie un objet `CacheWriteResult` avec `type: "success"` si une valeur a été supprimée du cache. Si l’opération échoue, la valeur renvoyée est `type: "error"`. En cas d’erreur, l’objet renvoyé comporte une
propriété `code` indiquant la nature de l’échec.

| Paramètre | Description                                                |
| --------- | ---------------------------------------------------------- |
| `key`     | *String*. La clé de l’enregistrement stocké dans le cache. |

<div id="apicachegetkey">
  ### `api.cache.get(key)`
</div>

Récupère un enregistrement décrivant une valeur en cache associée à la
`key` fournie, si elle existe. Si un enregistrement est trouvé, la valeur en cache
se trouve dans la propriété `value` de l&#39;objet renvoyé.

Renvoie un enregistrement du cache si un élément est trouvé dans le cache pour la
`key` fournie. Les enregistrements du cache sont des objets avec une
propriété `value` contenant la valeur en cache, ainsi qu&#39;une
propriété `expires_at` indiquant l&#39;expiration maximale de
l&#39;enregistrement en millisecondes depuis l&#39;époque Unix.

**Important :**
Ce cache est conçu pour des données éphémères de courte durée. Il se peut que des éléments ne soient pas
disponibles lors de transactions ultérieures, même s&#39;ils sont toujours dans leur
durée de vie prévue.

| Paramètre | Description                                                    |
| --------- | -------------------------------------------------------------- |
| `key`     | *String*. La clé de l&#39;enregistrement stocké dans le cache. |

<div id="apicachesetkey-value-options">
  ### `api.cache.set(key, value, [options])`
</div>

Stocke ou met à jour une valeur de type string dans le cache à la clé spécifiée.

Les valeurs stockées dans ce cache sont limitées au déclencheur dans lequel elles sont définies.
Elles sont assujetties aux
[limites du cache des Actions](/docs/fr-CA/customize/actions/limitations).

Les valeurs stockées de cette façon ont une durée de vie pouvant aller jusqu&#39;aux valeurs
`ttl` ou `expires_at` spécifiées. Si aucune durée de vie n&#39;est
spécifiée, une durée de vie par défaut de 15 minutes sera utilisée. Les durées de vie ne peuvent
pas dépasser la durée maximale indiquée dans
[limites du cache des Actions](/docs/fr-CA/customize/actions/limitations).

| Paramètre            | Description                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `key`                | *String*. La clé de l&#39;enregistrement stocké dans le cache.                                                                                                                                                                                                                                                                                                                                                                                      |
| `value`              | *String*. La valeur de l&#39;enregistrement à stocker.                                                                                                                                                                                                                                                                                                                                                                                              |
| `options`            | *Objet facultatif*. Options permettant d&#39;ajuster le comportement du cache.                                                                                                                                                                                                                                                                                                                                                                      |
| `options.expires_at` | *Nombre facultatif*. L&#39;heure d&#39;expiration absolue, en millisecondes depuis l&#39;époque Unix. Bien que les enregistrements mis en cache puissent être évincés plus tôt, ils ne seront jamais conservés au-delà de la valeur `expires_at` fournie.<br />*Remarque :* Cette valeur ne doit pas être fournie si une valeur a aussi été fournie pour `ttl`. Si les deux options sont fournies, l&#39;échéance la plus rapprochée sera utilisée. |
| `options.ttl`        | *Nombre facultatif*. La durée de vie de cette entrée de cache, en millisecondes. Bien que les valeurs mises en cache puissent être évincées plus tôt, elles ne seront jamais conservées au-delà de la valeur `ttl` fournie.<br />*Remarque :* Cette valeur ne doit pas être fournie si une valeur a aussi été fournie pour `expires_at`. Si les deux options sont fournies, l&#39;échéance la plus rapprochée sera utilisée.                        |
