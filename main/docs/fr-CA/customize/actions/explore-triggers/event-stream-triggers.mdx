Les Actions Event Stream sont des fonctions qui s’exécutent de façon asynchrone lorsque des événements précis se produisent dans votre tenant Auth0. Il s’agit d’actions non bloquantes qui n’ont aucune incidence sur la latence de l’expérience utilisateur.

Chaque Action Event Stream est associée à un Event Stream et écoute un ensemble prédéfini de Types d’événements (par exemple, des connexions réussies, des changements de mot de passe ou des suppressions d’utilisateurs). Lorsqu’un événement auquel vous êtes abonné se produit, l’Action est déclenchée. Contrairement aux Actions Login ou Pre-Registration, les Actions Event Stream s’exécutent en arrière-plan et n’affectent pas la transaction principale de l’utilisateur.

<Frame>![Diagramme illustrant le TBD.](/docs/images/customize/actions/event-stream-triggers.svg)</Frame>

Ces Actions sont non bloquantes (asynchrones) et sont configurées pour un ensemble précis de [Types d’événements](/docs/fr-CA/customize/events/event-types).

<div id="synchronization">
  ### Synchronisation
</div>

Une Action Event Stream peut servir à transmettre un événement précis à un service externe selon une logique personnalisée. L’Action suivante montre comment transmettre en toute sécurité le message d’un événement à un service externe à l’aide d’une clé API stockée.

```javascript lines
/**
* Handler à exécuter lors du traitement des événements dans un Event Stream.
* @param {Event} event - Détails sur l'événement entrant.
* @param {EventStreamAPI} api - Méthodes et utilitaires pour définir le traitement de l’Event Stream.
*/
exports.onExecuteEventStream = async (event, api) => {
  const message = event.message;

  try {
    await fetch(event.secrets.URL, {
      method: 'POST',
      headers: {
        'X-API-Key': event.secrets.API_KEY,
      },
      body: JSON.stringify(message),
    });
  } catch (err) {
    throw new Error('External service failure');
  }

  return;
};
```

Pour en savoir plus sur la création d’un Event Stream, consultez [Créer un Event Stream](/docs/fr-CA/customize/events/create-an-event-stream#create-an-event-stream-actions).
