---

Grâce à l’historique des versions des Actions, vous pouvez comparer les versions à l’état de brouillon et les versions déployées des Actions, et rétablir des versions antérieures au besoin.

<div id="deploy-actions">
  ## Déployer des Actions
</div>

Avant qu’une Action puisse être associée à un flow, elle doit être déployée.

1. Accédez à [Auth0 Dashboard &gt; Actions &gt; Actions personnalisées](https://manage.auth0.com/#/actions/library).
2. Choisissez votre Action.
3. Sélectionnez **Deploy**.

L’Action sélectionnée sera déployée et ajoutée à la liste des actions disponibles pour les flows. De plus, une nouvelle version de brouillon de l’Action déployée sera créée pour des modifications ultérieures. Comme les secrets et les dépendances sont associés aux versions, tous les secrets ou dépendances enregistrés seront accessibles dans ce nouveau brouillon.

<div id="revert-actions">
  ## Rétablir des Actions
</div>

Au besoin, vous pouvez rétablir la dernière version fiable connue d’une Action.

1. Accédez à [Auth0 Dashboard &gt; Actions &gt; Actions personnalisées](https://manage.auth0.com/#/actions/library).
2. Choisissez votre Action.
3. Sélectionnez **Historique des versions**.
4. Sélectionnez la version à laquelle vous voulez revenir, puis cliquez sur **Rétablir**.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Dans l’historique des versions, « Brouillon » désigne une version qui n’a jamais été déployée.
   </Callout>

Une nouvelle copie de la version sélectionnée sera créée et déployée. De plus, une nouvelle version brouillon de l’Action déployée sera créée afin de pouvoir être modifiée plus tard. Comme les secrets et les dépendances sont liés aux versions, tous les secrets ou dépendances enregistrés seront disponibles dans le nouveau brouillon.
