title: Exporter les données utilisateur vers Adobe Campaign
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans Adobe Campaign à l’aide de l’[Adobe Campaign Import Wizard](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par vous rendre dans la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez le **Format d’exportation** sur le format de fichier requis. Adobe Campaign accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Champs**, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | Adresse courriel |
| `created_at`      | Créé le          |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, vous trouverez plus d’information dans la [documentation Adobe Campaign : Importing Data](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html).

Connectez-vous à votre dashboard client Adobe Campaign, puis accédez à **Profiles and Targets &gt; Jobs**. Créez un nouveau job d’importation en cliquant sur le bouton **Create** et en sélectionnant **New Import**.

Une nouvelle fenêtre **Import Wizard** devrait s’ouvrir. À l’étape **Template Selection**, vous pouvez définir vos paramètres d’importation.

Une fois vos paramètres d’importation configurés, cliquez sur le bouton **Next** pour continuer.

À l’étape **File to Import**, téléversez le fichier CSV de données utilisateur que vous avez exporté depuis Auth0 dans la section précédente. Cliquez sur le bouton **Next** pour passer à **Field Mapping**.

Ensuite, mappez le schéma du fichier exporté au schéma de votre base de données Adobe Campaign. Vérifiez que les noms de champs et les types de champs sont exacts, puis cliquez sur le bouton **Next**.

Terminez les étapes de configuration restantes en définissant votre mode de rapprochement des données et en sélectionnant un dossier, une liste ou un service pour les utilisateurs importés.

Enfin, lancez l’importation en cliquant sur le bouton **Start** dans la fenêtre **Data Import Execution**.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Adobe Campaign.
