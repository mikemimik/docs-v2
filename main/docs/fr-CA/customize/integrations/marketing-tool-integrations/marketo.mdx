Dans cet article, vous apprendrez à exporter des données utilisateur d’Auth0 vers un fichier CSV, puis à les importer dans Marketo à l’aide du [point de terminaison Bulk Leads](https://developer.adobe.com/marketo-apis/api/mapi/#tag/Bulk-Import-Leads) de l’API REST de Marketo.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, définissez **Export Format** selon le format de fichier requis. Marketo accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name      |
| ------------- | ---------------- |
| `email`       | adresse courriel |
| `created_at`  | Date de création |
| `given_name`  | Prénom           |
| `family_name` | Nom de famille   |

Après avoir ajouté les attributs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ### Importer un fichier de données utilisateur
</div>

Avant de commencer, vous trouverez plus d’information dans la [documentation Marketo : Bulk Lead Import](https://experienceleague.adobe.com/en/docs/marketo-developer/marketo/rest/bulk-import/bulk-lead-import).

Pour importer le fichier de données utilisateur dans Marketo, effectuez une requête POST vers le [point de terminaison Bulk Leads](https://developer.adobe.com/marketo-apis/api/mapi/#operation/importLeadUsingPOST). Définissez l’en-tête `content-type` de la requête sur `multipart/form-data` et incluez un paramètre `file` avec votre fichier CSV exporté, ainsi qu’un paramètre de format défini sur `csv`. Par exemple :

<AuthCodeGroup>
  ```bash cURL lines
  curl --request POST \
    --url https://marketo_rest_api_base_url/bulk/v1/leads.json \
    --header 'authorization: Bearer {MARKETO_ACCESS_TOKEN}' \
    --form file=@auth0_users.csv \
    --form format=csv
  ```
