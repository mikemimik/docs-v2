---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans Alterian à l’aide de l’outil Data Import d’Alterian Campaign Manager.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez le **Format d’exportation** sur le format de fichier requis. Alterian accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Champs**, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | Adresse courriel |
| `created_at`      | Date de création |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, vous pouvez obtenir plus d’information en consultant [Alterian Campaign Manager: Data Import](https://cm.help.alterian.com/CM404/Default.htm#Customer_Analytics/Import_Export/Data_Import.htm).

Pour importer votre fichier CSV dans Alterian, suivez les étapes suivantes :

1. Ouvrez Alterian Campaign Manager et cliquez sur **Documents**.
2. Choisissez le répertoire de destination du fichier.
3. Cliquez sur **Upload Files** et sélectionnez votre fichier CSV.
4. Cliquez sur **Upload**.
5. Une fois le téléversement terminé, ouvrez l’outil **Data Import**.
6. Configurez les paramètres d’importation à l’aide des champs fournis dans la fenêtre **New Imports data**.
7. Après avoir vérifié vos paramètres, cliquez sur **Run Processes**.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Alterian.
