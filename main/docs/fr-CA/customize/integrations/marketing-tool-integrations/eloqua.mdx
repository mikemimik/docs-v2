title: Exporter les données utilisateur vers Oracle Eloqua
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 vers un fichier CSV, puis à les importer dans Eloqua à l’aide de l’assistant de téléversement des contacts.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** au format de fichier requis. Eloqua accepte l’importation de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

Dans la section **Fields**, en haut de la page, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | adresse courriel |
| `created_at`      | Date de création |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, vous trouverez plus d’information ici : [Oracle Eloqua Help Center: Uploading Contacts](https://docs.oracle.com/en/cloud/saas/marketing/eloqua-user/Help/Contacts/Tasks/UploadingContacts.htm).

Pour importer votre fichier CSV dans Eloqua, suivez ces étapes :

1. Connectez-vous à Eloqua, puis accédez à **Audience &gt; Contacts &gt; Upload** pour ouvrir l’**Assistant de téléversement des contacts**.
2. Configurez l’importation à l’aide des champs fournis dans l’onglet **Pick Data Source**.
3. Cliquez sur le bouton **Cloud** et sélectionnez le fichier à téléverser.
4. Après avoir vérifié le contenu du fichier dans l’onglet **Review**, cliquez sur **Étape suivante**.
5. Dans l’onglet **Map Fields**, saisissez les paramètres de mappage des champs afin de faire correspondre les données de votre fichier CSV aux champs des contacts. Cliquez sur **Étape suivante** pour continuer.
6. Terminez la dernière étape de l’assistant, puis cliquez sur **Finish**.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Eloqua.
