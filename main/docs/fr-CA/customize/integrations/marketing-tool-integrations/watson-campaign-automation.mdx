title: Exporter les données utilisateur vers Watson Campaign Automation
---

Dans cet article, vous apprendrez comment exporter les données utilisateur d’Auth0 dans un fichier CSV, puis les importer dans le tableau de bord de Watson Campaign Automation.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, définissez le **Format d’exportation** selon le format de fichier requis. Watson Campaign Automation accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Champs**, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | adresse courriel |
| `created_at`      | Date de création |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV afin de l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Pour importer votre fichier CSV dans Watson Campaign Automation, suivez les étapes suivantes :

1. Connectez-vous à Watson Campaign Automation, puis accédez à **Data &gt; Import New**.
2. À l’étape **Select File**, saisissez les paramètres de la nouvelle base de données dans les champs prévus à cet effet.
3. Sélectionnez l’option permettant de téléverser un fichier depuis un disque dur local, puis cliquez sur **Browse** pour le sélectionner.
4. Réglez le type de fichier sur CSV (valeurs séparées par des virgules).
5. Cliquez sur **Next** pour passer à l’étape **Define Data Format**.
6. Assurez-vous que vos données ont été traitées correctement, puis cliquez sur **Next**.
7. À l’étape **Map Fields**, choisissez les champs à inclure dans l’importation ainsi que leur type.
8. Cliquez sur **Next** pour passer à l’étape **Edit Field Settings**.
9. Modifiez au besoin les paramètres des champs par défaut ou des nouveaux champs.
10. Pour lancer l’importation, cliquez sur **Next**.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Watson Campaign Automation.
