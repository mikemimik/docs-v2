title: Exporter les données utilisateur vers Watson Campaign Automation
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 vers un fichier CSV, puis à les importer dans le tableau de bord de Watson Campaign Automation.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par aller à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** sur le format de fichier requis. Watson Campaign Automation accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | Adresse courriel |
| `created_at`      | Créé le          |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Pour importer votre fichier CSV dans Watson Campaign Automation, suivez les étapes ci-dessous :

1. Ouvrez une session dans Watson Campaign Automation et accédez à **Data &gt; Import New**.
2. À l’étape **Select File**, entrez les paramètres de la nouvelle base de données dans les champs prévus à cette fin.
3. Sélectionnez l’option pour téléverser un fichier à partir d’un disque dur local, puis cliquez sur **Browse** pour sélectionner le fichier.
4. Réglez le type de fichier sur CSV (valeurs séparées par des virgules).
5. Cliquez sur **Next** pour passer à l’étape **Define Data Format**.
6. Assurez-vous que vos données ont été traitées correctement, puis cliquez sur **Next**.
7. À l’étape **Map Fields**, choisissez les champs à inclure dans l’importation ainsi que leur type.
8. Cliquez sur **Next** pour passer à l’étape **Edit Field Settings**.
9. Modifiez au besoin les paramètres des champs par défaut ou des nouveaux champs.
10. Pour lancer l’importation, cliquez sur **Next**.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Watson Campaign Automation.
