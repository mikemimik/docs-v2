---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 vers un fichier CSV, puis à les importer dans le tableau de bord de Sailthru.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** sur le format de fichier requis. Sailthru accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name      |
| ------------- | ---------------- |
| `email`       | adresse courriel |
| `created_at`  | Créé le          |
| `given_name`  | Prénom           |
| `family_name` | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ### Importer un fichier de données utilisateur
</div>

Avant de commencer, vous trouverez plus d’information dans [Sailthru Documentation: Adding Users to Sailthru and Sailthru Lists](https://getstarted.meetmarigold.com/engagebysailthru/Content/audience/managing-users/add-users-lists-st.html?TocPath=Audience%7CManaging%20Users%7C_____3#ListFileUpload).

Pour importer votre fichier CSV dans Sailthru, suivez les étapes suivantes :

1. Connectez-vous à Sailthru, puis accédez à [Lists](https://my.sailthru.com/lists).
2. Choisissez une liste, cliquez sur **Upload List**, puis sélectionnez votre fichier CSV.
3. Sélectionnez **Add To List** dans le champ **Action**.
4. Définissez l’option **Replace Vars** et, au besoin, entrez une adresse courriel pour recevoir des notifications.
5. Vérifiez vos paramètres, puis cliquez sur **Submit** pour téléverser le fichier.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Sailthru.
