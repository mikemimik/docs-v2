---

Dans cet article, vous verrez comment exporter les données utilisateur d’Auth0 dans un fichier CSV, puis les importer dans le tableau de bord de Sailthru.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez le **Export Format** sur le format de fichier requis. Sailthru accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut de la page, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ### Importer un fichier de données utilisateur
</div>

Avant de commencer, vous pouvez obtenir de plus amples renseignements en consultant la [documentation Sailthru : ajout d’utilisateurs à Sailthru et aux listes Sailthru](https://getstarted.meetmarigold.com/engagebysailthru/Content/audience/managing-users/add-users-lists-st.html?TocPath=Audience%7CManaging%20Users%7C_____3#ListFileUpload).

Pour importer votre fichier CSV dans Sailthru, suivez les étapes ci-dessous :

1. Connectez-vous à Sailthru et accédez à [Lists](https://my.sailthru.com/lists).
2. Choisissez une liste, puis cliquez sur **Upload List** et sélectionnez votre fichier CSV.
3. Sélectionnez **Add To List** pour l’**Action**.
4. Définissez l’option **Replace Vars** et saisissez une adresse courriel pour recevoir des notifications, au besoin.
5. Vérifiez vos paramètres et cliquez sur **Submit** pour téléverser le fichier.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Sailthru.
