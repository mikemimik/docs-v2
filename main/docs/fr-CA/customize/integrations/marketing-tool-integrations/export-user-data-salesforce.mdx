title: Exporter des données utilisateur vers Salesforce
---

Dans cet article, vous apprendrez à exporter des données utilisateur depuis Auth0 dans un fichier CSV, puis à les importer dans Salesforce à l’aide du Data Import Wizard. Pour consulter la configuration requise, consultez l’article Salesforce [Data Import Wizard](https://help.salesforce.com/articleView?id=data_import_wizard.htm).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par vous rendre dans la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez le **Export Format** sur le format de fichier requis. Salesforce accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | adresse courriel |
| `created_at`      | Date de création |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Selon le type de compte, des étapes supplémentaires peuvent être nécessaires pour réussir l’importation. Prenez un moment pour consulter l’article Salesforce [Prepare Your Data for Import](https://help.salesforce.com/articleView?id=import_prepare.htm). Pour obtenir des instructions supplémentaires, consultez [Salesforce: Import Data with the Data Import Wizard](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm).

Avant de commencer, assurez-vous que votre compte Salesforce dispose des [autorisations utilisateur](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm) requises pour importer des enregistrements.

Pour importer votre fichier CSV dans Salesforce, suivez les étapes suivantes :

1. Connectez-vous à [Salesforce](https://login.salesforce.com/).
2. Accédez à **Configuration** et ouvrez le **Data Import Wizard**.
3. Cliquez sur **Lancer l’assistant**.
4. Configurez l’importation et sélectionnez le fichier CSV à téléverser à l’aide des champs fournis.
5. Cliquez sur **Suivant**.
6. Dans la page **Modifier le mappage des champs**, mappez les champs de données de votre fichier CSV aux champs de données de Salesforce. Cliquez sur **Suivant** pour continuer.
7. Après avoir vérifié les renseignements dans la page **Vérification**, cliquez sur **Démarrer l’importation**.
8. Consultez l’onglet **Tâches d’importation récentes** sur la page d’accueil du **Data Import Wizard** pour suivre l’état de votre importation.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Salesforce.
