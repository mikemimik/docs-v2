title: Exporter des données utilisateur vers Salesforce
---

Dans cet article, vous apprendrez comment exporter des données utilisateur depuis Auth0 dans un fichier CSV, puis les importer dans Salesforce à l’aide de l’Assistant d’importation de données. Pour consulter la configuration système requise, visitez l’article Salesforce [Assistant d’importation de données](https://help.salesforce.com/articleView?id=data_import_wizard.htm).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez la **User Import / Export Extension**. Dans la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez le **Format d’exportation** sur le format de fichier requis. Salesforce accepte l’importation de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name      |
| ------------- | ---------------- |
| `email`       | Adresse courriel |
| `created_at`  | Date de création |
| `given_name`  | Prénom           |
| `family_name` | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Certains types de compte peuvent nécessiter des étapes supplémentaires pour que l’importation réussisse. Prenez un moment pour consulter l’article Salesforce [Prepare Your Data for Import](https://help.salesforce.com/articleView?id=import_prepare.htm). Pour plus d’instructions, consultez [Salesforce: Import Data with the Data Import Wizard](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm).

Avant de commencer, assurez-vous que votre compte Salesforce possède les [permissions utilisateur](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm) requises pour importer des enregistrements.

Pour importer votre fichier CSV dans Salesforce, suivez ces étapes :

1. Connectez-vous à [Salesforce](https://login.salesforce.com/).
2. Accédez à **Configuration** et ouvrez l’**Assistant d’importation de données**.
3. Cliquez sur **Lancer l’assistant**.
4. Configurez l’importation et sélectionnez le fichier CSV à téléverser à l’aide des champs prévus à cet effet.
5. Cliquez sur **Suivant**.
6. À la page **Modifier le mappage des champs**, mappez les champs de données de votre fichier CSV aux champs de données Salesforce. Cliquez sur **Suivant** pour continuer.
7. Après avoir vérifié les renseignements à la page **Vérifier**, cliquez sur **Démarrer l’importation**.
8. Consultez l’onglet **Tâches d’importation récentes** sur la page d’accueil de l’**Assistant d’importation de données** pour suivre l’état de votre importation.

C’est tout ! Vous avez importé avec succès vos utilisateurs Auth0 dans Salesforce.
