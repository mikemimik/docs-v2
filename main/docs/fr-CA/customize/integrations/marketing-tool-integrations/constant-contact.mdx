title: Exporter les données utilisateur vers Constant Contact
---

Dans cet article, vous apprendrez comment exporter les données utilisateur d’Auth0 vers un fichier CSV, puis les importer dans le tableau de bord de Constant Contact.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** au format de fichier requis. Constant Contact accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name      |
| ------------- | ---------------- |
| `email`       | Adresse courriel |
| `created_at`  | Date de création |
| `given_name`  | Prénom           |
| `family_name` | Nom de famille   |

Après avoir ajouté les attributs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Pour importer votre fichier CSV dans Constant Contact, suivez les étapes suivantes :

1. [Connectez-vous à Constant Contact](https://login.constantcontact.com) et accédez à **Contacts**.
2. Dans la page **Contacts**, cliquez sur **Add Contacts &gt; Upload from file**.
3. Dans la page **Upload from file**, téléversez votre fichier CSV, puis cliquez sur **Continuer**.
4. Vérifiez les colonnes et associez-les aux champs de Constant Contact. Une fois terminé, cliquez sur **Continuer**.
5. Dans la page **Select lists**, choisissez la ou les listes auxquelles ajouter vos contacts.
6. Cliquez sur **Upload**.

Et voilà ! Vous avez importé vos utilisateurs Auth0 dans Constant Contact avec succès.
