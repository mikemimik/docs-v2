title: Exporter les données utilisateur vers Constant Contact
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans le tableau de bord de Constant Contact.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par vous rendre dans la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez le **Format d’exportation** sur le format de fichier requis. Constant Contact accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Champs**, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | Adresse courriel |
| `created_at`      | Date de création |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Pour importer votre fichier CSV dans Constant Contact, suivez ces étapes :

1. [Connectez-vous à Constant Contact](https://login.constantcontact.com) et accédez à **Contacts**.
2. Dans la page **Contacts**, cliquez sur **Ajouter des contacts &gt; Téléverser à partir d’un fichier**.
3. Téléversez votre fichier CSV dans la page **Téléverser à partir d’un fichier**, puis cliquez sur **Continuer**.
4. Vérifiez les colonnes et associez-les aux champs de Constant Contact. Une fois terminé, cliquez sur **Continuer**.
5. Dans la page **Sélectionner des listes**, choisissez la ou les listes auxquelles vous voulez ajouter vos contacts.
6. Cliquez sur **Téléverser**.

C’est tout ! Vous avez importé avec succès vos utilisateurs d’Auth0 dans Constant Contact.
