title: Exporter des données utilisateur vers Salesforce Marketing Cloud
---

Dans cet article, vous apprendrez à exporter des données utilisateur d’Auth0 vers un fichier CSV, puis à les importer dans Salesforce Marketing Cloud à l’aide d’Email Studio.

Pour en savoir plus, consultez l’article de Salesforce [Liste de vérification pour bien démarrer avec Email Studio](https://help.salesforce.com/s/articleView?id=mktg.mc_es_get_started_checklist_for_email_studio.htm\&type=5).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez le **Export Format** sur le format de fichier requis. Salesforce Marketing Cloud accepte l’importation de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name      |
| ------------- | ---------------- |
| `email`       | adresse courriel |
| `created_at`  | Date de création |
| `given_name`  | Prénom           |
| `family_name` | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, assurez-vous que votre compte Salesforce dispose des autorisations utilisateur requises pour importer des enregistrements. Prenez aussi un moment pour consulter les étapes de l’[article Salesforce sur l’importation d’abonnés](https://help.salesforce.com/s/articleView?id=mktg.mc_es_import_subscribers_from_file.htm\&type=5) ainsi que [De quelles autorisations ai-je besoin pour importer des enregistrements](https://help.salesforce.com/s/articleView?id=xcloud.faq_import_general_permissions.htm\&type=5).

Pour importer votre fichier CSV dans Salesforce Marketing Cloud, suivez ces étapes :

1. Connectez-vous à Salesforce Marketing Cloud et ouvrez **Email Studio**.
2. Accédez à **Subscribers &gt; Lists**.
3. Choisissez la liste dans laquelle importer les données et sélectionnez l’action **Import**.
4. Après l’écran d’introduction de l’assistant d’importation, sélectionnez votre fichier CSV comme **Upload Source** et CSV comme **Data Format**. Saisissez les autres paramètres, puis cliquez sur **Next**.
5. Dans la boîte de dialogue **Map Attributes**, faites correspondre les champs de données de votre fichier CSV aux bons champs de données. Une fois vos correspondances terminées, cliquez sur **Next**.
6. Après avoir vérifié vos correspondances dans la boîte de dialogue **Confirmed Mappings**, cliquez sur **Begin** pour démarrer l’importation.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Salesforce Marketing Cloud.
