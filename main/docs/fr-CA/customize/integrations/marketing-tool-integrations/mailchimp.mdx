title: Exporter les données utilisateur vers MailChimp
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans [MailChimp](https://login.mailchimp.com/).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, définissez le **Format d’exportation** selon le format de fichier requis. MailChimp accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

Pour en savoir plus, consultez [Base de connaissances MailChimp : directives de format pour votre fichier d’importation](https://kb.mailchimp.com/lists/growth/format-guidelines-for-your-import-file).

En haut, dans la section **Champs**, indiquez un **Champ utilisateur** et un **Nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Pour MailChimp, un champ de courriel avec le nom de colonne `Email Address` est requis; assurez-vous donc de l’inclure. Par exemple :

| Champ utilisateur | Nom de colonne |
| ----------------- | -------------- |
| `email`           | Email Address  |
| `created_at`      | Created At     |
| `given_name`      | First Name     |
| `family_name`     | Last Name      |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, vous trouverez plus d’information ici : [MailChimp Knowledge Base: Import Subscribers to a List](https://kb.mailchimp.com/lists/growth/import-subscribers-to-a-list).

Connectez-vous à votre compte MailChimp et accédez à la page **Lists**. Sélectionnez une liste dans laquelle importer vos utilisateurs Auth0 ([ou créez une nouvelle liste](https://kb.mailchimp.com/lists/growth/create-a-new-list)).

1. Sur votre page List dans MailChimp, cliquez sur **Import Contacts** dans le menu **Add Contacts**.
2. Sélectionnez l’option `CSV or tab-delimited text file` pour importer des contacts.
3. Téléversez le fichier CSV de données utilisateur que vous avez exporté depuis Auth0 dans la section précédente. MailChimp interprétera vos données utilisateur à la page suivante.
4. Vérifiez que les noms de colonne et les types de champ sont corrects.
5. Cliquez sur **Next**.
6. Passez vos sélections en revue et définissez la catégorie d’importation.
7. Cliquez sur **Import** pour lancer l’importation des utilisateurs.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans MailChimp.
