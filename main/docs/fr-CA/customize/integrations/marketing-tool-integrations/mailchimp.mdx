title: Exporter des données utilisateur vers MailChimp
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans le [tableau de bord de MailChimp](https://login.mailchimp.com/).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez la **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, définissez le **Export Format** selon le format de fichier requis. MailChimp accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

Pour en savoir plus, consultez [MailChimp Knowledge Base: Format Guidelines for Your Import File](https://kb.mailchimp.com/lists/growth/format-guidelines-for-your-import-file).

En haut, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Pour MailChimp, un champ d’adresse courriel avec le nom de colonne `Email Address` est requis; assurez-vous donc de l’inclure. Par exemple :

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, vous pouvez obtenir plus de renseignements en consultant [MailChimp Knowledge Base: Import Subscribers to a List](https://kb.mailchimp.com/lists/growth/import-subscribers-to-a-list).

Connectez-vous à votre compte MailChimp et accédez à la page **Lists**. Sélectionnez une liste dans laquelle importer vos utilisateurs Auth0 ([ou créez-en une nouvelle](https://kb.mailchimp.com/lists/growth/create-a-new-list)).

1. Sur votre page List de MailChimp, cliquez sur **Import Contacts** dans le menu **Add Contacts**.
2. Sélectionnez l’option `CSV or tab-delimited text file` pour importer des contacts.
3. Téléversez le fichier CSV de données utilisateur que vous avez exporté depuis Auth0 à la section précédente. MailChimp interprétera vos données utilisateur à la page suivante.
4. Vérifiez que les noms de colonnes et les types de champs sont corrects.
5. Cliquez sur le bouton **Next**.
6. Vérifiez vos sélections et définissez la catégorie d’importation.
7. Cliquez sur le bouton **Import** pour lancer l’importation des utilisateurs.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans MailChimp.
