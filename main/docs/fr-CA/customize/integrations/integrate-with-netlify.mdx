<div id="overview">
  ## Aperçu
</div>

Connectez une instance Auth0 nouvelle ou existante à votre projet Netlify pour améliorer la sécurité, l’évolutivité et la personnalisation de votre application grâce à l’intégration Auth0. Cette intégration vous permet de mettre en place, directement dans votre application, une approche simplifiée de l’identité client.

Utilisez la nouvelle intégration pour créer et personnaliser le parcours d’identité de votre projet Netlify. Pour mettre votre application en service rapidement et en toute sécurité, cette intégration :

* Configure un environnement de production dans Netlify, relié à votre tenant Auth0, en quelques clics seulement.
* Crée automatiquement une Application dans votre tenant Auth0, connectée à votre projet Netlify.
* Précharge vos identifiants Auth0 dans Netlify, ce qui vous permet d’éviter certaines étapes manuelles du processus.

<div id="prerequisites">
  ## Prérequis
</div>

Avant de commencer à intégrer Auth0 à votre projet Netlify, vous devez mettre en place quelques éléments :

* Un compte Netlify avec un projet que vous souhaitez intégrer. Consultez [Créer votre compte Netlify](https://www.netlify.com/) pour créer un compte.
* Une application Web qui nécessite une authentification, comme une application React, Vue ou un site statique avec des fonctions sans serveur.

<div id="add-a-netlify-integration">
  ## Ajouter une intégration Netlify
</div>

Après avoir créé votre compte Netlify et ouvert une session dans votre tableau de bord, vous pouvez commencer à ajouter une intégration. Pour ajouter l’intégration Auth0 à un projet Netlify existant :

1. Dans le tableau de bord Netlify, accédez à **Extensions &gt; Directory.**
2. Recherchez [**Auth0**](https://app.netlify.com/extensions/auth0) et sélectionnez **Install.**
3. Sélectionnez **Install** pour lancer l’installation dans votre portail d’intégration Auth0.

<div id="connect-your-site">
  ## Connectez votre site
</div>

Pour connecter votre site à Auth0 :

1. Accédez à la **Site Configuration** du site que vous souhaitez connecter à Auth0.
2. Dans la barre latérale, sélectionnez **Auth0** sous **Access &amp; security.**
3. Sélectionnez **Link an Auth0 tenant** et suivez les instructions pour connecter votre compte Auth0. Si vous n’avez pas encore de compte Auth0, vous pouvez vous inscrire à l’adresse https://auth0.com/signup.

   <Frame>![](/docs/images/cdy7uua7fh8z/3wZZNNVMadwZjeLi9vyQtH/01821cd184326a004592f5e0e20a6f72/2025-03-03_12-12-09.png)</Frame>
4. Sélectionnez votre tenant dans la liste déroulante, puis sélectionnez **Accept** pour autoriser votre compte.

   <Frame>![](/docs/images/cdy7uua7fh8z/7mrUvgQh70cgtri09aOHso/a76600ebf472dcddb3977a8b9e726e01/netlify-link.png)</Frame>
5. Une fois votre tenant lié, sélectionnez **Add tenant**.
6. Remplissez les renseignements requis pour votre tenant et votre application, cochez la case **Redeploy site**, puis sélectionnez **Create site tenant**.

   <Frame>![](/docs/images/cdy7uua7fh8z/hPxAU8mwd9035AXOPMqxp/f74d0864a6cc5a4aed963bfc77237d7c/auth0_tenant.png)</Frame>
7. Vous devriez maintenant voir le tenant de site lié avec succès dans la configuration de votre site.

   <Frame>![](/docs/images/cdy7uua7fh8z/1y8khKk4yjGvySB2QRItD3/0f9a5606373c69761ff6fefa4fe180af/netlify-success.png)</Frame>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Vous devriez maintenant voir votre application cliente et votre API dans votre Auth0 Dashboard, sous **Applications**.
</Callout>

<div id="set-up-the-integration">
  ## Configurer l’intégration
</div>

Lorsque vous intégrez Auth0 à votre projet Netlify, vous devez fournir à votre application des paramètres d’authentification comme le **<Tooltip tip="Client ID : valeur d’identification attribuée par Auth0 à votre ressource enregistrée." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Client+ID">Client ID</Tooltip>** et le **domaine Auth0**.

Si votre application lit ces paramètres à partir de variables d’environnement, ils sont injectés automatiquement par l’extension.

1. Accédez à votre tableau de bord Netlify.
2. Accédez à **Site configuration &gt; Environment variables.**
3. Cliquez sur **Add a variable**, puis saisissez :

   * `AUTH0_CLIENT_ID` (votre Auth0 Client ID)
   * `AUTH0_DOMAIN` (votre domaine Auth0)

     <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
       Ces variables, ainsi que tous les autres renseignements sur l’application, se trouvent dans votre Auth0 Dashboard, sous **Applications &gt; [Votre application] &gt; Settings**.
     </Callout>

Vous devriez maintenant avoir configuré les identifiants Auth0 appropriés pour le déploiement.

<div id="populate-a-configuration-file">
  ### Renseigner un fichier de configuration
</div>

Si votre application lit les paramètres Auth0 à partir d’un fichier, vous pouvez automatiser ce processus pendant la phase de build. Ainsi, votre application dispose toujours des autorisations nécessaires sans que vous ayez à modifier les fichiers manuellement.

Pour ce faire :

1. Accédez à votre tableau de bord Netlify.
2. Allez à **Site configuration &gt; Build &amp; deploy &gt; Build settings**.
3. Modifiez la **Build command** pour y inclure un script shell avant de lancer le build :

   ```bash wrap lines
   echo "{\"clientId\": \"${AUTH0_CLIENT_ID}\", \"domain\": \"${AUTH0_DOMAIN}\"}" > auth_config.json && npm run build
   ```

Ce script crée un fichier JSON nommé `auth_config.json` et le renseigne à l’aide des variables d’environnement `AUTH0_CLIENT_ID` et `AUTH0_DOMAIN` définies ci-dessus, afin que l’application dispose des bons paramètres Auth0 dans l’environnement d’exécution.

<div id="manage-integration">
  ## Gérer l’intégration
</div>

Après avoir ajouté l’intégration Auth0 à votre projet Netlify, l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> crée une nouvelle application pour gérer et personnaliser votre workflow CIAM.

1. Accédez à l’Auth0 Dashboard.
2. Sélectionnez **Applications &gt; Applications &gt; [Votre application Netlify]**
3. Confirmez l’URI de l’application Netlify sous **Settings &gt; Application URIs**.

Vous pouvez maintenant personnaliser et modifier cette application comme toute autre application dans votre Auth0 Dashboard.
