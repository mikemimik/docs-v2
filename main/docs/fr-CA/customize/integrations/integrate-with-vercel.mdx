L’intégration native d’Auth0 avec Vercel vous permet de connecter votre plateforme d’identité à votre projet Vercel avec un minimum de configuration. Configurez et personnalisez votre interface de connexion sans avoir à partir de zéro. Ensuite, étendez et faites évoluer vos flux d’authentification et d’autorisation grâce à des fonctionnalités avancées, notamment [Organizations B2B](/docs/fr-CA/manage-users/organizations), [autorisation fine (FGA)](https://docs.fga.dev/), [Auth0 for AI Agents](https://auth0.com/ai/docs), et plus encore.

L’intégration native d’Auth0 avec Vercel offre notamment les avantages suivants :

* Le provisionnement automatique d’une application Auth0 connectée à votre projet Vercel.
* Le préchargement de vos informations d’identification Auth0 dans le Vercel Dashboard pour une configuration simple et rapide.
* Une prise en charge prête à l’emploi de vos applications Next.js à l’aide des [Auth0 SDKs](/docs/fr-CA/libraries).

L’intégration native d’Auth0 avec Vercel crée un nouvel environnement de tenant Auth0 pour votre projet Vercel. Si vous souhaitez utiliser un compte Auth0 existant, intégrez Vercel manuellement lors de la création d’une application dans l’Auth0 Dashboard.

<div id="prerequisites">
  ## Prérequis
</div>

Avant de commencer, assurez-vous d’avoir :

* Un compte Vercel avec un projet actif. Au besoin, [créez un compte Vercel](https://vercel.com/signup)
* Une application Next.js utilisant la plus récente version du [Auth0 Next.js SDK](/docs/fr-CA/quickstart/webapp/nextjs)

Après avoir installé l’intégration, [activez l’Iframe embedding](/docs/fr-CA/customize/login-pages/manage-core-configuration#iframe-embedding) pour permettre à Universal Login ou à Classic Login de se charger dans un iframe, comme l’exige Vercel.

<div id="install-integration">
  ## Installer l’intégration
</div>

Installez l’intégration native d’Auth0 avec Vercel à partir du Marketplace de Vercel :

1. Accédez à **Integrations &gt; Browse Marketplace** et recherchez Auth0. Sélectionnez l’**intégration Auth0** sous **Native Integrations**, puis sélectionnez **Install**.

2. Auth0 renseigne automatiquement l’URL localhost et l’URL de rappel de votre application. Sélectionnez votre **Installation Plan**, puis sélectionnez **Continue**.

3. Entrez un nom pour votre application. Sélectionnez **Create**. Une fois le nouveau tenant Auth0 et l’application créés avec succès, sélectionnez **Continue**.

4. Pour vous connecter à un projet Vercel existant, sélectionnez **Connect Project**, puis sélectionnez votre projet et vos environnements. Entrez un préfixe personnalisé à ajouter à vos variables d’environnement, au besoin. Sélectionnez **Connect**. Une fois votre projet Vercel connecté avec succès, vous devriez être redirigé vers la section [Pour commencer](#getting-started) de l’intégration.

<div id="getting-started">
  ## Pour commencer
</div>

La section Pour commencer du Vercel Dashboard affiche un Quickstart pour mettre votre application en service rapidement et en toute sécurité. Vous pouvez aussi gérer les paramètres de votre application dans le menu de navigation de gauche.

<div id="quickstart">
  ### Quickstart
</div>

Pour commencer à utiliser l’intégration native d’Auth0 avec Vercel, suivez les instructions dans le Vercel Dashboard pour créer et déployer votre application.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Vous pouvez uniquement configurer les variables d’environnement d’Auth0 dans un environnement de production.
</Callout>

<div id="manage-application-settings">
  ## Gérer les paramètres de l’application
</div>

Vous pouvez gérer les paramètres de votre application dans le [Vercel Dashboard](#vercel-dashboard) et l’[Auth0 Dashboard](#auth0-dashboard).

<div id="vercel-dashboard">
  ### Vercel Dashboard
</div>

Pour gérer les paramètres de votre application dans le Vercel Dashboard, accédez à **Settings** dans le menu de navigation de gauche.

Vous pouvez mettre à jour les paramètres d’application suivants :

* **Update Name** : Mettez à jour le nom de l’application.
* **Update Configuration** :
  * **Parameters** : Mettez à jour l’URL localhost et le point de terminaison de rappel de votre application.
  * **Current Installation Level Plan** : Accédez aux paramètres de l’intégration pour mettre à jour le forfait au niveau de l’installation de votre intégration. Pour en savoir plus, consultez [Settings](#settings).
* **Secure this resource**
  * **Rotate Secrets** : Effectuez la rotation des secrets de votre application. Pour en savoir plus, consultez [Rotating environment variables](https://vercel.com/docs/projects/environment-variables/managing-environment-variables#rotating-environment-variables).
  * **Allowed Environments** : Sélectionnez si l’intégration est offerte dans **All environments** ou **Production environment only**.
* **Supprimer l’intégration** : Lorsque vous supprimez l’intégration, vous êtes automatiquement rétrogradé vers le forfait Free de Vercel. Pour en savoir plus, consultez [Delete integration](#delete-integration).

<div id="auth0-dashboard">
  ### Auth0 Dashboard
</div>

Gérez d’autres paramètres d’application dans le [Auth0 Dashboard](https://manage.auth0.com/).

Lorsque vous installez l’intégration native, Auth0 crée une nouvelle application dans un tenant dédié, préremplie avec vos URI Vercel, vos URI de rappel autorisés et vos URL de déconnexion.

Vous pouvez modifier cette application, par exemple en personnalisant votre expérience Universal Login, comme vous le feriez pour n’importe quelle autre application dans l’Auth0 Dashboard.

<div id="manage-integration">
  ## Gérer l’intégration
</div>

Pour gérer l’intégration depuis votre Vercel Dashboard, sélectionnez l’onglet **Integration** de votre projet, faites défiler la page jusqu’à l’intégration Auth0, puis sélectionnez **Manage**.

<div id="invoices">
  ### Factures
</div>

Sur la page **Factures**, consultez la liste des factures liées à votre intégration Auth0. Pour en savoir plus sur les factures, consultez [Billing &amp; Invoices](https://vercel.com/docs/pricing/understanding-my-invoice).

<div id="settings">
  ### Paramètres
</div>

Sur la page **Paramètres**, ajoutez des méthodes de paiement, changez le forfait au niveau de l’installation et supprimez l’intégration.

<div id="payment-methods">
  #### Modes de paiement
</div>

Pour ajouter un mode de paiement, sélectionnez **Ajouter**. Pour en savoir plus sur les modes de paiement, consultez [les tarifs de Vercel](https://vercel.com/docs/accounts/plans).

<div id="current-installation-level-plan">
  #### Forfait actuel au niveau de l’installation
</div>

Le forfait défini au niveau de l’installation s’applique à tous les produits de l’installation.

Pour modifier le forfait au niveau de l’installation :

1. Sous **Paramètres**, faites défiler jusqu’à **Forfait actuel au niveau de l’installation** et sélectionnez **Changer de forfait**.
2. Sélectionnez le forfait au niveau de l’installation souhaité.
3. Sélectionnez **Continuer**.
4. Vérifiez votre sélection, puis sélectionnez **Mettre à jour**.
5. Une fois l’installation mise à jour, sélectionnez **Terminé**.

<div id="delete-integration">
  #### Supprimer l’intégration
</div>

Lorsque vous supprimez l’intégration, votre compte est automatiquement rétrogradé vers le forfait Free de Vercel.

Pour supprimer l’intégration :

1. Sous **Settings**, faites défiler l’écran jusqu’à **Delete &lt;YOUR&#95;PRODUCT&gt;** et sélectionnez le bouton **Remove &lt;YOUR&#95;PRODUCT&gt;**.
2. Saisissez le slug de l’intégration et votre équipe, puis sélectionnez **I acknowledge that this will remove my Auth0-connected account**.
3. Sélectionnez **Remove &lt;YOUR&#95;PRODUCT&gt;**.
