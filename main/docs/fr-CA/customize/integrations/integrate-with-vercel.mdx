L’intégration native d’Auth0 avec Vercel vous permet de connecter votre plateforme d’identité à votre projet Vercel avec un minimum de configuration. Configurez et personnalisez votre interface de connexion sans avoir à la créer à partir de zéro. Ensuite, étendez et faites évoluer vos flux d’authentification et d’autorisation grâce à des fonctionnalités avancées, notamment les [organisations B2B](/docs/fr-CA/manage-users/organizations), l’[autorisation fine (FGA)](https://docs.fga.dev/), [Auth0 for AI Agents](https://auth0.com/ai/docs), et plus encore.

L’intégration native d’Auth0 avec Vercel permet :

* Le provisionnement automatique d’une application Auth0 connectée à votre projet Vercel.
* Le préchargement de vos identifiants Auth0 dans le Vercel Dashboard pour une configuration simple et rapide.
* Une prise en charge prête à l’emploi de vos applications Next.js au moyen des [Auth0 SDKs](/docs/fr-CA/libraries).

L’intégration native d’Auth0 avec Vercel crée un nouvel environnement de tenant Auth0 pour votre projet Vercel. Si vous souhaitez utiliser un compte Auth0 existant, effectuez l’intégration à Vercel manuellement lors de la création d’une application dans l’Auth0 Dashboard.

<div id="prerequisites">
  ## Prérequis
</div>

Avant de commencer, assurez-vous d’avoir :

* Un compte Vercel avec un projet actif. Au besoin, [créez un compte Vercel](https://vercel.com/signup)
* Une application Next.js utilisant la plus récente version du [Auth0 Next.js SDK](/docs/fr-CA/quickstart/webapp/nextjs)

Après avoir installé l’intégration, [activez Iframe embedding](/docs/fr-CA/customize/login-pages/manage-core-configuration#iframe-embedding) pour permettre à Universal Login ou à Classic Login de se charger dans un iframe, comme l’exige Vercel.

<div id="install-integration">
  ## Installer l’intégration
</div>

Installez l’intégration native d’Auth0 avec Vercel depuis le Marketplace de Vercel :

1. Accédez à **Intégrations &gt; Parcourir le Marketplace** et recherchez Auth0. Sélectionnez **l’intégration Auth0** sous **Intégrations natives**, puis sélectionnez **Installer**.

2. Auth0 préremplit automatiquement l’URL localhost et l’URL de rappel de votre application. Sélectionnez votre **forfait d’installation**, puis sélectionnez **Continuer**.

3. Saisissez un nom pour votre application. Sélectionnez **Créer**. Une fois le nouveau tenant Auth0 et l’application créés avec succès, sélectionnez **Continuer**.

4. Pour vous connecter à un projet Vercel existant, sélectionnez **Connect Project**, puis choisissez votre projet et vos environnements. Saisissez un préfixe personnalisé à ajouter à vos variables d’environnement, au besoin. Sélectionnez **Connect**. Une fois le projet Vercel connecté avec succès, vous devriez être redirigé vers la section [Pour commencer](#getting-started) de l’intégration.

<div id="getting-started">
  ## Pour commencer
</div>

La section Pour commencer du Vercel Dashboard affiche un Quickstart pour lancer votre application rapidement et en toute sécurité. Vous pouvez aussi gérer les paramètres de votre application dans le menu de navigation de gauche.

<div id="quickstart">
  ### Quickstart
</div>

Pour commencer à utiliser l’intégration native d’Auth0 avec Vercel, suivez les instructions du Vercel Dashboard pour créer et déployer votre application.

<div id="manage-application-settings">
  ## Gérer les paramètres de l’application
</div>

Vous pouvez gérer les paramètres de votre application dans le [Vercel Dashboard](#vercel-dashboard) et l’[Auth0 Dashboard](#auth0-dashboard).

<div id="vercel-dashboard">
  ### Vercel Dashboard
</div>

Pour gérer les paramètres de votre application dans le Vercel Dashboard, accédez à **Settings** dans le menu de navigation de gauche.

Vous pouvez mettre à jour les paramètres d’application suivants :

* **Mettre à jour le nom** : Mettez à jour le nom de l’application.
* **Mettre à jour la Configuration** :
  * **Parameters** : Mettez à jour l’URL localhost et l’endpoint de callback de votre application.
  * **forfait actuel au niveau de l’installation** : Accédez aux paramètres de l’intégration pour mettre à jour le forfait au niveau de l’installation de votre intégration. Pour en savoir plus, consultez [Settings](#settings).
* **Sécuriser cette ressource**
  * **Effectuer la rotation des secrets** : Effectuez la rotation des secrets de votre application. Pour en savoir plus, consultez [Rotation des variables d’environnement](https://vercel.com/docs/projects/environment-variables/managing-environment-variables#rotating-environment-variables).
  * **Environnements autorisés** : Sélectionnez si l’intégration est offerte dans **Tous les environnements** ou **l’environnement Production seulement**.
* **Supprimer l’intégration** : Lorsque vous supprimez l’intégration, vous êtes automatiquement rétrogradé vers un forfait Free de Vercel. Pour en savoir plus, consultez [Supprimer l’intégration](#delete-integration).

<div id="auth0-dashboard">
  ### Auth0 Dashboard
</div>

Gérez des paramètres d’application supplémentaires dans le [Auth0 Dashboard](https://manage.auth0.com/).

Lorsque vous installez l’intégration native, Auth0 crée une nouvelle application dans un tenant dédié, préconfigurée avec vos URI Vercel, vos URI de rappel autorisées et vos URL de déconnexion.

Vous pouvez modifier cette application, par exemple en personnalisant votre expérience Universal Login, comme vous le feriez pour n’importe quelle autre application dans votre tableau de bord Auth0.

<div id="manage-integration">
  ## Gérer l’intégration
</div>

Pour gérer l’intégration depuis votre Vercel Dashboard, sélectionnez l’onglet **Integration** de votre projet, faites défiler la page jusqu’à l’intégration Auth0, puis sélectionnez **Manage**.

<div id="invoices">
  ### Factures
</div>

Sur la page **Factures**, consultez la liste des factures liées à votre intégration Auth0. Pour en savoir plus sur les factures, consultez [Billing &amp; Invoices](https://vercel.com/docs/pricing/understanding-my-invoice).

<div id="settings">
  ### Paramètres
</div>

Sur la page **Paramètres**, ajoutez des modes de paiement, modifiez le forfait de l’installation et supprimez l’intégration.

<div id="payment-methods">
  #### Modes de paiement
</div>

Pour ajouter un mode de paiement, sélectionnez **Add**. Pour en savoir plus sur les modes de paiement, consultez [les tarifs de Vercel](https://vercel.com/docs/accounts/plans).

<div id="current-installation-level-plan">
  #### Forfait actuel au niveau de l’installation
</div>

Le forfait au niveau de l’installation s’applique à tous les produits de l’installation.

Pour modifier le forfait au niveau de l’installation :

1. Sous **Paramètres**, faites défiler l’écran jusqu’à **Forfait actuel au niveau de l’installation**, puis sélectionnez **Changer de forfait**.
2. Sélectionnez le forfait au niveau de l’installation souhaité.
3. Sélectionnez **Continuer**.
4. Vérifiez votre sélection, puis sélectionnez **Mettre à jour**.
5. Une fois l’installation mise à jour, sélectionnez **Terminé**.

<div id="delete-integration">
  #### Supprimer l’intégration
</div>

Lorsque vous supprimez l’intégration, votre compte est automatiquement rétrogradé au forfait Free de Vercel.

Pour supprimer l’intégration :

1. Sous **Settings**, faites défiler la page jusqu’à **Delete &lt;YOUR&#95;PRODUCT&gt;** et cliquez sur le bouton **Remove &lt;YOUR&#95;PRODUCT&gt;**.
2. Entrez le slug de l’intégration et votre équipe, puis sélectionnez **Je reconnais que cette action supprimera mon compte connecté à Auth0**.
3. Sélectionnez **Remove &lt;YOUR&#95;PRODUCT&gt;**.
