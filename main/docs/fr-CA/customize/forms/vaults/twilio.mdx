<div id="get-your-twilio-credentials">
  ## Récupérez vos identifiants Twilio
</div>

Vous les trouverez dans [le tableau de bord de la console Twilio](https://console.twilio.com/).

<div id="configure-your-twilio-vault-connection">
  ## Configurez votre connexion Vault Twilio
</div>

Pour configurer une connexion Vault Twilio pour vos Actions de Flow Twilio, vous devez renseigner les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre   | Description                                                 |
| ----------- | ----------------------------------------------------------- |
| Account SID | Account SID pour l’authentification auprès de l’API Twilio. |
| Auth token  | Auth token pour l’authentification auprès de l’API Twilio.  |
