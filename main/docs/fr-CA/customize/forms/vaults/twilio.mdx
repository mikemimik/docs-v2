<div id="get-your-twilio-credentials">
  ## Récupérez vos identifiants Twilio
</div>

Vous les trouverez dans le [tableau de bord de la console Twilio](https://console.twilio.com/).

<div id="configure-your-twilio-vault-connection">
  ## Configurer votre connexion Vault Twilio
</div>

Pour configurer une connexion Vault Twilio pour vos actions de Flow Twilio, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre   | Description                                                  |
| ----------- | ------------------------------------------------------------ |
| Account SID | Account SID servant à autoriser les requêtes à l’API Twilio. |
| Auth token  | Auth token servant à autoriser les requêtes à l’API Twilio.  |
