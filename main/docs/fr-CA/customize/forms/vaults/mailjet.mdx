<div id="get-your-mailjet-keys">
  ## Récupérez vos clés Mailjet
</div>

Vous les trouverez sur la page Compte du [tableau de bord Mailjet](https://app.mailjet.com/account/api_keys), sous « Clés API ».

<div id="configure-your-mailjet-vault-connection">
  ## Configurer votre connexion Vault Mailjet
</div>

Pour configurer une connexion Vault Mailjet pour vos actions de Flow Mailjet, vous devez indiquer les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre   | Description                                          |
| ----------- | ---------------------------------------------------- |
| Clé API     | Clé API pour l’authentification à l’API Mailjet.     |
| Clé secrète | Clé secrète pour l’authentification à l’API Mailjet. |
