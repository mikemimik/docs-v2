<div id="get-your-mailjet-keys">
  ## Obtenez vos clés MailJet
</div>

Vous les trouverez sur la page [Compte du tableau de bord de MailJet](https://app.mailjet.com/account/api_keys), sous « clés API ».

<div id="configure-your-mailjet-vault-connection">
  ## Configurer votre connexion Vault MailJet
</div>

Pour configurer une connexion Vault MailJet pour vos actions de Flow MailJet, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre   | Description                                          |
| ----------- | ---------------------------------------------------- |
| API Key     | API Key pour l’authentification à l’API MailJet.     |
| Clé secrète | Clé secrète pour l’authentification à l’API MailJet. |
