<div id="get-access-to-auth0-management-api">
  ## Accéder à l’Auth0 Management API
</div>

[Créer une application machine-to-machine (M2M)](/docs/fr-CA/get-started/auth0-overview/create-applications/machine-to-machine-apps) autorisée à utiliser l’Auth0 Management API.

<Warning>
  N’oubliez pas d’autoriser votre application M2M à accéder à l’Auth0 Management API et de sélectionner les bons scopes, conformément au principe du moindre privilège :

  * For Get User: `read:users`
  * For Create User: `create:users`
  * For Update User: `update:users`
</Warning>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Aucune autorisation de l’Auth0 Management API n’est requise pour envoyer des notifications au moyen des providers configurés dans votre tenant Auth0.
</Callout>

<div id="configure-your-vault-connection">
  ## Configurez votre connexion Vault
</div>

Pour configurer une Auth0 Vault Connection pour vos actions de flux Auth0, définissez les paramètres requis en fonction de la application M2M créée précédemment.

<Frame>![Dashboard > Actions > Forms > Vault > Auth0](/docs/images/customize/forms/vaults/auth0-vault-connection.png)</Frame>

<div id="input-settings">
  #### Paramètres d’entrée
</div>

| Paramètre              | Description                                                                         |
| ---------------------- | ----------------------------------------------------------------------------------- |
| Domaine du tenant      | Domaine du tenant de votre application M2M.                                         |
| Client ID              | Client ID de votre application M2M autorisée à utiliser l’Auth0 Management API.     |
| Client Secret          | Client Secret de votre application M2M autorisée à utiliser l’Auth0 Management API. |
| Audience (facultative) | Audience de l’Auth0 Management API.                                                 |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Par défaut, l’Audience sera déterminée à partir du Domaine du tenant.
</Callout>
