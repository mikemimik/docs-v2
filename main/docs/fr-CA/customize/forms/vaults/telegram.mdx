  ## Obtenez votre token Telegram
</div>

Découvrez comment générer un token dans [l’API bot de Telegram](https://core.telegram.org/#bot-api).

<div id="configure-your-telegram-vault-connection">
  ## Configurer votre connexion Vault Telegram
</div>

Pour configurer une connexion Vault Telegram pour vos actions de Flow Telegram, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/telegram-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre | Description                                                |
| --------- | ---------------------------------------------------------- |
| Token     | Token généré pour l’autorisation auprès de l’API Telegram. |
