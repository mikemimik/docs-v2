<div id="get-your-sendgrid-keys">
  ## Obtenez vos clés SendGrid
</div>

Trouvez-les sur la page Settings du [SendGrid dashboard](https://app.sendgrid.com/settings/api_keys), dans la section API keys.

<div id="configure-your-sendgrid-vault-connection">
  ## Configurez votre connexion Vault SendGrid
</div>

Pour configurer une connexion Vault SendGrid pour vos Flow Actions SendGrid, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre | Description                                       |
| --------- | ------------------------------------------------- |
| API Key   | API Key pour l’authentification à l’API SendGrid. |
