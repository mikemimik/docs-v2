<div id="get-your-sendgrid-keys">
  ## Obtenez vos clés SendGrid
</div>

Vous les trouverez sur la page Paramètres [du SendGrid dashboard](https://app.sendgrid.com/settings/api_keys), sous Clés API.

<div id="configure-your-sendgrid-vault-connection">
  ## Configurer votre connexion Vault de SendGrid
</div>

Pour configurer une connexion Vault de SendGrid pour vos Flow Actions SendGrid, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre | Description                                       |
| --------- | ------------------------------------------------- |
| Clé API   | Clé API pour l’authentification à l’API SendGrid. |
