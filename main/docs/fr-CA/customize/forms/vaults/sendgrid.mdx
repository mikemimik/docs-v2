<div id="get-your-sendgrid-keys">
  ## Récupérez vos clés SendGrid
</div>

Vous les trouverez dans la page Settings du [SendGrid dashboard](https://app.sendgrid.com/settings/api_keys), à la section API keys.

<div id="configure-your-sendgrid-vault-connection">
  ## Configurer votre connexion Vault SendGrid
</div>

Pour configurer une connexion Vault SendGrid pour vos actions de Flow de SendGrid, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre | Description                                               |
| --------- | --------------------------------------------------------- |
| API Key   | API Key utilisée pour autoriser l’accès à l’API SendGrid. |
