<div id="configure-your-jwt-vault-connection">
  ## Configurez votre connexion Vault JWT
</div>

Pour configurer une connexion Vault JWT afin que vos actions JWT dans Flows puissent signer, décoder et vérifier des JSON Web Tokens, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > JWT](/docs/images/customize/forms/vaults/jwt-vault-connection.png)</Frame>

<div id="input-settings">
  ## Paramètres d’entrée
</div>

| Paramètre | Description                                                 |
| --------- | ----------------------------------------------------------- |
| Secret    | Secret utilisé avec l’algorithme HS256 pour encoder le JWT. |
