<div id="http-request">
  ## HTTP Request
</div>

L’action HTTP Request vous permet d’envoyer une requête à une API ou à un service externe et d’en traiter la réponse.

<Frame>![](/docs/images/cdy7uua7fh8z/6eIOv07XenyaazzKixigei/2e9efd632137829a8ddb03aacd3197df/http-request-action.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                        | Description                                                                                                                                                            |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Méthode                          | La méthode HTTP. Les valeurs possibles sont : <ul> <li>`GET`</li> <li>`POST`</li> <li>`PUT`</li> <li>`PATCH`</li> <li>`DELETE`</li> </ul>                              |
| Type de contenu                  | L’en-tête `Content-Type`. Les valeurs possibles sont : <ul> <li>`application/json`</li> <li>`application/x-www-form-urlenconded`</li> <li>`application/xml`</li> </ul> |
| Corps                            | Le corps de la requête.                                                                                                                                                |
| En-têtes                         | Les en-têtes de la requête.                                                                                                                                            |
| Paramètres d’URL                 | Les paramètres de l’URL. Ces paramètres seront automatiquement encodés dans l’URL.                                                                                     |
| Activer l’authentification Basic | Active l’authentification Basic à l’aide d’un nom d’utilisateur et d’un mot de passe.                                                                                  |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type    | Description                                                 |
| --------- | ------- | ----------------------------------------------------------- |
| `headers` | Objet   | Les en-têtes de la réponse.                                 |
| `body`    | Objet   | Le corps de la réponse.                                     |
| `status`  | Nombre  | Le code d’état de la réponse.                               |
| `success` | Booléen | Retourne `true` ou `false` selon le `status` de la réponse. |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "headers": {
    ...
  },
  "status": 200,
  "body": {
    ...
  },
  "success": true
}
```
