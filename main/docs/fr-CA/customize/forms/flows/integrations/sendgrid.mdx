<div id="send-email">
  ## Envoyer un courriel
</div>

Envoyez un courriel avec SendGrid comme fournisseur de messagerie.

<Frame>![](/docs/images/cdy7uua7fh8z/3pT7MZCTVhcL9ueeke0HqQ/8583aedc27af021fe6a62430c6d59adf/sendGrid.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                              | Description                                                                                                                  |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| Nom de l’expéditeur                    | Le nom de l’expéditeur                                                                                                       |
| Courriel de l’expéditeur (obligatoire) | Le courriel de l’expéditeur                                                                                                  |
| Nom du destinataire                    | Le nom du destinataire                                                                                                       |
| Courriel du destinataire (obligatoire) | Le courriel du destinataire                                                                                                  |
| Objet (obligatoire)                    | L’objet du courriel                                                                                                          |
| Type de courriel (obligatoire)         | Envoyer un courriel en HTML ou à l’aide d’un modèle enregistré (facultatif)                                                  |
| Message (obligatoire)                  | Le message du courriel au format HTML ou un modèle (facultatif).                                                             |
| ID du modèle (facultatif)              | Si vous utilisez un modèle, vous trouverez son ID dans le [ SendGrid dashboard ](https://mc.sendgrid.com/dynamic-templates). |
| Variables personnalisées (facultatif)  | Ajoutez des variables personnalisées à insérer dans votre modèle.                                                            |

<div id="output-object">
  ### Objet de sortie
</div>

Vous pouvez consulter la réponse attendue et les codes d’état dans la [documentation de SendGrid](https://www.twilio.com/docs/sendgrid/api-reference/mail-send/mail-send).
