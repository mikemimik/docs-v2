<div id="send-email">
  ## Envoyer un courriel
</div>

Envoyez un courriel avec SendGrid comme fournisseur de courriel.

<Frame>![](/docs/images/cdy7uua7fh8z/3pT7MZCTVhcL9ueeke0HqQ/8583aedc27af021fe6a62430c6d59adf/sendGrid.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                             | Description                                                                                                             |
| ------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| Nom de l’expéditeur                   | Le nom de l’expéditeur                                                                                                  |
| Courriel de l’expéditeur (requis)     | Le courriel de l’expéditeur                                                                                             |
| Nom du destinataire                   | Le nom du destinataire                                                                                                  |
| Courriel du destinataire (requis)     | Le courriel du destinataire                                                                                             |
| Objet (requis)                        | L’objet du courriel                                                                                                     |
| Type de courriel (requis)             | Envoyer un courriel en HTML ou à l’aide d’un modèle enregistré (facultatif)                                             |
| Message (requis)                      | Message du courriel en format HTML ou modèle (facultatif).                                                              |
| ID du modèle (facultatif)             | Si vous utilisez un modèle, son ID se trouve dans le [ SendGrid dashboard ](https://mc.sendgrid.com/dynamic-templates). |
| Variables personnalisées (facultatif) | Ajoutez des variables personnalisées à insérer dans votre modèle.                                                       |

<div id="output-object">
  ### Objet de sortie
</div>

La réponse attendue et les codes d’état se trouvent dans [la documentation de SendGrid](https://www.twilio.com/docs/sendgrid/api-reference/mail-send/mail-send).
