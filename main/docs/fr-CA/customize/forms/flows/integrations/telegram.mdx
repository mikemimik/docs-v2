<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                | Description                                                                                         |
| ------------------------ | --------------------------------------------------------------------------------------------------- |
| ID du chat (obligatoire) | L’ID du chat Telegram. À obtenir auprès du [BotFather de Telegram](https://core.telegram.org/bots). |
| Messages (obligatoire)   | Le contenu du message Telegram.                                                                     |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez la réponse attendue et les codes d’état dans [la documentation de Telegram](https://core.telegram.org/method/messages.sendMessage).
