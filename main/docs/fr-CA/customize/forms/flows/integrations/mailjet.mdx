<div id="send-email">
  ## Envoyer un courriel
</div>

Envoyez un courriel avec Mailjet comme fournisseur de courriel.

<Frame>![](/docs/images/cdy7uua7fh8z/5z1CqSZTrLs1oMdHKvHOiM/7e1fa3b5d7ea5a5b2ec51dd58fa1d2ed/mailjet.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                              | Description                                                                                                            |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Nom de l’expéditeur                    | Le nom de l’expéditeur                                                                                                 |
| Courriel de l’expéditeur (obligatoire) | Le courriel de l’expéditeur                                                                                            |
| Nom du destinataire                    | Le nom du destinataire                                                                                                 |
| Courriel du destinataire (obligatoire) | Le courriel du destinataire                                                                                            |
| Nom du destinataire en CC              | Le nom du destinataire en CC                                                                                           |
| Courriel en CC                         | Le courriel en CC                                                                                                      |
| Objet                                  | L’objet du courriel                                                                                                    |
| Type de courriel                       | Envoyer le courriel en HTML ou à l’aide d’un modèle enregistré (facultatif)                                            |
| Message (obligatoire)                  | Message du courriel au format HTML ou modèle (facultatif).                                                             |
| ID du modèle (facultatif)              | Si vous utilisez un modèle, son ID se trouve dans le [ tableau de bord de Mailjet ](https://app.mailjet.com/template). |
| Variables personnalisées (facultatif)  | Ajoutez des variables personnalisées à insérer dans votre modèle.                                                      |
| Suivre l’activité des courriels        | Cochez cette case pour suivre l’activité des courriels.                                                                |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez la réponse attendue et les codes d’état dans la [documentation de Mailjet](https://dev.mailjet.com/email/reference/send-emails/).
