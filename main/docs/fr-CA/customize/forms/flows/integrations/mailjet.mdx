<div id="send-email">
  ## Envoyer un courriel
</div>

Envoyez un courriel en utilisant Mailjet comme fournisseur de courriel.

<Frame>![](/docs/images/cdy7uua7fh8z/5z1CqSZTrLs1oMdHKvHOiM/7e1fa3b5d7ea5a5b2ec51dd58fa1d2ed/mailjet.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                              | Description                                                                                                                   |
| -------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| Nom de l’expéditeur                    | Le nom de l’expéditeur                                                                                                        |
| Courriel de l’expéditeur (obligatoire) | Le courriel de l’expéditeur                                                                                                   |
| Nom du destinataire                    | Le nom du destinataire                                                                                                        |
| Courriel du destinataire (obligatoire) | Le courriel du destinataire                                                                                                   |
| Nom en Cc                              | Le nom en Cc                                                                                                                  |
| Courriel en Cc                         | Le courriel en Cc                                                                                                             |
| Objet                                  | L’objet du courriel                                                                                                           |
| Type de courriel                       | Envoyer un courriel en HTML ou à l’aide d’un modèle enregistré (facultatif)                                                   |
| Message (obligatoire)                  | Message du courriel en format HTML ou modèle (facultatif).                                                                    |
| ID du modèle (facultatif)              | Si vous utilisez un modèle, vous pouvez trouver son ID dans le [ tableau de bord Mailjet ](https://app.mailjet.com/template). |
| Variables personnalisées (facultatif)  | Ajoutez des variables personnalisées à insérer dans votre modèle.                                                             |
| Suivre l’activité des courriels        | Cochez cette case pour suivre l’activité des courriels.                                                                       |

<div id="output-object">
  ### Objet de sortie
</div>

Vous pouvez consulter la réponse attendue et les codes d’état dans la [documentation de Mailjet](https://dev.mailjet.com/email/reference/send-emails/).
