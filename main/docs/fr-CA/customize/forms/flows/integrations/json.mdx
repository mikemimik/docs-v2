title: JSON
---

Cette liste d’actions JSON vous permet de créer des objets JSON, d’analyser du JSON et de le convertir en chaînes de caractères.

<div id="create-json-object">
  ## Créer un objet JSON
</div>

Crée un objet JSON. C’est particulièrement utile pour réutiliser son contenu dans d’autres actions du flux.

<Frame>![](/docs/images/cdy7uua7fh8z/1zJEENRPI2uONuCApPY98p/f1fce88a42fb13fb9856c95c562bd506/createjsonobject.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre | Description   |
| --------- | ------------- |
| Corps     | L’objet JSON. |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type  | Description   |
| --------- | ----- | ------------- |
| `result`  | Objet | L’objet JSON. |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
	"result": {
		"name": "John Doe"
	}
}
```

<div id="parse-json">
  ## Analyser le JSON
</div>

Analyse un objet JSON contenu dans une chaîne de texte pour en faire un objet JSON.

<Frame>![](/docs/images/cdy7uua7fh8z/5HjIGbBiWik1XozAZczxg5/fe2b0c6f7c657c4209921f1a0978e54b/parsejson.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                 | Description     |
| ------------------------- | --------------- |
| Chaîne JSON (obligatoire) | La chaîne JSON. |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type  | Description           |
| --------- | ----- | --------------------- |
| `object`  | Objet | L’objet JSON analysé. |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "object": {
    "name": "John Doe"
  }
}
```

<div id="convert-json-to-string">
  ## Convertir JSON en chaîne de caractères
</div>

Convertit un objet JSON en chaîne de caractères.

<Frame>![](/docs/images/cdy7uua7fh8z/45acWYMlTGc5ZWMzK5Cqqr/6974ea640a76d2d9cc5814225e52734a/convertjsontostring.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                | Description                                              |
| ------------------------ | -------------------------------------------------------- |
| Objet JSON (obligatoire) | L’objet JSON qui sera sérialisé en chaîne de caractères. |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type   | Description             |
| --------- | ------ | ----------------------- |
| `json`    | String | L’objet JSON sérialisé. |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "json": "{\"name\":\"John Doe\"}"
}
```
