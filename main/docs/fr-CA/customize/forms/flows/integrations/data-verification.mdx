Cette liste d’actions de vérification des données vous permet de vérifier des adresses courriel ainsi que de générer et valider des mots de passe à usage unique.

<div id="generate-one-time-password">
  ## Générer un mot de passe à usage unique
</div>

Génère un code numérique valide pendant 5 minutes, qui peut être envoyé par courriel, par SMS ou par n’importe quelle application de messagerie.

<Frame>![](/docs/images/cdy7uua7fh8z/1h69FWzZwk9beKKjB4Cq38/16aa515e61cc6e9ebdfd0849201e3141/generate-one-time-password.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                         | Description                                                                                                            |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Valeur de référence (obligatoire) | Vous pouvez associer le code généré à un numéro de téléphone, une adresse courriel ou à n’importe quelle autre valeur. |
| Longueur du code (obligatoire)    | Le nombre de chiffres du code généré.                                                                                  |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type   | Description    |
| --------- | ------ | -------------- |
| `code`    | String | Le code généré |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "code": "███",
}
```

<div id="verify-one-time-password">
  ## Vérifier le mot de passe à usage unique
</div>

Vérifie si le code de mot de passe à usage unique fourni est valide.

<Frame>![](/docs/images/cdy7uua7fh8z/3bkT8QDN4dcl0YHTrSOIGj/9698a07f55353a11ca98164377cd4217/verify-otp.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                         | Description                                                                                                     |
| --------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| Valeur de référence (obligatoire) | Utilisez la même valeur de référence que celle configurée dans l’action Générer un mot de passe à usage unique. |
| Longueur du code (obligatoire)    | Le nombre de chiffres du code généré.                                                                           |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type    | Description                                                                                                                                                                                                                                                                    |
| --------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `valid`   | Boolean | Indique si le code fourni est valide en renvoyant une valeur `true` ou `false`.                                                                                                                                                                                                |
| `result`  | String  | Renvoie différents codes de résultat selon le code fourni : <ul> <li> `MISSING:` Le code fourni n’existe pas. </li> <li> `EXPIRED:` Le code fourni a expiré. </li> <li> `VALID:` Le code fourni est valide. </li> <li> `UNEQUAL:` Le code fourni n’est pas valide. </li> </ul> |

<div id="output-object-examples">
  ### Exemples d’objets de sortie
</div>

```json lines
{
  "valid": false,
  "result": "UNEQUAL"
}
```

```json lines
{
  "valid": true
}
```

<div id="verify-email-address">
  ## Vérification de l’adresse courriel
</div>

Exécute un ensemble de règles de vérification sélectionnées pour l’adresse courriel fournie.

<Frame>![](/docs/images/cdy7uua7fh8z/6qkqny1hhObFalHwdEG342/1a2e18195b29102c3f68744f42277917/verify-email.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                                 | Description                                                                                                                                                      |
| ----------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Courriel (obligatoire)                    | adresse courriel à vérifier.                                                                                                                                     |
| Exiger un enregistrement MX               | Exige des enregistrements MX. Lorsqu’un domaine n’a pas d’enregistrements MX, les serveurs de messagerie ne savent pas où envoyer les courriels pour ce domaine. |
| Bloquer les services de courriel gratuits | Bloque les services de courriel gratuits comme @gmail.com, @hotmail.com, etc.                                                                                    |
| Bloquer les services de courriel jetables | Bloque les courriels jetables comme @mailnator.com, @maildrop.cc, etc.                                                                                           |
| Bloquer les alias de compte courriel      | Bloque les courriels avec des alias comme `jane.doe+alias@gmail.com`                                                                                             |
| Domaines de la liste d’autorisation       | Autorise seulement les courriels provenant des domaines approuvés dans votre liste.                                                                              |
| Domaines de la liste de blocage           | Bloque des domaines de courriel précis.                                                                                                                          |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| --------- | ------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `valid`   | Boolean | Retourne `true` ou `false` selon que le courriel respecte ou non les règles de vérification.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `cause`   | String  | Si la propriété `valid` est `<br/>false<br/>`, retourne une référence à la première règle que le courriel ne respecte pas : <ul> <li> `MISSING_MX_RECORD:` Le domaine du courriel ne possède pas d’enregistrements MX. </li> <li> `FREE_EMAIL:` Le domaine du courriel appartient à un fournisseur de courriel gratuit.</li> <li> `DISPOSABLE_EMAIL:` Le courriel est une adresse courriel jetable.</li> <li> `BLOCKLISTED:` Le domaine du courriel est inclus dans le paramètre Blocklist domains.</li> <li> `NOT_ALLOWED:` Le domaine du courriel n’est pas inclus dans le paramètre Allowlist domains.</li> </ul> |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "valid": false,
  "cause": "FREE_EMAIL"
}
```

```json lines
{
  "valid": true,
  "cause": null
}
```
