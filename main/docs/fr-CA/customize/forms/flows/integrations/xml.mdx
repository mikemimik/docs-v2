Cette liste d’actions XML vous permet de convertir des objets JSON en chaîne XML et des chaînes XML en objets JSON.

<div id="convert-json-to-xml">
  ## Convertir JSON en XML
</div>

Convertit un objet JSON en chaîne XML.

<Frame>![](/docs/images/cdy7uua7fh8z/3GW2EhBMIX3YDPaqc3rXrZ/7706a863700df51376d645a7d9292a27/convertjsontoxml.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre          | Description                                   |
| ------------------ | --------------------------------------------- |
| JSON (obligatoire) | L’objet JSON qui sera converti en chaîne XML. |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type   | Description              |
| --------- | ------ | ------------------------ |
| `xml`     | String | La chaîne XML convertie. |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "xml": "<?xml version=\"1.0\" encoding=\"UTF-8\" standalone=\"yes\"?>\n<name>John Doe</name>"
}
```

<div id="convert-xml-to-json">
  ## Convertir du XML en JSON
</div>

Convertit une chaîne XML en un objet JSON.

<Frame>![](/docs/images/cdy7uua7fh8z/42fO7BelEN4bnlUj70WhV8/e53ada552f7cc05c192466db4777471a/covertxmltojson.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre         | Description                                     |
| ----------------- | ----------------------------------------------- |
| XML (obligatoire) | La chaîne XML qui sera convertie en objet JSON. |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type  | Description            |
| --------- | ----- | ---------------------- |
| `object`  | Objet | L’objet JSON converti. |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "object": {
    "name": "John Doe"
  }
}
```
