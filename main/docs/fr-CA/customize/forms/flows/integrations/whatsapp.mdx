<div id="configure-your-whatsapp-vault-connection">
  ## Configurer votre connexion Vault WhatsApp
</div>

Apprenez à configurer une connexion Vault pour vos actions de Flow WhatsApp dans [WhatsApp Vault Connection](/docs/fr-CA/customize/forms/vaults/whatsapp).

<div id="send-message">
  ## Envoyer un message
</div>

Permet d’envoyer un message.

<Frame>![](/docs/images/cdy7uua7fh8z/5zZPQw5KKYwBqxbR9jF1wZ/edba085456e3536f5167ff4c21bea77c/whatsapp.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                                         | Description                                                                                                                                                                                                                                                        |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ID du téléphone de l’expéditeur (obligatoire)     | Vous trouverez l’ID du téléphone de l’expéditeur dans les paramètres WhatsApp de [Facebook Business Manager](https://developers.facebook.com/apps). Notez qu’il ne s’agit pas d’un numéro de téléphone.                                                            |
| Numéro de téléphone du destinataire (obligatoire) | Le numéro de téléphone du destinataire au [format E.164](https://en.wikipedia.org/wiki/E.164).                                                                                                                                                                     |
| Type de message (obligatoire)                     | Sélectionnez le type de message dans le menu déroulant. Les options sont les suivantes : <ul> <li>Audio</li> <li>Contacts</li> <li>Document</li> <li>Image</li> <li>Interactif</li> <li>Emplacement</li> <li>Autocollant</li> <li>Modèle</li> <li>Texte</li> </ul> |
| Objet de message (obligatoire)                    | Le contenu de l’objet de message varie selon le type de message sélectionné. Pour en savoir plus, consultez [Messages](https://developers.facebook.com/docs/whatsapp/cloud-api/reference/messages).                                                                |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez tous les codes d’état attendus dans la [documentation de Facebook](https://developers.facebook.com/docs/whatsapp/cloud-api/support/error-codes).
