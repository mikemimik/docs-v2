  ## Envoyer un message
</div>

Envoie un message.

<Frame>![](/docs/images/cdy7uua7fh8z/5zZPQw5KKYwBqxbR9jF1wZ/edba085456e3536f5167ff4c21bea77c/whatsapp.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                                         | Description                                                                                                                                                                                                                                                                |
| ------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ID du téléphone de l’expéditeur (obligatoire)     | L’ID du téléphone de l’expéditeur se trouve dans les [paramètres WhatsApp de Facebook Business Manager](https://developers.facebook.com/apps). Notez qu’il ne s’agit pas d’un numéro de téléphone.                                                                         |
| Numéro de téléphone du destinataire (obligatoire) | Le numéro de téléphone du destinataire au [format E.164](https://en.wikipedia.org/wiki/E.164).                                                                                                                                                                             |
| Type de message (obligatoire)                     | Sélectionnez le type de message dans le menu déroulant. Les types possibles sont les suivants : <ul> <li>Audio</li> <li>Contacts</li> <li>Document</li> <li>Image</li> <li>Interactif</li> <li>Localisation</li> <li>Autocollant</li> <li>Modèle</li> <li>Texte</li> </ul> |
| Objet du message (obligatoire)                    | Le contenu de l’objet du message varie selon le type de message sélectionné. Pour en savoir plus, consultez [Messages](https://developers.facebook.com/docs/whatsapp/cloud-api/reference/messages).                                                                        |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez tous les codes d’état attendus dans la [documentation de Facebook](https://developers.facebook.com/docs/whatsapp/cloud-api/support/error-codes).
