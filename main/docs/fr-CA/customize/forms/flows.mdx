Flows vous permettent de créer visuellement la logique côté serveur qui peut être exécutée dans vos Forms pour traiter les données recueillies, intégrer n’importe quel service disposant d’une API ou déclencher des automatisations.

<Frame>![Dashboard > Forms > Flows](/docs/images/cdy7uua7fh8z/6u6Huh5CbH3GEQJv3AH7cx/e49ea613b2c46a43c83c89ffecddbafe/Flows.png)</Frame>

Vous pouvez utiliser les actions de Flows avec votre formulaire pour :

* Créer des validations personnalisées côté serveur pour les étapes de votre formulaire.
* Intégrer votre propre API en effectuant des requêtes HTTP personnalisées.
* Intégrer des services tiers à l’aide de nos intégrations prêtes à l’emploi.
* Exécuter une logique d’affaires différente en fonction des entrées ou des propriétés de l’utilisateur.
* Valider les données à l’aide de <Tooltip tip="JSON Web Token (JWT) : format standard d’ID Token (et souvent d’access token) utilisé pour représenter des claims de façon sécurisée entre deux parties." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=JSON+Web+Tokens">JSON Web Tokens</Tooltip>.
