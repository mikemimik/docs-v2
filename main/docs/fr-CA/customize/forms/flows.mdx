Flows vous permettent de créer visuellement la logique côté serveur qui peut s’exécuter dans vos Forms pour traiter les renseignements recueillis, intégrer n’importe quel service à une API ou déclencher des automatisations.

<Frame>![Dashboard > Forms > Flows](/docs/images/cdy7uua7fh8z/6u6Huh5CbH3GEQJv3AH7cx/e49ea613b2c46a43c83c89ffecddbafe/Flows.png)</Frame>

Vous pouvez utiliser les actions de Flows avec votre formulaire pour :

* Créer des validations personnalisées côté serveur pour les étapes du formulaire.
* Intégrer votre propre API en effectuant des requêtes HTTP personnalisées.
* Intégrer des services tiers à l’aide de nos intégrations prêtes à l’emploi.
* Exécuter différentes logiques métier selon les saisies de l’utilisateur ou les propriétés.
* Valider les données à l’aide de <Tooltip tip="JSON Web Token (JWT) : format standard des ID Token (et souvent des Access Token) utilisé pour représenter de façon sécuritaire des claims entre deux parties." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=JSON+Web+Tokens">JSON Web Tokens</Tooltip>.
