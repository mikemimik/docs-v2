---

Le SDK Auth0 PHP fournit une classe `Auth0\SDK\API\Management` qui regroupe les méthodes permettant d&#39;accéder à la [Management API](https://auth0.com/docs/api/management/v2) et d&#39;effectuer des opérations sur votre tenant Auth0. Grâce à cette interface, vous pouvez facilement :

* Rechercher et créer des utilisateurs
* Créer et mettre à jour des Applications
* Récupérer des entrées de journal
* Gérer des Rules

... et bien plus encore. Consultez notre [référence de l&#39;API](https://auth0.com/docs/api/management/v2) pour découvrir tout ce qu&#39;il est possible de faire.
Authentification

Pour utiliser la <Tooltip tip="Management API : un produit qui permet aux clients d'effectuer des tâches administratives." cta="Voir le glossaire" href="/docs/fr-CA/glossary?term=Management+API">Management API</Tooltip>, vous devez vous authentifier de l&#39;une des deux façons suivantes :

* Pour un accès temporaire ou des tests, vous pouvez [générer manuellement un jeton d&#39;API](/docs/fr-CA/secure/tokens/access-tokens/management-api-access-tokens) et l&#39;enregistrer dans votre fichier `.env`.
* Pour un accès prolongé, vous devez créer et exécuter un grant Client Credentials au moment où l&#39;accès est requis. Ce processus est décrit sur la [page de l&#39;Authentication API](/docs/fr-CA/libraries/auth0-php/using-the-authentication-api-with-auth0-php).

Quelle que soit la méthode, le jeton généré doit inclure les scopes requis pour les opérations que votre application veut exécuter. Consultez la [documentation de l&#39;API](https://auth0.com/docs/api/management/v2) pour connaître les scopes requis pour le point de terminaison précis auquel vous tentez d&#39;accéder.

Pour accorder les scopes nécessaires :

1. Accédez à [APIs](https://manage.auth0.com/#/apis) &gt; Auth0 Management API &gt; onglet **Machine to Machine Applications**.
2. Repérez votre Application et autorisez-la.
3. Cliquez sur la flèche pour développer la ligne, puis sélectionnez les scopes requis.

Vous pouvez maintenant vous authentifier de l&#39;une des deux façons ci-dessus et utiliser ce jeton pour effectuer des opérations :

```php lines
// 👆 Nous continuons à partir du guide de démarrage lié dans « Prérequis » ci-dessus. Ajoutez ceci au fichier index.php que vous avez créé précédemment.

if (isset($env['AUTH0_MANAGEMENT_API_TOKEN'])) {
    $auth0->configuration()->setManagementToken($env['AUTH0_MANAGEMENT_API_TOKEN']);
}

// Créez une instance configurée de la classe `Auth0\SDK\API\Management`, basée sur la configuration utilisée pour initialiser le SDK ($auth0).
// Si aucun AUTH0_MANAGEMENT_API_TOKEN n'est configuré, un échange de client credentials sera effectué automatiquement pour en générer un, à condition qu'un client secret soit configuré.
$management = $auth0->management();
```

La classe `Management` donne accès aux points de terminaison au moyen de méthodes de fabrique sur ses instances; par exemple, `$management->users()` renvoie une instance de `Auth0\SDK\API\Management\Users` que vous pouvez utiliser pour interagir avec les points de terminaison /users de la Management API.

<div id="example-search-users-by-email">
  ### Exemple - Rechercher des utilisateurs par courriel
</div>

Ce point de terminaison est documenté [ici](https://auth0.com/docs/api/management/v2#!/Users/get_users).

```php lines
// 👆 Nous continuons à partir du code ci-dessus. Ajoutez ceci à votre fichier de code source.

$response = $management->users()->getAll(['q' => 'josh']);

// Le code d'état de la réponse indique-t-il un échec ?
if ($response->getStatusCode() !== 200) {
    die("API request failed.");
}

// Décodez la réponse JSON en tableau PHP :
$response = json_decode(response->getBody()->__toString(), true, 512, JSON_THROW_ON_ERROR);

if (! empty($response)) {
    echo '<h2>User Results</h2>';

    foreach ($response as $result) {
        printf(
            '<p><strong>%s</strong> &lt;%s&gt; - %s</p>',
            !empty($result['nickname']) ? $result['nickname'] : 'No nickname',
            !empty($result['email']) ? $result['email'] : 'No email',
            $result['user_id']
        );
    }
}
```

<div id="example-get-all-clients">
  ### Exemple - Obtenir tous les clients
</div>

Ce point de terminaison est documenté [ici](https://auth0.com/docs/api/management/v2#!/Clients/get_clients).

```php lines
// 👆 Nous continuons à partir du code ci-dessus. Ajoutez ceci à votre fichier de code source.

$response = $management->clients()->getAll(['q' => 'josh']);

// Le code d'état de la réponse indique-t-il un échec ?
if ($response->getStatusCode() !== 200) {
    die("API request failed.");
}

// Décoder la réponse JSON en tableau PHP :
$response = json_decode(response->getBody()->__toString(), true, 512, JSON_THROW_ON_ERROR);

if (! empty($response)) {
    echo '<h2>Get All Clients</h2>';

    foreach ($response as $result) {
        printf(
            '<p><strong>%s</strong> - %s</p>',
            $result['name'],
            $result['client_id']
        );
    }
}
```
