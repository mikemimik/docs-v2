L’Auth0 ACUL JS SDK vous permet de créer des expériences Universal Login personnalisées avec un minimum d’effort. Que vous travailliez sur la connexion, l’inscription, le sans mot de passe ou l’enregistrement d’une clé d’accès, ce SDK est conçu pour prendre en charge facilement les flux d’authentification modernes.

<div id="why-choose-the-auth0-acul-js-sdk">
  ### Pourquoi choisir le Auth0 ACUL JS SDK?
</div>

* **Intégration rapide :** Intégrez des écrans prédéfinis comme `login-id` ou `signup-password`.
* **Fonctionnalités modernes :** Prise en charge du sans mot de passe, des clés d’accès et des connexions sociales.
* **Modules flexibles :** Importez seulement ce dont vous avez besoin.
* **Gestion unifiée de l’état :** Utilisez un seul objet transaction pour gérer tous les flux d’authentification.

<div id="architecture">
  ### Architecture
</div>

<div id="how-it-works">
  #### Comment ça fonctionne ?
</div>

Le SDK est conçu dans une optique de modularité et d’évolutivité. Voici comment l’ensemble s’articule :

* **Écrans :** Chaque flux d’authentification, comme `login-id` et `signup-password`, est un module autonome.
* **Gestionnaires de soumission :** Gèrent les soumissions de formulaire et les transitions.
* **Formulaires dynamiques :** Adaptent les champs et les validations selon la configuration de l’écran.
* **Propriétés utilitaires :** Permettent d’accéder de façon cohérente à des données communes, comme l’image de marque ou la transaction.
