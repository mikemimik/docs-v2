Le SDK Auth0 ACUL JS vous permet de créer des expériences Universal Login personnalisées avec un minimum d’effort. Que vous travailliez sur la connexion, l’inscription, l’authentification passwordless ou l’enregistrement d’une clé d’accès, ce SDK est conçu pour prendre facilement en charge les flux d’authentification modernes.

<div id="why-choose-the-auth0-acul-js-sdk">
  ### Pourquoi choisir le SDK Auth0 ACUL JS?
</div>

* **Intégration rapide :** Intégrez des écrans prédéfinis comme `login-id` ou `signup-password`.
* **Fonctionnalités modernes :** Prise en charge du passwordless, des passkeys et des connexions via les réseaux sociaux.
* **Modules flexibles :** Importez uniquement ce dont vous avez besoin.
* **Gestion unifiée de l’état :** Utilisez un seul objet de transaction pour gérer tous les flux d’authentification.

<div id="architecture">
  ### Architecture
</div>

<div id="how-it-works">
  #### Comment ça fonctionne ?
</div>

Le SDK est conçu dans un souci de modularité et d’évolutivité. Voici comment ses éléments s’imbriquent :

* **Écrans :** Chaque flux d’authentification, comme `login-id` et `signup-password`, est un module autonome.
* **Gestionnaires de soumission :** Gèrent l’envoi des formulaires et les transitions.
* **Formulaires dynamiques :** Adaptent les champs et les validations selon la configuration de l’écran.
* **Propriétés d’aide :** Permettent d’accéder de façon cohérente à des données courantes, comme l’image de marque ou la transaction.
