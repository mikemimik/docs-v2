Le SDK JS Auth0 ACUL vous permet de créer des expériences Universal Login personnalisées avec un minimum d’effort. Que vous travailliez sur la connexion, l’inscription, l’authentification sans mot de passe ou l’enrôlement de clés d’accès, ce SDK est conçu pour prendre en charge facilement les flux d’authentification modernes.

<div id="why-choose-the-auth0-acul-js-sdk">
  ### Pourquoi choisir le SDK JS Auth0 ACUL?
</div>

* **Intégration rapide :** Ajoutez des écrans prédéfinis comme `login-id` ou `signup-password`.
* **Fonctionnalités modernes :** Prend en charge l’authentification sans mot de passe, les clés d’accès et les connexions avec des comptes sociaux.
* **Modules flexibles :** N’importez que ce dont vous avez besoin.
* **Gestion unifiée de l’état :** Utilisez un seul objet transaction pour gérer tous les flux d’authentification.

<div id="architecture">
  ### Architecture
</div>

<div id="how-it-works">
  #### Comment ça fonctionne ?
</div>

Le SDK est conçu dans un souci de modularité et d’évolutivité. Voici comment le tout s’articule :

* **Écrans :** Chaque flux d’authentification, comme `login-id` et `signup-password`, est un module autonome.
* **Gestionnaires de soumission :** Gèrent la soumission des formulaires et les transitions.
* **Formulaires dynamiques :** Adaptent les champs et les validations selon la configuration de l’écran.
* **Propriétés utilitaires :** Permettent d’accéder de façon cohérente à des données communes, comme l’image de marque ou la transaction.
