Le SDK JS Auth0 ACUL vous permet de créer des expériences Universal Login personnalisées avec un minimum d’effort. Que ce soit pour la connexion, l’inscription, l’authentification sans mot de passe ou l’inscription de clés d’accès, ce SDK est conçu pour prendre en charge facilement les flux d’authentification modernes.

<div id="why-choose-the-auth0-acul-js-sdk">
  ### Pourquoi choisir le SDK JS Auth0 ACUL?
</div>

* **Intégration rapide :** Intégrez des écrans prédéfinis comme `login-id` ou `signup-password`.
* **Fonctionnalités modernes :** Prise en charge de l’authentification sans mot de passe, des passkeys et des connexions sociales.
* **Modules flexibles :** Importez uniquement ce dont vous avez besoin.
* **Gestion unifiée de l’état :** Utilisez un seul objet de transaction pour gérer tous les flux d’authentification.

<div id="architecture">
  ### Architecture
</div>

<div id="how-it-works">
  #### Comment ça fonctionne ?
</div>

Le SDK est conçu dans une optique de modularité et d’évolutivité. Voici comment tout s’articule :

* **Écrans :** Chaque flux d’authentification, comme `login-id` et `signup-password`, constitue un module autonome.
* **Gestionnaires de soumission :** Gèrent les soumissions de formulaires et les transitions.
* **Formulaires dynamiques :** Adaptent les champs et les règles de validation selon la configuration de l’écran.
* **Propriétés utilitaires :** Permettent d’accéder de manière cohérente aux données courantes, comme l’image de marque ou la transaction.
