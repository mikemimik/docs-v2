L’écran `mfa-push-enrollment-qr` affiche un code QR que l’utilisateur peut balayer avec l’application Auth0 Guardian afin de terminer l’inscription aux notifications push.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge **l’importation partielle** et **l’importation racine** pour chaque écran.

* L’utilisation de l’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’utilisation de l’importation racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous voulez un build unifié pour prendre en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useMfaPushEnrollmentQr } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useMfaPushEnrollmentQr,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useMfaPolling,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
  pickAuthenticator,
} from "@auth0/auth0-acul-react/mfa-push-enrollment-qr";

function MfaPushEnrollmentQrScreen() {
  const { pickAuthenticator } = useMfaPushEnrollmentQr();
  return (
    <button onClick={() => pickAuthenticator()}>
      Pick Authenticator
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Des hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `mfa-push-enrollment-qr`. Importez-les depuis `@auth0/auth0-acul-react/mfa-push-enrollment-qr`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit des paramètres de branding, comme le logo, les couleurs et les réglages du thème affichés sur l’écran `mfa-push-enrollment-qr`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `mfa-push-enrollment-qr`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si le flux d’AMF est limité à l’Organization. Retourne `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Aucun contexte d’organisation</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaPushEnrollmentQr">ScreenMembersOnMfaPushEnrollmentQr</a></span>}>
  Ce hook contient des détails propres à l’écran `mfa-push-enrollment-qr`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function TenantInfo() {

  }
  ```

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `mfa-push-enrollment-qr`, comme l’état actuel du flux d’AMF.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';

    const transaction = useTransaction();
  }
  ```

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>


<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>


<ParamField body="useMfaPushEnrollmentQr" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPushEnrollmentQrMembers">MfaPushEnrollmentQrMembers</a>}>

</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode termine l’inscription après que l’utilisateur a numérisé le code QR avec l’application Auth0 Guardian.

  ```jsx Example
  import { useMfaPushEnrollmentQr } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';

  function ContinueButton() {
    const { continueMethod } = useMfaPushEnrollmentQr();
    return (
      <button onClick={() => continueMethod({ rememberDevice: true })}>
        Continue
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaPushEnrollmentQrWithRememberOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPushEnrollmentQrWithRememberOptions).
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Si `true`, l’appareil sera mémorisé et l’AMF ne sera plus requise sur cet appareil.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickAuthenticator" type="Promise<void>">
  Cette méthode mène à l’écran de sélection de l’authentificateur pour permettre à l’utilisateur de choisir une autre méthode d’AMF à inscrire.

  ```jsx Example
  import { useMfaPushEnrollmentQr } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';

  function PickAuthenticatorButton() {
    const { pickAuthenticator } = useMfaPushEnrollmentQr();
    return (
      <button onClick={() => pickAuthenticator()}>
        Pick Authenticator
      </button>
    );
  }
  ```
</ParamField>
