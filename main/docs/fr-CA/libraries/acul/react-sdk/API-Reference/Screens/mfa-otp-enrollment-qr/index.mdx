L’écran `mfa-otp-enrollment-qr` affiche un code QR que l’utilisateur peut scanner avec son application d’authentification pour finaliser l’inscription à l’OTP.

<div id="import">
  ## Importation
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge **l’importation partielle** et **l’importation à la racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’usage.
* L’importation à la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous voulez qu’une même version prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useMfaOtpEnrollmentQr } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useMfaOtpEnrollmentQr,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
  toggleView,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-otp-enrollment-qr";

function MfaOtpEnrollmentQrScreen() {
  const { continueMethod } = useMfaOtpEnrollmentQr();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `mfa-otp-enrollment-qr`. Importez-les depuis `@auth0/auth0-acul-react/mfa-otp-enrollment-qr`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations de Branding, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `mfa-otp-enrollment-qr`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `mfa-otp-enrollment-qr`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si le flux MFA est associé à une Organization. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaOtpEnrollmentQr">ScreenMembersOnMfaOtpEnrollmentQr</a></span>}>
  Ce hook contient des détails propres à l’écran `mfa-otp-enrollment-qr`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les metadata associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `mfa-otp-enrollment-qr`, comme l’état actuel du flux MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme les valeurs préremplies provenant des paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function PrefilledForm() {

  }
  ```

</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des renseignements sur l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaOtpEnrollmentQr" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaOtpEnrollmentQrMembers">MfaOtpEnrollmentQrMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte disponibles dans l’écran `mfa-otp-enrollment-qr`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode envoie le code OTP afin de vérifier que l’application d’authentification est correctement configurée et de terminer l’inscription.

  ```jsx Example
  import { useMfaOtpEnrollmentQr } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';

  function EnrollmentQrForm() {
    const { continueMethod } = useMfaOtpEnrollmentQr();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Vérifier
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaOtpEnrollmentQrContinueOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaOtpEnrollmentQrContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code OTP saisi par l’utilisateur.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="toggleView" type="Promise<void>">
  Cette méthode fait basculer la vue d’inscription entre les modes code QR et saisie manuelle (clé secrète).

  ```jsx Example
  import { useMfaOtpEnrollmentQr } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';

  function ToggleViewButton() {
    const { toggleView } = useMfaOtpEnrollmentQr();
    return (
      <button onClick={() => toggleView()}>
        Entrer le code manuellement
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode redirige vers l’écran de sélection des méthodes MFA pour que l’utilisateur puisse choisir un autre facteur d’authentification.

  ```jsx Example
  import { useMfaOtpEnrollmentQr } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaOtpEnrollmentQr();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Essayer une autre méthode
      </button>
    );
  }
  ```
</ParamField>
