L’écran `reset-password-mfa-recovery-code-challenge` s’affiche pendant le flux de réinitialisation du mot de passe lorsqu’une authentification multifacteur au moyen d’un code de récupération est requise. Il invite l’utilisateur à saisir son code de récupération pour poursuivre le processus de réinitialisation.

<div id="import">
  ## Importation
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation à la racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’importation à la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous voulez une version unifiée capable de gérer tous les écrans possibles.

```jsx Import Example
// importation racine
import { useResetPasswordMfaRecoveryCodeChallenge } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useResetPasswordMfaRecoveryCodeChallenge,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';

function ResetPasswordMfaRecoveryCodeChallengeScreen() {
  const { continueMethod } = useResetPasswordMfaRecoveryCodeChallenge();
  return (
    <button onClick={() => continueMethod('ABC123')}>
      Verify Recovery Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `reset-password-mfa-recovery-code-challenge`. Importez-les depuis `@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les paramètres d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `reset-password-mfa-recovery-code-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `reset-password-mfa-recovery-code-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’organisation de l’utilisateur si la réinitialisation du mot de passe est limitée à cette organisation. Renvoie `null` lorsqu’aucun contexte d’organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt en cours dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Ce hook contient des détails propres à l’écran `reset-password-mfa-recovery-code-challenge`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```

</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `reset-password-mfa-recovery-code-challenge`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook donne accès aux données non fiables transmises à l’écran, comme une adresse courriel ou un nom d’utilisateur prérempli à partir de paramètres d’URL.


<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
  ```

<ParamField body="useResetPasswordMfaRecoveryCodeChallenge" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordMfaRecoveryCodeChallengeMembers">ResetPasswordMfaRecoveryCodeChallengeMembers</a>}>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode envoie le code de récupération pour poursuivre l’étape d’AMF dans le flux de réinitialisation du mot de passe.

  ```jsx Example
  import { useResetPasswordMfaRecoveryCodeChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';

  function VerifyRecoveryCodeForm() {
    const { continueMethod } = useResetPasswordMfaRecoveryCodeChallenge();
    return (
      <button onClick={() => continueMethod('ABC123')}>
        Vérifier le code de récupération
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [CustomOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code de récupération saisi par l’utilisateur.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode redirige l’utilisateur pour qu’il sélectionne une autre méthode d’AMF.

  ```jsx Example
  import { useResetPasswordMfaRecoveryCodeChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useResetPasswordMfaRecoveryCodeChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Essayer une autre méthode
      </button>
    );
  }
  ```
</ParamField>
