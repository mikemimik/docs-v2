L’écran `organization-picker` s’affiche lorsqu’un utilisateur doit sélectionner une organisation pour se connecter. Il lui permet de choisir une organisation ou d’ignorer cette étape.

<div id="import">
  ## Importation
</div>

Chaque écran comporte son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation racine** pour chaque écran.

* L’importation partielle vous permet d’inclure seulement le code nécessaire à votre cas d’utilisation.
* L’importation racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez une version unifiée capable de prendre en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useOrganizationPicker } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useOrganizationPicker,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  selectOrganization,
  skipOrganizationSelection,
} from '@auth0/auth0-acul-react/organization-picker';

function OrganizationPickerScreen() {
  const { selectOrganization } = useOrganizationPicker();
  return (
    <button onClick={() => selectOrganization({ organization: 'my-org', state: 'abc123' })}>Select</button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte d’Auth0 sur l’écran `organization-picker`. Importez-les depuis `@auth0/auth0-acul-react/organization-picker`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit des configurations d’image de marque, comme le logo, les couleurs et les paramètres du thème affichés sur l’écran `organization-picker`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/organization-picker';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `organization-picker`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/organization-picker';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur le contexte d’organisation de l’utilisateur sur l’écran `organization-picker`. Renvoie `null` lorsqu’aucun contexte d’organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/organization-picker';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/organization-picker';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Ce hook contient des détails propres à l’écran `organization-picker`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/organization-picker';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/organization-picker';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `organization-picker`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/organization-picker';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme les paramètres préremplis provenant de la chaîne de requête de l’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/organization-picker';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des renseignements sur l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentication disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/organization-picker';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useOrganizationPicker" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationPickerMembers">OrganizationPickerMembers</a>}>
  Ce hook renvoie toutes les méthodes et tout le contexte disponibles à l’écran `organization-picker`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="selectOrganization" type="Promise<void>">
  Cette méthode envoie la sélection d’organisation de l’utilisateur et poursuit le flux d’authentification.

  ```jsx Example
  import { useOrganizationPicker } from '@auth0/auth0-acul-react/organization-picker';

  function SelectButton() {
    const { selectOrganization } = useOrganizationPicker();
    return (
      <button onClick={() => selectOrganization({ organization: 'my-org', state: 'abc123' })}>
        Select Organization
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [SelectOrganizationOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/SelectOrganizationOptions).
    </ParamField>

    <ParamField body="organization" type="string" required>
      L’identifiant de l’organisation sélectionnée par l’utilisateur.
    </ParamField>

    <ParamField body="state" type="string" required>
      La valeur state actuelle de l’authentification.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="skipOrganizationSelection" type="Promise<void>">
  Cette méthode permet à l’utilisateur d’ignorer l’étape de sélection d’organisation et de poursuivre le flux d’authentification sans sélectionner d’organisation.

  ```jsx Example
  import { useOrganizationPicker } from '@auth0/auth0-acul-react/organization-picker';

  function SkipButton() {
    const { skipOrganizationSelection } = useOrganizationPicker();
    return (
      <button onClick={() => skipOrganizationSelection()}>
        Skip
      </button>
    );
  }
  ```
</ParamField>
