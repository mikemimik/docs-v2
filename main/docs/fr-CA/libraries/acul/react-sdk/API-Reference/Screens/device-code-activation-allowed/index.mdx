description: "Décrit tous les hooks et toutes les méthodes permettant de personnaliser l’écran `device-code-activation-allowed` d’Universal Login."
---

L’écran `device-code-activation-allowed` s’affiche après qu’un utilisateur a activé son appareil avec succès. Il confirme que l’appareil a été autorisé et que le processus d’activation est terminé.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation depuis la racine** pour chaque écran.

* L’utilisation de l’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’usage.
* L’utilisation de l’importation depuis la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez une version unifiée capable de gérer tous les écrans possibles.

```jsx Import Example
// importation racine
import { useDeviceCodeActivationAllowed } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useDeviceCodeActivationAllowed,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
} from '@auth0/auth0-acul-react/device-code-activation-allowed';

function DeviceCodeActivationAllowedScreen() {
  const { screen } = useDeviceCodeActivationAllowed();
  return <p>Device successfully activated on {screen.name}</p>;
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks associés à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `device-code-activation-allowed`. Importez-les depuis `@auth0/auth0-acul-react/device-code-activation-allowed`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `device-code-activation-allowed`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `device-code-activation-allowed`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si l’activation de l’appareil est limitée à l’Organization. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Ce hook contient des détails propres à l’écran `device-code-activation-allowed`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `device-code-activation-allowed`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme un code d’appareil prérempli à partir de paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit les détails de l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/device-code-activation-allowed';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useDeviceCodeActivationAllowed" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/DeviceCodeActivationAllowedMembers">DeviceCodeActivationAllowedMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte disponibles pour l’écran `device-code-activation-allowed`.
</ParamField>

<div id="commonutility-hooks">
  ## Hooks communs/utilitaires
</div>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Ce hook récupère les options du thème actuel avec une configuration à plat à partir du contexte de l’image de marque.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Ce hook renvoie une fonction qui permet de changer la langue d’affichage sur l’écran ACUL actuel.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Ce hook récupère le contexte et l’état de l’écran actuel.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Ce hook lit et gère les erreurs serveur, client et développeur à l’écran.
</ParamField>
