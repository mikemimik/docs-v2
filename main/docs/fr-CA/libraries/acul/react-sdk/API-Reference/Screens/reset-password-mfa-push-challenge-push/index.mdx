L’écran `reset-password-mfa-push-challenge-push` s’affiche pendant le flux de réinitialisation du mot de passe lorsqu’une authentification multifacteur par notification push est requise. Il invite l’utilisateur à approuver la demande sur son appareil mobile enregistré afin de poursuivre le processus de réinitialisation.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’usage.
* L’importation racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez une build unifiée capable de gérer tous les écrans possibles.

```jsx Import Example
// importation racine
import { useResetPasswordMfaPushChallengePush } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useResetPasswordMfaPushChallengePush,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useMfaPolling,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
  enterCodeManually,
  resendPushNotification,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';

function ResetPasswordMfaPushChallengePushScreen() {
  const { resendPushNotification } = useResetPasswordMfaPushChallengePush();
  return (
    <button onClick={() => resendPushNotification()}>
      Resend Push Notification
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Des hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `reset-password-mfa-push-challenge-push`. Importez-les depuis `@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `reset-password-mfa-push-challenge-push`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit les configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `reset-password-mfa-push-challenge-push`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des informations sur l’organisation de l’utilisateur si la réinitialisation du mot de passe est limitée à une organisation. Il renvoie `null` lorsqu’aucun contexte d’organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode poursuit le flux de réinitialisation du mot de passe après que l’utilisateur a approuvé la notification push sur son appareil mobile.

  ```jsx Example
  import { useResetPasswordMfaPushChallengePush } from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';

  function ContinueButton() {
    const { continueMethod } = useResetPasswordMfaPushChallengePush();
    return (
      <button onClick={() => continueMethod()}>
        Continuer
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="enterCodeManually" type="Promise<void>">
  Cette méthode redirige l’utilisateur vers la saisie manuelle du code OTP au lieu d’attendre l’approbation de la notification push.

  ```jsx Example
  import { useResetPasswordMfaPushChallengePush } from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';

  function EnterCodeButton() {
    const { enterCodeManually } = useResetPasswordMfaPushChallengePush();
    return (
      <button onClick={() => enterCodeManually()}>
        Saisir le code manuellement
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="resendPushNotification" type="Promise<void>">
  Cette méthode renvoie la notification push à l’appareil mobile enregistré de l’utilisateur.

  ```jsx Example
  import { useResetPasswordMfaPushChallengePush } from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';

  function ResendButton() {
    const { resendPushNotification } = useResetPasswordMfaPushChallengePush();
    return (
      <button onClick={() => resendPushNotification()}>
        Renvoyer la notification push
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode redirige l’utilisateur afin qu’il sélectionne une autre méthode d’AMF.

  ```jsx Example
  import { useResetPasswordMfaPushChallengePush } from '@auth0/auth0-acul-react/reset-password-mfa-push-challenge-push';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useResetPasswordMfaPushChallengePush();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Essayer une autre méthode
      </button>
    );
  }
  ```
</ParamField>

<div id="commonutility-hooks">
  ## Hooks utilitaires communs
</div>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling">useMfaPolling</a>} type="Hooks">
  Ce hook gère l’interrogation périodique des notifications push d’AMF en vérifiant de façon répétée l’endpoint du défi push jusqu’à ce que l’utilisateur approuve ou refuse la requête.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Ce hook récupère les options du thème actuel avec la configuration aplatie du contexte d’image de marque.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Ce hook renvoie une fonction permettant de changer la langue d’affichage sur l’écran ACUL actuel.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Ce hook récupère le contexte et l’état de l’écran actuel.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Ce hook lit et gère les erreurs serveur, client et développeur à l’écran.
</ParamField>
