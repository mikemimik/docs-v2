title: "Signup"

---

L’écran `signup` recueille les identifiants de l’utilisateur pour créer un nouveau compte. Il prend en charge le Signup standard avec nom d’utilisateur, courriel, numéro de téléphone et mot de passe, ainsi que le Signup fédéré par l’entremise de connexions sociales ou d’entreprise.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’importation racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez qu’une version unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useSignup } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useSignup,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  usePasswordValidation,
  useSignupIdentifiers,
  useUsernameValidation,
  // Méthodes
  signup,
  federatedSignup,
  pickCountryCode,
} from '@auth0/auth0-acul-react/signup';

function SignupForm() {
  const { signup } = useSignup();
  return (
    <button onClick={() => signup({ email: 'user@example.com', password: 'secret' })}>
      Create Account
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Des hooks à portée d’écran qui fournissent un accès en lecture seule aux données de contexte d’Auth0 sur l’écran `signup`. Importez-les depuis `@auth0/auth0-acul-react/signup`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit des configurations d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `signup`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/signup';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `signup`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/signup';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si l’inscription est associée à une Organization. Retourne `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/signup';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/signup';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnSignup">ScreenMembersOnSignup</a></span>}>
  Ce hook contient des détails propres à l’écran `signup`, notamment sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/signup';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/signup';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembersOnSignup">TransactionMembersOnSignup</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `signup`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/signup';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme une adresse courriel ou un username prérempli à partir des paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/signup';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des détails sur l’utilisateur actif, y compris le `username`, l’`email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/signup';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useSignup" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/SignupMembers">SignupMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte accessibles sur l’écran `signup`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="federatedSignup" type="Promise<void>">
  Cette méthode lance l’inscription au moyen d’un fournisseur d’identité fédéré, comme Google ou GitHub.

  ```jsx Example
  import { useSignup } from '@auth0/auth0-acul-react/signup';

  function SocialButton() {
    const { federatedSignup } = useSignup();
    return (
      <button onClick={() => federatedSignup({ connection: 'google-oauth2' })}>
        Sign up with Google
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [FederatedSignupPayloadOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/FederatedSignupPayloadOptions).
    </ParamField>

    <ParamField body="connection" type="string" required>
      Le nom de la connexion sociale ou d’entreprise à utiliser (par ex. `google-oauth2`, `github`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  Cette méthode ouvre le sélecteur d’indicatif de pays pour la saisie du numéro de téléphone pendant l’inscription.

  ```jsx Example
  import { useSignup } from '@auth0/auth0-acul-react/signup';

  function CountryCodePicker() {
    const { pickCountryCode } = useSignup();
    return (
      <button onClick={() => pickCountryCode()}>
        Select Country Code
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [CustomOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Champs personnalisés supplémentaires préfixés par `ulp-` (par exemple, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="signup" type="Promise<void>">
  Cette méthode envoie les renseignements d’inscription de l’utilisateur pour créer un nouveau compte.

  ```jsx Example
  import { useSignup } from '@auth0/auth0-acul-react/signup';

  function SignupForm() {
    const { signup } = useSignup();
    return (
      <button onClick={() => signup({ email: 'user@example.com', password: 'secret' })}>
        Create Account
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [SignupPayloadOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/SignupPayloadOptions).
    </ParamField>

    <ParamField body="email" type="string">
      L’adresse courriel de l’utilisateur.
    </ParamField>

    <ParamField body="password" type="string">
      Le mot de passe du nouveau compte.
    </ParamField>

    <ParamField body="username" type="string">
      Le nom d’utilisateur du nouveau compte, si la connexion l’exige.
    </ParamField>

    <ParamField body="phoneNumber" type="string">
      Le numéro de téléphone de l’utilisateur, si la connexion l’exige.
    </ParamField>

    <ParamField body="captcha?" type="string">
      Valeur captcha requise lorsque Bot Detection est activé sur le tenant.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Champs personnalisés supplémentaires préfixés par `ulp-` (par exemple, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<div id="commonutility-hooks">
  ## Hooks communs/utilitaires
</div>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Ce hook récupère les options du thème actuel avec une configuration mise à plat à partir du contexte d’image de marque.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Ce hook renvoie une fonction permettant de changer la langue d’affichage sur l’écran ACUL actuel.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Ce hook récupère le contexte et l’état de l’écran actuel.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Ce hook lit et gère les erreurs serveur, client et de développement à l’écran.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation">usePasswordValidation</a>} type="Hooks">
  Ce hook valide le mot de passe de l’utilisateur en fonction des exigences de la politique de mot de passe du tenant.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useSignupIdentifiers">useSignupIdentifiers</a>} type="Hooks">
  Ce hook renvoie les types d’identifiant d’inscription disponibles (courriel, nom d’utilisateur, téléphone) configurés pour la connexion.
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useUsernameValidation">useUsernameValidation</a>} type="Hooks">
  Ce hook valide le nom d’utilisateur selon les exigences de la politique de nom d’utilisateur du tenant.
</ParamField>
