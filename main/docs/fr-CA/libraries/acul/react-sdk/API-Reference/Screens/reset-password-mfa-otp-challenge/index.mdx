---

L’écran `reset-password-mfa-otp-challenge` s’affiche pendant le processus de réinitialisation du mot de passe lorsqu’une authentification multifacteur par mot de passe à usage unique (OTP) est requise. Il invite l’utilisateur à saisir l’OTP de son application d’authentification pour poursuivre la réinitialisation.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation racine** pour chaque écran.

* L’**importation partielle** vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’**importation racine** vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez une version unifiée capable de gérer tous les écrans possibles.

```jsx Import Example
// importation racine
import { useResetPasswordMfaOtpChallenge } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useResetPasswordMfaOtpChallenge,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';

function ResetPasswordMfaOtpChallengeScreen() {
  const { continueMethod } = useResetPasswordMfaOtpChallenge();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Vérifier le code
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks de portée écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `reset-password-mfa-otp-challenge`. Importez-les depuis `@auth0/auth0-acul-react/reset-password-mfa-otp-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les paramètres d’image de marque, comme le logo, les couleurs et les réglages du thème affichés sur l’écran `reset-password-mfa-otp-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `reset-password-mfa-otp-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’organisation de l’utilisateur si la réinitialisation du mot de passe est limitée à cette organisation. Il renvoie `null` lorsqu’aucun contexte d’organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Aucun contexte d’Organisation</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnResetPasswordMfaOtpChallenge">ScreenMembersOnResetPasswordMfaOtpChallenge</a></span>}>
  Ce hook contient des renseignements propres à l’écran `reset-password-mfa-otp-challenge`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';
  function ScreenDebug() {

  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>

  ```jsx Example
  function TenantInfo() {
  ```

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';
  ```

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>


<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode envoie le code OTP pour passer à l’étape d’AMF du processus de réinitialisation du mot de passe.

  ```jsx Example
  import { useResetPasswordMfaOtpChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';

  function VerifyOtpForm() {
    const { continueMethod } = useResetPasswordMfaOtpChallenge();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Vérifier le code
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ResetPasswordMfaOtpChallengeContinueOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordMfaOtpChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code OTP saisi par l’utilisateur.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode permet à l’utilisateur de choisir une autre méthode d’AMF.

  ```jsx Example
  import { useResetPasswordMfaOtpChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-otp-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useResetPasswordMfaOtpChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Essayer une autre méthode
      </button>
    );
  }
  ```
</ParamField>
