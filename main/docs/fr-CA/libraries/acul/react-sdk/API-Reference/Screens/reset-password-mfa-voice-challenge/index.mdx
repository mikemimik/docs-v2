L’écran `reset-password-mfa-voice-challenge` s’affiche pendant le flux de réinitialisation du mot de passe lorsque l’authentification multifacteur par appel vocal est requise. Il invite l’utilisateur à saisir le code de vérification envoyé à son téléphone pour poursuivre la réinitialisation.

<div id="import">
  ## Importation
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation à la racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’importation à la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez une build unifiée pour prendre en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useResetPasswordMfaVoiceChallenge,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useResend,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
  resendCode,
  switchToSms,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

function ResetPasswordMfaVoiceChallengeScreen() {
  const { continueMethod } = useResetPasswordMfaVoiceChallenge();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Des hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `reset-password-mfa-voice-challenge`. Importez-les depuis `@auth0/auth0-acul-react/reset-password-mfa-voice-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit des configurations d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `reset-password-mfa-voice-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `reset-password-mfa-voice-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si la réinitialisation du mot de passe est limitée à une Organization. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode envoie le code de vérification par appel vocal pour passer à l’étape d’AMF du flux de réinitialisation du mot de passe.

  ```jsx Example
  import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

  function VerifyCodeForm() {
    const { continueMethod } = useResetPasswordMfaVoiceChallenge();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Verify Code
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ResetPasswordMfaVoiceChallengeContinueOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordMfaVoiceChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code de vérification par appel vocal saisi par l’utilisateur.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Cette méthode renvoie le code de vérification par appel vocal au numéro de téléphone de l’utilisateur.

  ```jsx Example
  import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

  function ResendCodeButton() {
    const { resendCode } = useResetPasswordMfaVoiceChallenge();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  Cette méthode fait passer le mode de livraison de l’AMF de l’appel vocal au SMS.

  ```jsx Example
  import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

  function SwitchToSmsButton() {
    const { switchToSms } = useResetPasswordMfaVoiceChallenge();
    return (
      <button onClick={() => switchToSms()}>
        Switch to SMS
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode redirige l’utilisateur pour qu’il sélectionne une autre méthode d’AMF.

  ```jsx Example
  import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useResetPasswordMfaVoiceChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>
