L’écran `mfa-email-list` affiche la liste des adresses courriel pouvant être utilisées pour la vérification MFA et permet à l’utilisateur d’en sélectionner une.

<div id="import">
  ## Import
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**import partiel** et l’**import à la racine** pour chaque écran.

* L’import partiel vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’import à la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez qu’une compilation unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useMfaEmailList } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useMfaEmailList,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  goBack,
  selectMfaEmail,
} from "@auth0/auth0-acul-react/mfa-email-list";

function MfaEmailListScreen() {
  const { selectMfaEmail } = useMfaEmailList();
  return (
    <button onClick={() => selectMfaEmail({ index: 0 })}>
      Select Email
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Des hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `mfa-email-list`. Importez-les depuis `@auth0/auth0-acul-react/mfa-email-list`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations d’image de marque, comme le logo, les couleurs et les paramètres du thème affichés sur l’écran `mfa-email-list`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-email-list';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `mfa-email-list`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-email-list';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des informations sur l’organization de l’utilisateur si le flux MFA est limité à l’organization. Il renvoie `null` lorsqu’aucun contexte d’organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-email-list';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-email-list';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaEmailList">ScreenMembersOnMfaEmailList</a></span>}>
  Ce hook contient des détails propres à l’écran `mfa-email-list`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-email-list';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-email-list';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `mfa-email-list`, comme l’état actuel du flux MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-email-list';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme les valeurs préremplies provenant des paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-email-list';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des détails sur l’utilisateur actif, notamment son `username`, son `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-email-list';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaEmailList" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaEmailListMembers">MfaEmailListMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte disponibles dans l’écran `mfa-email-list`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="goBack" type="Promise<void>">
  Cette méthode ramène à l’écran précédent du flux MFA.

  ```jsx Example
  import { useMfaEmailList } from '@auth0/auth0-acul-react/mfa-email-list';

  function BackButton() {
    const { goBack } = useMfaEmailList();
    return (
      <button onClick={() => goBack()}>
        Back
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="selectMfaEmail" type="Promise<void>">
  Cette méthode sélectionne dans la liste l’adresse courriel à utiliser pour la vérification MFA.

  ```jsx Example
  import { useMfaEmailList } from '@auth0/auth0-acul-react/mfa-email-list';

  function EmailSelector() {
    const { selectMfaEmail } = useMfaEmailList();
    return (
      <button onClick={() => selectMfaEmail({ index: 0 })}>
        Use This Email
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [SelectMfaEmailOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/SelectMfaEmailOptions).
    </ParamField>

    <ParamField body="index" type="number" required>
      L’index de l’adresse courriel à sélectionner.
    </ParamField>
  </Expandable>
</ParamField>
