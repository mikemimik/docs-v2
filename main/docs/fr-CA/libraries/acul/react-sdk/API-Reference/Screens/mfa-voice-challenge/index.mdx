L’écran `mfa-voice-challenge` invite l’utilisateur à saisir un code à usage unique transmis par appel vocal pour terminer la vérification AMF.

<div id="import">
  ## Importation
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**importation partielle** et l’**importation à la racine** pour chaque écran.

* L’utilisation de l’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’utilisation de l’importation à la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous voulez un build unifié capable de gérer tous les écrans possibles.

```jsx Import Example
// importation racine
import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useMfaVoiceChallenge,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  useResend,
  // Méthodes
  continueMethod,
  pickPhone,
  resendCode,
  switchToSms,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-voice-challenge";

function MfaVoiceChallengeScreen() {
  const { continueMethod } = useMfaVoiceChallenge();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Vérifier le code
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `mfa-voice-challenge`. Importez-les depuis `@auth0/auth0-acul-react/mfa-voice-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit des paramètres d’image de marque, comme le logo, les couleurs et les réglages du thème affichés sur l’écran `mfa-voice-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des paramètres liés au client, comme `id`, `name` et `logoUrl`, pour l’écran `mfa-voice-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si le flux AMF est limité à l’Organization. Il renvoie `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaVoiceChallenge">ScreenMembersOnMfaVoiceChallenge</a></span>}>
  Ce hook contient des détails propres à l’écran `mfa-voice-challenge`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `mfa-voice-challenge`, comme l’état actuel du flux AMF.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembersOnMfaVoiceChallenge">UntrustedDataMembersOnMfaVoiceChallenge</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme les valeurs préremplies à partir des paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des informations sur l’utilisateur actif, y compris le `username`, l’`email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaVoiceChallenge" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaVoiceChallengeMembers">MfaVoiceChallengeMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte disponibles sur l’écran `mfa-voice-challenge`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode soumet le code à usage unique reçu par appel vocal pour terminer le défi AMF.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function VerifyCodeForm() {
    const { continueMethod } = useMfaVoiceChallenge();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Vérifier le code
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaVoiceChallengeContinueOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaVoiceChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code de vérification reçu par appel vocal.
    </ParamField>

    <ParamField body="rememberDevice" type="boolean">
      Indicateur facultatif permettant de mémoriser l’appareil pendant 30 jours et d’éviter les futurs défis AMF.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickPhone" type="Promise<void>">
  Cette méthode permet d’accéder à l’écran de sélection du numéro de téléphone.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function PickPhoneButton() {
    const { pickPhone } = useMfaVoiceChallenge();
    return (
      <button onClick={() => pickPhone()}>
        Utiliser un autre numéro de téléphone
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Cette méthode renvoie le code à usage unique par appel vocal au téléphone de l’utilisateur.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function ResendCodeButton() {
    const { resendCode } = useMfaVoiceChallenge();
    return (
      <button onClick={() => resendCode()}>
        Renvoyer le code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  Cette méthode fait passer le défi AMF à une livraison par SMS au lieu d’un appel vocal.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function SwitchToSmsButton() {
    const { switchToSms } = useMfaVoiceChallenge();
    return (
      <button onClick={() => switchToSms()}>
        Passer au SMS
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode permet d’accéder à l’écran de sélection de la méthode AMF afin que l’utilisateur puisse choisir un autre facteur d’authentification.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaVoiceChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Essayer une autre méthode
      </button>
    );
  }
  ```
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
