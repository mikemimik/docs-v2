L’écran `device-code-activation` s’affiche pendant le Device Authorization Flow. Il invite l’utilisateur à saisir le code d’appareil affiché sur son appareil pour l’autoriser.

<div id="import">
  ## Importation
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge les **partial import** et les **root import** pour chaque écran.

* L’utilisation de **partial import** vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’utilisation de **root import** vous permet de charger tous les écrans à partir d’un seul lot, ce qui est utile si vous souhaitez qu’une compilation unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useDeviceCodeActivation } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useDeviceCodeActivation,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
} from '@auth0/auth0-acul-react/device-code-activation';

function DeviceCodeActivationScreen() {
  const { continueMethod } = useDeviceCodeActivation();
  return (
    <button onClick={() => continueMethod({ code: 'ABCD-1234' })}>
      Activate Device
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks à portée d’écran offrant un accès en lecture seule aux données de contexte Auth0 sur l’écran `device-code-activation`. Importez-les depuis `@auth0/auth0-acul-react/device-code-activation`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations de branding, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `device-code-activation`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/device-code-activation';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des configurations liées au client, comme `id`, `name` et `logoUrl`, pour l’écran `device-code-activation`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/device-code-activation';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organization de l’utilisateur si l’activation de l’appareil est associée à une Organization. Il retourne `null` lorsqu’aucun contexte d’Organization n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/device-code-activation';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur le prompt actuel dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/device-code-activation';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Ce hook contient des détails propres à l’écran `device-code-activation`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/device-code-activation';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au tenant, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/device-code-activation';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `device-code-activation`, comme les connections actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/device-code-activation';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme un code d’appareil prérempli à partir de paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/device-code-activation';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des détails sur l’utilisateur actif, y compris son `username`, son `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/device-code-activation';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useDeviceCodeActivation" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/DeviceCodeActivationMembers">DeviceCodeActivationMembers</a>}>
  Ce hook renvoie toutes les méthodes et tout le contexte disponibles sur l’écran `device-code-activation`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode envoie le code d’appareil saisi par l’utilisateur afin d’autoriser l’appareil.

  ```jsx Example
  import { useDeviceCodeActivation } from '@auth0/auth0-acul-react/device-code-activation';

  function DeviceCodeForm() {
    const { continueMethod } = useDeviceCodeActivation();
    return (
      <button onClick={() => continueMethod({ code: 'ABCD-1234' })}>
        Activer l’appareil
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [DeviceCodeActivationContinueOptions](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/DeviceCodeActivationContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code d’appareil saisi par l’utilisateur pour autoriser l’appareil.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
