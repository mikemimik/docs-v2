<ParamField body="usePasskeyAutofill()" type={<span><a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsePasskeyAutofillResult">UsePasskeyAutofillResult</a></span>}>
  Hook React qui active l’interface conditionnelle du navigateur (remplissage automatique par passkey) pour les
  champs d’identifiant de connexion dans les screens Auth0 Advanced Customization of Universal Login (ACUL).

  Ce hook initialise l’API Conditional Mediation du navigateur, qui affiche les
  passkeys d’appareil enregistrées dans la liste déroulante de saisie semi-automatique du champ username, sans nécessiter
  d’interaction supplémentaire de l’utilisateur.

  * Le hook s’enregistre **une seule fois par cycle de vie de la page** — vous pouvez l’appeler plusieurs fois sans risque.
  * **Se dégrade gracieusement** sur les navigateurs non pris en charge sans bloquer l’interaction de l’utilisateur.
  * Associer le `inputRef` renvoyé à un élément input garantit que le bon
    attribut `autocomplete="username webauthn"` est appliqué automatiquement.
  * Si votre input déclare déjà `autocomplete="username webauthn"` dans le balisage,
    **associer la ref est facultatif** — le hook s’enregistre quand même correctement.

  ### Fonctionnalités clés

  * **Conditional Mediation automatique** — initialise le remplissage automatique par passkey sans
    aucun appel manuel à l’API du navigateur.
  * **Ref facultative** — fonctionne avec ou sans association de `inputRef` à l’élément input.
  * **Solution de secours sans friction** — les environnements non pris en charge sont gérés silencieusement,
    ce qui préserve le login flow normal.

  ## Paramètres

  Aucun.

  ## Retour

  [`UsePasskeyAutofillResult`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsePasskeyAutofillResult)

  Objet contenant :

  * `inputRef` *(facultatif)* — une ref React à attacher à l’élément input username. Garantit
    que le bon attribut `autocomplete` est défini lorsqu’elle est associée.

  ## Écrans pris en charge

  * `login-id`

  ```tsx Example
  import { usePasskeyAutofill } from '@auth0/auth0-acul-react/login-id';

  export function LoginIdentifierInput() {
    const { inputRef } = usePasskeyAutofill();

    return (
      <input
        ref={inputRef}
        type="text"
        name="username"
        placeholder="Email or username"
      />
    );
  }
  ```

  ## Remarques

  * Si `inputRef` n’est pas associé à un input, assurez-vous que l’élément input déclare explicitement
    `autocomplete="username webauthn"` afin que le navigateur puisse l’associer au remplissage automatique par passkey.
  * Le hook n’a aucun effet dans les environnements où l’API Conditional Mediation n’est pas disponible
    (p. ex. les navigateurs plus anciens ou les contextes non HTTPS) — aucune erreur n’est générée.
  * Appelez `usePasskeyAutofill` une seule fois au niveau supérieur de votre composant; ne l’appelez pas
    de façon conditionnelle ni dans des gestionnaires d’événements.
