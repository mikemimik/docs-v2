<ParamField body="useSignupIdentifiers()" type={<span>[] | <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier">Identifier[]</a></span>}>
  Renvoie une liste des identifiants activés (courriel, téléphone ou nom d&#39;utilisateur), chacun avec son statut `required`,
  selon les identifiants d&#39;inscription de l&#39;écran actuel.

  ## Renvoie

  [] | [`Identifier`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier)[]

  Un tableau d&#39;objets [Identifier](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier), où chacun contient un `type` (type d&#39;identifiant)
  et un indicateur `required` précisant s&#39;il est obligatoire pour l&#39;inscription.

  ## Écrans pris en charge

  * `signup`
  * `signup-id`

  ```tsx Exemple
  import { useSignupIdentifiers } from '@auth0/auth0-acul-react/signup';

  const identifiers = useSignupIdentifiers();
  const emailIdentifier = identifiers.find(({ type }) => type === 'email');
  const phoneIdentifier = identifiers.find(({ type }) => type === 'phone');
  const usernameIdentifier = identifiers.find(({ type }) => type === 'username');

  const emailRequired = emailIdentifier?.required ?? false;
  const phoneRequired = phoneIdentifier?.required ?? false;
  const usernameRequired = usernameIdentifier?.required ?? false;

  // Exemple de sortie :
  // [
  //   { type: 'email', required: true },
  //   { type: 'phone', required: false },
  //   { type: 'username', required: true },
  // ]
  ```
</ParamField>
