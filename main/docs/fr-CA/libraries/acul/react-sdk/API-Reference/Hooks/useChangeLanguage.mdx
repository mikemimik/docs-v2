---
title: "useChangeLanguage"
---

<ParamField body="useChangeLanguage" type="(options: LanguageChangeOptions) => Promise<void>">
  Hook React qui renvoie une fonction permettant de changer la langue d’affichage sur le `screen` ACUL actuel. Le `screen` est automatiquement affiché de nouveau dans la nouvelle `locale` après la soumission. La langue doit être l’une des `locales` activées dans votre `tenant` Auth0.

  ### Principales fonctionnalités

  * **Compatible avec le `tenant`** — accepte uniquement les `locales` activées dans la configuration de votre `tenant` Auth0.
  * **Nouveau rendu automatique** — le `screen` se met à jour dans la nouvelle langue une fois le changement soumis.
  * **Persistance de la session** — la préférence linguistique est enregistrée pour la durée de la `session` par défaut.

  ## Paramètres

  La fonction renvoyée accepte un seul argument [`LanguageChangeOptions`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/LanguageChangeOptions) :

  <ParamField body="language" type="string" required>
    Le code de `locale` vers lequel basculer. Il doit correspondre à l’une des `locales` activées dans votre `tenant` Auth0. Exemple : `"en"`, `"fr"`, `"es"`.
  </ParamField>

  <ParamField body="persist" type="&#x22;session&#x22;">
    Portée de persistance de la préférence linguistique. La valeur par défaut est `"session"`.
  </ParamField>

  ## Valeur renvoyée

  `(options: LanguageChangeOptions) => Promise<void>`

  Une fonction qui soumet le changement de langue. La promesse renvoyée est résolue lorsque la soumission est terminée.

  ```tsx Example
  import { useChangeLanguage } from "@auth0/auth0-acul-react";

  export function LanguageSwitcher() {
    const changeLanguage = useChangeLanguage();

    return (
      <div>
        <button onClick={() => changeLanguage({ language: "en" })}>English</button>
        <button onClick={() => changeLanguage({ language: "fr" })}>Français</button>
        <button onClick={() => changeLanguage({ language: "es" })}>Español</button>
      </div>
    );
  }
  ```

  Vous pouvez aussi remplir les options dynamiquement à partir de la configuration du `tenant` :

  ```tsx
  import { useChangeLanguage, useCurrentScreen } from "@auth0/auth0-acul-react";

  export function LocaleSelector() {
    const changeLanguage = useChangeLanguage();
    const { tenant, transaction } = useCurrentScreen();

    return (
      <select
        defaultValue={transaction.locale}
        onChange={e => changeLanguage({ language: e.target.value })}
      >
        {tenant.enabledLocales.map(locale => (
          <option key={locale} value={locale}>{locale}</option>
        ))}
      </select>
    );
  }
  ```

  ## Remarques

  * La valeur `language` doit correspondre à une `locale` activée dans votre `tenant` Auth0 — transmettre une `locale` non prise en charge entraînera une erreur.
  * Utilisez `useCurrentScreen()` pour accéder à `tenant.enabledLocales` et à `transaction.locale` afin de sélectionner la `locale` dynamiquement.
  * Appelez `useChangeLanguage` au niveau supérieur de votre composant; ne l’appelez pas de façon conditionnelle ni dans des gestionnaires d’événements.
</ParamField>
