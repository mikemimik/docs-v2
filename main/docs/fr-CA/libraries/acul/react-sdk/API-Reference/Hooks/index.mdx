<div id="methods">
  ## Méthodes
</div>

* [useAuth0Themes](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes) : hook React permettant d’obtenir les options du thème actuel avec une configuration aplatie provenant du contexte d’image de marque.
* [useChangeLanguage](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage) : hook React qui retourne une fonction permettant de changer la langue d’affichage sur l’écran ACUL actuel.
* [useCurrentScreen](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen) : hook React qui récupère le contexte et l’état de l’écran actuel.
* [useErrors](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) : hook React permettant de lire et de gérer les erreurs dans ACUL (Advanced Customization of Universal Login), y compris toutes les erreurs de validation et les erreurs côté serveur. Il regroupe les erreurs en trois catégories : - `server` — erreurs renvoyées par Auth0 ou votre propre backend. - `client` — erreurs provenant de la validation côté client (p. ex., saisie de formulaire invalide). - `developer` — erreurs causées par une intégration incorrecte ou une mauvaise utilisation du SDK.
* [useLoginIdentifiers](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useLoginIdentifiers) : retourne une liste des types d’identifiant actifs (comme `'email'`, `'phone'` ou `'username'`) actuellement utilisés dans le flux d’authentification ou le processus de connexion.
* [useMfaPolling](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling) : hook React permettant de gérer le polling MFA push (p. ex., en attente de l’approbation d’une push notification) sur un écran Auth0 Advanced Customization of Universal Login (ACUL).
* [useResend](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useResend) : ce hook React gère les actions de « renvoi » (p. ex., le renvoi d’un code de vérification) sur les écrans ACUL.
* [useSignupIdentifiers](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useSignupIdentifiers) : retourne une liste des identifiants activés (courriel, téléphone ou nom d’utilisateur), chacun avec son statut `required`, selon les identifiants d’inscription de l’écran actuel.
* [usePasswordValidation](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation) : ce hook React valide un mot de passe en fonction de la politique de mot de passe actuelle d’Auth0 et retourne un résultat structuré indiquant si le mot de passe respecte chaque règle.
* [useUsernameValidation](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useUsernameValidation) : hook React permettant de valider un nom d’utilisateur en fonction de la politique de nom d’utilisateur actuelle d’Auth0.
