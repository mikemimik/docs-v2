PickPhoneOptions
Définit les options de l’action &#39;pick-phone&#39;.
Bien que l’interface n’affiche qu’un seul numéro de téléphone, cette action peut être utilisée
dans certains scénarios où le serveur doit rediriger vers un écran de sélection du numéro de téléphone.

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```
