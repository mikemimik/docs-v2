---
title: "ContinueWithOrganizationNameOptions"
---

Options permettant de poursuivre avec le nom d’organisation sélectionné.

```ts Example
export interface ContinueWithOrganizationNameOptions {
  /**
   * Le nom de l'organisation.
   */
  organizationName: string;
  /**
   * Toute option personnalisée supplémentaire.
   */
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

Toutes les options personnalisées supplémentaires.

  ## Propriétés
</div>

  Le nom de l’organisation.
