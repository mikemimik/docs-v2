---
title: "UsernameValidationResult"
---

Le résultat de la validation d’un nom d’utilisateur selon une ou plusieurs règles.

```ts Example
export interface UsernameValidationResult {
  isValid: boolean;
  errors: UsernameValidationError[];
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="isValid" type="boolean" required>
  Indique si le nom d’utilisateur respecte toutes les règles de validation.
</ParamField>

<ParamField body="errors" type={<span><a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError">UsernameValidationError[]</a></span>} required>
  Un tableau d’objets [UsernameValidationError](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError) représentant
  les règles individuelles qui n’ont pas été respectées, s’il y a lieu.

  Ce tableau est vide si [UsernameValidationResult.isValid](#isvalid) vaut `true`.
</ParamField>
