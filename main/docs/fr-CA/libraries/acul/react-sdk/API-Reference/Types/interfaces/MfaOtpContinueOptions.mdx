---
title: "MfaOtpContinueOptions"
---

Options permettant de poursuivre avec le code d’inscription MFA OTP.

```ts Example
export interface ContinueOptions {
  /**
   * Le code OTP saisi par l'utilisateur.
   */
  code: string;

  /**
   * Toute option personnalisée supplémentaire.
   */
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

Toute autre option personnalisée supplémentaire.

</div>
