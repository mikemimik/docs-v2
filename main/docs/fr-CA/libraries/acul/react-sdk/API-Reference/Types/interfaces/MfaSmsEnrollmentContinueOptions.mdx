---
title: "MfaSmsEnrollmentContinueOptions"
---

Options pour poursuivre l’inscription MFA par SMS.

Étend `CustomOptions` afin de permettre la transmission d’options personnalisées en plus de phone et captcha.

```ts Example
export interface MfaSmsEnrollmentContinueOptions extends CustomOptions {
  phone: string;
  captcha?: string;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propriétés
</div>

<ParamField body="phone" type="string" required>
  Le numéro de téléphone de l’utilisateur pour l’inscription par SMS.
</ParamField>

<ParamField body="captcha?" type="string">
  Jeton CAPTCHA facultatif pour la vérification.
</ParamField>
