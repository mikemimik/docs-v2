---
title: "MfaSmsEnrollmentContinueOptions"
---

Options permettant de poursuivre l’inscription MFA par SMS.

Étend `CustomOptions` afin de permettre le passage d’options personnalisées en plus de `phone` et de `captcha`.

```ts Example
export interface MfaSmsEnrollmentContinueOptions extends CustomOptions {
  phone: string;
  captcha?: string;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propriétés
</div>

<ParamField body="phone" type="string" required>
  Le numéro de téléphone de l’utilisateur pour l’inscription SMS.
</ParamField>

<ParamField body="captcha?" type="string">
  Jeton CAPTCHA facultatif pour vérification.
</ParamField>
