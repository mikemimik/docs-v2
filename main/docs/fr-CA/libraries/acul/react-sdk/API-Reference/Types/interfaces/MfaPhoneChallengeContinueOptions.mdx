ContinueOptions
Définit les options requises pour poursuivre le processus de vérification MFA par téléphone.
Cela consiste généralement à sélectionner le mode de réception du code de vérification (SMS ou appel vocal).

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * Le type de défi à effectuer.
   * 'sms': Envoyer le code par message texte.
   * 'voice': Envoyer le code par appel vocal.
   * @default 'sms'
   */
  type: 'sms' | 'voice';
}
```
