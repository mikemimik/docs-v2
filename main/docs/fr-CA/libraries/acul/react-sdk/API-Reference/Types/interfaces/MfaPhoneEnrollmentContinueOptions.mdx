---
title: "MfaPhoneEnrollmentContinueOptions"
---

Options permettant de poursuivre l’inscription MFA par téléphone.

Étend `CustomOptions` afin de permettre la transmission d’options personnalisées avec phone et type.

```ts Example
export interface ContinueOptions extends CustomOptions {
  phone: string;
  type: 'sms' | 'voice';
}
```
