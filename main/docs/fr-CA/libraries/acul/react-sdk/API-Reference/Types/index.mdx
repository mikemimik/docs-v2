Cette opération est utilisée lorsque l’utilisateur demande qu’un nouveau code de vérification soit envoyé à son adresse courriel, par exemple s’il n’a pas reçu le code initial ou si celui-ci a expiré.
* [LoginEmailVerificationMembers](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/LoginEmailVerificationMembers): Description de LoginEmailVerificationMembers. Définit les membres (propriétés et méthodes) disponibles pour interagir avec l’écran de vérification du courriel à la connexion. Cet écran joue un rôle essentiel dans les processus de vérification du courriel, généralement lors de la connexion, où l’utilisateur doit prouver qu’il est bien propriétaire d’une adresse courriel en fournissant un code à usage unique. Le SDK permet de soumettre ce code ou d’en demander un nouveau.
