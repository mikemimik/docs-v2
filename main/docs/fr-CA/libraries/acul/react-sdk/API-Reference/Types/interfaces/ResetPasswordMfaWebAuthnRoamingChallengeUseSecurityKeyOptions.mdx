UseSecurityKeyOptions

```ts Example
export interface UseSecurityKeyOptions extends CustomOptions {
  /**
   * Facultatif. Si `true`, et si le contexte d'écran (`screen.showRememberDevice`)
   * indique que la mémorisation de l'appareil est une option, cela signale l'intention
   * de l'utilisateur de mémoriser ce navigateur/appareil pour les authentifications futures.
   * Le SDK soumettra `rememberBrowser=true` dans ce cas.
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```
