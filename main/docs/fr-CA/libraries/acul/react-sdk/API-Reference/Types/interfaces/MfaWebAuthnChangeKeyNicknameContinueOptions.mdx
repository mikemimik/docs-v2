ContinueOptions

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * Le nouveau surnom que l'utilisateur souhaite attribuer à sa clé de sécurité WebAuthn.
   * Cette valeur sera envoyée au serveur pour une mise à jour.
   * Le serveur va valider ce surnom (p. ex., longueur, caractères autorisés).
   * @type {string}
   * @example "Primary YubiKey"
   * @example "Home Office Key"
   */
  nickname: string;
}
```
