title: "MfaVoiceChallengeContinueOptions"

Options pour soumettre le code du défi vocal.

```ts Example
export interface MfaVoiceChallengeContinueOptions {
  /**
   * Le code de vérification reçu par appel vocal.
   */
  code: string;

  /**
   * Indicateur optionnel pour mémoriser l'appareil pendant 30 jours, en ignorant les futurs défis AMF.
   */
  rememberDevice?: boolean;

  /**
   * Options personnalisées supplémentaires à transmettre avec la requête.
   */
  [key: string]: string | number | boolean | undefined;
}
```
