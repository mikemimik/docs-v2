---
title: "ContextHooks<T>"
---

Classe de fabrique permettant de créer des hooks de contexte pour accéder aux données d’Auth0 Universal Login.
Ces hooks donnent accès à différents aspects du flux d’authentification ainsi qu’à la configuration du tenant.

<div id="methods">
  ## Méthodes
</div>

<ParamField body="useUser()" type="T[&#x22;user&#x22;]">
  Hook permettant d’accéder aux informations sur l’utilisateur et aux données de profil.

  Objet `user` contenant les informations de profil, les attributs et les données propres à l’utilisateur

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/login-id';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useTenant()" type="T[&#x22;tenant&#x22;]">
  Hook permettant d’accéder à la configuration et aux paramètres du tenant.

  Objet tenant contenant le domaine, la région et la configuration propre au tenant

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/login-id';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useBranding()" type="T[&#x22;branding&#x22;]">
  Hook permettant d’accéder à l’image de marque et à la configuration du thème.

  Objet image de marque contenant les couleurs, les logos, les polices et les paramètres de personnalisation visuelle

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/login-id';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient()" type="T[&#x22;client&#x22;]">
  Hook permettant d’accéder à la configuration de l’application Auth0 (client).

  Objet client contenant les paramètres de l’application, les callbacks et les données propres au client

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/login-id';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization()" type="T[&#x22;organization&#x22;]">
  Hook permettant d’accéder au contexte et aux paramètres de l’organisation.

  Objet Organization contenant des données propres à l’organisation, des métadonnées et de la configuration

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/login-id';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Aucun contexte d’organisation</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt()" type="T[&#x22;prompt&#x22;]">
  Hook permettant d’accéder à la configuration du prompt et aux paramètres du flux.

  Objet prompt contenant la configuration du flux, les paramètres de l’écran et les données propres au prompt

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/login-id';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData()" type="T[&#x22;untrustedData&#x22;]">
  Hook permettant d’accéder aux données non fiables provenant des paramètres d’URL et des soumissions de formulaire.

  Objet contenant des données saisies par l’utilisateur non fiables, qui doivent être validées avant utilisation

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/login-id';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useScreen()" type="T[&#x22;screen&#x22;]">
  Hook permettant d’accéder aux informations et aux métadonnées de l’écran actuel.

  Objet Screen contenant le nom de l’écran actuel, sa configuration et les données propres à l’écran

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/login-id';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTransaction()" type="T[&#x22;transaction&#x22;]">
  Hook permettant d’accéder à l’état de la transaction et aux données du flux d’authentification.

  Objet transaction contenant l’état du flux, les données de session et les informations propres à la transaction

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/login-id';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>
