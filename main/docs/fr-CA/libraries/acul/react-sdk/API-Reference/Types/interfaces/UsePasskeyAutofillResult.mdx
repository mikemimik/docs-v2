Le type de valeur renvoyée par [`usePasskeyAutofill`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/usePasskeyAutofill).

```typescript
interface UsePasskeyAutofillResult {
  inputRef: RefObject<HTMLInputElement>;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  Une référence React à attacher au champ de saisie du nom d’utilisateur. Lorsqu’elle est liée, le SDK configure automatiquement l’attribut `autocomplete` du champ sur `"username webauthn"`, ce qui active l’UI conditionnelle du navigateur pour le remplissage automatique par clé d’accès. Il est facultatif de lier cette référence si `autocomplete="username webauthn"` est déjà déclaré dans votre balisage.
</ParamField>
