---
title: "LanguageChangeOptions"
---

Options acceptées par la fonction retournée par [`useChangeLanguage`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage).

```typescript
interface LanguageChangeOptions {
  language: string;
  persist?: "session";
  [key: string]: string | number | boolean | undefined;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="language" type="string" required>
  Le code de paramètre régional à utiliser. Il doit correspondre à l’un des paramètres régionaux activés dans votre Auth0 tenant. Exemple : `"en"`, `"fr"`, `"es"`.
</ParamField>

<ParamField body="persist" type="&#x22;session&#x22;">
  Portée d’enregistrement de la préférence linguistique. Seule la valeur `"session"` est actuellement prise en charge. La valeur par défaut est `"session"`.
</ParamField>

<div id="remarks">
  ## Remarques
</div>

* Des champs personnalisés supplémentaires peuvent être fournis en plus de `language` et `persist`. Les champs personnalisés doivent être préfixés par `"ulp-"` et accepter des valeurs de type `string`, `number`, `boolean` ou `undefined`.
