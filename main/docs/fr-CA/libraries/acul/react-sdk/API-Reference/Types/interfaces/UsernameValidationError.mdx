---
title: "UsernameValidationError"
---

Représente une erreur de validation du username renvoyée lors de la validation.

Chaque erreur comporte un `code` lisible par machine et un `message` en langage clair.

```ts Example
export interface UsernameValidationError {
  code: string;
  message: string;
  field: string;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="code" type="string" required>
  Un identifiant unique de la règle de validation ayant échoué.

  Exemple : `"username-too-short"`, `"username-invalid-characters"`
</ParamField>

<ParamField body="message" type="string" required>
  Une description de l’erreur facile à comprendre.

  Exemple : `"Username must be at least 3 characters long."`
</ParamField>

<ParamField body="field" type="string" required>
  Le champ associé à l’erreur de validation, généralement &quot;username&quot;, &quot;email&quot; ou &quot;phone&quot;.
</ParamField>
