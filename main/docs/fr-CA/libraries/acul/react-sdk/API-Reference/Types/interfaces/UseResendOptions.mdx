---
title: "UseResendOptions"
---

Configuration optionnelle de [useResend](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendOptions {
  timeoutSeconds?: number;
  onTimeout?: () => void;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="timeoutSeconds" type="number" default="10">
  Durée du compte à rebours (en secondes) avant qu’un nouveau renvoi soit permis.
  La valeur par défaut est `10`.
</ParamField>

<ParamField body="onTimeout" type="() => void">
  La fonction de rappel est déclenchée lorsque le compte à rebours prend fin et que l’action de renvoi redevient disponible.
</ParamField>
