---
title: "UseResendOptions"
---

Configuration optionnelle pour [useResend](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendOptions {
  timeoutSeconds?: number;
  onTimeout?: () => void;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="timeoutSeconds" type="number" default="10">
  Durée du compte à rebours (en secondes) avant qu’un nouvel envoi soit permis.
  La valeur par défaut est `10`.
</ParamField>

<ParamField body="onTimeout" type="() => void">
  Fonction appelée lorsque le compte à rebours se termine et qu’il est de nouveau possible de renvoyer.
</ParamField>
