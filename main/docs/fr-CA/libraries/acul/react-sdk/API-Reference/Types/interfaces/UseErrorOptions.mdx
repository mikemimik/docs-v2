---
title: "UseErrorOptions"
---

Objet d’options pour le hook [`useError`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useError). Contrôle le comportement de l’affichage des erreurs dans les écrans d’Auth0 Universal Login.

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="includeDevErrors" type="boolean">
  Lorsque `true`, inclut les détails d’erreur propres au développement dans la sortie d’erreur. Utile pour le débogage en cours de développement; devrait être désactivé en production.
</ParamField>
