---
title: "UseErrorOptions"
---

Objet d’options pour le hook [`useError`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useError). Contrôle le comportement d’affichage des erreurs dans les écrans d’Auth0 Universal Login.

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="includeDevErrors" type="boolean">
  Lorsque la valeur est `true`, la sortie d’erreur inclut des détails propres au développement. Utile pour le débogage en cours de développement ; devrait être désactivé en production.
</ParamField>
