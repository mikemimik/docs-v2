---
title: "MfaRecoveryCodeEnrollmentMembers"
---

Interface qui décrit les membres de l’écran d’inscription au code de récupération MFA.

```ts Example
export interface MfaRecoveryCodeEnrollmentMembers extends BaseMembers {
  screen: ScreenMembersOnMfaRecoveryCodeEnrollment;

  /**
   * Déclare que l'utilisateur a sauvegardé le code de récupération.
   * Cette action est déclenchée lorsque l'utilisateur déclare avoir sauvegardé le code de récupération.
   * Elle redirige vers l'écran suivant dans le flux d'authentification.
   *
   * @param {object} payload - Un objet contenant les options personnalisées éventuelles.
   *
   * @returns {Promise<void>} Une promesse qui se résout lorsque l'action est soumise avec succès.
   * Rejette avec une erreur si la soumission échoue.
   */
  continue(payload: MfaRecoveryCodeEnrollmentContinueOptions): Promise<void>;
}
```
