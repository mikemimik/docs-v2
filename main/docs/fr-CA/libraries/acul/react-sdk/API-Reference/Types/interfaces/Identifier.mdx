---
title: "Identifiant"
---

Décrit un champ d’identifiant configuré (p. ex. courriel, téléphone ou nom d’utilisateur)
à utiliser dans un formulaire d’inscription ou dans un flux d’authentification.

```ts Example
export interface Identifier {
  type: IdentifierType;
  required: boolean;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="type" type={<span><a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a></span>} required>
  Le type d’identifiant, par exemple `'email'`, `'phone'` ou `'username'`.
</ParamField>

<ParamField body="required" type="boolean" required>
  Indique si cet identifiant est obligatoire pour l’inscription.
</ParamField>
