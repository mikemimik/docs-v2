---
title: "MfaPushEnrollmentQrWithRememberOptions"
---

Options pour poursuivre l’inscription à MFA push avec un paramètre facultatif de confiance de l’appareil.

Étend `CustomOptions` pour permettre de passer des options personnalisées avec la préférence de mémorisation de l’appareil.

```ts Example
export interface MfaPushEnrollmentQrWithRememberOptions extends CustomOptions {
  rememberDevice?: boolean;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propriétés
</div>

<ParamField body="rememberDevice?" type="boolean">
  Lorsqu’il est défini sur `true`, cet appareil est enregistré comme fiable pour que l’utilisateur n’ait pas à effectuer une authentification MFA push lors des connexions ultérieures à partir du même appareil.
</ParamField>
