---
title: "MfaPushEnrollmentQrWithRememberOptions"
---

Options pour poursuivre l’inscription à l’authentification MFA push avec un paramètre facultatif de confiance de l’appareil.

Étend `CustomOptions` pour permettre de transmettre des options personnalisées avec la préférence de mémorisation de l’appareil.

```ts Example
export interface MfaPushEnrollmentQrWithRememberOptions extends CustomOptions {
  rememberDevice?: boolean;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propriétés
</div>

<ParamField body="rememberDevice?" type="boolean">
  S’il est défini à `true`, cet appareil est enregistré comme fiable afin que l’utilisateur ne soit pas invité à effectuer une authentification MFA push lors des connexions suivantes à partir du même appareil.
</ParamField>
