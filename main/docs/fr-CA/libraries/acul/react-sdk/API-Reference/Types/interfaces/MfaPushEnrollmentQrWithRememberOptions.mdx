---
title: "MfaPushEnrollmentQrWithRememberOptions"
---

Options pour poursuivre l’inscription MFA push avec un paramètre facultatif de mémorisation de l’appareil.

Étend `CustomOptions` afin de permettre la transmission d’options personnalisées avec le paramètre de mémorisation de l’appareil.

```ts Example
export interface MfaPushEnrollmentQrWithRememberOptions extends CustomOptions {
  rememberDevice?: boolean;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propriétés
</div>

<ParamField body="rememberDevice?" type="boolean">
  Lorsqu’il est défini sur `true`, cet appareil est enregistré comme fiable afin que l’utilisateur n’ait pas à effectuer une authentification AMF par notification push lors des connexions suivantes à partir du même appareil.
</ParamField>
