---
title: "IdentifierType"
---

Type d’union de chaînes représentant les méthodes valides pour identifier un utilisateur lors de l’authentification. Sert à limiter les entrées d’identifiant à l’une de trois valeurs précises.

```typescript
type IdentifierType = "phone" | "email" | "username";
```

<div id="values">
  ## Valeurs
</div>

<ParamField body="&#x22;phone&#x22;" type="IdentifierType">
  Un identifiant de téléphone.
</ParamField>

<ParamField body="&#x22;email&#x22;" type="IdentifierType">
  Un identifiant d’adresse courriel.
</ParamField>

<ParamField body="&#x22;username&#x22;" type="IdentifierType">
  Un identifiant de nom d’utilisateur.
</ParamField>
