---
title: "IdentifierType"
---

Type d’union de chaînes de caractères représentant les méthodes valides pour identifier un utilisateur lors de l’authentification. Sert à limiter les valeurs d’entrée de l’identifiant à l’une des trois valeurs précises.

```typescript
type IdentifierType = "phone" | "email" | "username";
```

<div id="values">
  ## Valeurs
</div>

<ParamField body="&#x22;phone&#x22;" type="IdentifierType">
  Un identifiant de numéro de téléphone.
</ParamField>

<ParamField body="&#x22;email&#x22;" type="IdentifierType">
  Un identifiant d’adresse courriel.
</ParamField>

<ParamField body="&#x22;username&#x22;" type="IdentifierType">
  Un identifiant de nom d’utilisateur.
</ParamField>
