---
title: "PasswordValidationResult"
---

Type de retour de [`usePasswordValidation`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation). Contient le résultat global de validation ainsi qu’une liste détaillée des règles de complexité du mot de passe évaluées.

```typescript
export interface PasswordValidationResult {
  isValid: boolean;
  results: PasswordComplexityRule[];
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="isValid" type="boolean" required>
  Résultat global de toutes les vérifications de la politique de mot de passe. `true` seulement si chaque règle de [`PasswordComplexityRule`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) a un `status` égal à `"valid"`.
</ParamField>

<ParamField body="results" type={<span><a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>} required>
  Tableau des règles de mot de passe évaluées. Chaque élément est une [`PasswordComplexityRule`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) dont le champ [`status`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule#status) reflète le résultat de validation obtenu pour cette règle.
</ParamField>
