title: "UseResendReturn"
---

Type renvoyé par [useResend](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendReturn {
  remaining: number;
  disabled: boolean;
  startResend: () => void;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="remaining" type="number" required>
  Nombre de secondes restantes avant que le prochain renvoi soit autorisé. Servez-vous-en pour afficher un compte à rebours dans l’UI.
</ParamField>

<ParamField body="disabled" type="boolean" required>
  `true` lorsque le renvoi n’est pas disponible pour le moment, soit parce qu’un renvoi est en cours, soit parce que la période d’attente n’est pas encore écoulée.
</ParamField>

<ParamField body="startResend" type="() => void" required>
  Lance immédiatement une tentative de renvoi si l’action est actuellement autorisée.
</ParamField>
