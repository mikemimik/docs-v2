title: "UseResendReturn"
---

Type de retour de [useResend](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendReturn {
  remaining: number;
  disabled: boolean;
  startResend: () => void;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="remaining" type="number" required>
  Nombre de secondes restantes avant que la prochaine tentative de renvoi soit autorisée. Servez-vous-en pour afficher un compte à rebours dans l’UI.
</ParamField>

<ParamField body="disabled" type="boolean" required>
  `true` lorsque l’action de renvoi est momentanément indisponible, soit parce qu’un renvoi est en cours, soit parce que le délai d’attente n’est pas écoulé.
</ParamField>

<ParamField body="startResend" type="() => void" required>
  Déclenche immédiatement une tentative de renvoi si l’action est actuellement autorisée.
</ParamField>
