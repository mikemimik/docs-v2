ScreenMembersOnCustomizedConsent

```ts Example
export interface ScreenMembersOnCustomizedConsent extends ScreenMembers {
  /**
   * Un tableau d'objets `Scope` détaillant chaque permission (scope) demandée.
   * Il s'agit généralement de permissions de haut niveau comme 'read:profile' ou 'openid'.
   * Cette liste doit être affichée à l'utilisateur pour qu'il puisse la consulter.
   */
  scopes: Scope[];

  /**
   * Un tableau d'objets `AuthorizationDetail`.
   * Chaque objet fournit des détails précis sur des données spécifiques ou
   * des actions que l'application souhaite effectuer, potentiellement liées aux
   * Rich Authorization Requests (RAR) ou à d'autres modèles de permission à granularité fine.
   * Cette liste doit être affichée à l'utilisateur pour qu'il puisse la consulter.
   */
  authorizationDetails: AuthorizationDetail[];
}
```
