---
title: "PasswordComplexityRule"
---

Représente une règle de complexité du mot de passe évaluée. Elle est renvoyée dans [`PasswordValidationResult`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordValidationResult) par [`usePasswordValidation`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation). Elle prend en charge la composition hiérarchique de règles au moyen du tableau `items` facultatif.

```typescript
export interface PasswordComplexityRule {
  code: string;
  label: string;
  status: "valid" | "error";
  isValid: boolean;
  args?: {
    count: number;
    total?: number;
    example?: string;
  };
  items?: PasswordComplexityRule[];
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="code" type="string" required>
  Un identifiant lisible par machine pour cette règle de complexité. Exemple : `"lengthAtLeast"`, `"containsAtLeast"`.
</ParamField>

<ParamField body="label" type="string" required>
  Un nom d’affichage lisible par l’humain pour la règle, pouvant être affiché dans un indicateur de robustesse du mot de passe.
</ParamField>

<ParamField body="status" type="&#x22;valid&#x22; | &#x22;error&#x22;" required>
  Le résultat de la validation pour cette règle. `"valid"` si la règle est respectée, `"error"` si elle ne l’est pas.
</ParamField>

<ParamField body="isValid" type="boolean" required>
  Représentation booléenne du résultat de la règle. `true` lorsque `status` est `"valid"`.
</ParamField>

<ParamField body="args" type="object">
  Paramètres supplémentaires utilisés pour décrire le seuil de la règle ou fournir du contexte au message d’erreur.

  <Expandable title="propriétés">
    <ParamField body="count" type="number" required>
      Le seuil numérique ou le nombre associé à cette règle. Exemple : nombre minimal de caractères.
    </ParamField>

    <ParamField body="total" type="number">
      Le nombre total de caractères ou d’éléments pertinents pour cette règle, s’il y a lieu.
    </ParamField>

    <ParamField body="example" type="string">
      Une valeur d’exemple illustrant ce qui permet de respecter cette règle, s’il y a lieu.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="items" type={<span><a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>}>
  Règles de complexité imbriquées pour les groupes de règles composites. Permet une composition hiérarchique dans laquelle une règle parente regroupe plusieurs règles enfants.
</ParamField>
