Objet de résultat retourné par [useMfaPolling](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling).

<div id="properties">
  ## Propriétés
</div>

<ParamField body="isRunning" type="boolean">
  Indique si le processus d’interrogation du push d’AMF est actuellement actif.

  * `true` — L’interrogation est en cours et attend qu’un résultat soit obtenu.
  * `false` — L’interrogation s’est arrêtée, soit parce qu’elle est terminée,
    soit à la suite d’une annulation manuelle ou du démontage du composant.
</ParamField>

<ParamField body="startPolling()" type="void">
  Démarre ou reprend le processus d’interrogation.

  * Si l’interrogation est déjà active, cet appel n’a aucun effet.
  * Si elle a déjà été arrêtée, cet appel redémarre la boucle d’interrogation.
</ParamField>

<ParamField body="stopPolling()" type="void">
  Arrête immédiatement le processus d’interrogation.

  * Annule toutes les minuteries planifiées ou les requêtes en cours.
  * Peut être appelé plusieurs fois ; les appels suivants n’ont aucun effet.
</ParamField>
