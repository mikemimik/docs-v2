Objet de résultat renvoyé par [useMfaPolling](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling).

<div id="properties">
  ## Propriétés
</div>

<ParamField body="isRunning" type="boolean">
  Indique si le processus de polling de la MFA push est actuellement actif.

  * `true` — Le polling est en cours et attend la fin de l’opération.
  * `false` — Le polling s’est arrêté, soit parce que l’opération est terminée,
    soit à la suite d’une annulation manuelle ou du démontage du composant.
</ParamField>

<ParamField body="startPolling()" type="void">
  Démarre ou reprend le processus de polling.

  * Si le polling est déjà actif, cet appel n’a aucun effet.
  * S’il a été arrêté auparavant, cet appel redémarre la boucle de polling.
</ParamField>

<ParamField body="stopPolling()" type="void">
  Arrête immédiatement le processus de polling.

  * Annule toute minuterie planifiée ou requête en cours.
  * Peut être appelée sans risque plusieurs fois; les appels suivants n’ont aucun effet.
</ParamField>
