---
title: "ErrorsResult"
---

Un objet semblable à un tableau renvoyé dans [`UseErrorsResult`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorsResult) par le hook [`useErrors`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Il étend `ReadonlyArray<ErrorItem>`, donc il peut être parcouru, utilisé avec l’opérateur spread ou indexé directement. Il fournit deux méthodes pour interroger la liste d’erreurs par type ou par champ.

```typescript
interface ErrorsResult extends ReadonlyArray<ErrorItem> {
  byType(type: ErrorType, opts?: { field?: string }): readonly ErrorItem[];
  byField(field: string, opts?: { type?: ErrorType }): readonly ErrorItem[];
}
```

<ParamField body="byType" type={<span>readonly <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem">ErrorItem</a>[]</span>}>
  Retourne toutes les erreurs correspondant au [`ErrorType`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType) indiqué, avec filtrage facultatif selon un champ précis.

  <Expandable title="paramètres">
    <ParamField body="type" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>} required>
      Le type d’erreur à utiliser pour le filtrage : `"auth0"`, `"validation"` ou `"configuration"`.
    </ParamField>

    <ParamField body="opts" type="{ field?: string }">
      Options de filtre facultatives.

      <Expandable title="propriétés">
        <ParamField body="field" type="string">
          S’il est fourni, filtre davantage les résultats pour ne conserver que les erreurs associées à ce nom de champ.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="byField" type={<span>readonly <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem">ErrorItem</a>[]</span>}>
  Retourne toutes les erreurs associées au nom de champ indiqué, avec filtrage facultatif selon un [`ErrorType`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType) précis.

  <Expandable title="paramètres">
    <ParamField body="field" type="string" required>
      Le nom de champ à utiliser pour le filtrage. Exemple : `"username"`, `"email"`.
    </ParamField>

    <ParamField body="opts" type={<span>&#123; type?: <a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a> &#125;</span>}>
      Options de filtre facultatives.

      <Expandable title="propriétés">
        <ParamField body="type" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>}>
          S’il est fourni, filtre davantage les résultats pour ne conserver que les erreurs de ce type.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>
