---
title: "Alias de type : MfaPollingOptions"
---

```ts Example
type MfaPollingOptions = {
  intervalMs?: number;
  onCompleted?: () => void;
  onError?: (error: Error) => void;
}
```

<ParamField body="MfaPollingOptions" type="object">
  Options de configuration pour gérer un processus de polling des notifications push d’AMF.

  Fournissez ces options au démarrage d’une session de polling pour contrôler la fréquence
  à laquelle le point de terminaison du défi est interrogé et la façon dont la réussite ou l’échec sont gérés.

  <Expandable title="propriétés">
    <ParamField body="intervalMs" type="number">
      Intervalle facultatif, en **millisecondes**, entre des requêtes de polling consécutives.

      * S’il est omis, l’intervalle interne par défaut du SDK est utilisé (habituellement 5000 ms).
      * Choisissez une valeur qui offre un bon équilibre entre la réactivité et les limites de débit.

      ```ts Valeur par défaut
      Selon la mise en œuvre (généralement 5000 ms)
      ```
    </ParamField>

    <ParamField body="onCompleted" type="() => void">
      Fonction de rappel facultative exécutée une fois que le défi push d’AMF a été
      approuvé avec succès et que le polling est terminé.

      * Appelée exactement **une fois**, après quoi le polling s’arrête automatiquement.
      * Utile pour faire avancer le flux de connexion (p. ex., appeler `continue()`).
    </ParamField>

    <ParamField body="onError" type="(error: Error) => void">
      Fonction de rappel facultative invoquée si une erreur survient pendant le polling.

      Reçoit un objet [ULError](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error) contenant :

      * `status` — le code d’état HTTP de la requête ayant échoué

      * `responseText` — le corps brut de la réponse, s’il est disponible

      * Appelée une fois par événement d’erreur; le polling peut être réessayé selon le type d’erreur.

      * À utiliser pour afficher des messages d’erreur ou déclencher une logique personnalisée de nouvelle tentative ou d’abandon.

      **Propriétés**

      <ParamField body="error" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error">Error</a>}>
        Un objet d’erreur contenant le code d’état HTTP et le corps brut de la réponse de la requête de polling ayant échoué.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>
