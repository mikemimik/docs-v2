title: "Alias de type : MfaPollingOptions"
---

```ts Example
type MfaPollingOptions = {
  intervalMs?: number;
  onCompleted?: () => void;
  onError?: (error: Error) => void;
}
```

<ParamField body="MfaPollingOptions" type="object">
  Options de configuration pour gérer un processus de scrutation des notifications push d’AMF.

  Fournissez ces options au démarrage d’une session de scrutation pour contrôler la fréquence
  à laquelle le endpoint de challenge est interrogé et la façon dont le succès ou l’échec est géré.

  <Expandable title="propriétés">
    <ParamField body="intervalMs" type="number">
      Intervalle facultatif, en **millisecondes**, entre des requêtes de scrutation consécutives.

      * S’il est omis, l’intervalle interne par défaut du SDK est utilisé (généralement 5000 ms).
      * Choisissez une valeur qui offre un bon équilibre entre la réactivité et les limites de débit.

      ```ts Default Value
      Spécifique à la mise en œuvre (couramment 5000 ms)
      ```
    </ParamField>

    <ParamField body="onCompleted" type="() => void">
      Fonction de rappel facultative exécutée une fois que le challenge push d’AMF est
      approuvé avec succès et que la scrutation est terminée.

      * Appelée exactement **une** fois, après quoi la scrutation s’arrête automatiquement.
      * Utile pour faire avancer le flux de connexion (p. ex., en appelant `continue()`).
    </ParamField>

    <ParamField body="onError" type="(error: Error) => void">
      Fonction de rappel facultative invoquée si une erreur survient pendant la scrutation.

      Reçoit un objet [ULError](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error) contenant :

      * `status` — le code d’état HTTP de la requête échouée

      * `responseText` — le corps de la réponse brut, s’il est disponible

      * Appelée une fois par événement d’erreur; la scrutation peut réessayer selon le type d’erreur.

      * À utiliser pour afficher des messages d’erreur ou déclencher une logique personnalisée de nouvelle tentative ou d’abandon.

      **Propriétés**

      <ParamField body="error" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error">Error</a>}>
        Un objet d’erreur contenant le code d’état HTTP et le corps de la réponse brut de la requête de scrutation échouée.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>
