---
title: "ErrorType"
---

Type d’union de chaînes qui classe les erreurs gérées par [`useErrors`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Utilisé pour filtrer les erreurs avec `errors.byType()`, il est présent dans chaque [`ErrorItem`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem) sous la propriété `kind`.

```typescript
type ErrorType = "auth0" | "validation" | "configuration";
```

<div id="values">
  ## Valeurs
</div>

<ParamField body="&#x22;auth0&#x22;" type="ErrorType">
  Erreurs renvoyées par Auth0 ou par votre propre backend lors de l’authentification ou de l’autorisation.
</ParamField>

<ParamField body="&#x22;validation&#x22;" type="ErrorType">
  Erreurs provenant de la validation du formulaire côté client, par exemple lorsqu’une donnée saisie est invalide avant qu’une requête soit envoyée.
</ParamField>

<ParamField body="&#x22;configuration&#x22;" type="ErrorType">
  Erreurs causées par une mauvaise intégration du SDK ou une erreur de configuration, par exemple lorsqu’un hook est appelé hors du contexte requis. Affichées uniquement lorsque `includeDevErrors: true` est défini dans [`UseErrorOptions`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorOptions).
</ParamField>
