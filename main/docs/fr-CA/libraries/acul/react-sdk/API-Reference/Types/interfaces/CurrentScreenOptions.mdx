---
title: "CurrentScreenOptions"
---

Objet Options qui décrit le contexte de l’écran actuel dans un flux Auth0 Universal Login. Il donne accès au client, à l’organisation, au tenant, à la transaction et aux métadonnées de l’écran disponibles à l’exécution.

```ts Example
export interface CurrentScreenOptions {
  client: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  organization: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  prompt: {
    name: string;
  } | null;
  screen: {
    name: string;
  } | null;
  tenant: {
    enabledLocales: string[];
  } | null;
  transaction: {
    errors: TransactionError[] | null;
    state: string;
    locale: string;
  } | null;
  untrustedData: {
    authorizationParams: Record<string, string> | null;
  } | null;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="client" type="{ id: string; metadata: Record<string, string> | null } | null">
  Détails du client pour la requête d’autorisation en cours, y compris l’identifiant du client et les métadonnées associées.
</ParamField>

<ParamField body="organization" type="{ id: string; metadata: Record<string, string> | null } | null">
  Contexte de l’organisation pour la requête en cours, y compris l’identifiant de l’organisation et les métadonnées.
</ParamField>

<ParamField body="prompt" type="{ name: string } | null">
  La configuration du prompt actuel, indiquant quelle étape du prompt est affichée.
</ParamField>

<ParamField body="screen" type="{ name: string } | null">
  La configuration de l’écran actuel, indiquant quel écran du prompt est actif.
</ParamField>

<ParamField body="tenant" type="{ enabledLocales: string[] } | null">
  Paramètres à l’échelle du tenant, y compris la liste des locales activées pour ce déploiement de Universal Login.
</ParamField>

<ParamField body="transaction" type="{ errors: TransactionError[] | null; state: string; locale: string } | null">
  État de la transaction pour le flux d’authentification en cours, y compris les erreurs de validation, l’identifiant state et la locale active.
</ParamField>

<ParamField body="untrustedData" type="{ authorizationParams: Record<string, string> | null } | null">
  Données non fiables transmises au flux, comme des paramètres d’autorisation qui ne doivent pas servir à prendre des décisions de sécurité.
</ParamField>
