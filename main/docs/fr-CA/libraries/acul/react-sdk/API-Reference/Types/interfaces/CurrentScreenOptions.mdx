---
title: "CurrentScreenOptions"
---

Objet Options qui décrit le contexte de l’écran actuel dans un flux Auth0 Universal Login. Donne accès aux métadonnées du client, de l’organisation, du tenant, de la transaction et de l’écran disponibles à l’exécution.

```ts Example
export interface CurrentScreenOptions {
  client: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  organization: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  prompt: {
    name: string;
  } | null;
  screen: {
    name: string;
  } | null;
  tenant: {
    enabledLocales: string[];
  } | null;
  transaction: {
    errors: TransactionError[] | null;
    state: string;
    locale: string;
  } | null;
  untrustedData: {
    authorizationParams: Record<string, string> | null;
  } | null;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="client" type="{ id: string; metadata: Record<string, string> | null } | null">
  Détails du client pour la demande d’autorisation en cours, y compris l’identifiant du client et les métadonnées associées.
</ParamField>

<ParamField body="organization" type="{ id: string; metadata: Record<string, string> | null } | null">
  Contexte de l’organisation pour la demande en cours, y compris l’identifiant de l’organisation et les métadonnées.
</ParamField>

<ParamField body="prompt" type="{ name: string } | null">
  La configuration actuelle du prompt, qui indique quelle étape du prompt est en cours d’affichage.
</ParamField>

<ParamField body="screen" type="{ name: string } | null">
  La configuration actuelle de l’écran, qui indique quel écran du prompt est actif.
</ParamField>

<ParamField body="tenant" type="{ enabledLocales: string[] } | null">
  Paramètres au niveau du tenant, y compris la liste des locales activées pour ce déploiement Universal Login.
</ParamField>

<ParamField body="transaction" type="{ errors: TransactionError[] | null; state: string; locale: string } | null">
  L’état de la transaction pour le flux d’authentification en cours, y compris toute erreur de validation, le paramètre state et la locale active.
</ParamField>

<ParamField body="untrustedData" type="{ authorizationParams: Record<string, string> | null } | null">
  Données non fiables transmises au flux, comme des paramètres d’autorisation dont il ne faut pas tenir compte pour prendre des décisions de sécurité.
</ParamField>
