---
title: "CurrentScreenOptions"
---

Objet Options qui décrit le contexte de l’écran actuel dans un flux d’Auth0 Universal Login. Donne accès aux métadonnées du client, de l’organisation, du tenant, de la transaction et de l’écran disponibles à l’exécution.

```ts Example
export interface CurrentScreenOptions {
  client: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  organization: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  prompt: {
    name: string;
  } | null;
  screen: {
    name: string;
  } | null;
  tenant: {
    enabledLocales: string[];
  } | null;
  transaction: {
    errors: TransactionError[] | null;
    state: string;
    locale: string;
  } | null;
  untrustedData: {
    authorizationParams: Record<string, string> | null;
  } | null;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="client" type="{ id: string; metadata: Record<string, string> | null } | null">
  Les détails du client pour la demande d’autorisation en cours, y compris l’identifiant du client et toutes les métadonnées associées.
</ParamField>

<ParamField body="organization" type="{ id: string; metadata: Record<string, string> | null } | null">
  Le contexte de l’organisation pour la demande en cours, y compris l’identifiant de l’organisation et les métadonnées.
</ParamField>

<ParamField body="prompt" type="{ name: string } | null">
  La configuration actuelle du prompt, indiquant quelle étape du prompt est affichée.
</ParamField>

<ParamField body="screen" type="{ name: string } | null">
  La configuration actuelle de l’écran, indiquant quel écran du prompt est actif.
</ParamField>

<ParamField body="tenant" type="{ enabledLocales: string[] } | null">
  Les paramètres au niveau du tenant, y compris la liste des paramètres régionaux activés pour ce déploiement Universal Login.
</ParamField>

<ParamField body="transaction" type="{ errors: TransactionError[] | null; state: string; locale: string } | null">
  L’état de la transaction pour le flux d’authentification en cours, y compris les erreurs de validation, l’identifiant state et le paramètre régional actif.
</ParamField>

<ParamField body="untrustedData" type="{ authorizationParams: Record<string, string> | null } | null">
  Les données non fiables transmises au flux, comme des paramètres d’autorisation sur lesquels il ne faut pas s’appuyer pour prendre des décisions de sécurité.
</ParamField>
