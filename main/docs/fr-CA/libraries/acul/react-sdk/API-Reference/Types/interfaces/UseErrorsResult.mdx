---
title: "UseErrorsResult"
---

Objet de résultat renvoyé par [`useErrors`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Donne accès à l’état actuel des erreurs ainsi qu’à des méthodes pour supprimer une erreur précise ou toutes les erreurs dans les écrans d’Auth0 Universal Login.

```ts Example
export interface UseErrorsResult {
  errors: ErrorsResult;
  hasError: boolean;
  dismiss: (id: string) => void;
  dismissAll: () => void;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="errors" type={<a href="/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorsResult">ErrorsResult</a>} required>
  L’ensemble des erreurs actuelles, y compris les messages et les identifiants.
</ParamField>

<ParamField body="hasError" type="boolean" required>
  `true` lorsqu’une ou plusieurs erreurs sont présentes; utile pour afficher conditionnellement l’interface d’erreur.
</ParamField>

<ParamField body="dismiss" type="(id: string) => void" required>
  Supprime une erreur précise à l’aide de son `id`.

  <Expandable title="Paramètres">
    <ParamField body="id" type="string">
      L’identifiant de l’erreur à supprimer.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="dismissAll" type="() => void" required>
  Supprime toutes les erreurs actives à la fois.
</ParamField>
