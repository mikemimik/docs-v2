---
title: "FlattenedTheme"
---

Objet de thème structuré renvoyé par [`useAuth0Themes`](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes). Donne accès, sous une forme aplatie et au niveau des `jetons`, à tout le style visuel appliqué aux composants d’Auth0 Universal Login.

```ts Example
export interface FlattenedTheme {
  colors: Record<string, string>;
  fonts: Record<string, string | number | boolean | object>;
  borders: Record<string, string | number | boolean>;
  pageBackground: Record<string, string>;
  widget: Record<string, string | number>;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="colors" type="Record<string, string>" required>
  Jetons de couleur appliqués à l’ensemble de l’UI de connexion — comprend les couleurs du texte, de l’arrière-plan, des bordures et des états interactifs.
</ParamField>

<ParamField body="fonts" type="Record<string, string | number | boolean | object>" required>
  Jetons typographiques qui contrôlent la famille de polices, la taille, la graisse, la hauteur de ligne et la mise en forme du texte connexe.
</ParamField>

<ParamField body="borders" type="Record<string, string | number | boolean>" required>
  Jetons de bordure qui définissent le rayon, la largeur et le style des éléments de l’UI.
</ParamField>

<ParamField body="pageBackground" type="Record<string, string>" required>
  Jetons de style d’arrière-plan appliqués au conteneur de page qui entoure le widget de connexion.
</ParamField>

<ParamField body="widget" type="Record<string, string | number>" required>
  Jetons de style propres à la carte du widget de connexion, comme l’espacement intérieur, l’ombre et les dimensions.
</ParamField>
