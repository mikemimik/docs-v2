---
title: "ContinueOTPOptions"
---

Options permettant de poursuivre la vérification OTP

```ts Example
export interface ContinueOptions {
  /** Le code saisi par l'utilisateur */
  code: string;
  /** Indique si le navigateur doit être mémorisé */
  rememberDevice?: boolean;
  /** Toute option personnalisée supplémentaire */
  [key: string]: string | number | boolean | undefined;
}
```
