L’Auth0 ACUL React SDK vous permet de créer des expériences Universal Login personnalisées avec un minimum d’effort. Que vous travailliez sur la connexion, l’inscription, l’authentification sans mot de passe ou l’enregistrement de clé d’accès, ce SDK est conçu pour gérer facilement les flux d’authentification modernes.

<div id="why-choose-the-auth0-acul-react-sdk">
  ### Pourquoi choisir l’Auth0 ACUL React SDK?
</div>

* **Intégration rapide :** Intégrez des écrans prédéfinis comme `login-id` ou `signup-password`.
* **Fonctionnalités modernes :** Prend en charge l’authentification sans mot de passe, les clés d’accès et les connexions sociales.
* **Modules flexibles :** Importez uniquement ce dont vous avez besoin.
* **Gestion unifiée de l’état :** Utilisez un seul objet de transaction pour gérer tous les flux d’authentification.

<div id="architecture">
  ### Architecture
</div>

<div id="how-it-works">
  #### Comment ça fonctionne ?
</div>

Le SDK est conçu dans une optique de modularité et d’évolutivité. Voici comment l’ensemble s’articule :

* **Écrans :** Chaque flow d’authentification, comme `login-id` et `signup-password`, constitue un module autonome.
* **Gestionnaires de soumission :** Gèrent les soumissions de formulaires et les transitions.
* **Formulaires dynamiques :** Adaptent les champs et les règles de validation selon la configuration de l’écran.
* **Propriétés utilitaires :** Permettent d’accéder de façon cohérente aux données courantes, comme l’image de marque ou la transaction.
