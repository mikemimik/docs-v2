L’Auth0 ACUL React SDK vous permet de créer des expériences Universal Login personnalisées avec un minimum d’effort. Que vous travailliez sur la connexion, l’inscription, l’authentification sans mot de passe ou l’enregistrement de clés d’accès, ce SDK est conçu pour prendre en charge facilement les flux d’authentification modernes.

<div id="why-choose-the-auth0-acul-react-sdk">
  ### Pourquoi choisir Auth0 ACUL React SDK?
</div>

* **Intégration rapide :** Intégrez des écrans préconçus comme `login-id` ou `signup-password`.
* **Fonctionnalités modernes :** Prise en charge de l’authentification sans mot de passe, des passkeys et des connexions via les réseaux sociaux.
* **Modules flexibles :** Importez uniquement ce dont vous avez besoin.
* **Gestion unifiée de l’état :** Utilisez un seul objet de transaction pour gérer tous les flux d’authentification.

<div id="architecture">
  ### Architecture
</div>

<div id="how-it-works">
  #### Comment ça fonctionne ?
</div>

Le SDK est conçu dans un souci de modularité et d’évolutivité. Voici comment l’ensemble s’articule :

* **Écrans :** Chaque flux d’authentification, comme `login-id` et `signup-password`, est un module autonome.
* **Gestionnaires de soumission :** Gèrent les soumissions de formulaires et les transitions.
* **Formulaires dynamiques :** Adaptent les champs et les validations selon les configurations d’écran.
* **Propriétés utilitaires :** Permettent d’accéder de manière cohérente à des données communes, comme l’image de marque ou transcation.
