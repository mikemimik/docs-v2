Le Auth0 ACUL React SDK vous permet de créer des expériences Universal Login personnalisées avec un minimum d’effort. Que vous travailliez sur la connexion, l’inscription, l’authentification sans mot de passe ou l’enregistrement de clés d’accès, ce SDK est conçu pour prendre en charge facilement les flux d’authentification modernes.

<div id="why-choose-the-auth0-acul-react-sdk">
  ### Pourquoi choisir Auth0 ACUL React SDK?
</div>

* **Intégration rapide :** Ajoutez des écrans prédéfinis comme `login-id` ou `signup-password`.
* **Fonctionnalités modernes :** Prise en charge de l’authentification sans mot de passe, des clés d’accès et des connexions sociales.
* **Modules flexibles :** Importez uniquement ce dont vous avez besoin.
* **Gestion unifiée de l’état :** Utilisez un seul objet de transaction pour gérer tous les flux d’authentification.

<div id="architecture">
  ### Architecture
</div>

<div id="how-it-works">
  #### Comment ça fonctionne ?
</div>

Le SDK est conçu dans une optique de modularité et d’évolutivité. Voici comment les éléments s’articulent :

* **Écrans :** Chaque flux d’authentification, comme `login-id` et `signup-password`, constitue un module autonome.
* **Gestionnaires de soumission :** Gèrent la soumission des formulaires et les transitions.
* **Formulaires dynamiques :** Adaptent les champs et les validations selon la configuration de l’écran.
* **Propriétés utilitaires :** Permettent d’accéder de façon cohérente à des données communes, comme l’image de marque ou la transaction.
