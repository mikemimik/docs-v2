---
title: "Alias de type : ScreenIdentifierType"
---

<ParamField body="ScreenIdentifierType" type="email | phone | username">
  Représente les types d’identifiants utilisateur qui peuvent être utilisés lors de l’inscription.

  * `'email'`: L’utilisateur fournira une adresse courriel.
  * `'phone'`: L’utilisateur fournira un numéro de téléphone.
  * `'username'`: L’utilisateur choisira un username.

  Ce type est généralement utilisé dans les flux de sélection d’identité ou d’inscription
  pour définir quels identifiants sont activés et leur comportement.

  ```ts Example
  const type: IdentifierType = 'email';
  ```
</ParamField>
