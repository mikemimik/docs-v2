ContinueWithPasskeyOptions

```ts Example
export interface ContinueWithPasskeyOptions extends CustomOptions {
  /**
   * Facultatif. Si la valeur est vraie, et si `screen.showRememberDevice` est également vraie,
   * cela indique au serveur de mémoriser ce navigateur pour l'AMF pendant une période définie.
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="rememberDevice?" type="boolean">
  Facultatif. Si `true` et si `screen.showRememberDevice` est aussi `true`,
  cela indique que l’utilisateur a choisi de mémoriser le navigateur actuel,
  ce qui pourrait lui permettre d’éviter l’AMF pendant un certain temps. Correspond au champ de formulaire `rememberBrowser`.
</ParamField>
