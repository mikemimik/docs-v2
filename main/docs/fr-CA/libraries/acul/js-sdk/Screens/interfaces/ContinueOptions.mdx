---
title: "ContinueOptions"
---

Options pour poursuivre avec la vérification par courriel.

```ts Example
export interface ContinueOptions {
  /** Le code saisi par l'utilisateur. */
  code: string;
  /** Indique si l'appareil doit être mémorisé. */
  rememberDevice?: boolean;
  /** Toute option personnalisée supplémentaire */
  [key: string]: string | number | boolean | undefined;
}
```

  ## Propriétés
</div>

<ParamField body="code" type="string">
  Le code entré par l’utilisateur.
</ParamField>

<ParamField body="rememberDevice?" type="boolean">
  Indique s’il faut mémoriser l’appareil.
</ParamField>
