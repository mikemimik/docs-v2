ContinueOptions

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * Le nouveau nickname que l'utilisateur souhaite attribuer à sa clé de sécurité WebAuthn.
   * Cette valeur sera envoyée au serveur pour une mise à jour.
   * Le serveur va valider ce nickname (p. ex., longueur, caractères autorisés).
   * @type {string}
   * @example "Primary YubiKey"
   * @example "Home Office Key"
   */
  nickname: string;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="nickname" type="string">
  Le nouveau nickname de la clé de sécurité WebAuthn.
  Il s’agit de la valeur saisie par l’utilisateur dans le champ de saisie.
  Il doit respecter les contraintes de longueur et de caractères définies par le serveur.
</ParamField>
