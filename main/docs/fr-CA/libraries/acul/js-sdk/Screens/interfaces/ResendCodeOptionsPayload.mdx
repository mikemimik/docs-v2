ResendCodeOptions
description Options de la méthode `resendCode` de l’écran de vérification par courriel de Login.
Cette opération est utilisée lorsque l’utilisateur demande le renvoi d’un nouveau code de vérification à son adresse courriel,
par exemple s’il n’a pas reçu le code initial ou si celui-ci a expiré.

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
