ContinueOptions
Définit les options nécessaires pour poursuivre le processus de MFA Phone Challenge.
Il s’agit généralement de sélectionner la méthode (SMS ou Voice) pour recevoir le code de vérification.

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * Le type de défi à effectuer.
   * 'sms': Envoyer le code par message texte.
   * 'voice': Envoyer le code par appel vocal.
   * @default 'sms'
   */
  type: 'sms' | 'voice';
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;">
  Le type de défi de vérification à envoyer.

  * `sms` : Envoyer le code par message texte.
  * `voice` : Envoyer le code par appel téléphonique.
</ParamField>
