---
title: "LanguageChangeOptions"
---

```ts Example
export interface LanguageChangeOptions {
  language: string;
  persist?: 'session';
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propriétés
</div>

<ParamField body="language" type="string" required>
  Nom abrégé de la langue (code de langue) à utiliser (par exemple, `'en'`, `'fr'`, `'es'`).
</ParamField>

<ParamField body="persist?" type="&#x22;session&#x22;">
  Définit la durée de conservation de la préférence de langue. Lorsqu’elle est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la session.
</ParamField>
