---
title: "UsernameValidationError"
---

Représente une erreur de validation du nom d’utilisateur renvoyée lors de la validation.

Chaque erreur comporte un `code` lisible par machine et un `message` compréhensible par un humain.

```ts Example
export interface UsernameValidationError {
  /**
   * Un identifiant unique pour la règle de validation qui a échoué.
   * 
   * Exemple : `"username-too-short"`, `"username-invalid-characters"`
   */
  code: string;

  /**
   * Une description compréhensible par un humain de l'erreur.
   * 
   * Exemple : `"Username must be at least 3 characters long."`
   */
  message: string;

  /**
   * Le champ associé à l'erreur de validation, généralement « username », « email » ou « phone ».
   */
  field: string;
}
```
