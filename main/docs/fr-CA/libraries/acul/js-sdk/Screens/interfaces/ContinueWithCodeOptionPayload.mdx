ContinueWithCodeOptions
description Options de la méthode `continueWithCode` à l’écran de vérification du courriel de Login.
Cette opération est utilisée lorsque l’utilisateur saisit le code de vérification reçu par courriel
pour poursuivre le flux d’authentification.

```ts Example
export interface ContinueWithCodeOptions extends CustomOptions {
  /**
   * Le code de vérification envoyé à l'adresse courriel de l'utilisateur. Il s'agit généralement d'une courte chaîne
   * numérique ou alphanumérique que l'utilisateur doit saisir dans le formulaire.
   * @type {string}
   * @example "123456"
   * @example "ABCXYZ"
   */
  code: string;
}
```
