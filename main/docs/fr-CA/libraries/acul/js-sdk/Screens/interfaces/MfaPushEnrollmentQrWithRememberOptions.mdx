<div id="properties">
  ## Propriétés

<ParamField body="rememberDevice?" type="boolean">
  Si cette valeur est définie à `true`, cet appareil est enregistré comme fiable afin que l’utilisateur n’ait pas à effectuer une authentification MFA push lors de connexions ultérieures à partir du même appareil.
</ParamField>
