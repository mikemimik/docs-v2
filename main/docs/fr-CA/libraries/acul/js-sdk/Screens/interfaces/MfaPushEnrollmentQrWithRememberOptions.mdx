<div id="properties">
  ## Propriétés

<ParamField body="rememberDevice?" type="boolean">
  Si cette option est définie à `true`, cet appareil est enregistré comme fiable, de sorte que l’utilisateur ne sera pas invité à effectuer une authentification AMF push lors des connexions suivantes à partir du même appareil.
</ParamField>
