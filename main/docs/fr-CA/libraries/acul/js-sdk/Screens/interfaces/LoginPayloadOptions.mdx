Options permettant d’effectuer des opérations de connexion

```ts Example
export interface LoginOptions {
  /** Le nom d'utilisateur/courriel pour la connexion */
  username: string;
  /** Le mot de passe pour l'authentification */
  password: string;
  /** Valeur captcha facultative si nécessaire */
  captcha?: string;
  /** Toute option personnalisée supplémentaire */
  [key: string]: string | number | boolean | undefined;
}
```
