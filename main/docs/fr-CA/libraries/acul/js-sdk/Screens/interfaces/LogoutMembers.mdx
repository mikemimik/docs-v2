Interface définissant les méthodes et propriétés disponibles pour l’écran de déconnexion

```ts Example
export interface LogoutMembers extends BaseMembers {
  /**
   * Confirme l'action de déconnexion (accepter ou rejeter)
   * @param payload Les options contenant l'action à effectuer
   */
  confirmLogout(payload: ConfirmLogoutOptions): Promise<void>;
}
```
