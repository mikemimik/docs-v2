TryAnotherMethodOptions

```ts Example
export interface TryAnotherMethodOptions extends CustomOptions {
  /**
   * Le mode de livraison initialement présenté ou sélectionné sur l'écran actuel.
   * Ceci est requis par l'endpoint de l'API pour traiter correctement l'action 'pick-authenticator'.
   * - `sms`: Indique que l'écran actuel était destiné à un défi SMS.
   * - `voice`: Indique que l'écran actuel était destiné à un défi vocal.
   * @type {'sms' | 'voice'}
   */
  type: 'sms' | 'voice';
}
```
