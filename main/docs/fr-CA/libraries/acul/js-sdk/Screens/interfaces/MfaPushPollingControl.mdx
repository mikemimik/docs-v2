---
title: "MfaPushPollingControl"
---

Interface de contrôle permettant de gérer une session de polling d’une notification push MFA.

Cette interface fournit des méthodes impératives pour démarrer, arrêter et consulter le statut d’une
boucle de polling de longue durée qui vérifie si un défi push MFA a été
approuvé.

````ts Example
export interface MfaPushPollingControl {
  /**
   * Arrête immédiatement le processus de polling.
   *
   * - Annule tout minuteur planifié ou toute requête en attente.
   * - Une fois arrêté, `isRunning()` retourne `false`.
   * - Peut être appelé plusieurs fois sans problème ; les appels suivants n'ont aucun effet.
   *
   * @example
   * ```ts
   * const control = mfaPushChallengePush.pollingManager({ intervalMs: 5000 });
   * control.startPolling();
   *
   * // Plus tard, si l'utilisateur annule :
   * control.stopPolling();
   * ```
   */
  stopPolling: () => void;

  /**
   * Démarre ou reprend le processus de polling.
   *
   * - Si le polling est déjà actif, un nouvel appel n'a aucun effet.
   * - S'il a été arrêté précédemment, cet appel redémarre la boucle de polling.
   *
   * @example
   * ```ts
   * control.startPolling(); // Commence à vérifier le défi push MFA
   * ```
   */
  startPolling: () => void;

  /**
   * Indique si le processus de polling est en cours d'exécution.
   *
   * - Retourne `true` si le polling est actif et non annulé.
   * - Retourne `false` si le polling a été arrêté ou est terminé.
   *
   * @example
   * ```ts
   * if (control.isRunning()) {
   *   console.log('Interrogation en cours...');
   * } else {
   *   console.log('Interrogation arrêtée ou terminée.');
   * }
   * ```
   */
  isRunning: () => boolean;
}
````

<div id="properties">
  ## Propriétés
</div>

<ParamField body="isRunning" type="boolean">
  Indique si le processus de polling est en cours.

  ```ts Example
  if (control.isRunning()) {
    console.log('Polling in progress...');
  } else {
    console.log('Polling is stopped or completed.');
  }
  ```
</ParamField>

<ParamField body="startPolling" type="void">
  Démarre ou reprend le processus de polling.

  * Si le polling est déjà actif, l’appeler de nouveau n’a aucun effet.
  * S’il a déjà été arrêté, l’appeler redémarre la boucle de polling.

  ```ts Example
  control.startPolling(); // Commence à vérifier le défi push MFA
  ```
</ParamField>

<ParamField body="stopPolling" type="void">
  Arrête immédiatement le processus de polling.

  * Annule tout minuteur planifié ou toute requête en attente.
  * Une fois arrêté, `isRunning()` renvoie `false`.
  * Peut être appelé plusieurs fois sans risque; les appels suivants n’ont aucun effet.

  ```ts Example
  const control = mfaPushChallengePush.pollingManager({ intervalMs: 5000 });
  control.startPolling();

  // Plus tard, si l’utilisateur annule :
  control.stopPolling();
  ```
</ParamField>
