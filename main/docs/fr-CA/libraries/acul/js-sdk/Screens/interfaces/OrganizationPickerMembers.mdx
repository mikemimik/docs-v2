title: "OrganizationPickerMembers"

Interface définissant les méthodes et les propriétés offertes pour l’écran « organization-picker »

```ts Example
export interface OrganizationPickerMembers extends BaseMembers {
  /**
   * Sélectionne une organisation pour l'utilisateur.
   * @param payload Les options contenant l'ID de l'organisation.
   */
  selectOrganization(payload: { organization: string; state: string }): Promise<void>;

  /**
   * Ignore la sélection d'organisation et poursuit avec le compte personnel de l'utilisateur.
   * @param payload Options personnalisées facultatives à inclure dans la requête.
   */
  skipOrganizationSelection(payload?: CustomOptions): Promise<void>;
}
```
