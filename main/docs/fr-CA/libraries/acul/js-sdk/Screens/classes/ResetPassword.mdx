---
title: "ResetPassword"
description: "Décrit toutes les propriétés et méthodes permettant de personnaliser l’écran `reset-password` d’Universal Login."
---

La classe ResetPassword implémente la fonctionnalité de l’écran `reset-password`. Cet écran permet à l’utilisateur de choisir un nouveau mot de passe après avoir vérifié son identité.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran ResetPassword :

```typescript Example
import ResetPassword from '@auth0/auth0-acul-js/reset-password';
const resetPasswordManager = new ResetPassword();
resetPasswordManager.resetPassword({
  'password-reset': 'Test@123!',
  're-enter-password': 'Test@123!',
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import ResetPassword from '@auth0/auth0-acul-js/reset-password';
  const resetPasswordManager = new ResetPassword();
  resetPasswordManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque la valeur est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="resetPassword" type="Promise<void>">
  Cette méthode envoie le nouveau mot de passe de l’utilisateur pour terminer le flux de réinitialisation du mot de passe.

  ```typescript Example
  import ResetPassword from '@auth0/auth0-acul-js/reset-password';
  const resetPasswordManager = new ResetPassword();
  resetPasswordManager.resetPassword({
    'password-reset': 'Test@123!',
    're-enter-password': 'Test@123!',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ResetPasswordOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordOptions).
    </ParamField>

    <ParamField body="password-reset" type="string" required>
      Le nouveau mot de passe saisi par l’utilisateur.
    </ParamField>

    <ParamField body="re-enter-password" type="string" required>
      Confirmation du nouveau mot de passe. Doit correspondre à `password-reset`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="validatePassword" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/PasswordValidationResult">PasswordValidationResult</a></span>}>
  Cette méthode valide une chaîne de mot de passe selon la politique de mot de passe du tenant et renvoie un résultat de validation.

  ```typescript Example
  import ResetPassword from '@auth0/auth0-acul-js/reset-password';
  const resetPasswordManager = new ResetPassword();
  resetPasswordManager.validatePassword('MyP@ssw0rd!');
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="password" type="string" required>
      La chaîne de mot de passe à valider.
    </ParamField>
  </Expandable>
</ParamField>
