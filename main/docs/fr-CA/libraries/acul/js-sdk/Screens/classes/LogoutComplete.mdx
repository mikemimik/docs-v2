La classe LogoutComplete implémente la fonctionnalité de l’écran `logout-complete`. Cet écran s’affiche lorsqu’un utilisateur termine le processus de déconnexion.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran LogoutComplete :

```typescript Example
import LogoutComplete from '@auth0/auth0-acul-js/logout-complete';
const logoutCompleteManager = new LogoutComplete();
logoutCompleteManager.changeLanguage({
  language: 'fr',
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import LogoutComplete from '@auth0/auth0-acul-js/logout-complete';
  const logoutCompleteManager = new LogoutComplete();
  logoutCompleteManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette valeur est définie sur `'session'`, la langue sélectionnée est enregistrée pour toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune. Elle renvoie un tableau d’objets d’erreur provenant du contexte de transaction.
</ParamField>
