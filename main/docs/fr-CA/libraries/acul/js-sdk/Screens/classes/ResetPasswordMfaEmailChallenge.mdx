La classe ResetPasswordMfaEmailChallenge implémente la fonctionnalité de l’écran `reset-password-mfa-email-challenge`. Cet écran invite l’utilisateur à saisir le code de vérification envoyé à son adresse courriel dans le cadre de l’AMF d’un processus de réinitialisation du mot de passe.

  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode change la langue d’affichage de la page Universal Login.

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  resetPasswordMfaEmailChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque défini à `'session'`, la langue sélectionnée est enregistrée pour la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Cette méthode soumet le code de vérification par courriel pour poursuivre le défi AMF pendant le processus de réinitialisation du mot de passe.

  ```typescript Example

  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  await resetPasswordMfaEmailChallengeManager.continue({
    code: '123456',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ContinueOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code de vérification envoyé à l’adresse courriel.
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Indique s’il faut se souvenir de l’appareil pour les futurs défis AMF.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>

<ParamField body="resendCode" type="Promise<void>">
  Cette méthode renvoie le code de vérification à l’adresse courriel.

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  await resetPasswordMfaEmailChallengeManager.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Cette méthode fournit la fonctionnalité de renvoi avec gestion du délai d’expiration pour l’écran `reset-password-mfa-email-challenge`.

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  const { startResend } = resetPasswordMfaEmailChallengeManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Renvoi disponible dans ${remainingSeconds}s, désactivé : ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Le renvoi est maintenant disponible');
    }
  });

  // Appeler startResend lorsque l’utilisateur clique sur le bouton de renvoi
  startResend();
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [StartResendOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      Le nombre de secondes à attendre avant d’autoriser l’utilisateur à renvoyer le code.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Fonction de rappel invoquée chaque fois que l’état du renvoi change; elle renvoie le nombre de secondes restantes et indique si la fonctionnalité de renvoi est actuellement désactivée.
    </ParamField>

    <ParamField body="onTimeout?" type="void">
      Fonction de rappel déclenchée lorsque le délai d’expiration prend fin et que le bouton de renvoi devient disponible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">

  ```typescript Example
  await resetPasswordMfaEmailChallengeManager.tryAnotherMethod();
  ```
</ParamField>
