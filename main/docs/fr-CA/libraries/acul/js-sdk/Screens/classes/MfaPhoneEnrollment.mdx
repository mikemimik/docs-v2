---
title: "MfaPhoneEnrollment"
description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `mfa-phone-enrollment` d’Universal Login."
---

La classe MfaPhoneEnrollment implémente la fonctionnalité associée à l’écran `mfa-phone-enrollment`. Cet écran permet aux utilisateurs d’enregistrer un numéro de téléphone pour la AMF.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran d’inscription téléphonique AMF :

```typescript Example
import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

const mfaPhoneEnrollment = new MfaPhoneEnrollment();
await mfaPhoneEnrollment.continueEnrollment({ phone: '+1234567890', type: 'sms' });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode change la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';
  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  mfaPhoneEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’elle est définie à `'session'`, la langue sélectionnée est enregistrée pour toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueEnrollment" type="Promise<void>">
  Cette méthode poursuit le processus d’inscription avec le numéro de téléphone et le type fournis (SMS ou appel vocal).

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.continueEnrollment({ phone: '+1234567890', type: 'sms' });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaPhoneEnrollmentContinueOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/MfaPhoneEnrollmentContinueOptions).
    </ParamField>

    <ParamField body="phone" type="string" required>
      Le numéro de téléphone de l’utilisateur.
    </ParamField>

    <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;" required>
      Le mode d’envoi du code de vérification.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  Cette méthode mène à l’écran de sélection de l’indicatif du pays.

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.pickCountryCode();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode permet à l’utilisateur d’essayer une autre méthode d’AMF.

  ```typescript Example
  import MfaPhoneEnrollment from '@auth0/auth0-acul-js/mfa-phone-enrollment';

  const mfaPhoneEnrollment = new MfaPhoneEnrollment();
  await mfaPhoneEnrollment.tryAnotherMethod();
  ```
</ParamField>
