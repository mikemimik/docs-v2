description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `mfa-email-challenge` de Universal Login."

La classe MfaEmailChallenge implémente la fonctionnalité de l’écran `mfa-email-challenge`. Cet écran s’affiche lorsqu’un utilisateur doit vérifier son courriel dans le cadre de l’authentification AMF.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran de défi AMF par courriel :

```typescript Example
import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';

const mfaEmailChallenge = new MfaEmailChallenge();
await mfaEmailChallenge.continue({
  code: '123456',
  rememberDevice: true
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';
  const mfaEmailChallenge = new MfaEmailChallenge();
  mfaEmailChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue (locale) souhaité (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque la valeur est définie à `'session'`, la langue sélectionnée est conservée pendant toute la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">

  ```typescript Example
  import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';
  ```

      Indique s’il faut mémoriser l’appareil.

<ParamField body="resendManager" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Cette méthode fournit la fonctionnalité de renvoi avec gestion du délai d’attente pour cet écran.

  ```typescript Example
  import MfaEmailChallenge from '@auth0/auth0-acul-js/mfa-email-challenge';

  const mfaEmailChallenge = new MfaEmailChallenge();
  const { startResend } = mfaEmailChallenge.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // Appeler startResend lorsque l’utilisateur clique sur le bouton de renvoi
  startResend();
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [StartResendOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La durée du délai d’attente, en secondes, avant que l’utilisateur puisse renvoyer le code.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Fonction de rappel invoquée chaque fois que l’état du renvoi change; elle retourne le nombre de secondes restantes et indique si la fonctionnalité de renvoi est actuellement désactivée.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Fonction de rappel invoquée lorsque le délai d’attente expire et que le renvoi redevient disponible.
    </ParamField>
  </Expandable>
</ParamField>
