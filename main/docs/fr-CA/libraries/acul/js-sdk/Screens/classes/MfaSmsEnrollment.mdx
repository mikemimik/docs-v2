---
title: "MfaSmsEnrollment"
description: "Décrit toutes les propriétés et méthodes permettant de personnaliser l’écran `mfa-sms-enrollment` de Universal Login."
---

La classe MfaSmsEnrollment implémente les fonctionnalités de l’écran `mfa-sms-enrollment`. Cet écran s’affiche lorsque l’utilisateur doit enregistrer un numéro de téléphone pour l’authentification MFA par SMS.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran d’inscription SMS MFA :

```typescript Example
import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

const mfaSmsEnrollment = new MfaSmsEnrollment();
await mfaSmsEnrollment.continueEnrollment({ phone: '1234567890' });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode change la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  mfaSmsEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de paramètre régional de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette valeur est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueEnrollment" type="Promise<void>">
  Cette méthode poursuit le processus d’inscription par SMS avec le numéro de téléphone fourni.

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.continueEnrollment({ phone: '1234567890' });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaSmsEnrollmentOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/MfaSmsEnrollmentOptions).
    </ParamField>

    <ParamField body="captcha?" type="string">
      Le code CAPTCHA ou la réponse du fournisseur de CAPTCHA. Obligatoire si Bot Detection est activé dans votre tenant Auth0.
    </ParamField>

    <ParamField body="phone?" type="string">
      Le numéro de téléphone de l’utilisateur.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction du contexte, ou un tableau vide s’il n’y en a pas.
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  Cette méthode redirige vers l’écran où l’utilisateur peut sélectionner un indicatif de pays pour son numéro de téléphone.

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.pickCountryCode();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode permet à l’utilisateur d’essayer une autre méthode de MFA.

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.tryAnotherMethod();
  ```
</ParamField>
