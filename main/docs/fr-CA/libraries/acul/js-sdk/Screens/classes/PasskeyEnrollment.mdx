---
title: "PasskeyEnrollment"
description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `passkey-enrollment` dans Universal Login."
---

La classe PasskeyEnrollment implémente les fonctionnalités de l’écran `passkey-enrollment`. Cet écran permet aux utilisateurs d’enregistrer une clé d’accès comme méthode d’authentification.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran PasskeyEnrollment :

```typescript Example
import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
const passkeyEnrollmentManager = new PasskeyEnrollment();
await passkeyEnrollmentManager.continuePasskeyEnrollment();
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="abortPasskeyEnrollment" type="Promise<void>">
  Cette méthode interrompt le processus d’inscription de la clé d’accès.

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  await passkeyEnrollmentManager.abortPasskeyEnrollment();
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [AbortEnrollmentOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/AbortEnrollmentOptions).
    </ParamField>

    <ParamField body="doNotShowAgain?" type="boolean">
      Lorsque cette option est définie à `true`, l’invite d’inscription de la clé d’accès est masquée pour les prochaines sessions.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  passkeyEnrollmentManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue correspondant à la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette option est définie à `'session'`, la langue sélectionnée est enregistrée pour toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continuePasskeyEnrollment" type="Promise<void>">
  Cette méthode poursuit le processus d’inscription de la clé d’accès.

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  await passkeyEnrollmentManager.continuePasskeyEnrollment();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère, à partir du contexte, le tableau des erreurs de transaction, ou un tableau vide s’il n’y en a aucune.
</ParamField>
