title: "Logout"
description: "Décrit toutes les propriétés et méthodes offertes pour personnaliser l’écran `logout` de Universal Login."
---

La classe Logout implémente la fonctionnalité de l’écran `logout`. Cet écran permet aux utilisateurs de confirmer ou d’annuler l’action de déconnexion.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran Logout :

```typescript Example
import Logout from '@auth0/auth0-acul-js/logout';
const logoutManager = new Logout();
logoutManager.confirmLogout({
  action: 'accept',
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode change la langue d’affichage de la page Universal Login.

  ```typescript Example
  import Logout from '@auth0/auth0-acul-js/logout';
  const logoutManager = new Logout();
  logoutManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’elle est définie à `'session'`, la langue sélectionnée est conservée pendant toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="confirmLogout" type="Promise<void>">
  Cette méthode confirme l’action de logout (accepter ou refuser).

  ```typescript Example
  import Logout from '@auth0/auth0-acul-js/logout';
  const logoutManager = new Logout();
  logoutManager.confirmLogout({
    action: 'accept',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ConfirmLogoutOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ConfirmLogoutOptions).
    </ParamField>

    <ParamField body="action" type="&#x22;accept&#x22; | &#x22;deny&#x22;" required>
      L’état de la transaction d’authentification.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère, à partir du contexte, le tableau des erreurs de transaction, ou un tableau vide s’il n’y en a aucune. Elle renvoie un tableau d’objets d’erreur provenant du contexte de transaction.
</ParamField>
