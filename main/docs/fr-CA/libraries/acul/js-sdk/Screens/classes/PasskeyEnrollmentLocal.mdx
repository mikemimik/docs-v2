---
title: "PasskeyEnrollmentLocal"
description: "Décrit toutes les propriétés et méthodes permettant de personnaliser l’écran `passkey-enrollment-local` de Universal Login."
---

La classe PasskeyEnrollmentLocal implémente les fonctionnalités de l’écran `passkey-enrollment-local`. Cet écran permet aux utilisateurs d’enregistrer une clé d’accès stockée localement sur leur appareil comme méthode d’authentification.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran PasskeyEnrollmentLocal :

```typescript Example
import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
await passkeyEnrollmentLocalManager.continuePasskeyEnrollment();
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="abortPasskeyEnrollment" type="Promise<void>">
  Cette méthode annule le processus d’enrôlement de la clé d’accès.

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  await passkeyEnrollmentLocalManager.abortPasskeyEnrollment({ doNotShowAgain: true });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [AbortEnrollmentOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/AbortEnrollmentOptions).
    </ParamField>

    <ParamField body="doNotShowAgain?" type="boolean">
      Lorsque cette option est définie sur `true`, l’invite d’enrôlement de la clé d’accès est masquée pour les prochaines sessions.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode change la langue d’affichage de la page Universal Login.

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  passkeyEnrollmentLocalManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette option est définie sur `'session'`, la langue sélectionnée est enregistrée pendant toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continuePasskeyEnrollment" type="Promise<void>">
  Cette méthode poursuit le processus d’enrôlement de la clé d’accès.

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  await passkeyEnrollmentLocalManager.continuePasskeyEnrollment();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>
