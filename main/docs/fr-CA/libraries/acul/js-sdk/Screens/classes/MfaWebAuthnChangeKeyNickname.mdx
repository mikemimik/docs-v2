La classe MfaWebAuthnChangeKeyNickname implémente les fonctionnalités de l’écran `mfa-webauthn-change-key-nickname`. Cet écran s’affiche lorsque l’utilisateur doit modifier le pseudo de sa clé de sécurité WebAuthn.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran MFA WebAuthn Change Key Nickname :

```typescript Example
import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
await mfaWebAuthnChangeKeyNickname.continueWithNewNickname({ nickname: 'My YubiKey' });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

  const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
  mfaWebAuthnChangeKeyNickname.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue souhaité (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette valeur est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueWithNewNickname" type="Promise<void>">
  Cette méthode soumet le nouveau pseudo de la clé de sécurité WebAuthn de l’utilisateur.

  ```typescript Example
  import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

  const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
  await mfaWebAuthnChangeKeyNickname.continueWithNewNickname({ nickname: 'My YubiKey' });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaWebAuthnChangeKeyNicknameContinueOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/MfaWebAuthnChangeKeyNicknameContinueOptions).
    </ParamField>

    <ParamField body="nickname" type="string" required>
      Le nouveau pseudo de la clé de sécurité WebAuthn.
    </ParamField>
  </Expandable>
</ParamField>
