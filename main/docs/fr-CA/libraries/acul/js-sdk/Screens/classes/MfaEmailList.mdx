---
title: "MfaEmailList"
description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `mfa-email-list` de Universal Login."
---

La classe MfaEmailList implémente la fonctionnalité de l’écran `mfa-email-list`. Cet écran permet aux utilisateurs de sélectionner une adresse courriel déjà inscrite pour le MFA.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran MFA Email List :

```typescript Example
import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

const mfaEmailList = new MfaEmailList();
await mfaEmailList.selectMfaEmail({
  index: 0
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';
  const mfaEmailList = new MfaEmailList();
  mfaEmailList.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de paramètres régionaux de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’elle est définie à `'session'`, la langue sélectionnée est enregistrée pour la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="goBack" type="Promise<void>">
  Cette méthode retourne à l’écran précédent.

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

  const mfaEmailList = new MfaEmailList();
  await mfaEmailList.goBack();
  ```
</ParamField>

<ParamField body="selectMfaEmail" type="Promise<void>">
  Cette méthode sélectionne une adresse courriel déjà inscrite dans la liste.

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

  const mfaEmailList = new MfaEmailList();
  await mfaEmailList.selectMfaEmail({
    index: 0
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [SelectMfaEmailOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/SelectMfaEmailOptions).
    </ParamField>

    <ParamField body="index" type="number" required>
      L’indice du courriel à sélectionner.
    </ParamField>
  </Expandable>
</ParamField>
