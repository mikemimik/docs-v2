title: "MfaWebAuthnRoamingChallenge"
description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `mfa-webauthn-roaming-challenge` d’Universal Login."
---

La classe MfaWebAuthnRoamingChallenge implémente les fonctionnalités de l’écran `mfa-webauthn-roaming-challenge`. Cet écran s’affiche lorsque l’utilisateur doit vérifier son identité à l’aide d’un authentificateur itinérant, comme une clé de sécurité.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran du défi d’itinérance WebAuthn AMF :

```typescript Example
import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
await mfaWebAuthnRoamingChallenge.verify();
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode change la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

  const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
  mfaWebAuthnRoamingChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de paramètre régional de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’il est défini à `'session'`, la langue sélectionnée est enregistrée pour toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="reportWebAuthnError" type="Promise<void>">
  Cette méthode signale une erreur de l’API WebAuthn survenue pendant l’opération `navigator.credentials.get()`.

  ```typescript Example
  import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

  const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
  await mfaWebAuthnRoamingChallenge.reportWebAuthnError({
    error: { name: 'NotAllowedError', message: 'The operation was not allowed.' },
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaWebAuthnRoamingChallengeReportErrorOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/MfaWebAuthnRoamingChallengeReportErrorOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      L’objet d’erreur de l’API WebAuthn (`navigator.credentials.get()`) à signaler.

      <ParamField body="name" type="string" required>
        Le nom de l’erreur (p. ex., `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        Le message d’erreur décrivant le problème.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode permet à l’utilisateur d’essayer une autre méthode AMF.

  ```typescript Example
  import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

  const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
  await mfaWebAuthnRoamingChallenge.tryAnotherMethod();
  ```
</ParamField>

<ParamField body="verify" type="Promise<void>">
  Cette méthode lance la vérification WebAuthn à l’aide de la clé de sécurité.

  ```typescript Example
  import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

  const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
  await mfaWebAuthnRoamingChallenge.verify();
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [CustomOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Lorsque la valeur est `true`, le navigateur est mémorisé pour les prochaines vérifications AMF. S’applique uniquement si `screen.showRememberDevice` vaut `true`.
    </ParamField>
  </Expandable>
</ParamField>
