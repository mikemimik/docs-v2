description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `mfa-voice-challenge` d’Universal Login."
---

La classe MfaVoiceChallenge implémente la fonctionnalité de l’écran `mfa-voice-challenge`. Cet écran s’affiche lorsque l’utilisateur doit saisir un code reçu par appel vocal pour vérifier son identité.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran de défi vocal AMF :

```typescript Example
import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

const mfaVoiceChallenge = new MfaVoiceChallenge();
await mfaVoiceChallenge.continue({ code: '123456' });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  mfaVoiceChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue correspondant à la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque défini à `'session'`, la langue sélectionnée est enregistrée pour la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Cette méthode soumet le code de vérification reçu par appel vocal pour valider le défi AMF.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.continue({
    code: '123456',
    rememberDevice: true,
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaVoiceChallengeContinueOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/MfaVoiceChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code de vérification reçu par appel vocal.
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Lorsque `true`, le navigateur est mémorisé pour les futurs défis AMF. S’applique uniquement si `screen.showRememberDevice` est `true`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère, à partir du contexte, le tableau des erreurs de transaction, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="pickPhone" type="Promise<void>">
  Cette méthode redirige vers l’écran où l’utilisateur peut sélectionner un autre numéro de téléphone.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.pickPhone();
  ```
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Cette méthode demande un nouvel appel vocal contenant un code de vérification.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Cette méthode offre une fonctionnalité de renvoi avec gestion configurable du délai d’expiration et du statut.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  const { startResend } = mfaVoiceChallenge.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    },
  });

  startResend();
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [StartResendOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Fonction de rappel invoquée chaque fois que le statut du renvoi change; elle reçoit le nombre de secondes restantes et indique si le renvoi est actuellement désactivé.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Fonction de rappel invoquée lorsque le délai d’expiration du renvoi prend fin et que l’option de renvoi devient disponible.
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      Le nombre de secondes avant que l’option de renvoi devienne disponible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  Cette méthode fait passer la méthode de vérification au SMS.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.switchToSms();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode permet à l’utilisateur d’essayer une autre méthode d’AMF.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.tryAnotherMethod();
  ```
</ParamField>
