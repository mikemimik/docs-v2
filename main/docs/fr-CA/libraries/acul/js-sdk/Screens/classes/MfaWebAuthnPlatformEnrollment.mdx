title: "MfaWebAuthnPlatformEnrollment"
description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `mfa-webauthn-platform-enrollment` de Universal Login."
---

La classe MfaWebAuthnPlatformEnrollment implémente la fonctionnalité de l’écran `mfa-webauthn-platform-enrollment`. Cet écran s’affiche lorsque l’utilisateur doit enregistrer un authentificateur de plateforme, comme Touch ID ou Face ID.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran MFA WebAuthn Platform Enrollment :

```typescript Example
import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
await mfaWebAuthnPlatformEnrollment.submitPasskeyCredential();
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode change la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  mfaWebAuthnPlatformEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de paramètre régional de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’elle est définie à `'session'`, la langue sélectionnée est enregistrée pour toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="refuseEnrollmentOnThisDevice" type="Promise<void>">
  Cette méthode permet à l’utilisateur de refuser l’inscription à WebAuthn de plateforme sur l’appareil en cours d’utilisation.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  await mfaWebAuthnPlatformEnrollment.refuseEnrollmentOnThisDevice();
  ```
</ParamField>

<ParamField body="reportBrowserError" type="Promise<void>">
  Cette méthode signale une erreur côté navigateur survenue pendant l’opération WebAuthn `navigator.credentials.create()`.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  await mfaWebAuthnPlatformEnrollment.reportBrowserError({
    error: { name: 'NotAllowedError', message: 'The operation was not allowed.' },
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [CustomOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      L’objet d’erreur de l’API WebAuthn (`navigator.credentials.create()`) à signaler.

      <ParamField body="name" type="string" required>
        Le nom de l’erreur (p. ex., `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        Le message d’erreur décrivant le problème.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="snoozeEnrollment" type="Promise<void>">
  Cette méthode permet à l’utilisateur de reporter l’inscription à WebAuthn de plateforme.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  await mfaWebAuthnPlatformEnrollment.snoozeEnrollment();
  ```
</ParamField>

<ParamField body="submitPasskeyCredential" type="Promise<void>">
  Cette méthode lance la création d’une clé d’accès WebAuthn de plateforme et envoie l’information d’identification obtenue au serveur.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  await mfaWebAuthnPlatformEnrollment.submitPasskeyCredential();
  ```
</ParamField>
