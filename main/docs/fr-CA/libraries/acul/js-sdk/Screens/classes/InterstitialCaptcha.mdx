---
title: "InterstitialCaptcha"
description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `interstitial-captcha` de Universal Login."
---

La classe InterstitialCaptcha implémente la fonctionnalité de l’écran `interstitial-captcha`. Cet écran demande à l’utilisateur de résoudre un CAPTCHA pendant le flux d’authentification.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran InterstitialCaptcha :

```typescript Example
import InterstitialCaptcha from '@auth0/auth0-acul-js/interstitial-captcha';
const interstitialCaptchaManager = new InterstitialCaptcha();
await interstitialCaptchaManager.submitCaptcha({ captcha: 'captchaValue' });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode change la langue d’affichage de la page de connexion Universal Login.

  ```typescript Example
  import InterstitialCaptcha from '@auth0/auth0-acul-js/interstitial-captcha';
  const interstitialCaptchaManager = new InterstitialCaptcha();
  interstitialCaptchaManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette valeur est définie sur `'session'`, la langue sélectionnée est enregistrée pour toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="submitCaptcha" type="Promise<void>">
  Cette méthode soumet la réponse CAPTCHA pour poursuivre le flux d’authentification.

  ```typescript Example
  import InterstitialCaptcha from '@auth0/auth0-acul-js/interstitial-captcha';
  const interstitialCaptchaManager = new InterstitialCaptcha();
  await interstitialCaptchaManager.submitCaptcha({ captcha: 'captchaValue' });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [SubmitCaptchaOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/SubmitCaptchaOptions).
    </ParamField>

    <ParamField body="captcha" type="string" required>
      Le jeton de réponse CAPTCHA.
    </ParamField>
  </Expandable>
</ParamField>
