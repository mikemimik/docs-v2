title: "BruteForceProtectionUnblockFailure"
description: "Décrit toutes les propriétés et méthodes offertes pour personnaliser l’écran `brute-force-protection-unblock-failure` de Universal Login."
---

La classe BruteForceProtectionUnblockFailure implémente la fonctionnalité de l’écran `brute-force-protection-unblock-failure`. Cet écran s’affiche après une tentative infructueuse de débloquer un compte qui a été bloqué en raison de la protection contre les attaques par force brute.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran BruteForceProtectionUnblockFailure :

```typescript Example
import BruteForceProtectionUnblockFailure from '@auth0/auth0-acul-js/brute-force-protection-unblock-failure';
const bruteForceProtectionUnblockFailureManager = new BruteForceProtectionUnblockFailure();
bruteForceProtectionUnblockFailureManager.changeLanguage({ language: 'en' });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import BruteForceProtectionUnblockFailure from '@auth0/auth0-acul-js/brute-force-protection-unblock-failure';
  const bruteForceProtectionUnblockFailureManager = new BruteForceProtectionUnblockFailure();
  bruteForceProtectionUnblockFailureManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de la locale correspondant à la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque la valeur est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction à partir du contexte, ou un tableau vide s’il n’y en a pas.
