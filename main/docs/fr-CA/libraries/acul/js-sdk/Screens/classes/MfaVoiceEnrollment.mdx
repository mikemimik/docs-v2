---
title: "MfaVoiceEnrollment"
description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `mfa-voice-enrollment` d’Universal Login."
---

La classe MfaVoiceEnrollment implémente les fonctionnalités de l’écran `mfa-voice-enrollment`. Cet écran s’affiche lorsque l’utilisateur doit saisir un numéro de téléphone pour recevoir un appel vocal contenant un code de vérification afin de poursuivre l’inscription.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran d’inscription AMF par appel vocal :

```typescript Example
import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

const mfaVoiceEnrollment = new MfaVoiceEnrollment();
await mfaVoiceEnrollment.continue({ phone: '+1234567890' });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

  const mfaVoiceEnrollment = new MfaVoiceEnrollment();
  mfaVoiceEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de langue correspondant à la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’elle est définie sur `'session'`, la langue sélectionnée est enregistrée pour la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Cette méthode poursuit le processus d’inscription vocale avec le numéro de téléphone fourni.

  ```typescript Example
  import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

  const mfaVoiceEnrollment = new MfaVoiceEnrollment();
  await mfaVoiceEnrollment.continue({ phone: '+1234567890' });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaVoiceEnrollmentContinueOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/MfaVoiceEnrollmentContinueOptions).
    </ParamField>

    <ParamField body="phone" type="string" required>
      Le numéro de téléphone de l’utilisateur pour recevoir l’appel vocal.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère dans le contexte le tableau des erreurs de transaction, ou un tableau vide s’il n’y en a aucune.
</ParamField>

<ParamField body="selectPhoneCountryCode" type="Promise<void>">
  Cette méthode accède à l’écran où l’utilisateur peut sélectionner un indicatif de pays pour son numéro de téléphone.

  ```typescript Example
  import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

  const mfaVoiceEnrollment = new MfaVoiceEnrollment();
  await mfaVoiceEnrollment.selectPhoneCountryCode();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode permet à l’utilisateur d’essayer une autre méthode d’AMF.

  ```typescript Example
  import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

  const mfaVoiceEnrollment = new MfaVoiceEnrollment();
  await mfaVoiceEnrollment.tryAnotherMethod();
  ```
</ParamField>
