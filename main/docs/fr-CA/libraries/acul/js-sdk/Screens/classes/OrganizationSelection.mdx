---
title: "OrganizationSelection"
description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `organization-selection` dans Universal Login."
---

La classe OrganizationSelection implémente les fonctionnalités de l’écran `organization-selection`. Cet écran permet aux utilisateurs de sélectionner une organisation pour poursuivre.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran OrganizationSelection :

```typescript Example
import OrganizationSelection from '@auth0/auth0-acul-js/organization-selection';
const organizationSelectionManager = new OrganizationSelection();
await organizationSelectionManager.continueWithOrganizationName({
  organizationName: 'testOrganizationName',
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import OrganizationSelection from '@auth0/auth0-acul-js/organization-selection';
  const organizationSelectionManager = new OrganizationSelection();
  organizationSelectionManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de `locale` de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’elle est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueWithOrganizationName" type="Promise<void>">
  Cette méthode poursuit le processus avec le nom de l’organisation sélectionnée.

  ```typescript Example
  import OrganizationSelection from '@auth0/auth0-acul-js/organization-selection';
  const organizationSelectionManager = new OrganizationSelection();
  await organizationSelectionManager.continueWithOrganizationName({
    organizationName: 'testOrganizationName',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ContinueWithOrganizationNameOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/ContinueWithOrganizationNameOptions).
    </ParamField>

    <ParamField body="organizationName" type="string" required>
      Le nom de l’organisation avec lequel poursuivre le processus.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère, à partir du contexte, le tableau des erreurs de la transaction, ou un tableau vide s’il n’y en a aucune.
</ParamField>
