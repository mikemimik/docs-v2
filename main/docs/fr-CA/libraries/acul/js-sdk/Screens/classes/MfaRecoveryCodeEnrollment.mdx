---
title: "MfaRecoveryCodeEnrollment"
description: "Décrit toutes les propriétés et méthodes permettant de personnaliser l’écran `mfa-recovery-code-enrollment` de Universal Login."
---

La classe MfaRecoveryCodeEnrollment implémente la fonctionnalité de l’écran `mfa-recovery-code-enrollment`. Cet écran s’affiche lorsque l’utilisateur doit s’enrôler à MFA à l’aide d’un code de récupération.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran d’enrôlement du code de récupération MFA :

```typescript Example
import MfaRecoveryCodeEnrollment from '@auth0/auth0-acul-js/mfa-recovery-code-enrollment';

const mfaRecoveryCodeEnrollment = new MfaRecoveryCodeEnrollment();
await mfaRecoveryCodeEnrollment.continue({ isCodeCopied: true });
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaRecoveryCodeEnrollment from '@auth0/auth0-acul-js/mfa-recovery-code-enrollment';

  const mfaRecoveryCodeEnrollment = new MfaRecoveryCodeEnrollment();
  mfaRecoveryCodeEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code locale de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’elle est définie à `'session'`, la langue sélectionnée reste enregistrée pendant toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Cette méthode indique que l’utilisateur a sauvegardé le code de récupération et poursuit le flux d’enrôlement.

  ```typescript Example
  import MfaRecoveryCodeEnrollment from '@auth0/auth0-acul-js/mfa-recovery-code-enrollment';

  const mfaRecoveryCodeEnrollment = new MfaRecoveryCodeEnrollment();
  await mfaRecoveryCodeEnrollment.continue({ isCodeCopied: true });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaRecoveryCodeEnrollmentContinueOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/MfaRecoveryCodeEnrollmentContinueOptions).
    </ParamField>

    <ParamField body="isCodeCopied?" type="boolean">
      Indique si l’utilisateur a copié le code de récupération.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction dans le contexte, ou un tableau vide s’il n’en existe aucune.
</ParamField>
