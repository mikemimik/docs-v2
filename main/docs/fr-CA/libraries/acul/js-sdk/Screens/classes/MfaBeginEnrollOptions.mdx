description: "Décrit toutes les propriétés et méthodes disponibles pour personnaliser l’écran `mfa-begin-enroll-options` de Universal Login."
---

La classe MfaBeginEnrollOptions implémente les fonctionnalités de l’écran `mfa-begin-enroll-options`. Cet écran gère la sélection et l’inscription des facteurs MFA.

</Frame>

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire de l’écran MFA Begin Enroll Options :

```typescript Example
import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';
const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
await mfaBeginEnrollOptions.enroll({
  action: 'push-notification'
});
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';
  const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
  mfaBeginEnrollOptions.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de locale de la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsque cette valeur est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="enroll" type="Promise<void>">
  Cette méthode poursuit le processus d’inscription avec le facteur sélectionné.

  ```typescript Example
  import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';

  const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
  await mfaBeginEnrollOptions.enroll({
    action: 'push-notification'
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaEnrollOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/MfaEnrollOptions).
    </ParamField>

    <ParamField body="action" type="&#x22;push-notification&#x22; | &#x22;otp&#x22; | &#x22;sms&#x22; | &#x22;phone&#x22; | &#x22;voice&#x22; | &#x22;webauthn-roaming&#x22;" required>
      L’action qui indique quel facteur inscrire. L’une des valeurs suivantes : &quot;push-notification&quot;, &quot;otp&quot;, &quot;sms&quot;, &quot;phone&quot;, &quot;voice&quot; ou &quot;webauthn-roaming&quot;.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction dans le contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>
