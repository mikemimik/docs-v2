---
title: "ResetPasswordError"
description: "Décrit toutes les propriétés et méthodes offertes pour personnaliser l’écran `reset-password-error` de Universal Login."
---

La classe ResetPasswordError implémente la fonctionnalité de l’écran `reset-password-error`. Cet écran s’affiche lorsqu’une erreur se produit pendant le flux de réinitialisation du mot de passe, par exemple si le lien de réinitialisation a expiré ou n’est pas valide.

<div id="constructors">
  ## Constructeurs
</div>

Créez une instance du gestionnaire d’écran ResetPasswordError :

```typescript Example
import ResetPasswordError from '@auth0/auth0-acul-js/reset-password-error';
const resetPasswordErrorManager = new ResetPasswordError();
```

<div id="methods">
  ## Méthodes
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Cette méthode modifie la langue d’affichage de la page Universal Login.

  ```typescript Example
  import ResetPasswordError from '@auth0/auth0-acul-js/reset-password-error';
  const resetPasswordErrorManager = new ResetPasswordError();
  resetPasswordErrorManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      Le code de locale correspondant à la langue souhaitée (par exemple, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Lorsqu’elle est définie sur `'session'`, la langue sélectionnée est conservée pendant toute la durée de la session.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Cette méthode récupère le tableau des erreurs de transaction du contexte, ou un tableau vide s’il n’y en a aucune.
</ParamField>
