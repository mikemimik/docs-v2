> **changeLanguage**(`options`: [`LanguageChangeOptions`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)): `Promise`&lt;`void`&gt;

Soumet au serveur la nouvelle préférence de langue avec l’action `change-language` et réaffiche l’interface dans la langue sélectionnée.

<div id="parameters">
  ## Paramètres
</div>

<ParamField body="options" type="LanguageChangeOptions" required>
  La configuration de changement de langue. Consultez [`LanguageChangeOptions`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).

  <Expandable title="options">
    <ParamField body="language" type="string" required>
      Un code de langue pour la langue cible (p. ex. `'en'`, `'fr'`, `'es'`). Il doit correspondre à l’une des langues activées dans votre tenant Auth0.
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Portée de persistance de la préférence de langue. Actuellement, seule la valeur `"session"` est prise en charge.
    </ParamField>
  </Expandable>
</ParamField>

<div id="returns">
  ## Renvoie
</div>

`Promise`&lt;`void`&gt;

Se résout lorsque la préférence de langue a été envoyée et que l’écran se réaffiche dans la langue sélectionnée.
