> **getCurrentThemeOptions**(): [`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

**`Fonctionnalité commune`**

Récupère les options du thème actuel avec une configuration aplatie à partir du contexte d’image de marque

<div id="returns">
  ## Renvoie
</div>

[`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

Objet FlattenedTheme contenant les couleurs, les polices, les bordures, pageBackground et les configurations des widgets, ou null si aucune image de marque n’est disponible
