> **getCurrentThemeOptions**(): [`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

**`Fonctionnalité commune`**

Retourne les options du thème actuel issues de la configuration aplatie du contexte d’image de marque

<div id="returns">
  ## Retour
</div>

[`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

Objet `FlattenedTheme` contenant les couleurs, les polices, les bordures, `pageBackground` et la configuration du widget, ou `null` si aucune image de marque n’est définie
