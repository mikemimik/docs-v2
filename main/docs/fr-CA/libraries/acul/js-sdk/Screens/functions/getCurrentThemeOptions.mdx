> **getCurrentThemeOptions**(): [`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

**`Fonctionnalité commune`**

Récupère les options du thème actuel à partir du contexte d’image de marque, avec une configuration aplatie

<div id="returns">
  ## Renvoie
</div>

[`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

Objet FlattenedTheme contenant les couleurs, les polices, les bordures, `pageBackground` et la configuration des widgets, ou `null` si aucune image de marque n’est définie
