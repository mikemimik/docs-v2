> **getCurrentThemeOptions**(): [`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

**`Fonctionnalité commune`**

Récupère, depuis le contexte d’image de marque, les options du thème actuel avec une configuration aplatie

<div id="returns">
  ## Valeur de retour
</div>

[`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

Objet FlattenedTheme contenant les couleurs, les polices, les bordures, `pageBackground` et la configuration du widget, ou `null` si aucune image de marque n’est disponible
