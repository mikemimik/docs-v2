Cette bibliothèque vous permet de personnaliser le comportement du client réseau pour les configurations courantes, ainsi que de définir et d’utiliser votre propre mise en œuvre du client réseau.

Vous pouvez configurer la classe Auth0 avec un `NetworkingClient`, qui sera utilisé lors de l’envoi des requêtes. Vous pouvez configurer le client par défaut avec des valeurs de délai d’expiration personnalisées, les en-têtes à envoyer avec toutes les requêtes, et préciser s’il faut consigner les renseignements sur les requêtes et les réponses (à des fins de débogage hors production seulement). Pour une configuration plus avancée, vous pouvez fournir votre propre mise en œuvre de `NetworkingClient`.

<div id="timeout-configuration">
  ### Configuration du délai d’attente
</div>

export const codeExample1 = `val netClient = DefaultClient(
    connectTimeout = 30,
    readTimeout = 30
)

val account = Auth0("{yourClientId}", "{yourDomain}")
account.networkingClient = netClient`;
