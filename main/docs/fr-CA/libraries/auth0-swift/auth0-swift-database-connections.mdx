<Card title="Authentification par base de données sur les plateformes natives">
  L’authentification par nom d’utilisateur/courriel et mot de passe à partir d’applications natives est désactivée par défaut pour les nouveaux tenants depuis le 8 juin 2017. Les utilisateurs sont encouragés à utiliser Universal Login et à effectuer une Web Authentication à la place. Si vous souhaitez quand même continuer, vous devrez d’abord activer le Password Grant Type dans votre Dashboard. Consultez [Application Grant Types](/docs/fr-CA/get-started/applications/application-grant-types) pour en savoir plus.
</Card>

L’Authentication API fournit des méthodes pour authentifier les utilisateurs de base de données et les inscrire.

<div id="signing-up-with-a-database-connection">
  ## S’inscrire avec une connexion de base de données
</div>

Pour s’inscrire, il faut appeler la méthode `createUser` en lui passant l’`email`, le `password` et le nom de la `connection` de l’utilisateur afin de lancer le processus d’inscription. Vous pouvez aussi préciser des métadonnées utilisateur supplémentaires à stocker.

```swift lines
Auth0
    .authentication()
    .createUser(
        email: "support@auth0.com",
        password: "secret-password",
        connection: "Username-Password-Authentication",
        userMetadata: ["first_name": "First",
                       "last_name": "Last"])
    .start { result in
        switch result {
        case .success(let user):
            print("User: \(user)")
        case .failure(let error):
            print("Failed with \(error)")
        }
    }
```

<div id="login-with-a-database-connection">
  ## Se connecter avec une connexion de base de données
</div>

Pour se connecter avec une connexion de base de données, il faut appeler `login` en fournissant le nom d’utilisateur ou l’adresse courriel de l’utilisateur, son mot de passe et le nom de la connexion (par exemple `Username-Password-Authentication`) à utiliser pour l’authentification. La réponse sera un objet Credentials.

```swift lines
Auth0
    .authentication()
    .login(
        usernameOrEmail: "support@auth0.com",
        password: "secret-password",
        realm: "Username-Password-Authentication",
        scope: "openid")
     .start { result in
         switch result {
         case .success(let credentials):
            print("Credentials: \(credentials)")
         case .failure(let error):
            print("Failed with \(error)")
         }
     }
```
