**Auth0.Android** inclut la classe `WebAuthProvider` pour gérer le processus d’autorisation à l’aide du navigateur. Mais que faire si vous souhaitez utiliser votre propre implémentation ou une version Native d’un `AuthProvider` ?

<div id="the-authprovider-class">
  ## La classe AuthProvider
</div>

Créez une classe qui implémente l’interface `AuthProvider` et redéfinissez ses méthodes.

Vous pouvez aussi utiliser n’importe quelle implémentation **Native** déjà fournie par Auth0. Les fournisseurs d’[authentification sociale native](/docs/fr-CA/libraries/lock-android/lock-android-native-social-authentication) actuellement disponibles sont Google et Facebook.
