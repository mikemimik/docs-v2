---
title: Bien démarrer
description: Découvrez les bases et commencez à créer votre solution d’authentification.
---

Auth0 est une plateforme d’identité qui vous permet de gérer l’accès à vos applications.

Si vous découvrez la gestion des identités et des accès (IAM), familiarisez-vous avec quelques notions de base et planifiez la solution la mieux adaptée à votre environnement technologique et à vos besoins. Si vous connaissez déjà l’IAM, vous pouvez vous lancer et commencer à créer.

***

<div id="start-building">
  ## Commencez à développer
</div>

Pour démarrer rapidement, choisissez votre type d’application et suivez un tutoriel Quickstart étape par étape.

<Columns cols={2}>
  <Card title="Application native/mobile" href="/docs/fr-CA/quickstart/native">
    Application mobile ou de bureau qui s’exécute directement sur un appareil, comme iOS ou Android.
  </Card>

  <Card title="SPA" href="/docs/fr-CA/quickstart/spa">
    Application web JavaScript qui s’exécute dans le navigateur, comme AngularJS + Node.js ou React.
  </Card>

  <Card title="Application web traditionnelle" href="/docs/fr-CA/quickstart/webapp">
    Application web traditionnelle qui s’exécute sur le serveur, comme Express.js ou ASP.NET.
  </Card>

  <Card title="Backend/API" href="/docs/fr-CA/quickstart/backend">
    API ou service protégé par Auth0, comme une Express.js API ou une API ASP.NET.
  </Card>

  <Card title="Agent d’IA" href="https://auth0.com/ai/docs">
    Un agent d’IA pour appeler des API externes au nom de l’utilisateur et utiliser des mécanismes de contrôle avec intervention humaine, comme LangChain ou LlamaIndex.
  </Card>
</Columns>
