    <Section id={sections[1].id} title={sections[1].title} stepNumber="2" isSingleColumn>
      D’abord, vous devez installer le module `nginx-plus-module-njs` pour NGINX Plus.

      Suivez le guide d’installation des modules dynamiques pour installer les paquets sur votre système d’exploitation hôte.

      Pour les distributions Linux qui utilisent le gestionnaire de paquets `yum`, vous pouvez installer le module comme suit :

      ```sh lines
      sudo yum install nginx-plus-module-njs jq
      ```

      Une fois installé, vous devez l’activer dans NGINX en ajoutant la ligne suivante au début de votre
      fichier `/etc/nginx/nginx.conf` :

      ```sh lines
      load_module modules/ngx_http_js_module.so;
      ```

    <Section id={sections[2].id} title={sections[2].title} stepNumber="3" isSingleColumn>
      Clonez le dépôt `nginx-openid-connect`. Ce dépôt comprend un modèle de configuration.

      ```sh lines
      git clone https://github.com/nginxinc/nginx-openid-connect
      ```

      <LoggedInForm />

    <Section id={sections[3].id} title={sections[3].title} stepNumber="4" isSingleColumn>
      Exécutez le script `configure.sh` dans le dossier `nginx-openid-connect` afin de renseigner le modèle de configuration de
      votre application Auth0 :

      ```sh lines
      ./configure.sh --auth_jwt_key request \
      --client_id {yourClientId} \
      --pkce_enable \
      https://{yourDomain}/.well-known/openid-configuration
      ```

      Ensuite, ajoutez l’[URL de déconnexion de votre tenant](https://auth0.com/docs/api/authentication#auth0-logout) à votre
      fichier `openid_connect_configuration.conf`.

      <AuthCodeGroup>

      Copiez vos quatre fichiers de configuration dans le dossier `conf.d`.

      ```sh lines
      sudo cp openid_connect.js \ 
      frontend.conf \
      openid_connect_configuration.conf \
      openid_connect.server_conf /etc/nginx/conf.d
      ```

      4. Passez à la vue **Credentials**.

      Dans la section **Application Authentication**, définissez **Authentication Method** sur None.
