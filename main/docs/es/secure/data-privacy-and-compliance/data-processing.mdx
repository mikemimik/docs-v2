Este documento explica qué datos tiene Auth0 y cómo los procesa.

<div id="data-auth0-possesses">
  ## Datos que Auth0 tiene
</div>

Todos los datos que Auth0 tiene sobre un usuario final se encuentran en el perfil de usuario de Auth0. Los atributos específicos incluidos en el perfil de usuario varían según la implementación del cliente y dependen de varios factores, como el tipo de conexión, el consentimiento del usuario durante el flujo de autenticación y si se han ampliado los perfiles de usuario con información adicional.

<Warning>
  Los metadatos de Auth0 no son un almacén de datos seguro y no deben usarse para almacenar información sensible, como secrets de alto riesgo e información de identificación personal (PII), como números de la seguridad social o de tarjetas de crédito. Se recomienda encarecidamente a los clientes de Auth0 que evalúen los datos almacenados en los metadatos y que solo guarden lo necesario para fines de gestión de identidades y accesos.
</Warning>

<div id="when-data-is-stored">
  ## Cuándo se almacenan los datos
</div>

La información del perfil de usuario de Auth0 se almacena en Auth0 cuando usas una conexión de base de datos. Si un usuario inicia sesión con cualquier otro tipo de conexión (incluidas las conexiones de base de datos personalizadas), Auth0 almacena la información proporcionada por el <Tooltip tip="Proveedor de identidad (IdP): Servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=identity+provider">proveedor de identidad</Tooltip> externo para futuras consultas.

<div id="how-data-is-used">
  ## Cómo se usan los datos
</div>

Los datos personales almacenados en Auth0 se utilizan únicamente para prestar sus servicios, concretamente para autenticar a los usuarios

<div id="how-can-i-delete-my-data">
  ## ¿Cómo puedo eliminar mis datos?
</div>

Como responsable del tratamiento, el cliente es responsable de gestionar y eliminar datos mediante las funcionalidades de autoservicio, entre ellas:

* [Eliminar o restablecer tenants](/docs/es/troubleshoot/customer-support/manage-subscriptions/delete-or-reset-tenant) - Usa esta funcionalidad para eliminar un tenant. Cuando eliminas un tenant, el sistema elimina de forma permanente todos los datos del cliente asociados.
* [Administrar Users con la Management API](/docs/es/manage-users/user-accounts/manage-users-using-the-management-api) - Usa la API para recuperar, crear, actualizar o eliminar usuarios con la Management API.
* [Administrar Users en Dashboard](/docs/es/manage-users/user-accounts/manage-users-using-the-dashboard) - Usa el Dashboard para crear, ver, modificar o eliminar usuarios manualmente.

<div id="when-is-customer-data-automatically-deleted">
  ## ¿Cuándo se eliminan automáticamente los datos del cliente?
</div>

Auth0 conserva los datos del cliente mientras el cliente se considere activo.

Auth0 clasifica a un cliente como inactivo cuando cumple ambas condiciones siguientes:

1. El cliente usa un plan Free (incluidos quienes optaron por un Tier Free o se cambiaron a uno inferior).

2. Todos los tenants asociados no muestran actividad durante un período continuo de 150 días.

Después de 150 días de inactividad, el sistema programa los tenants del cliente inactivo y todos los datos asociados para su eliminación permanente.
