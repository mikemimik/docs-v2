---
description: Describe cómo exportar datos de usuario para cumplir con los requisitos del RGPD sobre portabilidad de los datos.
title: 'RGPD: Portabilidad de los datos'
---

De acuerdo con el artículo 20 del RGPD, los usuarios tienen derecho a recibir los datos personales que les conciernen en un formato estructurado, de uso común y legible por máquina.

Puede exportar datos de usuario almacenados en el repositorio de usuarios de Auth0, ya sea manualmente o de forma programática. Los datos sin procesar de Auth0 pueden exportarse en formato JSON (legible por máquina).

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El contenido de estos documentos no pretende constituir asesoramiento jurídico ni debe considerarse un sustituto de la asistencia jurídica. La responsabilidad final de comprender y cumplir con el RGPD recae en usted, aunque Auth0 le ayudará a cumplir sus requisitos siempre que sea posible.
</Callout>

<div id="export-data-manually">
  ## Exportar datos manualmente
</div>

1. Ve a [Auth0 Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users).
2. Busca al usuario y haz clic en su nombre para ver los detalles.
3. Haz clic en la pestaña **Raw JSON**. Aquí puedes ver el perfil completo del usuario en formato JSON.
4. Haz clic en **Copy JSON**. El perfil se copia al portapapeles.
5. Pega el contenido del portapapeles en un editor y guárdalo.

<div id="export-data-using-the-api">
  ## Exportar datos con la API
</div>

Puedes exportar el perfil completo de un usuario con nuestra <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. La respuesta estará en formato JSON. Puedes buscar un usuario por su ID o exportar una lista de tus usuarios.
