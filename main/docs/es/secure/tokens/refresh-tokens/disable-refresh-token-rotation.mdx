Puedes deshabilitar la rotación del <Tooltip tip="Token de actualización: token usado para obtener un token de acceso renovado sin obligar a los usuarios a iniciar sesión nuevamente." cta="Ver glosario" href="/docs/es/glossary?term=refresh+token">token de actualización</Tooltip> para cada aplicación desde el Dashboard o la <Tooltip tip="Token de actualización: token usado para obtener un token de acceso renovado sin obligar a los usuarios a iniciar sesión nuevamente." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<div id="disable-with-the-dashboard">
  ## Desactivar desde el Dashboard
</div>

1. Desde el [Auth0 Dashboard](https://manage.auth0.com/#), ve a **Applications &gt; Applications** y selecciona la aplicación que quieres configurar.
2. En la pestaña Settings, localiza la sección de rotación del token de actualización y desactiva el interruptor Allow Refresh Token Rotation.
3. Selecciona **Save Changes** en la parte inferior de la pantalla.

<div id="disable-with-the-management-api">
  ## Deshabilitar con la Management API
</div>

1. Desactive la rotación del token de actualización en cada aplicación mediante la Management API:

   {/* codeblockOld.header.login.logInButton codeblockOld.header.login.configureSnippet */}

   <AuthCodeBlock children={codeExample} language="javascript" />

2. Configure los ajustes del token de actualización no rotativo de la siguiente manera:

   ```http lines
   PATCH /api/v2/clients/{client_id}
   {
      "refresh_token": {
        "rotation_type": "non-rotating",
        "expiration_type": "non-expiring"
      }
   }
   ```
