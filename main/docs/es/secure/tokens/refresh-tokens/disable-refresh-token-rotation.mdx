Puedes desactivar la rotación del <Tooltip tip="Token de actualización: token utilizado para obtener un token de acceso renovado sin obligar a los usuarios a volver a iniciar sesión." cta="Ver glosario" href="/docs/es/glossary?term=refresh+token">token de actualización</Tooltip> en cada aplicación desde el Dashboard o la <Tooltip tip="Token de actualización: token utilizado para obtener un token de acceso renovado sin obligar a los usuarios a volver a iniciar sesión." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<div id="disable-with-the-dashboard">
  ## Deshabilitar con el Dashboard
</div>

1. En el [Auth0 Dashboard](https://manage.auth0.com/#), ve a **Applications &gt; Applications** y selecciona la aplicación que deseas configurar.
2. En la pestaña Settings, busca la sección de rotación del token de actualización y desactiva el toggle Allow Refresh Token Rotation.
3. Selecciona **Save Changes** en la parte inferior de la pantalla.

<div id="disable-with-the-management-api">
  ## Deshabilitar con la Management API
</div>

1. Deshabilita la rotación del Refresh Token en cada aplicación mediante la Management API:

   {/* codeblockOld.header.login.logInButton codeblockOld.header.login.configureSnippet */}

   <AuthCodeBlock children={codeExample} language="javascript" />

2. Configura los ajustes del token de actualización sin rotación de la siguiente manera:

   ```http lines
   PATCH /api/v2/clients/{client_id}
   {
      "refresh_token": {
        "rotation_type": "non-rotating",
        "expiration_type": "non-expiring"
      }
   }
   ```
