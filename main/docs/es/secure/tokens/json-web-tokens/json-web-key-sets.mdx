description: Un JSON Web Key Set es un objeto JSON que representa un conjunto de JSON Web Keys (un objeto JSON que representa una clave criptográfica).
title: JSON Web Key Sets
---

El JSON Web Key Set (JWKS) es un conjunto de claves que contiene las claves públicas utilizadas para verificar cualquier <Tooltip tip="JSON Web Token (JWT): Formato estándar de token de ID (y, a menudo, de token de acceso) que se utiliza para representar claims de forma segura entre dos partes." cta="Ver glosario" href="/docs/es/glossary?term=JSON+Web+Token+%28JWT%29">JSON Web Token (JWT)</Tooltip> emitido por el <Tooltip tip="Authorization Server: Servidor centralizado que ayuda a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funcionalidades disponibles para un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Authorization+Server">servidor de autorización</Tooltip> y firmado con el [algoritmo de firma](/docs/es/get-started/applications/signing-algorithms) RS256.

Al crear aplicaciones y API en Auth0, se admiten dos algoritmos para firmar <Tooltip tip="Authorization Server: Servidor centralizado que ayuda a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funcionalidades disponibles para un usuario." cta="Ver glosario" href="/docs/es/glossary?term=JWTs">JWT</Tooltip>: **RS256** y **HS256**. RS256 genera una firma asimétrica, lo que significa que se debe usar una clave privada para firmar el JWT y una clave pública distinta para verificar la firma.

Auth0 usa la [especificación JSON Web Key (JWK)](https://tools.ietf.org/html/rfc7517) para representar las claves criptográficas utilizadas para firmar tokens RS256. Esta especificación define dos estructuras de datos de alto nivel: **JSON Web Key (JWK)** y **JSON Web Key Set (JWKS)**. Estas son las definiciones de la especificación:

| Elemento                    | Descripción                                                                                                                            |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| **JSON Web Key (JWK)**      | Un objeto JSON que representa una clave criptográfica. Los miembros del objeto representan propiedades de la clave, incluido su valor. |
| **JSON Web Key Set (JWKS)** | Un objeto JSON que representa un conjunto de JWK. El objeto JSON DEBE tener un miembro `keys`, que es un arreglo de JWK.               |

Auth0 expone un endpoint de JWKS para cada tenant, que se encuentra en `https://{yourDomain}/.well-known/jwks.json`. Este endpoint contendrá el JWK utilizado para verificar todos los JWT emitidos por Auth0 para este tenant.

<Warning>
  Actualmente, Auth0 firma con un solo JWK a la vez; sin embargo, es importante asumir que este endpoint podría contener varios JWK. Por ejemplo, se pueden encontrar varias claves en el JWKS al [rotar las claves de firma de la aplicación](/docs/es/get-started/tenant-settings/signing-keys/rotate-signing-keys).
</Warning>
