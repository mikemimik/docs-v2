Puedes cambiar la duración del <Tooltip tip="Token de acceso: Credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> en el <Tooltip tip="Token de acceso: Credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="configure-access-token-lifetime">
  ## Configurar la duración del token de acceso
</div>

1. Ve a [Dashboard &gt; Applications &gt; APIs](https://manage.auth0.com/#/apis) y selecciona el nombre de la API para verla.

   <Frame>![Lista de API en Dashboard > Applications > APIs](/docs/images/cdy7uua7fh8z/3rhmhghYZDSi6YWHRA5yMQ/c71340259481b0b6787d5f3887cfda0f/dashboard-apis-list.png)</Frame>
2. Busca el campo **Duración máxima del token de acceso** en la sección **Access Token Expiration**.

   <Frame>![Dashboard - API - Configuración del token - Expiración](/docs/images/cdy7uua7fh8z/2FFfqqmntXL9CnGrSxea9M/810e5542553168ffcd6081a10bc2db2f/image2.png)</Frame>
3. Introduce la duración deseada (en segundos) para los tokens de acceso emitidos para esta API.

   * El valor predeterminado es `86,400` segundos (24 horas).
   * El valor máximo es `2,592,000 segundos` (30 días).
4. Selecciona **Guardar cambios**.

<div id="token-expiration-for-browser-flows">
  ## Expiración del token para flujos del navegador
</div>

El campo **Duración del token de acceso del flujo implícito/híbrido** se refiere a los tokens de acceso emitidos para la API mediante flujos implícitos e híbridos, y no abarca todos los flujos iniciados desde el navegador.

Por ejemplo, el flujo PKCE (utilizado en el SDK `auth0-js-spa`) puede iniciarse desde el navegador, pero toma el valor de **Token Expiration**, no el de **Duración del token de acceso del flujo implícito/híbrido**.

<div id="restricted-lifetime-for-mfa-access-tokens">
  ## Duración restringida de los tokens de acceso para MFA
</div>

La duración de los tokens de acceso con la <Tooltip tip="Audiencia: Identificador único de la audiencia de un token emitido. Se denomina aud en un token, y su valor contiene el ID de una aplicación (ID de cliente) para un token de ID o de una API (identificador de API) para un token de acceso." cta="Ver glosario" href="/docs/es/glossary?term=audience">audiencia</Tooltip> `{yourAuth0Domain}/mfa` está limitada a `600` segundos (10 minutos) por motivos de seguridad y no se puede modificar.
