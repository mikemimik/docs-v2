description: Aprende cómo funcionan los token de acceso de la Management API y cómo usarlos.
title: Token de acceso de la Management API
---

Para llamar a la [Auth0 Management API](https://auth0.com/docs/api/management/v2), debes proporcionar un <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, que se usa para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> que contenga la <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, que se usa para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=audience">audiencia</Tooltip> y los scopes adecuados.

<div id="audience">
  ## Audience
</div>

El valor de `audience` de la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> es `https://{yourDomain}/api/v2/`.

<div id="scopes">
  ## Scopes
</div>

Cada endpoint de la Management API requiere un conjunto específico de scopes. Consulta el [Management API Explorer](https://auth0.com/docs/api/management/v2/) para más información.

<div id="lifetime">
  ## Duración
</div>

La duración predeterminada del token de acceso de la Management API es de 86.400 segundos (24 horas).

Puedes [configurar la duración del token de acceso](/docs/es/secure/tokens/access-tokens/update-access-token-lifetime) para cada API por separado.

<div id="security">
  ## Seguridad
</div>

* **Token de acceso**: Una vez emitido, un token de acceso no se puede revocar. Auth0 recomienda que limites la duración de tus tokens de acceso al mínimo posible según tu caso de uso.
* **<Tooltip tip="Token de actualización: Token que se usa para obtener un nuevo token de acceso sin obligar a los usuarios a volver a iniciar sesión." cta="Ver glosario" href="/docs/es/glossary?term=Refresh+token">Token de actualización</Tooltip>**: Puedes [revocar un token de actualización](/docs/es/secure/tokens/refresh-tokens/revoke-refresh-tokens) usando el Dashboard, la Authentication API o la Management API.
* **<Tooltip tip="Secreto de cliente: Secreto que usa un cliente (aplicación) para autenticarse ante el servidor de autorización; solo el cliente y el servidor de autorización deben conocerlo, y debe ser lo suficientemente aleatorio como para que no se pueda adivinar." cta="Ver glosario" href="/docs/es/glossary?term=Client+secret">Secreto de cliente</Tooltip>**: Puedes [rotar el secreto de cliente de una aplicación](/docs/es/get-started/applications/rotate-client-secret) usando el Dashboard o la Management API.

<div id="quota">
  ## Cuota
</div>

Las cuotas de tokens se determinan según el nivel de suscripción. Puedes [consultar tu cuota actual](https://support.auth0.com/reports/quota) en el Auth0 Support Center. Si tienes preguntas sobre precios o cuotas, [ponte en contacto con nuestro equipo de ventas](https://auth0.com/contact-us).

* Los tokens emitidos para [audiencias internas de Auth0](/docs/es/secure/tokens/access-tokens/get-access-tokens#control-access-token-audience) no cuentan para tu cuota.
* Los tokens emitidos para audiencias personalizadas cuentan para tu cuota y están sujetos a los [límites de frecuencia de Auth0 Management API](/docs/es/troubleshoot/customer-support/operational-policies/rate-limit-policy/management-api-endpoint-rate-limits).
