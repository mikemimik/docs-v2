description: Aprende cómo funcionan los tokens de acceso de la Auth0 Management API y cómo usarlos.
title: Tokens de acceso de la Management API
---

Para llamar a la [Auth0 Management API](https://auth0.com/docs/api/management/v2), debes proporcionar un <Tooltip tip="Token de acceso: credencial de autorización, en forma de cadena opaca o JWT, que se usa para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> que contenga la <Tooltip tip="Token de acceso: credencial de autorización, en forma de cadena opaca o JWT, que se usa para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=audience">audiencia</Tooltip> y los scopes adecuados.

<div id="audience">
  ## Audiencia
</div>

El valor de audiencia de la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> es `https://{yourDomain}/api/v2/`.

<div id="scopes">
  ## Scopes
</div>

Cada endpoint de la Management API requiere un conjunto específico de scopes. Consulta el [Management API Explorer](https://auth0.com/docs/api/management/v2/) para obtener más información.

<div id="lifetime">
  ## Duración
</div>

La duración predeterminada del token para la Management API es de 86,400 segundos (24 horas).

Puedes [configurar la duración del token](/docs/es/secure/tokens/access-tokens/update-access-token-lifetime) para cada API de forma individual.

<div id="security">
  ## Seguridad
</div>

* **Token de acceso**: Una vez emitido, un token de acceso no puede revocarse. Auth0 recomienda restringir la duración de validez de sus tokens de acceso al menor valor posible según su caso de uso.
* **<Tooltip tip="Token de actualización: token que se usa para obtener un token de acceso renovado sin obligar a los usuarios a volver a iniciar sesión." cta="Ver glosario" href="/docs/es/glossary?term=Refresh+token">token de actualización</Tooltip>**: Puede [revocar un token de actualización](/docs/es/secure/tokens/refresh-tokens/revoke-refresh-tokens) usando el Dashboard, la Authentication API o la Management API.
* **<Tooltip tip="Secreto de cliente: Secret usado por un cliente (aplicación) para autenticarse con el servidor de autorización; solo deben conocerlo el cliente y el servidor de autorización, y debe ser lo suficientemente aleatorio como para que no se pueda adivinar." cta="Ver glosario" href="/docs/es/glossary?term=Client+secret">secreto de cliente</Tooltip>**: Puede [rotar el secreto de cliente de una aplicación](/docs/es/get-started/applications/rotate-client-secret) usando el Dashboard o la Management API.

<div id="quota">
  ## Cuota
</div>

Las cuotas de tokens se determinan según el nivel de suscripción. Puedes [revisar tu cuota actual](https://support.auth0.com/reports/quota) en el Auth0 Support Center. Si tienes preguntas sobre precios o cuotas, [ponte en contacto con nuestro equipo de ventas](https://auth0.com/contact-us).

* Los tokens emitidos para [audiencias internas de Auth0](/docs/es/secure/tokens/access-tokens/get-access-tokens#control-access-token-audience) no se contabilizan en tu cuota.
* Los tokens emitidos para audiencias personalizadas sí se contabilizan en tu cuota y están sujetos a los [límites de frecuencia de Auth0 Management API](/docs/es/troubleshoot/customer-support/operational-policies/rate-limit-policy/management-api-endpoint-rate-limits).
