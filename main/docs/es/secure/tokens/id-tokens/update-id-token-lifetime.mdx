description: Describe cómo actualizar la duración del token de ID de una aplicación mediante el Auth0 Dashboard.
title: Actualizar la duración del token de ID
---

Puede cambiar la duración del <Tooltip tip="Token de ID: credencial destinada al propio cliente, en lugar de para acceder a un recurso." cta="Ver glosario" href="/docs/es/glossary?term=ID+token">token de identidad</Tooltip> mediante el <Tooltip tip="Token de ID: credencial destinada al propio cliente, en lugar de para acceder a un recurso." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. De forma predeterminada, un token de identidad es válido durante 36000 segundos (10 horas). Si hay consideraciones de seguridad, puede acortar el tiempo antes de que el token expire, teniendo en cuenta que uno de sus propósitos es mejorar la experiencia del usuario al almacenar en caché la información del usuario.

1. Vaya a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y haga clic en el nombre de la aplicación que quiere ver.
2. Desplácese hasta **ID Token** y localice el campo **ID Token Expiration**. Introduzca la duración adecuada del token de identidad (en segundos) para la aplicación.
3. Haga clic en **Guardar cambios**.
