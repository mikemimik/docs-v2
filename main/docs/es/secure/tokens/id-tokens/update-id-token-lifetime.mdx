title: Actualizar la duración del token de ID
---

Puedes cambiar la duración del <Tooltip tip="Token de ID: credencial destinada al propio cliente, en lugar de para acceder a un recurso." cta="Ver glosario" href="/docs/es/glossary?term=ID+token">token de identidad</Tooltip> usando el <Tooltip tip="Token de ID: credencial destinada al propio cliente, en lugar de para acceder a un recurso." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. De forma predeterminada, un token de identidad es válido durante 36000 segundos (10 horas). Si hay consideraciones de seguridad, puedes acortar el tiempo antes de que el token expire, teniendo en cuenta que uno de sus propósitos es mejorar la experiencia del usuario al almacenar en caché su información.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y haz clic en el nombre de la aplicación que quieres ver.
2. Desplázate hasta **ID Token** y localiza el campo **ID Token Expiration**. Introduce la duración adecuada del token de identidad (en segundos) para la aplicación.
3. Haz clic en **Save Changes**.
