<Card title="Información general">
  La autenticación multifactor (MFA) es otro método para proteger tu aplicación y las identidades de tus usuarios. La MFA añade una capa de seguridad durante el inicio de sesión que exige a los usuarios proporcionar más de una credencial para demostrar su identidad digital. Los factores pueden ser:

  * Algo que eres, como un dato biométrico
  * Algo que sabes, como una contraseña
  * Algo que tienes, como un dispositivo

  Lee estas preguntas y respuestas para ver si usar MFA con tu instancia de Auth0 es la opción adecuada para ti.
</Card>

<div id="what-is-multi-factor-authentication">
  ## ¿Qué es la autenticación multifactor?
</div>

<Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuario que utiliza un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=Multi-factor+authentication">Autenticación multifactor</Tooltip> (MFA) es un método de verificación de usuario que requiere más de un tipo de verificación. Evita que los <Tooltip tip="Actores maliciosos: Entidad (una persona o grupo) que representa una amenaza para la empresa o el entorno con la intención de causar daño." cta="Ver glosario" href="/docs/es/glossary?term=bad+actors">actores maliciosos</Tooltip> accedan a una cuenta, incluso si han obtenido el nombre de usuario y la contraseña.

<div id="why-use-multi-factor-authentication">
  ## ¿Por qué usar la autenticación multifactor?
</div>

La MFA reduce la probabilidad de muchos tipos de ciberataques. Es habitual que terceros roben nombres de usuario y contraseñas o ataquen de forma automatizada las cuentas de usuario. Un factor adicional de MFA, como una huella dactilar o un código de un solo uso, dificulta este tipo de ataques.

<div id="how-does-multi-factor-authentication-work">
  ## ¿Cómo funciona la autenticación multifactor?
</div>

La MFA funciona al requerir información de verificación adicional (conocida como factores). Los usuarios no pueden iniciar sesión solo con nombres de usuario y contraseñas. Deben proporcionar pruebas adicionales de identidad, como reconocimiento facial o notificaciones por mensaje de texto.

<div id="mfa-factors">
  ## Factores de MFA
</div>

<Warning>
  Los factores de MFA dependen de la disponibilidad del plan; algunos solo están disponibles en los planes Professional y Enterprise. Para obtener más información, consulta [Auth0 Pricing](https://auth0.com/pricing).
</Warning>

Auth0 ofrece una variedad de factores de MFA, entre ellos:

* Notificaciones push
* Notificaciones por SMS
* Notificaciones de voz
* Códigos de un solo uso
* WebAuthn con claves de seguridad
* WebAuthn con biometría del dispositivo
* Notificaciones por correo electrónico
* Cisco Duo Security
* Códigos de recuperación

Para obtener más información, consulta [Factores de la autenticación multifactor](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-factors).
