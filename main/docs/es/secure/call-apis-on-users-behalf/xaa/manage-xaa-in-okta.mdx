Una vez que hayas terminado de configurar tu entorno de prueba de extremo a extremo, podrás gestionar cómo pueden conectarse entre sí las aplicaciones XAA válidas en la Consola de administración de Okta.

<Frame>![](/docs/images/xaa/manage_xaa_in_okta.png)</Frame>

En la Consola de administración de Okta:

1. Ve a **Applications &gt; Applications** y selecciona tu aplicación de recursos (p. ej., Todo0).
2. En la pestaña **Manage Connections**, agrega:
   * Aplicaciones solicitantes: aplicaciones que pueden conectarse a tu aplicación SaaS
   * Aplicaciones de recursos: aplicaciones a las que tu aplicación SaaS puede conectarse

Para el entorno de prueba de extremo a extremo, agrega Agent0, o la aplicación que quieras usar para las pruebas, como aplicación solicitante autorizada.
