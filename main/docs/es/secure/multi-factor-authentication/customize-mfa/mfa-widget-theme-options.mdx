---
description: Describe las opciones de tema del MFA Widget para personalizar las propiedades de tema de las páginas de MFA.
title: Opciones de tema del MFA Widget
---

Al usar tu propio HTML para las páginas de <Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuarios que utiliza un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=MFA">MFA</Tooltip> con Classic Login, este se basa en el MFA Widget de Auth0, que tiene las siguientes limitaciones:

* No admite MFA por correo electrónico.
* Si los usuarios registraron más de un factor, no pueden seleccionar cuál usar; el widget de MFA les pide iniciar sesión con el factor más seguro.
* No usa las funciones de [internacionalización](/docs/es/customize/internationalization-and-localization/universal-login-internationalization) de <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip>.

Las opciones de tema del MFA Widget están bajo la propiedad `theme`.

| Opción         | Descripción                                                                                                                                                                         |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `icon`         | La URL de la imagen que se usa en el encabezado. El valor predeterminado es el logotipo de Auth0. La altura recomendada es de 58 píxeles.                                           |
| `primaryColor` | Define el color principal del widget de MFA. Es útil al usar un `icon` personalizado para garantizar que se muestren colores complementarios. El valor predeterminado es `#ea5323`. |

<div id="examples">
  ## Ejemplos
</div>

```js lines
return new Auth0MFAWidget({

...

  theme: {
    icon: 'https://example.com/assets/logo.png',
    primaryColor: 'blue'
  },
  
...  
  
})
```

Consulta un [ejemplo básico del widget de Guardian](https://github.com/auth0/auth0-guardian.js/blob/master/example/basic_widget.html) con [auth0-guardian.js](https://github.com/auth0/auth0-guardian.js). Consulta el [readme](https://github.com/auth0/auth0-guardian.js/tree/master/example) para más detalles.
