title: Recursos para desarrolladores de la autenticación multifactor
---

Con los SDK de Auth0, puedes personalizar la experiencia de <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación del usuario que utiliza un factor adicional al nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> (MFA) de tus usuarios e incluso crear aplicaciones basadas en nuestras capacidades multifactor.

<div id="mfa-api">
  ## API de MFA
</div>

Los endpoints de la [API de MFA](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-developer-resources/mfa-api) te permiten exigir MFA cuando los usuarios interactúan con los [endpoints de token](https://auth0.com/docs/api/authentication#get-token), así como registrar y gestionar factores de MFA.

<div id="create-custom-mfa-pages">
  ## Crear páginas MFA personalizadas
</div>

Use las siguientes bibliotecas cliente para personalizar la apariencia de la página MFA de modo que coincida con su organización.

* [Biblioteca cliente de Auth0 MFA](https://github.com/auth0/auth0-guardian.js)
* [Crear un MFA Widget personalizado](https://github.com/auth0/auth0-guardian.js/tree/master/example)

<div id="create-custom-enrollments">
  ## Crear inscripciones personalizadas
</div>

Puedes [personalizar el proceso de inscripción](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-developer-resources/create-custom-enrollment-tickets) de tus usuarios creando tickets de inscripción personalizados.

<div id="build-custom-mobile-applications">
  ## Crea aplicaciones móviles personalizadas
</div>

Crea aplicaciones personalizadas similares a Guardian o añade funcionalidad multifactor a tus aplicaciones con nuestros SDK.

* [Guardian para Android](/docs/es/secure/multi-factor-authentication/auth0-guardian/guardian-for-android-sdk)
* [Guardian para iOS](/docs/es/secure/multi-factor-authentication/auth0-guardian/guardian-for-ios-sdk)
