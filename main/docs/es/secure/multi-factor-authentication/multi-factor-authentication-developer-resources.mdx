title: Recursos para desarrolladores sobre autenticación multifactor
---

Con los SDK de Auth0, puedes personalizar la experiencia de <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuario que usa un factor además de nombre de usuario y contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> (MFA) de tus usuarios e incluso crear aplicaciones basadas en nuestras funciones de autenticación multifactor.

<div id="mfa-api">
  ## API de MFA
</div>

Los endpoints de la [API de MFA](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-developer-resources/mfa-api) te permiten exigir MFA cuando los usuarios interactúan con los [endpoints de token](https://auth0.com/docs/api/authentication#get-token), así como registrar y gestionar factores de MFA.

<div id="create-custom-mfa-pages">
  ## Crear páginas de MFA personalizadas
</div>

Usa las siguientes bibliotecas cliente para personalizar el aspecto de la página de MFA para que se adapte a tu organización.

* [biblioteca cliente de Auth0 MFA](https://github.com/auth0/auth0-guardian.js)
* [Crear un MFA Widget personalizado](https://github.com/auth0/auth0-guardian.js/tree/master/example)

<div id="create-custom-enrollments">
  ## Crear inscripciones personalizadas
</div>

Puedes [personalizar el proceso de inscripción](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-developer-resources/create-custom-enrollment-tickets) para tus usuarios mediante la creación de tickets de inscripción personalizados.

<div id="build-custom-mobile-applications">
  ## Cree aplicaciones móviles personalizadas
</div>

Cree aplicaciones personalizadas similares a Guardian o añada funcionalidad de autenticación multifactor a sus aplicaciones con nuestros SDKs.

* [Guardian for Android](/docs/es/secure/multi-factor-authentication/auth0-guardian/guardian-for-android-sdk)
* [Guardian for iOS](/docs/es/secure/multi-factor-authentication/auth0-guardian/guardian-for-ios-sdk)
