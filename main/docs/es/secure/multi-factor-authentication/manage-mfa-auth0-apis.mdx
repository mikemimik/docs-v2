Auth0 ofrece dos API distintas para ayudarte a ti y a tus usuarios a gestionar factores de autenticación <Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuario que usa un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=MFA">MFA</Tooltip>.

<div id="auth0-authentication-api">
  ## Auth0 Authentication API
</div>

Utiliza los [endpoints de Multi-factor Authentication de la Auth0 Authentication API](https://auth0.com/docs/api/authentication#multi-factor-authentication) para permitir que tus usuarios gestionen sus propios factores de autenticación MFA.

Este método se basa en la autenticación mediante el flujo <Tooltip tip="Resource Owner: Entidad (como un usuario o una aplicación) capaz de otorgar acceso a un recurso protegido." cta="Ver glosario" href="/docs/es/glossary?term=Resource+Owner">Resource Owner</Tooltip> Password Grant (ROPG). Para obtener más información, consulta [gestionar factores de autenticación con Authentication API](/docs/es/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

<div id="auth0-management-api">
  ## Auth0 Management API
</div>

Usa los [endpoints de Users de la Management API](https://auth0.com/docs/api/management/v2) para gestionar los factores de autenticación MFA de tus usuarios.

Este método requiere autenticarse con una aplicación confidencial. Para obtener más información, consulta [gestionar métodos de autenticación con Management API](/docs/es/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authentication-methods-with-management-api).
