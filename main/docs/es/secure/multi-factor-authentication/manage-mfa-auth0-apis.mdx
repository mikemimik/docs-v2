Auth0 proporciona dos API diferentes para ayudarte a ti y a tus usuarios a gestionar los factores de autenticación de <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuario que utiliza un factor adicional además de nombre de usuario y contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=MFA">MFA</Tooltip>.

<div id="auth0-authentication-api">
  ## Auth0 Authentication API
</div>

Usa los [endpoints de autenticación multifactor de la Authentication API](https://auth0.com/docs/api/authentication#multi-factor-authentication) para permitir que tus usuarios gestionen sus propios factores de autenticación MFA.

Este método se basa en la autenticación mediante el flujo de autenticación <Tooltip tip="Resource Owner: Entidad (como un usuario o una aplicación) capaz de otorgar acceso a un recurso protegido." cta="Ver glosario" href="/docs/es/glossary?term=Resource+Owner">Resource Owner</Tooltip> Password Grant (ROPG). Para obtener más información, consulta [Gestionar factores de autenticación con Authentication API](/docs/es/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

<div id="auth0-management-api">
  ## Auth0 Management API
</div>

Usa los [endpoints de Users de la Management API](https://auth0.com/docs/api/management/v2) para administrar los factores de autenticación MFA de tus usuarios.

Este método se basa en la autenticación mediante una aplicación confidencial. Para obtener más información, consulta [Administra métodos de autenticación con Management API](/docs/es/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authentication-methods-with-management-api).
