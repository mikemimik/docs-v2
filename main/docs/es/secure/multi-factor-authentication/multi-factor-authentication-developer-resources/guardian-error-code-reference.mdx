---
description: Lista los códigos y las descripciones de errores de Guardian.
title: Referencia de códigos de error de Guardian
---

Usa los códigos de error para mostrar mensajes informativos y distinguir entre errores recuperables e irrecuperables.

| Código de error                       | Descripción                                                                                                                                         |
| ------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `invalid_token`                       | La solicitud o el token de la transacción no son válidos                                                                                            |
| `insufficient_scope`                  | No tienes suficientes concesiones para realizar la operación solicitada                                                                             |
| `invalid_bearer_format`               | El bearer incluido en el header de authentication no era válido                                                                                     |
| `enrollment_conflict`                 | Ya existe otra inscripción para el mismo usuario. No puedes inscribirte dos veces.                                                                  |
| `tenant_not_found`                    | No se puede encontrar el tenant asociado. Normalmente esto no debería ocurrir, salvo que elimines el tenant                                         |
| `login_transaction_not_found`         | La transacción de autenticación de MFA no está activa o ya venció                                                                                   |
| `error_sending_push_notification`     | Falló la entrega de la notificación push                                                                                                            |
| `push_notification_wrong_credentials` | Falló la entrega de la notificación push debido a credenciales incorrectas                                                                          |
| `invalid_otp`                         | El código OTP proporcionado no era válido                                                                                                           |
| `invalid_recovery_code`               | El código de recuperación proporcionado no era válido                                                                                               |
| `invalid_body`                        | Falló la validación del Body. Solicitud incorrecta.                                                                                                 |
| `invalid_query_string`                | Falló la validación del query string. Solicitud incorrecta.                                                                                         |
| `enrollment_transaction_not_found`    | La transacción de inscripción de MFA no está activa o venció                                                                                        |
| `invalid_phone_number`                | El número de teléfono proporcionado no es válido                                                                                                    |
| `error_sending_sms`                   | Error en la entrega del SMS                                                                                                                         |
| `feature_disabled`                    | La funcionalidad solicitada no está disponible globalmente en este momento (contacta con el proveedor)                                              |
| `feature_disabled_by_admin`           | La funcionalidad solicitada está deshabilitada actualmente por tu administrador                                                                     |
| `pn_endpoint_disabled`                | No pudimos entregar la notificación push después de reintentarlo muchas veces. Intenta eliminar tu cuenta del device y volver a agregarla.          |
| `too_many_sms`                        | Has excedido la cantidad de SMS asignados a tu usuario                                                                                              |
| `too_many_pn`                         | Has excedido la cantidad de notificaciones push asignadas a tu usuario                                                                              |
| `too_many_sms_per_tenant`             | Has excedido la cantidad de SMS asignados a tu tenant                                                                                               |
| `too_many_pn_per_tenant`              | Has excedido la cantidad de notificaciones push asignadas a tu tenant                                                                               |
| `field_required`                      | Se requiere un campo para realizar la operación (este error también tiene un atributo field con un código para el campo: `otpCode`, `recoveryCode`) |
| `method_not_found`                    | Has solicitado un método que actualmente no es compatible (no debería ocurrir)                                                                      |
| `no_method_available`                 | Actualmente no hay ningún método disponible para inscribirse (todos están deshabilitados)                                                           |
| `enrollment_method_disabled`          | El método de inscripción especificado está deshabilitado; este error también tiene un campo .method                                                 |
| `auth_method_disabled`                | El método de authentication especificado está deshabilitado; este error también tiene un campo .method                                              |
| `invalid_otp_format`                  | Error de validación del formato del OTP                                                                                                             |
| `invalid_recovery_code_format`        | Error de validación del formato del código de recuperación                                                                                          |
| `transaction_expired`                 | La transacción ya venció                                                                                                                            |
| `already_enrolled`                    | Ya estás inscrito; no puedes volver a inscribirte                                                                                                   |
| `not_enrolled`                        | No estás inscrito. Primero debes inscribirte                                                                                                        |
| `invalid_enrollment`                  | La inscripción proporcionada al método transaction#requestAuth no es válida o es null/undefined                                                     |
