description: Describe la API de MFA de Auth0, que te permite inscribir y gestionar MFA para los usuarios.
title: Auth0 MFA API
---

Auth0 proporciona un flujo integrado de inscripción y autenticación de <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuario que utiliza un factor además de nombre de usuario y contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> (MFA) con [Universal Login](/docs/es/authenticate/login/auth0-universal-login). Usa la [API de MFA](https://auth0.com/docs/api/authentication#multi-factor-authentication) en los siguientes casos si quieres:

* [Autenticar usuarios con la concesión de contraseña del propietario del recurso](/docs/es/secure/multi-factor-authentication/authenticate-using-ropg-flow-with-mfa).
* Crear una interfaz para que [los usuarios gestionen sus propios factores de autenticación](/docs/es/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

Para usar la API de MFA, debes habilitar el tipo de concesión de MFA para tu aplicación. Para habilitar la concesión de MFA en el <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, ve a [Dashboard &gt; Applications &gt; Advanced Settings &gt; Grant Types](https://manage.auth0.com/#/applications) y selecciona **MFA**. Consulta [Enable Multi-Factor Authentication](/docs/es/secure/multi-factor-authentication/enable-mfa) para más información.

<div id="limitations">
  ## Limitaciones
</div>

La API de MFA está diseñada para funcionar con los factores SMS, voz, push mediante la aplicación Auth0 Guardian, correo electrónico y OTP. Actualmente no admite registrar factores con Duo ni WebAuthn.
