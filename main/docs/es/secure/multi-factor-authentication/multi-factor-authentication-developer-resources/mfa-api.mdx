---
description: Describe la Auth0 MFA API, que le permite registrar y gestionar MFA para los usuarios.
title: Auth0 MFA API
---

Auth0 ofrece un flujo integrado de registro y autenticación de <Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuario que utiliza un factor además de nombre de usuario y contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> (MFA) mediante [Universal Login](/docs/es/authenticate/login/auth0-universal-login). Use la [API de MFA](https://auth0.com/docs/api/authentication#multi-factor-authentication) en los siguientes casos si desea:

* [Autenticar usuarios con la concesión de contraseña del propietario del recurso](/docs/es/secure/multi-factor-authentication/authenticate-using-ropg-flow-with-mfa).
* Crear una interfaz para que [los usuarios gestionen sus propios factores de autenticación](/docs/es/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

Para usar la API de MFA, debe habilitar el tipo de concesión MFA para su aplicación. Para habilitar la concesión MFA en el <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, vaya a [Dashboard &gt; Applications &gt; Advanced Settings &gt; Grant Types](https://manage.auth0.com/#/applications) y seleccione **MFA**. Consulte [Habilitar la autenticación multifactor](/docs/es/secure/multi-factor-authentication/enable-mfa) para obtener más información.

<div id="limitations">
  ## Limitaciones
</div>

La API de MFA está diseñada para funcionar con los factores SMS, voz, push a través de la app Auth0 Guardian, correo electrónico y OTP. Actualmente no admite la inscripción mediante Duo ni WebAuthn.
