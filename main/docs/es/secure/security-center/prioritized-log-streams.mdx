title: Prioritized Log Streams
---

**Prioritized Log Streams** es una funcionalidad del Dashboard que resalta los eventos de seguridad. En caso de ataque o de un aumento de la actividad de los usuarios, Prioritized Log Streams y los eventos predefinidos seguirán activos para que la automatización y los procesos de seguridad posteriores no se vean afectados.

Puedes crear un nuevo flujo de logs priorizado con un servicio compatible o configurar un webhook personalizado para enviar datos a un servicio que ya uses, y todos los logs pueden exportarse a un servicio de análisis de eventos. Los flujos de logs priorizados no cuentan para el límite de flujos de logs indicado en el plan de precios.

<div id="configure-a-prioritized-log-stream">
  #### Configurar un flujo de logs priorizado
</div>

1. Sigue las instrucciones de la [página de Log Streams](/docs/es/customize/log-streams) para configurar un nuevo flujo de logs.
2. Mientras configuras el nuevo flujo, ve a la pestaña **Settings** de tu Auth0 Dashboard y selecciona la casilla **Prioritized Log** para activar los eventos de seguridad predefinidos que se detallan a continuación. Selecciona **Save**.
3. Ve a la página de tu **flujo** y navega hasta la etiqueta **Prioritized**.

<div id="prioritized-logs-event-types">
  #### Tipos de eventos de registros priorizados
</div>

Auth0 clasifica los siguientes códigos de evento como eventos relacionados con la seguridad:

| Código de evento  | Evento                                                |
| ----------------- | ----------------------------------------------------- |
| `limit_mu`        | Dirección IP bloqueada                                |
| `limit_sul`       | Cuenta bloqueada (limitación por IP)                  |
| `limit_wc`        | Cuenta bloqueada (Brute Force Protection)             |
| `pwd_leak`        | Contraseña expuesta al iniciar sesión                 |
| `ublkdu`          | Se levantó el bloqueo de inicio de sesión del usuario |
| `signup_pwd_leak` | Contraseña expuesta en el registro                    |
| `reset_pwd_leak`  | Contraseña expuesta al restablecerla                  |
| `gd_send_sms`     | SMS de MFA enviado                                    |
