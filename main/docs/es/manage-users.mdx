description: Aprende a trabajar con usuarios, perfiles de usuario y metadatos de usuario en Auth0.
title: Administrar Users
---

La base de datos en la nube alojada por Auth0 almacena datos de usuario a los que puedes acceder a través de los perfiles de usuario. La información de los usuarios puede provenir de diversas fuentes, incluidos proveedores de identidad, tus propias bases de datos y conexiones empresariales (como Active Directory o SAML).
