El [endpoint `GET /api/v2/users/{id}`](https://auth0.com/docs/api/management/v2#!/Users/get_users_by_id) le permite recuperar un usuario específico mediante su ID de usuario de Auth0.

Este endpoint tiene **consistencia inmediata** y, por lo tanto, le recomendamos usarlo para:

* Búsquedas de usuarios realizadas durante el proceso de autenticación.
* Búsquedas de usuarios realizadas como parte del proceso de vinculación de cuentas.

El ID de usuario debe codificarse para URL, ya que puede contener caracteres que no funcionan bien en una URL.

La solicitud debe incluir un [token de acceso de la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens).

  ```go Go
  package main

  import (
  	"fmt"
  	"net/http"
  	"io/ioutil"
  )

  func main() {

  	url := "https://{yourDomain}/api/v2/users/%7BuserId%7D"

  	req, _ := http.NewRequest("GET", url, nil)

  	req.Header.Add("authorization", "Bearer {yourMgmtApiAccessToken}")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```
