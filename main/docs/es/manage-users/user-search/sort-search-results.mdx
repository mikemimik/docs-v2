Para ordenar los resultados de búsqueda de usuarios, pasa un valor `field:order` al parámetro `sort` al hacer tu solicitud. `field` es el nombre del campo por el que se ordena, mientras que `order` puede establecerse en `1` para orden ascendente y en `-1` para orden descendente.

No es compatible ordenar por `app_metadata` o `user_metadata`.

Por ejemplo, para ordenar los usuarios en orden ascendente por el campo `created_at`, puedes pasar el valor `created_at:1` al parámetro `sort`:

  import (

  }
