---
description: Describe cómo reenviar correos de verificación a los usuarios mediante el Dashboard de User Management de Auth0 y la Auth0 Management API.
title: Reenviar correos de verificación
---

Si necesitas reenviar un correo de verificación a un usuario, puedes hacerlo con el Dashboard de User Management de Auth0 o la Auth0 <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<Tabs>
  <Tab title="Usar Auth0 Dashboard">
    Puedes usar el Auth0 Dashboard para reenviar un correo de verificación a un usuario.

    1. En el Dashboard, ve a [**User Management &gt; Users**](https://manage.auth0.com/#/users).
    2. Selecciona al usuario.
    3. En el menú desplegable **Actions**, selecciona **Send Verification Email**.

    Si habilitas [múltiples dominios personalizados](/docs/es/customize/custom-domains/multiple-custom-domains), tendrás la opción **Domain** para seleccionar el dominio de notificación de tu tenant.
  </Tab>

  <Tab title="Usar Auth0 Management API">
    Puedes usar la Auth0 Management API para reenviar un correo de verificación a un usuario.

    1. Obtén un [token de acceso de la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens) con el scope `update:users`.
    2. Llama al endpoint de la Management API [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) con el `user_id` del usuario.

    #### Ejemplo de solicitud

    ```json lines
    {
       "user_id":"auth0|62463410c0242d00699461c8"
    }
    ```

    #### Ejemplo de respuesta

    ```json lines
    {
       "type":"verification_email",
       "status":"pending",
       "created_at":"2022-03-31T23:08:05.612Z",
       "id":"job_PyfpRHlmpkSoDmCv"
    }
    ```

    Si habilitas [múltiples dominios personalizados](/docs/es/customize/custom-domains/multiple-custom-domains), debes incluir el encabezado HTTP `auth0-custom-domain`. Para obtener más información, consulta [múltiples dominios personalizados](/docs/es/customize/custom-domains/multiple-custom-domains#customize-email-handling-using-the-management-api).

    #### Comprobar el estado de la tarea

    Puedes comprobar el estado de la tarea creada con el endpoint de la Management API [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email).

    1. Obtén el `id` de la tarea (se devuelve en la respuesta).
    2. Llama al endpoint de la Management API [Get a job](https://auth0.com/docs/api/management/v2#!/Jobs/get_jobs_by_id) con el `id` de la tarea.

    ##### Ejemplo de respuesta

    ```json lines
    {
       "type":"verification_email",
       "status":"completed",
       "created_at":"2022-03-31T23:08:05.612Z",
       "id":"job_PyfpRHlmpkSoDmCv"
    }
    ```
  </Tab>
</Tabs>
