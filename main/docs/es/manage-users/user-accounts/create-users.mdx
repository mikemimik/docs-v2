---
description: Describe cómo crear usuarios en el Auth0 Dashboard.
title: Crear usuarios
---

Puedes crear un usuario desde el Auth0 Dashboard o a través de la [Auth0 Management API](https://auth0.com/docs/api/management/v2/users/post-users).

1. Ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) y selecciona **Crear usuario**.
2. Selecciona la **Conexión** y, a continuación, introduce el **Correo electrónico** y la **Contraseña** del usuario.

   | Atributo           | Descripción                                                                                                                                                                                                                                                                                                                     |
   | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Conexión           | La conexión de base de datos que se utilizará para autenticar al usuario. La lista desplegable muestra todas las conexiones de base de datos configuradas en tu tenant. La conexión que uses debe estar asociada a una aplicación; de lo contrario, recibirás un mensaje de error indicando que la conexión está deshabilitada. |
   | Correo electrónico | La dirección de correo electrónico del usuario. El formato de la dirección de correo electrónico debe ajustarse a la especificación proporcionada en [RFC 5321 on IETF](https://datatracker.ietf.org/doc/html/rfc5321#section-4.1.2)                                                                                            |
   | Contraseña         | La contraseña del usuario. No existe un límite máximo para la longitud de la contraseña.                                                                                                                                                                                                                                        |
3. Selecciona **Create**.
   El usuario se creará y se te redirigirá al perfil del usuario recién creado.

Si habilitas [múltiples dominios personalizados](/docs/es/customize/custom-domains/multiple-custom-domains), tendrás la opción de seleccionar **Dominio** en la sección **Conexión** para elegir el dominio de notificaciones de tu tenant.

Si habilitas [Identificadores flexibles](/docs/es/authenticate/database-connections/flexible-identifiers-and-attributes), además de **email** puedes seleccionar el **username**, el **phone** del usuario o una combinación de dos o más.
