---
description: Explica cómo crear usuarios en el Auth0 Dashboard.
title: Crear usuarios
---

Puede crear un usuario con el Auth0 Dashboard o mediante la [Auth0 Management API](https://auth0.com/docs/api/management/v2/users/post-users).

1. Vaya a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) y seleccione **Crear usuario**.
2. Seleccione la **Conexión** y luego introduzca el **Email** y la **Password** del usuario.

   | Atributo | Descripción                                                                                                                                                                                                                                                                                                                       |
   | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Conexión | La conexión de base de datos que se utilizará para autenticar al usuario. La lista desplegable muestra todas las conexiones de base de datos configuradas en su tenant. La conexión que utilice debe estar asociada a una aplicación; de lo contrario, recibirá un mensaje de error indicando que la conexión está deshabilitada. |
   | Email    | La dirección de correo electrónico del usuario. El formato de la dirección de correo electrónico debe ajustarse a la especificación indicada en [RFC 5321 on IETF](https://datatracker.ietf.org/doc/html/rfc5321#section-4.1.2)                                                                                                   |
   | Password | La contraseña del usuario. No existe un límite máximo para la longitud de la contraseña.                                                                                                                                                                                                                                          |
3. Seleccione **Crear**.
   El usuario se crea y se le redirigirá al perfil del usuario recién creado.

Si habilita [múltiples dominios personalizados](/docs/es/customize/custom-domains/multiple-custom-domains), tendrá la opción de seleccionar **Domain** en la sección **Conexión** para elegir el dominio de notificación de su tenant.

Si habilita [Flexible Identifiers](/docs/es/authenticate/database-connections/flexible-identifiers-and-attributes), además de **email** puede seleccionar el **username** o el **phone** del usuario, o una combinación de dos o más.
