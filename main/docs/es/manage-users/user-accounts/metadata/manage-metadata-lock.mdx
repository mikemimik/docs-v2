description: Aprende a gestionar los metadatos de usuario y de la aplicación con la biblioteca Auth0 Lock.
title: Gestiona metadatos con Lock
---

Puedes usar la [biblioteca Lock](/docs/es/libraries/lock) para definir, agregar, leer y actualizar `user_metadata`. Puedes leer las propiedades de `user_metadata` del usuario igual que leerías cualquier otra propiedad del perfil de usuario. Por ejemplo, el siguiente fragmento de código recupera el valor asociado a `user_metadata.hobby` y lo asigna a un elemento de la página:

```javascript lines
// Usa el accessToken obtenido durante la autenticación para llamar a getUserInfo
lock.getUserInfo(accessToken, function(error, profile) {
  if (!error) {
    document.getElementById('hobby').textContent = profile.user_metadata.hobby;
  }
});
```

Puedes usar `additionalSignUpFields` para agregar campos personalizados a los formularios de registro de usuarios. Cuando un usuario agrega datos en un campo personalizado, Auth0 almacena los valores ingresados en el `user_metadata` de ese usuario. Para obtener más información sobre cómo agregar `user_metadata` durante el registro, lee [Campos adicionales de registro](/docs/es/libraries/lock/lock-configuration#additionalsignupfields-array-).
