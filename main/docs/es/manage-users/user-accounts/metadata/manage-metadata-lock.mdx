description: Aprende a administrar los metadatos de usuario y de la aplicación con la biblioteca Auth0 Lock.
title: Administrar metadatos con Lock
---

Puedes usar la [biblioteca Lock](/docs/es/libraries/lock) para definir, agregar, leer y actualizar `user_metadata.` Puedes leer las propiedades de `user_metadata` del usuario igual que cualquier otra propiedad del perfil de usuario. Por ejemplo, el siguiente fragmento de código recupera el valor asociado a `user_metadata.hobby` y lo asigna a un elemento de la página:

```javascript lines
// Usa el accessToken obtenido en la autenticación para llamar a getUserInfo
lock.getUserInfo(accessToken, function(error, profile) {
  if (!error) {
    document.getElementById('hobby').textContent = profile.user_metadata.hobby;
  }
});
```

Puedes usar `additionalSignUpFields` para añadir campos personalizados a los formularios de registro. Cuando un usuario introduce datos en un campo personalizado, Auth0 almacena los valores introducidos en el `user_metadata` de ese usuario. Para obtener más información sobre cómo añadir `user_metadata` durante el registro, consulta [Additional Signup Fields](/docs/es/libraries/lock/lock-configuration#additionalsignupfields-array-).
