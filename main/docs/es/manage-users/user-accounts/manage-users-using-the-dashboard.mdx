description: Describe cómo administrar usuarios en el Auth0 Dashboard.
title: Administrar Users con el Dashboard
---

Puedes usar el Dashboard para administrar tus usuarios. Para comenzar, ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users). Aquí puedes crear, ver, modificar o eliminar usuarios.

<Card title="La disponibilidad varía según el plan de Auth0">
  Tanto tu implementación específica de login como tu plan de Auth0 o acuerdo personalizado determinan si esta funcionalidad está disponible. Para obtener más información, consulta [Pricing](https://auth0.com/pricing).
</Card>

De forma predeterminada, los atributos del perfil de usuario proporcionados por <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=identity+providers">proveedores de identidad</Tooltip> distintos de Auth0 (como Google, Facebook o X) no se pueden editar directamente porque se actualizan desde el proveedor de identidad cada vez que el usuario inicia sesión. Para obtener más información sobre los atributos de los proveedores de identidad, consulta [Configurar la conexión del proveedor de identidad para las actualizaciones del perfil de usuario](/docs/es/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0).

Para poder editar los atributos raíz `name`, `nickname`, `given_name`, `family_name` o `picture` del perfil de usuario normalizado, debes [configurar la sincronización de tu conexión con Auth0](/docs/es/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0) para que los atributos del usuario se actualicen desde el proveedor de identidad solo cuando se cree el perfil de usuario. Estos atributos raíz estarán entonces disponibles para editarse individualmente o mediante importaciones masivas

Para aprender a administrar el acceso al <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> de los miembros de tu equipo, consulta [Administrar el acceso al Dashboard](/docs/es/get-started/manage-dashboard-access).
