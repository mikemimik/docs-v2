Esta guía le mostrará cómo establecer atributos raíz para un usuario durante la importación mediante la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> de Auth0. Esto le permite minimizar la cantidad de llamadas a la API necesarias para establecer atributos raíz al importar usuarios. Para ver qué atributos puede importar, visite [Estructura del perfil de usuario normalizado](/docs/es/manage-users/user-accounts/user-profiles/user-profile-structure).

Haga una llamada `POST` al [endpoint Create Job to Import Users](https://auth0.com/docs/api/management/v2#!/Jobs/post_users_imports). Asegúrese de reemplazar los valores de marcador de posición `MGMT_API_ACCESS_TOKEN`, `CONNECTION_ID` y `JSON_USER_FILE_PATH` por su <Tooltip tip="Token de acceso: Credencial de autorización, en forma de una cadena opaca o JWT, usada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">Token de acceso</Tooltip> de la Management API, el ID de la conexión y el nombre del archivo JSON de usuarios, respectivamente.

  ```bash cURL
  curl --request POST \
    --url 'https://{yourDomain}/api/v2/jobs/usersimports' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
    --header 'content-type: multipart/form-data ' \
    --data '{ "connection_id": "CONNECTION_ID", "users": "JSON_USER_FILE_PATH" }'
  ```

  ```csharp C#

  request.AddHeader("content-type", "multipart/form-data ");
  ```

  ```go Go
  package main

  import (
  	"fmt"
  	"strings"
  	"net/http"
  	"io/ioutil"

  func main() {

  	url := "https://{yourDomain}/api/v2/jobs/usersimports"

  	payload := strings.NewReader("{ "connection_id": "CONNECTION_ID", "users": "JSON_USER_FILE_PATH" }")

  	req, _ := http.NewRequest("POST", url, payload)

  	req.Header.Add("content-type", "multipart/form-data ")
  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

| Valor                   | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](https://auth0.com/docs/api/management/v2/tokens) con el [scope](/docs/es/glossary?term=scope) `create:users`.                                                                                                                                                                                                                                                                                                                                                                                       |
| `CONNECTION_ID`         | ID de la conexión en la que se insertarán los usuarios. Puedes obtener esta información mediante el [endpoint Get All Connections](https://auth0.com/docs/api/management/v2#!/Connections/get_connections).                                                                                                                                                                                                                                                                                                                               |
| `JSON_USER_FILE_PATH`   | Nombre del archivo que contiene los usuarios que se importarán. El archivo debe estar en formato JSON e incluir atributos raíz de los usuarios. Para ver una lista de los atributos disponibles, consulta [Atributos del perfil de usuario](/docs/es/manage-users/user-accounts/user-profiles/user-profile-structure#attributes). Para ver un ejemplo del formato del archivo, consulta [Esquema y ejemplos de importación masiva de usuarios de Database](/docs/es/manage-users/user-migration/bulk-user-import-database-schema-and-examples). |
