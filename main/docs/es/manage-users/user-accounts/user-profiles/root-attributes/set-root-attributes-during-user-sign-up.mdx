Puede establecer atributos raíz para un usuario durante el registro con la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> de Auth0. Esto le permite minimizar la cantidad de llamadas a la API necesarias para establecer atributos raíz al crear usuarios.

Haga una llamada `POST` al [endpoint Create a User](https://auth0.com/docs/api/management/v2#!/Users/post_users). Asegúrese de reemplazar los valores de marcador de posición `MGMT_API_ACCESS_TOKEN`, `CONNECTION_NAME`, `EMAIL_VALUE`, `PASSWORD_VALUE`, `GIVEN_NAME_VALUE`, `FAMILY_NAME_VALUE`, `NAME_VALUE`, `NICKNAME_VALUE` y `PICTURE` por su <Tooltip tip="Access Token: Credencial de Authorization, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">token de acceso</Tooltip> de la Management API, el nombre inicial de la conexión, la dirección de correo electrónico, la contraseña, el nombre de pila, el apellido, el nombre, el apodo y la URL de la foto, respectivamente.

</AuthCodeGroup>

| Valor                   | Descripción                                                                                                                                      |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `MGMT_API_ACCESS_TOKEN` | [token de acceso for the Management API](https://auth0.com/docs/api/management/v2/tokens) con el [scope](/docs/es/glossary?term=scope) `create:users`. |
| `CONNECTION_NAME`       | Nombre de la conexión a través de la cual se recibió la información inicial del usuario.                                                         |
| `EMAIL_VALUE`           | Dirección de correo electrónico del usuario que se va a crear.                                                                                   |
| `PASSWORD_VALUE`        | Contraseña del usuario que se va a crear.                                                                                                        |
| `GIVEN_NAME_VALUE`      | Nombre de pila del usuario que se va a crear.                                                                                                    |
| `FAMILY_NAME_VALUE`     | Apellido del usuario que se va a crear.                                                                                                          |
| `NAME_VALUE`            | Nombre completo del usuario que se va a crear.                                                                                                   |
| `NICKNAME_VALUE`        | Apodo del usuario que se va a crear.                                                                                                             |
| `PICTURE_VALUE`         | URL de la foto del usuario que se va a crear.                                                                                                    |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Si usas Lock o el [endpoint público de Signup](https://auth0.com/docs/api/authentication#signup) para registrar usuarios, puedes establecer atributos raíz con el mismo método.
</Callout>
