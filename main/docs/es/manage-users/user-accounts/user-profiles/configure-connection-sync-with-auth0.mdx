Puede actualizar las preferencias de conexión de un <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=identity+provider">proveedor de identidad</Tooltip> de origen para controlar cuándo se permiten las actualizaciones de los atributos raíz del perfil de usuario mediante el <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

De forma predeterminada, los atributos del perfil de usuario proporcionados por proveedores de identidad distintos de Auth0 (como Google, Facebook o X) no se pueden editar directamente, ya que se actualizan desde el proveedor de identidad cada vez que el usuario inicia sesión.

Para poder editar los atributos raíz name, nickname, given&#95;name, family&#95;name o picture en el perfil de usuario normalizado, debe configurar la sincronización de su conexión con Auth0 para que los atributos del usuario se actualicen desde el proveedor de identidad solo al crear el perfil de usuario.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Sync Users at Login** no se aplica a las conexiones passwordless y no se puede habilitar desde la interfaz de usuario del tenant ni mediante la Management API.
</Callout>

Puede editar los [atributos raíz individualmente](/docs/es/manage-users/user-accounts/user-profiles/root-attributes/update-root-attributes-for-users) o mediante una [importación masiva](/docs/es/manage-users/user-accounts/user-profiles/root-attributes/set-root-attributes-during-user-import) usando la Management API.

<div id="use-the-dashboard">
  ## Usa el Dashboard
</div>

1. Ve a [Auth0 Dashboard &gt; Authentication](https://manage.auth0.com/#/connections) y selecciona el tipo de conexión: Database, Social, empresarial o Passwordless.
2. Selecciona el nombre de una conexión para ver su configuración.
3. Busca la sección **Advanced**, activa o desactiva el interruptor **sincronizar atributos del perfil de usuario en cada inicio de sesión** según corresponda y selecciona **Save**.

<div id="use-the-management-api">
  ## Usa la Management API
</div>

Antes de completar este paso, primero debes [recuperar los valores existentes del objeto `options` de la conexión](/docs/es/authenticate/identity-providers/retrieve-connection-options) para evitar sobrescribir los valores actuales. Si no lo haces, después de actualizar se perderán los parámetros que falten del objeto original.

Haz una llamada `PATCH` al [endpoint actualizar una conexión](https://auth0.com/docs/api/management/v2#!/Connections/patch_connections_by_id). Asegúrate de incluir en la llamada los valores originales de `options` para evitar sobrescribir los valores actuales. Además, asegúrate de reemplazar los valores de marcador de posición `CONNECTION_ID`, `MGMT_API_ACCESS_TOKEN` y `ATTRIBUTE_UPDATE_VALUE` por tu ID de conexión, el <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, que se usa para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> de la Management API y el valor de actualización del atributo, respectivamente.

| Valor                    | Descripción                                                                                                                                                                                                                     |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `CONNECTION_ID`          | ID de la conexión para la que quieres permitir actualizaciones de los atributos raíz.                                                                                                                                           |
| `MGMT_API_ACCESS_TOKEN`  | [Access Token for the Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `update:connections`.                                                                                                       |
| `ATTRIBUTE_UPDATE_VALUE` | Indica cuándo quieres permitir actualizaciones de los atributos raíz del perfil del usuario. Los valores válidos son `on_first_login` y `on_each_login`. El valor predeterminado para las conexiones nuevas es `on_each_login`. |
