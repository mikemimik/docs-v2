Para autorizar el acceso de máquina a máquina de una aplicación, debes habilitarla para que use una organización con una API específica. Para ello, debes asociar la autorización de cliente de la aplicación para la API correspondiente con la organización. Una vez asociada, la aplicación puede usar la organización al solicitar tokens para la API y los scopes definidos en la autorización de cliente. Deberás definir esta asociación para cada API de la organización a la que tu aplicación necesite acceder.

Puedes autorizar el acceso de máquina a máquina de una aplicación mediante [Auth0 Dashboard](https://manage.auth0.com/) o la [Management API](https://auth0.com/docs/api/management/v2).

<Tabs>
  <Tab title="Auth0 Dashboard">
    Para asociar la autorización de cliente de una aplicación con una organización desde Auth0 Dashboard:

    1. Ve a **Organizations** y elige la organización con la que quieres asociarla.
    2. Selecciona la pestaña **Machine-to-Machine Access**.
    3. Haz clic en **Add Access.**
    4. Haz clic en la aplicación que quieres asociar con la organización.
    5. Selecciona una API.
    6. Haz clic en **Save.**
