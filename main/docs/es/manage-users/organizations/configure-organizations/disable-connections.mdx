<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Para deshabilitar una conexión desde el Auth0 Dashboard:

1. Ve a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) y selecciona la organización para la que quieres configurar las conexiones.
2. Desactiva el campo **Enabled** de la conexión que quieres deshabilitar.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Deshabilitar una conexión no elimina la asociación con la organización. Al deshabilitar una conexión, es posible conservar los valores de `show_as_button`, `assign_membership_on_login`, etc., y mantener la conexión deshabilitada.
</Callout>

Para eliminar una conexión de una organización desde el Auth0 Dashboard:

1. Ve a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) y selecciona la organización para la que quieres configurar las conexiones.
2. Selecciona la vista **Connections**, localiza la conexión que quieres eliminar y despliega su menú **More Options** (**...**).
3. Selecciona **Remove Connection** y confirma.

<div id="management-api">
  ## Management API
</div>

Para deshabilitar una conexión desde el Auth0 Dashboard:
Haz una llamada `PATCH` al endpoint `Update Organization Connections`. Asegúrate de reemplazar los valores de marcador de posición `ORG_ID`, `MGMT_API_ACCESS_TOKEN` y `CONNECTION_ID` por el ID de tu organización, tu <Tooltip tip="Token de acceso: credencial de Authorization, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">token de acceso</Tooltip> de Management API y el ID de la conexión, respectivamente.

Para eliminar una conexión de una organización mediante el Auth0 Dashboard:
Haz una llamada `DELETE` al endpoint `Delete Organization Connections`. Asegúrate de reemplazar los placeholder values `ORG_ID`, `MGMT_API_ACCESS_TOKEN` y `CONNECTION_ID` por tu ID de la organización, el <Tooltip tip="Token de acceso: credencial de Authorization, en forma de cadena opaca o JWT, que se utiliza para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">token de acceso</Tooltip> de la Management API y el ID de la conexión, respectivamente.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Encuentra tu dominio de Auth0**

  Si tu dominio de Auth0 está formado por el nombre de tu tenant, tu subdominio regional (a menos que tu tenant esté en la región de EE. UU. y se haya creado antes de junio de 2020) y `.auth0.com`. Por ejemplo, si el nombre de tu tenant fuera `travel0`, tu dominio de Auth0 sería `travel0.us.auth0.com`. (Si tu tenant estuviera en EE. UU. y se hubiera creado antes de junio de 2020, entonces tu nombre de dominio sería `https://travel0.auth0.com`).

  Si usas dominios personalizados, este debe ser tu dominio personalizado.
</Callout>

| Valor                   | Descripción                                                                                                                                                 |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | ID de la organización para la que desea deshabilitar una conexión.                                                                                          |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens) con el scope `delete:organization_connections`. |
| `CONNECTION_ID`         | ID de la conexión que desea deshabilitar para la organización.                                                                                              |

<div id="response-status-codes">
  ### Códigos de estado de respuesta
</div>

Los posibles códigos de estado de respuesta son los siguientes:

| Código de estado | Código de error      | Mensaje                                                                                                     | Causa                                                                                                    |
| ---------------- | -------------------- | ----------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| `204`            |                      | Conexión eliminada correctamente de la Organización.                                                        |                                                                                                          |
| `401`            |                      | Token no válido.                                                                                            |                                                                                                          |
| `401`            |                      | Se recibió una firma no válida para la validación del JSON Web Token.                                       |                                                                                                          |
| `401`            |                      | El cliente no es global.                                                                                    |                                                                                                          |
| `403`            | `insufficient_scope` | Scope insuficiente; se esperaba alguno de: `delete:organization_connections`.                               | Se intentó leer o escribir un campo que no está permitido con los scopes del bearer token proporcionado. |
| `429`            |                      | Demasiadas solicitudes. Verifique los headers X-RateLimit-Limit, X-RateLimit-Remaining y X-RateLimit-Reset. |                                                                                                          |
