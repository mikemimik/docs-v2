Puede quitar miembros de las [organizaciones](/docs/es/manage-users/organizations/organizations-overview) mediante el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Al quitar un usuario de una organización, solo se eliminará la membresía de ese usuario en la organización indicada; no se eliminará al usuario de la conexión ni del tenant de Auth0. El proceso tarda aproximadamente entre 5 y 10 segundos.

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Para eliminar un miembro desde Auth0 Dashboard:

1. Ve a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) y selecciona la organización cuya membresía quieres configurar.
2. Selecciona la vista **Members**, busca al miembro que quieres eliminar y despliega el menú **Más opciones** (**...**).
3. Selecciona **Eliminar miembro** y confirma.

<div id="management-api">
  ## Management API
</div>

Para eliminar miembros mediante la Management API:
Haz una llamada `DELETE` al endpoint `Delete Organization Members`. Asegúrate de reemplazar los valores de marcador de posición `ORG_ID`, `MGMT_API_ACCESS_TOKEN` y `USER_ID` por el ID de tu organización, el <Tooltip tip="Access Token: Credencial de Authorization, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">token de acceso</Tooltip> de la Management API y el ID de usuario, respectivamente. Puedes eliminar hasta 10 miembros por solicitud.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Encuentra tu dominio de Auth0**

  Tu dominio de Auth0 se compone del nombre de tu tenant, tu subdominio regional (a menos que tu tenant esté en la región de EE. UU. y se haya creado antes de junio de 2020) y `.auth0.com`. Por ejemplo, si el nombre de tu tenant fuera `travel0`, tu dominio de Auth0 sería `travel0.us.auth0.com`. (Si tu tenant estuviera en EE. UU. y se hubiera creado antes de junio de 2020, entonces tu nombre de dominio sería `https://travel0.auth0.com`).

  Si usas dominios personalizados, este debe ser tu dominio personalizado.
</Callout>

| Valor                   | Descripción                                                                                                                                          |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | ID de la organización de la que quieres eliminar la membresía.                                                                                       |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens) con el scope `delete:organization_members`. |
| `USER_ID`               | ID del usuario o de los usuarios que quieres eliminar de la organización especificada.                                                               |

<div id="response-status-codes">
  ### Códigos de estado de respuesta
</div>

Los posibles códigos de estado de respuesta son los siguientes:

| Código de estado | Código de error        | Mensaje                                                                                                  | Causa                                                                                                    |
| ---------------- | ---------------------- | -------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| `204`            |                        | Usuarios eliminados correctamente de la organización.                                                    |                                                                                                          |
| `400`            | `invalid_body`         | Cuerpo de la solicitud no válido. El mensaje variará según la causa.                                     | El payload de la solicitud no es válido.                                                                 |
| `400`            | `invalid_query_string` | Cadena de consulta de la solicitud no válida. El mensaje variará según la causa.                         | La cadena de consulta no es válida.                                                                      |
| `401`            |                        | Token no válido.                                                                                         |                                                                                                          |
| `401`            |                        | Se recibió una firma no válida para la validación de JSON Web Token.                                     |                                                                                                          |
| `401`            |                        | El cliente no es global.                                                                                 |                                                                                                          |
| `403`            | `insufficient_scope`   | Scope insuficiente; se esperaba alguno de: `delete:organization_members`.                                | Se intentó leer o escribir un campo que no está permitido con los scopes del bearer token proporcionado. |
| `429`            |                        | Demasiadas solicitudes. Revisa los headers X-RateLimit-Limit, X-RateLimit-Remaining y X-RateLimit-Reset. |                                                                                                          |
