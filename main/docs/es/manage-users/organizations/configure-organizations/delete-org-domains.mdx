---
title: Eliminar dominios de organización
description: Elimina la asociación entre una organización y un dominio a través del Auth0 Dashboard o de la Management API.
---

Al eliminar un dominio de organización, la asociación entre el dominio y la organización se quita de forma permanente. Esta operación no se puede deshacer.

<div id="delete-an-organization-domain-with-the-auth0-dashboard">
  ## Eliminar un dominio de organización con Auth0 Dashboard
</div>

Para eliminar un dominio de organización en Auth0 Dashboard:

1. Vaya a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) y elija su organización.
2. Seleccione la pestaña **Dominios**.
3. Junto al dominio que desea quitar, abra el menú de acciones (**...**) y seleccione **Delete**.
4. En la ventana que se abre, confirme la eliminación.

<div id="delete-an-organization-domain-with-the-management-api">
  ## Eliminar un dominio de organización con la Management API
</div>

Para eliminar un dominio de organización con la Management API, llama al [endpoint para eliminar un dominio de descubrimiento de una organización](/docs/es/api/management/v2/organizations/delete-discovery-domains-by-discovery-domain-id) (`DELETE /organizations/{id}/discovery-domains/{discovery_domain_id}`).
