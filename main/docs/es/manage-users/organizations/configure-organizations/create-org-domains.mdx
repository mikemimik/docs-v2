---
title: Crear dominios de organización
description: Indique a los usuarios que elijan la organización correcta durante el registro y los flujos de inicio de sesión con dominios de organización verificados.
---

Los dominios de organización funcionan con [Organization Domain Discovery](./define-organization-behavior) para ayudar a los usuarios a encontrar la organización correcta durante el registro y los [flujos de inicio de sesión](/docs/es/manage-users/organizations/login-flows-for-organizations).

Cuando se configuran, los dominios de organización verificados y Organization Domain Discovery ayudan a Auth0 a detectar automáticamente la organización de un usuario o a reducir las opciones de organización cuando los usuarios introducen su dirección de correo electrónico. Los dominios de organización y Organization Domain Discovery no restringen quién puede registrarse o iniciar sesión.

<div id="create-an-organization-domain-with-the-auth0-dashboard">
  ## Crear un dominio de organización con Auth0 Dashboard
</div>

Para crear un dominio de organización en Auth0 Dashboard:

1. Vaya a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations).
2. Seleccione la pestaña **Domains**.
3. Haga clic en **Add Domain**.
   En el campo **Domain**, introduzca el dominio de correo electrónico que desea asociar para el descubrimiento de organizaciones. El estado permanecerá pendiente hasta que se confirme la propiedad.
4. Seleccione **Add Domain**.
5. De forma opcional, configure [Organization Domain Discovery](./define-organization-behavior).

<div id="create-an-organization-domain-with-the-management-api">
  ## Crear un dominio de organización con la Management API
</div>

Para crear un dominio de organización con la Management API, llame al [endpoint Create an organization discovery domain](/docs/es/api/management/v2/organizations/post-discovery-domains) (`POST /organizations/{id}/discovery-domains`). También puede habilitar el dominio para Organization Domain Discovery con el parámetro `use_for_organization_discovery`.

Para habilitar Organization Domain Discovery, llame al [endpoint Patch clients by ID](/docs/es/api/management/v2/clients/patch-clients-by-id#body-organization-discovery-methods-one-of-0) (`PATCH /client/{id}`) y configure el parámetro `organization_discovery_methods`.
