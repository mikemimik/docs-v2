---
title: Crear dominios de organización
description: Pida a los usuarios que elijan la organización correcta durante el registro y los flujos de inicio de sesión con dominios de organización verificados.
---

Los dominios de organización funcionan con [Organization Domain Discovery](./define-organization-behavior) para ayudar a los usuarios a encontrar la organización correcta durante el registro y los [flujos de inicio de sesión](/docs/es/manage-users/organizations/login-flows-for-organizations).

Una vez configurados, los dominios de organización verificados y Organization Domain Discovery ayudan a Auth0 a detectar automáticamente la organización de un usuario o a acotar las opciones de organización cuando los usuarios introducen su dirección de correo electrónico. Los dominios de organización y Organization Domain Discovery no restringen quién puede registrarse o iniciar sesión.

<div id="create-an-organization-domain-with-the-auth0-dashboard">
  ## Crear un dominio de organización con Auth0 Dashboard
</div>

Para crear un dominio de organización en Auth0 Dashboard:

1. Ve a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations).
2. Selecciona la pestaña **Domains**.
3. Elige **Add Domain**.
   En el campo **Domain**, introduce el dominio de correo electrónico que quieres asociar para detectar la organización. El estado aparecerá como pendiente hasta que se confirme la propiedad.
4. Selecciona **Add Domain**.
5. Opcionalmente, configura [Organization Domain Discovery](./define-organization-behavior).

<div id="create-an-organization-domain-with-the-management-api">
  ## Crear un dominio de organización con la Management API
</div>

Para crear un dominio de organización con la Management API, llama al [endpoint de creación de un dominio de descubrimiento de organización](/docs/es/api/management/v2/organizations/post-discovery-domains) (`POST /organizations/{id}/discovery-domains`). De forma opcional, habilita el dominio para Organization Domain Discovery con el parámetro `use_for_organization_discovery`.

Para habilitar Organization Domain Discovery, llama al [endpoint Patch clients by ID](/docs/es/api/management/v2/clients/patch-clients-by-id#body-organization-discovery-methods-one-of-0) (`PATCH /client/{id}`) y establece el parámetro `organization_discovery_methods`.
