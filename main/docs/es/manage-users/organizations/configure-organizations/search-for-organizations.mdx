description: Aprende a buscar Organizations con el Auth0 Dashboard.
title: Buscar Organizations
---

Desde la sección Organizations de tu [Auth0 Dashboard](https://manage.auth0.com/#/organizations), puedes buscar [organizaciones](/docs/es/manage-users/organizations/organizations-overview) por nombre o nombre para mostrar.

<Frame>![Auth0 Dashboard - Organizations - Barra de búsqueda ](/docs/images/cdy7uua7fh8z/5CeeKLjPGxjTqy682lbnyj/3a41b87a369e65ca624b1822c99dbc57/2025-01-27_17-27-28.png)</Frame>

El menú desplegable junto a la barra de búsqueda te permite buscar por Name o Display Name. Si introduces tres caracteres o menos en la barra de búsqueda, solo se devolverán coincidencias exactas. Si introduces cuatro o más caracteres, se devolverán coincidencias parciales que contengan la cadena indicada. Por ejemplo, si tienes una organización llamada &quot;Big Holdings, Co.&quot; y escribes los términos de búsqueda `hold` o `ings`, se devolverá la organización Big Holdings, Co.

Las búsquedas devuelven un máximo de 50 resultados. Si no encuentras el resultado que buscas, prueba con un término de búsqueda más específico.

<Warning>
  La búsqueda de organizaciones utiliza consistencia eventual. Es posible que las organizaciones creadas o modificadas recientemente no aparezcan de inmediato en los resultados de búsqueda; sin embargo, con el tiempo, los resultados reflejarán con precisión las entidades de tu tenant.
</Warning>
