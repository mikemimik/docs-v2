Para asignar un miembro a tu [organización](/docs/es/manage-users/organizations/organizations-overview), primero debes haber [creado al usuario](/docs/es/manage-users/user-accounts/create-users) en tu tenant. Si no encuentras al usuario, puedes [invitarlo](/docs/es/manage-users/organizations/configure-organizations/invite-members) en su lugar.

Para gestionar miembros directamente, puedes usar el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Para asignar miembros mediante el Auth0 Dashboard:

1. Vaya a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) y seleccione la organización cuya membresía desea configurar.
2. Seleccione la vista **Members**, luego **Agregar miembros** y, a continuación, **Agregar usuarios**.
3. Introduzca el nombre de los usuarios que desea asignar como miembros de la organización y seleccione **Agregar usuario(s) a la organización**.

<div id="management-api">
  ## Management API
</div>

Para asignar miembros mediante la Management API:

Haz una solicitud `POST` al endpoint `Create Organization Members`. Asegúrate de reemplazar los valores de marcador de posición `ORG_ID`, `MGMT_API_ACCESS_TOKEN` y `USER_ID` por tu ID de organización, tu <Tooltip tip="Token de acceso: credencial de Authorization, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">token de acceso</Tooltip> de la Management API y los ID de usuario, respectivamente.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Encuentra tu dominio de Auth0**

  Tu dominio de Auth0 es el nombre de tu tenant, seguido de tu subdominio regional (a menos que tu tenant esté en la región de EE. UU. y se haya creado antes de junio de 2020), más `.auth0.com`. Por ejemplo, si el nombre de tu tenant fuera `travel0`, tu dominio de Auth0 sería `travel0.us.auth0.com`. (Si tu tenant estuviera en EE. UU. y se hubiera creado antes de junio de 2020, entonces tu nombre de dominio sería `https://travel0.auth0.com`.)

  Si usas dominios personalizados, este debe ser tu dominio personalizado.
</Callout>

| Valor                   | Descripción                                                                                                                                                 |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | ID de la organización a la que quieres asignar la membresía.                                                                                                |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens) con el scope `create:organization_members`.        |
| `USER_ID`               | ID del usuario o de los usuarios que quieres asignar a la organización especificada. Puedes enviar un máximo de 10 miembros por organización por solicitud. |

<div id="response-status-codes">
  ### Códigos de estado de la respuesta
</div>

Los posibles códigos de estado de la respuesta son los siguientes:

| Código de estado | Código de error        | Mensaje                                                                                                        | Causa                                                                                                    |
| ---------------- | ---------------------- | -------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| `204`            |                        | Miembros agregados correctamente a la organización.                                                            |                                                                                                          |
| `400`            | `invalid_body`         | Cuerpo de la solicitud no válido. El mensaje variará según la causa.                                           | El payload de la solicitud no es válido.                                                                 |
| `400`            | `invalid_query_string` | Cadena de consulta de la solicitud no válida. El mensaje variará según la causa.                               | La cadena de consulta no es válida.                                                                      |
| `401`            |                        | Token no válido.                                                                                               |                                                                                                          |
| `401`            |                        | Se recibió una firma no válida para la validación de JSON Web Token.                                           |                                                                                                          |
| `401`            |                        | El cliente no es global.                                                                                       |                                                                                                          |
| `403`            | `insufficient_scope`   | Scope insuficiente; se esperaba alguno de los siguientes: `create:organization_members`.                       | Se intentó leer o escribir un campo que no está permitido con los scopes del bearer token proporcionado. |
| `429`            |                        | Demasiadas solicitudes. Consulta los encabezados X-RateLimit-Limit, X-RateLimit-Remaining y X-RateLimit-Reset. |                                                                                                          |
