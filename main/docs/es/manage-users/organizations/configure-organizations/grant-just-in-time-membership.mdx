---
description: Aprende cómo conceder membresía Just-In-Time (JIT) a organizaciones.
title: Conceder membresía Just-In-Time a una conexión de organización
---

Conceder membresía just-in-time permite que los usuarios que inician sesión mediante una conexión habilitada obtengan automáticamente membresía en una [organización](/docs/es/manage-users/organizations/organizations-overview).

Para conceder membresía automáticamente cuando los usuarios inician sesión en una organización, primero debes [configurar y habilitar una conexión](/docs/es/manage-users/organizations/configure-organizations/enable-connections) para la organización. Al habilitar la conexión, selecciona **Habilitar membresía automática** para que todos los usuarios que inicien sesión con esa conexión se agreguen automáticamente como miembros de esta organización.
