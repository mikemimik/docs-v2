Puede configurar Organizations mediante el <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Para obtener más información sobre los pasos necesarios para configurar su organización, consulte [Crea tu primera organización](/docs/es/manage-users/organizations/create-first-organization).

<Card title="La disponibilidad varía según el plan de Auth0">
  La disponibilidad de esta funcionalidad depende de su plan de Auth0 o de su acuerdo personalizado. Para obtener más información, consulte [Pricing](https://auth0.com/pricing).
</Card>
