Puedes configurar Organizations desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Para obtener más información sobre los pasos necesarios para configurar tu organización, consulta [Create Your First Organization](/docs/es/manage-users/organizations/create-first-organization).

<Card title="La disponibilidad varía según el plan de Auth0">
  La disponibilidad de esta funcionalidad depende de tu plan de Auth0 o de tu acuerdo personalizado. Para obtener más información, consulta [Pricing](https://auth0.com/pricing).
</Card>
