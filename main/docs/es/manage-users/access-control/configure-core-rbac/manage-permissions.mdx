* [Asignar permisos a usuarios](/docs/es/manage-users/access-control/configure-core-rbac/rbac-users/assign-permissions-to-users)
* [Ver permisos de los roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Ver permisos de los usuarios](/docs/es/manage-users/access-control/configure-core-rbac/rbac-users/view-user-permissions)
