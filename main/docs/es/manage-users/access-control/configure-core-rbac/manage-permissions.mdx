* [Asignar permisos a los usuarios](/docs/es/manage-users/access-control/configure-core-rbac/rbac-users/assign-permissions-to-users)

* [Ver los permisos de un usuario](/docs/es/manage-users/access-control/configure-core-rbac/rbac-users/view-user-permissions)
