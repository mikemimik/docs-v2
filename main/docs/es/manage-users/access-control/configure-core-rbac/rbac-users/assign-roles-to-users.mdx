Puedes asignar [roles](/docs/es/manage-users/access-control/rbac) a un usuario mediante el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Los roles asignados pueden usarse con el conjunto de funcionalidades Authorization Core de la API.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Para que el control de acceso basado en roles (RBAC) funcione correctamente, debe habilitarlo para su API mediante el Dashboard o la Management API. La funcionalidad de Authorization Core es distinta de la de Authorization Extension. Para consultar una comparación, lea [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).
* Los roles se seleccionan de valores predefinidos. Si la lista de roles está vacía, asegúrese de [crear roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/create-roles) primero.

<div id="dashboard">
  ## Dashboard
</div>

Hay dos formas de asignar un rol a un usuario. Puedes seleccionar un usuario de la lista **Users** y luego asignarle un rol, o ir a la página **User Details** (perfil del usuario) de un usuario concreto y elegir un rol para asignarlo en la pestaña **Roles**.

Si tienes más de 50 roles, en la pestaña **Roles** solo se muestran los primeros 50. Los demás se pueden asignar mediante la Management API.

<div id="assign-roles-in-user-list">
  ### Asignar roles en la lista de usuarios
</div>

1. Ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users).
2. Haz clic en `...` junto al usuario que quieres modificar y selecciona **Assign Roles**.
3. Elige los roles que quieras asignar y haz clic en **Assign**.

<div id="assign-roles-in-user-profile">
  ### Asignar roles en el perfil del usuario
</div>

También puedes asignar roles a los usuarios desde la página de perfil individual de cada uno.

1. Ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) y haz clic en el nombre del usuario.
2. Haz clic en la vista **Roles** y luego en **Assign Role**.
3. Elige el rol que quieras asignar y haz clic en **Assign.**

<div id="management-api">
  ## Management API
</div>

Haz una solicitud `POST` al [endpoint Assign User Roles](https://auth0.com/docs/api/management/v2#!/Users/post_user_roles). Asegúrate de reemplazar los valores de marcador de posición `USER_ID`, `MGMT_API_ACCESS_TOKEN` y `ROLE_ID` por tu ID de usuario, el token de acceso de la Management API <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> y los ID de rol, respectivamente.

| Valor                   | Descripción                                                                                                                          |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| `USER_ID`               | El ID del usuario que se actualizará.                                                                                                |
| `MGMT_API_ACCESS_TOKEN` | [Access Token for the Management API](https://auth0.com/docs/api/management/v2/tokens) con los scopes `read:roles` y `update:users`. |
| `ROLE_ID`               | ID(s) del/de los rol(es) que desea agregar al usuario especificado.                                                                  |
