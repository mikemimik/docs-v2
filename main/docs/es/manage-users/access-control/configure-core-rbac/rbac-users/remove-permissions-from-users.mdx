Puede eliminar los permisos asignados directamente a un usuario desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Los permisos asignados se usan con el conjunto de funcionalidades Authorization Core de la API. Para obtener más información, lea [Control de acceso basado en roles](/docs/es/manage-users/access-control/rbac).

<div id="prerequisites">
  ## Requisitos previos
</div>

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debes habilitarlo para tu API mediante el Dashboard o la Management API. La funcionalidad de Authorization Core es distinta de la de Authorization Extension. Para compararlas, lee [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="use-the-dashboard">
  ## Usa el Dashboard
</div>

1. Ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) y haz clic en el nombre del usuario.
2. En la vista **Permissions**, haz clic en el icono de la papelera junto al permiso que quieras eliminar y confirma.

   <Frame>![Lista de permisos asignados a este usuario en los detalles del usuario del Auth0 Dashboard](/docs/images/cdy7uua7fh8z/4vShfxoUcSpCDnM5FnAbsR/de85942623430e3cedc097addbaa2f9d/User_Details_-_Permissions_-_English.png)</Frame>

<div id="use-the-management-api">
  ## Usa la Management API
</div>

Haz una llamada `DELETE` al [endpoint Delete User Permissions](https://auth0.com/docs/api/management/v2#!/Users/delete_permissions). Asegúrate de reemplazar los valores de marcador de posición `USER_ID`, `MGMT_API_ACCESS_TOKEN`, `API_ID` y `PERMISSION_NAME` por tu user ID, el <Tooltip tip="Access Token: credencial de Authorization, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">access token</Tooltip> de la Management API, los ID de las API y los nombres de los permisos, respectivamente.

</AuthCodeGroup>

| Valor                   | Descripción                                                                                                         |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `USER_ID`               | El ID del usuario que se actualizará.                                                                               |
| `MGMT_API_ACCESS_TOKEN` | [Access Token for the Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `update:users`. |
| `API_ID`                | ID(s) de la(s) API(s) asociada(s) al/a los permiso(s) que deseas eliminar del usuario especificado.                 |
| `PERMISSION_NAME`       | Nombre(s) del/de los permiso(s) que deseas eliminar del usuario especificado.                                       |
