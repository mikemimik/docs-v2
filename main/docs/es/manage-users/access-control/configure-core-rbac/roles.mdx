<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El conjunto de funcionalidades [Authorization Core](/docs/es/manage-users/access-control/rbac) y la [Authorization Extension](/docs/es/customize/extensions/authorization-extension) son funcionalidades totalmente independientes. Para administrar grupos, roles o permisos, debes usar la funcionalidad en la que se crearon originalmente.
</Callout>

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debes habilitarlo para tu API mediante el Dashboard o la <Tooltip tip="Management API: Un producto para permitir a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. La funcionalidad Authorization Core es diferente de la Authorization Extension. Para ver una comparación, consulta [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).

Ofrecemos varias funciones para ayudarte a administrar tus roles, a las que puedes acceder desde el <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la Auth0 Management API:

* [Crear roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/create-roles)
* [Editar definiciones de roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/edit-role-definitions)
* [Agregar permisos a los roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/add-permissions-to-roles)
* [Eliminar permisos de los roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/remove-permissions-from-roles)
* [Ver permisos de los roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Ver usuarios asignados a los roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/view-users-assigned-to-roles)
* [Eliminar roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/delete-roles)
