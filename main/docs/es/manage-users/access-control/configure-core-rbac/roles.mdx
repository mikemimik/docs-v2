<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  [Authorization Core](/docs/es/manage-users/access-control/rbac) y [Authorization Extension](/docs/es/customize/extensions/authorization-extension) son funcionalidades completamente independientes. Para administrar grupos, roles o permisos, debe usar la funcionalidad en la que se crearon originalmente.
</Callout>

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debe habilitarlo para su API mediante el Dashboard o la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. La funcionalidad Authorization Core es diferente de Authorization Extension. Para ver una comparación, consulte [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).

Ofrecemos varias funciones para ayudarle a administrar sus roles, a las que puede acceder a través del <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o de la Auth0 Management API:

* [Crear roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/create-roles)
* [Editar definiciones de roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/edit-role-definitions)
* [Agregar permisos a los roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/add-permissions-to-roles)
* [Eliminar permisos de los roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/remove-permissions-from-roles)
* [Ver permisos del rol](/docs/es/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Ver usuarios del rol](/docs/es/manage-users/access-control/configure-core-rbac/roles/view-users-assigned-to-roles)
* [Eliminar roles](/docs/es/manage-users/access-control/configure-core-rbac/roles/delete-roles)
