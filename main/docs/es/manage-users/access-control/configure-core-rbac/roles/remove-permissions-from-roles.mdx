Puedes eliminar los [permisos](/docs/es/manage-users/access-control/rbac) asignados a un rol desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Los permisos y roles asignados se utilizan con el conjunto de funcionalidades Authorization Core de la API.

<div id="prerequisite">
  ## Requisito previo
</div>

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debe habilitarlo para su API a través del Dashboard o de la Management API. La funcionalidad de Authorization Core es diferente de la de Authorization Extension. Para ver una comparación, lea [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Ve a [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles) y haz clic en el nombre del rol que quieres ver.
2. Haz clic en **Permissions** y, a continuación, en el icono de la papelera junto al permiso que quieras eliminar; luego, confirma.

<div id="management-api">
  ## Management API
</div>

Haz una llamada `DELETE` al [endpoint Delete Role Permissions](https://auth0.com/docs/api/management/v2#!/Roles/delete_role_permission_assignment). Asegúrate de reemplazar los valores de marcador de posición `ROLE_ID`, `MGMT_API_ACCESS_TOKEN`, `API_ID` y `PERMISSION_NAME` por el ID de tu rol, el <Tooltip tip="Access Token: Credencial de Authorization, en forma de una cadena opaca o JWT, que se usa para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">access token</Tooltip> de la Management API, los ID de las API y los nombres de los permisos, respectivamente.

</AuthCodeGroup>

| Valor                   | Descripción                                                                                                         |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `ROLE_ID`               | El ID del rol del que desea eliminar permisos.                                                                      |
| `MGMT_API_ACCESS_TOKEN` | [Access Token for the Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `update:roles`. |
| `API_ID`                | ID de la(s) API asociada(s) al/a los permiso(s) que desea eliminar del rol especificado.                            |
| `PERMISSION_NAME`       | Nombre(s) del/de los permiso(s) que desea eliminar del rol especificado.                                            |
