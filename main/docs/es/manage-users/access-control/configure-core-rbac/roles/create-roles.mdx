Puedes crear [roles](/docs/es/manage-users/access-control/rbac) desde el Auth0 Dashboard o la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Los roles pueden usarse con el conjunto de funcionalidades Authorization Core para API.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Para que el control de acceso basado en roles (RBAC) funcione correctamente, debes habilitarlo en tu API mediante el Dashboard o la Management API. La funcionalidad de Authorization Core es diferente de la de Authorization Extension. Para compararlas, consulta [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).
* [Configura una API](/docs/es/get-started/auth0-overview/set-up-apis) en el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.
* Los permisos se seleccionan de valores predefinidos. Si tu lista de permisos está vacía, debes [agregar permisos](/docs/es/get-started/apis/add-api-permissions) a tu API.

<div id="dashboard">
  ## Dashboard
</div>

1. Ve a [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles) y haz clic en **Create Role**.
2. Ponle un nombre al rol y añade una descripción; luego, haz clic en **Create**.

<div id="management-api">
  ## Management API
</div>

Haz una llamada `POST` al [endpoint Create Role](https://auth0.com/docs/api/management/v2#!/Roles/post_roles). Asegúrate de reemplazar los valores de marcador de posición `MGMT_API_ACCESS_TOKEN`, `ROLE_NAME` y `ROLE_DESC` por tu <Tooltip tip="Access Token: Credencial de autorización, en forma de una cadena opaca o JWT, que se usa para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> de la Management API, el nombre del rol y la descripción del rol, respectivamente.

| **Valor**               | **Descripción**                                                                                                        |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `create:roles`. |
| `ROLE_NAME`             | Nombre del rol que quieres crear.                                                                                      |
| `ROLE_DESC`             | Descripción del rol fácil de entender.                                                                                 |
