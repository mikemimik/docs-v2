---
description: Describe cómo ver los usuarios asignados a un rol mediante Auth0 Dashboard o Management API.
title: Ver los usuarios asignados a roles
---

Puedes ver los usuarios asignados a un [rol](/docs/es/manage-users/access-control/rbac) mediante <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Los roles se utilizan con el conjunto de funcionalidades Authorization Core de la API.

<div id="prerequisites">
  ## Requisitos previos
</div>

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debes habilitarlo en tu API mediante el Dashboard o la Management API. La funcionalidad de Authorization Core es distinta de la de Authorization Extension. Para ver una comparación, consulta [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Ve a [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/users) y haz clic en el nombre del rol para verlo.
2. Haz clic en la pestaña **Users**.

Se muestra la siguiente información de cada usuario:

| Columna                         | Descripción                                                             |
| ------------------------------- | ----------------------------------------------------------------------- |
| Foto                            | La foto del usuario en el perfil de usuario.                            |
| Nombre                          | El nombre del usuario en el perfil de usuario.                          |
| Dirección de correo electrónico | La dirección de correo electrónico del usuario en el perfil de usuario. |

<div id="management-api">
  ## Management API
</div>

Para obtener la lista de usuarios asignados a un rol, consulta [Obtener los usuarios asignados a un rol](https://auth0.com/docs/api/management/v2#!/Roles/get_role_user).
