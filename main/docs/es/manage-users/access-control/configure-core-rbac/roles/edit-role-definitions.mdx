Puedes editar la definición de un [rol](/docs/es/manage-users/access-control/rbac) desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Los roles se usan con el conjunto de funcionalidades Authorization Core para API.

<div id="prerequisite">
  ## Requisito previo
</div>

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debes habilitarlo en tu API mediante el Dashboard o la Management API. La funcionalidad de Authorization Core es distinta de la de Authorization Extension. Para comparar ambas, consulta [Authorization Core vs. Authorization Extension](/docs/es/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Ve a [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles) y haz clic en el nombre del rol para verlo.
2. Edita el nombre y la descripción del rol, y luego haz clic en **Save**.

<div id="management-api">
  ## Management API
</div>

Realiza una llamada `PATCH` al [endpoint Update Role](https://auth0.com/docs/api/management/v2#!/Roles/patch_roles_by_id). Asegúrate de reemplazar los valores de marcador de posición `ROLE_ID`, `MGMT_API_ACCESS_TOKEN`, `ROLE_NAME` y `ROLE_DESC` por tu ID de rol, el token de acceso de la Management API <Tooltip tip="Access Token: credencial de Authorization, en forma de una cadena opaque o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">access token</Tooltip>, el nombre del rol y la descripción del rol, respectivamente.

| Valor                   | Descripción                                                                                                            |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `ROLE_ID`               | El ID del rol cuya definición quieres editar.                                                                          |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `update:roles`. |
| `ROLE_NAME`             | Nombre del rol.                                                                                                        |
| `ROLE_DESC`             | Descripción del rol fácil de entender.                                                                                 |
