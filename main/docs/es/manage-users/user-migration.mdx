Auth0 admite la importación de usuarios desde aplicaciones externas mediante conexiones de base de datos personalizadas, la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> de Auth0 o la extensión User Import/Export.

<div id="automatic-migrations">
  ## Migraciones automáticas
</div>

Auth0 admite la migración automática de usuarios desde una [conexión de base de datos personalizada](/docs/es/authenticate/database-connections/custom-db) a Auth0. Esta práctica a veces también se conoce como **migración gradual** o **migración diferida**. Al activar esta función, tus usuarios:

* Se migran a Auth0 la primera vez que inician sesión después de configurar la integración.
* No tienen que restablecer su contraseña como consecuencia de la migración.

<Card title="La disponibilidad varía según el plan de Auth0">
  Tanto la implementación específica de tu inicio de sesión como tu plan de Auth0 o tu acuerdo personalizado influyen en si esta función está disponible. Para obtener más información, consulta [Precios](https://auth0.com/pricing).
</Card>

Cuando un usuario se autentica mediante una conexión de base de datos personalizada marcada para importarse a Auth0, se lleva a cabo el siguiente proceso:

* Auth0 autentica a los usuarios migrados con la base de datos de Auth0.
* Si el usuario no ha sido migrado, Auth0 ejecuta tu script de inicio de sesión personalizado y, cuando el inicio de sesión se completa correctamente, añade al usuario a la base de datos de Auth0.
* En los inicios de sesión posteriores, las credenciales del usuario se obtienen de Auth0, **NO** de tu base de datos personalizada.
* Los usuarios nuevos se añaden automáticamente a la base de datos de Auth0.

<div id="bulk-user-imports-with-the-management-api">
  ## Importaciones masivas de usuarios con la Management API
</div>

Si ya tienes una base de datos de usuarios, puedes usar nuestro endpoint de la Management API [`/post_users_imports`](https://auth0.com/docs/api/management/v2#!/Jobs/post_users_imports) para cargar esta información en una conexión de base de datos.

<div id="migrate-users-with-the-user-importexport-feature">
  ## Migrar usuarios con la función User Import/Export
</div>

La función User Import/Export permite:

* Importar de forma masiva sus usuarios existentes de base de datos a Auth0.
* Buscar y exportar algunos (o todos) sus usuarios de base de datos de Auth0.

Para usar la importación masiva, debe ser Dashboard Editor - Users o un Admin. Para usar la exportación masiva, debe ser Dashboard Viewer - Users, Editor - Users o un Admin.

Puede importar y exportar datos de usuario con la [función User Import/Export](/docs/es/manage-users/user-migration/bulk-user-import-export) disponible en el Dashboard.
