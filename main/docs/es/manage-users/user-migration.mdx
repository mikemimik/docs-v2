Auth0 admite la importación de usuarios desde aplicaciones externas mediante conexiones de base de datos personalizadas, la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> de Auth0 o la extensión User Import/Export.

<div id="automatic-migrations">
  ## Migraciones automáticas
</div>

Auth0 admite la migración automática de usuarios desde una [conexión de base de datos personalizada](/docs/es/authenticate/database-connections/custom-db) a Auth0. A esta práctica a veces se la conoce como **migración gradual** o **migración diferida**. Al activar esta funcionalidad, tus usuarios:

* Se migran a Auth0 la primera vez que inician sesión después de que configuras la integración.
* No necesitan restablecer su contraseña como resultado de la migración.

<Card title="La disponibilidad varía según el plan de Auth0">
  Tanto tu implementación específica de inicio de sesión como tu plan de Auth0 o acuerdo personalizado determinan si esta funcionalidad está disponible. Para obtener más información, consulta [Pricing](https://auth0.com/pricing).
</Card>

Cuando un usuario se autentica mediante una conexión de base de datos personalizada marcada para importarse a Auth0, se lleva a cabo el siguiente proceso:

* Auth0 autentica a los usuarios migrados con la base de datos de Auth0.
* Si el usuario no ha sido migrado, Auth0 ejecuta tu script personalizado de inicio de sesión y, si el inicio de sesión se realiza correctamente, agrega al usuario a la base de datos de Auth0.
* En los inicios de sesión posteriores, las credenciales del usuario se recuperan de Auth0, **NO** de tu base de datos personalizada.
* Los usuarios nuevos se agregan automáticamente a la base de datos de Auth0.

<div id="bulk-user-imports-with-the-management-api">
  ## Importaciones masivas de usuarios con la Management API de Auth0
</div>

Si ya tiene una base de datos de usuarios, puede usar nuestro endpoint de la Management API de Auth0 [`/post_users_imports`](https://auth0.com/docs/api/management/v2#!/Jobs/post_users_imports) para cargar esta información en una conexión de base de datos.

<div id="migrate-users-with-the-user-importexport-feature">
  ## Migrar usuarios con la funcionalidad User Import/Export
</div>

La funcionalidad User Import/Export te permite:

* Importar en bloque tus usuarios de base de datos existentes a Auth0.
* Buscar y exportar algunos (o todos) de tus usuarios de base de datos de Auth0.

Para usar la importación en bloque, debes ser Dashboard Editor - Users o un Admin. Para usar la exportación en bloque, debes ser Dashboard Viewer - Users, Editor - Users o un Admin.

Puedes importar y exportar datos de usuario con la [funcionalidad User Import/Export](/docs/es/manage-users/user-migration/bulk-user-import-export) disponible en el Dashboard.
