---
description: Correcciones que debe realizar en su aplicación de Auth0 antes del despliegue a producción
title: 'Verificaciones de preparación para producción: correcciones críticas'
---

Las siguientes comprobaciones verifican si ha completado todos los requisitos para desplegar correctamente en Producción.

Para obtener más información sobre el resultado de sus comprobaciones, consulte [Ejecutar las comprobaciones de producción](/docs/es/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Verificación                                                                                                                                                                                          | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Todas las Actions se ejecutan en una [versión recomendada de NodeJS](/docs/es/troubleshoot/product-lifecycle/past-migrations/migrate-nodejs-16-to-nodejs-18)                                             | Actualiza todas las [Actions](/docs/es/troubleshoot/product-lifecycle/past-migrations/migrate-nodejs-16-to-nodejs-18) a una versión recomendada de NodeJS para aprovechar los parches de seguridad y las nuevas funcionalidades.                                                                                                                                                                                                                                                                                                                      |
| [Permitir tokens de ID para la autenticación de Management API v2](/docs/es/troubleshoot/product-lifecycle/past-migrations/migrate-to-calling-api-with-access-tokens) está desactivada                   | La compatibilidad con el uso de token de ID para autorizar algunos de los endpoints de Users y Device Credentials de la Management API está quedando obsoleta. Después de completar la migración a Access Tokens, asegúrate de que el interruptor [Allow ID Tokens for Management API v2 Authentication](https://manage.auth0.com/#/tenant/advanced) esté desactivado. Si no puedes ver esta configuración, entonces tu tenant se creó después de que esta funcionalidad quedara obsoleta, por lo que ya está desactivada de forma predeterminada. |
| allowLegacyRoGrantTypes                                                                                                                                                                               | El [endpoint oauth/ro heredado](/docs/es/troubleshoot/product-lifecycle/past-migrations/migration-oauthro-oauthtoken) en la configuración de migración del tenant quedará obsoleto. Después de actualizar tus aplicaciones para que usen endpoints compatibles, asegúrate de desactivar este endpoint.                                                                                                                                                                                                                                                |
| allowOtherLegacyGrantTypes                                                                                                                                                                            | El [endpoint oauth/access&#95;token heredado](https://auth0.com/docs/api/authentication/login-legacy/social-with-providers-access-token) de la configuración de migración del tenant está quedando obsoleto. Después de actualizar tus aplicaciones para que usen endpoints compatibles, asegúrate de desactivar este endpoint.                                                                                                                                                                                                                    |
| allowLegacyTokenInfoEndpoint                                                                                                                                                                          | El [endpoint tokeninfo](/docs/es/troubleshoot/product-lifecycle/past-migrations#tokeninfo-endpoint-validation) heredado en la configuración de migración del tenant está quedando obsoleto. Después de actualizar tus aplicaciones para usar endpoints admitidos, asegúrate de desactivar este endpoint.                                                                                                                                                                                                                                              |
| allowLegacyDelegationGrantTypes                                                                                                                                                                       | El [endpoint de delegación](/docs/es/authenticate/login/oidc-conformant-authentication/oidc-adoption-delegation) heredado en la configuración de migración del tenant dejará de usarse. Después de actualizar tus aplicaciones para que usen endpoints compatibles, asegúrate de desactivar este endpoint.                                                                                                                                                                                                                                            |
| [Allowed Callback URLs](/docs/es/authenticate/login/redirect-users-after-login) no deben incluir Localhost                                                                                               | Valida que las [URL de devolución de llamada permitidas de la aplicación](https://manage.auth0.com/#/applications) no apunten a `localhost`, `127.0.0.1`, etc.                                                                                                                                                                                                                                                                                                                                                                                     |
| [Orígenes permitidos (CORS)](/docs/es/authenticate/login/cross-origin-authentication) no debería ser Localhost                                                                                           | Valida que la [URL de la página](https://manage.auth0.com/#/applications) no apunte a `localhost`.                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [Allowed Web Origins](/docs/es/get-started/applications/application-settings) no debe incluir `localhost`                                                                                                | Comprueba que las [URL de Allowed Web Origins](https://manage.auth0.com/#/applications) no apunten a `localhost`.                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [Proveedor de correo electrónico](/docs/es/customize/email/smtp-email-providers) debe configurarse                                                                                                       | Verifica que se haya configurado el [proveedor de correo electrónico personalizado](https://manage.auth0.com/#/emails/provider).                                                                                                                                                                                                                                                                                                                                                                                                                   |
| Hooks están en proceso de quedar obsoletos y deben [migrarse a Actions](/docs/es/customize/actions/migrate/migrate-from-hooks-to-actions)                                                                | Los Hooks quedarán obsoletos. Le recomendamos encarecidamente que empiece a usar [Actions](/docs/es/customize/actions/migrate/migrate-from-hooks-to-actions) e inicie el proceso de migración. Los Hooks existentes seguirán activos hasta el 18 de noviembre de 2024.                                                                                                                                                                                                                                                                                |
| [Autenticación multifactor](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-sms-voice-notifications-mfa) debe configurarse (Dependencia: MFA está configurado) | Si usa MFA con el [factor Phone Message](https://manage.auth0.com/#/mfa), configúrelo con Twilio o con un proveedor de envío personalizado para evitar los límites de frecuencia del proveedor de envío predeterminado.                                                                                                                                                                                                                                                                                                                            |
| [Perfil de usuario heredado](/docs/es/troubleshoot/product-lifecycle/past-migrations/migrate-from-legacy-auth-flows#user-profiles) debe estar desactivado                                                | Los flujos de autenticación heredados que permiten que los tokens de ID y el endpoint `/userinfo` incluyan el perfil de usuario completo se están retirando. Después de completar la migración a las nuevas API conformes con OIDC, asegúrate de que la opción [Legacy User Profile](https://manage.auth0.com/#/account/advanced) esté desactivada.                                                                                                                                                                                                |
| Rules quedará obsoleto y debe [migrarse a Actions](/docs/es/customize/actions/migrate/migrate-from-rules-to-actions)                                                                                     | Rules está quedando obsoleto. Le recomendamos encarecidamente que empiece a usar [Actions](/docs/es/customize/actions/migrate/migrate-from-rules-to-actions) e inicie el proceso de migración. Las Rules existentes seguirán activas hasta el 18 de noviembre de 2024.                                                                                                                                                                                                                                                                                |
| [Conexiones sociales](/docs/es/authenticate/identity-providers/social-identity-providers/devkeys) no deben usar las claves de desarrollador de Auth0                                                     | Verifica que las [Conexiones Sociales](https://manage.auth0.com/#/connections/social) no utilicen las claves de desarrollador predeterminadas de Auth0.                                                                                                                                                                                                                                                                                                                                                                                            |
| Se debe configurar Support Email                                                                                                                                                                      | Asegura que el [Support Email](https://manage.auth0.com/#/account) esté configurado en Configuración del tenant.                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| Support URL está configurada                                                                                                                                                                          | Asegura que la [Support URL](https://manage.auth0.com/#/account) esté configurada en Configuración del tenant.                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Se debe configurar la etiqueta de entorno del tenant                                                                                                                                                  | Garantiza que la [etiqueta de entorno del tenant](https://support.auth0.com/tenants/public) esté configurada correctamente como Production, Staging o Development. Los cambios en esta etiqueta afectan al límite de frecuencia de tu tenant.                                                                                                                                                                                                                                                                                                      |
| El tenant está configurado para usar la [versión predeterminada recomendada de NodeJS](/docs/es/troubleshoot/product-lifecycle/deprecations-and-migrations/migrate-nodejs-22)                            | Actualiza tu tenant a una [versión recomendada de NodeJS](/docs/es/troubleshoot/product-lifecycle/past-migrations/migrate-nodejs-16-to-nodejs-18) para aprovechar las actualizaciones de seguridad y las nuevas funcionalidades.                                                                                                                                                                                                                                                                                                                      |
| URI de inicio de sesión del tenant                                                                                                                                                                    | Garantiza que la [URI de inicio de sesión del tenant](https://manage.auth0.com/#/tenant/advanced) esté configurada en Configuración del tenant.                                                                                                                                                                                                                                                                                                                                                                                                    |
| Usa el dominio personalizado en las plantillas de correo electrónico de tu marca                                                                                                                      | Asegúrese de que sus plantillas de correo electrónico utilicen un [dominio personalizado](/docs/es/customize/custom-domains/configure-features-to-use-custom-domains#use-custom-domains-in-emails). Recomendamos encarecidamente personalizar todos los correos electrónicos dirigidos a los usuarios para que utilicen su dominio personalizado y la imagen de marca de su empresa.                                                                                                                                                                  |
