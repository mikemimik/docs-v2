description: Verificaciones para garantizar que sus Applications cumplan con las prácticas recomendadas de Auth0
title: 'Verificación de preparación para producción: Prácticas recomendadas'
---

Las siguientes verificaciones no pueden automatizarse, por lo que recomendamos revisar manualmente estas áreas antes de la implementación en Production.

| Verificación                                                                                                            | Descripción                                                                                                                                                                                                                                                                                                          |
| ----------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Externalizar [parámetros de configuración](/docs/es/authenticate/database-connections/custom-db)                           | [Externalice todos los parámetros de configuración, en lugar de codificarlos de forma rígida](https://manage.auth0.com/#/connections/database), como credenciales, cadenas de conexión, claves API, etc., al desarrollar Rules, Hooks o conexiones de base de datos personalizadas.                                  |
| Revisar los valores de tiempo de espera de [inicio de sesión único (SSO)](/docs/es/glossary?term=Single+Sign-on+%28SSO%29) | Revise los [valores predeterminados de tiempo de espera de la cookie de SSO](https://manage.auth0.com/#/account/advanced) y asegúrese de que se ajusten a sus requisitos.                                                                                                                                            |
| Tenants y administradores                                                                                               | Revise todos los tenants y los administradores del tenant para asegurarse de que sean los correctos. Dé de baja los tenants que ya no estén en uso. Asegúrese de que los administradores del tenant se limiten a los usuarios necesarios.                                                                            |
| Verificar los ID de cliente en el código de la aplicación                                                               | Asegúrese de que los ID de cliente en el código de su aplicación coincidan con la configuración de su aplicación de Auth0.                                                                                                                                                                                           |
| Agregar las IP públicas de Auth0 a la lista de permitidos                                                               | Incluya las IP de Auth0 en la lista de permitidos si se está conectando a servicios internos o a servicios detrás de un firewall al usar Rules, Hooks o bases de datos personalizadas. Puede consultar una lista de direcciones IP en la información sobre herramientas al configurar cualquiera de estos elementos. |
| Revisar la protección contra ataques                                                                                    | Se recomienda que [revise la funcionalidad y la configuración de protección de Auth0](/docs/es/secure/attack-protection).                                                                                                                                                                                               |
