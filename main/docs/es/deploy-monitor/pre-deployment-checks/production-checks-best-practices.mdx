description: Comprobaciones para garantizar que sus Aplicaciones cumplan con las prácticas recomendadas de Auth0
title: 'Verificación de preparación para producción: prácticas recomendadas'
---

Las siguientes comprobaciones no pueden automatizarse, por lo que recomendamos revisar manualmente estas áreas antes del despliegue en Production.

| Verificación                                                                                                           | Descripción                                                                                                                                                                                                                                                                                        |
| ---------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Externalice los [parámetros de configuración](/docs/es/authenticate/database-connections/custom-db)                       | [Externalice, en lugar de codificar de forma fija, todos los parámetros de configuración](https://manage.auth0.com/#/connections/database), como credenciales, cadenas de conexión, claves de API, etc., al desarrollar Rules, Hooks o conexiones de base de datos personalizadas.                 |
| Revise los valores de tiempo de espera de [inicio de sesión único (SSO)](/docs/es/glossary?term=Single+Sign-on+%28SSO%29) | Revise los [valores predeterminados de tiempo de espera de la cookie de SSO](https://manage.auth0.com/#/account/advanced) y asegúrese de que se ajusten a sus requisitos.                                                                                                                          |
| Tenants y administradores                                                                                              | Revise todos los tenants y sus administradores para asegurarse de que sean correctos. Retire los tenants que ya no se utilicen. Asegúrese de que los administradores del tenant se limiten a los usuarios necesarios.                                                                              |
| Verifique los ID de cliente en el código de la aplicación                                                              | Asegúrese de que los ID de cliente en el código de su aplicación coincidan con la configuración de su aplicación de Auth0.                                                                                                                                                                         |
| Agregue las IP públicas de Auth0 a la lista de permitidos                                                              | Agregue las IP de Auth0 a la lista de permitidos si se está conectando a servicios internos o a servicios detrás de un firewall al usar Rules, Hooks o bases de datos personalizadas. Puede obtener una lista de direcciones IP en la ayuda emergente al configurar cualquiera de estos elementos. |
| Revise la protección contra ataques                                                                                    | Se recomienda [revisar la funcionalidad y la configuración de protección contra ataques de Auth0](/docs/es/secure/attack-protection).                                                                                                                                                                 |
