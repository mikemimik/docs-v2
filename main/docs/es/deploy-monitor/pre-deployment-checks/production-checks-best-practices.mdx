title: 'Verificación de preparación para producción: Prácticas recomendadas'
---

Las siguientes verificaciones no pueden automatizarse, por lo que recomendamos revisar manualmente estas áreas antes del despliegue en Production.

| Verificación                                                                                                   | Descripción                                                                                                                                                                                                                                                                                                |
| -------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Externalizar los [parámetros de configuración](/docs/es/authenticate/database-connections/custom-db)              | [Externalice, en lugar de codificar de forma fija, todos los parámetros de configuración](https://manage.auth0.com/#/connections/database), como credenciales, cadenas de conexión, clave API, etc., al desarrollar Rules, Hooks o conexiones de base de datos personalizadas.                             |
| Revisar los valores de timeout de [inicio de sesión único (SSO)](/docs/es/glossary?term=Single+Sign-on+%28SSO%29) | Revise los valores predeterminados de timeout de la [cookie de SSO](https://manage.auth0.com/#/account/advanced) y asegúrese de que se ajusten a sus requisitos.                                                                                                                                           |
| Tenants y administradores                                                                                      | Revise todos los tenants y tenant administrators para asegurarse de que sean correctos. Retire los tenants que ya no estén en uso. Asegúrese de que los tenant administrators se limiten a los usuarios necesarios.                                                                                        |
| Verificar los IDs de cliente en el código de la aplicación                                                     | Asegúrese de que los IDs de cliente en el código de su aplicación coincidan con la configuración de su aplicación de Auth0.                                                                                                                                                                                |
| Agregar las IP públicas de Auth0 a la lista de permitidos                                                      | Incluya las IP de Auth0 en la lista de permitidos si se conecta a servicios internos o a servicios detrás de un firewall al usar Rules, Hooks o bases de datos personalizadas. Puede obtener una lista de direcciones IP en la información sobre herramientas al configurar cualquiera de estos elementos. |
| Revisar la protección contra ataques                                                                           | Se recomienda que [revise la funcionalidad y la configuración de protección contra ataques de Auth0](/docs/es/secure/attack-protection).                                                                                                                                                                      |
