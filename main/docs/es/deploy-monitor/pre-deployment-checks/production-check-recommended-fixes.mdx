---
description: Recomendaciones sobre cómo puede mejorar su aplicación de Auth0 antes del despliegue en producción
title: 'Verificaciones de preparación para producción: correcciones no críticas'
---

Las siguientes verificaciones comprueban si ha completado todas las recomendaciones (opcionales) para un despliegue exitoso en Producción.

Para obtener más información sobre los resultados de sus verificaciones, consulte [Ejecutar verificaciones de producción](/docs/es/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Verificación                                                                                                                                   | Descripción                                                                                                                                                                                                                                                                                                                                                         |
| ---------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configurar la transmisión de logs                                                                                                              | Asegúrese de transmitir las entradas de registro del tenant a una ubicación de almacenamiento o a un proveedor externos. Recomendamos que [configure un servicio de transmisión de logs](/docs/es/customize/log-streams) para conservar los logs durante más de 30 días y facilitar un mejor análisis y alertas.                                                       |
| [Dominio personalizado](/docs/es/customize/custom-domains) debe estar configurado                                                                 | Use [dominios personalizados con Universal Login](https://manage.auth0.com/#/tenant/custom_domains) para ofrecer la experiencia más fluida y segura a sus usuarios finales.                                                                                                                                                                                         |
| [Página de error personalizada](/docs/es/customize/login-pages/custom-error-pages) debe estar configurada                                         | [Configure una página de error personalizada](https://manage.auth0.com/#/account) con los detalles específicos de su aplicación y la identidad visual corporativa.                                                                                                                                                                                                  |
| Habilitar Universal Login                                                                                                                      | Asegúrese de usar la versión más reciente de [Universal Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login). Recomendamos actualizar su página de inicio de sesión de la experiencia Classic a Universal Login para beneficiarse de una mejor accesibilidad y rendimiento, personalización sin código y nuevas funcionalidades. |
| [Plantillas de correo electrónico](/docs/es/customize/email/manage-email-flow) deben estar configuradas                                           | [Configure plantillas de correo electrónico personalizadas](https://manage.auth0.com/#/emails) con los detalles específicos de su aplicación y la identidad visual corporativa.                                                                                                                                                                                     |
| [Autenticación multifactor](/docs/es/secure/multi-factor-authentication) debe estar configurada                                                   | Se recomienda considerar la autenticación multifactor como una práctica segura para autenticar a sus usuarios.                                                                                                                                                                                                                                                      |
| [MFA para los administradores del tenant](/docs/es/get-started/manage-dashboard-access) está habilitado                                           | [Habilite la autenticación multifactor](https://manage.auth0.com/#/account/admins) para los administradores del tenant.                                                                                                                                                                                                                                             |
| Debe definirse la [URL de redirección para cierre de sesión](/docs/es/authenticate/login/logout#set-the-allowed-logout-urls-at-the-account-level) | Revise las [URL de redirección para cierre de sesión permitidas](https://manage.auth0.com/#/account/advanced) de su aplicación.                                                                                                                                                                                                                                     |
| Las aplicaciones deben usar RS256 en lugar de HS256 para el algoritmo de firma de JSON Web Token (JWT)                                         | Establezca el [algoritmo de firma](/docs/es/get-started/apis#signing-algorithms) de JSON Web Token (JWT) en RS256 en lugar de HS256 para cada aplicación.                                                                                                                                                                                                              |
| Establecer [Application Login URI](/docs/es/get-started/applications/application-settings#application-uris)                                       | Indique un [Application Login URI](/docs/es/get-started/applications/application-settings#application-uris) para todas las aplicaciones interactivas.                                                                                                                                                                                                                  |
| Establecer [Tenant Allowed Logout URL](/docs/es/get-started/applications/application-settings#application-uris)                                   | Cree al menos una ruta de cierre de sesión de acceso público y [actualice la configuración del tenant](/docs/es/get-started/applications/application-settings#application-uris) para que apunte a ella.                                                                                                                                                                |
| Establecer [Tenant Login URI](/docs/es/get-started/tenant-settings#login-and-logout)                                                              | Publique una ruta de inicio de sesión predeterminada y [actualice la configuración de su tenant](/docs/es/get-started/tenant-settings#login-and-logout) para que apunte a ella.                                                                                                                                                                                        |
