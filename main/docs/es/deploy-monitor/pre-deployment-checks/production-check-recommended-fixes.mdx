---
description: Recomendaciones sobre cómo puedes mejorar tu aplicación de Auth0 antes del despliegue en producción
title: 'Verificaciones de preparación para producción: correcciones no críticas'
---

Las siguientes verificaciones comprueban si has completado todas las recomendaciones (opcionales) para realizar un despliegue correcto en producción.

Para obtener más información sobre el resultado de tus verificaciones, consulta [Ejecutar verificaciones de producción](/docs/es/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Verificación                                                                                                                                  | Descripción                                                                                                                                                                                                                                                                                                                                                      |
| --------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configurar la transmisión de logs                                                                                                             | Asegúrate de transmitir las entradas de registro del tenant a una ubicación de almacenamiento externo o a un proveedor externo. Te recomendamos [configurar un servicio de transmisión de logs](/docs/es/customize/log-streams) para conservar los logs durante más de 30 días y mejorar el análisis y las alertas.                                                 |
| Se debe configurar el [Custom Domain](/docs/es/customize/custom-domains)                                                                         | Usa [custom domains con Universal Login](https://manage.auth0.com/#/tenant/custom_domains) para ofrecer a tus usuarios finales una experiencia más fluida y segura.                                                                                                                                                                                              |
| Se debe configurar la [página de error personalizada](/docs/es/customize/login-pages/custom-error-pages)                                         | [Configura una página de error personalizada](https://manage.auth0.com/#/account) con los detalles específicos de tu aplicación y la identidad visual de tu empresa.                                                                                                                                                                                             |
| Habilitar Universal Login                                                                                                                     | Asegúrate de usar la versión más reciente de [Universal Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login). Te recomendamos actualizar tu página de inicio de sesión de la experiencia Classic a Universal Login para beneficiarte de una mejor accesibilidad y rendimiento, personalización sin código y nuevas funciones. |
| Se deben configurar las [plantillas de correo electrónico](/docs/es/customize/email/manage-email-flow)                                           | [Configura plantillas de correo electrónico personalizadas](https://manage.auth0.com/#/emails) con los detalles específicos de tu aplicación y la identidad visual de tu empresa.                                                                                                                                                                                |
| Se debe configurar la [autenticación multifactor](/docs/es/secure/multi-factor-authentication)                                                   | Te recomendamos considerar la autenticación multifactor como una práctica segura para autenticar a tus usuarios.                                                                                                                                                                                                                                                 |
| La [MFA para los administradores del tenant](/docs/es/get-started/manage-dashboard-access) está habilitada                                       | [Habilita la autenticación multifactor](https://manage.auth0.com/#/account/admins) para los administradores del tenant.                                                                                                                                                                                                                                          |
| Se debe definir la [URL de redirección de cierre de sesión](/docs/es/authenticate/login/logout#set-the-allowed-logout-urls-at-the-account-level) | Revisa las [Allowed Redirect Logout URLs](https://manage.auth0.com/#/account/advanced) de tu aplicación.                                                                                                                                                                                                                                                         |
| Las aplicaciones deben usar RS256 en lugar de HS256 para el algoritmo de firma de JSON Web Token (JWT)                                        | Configura el [algoritmo de firma](/docs/es/get-started/apis#signing-algorithms) de JSON Web Token (JWT) como RS256 en lugar de HS256 para cada aplicación.                                                                                                                                                                                                          |
| Establece el [Application Login URI](/docs/es/get-started/applications/application-settings#application-uris)                                    | Indica un [Application Login URI](/docs/es/get-started/applications/application-settings#application-uris) para todas las Applications interactivas.                                                                                                                                                                                                                |
| Establece la [URL de cierre de sesión permitida del tenant](/docs/es/get-started/applications/application-settings#application-uris)             | Crea al menos una ruta de cierre de sesión de acceso público y [actualiza la configuración del tenant](/docs/es/get-started/applications/application-settings#application-uris) para que apunte allí.                                                                                                                                                               |
| Establece el [Tenant Login URI](/docs/es/get-started/tenant-settings#login-and-logout)                                                           | Publica una ruta de inicio de sesión predeterminada y [actualiza la configuración de tu tenant](/docs/es/get-started/tenant-settings#login-and-logout) para que apunte allí.                                                                                                                                                                                        |
