Auth0 CLI es una herramienta de línea de comandos que proporciona acceso directo a la Auth0 Management API, lo que te permite administrar tu tenant de Auth0 desde el terminal. Está diseñada para usarse de forma interactiva durante el desarrollo y puede integrarse en secuencias de comandos para automatizar tareas.

<Callout>
  Para consultar la referencia completa de comandos de Auth0 CLI e información detallada sobre su uso, lee la [documentación de referencia de Auth0 CLI](https://auth0.github.io/auth0-cli/).
</Callout>

<div id="key-features">
  ## Características clave
</div>

* **Gestión interactiva**: Crea, actualiza y gestiona recursos de Auth0 desde la línea de comandos
* **Compatibilidad con la autenticación**: Varios métodos de autenticación, incluido el inicio de sesión de usuarios y las credenciales de máquina a máquina
* **Gestión de recursos**: Operaciones CRUD completas para aplicaciones, API, conexiones, usuarios y más
* **Multiplataforma**: Disponible para macOS, Linux y Windows
* **Salida en JSON**: Salida legible por máquinas para scripts y automatización

<div id="use-cases">
  ## Casos de uso
</div>

Auth0 CLI es ideal para:

* **Flujos de trabajo de desarrollo**: Configurar y preparar rápidamente los recursos de Auth0 durante el desarrollo
* **Pruebas**: Crear aplicaciones y usuarios de prueba para pruebas automatizadas
* **Depuración**: Inspeccionar y modificar la configuración de Auth0 en tiempo real
* **Automatización con scripts**: Automatizar tareas repetitivas con scripts de shell
* **Integración de CI/CD**: Integrar la configuración de Auth0 en tus flujos de despliegue

<div id="comparison-with-other-tools">
  ## Comparación con otras herramientas
</div>

<div id="auth0-cli-vs-deploy-cli">
  ### Auth0 CLI vs Deploy CLI
</div>

* **Auth0 CLI**: Gestión interactiva de recursos, comando a comando. Ideal para el desarrollo y tareas ad hoc.
* **[Deploy CLI](/docs/es/deploy-monitor/deploy-cli-tool)**: Gestión declarativa del tenant basada en archivos de configuración. Ideal para gestionar configuraciones completas del tenant en diferentes entornos.

<div id="auth0-cli-vs-terraform-provider">
  ### Auth0 CLI vs Terraform Provider
</div>

* **Auth0 CLI**: Comandos imperativos para aplicar cambios inmediatos. Sin gestión del estado.
* **Terraform Provider**: Infraestructura como código declarativa con seguimiento del estado. Ideal para gestionar infraestructura de producción.

<div id="install-auth0-cli">
  ## Instalar Auth0 CLI
</div>

<div id="macos">
  ### macOS
</div>

Para configurarlo con Homebrew:

```bash lines
brew tap auth0/auth0-cli
brew install auth0
```

<div id="linux">
  ### Linux
</div>

```bash lines
curl -sSfL https://raw.githubusercontent.com/auth0/auth0-cli/main/install.sh | sh -s -- -b /usr/local/bin
```

<div id="windows">
  ### Windows
</div>

Configúralo con Scoop:

```powershell lines
scoop bucket add auth0 https://github.com/auth0/scoop-auth0-cli.git
scoop install auth0
```

O descarga la versión más reciente desde la [página de versiones de GitHub](https://github.com/auth0/auth0-cli/releases).

<div id="authenticate">
  ## Autentícate
</div>

Para usar Auth0 CLI, debes autenticarte. Cuando ejecutes el siguiente comando, se abrirá una ventana del navegador para que inicies sesión en tu cuenta de Auth0:

```bash lines
auth0 login
```

Para autenticarse con una aplicación máquina a máquina, use el siguiente comando:

```bash lines
auth0 login --domain <your-domain> --client-id <client-id> --client-secret <client-secret>
```

<div id="common-commands">
  ## Comandos comunes
</div>

<div id="manage-applications">
  ### Gestionar aplicaciones
</div>

```bash lines
# Listar todas las aplicaciones
auth0 apps list

# Crear una nueva aplicación
auth0 apps create --name "My App" --type spa

# Mostrar detalles de la aplicación
auth0 apps show <app-id>

# Actualizar una aplicación
auth0 apps update <app-id> --callbacks "http://localhost:3000/callback"

# Eliminar una aplicación
auth0 apps delete <app-id>
```

<div id="manage-apis">
  ### Gestionar API
</div>

```bash lines
# Listar todas las APIs
auth0 apis list

# Crear una nueva API
auth0 apis create --name "My API" --identifier "https://my-api.example.com"

# Mostrar detalles de la API
auth0 apis show <api-id>
```

<div id="manage-users">
  ### Administrar Users
</div>

```bash lines
# Listar usuarios
auth0 users list

# Crear un usuario
auth0 users create --email "user@example.com" --connection "Username-Password-Authentication"

# Mostrar detalles del usuario
auth0 users show <user-id>

# Actualizar un usuario
auth0 users update <user-id> --email "newemail@example.com"
```

<div id="manage-tenant-settings">
  ### Gestionar la configuración del tenant
</div>

```bash lines
# Mostrar el tenant actual
auth0 tenants list

# Actualizar la configuración del tenant
auth0 tenants update
```

<div id="using-in-scripts">
  ## Uso en scripts
</div>

La Auth0 CLI admite salida en JSON para facilitar el análisis en scripts:

```bash lines
# Obtener los detalles de la aplicación en formato JSON
auth0 apps show <app-id> --json

# Crear una aplicación y capturar la salida
auth0 apps create --name "My App" --type spa --json > app-details.json
```
