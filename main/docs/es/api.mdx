<div class="api_card_wrapper">
  <Card title="API de autenticación" href="/docs/es/api/authentication" arrow />
</div>

<Columns cols={2}>
  <Column>
    La API de autenticación expone funcionalidades de identidad para Auth0 y los protocolos de identidad compatibles (incluidos OpenID Connect, OAuth y SAML).

    Normalmente, deberías usar esta API a través de uno de los SDK de Auth0, como [Auth0.js](/docs/es/libraries/auth0js), o una biblioteca como [Lock](/docs/es/libraries/lock). Sin embargo, si estás creando manualmente la interfaz de usuario para la autenticación, tendrás que llamar directamente a la API de autenticación.
  </Column>

  <Column>
    Algunos ejemplos de tareas incluyen:

    * Obtener [tokens](/docs/es/secure/tokens) durante la autenticación
    * Solicitar el perfil de un usuario mediante un [token de acceso](/docs/es/secure/tokens/access-tokens)
    * Intercambiar [tokens de actualización](/docs/es/secure/tokens/refresh-tokens) por nuevos tokens de acceso
    * Solicitar un desafío para la [autenticación multifactor (MFA)](/docs/es/secure/multi-factor-authentication)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="API de administración" href="/docs/es/api/management/v2" arrow />
</div>

<Columns cols={2}>
  <Column>
    La API de administración te permite gestionar tu cuenta de Auth0 mediante programación para que puedas automatizar la configuración de tu entorno. La mayoría de las tareas que puedes realizar en el panel de administración de Auth0 también pueden realizarse mediante programación mediante esta API.
  </Column>

  <Column>
    Algunos ejemplos de tareas incluyen:

    * Registrar tus aplicaciones y API en Auth0
    * Configurar [conexiones](/docs/es/connections) con las que tus usuarios pueden autenticarse
    * [Gestionar usuarios](/docs/es/manage-users)
    * [Vincular cuentas de usuario](/docs/es/manage-users/user-accounts/user-account-linking/link-user-accounts)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="My Account API" href="/docs/es/api/myaccount" arrow />
</div>

<Columns cols={2}>
  <Column>
    La API My Account de Auth0 proporciona un conjunto específico de endpoints para que los usuarios gestionen la información de su propia cuenta. Puedes usar estas API para crear experiencias de autoservicio en tus aplicaciones o añadir progresivamente más información a una cuenta de usuario.
  </Column>

  <Column>
    Algunos ejemplos de tareas incluyen:

    * Permitir al usuario final gestionar sus propios factores
    * Actualizar los detalles de un método de autenticación
    * Saber qué métodos de autenticación están disponibles
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="My Organization API" href="/docs/es/api/myorganization" arrow />
</div>

<Columns cols={2}>
  <Column>
    La API My Organization de Auth0 proporciona una interfaz segura con alcance de organización que permite a tus clientes empresariales gestionar sus propias organizaciones dentro de tu tenant de Auth0. Esta API sirve como base técnica para la administración delegada integrada y las integraciones con enfoque API-first. Para obtener más información, consulta [My Organzation API and Embeddable UI Components](/docs/es/manage-users/my-organization-api).
  </Column>

  <Column>
    Algunos ejemplos de tareas incluyen:

    * Detalles de la organización de Auth0 (nombre, marca y nombre para mostrar)
    * Configuración, propiedad y relaciones específicas de la organización
    * Proveedores de identidad (IdP) y aprovisionamiento SCIM
    * Configuración de dominios y Home Realm Discovery (HRD)
  </Column>
</Columns>

<div id="pagination">
  ## Paginación
</div>

En los tenants de Public Cloud, los endpoints de la API de administración devuelven un máximo de 50 resultados por respuesta. Si hay más resultados de los que caben en una sola respuesta, incluye los parámetros `page` y `per_page` en tu solicitud para obtener más resultados.

Consulta [Migrate to Management API v2 Endpoint Paginated Queries](/docs/es/troubleshoot/product-lifecycle/past-migrations/migrate-to-paginated-queries) para obtener información sobre cómo actualizar tus aplicaciones.
