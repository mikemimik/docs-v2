<div class="api_card_wrapper">
  <Card title="Authentication API" href="/docs/es/api/authentication" arrow />
</div>

<Columns cols={2}>
  <Column>
    La Authentication API expone funcionalidades de identidad para Auth0 y protocolos de identidad compatibles (incluidos OpenID Connect, OAuth y SAML).

    Normalmente, deberías usar esta API a través de uno de los SDKs de Auth0, como [Auth0.js](/docs/es/libraries/auth0js), o una biblioteca como [Lock](/docs/es/libraries/lock). Sin embargo, si estás creando manualmente tu interfaz de usuario de authentication, tendrás que llamar directamente a la Authentication API.
  </Column>

  <Column>
    Algunas tareas de ejemplo incluyen:

    * Obtener [tokens](/docs/es/secure/tokens) durante la autenticación
    * Solicitar el perfil de un usuario con un [token de acceso](/docs/es/secure/tokens/access-tokens)
    * Intercambiar [tokens de actualización](/docs/es/secure/tokens/refresh-tokens) por nuevos tokens de acceso
    * Solicitar un desafío para la [autenticación multifactor (MFA)](/docs/es/secure/multi-factor-authentication)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="Management API" href="/docs/es/api/management/v2" arrow />
</div>

<Columns cols={2}>
  <Column>
    La Management API te permite administrar tu cuenta de Auth0 de forma programática para automatizar la configuración de tu entorno. La mayoría de las tareas que puedes realizar en el Dashboard de administración de Auth0 también pueden realizarse de forma programática mediante esta API.
  </Column>

  <Column>
    Algunas tareas de ejemplo incluyen:

    * Registrar tus aplicaciones y APIs en Auth0
    * Configurar [conexiones](/docs/es/connections) con las que tus usuarios pueden autenticarse
    * [Administrar usuarios](/docs/es/manage-users)
    * [Vincular cuentas de usuario](/docs/es/manage-users/user-accounts/user-account-linking/link-user-accounts)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="My Account API" href="/docs/es/api/myaccount" arrow />
</div>

<Columns cols={2}>
  <Column>
    La My Account API de Auth0 proporciona un conjunto específico de endpoints para que los usuarios administren la información de su propia cuenta. Puedes usar estas APIs para crear experiencias de autoservicio en tus aplicaciones o añadir progresivamente información a una cuenta de usuario.
  </Column>

  <Column>
    Algunas tareas de ejemplo incluyen:

    * Permitir que el usuario final administre sus propios factores
    * Actualizar los detalles de un método de autenticación
    * Saber qué métodos de autenticación están disponibles
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="My Organization API" href="/docs/es/api/myorganization" arrow />
</div>

<Columns cols={2}>
  <Column>
    La My Organization API de Auth0 proporciona una interfaz segura con alcance de organización que permite a tus clientes empresariales administrar sus propias organizaciones dentro de tu tenant de Auth0. Esta API sirve como base técnica para la administración delegada integrada y las integraciones centradas en APIs. Para obtener más información, consulta [My Organization API y componentes de interfaz de usuario integrables](/docs/es/manage-users/my-organization-api).
  </Column>

  <Column>
    Algunas tareas de ejemplo incluyen:

    * Detalles de la organización de Auth0 (nombre, branding, nombre para mostrar)
    * Configuración, propiedad y relaciones específicas de la organización
    * Identity Providers (IdPs) y aprovisionamiento de SCIM
    * Configuración de dominios y Home Realm Discovery (HRD)
  </Column>
</Columns>

<div id="pagination">
  ## Paginación
</div>

En los tenants de Public Cloud, los endpoints de Management API devuelven un máximo de 50 resultados por respuesta. Cuando hay más resultados de los que se pueden devolver en una sola respuesta, incluye los parámetros `page` y `per_page` en tu solicitud para obtener más resultados.

Consulta [Migrar a consultas paginadas del endpoint v2 de Management API](/docs/es/troubleshoot/product-lifecycle/past-migrations/migrate-to-paginated-queries) para obtener información sobre cómo actualizar tus aplicaciones.
