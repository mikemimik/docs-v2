<div id="other-support-resources">
  ## Otros recursos de soporte
</div>

* La [página de estado de Auth0](https://status.auth0.com) contiene información sobre el estado actual del entorno de producción. Consulta la página de estado antes de enviar una solicitud de soporte.

  Si la página de estado muestra una notificación sobre una interrupción del servicio, nuestro equipo ya está trabajando para restablecerlo y la mantendrá actualizada. Publicamos un análisis de la causa raíz en la página de estado una vez que hemos investigado el problema. Puedes suscribirte para recibir notificaciones sobre cualquier cambio.

* El [Auth0 Support Center oficial](https://support.auth0.com/) ofrece información para la resolución de problemas y soluciones para muchos problemas comunes.

* La [comunidad pública de preguntas y respuestas](https://community.auth0.com/) de Auth0 ofrece soporte para todo el mundo. Puedes buscar entre las preguntas existentes y publicar nuevas si la tuya aún no se ha hecho. No se garantizan tiempos de respuesta para las preguntas publicadas.
