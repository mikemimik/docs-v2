title: Rendimiento de Private Cloud 10,000 RPS (100x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total tanto de Authentication API como de MyAccount API en un entorno suele estar soportado hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, el uso combinado intensivo de las APIs de Authentication y Management, o ciertos flujos de autenticación como ROPG en Authentication API, pueden provocar una degradación del rendimiento con menos RPS.
</Callout>

Consulte a continuación los límites de frecuencia del tipo de suscripción Private Cloud Performance 10,000 RPS (100x).

Por lo tanto, recomendamos desplegar un tenant por entorno de nube privada para mitigar riesgos.
