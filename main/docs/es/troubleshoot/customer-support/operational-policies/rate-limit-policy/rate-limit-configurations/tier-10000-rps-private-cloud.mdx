title: Rendimiento de Private Cloud 10.000 RPS (100x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele admitirse hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, un uso combinado intensivo de Authentication API y Management API, o ciertos flujos de autenticación, como ROPG en Authentication API, pueden degradar el rendimiento incluso con menos RPS.
</Callout>

Consulta a continuación los límites de frecuencia del tipo de suscripción Private Cloud Performance 10.000 RPS (100x).

Por lo tanto, recomendamos desplegar un tenant por entorno de Private Cloud para mitigar riesgos.
