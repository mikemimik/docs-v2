title: Rendimiento de Private Cloud 1500 RPS (15x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele ser compatible hasta 500 RPS; sin embargo, determinadas cargas de trabajo, un uso combinado intensivo de Authentication API y Management API, o ciertos flujos de autenticación como ROPG en Authentication API, pueden provocar una degradación del rendimiento incluso con menos RPS.
</Callout>

Consulta a continuación los límites de frecuencia del tipo de suscripción Private Cloud Performance 1500 RPS (15x). Estos límites se aplican a cada tenant que crees en el entorno de Private Cloud.

Por lo tanto, recomendamos implementar un tenant por entorno de Private Cloud para mitigar riesgos.
