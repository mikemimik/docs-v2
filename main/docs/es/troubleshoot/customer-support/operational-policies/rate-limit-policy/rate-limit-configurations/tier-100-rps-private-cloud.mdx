title: Private Cloud Basic 100 RPS (1x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento total sostenido tanto de la Authentication API como de la MyAccount API en un entorno suele estar soportado hasta 500 RPS; sin embargo, ciertas cargas de trabajo particulares, el uso combinado intensivo de las Authentication and Management APIs o algunos flujos de autenticación, como ROPG en la Authentication API, pueden provocar una degradación del rendimiento con menos RPS.
</Callout>

Consulta a continuación los límites de frecuencia del tipo de suscripción Private Cloud Basic 100 RPS (1x). Estos límites se aplican a cada tenant que crees en el entorno de Private Cloud.

Por lo tanto, recomendamos desplegar un tenant por entorno de Private Cloud para mitigar riesgos.
