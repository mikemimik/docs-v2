title: Private Cloud Performance 6000 RPS (60x) y 6000 RPS (60x) Burst
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele admitirse hasta 500 RPS; sin embargo, ciertas cargas de trabajo particulares, el uso intensivo combinado de Authentication API y Management API, o determinados flujos de autenticación como ROPG en Authentication API pueden degradar el rendimiento incluso con menos RPS.
</Callout>

Consulta a continuación los límites de frecuencia de los tipos de suscripción Private Cloud Performance 6000 RPS (60x) y 6000 RPS (60x) Burst.

Por lo tanto, recomendamos desplegar un tenant por entorno de Private Cloud para mitigar riesgos.
