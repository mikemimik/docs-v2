title: Rendimiento de Private Cloud 3000 RPS (30x) y ráfaga de 3000 RPS (30x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele ser compatible hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, un uso combinado intensivo de Authentication API y Management API, o determinados flujos de autenticación como ROPG en Authentication API pueden provocar una degradación del rendimiento con menos RPS.
</Callout>

Consulte a continuación los límites de frecuencia de los tipos de subscription Private Cloud Performance 3000 RPS (30x) y 3000 RPS (30x) Burst. Estos límites se aplican a cada tenant que cree en el entorno de Private Cloud.

Por lo tanto, recomendamos desplegar un tenant por entorno de Private Cloud para mitigar riesgos.
