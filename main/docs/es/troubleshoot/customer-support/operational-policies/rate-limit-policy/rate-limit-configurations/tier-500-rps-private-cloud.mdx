title: Rendimiento de Private Cloud 500 RPS (5x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele estar admitido hasta 500 RPS; sin embargo, ciertas cargas de trabajo particulares, el uso intensivo combinado de Authentication API y Management API, o ciertos flujos de autenticación como ROPG en Authentication API pueden provocar una degradación del rendimiento con menos RPS.
</Callout>

Consulta a continuación los límites de frecuencia del tipo de suscripción Private Cloud Performance 500 RPS (5x).

Estos límites se aplican a cada tenant que crees en el entorno de Private Cloud. Por lo tanto, recomendamos implementar un tenant por entorno de Private Cloud para mitigar riesgos.
