title: Rendimiento de Private Cloud 500 RPS (5x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total tanto de la Authentication API como de la MyAccount API en un entorno suele admitirse hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, el uso combinado intensivo de las API de Authentication and Management o ciertos flujos de autenticación, como ROPG en la Authentication API, pueden causar una degradación del rendimiento incluso con menos RPS.
</Callout>

A continuación, consulta los límites de frecuencia del tipo de suscripción Private Cloud Performance 500 RPS (5x).

Estos límites se aplican a cada tenant que crees en el entorno de Private Cloud. Por lo tanto, recomendamos desplegar un tenant por entorno de Private Cloud para mitigar riesgos.
