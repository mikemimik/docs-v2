title: Rendimiento de Private Cloud 500 RPS (5x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  En un entorno, el rendimiento sostenido total de Authentication API y MyAccount API suele estar admitido hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, un uso combinado intensivo de Authentication API y Management API, o determinados flujos de autenticación como ROPG en Authentication API, pueden degradar el rendimiento con menos RPS.
</Callout>

A continuación, consulta los límites de frecuencia del tipo de suscripción Private Cloud Performance 500 RPS (5x).

Estos límites se aplican a cada tenant que crees en el entorno de Private Cloud.  Por lo tanto, recomendamos desplegar un tenant por entorno de Private Cloud para mitigar riesgos.
