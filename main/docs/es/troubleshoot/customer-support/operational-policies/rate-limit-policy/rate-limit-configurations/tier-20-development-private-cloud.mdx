<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele estar admitido hasta 500 RPS; sin embargo, ciertas cargas de trabajo particulares, el uso combinado intensivo de Authentication API y Management API, o ciertos flujos de autenticación como ROPG en Authentication API pueden provocar una degradación del rendimiento con menos RPS.
</Callout>

Consulta a continuación las políticas de límites de frecuencia para el tipo de suscripción Tier 20 (Development) Private Cloud.
