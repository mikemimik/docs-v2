Límites de frecuencia para Authentication API y los API endpoints en el tipo de suscripción Enterprise:

| Production (3x Public Performance Burst) | 300/segundo durante 48 h/mes                                                                                                       | 100/segundo                                                                                                                        |
| Production (4x Public Performance Burst) | 400/segundo durante 48 h/mes                                                                                                       | 100/segundo                                                                                                                        |
| Non-production                           | 100/segundo                                                                                                                        | 100/segundo                                                                                                                        |

<div id="scim-api">
  ## API de SCIM
</div>

Límites de frecuencia para los endpoints de la API de SCIM entrante en suscripciones de Public Cloud que incluyen conexiones empresariales:

| Tipo de límite                                          | Ruta del endpoint                      | Operación           | Límite                      |
| ------------------------------------------------------- | -------------------------------------- | ------------------- | --------------------------- |
| Endpoint de una conexión SCIM individual                | `/scim/v2/connections/{connection-id}` | Cualquier solicitud | 25 solicitudes por segundo  |
| Límite global del tenant para todas las conexiones SCIM | `/scim/v2/connections/*`               | Cualquier solicitud | 100 solicitudes por segundo |
