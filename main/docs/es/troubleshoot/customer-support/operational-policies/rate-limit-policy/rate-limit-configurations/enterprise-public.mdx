<div id="scim-api">
  ## API de SCIM
</div>

Límites de frecuencia para los endpoints de la API de Inbound SCIM en las suscripciones de Public Cloud que incluyen conexiones empresariales:

| Tipo de límite                                          | Ruta del endpoint                      | Operación           | Límite                      |
| ------------------------------------------------------- | -------------------------------------- | ------------------- | --------------------------- |
| Endpoint de una conexión SCIM individual                | `/scim/v2/connections/{connection-id}` | Cualquier solicitud | 25 solicitudes por segundo  |
| Límite global del tenant para todas las conexiones SCIM | `/scim/v2/connections/*`               | Cualquier solicitud | 100 solicitudes por segundo |
