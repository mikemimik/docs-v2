<div id="export-data">
  ## Exportar datos
</div>

Si desea exportar sus datos de Auth0, hay varias maneras de hacerlo. Tenga en cuenta que estas herramientas no exportan los hashes de contraseña de los usuarios de base de datos alojados en Auth0. Aun así, puede solicitar esta información abriendo un [ticket de soporte](https://support.auth0.com/). Si ha decidido dejar de usar nuestro servicio, consulte [Exportar datos](/docs/es/troubleshoot/customer-support/manage-subscriptions/export-data) para obtener más información.

<div id="use-the-importexport-extension">
  ### Usa la extensión Import/Export
</div>

Puedes usar nuestra [extensión Import/Export](/docs/es/manage-users/user-migration/user-import-export-extension) para exportar casi todos los datos de tu tenant de Auth0.

<div id="use-the-management-api">
  ### Usa la Management API
</div>

Si quieres exportar determinados conjuntos de datos de forma programática, usa el endpoint Bulk User Export de la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<div id="transfer-data">
  ## Transferencia de datos
</div>

Auth0 no transferirá datos de un tenant de Auth0 a otro. Esto se aplica tanto a los clientes de Public Cloud como a los de Private Cloud.

Todos los datos de tu tenant de Auth0 están siempre bajo tu control y disponibles a través de la Management API en cualquier momento. La única información no disponible a través de la API, por motivos de seguridad, son los hashes de contraseña de tus usuarios de base de datos alojados en Auth0 y las claves privadas.
