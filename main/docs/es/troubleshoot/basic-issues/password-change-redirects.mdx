---
description: Descripción de cómo establecer una Redirect URI para el cambio de contraseña
title: Redirecciones tras el cambio de contraseña
---

La URI a la que se redirige al usuario después de un restablecimiento de contraseña correcto depende de las siguientes condiciones:

1. si tu tenant usa Universal o Classic Login, y
2. si generas un ticket de cambio de contraseña directamente mediante la Management API, o
3. si envías un correo electrónico de cambio de contraseña desde la Authentication API, la página de inicio de sesión o cualquier otra fuente.

El siguiente diagrama de flujo sirve como guía para establecer la Redirect URI de tu tenant según su configuración.
