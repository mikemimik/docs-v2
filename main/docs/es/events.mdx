<div id="use-cases">
  ## Casos de uso
</div>

Events son notificaciones en tiempo real sobre acciones específicas —inicialmente, cambios en un usuario y en organizaciones, aunque con el tiempo también incluirán otros— que han ocurrido dentro de tu tenant de Auth0. Puedes crear Event Streams que escuchen estos eventos para orquestar cambios asíncronos (es decir, cambios en una entidad o flujo que ya se hayan completado) en Auth0 y reenviarlos a uno de muchos destinos para su procesamiento externo. Events admite diversos casos de uso, entre ellos:

* Enviar correos electrónicos a nuevos clientes para darles la bienvenida o pedirles que verifiquen su dirección de correo electrónico.
* Hacer seguimiento de los cambios en el ciclo de vida de los usuarios para poder actualizar los sistemas de CRM (gestión de relaciones con clientes) o de facturación.
