<div id="use-cases">
  ## Casos de uso
</div>

Events son notificaciones en tiempo real sobre acciones específicas —inicialmente, cambios en un usuario y en las organizaciones, aunque con el tiempo también podrán ser otros— que han ocurrido dentro de tu tenant de Auth0. Puedes crear Event Streams que escuchen estos eventos para orquestar cambios asíncronos (es decir, cambios en una entidad o flujo que ya se han completado) en Auth0 y reenviarlos a uno de muchos destinos para su procesamiento externo. Events admite una variedad de casos de uso, entre ellos:

* Enviar correos electrónicos a nuevos clientes para darles la bienvenida o pedirles que verifiquen su dirección de correo electrónico.
* Monitorear los cambios en el ciclo de vida del usuario para que puedas actualizar los sistemas de CRM (gestión de relaciones con los clientes) o de facturación.
