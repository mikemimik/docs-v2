    <Section id={sections[1].id} title={sections[1].title} stepNumber="2" isSingleColumn>
      Primero, debes instalar el módulo `nginx-plus-module-njs` para NGINX Plus.

      Sigue la guía de instalación de módulos dinámicos para instalar paquetes en el sistema operativo host.

      En las distribuciones de Linux que usan el gestor de paquetes `yum`, puedes instalar el módulo de la siguiente manera:

      ```sh lines
      sudo yum install nginx-plus-module-njs jq
      ```

      Una vez instalado, debes habilitarlo en NGINX añadiendo la siguiente línea cerca del inicio de tu archivo
      `/etc/nginx/nginx.conf`:

      ```sh lines
      load_module modules/ngx_http_js_module.so;
      ```

      Clona el repositorio `nginx-openid-connect`. Este repositorio incluye una plantilla de configuración.

      ```sh lines
      git clone https://github.com/nginxinc/nginx-openid-connect
      ```

      <LoggedInForm />

    <Section id={sections[3].id} title={sections[3].title} stepNumber="4" isSingleColumn>
      Ejecuta el script configure.sh dentro de la carpeta nginx-openid-connect para rellenar la configuración de plantilla de
      tu aplicación de Auth0:

      ```sh lines
      ./configure.sh --auth_jwt_key request \
      --client_id {yourClientId} \
      --pkce_enable \
      https://{yourDomain}/.well-known/openid-configuration
      ```

      A continuación, añade la [URL de cierre de sesión de tu tenant](https://auth0.com/docs/api/authentication#auth0-logout) al
      archivo `openid_connect_configuration.conf`.
