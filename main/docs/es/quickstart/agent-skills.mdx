---
title: "Agent Skills para Auth0"
description: "Usa Agent Skills de Auth0 para integrar la autenticación en tus aplicaciones con asistentes de programación con IA como Claude Code, Cursor y GitHub Copilot"
---

Agent Skills son instrucciones estructuradas que ayudan a los asistentes de programación con IA a implementar correctamente la autenticación con Auth0. Instálalos una vez y luego pídele a tu asistente que añada el inicio de sesión, proteja rutas, asegure las API, configure MFA, migre desde otro proveedor o personalice tu página de inicio de sesión; él se encarga del resto.

Funciona con Claude Code, Cursor, GitHub Copilot y [más de 40 agentes](https://agentskills.io/clients) compatibles con el formato [Agent Skills](https://agentskills.io).

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una [cuenta de Auth0](https://auth0.com/signup) (plan Free disponible)
* Tu **dominio** de Auth0 y tu **ID de cliente** del [Auth0 Dashboard](https://manage.auth0.com)
* Un asistente de programación con IA (Claude Code, Cursor o GitHub Copilot)

<div id="install">
  ## Instalación
</div>

<Tabs>
  <Tab title="Claude Code">
    Auth0 está en el marketplace oficial de plugins de Claude Code (`claude-plugins-official`). Primero, asegúrate de que el marketplace esté actualizado y, después, instala:

    ```bash
    /plugin marketplace update claude-plugins-official
    /plugin install auth0@claude-plugins-official
    ```

    O escribe `/plugin` en una session, ve a la pestaña **Discover** y busca &quot;Auth0&quot;.

    También puedes instalarlo desde la terminal sin iniciar una session:

    ```bash
    claude plugin marketplace update claude-plugins-official
    claude plugin install auth0@claude-plugins-official
    ```

    <Note>
      Si recibes un error de &quot;not found&quot;, añade primero el marketplace y vuelve a intentar la instalación:

      ```bash
      /plugin marketplace add anthropics/claude-plugins-official
      /plugin install auth0@claude-plugins-official
      ```
    </Note>
  </Tab>

  <Tab title="Cursor">
    Auth0 está en el [marketplace de Cursor](https://cursor.com/marketplace/auth0). Abre la página y haz clic en **Add** para instalarlo.

    También puedes instalarlo desde **Cursor Settings &gt; Rules &gt; Add Rule &gt; Remote Rule (GitHub)** e introducir la [URL del repositorio de Auth0 Agent Skills](https://github.com/auth0/agent-skills).
  </Tab>

  <Tab title="GitHub Copilot">
    Instálalo con [Skills CLI](https://github.com/vercel-labs/skills) para GitHub Copilot:

    ```bash
    npx skills add auth0/agent-skills --agent github-copilot
    ```
  </Tab>

  <Tab title="Other Agents">
    [Skills CLI](https://github.com/vercel-labs/skills) funciona con Claude Code, Cursor, Copilot, Codex y [más de 40 agentes](https://agentskills.io/clients):

    ```bash
    npx skills add auth0/agent-skills --all
    ```

    Usa `--agent` para indicar agentes específicos:

    ```bash
    npx skills add auth0/agent-skills --agent claude-code cursor
    ```
  </Tab>
</Tabs>

<div id="try-it-now">
  ## Pruébalo ahora
</div>

Después de instalarlo, pega cualquiera de estos en tu asistente de programación con IA:

```
Add Auth0 login to my app
```

```
Secure my Express API with Auth0 JWT validation
```

```
Migrate from Firebase Auth to Auth0
```

<Tip>
  Da contexto al redactar la solicitud: menciona tu framework, el tipo de router y si necesitas inicio de sesión, proteger la API o ambas cosas. Cuanto más específica sea la solicitud, mejor será el resultado.
</Tip>

<div id="how-it-works">
  ## Cómo funciona
</div>

<Steps>
  <Step title="Describe qué necesitas">
    Dile a tu asistente lo que quieres: añadir inicio de sesión a tu aplicación de React, proteger una API de Spring Boot, migrar desde Cognito, añadir MFA o personalizar tu página de inicio de sesión. No necesitas saber qué skill o SDK usar.
  </Step>

  <Step title="Se activa la skill correcta">
    El asistente lee los archivos de tu proyecto (`package.json`, `requirements.txt`, `build.gradle`, etc.), identifica tu framework y activa la skill correspondiente de entre las 25 disponibles.
  </Step>

  <Step title="Obtienes código funcional">
    La skill guía a tu asistente a lo largo de toda la implementación: instalar el SDK adecuado, crear rutas de autenticación, configurar variables de entorno, proteger rutas y seguir las prácticas recomendadas de Auth0.
  </Step>
</Steps>

<Info>
  No tienes que elegir las skills manualmente. La detección del framework gestiona automáticamente las skills de SDK, y puedes invocar directamente skills de funcionalidades como MFA o Migration simplemente describiendo lo que necesitas.
</Info>
