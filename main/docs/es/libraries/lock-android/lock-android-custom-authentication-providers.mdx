**Auth0.Android** incluye la clase `WebAuthProvider` para gestionar el flujo de autorización a través del navegador. Pero ¿qué pasa si quieres usar tu propia implementación o una versión Native de un `AuthProvider`?

<div id="the-authprovider-class">
  ## La clase AuthProvider
</div>

Cree una clase que implemente la interfaz `AuthProvider` y sobrescriba sus métodos.

También puede usar cualquier implementación de **Native** que Auth0 ya proporcione. Los proveedores de [autenticación social nativa](/docs/es/libraries/lock-android/lock-android-native-social-authentication) disponibles actualmente son Google y Facebook.
