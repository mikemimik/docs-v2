Esta biblioteca le permite personalizar el comportamiento del cliente de red para configuraciones comunes, así como definir y usar su propia implementación del cliente de red.

Puede configurar la clase Auth0 con un `NetworkingClient`, que se usará al realizar solicitudes. Puede configurar el cliente predeterminado con valores personalizados de tiempo de espera, los encabezados que deban enviarse en todas las solicitudes y si se debe registrar información de las solicitudes y respuestas (solo con fines de depuración en entornos que no sean de producción). Para una configuración más avanzada, puede proporcionar su propia implementación de `NetworkingClient`.

<div id="timeout-configuration">
  ### Configuración del timeout
</div>

export const codeExample1 = `val netClient = DefaultClient(
    connectTimeout = 30,
    readTimeout = 30
)

val account = Auth0("{yourClientId}", "{yourDomain}")
account.networkingClient = netClient`;
