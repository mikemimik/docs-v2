Esta biblioteca te permite personalizar el comportamiento del cliente de red para configuraciones habituales, así como definir y usar tu propia implementación de `NetworkingClient`.

Puedes configurar la clase `Auth0` con un `NetworkingClient`, que se utilizará al realizar solicitudes. Puedes configurar el cliente predeterminado con valores de tiempo de espera personalizados, los encabezados que deban enviarse en todas las solicitudes y si se debe registrar información de solicitudes/respuestas (solo con fines de depuración en entornos no productivos). Para una configuración más avanzada, puedes proporcionar tu propia implementación de `NetworkingClient`.

<div id="timeout-configuration">
  ### Configuración del tiempo de espera
</div>

export const codeExample1 = `val netClient = DefaultClient(
    connectTimeout = 30,
    readTimeout = 30
)

val account = Auth0("{yourClientId}", "{yourDomain}")
account.networkingClient = netClient`;
