<ParamField body="continue" type="Promise<void>">
  Confirma que el usuario ha guardado el nuevo código de recuperación y continúa el flujo de autenticación.
  Este método debe llamarse después de que el usuario indique que ha guardado de forma segura el código de recuperación mostrado.
  Envía una confirmación al servidor de Auth0 para proceder al siguiente paso.

  Una promesa que se resuelve cuando la confirmación se envía correctamente.

  #### Lanza

  Lanza un error si el envío falla (por ejemplo, por un problema de red o un `state` no válido).

    await screenManager.continue();
    // Redirige a la siguiente pantalla si se completa correctamente
  } catch (error) {
    console.error("Failed to confirm recovery code saved:", error);
