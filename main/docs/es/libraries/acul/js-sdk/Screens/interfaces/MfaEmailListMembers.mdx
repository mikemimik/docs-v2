title: "MfaEmailListMembers"
---

Interfaz que define los métodos y las propiedades disponibles para la pantalla mfa-email-list

```ts Example
export interface MfaEmailListMembers extends BaseMembers {
  screen: ScreenMembersOnMfaEmailList;
  /**
   * Selecciona una dirección de correo electrónico registrada de la lista
   * @param payload Las opciones que contienen la Action de selección
   */
  selectMfaEmail(payload: SelectMfaEmailOptions): Promise<void>;
  /**
   * Navega de regreso a la pantalla anterior
   * @param payload Opciones Custom opcionales para incluir con la solicitud
   */
  goBack(payload?: CustomOptions): Promise<void>;
}
```
