---
title: "MfaRecoveryCodeEnrollmentMembers"
---

Interfaz que describe los miembros de la pantalla de inscripción del código de recuperación de MFA.

```ts Example
export interface MfaRecoveryCodeEnrollmentMembers extends BaseMembers {
  screen: ScreenMembersOnMfaRecoveryCodeEnrollment;

  /**
   * Declara que el usuario guardó el código de recuperación.
   * Esta Action se activa cuando el usuario declara que ha guardado el código de recuperación.
   * Redirige a la siguiente pantalla en el flujo de autenticación.
   *
   * @param {object} payload - Un objeto que contiene opciones personalizadas.
   *
   * @returns {Promise<void>} Una promesa que se resuelve cuando la Action se envía correctamente.
   * Se rechaza con un error si el envío falla.
   */
  continue(payload: MfaRecoveryCodeEnrollmentContinueOptions): Promise<void>;
}
```
