ContinueOptions
Define las opciones necesarias para continuar con el proceso de verificación MFA por teléfono.
Normalmente, esto implica seleccionar el método (SMS o Voz) para recibir el código de verificación.

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * El tipo de desafío a realizar.
   * 'sms': Enviar el código por mensaje de texto.
   * 'voice': Enviar el código mediante una llamada de voz.
   * @default 'sms'
   */
  type: 'sms' | 'voice';
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;">
  El tipo de desafío que se realizará.

  * `sms`: Envía el código por mensaje de texto.
  * `voice`: Envía el código mediante una llamada telefónica.
</ParamField>
