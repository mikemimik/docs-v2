TryAnotherMethodOptions

```ts Example
export interface TryAnotherMethodOptions extends CustomOptions {
  /**
   * Opcional. Si es `true`, y si el screen context (`screen.showRememberDevice`)
   * indica que recordar el device es una opción, esto señala la intención del usuario
   * de recordar este navegador/device.
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="rememberDevice?" type="boolean">
  Opcional. Si `true` y `screen.showRememberDevice` también es `true`,
  el SDK incluirá `rememberBrowser=true` al enviar el formulario.
</ParamField>
