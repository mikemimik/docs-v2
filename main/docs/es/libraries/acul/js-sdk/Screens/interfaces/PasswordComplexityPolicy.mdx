---
title: "PasswordComplexityPolicy"
---

Define las reglas de complejidad de la contraseña que se aplican durante el registro o los flujos de restablecimiento de contraseña.

```ts Example
export interface PasswordComplexityPolicy {
  character_type_rule: 'all' | 'at_least';
  character_types: ('number' | 'uppercase' | 'lowercase' | 'special')[];
  identical_characters: 'block' | 'allow';
  max_length_exceeded: 'allow' | 'error';
  min_length: number;
  sequential_characters: 'block' | 'allow';
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="character_type_rule" type="&#x22;all&#x22; | &#x22;at_least&#x22;" required>
  Determina si deben estar presentes todos los tipos de caracteres especificados (`'all'`) o solo algunos (`'at_least'`).
</ParamField>

<ParamField body="character_types" type="(&#x22;number&#x22; | &#x22;uppercase&#x22; | &#x22;lowercase&#x22; | &#x22;special&#x22;)[]" required>
  Especifica qué categorías de caracteres se deben exigir en la contraseña.
</ParamField>

<ParamField body="identical_characters" type="&#x22;block&#x22; | &#x22;allow&#x22;" required>
  Controla si se permiten (`'allow'`) o se bloquean (`'block'`) los caracteres idénticos repetidos.
</ParamField>

<ParamField body="max_length_exceeded" type="&#x22;allow&#x22; | &#x22;error&#x22;" required>
  Define el comportamiento cuando la contraseña supera la longitud máxima: permitirlo silenciosamente o devolver un error.
</ParamField>

<ParamField body="min_length" type="number" required>
  La cantidad mínima de caracteres requerida para la contraseña.
</ParamField>

<ParamField body="sequential_characters" type="&#x22;block&#x22; | &#x22;allow&#x22;" required>
  Determina si se permiten (`'allow'`) o se bloquean (`'block'`) las secuencias consecutivas de caracteres (p. ej., `'abc'`, `'123'`).
</ParamField>
