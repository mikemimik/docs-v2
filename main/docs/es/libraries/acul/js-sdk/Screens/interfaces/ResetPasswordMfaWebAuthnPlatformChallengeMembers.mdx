<ParamField body="continueWithPasskey" type="Promise<void>">
  Inicia el desafío del autenticador de plataforma WebAuthn llamando a `navigator.credentials.get()`
  con las opciones de desafío proporcionadas en `screen.publicKey`.
  Si se completa correctamente, envía a Auth0 la aserción de la credencial resultante con `action: "default"`.

  Una promesa que se resuelve cuando se envía el intento de verificación.
  Por lo general, una operación correcta da lugar a una redirección.

  #### Throws

  Lanza un error si falta `screen.publicKey`,
  si `navigator.credentials.get()` falla (por ejemplo, por cancelación del usuario: `NotAllowedError`),
  o si falla el envío del formulario a Auth0. Si `navigator.credentials.get()` falla
  con una `DOMException`, se recomienda capturar ese error y llamar a `reportBrowserError`.

  ```typescript Example
  // Suponiendo que 'sdk' es una instancia de ResetPasswordMfaWebAuthnPlatformChallenge
  try {
    await sdk.continueWithPasskey({
      rememberDevice: true // si el usuario marcó la casilla y sdk.screen.showRememberDevice es true
    });
    // Si se completa correctamente, Auth0 se encarga de la redirección.
  } catch (error) {
    console.error("Falló la verificación del autenticador de plataforma:", error);
    // Si es un error de la API de WebAuthn (DOMException), repórtalo
    if (error instanceof DOMException && error.name && error.message) {
      await sdk.reportBrowserError({ error: { name: error.name, message: error.message } });
    }
    // Revisa sdk.transaction.errors para ver mensajes de validación del servidor si la página se recarga.
  }
  ```

<ParamField body="reportBrowserError" type="Promise<void>">
  Notifica un error del navegador ocurrido durante la llamada a la API `navigator.credentials.get()`.
  Se usa para informar a Auth0 sobre problemas como la cancelación del usuario, un tiempo de espera agotado o hardware no compatible.
  Envía los detalles del error con `action: "showError::{errorDetailsJsonString}"`.

  Una promesa que se resuelve cuando se envía el informe de error.

  #### Throws

  Si falla el envío del formulario (por ejemplo, por un error de red).

  ```typescript Example
  // Suponiendo que 'sdk' es una instancia de ResetPasswordMfaWebAuthnPlatformChallenge
  // y 'webAuthnError' es una DOMException de una llamada fallida a navigator.credentials.get().
  try {
    await sdk.reportBrowserError({
      error: { name: webAuthnError.name, message: webAuthnError.message }
    });
    // Auth0 puede volver a renderizar la página o redirigir en función del error.
  } catch (submitError) {
    console.error("No se pudo informar el error del navegador de WebAuthn:", submitError);
  }
  ```
