title: "DeviceCodeActivationContinueOptions"
---

payload para continuar la activación del device code.

Esta interfaz extiende `CustomOptions`, lo que significa que todas sus propiedades personalizadas
se combinan en el nivel superior del objeto. Esto evita el anidamiento y ofrece
una estructura más simple para quienes usan esta interfaz.

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```
