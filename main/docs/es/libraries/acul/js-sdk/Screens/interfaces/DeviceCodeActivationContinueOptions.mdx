payload para continuar la activación del código de dispositivo.

Esta interfaz extiende `CustomOptions`, lo que significa que todas las propiedades de las opciones personalizadas
se integran en el nivel superior del objeto. Esto evita el anidamiento y ofrece
una estructura más simple para quienes usan esta interfaz.

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```
