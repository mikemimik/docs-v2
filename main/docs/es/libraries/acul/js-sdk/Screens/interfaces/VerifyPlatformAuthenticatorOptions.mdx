VerifyPlatformAuthenticatorOptions

```ts Example
export interface VerifyPlatformAuthenticatorOptions extends CustomOptions {
  /**
   * Opcional. Si es true y `screen.showRememberDevice` es true, indica que el usuario
   * ha elegido recordar este device, generalmente durante 30 días, para omitir el MFA en inicios de sesión posteriores.
   * @type {boolean}
   */
  rememberDevice?: boolean;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="rememberDevice?" type="boolean">
  Opcional. Si es true, intenta recordar el navegador para futuros desafíos MFA.
  Corresponde al campo del formulario `rememberBrowser`. Solo se aplica si `screen.showRememberDevice` es true.
</ParamField>
