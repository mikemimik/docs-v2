Interfaz que define los métodos y las propiedades disponibles para la pantalla de reset-password-mfa-email-challenge

```ts Example
export interface ResetPasswordMfaEmailChallengeMembers extends BaseMembers {
  screen: ScreenMembersOnResetPasswordMfaEmailChallenge;

  /**
   * Continúa con la verificación por correo electrónico usando el código proporcionado.
   * @param payload Las opciones que contienen el código y el indicador rememberDevice.
   */
  continue(payload: ContinueOptions): Promise<void>;

  /**
   * Reenvía el código por correo electrónico.
   * @param payload Opciones personalizadas opcionales para incluir en la solicitud.
   */
  resendCode(payload?: ResendCodeOptions): Promise<void>;

  /**
   * Permite al usuario probar otro método de MFA.
   * @param payload Opciones personalizadas opcionales para incluir en la solicitud.
   */
  tryAnotherMethod(payload?: TryAnotherMethodOptions): Promise<void>;

  /**
   * Obtiene la funcionalidad de reenvío con gestión del tiempo de espera para esta pantalla
   * @param options Opciones de configuración para la funcionalidad de reenvío
   */
  resendManager(options?: StartResendOptions): ResendControl;
}
```
