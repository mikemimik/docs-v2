---
title: "DeviceCodeConfirmationMembers"
---

Interfaz que describe los miembros de la pantalla de confirmación de código de dispositivo.

```ts Example
export interface DeviceCodeConfirmationMembers extends BaseMembers {
  screen: ScreenMembersOnDeviceCodeConfirmation;

  /**
   * Confirma el código de dispositivo.
   * @param payload Opciones personalizadas opcionales para incluir con la solicitud.
   */
  confirm(payload?: CustomOptions): Promise<void>;

  /**
   * Cancela el flujo de código de dispositivo.
   * @param payload Opciones personalizadas opcionales para incluir con la solicitud.
   */
  cancel(payload?: CustomOptions): Promise<void>;
}
```
