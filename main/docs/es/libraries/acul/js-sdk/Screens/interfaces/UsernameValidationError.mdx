---
title: "UsernameValidationError"
---

Representa un único error de validación del nombre de usuario que se devuelve durante la validación.

Cada error tiene un `code` legible por máquina y un `message` legible para humanos.

```ts Example
export interface UsernameValidationError {
  /**
   * Un identificador único para la regla de validación que falló.
   * 
   * Ejemplo: `"username-too-short"`, `"username-invalid-characters"`
   */
  code: string;

  /**
   * Una descripción legible por humanos del error.
   * 
   * Ejemplo: `"Username must be at least 3 characters long."`
   */
  message: string;

  /**
   * El campo asociado al error de validación, generalmente "username", "email" o "phone".
   */
  field: string;
}
```
