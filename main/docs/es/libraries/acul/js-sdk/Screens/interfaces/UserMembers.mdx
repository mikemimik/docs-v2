---
title: "Miembros de usuarios"
---

```ts Example
export interface UserMembers {
  id: string | null;
  email: string | null;
  username: string | null;
  phoneNumber: string | null;
  picture: string | null;
  enrolledFactors: Array<string> | null;
  enrolledEmails: Array<EnrolledEmail> | null;
  enrolledPhoneNumbers: Array<EnrolledPhoneNumber> | null;
  enrolledDevices: Array<EnrolledDevice> | null;
  organizations: Organizations[] | null; // OPT-IN 
  userMetadata: { [key: string]: string } | null; // OPT-IN
  appMetadata: { [key: string]: string } | null; // OPT-IN
}
```

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * OPT-IN: Debe configurar explícitamente estas propiedades para que estén disponibles en la pantalla. Para obtener más información, lea [Configurar ACUL](/docs/es/customize/login-pages/advanced-customizations/configure).

  * Cuando el servidor de Auth0 no identifica al usuario actual, las propiedades de `user` no están disponibles en las siguientes pantallas:

  | Pantallas                              | Pantallas                                  | Pantallas                                | Pantallas                                                  |
  | -------------------------------------- | ------------------------------------------ | ---------------------------------------- | ---------------------------------------------------------- |
  | Pantallas de inicio de sesión          | Pantallas de OTP de MFA                    | Pantallas de Aceptar invitación          | Pantallas de InterstitialCaptcha                           |
  | Pantallas de registro                  | Pantallas de teléfono de MFA               | Pantallas de Device Code                 | Pantallas de Restablecer contraseña                        |
  | Pantallas de clave de acceso           | Pantallas de MFA push                      | Pantallas de verificación de email       | Pantallas de códigos de país de MFA                        |
  | Pantallas de desafío                   | Pantallas de código de recuperación de MFA | Pantallas de cierre de sesión abortado   | Pantallas de detección de capacidades del navegador de MFA |
  | Pantallas de identificador de email    | Pantallas de MFA WebAuthn                  | Pantallas de cierre de sesión completado | Pantallas de inscripción de MFA                            |
  | Pantallas de identificador de teléfono | Pantallas de Aceptar invitación            | Pantallas de Brute Force Protection      |                                                            |
</Callout>
