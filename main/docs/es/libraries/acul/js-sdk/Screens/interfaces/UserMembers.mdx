```ts Example
export interface UserMembers {
  id: string | null;
  email: string | null;
  username: string | null;
  phoneNumber: string | null;
  picture: string | null;
  enrolledFactors: Array<string> | null;
  enrolledEmails: Array<EnrolledEmail> | null;
  enrolledPhoneNumbers: Array<EnrolledPhoneNumber> | null;
  enrolledDevices: Array<EnrolledDevice> | null;
  organizations: Organizations[] | null; // OPT-IN 
  userMetadata: { [key: string]: string } | null; // OPT-IN
  appMetadata: { [key: string]: string } | null; // OPT-IN
}
```

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * OPT-IN: Debes configurar explícitamente estas propiedades para que estén disponibles en la pantalla. Para obtener más información, consulta [Configurar ACUL](/docs/es/customize/login-pages/advanced-customizations/configure).

  * Cuando el servidor de Auth0 no tiene identificado al usuario actual, las propiedades de `user` no están disponibles en las siguientes pantallas:

  | Pantallas de registro                            | Pantallas de teléfono de MFA               | Pantallas de Device Code                        | Pantallas de restablecimiento de contraseña                  |
  | Pantallas de clave de acceso                     | Pantallas de MFA push                      | Pantallas de verificación de correo electrónico | Pantallas de códigos de país para MFA                        |
  | Pantallas de desafío                             | Pantallas de código de recuperación de MFA | Pantallas de cierre de sesión abortado          | Pantallas de detección de capacidades del navegador para MFA |
  | Pantallas de identificador de correo electrónico | Pantallas de WebAuthn de MFA               | Pantallas de cierre de sesión completado        | Pantallas de inscripción en MFA                              |
  | Pantallas de identificador de teléfono           | Pantallas de aceptar invitación            | Pantallas de Brute Force Protection             |                                                              |
