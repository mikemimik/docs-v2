```ts Example
export interface UserMembers {
  id: string | null;
  email: string | null;
  username: string | null;
  phoneNumber: string | null;
  picture: string | null;
  enrolledFactors: Array<string> | null;
  enrolledEmails: Array<EnrolledEmail> | null;
  enrolledPhoneNumbers: Array<EnrolledPhoneNumber> | null;
  enrolledDevices: Array<EnrolledDevice> | null;
  organizations: Organizations[] | null; // OPT-IN 
  userMetadata: { [key: string]: string } | null; // OPT-IN
  appMetadata: { [key: string]: string } | null; // OPT-IN
}
```

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * OPT-IN: Debe configurar explícitamente estas propiedades para que estén disponibles en la pantalla. Para obtener más información, consulte [configurar ACUL](/docs/es/customize/login-pages/advanced-customizations/configure).

  * Cuando el servidor de Auth0 no reconoce al usuario actual, las propiedades de `user` no están disponibles en las siguientes pantallas:

  | Pantallas                                        | Pantallas                                  | Pantallas                                       | Pantallas                                                  |

  | Pantallas de login                               | Pantallas de MFA OTP                       | Pantallas de Aceptar invitación                 | Pantallas de InterstitialCaptcha                           |
  | Pantallas de signup                              | Pantallas de MFA Phone                     | Pantallas de código de dispositivo              | Pantallas de Restablecer contraseña                        |
  | Pantallas de clave de acceso                     | Pantallas de MFA Push                      | Pantallas de verificación de correo electrónico | Pantallas de código de país de MFA                         |
  | Pantallas de challenge                           | Pantallas de código de recuperación de MFA | Pantallas de cierre de sesión cancelado         | Pantallas de detección de capacidades del navegador de MFA |
  | Pantallas de identificador de correo electrónico | Pantallas de MFA WebAuthn                  | Pantallas de cierre de sesión completado        | Pantallas de inscripción en MFA                            |
  | Pantallas de identificador de teléfono           | Pantallas de Aceptar invitación            | Pantallas de Brute Force Protection             |                                                            |
