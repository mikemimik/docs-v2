---
title: "SelectMfaPushDeviceOptions"
---

Opciones para seleccionar un dispositivo para MFA push.

```ts Example
export interface SelectMfaPushDeviceOptions {
  /**
   * El índice del device a seleccionar de la lista de dispositivos registrados.
   * El índice es base 0.
   */
  deviceIndex: number;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="deviceIndex" type="number">
  El índice del dispositivo que se debe seleccionar de la lista de dispositivos registrados.
  El índice empieza en 0.
</ParamField>
