ResendCodeOptions
description Opciones para el método `resendCode` en la pantalla de Verificación de Email de Login.
Esta operación se utiliza cuando el usuario solicita que se le envíe un nuevo código de verificación a su correo electrónico,
por ejemplo, si no recibió el código inicial o si ya ha caducado.

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
