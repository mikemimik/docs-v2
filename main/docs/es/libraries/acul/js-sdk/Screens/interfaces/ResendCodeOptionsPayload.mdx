ResendCodeOptions
description Opciones del método `resendCode` en la pantalla Verificación de correo electrónico de Login.
Esta operación se utiliza cuando el usuario solicita que se le envíe un nuevo código de verificación por correo electrónico,
por ejemplo, si no recibió el código inicial o si ya ha caducado.

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
