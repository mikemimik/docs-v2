---
title: "MfaPushPollingControl"
---

Interfaz de control para gestionar una sesión de consulta periódica de una notificación push de MFA.

Esta interfaz proporciona métodos imperativos para iniciar, detener e inspeccionar el estado de un
bucle de consulta periódica de larga duración que comprueba si se ha
aprobado un desafío de notificación push de MFA.

````ts Example
export interface MfaPushPollingControl {
  /**
   * Detiene el proceso de consulta de inmediato.
   *
   * - Cancela cualquier temporizador programado o solicitud pendiente.
   * - Una vez detenido, `isRunning()` devuelve `false`.
   * - Es seguro llamarlo varias veces; las llamadas posteriores no tienen efecto.
   *
   * @example
   * ```ts
   * const control = mfaPushChallengePush.pollingManager({ intervalMs: 5000 });
   * control.startPolling();
   *
   * // Más tarde, si el usuario cancela:
   * control.stopPolling();
   * ```
   */
  stopPolling: () => void;

  /**
   * Inicia o reanuda el proceso de consulta.
   *
   * - Si la consulta ya está activa, volver a llamarlo no tiene efecto.
   * - Si se detuvo previamente, llamarlo reinicia el bucle de consulta.
   *
   * @example
   * ```ts
   * control.startPolling(); // Comienza a verificar el desafío de notificación push de MFA
   * ```
   */
  startPolling: () => void;

  /**
   * Indica si el proceso de consulta está en ejecución actualmente.
   *
   * - Devuelve `true` si la consulta está activa y no ha sido cancelada.
   * - Devuelve `false` si la consulta se ha detenido o ha finalizado.
   *
   * @example
   * ```ts
   * if (control.isRunning()) {
   *   console.log('Consulta en progreso...');
   * } else {
   *   console.log('La consulta está detenida o completada.');
   * }
   * ```
   */
  isRunning: () => boolean;
}
````

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isRunning" type="boolean">
  Indica si el proceso de consulta se está ejecutando en ese momento.

  ```ts Example
  if (control.isRunning()) {
    console.log('Polling in progress...');
  } else {
    console.log('Polling is stopped or completed.');
  }
  ```
</ParamField>

<ParamField body="startPolling" type="void">
  Inicia o reanuda el proceso de consulta.

  * Si la consulta ya está activa, volver a llamar a este método no tiene efecto.
  * Si se detuvo anteriormente, volver a llamarlo reinicia el bucle de consulta.

  ```ts Example
  control.startPolling(); // Empezar a comprobar el desafío de notificación push de MFA
  ```
</ParamField>

<ParamField body="stopPolling" type="void">
  Detiene el proceso de consulta de inmediato.

  * Cancela cualquier temporizador programado o solicitud pendiente.
  * Una vez detenido, `isRunning()` devuelve `false`.
  * Se puede llamar varias veces sin problema; las llamadas posteriores no tienen efecto.

  ```ts Example
  const control = mfaPushChallengePush.pollingManager({ intervalMs: 5000 });
  control.startPolling();

  // Más tarde, si el usuario cancela:
  control.stopPolling();
  ```
</ParamField>
