---
title: "MfaPushPollingControl"
---

Interfaz de control para gestionar una sesión de consulta periódica de notificaciones push de MFA.

Esta interfaz proporciona métodos imperativos para iniciar, detener y consultar el estado de un
bucle de consulta periódica de larga duración que comprueba si se ha
aprobado un desafío push de MFA.

````ts Example
export interface MfaPushPollingControl {
  /**
   * Detiene el proceso de consulta periódica de inmediato.
   *
   * - Cancela cualquier temporizador programado o solicitud pendiente.
   * - Una vez detenido, `isRunning()` devuelve `false`.
   * - Es seguro llamarlo varias veces; las llamadas posteriores no tienen efecto.
   *
   * @example
   * ```ts
   * const control = mfaPushChallengePush.pollingManager({ intervalMs: 5000 });
   * control.startPolling();
   *
   * // Más tarde, si el usuario cancela:
   * control.stopPolling();
   * ```
   */
  stopPolling: () => void;

  /**
   * Inicia o reanuda el proceso de consulta periódica.
   *
   * - Si la consulta periódica ya está activa, volver a llamarlo no tiene efecto.
   * - Si se detuvo previamente, llamarlo reinicia el bucle de consulta periódica.
   *
   * @example
   * ```ts
   * control.startPolling(); // Comienza a verificar el MFA push challenge
   * ```
   */
  startPolling: () => void;

  /**
   * Indica si el proceso de consulta periódica está en ejecución actualmente.
   *
   * - Devuelve `true` si la consulta periódica está activa y no ha sido cancelada.
   * - Devuelve `false` si la consulta periódica se ha detenido o ha finalizado.
   *
   * @example
   * ```ts
   * if (control.isRunning()) {
   *   console.log('Consulta periódica en progreso...');
   * } else {
   *   console.log('La consulta periódica está detenida o completada.');
   * }
   * ```
   */
  isRunning: () => boolean;
}
````

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isRunning" type="boolean">
  Indica si el proceso de consulta periódica se está ejecutando en este momento.

  ```ts Example
  if (control.isRunning()) {
    console.log('Consulta periódica en curso...');
  } else {
    console.log('La consulta periódica se detuvo o se completó.');
  }
  ```
</ParamField>

<ParamField body="startPolling" type="void">
  Inicia o reanuda el proceso de consulta periódica.

  * Si la consulta periódica ya está activa, volver a llamarlo no tiene ningún efecto.
  * Si se detuvo anteriormente, volver a llamarlo reinicia el ciclo de consulta periódica.

  ```ts Example
  control.startPolling(); // Empieza a comprobar el desafío push de MFA
  ```
</ParamField>

<ParamField body="stopPolling" type="void">
  Detiene el proceso de consulta periódica de inmediato.

  * Cancela cualquier temporizador programado o solicitud pendiente.
  * Una vez detenido, `isRunning()` devuelve `false`.
  * Se puede llamar varias veces sin problema; las llamadas posteriores no tienen efecto.

  ```ts Example
  const control = mfaPushChallengePush.pollingManager({ intervalMs: 5000 });
  control.startPolling();

  // Más adelante, si el usuario cancela:
  control.stopPolling();
  ```
</ParamField>
