PickPhoneOptions
Define las opciones de la acción &#39;pick-phone&#39;.
Aunque la interfaz solo muestra un número de teléfono, esta acción puede usarse
en casos en los que el servidor necesite redirigir a una pantalla de selección de teléfono.

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```
