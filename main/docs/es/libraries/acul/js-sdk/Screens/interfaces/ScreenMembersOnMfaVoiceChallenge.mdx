Interfaz extendida de ScreenMembers para la pantalla de desafío de voz de MFA.

```ts Example
export interface ScreenMembersOnMfaVoiceChallenge extends ScreenMembers {
  /**
   * Datos adicionales de pantalla específicos del desafío de voz de MFA.
   */
  data: {
    /**
     * El número de teléfono al que se envió el código de voz.
     */
    phoneNumber?: string;

    /**
     * Si se debe mostrar la opción de recordar dispositivo.
     */
    showRememberDevice?: boolean;

    /**
     * Si se debe mostrar el enlace para cambiar a la verificación por SMS.
     */
    showLinkSms?: boolean;
  } | null;
}
```
