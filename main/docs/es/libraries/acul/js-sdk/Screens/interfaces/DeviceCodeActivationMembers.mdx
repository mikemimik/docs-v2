---
title: "DeviceCodeActivationMembers"
---

Interfaz que describe los miembros de la pantalla de activación del código de dispositivo.

```ts Example
export interface DeviceCodeActivationMembers extends BaseMembers {
  /**
   * Envía el código de dispositivo introducido por el usuario.
   * Esta Action se activa cuando el usuario introduce el código que aparece en su dispositivo y envía el formulario.
   *
   * @param {object} payload - Un objeto que contiene el código introducido por el usuario y cualquier payload personalizado.
   * @param {string} payload.code - El código de dispositivo introducido por el usuario.
   *
   * @returns {Promise<void>} Una promesa que se resuelve cuando el código se envía correctamente.
   * Se rechaza con un error si el envío falla.
   */
  continue(payload: ContinueOptions): Promise<void>;
}
```
