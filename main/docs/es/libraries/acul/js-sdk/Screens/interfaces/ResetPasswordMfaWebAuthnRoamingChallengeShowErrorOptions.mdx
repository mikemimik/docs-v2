ShowErrorOptions

```ts Example
export interface ShowErrorOptions {
  /**
   * El objeto de error capturado de una llamada fallida a `navigator.credentials.get()`.
   * Debe incluir al menos las propiedades `name` y `message` del DOMException.
   * @type {WebAuthnErrorDetails}
   * @example { name: "NotAllowedError", message: "The operation was aborted by the user." }
   */
  error: WebAuthnErrorDetails;

  /**
   * Opcional. Si es `true`, y si el contexto de pantalla (`screen.showRememberDevice`)
   * indica que recordar el device es una opción, esto señala la intención del usuario
   * de recordar este navegador/device.
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```

<ParamField body="rememberDevice?" type="boolean">
  Opcional. Si `true` y `screen.showRememberDevice` también es `true`,
  el SDK incluirá `rememberBrowser=true` en el envío del formulario.
</ParamField>
