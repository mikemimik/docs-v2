> **getCurrentScreen**(): `string` | `null`

**`Función común`**

Obtiene el nombre de la pantalla actual en el contexto de autenticación

<div id="returns">
  ## Devuelve
</div>

`string` | `null`

El nombre de la pantalla actual, o `null` si no hay ninguna pantalla activa
