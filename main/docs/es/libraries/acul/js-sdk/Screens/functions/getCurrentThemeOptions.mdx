> **getCurrentThemeOptions**(): [`FlattenedTheme`](/docs/es/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

**`Funcionalidad común`**

Obtiene las opciones del tema actual con la configuración aplanada del contexto de branding

<div id="returns">
  ## Devuelve
</div>

[`FlattenedTheme`](/docs/es/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

Objeto `FlattenedTheme` que contiene colores, fuentes, bordes, `pageBackground` y configuraciones del widget, o `null` si no hay branding disponible
