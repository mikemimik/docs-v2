> **getCurrentThemeOptions**(): [`FlattenedTheme`](/docs/es/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

**`Funcionalidad común`**

Obtiene las opciones del tema actual con la configuración simplificada desde el contexto de branding

<div id="returns">
  ## Devuelve
</div>

[`FlattenedTheme`](/docs/es/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

Objeto `FlattenedTheme` con los colores, fuentes, bordes, pageBackground y la configuración del widget, o `null` si no hay branding disponible
