---
title: "Alias de tipo: ScreenIdentifierType"
---

<ParamField body="ScreenIdentifierType" type="email | phone | username">
  Representa los tipos de identificadores de usuario que se pueden usar durante el registro.

  * `'email'`: El usuario proporcionará una dirección de correo electrónico.
  * `'phone'`: El usuario proporcionará un número de teléfono.
  * `'username'`: El usuario elegirá un nombre de usuario.

  Este tipo se usa normalmente en la selección de identidad o en los flujos de registro
  para definir qué identificadores están habilitados y cómo se comportan.

  ```ts Example
  const type: IdentifierType = 'email';
  ```
</ParamField>
