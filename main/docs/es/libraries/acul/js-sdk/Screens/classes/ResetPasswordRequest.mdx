---
title: "ResetPasswordRequest"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `reset-password-request` de Universal Login."
---

La clase ResetPasswordRequest implementa la funcionalidad de la pantalla `reset-password-request`. Esta pantalla solicita al usuario que introduzca su identificador para iniciar el flujo de restablecimiento de contraseña.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla ResetPasswordRequest:

```typescript Example
import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
const resetPasswordRequestManager = new ResetPasswordRequest();
await resetPasswordRequestManager.resetPassword({ username: 'testuser' });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="backToLogin" type="Promise<void>">
  Este método devuelve al usuario a la pantalla de inicio de sesión.

  ```typescript Example
  import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
  const resetPasswordRequestManager = new ResetPasswordRequest();
  await resetPasswordRequestManager.backToLogin();
  ```
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
  const resetPasswordRequestManager = new ResetPasswordRequest();
  resetPasswordRequestManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Si se establece en `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción del contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="getLoginIdentifiers" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a>[]</span>}>
  Este método devuelve los tipos de identificador activos (como `username` o `email`) configurados para la pantalla `reset-password-request`.

  ```typescript Example
  import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
  const resetPasswordRequestManager = new ResetPasswordRequest();
  resetPasswordRequestManager.getLoginIdentifiers();
  ```
</ParamField>

<ParamField body="resetPassword" type="Promise<void>">
  Este método envía el identificador del usuario para iniciar el flujo de restablecimiento de contraseña.

  ```typescript Example
  import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
  const resetPasswordRequestManager = new ResetPasswordRequest();
  await resetPasswordRequestManager.resetPassword({ username: 'testuser' });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordRequestOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordRequestOptions).
    </ParamField>

    <ParamField body="username" type="string" required>
      El nombre de usuario o la dirección de correo electrónico de la cuenta que se quiere restablecer.
    </ParamField>

    <ParamField body="captcha?" type="string">
      El token de respuesta de CAPTCHA, si CAPTCHA está habilitado para este tenant.
    </ParamField>
  </Expandable>
</ParamField>
