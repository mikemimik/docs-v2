---
title: "OrganizationSelection"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `organization-selection` de Universal Login."
---

La clase OrganizationSelection implementa la funcionalidad de la pantalla `organization-selection`. Esta pantalla permite a los usuarios seleccionar una organización para continuar.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de pantalla OrganizationSelection:

```typescript Example
import OrganizationSelection from '@auth0/auth0-acul-js/organization-selection';
const organizationSelectionManager = new OrganizationSelection();
await organizationSelectionManager.continueWithOrganizationName({
  organizationName: 'testOrganizationName',
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import OrganizationSelection from '@auth0/auth0-acul-js/organization-selection';
  const organizationSelectionManager = new OrganizationSelection();
  organizationSelectionManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de `locale` del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueWithOrganizationName" type="Promise<void>">
  Este método continúa con el nombre de la organización seleccionada.

  ```typescript Example
  import OrganizationSelection from '@auth0/auth0-acul-js/organization-selection';
  const organizationSelectionManager = new OrganizationSelection();
  await organizationSelectionManager.continueWithOrganizationName({
    organizationName: 'testOrganizationName',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ContinueWithOrganizationNameOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ContinueWithOrganizationNameOptions).
    </ParamField>

    <ParamField body="organizationName" type="string" required>
      El nombre de la organización con la que se continuará.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera del contexto el arreglo de errores de la transacción, o un arreglo vacío si no hay ninguno.
</ParamField>
