---
title: "OrganizationPicker"
description: "Describe todas las propiedades y los métodos disponibles para personalizar la pantalla `organization-picker` de Universal Login."
---

La clase OrganizationPicker implementa la funcionalidad de la pantalla `organization-picker`. Esta pantalla permite a los usuarios seleccionar una organización de una lista de organizaciones disponibles.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla OrganizationPicker:

```typescript Example
import OrganizationPicker from '@auth0/auth0-acul-js/organization-picker';
const organizationPickerManager = new OrganizationPicker();
await organizationPickerManager.selectOrganization({ organization: 'org_XXXXXXXXXXXXXXX' });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import OrganizationPicker from '@auth0/auth0-acul-js/organization-picker';
  const organizationPickerManager = new OrganizationPicker();
  organizationPickerManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="selectOrganization" type="Promise<void>">
  Este método envía el ID de la organización seleccionada.

  ```typescript Example
  import OrganizationPicker from '@auth0/auth0-acul-js/organization-picker';
  const organizationPickerManager = new OrganizationPicker();
  await organizationPickerManager.selectOrganization({ organization: 'org_XXXXXXXXXXXXXXX' });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SelectOrganizationOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/SelectOrganizationOptions).
    </ParamField>

    <ParamField body="organization" type="string" required>
      El ID de la organización que se desea seleccionar.
    </ParamField>

    <ParamField body="state?" type="string">
      Un valor de state opcional para incluir en la solicitud.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="skipOrganizationSelection" type="Promise<void>">
  Este método envía la acción para omitir la selección de organización.

  ```typescript Example
  import OrganizationPicker from '@auth0/auth0-acul-js/organization-picker';
  const organizationPickerManager = new OrganizationPicker();
  await organizationPickerManager.skipOrganizationSelection();
  ```
</ParamField>
