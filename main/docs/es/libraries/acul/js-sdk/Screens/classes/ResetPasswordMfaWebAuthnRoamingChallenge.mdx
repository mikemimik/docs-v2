La clase ResetPasswordMfaWebAuthnRoamingChallenge implementa la funcionalidad de la pantalla `reset-password-mfa-webauthn-roaming-challenge`. Esta pantalla solicita al usuario que verifique su identidad mediante una clave de seguridad portátil (como una clave USB FIDO2) como parte del paso de MFA durante un flujo de restablecimiento de contraseña.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del gestor de la pantalla ResetPasswordMfaWebAuthnRoamingChallenge:

```typescript Example
import ResetPasswordMfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/reset-password-mfa-webauthn-roaming-challenge';
const resetPasswordMfaWebAuthnRoamingChallengeManager = new ResetPasswordMfaWebAuthnRoamingChallenge();
try {
  await resetPasswordMfaWebAuthnRoamingChallengeManager.useSecurityKey({
    rememberDevice: true,
  });
} catch (error) {
  if (error instanceof DOMException) {
    await resetPasswordMfaWebAuthnRoamingChallengeManager.showError({
      error: { name: error.name, message: error.message },
    });
  }
}
```

  ```typescript Example

    language: 'fr',
  ```

<ParamField body="useSecurityKey" type="Promise<void>">
  Este método inicia el desafío de clave de seguridad de WebAuthn y envía la credencial resultante para completar el paso de MFA durante el flujo de restablecimiento de contraseña. Si el navegador lanza una `DOMException` (por ejemplo, si el usuario cancela), captura el error y llama a `showError` para notificárselo a Auth0.

  ```typescript Example
  import ResetPasswordMfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/reset-password-mfa-webauthn-roaming-challenge';
  const resetPasswordMfaWebAuthnRoamingChallengeManager = new ResetPasswordMfaWebAuthnRoamingChallenge();
  try {
    await resetPasswordMfaWebAuthnRoamingChallengeManager.useSecurityKey({
      rememberDevice: true,
    });
  } catch (error) {
    if (error instanceof DOMException) {
      await resetPasswordMfaWebAuthnRoamingChallengeManager.showError({
        error: { name: error.name, message: error.message },
      });
    }
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordMfaWebAuthnRoamingChallengeUseSecurityKeyOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaWebAuthnRoamingChallengeUseSecurityKeyOptions).
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Cuando es `true`, marca el dispositivo como de confianza para que no se le vuelva a solicitar MFA al usuario en ese dispositivo.
    </ParamField>
  </Expandable>
</ParamField>
