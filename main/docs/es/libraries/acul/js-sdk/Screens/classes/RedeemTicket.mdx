title: "RedeemTicket"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `redeem-ticket` de Universal Login."
---

La clase RedeemTicket implementa la funcionalidad de la pantalla `redeem-ticket`. Esta pantalla se muestra cuando un usuario necesita canjear un ticket para continuar con el flujo de autenticación.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla RedeemTicket:

```typescript Example
import RedeemTicket from '@auth0/auth0-acul-js/redeem-ticket';
const redeemTicketManager = new RedeemTicket();
await redeemTicketManager.continue();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import RedeemTicket from '@auth0/auth0-acul-js/redeem-ticket';
  const redeemTicketManager = new RedeemTicket();
  redeemTicketManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale correspondiente al idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método realiza la acción predeterminada en la pantalla `redeem-ticket` y continúa el flujo de autenticación.

  ```typescript Example
  import RedeemTicket from '@auth0/auth0-acul-js/redeem-ticket';
  const redeemTicketManager = new RedeemTicket();
  await redeemTicketManager.continue();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no hay ninguno.
</ParamField>
