title: "PhoneIdentifierChallenge"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `phone-identifier-challenge` de Universal Login."
---

La clase PhoneIdentifierChallenge implementa la funcionalidad de la pantalla `phone-identifier-challenge`. En esta pantalla, se solicita al usuario que introduzca un código de verificación enviado a su número de teléfono para confirmar su identidad.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla PhoneIdentifierChallenge:

```typescript Example
import PhoneIdentifierChallenge from '@auth0/auth0-acul-js/phone-identifier-challenge';
const phoneIdentifierChallengeManager = new PhoneIdentifierChallenge();
await phoneIdentifierChallengeManager.submitPhoneChallenge({
  code: '123456',
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import PhoneIdentifierChallenge from '@auth0/auth0-acul-js/phone-identifier-challenge';
  const phoneIdentifierChallengeManager = new PhoneIdentifierChallenge();
  phoneIdentifierChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado persiste durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Este método reenvía el código de verificación al número de teléfono del usuario.

  ```typescript Example
  import PhoneIdentifierChallenge from '@auth0/auth0-acul-js/phone-identifier-challenge';
  const phoneIdentifierChallengeManager = new PhoneIdentifierChallenge();
  await phoneIdentifierChallengeManager.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Este método proporciona la funcionalidad de reenvío con gestión del tiempo de espera. Devuelve un objeto [`ResendControl`](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResendControl) con un método `startResend()` que activa el reenvío y aplica un período de espera antes de que el usuario pueda volver a reenviar.

  ```typescript Example
  import PhoneIdentifierChallenge from '@auth0/auth0-acul-js/phone-identifier-challenge';
  const phoneIdentifierChallengeManager = new PhoneIdentifierChallenge();
  const { startResend } = phoneIdentifierChallengeManager.resendManager({
    timeoutSeconds: 30,
  });
  await startResend();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [StartResendOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La duración del período de espera, en segundos, antes de que el usuario pueda volver a reenviar el código.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Devolución de llamada que se invoca cada vez que cambia el estado del reenvío; devuelve los segundos restantes e indica si la funcionalidad de reenvío está deshabilitada en ese momento.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Devolución de llamada que se invoca cuando el período de espera expira y el reenvío vuelve a estar disponible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="returnToPrevious" type="Promise<void>">
  Este método devuelve al usuario a la pantalla anterior.

  ```typescript Example
  import PhoneIdentifierChallenge from '@auth0/auth0-acul-js/phone-identifier-challenge';
  const phoneIdentifierChallengeManager = new PhoneIdentifierChallenge();
  await phoneIdentifierChallengeManager.returnToPrevious();
  ```
</ParamField>

<ParamField body="submitPhoneChallenge" type="Promise<void>">
  Este método envía el código de verificación del teléfono para completar la verificación.

  ```typescript Example
  import PhoneIdentifierChallenge from '@auth0/auth0-acul-js/phone-identifier-challenge';
  const phoneIdentifierChallengeManager = new PhoneIdentifierChallenge();
  await phoneIdentifierChallengeManager.submitPhoneChallenge({
    code: '123456',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [PhoneChallengeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/PhoneChallengeOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código de verificación enviado al número de teléfono del usuario.
    </ParamField>

    <ParamField body="captcha?" type="string">
      El token de respuesta de CAPTCHA, si CAPTCHA está habilitado para este tenant.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchToText" type="Promise<void>">
  Este método cambia el método de entrega del código de verificación para que se envíe por SMS.

  ```typescript Example
  import PhoneIdentifierChallenge from '@auth0/auth0-acul-js/phone-identifier-challenge';
  const phoneIdentifierChallengeManager = new PhoneIdentifierChallenge();
  await phoneIdentifierChallengeManager.switchToText();
  ```
</ParamField>

<ParamField body="switchToVoice" type="Promise<void>">
  Este método cambia el método de entrega del código de verificación para que se realice mediante una llamada de voz.

  ```typescript Example
  import PhoneIdentifierChallenge from '@auth0/auth0-acul-js/phone-identifier-challenge';
  const phoneIdentifierChallengeManager = new PhoneIdentifierChallenge();
  await phoneIdentifierChallengeManager.switchToVoice();
  ```
</ParamField>
