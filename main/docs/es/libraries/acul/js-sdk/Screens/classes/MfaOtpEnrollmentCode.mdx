---
title: "MfaOtpEnrollmentCode"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `mfa-otp-enrollment-code` de Universal Login."
---

La clase MfaOtpEnrollmentCode implementa la funcionalidad de la pantalla `mfa-otp-enrollment-code`. Esta pantalla se muestra cuando el usuario necesita introducir el código OTP recibido durante la configuración de MFA.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de código para la inscripción de MFA OTP:

```typescript Example
import MfaOtpEnrollmentCode from '@auth0/auth0-acul-js/mfa-otp-enrollment-code';

const mfaOtpEnrollmentCode = new MfaOtpEnrollmentCode();
await mfaOtpEnrollmentCode.continue({
  code: '123456',
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaOtpEnrollmentCode from '@auth0/auth0-acul-js/mfa-otp-enrollment-code';
  const mfaOtpEnrollmentCode = new MfaOtpEnrollmentCode();
  mfaOtpEnrollmentCode.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método continúa el proceso de inscripción en MFA con OTP enviando el código OTP.

  ```typescript Example
  import MfaOtpEnrollmentCode from '@auth0/auth0-acul-js/mfa-otp-enrollment-code';

  const mfaOtpEnrollmentCode = new MfaOtpEnrollmentCode();
  await mfaOtpEnrollmentCode.continue({
    code: '123456',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaOtpContinueOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/MfaOtpContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código OTP introducido por el usuario.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="toggleView" type="Promise<void>">
  Este método alterna entre las vistas de código QR y de entrada manual.

  ```typescript Example
  import MfaOtpEnrollmentCode from '@auth0/auth0-acul-js/mfa-otp-enrollment-code';

  const mfaOtpEnrollmentCode = new MfaOtpEnrollmentCode();
  await mfaOtpEnrollmentCode.toggleView();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario probar otro método de MFA.

  ```typescript Example
  import MfaOtpEnrollmentCode from '@auth0/auth0-acul-js/mfa-otp-enrollment-code';

  const mfaOtpEnrollmentCode = new MfaOtpEnrollmentCode();
  await mfaOtpEnrollmentCode.tryAnotherMethod();
  ```
</ParamField>
