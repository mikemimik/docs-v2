title: "MfaEmailList"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `mfa-email-list` de Universal Login."
---

La clase MfaEmailList implementa la funcionalidad de la pantalla `mfa-email-list`. Esta pantalla permite seleccionar una dirección de correo electrónico registrada para MFA.

</Frame>

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla Lista de correo electrónico de MFA:

```typescript Example
import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

const mfaEmailList = new MfaEmailList();
await mfaEmailList.selectMfaEmail({
  index: 0
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';
  const mfaEmailList = new MfaEmailList();
  mfaEmailList.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene del contexto el arreglo de errores de la transacción, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="goBack" type="Promise<void>">
  Este método vuelve a la pantalla anterior.

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

  const mfaEmailList = new MfaEmailList();
  await mfaEmailList.goBack();
  ```
</ParamField>

<ParamField body="selectMfaEmail" type="Promise<void>">
  Este método selecciona una dirección de correo electrónico registrada de la lista.

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

  const mfaEmailList = new MfaEmailList();
  await mfaEmailList.selectMfaEmail({
    index: 0
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SelectMfaEmailOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/SelectMfaEmailOptions).
    </ParamField>

    <ParamField body="index" type="number" required>
      El índice del correo electrónico que se va a seleccionar.
    </ParamField>
  </Expandable>
</ParamField>
