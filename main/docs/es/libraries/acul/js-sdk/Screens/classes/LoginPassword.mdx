---
title: "LoginPassword"
description: "Describe todas las propiedades y los métodos disponibles para personalizar la pantalla `login-password` de Universal Login."
---

La clase LoginPassword implementa la funcionalidad de la pantalla `login-password`. Esta pantalla recopila el identificador y la contraseña del usuario.

</Frame>

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla LoginPassword:

```typescript Example
import LoginPassword from "@auth0/auth0-acul-js/login-password";
const loginPasswordManager = new LoginPassword();
loginPasswordManager.login({
  username: "testUser",
  password: "testPassword"
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedLogin" type="Promise<void>">
  Este método redirige al usuario al proveedor de identidad social o empresarial (IdP) para autenticarse.

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.federatedLogin({
    connection: "google-oauth2"
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [FederatedLoginPayloadOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/FederatedLoginPayloadOptions).
    </ParamField>

    <ParamField body="connection" type="string" required>
      El nombre de la conexión social que se va a usar.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera del contexto el arreglo de errores de la transacción o, si no hay ninguno, un arreglo vacío. Devuelve un arreglo de objetos de error del contexto de la transacción.
</ParamField>

<ParamField body="login" type="Promise<void>">
  Este método autentica al usuario con el identificador y la contraseña proporcionados.

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.login({
    username: "testUser",
    password: "testPassword"
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LoginPayloadOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LoginPayloadOptions).
    </ParamField>

    <ParamField body="username" type="string" required>
      El identificador del usuario.
    </ParamField>

    <ParamField body="password" type="string" required>
      La contraseña del usuario.
    </ParamField>

    <ParamField body="captcha?" type="string">
      El código de captcha o la respuesta del proveedor de CAPTCHA. Esta propiedad es obligatoria si tu tenant de Auth0 tiene [Bot Detection](/docs/es/secure/attack-protection/bot-detection) habilitado.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  Este método alterna entre una conexión de base de datos (contraseña) y una conexión passwordless (OTP de Email/SMS).

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.switchConnection({
    connection: "email"
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SwitchConnectionOptionsLoginPassword](/docs/es/libraries/acul/js-sdk/Screens/interfaces/SwitchConnectionOptionsLoginPassword).
    </ParamField>

    <ParamField body="connection" type="string" required>
      El nombre de la conexión a la que se cambiará. Usa `'email'` o `'sms'` para passwordless, o el nombre de una conexión de base de datos para la autenticación basada en contraseña.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>
