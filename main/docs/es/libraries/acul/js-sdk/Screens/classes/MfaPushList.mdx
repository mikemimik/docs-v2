<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de lista de MFA Push:

```typescript Example
import MfaPushList from '@auth0/auth0-acul-js/mfa-push-list';

const mfaPushList = new MfaPushList();
await mfaPushList.selectMfaPushDevice({ deviceIndex: 0 });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaPushList from '@auth0/auth0-acul-js/mfa-push-list';
  const mfaPushList = new MfaPushList();
  mfaPushList.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no hay ninguno.
</ParamField>

<ParamField body="goBack" type="Promise<void>">
  Este método vuelve a la pantalla anterior.

  ```typescript Example
  import MfaPushList from '@auth0/auth0-acul-js/mfa-push-list';

  const mfaPushList = new MfaPushList();
  await mfaPushList.goBack();
  ```
</ParamField>

<ParamField body="selectMfaPushDevice" type="Promise<void>">
  Este método selecciona un dispositivo registrado de la lista para iniciar MFA push.

  ```typescript Example
  import MfaPushList from '@auth0/auth0-acul-js/mfa-push-list';

  const mfaPushList = new MfaPushList();
  await mfaPushList.selectMfaPushDevice({ deviceIndex: 0 });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SelectMfaPushDeviceOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/SelectMfaPushDeviceOptions).
    </ParamField>

    <ParamField body="deviceIndex" type="number" required>
      El índice del dispositivo que se va a seleccionar de la lista de dispositivos registrados. El índice empieza en 0.
    </ParamField>
  </Expandable>
</ParamField>
