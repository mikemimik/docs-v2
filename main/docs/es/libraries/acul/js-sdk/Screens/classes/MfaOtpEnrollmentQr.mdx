title: "MfaOtpEnrollmentQr"

---

La clase MfaOtpEnrollmentQr implementa la funcionalidad de la pantalla `mfa-otp-enrollment-qr`. Esta pantalla muestra un código QR para que el usuario lo escanee con su aplicación de autenticación durante la inscripción de MFA.

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla QR de inscripción de MFA OTP:

```typescript Example
import MfaOtpEnrollmentQr from '@auth0/auth0-acul-js/mfa-otp-enrollment-qr';

const mfaOtpEnrollmentQr = new MfaOtpEnrollmentQr();
await mfaOtpEnrollmentQr.toggleView();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaOtpEnrollmentQr from '@auth0/auth0-acul-js/mfa-otp-enrollment-qr';
  const mfaOtpEnrollmentQr = new MfaOtpEnrollmentQr();
  mfaOtpEnrollmentQr.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método envía el código OTP para continuar el proceso de inscripción.

  ```typescript Example
  import MfaOtpEnrollmentQr from '@auth0/auth0-acul-js/mfa-otp-enrollment-qr';

  const mfaOtpEnrollmentQr = new MfaOtpEnrollmentQr();
  await mfaOtpEnrollmentQr.continue({ code: '123456' });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaOtpEnrollmentQrContinueOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/MfaOtpEnrollmentQrContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código OTP introducido por el usuario.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no hay ninguno.
</ParamField>

<ParamField body="toggleView" type="Promise<void>">
  Este método cambia entre las vistas de código QR y entrada manual.

  ```typescript Example
  import MfaOtpEnrollmentQr from '@auth0/auth0-acul-js/mfa-otp-enrollment-qr';

  const mfaOtpEnrollmentQr = new MfaOtpEnrollmentQr();
  await mfaOtpEnrollmentQr.toggleView();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario usar otro método de MFA.

  ```typescript Example
  import MfaOtpEnrollmentQr from '@auth0/auth0-acul-js/mfa-otp-enrollment-qr';

  const mfaOtpEnrollmentQr = new MfaOtpEnrollmentQr();
  await mfaOtpEnrollmentQr.tryAnotherMethod();
  ```
</ParamField>
