<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de bienvenida de MFA Push:

```typescript Example
import MfaPushWelcome from '@auth0/auth0-acul-js/mfa-push-welcome';

const mfaPushWelcome = new MfaPushWelcome();
await mfaPushWelcome.enroll();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaPushWelcome from '@auth0/auth0-acul-js/mfa-push-welcome';
  const mfaPushWelcome = new MfaPushWelcome();
  mfaPushWelcome.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="enroll" type="Promise<void>">
  Este método navega a la pantalla de inscripción.

  ```typescript Example
  import MfaPushWelcome from '@auth0/auth0-acul-js/mfa-push-welcome';

  const mfaPushWelcome = new MfaPushWelcome();
  await mfaPushWelcome.enroll();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="pickAuthenticator" type="Promise<void>">
  Este método navega a la pantalla de selección de autenticador.

  ```typescript Example
  import MfaPushWelcome from '@auth0/auth0-acul-js/mfa-push-welcome';

  const mfaPushWelcome = new MfaPushWelcome();
  await mfaPushWelcome.pickAuthenticator();
  ```
</ParamField>
