La clase MfaDetectBrowserCapabilities implementa la funcionalidad de la pantalla `mfa-detect-browser-capabilities`. Esta pantalla detecta las capacidades del navegador para los métodos de autenticación MFA.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de pantalla MFA Detect Browser Capabilities:

```typescript Example
import MfaDetectBrowserCapabilities from '@auth0/auth0-acul-js/mfa-detect-browser-capabilities';
const mfaDetectBrowserCapabilities = new MfaDetectBrowserCapabilities();
await mfaDetectBrowserCapabilities.detectCapabilities();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaDetectBrowserCapabilities from '@auth0/auth0-acul-js/mfa-detect-browser-capabilities';
  const mfaDetectBrowserCapabilities = new MfaDetectBrowserCapabilities();
  mfaDetectBrowserCapabilities.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="detectCapabilities" type="Promise<void>">
  Este método selecciona un autenticador en función de las capacidades del navegador.

  ```typescript Example
  import MfaDetectBrowserCapabilities from '@auth0/auth0-acul-js/mfa-detect-browser-capabilities';
  const mfaDetectBrowserCapabilities = new MfaDetectBrowserCapabilities();
  await mfaDetectBrowserCapabilities.detectCapabilities();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>
