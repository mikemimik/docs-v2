La clase EmailOTPChallenge implementa la funcionalidad de la pantalla `email-otp-challenge`. En esta pantalla, se solicita al usuario que introduzca un código de un solo uso enviado a su dirección de correo electrónico.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla EmailOTPChallenge:

```typescript Example
import EmailOTPChallenge from '@auth0/auth0-acul-js/email-otp-challenge';
const emailOTPChallengeManager = new EmailOTPChallenge();
await emailOTPChallengeManager.submitCode({ code: '123456' });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import EmailOTPChallenge from '@auth0/auth0-acul-js/email-otp-challenge';
  const emailOTPChallengeManager = new EmailOTPChallenge();
  emailOTPChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción del contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Este método solicita que se envíe un nuevo código OTP al correo electrónico del usuario.

  ```typescript Example
  import EmailOTPChallenge from '@auth0/auth0-acul-js/email-otp-challenge';
  const emailOTPChallengeManager = new EmailOTPChallenge();
  await emailOTPChallengeManager.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Este método proporciona la funcionalidad de reenvío con gestión del tiempo de espera. Devuelve un objeto [`ResendControl`](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResendControl) con un método `startResend()` que activa el reenvío y aplica un período de espera antes de que el usuario pueda volver a reenviar.

  ```typescript Example
  import EmailOTPChallenge from '@auth0/auth0-acul-js/email-otp-challenge';
  const emailOTPChallengeManager = new EmailOTPChallenge();
  const { startResend } = emailOTPChallengeManager.resendManager({
    timeoutSeconds: 15,
  });
  await startResend();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [StartResendOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La duración del período de espera, en segundos, antes de que el usuario pueda reenviar el código otra vez.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Devolución de llamada que se invoca cada vez que cambia el estado del reenvío; devuelve los segundos restantes y si la funcionalidad de reenvío está deshabilitada en ese momento.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Devolución de llamada que se invoca cuando expira el período de espera y el reenvío vuelve a estar disponible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="submitCode" type="Promise<void>">
  Este método envía el código OTP introducido por el usuario.

  ```typescript Example
  import EmailOTPChallenge from '@auth0/auth0-acul-js/email-otp-challenge';
  const emailOTPChallengeManager = new EmailOTPChallenge();
  await emailOTPChallengeManager.submitCode({ code: '123456' });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [OtpCodeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/OtpCodeOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código OTP que introduce el usuario para enviarlo.
    </ParamField>
  </Expandable>
</ParamField>
