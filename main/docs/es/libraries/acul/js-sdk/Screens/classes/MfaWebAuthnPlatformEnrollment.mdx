---

La clase MfaWebAuthnPlatformEnrollment implementa la funcionalidad de la pantalla `mfa-webauthn-platform-enrollment`. Esta pantalla se muestra cuando el usuario necesita registrar un autenticador de plataforma, como Touch ID o Face ID.

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla de inscripción en MFA con WebAuthn de plataforma:

```typescript Example
import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
await mfaWebAuthnPlatformEnrollment.submitPasskeyCredential();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  mfaWebAuthnPlatformEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="refuseEnrollmentOnThisDevice" type="Promise<void>">
  Este método permite al usuario rechazar la inscripción en WebAuthn de plataforma en el dispositivo actual.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  await mfaWebAuthnPlatformEnrollment.refuseEnrollmentOnThisDevice();
  ```
</ParamField>

<ParamField body="reportBrowserError" type="Promise<void>">
  Este método informa un error del navegador detectado durante la operación de WebAuthn `navigator.credentials.create()`.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  await mfaWebAuthnPlatformEnrollment.reportBrowserError({
    error: { name: 'NotAllowedError', message: 'The operation was not allowed.' },
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [CustomOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      El objeto de error de la API de WebAuthn (`navigator.credentials.create()`) que se debe informar.

      <ParamField body="name" type="string" required>
        El nombre del error (p. ej., `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        El mensaje de error que describe el problema.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="snoozeEnrollment" type="Promise<void>">
  Este método permite al usuario posponer la inscripción en WebAuthn de plataforma.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  await mfaWebAuthnPlatformEnrollment.snoozeEnrollment();
  ```
</ParamField>

<ParamField body="submitPasskeyCredential" type="Promise<void>">
  Este método inicia la creación de la credencial de WebAuthn de plataforma y envía la credencial resultante al servidor.

  ```typescript Example
  import MfaWebAuthnPlatformEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-platform-enrollment';

  const mfaWebAuthnPlatformEnrollment = new MfaWebAuthnPlatformEnrollment();
  await mfaWebAuthnPlatformEnrollment.submitPasskeyCredential();
  ```
</ParamField>
