title: "MfaVoiceChallenge"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `mfa-voice-challenge` de Universal Login."
---

La clase MfaVoiceChallenge implementa la funcionalidad de la pantalla `mfa-voice-challenge`. Esta pantalla se muestra cuando el usuario debe introducir un código recibido por llamada de voz para verificar su identidad.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de desafío de voz de MFA:

```typescript Example
import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

const mfaVoiceChallenge = new MfaVoiceChallenge();
await mfaVoiceChallenge.continue({ code: '123456' });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  mfaVoiceChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método envía el código de verificación por voz para validar el desafío MFA.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.continue({
    code: '123456',
    rememberDevice: true,
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaVoiceChallengeContinueOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/MfaVoiceChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código de verificación recibido por llamada de voz.
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Cuando es `true`, recuerda el navegador para futuros desafíos MFA. Solo se aplica si `screen.showRememberDevice` es `true`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no hay ninguno.
</ParamField>

<ParamField body="pickPhone" type="Promise<void>">
  Este método lleva a la pantalla donde el usuario puede seleccionar un número de teléfono diferente.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.pickPhone();
  ```
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Este método solicita una nueva llamada de voz con un código de verificación.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Este método proporciona la funcionalidad de reenvío con gestión configurable del tiempo de espera y del estado.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  const { startResend } = mfaVoiceChallenge.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    },
  });

  startResend();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [StartResendOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Función que se invoca cada vez que cambia el estado del reenvío y recibe los segundos restantes y si el reenvío está deshabilitado en ese momento.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Función que se invoca cuando expira el tiempo de espera del reenvío y la opción de reenvío vuelve a estar disponible.
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La cantidad de segundos que deben pasar antes de que la opción de reenvío esté disponible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  Este método cambia el método de verificación a SMS.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.switchToSms();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario usar otro método de MFA.

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.tryAnotherMethod();
  ```
</ParamField>
