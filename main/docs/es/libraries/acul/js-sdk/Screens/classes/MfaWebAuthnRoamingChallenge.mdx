---

La clase MfaWebAuthnRoamingChallenge implementa la funcionalidad de la pantalla `mfa-webauthn-roaming-challenge`. Esta pantalla se muestra cuando el usuario necesita verificar su identidad mediante un autenticador itinerante, como una clave de seguridad.

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla de desafío itinerante de MFA WebAuthn:

```typescript Example
import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
await mfaWebAuthnRoamingChallenge.verify();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

  const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
  mfaWebAuthnRoamingChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="reportWebAuthnError" type="Promise<void>">
  Este método informa un error de la API de WebAuthn detectado durante la operación `navigator.credentials.get()`.

  ```typescript Example
  import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

  const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
  await mfaWebAuthnRoamingChallenge.reportWebAuthnError({
    error: { name: 'NotAllowedError', message: 'The operation was not allowed.' },
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaWebAuthnRoamingChallengeReportErrorOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/MfaWebAuthnRoamingChallengeReportErrorOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      El objeto de error de la API de WebAuthn (`navigator.credentials.get()`) que se va a informar.

      <ParamField body="name" type="string" required>
        El nombre del error (p. ej., `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        El mensaje de error que describe qué salió mal.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario probar otro método de MFA.

  ```typescript Example
  import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

  const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
  await mfaWebAuthnRoamingChallenge.tryAnotherMethod();
  ```
</ParamField>

<ParamField body="verify" type="Promise<void>">
  Este método inicia el desafío de WebAuthn con la clave de seguridad.

  ```typescript Example
  import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

  const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
  await mfaWebAuthnRoamingChallenge.verify();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [CustomOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Cuando es `true`, recuerda el navegador para futuros desafíos MFA. Solo se aplica si `screen.showRememberDevice` es `true`.
    </ParamField>
  </Expandable>
</ParamField>
