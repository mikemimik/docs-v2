La clase MfaBeginEnrollOptions implementa la funcionalidad de la pantalla `mfa-begin-enroll-options`. Esta pantalla permite seleccionar e inscribir factores de MFA.

</Frame>

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de pantalla de opciones para iniciar la inscripción en MFA:

```typescript Example
import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';
const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
await mfaBeginEnrollOptions.enroll({
  action: 'push-notification'
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';
  const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
  mfaBeginEnrollOptions.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="enroll" type="Promise<void>">
  Este método continúa el proceso de inscripción con el factor seleccionado.

  ```typescript Example
  import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';

  const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
  await mfaBeginEnrollOptions.enroll({
    action: 'push-notification'
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaEnrollOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/MfaEnrollOptions).
    </ParamField>

    <ParamField body="action" type="&#x22;push-notification&#x22; | &#x22;otp&#x22; | &#x22;sms&#x22; | &#x22;phone&#x22; | &#x22;voice&#x22; | &#x22;webauthn-roaming&#x22;" required>
      La acción que indica qué factor se va a inscribir. Debe ser una de estas opciones: &quot;push-notification&quot;, &quot;otp&quot;, &quot;sms&quot;, &quot;phone&quot;, &quot;voice&quot; o &quot;webauthn-roaming&quot;.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera del contexto el arreglo de errores de la transacción, o un arreglo vacío si no existe ninguno.
</ParamField>
