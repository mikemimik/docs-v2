---
title: "MfaSmsList"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `mfa-sms-list` de Universal Login."
---

La clase MfaSmsList implementa la funcionalidad de la pantalla `mfa-sms-list`. Esta pantalla se muestra cuando el usuario necesita seleccionar un número de teléfono ya registrado para MFA por SMS.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del gestor de la pantalla de lista de SMS de MFA:

```typescript Example
import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

const mfaSmsList = new MfaSmsList();
await mfaSmsList.selectPhoneNumber({ index: 0 });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="backAction" type="Promise<void>">
  Este método vuelve a la pantalla anterior.

  ```typescript Example
  import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

  const mfaSmsList = new MfaSmsList();
  await mfaSmsList.backAction();
  ```
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

  const mfaSmsList = new MfaSmsList();
  mfaSmsList.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de transacción del contexto, o un arreglo vacío si no hay ninguno.
</ParamField>

<ParamField body="selectPhoneNumber" type="Promise<void>">
  Este método selecciona un número de teléfono de la lista de números de teléfono registrados.

  ```typescript Example
  import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

  const mfaSmsList = new MfaSmsList();
  await mfaSmsList.selectPhoneNumber({ index: 0 });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaSmsListOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/MfaSmsListOptions).
    </ParamField>

    <ParamField body="index" type="number" required>
      El índice del número de teléfono que se va a seleccionar.
    </ParamField>
  </Expandable>
</ParamField>
