La clase ResetPasswordMfaPhoneChallenge implementa la funcionalidad de la pantalla `reset-password-mfa-phone-challenge`. Esta pantalla permite al usuario solicitar un código de verificación por SMS o mediante una llamada de voz como parte del paso de MFA del flujo de restablecimiento de contraseña.

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import ResetPasswordMfaPhoneChallenge from '@auth0/auth0-acul-js/reset-password-mfa-phone-challenge';
  const resetPasswordMfaPhoneChallengeManager = new ResetPasswordMfaPhoneChallenge();
  resetPasswordMfaPhoneChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método envía un código de verificación al teléfono del usuario mediante el método de entrega seleccionado para continuar el desafío MFA durante el flujo de restablecimiento de contraseña.

  ```typescript Example
  import ResetPasswordMfaPhoneChallenge from '@auth0/auth0-acul-js/reset-password-mfa-phone-challenge';
  const resetPasswordMfaPhoneChallengeManager = new ResetPasswordMfaPhoneChallenge();
  await resetPasswordMfaPhoneChallengeManager.continue({
    type: 'sms',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordMfaPhoneChallengeContinueOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaPhoneChallengeContinueOptions).
    </ParamField>

    <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;" required>
      El método de entrega del código de verificación. Use `sms` para enviarlo por mensaje de texto o `voice` para enviarlo mediante una llamada de voz.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no hay ninguno.
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario seleccionar otro método de MFA para completar el desafío.

  ```typescript Example
  import ResetPasswordMfaPhoneChallenge from '@auth0/auth0-acul-js/reset-password-mfa-phone-challenge';
  const resetPasswordMfaPhoneChallengeManager = new ResetPasswordMfaPhoneChallenge();
  await resetPasswordMfaPhoneChallengeManager.tryAnotherMethod({
    type: 'sms',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordMfaPhoneChallengeTryAnotherMethodOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaPhoneChallengeTryAnotherMethodOptions).
    </ParamField>

    <ParamField body="type?" type="&#x22;sms&#x22; | &#x22;voice&#x22;">
      El método de entrega que se mostró en la pantalla actual. La API lo requiere para procesar correctamente el cambio de método.
    </ParamField>
  </Expandable>
</ParamField>
