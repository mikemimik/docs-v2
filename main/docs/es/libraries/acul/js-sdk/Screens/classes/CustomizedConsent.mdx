---
title: "CustomizedConsent"
description: "Describe todas las propiedades y los métodos disponibles para personalizar la pantalla `customized-consent` de Universal Login."
---

La clase CustomizedConsent implementa la funcionalidad de la pantalla `customized-consent`. Esta pantalla le pide al usuario que acepte o rechace los permisos y los detalles de autorización solicitados por una aplicación cliente.

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla CustomizedConsent:

```typescript Example
import CustomizedConsent from '@auth0/auth0-acul-js/customized-consent';
const customizedConsentManager = new CustomizedConsent();
await customizedConsentManager.accept();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="accept" type="Promise<void>">
  Este método envía la decisión del usuario de aceptar los permisos solicitados y los detalles de autorización.

  ```typescript Example
  import CustomizedConsent from '@auth0/auth0-acul-js/customized-consent';
  const customizedConsentManager = new CustomizedConsent();
  await customizedConsentManager.accept();
  ```
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import CustomizedConsent from '@auth0/auth0-acul-js/customized-consent';
  const customizedConsentManager = new CustomizedConsent();
  customizedConsentManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="deny" type="Promise<void>">
  Este método envía la decisión del usuario de rechazar los permisos solicitados y los detalles de autorización.

  ```typescript Example
  import CustomizedConsent from '@auth0/auth0-acul-js/customized-consent';
  const customizedConsentManager = new CustomizedConsent();
  await customizedConsentManager.deny();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera del contexto el arreglo de errores de la transacción, o un arreglo vacío si no existe ninguno.
</ParamField>
