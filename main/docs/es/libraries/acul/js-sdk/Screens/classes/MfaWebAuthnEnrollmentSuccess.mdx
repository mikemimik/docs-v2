La clase MfaWebAuthnEnrollmentSuccess implementa la funcionalidad de la pantalla `mfa-webauthn-enrollment-success`. Esta pantalla se muestra cuando el usuario ha registrado correctamente una clave de seguridad de WebAuthn.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de éxito del registro WebAuthn de MFA:

```typescript Example
import MfaWebAuthnEnrollmentSuccess from '@auth0/auth0-acul-js/mfa-webauthn-enrollment-success';

const mfaWebAuthnEnrollmentSuccess = new MfaWebAuthnEnrollmentSuccess();
await mfaWebAuthnEnrollmentSuccess.continue();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import MfaWebAuthnEnrollmentSuccess from '@auth0/auth0-acul-js/mfa-webauthn-enrollment-success';

  const mfaWebAuthnEnrollmentSuccess = new MfaWebAuthnEnrollmentSuccess();
  mfaWebAuthnEnrollmentSuccess.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método continúa el flujo de autenticación después de un registro WebAuthn exitoso.

  ```typescript Example
  import MfaWebAuthnEnrollmentSuccess from '@auth0/auth0-acul-js/mfa-webauthn-enrollment-success';

  const mfaWebAuthnEnrollmentSuccess = new MfaWebAuthnEnrollmentSuccess();
  await mfaWebAuthnEnrollmentSuccess.continue();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción del contexto, o un arreglo vacío si no existe ninguno.
</ParamField>
