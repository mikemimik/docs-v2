title: "SignupPassword"
description: "Describe todas las propiedades y los métodos disponibles para personalizar la pantalla `signup-password` de Universal Login."
---

La clase SignupPassword implementa la funcionalidad de la pantalla `signup-password`. Esta pantalla recopila la contraseña del usuario.

</Frame>

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla SignupPassword:

```typescript Example
import SignupPassword from '@auth0/auth0-acul-js/signup-password';
const signupPasswordManager = new SignupPassword();
signupPasswordManager.signup({
  email: 'test@example.com',
  password: 'P@$wOrd123!',
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  signupPasswordManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedSignup" type="Promise<void>">
  Este método permite registrarse mediante distintos identificadores sociales. Por ejemplo: Google, Facebook, etc.

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  signupPasswordManager.federatedSignup({
    connection: 'google-oauth2',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [FederatedSignupPasswordPayloadOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/FederatedSignupPasswordPayloadOptions).
    </ParamField>

    <ParamField body="connection" type="string" required>
      El nombre de la conexión social que se va a usar.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno. Devuelve un arreglo de objetos de error del contexto de la transacción.
</ParamField>

<ParamField body="signup" type="Promise<void>">
  Este método gestiona el envío del formulario de contraseña de registro.

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  signupPasswordManager.signup({
    email: 'test@example.com',
    password: 'P@$wOrd123!',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SignupPasswordOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/SignupPasswordOptions).
    </ParamField>

    <ParamField body="email?" type="string">
      El correo electrónico del usuario.
    </ParamField>

    <ParamField body="username?" type="string">
      El identificador del usuario.
    </ParamField>

    <ParamField body="phoneNumber?" type="string">
      El número de teléfono del usuario.
    </ParamField>

    <ParamField body="password" type="string" required>
      La contraseña del usuario.
    </ParamField>

    <ParamField body="captcha?" type="string">
      El código captcha o la respuesta del proveedor de CAPTCHA. Esta propiedad es obligatoria si tu tenant de Auth0 tiene [Bot Detection](/docs/es/secure/attack-protection/bot-detection) habilitado.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  Este método cambia la conexión de autenticación durante el flujo signup-password.

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  signupPasswordManager.switchConnection({
    connection: 'Username-Password-Authentication',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SwitchConnectionOptionsSignupPassword](/docs/es/libraries/acul/js-sdk/Screens/interfaces/SwitchConnectionOptionsSignupPassword).
    </ParamField>

    <ParamField body="connection" type="string" required>
      El identificador de la conexión a la que se cambiará durante el flujo de autenticación signup-password.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="validatePassword" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/PasswordValidationResult">PasswordValidationResult</a></span>}>
  Este método valida una contraseña con respecto a la política de contraseñas de la transacción actual. Devuelve un objeto que indica si la contraseña es válida y el motivo.

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  const result = signupPasswordManager.validatePassword('MyP@ssw0rd!');
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="password" type="string" required>
      La contraseña que se va a validar.
    </ParamField>
  </Expandable>
</ParamField>
