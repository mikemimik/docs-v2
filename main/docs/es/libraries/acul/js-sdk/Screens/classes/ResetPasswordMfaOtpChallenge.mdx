La clase ResetPasswordMfaOtpChallenge implementa la funcionalidad de la pantalla `reset-password-mfa-otp-challenge`. Esta pantalla solicita al usuario que introduzca la contraseña de un solo uso (OTP) de su aplicación de autenticación como parte del paso de MFA durante un flujo de restablecimiento de contraseña.

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import ResetPasswordMfaOtpChallenge from '@auth0/auth0-acul-js/reset-password-mfa-otp-challenge';
  const resetPasswordMfaOtpChallengeManager = new ResetPasswordMfaOtpChallenge();
  resetPasswordMfaOtpChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece como `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  Este método envía el código OTP para continuar el desafío MFA durante el flujo de restablecimiento de contraseña.

  ```typescript Example
  import ResetPasswordMfaOtpChallenge from '@auth0/auth0-acul-js/reset-password-mfa-otp-challenge';
  const resetPasswordMfaOtpChallengeManager = new ResetPasswordMfaOtpChallenge();
  await resetPasswordMfaOtpChallengeManager.continue({
    code: '123456',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordMfaOtpChallengeContinueOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaOtpChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código OTP de la aplicación de autenticación del usuario.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no hay ninguno.
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario seleccionar otro método MFA para completar el desafío.

  ```typescript Example
  import ResetPasswordMfaOtpChallenge from '@auth0/auth0-acul-js/reset-password-mfa-otp-challenge';
  const resetPasswordMfaOtpChallengeManager = new ResetPasswordMfaOtpChallenge();
  await resetPasswordMfaOtpChallengeManager.tryAnotherMethod();
  ```
</ParamField>
