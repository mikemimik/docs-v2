La clase ResetPasswordMfaWebAuthnPlatformChallenge implementa la funcionalidad de la pantalla `reset-password-mfa-webauthn-platform-challenge`. Esta pantalla solicita al usuario que verifique su identidad mediante un autenticador de plataforma (como Touch ID o Windows Hello) como parte del paso de MFA durante un flujo de restablecimiento de contraseña.

  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import ResetPasswordMfaWebAuthnPlatformChallenge from '@auth0/auth0-acul-js/reset-password-mfa-webauthn-platform-challenge';
  const resetPasswordMfaWebAuthnPlatformChallengeManager = new ResetPasswordMfaWebAuthnPlatformChallenge();
  resetPasswordMfaWebAuthnPlatformChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueWithPasskey" type="Promise<void>">
  Este método inicia el desafío del autenticador de plataforma de WebAuthn y envía la credencial resultante para completar el paso de MFA durante el flujo de restablecimiento de contraseña. Si el navegador lanza una `DOMException` (como cuando el usuario cancela), captura el error y llama a `reportBrowserError` para notificar a Auth0.

  ```typescript Example
  import ResetPasswordMfaWebAuthnPlatformChallenge from '@auth0/auth0-acul-js/reset-password-mfa-webauthn-platform-challenge';
  const resetPasswordMfaWebAuthnPlatformChallengeManager = new ResetPasswordMfaWebAuthnPlatformChallenge();
  try {
    await resetPasswordMfaWebAuthnPlatformChallengeManager.continueWithPasskey({
      rememberDevice: true,
    });
  } catch (error) {
    if (error instanceof DOMException) {
      await resetPasswordMfaWebAuthnPlatformChallengeManager.reportBrowserError({
        error: { name: error.name, message: error.message },
      });
    }
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordMfaWebAuthnPlatformChallengeContinueOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaWebAuthnPlatformChallengeContinueOptions).
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Cuando es `true`, marca el dispositivo como de confianza para que no se le vuelva a pedir MFA al usuario en ese dispositivo.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="reportBrowserError" type="Promise<void>">
  Este método informa a Auth0 sobre un error de WebAuthn del lado del navegador, como la cancelación del usuario (`NotAllowedError`) o un timeout de `navigator.credentials.get()`.

  ```typescript Example
  import ResetPasswordMfaWebAuthnPlatformChallenge from '@auth0/auth0-acul-js/reset-password-mfa-webauthn-platform-challenge';
  const resetPasswordMfaWebAuthnPlatformChallengeManager = new ResetPasswordMfaWebAuthnPlatformChallenge();
  await resetPasswordMfaWebAuthnPlatformChallengeManager.reportBrowserError({
    error: { name: error.name, message: error.message },
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordMfaWebAuthnPlatformChallengeReportErrorOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordMfaWebAuthnPlatformChallengeReportErrorOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      El objeto de error de la API de WebAuthn (`navigator.credentials.get()`) que se va a reportar.

      <ParamField body="name" type="string" required>
        El nombre del error (por ejemplo, `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        El mensaje de error que describe qué salió mal.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>
