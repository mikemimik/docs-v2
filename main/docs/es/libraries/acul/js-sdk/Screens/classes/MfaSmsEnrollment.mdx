title: "MfaSmsEnrollment"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `mfa-sms-enrollment` de Universal Login."
---

La clase MfaSmsEnrollment implementa la funcionalidad de la pantalla `mfa-sms-enrollment`. Esta pantalla se muestra cuando el usuario necesita registrar un número de teléfono para MFA por SMS.

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla de inscripción de MFA por SMS:

```typescript Example
import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

const mfaSmsEnrollment = new MfaSmsEnrollment();
await mfaSmsEnrollment.continueEnrollment({ phone: '1234567890' });
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  mfaSmsEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueEnrollment" type="Promise<void>">
  Este método continúa el proceso de inscripción por SMS con el número de teléfono proporcionado.

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.continueEnrollment({ phone: '1234567890' });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaSmsEnrollmentOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/MfaSmsEnrollmentOptions).
    </ParamField>

    <ParamField body="captcha?" type="string">
      El código o la respuesta del CAPTCHA provider. Es obligatorio si tu tenant de Auth0 tiene Bot Detection habilitado.
    </ParamField>

    <ParamField body="phone?" type="string">
      El número de teléfono del usuario.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  Este método navega a la screen donde el usuario puede seleccionar un country code para su número de teléfono.

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.pickCountryCode();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método permite al usuario probar con otro MFA method.

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.tryAnotherMethod();
  ```
</ParamField>
