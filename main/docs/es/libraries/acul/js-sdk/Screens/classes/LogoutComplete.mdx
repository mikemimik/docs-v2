La clase LogoutComplete implementa la funcionalidad de la pantalla `logout-complete`. Esta pantalla se muestra cuando un usuario completa el cierre de sesión.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del gestor de pantalla LogoutComplete:

```typescript Example
import LogoutComplete from '@auth0/auth0-acul-js/logout-complete';
const logoutCompleteManager = new LogoutComplete();
logoutCompleteManager.changeLanguage({
  language: 'fr',
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import LogoutComplete from '@auth0/auth0-acul-js/logout-complete';
  const logoutCompleteManager = new LogoutComplete();
  logoutCompleteManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno. Devuelve un arreglo de objetos de error del contexto de la transacción.
</ParamField>
