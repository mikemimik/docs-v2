La clase ResetPasswordMfaSmsChallenge implementa la funcionalidad de la pantalla `reset-password-mfa-sms-challenge`. Esta pantalla solicita al usuario que introduzca el código de verificación enviado a su teléfono por SMS como parte del paso de MFA durante un flujo de restablecimiento de contraseña.

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import ResetPasswordMfaSmsChallenge from '@auth0/auth0-acul-js/reset-password-mfa-sms-challenge';
  const resetPasswordMfaSmsChallengeManager = new ResetPasswordMfaSmsChallenge();
  resetPasswordMfaSmsChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

  ```typescript Example

  const resetPasswordMfaSmsChallengeManager = new ResetPasswordMfaSmsChallenge();
  await resetPasswordMfaSmsChallengeManager.continueMfaSmsChallenge({
    code: '123456',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">

    </ParamField>

    <ParamField body="code" type="string" required>
      El código de verificación por SMS introducido por el usuario.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Este método reenvía el código de verificación por SMS al teléfono del usuario.

  ```typescript Example
  import ResetPasswordMfaSmsChallenge from '@auth0/auth0-acul-js/reset-password-mfa-sms-challenge';
  const resetPasswordMfaSmsChallengeManager = new ResetPasswordMfaSmsChallenge();
  await resetPasswordMfaSmsChallengeManager.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  Este método proporciona funcionalidad de reenvío con gestión del tiempo de espera. Devuelve un objeto [`ResendControl`](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ResendControl) con un método `startResend()` que activa el reenvío y aplica un período de espera antes de que el usuario pueda volver a reenviar.

  ```typescript Example
  import ResetPasswordMfaSmsChallenge from '@auth0/auth0-acul-js/reset-password-mfa-sms-challenge';
  const resetPasswordMfaSmsChallengeManager = new ResetPasswordMfaSmsChallenge();
  const { startResend } = resetPasswordMfaSmsChallengeManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // Llama a startResend cuando el usuario hace clic en el botón de reenvío
  startResend();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [StartResendOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions).
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      La duración, en segundos, del período de espera antes de que el usuario pueda volver a reenviar el código.
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      Función de devolución de llamada que se invoca cada vez que cambia el estado del reenvío; devuelve los segundos restantes e indica si la funcionalidad de reenvío está deshabilitada en ese momento.
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      Función de devolución de llamada que se invoca cuando finaliza el período de espera y el reenvío vuelve a estar disponible.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
