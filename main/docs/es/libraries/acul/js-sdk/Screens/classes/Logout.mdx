title: "Cierre de sesión"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `logout` de Universal Login."
---

La clase Logout implementa la funcionalidad de la pantalla `logout`. Esta pantalla permite a los usuarios confirmar o rechazar la acción de cierre de sesión.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del gestor de la pantalla de cierre de sesión:

```typescript Example
import Logout from '@auth0/auth0-acul-js/logout';
const logoutManager = new Logout();
logoutManager.confirmLogout({
  action: 'accept',
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import Logout from '@auth0/auth0-acul-js/logout';
  const logoutManager = new Logout();
  logoutManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de idioma del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="confirmLogout" type="Promise<void>">
  Este método confirma la acción de cierre de sesión (aceptar o denegar).

  ```typescript Example
  import Logout from '@auth0/auth0-acul-js/logout';
  const logoutManager = new Logout();
  logoutManager.confirmLogout({
    action: 'accept',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ConfirmLogoutOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/ConfirmLogoutOptions).
    </ParamField>

    <ParamField body="action" type="&#x22;accept&#x22; | &#x22;deny&#x22;" required>
      El estado de la transacción de autenticación.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera del contexto el arreglo de errores de la transacción, o un arreglo vacío si no existe ninguno. Devuelve un arreglo de objetos de error del contexto de la transacción.
</ParamField>
