---
title: "Login"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `login` de Universal Login."
---

La clase Login implementa la funcionalidad de la pantalla `login`. Esta pantalla recopila el identificador y la contraseña del usuario. Según tu tenant, este identificador puede ser un correo electrónico, un número de teléfono o un nombre de usuario.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de inicio de sesión:

```typescript Example
import Login from "@auth0/auth0-acul-js/login";
const loginManager = new Login();
loginManager.login({
  username: "testUser",
  password: "testPassword"
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de la página de Universal Login.

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Si se establece en `'session'`, el idioma seleccionado se mantiene durante la sesión.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedLogin" type="Promise<void>">
  Este método redirige al usuario al proveedor de identidad social o empresarial (IdP) para autenticarse.

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.federatedLogin({
    connection: "google-oauth2"
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [FederatedLoginPayloadOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/FederatedLoginPayloadOptions).
    </ParamField>

    <ParamField body="connection" type="string" required>
      El nombre de la conexión social que se va a usar.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera del contexto el arreglo de errores de la transacción, o un arreglo vacío si no existe ninguno. Un arreglo de objetos de error del contexto de la transacción.
</ParamField>

<ParamField body="getLoginIdentifiers" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a>[] | null</span>}>
  Este método devuelve los tipos de identificador activos de la pantalla `login`.

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.getLoginIdentifiers();
  ```
</ParamField>

<ParamField body="login" type="Promise<void>">
  Este método autentica al usuario mediante el identificador y la contraseña proporcionada.

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.login({
    username: "testUser",
    password: "testPassword"
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LoginPayloadOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LoginPayloadOptions).
    </ParamField>

    <ParamField body="username" type="string" required>
      El identificador del usuario.
    </ParamField>

    <ParamField body="password" type="string" required>
      La contraseña del usuario.
    </ParamField>

    <ParamField body="captcha?" type="string">
      El código captcha o la respuesta del proveedor de CAPTCHA. Esta propiedad es obligatoria si en tu tenant de Auth0 está habilitado [Bot Detection](/docs/es/secure/attack-protection/bot-detection).
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  Este método redirige al usuario a la lista de selección del código de país, donde puede actualizar el prefijo del código de país de su número de teléfono.

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.pickCountryCode();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [CustomOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>
