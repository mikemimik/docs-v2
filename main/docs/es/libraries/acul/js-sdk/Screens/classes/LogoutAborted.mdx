La clase LogoutAborted implementa la funcionalidad de la pantalla `logout-aborted`. Esta pantalla se muestra cuando un usuario cancela el proceso de cierre de sesión.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del gestor de la pantalla LogoutAborted:

```typescript Example
import LogoutAborted from '@auth0/auth0-acul-js/logout-aborted';
const logoutAbortedManager = new LogoutAborted();
logoutAbortedManager.changeLanguage({
  language: 'fr',
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import LogoutAborted from '@auth0/auth0-acul-js/logout-aborted';
  const logoutAbortedManager = new LogoutAborted();
  logoutAbortedManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno. Un arreglo de objetos de error del contexto de la transacción.
</ParamField>
