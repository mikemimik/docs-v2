title: "PasskeyEnrollment"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `passkey-enrollment` de Universal Login."
---

La clase PasskeyEnrollment implementa la funcionalidad de la pantalla `passkey-enrollment`. Esta pantalla permite a los usuarios registrar una clave de acceso como un método de autenticación.

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla PasskeyEnrollment:

```typescript Example
import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
const passkeyEnrollmentManager = new PasskeyEnrollment();
await passkeyEnrollmentManager.continuePasskeyEnrollment();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="abortPasskeyEnrollment" type="Promise<void>">
  Este método cancela el proceso de inscripción de clave de acceso.

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  await passkeyEnrollmentManager.abortPasskeyEnrollment();
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [AbortEnrollmentOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/AbortEnrollmentOptions).
    </ParamField>

    <ParamField body="doNotShowAgain?" type="boolean">
      Cuando se establece en `true`, oculta el prompt de inscripción de clave de acceso en sesiones futuras.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  passkeyEnrollmentManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continuePasskeyEnrollment" type="Promise<void>">
  Este método continúa el proceso de inscripción de clave de acceso.

  ```typescript Example
  import PasskeyEnrollment from '@auth0/auth0-acul-js/passkey-enrollment';
  const passkeyEnrollmentManager = new PasskeyEnrollment();
  await passkeyEnrollmentManager.continuePasskeyEnrollment();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción del contexto, o un arreglo vacío si no hay ninguno.
</ParamField>
