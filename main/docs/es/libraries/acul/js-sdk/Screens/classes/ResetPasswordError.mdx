---
title: "ResetPasswordError"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `reset-password-error` de Universal Login."
---

La clase ResetPasswordError implementa la funcionalidad de la pantalla `reset-password-error`. Esta pantalla se muestra cuando se produce un error durante el flujo de restablecimiento de contraseña, por ejemplo, si el enlace de restablecimiento ha caducado o no es válido.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del gestor de pantalla ResetPasswordError:

```typescript Example
import ResetPasswordError from '@auth0/auth0-acul-js/reset-password-error';
const resetPasswordErrorManager = new ResetPasswordError();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import ResetPasswordError from '@auth0/auth0-acul-js/reset-password-error';
  const resetPasswordErrorManager = new ResetPasswordError();
  resetPasswordErrorManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/es/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de locale del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece como `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/es/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no existe ninguno.
</ParamField>
