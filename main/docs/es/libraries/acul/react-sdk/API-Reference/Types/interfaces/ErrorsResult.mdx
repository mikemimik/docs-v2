---
title: "ErrorsResult"
---

Un objeto similar a un arreglo que se devuelve como parte de [`UseErrorsResult`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorsResult) mediante el hook [`useErrors`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Extiende `ReadonlyArray<ErrorItem>`, por lo que se puede iterar, expandir o acceder por índice directamente. Proporciona dos métodos para consultar la lista de errores por tipo o campo.

```typescript
interface ErrorsResult extends ReadonlyArray<ErrorItem> {
  byType(type: ErrorType, opts?: { field?: string }): readonly ErrorItem[];
  byField(field: string, opts?: { type?: ErrorType }): readonly ErrorItem[];
}
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="byType" type={<span>readonly <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem">ErrorItem</a>[]</span>}>
  Devuelve todos los errores que coinciden con el [`ErrorType`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType) indicado, con la opción de filtrarlos por un campo específico.

  <Expandable title="parámetros">
    <ParamField body="type" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>} required>
      El tipo de error por el que se filtrará: `"auth0"`, `"validation"` o `"configuration"`.
    </ParamField>

    <ParamField body="opts" type="{ field?: string }">
      Opciones de filtro opcionales.

      <Expandable title="propiedades">
        <ParamField body="field" type="string">
          Si se proporciona, filtra aún más los resultados para mostrar solo los errores asociados a este nombre de campo.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="byField" type={<span>readonly <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem">ErrorItem</a>[]</span>}>
  Devuelve todos los errores asociados al nombre de campo indicado, con la opción de filtrarlos por un [`ErrorType`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType) específico.

  <Expandable title="parámetros">
    <ParamField body="field" type="string" required>
      El nombre del campo por el que se filtrará. Ejemplo: `"username"`, `"email"`.
    </ParamField>

    <ParamField body="opts" type={<span>&#123; type?: <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a> &#125;</span>}>
      Opciones de filtro opcionales.

      <Expandable title="propiedades">
        <ParamField body="type" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>}>
          Si se proporciona, filtra aún más los resultados para mostrar solo los errores de este tipo.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>
