---
title: "UseResendOptions"
---

Configuración opcional de [useResend](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendOptions {
  timeoutSeconds?: number;
  onTimeout?: () => void;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="timeoutSeconds" type="number" default="10">
  Duración de la cuenta regresiva (en segundos) antes de que se permita otro reenvío.
  El valor predeterminado es `10`.
</ParamField>

<ParamField body="onTimeout" type="() => void">
  La devolución de llamada se ejecuta cuando termina la cuenta regresiva y la acción de reenvío vuelve a estar disponible.
</ParamField>
