---
title: "MfaPushEnrollmentQrWithRememberOptions"
---

Opciones para continuar la inscripción en MFA push con un ajuste opcional para recordar el dispositivo.

Extiende `CustomOptions` para permitir pasar opciones personalizadas junto con la preferencia de recordar el dispositivo.

```ts Example
export interface MfaPushEnrollmentQrWithRememberOptions extends CustomOptions {
  rememberDevice?: boolean;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propiedades
</div>

<ParamField body="rememberDevice?" type="boolean">
  Cuando se establece en `true`, este dispositivo se registra como de confianza para que no se le solicite al usuario la autenticación MFA push en futuros inicios de sesión desde el mismo dispositivo.
</ParamField>
