---
title: "Identificador"
---

Describe un campo identificador configurado (p. ej., correo electrónico, phone o nombre de usuario)
para su uso en un formulario de registro o flujo de autenticación.

```ts Example
export interface Identifier {
  type: IdentifierType;
  required: boolean;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="type" type={<span><a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a></span>} required>
  El tipo de identificador, como `'email'`, `'phone'` o `'username'`.
</ParamField>

<ParamField body="required" type="boolean" required>
  Indica si este identificador es obligatorio para el registro.
</ParamField>
