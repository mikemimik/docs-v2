<ParamField body="continue" type="Promise<void>">
  Permite al usuario continuar el flujo de autenticación después de completar correctamente el registro WebAuthn.
  Este método envía una solicitud POST al `endpoint` `/u/mfa-webauthn-enrollment-success` con `action: "default"`.

  Una promesa que se resuelve cuando la acción de continuar se envía correctamente.
  Si se completa correctamente, el navegador normalmente será redirigido al siguiente paso del flujo.

  #### Lanza

  Lanza un error si falla el envío del formulario (p. ej., un problema de red o `invalid state`).

  ```typescript Example
  // Suponiendo que 'sdk' es una instancia de MfaWebAuthnEnrollmentSuccess
  try {
    await sdk.continue();
    // El usuario es redirigido al siguiente paso.
  } catch (error) {
    console.error("Failed to continue after WebAuthn enrollment:", error);
    // Gestiona el error, posiblemente inspeccionando sdk.transaction.errors si la página se vuelve a renderizar con un error.
  }
  ```
