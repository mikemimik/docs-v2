Opciones para continuar con el desafío OTP

```ts Example
export interface ContinueOptions {
  /** El código ingresado por el usuario */
  code: string;
  /** Indica si se debe recordar el navegador */
  rememberDevice?: boolean;
  /** Cualquier opción personalizada adicional */
  [key: string]: string | number | boolean | undefined;
}
```
