---
title: "UsernameValidationError"
---

Representa un error individual de validación del nombre de usuario que se devuelve durante la validación.

Cada error tiene un `code` legible por máquina y un `message` legible para las personas.

```ts Example
export interface UsernameValidationError {
  code: string;
  message: string;
  field: string;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="code" type="string" required>
  Un identificador único de la regla de validación que falló.

  Ejemplo: `"username-too-short"`, `"username-invalid-characters"`
</ParamField>

<ParamField body="message" type="string" required>
  Una descripción del error legible.

  Ejemplo: `"Username must be at least 3 characters long."`
</ParamField>

<ParamField body="field" type="string" required>
  El campo asociado al error de validación, normalmente &quot;username&quot;, &quot;email&quot; o &quot;phone&quot;.
</ParamField>
