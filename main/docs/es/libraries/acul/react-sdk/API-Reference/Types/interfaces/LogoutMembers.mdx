Interfaz que define los métodos y las propiedades disponibles para la pantalla de cierre de sesión

```ts Example
export interface LogoutMembers extends BaseMembers {
  /**
   * Confirma la acción de cierre de sesión (aceptar o denegar)
   * @param payload Las opciones que contienen la acción a realizar
   */
  confirmLogout(payload: ConfirmLogoutOptions): Promise<void>;
}
```
