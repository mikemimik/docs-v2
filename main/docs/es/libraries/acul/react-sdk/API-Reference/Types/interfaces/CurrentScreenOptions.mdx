---
title: "CurrentScreenOptions"
---

Objeto Options que describe el contexto de la pantalla actual en un flujo de Auth0 Universal Login. Proporciona acceso a los metadatos del cliente, la organización, el tenant, la transacción y la pantalla disponibles en runtime.

```ts Example
export interface CurrentScreenOptions {
  client: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  organization: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  prompt: {
    name: string;
  } | null;
  screen: {
    name: string;
  } | null;
  tenant: {
    enabledLocales: string[];
  } | null;
  transaction: {
    errors: TransactionError[] | null;
    state: string;
    locale: string;
  } | null;
  untrustedData: {
    authorizationParams: Record<string, string> | null;
  } | null;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="client" type="{ id: string; metadata: Record<string, string> | null } | null">
  Detalles del cliente para la solicitud de autorización actual, incluido el ID de cliente y cualquier metadato asociado.
</ParamField>

<ParamField body="organization" type="{ id: string; metadata: Record<string, string> | null } | null">
  Contexto de la organización para la solicitud actual, incluido el identificador de la organización y los metadatos.
</ParamField>

<ParamField body="prompt" type="{ name: string } | null">
  La configuración actual del prompt, que identifica qué paso del prompt se está renderizando.
</ParamField>

<ParamField body="screen" type="{ name: string } | null">
  La configuración actual de la pantalla, que identifica qué pantalla dentro del prompt está activa.
</ParamField>

<ParamField body="tenant" type="{ enabledLocales: string[] } | null">
  Configuración a nivel del tenant, incluida la lista de locales habilitados para esta implementación de Universal Login.
</ParamField>

<ParamField body="transaction" type="{ errors: TransactionError[] | null; state: string; locale: string } | null">
  Estado de la transacción para el flujo de autenticación actual, incluidos los errores de validación, el identificador de estado y el locale activo.
</ParamField>

<ParamField body="untrustedData" type="{ authorizationParams: Record<string, string> | null } | null">
  Datos no confiables pasados al flujo, como parámetros de autorización en los que no se debe confiar para tomar decisiones de seguridad.
</ParamField>
