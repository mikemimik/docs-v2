---
title: "FlattenedTheme"
---

Objeto de tema estructurado devuelto por [`useAuth0Themes`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes). Proporciona acceso a nivel de token, de forma aplanada, a todos los estilos visuales aplicados a los componentes de Auth0 Universal Login.

```ts Example
export interface FlattenedTheme {
  colors: Record<string, string>;
  fonts: Record<string, string | number | boolean | object>;
  borders: Record<string, string | number | boolean>;
  pageBackground: Record<string, string>;
  widget: Record<string, string | number>;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="colors" type="Record<string, string>" required>
  Tokens de color aplicados en toda la UI de inicio de sesión; incluyen colores de texto, fondo, borde y estados interactivos.
</ParamField>

<ParamField body="fonts" type="Record<string, string | number | boolean | object>" required>
  Tokens tipográficos que controlan la familia tipográfica, el tamaño, el peso, la altura de línea y otros estilos de texto relacionados.
</ParamField>

<ParamField body="borders" type="Record<string, string | number | boolean>" required>
  Tokens de borde que definen el radio, el ancho y el estilo de los elementos de la UI.
</ParamField>

<ParamField body="pageBackground" type="Record<string, string>" required>
  Tokens de estilo de fondo aplicados al contenedor de la página que rodea el widget de inicio de sesión.
</ParamField>

<ParamField body="widget" type="Record<string, string | number>" required>
  Tokens de estilo aplicados específicamente a la tarjeta del widget de inicio de sesión, como el relleno, la sombra y las dimensiones.
</ParamField>
