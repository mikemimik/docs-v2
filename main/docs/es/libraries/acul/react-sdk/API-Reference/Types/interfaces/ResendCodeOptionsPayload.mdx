ResendCodeOptions
description Opciones para el método `resendCode` en la pantalla de verificación por correo electrónico de inicio de sesión.
Esta operación se usa cuando el usuario solicita que se le envíe un nuevo código de verificación por correo electrónico,
por ejemplo, si no recibió el código inicial o si ya ha caducado.

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
