ScreenMembersOnMfaWebAuthnEnrollmentSuccess

```ts Example
export interface ScreenMembersOnMfaWebAuthnEnrollmentSuccess extends ScreenMembers {
  /**
   * Datos específicos de la pantalla que contienen detalles sobre el registro WebAuthn exitoso.
   * @type {{ nickname: string; webauthnType: WebAuthnType; } | null}
   */
  data: {
    /**
     * El apodo que el usuario (o el sistema) asignó al autenticador de WebAuthn recién registrado.
     * Esto ayuda al usuario a identificar el autenticador más adelante.
     * @type {string}
     * @example "YubiKey 5"
     * @example "My Laptop's Touch ID"
     */
    nickname: string;
    /**
     * El tipo de autenticador de WebAuthn que se registró correctamente.
     * - `'webauthn-roaming'`: Indica un autenticador itinerante como una clave de seguridad USB.
     * - `'webauthn-platform'`: Indica un autenticador de plataforma como Touch ID o Windows Hello.
     * @type {WebAuthnType}
     */
    webauthnType: WebAuthnType;
  } | null;
}
```
