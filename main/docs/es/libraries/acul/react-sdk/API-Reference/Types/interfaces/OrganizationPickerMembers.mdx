title: "OrganizationPickerMembers"

Interfaz que define los métodos y las propiedades disponibles para la pantalla «organization-picker»

```ts Example
export interface OrganizationPickerMembers extends BaseMembers {
  /**
   * Selecciona una organización para el usuario.
   * @param payload Las opciones que contienen el Organization ID.
   */
  selectOrganization(payload: { organization: string; state: string }): Promise<void>;

  /**
   * Omite la selección de organización y continúa con la cuenta personal del usuario.
   * @param payload Opciones personalizadas opcionales para incluir con la solicitud.
   */
  skipOrganizationSelection(payload?: CustomOptions): Promise<void>;
}
```
