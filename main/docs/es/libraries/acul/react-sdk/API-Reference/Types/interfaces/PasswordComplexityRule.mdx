---
title: "PasswordComplexityRule"
---

Representa una única regla de complejidad de contraseña evaluada. Se devuelve como parte de [`PasswordValidationResult`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordValidationResult) mediante [`usePasswordValidation`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation). Admite la composición jerárquica de reglas mediante el arreglo opcional `items`.

```typescript
export interface PasswordComplexityRule {
  code: string;
  label: string;
  status: "valid" | "error";
  isValid: boolean;
  args?: {
    count: number;
    total?: number;
    example?: string;
  };
  items?: PasswordComplexityRule[];
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="code" type="string" required>
  Un identificador legible por máquina para esta regla de complejidad. Ejemplo: `"lengthAtLeast"`, `"containsAtLeast"`.
</ParamField>

<ParamField body="label" type="string" required>
  Un nombre para mostrar legible para las personas de la regla, adecuado para renderizarse en un indicador de fortaleza de la contraseña.
</ParamField>

<ParamField body="status" type="&#x22;valid&#x22; | &#x22;error&#x22;" required>
  El resultado de la validación de esta regla. `"valid"` si la regla se cumple; `"error"` si falla.
</ParamField>

<ParamField body="isValid" type="boolean" required>
  Representación booleana del resultado de la regla. `true` cuando `status` es `"valid"`.
</ParamField>

<ParamField body="args" type="object">
  Parámetros adicionales que se usan para describir el umbral de la regla o proporcionar contexto para el mensaje de error.

  <Expandable title="propiedades">
    <ParamField body="count" type="number" required>
      El umbral numérico o recuento asociado a esta regla. Ejemplo: cantidad mínima de caracteres.
    </ParamField>

    <ParamField body="total" type="number">
      El número total de caracteres o elementos relevantes para esta regla, cuando corresponda.
    </ParamField>

    <ParamField body="example" type="string">
      Un valor de ejemplo que ilustra qué cumple esta regla, cuando corresponda.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="items" type={<span><a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>}>
  Reglas de complejidad anidadas para grupos de reglas compuestas. Permite una composición jerárquica de reglas en la que una regla principal agrupa varias reglas secundarias.
</ParamField>
