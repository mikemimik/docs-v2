---
title: "UseErrorOptions"
---

Objeto de opciones para el hook [`useError`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useError). Controla cómo se muestran los errores en las screens de Auth0 Universal Login.

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="includeDevErrors" type="boolean">
  Cuando es `true`, incluye en la salida de errores detalles específicos del entorno de desarrollo. Resulta útil para depurar durante el desarrollo; debe desactivarse en producción.
</ParamField>
