---
title: "UseErrorOptions"
---

Objeto Options para el hook [`useError`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useError). Controla cómo se muestran los errores en las screens de Auth0 Universal Login.

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="includeDevErrors" type="boolean">
  Cuando `true`, incluye detalles de error específicos del entorno de desarrollo en la salida de errores. Resulta útil para depurar durante el desarrollo; debe desactivarse en producción.
</ParamField>
