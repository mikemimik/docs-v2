---
title: "MfaRecoveryCodeChallengeMembers"
---

Interfaz que describe los miembros de la pantalla de desafío de código de recuperación de MFA.

```ts Example
export interface MfaRecoveryCodeChallengeMembers extends BaseMembers {
  client: ClientMembers;
  organization: OrganizationMembers;
  prompt: PromptMembers;
  screen: ScreenMembers;
  transaction: TransactionMembers;

  /**
   * Continúa con el código de recuperación proporcionado.
   * @param payload - Las opciones de continuación que contienen el código de recuperación y las opciones personalizadas opcionales.
   * @returns Una promesa que se resuelve cuando la continuación es exitosa.
   */
  continue(payload: ContinueOptions): Promise<void>;

  /**
   * Navega a la pantalla donde el usuario puede elegir otro método de MFA.
   * @param payload Payload opcional.
   * @returns Una promesa que se resuelve cuando la navegación finaliza.
   */
  tryAnotherMethod(payload?: CustomOptions): Promise<void>;
}
```
