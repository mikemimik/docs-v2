Opciones para seleccionar el código de país

```ts Example
export interface SelectCountryCodeOptions {
  /** El código de país (p. ej. 'US', 'GB') */
  country_code: string;
  /** El prefijo telefónico (p. ej. '+1', '+44') */
  phone_prefix: string;
  /** Any additional opciones personalizadas */
  [key: string]: string | number | boolean | undefined;
}
```
