ContinueOptions

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * El nuevo apodo que el usuario desea asignar a su clave de seguridad de WebAuthn.
   * Este valor se enviará al servidor para su actualización.
   * El servidor validará este apodo (p. ej., longitud, caracteres permitidos).
   * @type {string}
   * @example "Primary YubiKey"
   * @example "Home Office Key"
   */
  nickname: string;
}
```
