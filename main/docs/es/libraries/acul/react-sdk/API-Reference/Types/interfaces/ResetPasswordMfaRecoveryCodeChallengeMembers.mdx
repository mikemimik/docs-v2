title: "ResetPasswordMfaRecoveryCodeChallengeMembers"

Interfaz que describe los miembros de la pantalla desafío de código de recuperación de MFA.

```ts Example
export interface ResetPasswordMfaRecoveryCodeChallengeMembers extends BaseMembers {
  client: ClientMembers;
  organization: OrganizationMembers;
  prompt: PromptMembers;
  screen: ScreenMembers;
  transaction: TransactionMembers;

  /**
   * Continúa con el código de recuperación proporcionado.
   * @param {string} code - El código de recuperación ingresado por el usuario.
   * @param {CustomOptions} [payload] - Payload opcional.
   * @returns {Promise<void>}
   */
  continue(code: string, payload?: CustomOptions): Promise<void>;

  /**
   * Navega a la pantalla donde el usuario puede seleccionar otro método de MFA.
   * @param {CustomOptions} [payload] - Payload opcional.
   * @returns {Promise<void>}
   */
  tryAnotherMethod(payload?: CustomOptions): Promise<void>;
}
```
