---
title: "UsernameValidationResult"
---

El resultado de validar un nombre de usuario según una o varias reglas.

```ts Example
export interface UsernameValidationResult {
  isValid: boolean;
  errors: UsernameValidationError[];
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isValid" type="boolean" required>
  Indica si el nombre de usuario cumple todas las reglas de validación.
</ParamField>

<ParamField body="errors" type={<span><a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError">UsernameValidationError[]</a></span>} required>
  Un arreglo de objetos [UsernameValidationError](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError) que representa
  las reglas individuales que no se cumplieron, si las hay.

  Este arreglo está vacío si [UsernameValidationResult.isValid](#isvalid) es `true`.
</ParamField>
