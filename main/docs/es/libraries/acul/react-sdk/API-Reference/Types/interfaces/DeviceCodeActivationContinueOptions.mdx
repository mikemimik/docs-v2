title: "DeviceCodeActivationContinueOptions"
---

payload para continuar con la activación del código de dispositivo.

Esta interfaz amplía `CustomOptions`, lo que significa que todas las propiedades de las opciones personalizadas
se combinan en el nivel superior del objeto. Así se evita el anidamiento y se ofrece
una estructura más sencilla para quienes utilizan esta interfaz.

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```
