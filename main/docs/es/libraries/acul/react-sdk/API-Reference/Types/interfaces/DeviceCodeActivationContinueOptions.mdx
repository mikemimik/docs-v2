payload para continuar la activación del device code.

Esta interfaz extiende `CustomOptions`, lo que significa que todas las propiedades de las opciones personalizadas
se incorporan en el nivel superior del objeto. Esto evita el anidamiento y ofrece
una estructura más sencilla para quienes utilizan esta interfaz.

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```
