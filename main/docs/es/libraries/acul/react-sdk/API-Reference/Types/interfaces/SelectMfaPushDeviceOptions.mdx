---
title: "SelectMfaPushDeviceOptions"
---

Opciones para seleccionar un dispositivo de MFA push.

```ts Example
export interface SelectMfaPushDeviceOptions {
  /**
   * El índice del device a seleccionar de la lista de devices registrados.
   * El índice es basado en 0.
   */
  deviceIndex: number;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propiedades
</div>

<ParamField body="deviceIndex" type="number">
  El índice del device que se debe seleccionar de la lista de dispositivos registrados.
  El índice comienza basado en 0.
</ParamField>
