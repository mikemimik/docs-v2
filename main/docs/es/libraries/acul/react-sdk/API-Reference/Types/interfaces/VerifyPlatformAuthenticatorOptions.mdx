VerifyPlatformAuthenticatorOptions

```ts Example
export interface VerifyPlatformAuthenticatorOptions extends CustomOptions {
  /**
   * Opcional. Si es true y `screen.showRememberDevice` es true, indica que el usuario
   * ha elegido recordar este device, normalmente durante 30 días, para omitir el MFA en inicios de sesión posteriores.
   * @type {boolean}
   */
  rememberDevice?: boolean;
}
```

<ParamField body="rememberDevice?" type="boolean">
  Opcional. Si es true, se intenta recordar el navegador para futuros desafíos MFA.
  Corresponde al campo del formulario `rememberBrowser`. Esto solo aplica si `screen.showRememberDevice` es true.
</ParamField>
