```ts Example
type MfaPollingOptions = {
  intervalMs?: number;
  onCompleted?: () => void;
  onError?: (error: Error) => void;
}
```

<ParamField body="MfaPollingOptions" type="object">
  Opciones de configuración para gestionar un proceso de polling de notificaciones push de MFA.

  Proporcione estas opciones al iniciar una sesión de polling para controlar la frecuencia con la que
  se consulta el endpoint del challenge y cómo se gestionan los casos de éxito o error.

  <Expandable title="propiedades">
    <ParamField body="intervalMs" type="number">
      Intervalo opcional, en **milisegundos**, entre requests de polling consecutivas.

      * Si se omite, se usa el intervalo predeterminado interno del SDK (normalmente, 5000 ms).
      * Elija un valor que equilibre la capacidad de respuesta con los límites de frecuencia.

      ```ts Default Value
      Implementation-specific (commonly 5000 ms)
      ```
    </ParamField>

    <ParamField body="onCompleted" type="() => void">
      Función de devolución de llamada opcional que se ejecuta cuando el MFA push challenge se aprueba correctamente
      y el polling finaliza.

      * Se llama exactamente **una vez**, después de lo cual el polling se detiene automáticamente.
      * Es útil para continuar el flujo de login (por ejemplo, llamando a `continue()`).
    </ParamField>

    <ParamField body="onError" type="(error: Error) => void">
      Función de devolución de llamada opcional que se invoca si se produce un error durante el polling.

      Recibe un objeto [ULError](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error) que contiene:

      * `status` — el código de estado HTTP de la request fallida

      * `responseText` — el cuerpo de la respuesta sin procesar, si está disponible

      * Se llama una vez por cada evento de error; el polling puede reintentarse según el tipo de error.

      * Úselo para mostrar mensajes de error o activar lógica personalizada de reintento o cancelación.

      **Propiedades**

      <ParamField body="error" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error">Error</a>}>
        Un objeto de error que contiene el código de estado HTTP y el cuerpo de la respuesta sin procesar de la request de polling fallida.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>
