---
title: "UseErrorsResult"
---

Objeto de resultado que devuelve [`useErrors`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Proporciona acceso al estado actual de los errores y a métodos para descartar errores individuales o todos los errores en las pantallas de Auth0 Universal Login.

```ts Example
export interface UseErrorsResult {
  errors: ErrorsResult;
  hasError: boolean;
  dismiss: (id: string) => void;
  dismissAll: () => void;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="errors" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorsResult">ErrorsResult</a>} required>
  La colección de errores actuales, incluidos los mensajes y los identificadores.
</ParamField>

<ParamField body="hasError" type="boolean" required>
  `true` cuando hay uno o más errores; útil para renderizar condicionalmente la UI de errores.
</ParamField>

<ParamField body="dismiss" type="(id: string) => void" required>
  Elimina un error específico por su `id`.

  <Expandable title="Parámetros">
    <ParamField body="id" type="string">
      El identificador del error que se debe eliminar.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="dismissAll" type="() => void" required>
  Elimina de una vez todos los errores activos.
</ParamField>
