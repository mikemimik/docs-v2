ContinueOptions
Define las opciones necesarias para continuar el proceso de desafío de MFA por teléfono.
Normalmente, esto implica seleccionar el método (SMS o Voz) para recibir el código de verificación.

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * El tipo de desafío a realizar.
   * 'sms': Envía el código por mensaje de texto.
   * 'voice': Envía el código mediante una llamada de voz.
   * @default 'sms'
   */
  type: 'sms' | 'voice';
}
```

  * `sms`: Envía el código por mensaje de texto.
  * `voice`: Envía el código mediante una llamada de voz.

  #### Valor predeterminado
