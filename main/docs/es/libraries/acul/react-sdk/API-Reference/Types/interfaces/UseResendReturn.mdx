title: "UseResendReturn"
---

Tipo de valor devuelto de [useResend](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendReturn {
  remaining: number;
  disabled: boolean;
  startResend: () => void;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="remaining" type="number" required>
  Número de segundos restantes hasta que se permita el siguiente intento de reenvío. Úsalo para mostrar una cuenta regresiva en la UI.
</ParamField>

<ParamField body="disabled" type="boolean" required>
  `true` cuando la opción de reenviar no está disponible en ese momento, ya sea porque hay un reenvío en curso o porque el período de espera aún no ha finalizado.
</ParamField>

<ParamField body="startResend" type="() => void" required>
  Inicia de inmediato un intento de reenvío si en ese momento está permitido.
</ParamField>
