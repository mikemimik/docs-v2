---
title: "ErrorType"
---

Un tipo de unión de valores `string` que clasifica los errores gestionados por [`useErrors`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Se usa para filtrar errores con `errors.byType()` y está presente en cada [`ErrorItem`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem) como la propiedad `kind`.

```typescript
type ErrorType = "auth0" | "validation" | "configuration";
```

<div id="values">
  ## Valores
</div>

<ParamField body="&#x22;auth0&#x22;" type="ErrorType">
  Errores devueltos por Auth0 o por tu propio backend durante la autenticación o la autorización.
</ParamField>

<ParamField body="&#x22;validation&#x22;" type="ErrorType">
  Errores de validation del form en el cliente, como entradas no válidas antes de que se haga una solicitud.
</ParamField>

<ParamField body="&#x22;configuration&#x22;" type="ErrorType">
  Errores causados por una integración incorrecta del SDK o por una mala configuración, como llamar a un hook fuera de su contexto requerido. Solo aparecen cuando se establece `includeDevErrors: true` en [`UseErrorOptions`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorOptions).
</ParamField>
