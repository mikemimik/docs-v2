---
title: "PasswordValidationResult"
---

El tipo de valor devuelto por [`usePasswordValidation`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation). Contiene el resultado general de si la contraseña es válida y una lista detallada de las reglas de complejidad de la contraseña evaluadas.

```typescript
export interface PasswordValidationResult {
  isValid: boolean;
  results: PasswordComplexityRule[];
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isValid" type="boolean" required>
  Resultado global de todas las comprobaciones de la política de contraseñas. `true` solo si cada regla de [`PasswordComplexityRule`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) tiene `status` igual a `"valid"`.
</ParamField>

<ParamField body="results" type={<span><a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>} required>
  Arreglo de reglas de contraseña evaluadas. Cada elemento es una [`PasswordComplexityRule`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) cuyo campo [`status`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule#status) refleja el resultado de la validación calculado para esa regla.
</ParamField>
