title: "MfaPollingResult"
---

Objeto de resultado que devuelve [useMfaPolling](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling).

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isRunning" type="boolean">
  Indica si el proceso de polling de MFA push está activo en este momento.

  * `true` — El polling está en curso y a la espera de completarse.
  * `false` — El polling se ha detenido, ya sea porque se completó,
    se canceló manualmente o se desmontó el componente.
</ParamField>

<ParamField body="startPolling()" type="void">
  Inicia o reanuda el proceso de polling.

  * Si el polling ya está activo, esta llamada no tiene efecto.
  * Si se detuvo previamente, al llamar a este método se reinicia el ciclo de polling.
</ParamField>

<ParamField body="stopPolling()" type="void">
  Detiene el proceso de polling de inmediato.

  * Cancela cualquier temporizador programado o solicitud en curso.
  * Se puede llamar varias veces sin problema; las llamadas posteriores no tienen efecto.
</ParamField>
