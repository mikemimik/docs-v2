title: "MfaPollingResult"
---

Objeto de resultado que devuelve [useMfaPolling](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling).

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isRunning" type="boolean">
  Indica si el proceso de sondeo de MFA push está activo en ese momento.

  * `true` — El sondeo está en ejecución y a la espera de completarse.
  * `false` — El sondeo se ha detenido, ya sea porque finalizó,
    por cancelación manual o porque se desmontó el componente.
</ParamField>

<ParamField body="startPolling()" type="void">
  Inicia o reanuda el proceso de sondeo.

  * Si el sondeo ya está activo, esta llamada no tiene efecto.
  * Si se detuvo anteriormente, volver a llamarla reinicia el ciclo de sondeo.
</ParamField>

<ParamField body="stopPolling()" type="void">
  Detiene el proceso de sondeo de inmediato.

  * Cancela cualquier temporizador programado o solicitud en curso.
  * Se puede llamar varias veces sin problema; las llamadas posteriores no tienen efecto.
</ParamField>
