ContinueWithCodeOptions
description Opciones para el método `continueWithCode` en la pantalla Verificación de correo electrónico de Login.
Esta operación se utiliza cuando el usuario introduce el código de verificación que recibió por correo electrónico
para continuar con el flujo de autenticación.

```ts Example
export interface ContinueWithCodeOptions extends CustomOptions {
  /**
   * El código de verificación enviado al correo electrónico del usuario. Generalmente es una cadena numérica corta
   * o alfanumérica que el usuario debe ingresar en el formulario.
   * @type {string}
   * @example "123456"
   * @example "ABCXYZ"
   */
  code: string;
}
```
