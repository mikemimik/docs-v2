---
title: "ContextHooks<T>"
---

Clase de fábrica que crea hooks de contexto para acceder a los datos de Auth0 Universal Login.
Estos hooks permiten acceder a distintos aspectos del flujo de autenticación y de la configuración del tenant.

<div id="methods">
  ## Métodos
</div>

<ParamField body="useUser()" type="T[&#x22;user&#x22;]">
  Hook para acceder a la información del usuario y a los datos del perfil.

  Objeto de usuario que contiene información del perfil, atributos y datos específicos del usuario

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/login-id';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useTenant()" type="T[&#x22;tenant&#x22;]">
  Hook para acceder a la configuración del tenant y sus ajustes.

  Objeto tenant que contiene el dominio, la región y la configuración específica del tenant

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/login-id';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useBranding()" type="T[&#x22;branding&#x22;]">
  Hook para acceder al branding y a la configuración del tema.

  Objeto de branding que contiene colores, logotipos, fuentes y opciones de personalización visual

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/login-id';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient()" type="T[&#x22;client&#x22;]">
  Hook para acceder a la configuración de la aplicación de Auth0 (cliente).

  Objeto cliente que contiene la configuración de la aplicación, las devoluciones de llamada y datos específicos del cliente

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/login-id';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization()" type="T[&#x22;organization&#x22;]">
  Hook para acceder al contexto y la configuración de la organización.

  Objeto de organización que contiene datos específicos de la organización, metadatos y configuración

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/login-id';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Sin contexto de organización</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt()" type="T[&#x22;prompt&#x22;]">
  Hook para acceder a la configuración del prompt y a los ajustes del flujo.

  Objeto prompt que contiene la configuración del flujo, la configuración de la pantalla y datos específicos del prompt

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/login-id';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData()" type="T[&#x22;untrustedData&#x22;]">
  Hook para acceder a datos no confiables procedentes de parámetros de URL y envíos de formularios.

  Objeto que contiene datos introducidos por el usuario que no son confiables y deben validarse antes de usarse

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/login-id';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useScreen()" type="T[&#x22;screen&#x22;]">
  Hook para acceder a la información y los metadatos de la pantalla actual.

  Objeto `screen` que contiene el nombre de la pantalla actual, la configuración y los datos específicos de la pantalla

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/login-id';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTransaction()" type="T[&#x22;transaction&#x22;]">
  Hook para acceder al estado de la transacción y a los datos del flujo de autenticación.

  Objeto de transacción que contiene el estado del flujo, los datos de la sesión e información específica de la transacción

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/login-id';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>
