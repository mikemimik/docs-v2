Interfaz que define los métodos y las propiedades disponibles de la pantalla mfa-country-codes

```ts Example
export interface MfaCountryCodesMembers extends BaseMembers {
  screen: ScreenMembersOnMfaCountryCodes;
  /**
   * Selecciona un código de país de las opciones disponibles
   * @param payload Las opciones que contienen la acción de selección del código de país
   */
  selectCountryCode(payload: SelectCountryCodeOptions): Promise<void>;
  /**
   * Regresa a la pantalla anterior
   * @param payload Opciones personalizadas opcionales para incluir con la solicitud
   */
  goBack(payload?: CustomOptions): Promise<void>;
}
```
