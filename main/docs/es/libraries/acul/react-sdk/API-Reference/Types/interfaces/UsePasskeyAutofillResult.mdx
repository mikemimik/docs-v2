El tipo de valor devuelto por [`usePasskeyAutofill`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/usePasskeyAutofill).

```typescript
interface UsePasskeyAutofillResult {
  inputRef: RefObject<HTMLInputElement>;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  Una referencia de React para adjuntarla al campo de entrada del nombre de usuario. Al vincularla, el SDK configura automáticamente el atributo `autocomplete` del campo como `"username webauthn"`, lo que habilita la interfaz condicional del navegador para el autocompletado con clave de acceso. Vincular esta referencia es opcional si `autocomplete="username webauthn"` ya está declarado en tu código.
</ParamField>
