```typescript
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  Una referencia de React para adjuntarla al elemento de entrada del nombre de usuario. Cuando se asocia, el SDK configura automáticamente el atributo `autocomplete` del campo como `"username webauthn"`, lo que habilita la UI condicional del navegador para el autocompletado con clave de acceso. Asociar esta referencia es opcional si `autocomplete="username webauthn"` ya está declarado en tu marcado.
</ParamField>
