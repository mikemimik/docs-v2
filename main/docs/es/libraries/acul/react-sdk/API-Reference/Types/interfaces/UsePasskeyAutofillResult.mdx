```typescript
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  Una ref de React para adjuntarla al elemento de entrada del nombre de usuario. Cuando se vincula, el SDK configura automáticamente el atributo `autocomplete` de la entrada como `"username webauthn"`, lo que habilita la UI condicional del navegador para el autocompletado de claves de acceso. Vincular esta ref es opcional si `autocomplete="username webauthn"` ya está declarado en tu marcado.
</ParamField>
