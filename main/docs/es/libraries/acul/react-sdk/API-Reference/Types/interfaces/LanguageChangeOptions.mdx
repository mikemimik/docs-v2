---
title: "LanguageChangeOptions"
---

Opciones aceptadas por la función que devuelve [`useChangeLanguage`](/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage).

```typescript
interface LanguageChangeOptions {
  language: string;
  persist?: "session";
  [key: string]: string | number | boolean | undefined;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="language" type="string" required>
  El código del locale que se usará. Debe coincidir con uno de los locales habilitados en tu tenant de Auth0. Ejemplo: `"en"`, `"fr"`, `"es"`.
</ParamField>

<ParamField body="persist" type="&#x22;session&#x22;">
  Ámbito de persistencia de la preferencia de idioma. Por el momento, solo se admite `"session"`. El valor predeterminado es `"session"`.
</ParamField>

<div id="remarks">
  ## Observaciones
</div>

* Se pueden enviar campos personalizados adicionales junto con `language` y `persist`. Los campos personalizados deben llevar el prefijo `"ulp-"` y aceptar valores de tipo `string`, `number`, `boolean` o `undefined`.
