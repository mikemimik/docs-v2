<ParamField body="useSignupIdentifiers()" type={<span>[] | <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier">Identifier[]</a></span>}>
  Devuelve una lista de identificadores habilitados (email, phone o username), cada uno con su estado `required`,
  en función de los identificadores de registro de la pantalla actual.

  ## Devuelve

  [] | [`Identifier`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier)[]

  Un arreglo de objetos [Identifier](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/Identifier), donde cada uno contiene un `type` (tipo de identificador)
  y un indicador `required` que señala si es obligatorio para el registro.

  ## Pantallas compatibles

  * `signup`
  * `signup-id`

  ```tsx Example
  import { useSignupIdentifiers } from '@auth0/auth0-acul-react/signup';

  const identifiers = useSignupIdentifiers();
  const emailIdentifier = identifiers.find(({ type }) => type === 'email');
  const phoneIdentifier = identifiers.find(({ type }) => type === 'phone');
  const usernameIdentifier = identifiers.find(({ type }) => type === 'username');

  const emailRequired = emailIdentifier?.required ?? false;
  const phoneRequired = phoneIdentifier?.required ?? false;
  const usernameRequired = usernameIdentifier?.required ?? false;

  // Salida de ejemplo:
  // [
  //   { type: 'email', required: true },
  //   { type: 'phone', required: false },
  //   { type: 'username', required: true },
  // ]
  ```
</ParamField>
