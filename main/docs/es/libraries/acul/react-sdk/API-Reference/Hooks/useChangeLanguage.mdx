title: "useChangeLanguage"
---

<ParamField body="useChangeLanguage" type="(options: LanguageChangeOptions) => Promise<void>">
  Hook de React que devuelve una función para cambiar el idioma que se muestra en la pantalla actual de ACUL. La pantalla se vuelve a renderizar automáticamente con el nuevo locale después del envío. El idioma debe ser uno de los locales habilitados configurados en tu tenant de Auth0.

  ### Características principales

  * **Compatible con el tenant** — solo acepta locales habilitados en la configuración de tu tenant de Auth0.
  * **Renderizado automático** — la pantalla se actualiza al nuevo idioma después de enviar el cambio.
  * **Persistencia de sesión** — la preferencia de idioma se almacena durante la sesión de forma predeterminada.

  ## Parámetros

  La función devuelta acepta un único argumento [`LanguageChangeOptions`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/LanguageChangeOptions):

  <ParamField body="language" type="string" required>
    El código de locale al que se cambiará. Debe coincidir con uno de los locales habilitados en tu tenant de Auth0. Ejemplo: `"en"`, `"fr"`, `"es"`.
  </ParamField>

  <ParamField body="persist" type="&#x22;session&#x22;">
    Ámbito de persistencia de la preferencia de idioma. El valor predeterminado es `"session"`.
  </ParamField>

  ## Devuelve

  `(options: LanguageChangeOptions) => Promise<void>`

  Una función que envía el cambio de idioma. La Promise devuelta se resuelve cuando se completa el envío.

  ```tsx Example
  import { useChangeLanguage } from "@auth0/auth0-acul-react";

  export function LanguageSwitcher() {
    const changeLanguage = useChangeLanguage();

    return (
      <div>
        <button onClick={() => changeLanguage({ language: "en" })}>English</button>
        <button onClick={() => changeLanguage({ language: "fr" })}>Français</button>
        <button onClick={() => changeLanguage({ language: "es" })}>Español</button>
      </div>
    );
  }
  ```

  También puedes rellenar las opciones dinámicamente a partir de la configuración del tenant:

  ```tsx
  import { useChangeLanguage, useCurrentScreen } from "@auth0/auth0-acul-react";

  export function LocaleSelector() {
    const changeLanguage = useChangeLanguage();
    const { tenant, transaction } = useCurrentScreen();

    return (
      <select
        defaultValue={transaction.locale}
        onChange={e => changeLanguage({ language: e.target.value })}
      >
        {tenant.enabledLocales.map(locale => (
          <option key={locale} value={locale}>{locale}</option>
        ))}
      </select>
    );
  }
  ```

  ## Observaciones

  * El valor de `language` debe coincidir con un locale habilitado en tu tenant de Auth0; si pasas un locale no compatible, se producirá un error.
  * Usa `useCurrentScreen()` para acceder a `tenant.enabledLocales` y `transaction.locale` para seleccionar el locale dinámicamente.
  * Llama a `useChangeLanguage` en el nivel superior de tu componente; no lo llames de forma condicional ni dentro de controladores de eventos.
</ParamField>
