<ParamField body="useLoginIdentifiers()" type={<span><a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/IdentifierType">IdentifierType[]</a></span>}>
  Devuelve una lista de tipos de identificador activos (como `'email'`, `'phone'` o `'username'`)
  que se usan actualmente en el flujo de autenticación o en el proceso de inicio de sesión.

  ## Devuelve

  [`IdentifierType`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/IdentifierType)[]

  Un arreglo de [IdentifierType](/docs/es/libraries/acul/react-sdk/API-Reference/Types/type-aliases/IdentifierType) que representa los identificadores activos.

  ## Pantallas compatibles

  * `login`
  * `login-id`
  * `reset-password-request`

  ```tsx Example
  import { useLoginIdentifiers } from '@auth0/auth0-acul-react/login';

  const loginIdentifiers = useLoginIdentifiers();
  const hasEmail = loginIdentifiers.includes('email');
  const hasPhone = loginIdentifiers.includes('phone');
  const hasUsername = loginIdentifiers.includes('username');

  // loginIdentifiers podría ser:
  // ['email', 'username']
  ```
