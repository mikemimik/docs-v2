<ParamField body="usePasskeyAutofill()" type={<span><a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsePasskeyAutofillResult">UsePasskeyAutofillResult</a></span>}>
  Hook de React que habilita la funcionalidad de UI condicional del navegador (autocompletado con clave de acceso) para los
  campos de identificador de inicio de sesión en las pantallas de Auth0 Advanced Customization of Universal Login (ACUL).

  Este hook inicializa la API Conditional Mediation del navegador, que muestra las
  claves de acceso almacenadas en el dispositivo en el menú desplegable de autocompletar del campo de username sin requerir
  interacción adicional del usuario.

  * El hook se registra **una vez por ciclo de vida de la página**: es seguro llamarlo varias veces.
  * **Se degrada correctamente** en navegadores no compatibles sin bloquear la interacción del usuario.
  * Vincular el `inputRef` devuelto a un elemento input garantiza que el atributo
    `autocomplete="username webauthn"` correcto se aplique automáticamente.
  * Si tu input ya declara `autocomplete="username webauthn"` en el marcado,
    **vincular la ref es opcional**: el hook se sigue registrando correctamente.

  ### Características principales

  * **Conditional Mediation automática**: inicializa el autocompletado con clave de acceso sin
    necesidad de hacer llamadas manuales a la API del navegador.
  * **Diseño con ref opcional**: funciona tanto si vinculas `inputRef` al elemento input como si no.
  * **Fallback sin fricción**: los entornos no compatibles se gestionan silenciosamente,
    preservando el flujo de login normal.

  ## Parámetros

  Ninguno.

  ## Devuelve

  [`UsePasskeyAutofillResult`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsePasskeyAutofillResult)

  Objeto que contiene:

  * `inputRef` *(opcional)* — una ref de React para adjuntarla al elemento input de username. Garantiza
    que el atributo `autocomplete` correcto se establezca al vincularla.

  ## Pantallas compatibles

  * `login-id`

  ```tsx Ejemplo
  import { usePasskeyAutofill } from '@auth0/auth0-acul-react/login-id';

  export function LoginIdentifierInput() {
    const { inputRef } = usePasskeyAutofill();

    return (
      <input
        ref={inputRef}
        type="text"
        name="username"
        placeholder="Email or username"
      />
    );
  }
  ```

  ## Observaciones

  * Si `inputRef` no está vinculado a un input, asegúrate de que el elemento input declare explícitamente
    `autocomplete="username webauthn"` para que el navegador pueda asociarlo con el autocompletado con clave de acceso.
  * El hook no tiene efecto en entornos donde la API Conditional Mediation no está disponible
    (por ejemplo, navegadores antiguos o contextos sin HTTPS): no se produce ningún error.
  * Llama a `usePasskeyAutofill` una vez en el nivel superior de tu componente; no lo llames
    de forma condicional ni dentro de controladores de eventos.
