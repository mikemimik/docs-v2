```tsx
usePasswordValidation(
  password: string,
  options?: { includeInErrors?: boolean },
): PasswordValidationResult
```

<ParamField body="usePasswordValidation" type={<span><a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordValidationResult">PasswordValidationResult</a></span>}>
  Este hook de React valida una contraseña según la política de contraseñas actual de Auth0
  y devuelve un resultado estructurado que indica si la contraseña cumple cada regla.

  Opcionalmente, puede enviar los resultados de la validación al gestor global de errores para que
  los componentes de error del formulario se actualicen automáticamente.

  ### Características principales

  * **Validación según la política** — comprueba la contraseña según las reglas de la política de contraseñas de Auth0 configuradas en el tenant.
  * **Integración con el gestor de errores** — opcionalmente muestra automáticamente los errores de validación en los componentes de error del formulario.

  ## Parámetros

  <ParamField body="password" type="string">
    La contraseña que se va a validar.
  </ParamField>

  <ParamField body="options" type="{ includeInErrors?: boolean }">
    Configuración opcional para el hook.

    <Expandable title="properties">
      <ParamField body="includeInErrors" type="boolean">
        Si es `true`, los errores de validación se almacenan en el gestor global de errores en el campo `password`. El valor predeterminado es `false`.
      </ParamField>
    </Expandable>
  </ParamField>

  ## Devuelve

  [`PasswordValidationResult`](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordValidationResult)

  Un objeto [PasswordValidationResult](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordValidationResult) que contiene:

  * `isValid` — `true` si la contraseña cumple todas las reglas configuradas.
  * `results` — un arreglo de resultados por regla con `code`, `label`, `status` e `isValid`.

  ## Pantallas compatibles

  * `signup`
  * `signup-password`
  * `reset-password`

  ```tsx Ejemplo
  import { usePasswordValidation } from '@auth0/auth0-acul-react/signup';

  const { isValid, results} = usePasswordValidation(password, { includeInErrors: true });

  if (!isValid) {
    console.log(results);
  }
  ```

  ## Observaciones

  * Llama a `usePasswordValidation` en el nivel superior de tu componente; no lo llames de forma condicional ni dentro de controladores de eventos.
  * La ruta de importación debe coincidir con la pantalla; usa `signup`, `signup-password` o `reset-password` según corresponda.
</ParamField>
