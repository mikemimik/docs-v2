La pantalla `interstitial-captcha` se muestra cuando se requiere un desafío CAPTCHA durante el flujo de autenticación. Le solicita al usuario que complete el CAPTCHA antes de continuar.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite **importación parcial** e **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso.
* La importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una única compilación que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useInterstitialCaptcha } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useInterstitialCaptcha,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Métodos
  submitCaptcha,
} from '@auth0/auth0-acul-react/interstitial-captcha';

function InterstitialCaptchaScreen() {
  const { submitCaptcha } = useInterstitialCaptcha();
  return (
    <button onClick={() => submitCaptcha({ captcha: 'token' })}>Submit</button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `interstitial-captcha`. Impórtalos desde `@auth0/auth0-acul-react/interstitial-captcha`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `interstitial-captcha`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/interstitial-captcha';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `interstitial-captcha`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/interstitial-captcha';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el desafío CAPTCHA tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/interstitial-captcha';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/interstitial-captcha';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Este hook contiene detalles específicos de la pantalla `interstitial-captcha`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/interstitial-captcha';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/interstitial-captcha';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `interstitial-captcha`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/interstitial-captcha';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja los datos no confiables que se pasan a la pantalla, como parámetros completados previamente a partir de los query strings de la URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/interstitial-captcha';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/interstitial-captcha';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useInterstitialCaptcha" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/InterstitialCaptchaMembers">InterstitialCaptchaMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `interstitial-captcha`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="submitCaptcha" type="Promise<void>">
  Este método envía el token de CAPTCHA para completar la verificación y continuar el flujo de autenticación.

  ```jsx Example
  import { useInterstitialCaptcha } from '@auth0/auth0-acul-react/interstitial-captcha';

  function SubmitButton() {
    const { submitCaptcha } = useInterstitialCaptcha();
    return (
      <button onClick={() => submitCaptcha({ captcha: 'token' })}>
        Continuar
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SubmitCaptchaOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/SubmitCaptchaOptions).
    </ParamField>

    <ParamField body="captcha" type="string" required>
      El token de respuesta del CAPTCHA.
    </ParamField>
  </Expandable>
</ParamField>
