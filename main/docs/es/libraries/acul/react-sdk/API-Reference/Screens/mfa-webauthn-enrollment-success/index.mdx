La pantalla `mfa-webauthn-enrollment-success` confirma que el usuario ha registrado correctamente una clave de seguridad de WebAuthn y le permite continuar con el flujo de autenticación.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useMfaWebAuthnEnrollmentSuccess } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useMfaWebAuthnEnrollmentSuccess,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Métodos
  continueMethod,
} from "@auth0/auth0-acul-react/mfa-webauthn-enrollment-success";

function MfaWebAuthnEnrollmentSuccessScreen() {
  const { continueMethod } = useMfaWebAuthnEnrollmentSuccess();
  return (
    <button onClick={() => continueMethod()}>
      Continue
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-webauthn-enrollment-success`. Impórtalos desde `@auth0/auth0-acul-react/mfa-webauthn-enrollment-success`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `mfa-webauthn-enrollment-success`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-webauthn-enrollment-success`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el flujo de MFA tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaWebAuthnEnrollmentSuccess">ScreenMembersOnMfaWebAuthnEnrollmentSuccess</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-webauthn-enrollment-success`, incluida su configuración y su contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-webauthn-enrollment-success`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja datos no confiables que se pasan a la pantalla, como valores completados previamente a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona detalles del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaWebAuthnEnrollmentSuccess" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaWebAuthnEnrollmentSuccessMembers">MfaWebAuthnEnrollmentSuccessMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `mfa-webauthn-enrollment-success`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Este método hace avanzar el flujo una vez que el usuario haya registrado correctamente una clave de seguridad de WebAuthn.

  ```jsx Example
  import { useMfaWebAuthnEnrollmentSuccess } from '@auth0/auth0-acul-react/mfa-webauthn-enrollment-success';

  function ContinueButton() {
    const { continueMethod } = useMfaWebAuthnEnrollmentSuccess();
    return (
      <button onClick={() => continueMethod()}>
        Continue
      </button>
    );
  }
  ```
</ParamField>
