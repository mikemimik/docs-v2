La pantalla `mfa-push-enrollment-qr` muestra un código QR para que el usuario lo escanee con la Auth0 Guardian app y complete la inscripción para las notificaciones push.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso.
* La importación desde la raíz te permite cargar todas las pantallas desde un solo bundle, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useMfaPushEnrollmentQr } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useMfaPushEnrollmentQr,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useMfaPolling,
  // Hooks de utilidad
  useChangeLanguage,
  // Métodos
  continueMethod,
  pickAuthenticator,
} from "@auth0/auth0-acul-react/mfa-push-enrollment-qr";

function MfaPushEnrollmentQrScreen() {
  const { pickAuthenticator } = useMfaPushEnrollmentQr();
  return (
    <button onClick={() => pickAuthenticator()}>
      Pick Authenticator
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-push-enrollment-qr`. Impórtalos desde `@auth0/auth0-acul-react/mfa-push-enrollment-qr`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de identidad visual, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `mfa-push-enrollment-qr`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-push-enrollment-qr`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la Organización del usuario si el flujo de MFA está limitado a una Organización. Devuelve `null` cuando no hay contexto de Organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaPushEnrollmentQr">ScreenMembersOnMfaPushEnrollmentQr</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-push-enrollment-qr`, incluida su configuración y su contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-push-enrollment-qr`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook gestiona datos no confiables que se pasan a la pantalla, como valores precargados a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```

</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaPushEnrollmentQr" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPushEnrollmentQrMembers">MfaPushEnrollmentQrMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `mfa-push-enrollment-qr`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Este método completa la inscripción de push después de que el usuario haya escaneado el código QR con Auth0 Guardian app.

  ```jsx Example
  import { useMfaPushEnrollmentQr } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';

  function ContinueButton() {
    const { continueMethod } = useMfaPushEnrollmentQr();
    return (
      <button onClick={() => continueMethod({ rememberDevice: true })}>
        Continue
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaPushEnrollmentQrWithRememberOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPushEnrollmentQrWithRememberOptions).
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      Si es `true`, se recordará el dispositivo y no se volverá a requerir MFA en este dispositivo.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickAuthenticator" type="Promise<void>">
  Este método dirige a la pantalla de selección de autenticador para que el usuario pueda elegir un método de MFA diferente para registrar.

  ```jsx Example
  import { useMfaPushEnrollmentQr } from '@auth0/auth0-acul-react/mfa-push-enrollment-qr';

  function PickAuthenticatorButton() {
    const { pickAuthenticator } = useMfaPushEnrollmentQr();
    return (
      <button onClick={() => pickAuthenticator()}>
        Pick Authenticator
      </button>
    );
  }
  ```
</ParamField>

<div id="commonutility-hooks">
  ## Hooks comunes y de utilidad
</div>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling">useMfaPolling</a>} type="Hooks">
  Este hook administra el estado de sondeo para las pantallas que esperan una respuesta asíncrona a una notificación push.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Este hook obtiene las opciones del tema actual con la configuración en formato plano del contexto de identidad visual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Este hook devuelve una función para cambiar el idioma que se muestra en la pantalla actual de ACUL.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Este hook obtiene el contexto y el estado de la pantalla actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Este hook lee y administra los errores del servidor, del cliente y del desarrollador en la pantalla.
</ParamField>
