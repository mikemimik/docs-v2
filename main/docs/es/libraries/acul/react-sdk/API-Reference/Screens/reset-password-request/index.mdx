La pantalla `reset-password-request` se muestra al inicio del flujo de restablecimiento de contraseña. En ella, se le pide al usuario que introduzca su nombre de usuario o correo electrónico para recibir un enlace de restablecimiento de contraseña.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** en cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso.
* La importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una sola compilación que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useResetPasswordRequest } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useResetPasswordRequest,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useLoginIdentifiers,
  // Hooks de utilidad
  useChangeLanguage,
  // Métodos
  backToLogin,
  resetPassword,
} from '@auth0/auth0-acul-react/reset-password-request';

function ResetPasswordRequestScreen() {
  const { resetPassword } = useResetPasswordRequest();
  return (
    <button onClick={() => resetPassword({ username: 'user@example.com' })}>
      Send Reset Link
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexto
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `reset-password-request`. Impórtalos desde `@auth0/auth0-acul-react/reset-password-request`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `reset-password-request`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-request';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `reset-password-request`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-request';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el restablecimiento de contraseña tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-request';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-request';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnResetPasswordRequest">ScreenMembersOnResetPasswordRequest</a></span>}>
  Este hook contiene detalles específicos de la pantalla `reset-password-request`, incluida su configuración y su contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/reset-password-request';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/reset-password-request';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembersOnResetPasswordRequest">TransactionMembersOnResetPasswordRequest</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `reset-password-request`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/reset-password-request';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja los datos no confiables que se pasan a la pantalla, como un correo electrónico o nombre de usuario precargado a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/reset-password-request';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook ofrece información sobre el usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/reset-password-request';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useResetPasswordRequest" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordRequestMembers">ResetPasswordRequestMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `reset-password-request`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="backToLogin" type="Promise<void>">
  Este método devuelve al usuario a la pantalla de inicio de sesión y cancela el flujo de restablecimiento de contraseña.

  ```jsx Example
  import { useResetPasswordRequest } from '@auth0/auth0-acul-react/reset-password-request';

  function BackToLoginButton() {
    const { backToLogin } = useResetPasswordRequest();
    return (
      <button onClick={() => backToLogin()}>
        Back to Login
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="resetPassword" type="Promise<void>">
  Este método envía el nombre de usuario o correo electrónico para iniciar el proceso de restablecimiento de contraseña.

  ```jsx Example
  import { useResetPasswordRequest } from '@auth0/auth0-acul-react/reset-password-request';

  function ResetPasswordForm() {
    const { resetPassword } = useResetPasswordRequest();
    return (
      <button onClick={() => resetPassword({ username: 'user@example.com' })}>
        Send Reset Link
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ResetPasswordRequestOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordRequestOptions).
    </ParamField>

    <ParamField body="username" type="string" required>
      El nombre de usuario o la dirección de correo electrónico de la cuenta que se va a restablecer.
    </ParamField>

    <ParamField body="captcha?" type="string">
      El token de respuesta de CAPTCHA, obligatorio cuando CAPTCHA está habilitado para el tenant.
    </ParamField>
  </Expandable>
</ParamField>

<div id="commonutility-hooks">
  ## Hooks comunes/de utilidad
</div>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useLoginIdentifiers">useLoginIdentifiers</a>} type="Hooks">
  Este hook devuelve los tipos de identificador de inicio de sesión (email, username, phone) configurados para la conexión actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Este hook obtiene las opciones del tema actual con la configuración simplificada desde el contexto de branding.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Este hook devuelve una función para cambiar el idioma de visualización en la pantalla actual de ACUL.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Este hook obtiene el contexto y el estado de la pantalla actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Este hook lee y gestiona los errores del servidor, del cliente y del desarrollador en la pantalla.
</ParamField>
