La pantalla `mfa-otp-enrollment-code` pide al usuario que introduzca un código de verificación para completar el registro de OTP de su aplicación de autenticación.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* La importación parcial permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useMfaOtpEnrollmentCode } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useMfaOtpEnrollmentCode,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks de utilidad
  useChangeLanguage,
  // Métodos
  continueMethod,
  toggleView,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-otp-enrollment-code";

function MfaOtpEnrollmentCodeScreen() {
  const { continueMethod } = useMfaOtpEnrollmentCode();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-otp-enrollment-code`. Impórtalos desde `@auth0/auth0-acul-react/mfa-otp-enrollment-code`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes de tema que se muestran en la pantalla `mfa-otp-enrollment-code`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-otp-enrollment-code`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el flujo de MFA tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaOtpEnrollmentCode">ScreenMembersOnMfaOtpEnrollmentCode</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-otp-enrollment-code`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-otp-enrollment-code`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja los datos no confiables que se pasan a la pantalla, como valores rellenados previamente a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaOtpEnrollmentCode" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaOtpEnrollmentCodeMembers">MfaOtpEnrollmentCodeMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `mfa-otp-enrollment-code`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Este método envía el código OTP introducido por el usuario para completar el registro.

  ```jsx Example
  import { useMfaOtpEnrollmentCode } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';

  function EnrollmentCodeForm() {
    const { continueMethod } = useMfaOtpEnrollmentCode();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Verificar
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaOtpContinueOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaOtpContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código OTP introducido por el usuario.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="toggleView" type="Promise<void>">
  Este método cambia la vista en la pantalla `mfa-otp-enrollment-code`, por ejemplo, entre la vista de código QR y la de entrada manual.

  ```jsx Example
  import { useMfaOtpEnrollmentCode } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';

  function ToggleViewButton() {
    const { toggleView } = useMfaOtpEnrollmentCode();
    return (
      <button onClick={() => toggleView()}>
        Cambiar vista
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método lleva a la pantalla de selección del método de MFA para que el usuario pueda elegir un factor de autenticación diferente.

  ```jsx Example
  import { useMfaOtpEnrollmentCode } from '@auth0/auth0-acul-react/mfa-otp-enrollment-code';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaOtpEnrollmentCode();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Probar otro método
      </button>
    );
  }
  ```
</ParamField>
