La pantalla `login-passwordless-email-code` solicita al usuario que introduzca un código de un solo uso enviado a su dirección de correo electrónico para completar la autenticación passwordless.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite **importación parcial** e **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso.
* La importación desde la raíz te permite cargar todas las pantallas en un solo paquete, lo que resulta útil cuando quieres una única compilación que gestione todas las pantallas posibles.

```jsx Import Example
// importación raíz
import { useLoginPasswordlessEmailCode } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useLoginPasswordlessEmailCode,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks de utilidad
  useChangeLanguage,
  useResend,
  // Métodos
  submitCode,
  resendCode,
  switchConnection,
} from '@auth0/auth0-acul-react/login-passwordless-email-code';

function EmailCodeForm() {
  const { submitCode } = useLoginPasswordlessEmailCode();
  return (
    <button onClick={() => submitCode({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexto
</div>

Hooks específicos de la pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `login-passwordless-email-code`. Impórtalos desde `@auth0/auth0-acul-react/login-passwordless-email-code`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y la configuración del tema que se muestran en la pantalla `login-passwordless-email-code`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `login-passwordless-email-code`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la Organización del usuario si el inicio de sesión está limitado a una Organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnLoginPasswordlessEmailCode">ScreenMembersOnLoginPasswordlessEmailCode</a></span>}>
  Este hook contiene detalles específicos de la pantalla `login-passwordless-email-code`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembersOnLoginPasswordlessEmailCode">TransactionMembersOnLoginPasswordlessEmailCode</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `login-passwordless-email-code`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook contiene datos no confiables que se pasan a la pantalla, como una dirección de correo electrónico completada previamente a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, como `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/login-passwordless-email-code';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useLoginPasswordlessEmailCode" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/LoginPasswordlessEmailCodeMembers">LoginPasswordlessEmailCodeMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `login-passwordless-email-code`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="resendCode" type="Promise<void>">
  Este método reenvía el código de un solo uso a la dirección de correo electrónico del usuario.

  ```jsx Example
  import { useLoginPasswordlessEmailCode } from '@auth0/auth0-acul-react/login-passwordless-email-code';

  function ResendButton() {
    const { resendCode } = useLoginPasswordlessEmailCode();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [CustomOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="submitCode" type="Promise<void>">
  Este método envía el código de un solo uso para completar la autenticación passwordless por correo electrónico.

  ```jsx Example
  import { useLoginPasswordlessEmailCode } from '@auth0/auth0-acul-react/login-passwordless-email-code';

  function EmailCodeForm() {
    const { submitCode } = useLoginPasswordlessEmailCode();
    return (
      <button onClick={() => submitCode({ code: '123456' })}>
        Verify Code
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SubmitCodeOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/SubmitCodeOptions).
    </ParamField>

    <ParamField body="code" type="string | number" required>
      El código de un solo uso enviado a la dirección de correo electrónico del usuario.
    </ParamField>

    <ParamField body="captcha?" type="string">
      Valor de captcha obligatorio cuando Bot Detection está habilitado en el tenant.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  Este método cambia a una conexión de autenticación alternativa sin reiniciar el flujo.

  ```jsx Example
  import { useLoginPasswordlessEmailCode } from '@auth0/auth0-acul-react/login-passwordless-email-code';

  function ConnectionSwitcher() {
    const { switchConnection } = useLoginPasswordlessEmailCode();
    return (
      <button onClick={() => switchConnection({ connection: 'enterprise-saml' })}>
        Use SSO
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SwitchConnectionOptionsEmailCode](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/SwitchConnectionOptionsEmailCode).
    </ParamField>

    <ParamField body="connection" type="string" required>
      El nombre de la conexión a la que se cambiará. Usa `email` o `sms` para passwordless, o el nombre de una conexión de base de datos para la autenticación basada en contraseña.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<div id="commonutility-hooks">
  ## Hooks comunes y de utilidad
</div>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Este hook obtiene las opciones del tema actual con la configuración consolidada del contexto de branding.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Este hook devuelve una función para cambiar el idioma de visualización en la pantalla actual de ACUL.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Este hook obtiene el contexto y el estado de la pantalla actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Este hook lee y administra los errores del servidor, del cliente y del desarrollador en la pantalla.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useResend">useResend</a>} type="Hooks">
  Este hook administra el tiempo de espera y la disponibilidad para las acciones de reenvío del código.
</ParamField>
