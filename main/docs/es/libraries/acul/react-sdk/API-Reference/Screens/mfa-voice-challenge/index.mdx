La pantalla `mfa-voice-challenge` pide al usuario que introduzca un código de un solo uso enviado por llamada de voz para completar la verificación de MFA.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite **importación parcial** e **importación raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// root import
import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react';

// partial import
import {
  useMfaVoiceChallenge,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  useResend,
  // Métodos
  continueMethod,
  pickPhone,
  resendCode,
  switchToSms,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-voice-challenge";

function MfaVoiceChallengeScreen() {
  const { continueMethod } = useMfaVoiceChallenge();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexto
</div>

Hooks de ámbito de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-voice-challenge`. Impórtalos desde `@auth0/auth0-acul-react/mfa-voice-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `mfa-voice-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-voice-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la Organización del usuario si el flujo de MFA tiene alcance de Organización. Devuelve `null` cuando no hay contexto de Organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaVoiceChallenge">ScreenMembersOnMfaVoiceChallenge</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-voice-challenge`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-voice-challenge`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembersOnMfaVoiceChallenge">UntrustedDataMembersOnMfaVoiceChallenge</a></span>}>
  Este hook gestiona los datos no confiables que se pasan a la pantalla, como valores rellenados previamente a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaVoiceChallenge" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaVoiceChallengeMembers">MfaVoiceChallengeMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `mfa-voice-challenge`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Este método envía el código de un solo uso recibido por llamada de voz para completar el desafío MFA.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function VerifyCodeForm() {
    const { continueMethod } = useMfaVoiceChallenge();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Verify Code
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaVoiceChallengeContinueOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaVoiceChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código de verificación recibido mediante llamada de voz.
    </ParamField>

    <ParamField body="rememberDevice" type="boolean">
      Indicador opcional para recordar el dispositivo durante 30 días y omitir futuros desafíos MFA.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickPhone" type="Promise<void>">
  Este método navega a la pantalla de selección de número de teléfono.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function PickPhoneButton() {
    const { pickPhone } = useMfaVoiceChallenge();
    return (
      <button onClick={() => pickPhone()}>
        Use a Different Phone Number
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Este método reenvía al teléfono del usuario el código de un solo uso de la llamada de voz.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function ResendCodeButton() {
    const { resendCode } = useMfaVoiceChallenge();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  Este método cambia el desafío MFA para que se envíe por SMS en lugar de por llamada de voz.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function SwitchToSmsButton() {
    const { switchToSms } = useMfaVoiceChallenge();
    return (
      <button onClick={() => switchToSms()}>
        Switch to SMS
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método navega a la pantalla de selección del método MFA para que el usuario pueda elegir un factor de autenticación diferente.

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaVoiceChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>

<div id="commonutility-hooks">
  ## Hooks comunes/de utilidad
</div>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useResend">useResend</a>} type="Hooks">
  Este hook gestiona el estado y la lógica del tiempo de espera para reenviar códigos en pantallas con funcionalidad de reenvío.

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Este hook obtiene las opciones del tema actual con la configuración consolidada desde el contexto de branding.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Este hook devuelve una función para cambiar el idioma de visualización en la pantalla actual de ACUL.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Este hook obtiene el contexto y el estado de la pantalla actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Este hook lee y gestiona los errores del servidor, del cliente y del desarrollador en la pantalla.
</ParamField>
