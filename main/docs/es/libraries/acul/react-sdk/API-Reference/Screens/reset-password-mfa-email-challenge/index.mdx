La pantalla `reset-password-mfa-email-challenge` se muestra durante el flujo de restablecimiento de contraseña cuando se requiere autenticación multifactor mediante correo electrónico. Solicita al usuario que introduzca un código de verificación enviado a su dirección de correo electrónico para continuar con el proceso de restablecimiento.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz te permite cargar todas las pantallas en un único paquete, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useResetPasswordMfaEmailChallenge } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useResetPasswordMfaEmailChallenge,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useResend,
  // Hooks de utilidad
  useChangeLanguage,
  // Métodos
  continueMethod,
  resendCode,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';

function ResetPasswordMfaEmailChallengeScreen() {
  const { continueMethod } = useResetPasswordMfaEmailChallenge();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `reset-password-mfa-email-challenge`. Impórtalos desde `@auth0/auth0-acul-react/reset-password-mfa-email-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes de tema que se muestran en la pantalla `reset-password-mfa-email-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `reset-password-mfa-email-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el restablecimiento de contraseña tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnResetPasswordMfaEmailChallenge">ScreenMembersOnResetPasswordMfaEmailChallenge</a></span>}>
  Este hook contiene detalles específicos de la pantalla `reset-password-mfa-email-challenge`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `reset-password-mfa-email-challenge`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook gestiona los datos no confiables que se pasan a la pantalla, como un correo electrónico o nombre de usuario precargado desde parámetros de URL.

  ```jsx Example

  function PrefilledForm() {

  }
  ```

</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  ```

  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `reset-password-mfa-email-challenge`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Este método envía el código de verificación por correo electrónico para continuar con el paso de MFA del flujo de restablecimiento de contraseña.

  ```jsx Example
  import { useResetPasswordMfaEmailChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';

  function VerifyCodeForm() {
    const { continueMethod } = useResetPasswordMfaEmailChallenge();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Verify Code
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [ContinueOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      El código de verificación introducido por el usuario.
    </ParamField>

    <ParamField body="rememberDevice" type="boolean">
      Indica si se debe recordar el dispositivo para futuras autenticaciones.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Este método reenvía al usuario el código de verificación por correo electrónico.

  ```jsx Example
  import { useResetPasswordMfaEmailChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';

  function ResendButton() {
    const { resendCode } = useResetPasswordMfaEmailChallenge();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método redirige al usuario para que seleccione otro método de MFA.

  ```jsx Example
  import { useResetPasswordMfaEmailChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useResetPasswordMfaEmailChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>

<div id="commonutility-hooks">

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useResend">useResend</a>} type="Hooks">
  Este hook gestiona el estado del tiempo de espera y el temporizador para reenviar el código de verificación en la pantalla actual.

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Este hook obtiene las opciones del tema actual con la configuración consolidada del contexto de branding.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Este hook devuelve una función para cambiar el idioma de visualización en la pantalla actual de ACUL.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Este hook obtiene el contexto y el estado de la pantalla actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Este hook lee y gestiona los errores del servidor, del cliente y del desarrollador en la pantalla.
