La pantalla `brute-force-protection-unblock` se muestra cuando Auth0 bloquea la cuenta de un usuario por demasiados intentos de inicio de sesión fallidos y le pide que verifique su identidad para desbloquear el acceso.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite **importación parcial** e **importación desde la raíz** para cada pantalla.

* La importación parcial permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz permite cargar todas las pantallas en un único paquete, lo que resulta útil cuando quieres una única compilación para gestionar todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useBruteForceProtectionUnblock } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useBruteForceProtectionUnblock,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Métodos
  unblockAccount,
} from '@auth0/auth0-acul-react/brute-force-protection-unblock';

function UnblockForm() {
  const { unblockAccount } = useBruteForceProtectionUnblock();
  return (
    <button onClick={() => unblockAccount()}>
      Unblock Account
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `brute-force-protection-unblock`. Impórtalos desde `@auth0/auth0-acul-react/brute-force-protection-unblock`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de imagen de marca, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `brute-force-protection-unblock`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `brute-force-protection-unblock`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el flujo de desbloqueo tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Sin contexto de organización</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnBruteForceProtectionUnblock">ScreenMembersOnBruteForceProtectionUnblock</a></span>}>
  Este hook contiene detalles específicos de la pantalla `brute-force-protection-unblock`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `brute-force-protection-unblock`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook gestiona los datos no confiables que se pasan a la pantalla, como un nombre de usuario rellenado previamente a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook ofrece información sobre el usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/brute-force-protection-unblock';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useBruteForceProtectionUnblock" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BruteForceProtectionUnblockMembers">BruteForceProtectionUnblockMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `brute-force-protection-unblock`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="unblockAccount" type="Promise<void>">
  Este método envía la solicitud para desbloquear la cuenta del usuario una vez que se ha activado la protección contra ataques de fuerza bruta.

  ```jsx Example
  import { useBruteForceProtectionUnblock } from '@auth0/auth0-acul-react/brute-force-protection-unblock';

  function UnblockForm() {
    const { unblockAccount } = useBruteForceProtectionUnblock();
    return (
      <button onClick={() => unblockAccount()}>
        Desbloquear cuenta
      </button>
    );
  }
  ```
</ParamField>
