La pantalla `consent` se muestra durante el flujo de autorización cuando un usuario necesita otorgar o denegar permisos a una aplicación. Presenta los scopes solicitados y permite al usuario aceptar o rechazar la solicitud de consentimiento.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz te permite cargar todas las pantallas desde un único bundle, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useConsent } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useConsent,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks de utilidad
  useChangeLanguage,
  // Métodos
  accept,
  deny,
} from '@auth0/auth0-acul-react/consent';

function ConsentScreen() {
  const { accept, deny } = useConsent();
  return (
    <>
      <button onClick={() => accept()}>Accept</button>
      <button onClick={() => deny()}>Deny</button>
    </>
  );
}
```

<div id="context-hooks">
  ## Hook de contexto
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `consent`. Impórtalos desde `@auth0/auth0-acul-react/consent`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y la configuración del tema que se muestran en la pantalla `consent`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/consent';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `consent`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/consent';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si la solicitud de consentimiento tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/consent';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/consent';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnConsent">ScreenMembersOnConsent</a></span>}>
  Este hook contiene detalles específicos de la pantalla `consent`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/consent';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/consent';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `consent`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/consent';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook gestiona datos no confiables que se pasan a la pantalla, como parámetros rellenados previamente desde la cadena de consulta de la URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/consent';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/consent';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useConsent" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ConsentMembers">ConsentMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `consent`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="accept" type="Promise<void>">
  Este método registra el consentimiento del usuario y concede los permisos solicitados a la aplicación.

  ```jsx Example
  import { useConsent } from '@auth0/auth0-acul-react/consent';

  function AcceptButton() {
    const { accept } = useConsent();
    return (
      <button onClick={() => accept()}>
        Aceptar
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="deny" type="Promise<void>">
  Este método rechaza la solicitud de consentimiento y revoca los permisos solicitados para la aplicación.

  ```jsx Example
  import { useConsent } from '@auth0/auth0-acul-react/consent';

  function DenyButton() {
    const { deny } = useConsent();
    return (
      <button onClick={() => deny()}>
        Rechazar
      </button>
    );
  }
  ```
</ParamField>

<div id="commonutility-hooks">
  ## Hooks comunes y de utilidad
</div>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Este hook obtiene las opciones del tema actual con la configuración plana del contexto de branding.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Este hook devuelve una función para cambiar el idioma que se muestra en la pantalla actual de ACUL.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Este hook obtiene el contexto y el estado de la pantalla actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Este hook lee y gestiona los errores del servidor, del cliente y del desarrollador en la pantalla.
</ParamField>
