<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// root import
import { useMfaEnrollResult } from '@auth0/auth0-acul-react';

// partial import
import {
  useMfaEnrollResult,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
} from "@auth0/auth0-acul-react/mfa-enroll-result";

function MfaEnrollResultScreen() {
  const screen = useMfaEnrollResult();
  return (
    <div>Enrollment complete</div>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexto
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-enroll-result`. Impórtalos desde `@auth0/auth0-acul-react/mfa-enroll-result`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona la configuración de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `mfa-enroll-result`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-enroll-result`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el flujo de MFA tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaEnrollResult">ScreenMembersOnMfaEnrollResult</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-enroll-result`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-enroll-result`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja datos no confiables que se pasan a la pantalla, como valores precargados a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-enroll-result';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaEnrollResult" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaEnrollResultMembers">MfaEnrollResultMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `mfa-enroll-result`.
</ParamField>

<div id="commonutility-hooks">
  ## Hooks comunes/de utilidad
</div>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Este hook obtiene las opciones del tema actual con la configuración plana del contexto de branding.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Este hook devuelve una función para cambiar el idioma que se muestra en la pantalla actual de ACUL.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Este hook obtiene el contexto y el estado de la pantalla actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Este hook lee y gestiona los errores del servidor, del cliente y del desarrollador en la pantalla.
</ParamField>
