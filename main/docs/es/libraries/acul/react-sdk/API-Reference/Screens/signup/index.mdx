La pantalla `registro` recopila las credenciales del usuario para crear una nueva cuenta. Admite el registro estándar con nombre de usuario, correo electrónico, número de teléfono y contraseña, así como el registro federado mediante conexiones sociales o empresariales.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite **importación parcial** e **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso.
* La importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useSignup } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useSignup,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  usePasswordValidation,
  useSignupIdentifiers,
  useUsernameValidation,
  // Métodos
  signup,
  federatedSignup,
  pickCountryCode,
} from '@auth0/auth0-acul-react/signup';

function SignupForm() {
  const { signup } = useSignup();
  return (
    <button onClick={() => signup({ email: 'user@example.com', password: 'secret' })}>
      Create Account
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `registro`. Impórtalos desde `@auth0/auth0-acul-react/signup`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `registro`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/signup';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `registro`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/signup';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el registro tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/signup';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/signup';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnSignup">ScreenMembersOnSignup</a></span>}>
  Este hook contiene detalles específicos de la pantalla `registro`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/signup';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/signup';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembersOnSignup">TransactionMembersOnSignup</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `registro`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/signup';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja datos no confiables que se pasan a la pantalla, como un correo electrónico o un `username` precargado a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/signup';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona detalles del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/signup';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useSignup" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/SignupMembers">SignupMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `registro`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="federatedSignup" type="Promise<void>">
  Este método inicia el registro a través de un proveedor de identidad federado, como Google o GitHub.

  ```jsx Example
  import { useSignup } from '@auth0/auth0-acul-react/signup';

  function SocialButton() {
    const { federatedSignup } = useSignup();
    return (
      <button onClick={() => federatedSignup({ connection: 'google-oauth2' })}>
        Sign up with Google
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [FederatedSignupPayloadOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/FederatedSignupPayloadOptions).
    </ParamField>

    <ParamField body="connection" type="string" required>
      El nombre de la conexión social o empresarial que se utilizará (por ejemplo, `google-oauth2`, `github`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  Este método abre el selector de código de país para introducir el número de teléfono durante el registro.

  ```jsx Example
  import { useSignup } from '@auth0/auth0-acul-react/signup';

  function CountryCodePicker() {
    const { pickCountryCode } = useSignup();
    return (
      <button onClick={() => pickCountryCode()}>
        Select Country Code
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [CustomOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="signup" type="Promise<void>">
  Este método envía los datos de registro del usuario para crear una cuenta nueva.

  ```jsx Example
  import { useSignup } from '@auth0/auth0-acul-react/signup';

  function SignupForm() {
    const { signup } = useSignup();
    return (
      <button onClick={() => signup({ email: 'user@example.com', password: 'secret' })}>
        Create Account
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SignupPayloadOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/SignupPayloadOptions).
    </ParamField>

    <ParamField body="email" type="string">
      La dirección de correo electrónico del usuario.
    </ParamField>

    <ParamField body="password" type="string">
      La contraseña de la cuenta nueva.
    </ParamField>

    <ParamField body="username" type="string">
      El nombre de usuario de la cuenta nueva, si la conexión lo requiere.
    </ParamField>

    <ParamField body="phoneNumber" type="string">
      El número de teléfono del usuario, si la conexión lo requiere.
    </ParamField>

    <ParamField body="captcha?" type="string">
      Valor de captcha obligatorio cuando Bot Detection está habilitado en el tenant.
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      Campos personalizados adicionales con el prefijo `ulp-` (por ejemplo, `'ulp-custom-field'`).
    </ParamField>
  </Expandable>
</ParamField>

<div id="commonutility-hooks">
  ## Hooks comunes/de utilidad
</div>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Este hook obtiene las opciones del tema actual con la configuración en formato plano del contexto de branding.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  Este hook devuelve una función para cambiar el idioma de visualización en la pantalla actual de ACUL.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Este hook obtiene el contexto y el estado de la pantalla actual.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Este hook lee y gestiona los errores del servidor, del cliente y del desarrollador en la pantalla.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation">usePasswordValidation</a>} type="Hooks">
  Este hook valida la contraseña del usuario según los requisitos de la política de contraseñas del tenant.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useSignupIdentifiers">useSignupIdentifiers</a>} type="Hooks">
  Este hook devuelve los tipos de identificador de registro disponibles (email, username, phone) configurados para la conexión.
</ParamField>

<ParamField body={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Hooks/useUsernameValidation">useUsernameValidation</a>} type="Hooks">
  Este hook valida el nombre de usuario según los requisitos de la política de nombres de usuario del tenant.
</ParamField>
