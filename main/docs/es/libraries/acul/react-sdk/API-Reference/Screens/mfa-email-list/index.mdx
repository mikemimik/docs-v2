La pantalla `mfa-email-list` muestra una lista de direcciones de correo electrónico disponibles para la verificación por MFA y permite al usuario seleccionar una.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite **importación parcial** e **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso.
* La importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useMfaEmailList } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useMfaEmailList,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Métodos
  goBack,
  selectMfaEmail,
} from "@auth0/auth0-acul-react/mfa-email-list";

function MfaEmailListScreen() {
  const { selectMfaEmail } = useMfaEmailList();
  return (
    <button onClick={() => selectMfaEmail({ index: 0 })}>
      Select Email
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexto
</div>

Hooks con alcance a nivel de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-email-list`. Impórtalos desde `@auth0/auth0-acul-react/mfa-email-list`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `mfa-email-list`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-email-list';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-email-list`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-email-list';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la Organización del usuario si el flujo de MFA está limitado a una Organización. Devuelve `null` cuando no hay contexto de Organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-email-list';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-email-list';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaEmailList">ScreenMembersOnMfaEmailList</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-email-list`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-email-list';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-email-list';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-email-list`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-email-list';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook gestiona datos no confiables que se pasan a la pantalla, como valores rellenados previamente a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-email-list';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook ofrece información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-email-list';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaEmailList" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaEmailListMembers">MfaEmailListMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `mfa-email-list`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="goBack" type="Promise<void>">
  Este método regresa a la pantalla anterior del flujo de MFA.

  ```jsx Example
  import { useMfaEmailList } from '@auth0/auth0-acul-react/mfa-email-list';

  function BackButton() {
    const { goBack } = useMfaEmailList();
    return (
      <button onClick={() => goBack()}>
        Volver
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="selectMfaEmail" type="Promise<void>">
  Este método selecciona una dirección de correo electrónico de la lista para usarla en el desafío MFA.

  ```jsx Example
  import { useMfaEmailList } from '@auth0/auth0-acul-react/mfa-email-list';

  function EmailSelector() {
    const { selectMfaEmail } = useMfaEmailList();
    return (
      <button onClick={() => selectMfaEmail({ index: 0 })}>
        Usar este correo electrónico
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [SelectMfaEmailOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/SelectMfaEmailOptions).
    </ParamField>

    <ParamField body="index" type="number" required>
      El índice de la dirección de correo electrónico que se quiere seleccionar.
    </ParamField>
  </Expandable>
</ParamField>
