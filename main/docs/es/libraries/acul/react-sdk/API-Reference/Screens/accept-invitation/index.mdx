La pantalla `accept-invitation` se muestra cuando se ha invitado a un usuario a unirse a una organización. Permite al usuario aceptar la invitación y continuar con el flujo de autenticación.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite **importación parcial** e **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil si quieres una compilación unificada para gestionar todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useAcceptInvitation } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useAcceptInvitation,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Métodos
  acceptInvitation,
} from '@auth0/auth0-acul-react/accept-invitation';

function AcceptInvitationScreen() {
  const { acceptInvitation } = useAcceptInvitation();
  return (
    <button onClick={() => acceptInvitation()}>Accept Invitation</button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `accept-invitation`. Impórtalos desde `@auth0/auth0-acul-react/accept-invitation`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `accept-invitation`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/accept-invitation';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `accept-invitation`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/accept-invitation';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la Organización a la que se ha invitado al usuario a unirse. Devuelve `null` cuando no hay contexto de Organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/accept-invitation';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/accept-invitation';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnAcceptInvitation">ScreenMembersOnAcceptInvitation</a></span>}>
  Este hook contiene detalles específicos de la pantalla `accept-invitation`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/accept-invitation';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y sus metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/accept-invitation';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `accept-invitation`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/accept-invitation';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook gestiona los datos no confiables que se pasan a la pantalla, como parámetros precargados de la query string de la URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/accept-invitation';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información sobre el usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/accept-invitation';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useAcceptInvitation" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/AcceptInvitationMembers">AcceptInvitationMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `accept-invitation`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="acceptInvitation" type="Promise<void>">
  Este método registra la aceptación del usuario de la invitación a la organización y continúa con el flujo de autenticación.

  ```jsx Example
  import { useAcceptInvitation } from '@auth0/auth0-acul-react/accept-invitation';

  function AcceptButton() {
    const { acceptInvitation } = useAcceptInvitation();
    return (
      <button onClick={() => acceptInvitation()}>
        Accept Invitation
      </button>
    );
  }
  ```
</ParamField>
