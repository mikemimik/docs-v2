description: "Describe todos los hooks y métodos disponibles para personalizar la pantalla `passkey-enrollment-local` de Universal Login."
---

La pantalla `passkey-enrollment-local` se muestra cuando se le pide al usuario que registre una clave de acceso en su dispositivo actual. Le permite continuar con la inscripción de la clave de acceso o cancelarla y, opcionalmente, suprimir prompts futuros.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso.
* La importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una compilación unificada que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { usePasskeyEnrollmentLocal } from '@auth0/auth0-acul-react';

// importación parcial
import {
  usePasskeyEnrollmentLocal,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks de utilidad
  useChangeLanguage,
  // Métodos
  continuePasskeyEnrollment,
  abortPasskeyEnrollment,
} from '@auth0/auth0-acul-react/passkey-enrollment-local';

function PasskeyEnrollmentLocalScreen() {
  const { continuePasskeyEnrollment } = usePasskeyEnrollmentLocal();
  return (
    <button onClick={() => continuePasskeyEnrollment()}>Enroll Passkey</button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `passkey-enrollment-local`. Impórtelos desde `@auth0/auth0-acul-react/passkey-enrollment-local`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `passkey-enrollment-local`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `passkey-enrollment-local`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si la inscripción de clave de acceso tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnPasskeyEnrollmentLocal">ScreenMembersOnPasskeyEnrollmentLocal</a></span>}>
  Este hook contiene detalles específicos de la pantalla `passkey-enrollment-local`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `passkey-enrollment-local`, como las conexiones activas y el estado actual del flujo.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja los datos no confiables que se pasan a la pantalla, como parámetros precompletados de la query string de la URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/passkey-enrollment-local';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="usePasskeyEnrollmentLocal" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasskeyEnrollmentLocalMembers">PasskeyEnrollmentLocalMembers</a>}>
  Este hook devuelve todos los métodos y el contexto disponibles en la pantalla `passkey-enrollment-local`.
</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="continuePasskeyEnrollment" type="Promise<void>">
  Este método inicia el proceso de inscripción de clave de acceso en el dispositivo actual y continúa el flujo de autenticación.

  ```jsx Example
  import { usePasskeyEnrollmentLocal } from '@auth0/auth0-acul-react/passkey-enrollment-local';

  function EnrollButton() {
    const { continuePasskeyEnrollment } = usePasskeyEnrollmentLocal();
    return (
      <button onClick={() => continuePasskeyEnrollment()}>
        Enroll Passkey
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="abortPasskeyEnrollment" type="Promise<void>">
  Este método cancela el proceso de inscripción de clave de acceso y continúa el flujo de autenticación sin inscribir una clave de acceso.

  ```jsx Example
  import { usePasskeyEnrollmentLocal } from '@auth0/auth0-acul-react/passkey-enrollment-local';

  function SkipButton() {
    const { abortPasskeyEnrollment } = usePasskeyEnrollmentLocal();
    return (
      <button onClick={() => abortPasskeyEnrollment({ doNotShowAgain: true })}>
        Skip and Don't Ask Again
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [AbortEnrollmentOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/AbortEnrollmentOptions).
    </ParamField>

    <ParamField body="doNotShowAgain" type="boolean">
      Cuando es `true`, evita que se muestren futuros prompts de inscripción de clave de acceso para este usuario.
    </ParamField>
  </Expandable>
</ParamField>
