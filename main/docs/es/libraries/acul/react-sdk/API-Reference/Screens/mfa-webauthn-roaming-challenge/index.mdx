La pantalla `mfa-webauthn-roaming-challenge` le pide al usuario que complete un desafío MFA con una clave de seguridad itinerante, como una YubiKey u otro autenticador de hardware FIDO2.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite **importación parcial** e **importación desde la raíz** para cada pantalla.

* La importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* La importación desde la raíz te permite cargar todas las pantallas desde un solo bundle, lo que resulta útil cuando quieres una única compilación que gestione todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useMfaWebAuthnRoamingChallenge,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Métodos
  reportWebAuthnError,
  tryAnotherMethod,
  verify,
} from "@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge";

function MfaWebAuthnRoamingChallengeScreen() {
  const { verify } = useMfaWebAuthnRoamingChallenge();
  return (
    <button onClick={() => verify({ rememberDevice: true })}>
      Verify with Security Key
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

Hooks con alcance de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-webauthn-roaming-challenge`. Impórtalos desde `@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de identidad visual, como el logotipo, los colores y los ajustes del tema que se muestran en la pantalla `mfa-webauthn-roaming-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-webauthn-roaming-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el flujo de MFA tiene alcance de organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaWebAuthnRoamingChallenge">ScreenMembersOnMfaWebAuthnRoamingChallenge</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-webauthn-roaming-challenge`, incluida su configuración y su contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-webauthn-roaming-challenge`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function TransactionInfo() {

  }
  ```

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>


<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>


</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="reportWebAuthnError" type="Promise<void>">
  Este método informa de un error de WebAuthn detectado durante el desafío de clave de seguridad itinerante.

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function HandleWebAuthnError({ error }) {
    const { reportWebAuthnError } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => reportWebAuthnError({ error })}>
        Report Error
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaWebAuthnRoamingChallengeReportErrorOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaWebAuthnRoamingChallengeReportErrorOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      El objeto de error de la API de WebAuthn (`navigator.credentials.get()`) que se va a informar.

      <ParamField body="name" type="string" required>
        El nombre del error (por ejemplo, `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        El mensaje de error que describe qué salió mal.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método navega a la pantalla de selección del método MFA para que el usuario pueda elegir un factor de autenticación diferente.

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="verify" type="Promise<void>">
  Este método inicia la verificación de la clave de seguridad itinerante para completar el desafío MFA.

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function VerifyButton() {
    const { verify } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => verify({ rememberDevice: true })}>
        Verify with Security Key
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [VerifySecurityKeyOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/VerifySecurityKeyOptions).
    </ParamField>

    <ParamField body="rememberDevice" type="boolean">
      Opcional. Si es true, intenta recordar el navegador para futuros desafíos MFA.
      Corresponde al campo de formulario `rememberBrowser`.
    </ParamField>
  </Expandable>
</ParamField>
