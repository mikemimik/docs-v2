La pantalla `mfa-phone-enrollment` permite al usuario registrar un número de teléfono para MFA por SMS o voz al introducir su número y seleccionar un método de entrega.

<div id="import">
  ## Importación
</div>

Cada pantalla tiene su propio conjunto de hooks y métodos. El SDK admite la **importación parcial** y la **importación desde la raíz** para cada pantalla.

* Usar la importación parcial te permite incluir solo el código que necesitas para tu caso de uso específico.
* Usar la importación desde la raíz te permite cargar todas las pantallas desde un único paquete, lo que resulta útil cuando quieres una compilación unificada para manejar todas las pantallas posibles.

```jsx Import Example
// importación desde la raíz
import { useMfaPhoneEnrollment } from '@auth0/auth0-acul-react';

// importación parcial
import {
  useMfaPhoneEnrollment,
  // Hooks de contexto
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks comunes
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks de utilidad
  useChangeLanguage,
  // Métodos
  continueEnrollment,
  pickCountryCode,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-phone-enrollment";

function MfaPhoneEnrollmentScreen() {
  const { continueEnrollment } = useMfaPhoneEnrollment();
  return (
    <button onClick={() => continueEnrollment({ phone: '+15551234567', type: 'sms' })}>
      Enroll Phone
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexto
</div>

Hooks de pantalla que proporcionan acceso de solo lectura a los datos de contexto de Auth0 en la pantalla `mfa-phone-enrollment`. Impórtalos desde `@auth0/auth0-acul-react/mfa-phone-enrollment`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Este hook proporciona configuraciones de branding, como el logotipo, los colores y los ajustes de tema que se muestran en la pantalla `mfa-phone-enrollment`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Este hook proporciona configuraciones relacionadas con el cliente, como `id`, `name` y `logoUrl`, para la pantalla `mfa-phone-enrollment`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Este hook proporciona información sobre la organización del usuario si el flujo de MFA está limitado a una organización. Devuelve `null` cuando no hay contexto de organización.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Este hook contiene datos sobre el prompt actual en el flujo de autenticación.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Este hook contiene detalles específicos de la pantalla `mfa-phone-enrollment`, incluida su configuración y contexto.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Este hook contiene datos relacionados con el tenant, como `id` y los metadatos asociados.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Este hook proporciona datos específicos de la transacción para la pantalla `mfa-phone-enrollment`, como el estado actual del flujo de MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Este hook maneja datos no confiables que se pasan a la pantalla, como valores rellenados previamente a partir de parámetros de URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Este hook proporciona información del usuario activo, incluidos `username`, `email` y los métodos de autenticación disponibles.

  ```jsx Ejemplo
  import { useUser } from '@auth0/auth0-acul-react/mfa-phone-enrollment';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaPhoneEnrollment" type={<a href="/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPhoneEnrollmentMembers">MfaPhoneEnrollmentMembers</a>}>

</ParamField>

<div id="methods">
  ## Métodos
</div>

<ParamField body="continueEnrollment" type="Promise<void>">
  Este método envía el número de teléfono del usuario y el método de entrega seleccionado para registrar el teléfono en MFA.

  ```jsx Example
  import { useMfaPhoneEnrollment } from '@auth0/auth0-acul-react/mfa-phone-enrollment';

  function PhoneEnrollmentForm() {
    const { continueEnrollment } = useMfaPhoneEnrollment();
    return (
      <button onClick={() => continueEnrollment({ phone: '+15551234567', type: 'sms' })}>
        Enroll Phone
      </button>
    );
  }
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaPhoneEnrollmentContinueOptions](/docs/es/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPhoneEnrollmentContinueOptions).
    </ParamField>

    <ParamField body="phone" type="string" required>
      El número de teléfono que se va a registrar para MFA.
    </ParamField>

    <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;" required>
      El método de entrega del código de verificación.

      * `sms`: Envía el código por mensaje de texto.
      * `voice`: Envía el código mediante una llamada de voz.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  Este método navega a la pantalla de selección de código de país para que el usuario pueda elegir el código de país de su número de teléfono.

  ```jsx Example
  import { useMfaPhoneEnrollment } from '@auth0/auth0-acul-react/mfa-phone-enrollment';

  function PickCountryCodeButton() {
    const { pickCountryCode } = useMfaPhoneEnrollment();
    return (
      <button onClick={() => pickCountryCode()}>
        Select Country Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Este método navega a la pantalla de selección de métodos de MFA para que el usuario pueda elegir un factor de autenticación diferente.

  ```jsx Example
  import { useMfaPhoneEnrollment } from '@auth0/auth0-acul-react/mfa-phone-enrollment';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaPhoneEnrollment();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>
