El Auth0 ACUL JS SDK te permite crear experiencias personalizadas de Universal Login con un esfuerzo mínimo. Ya sea que estés trabajando con inicio de sesión, registro, passwordless o inscripción de clave de acceso, este SDK está diseñado para gestionar fácilmente flujos de autenticación modernos.

<div id="why-choose-the-auth0-acul-js-sdk">
  ### ¿Por qué elegir Auth0 ACUL JS SDK?
</div>

* **Integración rápida:** Integra pantallas prediseñadas como `login-id` o `signup-password`.
* **Funcionalidades modernas:** Compatible con passwordless, claves de acceso e inicios de sesión con redes sociales.
* **Módulos flexibles:** Importa solo lo que necesitas.
* **Gestión unificada del estado:** Usa un único objeto de transacción para gestionar todos los flujos de autenticación.

<div id="architecture">
  ### Arquitectura
</div>

<div id="how-it-works">
  #### ¿Cómo funciona?
</div>

El SDK está diseñado pensando en la modularidad y la escalabilidad. Así es como se integra todo:

* **Pantallas:** Cada flujo de autenticación, como `login-id` y `signup-password`, es un módulo independiente.
* **Controladores de envío:** Gestionan los envíos de formularios y las transiciones.
* **Formularios dinámicos:** Adaptan los campos y las validaciones según la configuración de la pantalla.
* **Propiedades auxiliares:** Permiten acceder de forma coherente a datos comunes, como la identidad visual o la transacción.
