El Auth0 ACUL JS SDK te permite crear experiencias personalizadas de Universal Login con un esfuerzo mínimo. Tanto si trabajas con login, signup, passwordless o la inscripción de clave de acceso, este SDK está diseñado para gestionar fácilmente los flujos de autenticación modernos.

<div id="why-choose-the-auth0-acul-js-sdk">
  ### ¿Por qué elegir Auth0 ACUL JS SDK?
</div>

* **Integración rápida:** Incorpora pantallas prediseñadas como `login-id` o `signup-password`.
* **Funcionalidades modernas:** Compatibilidad con passwordless, claves de acceso e inicios de sesión con redes sociales.
* **Módulos flexibles:** Importa solo lo que necesites.
* **Gestión unificada del estado:** Usa un único objeto de transacción para gestionar todos los flujos de autenticación.

<div id="architecture">
  ### Arquitectura
</div>

<div id="how-it-works">
  #### ¿Cómo funciona?
</div>

El SDK está diseñado pensando en la modularidad y la escalabilidad. Así es como encaja todo:

* **Pantallas:** Cada flujo de autenticación, como `login-id` y `signup-password`, es un módulo independiente.
* **Manejadores de envío:** Gestionan los envíos de formularios y las transiciones.
* **Formularios dinámicos:** Adaptan los campos y las validaciones según la configuración de la pantalla.
* **Propiedades auxiliares:** Dan acceso a datos comunes, como la imagen de marca o la transacción, de forma coherente.
