El SDK de Auth0 ACUL React te permite crear experiencias personalizadas de Universal Login con el mínimo esfuerzo. Tanto si trabajas con inicio de sesión, registro, passwordless o inscripción de clave de acceso, este SDK está diseñado para gestionar con facilidad los flujos de autenticación modernos.

<div id="why-choose-the-auth0-acul-react-sdk">
  ### ¿Por qué elegir el SDK de Auth0 ACUL React?
</div>

* **Integración rápida:** Añade pantallas prediseñadas como `login-id` o `signup-password`.
* **Funcionalidades modernas:** Compatibilidad con passwordless, claves de acceso e inicios de sesión con redes sociales.
* **Módulos flexibles:** Importa solo lo que necesites.
* **Gestión unificada del estado:** Usa un único objeto de transacción para gestionar todos los flujos de autenticación.

<div id="architecture">
  ### Arquitectura
</div>

<div id="how-it-works">
  #### ¿Cómo funciona?
</div>

El SDK está diseñado con la modularidad y la escalabilidad en mente. Así es como se integra todo:

* **Pantallas:** Cada flujo de autenticación, como `login-id` y `signup-password`, es un módulo independiente.
* **Manejadores de envío:** Gestionan el envío de formularios y las transiciones.
* **Formularios dinámicos:** Adaptan los campos y las validaciones según la configuración de la pantalla.
* **Propiedades auxiliares:** Permiten acceder de forma coherente a datos comunes, como la marca o la transacción.
