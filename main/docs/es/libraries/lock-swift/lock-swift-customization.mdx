A continuación, se enumeran numerosas opciones para configurar el estilo y la apariencia de Lock. Además, en la página [Opciones de configuración del comportamiento](/docs/es/libraries/lock-swift/lock-swift-configuration-options) también hay varias opciones para modificar el comportamiento y la funcionalidad de Lock.

<div id="customizing-locks-appearance">
  ## Personalización de la apariencia de Lock
</div>

Puedes añadir opciones de personalización de estilo a la inicialización de Lock mediante `withStyle`.

```swift lines
Lock
  .classic()
  .withStyle {
	  $0.title = "Company LLC"
	  $0.logo = LazyImage(name: "company_logo")
	  $0.primaryColor = UIColor(red: 0.6784, green: 0.5412, blue: 0.7333, alpha: 1.0)
	}
  .present(from: self)
```

<div id="header-style-options">
  ## Opciones de estilo del encabezado
</div>

<div id="headerblur">
  ### headerBlur
</div>

Estilo de efecto de desenfoque que se utiliza. Puede ser cualquier valor definido en `UIBlurEffectStyle`.

```swift lines
.withStyle {
  $0.headerBlur = .extraLight
}
```

<div id="headercolor">
  ### headerColor
</div>

Color de fondo del encabezado. De forma predeterminada, no tiene color, solo un efecto de desenfoque.

```swift lines
.withStyle {
  $0.headerColor = UIColor? = nil
}
```

<div id="logo">
  ### logo
</div>

Logotipo del encabezado. La resolución mínima recomendada es de 200 píxeles (ancho) por 200 píxeles (alto).

```swift lines
.withStyle {
  $0.logo = LazyImage(name: "company_logo")
}
```

<div id="headercloseicon">
  ### headerCloseIcon
</div>

El icono de &quot;cerrar&quot; en el encabezado se puede cambiar.

```swift lines
.withStyle {
  $0.headerCloseIcon = LazyImage(name: "ic_close")
}
```

<div id="headerbackicon">
  ### headerBackIcon
</div>

El icono de &quot;volver&quot; en el encabezado se puede modificar.

```swift lines
.withStyle {
  $0.headerBackIcon = LazyImage(name: "ic_close")
}
```

<div id="title-style-options">
  ## Opciones de estilo para el título
</div>

<div id="hidetitle">
  ### hideTitle
</div>

Oculta el título del encabezado y muestra solo el logotipo. Esta opción es falsa de forma predeterminada.

```swift lines
.withStyle {
  $0.hideTitle = false
}
```

<div id="title">
  ### title
</div>

Texto del título que se usa en el encabezado

```swift lines
.withStyle {
  $0.title = "Company LLC"
}
```

<div id="titlecolor">
  ### titleColor
</div>

Color utilizado para el título del encabezado.

```swift lines
.withStyle {
  $0.titleColor = UIColor.black
}
```

<div id="oauth2">
  ### OAuth2
</div>

Los estilos de cualquier conexión OAuth2 que no sea de base de datos se pueden personalizar mediante el mapeo del nombre de la conexión a un `AuthStyle`

```swift lines
.withStyle {
  $0.oauth2["slack"] = AuthStyle(
    name: "Slack",
    color: UIColor(red: 0.4118, green: 0.8078, blue: 0.6588, alpha: 1.0),
    withImage: LazyImage(name: "ic_slack")
  )
}
```
