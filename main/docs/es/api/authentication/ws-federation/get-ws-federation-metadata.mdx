---
title: "Obtener los metadatos de WS-Federation"
description: "Devuelve los metadatos de WS-Federation de un tenant de Auth0."
---

<div id="endpoint">
  ## Endpoint
</div>

`GET /wsfed/FederationMetadata/2007-06/FederationMetadata.xml`

Este endpoint devuelve los metadatos de WS-Federation.

<div id="learn-more">
  ### Más información
</div>

* [WS-Federation](https://auth0.com/docs/authenticate/protocols/ws-fed-protocol)

  ## Mensajes de respuesta
</div>

| Estado | Descripción                                       |
| ------ | ------------------------------------------------- |
| 200    | Respuesta exitosa con metadatos de WS-Federation. |
