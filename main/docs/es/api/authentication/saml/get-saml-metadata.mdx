---
title: "Obtener metadatos"
description: "Devuelve los metadatos de SAML 2.0 de un tenant de Auth0."
---

<div id="endpoint">
  ## Endpoint
</div>

`GET /samlp/metadata/{clientId}`

Este endpoint devuelve los metadatos de SAML 2.0.

<div id="learn-more">
  ### Más información
</div>

* [SAML](https://auth0.com/docs/authenticate/protocols/saml/saml-configuration)

<div id="path-parameters">
  ## Parámetros de ruta
</div>

<ParamField path="clientId" type="string" required>
  El client&#95;id de tu aplicación.
</ParamField>

<div id="response-messages">
  ## Mensajes de respuesta
</div>

| Estado | Descripción                                      |
| ------ | ------------------------------------------------ |
| 200    | Los metadatos SAML se recuperaron correctamente. |
