title: "Aceptar solicitud"
description: "Acepta una solicitud de autenticación SAML para iniciar un flujo de inicio de sesión."
---

<div id="endpoint">
  ## Endpoint
</div>

`GET /samlp/{clientId}`

Use este endpoint para aceptar una solicitud SAML e iniciar sesión.

Opcionalmente, puede incluir un parámetro `connection` para iniciar sesión con un proveedor específico. Si no se especifica ninguna conexión, se mostrará la [Auth0 Login Page](https://auth0.com/docs/authenticate/login/auth0-universal-login).

Opcionalmente, las solicitudes de inicio de sesión iniciadas por SP pueden incluir un parámetro `organization` para autenticar a los usuarios en el contexto de una organización. Para obtener más información, consulte [Organizations](https://auth0.com/docs/manage-users/organizations).

<div id="path-parameters">
  ## Parámetros de ruta
</div>

<ParamField path="clientId" type="string" required>
  ID de cliente de tu aplicación.
</ParamField>

<div id="body-parameters">
  ## Parámetros del cuerpo
</div>

<ParamField body="connection" type="string">
  Conexión que se utilizará durante el inicio de sesión.
</ParamField>

<ParamField body="organization" type="string">
  ID de la organización, si la autenticación se realiza en el contexto de una organización.
</ParamField>

<div id="response-messages">
  ## Mensajes de respuesta
</div>

| Estado | Descripción                            |
| ------ | -------------------------------------- |
| 200    | Solicitud SAML aceptada correctamente. |
