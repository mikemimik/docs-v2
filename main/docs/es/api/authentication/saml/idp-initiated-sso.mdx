---

<div id="endpoint">
  ## Endpoint
</div>

`POST /login/callback`

Este endpoint acepta una SAMLResponse de iniciado por IdP Sign On de un proveedor de identidad SAML. La conexión correspondiente al proveedor de identidad se especifica en la cadena de consulta. Se redirigirá al usuario a la aplicación especificada en la sección iniciado por IdP Sign On de SAML Provider.

<div id="learn-more">
  ### Más información
</div>

* [SAML](https://auth0.com/docs/authenticate/protocols/saml/saml-configuration)

<div id="body-parameters">
  ## Parámetros del cuerpo
</div>

<ParamField body="connection" type="string" required>
  El nombre de un proveedor de identidad configurado en tu aplicación.
</ParamField>

<ParamField body="SAMLResponse" type="string" required>
  Una respuesta SAML de iniciado por IdP Sign On.
</ParamField>
