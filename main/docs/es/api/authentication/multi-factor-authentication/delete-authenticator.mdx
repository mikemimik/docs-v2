---
title: "Eliminar un autenticador"
description: "Eliminar un autenticador de MFA asociado mediante su ID."
---

<div id="endpoint">
  ## Endpoint
</div>

`DELETE /mfa/authenticators/{AUTHENTICATOR_ID}`

Elimina un autenticador asociado con su ID. Puedes obtener los ID de los autenticadores consultando la lista de autenticadores.

Para acceder a este endpoint, debes incluir un token de acceso en el encabezado Authorization, con los siguientes claims:

* `scope`: `remove:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="path-parameters">
  ## Parámetros de ruta
</div>

<ParamField path="AUTHENTICATOR_ID" type="string" required>
  El ID del autenticador que se va a eliminar.
</ParamField>

<div id="headers">
  ## Encabezados
</div>

<ParamField header="Authorization" type="string" required>
  El token de acceso obtenido durante el inicio de sesión.
</ParamField>

<div id="response-messages">
  ## Mensajes de respuesta
</div>

| Estado | Descripción                               |
| ------ | ----------------------------------------- |
| 204    | Autenticador eliminado correctamente.     |
| 401    | No autorizado: token de acceso no válido. |
| 404    | Autenticador no encontrado.               |
