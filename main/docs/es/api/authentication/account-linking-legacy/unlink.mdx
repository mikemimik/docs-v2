---

<div id="endpoint">
  ## Endpoint
</div>

`POST /login/unlink`

<Note>
  Este endpoint está **obsoleto**. En su lugar, debe usarse [DELETE /api/v2/users/{id}/identities/{provider}/{user_id}](/api/management/v2/users/delete-user-identity-by-user-id).
</Note>

Con el `access_token` y el `user_id` de un usuario que ha iniciado sesión, este endpoint desvincula la cuenta del usuario del proveedor de identidad.

  ### Más información
