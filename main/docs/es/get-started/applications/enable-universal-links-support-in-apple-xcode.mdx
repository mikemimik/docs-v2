title: Habilitar la compatibilidad con Universal Links en Apple Xcode
---

Los Universal Links establecen una relación verificada entre dominios y aplicaciones, por lo que tanto la configuración de tu aplicación de Auth0 como tu aplicación de iOS deben estar sincronizadas. Para ello, debes proporcionar a Auth0 el ID del equipo de Apple y el identificador del paquete. Puedes habilitar la compatibilidad con Universal Links para tu aplicación registrada en Auth0 desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="obtain-apple-team-id-and-bundle-identifier">
  ## Obtén el ID del equipo de Apple y el identificador del paquete
</div>

1. Para encontrar tu **ID del equipo de Apple**, ve a la [página de resumen de tu cuenta de desarrollador de Apple](https://developer.apple.com/membercenter/index.action#accountSummary).
2. Para encontrar el **identificador del paquete** de tu aplicación iOS, ve a la página de [configuración de tu proyecto en Xcode](https://developer.apple.com/library/content/documentation/IDEs/Conceptual/AppDistributionGuide/ConfiguringYourApp/ConfiguringYourApp.html).

<div id="provide-apple-team-id-and-bundle-identifier-to-auth0">
  ## Proporcione a Auth0 el ID del equipo de Apple y el identificador del paquete
</div>

1. Vaya a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/clients) y seleccione el nombre de la aplicación que quiere ver.

   <Frame>![Lista de aplicaciones del Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Desplácese hasta la parte inferior de la página **Settings** y seleccione **Show Advanced Settings**.
3. Seleccione **Device Settings**, introduzca los valores de **Team ID** y **App bundler identifier** para su aplicación de iOS y seleccione **Save Changes**.

   <Frame>![Dashboard Applications pestaña Settings de la aplicación pestaña Advanced Settings pestaña Device Settings](/docs/images/cdy7uua7fh8z/YKKPtWtswHn2FZGFSVWNw/086b32e41b3fd446ac8aa678be33648d/Device_Settings_-_EN.png)</Frame>

<div id="test-link">
  ## Probar el enlace
</div>

Comprueba si el archivo `apple-app-site-association` de los Universal Links está disponible para tu aplicación; para ello, abre en el navegador la siguiente URL: `{yourDomain}/apple-app-site-association`

Si el enlace funciona correctamente, verás el siguiente JSON (con formato para facilitar la lectura):

```json lines
{
  "applinks": {
    "apps": [],
    "details": [{
      "appID": "86WQXF56BC.com.auth0.Passwordless-Email",
      "paths": ["/ios/com.auth0.Passwordless-Email/*"]
    }]
  }
}
```
