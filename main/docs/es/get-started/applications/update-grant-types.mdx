Puedes cambiar los tipos de concesión de tu aplicación desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o con la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<div id="use-the-dashboard">
  ## Usa el Dashboard
</div>

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y selecciona el nombre de la aplicación que quieres consultar.

   <Frame>![Lista de aplicaciones del Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Desplázate hasta la parte inferior de la página y selecciona **Mostrar configuración avanzada**.
3. Selecciona **Tipos de concesión** y habilita o deshabilita las concesiones correspondientes para la aplicación. Cuando termines, selecciona **Guardar cambios**. El tipo de concesión de código de dispositivo solo está disponible para aplicaciones nativas.

   <Frame>![Pestaña Tipos de concesión de Configuración avanzada en Configuración de la aplicación de Applications del Dashboard](/docs/images/cdy7uua7fh8z/4ZMq4VfeGsKsKfZVFwSqe8/a0057749551915ac8d24b151bc27e875/Grant_Types_-_English.png)</Frame>

<div id="use-the-management-api">
  ## Usa la Management API
</div>

Haz una llamada `PATCH` al endpoint [`/Clients/patch_clients_by_id`](https://auth0.com/docs/api/management/v2#!/Clients/patch_clients_by_id). Asegúrate de reemplazar los valores de marcador de posición `{yourClientId}`, `{yourManagementApiAccessToken}` y `{grantType}` por tu <Tooltip tip="ID de cliente: valor de identificación asignado a tu recurso registrado por Auth0." cta="Ver glosario" href="/docs/es/glossary?term=client+ID">ID de cliente</Tooltip>, el <Tooltip tip="ID de cliente: valor de identificación asignado a tu recurso registrado por Auth0." cta="Ver glosario" href="/docs/es/glossary?term=access+token">access token</Tooltip> de la Management API y el tipo de concesión deseado, respectivamente.

    --header 'authorization: Bearer {yourMgmtApiAccessToken}' \

    --header 'content-type: application/json' \
    --data '{ "grant_types": "{grantTypes}" }'

| Valor                   | Descripción                                                                                                              |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| `{yourClientId}`        | El ID de la aplicación que se va a actualizar.                                                                           |
| `MGMT_API_ACCESS_TOKEN` | [Access Tokens para la Management API](https://auth0.com/docs/api/management/v2/tokens) con el `scope` `update:clients`. |
| `GRANT_TYPES`           | Los tipos de concesión que deseas habilitar para la aplicación especificada.                                             |

<div id="troubleshoot">
  ## Solución de problemas
</div>

Si se intenta usar un flujo con una aplicación que no tiene los `grant_types` adecuados para ese flujo (o con el campo vacío), se producirá el siguiente error:
``Tipo de concesión `grant_type` no permitido para el cliente.``
