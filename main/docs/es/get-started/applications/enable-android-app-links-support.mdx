title: Habilitar la compatibilidad con Android App Links
---

[Android App Links](https://developer.android.com/training/app-links) permiten que una aplicación se defina como la aplicación predeterminada para abrir un tipo determinado de enlace. Por ejemplo, al hacer clic en una URL en un correo electrónico, el enlace se abrirá en la aplicación designada. Esta guía te mostrará cómo habilitar la compatibilidad con Android App Links para tu aplicación registrada en Auth0 usando el Auth0 Dashboard.

1. Ve a [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y selecciona el nombre de la aplicación que quieres ver.

   <Frame>![Dashboard Applications List](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Desplázate hasta la parte inferior de la página **Settings** y selecciona **Show Advanced Settings**.
3. Selecciona **Device Settings**, proporciona el [App Package Name](https://developer.android.com/studio/build/application-id) y las huellas digitales SHA256 del certificado de firma de tu aplicación Android, y selecciona **Save Changes**.

   <Frame>![Dashboard Applications Application Settings Tab Advanced Settings Device Settings Tab](/docs/images/cdy7uua7fh8z/YKKPtWtswHn2FZGFSVWNw/086b32e41b3fd446ac8aa678be33648d/Device_Settings_-_EN.png)</Frame>

Puedes usar el siguiente comando para generar la huella digital con la herramienta keytool de Java en tu terminal:
`keytool -list -v -keystore my-release-key.keystore`

Para obtener más información sobre los certificados de firma, consulta la documentación para desarrolladores de Android sobre [Sign Your App](https://developer.android.com/studio/publish/app-signing.html).

<div id="test-link">
  ## Probar el enlace
</div>

1. Ve a la siguiente URL en tu navegador: `https://{yourDomain}/.well-known/assetlinks.json`
2. Si el enlace funciona correctamente, verás el siguiente JSON (con formato para facilitar la lectura):

   ```json lines
   [{
     "relation": ["delegate_permission/common.handle_all_urls"],
     "target": {
       "namespace": "android_app",
       "package_name": "com.mycompany.app1",
       "sha256_cert_fingerprints":
       ["14:6D:E9:83:C5:73:06:50:D8:EE:B9:95:2F:34:FC:64:16:A0:83:42:E6:1D:BE:A8:8A:04:96:B2:3F:CF:44:E5"]
     }
   }]
   ```

Para obtener más información sobre cómo probar tus Android App Links, consulta [Verify Android App Links](https://developer.android.com/training/app-links/verify-applinks) en https://developer.android.com.
