description: Aprende a configurar, actualizar y eliminar los metadatos de la aplicación (client_metadata y clientMetadata) en Advanced Settings de Applications del Auth0 Dashboard.
title: configurar los metadatos de la aplicación
---

Los metadatos de la aplicación son opcionales y consisten en claves y valores personalizables (máx. 255 caracteres cada uno) que puedes establecer para cada aplicación. Los metadatos se exponen en el objeto `Client` como `client_metadata`, y en Rules como `context.clientMetadata`. Por ejemplo, puedes almacenar la URL de la página de inicio de la aplicación (un campo que Auth0 no proporciona de forma predeterminada en la configuración de la aplicación).

Los metadatos del cliente se almacenan como parte de las propiedades de la aplicación (cliente). Para obtener más información sobre tipos de datos, nombres de campos y limitaciones de almacenamiento, consulta [Metadata Field Names and Data Types](/docs/es/manage-users/user-accounts/metadata/metadata-fields-data).

<Card title="Dónde almacenar secretos de cliente">
  El lugar donde almacenar el secreto depende del alcance del secreto:

  * ¿Hay un solo secreto por aplicación? Entonces `client_metadata` sería un buen lugar.
  * ¿Es el mismo secreto para todo el sistema (es decir, para todas o muchas aplicaciones)? Entonces los valores de configuración de la rule podrían ser una mejor opción.
  * ¿Es un secreto diferente para cada usuario? Entonces podría ser mejor almacenarlo en `app_metadata`.
</Card>

<Warning>
  Los claims del token de ID no están cifrados, así que, dependiendo del flujo que uses, el usuario podría obtener el token e inspeccionar su contenido. Auth0 **no** recomienda almacenar un secreto de esa forma.
</Warning>

Las aplicaciones existentes no tendrán ningún valor para esta propiedad.

Puedes acceder a los metadatos de la aplicación en [Actions](/docs/es/customize/actions):

```js lines
exports.onExecutePostLogin = async (event, api) => {
  if (event.client.metadata.SKIP_VERIFICATION === "yes"){
    return;
  }
  // ... continuar este Action
}
```

... o en [Rules](/docs/es/customize/rules):

```javascript lines
function applicationMetadataExample (user, context, callback){
  context.clientMetadata = context.clientMetadata || {};
  if (context.clientMetadata.SKIP_VERIFICATION === "yes"){
    return callback();
  }
  // ... continuar este Rule
}
```

Puedes consultar y agregar metadatos de la aplicación desde el Dashboard o la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Para obtener información sobre cómo administrar los metadatos del cliente con la Management API, lee [Administrar metadatos con la Management API](/docs/es/manage-users/user-accounts/metadata/manage-metadata-api).

<div id="add-application-metadata-keyvalue-pairs">
  ## Agrega pares de clave/valor en los metadatos de la aplicación
</div>

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y selecciona la aplicación.
2. Desplázate hacia abajo y haz clic en **Advanced Settings**.
3. En la pestaña **Application Metadata**, introduce el nombre de la clave y el valor, y luego haz clic en **Add**.

   <Frame>![Dashboard Applications Applications Settings Tab Advanced Settings Application Metadata Tab](/docs/images/cdy7uua7fh8z/7GWHyQloNihda3fuYiVEWQ/b3cced7c53690e6f3cbf7b2e23ff8646/App_Metadata_-_English.png)</Frame>
4. Haz clic en **Save Changes**.

<div id="update-application-metadata-value">
  ## Actualizar el valor de los metadatos de la aplicación
</div>

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y selecciona la aplicación.
2. Desplázate hacia abajo y haz clic en **Advanced Settings**.
3. En la pestaña **Application Metadata**, introduce el nombre de la clave que quieres cambiar y un nuevo valor; luego, haz clic en **Add**.
4. Haz clic en **Save Changes**

<div id="delete-application-metadata">
  ## Eliminar los metadatos de la aplicación
</div>

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y selecciona la aplicación.
2. Desplázate hacia abajo y haz clic en **Advanced Settings**.
3. En la pestaña **Application Metadata**, localiza el par clave-valor que quieres eliminar y haz clic en el icono de la papelera.
4. Confirma la eliminación.
5. Haz clic en **Save Changes**.
