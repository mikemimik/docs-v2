description: Describe cómo cambiar el algoritmo de firma de una aplicación con Auth0 Dashboard.
title: Cambiar los algoritmos de firma de una aplicación
---

Puedes cambiar el <Tooltip tip="Algoritmo de firma: algoritmo utilizado para firmar digitalmente tokens y garantizar que el token no se haya manipulado." cta="Ver glosario" href="/docs/es/glossary?term=signing+algorithm">algoritmo de firma</Tooltip> de tu aplicación desde el Dashboard.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y selecciona el nombre de la aplicación para verla.

   <Frame>![Lista de aplicaciones del Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Desplázate hasta la parte inferior de la pestaña **Settings** y selecciona **Show Advanced Settings**.
3. Ve a la pestaña **OAuth**.

   <Frame>![Pestaña OAuth de Application Settings en Applications del Dashboard](/docs/images/cdy7uua7fh8z/5bbq6VrI9tUMht0PwMimre/08574482efef944b16ecfbb445be50ab/Oauth_Tab_-_English.png)</Frame>
4. Busca **JsonWebToken Signature Algorithm** y selecciona el algoritmo de firma adecuado para la aplicación.
5. Haz clic en **Save Changes**.
