<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Para usar las funciones de Identidad altamente regulada, debes tener un Enterprise Plan con el add-on Highly Regulated Identity. Consulta [Auth0 Pricing](https://auth0.com/pricing/) para más información.
</Callout>

Usa la [Auth0 Management API](https://auth0.com/docs/api/management/v2) o el [Auth0 Dashboard](https://manage.auth0.com/) para configurar la autenticación mTLS. Para obtener más información sobre cómo funciona la autenticación mTLS en Auth0, consulta [Authenticate with mTLS](/docs/es/get-started/authentication-and-authorization-flow/authenticate-with-mtls).
