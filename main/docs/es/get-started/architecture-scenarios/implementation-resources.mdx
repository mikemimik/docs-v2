Auth0 ofrece una amplia variedad de recursos para ayudarle a aprovechar eficazmente nuestro producto y nuestra comunidad. Esta lista incluye enlaces a los recursos disponibles por categoría.

<div id="get-started">
  ## Primeros pasos
</div>

Los recursos diseñados para ayudarte a conocer los conceptos básicos de Auth0 incluyen:

* [**Documentación de primeros pasos**](/docs/es/get-started): Explora el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> y los términos más comunes que se usan para los componentes del servicio de Auth0. Obtén una visión general de Auth0 y aprende la terminología que podrías encontrar al trabajar con el personal de Auth0 o al leer nuestra documentación.
* [**Escenarios de arquitectura**](/docs/es/get-started/architecture-scenarios): Revisa escenarios de arquitectura comunes y aprende a implementarlos con Auth0. Los escenarios incluyen tutoriales sobre patrones de arquitectura habituales, como una Aplicación de página única (SPA) que llama a una API. Las descripciones de alto nivel son útiles para arquitectos, mientras que los tutoriales ayudarán a los equipos de desarrollo.
* [**Guías de implementación**](/docs/es/): Aprende a implementar funciones de uso frecuente, como la gestión de usuarios y la <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuario que usa un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> (MFA). Esta información es útil tanto para arquitectos como para desarrolladores

<div id="learn">
  ## Aprende
</div>

Auth0 ofrece numerosos tutoriales, guías, informes técnicos y publicaciones del blog pensados tanto para aprender como para consultar referencias rápidas.

* [**Sitio de documentación**](/docs/es/): Recorre nuestra documentación para explorar una gran variedad de temas, o usa el buscador para encontrar rápidamente contenido relacionado con un tema o término.
* [**Descripciones de funcionalidades e informes técnicos**](https://auth0.com/learn/): Consulta breves descripciones de funcionalidades, casos de uso del sector e informes técnicos de referencia. El sitio Auth0 Learn ofrece una visión general rápida de las funcionalidades de Auth0 y de su valor para el negocio; resulta útil tanto para responsables de proyectos como para arquitectos y desarrolladores.
* [**Publicaciones del blog**](https://auth0.com/blog/): Lee artículos del blog escritos por expertos sobre una amplia variedad de temas, desde consejos atemporales hasta noticias de última hora del ámbito de la identidad. Muchas de estas publicaciones están dirigidas a arquitectos y desarrolladores.

<div id="run-sample-code">
  ## Ejecuta código de ejemplo
</div>

Una vez que tu equipo de desarrollo esté listo para crear, Auth0 proporciona programas de ejemplo, SDKs y bibliotecas para acelerar tu proyecto.

* [**Quickstarts**](/docs/es/quickstarts): Explora una amplia variedad de pequeños programas de ejemplo que muestran cómo implementar las funciones clave que querrás incluir en tu aplicación, como la autenticación, la gestión de sesiones, las actualizaciones de perfil y el cierre de sesión. Quickstarts les da a los desarrolladores una ventaja inicial para entender cómo integrar aplicaciones con Auth0.
* [Libraries and SDKs](/docs/es/libraries): Simplifica el desarrollo de tu aplicación personalizada con nuestro amplio conjunto de bibliotecas y SDKs, que abstraen muchos de los detalles de los protocolos de identidad. Los SDKs de Auth0 son compatibles con numerosos lenguajes y frameworks para simplificar tu integración con Auth0. También ofrecemos una biblioteca para Lock, un widget de login, que puedes usar en varias plataformas, incluidas iOS y Android.
* [**Management API**](https://auth0.com/docs/api/management/v2): Explora y manipula objetos, configuraciones y ajustes dentro de Auth0. El <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> Explorer te permite manipular rápidamente objetos y ajustes individuales de forma ad hoc, y probar llamadas a la API antes de incorporarlas a tus aplicaciones.
* [**Authentication API**](https://auth0.com/docs/api/authentication): Autentica y autoriza usuarios mediante los protocolos OIDC, <Tooltip tip="OAuth 2.0: Framework de autorización que define protocolos y flujos de trabajo de autorización." cta="Ver glosario" href="/docs/es/glossary?term=OAuth">OAuth</Tooltip>, y <Tooltip tip="OAuth 2.0: Framework de autorización que define protocolos y flujos de trabajo de autorización." cta="Ver glosario" href="/docs/es/glossary?term=SAML">SAML</Tooltip> . El Authentication API Explorer te permite experimentar con la autenticación y los <Tooltip tip="Flujo de autorización: Concesión de autorización (o flujo de trabajo) especificada en el framework OAuth 2.0." cta="Ver glosario" href="/docs/es/glossary?term=authorization+flows">flujos de autorización</Tooltip>, y probar llamadas a la API antes de incorporarlas a tus aplicaciones.

<div id="try-out-features-and-api-calls">
  ## Prueba funciones y llamadas a la API
</div>

Mientras desarrollas, Auth0 te permite experimentar y probar varias funciones del producto.

* **Botones/enlaces TRY**: Prueba rápidamente las funciones del producto de Auth0. Los botones **TRY** se encuentran en todo Auth0 y te permiten experimentar con conexiones, reglas, hooks y plantillas de correo electrónico.
* [**Authentication API Explorer**](https://auth0.com/docs/api/authentication#introduction): Experimenta con los flujos de authentication y authorization, y prueba llamadas a la API antes de incorporarlas a tus aplicaciones.
* [**Management API Explorer**](https://auth0.com/docs/api/management/v2): Manipula rápidamente objetos y configuraciones individuales según sea necesario, y prueba llamadas a la API antes de incorporarlas a tus aplicaciones.

<div id="get-help">
  ## Obtener ayuda
</div>

Los recursos de Auth0 que pueden ayudarle a resolver problemas con su implementación incluyen:

* [**Auth0 Community**](https://community.auth0.com/): Conéctese con el mundo de Auth0 a través de publicaciones, preguntas frecuentes y preguntas y respuestas de la comunidad. Los arquitectos y desarrolladores encontrarán aquí una valiosa fuente de información para aprender, conectar con otras personas y obtener ayuda con problemas.
* [**Support Center**](https://support.auth0.com/): Vea y administre su suscripción y tenants, envíe y consulte solicitudes de soporte, ejecute comprobaciones automatizadas para producción en un tenant y consulte información de cumplimiento. Los suscriptores de pago encontrarán en el centro de soporte un recurso valioso si un problema o una pregunta no pueden resolverse con la documentación o buscando en el foro.

  * [**Crear casos de soporte**](/docs/es/troubleshoot/customer-support/open-and-manage-support-tickets): Cree y envíe un caso de soporte si necesita ayuda.
  * [**Planes de soporte y acuerdos de nivel de servicio**](/docs/es/troubleshoot/customer-support#defect-responses): Conozca los distintos niveles de soporte disponibles para su compra.
  * [**Consejos para la solución de problemas**](/docs/es/deploy-monitor/deploy-private-cloud/private-cloud-on-aws#what-to-check-before-logging-an-issue) y [**Información para incluir en su caso de soporte**](/docs/es/deploy-monitor/deploy-private-cloud/private-cloud-on-aws#information-to-provide-when-logging-an-issue): Obtenga orientación para ayudar a sus equipos de desarrollo y soporte a analizar problemas.
* [**Versiones compatibles**](/docs/es/troubleshoot/customer-support/product-support-matrix): Comprenda qué versiones de SDKs, navegadores y lenguajes son compatibles. Los arquitectos y desarrolladores deberían revisar esto para asegurarse de que su proyecto emplea lenguajes, bibliotecas y SDKs que permitirán que su implementación funcione con Auth0.
* [**Portal de comentarios**](https://auth0.com/feedback): Haga sugerencias sobre el producto. (También puede hacerlo a través del Support Center si desea tener mejor visibilidad de lo que ha enviado con el tiempo). Los arquitectos y desarrolladores pueden usar este sitio para enviar comentarios sobre el producto Auth0 con vistas a futuras mejoras.
* [**Servicios profesionales**](/docs/es/get-started/professional-services): Recurra a nuestro equipo global de servicios profesionales para acelerar el éxito de su proyecto. Los responsables de proyectos encontrarán esto útil para conocer cómo los expertos en identidad de Auth0 pueden ayudar a acelerar su proyecto o cubrir cualquier carencia temporal de conocimientos.

<div id="set-up-and-monitor-operations">
  ## Configura y supervisa las operaciones
</div>

Cuando estés listo para planificar tu lanzamiento, Auth0 ofrece los siguientes recursos:

* [**Consejos previos al lanzamiento**](/docs/es/deploy-monitor/pre-deployment-checks) y [**Verificación de producción**](/docs/es/deploy-monitor/pre-deployment-checks/how-to-run-production-checks): Obtén consejos y herramientas para ayudarte a planificar tu lanzamiento. Los gerentes de proyecto y los equipos de desarrollo y operaciones deberían revisar estas recomendaciones para lograr un lanzamiento sin contratiempos.
* [**Políticas operativas**](/docs/es/troubleshoot/customer-support/operational-policies): Familiarízate con las políticas de Auth0 para conocer los plazos de las solicitudes operativas. Son útiles para todo el equipo, pero especialmente los responsables del proyecto y los equipos de operaciones deberían conocer las políticas operativas de Auth0.
* [**Dashboard de estado**](https://status.auth0.com/): Determina rápidamente la disponibilidad de los servicios de Auth0 y suscríbete a las actualizaciones de estado. Aunque las interrupciones del servicio son poco frecuentes, cuando ocurre una, Auth0 realiza un análisis de la causa raíz y publica los resultados en este sitio. Los equipos de operaciones y soporte deberían saber cómo consultar el estado de Auth0.
* [**Monitorea endpoints**](/docs/es/deploy-monitor): Aprende a integrar nuestros endpoints de monitoreo en tu infraestructura de monitoreo. Esta información es especialmente útil para los equipos de operaciones y los responsables de proyecto.
* [**Datos de logs**](/docs/es/deploy-monitor/logs): Obtén información sobre los tipos de logs que ofrece Auth0, la retención de datos de logs y las herramientas que puedes usar para exportar datos de logs a herramientas analíticas externas para su análisis y almacenamiento a largo plazo. Esta información es útil para desarrolladores y equipos de operaciones, así como para equipos de cumplimiento interesados en la retención de datos.
* [**Avisos del Dashboard**](/docs/es/get-started/architecture-scenarios/business-to-consumer/operations#notifications): Mantente informado sobre anuncios importantes de Auth0. De vez en cuando, Auth0 te notificará información importante a través de tu Auth0 Dashboard y (según la gravedad de la información) por correo electrónico a los Admins registrados de tu Auth0 Dashboard. Deberías iniciar sesión regularmente en el Dashboard y revisar el ícono de campana en la parte superior para ver si hay avisos importantes.
