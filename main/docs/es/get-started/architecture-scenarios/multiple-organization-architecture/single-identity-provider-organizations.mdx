title: Organizaciones con proveedor de identidad único
---

En el escenario de organizaciones con <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=Identity+Provider">proveedor de identidad único</Tooltip> (IdP), cada organización que usa la función [Auth0 Organizations](/docs/es/manage-users/organizations/organizations-overview) utiliza exactamente una conexión de Auth0. Esta función permite que cada aplicación seleccione la organización adecuada definida en el Auth0 Tenant, que autenticará directamente las credenciales del usuario o reenviará la solicitud de autenticación al IdP correspondiente para que la procese.

En este escenario, seguiremos usando el ejemplo de [Travel0 Corporate Booking](/docs/es/get-started/architecture-scenarios/multiple-organization-architecture). En este ejemplo, la aplicación de reservas corporativas equivale a su aplicación, y Travel0 corresponde a su empresa.

<Frame>![Escenarios de arquitectura: multitenencia: diagrama: usuarios aislados, aplicaciones compartidas, organizaciones A y B](/docs/images/cdy7uua7fh8z/1YBaMaMozamiTv8hXmNB2D/21e9270a59021a12209bdcb4345147cb/isolated-users_shared-apps_org-ab.png)</Frame>

<Warning>
  La función Auth0 Organizations actualmente admite aplicaciones que usan los protocolos [SAML](/docs/es/authenticate/protocols/saml/saml-configuration), [Open ID Connect](/docs/es/authenticate/protocols/openid-connect-protocol) y/o el marco de trabajo [OAuth 2](/docs/es/authenticate/protocols/oauth). Las aplicaciones que requieren [WS-Fed](/docs/es/authenticate/protocols/ws-fed-protocol) no son compatibles.
</Warning>

Independientemente de cómo se procesen las credenciales —ya sea que las credenciales del usuario se almacenen en una conexión de base de datos (como en Hoekstra &amp; Associates), se validen mediante una conexión empresarial (como en MetaHexa Bank) o mediante una conexión social—, su aplicación se comporta de la misma manera. Aprovechar la función [Organizations](/docs/es/manage-users/organizations/organizations-overview) le permite configurar y controlar la experiencia.
