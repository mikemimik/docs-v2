En el [Escenario de arquitectura de servidor + API](/docs/es/get-started/architecture-scenarios/server-application-api), abordamos un escenario sencillo de máquina a máquina: una API utilizada para importar entradas de hojas de hora en los sistemas de ExampleCo y una tarea cron utilizada por contratistas externos para enviar sus hojas de hora mediante esta API.

Aprendimos sobre la Concesión de credenciales de cliente y los <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+tokens">tokens de acceso</Tooltip>, cómo configurar una API en Auth0, cómo configurar una Machine-to-Machine Application para comunicarse de forma segura con esta API, cómo definir y proteger nuestros endpoints de API, cómo usar las bibliotecas proporcionadas para validar el token de acceso y cómo obtener uno nuevo de Auth0.

Usamos Node.js para la implementación de la API y Python para el proceso de servidor no interactivo. Esperamos que este ejemplo te ayude a crear tu propia implementación con las tecnologías que prefieras.
