<div id="get-an-access-token">
  ## Obtener un token de acceso
</div>

Para realizar la solicitud HTTP al endpoint de la API de Auth0 `/oauth/token`, usaremos las bibliotecas `json`, `urllib` y `urllib2`.

Esta es nuestra implementación de ejemplo:

export const codeExample = `def main():
  import json, urllib, urllib2, httplib

  # Valores de configuración
  domain = "{yourDomain}" # Tu dominio de Auth0
  api_identifier = "API_IDENTIFIER" # Identificador de la API de tu API
  client_id = "{yourClientId}" # ID de cliente de tu Aplicación de máquina a máquina
  client_secret = "{yourClientSecret}" # Secreto de cliente de tu Aplicación de máquina a máquina
  api_url = "http://localhost:8080/timesheets/upload"
  grant_type = "client_credentials" # Flujo de OAuth 2.0 que se va a usar

  # Obtener un token de acceso desde Auth0
  base_url = "https://{domain}".format(domain=domain)
  data = urllib.urlencode({'client_id': client_id,
                            'client_secret': client_secret,
                            'audience': api_identifier,
                            'grant_type': grant_type})
  req = urllib2.Request(base_url + "/oauth/token", data, headers={"Accept": "application/x-www-form-urlencoded"})
  response = urllib2.urlopen(req)
  resp_body = response.read()
  oauth = json.loads(resp_body)
  access_token = oauth['access_token']

# Código estándar para llamar a la función main().
if __name__ == '__main__':
  main()`;

<div id="invoke-the-api">
  ## Invocar la API
</div>

Los pasos que seguimos en nuestra implementación son:

* Crear un objeto JSON que contenga los datos de la hoja de horas y asignarlo a una variable `timesheet`.
* Agregar la URL de la API y el contenido de la variable `timesheet` al cuerpo de la solicitud con `urllib2.Request`.
* Agregar el encabezado `Authorization` a la solicitud.
* Establecer el encabezado `Content-Type` como `application/json`.
* Invocar la API con `urllib2.urlopen` y añadir algo de manejo de errores. Recuperar la respuesta con `json.loads` e imprimirla en la consola.

Esta es nuestra implementación de ejemplo (se omite parte del código por brevedad):

```python lines expandable
def main():
  # importar bibliotecas - código omitido

  # Valores de configuración - código omitido

  # Obtener un token de acceso de Auth0 - código omitido

  #Enviar nuevo registro de horas a la API
  timesheet = {'user_id': '007',

                          'project': 'StoreZero',
                          'hours': 5}
  req = urllib2.Request(api_url, data = json.dumps(timesheet))
  req.add_header('Authorization', 'Bearer ' + access_token)
  req.add_header('Content-Type', 'application/json')

  try:
    response = urllib2.urlopen(req)
    res = json.loads(response.read())
    print 'Created timesheet ' + str(res['id']) + ' for employee ' + str(res['user_id'])
  except urllib2.HTTPError, e:
    print 'HTTPError = ' + str(e.code) + ' ' + str(e.reason)
  except urllib2.URLError, e:
    print 'URLError = ' + str(e.reason)
  except httplib.HTTPException, e:
    print 'HTTPException'
  except Exception, e:
    print 'Generic Exception' + str(e)
```
