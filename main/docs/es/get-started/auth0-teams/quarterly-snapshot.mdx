title: Acerca de Quarterly Snapshot
---

Quarterly Snapshot ofrece una visión general del uso y del valor a nivel de cuenta, incluida información agregada sobre cómo se comparan con la cuota, su entorno de identidad y la detección de amenazas.

La función se habilita automáticamente cuando Auth0 Teams está disponible. Está disponible exclusivamente para determinados clientes empresariales gestionados por su equipo de cuenta.

<div id="things-to-know">
  ## Cosas que debes saber
</div>

* Solo se puede acceder a Quarterly Snapshot a través de [Auth0 Teams](/docs/es/get-started/auth0-teams). Auth0 Teams debe estar habilitado para la cuenta antes de que comience el trimestre natural.
* Quarterly Snapshot es una función disponible para un grupo selecto de clientes empresariales gestionados por el DTAM.
* Los Team Owners y los Report Viewers pueden ver y descargar el Quarterly Snapshot en la sección **Reports** del Dashboard de Auth0 Teams.
* Los Team Owners pueden gestionar la lista de Report Viewers a través del Dashboard de Auth0 Teams.
* Los objetos de métricas en los Quarterly Snapshots pueden variar en función de las prestaciones de la cuenta, las funciones habilitadas y el uso.
* Todas las métricas se agregan a nivel de cuenta. Para obtener más información, consulta la tabla de **Métricas** a continuación.

  ## Métricas
</div>

Estas son las métricas que puedes ver en el Quarterly Snapshot.

| Usuarios activos mensuales                | Mide los usuarios activos mensuales únicos de la cuenta en comparación con tu cuota.                                                                                                                                                                                                                               | Para obtener información detallada a nivel de tenant, consulta el informe [Support Center Quota](https://support.auth0.com/reports/quota). |
| Tokens de Machine to Machine              | Muestra el número de tokens de acceso emitidos por Auth0 para la [concesión de credenciales de cliente](/docs/es/get-started/authentication-and-authorization-flow/client-credentials-flow). No se contabilizan los tokens emitidos para Auth0 Management API ni para otras API integradas de Auth0.                  | Para obtener información detallada a nivel de tenant, consulta el informe [Support Center Quota](https://support.auth0.com/reports/quota). |
| Conexiones empresariales activas          | Esta métrica muestra la cantidad de conexiones empresariales que utiliza una cuenta.                                                                                                                                                                                                                               | Para obtener más información sobre tu cuota, ponte en contacto con tu ejecutivo de cuenta.                                                 |
| Registros de nuevos usuarios              | Muestra cuántos usuarios nuevos (registros de usuario) se añadieron a tus tenants durante los meses de este trimestre.                                                                                                                                                                                             | Consulta la [página Activity](/docs/es/get-started/auth0-overview/dashboard/activity) para obtener más información sobre los usuarios nuevos. |
| Usuarios inactivos                        | Esta métrica representa el promedio mensual de usuarios que no se han autenticado en el último mes.                                                                                                                                                                                                                |                                                                                                                                            |
| Breached Password                         | Número de intentos bloqueados con credenciales comprometidas por la funcionalidad [detección de contraseñas vulneradas](/docs/es/secure/attack-protection/breached-password-detection).                                                                                                                               |                                                                                                                                            |
| Protección contra ataques de fuerza bruta | Solicitudes bloqueadas por la funcionalidad [Protección contra ataques de fuerza bruta](/docs/es/secure/attack-protection/brute-force-protection), si está habilitada. Esta métrica resume todas las solicitudes bloqueadas específicamente en tenants de producción.                                                 |                                                                                                                                            |
| Suspicious IP Throttling                  | Solicitudes bloqueadas por la funcionalidad [Suspicious IP Throttling](/docs/es/secure/attack-protection/suspicious-ip-throttling), si está habilitada. Esta métrica suma todas las solicitudes bloqueadas específicamente en tenants de producción.                                                                  |                                                                                                                                            |
| Bot Detection                             | Solicitudes que no lograron resolver el desafío CAPTCHA. Esta métrica suma todas las solicitudes de bots que fueron bloqueadas por la funcionalidad [Bot Detection](/docs/es/secure/attack-protection/bot-detection) tras no resolver un desafío CAPTCHA.                                                             |                                                                                                                                            |
| MFA adaptativo                            | Solicitudes en las que se emitió un desafío de [MFA](/docs/es/secure/multi-factor-authentication/adaptive-mfa) debido a una baja puntuación de confianza. Esta métrica suma todos los intentos de login en los que se emitió un desafío de autenticación multifactor y la puntuación de confianza calculada fue baja. |                                                                                                                                            |
