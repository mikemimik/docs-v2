title: Acerca de Quarterly Snapshot
---

Quarterly Snapshot ofrece información destacada sobre el uso y el valor a nivel de cuenta, incluida información agregada sobre cómo se compara con la cuota, su entorno de identidad y la detección de amenazas.

La funcionalidad se habilita automáticamente cuando se aprovisiona Auth0 Teams. Está disponible exclusivamente para determinados clientes empresariales gestionados por su equipo de cuenta.

<div id="things-to-know">
  ## Aspectos que debe saber
</div>

* Solo se puede acceder a Quarterly Snapshot a través de [Auth0 Teams](/docs/es/get-started/auth0-teams). Auth0 Teams debe estar habilitado para la cuenta antes del inicio del trimestre natural.
* Quarterly Snapshot es una funcionalidad disponible para un grupo selecto de clientes empresariales gestionados por el DTAM.
* Los Team Owners y Report Viewers pueden ver y descargar Quarterly Snapshot en la sección **Informes** del Dashboard de Auth0 Teams.
* Los Team Owners pueden administrar la lista de Report Viewers a través del Dashboard de Auth0 Teams.
* Los objetos de métricas de Quarterly Snapshot pueden variar en función de los derechos de la cuenta, las funcionalidades habilitadas y el uso.
* Todas las métricas se agregan a nivel de cuenta. Para obtener más información, consulte la tabla de **Métricas** que aparece a continuación.

<div id="metrics">
  ## Métricas
</div>

Estas son las métricas que puedes ver en tu Quarterly Snapshot.

| Métrica                                   | Qué mide                                                                                                                                                                                                                                                                                                                | Comentarios                                                                                                                                |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| Usuarios activos mensuales                | Mide la cantidad de usuarios activos mensuales únicos de la cuenta en comparación con tu cuota.                                                                                                                                                                                                                         | Para obtener información detallada a nivel de tenant, consulta el informe [Support Center Quota](https://support.auth0.com/reports/quota). |
| Tokens de Machine to Machine              | Muestra la cantidad de tokens de acceso emitidos por Auth0 para la [concesión de credenciales de cliente](/docs/es/get-started/authentication-and-authorization-flow/client-credentials-flow). No se cuentan los tokens emitidos para Auth0 Management API ni para otras API integradas de Auth0.                          | Para obtener información detallada a nivel de tenant, consulta el informe [Support Center Quota](https://support.auth0.com/reports/quota). |
| Conexiones empresariales activas          | Esta métrica muestra la cantidad de conexiones empresariales que usa una cuenta.                                                                                                                                                                                                                                        | Para obtener más detalles sobre tu cuota, ponte en contacto con tu ejecutivo de cuenta.                                                    |
| Registros de nuevos usuarios              | Muestra cuántos usuarios nuevos (registros de usuario) se añadieron a tu(s) tenant(s) durante los meses de este trimestre.                                                                                                                                                                                              | Consulta la [página Activity](/docs/es/get-started/auth0-overview/dashboard/activity) para obtener más información sobre los usuarios nuevos. |
| Usuarios inactivos                        | Esta métrica representa el promedio mensual de usuarios que no se han autenticado en el último mes.                                                                                                                                                                                                                     |                                                                                                                                            |
| Breached Password                         | Número de intentos bloqueados por credenciales comprometidas, tal como lo hace la funcionalidad de [detección de contraseñas vulnerada](/docs/es/secure/attack-protection/breached-password-detection).                                                                                                                    |                                                                                                                                            |
| Protección contra ataques de fuerza bruta | Solicitudes bloqueadas por la funcionalidad [Protección contra ataques de fuerza bruta](/docs/es/secure/attack-protection/brute-force-protection), si está habilitada. Esta métrica resume todas las solicitudes bloqueadas específicamente en tenants de producción.                                                      |                                                                                                                                            |
| Limitación de IP sospechosas              | Solicitudes bloqueadas por la funcionalidad [Suspicious IP Throttling](/docs/es/secure/attack-protection/suspicious-ip-throttling), si está habilitada. Esta métrica suma todas las solicitudes bloqueadas específicamente en tenants de producción.                                                                       |                                                                                                                                            |
| Bot Detection                             | Solicitudes que no pudieron resolver el desafío CAPTCHA. Esta métrica suma todas las solicitudes de bots bloqueadas por la funcionalidad [Bot Detection](/docs/es/secure/attack-protection/bot-detection) después de no resolver un desafío CAPTCHA.                                                                       |                                                                                                                                            |
| MFA adaptativo                            | Solicitudes en las que se emitió un desafío de [MFA](/docs/es/secure/multi-factor-authentication/adaptive-mfa) debido a un nivel bajo de confidence score. Esta métrica suma todos los intentos de inicio de sesión en los que se emitió un desafío de autenticación multifactor y el confidence score calculado fue bajo. |                                                                                                                                            |
