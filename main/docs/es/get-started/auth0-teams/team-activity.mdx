description: Team Activity permite a los Team Owners ver y auditar los registros de eventos generados por miembros del equipo.
title: Team Activity
---

Team Activity permite a los Team Owners ver los registros de auditoría asociados a las actividades que los miembros realizan en su cuenta de Auth0.

El informe de Team Activity se encuentra en la sección **Reports** del Teams Dashboard.

Team Activity proporciona una vista resumida del miembro que generó el evento, el tipo de evento y la fecha en que se generó.

Puede seleccionar **View Details** para ver más información, como la descripción del evento, el estado del evento, la dirección IP del usuario que activó el evento, los detalles del agente de usuario y mucho más.

<div id="event-types">
  ## Tipos de eventos
</div>

A continuación, se describen los tipos de eventos de Team Activity.

| Tipo de evento           | Descripción                                                                                                 |
| ------------------------ | ----------------------------------------------------------------------------------------------------------- |
| Invitación al equipo     | Se creó una invitación para un miembro del equipo<br />Se eliminó una invitación para un miembro del equipo |
| Miembro del equipo       | Se eliminó un miembro del equipo<br />Se actualizó el rol de un miembro del equipo                          |
| Política de seguridad    | Se actualizó la política de seguridad<br />Se actualizaron las conexiones sociales                          |
| Configuración del equipo | Se actualizó la configuración del equipo                                                                    |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  En el futuro se admitirá la autenticación de usuarios del Dashboard en Auth0 y Teams Dashboard, además de otros eventos.
</Callout>

<div id="event-schema">
  ## Esquema de eventos
</div>

A continuación se describe el esquema de eventos de Team Activity.

```json lines
{  
    "id": "string", 
    "team_id": "string",
    "user_id": "string",
    "type": "string",
    "description": "string",
    "status": "string",
    "details": "object",
    "ip": "string",
    "hostname": "string",
    "user_agent": "string",
    "version": "string",
    "created_at": "date",
    "team_slug": "string",
    "user": "object"
}
```

| Nombre        | Tipo     | Descripción                                                                            |
| ------------- | -------- | -------------------------------------------------------------------------------------- |
| `id`          | string   | Identificador único del registro de eventos.                                           |
| `team_id`     | string   | Identificador único del equipo.                                                        |
| `user_id`     | string   | Identificador único del usuario que generó el evento.                                  |
| `type`        | string   | Tipo de evento.                                                                        |
| `description` | string   | Resumen del tipo de evento.                                                            |
| `status`      | string   | Estado del evento. Los valores posibles son `Success` o `Failure`.                     |
| `details`     | object   | Detalles del evento registrado.                                                        |
| `ip`          | string   | Dirección IP del usuario que generó el registro de eventos.                            |
| `hostname`    | string   | FQDN de la Auth0 App en la que se realizó el evento.                                   |
| `user_agent`  | string   | Navegador web, y tipo y versión del sistema operativo del dispositivo.                 |
| `version`     | string   | Versión del esquema con la que se generó el registro de actividad.                     |
| `created_at`  | datetime | Fecha y hora en que se generó el registro de eventos.                                  |
| `team_slug`   | string   | Identificador textual único del equipo.                                                |
| `user`        | object   | Contiene el nombre, la URL del avatar y el estado de pertenencia al equipo (booleano). |
