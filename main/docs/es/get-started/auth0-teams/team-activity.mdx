title: Team Activity
---

Team Activity permite a los Team Owners ver los registros de auditoría asociados a las actividades que los miembros realizan en su cuenta de Auth0.

El informe de Team Activity se encuentra en la sección **Reports** del Teams Dashboard.

Team Activity proporciona una vista resumida del miembro que generó el evento, el tipo de evento y la fecha en que se generó.

Puede seleccionar **View Details** para ver más información, como la descripción del evento, el estado del evento, la dirección IP del usuario que lo desencadenó, los detalles del agente de usuario y más.

<div id="event-types">
  ## Tipos de eventos
</div>

A continuación se describen los tipos de eventos de Team Activity.

| Tipo de evento           | Descripción                                                                             |
| ------------------------ | --------------------------------------------------------------------------------------- |
| Invitación al equipo     | Invitación de miembro del equipo creada<br />Invitación de miembro del equipo eliminada |
| miembro del equipo       | miembro del equipo eliminado<br />Role de miembro del equipo actualizado                |
| Política de seguridad    | Política de seguridad actualizada<br />Conexiones sociales actualizadas                 |
| Configuración del equipo | Configuración del equipo actualizada                                                    |

<div id="event-schema">
  ## esquema de eventos
</div>

A continuación, se describe el esquema de eventos de Team Activity.

```json lines
{  
    "id": "string", 
    "team_id": "string",
    "user_id": "string",
    "type": "string",
    "description": "string",
    "status": "string",
    "details": "object",
    "ip": "string",
    "hostname": "string",
    "user_agent": "string",
    "version": "string",
    "created_at": "date",
    "team_slug": "string",
    "user": "object"
}
```

| Name          | Type     | Description                                                                            |
| ------------- | -------- | -------------------------------------------------------------------------------------- |
| `id`          | string   | Identificador único del registro de eventos.                                           |
| `team_id`     | string   | Identificador único del equipo.                                                        |
| `user_id`     | string   | El identificador único del usuario que generó el evento.                               |
| `type`        | string   | Tipo de evento.                                                                        |
| `description` | string   | Resumen del tipo de evento.                                                            |
| `status`      | string   | Estado del evento. Los valores posibles son `Success` o `Failure`.                     |
| `details`     | object   | Detalles del evento registrado.                                                        |
| `ip`          | string   | Dirección IP del usuario que generó el registro de eventos.                            |
| `hostname`    | string   | FQDN de la Auth0 App en la que se realizó el evento.                                   |
| `user_agent`  | string   | Navegador web y tipo y versión del SO del dispositivo.                                 |
| `version`     | string   | Versión del esquema con la que se generó el registro de actividad.                     |
| `created_at`  | datetime | Fecha y hora en que se generó el registro de eventos.                                  |
| `team_slug`   | string   | Identificador textual único del equipo.                                                |
| `user`        | object   | Contiene el nombre, la URL del avatar y el estado de la membresía del Team (booleano). |
