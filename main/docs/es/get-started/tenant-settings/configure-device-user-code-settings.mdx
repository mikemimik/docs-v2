title: Configurar la configuración del código de usuario para dispositivos
---

Puede configurar los ajustes del código de usuario generado por su aplicación durante el [flujo de autorización de dispositivos](/docs/es/get-started/authentication-and-authorization-flow/device-authorization-flow) desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Vaya a [Dashboard &gt; Settings](https://manage.auth0.com/#/tenant) y haga clic en la pestaña **Advanced**.
2. Desplácese hasta la sección **Device Flow User Code Format**, busque **User Code Character Set** y **User Code Mask**, introduzca los ajustes deseados y haga clic en **Save**.

| Ajuste                           | Descripción                                                                                                                                                                                             |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Device Flow User Code Format** | Conjunto de caracteres que se utiliza para generar aleatoriamente un código de usuario.                                                                                                                 |
| **User Code Mask**               | Máscara que se utiliza para definir la longitud y el formato de un código de usuario generado aleatoriamente. Su propósito es mejorar la legibilidad del código de usuario y facilitar su introducción. |
