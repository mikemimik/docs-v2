title: Configurar la configuración del código de usuario del dispositivo
---

Puedes configurar la configuración del código de usuario generado por tu aplicación durante el [Device Authorization Flow](/docs/es/get-started/authentication-and-authorization-flow/device-authorization-flow) desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Ve a [Dashboard &gt; Settings](https://manage.auth0.com/#/tenant) y haz clic en la pestaña **Advanced**.
2. Desplázate hasta la sección **Device Flow User Code Format**, busca **User Code Character Set** y **User Code Mask**, introduce la configuración deseada y haz clic en **Save**.

| Configuración                    | Descripción                                                                                                                                                                                         |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Device Flow User Code Format** | Conjunto de caracteres que se usa para generar aleatoriamente un código de usuario.                                                                                                                 |
| **User Code Mask**               | Máscara que se usa para definir la longitud y el formato de un código de usuario generado aleatoriamente. Su finalidad es mejorar la legibilidad del código de usuario y facilitar su introducción. |
