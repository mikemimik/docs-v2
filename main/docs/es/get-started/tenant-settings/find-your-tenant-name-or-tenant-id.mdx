---
description: Cómo encontrar el nombre de tu tenant de Auth0
title: Encuentra el nombre de tu tenant o el Tenant ID
---

Cada tenant de Auth0 tiene un nombre. Este nombre se usa con frecuencia en el código para identificar el tenant. Cuando veas las variables `{yourTenantId}`, `{yourTenantName}`, `{yourTenant}` o una similar en la documentación de Auth0, sustituye la variable por el nombre del tenant.

1. Ve al [Auth0 Dashboard](https://manage.auth0.com/#).
2. En la esquina superior izquierda, busca el nombre del tenant junto al logotipo de Auth0.
