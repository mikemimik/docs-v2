Puede revocar la clave de firma de la aplicación o API de su tenant mediante el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. La clave de firma se utiliza para firmar <Tooltip tip="Management API: un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=ID+tokens">tokens de ID</Tooltip>, <Tooltip tip="ID Token: credencial destinada al propio cliente, en lugar de usarse para acceder a un recurso." cta="Ver glosario" href="/docs/es/glossary?term=access+tokens">tokens de acceso</Tooltip>, aserciones <Tooltip tip="Access Token: credencial de autorización, en forma de una cadena opaca o JWT, que se utiliza para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=SAML">SAML</Tooltip> y aserciones <Tooltip tip="Security Assertion Markup Language (SAML): protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin una contraseña." cta="Ver glosario" href="/docs/es/glossary?term=WS-Fed">WS-Fed</Tooltip> enviadas a su aplicación o API. Para obtener más información, consulte [claves de firma](/docs/es/get-started/tenant-settings/signing-keys).

<div id="prerequisites">
  ## Requisitos previos
</div>

* Antes de poder revocar una clave de firma utilizada anteriormente, primero debes haber rotado la clave. Para obtener más información, consulta [Rotar claves de firma](/docs/es/get-started/tenant-settings/signing-keys/rotate-signing-keys) o la sección **Rotar y revocar clave de firma** a continuación.
* Asegúrate de haber actualizado tu aplicación o API con la nueva clave antes de revocar la clave anterior.

<Warning>
  No puedes reutilizar una clave de firma una vez revocada, así que asegúrate de que realmente quieres revocarla.
</Warning>

<div id="use-the-dashboard">
  ## Usa el Dashboard
</div>

<div id="revoke-previously-used-signing-key">
  ### Revocar una clave de firma usada anteriormente
</div>

1. Ve a [Dashboard &gt; Settings &gt; claves de firma](https://manage.auth0.com/#/tenant/signing_keys).
2. En la sección **List of Valid Keys**, busca la clave **Previously Used**, abre el menú de más opciones (**...**) y selecciona **Revoke Key**.
   La sección **List of Valid Keys** muestra la clave de firma actual que usa tu tenant, además de la siguiente clave de firma que se asignará si decides rotar tus claves de firma. Si ya has rotado claves de firma anteriormente, esta sección también muestra las claves usadas previamente.
   La sección **List of Revoked Keys** muestra las tres últimas claves revocadas de tu tenant.
3. Selecciona **Revoke** para confirmar.

<div id="rotate-and-revoke-signing-key">
  ### Rotar y revocar la clave de firma
</div>

1. Ve a [Dashboard &gt; Settings &gt; Claves de firma](https://manage.auth0.com/#/tenant/signing_keys).
2. En la sección **Configuración de rotación**, busca la sección **Rotar y revocar la clave de firma** y selecciona **Rotar y revocar clave**.
3. Selecciona **Rotar y revocar** para confirmar.

<div id="use-the-management-api">
  ## Usa la Management API
</div>

<Warning>
  Solo puedes revocar la clave de firma que se usó anteriormente.
</Warning>

1. Para obtener una lista de las claves de firma, haz una llamada `GET` al endpoint [Obtener todas las claves de firma de la aplicación](https://auth0.com/docs/api/management/v2#!/Keys/get_signing_keys).
2. Haz una llamada `PUT` al endpoint [Revocar una clave de firma de la aplicación por su ID de la clave](https://auth0.com/docs/api/management/v2#!/Keys/put_signing_keys). Asegúrate de reemplazar los valores de marcador de posición `{yourKeyId}` y `{yourMgmtApiAccessToken}` por el ID de tu clave de firma y el token de acceso de la Management API, respectivamente.

| Valor                   | Descripción                                                                                                                                                                                                         |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `YOUR_KEY_ID`           | ID de la clave de firma que se va a revocar. Para saber cómo encontrar el ID de tu clave de firma, consulta [Localizar conjuntos de claves web JSON](/docs/es/secure/tokens/json-web-tokens/locate-json-web-key-sets). |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](https://auth0.com/docs/api/management/v2/tokens) con el scope `update:signing_keys`.                                                                                       |
