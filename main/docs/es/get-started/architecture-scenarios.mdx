Estos escenarios describen los distintos tipos de arquitecturas tecnológicas que puede usar su aplicación y cómo Auth0 puede ayudar. El objetivo de estos escenarios es guiarle a lo largo del proceso de implementación, de principio a fin.

| Lea...                                                                                                                          | Para aprender...                                                                                                                                                        |
| ------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| [Inicio de sesión único para aplicaciones web tradicionales](/docs/es/get-started/architecture-scenarios/sso-for-regular-web-apps) | Cómo configurar e implementar una aplicación web que use SSO para empleados que usan Active Directory y contratistas que usan un repositorio de usuarios independiente. |
| [Aplicación de servidor + API](/docs/es/get-started/architecture-scenarios/server-application-api)                                 | Cómo configurar una API de hoja de horas con una tarea cron para registrar entradas de hoja de horas.                                                                   |
