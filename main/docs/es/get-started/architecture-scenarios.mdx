Estos escenarios describen los distintos tipos de arquitecturas tecnológicas que puede usar su aplicación y cómo Auth0 puede ayudar. El objetivo de estos escenarios es guiarle a lo largo del proceso de implementación de principio a fin.

| Lea...                                                                                                                          | Para saber...                                                                                                                                                           |

| [Aplicación de servidor + API](/docs/es/get-started/architecture-scenarios/server-application-api)                                 | Cómo configurar una API de hoja de horas con una tarea cron para registrar entradas de la hoja de horas.                                                                |
