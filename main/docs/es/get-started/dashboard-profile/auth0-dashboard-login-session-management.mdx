---
description: Describe cómo los administradores de Auth0 Dashboard pueden ver y revocar sesiones de inicio de sesión.
title: Gestión de sesiones de inicio de sesión de Auth0 Dashboard
---

Con Login Sessions, los administradores de <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> pueden supervisar las sesiones activas de distintas aplicaciones del Dashboard en varios dispositivos y navegadores. Puedes revisar y revocar las sesiones creadas por usuarios que acceden a Teams Dashboard, Support Center y Auth0 Dashboard.

Puedes ver una lista de todas las sesiones activas en la [página de perfil](https://manage.auth0.com/#/profile/sessions) de tu Auth0 Dashboard.

<Frame>![Auth0 Dashboard > Perfil](/docs/images/cdy7uua7fh8z/3hBNRo2Bc4w6oKpF0mMt4o/3796b322593eae38eeba0a3af3698eca/English_user_profle_login_sessions.png)</Frame>

Para cerrar sesión o revocar una sesión, localiza la sesión que te interese y selecciona **Revocar**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Revocar una sesión no elimina automáticamente las sesiones de las extensiones. Sin embargo, estas sesiones son de corta duración y no se renuevan cuando se revoca una sesión de inicio de sesión o esta expira.
</Callout>
