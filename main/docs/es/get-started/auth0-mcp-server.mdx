El servidor MCP de Auth0 conecta agentes de IA con tu tenant de Auth0, lo que les permite realizar operaciones complejas de varios pasos en Auth0, como crear aplicaciones, gestionar usuarios o implementar Actions. Aprovecha el [Model Context Protocol (MCP)](https://modelcontextprotocol.io/introduction), un protocolo estandarizado que permite a los agentes de IA comunicarse con herramientas externas y APIs para ejecutar tareas en tu nombre.

Diseñado para desarrolladores que usan herramientas basadas en IA como Cursor, Windsurf o Claude Desktop, el servidor MCP de Auth0 transforma la forma en que gestionan su infraestructura de identidad, haciendo que trabajar con los servicios de Auth0 sea más rápido y sencillo.

<div id="authentication-and-credential-security">
  ## Seguridad de la autenticación y las credenciales
</div>

El servidor MCP de Auth0 da prioridad a una seguridad robusta mediante la gestión de la autenticación, la validación de las solicitudes y la garantía de una comunicación segura con la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> de Auth0. Esto incluye:

* [Flujo de autorización de dispositivos de OAuth 2.0](/docs/es/get-started/authentication-and-authorization-flow/device-authorization-flow): Autentica las solicitudes con Auth0 de forma segura.
* [Almacenamiento seguro de credenciales](/docs/es/get-started/applications/rotate-credentials): Las credenciales se almacenan de forma segura en el llavero del sistema y nunca se exponen en texto plano.
* [Permisos mínimos de API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens): Solo se solicitan los permisos esenciales, en estricto cumplimiento del principio de privilegio mínimo.

<div id="supported-tools-and-integrations">
  ## Herramientas e integraciones compatibles
</div>

El servidor MCP de Auth0 actualmente se integra con los siguientes [clientes MCP](https://modelcontextprotocol.io/clients), entre ellos:

* [Claude Desktop](https://claude.ai/download)
* [Cursor](https://www.cursor.com/downloads)
* [Windsurf](https://www.windsurf.com/editor)

Se irán añadiendo más integraciones a medida que surjan nuevos clientes relevantes. Para consultar una lista completa de las herramientas compatibles y sus funcionalidades en detalle, consulta nuestra [referencia de herramientas del servidor MCP de Auth0](/docs/es/get-started/auth0-mcp-server/auth0-mcp-tools-reference).
