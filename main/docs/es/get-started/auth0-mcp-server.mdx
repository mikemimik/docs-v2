El servidor MCP de Auth0 conecta agentes de IA con tu tenant de Auth0, lo que les permite realizar operaciones complejas de varios pasos en Auth0, como crear aplicaciones, gestionar usuarios o desplegar Actions. Aprovecha el [Model Context Protocol (MCP)](https://modelcontextprotocol.io/introduction), un protocolo estandarizado que permite a los agentes de IA comunicarse con herramientas externas y APIs para ejecutar tareas en tu nombre.

Diseñado para desarrolladores que usan herramientas impulsadas por IA como Cursor, Windsurf o Claude Desktop, el servidor MCP de Auth0 transforma la manera en que gestionan su infraestructura de identidad, haciendo que trabajar con los servicios de Auth0 sea más rápido y sencillo.

<div id="authentication-and-credential-security">
  ## Seguridad de la autenticación y las credenciales
</div>

El servidor MCP de Auth0 prioriza una seguridad sólida al gestionar la autenticación, validar las peticiones y garantizar una comunicación segura con la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> de Auth0. Esto incluye:

* [OAuth 2.0 Device Authorization Flow](/docs/es/get-started/authentication-and-authorization-flow/device-authorization-flow): autentica de forma segura las peticiones con Auth0.
* [Almacenamiento seguro de credenciales](/docs/es/get-started/applications/rotate-credentials): las credenciales se almacenan de forma segura en el llavero del sistema y nunca se exponen en texto plano.
* [Permisos mínimos de API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens): solo se solicitan los permisos esenciales, en estricto cumplimiento del principio del privilegio mínimo.

<div id="supported-tools-and-integrations">
  ## Herramientas e integraciones compatibles
</div>

El servidor MCP de Auth0 actualmente se integra con los siguientes [clientes MCP](https://modelcontextprotocol.io/clients), incluidos:

* [Claude Desktop](https://claude.ai/download)
* [Cursor](https://www.cursor.com/downloads)
* [Windsurf](https://www.windsurf.com/editor)

Se irán añadiendo integraciones adicionales a medida que aparezcan nuevos clientes destacados. Para consultar una lista completa de las herramientas compatibles y sus funciones detalladas, consulta nuestra [Referencia de herramientas del servidor MCP de Auth0](/docs/es/get-started/auth0-mcp-server/auth0-mcp-tools-reference).
