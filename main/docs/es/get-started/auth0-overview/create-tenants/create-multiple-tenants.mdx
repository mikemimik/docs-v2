---
description: Describe cómo crear un tenant adicional con el Auth0 Dashboard.
title: Crear múltiples tenants
---

Puedes configurar múltiples tenants para crear [diferentes entornos](/docs/es/get-started/auth0-overview/create-tenants/set-up-multiple-environments) en el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> y así admitir configuraciones complejas. Por ejemplo, podrías tener dos dominios distintos (uno interno y otro orientado al público), o quizás quieras que los usuarios inicien sesión de manera diferente para distintas aplicaciones. Para lograrlo, crea más de un tenant de Auth0, de modo que puedas tener conjuntos separados de aplicaciones, conexiones y usuarios para las aplicaciones y los grupos de usuarios que necesitas admitir.

1. Ve al [Auth0 Dashboard](https://manage.auth0.com/#/), selecciona el nombre de tu tenant y luego selecciona **Create Tenant**.

   <Frame>![Dashboard Tenant Drop-Down Menu Create Tenant](/docs/images/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/5efad1137eb8b04031b30137c2fc5ac5/2025-02-26_14-51-39.png)</Frame>
2. Introduce el **Tenant Domain** que desees, selecciona una **Region** y selecciona **Create**.
