Para integrar Auth0 con una aplicación web de página única, primero debe registrar su aplicación en Auth0 desde el Dashboard.

1. Vaya a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y haga clic en **Create Application**.
2. Introduzca un nombre descriptivo para su aplicación, seleccione **aplicaciones web de página única** y haga clic en **Create**.

   Una vez creada la nueva aplicación, puede ver la página **configuración de la aplicación**, que incluye las siguientes pestañas:

   | Pestaña de configuración | Descripción                                                                                                                                                                                                                                                                                                                                                                                                            |
   | ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Quick Start**          | Muestra toda la documentación disponible para su tipo de aplicación.                                                                                                                                                                                                                                                                                                                                                   |
   | **Settings**             | Muestra toda la configuración disponible para su aplicación. De forma predeterminada, la mayoría de los ajustes se crean automáticamente. Para obtener más información, lea [configuración de la aplicación](/docs/es/get-started/applications/application-settings).                                                                                                                                                               |
   | **Credentials**          | Muestra el método de autenticación de la aplicación y las credenciales configuradas. Para obtener más información, lea [Application Credentials](/docs/es/secure/application-credentials)                                                                                                                                                                                                                                 |
   | **Add-ons**              | Le permite habilitar complementos asociados a una aplicación. Normalmente se usan para integraciones de SAML o WS-Fed, que no suelen ser necesarias para las SPA. Para obtener más información, lea [Enable SAML2 Web App Addon](/docs/es/authenticate/protocols/saml/saml-sso-integrations/enable-saml2-web-app-addon) y [Configure WS-Fed Applications](/docs/es/get-started/applications/update-application-connections). |
   | **Connections**          | Le permite habilitar conexiones para su aplicación. Las conexiones son fuentes de usuarios; pueden habilitarse y compartirse entre varias aplicaciones. Para obtener más información, lea [Connections](/docs/es/authenticate/identity-providers).                                                                                                                                                                        |
   | **Organizations**        | Le permite autenticar a los usuarios en las organizaciones a las que pertenecen. Para obtener más información, lea [Organizations](/docs/es/manage-users/organizations).                                                                                                                                                                                                                                                  |
   | **Login Experience**     | Le permite configurar la experiencia de inicio de sesión para los usuarios de su organización. Para obtener más información, lea [Understand how Auth0 Organizations Work](/docs/es/manage-users/organizations/organizations-overview#login-experience).                                                                                                                                                                  |

   Para obtener más información, lea [configuración de la aplicación](/docs/es/get-started/applications/application-settings).

<div id="next-steps">
  ## Siguientes pasos
</div>

Una vez que hayas registrado tu aplicación, revisa y completa estos pasos de configuración:

* Configura al menos una conexión (como una conexión de base de datos o inicio de sesión social) y habilítala para tu aplicación. Sin una conexión habilitada, los usuarios no podrán autenticarse.
* Modifica el código de tu aplicación para usar la aplicación registrada en Auth0. Consulta nuestros [Auth0 Quickstarts](/docs/es/quickstarts), donde encontrarás instrucciones detalladas sobre cómo implementar el inicio de sesión y el cierre de sesión, gestionar las sesiones de los usuarios, recuperar y mostrar la información del perfil del usuario, entre otras cosas.
* Usa las [Auth0 APIs](/docs/es/api).

  * La [Authentication API](https://auth0.com/docs/api/authentication) gestiona todas las funciones principales relacionadas con la identidad (por ejemplo, inicio de sesión, cierre de sesión y obtención del perfil del usuario). La mayoría de los usuarios utiliza esta API a través de nuestros Quickstarts, el [Auth0 SPA SDK](/docs/es/libraries/auth0-spa-js) o el [widget de Lock](/docs/es/libraries/lock). Para aplicaciones heredadas, consulta la [biblioteca auth0.js](/docs/es/libraries/auth0js). Sin embargo, si estás creando manualmente toda tu interfaz de usuario de autenticación, tendrás que interactuar directamente con esta API.
  * La [Management API](https://auth0.com/docs/api/management/v2) te permite automatizar varias tareas a las que también puedes acceder a través del Dashboard de Auth0 (por ejemplo: crear usuarios y configurar los tipos de concesión de la aplicación).
