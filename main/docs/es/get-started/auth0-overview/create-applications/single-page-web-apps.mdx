Para integrar Auth0 con una aplicación web de página única, primero debes registrar tu aplicación en Auth0 mediante el Dashboard.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y haz clic en **Create Application**.
2. Introduce un nombre descriptivo para tu aplicación, selecciona **aplicación web de página única** y haz clic en **Create**.

   Una vez creada la nueva aplicación, puedes ver la página de **Application Settings**, que incluye las siguientes pestañas:

   | Pestaña de configuración | Descripción                                                                                                                                                                                                                                                                                                                                                                                                             |
   | ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Quick Start**          | Muestra toda la documentación disponible para tu tipo de aplicación.                                                                                                                                                                                                                                                                                                                                                    |
   | **Settings**             | Muestra toda la configuración disponible para tu aplicación. De forma predeterminada, la mayoría de los ajustes se crean automáticamente. Para obtener más información, consulta [Application Settings](/docs/es/get-started/applications/application-settings).                                                                                                                                                           |
   | **Credentials**          | Muestra el método de autenticación de la aplicación y las credenciales configuradas. Para obtener más información, consulta [Application Credentials](/docs/es/secure/application-credentials)                                                                                                                                                                                                                             |
   | **Add-ons**              | Te permite habilitar plugins asociados a una aplicación. Normalmente se usan para integraciones con SAML o WS-Fed, que no suelen ser necesarias para las SPA. Para obtener más información, consulta [Enable SAML2 Web App Addon](/docs/es/authenticate/protocols/saml/saml-sso-integrations/enable-saml2-web-app-addon) y [Configure WS-Fed Applications](/docs/es/get-started/applications/update-application-connections). |
   | **Connections**          | Te permite habilitar conexiones para tu aplicación. Las conexiones son fuentes de usuarios; se pueden habilitar y compartir entre varias aplicaciones. Para obtener más información, consulta [Connections](/docs/es/authenticate/identity-providers).                                                                                                                                                                     |
   | **Organizations**        | Te permite autenticar usuarios en las organizaciones a las que pertenecen. Para obtener más información, consulta [Organizations](/docs/es/manage-users/organizations).                                                                                                                                                                                                                                                    |
   | **Login Experience**     | Te permite configurar la experiencia de inicio de sesión para los usuarios de tu organización. Para obtener más información, consulta [Understand how Auth0 Organizations Work](/docs/es/manage-users/organizations/organizations-overview#login-experience).                                                                                                                                                              |

   Para obtener más información, consulta [Application Settings](/docs/es/get-started/applications/application-settings).

<div id="next-steps">
  ## Próximos pasos
</div>

Una vez que hayas registrado tu aplicación, revisa y completa estos pasos de configuración:

* Configura al menos una conexión (como una conexión de base de datos o inicio de sesión social) y habilítala para tu aplicación. Sin una conexión habilitada, los usuarios no podrán autenticarse.
* Modifica el código de tu aplicación para usar la aplicación registrada en Auth0. Revisa nuestros [Auth0 Quickstarts](/docs/es/quickstarts), donde encontrarás instrucciones detalladas sobre cómo implementar el inicio y el cierre de sesión, gestionar las sesiones de tus usuarios, recuperar y mostrar la información del perfil del usuario, y mucho más.
* Usa las [Auth0 APIs](/docs/es/api).

  * La [Authentication API](https://auth0.com/docs/api/authentication) gestiona todas las funciones principales relacionadas con la identidad (por ejemplo, inicio de sesión, cierre de sesión y obtención del perfil del usuario). La mayoría de los usuarios utilizan esta API a través de nuestros Quickstarts, el [Auth0 SPA SDK](/docs/es/libraries/auth0-spa-js) o el [widget de Lock](/docs/es/libraries/lock). Para aplicaciones Legacy, consulta la [biblioteca auth0.js](/docs/es/libraries/auth0js). Sin embargo, si estás creando manualmente toda tu interfaz de usuario de autenticación, tendrás que interactuar directamente con esta API.
  * La [Management API](https://auth0.com/docs/api/management/v2) te permite automatizar varias tareas a las que también se puede acceder desde el Dashboard de Auth0 (por ejemplo: crear usuarios y configurar los tipos de concesión de la aplicación).
