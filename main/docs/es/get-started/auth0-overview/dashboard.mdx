El [Auth0 Dashboard](https://manage.auth0.com/#) es donde administras todos los aspectos de tu suscripción a Auth0 y su configuración.

<Frame>![página Activity de Auth0 Dashboard](/docs/images/cdy7uua7fh8z/6k0cRLNEPko6vY6Kw7nIQD/4590d1b9d3e79f8d9c9970d733b08d02/2025-01-28_14-34-41.png)</Frame>

Consta de varias secciones por las que puedes navegar mediante el menú de la barra lateral izquierda.

<div id="configure-implementation">
  ## Configurar la implementación
</div>

La siguiente tabla ofrece una breve descripción general de las distintas secciones del Dashboard y de lo que puede hacer en cada una.

| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Applications**    | Administre sus aplicaciones, API e integraciones de inicio de sesión único (SSO).<br />**Applications**: Para cada aplicación con la que quiera autenticar usuarios mediante Auth0, registre una aplicación.<br />**APIs**: Para cada API que quiera proteger con Auth0, registre una API. Cree nuevas API y administre las existentes.<br />**SSO Integrations**: Vea y habilite servicios externos para SSO. Cree nuevas integraciones de SSO y configure, revise y administre la configuración de la integración.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

Conéctese a bases de datos existentes mediante scripts de JavaScript basados en plantillas que se ejecutan en el servidor de Auth0 durante cada autenticación. Migre gradualmente a Auth0 una base de datos existente de credenciales heredadas a medida que los usuarios se autentican (no se requiere restablecimiento de contraseña).<br />**Social**: Configure proveedores de identidad social (como Facebook, X y GitHub) mediante los cuales sus usuarios puedan iniciar sesión.<br />**Enterprise**: Configure proveedores de identidad empresariales (como Active Directory, SAML y Office 365) mediante los cuales sus usuarios puedan iniciar sesión con sus credenciales empresariales.<br />**Passwordless**: Permita que sus usuarios se registren e inicien sesión usando códigos de acceso de un solo uso (enviados por correo electrónico o SMS) o enlaces de un solo clic, en lugar de contraseñas. |

<div id="manage-account-settings">
  ## Administrar la configuración de la cuenta
</div>

En la parte superior izquierda, puedes ver el nombre y el icono de tu tenant, además de una pequeña flecha. Esta flecha muestra un menú desplegable que puedes usar para configurar distintos aspectos de tu cuenta:

* [**Configuración**](/docs/es/get-started/tenant-settings): Configura varios aspectos de tu tenant.
* [**Invitar a un miembro**](/docs/es/get-started/manage-dashboard-access): Agrega un usuario adicional como administrador u otro rol a la configuración de tu tenant.
* [**Uso de la cuota**](https://support.auth0.com/reports/quota): Consulta información sobre el uso de la cuota de tu suscripción y tus tenants.
* [**Crear tenant**](/docs/es/get-started/auth0-overview/create-tenants): Usa esta opción para crear un tenant nuevo.
* **Cambiar de tenant**: Si tienes [varios tenants](/docs/es/get-started/auth0-overview/create-tenants/create-multiple-tenants), usa esta opción para cambiar entre ellos. Si creas una aplicación para un tenant, no la verás en la lista de otro tenant.

En la parte superior derecha, verás varias opciones en las que puedes hacer clic:

* **Buscar**: Busca aplicaciones, entradas de Marketplace, usuarios y otro contenido relacionado con tu tenant.
* **Cuéntanos qué necesitas:** Ponte en contacto con un experto de Auth0 para que te ayude a alcanzar tus objetivos con Auth0.
* **Documentación:** Un enlace al sitio de documentación que estás leyendo ahora mismo.
* **Campana de notificaciones:** Te informa sobre nuevas comunicaciones de Auth0 o cambios en la configuración de tu tenant.
* **Perfil:** Un enlace a tu perfil local del tenant, así como opciones para [cambiar tu tema](/docs/es/get-started/dashboard-profile/light-and-dark-themes) y cerrar sesión.
