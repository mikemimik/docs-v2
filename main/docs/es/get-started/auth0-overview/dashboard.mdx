El [Auth0 Dashboard](https://manage.auth0.com/#) es donde administras todos los aspectos de tu suscripción a Auth0 y tu configuración.

<Frame>![página Activity del Auth0 Dashboard](/docs/images/cdy7uua7fh8z/6k0cRLNEPko6vY6Kw7nIQD/4590d1b9d3e79f8d9c9970d733b08d02/2025-01-28_14-34-41.png)</Frame>

Consta de varias secciones por las que puedes navegar mediante el menú de la barra lateral izquierda.

<div id="configure-implementation">
  ## Configurar la implementación
</div>

La siguiente tabla ofrece una breve descripción general de las distintas secciones del Dashboard y de lo que puedes hacer en cada una.

| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Applications**    | Administra tus aplicaciones, API e integraciones de inicio de sesión único (SSO).<br />**Applications**: Para cada aplicación con la que quieras autenticar usuarios mediante Auth0, registra una aplicación.<br />**APIs**: Para cada API que quieras proteger con Auth0, registra una API. Crea nuevas API y administra las existentes.<br />**SSO Integrations**: Ve y habilita servicios externos para SSO. Crea nuevas integraciones de SSO y configura, revisa y administra sus ajustes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Authentication**  | Administra los proveedores de identidad mediante los cuales permites que los usuarios se autentiquen en tus aplicaciones.<br />**Database**: Almacena y administra de forma segura credenciales de identificador y contraseña, ya sea en un almacén de datos de Auth0 o en tu propia base de datos. Conéctate a bases de datos existentes mediante scripts de JavaScript basados en Templates que se ejecutan en el servidor de Auth0 durante cada autenticación. Migra gradualmente a Auth0 una base de datos existente de credenciales heredadas a medida que los usuarios se autentican (sin necesidad de restablecer la contraseña).<br />**Social**: Configura proveedores de identidad social (como Facebook, X y GitHub) mediante los cuales tus usuarios pueden iniciar sesión.<br />**Enterprise**: Configura proveedores de identidad empresariales (como Active Directory, SAML y Office 365) mediante los cuales tus usuarios pueden iniciar sesión con sus credenciales empresariales.<br />**Passwordless**: Permite que tus usuarios se registren e inicien sesión usando códigos de acceso de un solo uso (enviados por correo electrónico o SMS) o enlaces de un clic, en lugar de contraseñas. |

<div id="manage-account-settings">
  ## Administrar la configuración de la cuenta
</div>

En la parte superior izquierda, puedes ver el nombre y el ícono de tu tenant, junto con una pequeña flecha. Esta flecha muestra un menú desplegable que puedes usar para configurar distintos aspectos de tu cuenta:

* [**Settings**](/docs/es/get-started/tenant-settings): Configura varios aspectos de tu tenant.
* [**Invitar a un miembro**](/docs/es/get-started/manage-dashboard-access): Agrega un usuario adicional como administrador u otro rol a la configuración de tu tenant.
* [**Uso de cuota**](https://support.auth0.com/reports/quota): Consulta información sobre el uso de cuota de tu suscripción y tus tenants.
* [**Crear tenant**](/docs/es/get-started/auth0-overview/create-tenants): Usa esta opción para crear un tenant nuevo.
* **Cambiar de tenant**: Si tienes [varios tenants](/docs/es/get-started/auth0-overview/create-tenants/create-multiple-tenants), usa esta opción para cambiar entre ellos. Si creas una aplicación para un tenant, no la verás en la lista de otro tenant.

En la parte superior derecha, verás varias opciones en las que puedes hacer clic:

* **Buscar**: Busca aplicaciones, entradas de Marketplace, usuarios y otro contenido relacionado con tu tenant.
* **Cuéntanos tus necesidades:** Contacta a un experto de Auth0 para que te ayude a alcanzar tus objetivos con Auth0.
* **Docs:** Un enlace al sitio de documentación que estás leyendo ahora mismo.
* **Campana de notificaciones:** Te informa sobre nuevas comunicaciones de Auth0 o cambios en la configuración de tu tenant.
* **Perfil:** Un enlace a tu perfil local del tenant, así como opciones para [cambiar el tema](/docs/es/get-started/dashboard-profile/light-and-dark-themes) y cerrar sesión.
