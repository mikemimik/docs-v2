---
description: Aprende a editar los roles de usuario del Auth0 Dashboard para tu tenant de Auth0.
title: Editar miembros del tenant
---

Si eres administrador del tenant, puedes cambiar los roles asignados a un usuario. Los cambios surtirán efecto cuando los usuarios actualicen su Dashboard.

Puedes asignar más de un rol a un usuario y los permisos se combinarán. Debes haber agregado previamente al miembro del tenant.

1. Ve a [Dashboard &gt; Settings &gt; Tenant Members](https://manage.auth0.com/#/tenant/admins).
2. Haz clic en **...** a la derecha del nombre o la dirección de correo electrónico de un usuario y luego en **Edit Roles**.
   Si no ves esa opción, significa que el usuario aún no ha aceptado su invitación.
3. Selecciona roles adicionales o quita roles.
4. Haz clic en **Save**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Los permisos del usuario se mostrarán cuando actualice su Dashboard. Si el usuario no lo actualiza, puede encontrarse con errores que indiquen que no tiene acceso a un área específica. Si se produce uno de esos errores, el usuario tendrá que cerrar sesión y volver a iniciarla.
</Callout>

**Restablecimiento de contraseña**: Para obtener más información, consulta [Restablecer la contraseña de la cuenta de Auth0](/docs/es/troubleshoot/customer-support/reset-account-passwords) si tú o alguno de tus usuarios del Dashboard tiene problemas para iniciar sesión.

**Dirección de correo electrónico no válida**: Para obtener más información, consulta [Actualizar las direcciones de correo electrónico de usuarios del Dashboard](/docs/es/get-started/manage-dashboard-access/update-dashboard-user-email) para ver las direcciones de correo electrónico de los miembros de tu tenant y actualizarlas si es necesario.
