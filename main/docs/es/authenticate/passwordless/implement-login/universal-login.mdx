title: Autenticación passwordless con Universal Login
---

[Universal Login](/docs/es/authenticate/login/auth0-universal-login) es la implementación de Auth0 del flujo de login. Cada vez que un usuario necesita demostrar su identidad, tus applications redirigen a <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip> y Auth0 se encargará de lo necesario para garantizar la identidad del usuario. Es la forma recomendada de implementar la autenticación <Tooltip tip="Passwordless: Forma de autenticación que no depende de una contraseña como primer factor." cta="Ver glosario" href="/docs/es/glossary?term=Passwordless">Passwordless</Tooltip>.

Para implementar la autenticación passwordless en Universal Login, debes personalizar el HTML de la página de login.

1. Ve a [Dashboard &gt; Branding &gt; Universal Login](https://manage.auth0.com/#/login_settings) y haz clic en **Advanced Options**.

2. Haz clic en la pestaña **Login** y, a continuación, activa el toggle **Customize Login Page**.

<Frame>![Login personalizado](/docs/images/cdy7uua7fh8z/5iwLcz2O100hLuxlKN4rSs/e4f35f8c44516b98b3584d56bcaac12d/manage.auth0.com_dashboard_us_pavo_login_page__2_.png)</Frame>

3. En el menú desplegable Default Template, elige la plantilla **Custom Login Form**. La plantilla HTML se actualizará con código que utiliza CSS y el SDK de auth0.js.

4. Usa HTML y CSS para personalizar la plantilla según tus necesidades y haz clic en **Save Changes**. Para obtener más información sobre cómo usar el SDK de auth0.js con la plantilla **Custom Login Form**, consulta [SDK de auth0.js](/docs/es/libraries/auth0js).

5. Selecciona la pestaña **Preview** para revisar tus cambios. Asegúrate de seleccionar la aplicación correcta para la que quieres previsualizar la página de login.

Para integrar Universal Login en tu aplicación, consulta nuestros [Quickstarts](/docs/es/quickstarts), donde encontrarás ejemplos completos para todos los tipos de aplicación y las plataformas más populares.
