title: Autenticación Passwordless con Universal Login
---

[Universal Login](/docs/es/authenticate/login/auth0-universal-login) es la implementación de Auth0 del flujo de login. Cada vez que un usuario necesita demostrar su identidad, sus aplicaciones redirigen a <Tooltip tip="Universal Login: Su aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip>, y Auth0 hará lo necesario para garantizar la identidad del usuario. Es la forma preferida de implementar la <Tooltip tip="Passwordless: Forma de autenticación que no depende de una contraseña como primer factor." cta="Ver glosario" href="/docs/es/glossary?term=Passwordless">autenticación Passwordless</Tooltip>.

Para implementar la autenticación Passwordless en Universal Login, debe personalizar el HTML de la página de login.

1. Vaya a [Dashboard &gt; Branding &gt; Universal Login](https://manage.auth0.com/#/login_settings) y haga clic en **Advanced Options**.

2. Haga clic en la pestaña **Login** y, a continuación, habilite el interruptor **Customize Login Page**.

<Frame>![Login personalizado](/docs/images/cdy7uua7fh8z/5iwLcz2O100hLuxlKN4rSs/e4f35f8c44516b98b3584d56bcaac12d/manage.auth0.com_dashboard_us_pavo_login_page__2_.png)</Frame>

3. En el menú desplegable **Default Template**, elija la plantilla **Custom Login Form**. La plantilla HTML se actualizará con código que utiliza CSS y el SDK de auth0.js.

4. Use HTML y CSS para personalizar la plantilla según sus necesidades y haga clic en **Save Changes**. Para obtener más información sobre cómo usar el SDK de auth0.js con la plantilla **Custom Login Form**, consulte [SDK de auth0.js](/docs/es/libraries/auth0js).

5. Seleccione la pestaña **Preview** para revisar sus cambios. Asegúrese de seleccionar la aplicación correcta para la que desea previsualizar la página de login.

Para integrar Universal Login en su aplicación, consulte nuestros [Quickstarts](/docs/es/quickstarts), donde encontrará ejemplos completos para todos los tipos de aplicación y las plataformas más populares.
