---
title: Autenticación passwordless con Embedded Login
description: Describe cómo implementar la autenticación passwordless con Universal Login.
---

Para integrar la interfaz de usuario de login en tu aplicación, puedes usar nuestra [API de Passwordless para Embedded Login](/docs/es/authenticate/passwordless/implement-login/embedded-login/relevant-api-endpoints) o nuestros SDKs.

Según el tipo de aplicación que quieras crear, debes implementarla de forma diferente:

* Para aplicaciones de página única (SPA) (por ejemplo, Angular o React), lee [Autenticación passwordless con Embedded Login para aplicaciones de página única (SPA)](/docs/es/authenticate/passwordless/implement-login/embedded-login/spa).
* Para aplicaciones nativas (por ejemplo, iOS, Android o aplicaciones de escritorio), lee [Login passwordless con Embedded Login en aplicaciones nativas](/docs/es/authenticate/passwordless/implement-login/embedded-login/native).
* Para aplicaciones web clásicas (por ejemplo, NodeJS, Java, Rails o .NET), lee [Login passwordless con Embedded Login en aplicaciones web clásicas](/docs/es/authenticate/passwordless/implement-login/embedded-login/webapps).

El Embedded Login en las aplicaciones web tradicionales y las SPA requiere configurar la autenticación de origen cruzado. Para obtener más información, consulta [Autenticación de origen cruzado](/docs/es/authenticate/login/cross-origin-authentication).
