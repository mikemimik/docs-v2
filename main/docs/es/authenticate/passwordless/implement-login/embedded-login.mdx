---
title: Autenticación Passwordless con inicio de sesión integrado
description: Describe cómo implementar la autenticación Passwordless con Universal Login.
---

Para integrar la interfaz de usuario de inicio de sesión en tu aplicación, puedes usar nuestra [API de Passwordless para inicio de sesión integrado](/docs/es/authenticate/passwordless/implement-login/embedded-login/relevant-api-endpoints) o nuestros SDKs.

Según el tipo de aplicación que quieras crear, debes implementarlo de forma diferente:

* Para aplicaciones de página única (SPA) (por ejemplo, Angular o React), lee [Autenticación Passwordless integrada para aplicaciones de página única (SPA)](/docs/es/authenticate/passwordless/implement-login/embedded-login/spa).
* Para aplicaciones nativas (por ejemplo, iOS, Android o aplicaciones de escritorio), lee [Inicio de sesión Passwordless integrado en aplicaciones nativas](/docs/es/authenticate/passwordless/implement-login/embedded-login/native).
* Para aplicaciones web clásicas (por ejemplo, NodeJS, Java, Rails o .NET), lee [Inicio de sesión Passwordless integrado en aplicaciones web clásicas](/docs/es/authenticate/passwordless/implement-login/embedded-login/webapps).

El inicio de sesión integrado en aplicaciones web clásicas y SPA requiere configurar la autenticación de origen cruzado. Para obtener más información, lee [Autenticación de origen cruzado](/docs/es/authenticate/login/cross-origin-authentication).
