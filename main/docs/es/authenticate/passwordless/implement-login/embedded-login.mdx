---
title: Autenticación passwordless con Embedded Login
description: Describe cómo implementar la autenticación passwordless con Universal Login.
---

Para integrar la interfaz de usuario de inicio de sesión en tu aplicación, puedes usar nuestra [API de passwordless para Embedded Login](/docs/es/authenticate/passwordless/implement-login/embedded-login/relevant-api-endpoints) o nuestros SDKs.

Según el tipo de aplicación que quieras crear, debes implementarla de forma diferente:

* Para Single Page Applications (SPA) (por ejemplo, Angular o React), consulta [Autenticación passwordless con Embedded Login para aplicaciones de página única (SPA)](/docs/es/authenticate/passwordless/implement-login/embedded-login/spa).
* Para aplicaciones nativas (por ejemplo, iOS, Android o aplicaciones de escritorio), consulta [Inicio de sesión passwordless con Embedded Login en aplicaciones nativas](/docs/es/authenticate/passwordless/implement-login/embedded-login/native).
* Para aplicaciones web clásicas (por ejemplo, NodeJS, Java, Rails o .NET), consulta [Inicio de sesión passwordless con Embedded Login en aplicaciones web clásicas](/docs/es/authenticate/passwordless/implement-login/embedded-login/webapps).

El Embedded Login en las aplicaciones web clásicas y las SPA requiere configurar la autenticación de origen cruzado. Para obtener más información, consulta [Cross-Origin Authentication](/docs/es/authenticate/login/cross-origin-authentication).
