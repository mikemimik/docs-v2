---
description: Describe algunas limitaciones del uso de las conexiones passwordless.
title: Limitaciones de las conexiones passwordless
---

El tipo de autenticación que desea usar depende de qué versión de <Tooltip tip="Universal Login: Su aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip> configuró para su aplicación. Si usa la experiencia de Universal Login, puede configurar la [autenticación passwordless mediante WebAuthN con Device Biometrics](/docs/es/authenticate/login/auth0-universal-login/passwordless-login/webauthn-device-biometrics) o usar conexiones <Tooltip tip="Universal Login: Su aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=passwordless">passwordless</Tooltip>, como se describe a continuación.

Puede configurar la autenticación passwordless por correo electrónico o SMS mediante los siguientes métodos:

1. Universal Login + parámetros de conexión passwordless
2. Classic Login + Lock
3. Classic Login + interfaz de usuario personalizada + Auth0.js

Para obtener más información sobre estas implementaciones, consulte [Configurar correo electrónico o SMS para la autenticación passwordless](/docs/es/authenticate/login/auth0-universal-login/passwordless-login/email-or-sms).

<div id="errors">
  ### Errors
</div>

Si usas inicio de sesión integrado, te recomendamos usar <Tooltip tip="Dominio personalizado: dominio de terceros con un nombre especializado o personalizado." cta="Ver glosario" href="/docs/es/glossary?term=custom+domains">dominios personalizados</Tooltip>. Para obtener más información, consulta [Dominios personalizados](/docs/es/customize/custom-domains).

<div id="considerations">
  ## Consideraciones
</div>

* No puedes crear usuarios passwordless desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Créelos directamente desde la [Management API](https://auth0.com/docs/api/management/v2#!/Users/post_users).
* Los <Tooltip tip="Token de actualización: token usado para obtener un token de acceso renovado sin obligar a los usuarios a iniciar sesión de nuevo." cta="Ver glosario" href="/docs/es/glossary?term=Refresh+tokens">tokens de actualización</Tooltip> no se pueden obtener al usar un Magic Link passwordless por correo electrónico; solo un OTP.
