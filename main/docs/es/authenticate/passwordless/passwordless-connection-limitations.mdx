---
description: Describe algunas limitaciones del uso de conexiones passwordless.
title: Limitaciones de las conexiones passwordless
---

El tipo de autenticación que quieras usar depende de la versión de <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip> que hayas configurado en tu aplicación. Si usas la experiencia de Universal Login, puedes configurar la [autenticación passwordless con WebAuthn with Device Biometrics](/docs/es/authenticate/login/auth0-universal-login/passwordless-login/webauthn-device-biometrics) o usar conexiones <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=passwordless">passwordless</Tooltip>, como se describe a continuación.

Puedes configurar la autenticación passwordless por correo electrónico o SMS mediante los siguientes métodos:

1. Universal Login + parámetros de conexión passwordless
2. Classic Login + Lock
3. Classic Login + interfaz personalizada + Auth0.js

Para obtener más información sobre estas implementaciones, consulta [Configurar el correo electrónico o SMS para la autenticación passwordless](/docs/es/authenticate/login/auth0-universal-login/passwordless-login/email-or-sms).

<div id="errors">
  ### Errors
</div>

Si estás usando un inicio de sesión integrado, te recomendamos usar <Tooltip tip="Dominio personalizado: dominio de terceros con un nombre especializado o de marca." cta="Ver glosario" href="/docs/es/glossary?term=custom+domains">dominios personalizados</Tooltip>. Para obtener más información, consulta [Custom Domains](/docs/es/customize/custom-domains).

<div id="considerations">
  ## Consideraciones
</div>

* No puedes crear usuarios passwordless desde el <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Créalos directamente con la [Management API](https://auth0.com/docs/api/management/v2#!/Users/post_users).
* Los <Tooltip tip="Token de actualización: Token utilizado para obtener un token de acceso renovado sin obligar a los usuarios a iniciar sesión de nuevo." cta="Ver glosario" href="/docs/es/glossary?term=Refresh+tokens">tokens de actualización</Tooltip> no se pueden obtener al usar un Magic Link passwordless por correo electrónico, solo un OTP.
