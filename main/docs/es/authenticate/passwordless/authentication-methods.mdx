Con las <Tooltip tip="Passwordless: Forma de autenticación que no depende de una contraseña como primer factor." cta="Ver glosario" href="/docs/es/glossary?term=Passwordless">Passwordless</Tooltip> conexiones, los usuarios pueden iniciar sesión sin contraseña. En su lugar, pueden usar otros métodos de autenticación. Las conexiones Passwordless de Auth0 admiten los siguientes métodos de autenticación:

| Factor                                                                                | Descripción                                                                                                                                                                                                                                                                       |
| ------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Email](/docs/es/authenticate/passwordless/authentication-methods/email-otp)             | Se solicita al usuario que introduzca su dirección de correo electrónico; Auth0 le envía allí un código de un solo uso. Luego, el usuario introduce el código en su aplicación.                                                                                                   |
| [Magic Link](/docs/es/authenticate/passwordless/authentication-methods/email-magic-link) | Se solicita al usuario que introduzca su dirección de correo electrónico; Auth0 le envía un correo con un enlace. El usuario hace clic en el enlace e inicia sesión directamente en su aplicación.                                                                                |
| [SMS](/docs/es/authenticate/passwordless/authentication-methods/sms-otp)                 | Se solicita al usuario que introduzca su número de teléfono; Auth0 le envía allí un código de un solo uso. De forma predeterminada, Auth0 usa Twilio para enviar el código, pero si tiene un gateway de SMS personalizado, puede modificar su conexión para usar ese en su lugar. |
