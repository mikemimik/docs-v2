Con [Rules](/docs/es/customize/rules), puedes gestionar casos más complejos de los que son posibles solo con [conexiones passwordless](/docs/es/authenticate/passwordless). Por ejemplo, puedes añadir medidas adicionales para reforzar la verificación de la posesión de una dirección de correo electrónico o de un dispositivo.

<div id="require-multi-factor-authentication-for-users-who-are-outside-the-corporate-network">
  ## Requerir autenticación multifactor para los usuarios que estén fuera de la red corporativa
</div>

Supongamos que quieres exigir [autenticación multifactor (MFA)](/docs/es/secure/multi-factor-authentication) a cualquier usuario que acceda a la aplicación mediante una conexión <Tooltip tip="Passwordless: Forma de autenticación que no depende de una contraseña como primer factor." cta="Ver glosario" href="/docs/es/glossary?term=passwordless">passwordless</Tooltip> desde fuera de tu red corporativa.

Con una rule, puedes comprobar si un usuario se está autenticando mediante un método passwordless (`sms`, `email`) y, si la IP de su sesión está fuera de la red corporativa designada, solicitarle un segundo factor de autenticación.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  También podrías activar esta rule en función de otros criterios, por ejemplo, si la IP actual coincide con la lista de IP permitidas del usuario o si la geolocalización del usuario indica que se encuentra en un país distinto del que figura en su perfil de usuario.
</Callout>

Para ello, [crea la siguiente rule](/docs/es/customize/rules/create-rules):

```javascript lines
function(user, context, callback) {
  const ipaddr = require('ipaddr.js');
  const corp_network = "192.168.1.134/26";
  const current_ip = ipaddr.parse(context.request.ip);
  // ¿el método de autenticación es passwordless y la IP está fuera de la red corporativa?
  const passwordlessOutside = context.authentication.methods.find(
    (method) => (
      ((method.name === 'sms') || (method.name === 'email')) && 
      (!current_ip.match(ipaddr.parseCIDR(corp_network)))
    )
  );

  // si es así, solicitar MFA
  if (passwordlessOutside) {
    context.multifactor = {
      provider: 'any',
      allowRememberBrowser: false
    };
  }
  callback(null, user, context);
}
```
