---
description: Aprende a habilitar el add-on SAML2 Web App para usar Auth0 como proveedor de identidad para inicio de sesión único (SSO) de SAML.
title: Habilitar SAML2 Web App Addon
---

Puedes configurar un add-on SAML2 Web App para usarlo con una aplicación.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications).
2. Identifica la aplicación con la que quieres usar SAML2 Web App y haz clic en el nombre de la aplicación para ir a su configuración.
3. Ve a la pestaña **Addons**.
4. En el cuadro **SAML2 Web App**, haz clic en el interruptor para habilitar el Addon.

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Settings Tab](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>
5. Ve a la pestaña **Settings** e indica la **URL de devolución de llamada de la aplicación** que recibe la respuesta SAML, así como los valores de **Settings**. En la pestaña **Settings** hay un objeto de ejemplo que puedes usar para determinar qué ajustes quieres habilitar.
6. Haz clic en **Debug** para comprobar si los parámetros que proporcionaste son válidos.
7. Haz clic en **Save** cuando termines.

<div id="provide-application-details">
  ## Proporciona los datos de la aplicación
</div>

Aunque los pasos concretos para proporcionar a tu SAML2 Web App los datos de tu aplicación pueden variar, puedes obtener la información de configuración que necesitas en la pestaña **Usage** después de configurar el addon.

<Frame>![Dashboard Applications Applications Addons pestaña SAML2 Web App pestaña Usage](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>

  ## Limitación
</div>
