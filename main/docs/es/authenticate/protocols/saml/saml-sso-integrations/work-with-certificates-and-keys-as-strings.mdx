---
description: Describe cómo trabajar con certificados o claves en reglas o solicitudes a la Management API que requieren una representación en cadena del archivo.
title: Trabajar con certificados y claves como cadenas
---

Puedes usar el siguiente comando en un shell de UNIX para concatenar un certificado en formato PEM con `\n` (salto de línea escapado):

```bash wrap lines
awk 'NF {sub(/\r/, ""); printf "%s\\n",$0;}' certificate.pem | pbcopy
```

Esto copia el valor en tu portapapeles para que puedas pegarlo donde lo necesites.

<div id="more-information">
  ### Más información
</div>

Al trabajar con certificados o claves en reglas o en solicitudes de <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> v2, lo más probable es que necesites una representación en cadena del archivo.

Si abres un archivo de certificado (`cer`, `pem`) con un editor de texto, verás algo así:

```text lines
-----BEGIN CERTIFICATE-----
MIICzDCCAbQCCQDH8GvxPIeH+DANBgkqhkiG9w0BAQsFADAoMQswCQYDVQQGEwJh
cjEZMBcGA1UEAwwQaHR0cHM6Ly9uaWNvLmNvbTAeFw0xOTA0MDgxODA3NTVaFw0y
//
// más líneas de información codificada en base64
//
nSWyabd+LiBGtLTMB+ZLbOIi3EioWPGw/nHOI8jzPrqhiCLuZCSQmiqrLQYNsc1W
-----END CERTIFICATE-----
```

Las líneas entre el encabezado `-----BEGIN CERTIFICATE-----` y el pie `-----END CERTIFICATE-----` contienen información binaria codificada en base64. Las claves públicas y las claves privadas (archivos .key) se verán parecidas; solo cambiarán el encabezado y el pie.

Para obtener una representación como cadena de un archivo de certificado/clave, tendrás que concatenarlo todo en una sola línea, reemplazando los saltos de línea reales del archivo por una secuencia `\n` (salto de línea de escape). Así, a partir del ejemplo anterior, obtendrías algo como esto:

```text lines
"-----BEGIN CERTIFICATE-----
\nMIICzDCCAbQCCQDH8GvxPIeH+DANBgkqhkiG9w0BAQsFADAoMQswCQYDVQQGEwJh\n
[..all the other lines..]
-----END CERTIFICATE-----\n"
```
