Auth0 admite perfiles de algoritmos predefinidos para ofrecer una forma sencilla de especificar una lista de algoritmos criptográficos seguros sin tener que indicar cada algoritmo por separado según su caso de uso. Estos perfiles representan un conjunto predefinido de algoritmos que sirve como línea base para el patrón de uso previsto.

<div id="naming-conventions">
  ## Convenciones de nomenclatura
</div>

Los perfiles se nombran según el año en que se crean, con un número correlativo para indicar las versiones posteriores del mismo año. Siempre que sea posible, debe usarse la versión más reciente del perfil para contar con la configuración más segura.

<div id="supported-profiles">
  ## Perfiles compatibles
</div>

Auth0 como proveedor de servicios SAML (SP)
