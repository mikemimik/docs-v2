title: Protocolo ligero de acceso a directorios
---

El Protocolo ligero de acceso a directorios (LDAP) es un protocolo de aplicación que se utiliza para acceder a servicios distribuidos de información de directorio y mantenerlos a través de una red de Protocolo de Internet (IP). La función de LDAP es permitir el acceso a un directorio existente, como Active Directory (AD). Auth0 usa un [Conector de Active Directory/LDAP](/docs/es/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) que instalas en tu red para integrarlo con AD/LDAP.
