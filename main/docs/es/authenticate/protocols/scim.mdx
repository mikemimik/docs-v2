---
title: Sistema de gestión de identidades entre dominios (SCIM)
description: Describe el uso del esquema de System for Cross-domain Identity Management (SCIM) en la gestión de identidades
---

SCIM (System for Cross-domain Identity Management) es un conjunto de protocolos de nivel de aplicación para gestionar y transmitir de forma segura los datos de los usuarios entre varios dominios. Los clientes SCIM pueden integrarse para gestionar operaciones CRUD (create, replace, update, delete), aplicar consultas y filtros, y crear grupos de usuarios dentro de tu organización. SCIM te permite automatizar el ciclo de vida de los usuarios y mantener las cuentas de usuario en distintas plataformas.

Para consultar la especificación de SCIM, consulta [System for Cross-domain Identity Management: Core Schema](https://datatracker.ietf.org/doc/rfc7643/).

<div id="scim-with-auth0">
  ## SCIM con Auth0
</div>

Auth0 ofrece un directorio extensible y flexible diseñado para admitir casos de uso de CIAM, y se centra en simplificar la gestión de identidades para aplicaciones de consumo directo y de software como servicio.

En un tenant de Auth0, puedes usar directorios aislados mediante conexiones Enterprise, Social o Database. Para obtener más información sobre las conexiones disponibles, consulta [Identity Providers](/docs/es/authenticate/identity-providers) o [Database Connections](/docs/es/authenticate/database-connections).
