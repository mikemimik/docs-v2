Para obtener más información sobre las operaciones de administración de SCIM compatibles y cómo utilizarlas, ve a [Auth0 Management API Explorer](https://auth0.com/docs/api/management/v2) y despliega la pestaña **Connections**. La <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=management+API">management API</Tooltip> puede habilitar, actualizar y eliminar la configuración de SCIM para una conexión seleccionada, así como crear y eliminar tokens de SCIM.

<div id="register-a-client-app-for-management-api-communication">
  ### Registrar una aplicación cliente para comunicarse con la Management API
</div>

Para usar Auth0 Management API, registra una aplicación cliente de máquina a máquina que tenga los permisos necesarios para administrar tu configuración de SCIM.

1. En el [Auth0 Dashboard](https://manage.auth0.com/#/), selecciona **Applications**.
2. Selecciona **Create Application**.
3. Introduce un nombre, selecciona **Machine to Machine App** y luego **Create**.
4. En tu nueva aplicación, selecciona la pestaña **APIs** y establece **Auth0 Management API** como **Authorized**.
5. En **Permissions**, selecciona `read:scim_config`, `create:scim_config`, `update:scim_config`, `delete:scim_config`, `create:scim_token`, `read:scim_token` y `delete:scim_token.`
6. Selecciona **Update**.
7. Sigue las instrucciones de [Tokens de acceso de la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens) para aprender a obtener tokens y llamar a los endpoints de [Auth0 Management API v2](https://auth0.com/docs/api/management/v2/connections/post-scim-configuration).
