title: Configurar Amazon Web Services como proveedor de servicios SAML
---

Configura el complemento SAML2 Web App para Amazon Web Services (AWS) en una aplicación.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y crea una aplicación nueva o haz clic en el nombre de una aplicación para actualizarla.

2. Ve a la pestaña **Addons** y habilita el interruptor **SAML2 Web App**.

3. En la pestaña **Settings**, establece la **URL de devolución de llamada de la aplicación** en: `https://signin.aws.amazon.com/saml`.

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Settings Tab](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>

4. Pega el siguiente código en el cuadro de texto **Settings** y haz clic en **Debug**. Asegúrate de reemplazar los marcadores de posición `AWS_SSO_ISSUER_URL` y `AWS_SSO_ACS_URL` por los valores que copiaste de AWS.

   ```json lines
   {
     "audience": "AWS_SSO_ISSUER_URL",
     "destination": "AWS_SSO_ACS_URL",
     "mappings": {
       "email": "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/emailaddress",
       "name": "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/name"
     },
     "createUpnClaim": false,
     "passthroughClaimsWithNoMapping": false,
     "mapUnknownClaimsAsIs": false,
     "mapIdentities": false,
     "nameIdentifierFormat": "urn:oasis:names:tc:SAML:1.1:nameid-format:emailAddress",
     "nameIdentifierProbes": [
       "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/emailaddress"
     ]
   }
   ```

5. Desplázate hasta el final de la página y haz clic en **Enable**.

6. En la pestaña **Usage**, busca **Identity Provider Metadata** y haz clic en **Download** para descargar el archivo de metadatos. Lo necesitarás cuando configures Auth0 como proveedor de identidad.

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Usage Tab](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>
