---
description: Aprende a configurar Auth0 como proveedor de identidad mediante el addon SAML2 Web App para Egencia.
title: Configurar Egencia como proveedor de servicios SAML
---

Usa la siguiente configuración de <Tooltip tip="Security Assertion Markup Language (SAML): Protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin contraseña." cta="Ver glosario" href="/docs/es/glossary?term=SAML">SAML</Tooltip> para Egencia.

1. Ve a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y crea una nueva aplicación o haz clic en el nombre de una existente para actualizarla.

2. Ve a la pestaña **Addons** y habilita el interruptor **SAML2 Web App**.

3. En la pestaña **Settings**, establece la **Application Callback URL** en: `https://www.egencia.com/auth/v1/artifactConsumer`.

   <Frame>![Dashboard Applications Applications Addons pestaña SAML2 Web App Settings pestaña](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>

4. Pega el siguiente código en el cuadro de texto **Settings** y haz clic en **Debug**.

   ```json lines
   {
         "audience": "https://{yourAccountName}.egencia.com",
          "mappings": {
          },
          "createUpnClaim":       false,
          "passthroughClaimsWithNoMapping": false,
          "mapUnknownClaimsAsIs": false,
          "mapIdentities":        false,
          "nameIdentifierFormat": "urn:oasis:names:tc:SAML:1.1:nameid-format:emailAddress",
          "nameIdentifierProbes": [
        "http://schemas.xmlsoap.org/ws/2005/05/identity/claims/emailaddress",
          ],
       }
   ```

5. Desplázate hasta la parte inferior de la pestaña y haz clic en **Enable**.

6. En la pestaña **Usage**, localiza **Identity Provider Metadata** y haz clic en **Download** para descargar el archivo de metadatos. Lo necesitarás cuando configures Auth0 como proveedor de identidad (IdP).

   <Frame>![Dashboard Applications Applications Addons pestaña SAML2 Web App Usage pestaña](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>
