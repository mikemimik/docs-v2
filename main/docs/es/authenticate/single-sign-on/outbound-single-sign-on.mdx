---

Para el <Tooltip tip="Inicio de sesión único (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia sesión automáticamente a ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=Single+Sign-On">inicio de sesión único</Tooltip> (SSO) iniciado por el proveedor de identidad, un <Tooltip tip="Inicio de sesión único (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia sesión automáticamente a ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=Identity+Provider">Proveedor de identidad</Tooltip> (IdP) de terceros es el proveedor de SSO. Cuando un usuario inicia sesión en una aplicación:

1. La aplicación redirige al usuario a un proveedor de identidad.
2. El proveedor de identidad de terceros realiza la autenticación y autorización.
3. Tras autenticarse correctamente, el usuario regresa a la aplicación.

Auth0 ofrece [integraciones de SSO](/docs/es/customize/integrations/sso-integrations) iniciadas por IdP para varios servicios, como [Dropbox](https://marketplace.auth0.com/integrations/dropbox-sso), [Slack](https://marketplace.auth0.com/integrations/slack-sso) o [Zoom](https://marketplace.auth0.com/integrations/zoom-sso). Para ver la lista completa, explora [Auth0 Marketplace: integraciones de SSO](https://marketplace.auth0.com/features/sso-integrations).
