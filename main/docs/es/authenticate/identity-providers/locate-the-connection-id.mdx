---
description: Usa Auth0 Dashboard para encontrar el ID de una conexión de Auth0 y usarlo al configurar servicios de Auth0, como Actions y la gestión de usuarios.
title: Localiza el ID o el nombre de la conexión
---

Algunas tareas de gestión de usuarios requieren un ID de conexión, que es un identificador único de un origen de usuarios. Por ejemplo, si importas o exportas usuarios, o te conectas a <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=identity+providers">proveedores de identidad</Tooltip>, debes proporcionar un ID de conexión.

1. Ve a [Auth0 Dashboard](https://manage.auth0.com/#/), selecciona **Authentication** y luego el tipo de conexión cuyo ID necesitas.
2. Selecciona el nombre de la conexión cuyo ID necesitas.
3. En la vista **Settings**, busca el campo **Name**; ahí aparece el ID de la conexión.
