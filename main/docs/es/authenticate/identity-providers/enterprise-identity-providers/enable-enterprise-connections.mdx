description: Describe cómo habilitar conexiones empresariales en las aplicaciones mediante Auth0 Dashboard.
title: Habilitar conexiones empresariales
---

Puedes habilitar conexiones empresariales en las aplicaciones mediante el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Para habilitar tu conexión empresarial, primero debes haberla configurado.

1. Ve a [Auth0 Dashboard &gt; Autenticación &gt; Empresarial](https://manage.auth0.com/#/connections/enterprise) y selecciona el tipo de conexión que deseas ver.

   <Frame>![Dashboard - Conexiones - Empresarial](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. Selecciona el nombre de la conexión que deseas ver.

   <Frame>![Dashboard - Conexiones - Empresarial - AD/LDAP - Lista](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/LDAP_page_-_EN.png)</Frame>
3. Selecciona la vista **Aplicaciones** y habilita o deshabilita la conexión para las aplicaciones correspondientes.

   <Frame>![Dashboard - Conexiones - Empresarial - Aplicaciones](/docs/images/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/38fbae514304297bbbf72beeedb8892f/SSL_-_EN.png)</Frame>

Cuando configures la conexión con tu <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=IdP">IdP</Tooltip> empresarial, recuerda que el parámetro `connection` que se pasa al endpoint `/authorize,` `/whr,` o `/wsfed` de Auth0 redirigirá al IdP de origen **a menos que** la conexión se realice mediante Database o Active Directory.
