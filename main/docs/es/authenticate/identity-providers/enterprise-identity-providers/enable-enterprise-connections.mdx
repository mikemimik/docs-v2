---
description: Describe cómo habilitar conexiones empresariales para aplicaciones desde el Auth0 Dashboard.
title: Habilitar conexiones empresariales
---

Puedes habilitar conexiones empresariales para aplicaciones desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Para habilitar tu conexión empresarial, primero debes haberla configurado.

1. Ve a [Auth0 Dashboard &gt; Autenticación &gt; empresarial](https://manage.auth0.com/#/connections/enterprise) y selecciona el tipo de conexión que deseas ver.

   <Frame>![Dashboard - Conexiones - empresarial](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. Selecciona el nombre de la conexión que deseas ver.

   <Frame>![Dashboard - Conexiones - empresarial - AD/LDAP - Lista](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/LDAP_page_-_EN.png)</Frame>
3. Selecciona la vista **Applications** y habilita o deshabilita la conexión para las aplicaciones correspondientes.

   <Frame>![Dashboard - Conexiones - empresarial - Applications](/docs/images/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/38fbae514304297bbbf72beeedb8892f/SSL_-_EN.png)</Frame>

Al configurar tu conexión empresarial de <Tooltip tip="Identity Provider (IdP): servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=IdP">IdP</Tooltip>, recuerda que el parámetro `connection` que se pasa al endpoint `/authorize,` `/whr,` o `/wsfed` de Auth0 redirigirá al IdP externo **a menos que** esté conectado mediante una base de datos o Active Directory.
