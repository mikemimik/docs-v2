---
description: Actualiza un Conector AD/LDAP de Auth0 instalado a la versión más reciente en Windows o Linux, manteniendo la configuración existente y el vínculo con el tenant.
title: Actualizar conectores AD/LDAP
---

Si hay varias instancias del Conector AD/LDAP en tu entorno, Auth0 recomienda que sigas los pasos que se indican a continuación para cada instancia, de una en una, de modo que solo una instancia esté inactiva en cada momento.

<div id="admin-console-windows">
  ## Admin Console (Windows)
</div>

1. Abra la Admin Console accediendo a `http://localhost:8357/`. Si la Admin Console tiene acceso saliente a internet, comprobará si hay una nueva versión disponible y lo mostrará en la parte superior de la página:

   <Frame>![Pantalla de la Admin Console con actualización disponible del conector AD/LDAP](/docs/images/cdy7uua7fh8z/6Yp2uRnEU3RMeCqgHa6Ztl/b9c04ab83b9fd725910291b01ac9d05d/connector-update-available.png)</Frame>
2. Vaya a la pestaña **Update** para actualizar la instalación de esta máquina a la versión más reciente. La actualización tardará unos 2 minutos y los logs del actualizador se mostrarán en la Admin Console.

<div id="updater-script-windows">
  ## Script de actualización (Windows)
</div>

El script de actualización actualizará el Conector AD/LDAP desde la línea de comandos ejecutando los siguientes pasos:

1. Verifique si hay una actualización disponible.
2. Haga una copia de seguridad de la configuración existente, los certificados y profileMapper.js.
3. Desinstale el Conector AD/LDAP.
4. Descargue la actualización.
5. Instale el Conector AD/LDAP.
6. Restaure la configuración existente, los certificados y profileMapper.js.
7. Inicie el servicio de Windows.
8. Para ejecutar el script de actualización, ejecute la siguiente instrucción en la línea de comandos:
   `@powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((new-object net.webclient).DownloadString('https://cdn.auth0.com/connector/windows/update-connector.ps1'))"`

El script de actualización usa comandos específicos de PowerShell que solo están disponibles en Windows PowerShell 3.0 y versiones posteriores. Si lo ejecuta en Windows 2008 o Windows 2008 R2, es posible que primero deba actualizar su versión de PowerShell.

Para obtener más información, consulte [Descargar Window Management Framework](https://www.microsoft.com/en-us/download/details.aspx?id=34595).

<div id="manual-update-windowslinux">
  ## Actualización manual (Windows/Linux)
</div>

1. Verifica la versión que tienes instalada. Coloca el cursor sobre el indicador de estado del Conector en la consola. La descripción emergente mostrará el estado actual y la versión instalada.
2. Descarga la última versión del [instalador](https://cdn.auth0.com/connector/windows/adldap-5.0.10.msi?_ga=2.58973869.1675886805.1589821069-1123834949.1586212750).
   Para otras plataformas, usa la [versión](https://github.com/auth0/ad-ldap-connector/releases/tag/v5.0.10) del repositorio de GitHub.
3. Haz una copia de seguridad de tu configuración actual desde la Admin Console o manualmente.

   1. En la Admin Console, haz clic en el botón **Descargar** para generar un archivo .zip que contiene el archivo `config.json`, la carpeta `certs` y el archivo `lib\\profileMapper.js`.
   2. Para hacerlo manualmente, haz una copia de seguridad de los mismos archivos indicados arriba desde la carpeta `%Program Files(x86)%\Auth0\AD LDAP Connector\`. Esta ruta corresponde a Windows. En otras plataformas, los archivos pueden estar en otra ubicación, pero serán los mismos.
4. Ejecuta el instalador y sigue las instrucciones.
5. Cierra el cuadro de diálogo de configuración sin cambiar nada.
6. Restaura los archivos desde la Admin Console o manualmente.

   1. En la Admin Console, carga el archivo .zip en la pestaña **Import / Export**.
   2. Para hacerlo manualmente, copia los archivos de los que hiciste la copia de seguridad anteriormente.
7. Reinicia el servicio **Auth0 AD LDAP** desde la consola de servicios.
