description: Describe cómo importar o exportar una configuración del Conector AD/LDAP desde la Consola de administración del Conector.
title: Importar y exportar configuraciones del Conector AD/LDAP
---

Puede exportar la configuración del Conector AD/LDAP o importar una configuración exportada previamente. Esto resulta útil para implementaciones con más de un nodo del Conector AD/LDAP desplegado para alta disponibilidad. Puede configurar y probar la configuración en un nodo, luego exportar la configuración funcional e importarla en todos los nodos siguientes.

1. Desde un navegador, vaya a `http://localhost:8357` para abrir la **Consola de administración del Conector** en el servidor del Conector AD/LDAP.
2. Haga clic en la pestaña **Import / Export**.
3. Exporte un archivo .zip que contenga el archivo `config.json`, la carpeta `certs` y el archivo `lib\\profileMapper.js` de la carpeta `%Program Files(x86)%\Auth0\AD LDAP Connector\`.
4. Para importar, cargue un archivo .zip de respaldo en la pestaña **Import / Export**.
