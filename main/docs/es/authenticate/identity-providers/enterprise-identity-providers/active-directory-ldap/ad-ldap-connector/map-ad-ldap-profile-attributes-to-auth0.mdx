title: Mapear atributos de perfil de AD/LDAP al perfil de usuario de Auth0
---

Puede modificar el mapeo de atributos del perfil de AD/LDAP a atributos del perfil de usuario de Auth0.

1. En un navegador, vaya a `http://localhost:8357` para abrir la **Consola de administración del Conector** en el servidor del AD/LDAP Connector.
2. Haga clic en **Asignador de perfiles** para abrir una pantalla del editor de código que asigna atributos de un servicio de directorio de origen (representado por la variable `raw_data`) a una variable que se devuelve para completar el perfil de usuario de Auth0.
   La primera parte de la función crea una instancia de una variable llamada `profile` e incluye un mapeo para la parte principal del perfil de usuario de Auth0. Debajo de eso, se pueden establecer atributos adicionales con una sintaxis con este formato:
   `profile['department'] = raw_data['companydept'];`
   En este ejemplo, `department` es el nombre del atributo en el perfil de usuario de Auth0 y `companydept` es el nombre del atributo en el servicio de directorio de origen (como AD).
3. Haga clic en **Guardar** cuando haya terminado de modificar la configuración de LDAP.
4. Después de que el Conector ejecute las pruebas, asegúrese de que todas aparezcan en verde.

| Prueba   | Descripción                                                                                                                                   | Solución de problemas                                                                                    |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| Prueba 1 | Intenta establecer una conexión TCP con el servidor LDAP y el puerto especificados.                                                           | Verifique la conectividad básica de red y la configuración del firewall que podría impedir esa conexión. |
| Prueba 2 | Intenta realizar una vinculación LDAP en el servidor LDAP y el puerto especificados, con el nombre de usuario y la contraseña proporcionados. | Verifique la cadena de conexión LDAP, la ruta de búsqueda, el nombre de usuario y la contraseña.         |
| Prueba 3 | Intenta realizar una búsqueda LDAP en el directorio para comprobar los privilegios del nombre de usuario especificado.                        | Verifique los privilegios del nombre de usuario en el directorio de destino.                             |
| Prueba 4 | Intenta establecer una conexión con el servidor de Auth0.                                                                                     | Verifique la conectividad de red y la configuración del firewall que podría impedir esa conexión.        |
