description: Describe cómo mapear atributos de perfil de AD/LDAP a atributos del perfil de usuario de Auth0 mediante el Mapeador de perfiles en la Consola de administración del Conector.
title: Mapear atributos de perfil de AD/LDAP al perfil de usuario de Auth0
---

Puede modificar el mapeo de atributos del perfil desde AD/LDAP hacia atributos del perfil de usuario de Auth0.

1. En un navegador, vaya a `http://localhost:8357` para iniciar la **Consola de administración del Conector** en el servidor del AD/LDAP Connector.
2. Haga clic en **Mapeador de perfiles** para abrir una pantalla del editor de código que mapea atributos desde un servicio de directorio de origen (representado por la variable `raw_data`) a una variable que se devuelve para rellenar el perfil de usuario de Auth0.
   La primera parte de la función instancia una variable llamada `profile` y contiene un mapeo para la parte principal del perfil de usuario de Auth0. Debajo, puede definir atributos adicionales con una sintaxis como la siguiente:
   `profile['department'] = raw_data['companydept'];`
   En este ejemplo, `department` es el nombre del atributo en el perfil de usuario de Auth0 y `companydept` es el nombre del atributo en el servicio de directorio de origen (como AD).
3. Haga clic en **Save** cuando termine de modificar la configuración de LDAP.
4. Después de que el Conector ejecute las pruebas, asegúrese de que todas aparezcan en verde.

| Prueba | Descripción                                                                                                                            | Solución de problemas                                                                                   |
| ------ | -------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| Test 1 | Intenta establecer una conexión TCP con el servidor LDAP y el puerto especificados.                                                    | Compruebe la conectividad básica de red y la configuración del firewall que pueda impedir esa conexión. |
| Test 2 | Intenta realizar un enlace LDAP en el servidor LDAP y el puerto especificados con el nombre de usuario y la contraseña proporcionados. | Compruebe la cadena de conexión LDAP, la ruta de búsqueda, el nombre de usuario y la contraseña.        |
| Test 3 | Intenta realizar una búsqueda LDAP en el directorio para comprobar los privilegios del nombre de usuario especificado.                 | Compruebe los privilegios del nombre de usuario en el directorio de destino.                            |
| Test 4 | Intenta establecer una conexión con el servidor de Auth0.                                                                              | Compruebe la conectividad de red y la configuración del firewall que pueda impedir esa conexión.        |
