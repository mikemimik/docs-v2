---
description: Describe cómo mapear atributos de perfil de AD/LDAP a atributos del perfil de usuario de Auth0 mediante Profile Mapper en la Connector Admin Console.
title: Mapear atributos de perfil de AD/LDAP al perfil de usuario de Auth0
---

Puede modificar el mapeo de atributos de perfil de AD/LDAP a atributos del perfil de usuario de Auth0.

1. En un navegador, vaya a `http://localhost:8357` para abrir la **Connector Admin Console** en el servidor de AD/LDAP Connector.
2. Haga clic en **Profile Mapper** para abrir una pantalla de editor de código que mapea atributos de un servicio de directorio de origen (representado por la variable `raw_data`) a una variable que se devuelve para rellenar el perfil de usuario de Auth0.
   La primera parte de la función inicializa una variable llamada `profile` e incluye un mapeo para la parte principal del perfil de usuario de Auth0. Debajo, puede definir atributos adicionales con una sintaxis como esta:
   `profile['department'] = raw_data['companydept'];`
   En este ejemplo, `department` es el nombre del atributo en el perfil de usuario de Auth0 y `companydept` es el nombre del atributo en el servicio de directorio de origen (como AD).
3. Haga clic en **Save** cuando termine de modificar la configuración de LDAP.
4. Después de que el Connector ejecute las pruebas, asegúrese de que todas aparezcan en verde.

| Prueba   | Descripción                                                                                                                            | Solución de problemas                                                                                    |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| Prueba 1 | Intenta establecer una conexión TCP con el servidor LDAP y el puerto especificados.                                                    | Verifique la conectividad básica de red y la configuración del firewall que podría impedir esa conexión. |
| Prueba 2 | Intenta realizar un enlace LDAP en el servidor LDAP y el puerto especificados con el nombre de usuario y la contraseña proporcionados. | Verifique la cadena de conexión LDAP, la ruta de búsqueda, el nombre de usuario y la contraseña.         |
| Prueba 3 | Intenta realizar una búsqueda LDAP en el directorio para comprobar los privilegios del nombre de usuario especificado.                 | Verifique los privilegios del nombre de usuario en el directorio de destino.                             |
| Prueba 4 | Intenta establecer una conexión con el servidor de Auth0.                                                                              | Verifique la conectividad de red y la configuración del firewall que podría impedir esa conexión.        |
