description: Aprende a deshabilitar la caché de credenciales a nivel de la conexión para una conexión empresarial AD/LDAP desde el Auth0 Dashboard.
title: Deshabilitar la caché de credenciales de una conexión AD/LDAP
---

Puedes deshabilitar la caché de credenciales a nivel de la conexión para una [conexión AD/LDAP](/docs/es/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) desde el <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Ve a [Auth0 Dashboard &gt; Authentication &gt; Empresarial](https://manage.auth0.com/#/connections/enterprise) y selecciona el tipo de conexión **Active Directory/LDAP**.
2. Selecciona el nombre de la conexión que quieres editar.
3. Activa el interruptor **Disable Cache**.
