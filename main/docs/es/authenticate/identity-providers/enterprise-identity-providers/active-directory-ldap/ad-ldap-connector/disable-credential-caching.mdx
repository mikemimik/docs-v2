description: Aprende a desactivar la caché de credenciales a nivel de conexión para una conexión empresarial AD/LDAP con el Auth0 Dashboard.
title: Desactivar la caché de credenciales de la conexión AD/LDAP
---

Puedes desactivar la caché de credenciales a nivel de conexión para una [conexión AD/LDAP](/docs/es/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) usando el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Ve a [Auth0 Dashboard &gt; Autenticación &gt; empresarial](https://manage.auth0.com/#/connections/enterprise) y selecciona el tipo de conexión **Active Directory/LDAP**.
2. Selecciona el nombre de la conexión que quieres editar.
3. Activa el interruptor **Disable Cache**.
