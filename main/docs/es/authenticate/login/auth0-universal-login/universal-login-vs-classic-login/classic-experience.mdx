description: Descripción general de la experiencia de Classic Login
title: Experiencia de Classic Login
---

Classic Login es una experiencia de inicio de sesión alojada por Auth0 que se basa en JavaScript para su personalización. Implementar Classic Login es menos complejo que integrar directamente el proceso de autenticación en tu aplicación, y puede ayudar a evitar los riesgos de la autenticación de origen cruzado.

Classic Login se basa en las bibliotecas JavaScript de Auth0 ([Lock.js](/docs/es/libraries/lock), [auth0.js](/docs/es/libraries/auth0js), <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuario que usa un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=MFA">MFA</Tooltip> Widget y Restablecimiento de contraseña). Al personalizar las páginas de Classic Login en el <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, las plantillas predeterminadas usan las mismas bibliotecas JavaScript para crear una transición más natural entre la interfaz de usuario predeterminada y una personalizada.

Después de elegir una plantilla predeterminada para tus páginas de Classic Login, puedes modificarla para adaptarla a tus necesidades. También puedes personalizar diversos elementos de comportamiento y apariencia del widget de Lock. Las plantillas de Auth0.js ofrecen mayor flexibilidad, ya que te permiten crear una interfaz de usuario personalizada y adaptarla al estilo de tu aplicación.

<Warning>
  En este momento, los esfuerzos de desarrollo activo de Auth0 están centrados en Universal Login, y Classic Login ya no recibe actualizaciones. A menos que tu caso de uso específico requiera la experiencia Classic, se recomienda implementar Universal Login.

  Para ver una comparación de funcionalidades entre las experiencias de inicio de sesión, consulta [Universal Login vs. Classic Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login)
</Warning>
