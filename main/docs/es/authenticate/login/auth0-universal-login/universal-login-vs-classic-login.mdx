---
title: Universal Login vs. Classic Login
description: Compara las funciones disponibles en las experiencias de Universal Login y Classic Login
---

[Auth0 Universal Login](/docs/es/authenticate/login/auth0-universal-login) proporciona una función esencial de un <Tooltip tip="Servidor de autorización: Servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/docs/es/glossary?term=authorization+server">servidor de autorización</Tooltip>: el flujo de inicio de sesión. Cuando un usuario necesita demostrar su identidad para acceder a su aplicación, puede redirigirlo a <Tooltip tip="Servidor de autorización: Servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip> y dejar que Auth0 se encargue del proceso de autenticación.

Universal Login es la solución principal de inicio de sesión alojado de Auth0. [Universal Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) ofrece herramientas de personalización fáciles de usar y una experiencia más simple y rápida para los usuarios finales. Como alternativa, Auth0 también admite [Classic Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/classic-experience), una experiencia de inicio de sesión alojado que utiliza controles de JavaScript para la personalización.

<Warning>
  Actualmente, Auth0 centra sus esfuerzos de desarrollo en Universal Login, y Classic Login ya no recibe actualizaciones. A menos que su caso de uso específico requiera la experiencia Classic, se recomienda implementar Universal Login.
</Warning>

Para ayudarle a elegir la mejor solución para sus necesidades, la siguiente tabla compara las funciones de ambas experiencias de inicio de sesión.

| **Solución No-Code**                                                                    | Sí, mediante el [no-code editor](/docs/es/customize/login-pages/universal-login/customize-themes)                                                                                     | No                                                                                                                                                                               |
| **Autenticación passwordless**                                                          | Sí                                                                                                                                                                                 | Sí, con [SMS o correo electrónico](/docs/es/authenticate/login/auth0-universal-login/passwordless-login)                                                                            |
| **Magic Link por correo electrónico**                                                   | No                                                                                                                                                                                 | [Sí](/docs/es/authenticate/passwordless/authentication-methods/email-magic-link)                                                                                                    |
| **Claves de acceso**                                                                    | [Sí](/docs/es/authenticate/database-connections/passkeys)                                                                                                                             | No                                                                                                                                                                               |

| **Cumplimiento de las Pautas de Accesibilidad al Contenido en la Web (WCAG)**           | Sí                                                                                                                                                                                 | No                                                                                                                                                                               |
| **Compatibilidad con Organizations**                                                    | [Sí](/docs/es/manage-users/organizations)                                                                                                                                             | No                                                                                                                                                                               |
| **Aceptación de Términos de servicio durante el registro**                              | Sí                                                                                                                                                                                 | [Sí](/docs/es/libraries/lock/lock-configuration#showterms-boolean-)                                                                                                                 |

| **Localización completa**                                                               | Sí, en [estos](/docs/es/customize/internationalization-and-localization/universal-login-internationalization) idiomas                                                                 | Solo para la página de login                                                                                                                                                     |
| **Texto localizado personalizable en todas las páginas**                                | [Sí](/docs/es/customize/login-pages/universal-login/customize-text-elements)                                                                                                          | No                                                                                                                                                                               |
| **MFA por correo electrónico**                                                          | [Sí](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-email-notifications-for-mfa)                                                           | No                                                                                                                                                                               |
| **MFA por voz**                                                                         | [Sí](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-sms-voice-notifications-mfa)                                                           | No                                                                                                                                                                               |
| **MFA con Duo**                                                                         | Sí, pero Duo debe ser el único factor de MFA habilitado                                                                                                                            | Sí                                                                                                                                                                               |
| **Personalización de MFA con Actions**                                                  | [Sí](/docs/es/secure/multi-factor-authentication/customize-mfa/customize-mfa-selection-universal-login)                                                                               | No                                                                                                                                                                               |
| **Botones para conexiones empresariales**                                               | [Sí](/docs/es/authenticate/login/auth0-universal-login/identifier-first#define-home-realm-discovery-identity-providers)                                                               | No                                                                                                                                                                               |
| **Posibilidad de desactivar la recuperación de contraseña de autoservicio**             | Sí                                                                                                                                                                                 | [Sí](/docs/es/libraries/lock/lock-configuration#allowforgotpassword-boolean)                                                                                                        |
