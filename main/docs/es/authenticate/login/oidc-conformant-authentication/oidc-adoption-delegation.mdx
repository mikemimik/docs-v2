---
description: Aprende cómo el flujo compatible con OIDC afecta al uso que haces de la delegación.
title: Delegación con OIDC
---

<Warning>
  De forma predeterminada, la delegación está deshabilitada para los tenants que no tenían ningún add-on en uso a fecha del 8 de junio de 2017. Los tenants Legacy que actualmente usan un add-on que requiere delegación pueden seguir usando esta funcionalidad. Si en algún momento la funcionalidad de delegación cambia o se elimina del servicio, se notificará con antelación a los clientes que la usan actualmente y se les dará tiempo suficiente para migrar.
</Warning>

Tradicionalmente, la delegación se usa para:

* Intercambiar un <Tooltip tip="token de ID: Credencial destinada al propio cliente, en lugar de para acceder a un recurso." cta="Ver glosario" href="/docs/es/glossary?term=ID+token">token de identidad</Tooltip> emitido para una aplicación por otro nuevo emitido para una aplicación diferente.
* Obtener un nuevo token de identidad usando un [token de actualización](/docs/es/secure/tokens/refresh-tokens).
* Intercambiar un token de identidad por un token de API de terceros (p. ej., Firebase, AWS).

Dado que el flujo compatible con OIDC requiere que los tokens de ID ya no se usen para proteger APIs y que los <Tooltip tip="token de actualización: Token usado para obtener un Access Token renovado sin obligar a los usuarios a iniciar sesión de nuevo." cta="Ver glosario" href="/docs/es/glossary?term=refresh+tokens">tokens de actualización</Tooltip> se usen solo en el endpoint [`/oauth/token`](https://auth0.com/docs/api/authentication#refresh-token), el endpoint [`/delegation`](https://auth0.com/docs/api/authentication#delegation) está obsoleto.

Las aplicaciones compatibles con OIDC no pueden ser el origen ni el destino de las solicitudes de delegación.

<div id="third-party-apis">
  ## APIs de terceros
</div>

Como no existe ningún mecanismo compatible con OIDC para obtener tokens para APIs de terceros (por ejemplo, Firebase o AWS), delegación aún puede usarse para obtener tokens para APIs de terceros.
