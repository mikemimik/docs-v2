---
description: Aprende cómo el flujo compatible con OIDC afecta el uso que haces de la delegación.
title: Delegación con OIDC
---

<Warning>
  De forma predeterminada, la delegación está deshabilitada para los tenants que no tenían ningún add-on en uso a fecha del 8 de junio de 2017. Los tenants Legacy que actualmente usan un add-on que requiere delegación pueden seguir usando esta funcionalidad. Si la funcionalidad de delegación se modifica o se elimina del servicio en algún momento, se notificará con antelación a los clientes que la usan actualmente y se les dará tiempo suficiente para migrar.
</Warning>

Tradicionalmente, la delegación se usa para:

* Intercambiar un <Tooltip tip="Token de ID: Credencial destinada al propio cliente, en lugar de para acceder a un recurso." cta="Ver glosario" href="/docs/es/glossary?term=ID+token">token de identidad</Tooltip> emitido para una aplicación por otro nuevo emitido para una aplicación diferente.
* Obtener un nuevo token de identidad usando un [token de actualización](/docs/es/secure/tokens/refresh-tokens).
* Intercambiar un token de identidad por un token de API de terceros (por ejemplo, Firebase, AWS).

Debido a que el flujo compatible con OIDC exige que los tokens de ID ya no se usen para proteger API y que los <Tooltip tip="Token de actualización: Token usado para obtener un access token renovado sin obligar a los usuarios a iniciar sesión de nuevo." cta="Ver glosario" href="/docs/es/glossary?term=refresh+tokens">tokens de actualización</Tooltip> se usen únicamente en el endpoint [`/oauth/token`](https://auth0.com/docs/api/authentication#refresh-token), el endpoint [`/delegation`](https://auth0.com/docs/api/authentication#delegation) está obsoleto.

Las aplicaciones compatibles con OIDC no pueden ser el origen ni el destino de las solicitudes de delegación.

<div id="third-party-apis">
  ## APIs de terceros
</div>

Como no existe ningún mecanismo compatible con OIDC para obtener tokens de APIs de terceros (por ejemplo, Firebase, AWS), delegación todavía puede utilizarse para obtener tokens de APIs de terceros.
