title: Añadir nombre de usuario a Database Connections
---

En las conexiones de base de datos, puedes hacer que los usuarios inicien sesión con un nombre de usuario en lugar de su dirección de correo electrónico.

<div id="require-username">
  ## Requerir nombre de usuario
</div>

1. Ve a [Auth0 Dashboard &gt; Autenticación &gt; Database](https://manage.auth0.com/#/connections/database) y selecciona el nombre de la conexión que quieres editar.
2. Busca **Requires Username** y usa el toggle para activar o desactivar la obligación de usar un nombre de usuario.

   <Frame>![Activar o desactivar la configuración para requerir nombre de usuario](/docs/images/cdy7uua7fh8z/1MlOrFmcSCOCRYKRxWz6xT/7a87dac16a31cbb6e608489c5843d6c7/2025-02-25_12-02-23.png)</Frame>

Para ver cómo afectará esto a la pantalla de login, selecciona la vista **Try Connection**. Ten en cuenta que, una vez activado **Requires Username**, los usuarios nuevos deberán introducir un nombre de usuario y su dirección de correo electrónico para registrarse.

<Frame>![Formulario de login con nombre de usuario](/docs/images/cdy7uua7fh8z/AiB37UZqyBPb8h55PZtzm/73e6fd0033f63f865c40f189db1620ef/2025-01-29_15-22-11.png)</Frame>

Después, los usuarios podrán iniciar sesión con su nombre de usuario o su dirección de correo electrónico. Los usuarios que se registraron mientras **Requires Username** estaba desactivado no tendrán almacenado un campo **Username** y tendrán que iniciar sesión con su dirección de correo electrónico.

<div id="username-length">
  ### Longitud del nombre de usuario
</div>

De forma predeterminada, la longitud permitida para los nombres de usuario es de entre 1 y 15 caracteres. Sin embargo, con el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o a través de la <Tooltip tip="Management API: un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> v2, puedes modificar la longitud mínima y máxima (hasta un máximo de 128 caracteres).

<div id="allowed-characters">
  ### Caracteres permitidos
</div>

El campo de nombre de usuario acepta los siguientes caracteres:

* Caracteres alfanuméricos (sin acentos, convertidos automáticamente a minúsculas);
* El carácter arroba (@) (pero no se permiten direcciones de correo electrónico);
* El carácter de acento circunflejo (^);
* El carácter de signo de dólar ($);
* El carácter de punto (.);
* El carácter de exclamación (!);
* El carácter de acento grave (&#96;);
* El carácter de guion (-);
* El carácter de almohadilla (#);
* El carácter de signo más (+);
* El carácter de comilla simple (&#39;);
* El carácter de tilde (~);
* El carácter de guion bajo (&#95;);

No se permite ningún otro carácter o símbolo, y Auth0 no valida ni sanea las entradas de la base de datos personalizada.
