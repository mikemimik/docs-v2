title: Usar Auth0 App for Sumo Logic
---

Puedes usar la **Auth0 App for Sumo Logic** para visualizar los datos de tu tenant de Auth0 en un Dashboard de Sumo Logic. El Dashboard te permite supervisar el estado del tráfico de inicio de sesión de un tenant. La app te permite usar las agregaciones recomendadas por Auth0 o utilizarlas como punto de partida para crear tus propias visualizaciones personalizadas.

<div id="prerequisite">
  ## Requisito previo
</div>

[Configura la transmisión de logs para Sumo Logic](https://marketplace.auth0.com/integrations/sumo-logic-log-streaming).

<div id="install-the-auth0-app-for-sumo-logic">
  ## Instalar la Auth0 App for Sumo Logic
</div>

1. Inicie sesión en [Sumo Logic](https://www.sumologic.com).
2. En el **Catálogo de aplicaciones**, busque y seleccione la app de Auth0.
3. Para instalar la app, complete los siguientes campos:

   | Campo                       | Descripción                                                                                                                                                                                                                                            |
   | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   | **Nombre de la aplicación** | Mantenga el nombre actual o escriba el nombre que prefiera.                                                                                                                                                                                            |
   | **Fuente de datos**         | Elija **Categoría de origen** y seleccione una categoría de la lista, o elija **Introducir un filtro de Custom Data** e introduzca una categoría de origen personalizada que comience con un guion bajo. Por ejemplo: ***sourceCategory=MyCategory***. |
   | **Advanced**                | Seleccione la **Ubicación en la biblioteca** (de forma predeterminada, la carpeta Personal) o haga clic en **Nueva carpeta** para añadir una carpeta nueva.                                                                                            |
4. Seleccione **Añadir a la biblioteca**.

Una vez instalada una app, aparecerá en su carpeta **Personal** o en la carpeta que haya seleccionado. Desde allí, puede compartirla con su organización.

Los paneles se irán completando automáticamente con el tiempo. Se completan lentamente con datos que coinciden con la consulta del intervalo de tiempo y que se han recibido desde que se creó el panel.

<Frame>![Dashboard de Sumo Logic con datos de streaming de Auth0](/docs/images/cdy7uua7fh8z/2BynHH8t2c8axVoRK0h1Et/715854ee4e6774c64973ef1ecd7e8390/sumo-logic-dashboard.png)</Frame>

El Dashboard se divide en las siguientes categorías o paneles:

* **Métricas de inicio de sesión**: Consulte el número de acciones, como inicios de sesión exitosos e inicios de sesión fallidos.
* **Métricas de creación de cuentas**: Consulte el desglose de la creación de cuentas por país, plataforma, IP y otros datos demográficos.
* **Métricas de seguridad**: Consulte los eventos de detección de anomalías a lo largo del tiempo para detectar ataques.

<div id="dashboard-filters">
  ## Filtros del Dashboard
</div>

Cada Dashboard tiene un conjunto de filtros que puedes aplicar a todo el Dashboard. Haz clic en el icono de embudo en la barra de menú superior del Dashboard para ver una lista desplazable de filtros que se aplican a todo el Dashboard.

Cada panel también tiene un conjunto de filtros que se aplican solo a los resultados de ese panel. Haz clic en el icono de embudo en la barra de menú superior del panel para ver una lista de filtros específicos del panel.

<div id="customize-dashboard">
  ## Personaliza el Dashboard
</div>

Puedes personalizar tu Dashboard de seguridad de Auth0 en Sumo Logic para añadir widgets de datos personalizados.

1. Ve a tu Dashboard de seguridad de Auth0 y selecciona **add panel** en la esquina superior derecha.
2. Selecciona un tipo de panel haciendo clic en el icono. Consulta [la ayuda de Sumo Logic](https://help.sumologic.com/Visualizations-and-Alerts/Dashboard_\(New\)/Panels) para obtener más información.
3. Introduce una consulta de logs, una consulta de métricas o varias consultas desde el menú desplegable.
4. Para cambiar el tipo de gráfico, consulta [la ayuda de Sumo Logic](https://help.sumologic.com/Visualizations-and-Alerts/Dashboard_\(New\)/Panels/01Modify_a_Chart) para ver las opciones disponibles.
5. Haz clic en **Add to Dashboard** en la esquina superior derecha para añadir el panel a tu Dashboard.
