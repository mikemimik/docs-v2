title: Usar Auth0 App for Sumo Logic
---

Puedes usar **Auth0 App for Sumo Logic** para visualizar los datos de tu tenant de Auth0 en un Dashboard de Sumo Logic. El Dashboard te permite supervisar el estado del tráfico de inicio de sesión de un tenant. La aplicación te permite usar las agregaciones recomendadas por Auth0 o tomarlas como punto de partida para crear tus propias visualizaciones personalizadas.

<div id="prerequisite">
  ## Requisito previo
</div>

[Configura la transmisión de logs a Sumo Logic](https://marketplace.auth0.com/integrations/sumo-logic-log-streaming).

<div id="install-the-auth0-app-for-sumo-logic">
  ## Instala Auth0 App for Sumo Logic
</div>

1. Inicia sesión en [Sumo Logic](https://www.sumologic.com).
2. En el **Catálogo de aplicaciones**, busca y selecciona la app de Auth0.
3. Para instalar la app, completa los siguientes campos:

   | Campo                       | Descripción                                                                                                                                                                                                                                           |
   | --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Nombre de la aplicación** | Conserva el nombre actual o introduce el nombre que prefieras.                                                                                                                                                                                        |
   | **Fuente de datos**         | Elige **Categoría de origen** y selecciona una categoría de la lista, o elige **Introducir un filtro de Custom Data** e introduce una categoría de origen personalizada que comience con un guion bajo. Por ejemplo: ***sourceCategory=MyCategory***. |
   | **Advanced**                | Selecciona la **Ubicación en la biblioteca** (la opción predeterminada es la carpeta Personal) o haz clic en **Nueva carpeta** para añadir una carpeta nueva.                                                                                         |
4. Selecciona **Añadir a la biblioteca**.

Una vez instalada, la app aparece en tu carpeta **Personal** o en la carpeta que hayas seleccionado. Desde ahí, puedes compartirla con tu organización.

Los paneles se irán completando automáticamente con el tiempo. Se llenan poco a poco con datos que coinciden con la consulta del intervalo de tiempo y que se reciben desde que se creó el panel.

<Frame>![Dashboard de Sumo Logic con datos de streaming de Auth0](/docs/images/cdy7uua7fh8z/2BynHH8t2c8axVoRK0h1Et/715854ee4e6774c64973ef1ecd7e8390/sumo-logic-dashboard.png)</Frame>

El Dashboard se divide en las siguientes categorías o paneles:

* **Métricas de inicio de sesión**: Consulta el recuento de acciones, como inicios de sesión correctos e inicios de sesión fallidos.
* **Métricas de creación de cuentas**: Consulta el desglose de la creación de cuentas por país, plataforma, IP y otros datos demográficos.
* **Métricas de seguridad**: Consulta los eventos de detección de anomalías a lo largo del tiempo para detectar ataques.

<div id="dashboard-filters">
  ## Filtros del Dashboard
</div>

Cada Dashboard cuenta con un conjunto de filtros que puedes aplicar a todo el Dashboard. Haz clic en el ícono de embudo en la barra de menú superior del Dashboard para ver una lista desplazable de filtros que se aplican a todo el Dashboard.

Cada panel también tiene un conjunto de filtros que se aplican solo a los resultados de ese panel. Haz clic en el ícono de embudo en la barra de menú superior del panel para ver una lista de filtros específicos del panel.

<div id="customize-dashboard">
  ## Personalizar el Dashboard
</div>

Puedes personalizar tu dashboard de seguridad de Sumo Logic para Auth0 y agregar widgets de datos personalizados.

1. Ve a tu dashboard de seguridad de Auth0 y selecciona **add panel** en la esquina superior derecha.
2. Selecciona un tipo de panel haciendo clic en el icono. Consulta la [ayuda de Sumo Logic](https://help.sumologic.com/Visualizations-and-Alerts/Dashboard_\(New\)/Panels) para obtener más información.
3. Introduce una consulta de logs, una consulta de métricas o varias consultas desde el menú desplegable.
4. Para cambiar el tipo de gráfico, consulta la [ayuda de Sumo Logic](https://help.sumologic.com/Visualizations-and-Alerts/Dashboard_\(New\)/Panels/01Modify_a_Chart) para ver las opciones disponibles.
5. Haz clic en **Add to Dashboard** en la esquina superior derecha para añadir el panel a tu Dashboard.
