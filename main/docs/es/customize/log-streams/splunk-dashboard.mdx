Puedes usar **Auth0 App for Splunk** para visualizar los datos de tu tenant de Auth0 en un Dashboard de Splunk. El Dashboard te permite supervisar el estado del tráfico de inicio de sesión de un tenant. La aplicación te permite usar las agregaciones recomendadas de Auth0 o tomarlas como punto de partida para crear tus propias visualizaciones personalizadas.

<div id="install-the-auth0-app-for-splunk">
  ## Instala Auth0 App for Splunk
</div>

1. Inicia sesión en [Splunk](https://www.splunk.com/).
2. Ve a la página de la aplicación [Auth0 App for Splunk](https://splunkbase.splunk.com/app/5193).
3. Selecciona el botón verde **Install**.
4. Introduce tu nombre de usuario y contraseña, acepta los términos y condiciones y selecciona **Login and Install**.
5. En el widget **Complete** que aparece, selecciona el botón **Open the App**.
   Aparecerá el Auth0 Dashboard con los filtros predeterminados aplicados.

<div id="dashboard-filters">
  ## Filtros del Dashboard
</div>

Los siguientes filtros te permiten profundizar en los detalles específicos de tu tráfico. Puedes ingresar `*` para buscar entre todos los valores de ese campo.

| Filtro            | Descripción                                                                                                                                                                                                        |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Rango de tiempo   | Un selector de tiempo de Splunk que te permite elegir el período durante el cual ver los eventos.                                                                                                                  |
| Índice            | El `index` de Splunk en el que quieres buscar. Al crear tu `Http Event Collector Token`, si especificaste un índice en particular, puedes seleccionar este valor en el menú desplegable.                           |
| Origen HTTP       | El nombre de `source` de Splunk.                                                                                                                                                                                   |
| IP                | La dirección IP cuyo tráfico quieres inspeccionar. Esto corresponde a `data.ip` en el evento de log.                                                                                                               |
| Cliente           | El cliente cuyo tráfico quieres inspeccionar. Esto corresponde a `data.client_name` en el evento de log.                                                                                                           |
| País              | El país cuyo tráfico quieres inspeccionar. Este es un campo que se obtiene mediante el comando de búsqueda `iplocation data.ip` en la consulta. Como resultado, se agrega un campo `Country` a cada evento de log. |
| Nombre de usuario | La dirección de correo electrónico cuyo tráfico de inicio de sesión quieres inspeccionar. Esto corresponde a `data.user_name` en el evento de log.                                                                 |

<div id="customize-dashboard">
  ## Personaliza el Dashboard
</div>

Puedes personalizar tu dashboard de seguridad de Splunk Auth0 para añadir widgets de datos personalizados.

1. Ve a tu dashboard de seguridad de Auth0 y selecciona **edit** en la esquina superior derecha.
2. En el panel Edit Dashboard, selecciona **add panel** y luego elige un tipo de contenido, por ejemplo, gráfico de líneas, evento o gráfico de áreas.
3. Introduce el intervalo de tiempo, el título del contenido y la cadena de búsqueda para la visualización de datos.
4. Selecciona **Add to dashboard** para añadir tu nuevo widget de datos.
