title: Usa plantillas de dashboard de Auth0 con Datadog
---

Auth0 proporciona plantillas predefinidas que puedes usar para visualizar los datos de tu tenant de Auth0 en Datadog. Estas plantillas te permiten mostrar visualizaciones para supervisar el estado del tráfico de inicio de sesión de un tenant. También te permiten usar las agregaciones recomendadas por Auth0 o tomarlas como punto de partida para crear tus propias visualizaciones personalizadas.

| Plantilla          | Purpose                                                                                                                                        | Archivo descargable                                                                                           |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Security Dashboard | Supervisa el tráfico de inicio de sesión del tenant, incluidas las métricas de inicio de sesión, creación de cuentas y detección de anomalías. | [Auth0SecDashboard.json](https://cdn.auth0.com/website/docs/logs/streams/datadog/Auth0SecurityDashboard.json) |

También puedes organizar tu visualización con las siguientes opciones:

* Usa la selección de tiempo de Datadog para elegir el intervalo de tu visualización.
* Usa filtros para acotar el análisis de datos mediante parámetros configurables, como dirección IP, país, usuario y aplicación cliente.
* Usa **Edit Widgets** para reorganizar los mosaicos y las visualizaciones, así como para crear copias en las que puedas inspeccionar y añadir parámetros a las consultas según sea necesario.

<div id="prerequisites">
  ## Requisitos previos
</div>

* [Configurar la transmisión de logs a Datadog](https://marketplace.auth0.com/integrations/datadog-log-streaming).
* Descargar desde Auth0 el archivo JSON de la plantilla de dashboard deseada.

<div id="upload-dashboard-templates-in-datadog">
  ## Carga plantillas de dashboard en Datadog
</div>

1. Inicia sesión en [Datadog](https://www.datadoghq.com/).
2. Ve a **Dashboards** en la barra de navegación de la izquierda.
3. Selecciona **Screenboard** como tipo de dashboard.
4. En el lienzo en blanco, en la esquina superior derecha, selecciona el icono de engranaje y haz clic en **Import Dashboard JSON**.
5. Carga el archivo JSON del dashboard que quieras usar (por ejemplo, `Auth0SecDashboard.json`).

<div id="security-dashboard-visualizations">
  ## Visualizaciones de Security Dashboard
</div>

Una vez que hayas importado Security Dashboard de Auth0, puedes usar las siguientes visualizaciones para comprobar el estado de tu tenant o como punto de partida para crear tus propias visualizaciones y alertas en Datadog.

| Visualización                                            | Propósito                                                                                                                                                                                             |
| -------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Inicios de sesión exitosos                               | Muestra el número de eventos de inicio de sesión exitosos del tenant.                                                                                                                                 |
| Fallos de inicio de sesión                               | Muestra el número de eventos de inicio de sesión fallidos del tenant. Los picos en los fallos de inicio de sesión pueden ser una señal de tráfico anómalo.                                            |
| Fallos de inicio de sesión por geolocalización           | Muestra el número de eventos de inicio de sesión fallidos del tenant por geolocalización. Los picos de tráfico inesperados en determinadas zonas geográficas pueden ser una señal de tráfico anómalo. |
| Número de IP únicas que envían tráfico                   | Muestra el número de direcciones IP que envían tráfico al tenant. Este número suele seguir patrones predecibles, y las variaciones importantes pueden indicar tráfico anómalo.                        |
| 5 IP principales con fallos                              | Muestra las 5 direcciones IP con el mayor número de errores de inicio de sesión.                                                                                                                      |
| Eventos de creación de cuentas en los principales países | Muestra el número de cuentas creadas en los principales países. Las variaciones en los patrones y las ubicaciones geográficas habituales pueden indicar tráfico anómalo.                              |
| Eventos de creación de cuentas por plataforma            | Muestra el número de cuentas creadas por plataforma, como Windows, OSX, Android o iOS.                                                                                                                |
| IP con más cuentas creadas                               | Muestra las direcciones IP desde las que se ha creado el mayor número de cuentas.                                                                                                                     |
| Eventos de detección de anomalías                        | Muestra el número de eventos de detección de anomalías del tenant, incluido el número de direcciones IP bloqueadas debido a Brute Force Protection y detección de contraseñas vulnerada.              |
| Log Stream                                               | Muestra eventos sin procesar del log del tenant.                                                                                                                                                      |

<div id="customize-dashboard">
  ## Personaliza el dashboard
</div>

Puedes personalizar tu dashboard de seguridad de Auth0 en Datadog para agregar widgets con datos personalizados.

1. Ve a tu dashboard de seguridad de Auth0 y, en la navegación lateral, haz clic en **Dashboards**.
2. Selecciona en la lista el nombre del dashboard que creaste.
3. Selecciona una visualización y arrástrala a la sección principal de la página.
4. Configura las preferencias de tu gráfico de datos.
