title: Usa plantillas de dashboard de Auth0 con Datadog
---

Auth0 proporciona plantillas predefinidas que puedes usar para visualizar los datos de tu tenant de Auth0 en Datadog. Puedes mostrar visualizaciones para supervisar el estado del tráfico de inicio de sesión de un tenant. Las plantillas te permiten usar las agregaciones recomendadas por Auth0 o tomarlas como punto de partida para crear tus propias visualizaciones personalizadas.

| Plantilla          | Purpose                                                                                                                                        | Archivo descargable                                                                                           |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Security Dashboard | Supervisa el tráfico de inicio de sesión del tenant, incluidas las métricas de inicio de sesión, creación de cuentas y detección de anomalías. | [Auth0SecDashboard.json](https://cdn.auth0.com/website/docs/logs/streams/datadog/Auth0SecurityDashboard.json) |

También puedes organizar tu visualización con las siguientes funciones:

* Usa la selección de tiempo de Datadog para elegir el intervalo de tu visualización.
* Usa filtros para acotar el análisis de datos mediante parámetros configurables como dirección IP, país, usuario y aplicación cliente.
* Usa **Edit Widgets** para reorganizar los mosaicos y las visualizaciones, así como para crear copias en las que puedas inspeccionar y añadir parámetros a las consultas según sea necesario.

<div id="prerequisites">
  ## Requisitos previos
</div>

* [Configure la transmisión de logs a Datadog](https://marketplace.auth0.com/integrations/datadog-log-streaming).
* Descargue de Auth0 el archivo JSON de la plantilla de dashboard que desee.

<div id="upload-dashboard-templates-in-datadog">
  ## Sube plantillas de dashboard a Datadog
</div>

1. Inicia sesión en [Datadog](https://www.datadoghq.com/).
2. Ve a **Dashboards** en la barra de navegación de la izquierda.
3. Selecciona **Screenboard** como tipo de dashboard.
4. En el lienzo en blanco, en la esquina superior derecha, selecciona el icono de engranaje y haz clic en **Import Dashboard JSON**.
5. Sube el archivo JSON del dashboard que quieras usar (por ejemplo, `Auth0SecDashboard.json`).

<div id="security-dashboard-visualizations">
  ## Visualizaciones de Security Dashboard
</div>

Una vez que hayas importado Security Dashboard de Auth0, puedes usar las siguientes visualizaciones para determinar el estado de tu tenant o como punto de partida para crear tus propias visualizaciones y alertas en Datadog.

| Visualización                                            | Purpose                                                                                                                                                                                                       |
| -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Inicio de sesión correcto                                | Muestra la cantidad de eventos de inicio de sesión correctos del tenant.                                                                                                                                      |
| Inicios de sesión fallidos                               | Muestra la cantidad de eventos de inicio de sesión fallidos del tenant. Los picos de inicios de sesión fallidos pueden ser una señal de tráfico anómalo.                                                      |
| Inicios de sesión fallidos por geolocalización           | Muestra la cantidad de eventos de inicio de sesión fallidos del tenant por geolocalización. Los picos de tráfico inesperados en determinadas ubicaciones geográficas pueden ser una señal de tráfico anómalo. |
| Número de IP únicas que envían tráfico                   | Muestra la cantidad de direcciones IP que envían tráfico al tenant. Este número suele seguir patrones predecibles, y variaciones importantes pueden indicar tráfico anómalo.                                  |
| Las 5 IP con más fallos                                  | Muestra las 5 direcciones IP con la mayor cantidad de errores de inicio de sesión.                                                                                                                            |
| Eventos de creación de cuentas en los principales países | Muestra la cantidad de cuentas creadas en los principales países. Las variaciones en patrones y ubicaciones geográficas predecibles pueden indicar tráfico anómalo.                                           |
| Eventos de creación de cuentas por plataforma            | Muestra la cantidad de cuentas creadas por plataforma, como Windows, OSX, Android o iOS.                                                                                                                      |
| IP con más cuentas creadas                               | Muestra las direcciones IP desde las que se ha creado la mayor cantidad de cuentas.                                                                                                                           |
| Eventos de detección de anomalías                        | Muestra la cantidad de eventos de detección de anomalías del tenant, incluida la cantidad de direcciones IP bloqueadas debido a Brute Force Protection y a la detección de contraseñas vulnerada.             |
| Log Stream                                               | Muestra eventos sin procesar del registro del tenant.                                                                                                                                                         |

<div id="customize-dashboard">
  ## Personalizar el dashboard
</div>

Puedes personalizar tu Security Dashboard de Auth0 en Datadog para añadir widgets de datos personalizados.

1. Ve a tu Security Dashboard de Auth0 y, en la navegación lateral, haz clic en **Dashboards**.
2. Selecciona en la lista el nombre del dashboard que creaste.
3. Selecciona una visualización y arrástrala a la sección principal de la página.
4. Configura las preferencias del gráfico de datos.
