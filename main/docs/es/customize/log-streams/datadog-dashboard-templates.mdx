title: Usa plantillas del Auth0 Dashboard con Datadog
---

Auth0 proporciona plantillas predefinidas que puedes usar para visualizar los datos de tu tenant de Auth0 en Datadog. Muestra visualizaciones para supervisar el estado del tráfico de inicio de sesión de un tenant. Las plantillas te permiten usar las agregaciones recomendadas por Auth0 o tomarlas como punto de partida para crear tus propias visualizaciones personalizadas.

| Plantilla          | Propósito                                                                                                                                      | Archivo descargable                                                                                           |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Panel de seguridad | Supervisa el tráfico de inicio de sesión del tenant, incluidas las métricas de inicio de sesión, creación de cuentas y detección de anomalías. | [Auth0SecDashboard.json](https://cdn.auth0.com/website/docs/logs/streams/datadog/Auth0SecurityDashboard.json) |

También puedes organizar tu visualización con las siguientes funciones:

* Usa el selector de tiempo de Datadog para elegir el período de tu visualización.
* Usa filtros para acotar el análisis de datos mediante parámetros configurables, como dirección IP, país, usuario y aplicación cliente.
* Usa **Editar widgets** para reorganizar los mosaicos y las visualizaciones, así como crear copias en las que puedas inspeccionar y agregar parámetros a las consultas según sea necesario.

<div id="prerequisites">
  ## Requisitos previos
</div>

* [Configura el streaming de logs a Datadog](https://marketplace.auth0.com/integrations/datadog-log-streaming).
* Descarga de Auth0 el archivo JSON de la plantilla de dashboard que desees.

<div id="upload-dashboard-templates-in-datadog">
  ## Cargar plantillas de dashboard en Datadog
</div>

1. Inicia sesión en [Datadog](https://www.datadoghq.com/).
2. Ve a **Dashboards** en la barra de navegación de la izquierda.
3. Selecciona **Screenboard** como tipo de dashboard.
4. En el lienzo en blanco, en la esquina superior derecha, selecciona el icono de engranaje y haz clic en **Import Dashboard JSON**.
5. Carga el archivo JSON del dashboard que quieras usar (por ejemplo, `Auth0SecDashboard.json`).

<div id="security-dashboard-visualizations">
  ## Visualizaciones del dashboard de seguridad
</div>

Una vez que hayas importado el dashboard de seguridad desde Auth0, puedes usar las siguientes visualizaciones para evaluar el estado de tu tenant o como punto de partida para crear tus propias visualizaciones y alertas en Datadog.

| Visualización                                            | Propósito                                                                                                                                                                                                    |
| -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Inicio de sesión correcto                                | Muestra el número de eventos de inicio de sesión correctos del tenant.                                                                                                                                       |
| Fallos de inicio de sesión                               | Muestra el número de eventos de inicio de sesión fallidos del tenant. Los picos en los fallos de inicio de sesión pueden ser una señal de tráfico anómalo.                                                   |
| Fallos de inicio de sesión por geolocalización           | Muestra el número de eventos de inicio de sesión fallidos del tenant por geolocalización. Los picos de tráfico inesperados en determinadas zonas geográficas pueden ser una señal de tráfico anómalo.        |
| Número de IP únicas que envían tráfico                   | Muestra el número de direcciones IP que envían tráfico al tenant. Este número suele seguir patrones predecibles, y las variaciones importantes pueden indicar tráfico anómalo.                               |
| 5 IP principales con fallos                              | Muestra las 5 direcciones IP con el mayor número de errores de inicio de sesión.                                                                                                                             |
| Eventos de creación de cuentas en los principales países | Muestra el número de cuentas creadas en los principales países. Las variaciones en los patrones previsibles y en las ubicaciones geográficas pueden indicar tráfico anómalo.                                 |
| Eventos de creación de cuentas por plataforma            | Muestra el número de cuentas creadas por plataforma, como Windows, OSX, Android o iOS.                                                                                                                       |
| IP con más cuentas creadas                               | Muestra las direcciones IP desde las que se ha creado el mayor número de cuentas.                                                                                                                            |
| Eventos de detección de anomalías                        | Muestra el número de eventos de detección de anomalías del tenant, incluido el número de direcciones IP bloqueadas debido a la protección contra fuerza bruta y a la detección de contraseñas comprometidas. |
| Flujo de registros                                       | Muestra los eventos en bruto del registro del tenant.                                                                                                                                                        |

<div id="customize-dashboard">
  ## Personaliza el dashboard
</div>

Puedes personalizar tu dashboard de seguridad de Datadog Auth0 para añadir widgets de datos personalizados.

1. Ve a tu dashboard de seguridad de Auth0 y, en el menú lateral, selecciona **Dashboards**.
2. Selecciona en la lista el nombre del dashboard que creaste.
3. Selecciona una visualización y arrástrala a la sección principal de la página.
4. Configura las preferencias del gráfico de datos.
