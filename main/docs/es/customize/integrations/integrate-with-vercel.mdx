La integración nativa de Auth0 con Vercel te permite conectar tu plataforma de identidad a tu proyecto de Vercel con una configuración mínima. Configura y personaliza tu pantalla de inicio de sesión sin tener que crearla desde cero. Luego, amplía y escala tus flujos de autenticación y autorización con funcionalidades avanzadas, como [Organizations B2B](/docs/es/manage-users/organizations), [autorización detallada (FGA)](https://docs.fga.dev/), [Auth0 for AI Agents](https://auth0.com/ai/docs) y más.

La integración nativa de Auth0 con Vercel permite:

* El aprovisionamiento automático de una aplicación de Auth0 que se conecta a tu proyecto de Vercel.
* La precarga de tus credenciales de Auth0 en el Dashboard de Vercel para una configuración rápida y sencilla.
* Compatibilidad lista para usar con tus aplicaciones Next.js mediante los [Auth0 SDKs](/docs/es/libraries).

La integración nativa de Auth0 con Vercel crea un nuevo entorno de tenant de Auth0 para tu proyecto de Vercel. Si quieres usar una cuenta de Auth0 existente, intégrala con Vercel manualmente al crear una aplicación en el Auth0 Dashboard.

<div id="prerequisites">
  ## Requisitos previos
</div>

Antes de comenzar, asegúrate de tener lo siguiente:

* Una cuenta de Vercel con un proyecto activo. Si es necesario, [crea una cuenta de Vercel](https://vercel.com/signup)
* Una aplicación de Next.js que use la última versión del [SDK de Auth0 para Next.js](/docs/es/quickstart/webapp/nextjs)

Después de instalar la integración, [habilita la incrustación en iframe](/docs/es/customize/login-pages/manage-core-configuration#iframe-embedding) para permitir que Universal Login o Classic Login se carguen en un iframe, como requiere Vercel.

<div id="install-integration">
  ## Instalar integración
</div>

Instala la integración nativa de Auth0 con Vercel desde Vercel Marketplace:

1. Ve a **Integrations &gt; Browse Marketplace** y busca Auth0. Selecciona la **integración de Auth0** en **Native Integrations** y, a continuación, selecciona **Install**.

2. Auth0 completa automáticamente la URL de localhost y la URL de devolución de llamada de tu aplicación. Selecciona tu **Installation Plan** y, a continuación, selecciona **Continuar**.

3. Ingresa un nombre para tu aplicación. Selecciona **Create**. Una vez que el nuevo tenant de Auth0 y la aplicación se hayan creado correctamente, selecciona **Continuar**.

4. Para conectarte a un proyecto de Vercel existente, selecciona **Connect Project** y, a continuación, selecciona tu proyecto y los entornos. Ingresa un prefijo personalizado para agregarlo a tus variables de entorno, si es necesario. Selecciona **Connect**. Una vez que hayas conectado correctamente un proyecto de Vercel, se te redirigirá a la sección [Primeros pasos](#getting-started) de la integración.

<div id="getting-started">
  ## Primeros pasos
</div>

La sección Primeros pasos del Dashboard de Vercel muestra una Quickstart para poner tu aplicación en marcha de forma rápida y segura. También puedes administrar la configuración de tu aplicación en la navegación de la izquierda.

<div id="quickstart">
  ### Quickstart
</div>

Para empezar a usar la integración nativa de Vercel de Auth0, sigue las instrucciones del Dashboard de Vercel para crear y desplegar tu aplicación.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Solo puedes configurar las variables de entorno de Auth0 en un entorno de producción.
</Callout>

<div id="manage-application-settings">
  ## Administrar la configuración de la aplicación
</div>

Puedes administrar la configuración de tu aplicación en el [Dashboard de Vercel](#vercel-dashboard) y en el [Auth0 Dashboard](#auth0-dashboard).

<div id="vercel-dashboard">
  ### Dashboard de Vercel
</div>

Para administrar la configuración de tu aplicación en el Dashboard de Vercel, ve a **Settings** en el menú de navegación de la izquierda.

Puedes actualizar la siguiente configuración de la aplicación:

* **Actualizar nombre**: Actualiza el nombre de la aplicación.
* **Actualizar configuración**:
  * **Parámetros**: Actualiza la URL de localhost y el endpoint de devolución de llamada de tu aplicación.
  * **Plan actual del nivel de instalación**: Ve a la configuración de la integración para actualizar el plan del nivel de instalación de tu integración. Para obtener más información, consulta [Settings](#settings).
* **Proteger este recurso**
  * **Rotar secretos**: Rota los secretos de tu aplicación. Para obtener más información, consulta [Rotating environment variables](https://vercel.com/docs/projects/environment-variables/managing-environment-variables#rotating-environment-variables).
  * **Entornos permitidos**: Selecciona si la integración está disponible en **Todos los entornos** o solo en el **entorno de producción**.
* **Eliminar integración**: Cuando eliminas la integración, se te cambia automáticamente al plan Free de Vercel. Para obtener más información, consulta [Eliminar integración](#delete-integration).

<div id="auth0-dashboard">
  ### Auth0 Dashboard
</div>

Administra ajustes adicionales de la aplicación en el [Auth0 Dashboard](https://manage.auth0.com/).

Cuando instalas la integración nativa, Auth0 crea una nueva aplicación en un tenant dedicado, preconfigurada con tus URI de Vercel, callbacks permitidos y URL de cierre de sesión.

Puedes modificar esta aplicación, por ejemplo, personalizando tu experiencia de Universal Login, como lo harías con cualquier otra aplicación en tu Dashboard de Auth0.

<div id="manage-integration">
  ## Gestionar la integración
</div>

Para gestionar la integración desde tu Dashboard de Vercel, selecciona la pestaña **Integration** de tu proyecto, desplázate hasta la integración de Auth0 y selecciona **Manage**.

<div id="invoices">
  ### Facturas
</div>

En la página de **Facturas**, puedes ver una lista de las facturas relacionadas con tu integración con Auth0. Para obtener más información sobre las facturas, consulta [Billing &amp; Invoices](https://vercel.com/docs/pricing/understanding-my-invoice).

<div id="settings">
  ### Configuración
</div>

En la página de **Configuración**, agrega métodos de pago, cambia el plan del nivel de instalación y elimina la integración.

<div id="payment-methods">
  #### Métodos de pago
</div>

Para agregar un método de pago, selecciona **Agregar**. Para obtener más información sobre los métodos de pago, consulta [los precios de Vercel](https://vercel.com/docs/accounts/plans).

<div id="current-installation-level-plan">
  #### Plan actual del nivel de instalación
</div>

El plan del nivel de instalación se aplica a todos los productos de la instalación.

Para cambiar el plan del nivel de instalación:

1. En **Configuración**, desplázate hasta **Plan actual del nivel de instalación** y selecciona **Cambiar plan**.
2. Selecciona el plan del nivel de instalación que quieras.
3. Selecciona **Continuar**.
4. Revisa tu selección y luego selecciona **Actualizar**.
5. Una vez actualizada la instalación, selecciona **Listo**.

<div id="delete-integration">
  #### Eliminar integración
</div>

Cuando elimines la integración, pasarás automáticamente al plan Free de Vercel.

Para eliminar la integración:

1. En **Configuración**, desplázate hasta **Eliminar &lt;YOUR&#95;PRODUCT&gt;** y selecciona el botón **Quitar &lt;YOUR&#95;PRODUCT&gt;**.
2. Introduce el slug de la integración y tu Team; después, selecciona **Reconozco que esto quitará mi cuenta conectada a Auth0**.
3. Selecciona **Quitar &lt;YOUR&#95;PRODUCT&gt;**.
