La integración nativa de Auth0 con Vercel te permite conectar tu plataforma de identidad a tu proyecto de Vercel con una configuración mínima. Configura y personaliza tu pantalla de inicio de sesión sin tener que crearla desde cero. Después, amplía y escala tus flujos de autenticación y autorización con funcionalidades avanzadas, como [B2B Organizations](/docs/es/manage-users/organizations), [Autorización detallada (FGA)](https://docs.fga.dev/), [Auth0 for AI Agents](https://auth0.com/ai/docs) y más.

La integración nativa de Auth0 con Vercel permite:

* Aprovisionamiento automático de una aplicación de Auth0 que se conecta a tu proyecto de Vercel.
* Precarga de tus credenciales de Auth0 en el Dashboard de Vercel para una configuración rápida y sencilla.
* Compatibilidad inmediata con tus aplicaciones Next.js mediante los [SDKs de Auth0](/docs/es/libraries).

La integración nativa de Auth0 con Vercel crea un nuevo tenant de Auth0 para tu proyecto de Vercel. Si quieres usar una cuenta de Auth0 existente, intégrala con Vercel manualmente al crear una aplicación en el Auth0 Dashboard.

<div id="prerequisites">
  ## Requisitos previos
</div>

Antes de empezar, asegúrate de tener lo siguiente:

* Una cuenta de Vercel con un proyecto activo. Si es necesario, [crea una cuenta de Vercel](https://vercel.com/signup)
* Una aplicación de Next.js que utilice la versión más reciente del [SDK de Auth0 para Next.js](/docs/es/quickstart/webapp/nextjs)

Después de instalar la integración, [habilita la incrustación en iframe](/docs/es/customize/login-pages/manage-core-configuration#iframe-embedding) para permitir que Universal Login o Classic Login se carguen dentro de un iframe, como lo requiere Vercel.

<div id="install-integration">
  ## Instalar la integración
</div>

Instala la integración nativa de Auth0 con Vercel desde el Marketplace de Vercel:

1. Ve a **Integrations &gt; Browse Marketplace** y busca Auth0. Selecciona la **integración de Auth0** en **Native Integrations** y, a continuación, selecciona **Install**.

2. Auth0 rellena automáticamente la URL de localhost y la URL de devolución de llamada de tu aplicación. Selecciona tu **Installation Plan** y, a continuación, selecciona **Continuar**.

3. Ingresa un nombre para tu aplicación. Selecciona **Create**. Una vez que el nuevo tenant de Auth0 y la aplicación se hayan creado correctamente, selecciona **Continuar**.

4. Para conectarte a un proyecto de Vercel existente, selecciona **Connect Project** y, a continuación, selecciona tu proyecto y tus entornos. Si es necesario, ingresa un prefijo personalizado que se añadirá a tus variables de entorno. Selecciona **Connect**. Una vez que hayas conectado correctamente un proyecto de Vercel, se te llevará a la sección [Primeros pasos](#getting-started) de la integración.

<div id="getting-started">
  ## Primeros pasos
</div>

La sección Primeros pasos del Dashboard de Vercel incluye una Quickstart para poner tu aplicación en marcha de forma rápida y segura. También puedes administrar la configuración de tu aplicación en el menú de navegación de la izquierda.

<div id="quickstart">
  ### Quickstart
</div>

Para empezar a usar la integración nativa de Auth0 con Vercel, sigue las instrucciones en el Dashboard de Vercel para crear y desplegar tu aplicación.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Solo puedes configurar las variables de entorno de Auth0 en un entorno Production.
</Callout>

<div id="manage-application-settings">
  ## Gestiona la configuración de la aplicación
</div>

Puedes gestionar la configuración de tu aplicación en el [Dashboard de Vercel](#vercel-dashboard) y el [Auth0 Dashboard](#auth0-dashboard).

<div id="vercel-dashboard">
  ### Dashboard de Vercel
</div>

Para administrar la configuración de tu aplicación en el Dashboard de Vercel, ve a **Settings** en el menú de navegación de la izquierda.

Puedes actualizar la siguiente configuración de la aplicación:

* **Actualizar nombre**: Actualiza el nombre de la aplicación.
* **Actualizar configuración**:
  * **Parámetros**: Actualiza la URL de localhost y el endpoint de devolución de llamada de tu aplicación.
  * **Plan actual del nivel de instalación**: Ve a la configuración de la integración para actualizar el plan del nivel de instalación de tu integración. Para obtener más información, consulta [Settings](#settings).
* **Proteger este recurso**
  * **Rotar secrets**: Rota los secrets de tu aplicación. Para obtener más información, consulta [Rotating environment variables](https://vercel.com/docs/projects/environment-variables/managing-environment-variables#rotating-environment-variables).
  * **Entornos permitidos**: Selecciona si la integración está disponible para **Todos los entornos** o **solo el entorno Production**.
* **Eliminar integración**: Cuando elimines la integración, pasarás automáticamente al plan Free de Vercel. Para obtener más información, consulta [Eliminar integración](#delete-integration).

<div id="auth0-dashboard">
  ### Auth0 Dashboard
</div>

Administra la configuración adicional de la aplicación en el [Auth0 Dashboard](https://manage.auth0.com/).

Cuando instalas la integración nativa, Auth0 crea una nueva aplicación en un tenant dedicado, preconfigurada con tus URI de Vercel, las URL de devolución de llamada permitidas y las URL de cierre de sesión.

Puedes modificar esta aplicación, por ejemplo, personalizando tu experiencia de Universal Login, igual que harías con cualquier otra aplicación en tu Dashboard de Auth0.

<div id="manage-integration">
  ## Gestionar la integración
</div>

Para gestionar la integración desde el Dashboard de Vercel, selecciona la pestaña **Integration** de tu proyecto, desplázate hasta la integración de Auth0 y selecciona **Manage**.

<div id="invoices">
  ### Facturas
</div>

En la página de **Facturas**, consulta una lista de las facturas relacionadas con tu integración de Auth0. Para obtener más información sobre las facturas, consulta [Billing &amp; Invoices](https://vercel.com/docs/pricing/understanding-my-invoice).

<div id="settings">
  ### Configuración
</div>

En la página de **Configuración**, agrega métodos de pago, cambia el plan de la instalación y elimina la integración.

<div id="payment-methods">
  #### Métodos de pago
</div>

Para agregar un método de pago, selecciona **Agregar**. Para obtener más información sobre los métodos de pago, consulta [Precios de Vercel](https://vercel.com/docs/accounts/plans).

<div id="current-installation-level-plan">
  #### Plan actual del nivel de instalación
</div>

El plan del nivel de instalación se aplica a todos los productos de la instalación.

Para cambiar el plan del nivel de instalación:

1. En **Configuración**, desplázate hasta **Plan actual del nivel de instalación** y selecciona **Cambiar plan**.
2. Selecciona el plan del nivel de instalación que prefieras.
3. Selecciona **Continuar**.
4. Revisa tu selección y, a continuación, selecciona **Actualizar**.
5. Una vez que se haya actualizado la instalación, selecciona **Listo**.

<div id="delete-integration">
  #### Eliminar integración
</div>

Cuando elimines la integración, pasarás automáticamente al plan Free de Vercel.

Para eliminar la integración:

1. En **Settings**, desplázate hasta **Delete &lt;YOUR&#95;PRODUCT&gt;** y selecciona el botón **Remove &lt;YOUR&#95;PRODUCT&gt;**.
2. Introduce el slug de la integración y tu Team, y luego selecciona **Reconozco que esto eliminará mi cuenta conectada a Auth0**.
3. Selecciona **Remove &lt;YOUR&#95;PRODUCT&gt;**.
