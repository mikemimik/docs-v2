Hay muchas formas de integrar Auth0 con los productos y servicios de Amazon, según tus necesidades específicas.

<div id="product-integrations">
  ## Integraciones de productos
</div>

Puedes obtener más información sobre cómo usar los siguientes productos junto con tus servicios de Auth0 y AWS:

* [CloudFront](/docs/es/customize/custom-domains/self-managed-certificates/configure-aws-cloudfront-for-use-as-reverse-proxy): Úsalo como proxy inverso con tu <Tooltip tip="Custom Domain: Dominio de terceros con un nombre especializado o personalizado." cta="Ver glosario" href="/docs/es/glossary?term=custom+domain">dominio personalizado</Tooltip>.
* [Simple Email Service (SES)](/docs/es/customize/email/smtp-email-providers/configure-amazon-ses-as-external-smtp-email-provider): Administra las comunicaciones por correo electrónico con tus usuarios.
* [EventBridge](https://marketplace.auth0.com/integrations/amazon-log-streaming): Envía registros a EventBridge.
* [Cognito](/docs/es/customize/integrations/aws/amazon-cognito): Úsalo como backend de tu aplicación.
* AWS S3 and DynamoDB:

  * Auth0 proporciona endpoints de gateway de VPC para acceder a AWS S3 y DynamoDB, que pueden ser necesarios para [Actions](/docs/es/customize/actions). Esto requiere políticas de IAM basadas en `aws:SourceVpc`, como se describe en [Gateway endpoints for Amazon S3](https://docs.aws.amazon.com/vpc/latest/privatelink/vpc-endpoints-s3.html) y [Gateway endpoints for Amazon DynamoDB](https://docs.aws.amazon.com/vpc/latest/privatelink/vpc-endpoints-ddb.html). Ponte en contacto con el [equipo de soporte de Auth0](https://www.support.auth0.com) para obtener más información sobre los identificadores de VPC.

<div id="identity-and-access-management-iam-integrations">
  ## Integraciones de gestión de identidades y accesos (IAM)
</div>

Tenemos varias guías para configurar AWS para que funcione con Auth0:

* [Protege endpoints de AWS API Gateway con autorizadores personalizados](/docs/es/customize/integrations/aws/aws-api-gateway-custom-authorizers): Acepta <Tooltip tip="Token de acceso: Credencial de autorización, en forma de cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+tokens">tokens de acceso</Tooltip> emitidos por Auth0.
* [Usa etiquetas de sesión de AWS con API y recursos de AWS](/docs/es/customize/integrations/aws/session-tags): Etiqueta recursos y asigna a los usuarios pares clave/valor para RBAC.
* [Configura AWS para inicio de sesión único](/docs/es/customize/integrations/aws/configure-amazon-web-services-for-sso): Permite que tus usuarios inicien sesión en AWS con cualquier <Tooltip tip="Proveedor de identidad (IdP): Servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=identity+provider">proveedor de identidad</Tooltip> compatible.
* [Agrega el inicio de sesión de Amazon a tu aplicación](/docs/es/customize/integrations/aws/configure-amazon-web-services-for-sso): Permite que los usuarios inicien sesión en tu aplicación con su perfil de Amazon.
