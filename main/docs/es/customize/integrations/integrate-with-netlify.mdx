<div id="overview">
  ## Información general
</div>

Conecta una instancia de Auth0 nueva o existente a tu proyecto de Netlify para mejorar la seguridad, la escalabilidad y la personalización de tu aplicación mediante la integración de Auth0. Esta integración te permite implementar directamente en tu aplicación un enfoque optimizado para la identidad de clientes.

Usa la nueva integración para crear y personalizar el flujo de identidad de tu proyecto de Netlify. Para poner tu aplicación en marcha de forma rápida y segura, esta integración:

* Configura un entorno de producción en Netlify que se integra con tu tenant de Auth0 en solo unos clics.
* Crea automáticamente una aplicación en tu tenant de Auth0 que se conecta con tu proyecto de Netlify.
* Carga previamente tus credenciales de Auth0 en Netlify, lo que te permite omitir algunos de los pasos manuales de este proceso.

<div id="prerequisites">
  ## Requisitos previos
</div>

Para empezar a integrar Auth0 en tu proyecto de Netlify, hay algunos requisitos que debes tener listos antes de comenzar:

* Una cuenta de Netlify con un proyecto que quieras integrar. Visita [Crea tu cuenta de Netlify](https://www.netlify.com/) para crear una cuenta.
* Una aplicación web que requiera autenticación, como un sitio de React, Vue o un sitio estático con funciones sin servidor.

<div id="add-a-netlify-integration">
  ## Añade una integración de Netlify
</div>

Después de crear tu cuenta de Netlify e iniciar sesión en tu Dashboard, puedes comenzar el proceso para añadir una integración. Para añadir la integración de Auth0 a un proyecto de Netlify existente:

1. En el Dashboard de Netlify, ve a **Extensions &gt; Directory.**
2. Busca [**Auth0**](https://app.netlify.com/extensions/auth0) y selecciona **Install.**
3. Selecciona **Install** para continuar con la instalación en tu portal de integración de Auth0.

<Frame>![](/docs/images/cdy7uua7fh8z/18pdegCJ6AHGyWsWY53aQt/f5ecef993ef893f1920dda33246e6840/netlify-directory.png)</Frame>

<div id="connect-your-site">
  ## Conecta tu sitio
</div>

Para conectar tu sitio a Auth0:

1. Ve a la **Configuración del sitio** del sitio que quieres conectar a Auth0.
2. En la barra lateral, selecciona **Auth0** en **Access &amp; security.**
3. Selecciona **Link an Auth0 tenant** y sigue los prompts para conectarte a tu cuenta de Auth0. Si todavía no tienes una cuenta de Auth0, puedes registrarte en https://auth0.com/signup.

   <Frame>![](/docs/images/cdy7uua7fh8z/3wZZNNVMadwZjeLi9vyQtH/01821cd184326a004592f5e0e20a6f72/2025-03-03_12-12-09.png)</Frame>
4. Selecciona tu tenant en la lista desplegable y luego **Accept** para autorizar tu cuenta.

   <Frame>![](/docs/images/cdy7uua7fh8z/7mrUvgQh70cgtri09aOHso/a76600ebf472dcddb3977a8b9e726e01/netlify-link.png)</Frame>
5. Después de vincular tu tenant, selecciona **Add tenant**.
6. Completa la información requerida para tu tenant y tu aplicación, marca la casilla **Redeploy site** y selecciona **Create site tenant**.

   <Frame>![](/docs/images/cdy7uua7fh8z/hPxAU8mwd9035AXOPMqxp/f74d0864a6cc5a4aed963bfc77237d7c/auth0_tenant.png)</Frame>
7. Ahora deberías ver el tenant del sitio vinculado correctamente en la configuración de tu sitio.

   <Frame>![](/docs/images/cdy7uua7fh8z/1y8khKk4yjGvySB2QRItD3/0f9a5606373c69761ff6fefa4fe180af/netlify-success.png)</Frame>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Ahora deberías ver tu aplicación cliente y tu API en tu Auth0 Dashboard, en **Applications**.
</Callout>

<div id="set-up-the-integration">
  ## Configura la integración
</div>

Al integrar Auth0 con tu proyecto de Netlify, tendrás que proporcionar a tu aplicación ajustes de autenticación, como el **<Tooltip tip="ID de cliente: Valor de identificación asignado por Auth0 a tu recurso registrado." cta="Ver glosario" href="/docs/es/glossary?term=Client+ID">ID de cliente</Tooltip>** y el **dominio de Auth0**.

Si tu aplicación lee estos ajustes desde variables de entorno, la extensión los inyecta automáticamente.

1. Ve a tu Dashboard de Netlify.
2. Ve a **Configuración del sitio &gt; Environment variables.**
3. Haz clic en **Add a variable** e introduce:

   * `AUTH0_CLIENT_ID` (tu ID de cliente de Auth0)
   * `AUTH0_DOMAIN` (tu dominio de Auth0)

     <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
       Estas variables, junto con toda la información de la aplicación, se pueden encontrar en tu Auth0 Dashboard, en **Applications &gt; [Your app] &gt; Settings**.
     </Callout>

Ahora ya deberías tener configuradas las credenciales correctas de Auth0 para el despliegue.

<div id="populate-a-configuration-file">
  ### Completar un archivo de configuración
</div>

Si su aplicación lee la configuración de Auth0 desde un archivo, puede automatizar este proceso durante la fase de compilación. Esto garantiza que su aplicación siempre tenga la configuración de autorización correcta sin tener que modificar archivos manualmente.

Para hacerlo:

1. Vaya a su Dashboard de Netlify.
2. Navegue a **Configuración del sitio &gt; Build &amp; deploy &gt; Build settings**.
3. Modifique el **Build command** para incluir un script de shell antes de iniciar la compilación:

   ```bash wrap lines
   echo "{\"clientId\": \"${AUTH0_CLIENT_ID}\", \"domain\": \"${AUTH0_DOMAIN}\"}" > auth_config.json && npm run build
   ```

Este script crea un archivo JSON llamado `auth_config.json` y lo completa con las variables de entorno `AUTH0_CLIENT_ID` y `AUTH0_DOMAIN` definidas anteriormente, lo que garantiza que la aplicación tenga la configuración correcta de Auth0 en el runtime.

<div id="manage-integration">
  ## Administrar la integración
</div>

Después de agregar la integración de Auth0 a su proyecto de Netlify, el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> crea una nueva aplicación para gestionar y personalizar su flujo de trabajo de CIAM.

1. Vaya al Auth0 Dashboard.
2. Seleccione **Applications &gt; Applications &gt; [Your Netlify application]**
3. Confirme el URI de la aplicación de Netlify en **Settings &gt; Application URIs**.

Ahora puede personalizar y modificar esta aplicación como cualquier otra aplicación de su Auth0 Dashboard.
