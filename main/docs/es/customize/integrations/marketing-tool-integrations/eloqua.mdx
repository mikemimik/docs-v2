title: Exportar datos de usuario a Oracle Eloqua
---

En este artículo, aprenderás a exportar datos de usuario de Auth0 a un archivo CSV y luego importarlos en Eloqua con el asistente de carga de contactos.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para comenzar, ve a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

Luego, establece **Export Format** con el formato de archivo requerido. Eloqua acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, especifica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de agregar los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación se haya completado, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Antes de empezar, puedes encontrar más información en [Oracle Eloqua Help Center: Uploading Contacts](https://docs.oracle.com/en/cloud/saas/marketing/eloqua-user/Help/Contacts/Tasks/UploadingContacts.htm).

Para importar tu archivo CSV a Eloqua, sigue estos pasos:

1. Inicia sesión en Eloqua y ve a **Audience &gt; Contacts &gt; Upload** para abrir el **Contact Upload Wizard**.
2. Configura la importación con los campos disponibles en la pestaña **Pick Data Source**.
3. Haz clic en el botón **Cloud** y selecciona el archivo que deseas cargar.
4. Después de verificar el contenido del archivo en la pestaña **Review**, haz clic en **Next Step**.
5. En la pestaña **Map Fields**, introduce la configuración de mapeo de campos para que los datos de tu archivo CSV coincidan con los campos de contacto. Haz clic en **Next Step** para continuar.
6. Completa el último paso del asistente y haz clic en **Finish**.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a Eloqua.
