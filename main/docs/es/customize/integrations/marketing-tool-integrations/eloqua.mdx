title: Exportar datos de usuario a Oracle Eloqua
---

En este artículo, aprenderás a exportar datos de usuario de Auth0 a un archivo CSV y luego a importarlos en Eloqua con Contact Upload Wizard.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Comience por dirigirse a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abra la **User Import / Export Extension**. En la página de la extensión, seleccione **Export** en el menú.

A continuación, establezca el **Formato de exportación** en el formato de archivo requerido. Eloqua acepta importaciones de archivos en formato CSV, así que elija la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indique un **User Field** y un **Column Name** para cada atributo de usuario que quiera incluir en la exportación. Por ejemplo:

| User Field    | Column Name                     |
| ------------- | ------------------------------- |
| `email`       | Dirección de correo electrónico |
| `created_at`  | Fecha de creación               |
| `given_name`  | Nombre                          |
| `family_name` | Apellido                        |

Después de añadir los campos de usuario, haga clic en el botón **Export Users** para iniciar la exportación. Una vez completada, descargue el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Antes de comenzar, puede obtener más información en [Oracle Eloqua Help Center: Uploading Contacts](https://docs.oracle.com/en/cloud/saas/marketing/eloqua-user/Help/Contacts/Tasks/UploadingContacts.htm).

Para importar su archivo CSV en Eloqua, siga estos pasos:

1. Inicie sesión en Eloqua y vaya a **Audience &gt; Contacts &gt; Upload** para abrir el **Contact Upload Wizard**.
2. Configure la importación con los campos proporcionados en la pestaña **Pick Data Source**.
3. Haga clic en el botón **Cloud** y seleccione el archivo que desea cargar.
4. Después de verificar el contenido del archivo en la pestaña **Review**, haga clic en **Siguiente paso**.
5. En la pestaña **Map Fields**, introduzca la configuración de mapeo de campos para hacer coincidir los datos de su archivo CSV con los campos de contacto. Haga clic en **Siguiente paso** para continuar.
6. Complete el paso final del asistente y haga clic en **Finish**.

¡Eso es todo! Ha importado correctamente sus usuarios de Auth0 a Eloqua.
