---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego importarlos en Adobe Campaign con el [Asistente de importación de Adobe Campaign](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html).

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para empezar, ve a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, configura **Export Format** con el formato de archivo requerido. Adobe Campaign acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name                     |
| ------------- | ------------------------------- |
| `email`       | Dirección de correo electrónico |
| `created_at`  | Fecha de creación               |
| `given_name`  | Nombre                          |
| `family_name` | Apellido                        |

Después de agregar los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación se complete, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Antes de comenzar, puede obtener más información en [Documentación de Adobe Campaign: Importing Data](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html).

Inicie sesión en el Dashboard del cliente de Adobe Campaign y vaya a **Profiles and Targets &gt; Jobs**. Cree una nueva tarea de importación haciendo clic en el botón **Create** y seleccionando **New Import**.

Debería abrirse una nueva ventana de **Import Wizard**. En el paso **Template Selection** puede establecer los parámetros de importación.

Una vez que haya configurado los parámetros de importación, haga clic en el botón **Next** para continuar.

En el paso **File to Import**, cargue el archivo CSV de datos de usuario que exportó desde Auth0 en la sección anterior. Haga clic en el botón **Next** para pasar a **Field Mapping**.

A continuación, asigne el schema del archivo de exportación al schema de la base de datos de Adobe Campaign. Verifique que los nombres y tipos de campo sean correctos y, a continuación, haga clic en el botón **Next**.

Complete los pasos de configuración restantes definiendo el modo de conciliación de datos y seleccionando una carpeta, lista o servicio para los usuarios que se van a importar.

Por último, inicie la importación haciendo clic en el botón **Start** en la ventana **Data Import Execution**.

¡Eso es todo! Ha importado correctamente sus usuarios de Auth0 a Adobe Campaign.
