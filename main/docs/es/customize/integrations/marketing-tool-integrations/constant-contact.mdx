title: Exportar datos de usuario a Constant Contact
---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego a importarlos en el panel de Constant Contact.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Comience por ir a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abrir la **User Import / Export Extension**. En la página de la extensión, seleccione **Export** en el menú.

A continuación, configure **Export Format** con el formato de archivo requerido. Constant Contact acepta importaciones de archivos en formato CSV, así que elija la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indique un **User Field** y un **Column Name** para cada atributo de usuario que quiera incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de agregar los campos de usuario, haga clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación se complete, descargue el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importa un archivo de datos de usuario
</div>

Para importar tu archivo CSV a Constant Contact, sigue estos pasos:

1. [Inicia sesión en Constant Contact](https://login.constantcontact.com) y ve a **Contacts**.
2. En la página **Contacts**, haz clic en **Add Contacts &gt; Upload from file**.
3. Sube tu archivo CSV en la página **Upload from file** y luego haz clic en **Continuar**.
4. Revisa las columnas y asígnalas a los campos de Constant Contact correspondientes. Cuando termines, haz clic en **Continuar**.
5. En la página **Select lists**, elige la lista o las listas a las que quieres agregar tus contactos.
6. Haz clic en **Upload**.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a Constant Contact.
