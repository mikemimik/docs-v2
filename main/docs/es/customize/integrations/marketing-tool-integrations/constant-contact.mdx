title: Exportar datos de usuario a Constant Contact
---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego a importarlos en el Dashboard de Constant Contact.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para comenzar, ve a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, establece **Export Format** en el formato de archivo requerido. Constant Contact acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, especifica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de añadir los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Una vez completada, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Para importar tu archivo CSV en Constant Contact, sigue estos pasos:

1. [Inicia sesión en Constant Contact](https://login.constantcontact.com) y ve a **Contacts**.
2. En la página **Contacts**, haz clic en **Add Contacts &gt; Upload from file**.
3. Carga tu archivo CSV en la página **Upload from file** y luego haz clic en **Continuar**.
4. Revisa las columnas y asígnalas a los campos correspondientes de Constant Contact. Cuando termines, haz clic en **Continuar**.
5. En la página **Select lists**, elige la lista o las listas a las que quieres agregar tus contactos.
6. Haz clic en **Upload**.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 en Constant Contact.
