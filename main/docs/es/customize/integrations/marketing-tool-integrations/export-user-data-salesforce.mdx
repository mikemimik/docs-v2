title: Exportar datos de usuario a Salesforce
---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego importarlos en Salesforce con el Data Import Wizard. Para consultar los requisitos del sistema, visita el artículo de Salesforce [Data Import Wizard](https://help.salesforce.com/articleView?id=data_import_wizard.htm).

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para empezar, vaya a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abra la **User Import / Export Extension**. En la página de la extensión, seleccione **Export** en el menú.

A continuación, configure **Export Format** con el formato de archivo requerido. Salesforce acepta importaciones de archivos en formato CSV, así que elija la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, especifique un **User Field** y un **Column Name** para cada atributo de usuario que quiera incluir en la exportación. Por ejemplo:

| Campo de usuario | Nombre de columna               |
| ---------------- | ------------------------------- |
| `email`          | Dirección de correo electrónico |
| `created_at`     | Fecha de creación               |
| `given_name`     | Nombre                          |
| `family_name`    | Apellido                        |

Después de agregar los campos de usuario, haga clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación se complete, descargue el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Es posible que algunos tipos de cuenta requieran pasos adicionales para completar la importación correctamente. Tómate un momento para revisar el artículo de Salesforce [Prepare Your Data for Import](https://help.salesforce.com/articleView?id=import_prepare.htm). Para obtener instrucciones adicionales, visita [Salesforce: Import Data with the Data Import Wizard](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm).

Antes de comenzar, asegúrate de que tu cuenta de Salesforce tenga los [permisos de usuario](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm) necesarios para importar registros.

Para importar tu archivo CSV en Salesforce, sigue estos pasos:

1. Inicia sesión en [Salesforce](https://login.salesforce.com/).
2. Ve a **Setup** y abre **Data Import Wizard**.
3. Haz clic en **Launch Wizard**.
4. Configura la importación y selecciona el archivo CSV que quieres cargar en los campos correspondientes.
5. Haz clic en **Next**.
6. En la página **Edit Field Mapping**, asigna los campos de datos de tu archivo CSV a los campos de datos de Salesforce. Haz clic en **Next** para continuar.
7. Después de verificar la información en la página **Review**, haz clic en **Start Import**.
8. Revisa la pestaña **Recent Import Jobs** en la página principal de **Data Import Wizard** para ver actualizaciones sobre el estado de la importación.

¡Eso es todo! Importaste correctamente tus usuarios de Auth0 en Salesforce.
