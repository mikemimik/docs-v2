---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego importarlos en Salesforce con el Data Import Wizard. Para consultar los requisitos del sistema, visita el artículo de Salesforce [Data Import Wizard](https://help.salesforce.com/articleView?id=data_import_wizard.htm).

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Empieza por ir a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abrir la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, configura **Export Format** con el formato de archivo requerido. Salesforce acepta la importación de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de añadir los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación haya finalizado, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importa un archivo de datos de usuario
</div>

Algunos tipos de cuenta pueden requerir pasos adicionales para que la importación se complete correctamente. Tómate un momento para revisar el artículo de Salesforce [Prepare Your Data for Import](https://help.salesforce.com/articleView?id=import_prepare.htm). Para obtener instrucciones adicionales, visita [Salesforce: Import Data with the Data Import Wizard](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm).

Antes de comenzar, asegúrate de que tu cuenta de Salesforce tenga los [permisos de usuario](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm) necesarios para importar registros.

Para importar tu archivo CSV en Salesforce, sigue estos pasos:

1. Inicia sesión en [Salesforce](https://login.salesforce.com/).
2. Ve a **Setup** y abre el **Data Import Wizard**.
3. Haz clic en **Launch Wizard**.
4. Configura la importación y selecciona el archivo CSV que quieres cargar mediante los campos proporcionados.
5. Haz clic en **Next**.
6. En la página **Edit Field Mapping**, asigna los campos de datos de tu archivo CSV a los campos de datos de Salesforce. Haz clic en **Next** para continuar.
7. Después de verificar la información en la página **Review**, haz clic en **Start Import**.
8. Consulta la pestaña **Recent Import Jobs** en la página principal de **Data Import Wizard** para ver actualizaciones sobre el estado de la importación.

¡Eso es todo! Importaste correctamente tus usuarios de Auth0 a Salesforce.
