title: Exportar datos de usuario a MailChimp
---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego a importarlos en el [panel de control de MailChimp](https://login.mailchimp.com/).

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para empezar, ve a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, establece **Export Format** en el formato de archivo requerido. MailChimp acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

Para obtener más información, consulta [MailChimp Knowledge Base: Format Guidelines for Your Import File](https://kb.mailchimp.com/lists/growth/format-guidelines-for-your-import-file).

En la parte superior, en la sección **Fields**, indica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. En MailChimp, es obligatorio un campo de correo electrónico con el nombre de columna `Email Address`, así que asegúrate de incluirlo. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de agregar los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando se complete, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Antes de empezar, puedes encontrar más información en la [base de conocimiento de MailChimp: Import Subscribers to a List](https://kb.mailchimp.com/lists/growth/import-subscribers-to-a-list).

Inicia sesión en tu cuenta de MailChimp y ve a la página de **Lists**. Selecciona una lista para importar tus usuarios de Auth0 ([o crea una lista nueva](https://kb.mailchimp.com/lists/growth/create-a-new-list)).

1. En la página de tu lista de MailChimp, haz clic en **Import Contacts** en el menú **Add Contacts**.
2. Selecciona la opción `CSV or tab-delimited text file` para importar los contactos.
3. Carga el archivo CSV de datos de usuario que exportaste desde Auth0 en la sección anterior. MailChimp interpretará tus datos de usuario en la página siguiente.
4. Comprueba que los nombres de columna y los tipos de campo sean correctos.
5. Haz clic en el botón **Next**.
6. Revisa tus selecciones y establece la categoría de importación.
7. Haz clic en el botón **Import** para iniciar la importación de usuarios.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a MailChimp.
