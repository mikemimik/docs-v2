title: Exportar datos de usuario a MailChimp
---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego importarlos en el [panel de control de MailChimp](https://login.mailchimp.com/).

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Comience por ir a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abrir la **User Import / Export Extension**. En la página de la extensión, seleccione **Export** en el menú.

A continuación, establezca **Export Format** en el formato de archivo requerido. MailChimp acepta importaciones de archivos en formato CSV, así que elija la opción `Tab Separated Value file (*.csv)`.

Para obtener más información, consulte [Base de conocimientos de MailChimp: Directrices de formato para su archivo de importación](https://kb.mailchimp.com/lists/growth/format-guidelines-for-your-import-file).

En la parte superior, en la sección **Fields**, indique un **User Field** y un **Column Name** para cada atributo de usuario que quiera incluir en la exportación. Para MailChimp, se requiere un campo de correo electrónico con el nombre de columna `Email Address`, así que asegúrese de incluirlo. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de añadir los campos de usuario, haga clic en el botón **Export Users** para iniciar la exportación. Una vez completada, descargue el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Antes de comenzar, puede obtener más información en [MailChimp Knowledge Base: Importar suscriptores a una lista](https://kb.mailchimp.com/lists/growth/import-subscribers-to-a-list).

Inicie sesión en su cuenta de MailChimp y vaya a la página **Lists**. Seleccione una lista en la que importar sus usuarios de Auth0 ([o cree una lista nueva](https://kb.mailchimp.com/lists/growth/create-a-new-list)).

1. En la página de su lista de MailChimp, haga clic en **Import Contacts** en el menú **Add Contacts**.
2. Seleccione la opción `CSV or tab-delimited text file` para importar contactos.
3. Cargue el archivo CSV de datos de usuario que exportó de Auth0 en la sección anterior. MailChimp interpretará sus datos de usuario en la página siguiente.
4. Compruebe que los nombres de las columnas y los tipos de campo sean correctos.
5. Haga clic en el botón **Next**.
6. Revise sus selecciones y establezca la categoría de importación.
7. Haga clic en el botón **Import** para iniciar la importación de usuarios.

¡Eso es todo! Ha importado correctamente sus usuarios de Auth0 a MailChimp.
