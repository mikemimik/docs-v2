---

En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y, después, a importarlos en Alterian con la herramienta de Data Import del administrador de campañas.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Empieza por ir a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, configura **Export Format** con el formato de archivo requerido. Alterian acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name                     |
| ------------- | ------------------------------- |
| `email`       | Dirección de correo electrónico |
| `created_at`  | Fecha de creación               |
| `given_name`  | Nombre                          |
| `family_name` | Apellido                        |

Después de agregar los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación se complete, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importa un archivo de datos de usuario
</div>

Antes de comenzar, puedes obtener más información en [Alterian Campaign Manager: Data Import](https://cm.help.alterian.com/CM404/Default.htm#Customer_Analytics/Import_Export/Data_Import.htm).

Para importar tu archivo CSV a Alterian, sigue estos pasos:

1. Abre Alterian Campaign Manager y haz clic en **Documents**.
2. Elige el directorio al que quieres importar el archivo.
3. Haz clic en **Upload Files** y selecciona tu archivo CSV.
4. Haz clic en **Upload**.
5. Cuando se complete la carga, abre la herramienta **Data Import**.
6. Configura las opciones de importación con los campos disponibles en la ventana **New Imports data**.
7. Después de revisar la configuración, haz clic en **Run Processes**.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a Alterian.
