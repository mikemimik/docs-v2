En este artículo, aprenderás a exportar datos de usuario de Auth0 a un archivo CSV y luego importarlos en Marketo mediante el [Bulk Leads endpoint](https://developer.adobe.com/marketo-apis/api/mapi/#tag/Bulk-Import-Leads) de la API REST de Marketo.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para empezar, ve a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abre la **User Import / Export Extension**. En la página de la extensión, selecciona **Export** en el menú.

A continuación, configura **Export Format** con el formato de archivo requerido. Marketo acepta importaciones de archivos en formato CSV, así que elige la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indica un **User Field** y un **Column Name** para cada atributo de usuario que quieras incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de añadir los campos de usuario, haz clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación haya finalizado, descarga el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ### Importar un archivo de datos de usuario
</div>

Antes de empezar, puedes encontrar más información en [Marketo Documentation: Bulk Lead Import](https://experienceleague.adobe.com/en/docs/marketo-developer/marketo/rest/bulk-import/bulk-lead-import).

Para importar el archivo de datos de usuario a Marketo, realiza una solicitud POST al [Bulk Leads endpoint](https://developer.adobe.com/marketo-apis/api/mapi/#operation/importLeadUsingPOST). Configura el header `content-type` de la solicitud como `multipart/form-data` e incluye un parámetro `file` con el archivo CSV exportado, así como el parámetro `format` establecido en `csv`. Por ejemplo:

<AuthCodeGroup>
  ```bash cURL lines
  curl --request POST \
    --url https://marketo_rest_api_base_url/bulk/v1/leads.json \
    --header 'authorization: Bearer {MARKETO_ACCESS_TOKEN}' \
    --form file=@auth0_users.csv \
    --form format=csv
  ```
