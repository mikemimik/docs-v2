En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego a importarlos en el panel de control de Sailthru.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Empiece por ir a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abra la **User Import / Export Extension**. En la página de la extensión, seleccione **Export** en el menú.

A continuación, establezca **Export Format** en el formato de archivo requerido. Sailthru acepta la importación de archivos en formato CSV, así que elija la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indique un **User Field** y un **Column Name** para cada atributo de usuario que se incluirá en la exportación. Por ejemplo:

| User Field    | Column Name                     |
| ------------- | ------------------------------- |
| `email`       | Dirección de correo electrónico |
| `created_at`  | Fecha de creación               |
| `given_name`  | Nombre                          |
| `family_name` | Apellido                        |

Después de agregar los campos de usuario, haga clic en el botón **Export Users** para iniciar la exportación. Una vez completada la exportación, descargue el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ### Importar un archivo de datos de usuario
</div>

Antes de empezar, puedes encontrar más información en [la documentación de Sailthru: Añadir usuarios a Sailthru y a las listas de Sailthru](https://getstarted.meetmarigold.com/engagebysailthru/Content/audience/managing-users/add-users-lists-st.html?TocPath=Audience%7CManaging%20Users%7C_____3#ListFileUpload).

Para importar tu archivo CSV a Sailthru, sigue estos pasos:

1. Inicia sesión en Sailthru y ve a [Lists](https://my.sailthru.com/lists).
2. Elige una lista y, a continuación, haz clic en **Upload List** y selecciona tu archivo CSV.
3. Selecciona **Add To List** en **Action**.
4. Configura la opción **Replace Vars** e introduce una dirección de correo electrónico para recibir notificaciones, si lo deseas.
5. Revisa la configuración y haz clic en **Submit** para cargar el archivo.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a Sailthru.
