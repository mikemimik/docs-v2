En este artículo, aprenderás a exportar los datos de usuario de Auth0 a un archivo CSV y luego importarlos en el panel de control de Sailthru.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Empiece por ir a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abrir la **User Import / Export Extension**. En la página de la extensión, seleccione **Export** en el menú.

A continuación, configure **Export Format** con el formato de archivo requerido. Sailthru acepta importaciones de archivos en formato CSV, así que elija la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, indique un **User Field** y un **Column Name** para cada atributo de usuario que quiera incluir en la exportación. Por ejemplo:

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Después de agregar los campos de usuario, haga clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación haya finalizado, descargue el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ### Importar un archivo de datos de usuario
</div>

Antes de empezar, puedes encontrar más información en [Sailthru Documentation: Adding Users to Sailthru and Sailthru Lists](https://getstarted.meetmarigold.com/engagebysailthru/Content/audience/managing-users/add-users-lists-st.html?TocPath=Audience%7CManaging%20Users%7C_____3#ListFileUpload).

Para importar tu archivo CSV a Sailthru, sigue estos pasos:

1. Inicia sesión en Sailthru y ve a [Lists](https://my.sailthru.com/lists).
2. Elige una lista, haz clic en **Upload List** y selecciona tu archivo CSV.
3. Selecciona **Add To List** en **Action**.
4. Configura la opción **Replace Vars** e introduce un correo electrónico para recibir notificaciones si lo deseas.
5. Revisa la configuración y haz clic en **Submit** para cargar el archivo.

¡Eso es todo! Has importado correctamente tus usuarios de Auth0 a Sailthru.
