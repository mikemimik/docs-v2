description: Esta página explica los aspectos básicos del plugin de WordPress Login by Auth0
title: Login by Auth0 WordPress Plugin
---

Auth0 ofrece un plugin de WordPress para integrar tus sitios con tu cuenta de Auth0. Este plugin habilita el <Tooltip tip="Inicio de sesión único (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, lo inicia sesión automáticamente en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=Single+Sign-on">inicio de sesión único</Tooltip> (SSO) para empresas, el inicio de sesión con redes sociales, usuario/contraseña y el inicio de sesión <Tooltip tip="Inicio de sesión sin contraseña: método que permite a los usuarios iniciar sesión sin una contraseña tradicional, normalmente mediante enlaces por correo electrónico, códigos por SMS, biometría u otras alternativas similares." cta="Ver glosario" href="/docs/es/glossary?term=passwordless">passwordless</Tooltip> en todas tus instancias.

Las funcionalidades de inicio de sesión se implementan mediante un nuevo formulario (con tecnología de Auth0) que habilita:

* SSO con directorios empresariales (LDAP, AD, Google Workspace, Office365 y proveedor de <Tooltip tip="Security Assertion Markup Language (SAML): protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin una contraseña." cta="Ver glosario" href="/docs/es/glossary?term=SAML">SAML</Tooltip>)
* SSO con más de 30 [proveedores sociales](/docs/es/authenticate/identity-providers)
* SSO entre instalaciones de WordPress y otras aplicaciones
* Dashboard de gestión de usuarios
* <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuario que utiliza un factor además de username and password, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=Multi-factor+Authentication">Autenticación multifactor</Tooltip> opcional
* Autenticación passwordless opcional
* Informes y analíticas
