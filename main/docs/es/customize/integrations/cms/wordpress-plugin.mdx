title: Login by Auth0 WordPress Plugin
---

Auth0 proporciona un Plugin de WordPress para integrar tus sitios con tu cuenta de Auth0. Este plugin habilita el <Tooltip tip="Inicio de sesión único (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=Single+Sign-on">inicio de sesión único</Tooltip> (SSO) para empresas, inicio de sesión social, usuario/contraseña e inicio de sesión <Tooltip tip="Inicio de sesión único (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=passwordless">passwordless</Tooltip> en todas tus instancias.

Las funciones de inicio de sesión se implementan mediante un nuevo formulario (con tecnología de Auth0) que permite:

* SSO con directorios empresariales (LDAP, AD, Google Workspace, Office365 y proveedor de <Tooltip tip="Security Assertion Markup Language (SAML): protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin una contraseña." cta="Ver glosario" href="/docs/es/glossary?term=SAML">SAML</Tooltip>)
* SSO con más de 30 [proveedores sociales](/docs/es/authenticate/identity-providers)
* SSO entre instalaciones de WordPress y otras aplicaciones
* Dashboard de User Management
* <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuario que utiliza un factor además de nombre de usuario y contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=Multi-factor+Authentication">Autenticación multifactor</Tooltip> opcional
* Autenticación Passwordless opcional
* Informes y análisis
