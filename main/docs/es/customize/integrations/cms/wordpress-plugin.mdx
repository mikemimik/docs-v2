title: plugin de WordPress Login by Auth0
---

Auth0 proporciona un plugin de WordPress para integrar sus sitios con su cuenta de Auth0. Este plugin habilita el <Tooltip tip="Inicio de sesión único (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=Single+Sign-on">inicio de sesión único</Tooltip> (SSO) para empresas, inicio de sesión social, inicio de sesión con usuario y contraseña e inicio de sesión <Tooltip tip="Inicio de sesión único (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=passwordless">passwordless</Tooltip> en todas sus instancias.

Las funciones de inicio de sesión se implementan mediante un nuevo formulario (con tecnología de Auth0) que habilita:

* SSO con directorios empresariales (LDAP, AD, Google Workspace, Office365 y <Tooltip tip="Security Assertion Markup Language (SAML): Protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin una contraseña." cta="Ver glosario" href="/docs/es/glossary?term=SAML">SAML</Tooltip> Provider)
* SSO con más de 30 [proveedores sociales](/docs/es/authenticate/identity-providers)
* SSO entre instalaciones de WordPress y otras aplicaciones
* Dashboard de gestión de usuarios
* <Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuario que utiliza un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=Multi-factor+Authentication">Autenticación multifactor</Tooltip> opcional
* Autenticación passwordless opcional
* Informes y análisis
