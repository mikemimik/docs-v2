<div id="get-your-telegram-token">
  ## Obtén tu token de Telegram
</div>

Aprende a generar un token en la [API de bots de Telegram](https://core.telegram.org/#bot-api).

<div id="configure-your-telegram-vault-connection">
  ## Configura tu conexión de Vault para Telegram
</div>

Para configurar una conexión de Vault para Telegram para tus acciones de flujo de Telegram, debes establecer los parámetros obligatorios.

<Frame>![Dashboard > Actions > Formularios > Vault > SendGrid](/docs/images/customize/forms/vaults/telegram-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                       |
| --------- | ------------------------------------------------- |
| Token     | Token generado para autorizar la API de Telegram. |
