<div id="get-your-mailjet-keys">
  ## Obtén tus claves de MailJet
</div>

Encuéntralas en la página Account del [Dashboard de MailJet](https://app.mailjet.com/account/api_keys), en la sección API keys.

<div id="configure-your-mailjet-vault-connection">
  ## Configure la conexión de Vault de MailJet
</div>

Para configurar una conexión de Vault de MailJet para sus Flow Actions de MailJet, debe establecer los parámetros obligatorios.

<Frame>![Dashboard > Actions > Forms > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### configuración de entrada
</div>

| Parámetro     | Descripción                                                 |
| ------------- | ----------------------------------------------------------- |
| API Key       | API Key para autorizar el acceso a la API de MailJet.       |
| Clave secreta | Clave secreta para autorizar el acceso a la API de MailJet. |
