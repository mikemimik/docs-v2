<div id="get-your-mailjet-keys">
  ## Obtén tus claves de MailJet
</div>

Las encontrarás en la página Account del [Dashboard de MailJet](https://app.mailjet.com/account/api_keys), en la sección de claves API.

<div id="configure-your-mailjet-vault-connection">
  ## Configura tu conexión de Vault de MailJet
</div>

Para configurar una conexión de Vault de MailJet para tus MailJet Flow Actions, debes configurar los parámetros obligatorios.

<Frame>![Dashboard > Actions > Formularios > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro     | Descripción                                               |
| ------------- | --------------------------------------------------------- |
| API Key       | API Key para la autenticación en la API de MailJet.       |
| Clave secreta | Clave secreta para la autenticación en la API de MailJet. |
