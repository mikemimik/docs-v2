<div id="configure-your-jwt-vault-connection">
  ## Configura tu Conexión de Vault JWT
</div>

Para configurar una Conexión de Vault JWT que permita a tus Actions JWT de Flows firmar, decodificar y verificar JSON Web Tokens, debes configurar los parámetros obligatorios.

<Frame>![Dashboard > Actions > Forms > Vault > JWT](/docs/images/customize/forms/vaults/jwt-vault-connection.png)</Frame>

<div id="input-settings">
  ## configuración de entrada
</div>

| Parámetro | Descripción                                                          |
| --------- | -------------------------------------------------------------------- |
| Secret    | Secret que se utilizará en el algoritmo HS256 para codificar el JWT. |
