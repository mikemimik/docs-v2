<div id="configure-your-jwt-vault-connection">
  ## Configura tu Conexión de Vault JWT
</div>

Para configurar una Conexión de Vault JWT para tus Actions JWT de Flows a fin de firmar, decodificar y verificar JSON Web Tokens, debes establecer los parámetros requeridos.

<Frame>![Dashboard > Actions > Formularios > Vault > JWT](/docs/images/customize/forms/vaults/jwt-vault-connection.png)</Frame>

<div id="input-settings">
  ## configuración de entrada
</div>

| Parámetro | Descripción                                                           |
| --------- | --------------------------------------------------------------------- |
| Secret    | Secret que se utilizará con el algoritmo HS256 para codificar el JWT. |
