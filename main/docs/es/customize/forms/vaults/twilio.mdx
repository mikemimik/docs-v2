<div id="get-your-twilio-credentials">
  ## Obtén tus credenciales de Twilio
</div>

Encuéntralas en [la consola de Twilio](https://console.twilio.com/).

<div id="configure-your-twilio-vault-connection">
  ## Configura tu conexión de Twilio Vault
</div>

Para configurar una conexión de Twilio Vault para tus Twilio Flow Actions, debes definir los parámetros obligatorios.

<Frame>![Dashboard > Actions > Formularios > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro   | Descripción                                  |
| ----------- | -------------------------------------------- |
| Account SID | Account SID para autorizar la API de Twilio. |
| Auth token  | Auth token para autorizar la API de Twilio.  |
