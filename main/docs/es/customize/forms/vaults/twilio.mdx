<div id="get-your-twilio-credentials">
  ## Obtén tus credenciales de Twilio
</div>

Encuéntralas en [el panel de la consola de Twilio](https://console.twilio.com/).

<div id="configure-your-twilio-vault-connection">
  ## Configura tu conexión de Vault de Twilio
</div>

Para configurar una conexión de Vault de Twilio para tus Flow Actions de Twilio, debes establecer los parámetros obligatorios.

<Frame>![Dashboard > Actions > Forms > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro   | Descripción                                                |
| ----------- | ---------------------------------------------------------- |
| Account SID | Account SID para autorizar solicitudes a la API de Twilio. |
| Auth token  | Auth token para autorizar solicitudes a la API de Twilio.  |
