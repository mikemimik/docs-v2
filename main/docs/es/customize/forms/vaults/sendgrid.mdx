<div id="get-your-sendgrid-keys">
  ## Obtén tus claves de SendGrid
</div>

Encuéntralas en la página Settings [del Dashboard de SendGrid](https://app.sendgrid.com/settings/api_keys), en la sección Claves API.

<div id="configure-your-sendgrid-vault-connection">
  ## Configura tu conexión de Vault de SendGrid
</div>

Para configurar una conexión de Vault de SendGrid para tus SendGrid Flow Actions, debes configurar los parámetros necesarios.

<Frame>![Dashboard > Actions > Formulario > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### configuración de entrada
</div>

| Parámetro | Descripción                                         |
| --------- | --------------------------------------------------- |
| API Key   | API Key para la autorización en la API de SendGrid. |
