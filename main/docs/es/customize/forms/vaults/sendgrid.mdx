<div id="get-your-sendgrid-keys">
  ## Obtén tus claves API de SendGrid
</div>

Encuéntralas en la página Settings del [Dashboard de SendGrid](https://app.sendgrid.com/settings/api_keys), en la sección de claves API.

<div id="configure-your-sendgrid-vault-connection">
  ## Configura tu conexión de Vault de SendGrid
</div>

Para configurar una conexión de Vault de SendGrid para tus SendGrid Flow Actions, debes establecer los parámetros necesarios.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                |
| --------- | ------------------------------------------ |
| API Key   | API Key para autorizar la API de SendGrid. |
