<div id="get-your-sendgrid-keys">
  ## Obtén tus claves de SendGrid
</div>

Encuéntralas en la página [Settings del dashboard de SendGrid](https://app.sendgrid.com/settings/api_keys), en la sección API keys.

<div id="configure-your-sendgrid-vault-connection">
  ## Configura tu conexión de Vault para SendGrid
</div>

Para configurar una conexión de Vault para SendGrid para tus SendGrid Flow Actions, debes configurar los parámetros obligatorios.

<Frame>![Dashboard > Actions > Formulario > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                |
| --------- | ------------------------------------------ |
| API Key   | API Key para autorizar la API de SendGrid. |
