<div id="get-your-whatsapp-token">
  ## Obtén tu token de WhatsApp
</div>

Aprende a generar un token en [WhatsApp](https://developers.facebook.com/blog/post/2022/12/05/auth-tokens/).

<div id="configure-your-whatsapp-vault-connection">
  ## Configura tu conexión de WhatsApp en Vault
</div>

Para configurar una conexión de WhatsApp en Vault para tus WhatsApp Flow Actions, debes establecer los parámetros necesarios.

<Frame>![Dashboard > Actions > formulario > Vault > WhatsApp](/docs/images/customize/forms/vaults/whatsapp-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                                |
| --------- | ---------------------------------------------------------- |
| Token     | Token generado para la autorización de la API de WhatsApp. |
