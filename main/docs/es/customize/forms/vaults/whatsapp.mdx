  ## Obtén tu token de WhatsApp
</div>

Aprende a generar un token de WhatsApp en [WhatsApp](https://developers.facebook.com/blog/post/2022/12/05/auth-tokens/).

<div id="configure-your-whatsapp-vault-connection">
  ## Configura tu conexión de Vault para WhatsApp
</div>

Para configurar una conexión de Vault para WhatsApp para tus WhatsApp Flow Actions, debes establecer los parámetros obligatorios.

<Frame>![Dashboard > Actions > Forms > Vault > WhatsApp](/docs/images/customize/forms/vaults/whatsapp-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                                |
| --------- | ---------------------------------------------------------- |
| Token     | Token generado para la autorización de la API de WhatsApp. |
