  ## Obtén tu token de WhatsApp
</div>

Descubre cómo generar un token en [WhatsApp](https://developers.facebook.com/blog/post/2022/12/05/auth-tokens/).

<div id="configure-your-whatsapp-vault-connection">
  ## Configura tu conexión de WhatsApp en Vault
</div>

Para configurar una conexión de WhatsApp en Vault para tus acciones de flujo de WhatsApp, debes establecer los parámetros obligatorios.

<Frame>![Dashboard > Actions > Formulario > Vault > WhatsApp](/docs/images/customize/forms/vaults/whatsapp-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                       |
| --------- | ------------------------------------------------- |
| Token     | Token generado para autorizar la API de WhatsApp. |
