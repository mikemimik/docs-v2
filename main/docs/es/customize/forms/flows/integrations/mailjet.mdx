  ## Enviar un correo electrónico
</div>

Envía un correo electrónico usando Mailjet como proveedor de correo electrónico.

<Frame>![](/docs/images/cdy7uua7fh8z/5z1CqSZTrLs1oMdHKvHOiM/7e1fa3b5d7ea5a5b2ec51dd58fa1d2ed/mailjet.png)</Frame>

<div id="input-settings">
  ### configuración de entrada
</div>

| Parámetro                                         | Descripción                                                                                                   |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Nombre del remitente                              | El nombre del remitente                                                                                       |
| Correo electrónico del remitente (obligatorio)    | El correo electrónico del remitente                                                                           |
| Nombre del destinatario                           | El nombre del destinatario                                                                                    |
| Correo electrónico del destinatario (obligatorio) | El correo electrónico del destinatario                                                                        |
| Nombre del CC                                     | El nombre del CC                                                                                              |
| Correo electrónico del CC                         | El correo electrónico del CC                                                                                  |
| Asunto                                            | El asunto del correo electrónico                                                                              |
| Tipo de correo electrónico                        | Enviar un correo electrónico con contenido HTML o con una plantilla guardada (opcional)                       |
| Mensaje (obligatorio)                             | Mensaje del correo electrónico en formato HTML o una plantilla (opcional).                                    |
| ID de plantilla (opcional)                        | Si usa una plantilla, puede encontrar su ID en el [ Dashboard de Mailjet ](https://app.mailjet.com/template). |
| Variables personalizadas (opcional)               | Agregue variables personalizadas para insertarlas en su plantilla.                                            |
| Rastrear la actividad del correo electrónico      | Casilla para rastrear la actividad del correo electrónico.                                                    |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes consultar la respuesta esperada y los códigos de estado en la [documentación de Mailjet](https://dev.mailjet.com/email/reference/send-emails/).
