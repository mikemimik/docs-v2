Esta lista de acciones de Twilio te permite enviar SMS y hacer llamadas con Twilio como proveedor.

<div id="configure-your-twilio-vault-connection">
  ## Configura tu Twilio Vault Connection
</div>

Aprende a configurar una Vault Connection para tus Twilio Flow Actions en [Twilio Vault Connection](/docs/es/customize/forms/vaults/twilio).

<div id="send-sms">
  ## Enviar SMS
</div>

Envía un SMS.

<Frame>![](/docs/images/cdy7uua7fh8z/4lPqmcYGCJ2oRHM3GSta6w/10b0141d163e4be601c2786895486440/sendsmstwilio.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro             | Descripción                                                                               |
| --------------------- | ----------------------------------------------------------------------------------------- |
| de (obligatorio)    | El nombre del remitente del SMS.                                                          |
| para (obligatorio)      | El número de teléfono de destino en [formato E.164](https://en.wikipedia.org/wiki/E.164). |
| Message (obligatorio) | El contenido del mensaje SMS.                                                             |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes consultar todos los códigos de estado posibles en la [documentación de Twilio](https://www.twilio.com/docs/messaging/api/message-resource#message-status-values).

<div id="make-call">
  ## Hacer una llamada
</div>

Hace una llamada.

<Frame>![](/docs/images/cdy7uua7fh8z/4JaAbJIVuVUJY40lhQuVUN/1a1bd6143c8033c12e18fa49252bd36b/makecalltwilio.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro                      | Descripción                                                                                                                                                                    |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| De (obligatorio)               | Tu número de teléfono de Twilio. Debe incluirse el código de país. Ejemplo: +18005550175; 1 es el código de país de Estados Unidos.                                            |
| Para (obligatorio)             | El número de teléfono de destino en [formato E.164](https://en.wikipedia.org/wiki/E.164).                                                                                      |
| Payload de TwiML (obligatorio) | Las instrucciones de TwiML para tu mensaje de voz.<br />Para obtener más información, consulta [la documentación de TwiML de Twilio](https://www.twilio.com/docs/voice/twiml). |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes consultar todos los códigos de estado esperados en [la documentación de Twilio](https://www.twilio.com/docs/voice/twiml).
