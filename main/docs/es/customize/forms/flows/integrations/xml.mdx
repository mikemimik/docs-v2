Esta lista de acciones XML te permite convertir objetos JSON en cadenas XML y cadenas XML en objetos JSON.

<div id="convert-json-to-xml">
  ## Convertir JSON a XML
</div>

Convierte un objeto JSON en una cadena XML.

<Frame>![](/docs/images/cdy7uua7fh8z/3GW2EhBMIX3YDPaqc3rXrZ/7706a863700df51376d645a7d9292a27/convertjsontoxml.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro          | Descripción                                         |
| ------------------ | --------------------------------------------------- |
| JSON (obligatorio) | El objeto JSON que se convertirá en una cadena XML. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción               |
| --------- | ------ | ------------------------- |
| `xml`     | String | La cadena XML convertida. |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
  "xml": "<?xml version=\"1.0\" encoding=\"UTF-8\" standalone=\"yes\"?>\n<name>John Doe</name>"
}
```

<div id="convert-xml-to-json">
  ## Convertir XML a JSON
</div>

Convierte una cadena XML en un objeto JSON.

<Frame>![](/docs/images/cdy7uua7fh8z/42fO7BelEN4bnlUj70WhV8/e53ada552f7cc05c192466db4777471a/covertxmltojson.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro         | Descripción                                        |
| ----------------- | -------------------------------------------------- |
| XML (obligatorio) | La cadena XML que se convertirá en un objeto JSON. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción                |
| --------- | ------ | -------------------------- |
| `object`  | Objeto | El objeto JSON convertido. |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
  "object": {
    "name": "John Doe"
  }
}
```
