<div id="send-message">
  ## Enviar mensaje
</div>

Envía un mensaje.

<Frame>![](/docs/images/cdy7uua7fh8z/5zZPQw5KKYwBqxbR9jF1wZ/edba085456e3536f5167ff4c21bea77c/whatsapp.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro                                      | Descripción                                                                                                                                                                                                                                  |
| ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ID del teléfono del remitente (required)       | El ID del teléfono del remitente se encuentra en la [configuración de WhatsApp de Facebook Business Manager](https://developers.facebook.com/apps). Tenga en cuenta que no es un número de teléfono.                                         |
| Número de teléfono del destinatario (required) | El número de teléfono del destinatario en [formato compatible con E.164](https://en.wikipedia.org/wiki/E.164).                                                                                                                               |
| Tipo de mensaje (required)                     | Seleccione el tipo de mensaje en el menú desplegable. Puede ser: <ul> <li>Audio</li> <li>Contactos</li> <li>Documento</li> <li>Imagen</li> <li>Interactivo</li> <li>Ubicación</li> <li>Pegatina</li> <li>Plantilla</li> <li>Texto</li> </ul> |
| Objeto de mensaje (required)                   | El contenido del objeto de mensaje variará según el tipo de mensaje seleccionado. Para obtener más información, consulte [Mensajes](https://developers.facebook.com/docs/whatsapp/cloud-api/reference/messages).                             |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes encontrar todos los códigos de estado previstos en [la documentación de Facebook](https://developers.facebook.com/docs/whatsapp/cloud-api/support/error-codes).
