<div id="http-request">
  ## HTTP Request
</div>

La Action HTTP Request te permite llamar a una API externa o a un servicio externo y procesar la respuesta.

<Frame>![](/docs/images/cdy7uua7fh8z/6eIOv07XenyaazzKixigei/2e9efd632137829a8ddb03aacd3197df/http-request-action.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro                      | Descripción                                                                                                                                                                     |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Método                         | El método HTTP. Los valores disponibles incluyen: <ul> <li>`GET`</li> <li>`POST`</li> <li>`PUT`</li> <li>`PATCH`</li> <li>`DELETE`</li> </ul>                                   |
| Tipo de contenido              | El encabezado `Content-Type`. Los valores disponibles incluyen: <ul> <li>`application/json`</li> <li>`application/x-www-form-urlenconded`</li> <li>`application/xml`</li> </ul> |
| Cuerpo                         | El cuerpo de la solicitud.                                                                                                                                                      |
| Encabezados                    | Los encabezados de la solicitud.                                                                                                                                                |
| Parámetros de URL              | Los parámetros de URL. Estos parámetros se codificarán automáticamente en la URL.                                                                                               |
| Habilitar autenticación básica | Habilita la autenticación con nombre de usuario y contraseña.                                                                                                                   |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo     | Descripción                                                  |
| --------- | -------- | ------------------------------------------------------------ |
| `headers` | Objeto   | Los encabezados de la respuesta.                             |
| `body`    | Objeto   | El cuerpo de la respuesta.                                   |
| `status`  | Número   | El código de estado de la respuesta.                         |
| `success` | Booleano | Devuelve `true` o `false` según el `status` de la respuesta. |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
  "headers": {
    ...
  },
  "status": 200,
  "body": {
    ...
  },
  "success": true
}
```
