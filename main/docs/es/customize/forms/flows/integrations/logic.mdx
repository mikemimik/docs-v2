Esta lista de acciones lógicas te permite agregar lógica condicional en formularios, almacenar variables compartidas, asignar valores y mostrar mensajes de error personalizados.

<div id="ifthen-condition">
  ## Condición If/Then
</div>

Agrega una acción condicional con ramas True / False para personalizar el comportamiento del flujo según los datos de entrada o las propiedades.

<Frame>![](/docs/images/cdy7uua7fh8z/4ehTUSu8W1LibUd1uj3x5s/00d76b15c68746ed74e44127804c9557/if-then-condition-action.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                                                                                                                                                  |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Condition | Acción condicional personalizable que activa las ramas `True` / `False` en función del resultado. Puedes usar los operadores `AND` / `OR` para anidar distintas condiciones. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo    | Descripción                                                         |
| --------- | ------- | ------------------------------------------------------------------- |
| `result`  | Boolean | Devuelve `true` o `false` en función del resultado de la condición. |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
  "result": true
}
```

<div id="store-shared-variable">
  ## Guardar variable compartida
</div>

Guarda variables en un estado privado disponible en tu formulario y en todos sus flujos.

Puedes hacer referencia a estas variables compartidas usando `{{vars.*}}`**.** Para obtener más información, consulta [Variables](/docs/es/customize/forms/variables).

<Frame>![](/docs/images/cdy7uua7fh8z/4MO9wvChhnCJOg9OAdWuEf/98182a2bd8b6cf4c4baebefb96bce121/stored-shared-variable.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro       | Descripción                                                                                                                               |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Campos de datos | Las variables que se almacenarán en un estado privado para usarlas en tu formulario y en todo su flujo. Usa pares clave/valor para añadir variables. |

<div id="show-error-message">
  ## Mostrar mensaje de error
</div>

Muestra un mensaje de error personalizado al usuario.

<Warning>
  Esta acción del flujo finaliza el Flow. Las acciones insertadas después de esta no se ejecutarán.
</Warning>

<Frame>![](/docs/images/cdy7uua7fh8z/1Kd4SyaBtAC9S8nfWFeGfh/6e42b16c0ab69dc6d026fb3601133fbe/show-error-message-flow-action.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro        | Descripción                                     |
| ---------------- | ----------------------------------------------- |
| Mensaje de error | El mensaje de error que se mostrará al usuario. |

<div id="input-value-mapping">
  ## Mapeo de valores de entrada
</div>

Asigna una variable de entrada a una salida en función de sus valores.

<Frame>![](/docs/images/cdy7uua7fh8z/4VtAr4vCSXtKlSaxtiXQhV/bcb07221f688483aa2010d80394ae630/input-value-mapping.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro                      | Descripción                                                                                                                       |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| Valor de entrada (obligatorio) | La variable de entrada.                                                                                                           |
| Tabla de mapeo                 | Asigna un valor de la variable de entrada a una salida.                                                                           |
| Valor de salida predeterminado | Establece un valor de salida predeterminado que se utilizará cuando no haya ningún valor de salida definido en la tabla de mapeo. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción                               |
| --------- | ------ | ----------------------------------------- |
| `result`  | String | Devuelve el valor asociado a una entrada. |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
	"result": "Please enter a valid email address."
}
```
