  ## Configura tu Conexión de Vault JWT
</div>

Aprende a configurar una conexión de Vault para tus Flow Actions de JSON Web Token en [Conexión de Vault JWT](/docs/es/customize/forms/vaults/jwt).

<div id="sign-json-web-token">
  ## Firmar un JSON Web Token
</div>

Genera un JSON Web Token.

<Frame>![](/docs/images/cdy7uua7fh8z/3XQqUMKrmrc8fUkCsDhI2T/43f7f8d38673d8fa4f9dba7242e044d9/sign-json-web-token.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                                                                       |
| --------- | ------------------------------------------------------------------------------------------------- |
| Payload   | Datos que se codificarán. Recomendamos darles formato conforme a los estándares de OpenID.        |
| Subject   | Identifica el sujeto del JWT.                                                                     |
| Issuer    | Identifica a la entidad que emitió el JWT.                                                        |
| Audience  | Identifica los destinatarios a los que va dirigido el JWT. Por ejemplo: admin.your&#95;domain.com |
| Expira en | Indica el tiempo de expiración, a partir del cual el JWT no debe aceptarse para su procesamiento. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción                                     |
| --------- | ------ | ----------------------------------------------- |
| `token`   | String | Una cadena de texto con formato JSON Web Token. |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
  "token": "eyJhbGciOiJIUzI1N..."
}
```

<div id="decode-json-web-token">
  ## Decodificar un JSON Web Token
</div>

Decodifica el JSON Web Token proporcionado.

<Frame>![](/docs/images/cdy7uua7fh8z/7I25WVtppllW6qhC5sALvH/70143bbe0d96920e1d9dfabbb1d6aeff/decode-json-web-token.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro           | Descripción                                    |
| ------------------- | ---------------------------------------------- |
| Token (obligatorio) | Cadena del JSON Web Token que se decodificará. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción                                           |
| --------- | ------ | ----------------------------------------------------- |
| `payload` | objeto | El contenido decodificado y válido del JSON Web Token |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
  "header": {
    "alg": "HS256",
    "typ": "JWT"
  },
  "payload": {
    "sub": "1234567890",
    "name": "John Doe",
    "iat": 1516239022
  },
  "signature": "SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c"
}
```

<div id="verify-json-web-token">
  ## Verificar el JSON Web Token
</div>

Verifica los datos del JSON Web Token para determinar si permanece intacto o si ha sido modificado, a fin de garantizar su autenticidad.

<Frame>![](/docs/images/cdy7uua7fh8z/15MDJcfZqtC46h6mRPOGcz/55e0785db066a54f836d30199fa5f295/verify-json-web-token.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro           | Descripción                                          |
| ------------------- | ---------------------------------------------------- |
| Token (obligatorio) | Cadena del JSON Web Token que se verificará.         |
| Issuer              | El emisor del JWT que se verificará.                 |
| Audience            | La audiencia destinataria del JWT que se verificará. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo    | Descripción                                                               |
| --------- | ------- | ------------------------------------------------------------------------- |
| `valid`   | Boolean | Devuelve `true` o `false` en función de si el JWT tiene una firma válida. |
| `cause`   | String  | Si la propiedad `valid` es `false`, se muestra un mensaje.                |
| `payload` | Object  | El contenido decodificado y válido del JSON Web Token.                    |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
  "valid": true,
  "header": {
    "alg": "HS256",
    "typ": "JWT"
  },
  "payload": {
    "sub": "1234567890",
    "name": "Jane Doe",
    "iat": 1516239022
  },
  "signature": "SflKxwRJSMe..."
}
```

```json lines
{
  "valid": false,
  "cause": "INVALID_SIGNATURE"
}
```
