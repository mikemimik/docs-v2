<div id="send-telegram-message">
  ## Enviar mensaje de Telegram
</div>

Envía un mensaje a un chat de Telegram

<Frame>![](/docs/images/cdy7uua7fh8z/kgSIPsWN5djsT1yQqg5Z8/fea0821d0e919c5101d01b84b968dd53/sendtelegrammessage.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro                 | Descripción                                                                                                 |
| ------------------------- | ----------------------------------------------------------------------------------------------------------- |
| ID del chat (obligatorio) | El ID del chat de Telegram. Se obtiene a través de [BotFather de Telegram](https://core.telegram.org/bots). |
| Mensaje (obligatorio)     | El contenido del cuerpo del mensaje de Telegram.                                                            |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes consultar la respuesta esperada y los códigos de estado en la [documentación de Telegram](https://core.telegram.org/method/messages.sendMessage).
