<div id="send-email">
  ## Enviar un correo electrónico
</div>

Envía un correo electrónico con SendGrid como proveedor de correo electrónico.

<Frame>![](/docs/images/cdy7uua7fh8z/3pT7MZCTVhcL9ueeke0HqQ/8583aedc27af021fe6a62430c6d59adf/sendGrid.png)</Frame>

<div id="input-settings">
  ### configuración de entrada
</div>

| Parámetro                                         | Descripción                                                                                                         |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| Nombre del remitente                              | El nombre del remitente                                                                                             |
| Correo electrónico del remitente (obligatorio)    | El correo electrónico del remitente                                                                                 |
| Nombre del destinatario                           | El nombre del destinatario                                                                                          |
| Correo electrónico del destinatario (obligatorio) | El correo electrónico del destinatario                                                                              |
| Asunto (obligatorio)                              | El asunto del correo electrónico                                                                                    |
| Tipo de correo electrónico (obligatorio)          | Envía el correo con contenido HTML o mediante una plantilla guardada (opcional)                                     |
| Mensaje (obligatorio)                             | Mensaje del correo electrónico en formato HTML o una plantilla (opcional).                                          |
| ID de plantilla (opcional)                        | Si usas una plantilla, puedes encontrar su ID en [ SendGrid dashboard ](https://mc.sendgrid.com/dynamic-templates). |
| Variables personalizadas (opcional)               | Agrega variables personalizadas para insertarlas en la plantilla.                                                   |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes consultar la respuesta esperada y los códigos de estado en [la documentación de SendGrid](https://www.twilio.com/docs/sendgrid/api-reference/mail-send/mail-send).
