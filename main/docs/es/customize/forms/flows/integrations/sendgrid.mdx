<div id="send-email">
  ## Enviar correo electrónico
</div>

Envía un correo electrónico con SendGrid como proveedor de correo electrónico.

<Frame>![](/docs/images/cdy7uua7fh8z/3pT7MZCTVhcL9ueeke0HqQ/8583aedc27af021fe6a62430c6d59adf/sendGrid.png)</Frame>

<div id="input-settings">
  ### configuración de entrada
</div>

| Parámetro                                         | Descripción                                                                                                       |
| ------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| Nombre del remitente                              | Nombre del remitente                                                                                              |
| Correo electrónico del remitente (obligatorio)    | Correo electrónico del remitente                                                                                  |
| Nombre del destinatario                           | Nombre del destinatario                                                                                           |
| Correo electrónico del destinatario (obligatorio) | Correo electrónico del destinatario                                                                               |
| Asunto (obligatorio)                              | Asunto del correo electrónico                                                                                     |
| Tipo de correo electrónico (obligatorio)          | Envíe el correo con contenido HTML o con una plantilla guardada (opcional)                                        |
| Mensaje (obligatorio)                             | Mensaje de correo electrónico en formato HTML o una plantilla (opcional).                                         |
| ID de plantilla (opcional)                        | Si usa una plantilla, puede encontrar su ID en [ SendGrid dashboard ](https://mc.sendgrid.com/dynamic-templates). |
| Variables personalizadas (opcional)               | Agregue variables personalizadas para insertarlas en la plantilla.                                                |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes consultar la respuesta esperada y los códigos de estado en [la documentación de SendGrid](https://www.twilio.com/docs/sendgrid/api-reference/mail-send/mail-send).
