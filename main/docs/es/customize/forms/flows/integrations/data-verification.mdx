Esta lista de acciones de verificación de datos te permite verificar direcciones de correo electrónico, así como generar y verificar códigos de un solo uso.

<div id="generate-one-time-password">
  ## Generar código de un solo uso
</div>

Genera un código numérico válido durante 5 minutos, que puede enviarse por correo electrónico, SMS o cualquier aplicación de mensajería.

<Frame>![](/docs/images/cdy7uua7fh8z/1h69FWzZwk9beKKjB4Cq38/16aa515e61cc6e9ebdfd0849201e3141/generate-one-time-password.png)</Frame>

<div id="input-settings">
  ### configuración de entrada
</div>

| Parámetro                         | Descripción                                                                                                             |
| --------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| Valor de referencia (obligatorio) | Puedes vincular el código generado a un número de teléfono, una dirección de correo electrónico o cualquier otro valor. |
| Longitud del código (obligatorio) | La cantidad de dígitos del código generado.                                                                             |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción        |
| --------- | ------ | ------------------ |
| `code`    | String | El código generado |

<div id="output-object-example">
  ### Ejemplo de objeto de salida
</div>

```json lines
{
  "code": "███",
}
```

<div id="verify-one-time-password">
  ## Verificar código de un solo uso
</div>

Comprueba si el código de código de un solo uso proporcionado es válido o no.

<Frame>![](/docs/images/cdy7uua7fh8z/3bkT8QDN4dcl0YHTrSOIGj/9698a07f55353a11ca98164377cd4217/verify-otp.png)</Frame>

### Configuración de entrada

| Parámetro                         | Descripción                                                                                  |
| --------------------------------- | -------------------------------------------------------------------------------------------- |
| Valor de referencia (obligatorio) | Usa el mismo valor de referencia configurado en la Action Generar código de un solo uso. |
| Longitud del código (obligatorio) | El número de dígitos del código generado.                                                    |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo    | Descripción                                                                                                                                                                                                                                                                                                              |
| --------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `valid`   | Boolean | Indica si el código proporcionado es válido o no; devuelve un valor `true` o `false`.                                                                                                                                                                                                                                    |
| `result`  | String  | Devuelve distintos códigos de resultado según el código proporcionado: <ul> <li> `MISSING:` El código proporcionado no existe. </li> <li> `EXPIRED:` El código proporcionado ha caducado. </li> <li> `VALID:` El código proporcionado es válido. </li> <li> `UNEQUAL:` El código proporcionado no es válido. </li> </ul> |

<div id="output-object-examples">
  ### Ejemplos del objeto de salida
</div>

```json lines
{
  "valid": false,
  "result": "UNEQUAL"
}
```

```json lines
{
  "valid": true
}
```

<div id="verify-email-address">
  ## Verificar la dirección de correo electrónico
</div>

Ejecuta varias comprobaciones de verificación seleccionadas en la dirección de correo electrónico proporcionada.

<Frame>![](/docs/images/cdy7uua7fh8z/6qkqny1hhObFalHwdEG342/1a2e18195b29102c3f68744f42277917/verify-email.png)</Frame>

### Configuración de entrada

| Parámetro                                              | Descripción                                                                                                                                              |
| ------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Correo electrónico (obligatorio)                       | dirección de correo electrónico que se va a verificar.                                                                                                   |
| Requerir registro MX                                   | Requiere registros MX. Cuando un dominio no tiene registros MX, los servidores de correo no saben adónde enviar los correos electrónicos de ese dominio. |
| Bloquear proveedores de correo electrónico gratuitos   | Bloquea proveedores de correo electrónico gratuitos como @gmail.com, @hotmail.com, etc.                                                                  |
| Bloquear proveedores de correo electrónico desechables | Bloquea correos electrónicos desechables como @mailnator.com, @maildrop.cc, etc.                                                                         |
| Bloquear alias de cuentas de correo electrónico        | Bloquea direcciones de correo electrónico con alias como `jane.doe+alias@gmail.com`                                                                      |
| Dominios permitidos                                    | Permite correos electrónicos solo de los dominios aprobados en tu lista.                                                                                 |
| Dominios bloqueados                                    | Bloquea dominios de correo electrónico específicos.                                                                                                      |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo    | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| --------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `valid`   | Boolean | Devuelve `true` o `false` en función de si el correo electrónico cumple o no las reglas de verificación.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `cause`   | String  | Si la propiedad `valid` es `<br/>false<br/>` , devuelve una referencia a la primera regla que no cumple el correo electrónico: <ul> <li> `MISSING_MX_RECORD:` El dominio del correo electrónico no tiene registros MX. </li> <li> `FREE_EMAIL:` El dominio del correo electrónico pertenece a un proveedor de correo electrónico gratuito.</li> <li> `DISPOSABLE_EMAIL:` El correo electrónico es una dirección de correo electrónico desechable.</li> <li> `BLOCKLISTED:` El dominio del correo electrónico está incluido en la configuración de dominios bloqueados.</li> <li> `NOT_ALLOWED:` El dominio del correo electrónico no está incluido en la configuración de dominios de la lista de permitidos.</li> </ul> |

<div id="output-object-example">
  ### Ejemplo de objeto de salida
</div>

```json lines
{
  "valid": false,
  "cause": "FREE_EMAIL"
}
```

```json lines
{
  "valid": true,
  "cause": null
}
```
