Puedes personalizar los mensajes del formulario para ofrecer a tus usuarios una experiencia única. Además, puedes traducir los componentes del formulario, los errores y otros mensajes personalizados a otros idiomas para brindar una experiencia localizada.

<div id="messages">
  ## Mensajes
</div>

En el editor de formularios, puedes personalizar los mensajes de error y crear otros mensajes personalizados para tus usuarios.

<div id="error-messages">
  ### Mensajes de error
</div>

Puede crear mensajes de error personalizados para reemplazar los mensajes de error predeterminados de sus formularios.

Para crear mensajes de error personalizados, siga estos pasos:

1. En el Auth0 Dashboard, vaya a **Actions &gt; Formularios**.
2. Seleccione **Mensajes** en el menú izquierdo del editor de formularios.
3. Seleccione la pestaña **Errors****.**
4. Introduzca sus mensajes de error personalizados en los campos correspondientes para reemplazar los mensajes predeterminados.
5. Seleccione **Publicar** para guardar.

<div id="custom-messages">
  ### Mensajes personalizados
</div>

Puedes crear mensajes personalizados para complementar los mensajes predeterminados y ofrecer una experiencia de formulario aún más personalizada. Por ejemplo, puedes añadir un mensaje personalizado que se muestre a los usuarios cuando una Action devuelva un error en un flujo.

<div id="create-custom-messages">
  #### Crear mensajes personalizados
</div>

Para crear mensajes personalizados, sigue estos pasos:

1. En el menú izquierdo del editor de formularios, selecciona **Mensajes**.
2. Selecciona la pestaña **Custom**. Luego, selecciona **Agregar nuevo elemento**.
3. A la izquierda, agrega una clave única para tu mensaje**.** Luego**, ingresa** tu mensaje personalizado en el campo de la derecha.
4. Selecciona **Publicar** para guardar.

<div id="add-custom-messages-to-your-forms">
  #### Añade mensajes personalizados a tus formularios
</div>

Para referenciar tus mensajes personalizados en tus formularios, usa la función `{{t('custom_unique_key')}}` con la clave única del mensaje personalizado.

<Frame>![Dashboard > Formularios > Mensajes personalizados y traducciones > Función de traducción](/docs/images/cdy7uua7fh8z/45p4V7w5gw7DQZyJ08ECSH/d040720d9aa19f6e97e21defcd428e55/translation-function.png)</Frame>

<div id="translations">
  ## Translations
</div>

Puedes usar la funcionalidad Translations para ofrecer una experiencia internacional en tus formularios. Esta funcionalidad usa el idioma del navegador del usuario para determinar el idioma adecuado para tus formularios.
De forma predeterminada, se proporcionan traducciones al español para algunos mensajes de error estándar. Sin embargo, para garantizar una experiencia de usuario de calidad, puedes definir las traducciones de los componentes de tu formulario, los errores y los mensajes personalizados en el idioma que elijas.

Para habilitar las traducciones, sigue estos pasos:

1. En el Auth0 Dashboard, ve a **Actions &gt; Formularios**.
2. Selecciona **Translations** en el menú izquierdo del editor de formularios.
3. Marca **Habilitar idiomas adicionales**.
4. Selecciona los idiomas deseados en el menú desplegable **Idiomas adicionales**.

<div id="translate-messages">
  ### Traducir mensajes
</div>

Con el editor de idiomas, puedes definir traducciones para los componentes de tu formulario, los errores y otros mensajes personalizados.

Para definir tus traducciones, sigue estos pasos:

1. Selecciona el idioma que quieras en el menú desplegable Select language.
2. Selecciona una de las siguientes pestañas para comenzar a definir tus traducciones:

   1. **Components**
   2. **Errors**
   3. **Custom**
3. Para cada campo disponible, añade las traducciones que necesites.
4. Selecciona **Publish** para guardar.
