Puedes personalizar los mensajes de Form para ofrecer a tus usuarios una experiencia única con Form. Además, puedes traducir componentes de formularios, errores y otros mensajes personalizados a otros idiomas para ofrecer una experiencia localizada.

<div id="messages">
  ## Mensajes
</div>

En el editor de formularios, puedes personalizar los mensajes de error y crear otros mensajes personalizados para tus usuarios.

<div id="error-messages">
  ### Mensajes de error
</div>

Puede crear mensajes de error personalizados para sustituir los mensajes de error predeterminados de sus formularios.

Para crear mensajes de error personalizados, siga estos pasos:

1. En Auth0 Dashboard, vaya a **Actions &gt; Formularios**.
2. Seleccione **Messages** en el menú izquierdo del editor de formularios.
3. Seleccione la pestaña **Errors** **.**
4. Introduzca sus mensajes de error personalizados en los campos correspondientes para sustituir los mensajes predeterminados.
5. Seleccione **Publish** para guardar.

<div id="custom-messages">
  ### Mensajes personalizados
</div>

Puedes crear mensajes personalizados para añadirlos a los mensajes predeterminados y ofrecer una experiencia de Form aún más personalizada. Por ejemplo, puedes añadir un mensaje personalizado que se muestre a los usuarios cuando una Action devuelve un error en un flujo.

<div id="create-custom-messages">
  #### Crear mensajes personalizados
</div>

Para crear mensajes personalizados, sigue estos pasos:

1. En el menú izquierdo del editor de formularios, selecciona **Messages**.
2. Selecciona la pestaña **Custom**. Luego, selecciona **Add New Item**.
3. A la izquierda, agrega una clave única para tu mensaje**.** Luego**, ingresa** tu mensaje personalizado en el campo de la derecha.
4. Selecciona **Publish** para guardar.

<div id="add-custom-messages-to-your-forms">
  #### Agrega mensajes personalizados a tus formularios
</div>

Para hacer referencia a tus mensajes personalizados en los formularios, usa la función `{{t('custom_unique_key')}}` con la clave única del mensaje personalizado.

<Frame>![Dashboard > Formularios > Mensajes personalizados y traducciones > Función de traducción](/docs/images/cdy7uua7fh8z/45p4V7w5gw7DQZyJ08ECSH/d040720d9aa19f6e97e21defcd428e55/translation-function.png)</Frame>

<div id="translations">
  ## Traducciones
</div>

Puedes usar la funcionalidad Traducciones para ofrecer una experiencia de usuario internacional en tus formularios. Esta funcionalidad utiliza el idioma del navegador del usuario para determinar el idioma adecuado para tus formularios.
De forma predeterminada, se proporcionan en español algunas traducciones de mensajes de error estándar. Sin embargo, para ofrecer una experiencia de usuario de calidad, puedes definir las traducciones de los componentes de tu formulario, los errores y los mensajes personalizados en el idioma que elijas.

Para habilitar las traducciones, sigue estos pasos:

1. En el Auth0 Dashboard, ve a **Actions &gt; Formularios**.
2. Selecciona **Traducciones** en el menú izquierdo del editor de formularios.
3. Marca **Habilitar idiomas adicionales**.
4. Selecciona los idiomas que quieras en el menú desplegable **Idiomas adicionales**.

<div id="translate-messages">
  ### Traducir mensajes
</div>

Con el editor de idiomas, puedes definir traducciones para los componentes de tu formulario, errores y otros mensajes personalizados.

Para definir tus traducciones, sigue estos pasos:

1. Selecciona el idioma deseado en el menú desplegable Select language.
2. Selecciona una de las siguientes pestañas para empezar a definir tus traducciones:

   1. **Components**
   2. **Errors**
   3. **Custom**
3. Para cada campo disponible, proporciona las traducciones que necesites.
4. Selecciona **Publish** para guardar.
