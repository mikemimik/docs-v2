---

Puedes personalizar los mensajes de Forms para ofrecer a tus usuarios una experiencia única. Además, puedes traducir los componentes del formulario, los errores y otros mensajes personalizados a otros idiomas para brindar una experiencia localizada.

<div id="messages">
  ## Mensajes
</div>

Con el editor de formularios, puedes personalizar los mensajes de error y crear otros mensajes personalizados para tus usuarios.

<div id="error-messages">
  ### Mensajes de error
</div>

Puedes crear mensajes de error personalizados para sustituir los mensajes de error predeterminados de tus formularios.

Para crear mensajes de error personalizados, sigue estos pasos:

1. En el Auth0 Dashboard, ve a **Actions &gt; Forms**.
2. Selecciona **Messages** en el menú izquierdo del editor de formularios.
3. Selecciona la pestaña **Errors**.**
4. Introduce tus mensajes de error personalizados en los campos correspondientes para sustituir los mensajes predeterminados.
5. Selecciona **Publish** para guardar.

<div id="custom-messages">
  ### Mensajes personalizados
</div>

Puedes crear mensajes personalizados para complementar los mensajes predeterminados y ofrecer una experiencia con Form aún más personalizada. Por ejemplo, puedes añadir un mensaje personalizado para mostrarlo a los usuarios cuando una acción devuelva un error en un flujo.

<div id="create-custom-messages">
  #### Crear mensajes personalizados
</div>

Para crear mensajes personalizados, sigue estos pasos:

1. En el menú izquierdo del editor de formularios, selecciona **Messages**.
2. Selecciona la pestaña **Custom**. Luego, selecciona **Add New Item**.
3. A la izquierda, agrega una clave única para tu mensaje**.** Luego, **ingresa** tu mensaje personalizado en el campo de la derecha.
4. Selecciona **Publish** para guardar.

<div id="add-custom-messages-to-your-forms">
  #### Añade mensajes personalizados a tus formularios
</div>

Para hacer referencia a tus mensajes personalizados en tus formularios, usa la función `{{t('custom_unique_key')}}` con la clave única del mensaje personalizado.

<Frame>![Dashboard > Forms > Mensajes personalizados y traducciones > Función de traducción](/docs/images/cdy7uua7fh8z/45p4V7w5gw7DQZyJ08ECSH/d040720d9aa19f6e97e21defcd428e55/translation-function.png)</Frame>

<div id="translations">
  ## Traducciones
</div>

Puede usar la función Translations para crear una experiencia de usuario internacional en sus formularios. Esta función utiliza el idioma del navegador del usuario para determinar el idioma adecuado para sus formularios.
De forma predeterminada, se proporcionan en español las traducciones de algunos mensajes de error estándar. Sin embargo, para ofrecer una experiencia de usuario de calidad, puede definir las traducciones de los componentes de su formulario, los errores y los mensajes personalizados en el idioma que elija.

Para habilitar las traducciones, siga los pasos a continuación:

1. En el Auth0 Dashboard, vaya a **Actions &gt; Forms**.
2. Seleccione **Translations** en el menú izquierdo del editor de formularios.
3. Marque **Enable additional languages**.
4. Seleccione los idiomas que desee en el menú desplegable **Additional languages**.

<div id="translate-messages">
  ### Traducir mensajes
</div>

Con el editor de idiomas, puedes definir traducciones para los componentes del formulario, los errores y otros mensajes personalizados.

Para definir tus traducciones, sigue estos pasos:

1. Selecciona el idioma deseado en el menú desplegable **Select language**.
2. Selecciona una de las siguientes pestañas para empezar a definir tus traducciones:

   1. **Componentes**
   2. **Errors**
   3. **Custom**
3. Para cada campo disponible, añade las traducciones que necesites.
4. Selecciona **Publish** para guardar.
