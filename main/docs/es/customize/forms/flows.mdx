Flows te permite crear visualmente la lógica Server-side que puede ejecutarse dentro de tus formularios para procesar la información recopilada, integrar cualquier servicio con una API o activar automatizaciones.

<Frame>![Dashboard > Forms > Flows](/docs/images/cdy7uua7fh8z/6u6Huh5CbH3GEQJv3AH7cx/e49ea613b2c46a43c83c89ffecddbafe/Flows.png)</Frame>

Puedes usar las Actions de Flows con tu formulario para:

* Crear validaciones personalizadas Server-side para los pasos de tu formulario.
* Integrar tu propia API mediante solicitudes HTTP personalizadas.
* Integrar servicios de terceros con nuestras integraciones listas para usar.
* Ejecutar distinta lógica de negocio en función de la información ingresada por el usuario o de las propiedades.
* Validar datos usando <Tooltip tip="JSON Web Token (JWT): formato estándar del token de ID (y, a menudo, del token de acceso) utilizado para representar claims de forma segura entre dos partes." cta="Ver glosario" href="/docs/es/glossary?term=JSON+Web+Tokens">JSON Web Tokens</Tooltip>.
