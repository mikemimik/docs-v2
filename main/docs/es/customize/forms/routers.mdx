title: Routers

Con los nodos Router, puedes agregar reglas a Forms para crear saltos de lógica condicional que redirijan a los usuarios entre nodos. Por ejemplo, puedes definir una condición de regla para un atributo específico de `user_metadata` a fin de dirigir a un usuario a un nodo de paso o de flujo determinado.

* El nodo Router agrega automáticamente una regla de caso predeterminado. Si no se cumple ninguna condición de regla, el recorrido continúa hacia el nodo vinculado.
* Las reglas se evalúan en el orden en que aparecen en el router. Cuando se cumple una condición de regla, el recorrido continúa hacia el nodo vinculado y no se evalúa ninguna otra condición.
* Puedes agregar hasta 10 condiciones de regla al nodo Router.

<div id="add-a-router-rule">
  ## Agregar una regla del router
</div>

Para agregar una condición de regla, sigue estos pasos:

1. En el editor de formularios, selecciona un nodo Router para abrir su configuración.
2. Selecciona **+ New Rule**.
3. Expande la regla recién creada.
4. Para identificar la regla, agrega un **Alias**.
5. Agrega una variable con el ícono **+**.
6. Selecciona el operador de la regla para la variable.
7. Agrega el valor o la variable que debe cumplir la regla.
8. Vincula la nueva regla con el paso o nodo de flujo correspondiente.
9. Selecciona **Publish** para guardar.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * Te recomendamos definir un alias para identificar fácilmente cada regla en el nodo Router.
  * Puedes elegir entre varios operadores de regla para evaluar operandos. Por ejemplo, puedes configurar first&#95;name `is equal to` Jane.
  * Los operandos pueden ser valores estáticos o variables.
  * Una regla puede incluir operadores `AND` y `OR` para evaluar lógica condicional adicional.
  * Después de agregar una regla, asegúrate de vincularla al nodo deseado.
</Callout>

<div id="available-rule-operators">
  ## Operadores de regla disponibles
</div>

Los operadores de regla son:

* `es mayor que`
* `es mayor o igual que`
* `es menor que`
* `es menor o igual que`
* `es igual a`
* `es diferente de`
* `incluye`
* `no incluye`
* `empieza por`
* `termina en`
* `existe`
* `no existe`
* `es nulo`
* `no es nulo`
* `no tiene valor`
* `es verdadero`
* `es falso`
