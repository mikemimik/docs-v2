<Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip> consiste en un conjunto de páginas que facilitan varias acciones relacionadas con la cuenta, como iniciar sesión, registrar factores de <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> o restablecer contraseñas. Auth0 proporciona el texto de estas páginas en varios idiomas.

En algunos casos, es posible que quieras modificar el texto de estas páginas para adaptarlo mejor al tono o a las necesidades específicas de tu aplicación.

Puedes personalizar el texto que se muestra en las páginas de [Universal Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) en todos los idiomas admitidos, ya sea desde el <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o mediante la <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Las siguientes definiciones, valores y variables se aplican a ambos métodos.

<div id="configure-prompts-and-screens">
  ## Configurar prompts y pantallas
</div>

Puedes personalizar cualquiera de los [prompts y pantallas](#prompt-values) disponibles mediante el Auth0 Dashboard o la [Management API](https://auth0.com/docs/api/management/v2).

<Tabs>
  <Tab title="Usa el Dashboard">
    Puedes usar el Dashboard para personalizar los prompts de texto de New Universal Login.

    1. En el Dashboard, ve a [**Branding &gt; Universal Login &gt; Advanced Options**](https://manage.auth0.com/#/custom_text) y luego selecciona **Custom Text**.
    2. Selecciona el **Prompt** y la **Screen** que quieres editar, y haz los cambios que desees. Hay una vista previa en tiempo real a la derecha.
    3. Cuando tus cambios estén listos para aparecer en tu entorno de Production, selecciona **Save Changes.**
  </Tab>

  <Tab title="Usa la Management API">
    Puedes usar la Management API para personalizar los prompts de texto del nuevo Universal Login.

    1. Obtén un [token de acceso de la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens/get-management-api-access-tokens-for-production) con los scopes `read:prompts` y `update:prompts`.
    2. Llama al endpoint de la Management API [Get custom text for a prompt](https://auth0.com/docs/api/management/v2#!/Prompts/get_custom_text_by_language) para obtener los datos de configuración existentes del prompt y del idioma especificados.
    3. Modifica los datos de configuración como desees, usando como referencia los [parámetros admitidos](#supported-parameters) y los [valores del prompt](#prompt-values).
    4. Llama al endpoint de la Management API [Establecer texto personalizado para un prompt específico](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) con la configuración actualizada.

       <Warning>
         El endpoint de la Management API [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) reemplaza toda la configuración existente del prompt especificado por la configuración proporcionada en el parámetro `body`.

         Si quieres configurar varios valores de `screen` para un prompt determinado, debes incluirlos todos en una sola llamada a la Management API.
       </Warning>

    #### Ejemplo

    Puedes definir el componente de texto `description` para la pantalla `login` del prompt `login` con el valor `Log in to ACME's website` llamando al endpoint [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) de la Management API de la siguiente manera:

    <AuthCodeGroup>
      ```bash cURL
      curl --request PUT \
        --url 'https://{yourDomain}/api/v2/prompts/login/custom-text/en' \
        --header 'authorization: Bearer {yourMgmtApiAccessToken}' \
        --header 'content-type: application/json' \
        --data '{ "login": { "description": "Login to ACME'''s Website" } }'
      ```

      ```csharp C#
      var client = new RestClient("https://{yourDomain}/api/v2/prompts/login/custom-text/en");
      var request = new RestRequest(Method.PUT);
      request.AddHeader("content-type", "application/json");
      request.AddHeader("authorization", "Bearer {yourMgmtApiAccessToken}");
      request.AddParameter("application/json", "{ "login": { "description": "Login to ACME's Website" } }", ParameterType.RequestBody);
      IRestResponse response = client.Execute(request);
      ```

      ```go Go
      package main

      import (
      	"fmt"
      	"strings"
      	"net/http"
      	"io/ioutil"
      )

      func main() {

      	url := "https://{yourDomain}/api/v2/prompts/login/custom-text/en"

      	payload := strings.NewReader("{ "login": { "description": "Login to ACME's Website" } }")

      	req, _ := http.NewRequest("PUT", url, payload)

      	req.Header.Add("content-type", "application/json")
      	req.Header.Add("authorization", "Bearer {yourMgmtApiAccessToken}")

      	res, _ := http.DefaultClient.Do(req)

      	defer res.Body.Close()
      	body, _ := ioutil.ReadAll(res.Body)

      	fmt.Println(res)
      	fmt.Println(string(body))

      }
      ```


      ```php PHP
          }
      ```


<div id="supported-parameters">
  ### Parámetros compatibles
</div>

| Parámetro  | Definición                                                                                                                                                                                                                             |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `prompt`   | El paso del flujo de inicio de sesión. Para ver una lista de valores disponibles, consulta [Valores de prompt](#prompt-values).                                                                                                        |
| `language` | El idioma al que se aplica la personalización. Para ver una lista de idiomas disponibles, consulta [Internacionalización de Universal Login](/docs/es/customize/internationalization-and-localization/universal-login-internationalization). |
| `screen`   | La pantalla específica dentro de un prompt.                                                                                                                                                                                            |
| `key`      | El identificador de un componente de texto.                                                                                                                                                                                            |
| `text`     | El mensaje que se muestra al usuario.                                                                                                                                                                                                  |

<div id="prompt-values">
  ### Valores de `prompt`
</div>

Los valores de `prompt` disponibles son:

<div id="variables">
  ## Variables
</div>

Algunas pantallas tienen variables en el texto que se sustituyen en tiempo de ejecución según la información contextual. Como hay distintas variables disponibles para cada pantalla, no se garantiza que una variable determinada funcione fuera de su pantalla original.

| Variable                      | Descripción                                                                                                   |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------- |
| $&#123;clientName&#125;       | Nombre de la aplicación de Auth0                                                                              |
| $&#123;connectionName&#125;   | Nombre de la conexión (p. ej., Google)                                                                        |
| $&#123;tenantName&#125;       | Nombre del tenant de Auth0                                                                                    |
| $&#123;organizationName&#125; | Nombre de la organización del usuario si este se autentica en el contexto de una organización                 |
| $&#123;companyName&#125;      | Nombre del tenant de Auth0 o de la organización si el usuario se autentica en el contexto de una organización |
| $&#123;userName&#125;         | Nombre del usuario que inició sesión                                                                          |
| $&#123;email&#125;            | Correo electrónico del usuario que inició sesión                                                              |
| $&#123;appName&#125;          | Nombre de la aplicación personalizada de Guardian Push                                                        |
