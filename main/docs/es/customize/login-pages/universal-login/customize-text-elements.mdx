<Tooltip tip="Universal Login: Su aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip> consiste en un conjunto de páginas que facilitan varias acciones relacionadas con la cuenta, como iniciar sesión, registrar factores de <Tooltip tip="Universal Login: Su aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> o restablecer contraseñas. Auth0 proporciona el texto de estas páginas en varios idiomas.

En algunos casos, es posible que desee modificar el texto de estas páginas para adaptarlo mejor al tono o a las necesidades específicas de su aplicación.

Puede personalizar el texto que se muestra en las páginas de [Universal Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) en todos los idiomas compatibles a través del <Tooltip tip="Auth0 Dashboard: El principal producto de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o de la <Tooltip tip="Auth0 Dashboard: El principal producto de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>. Las siguientes definiciones, valores y variables se aplican a ambos métodos.

<div id="configure-prompts-and-screens">
  ## Configurar prompts y pantallas
</div>

Puedes personalizar cualquiera de los [prompts y pantallas](#prompt-values) disponibles desde el Auth0 Dashboard o la [Management API](https://auth0.com/docs/api/management/v2).

<Tabs>
  <Tab title="Usa el Dashboard">
    Puedes usar el Dashboard para personalizar los textos de los prompts de New Universal Login.

    1. En el Dashboard, ve a [**Branding &gt; Universal Login &gt; Advanced Options**](https://manage.auth0.com/#/custom_text) y, a continuación, selecciona **Custom Text**.
    2. Selecciona el **Prompt** y la **Screen** que quieras editar y haz los cambios que desees. A la derecha hay una vista previa en tiempo real.
    3. Cuando quieras que los cambios aparezcan en tu entorno de Production, selecciona **Guardar cambios.**
  </Tab>

  <Tab title="Usa Management API">
    Puedes usar la Management API para personalizar los prompts de texto del nuevo Universal Login.

    1. Obtén un [token de acceso de la Management API](/docs/es/secure/tokens/access-tokens/management-api-access-tokens/get-management-api-access-tokens-for-production) con los scopes `read:prompts` y `update:prompts`.
    2. Llama al endpoint de la Management API [Obtener texto personalizado para un prompt](https://auth0.com/docs/api/management/v2#!/Prompts/get_custom_text_by_language) para obtener los datos de configuración existentes del prompt y el idioma especificados.
    3. Modifica los datos de configuración como desees, usando los [parámetros admitidos](#supported-parameters) y los [valores de prompt](#prompt-values) como referencia.
    4. Llame al endpoint de la Management API [Establezca texto personalizado para un prompt específico](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) con los datos de configuración actualizados.

       <Warning>
         El endpoint de la Management API [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) sustituye todos los datos de configuración existentes del prompt especificado por los datos de configuración proporcionados en el parámetro `body`.

         Si quieres configurar varios valores de `screen` para un prompt determinado, debes incluirlos todos en una sola llamada a la Management API.
       </Warning>

    #### Ejemplo

    Puedes establecer el componente de texto `description` de la pantalla `login` del prompt `login` con el valor `Log in to ACME's website` llamando al endpoint [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) de la Management API de la siguiente manera:

    <AuthCodeGroup>
      ```bash cURL
      curl --request PUT \
        --url 'https://{yourDomain}/api/v2/prompts/login/custom-text/en' \
        --header 'authorization: Bearer {yourMgmtApiAccessToken}' \
        --header 'content-type: application/json' \
        --data '{ "login": { "description": "Login to ACME'''s Website" } }'
      ```

      ```csharp C#
      var client = new RestClient("https://{yourDomain}/api/v2/prompts/login/custom-text/en");
      var request = new RestRequest(Method.PUT);
      request.AddHeader("content-type", "application/json");
      request.AddHeader("authorization", "Bearer {yourMgmtApiAccessToken}");
      request.AddParameter("application/json", "{ "login": { "description": "Login to ACME's Website" } }", ParameterType.RequestBody);
      IRestResponse response = client.Execute(request);
      ```

      ```go Go
      package main

      import (
      	"fmt"
      	"strings"
      	"net/http"
      	"io/ioutil"
      )

      func main() {

      	url := "https://{yourDomain}/api/v2/prompts/login/custom-text/en"

      	payload := strings.NewReader("{ "login": { "description": "Login to ACME's Website" } }")

      	req, _ := http.NewRequest("PUT", url, payload)

      	req.Header.Add("content-type", "application/json")
      	req.Header.Add("authorization", "Bearer {yourMgmtApiAccessToken}")

      	res, _ := http.DefaultClient.Do(req)

      	defer res.Body.Close()
      	body, _ := ioutil.ReadAll(res.Body)

      	fmt.Println(res)
      	fmt.Println(string(body))

      }
      ```



<div id="supported-parameters">
  ### Parámetros admitidos
</div>

| Parámetro  | Definición                                                                                                                                                                                                                             |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `prompt`   | El paso del flujo de inicio de sesión. Para ver una lista de valores disponibles, consulta [Valores de prompt](#prompt-values).                                                                                                        |
| `language` | El idioma al que se aplica la personalización. Para ver una lista de idiomas disponibles, consulta [Internacionalización de Universal Login](/docs/es/customize/internationalization-and-localization/universal-login-internationalization). |
| `screen`   | La pantalla específica de un prompt.                                                                                                                                                                                                   |
| `key`      | El identificador de un componente de texto.                                                                                                                                                                                            |
| `text`     | El mensaje que se muestra al usuario.                                                                                                                                                                                                  |

<div id="prompt-values">

<div id="variables">
  ## Variables
</div>

Algunas pantallas incluyen variables en el texto que se sustituyen en el runtime según la información de contexto. Como las variables disponibles varían según la pantalla, no se garantiza que una variable determinada funcione en otra pantalla distinta de aquella para la que fue definida.

| Variable                      | Descripción                                                                                                   |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------- |
| $&#123;clientName&#125;       | Nombre de la aplicación de Auth0                                                                              |
| $&#123;connectionName&#125;   | Nombre de la conexión (p. ej., Google)                                                                        |
| $&#123;tenantName&#125;       | Nombre del tenant de Auth0                                                                                    |
| $&#123;organizationName&#125; | Nombre de la organización del usuario si este se autentica en el contexto de una organización                 |
| $&#123;companyName&#125;      | Nombre del tenant de Auth0 o de la organización si el usuario se autentica en el contexto de una organización |
| $&#123;userName&#125;         | Nombre del usuario que ha iniciado sesión                                                                     |
| $&#123;email&#125;            | Correo electrónico del usuario que ha iniciado sesión                                                         |
| $&#123;appName&#125;          | Nombre de la aplicación Push personalizada de Guardian                                                        |
