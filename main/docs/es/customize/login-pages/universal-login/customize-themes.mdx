title: Personaliza los temas de las páginas de Universal Login
---

Puedes crear una experiencia de login coherente con tu marca para tus usuarios al personalizar el estilo de los prompts de <Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip>. Con el editor no-code de Universal Login, puedes personalizar fácilmente los colores, las fuentes, los bordes y los fondos de los prompts que se muestran a los usuarios a lo largo del flujo de login.

El editor no-code te permite personalizar el tema de marca de Universal Login. Para personalizar el texto incluido en cada prompt, consulta [Personalizar los elementos de texto de Universal Login](/docs/es/customize/login-pages/universal-login/customize-text-elements). Si necesitas más opciones de personalización, consulta [Personalizar las plantillas de página de Universal Login](/docs/es/customize/login-pages/universal-login/customize-templates).

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Para usar el editor no-code, debes tener habilitado [Universal Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience). La experiencia de Classic Login no es compatible con este editor.
</Callout>

<div id="universal-login-no-code-editor">
  ## Editor No-Code de Universal Login
</div>

Para personalizar tu tema de Universal Login, sigue estos pasos:

1. En el Auth0 Dashboard, ve a [Branding &gt; Universal Login &gt; Customization Options](https://manage.auth0.com/#/universal-login/customizations/colors).
2. Selecciona un [componente del tema](#theme-components) (por ejemplo, Colors) en el menú Styles. Luego, personaliza las opciones que aparecen a la derecha.
3. Después de personalizar tu tema, selecciona **Save and Publish**.

Después de guardar tu tema, puedes acceder a una vista previa en vivo de tu experiencia de login para revisar y probar tus pantallas. Para abrir la vista previa en una pestaña nueva, selecciona **Try** encima del editor No-Code.

<div id="theme-components">
  ### Componentes del tema
</div>

Para crear el tema de Universal Login que quieras, el menú Styles incluye los siguientes componentes:

<AccordionGroup>
  <Accordion title="Color">
    Selecciona un campo de entrada para cambiar el color del elemento. Para elegir colores específicos, puedes:

    * Usar el selector de color.
    * Introducir el código hex o el valor RGB.
    * Seleccionar la herramienta cuentagotas para elegir un color de tu pantalla para los elementos disponibles.

    <Frame>![](/docs/images/cdy7uua7fh8z/4a2iEFD4dPvxHHFueQ7rnD/ce09850794e10176aa489aaff03e67b8/2022-05-12_11-24-33.png)</Frame>

    | Elemento                                      | Descripción                                                                                                                                                    |
    | --------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | Botón principal                               | Rellena el botón principal, o el botón que activa la siguiente acción, con el color elegido.                                                                   |
    | Etiqueta del botón principal                  | Cambia el color del texto dentro del botón principal.                                                                                                          |
    | Borde del botón secundario                    | Cambia el borde de los campos de entrada.                                                                                                                      |
    | Etiqueta del botón secundario                 | Cambia el texto dentro de los campos secundarios en los que se puede hacer clic.                                                                               |
    | Enlaces y componentes con foco                | Cambia el color del texto de los enlaces que llevan a los usuarios a activar otro workflow, como Restablecer contraseña o probar otro método de autenticación. |
    | Color base de foco                            | Cambia el color de los botones cuando el cursor pasa sobre el botón antes de seleccionarlo.                                                                    |
    | Color base al pasar el cursor                 | Cambia el color de un botón cuando el usuario hace clic en él.                                                                                                 |
    | Encabezado                                    | Cambia el color del texto del título en el encabezado.                                                                                                         |
    | Texto del cuerpo                              | Cambia el color del texto del cuerpo.                                                                                                                          |
    | Fondo del widget                              | Cambia el color de fondo de los widgets.                                                                                                                       |
    | Borde del widget                              | Cambia el color del borde de los widgets.                                                                                                                      |
    | Etiquetas y marcadores de posición de entrada | Cambia el texto de las etiquetas de los campos de entrada y del texto del marcador de posición.                                                                |
    | Texto de entrada                              | Cambia el color del texto que los usuarios escriben en los campos de entrada. Por ejemplo, los campos de nombre de usuario y contraseña.                       |
    | Borde de la entrada                           | Cambia el color del borde de los campos de entrada.                                                                                                            |
    | Fondo de la entrada                           | Cambia el color de fondo de los campos de entrada.                                                                                                             |
    | Iconos                                        | Cambia el color de los iconos dentro de los campos de entrada.                                                                                                 |
    | Error                                         | Cambia el color de los mensajes de error dentro de los widgets.                                                                                                |
    | Éxito                                         | Cambia el color del mensaje de éxito dentro de los widgets.                                                                                                    |

    Una vez que hagas un cambio, la ventana de Preview muestra varios ejemplos de los widgets con los cambios especificados.

    <Frame>![](/docs/images/cdy7uua7fh8z/4BLBu39lD1SLi7lbjwpf5n/655a46aa8fed8415e7eb4595c09a1ea8/2022-05-12_11-23-39.png)</Frame>
  </Accordion>

  <Accordion title="Fuentes">
    Para cambiar la fuente predeterminada, introduce la URL de cualquier archivo WOFF (Web Open Font Format) en el campo de entrada. Tus archivos WOFF deben estar alojados en una CDN con CORS habilitado o en un sitio de alojamiento con el encabezado Access-Control-Allow-Origin.

    Debajo del panel de URL de la fuente, ajusta el tamaño general del texto con la opción en píxeles **Reference text size**. Ten en cuenta que el resto de las opciones no cambia cuando se modifica el texto de referencia.

    Cambia los siguientes elementos de forma individual:

    * Título
    * Subtítulo
    * Texto del cuerpo
    * Texto del botón
    * Etiquetas de los campos de entrada
    * Enlaces

    Cuando seleccionas una opción para configurar, el panel de Preview muestra la ubicación del elemento en el widget y se ajusta al tamaño especificado.

    <Frame>![](/docs/images/cdy7uua7fh8z/1dHa6nIrEGy2hGV4LyKXmE/13efef14d4725c524f76e84f06444b2b/2022-05-12_11-18-38.png)</Frame>
  </Accordion>

  <Accordion title="Bordes">
    Para personalizar los bordes, los estilos de los botones, los campos de entrada y las esquinas del widget, ajusta el control deslizante o elige una de las opciones disponibles.

    | Opción                           | Descripción                                                                                                                                                                  |
    | -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | Grosor del borde del botón       | Ajusta el ancho del borde alrededor de los campos de entrada dentro de los prompts de inicio de sesión.                                                                      |
    | Estilo de los botones            | Cambia la forma de los botones interactivos dentro de los widgets. Las opciones disponibles son: esquinas rectas, esquinas redondeadas o estilo de píldora.                  |
    | Radio del borde del botón        | Ajusta las esquinas de los botones interactivos cuando seleccionas esquinas redondeadas.                                                                                     |
    | Grosor del borde de entrada      | Ajusta el ancho de los bordes de los campos de entrada y los botones interactivos dentro de los prompts de inicio de sesión.                                                 |
    | Estilo de las entradas           | Cambia la forma de los campos de entrada dentro de los prompts de inicio de sesión. Las opciones disponibles son: esquinas rectas, esquinas redondeadas o estilo de píldora. |
    | Radio del borde de entrada       | Ajusta las esquinas de los campos de entrada cuando seleccionas esquinas redondeadas.                                                                                        |
    | Radio de las esquinas del widget | Cambia la forma del widget de esquinas rectas a esquinas redondeadas.                                                                                                        |
    | Grosor del borde del widget      | Ajusta el ancho del borde del widget.                                                                                                                                        |
    | Sombra                           | Casilla de verificación para permitir que los widgets tengan sombra.                                                                                                         |

    <Frame>![](/docs/images/cdy7uua7fh8z/19TZeRWGWrYQWQ8ebRELXz/76ccd288acdec4ceb96d6c8402d48518/2022-05-10_13-44-25.png)</Frame>
  </Accordion>

  <Accordion title="Widget">
    Agrega la URL de tu logotipo para configurar la posición, la altura, la alineación del texto y la ubicación del inicio de sesión social.

    | Elemento                            | Descripción                                                                                                                            |
    | ----------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
    | Posición del logotipo               | Elige la ubicación del widget: a la izquierda, a la derecha o en el centro de la página. También puedes optar por ocultar el logotipo. |
    | URL del logotipo                    | Ingresa la URL de tu logotipo personalizado. Auth0 recomienda archivos SVG.                                                            |
    | Altura del logotipo                 | Ajusta el tamaño de tu logotipo en la parte superior del widget.                                                                       |
    | Alineación del texto del encabezado | Elige si el texto dentro del encabezado debe alinearse a la izquierda, a la derecha o al centro.                                       |
    | Diseño de los botones sociales      | Elige si el texto debe estar en la parte superior o inferior del botón.                                                                |

    <Frame>![](/docs/images/cdy7uua7fh8z/1AJksBqHefFzOmt31XlqxP/b0ac004a03681370d724b4c0d06a8351/2022-05-31_09-39-26.png)</Frame>
  </Accordion>

  <Accordion title="Fondos de página">
    Agrega una URL de imagen de fondo para elegir el fondo que se mostrará en el widget del prompt. El Editor No-Code admite otros formatos de imagen, como PNG y JPEG.

    <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
      Auth0 recomienda usar una imagen en formato JPEG de al menos 2000 px de ancho.
    </Callout>

    Luego, elige si el widget se alinea al centro, a la izquierda o a la derecha de la pantalla.

    <Frame>![](/docs/images/cdy7uua7fh8z/6WQqdGCv972KnyKMwoYeyO/61dc83d2ce507e92bde7f41df044d79b/2022-05-31_09-41-21.png)</Frame>
  </Accordion>
</AccordionGroup>

<div id="management-api">
  ## Management API
</div>

También puedes personalizar el tema de Universal Login con la [Management API](https://auth0.com/docs/api/management/v2). En concreto, puedes usar los endpoints de Branding para:

* [Crear un tema de marca nuevo](https://auth0.com/docs/api/management/v2/branding/post-branding-theme)
* [Obtener el tema de marca predeterminado](https://auth0.com/docs/api/management/v2/branding/get-default-branding-theme)
* [Obtener un tema de marca específico](https://auth0.com/docs/api/management/v2/branding/get-branding-theme)
* [Eliminar un tema de marca](https://auth0.com/docs/api/management/v2/branding/delete-branding-theme)
* [Actualizar el tema actual](https://auth0.com/docs/api/management/v2/branding/patch-branding-theme)
