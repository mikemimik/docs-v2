title: Casos de uso de ACUL

ACUL es compatible con varios entornos tecnológicos para personalizar por completo la experiencia de inicio de sesión de sus usuarios finales y adaptarse a numerosas necesidades empresariales mediante una personalización avanzada.
Use los siguientes escenarios para personalizar los [temas](/docs/es/customize/login-pages/universal-login/customize-themes), las [plantillas](/docs/es/customize/login-pages/universal-login/customize-templates) y los [elementos de texto](/docs/es/customize/login-pages/universal-login/customize-text-elements) de Universal Login. Para obtener más información sobre la experiencia de inicio de sesión alojada de Auth0, consulte [Auth0 Universal Login](/docs/es/authenticate/login/auth0-universal-login).
