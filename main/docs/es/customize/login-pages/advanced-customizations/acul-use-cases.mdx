title: Casos de uso de ACUL

ACUL es compatible con varios entornos tecnológicos para personalizar por completo la experiencia de inicio de sesión de tus usuarios finales y adaptarse a numerosas necesidades empresariales mediante una personalización avanzada.
Usa los siguientes escenarios para personalizar los [temas](/docs/es/customize/login-pages/universal-login/customize-themes), las [plantillas](/docs/es/customize/login-pages/universal-login/customize-templates) y los [elementos de texto](/docs/es/customize/login-pages/universal-login/customize-text-elements) de tu Universal Login. Para obtener más información sobre el inicio de sesión alojado de Auth0, consulta [Auth0 Universal Login](/docs/es/authenticate/login/auth0-universal-login).
