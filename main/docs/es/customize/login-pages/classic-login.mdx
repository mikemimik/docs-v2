<Warning>
  En este momento, los esfuerzos de desarrollo activo de Auth0 se centran en Universal Login, y Classic Login ya no recibe actualizaciones. Para personalizar tus páginas de inicio de sesión, Auth0 recomienda [Advanced Customizations for Universal Login](/docs/es/customize/login-pages/advanced-customizations).
</Warning>

Classic Login es una experiencia de inicio de sesión alojada por Auth0 que depende de JavaScript para su personalización. Implementar Classic Login es menos complejo que integrar directamente el proceso de autenticación en tu aplicación, y puede ayudarte a prevenir los riesgos de la autenticación de origen cruzado.

Classic Login sirve como alternativa a [Universal Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience). Para ver una comparación de funcionalidades entre las dos experiencias de inicio de sesión, consulta [Universal Login vs. Classic Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login).
