description: Aprende a personalizar la página de inicio de sesión para la experiencia de Classic Login.
title: Personalizar las páginas de Classic Login
---

<Warning>
  En este momento, los esfuerzos de desarrollo activo de Auth0 se centran en Universal Login, y Classic Login ya no recibe actualizaciones. Para personalizar tus páginas de inicio de sesión, Auth0 recomienda [Advanced Customizations for Universal Login](/docs/es/customize/login-pages/advanced-customizations).
</Warning>

[Classic Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/classic-experience) es una experiencia de inicio de sesión alojada por Auth0 que se basa en JavaScript para la personalización de páginas.

Desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, puedes configurar opciones básicas de personalización que se aplican a todas las páginas de inicio de sesión. Para una personalización avanzada, puedes modificar directamente la plantilla HTML de una o varias páginas.

Además, puedes gestionar el contenido de tu página mediante el [sistema de control de versiones](/docs/es/customize/login-pages/classic-login/version-control) que prefieras.

<div id="basic-customization">
  ## Personalización básica
</div>

Puede configurar las siguientes opciones desde el Auth0 Dashboard en [Branding &gt; Universal Login &gt; pestaña Settings](https://manage.auth0.com/#/login_settings):

* Logotipo de la empresa (tamaño recomendado: 150 x 150 píxeles)
* Color principal
* Color de fondo

De forma predeterminada, estos ajustes afectan a todas las páginas de Classic Login, como la pantalla de inicio de sesión o la página de restablecimiento de contraseña. Si modifica directamente la plantilla HTML de una o varias páginas y edita cualquier atributo relacionado con estas opciones, esos atributos prevalecerán sobre estos ajustes.

<div id="advanced-customization">
  ## Personalización avanzada
</div>

Para una personalización aún mayor, puedes modificar directamente la plantilla HTML de las siguientes páginas:

* Login (consulta más abajo)
* [Restablecimiento de contraseña](/docs/es/customize/login-pages/classic-login/customize-password-reset-page)
* [autenticación multifactor](/docs/es/secure/multi-factor-authentication/customize-mfa/customize-mfa-classic-login)

<div id="responsibility-for-updates">
  ### Responsabilidad de las actualizaciones
</div>

**Cuando habilitas la personalización para una página específica, asumes la responsabilidad de mantener esa página**, ya que dejará de recibir actualizaciones automáticas de Auth0. Ese mantenimiento incluye actualizar los números de versión de los SDK de Auth0 o widgets que se hayan incluido.

Dado que por la página de inicio de sesión suele pasar información sensible relacionada con la seguridad, introducir vulnerabilidades de cross-site scripting (XSS) es motivo de preocupación. Además, si tu organización usa [Home Realm Discovery](/docs/es/get-started/architecture-scenarios/business-to-business/authentication#home-realm-discovery) junto con Classic Login, la lista de conexiones disponibles queda visible públicamente en la propiedad Auth0.setClient. Teniendo esto en cuenta, Auth0 te recomienda extremar la precaución al usar JavaScript de terceros en tu página de inicio de sesión.

<div id="enable-page-customization">
  ### Habilitar la personalización de páginas
</div>

Para habilitar la personalización de una página específica, sigue estos pasos:

1. En Auth0 Dashboard, ve a [Branding &gt; Universal Login](https://manage.auth0.com/#/login_settings).
2. Selecciona una de las siguientes pestañas:

   * **Login**
   * **Restablecimiento de contraseña**
   * **Autenticación multifactor**
3. Activa el interruptor **Customize Page** ubicado en la parte superior de la pestaña.
4. Repite los pasos 2 y 3 según sea necesario.

Después de habilitar la personalización de una página, puedes modificar su plantilla HTML en el editor de código ubicado debajo del interruptor.

<Warning>
  Si habilitas la personalización para inspeccionar el código de la página, pero decides no personalizarla, asegúrate de desactivar el interruptor **Customize Page** al terminar para que Auth0 pueda renderizar las páginas predeterminadas.
</Warning>

<div id="customize-the-login-page">
  ### Personaliza la página de inicio de sesión
</div>

Para personalizar la plantilla de la página de inicio de sesión, primero debes elegir una plantilla base sobre la que trabajar.

1. En el Auth0 Dashboard, ve a [Branding &gt; Universal Login &gt; pestaña Login](https://manage.auth0.com/#/login_page).
2. Activa el selector **Customize Login Page**.
3. Encima del editor de código HTML, selecciona una opción del menú **Default Templates**.

Las plantillas disponibles usan las siguientes bibliotecas:

* [Lock](/docs/es/libraries/lock)
* [Lock (modo passwordless)](/docs/es/libraries/lock)
* [Auth0.js](/docs/es/libraries/auth0js)
* [Authentication API](https://auth0.com/docs/api/authentication)

Puedes usar estas bibliotecas en Classic Login o integrarlas directamente en una aplicación. Para entender mejor cómo personalizar tu experiencia de inicio de sesión, consulta la documentación de la biblioteca que prefieras.

<div id="lock-template">
  #### Plantilla de Lock
</div>

Si va a usar la página de inicio de sesión predeterminada y solo necesita hacer cambios mínimos, puede usar la plantilla de Lock para modificar el comportamiento del widget de Lock en la página de inicio de sesión.

Por ejemplo, puede configurar el widget de Lock para:

* Redirigir a los usuarios a la página de registro en lugar de a la página de inicio de sesión de forma predeterminada
* Personalizar la página de inicio de sesión con distintos colores, textos o idiomas predeterminados
* Mostrar a los usuarios solo uno o varios métodos específicos de inicio de sesión

Para obtener más información sobre cómo personalizar el widget de Lock, consulte [Opciones de configuración de Lock](/docs/es/libraries/lock/lock-configuration).

<div id="custom-login-form-template">
  #### Plantilla de Custom Login Form
</div>

Si quieres modificar considerablemente la página de inicio de sesión, puedes usar la plantilla de Custom Login Form como guía. Esta plantilla te muestra cómo obtener los valores que necesitas con el SDK de Auth0.js. Después, puedes modificar el estilo y el diseño de la página de inicio de sesión como prefieras. Cualquier personalización de CSS **debe** hacerse inline, ya que no puedes alojar un archivo CSS independiente en tu tenant de Auth0.
