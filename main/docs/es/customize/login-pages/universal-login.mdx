[Auth0 Universal Login](/docs/es/authenticate/login/auth0-universal-login) proporciona la funcionalidad esencial de un <Tooltip tip="Servidor de autorización: Servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, tu servidor de autorización puede controlar los datos, las tareas y las funcionalidades disponibles para un usuario." cta="Ver glosario" href="/docs/es/glossary?term=authorization+server">servidor de autorización</Tooltip>: el flujo de inicio de sesión. Cuando un usuario necesita demostrar su identidad para acceder a tu aplicación, puedes redirigirlo a <Tooltip tip="Servidor de autorización: Servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, tu servidor de autorización puede controlar los datos, las tareas y las funcionalidades disponibles para un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip> y dejar que Auth0 se encargue del proceso de autenticación.

La configuración de Universal Login es dinámica y no requiere cambios a nivel de aplicación, ya que las páginas web alojadas en el <Tooltip tip="Servidor de autenticación: Servidor que confirma o rechaza la identidad de un usuario." cta="Ver glosario" href="/docs/es/glossary?term=authentication+server">servidor de autenticación</Tooltip> centralizado de Auth0 controlan por completo el flujo de inicio de sesión. Además, no necesitas actualizar manualmente tu código para que tus aplicaciones se beneficien de las mejoras que Auth0 realiza en Universal Login.

Desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, puedes personalizar la apariencia y el comportamiento de las páginas de inicio de sesión para crear una experiencia coherente y alineada con tu marca. Para casos de uso avanzados, también puedes cambiar el código de cada página individualmente.
