<div id="set-language-option">
  ## Configurar la opción de idioma
</div>

Para usar los siguientes ejemplos, primero debes incluir Lock en tu página:

`<script src="https://cdn.auth0.com/js/lock/11.14/lock.min.js"></script>`

A continuación, define tu objeto `options` e incluye la opción `language`. La opción `language` debe ser un `String` que coincida con el nombre del archivo correspondiente en el directorio `i18n` [de Lock](https://github.com/auth0/lock/tree/master/src/i18n). Luego, inicializa Lock.

{/* codeblockOld.header.login.logInButton codeblockOld.header.login.configureSnippet */}

export const codeExample1 = `// Selecciona un idioma compatible
var options = {
  language: 'es'
};

// Inicializa Auth0Lock
var lock = new Auth0Lock(
  '{yourClientId}',
  '{yourDomain}',
  options
);`;

<AuthCodeBlock children={codeExample1} language="javascript" />

Los datos de traducción de Lock provienen de archivos de idioma con pares clave-valor que representan distintas traducciones. En algunos idiomas, puede que falten ciertos valores; en ese caso, verás esta advertencia: `language does not have property <missing prop>`. Te animamos a enviar un [pull request](https://github.com/auth0/lock/tree/master/src/i18n) para añadir esos valores faltantes. Como alternativa, puedes definirlos en las `options` de Lock (consulta más abajo).

<div id="replace-dictionary-terms">
  ## Reemplaza términos del diccionario
</div>

También puedes personalizar elementos concretos de tu propio diccionario mediante la opción `languageDictionary`. Esto resulta útil si usas uno de los idiomas compatibles, pero quieres cambiar la redacción de algunos elementos concretos. Por ejemplo, podrías reformular el `title` o cambiar la forma en que se muestran otras etiquetas al usuario, mientras dejas intacto el resto del texto del widget.

{/* codeblockOld.header.login.logInButton codeblockOld.header.login.configureSnippet */}

export const codeExample2 = `// Personaliza algunos atributos de languageDictionary
var options = {
  languageDictionary: {
    emailInputPlaceholder: "something@youremail.com",
    title: "Inicia sesión"
  },
};

// Inicialización de Auth0Lock
var lock = new Auth0Lock(
  '{yourClientId}',
  '{yourDomain}',
  options
);`;

<AuthCodeBlock children={codeExample2} language="javascript" />

Para ver un ejemplo de los nombres de propiedad `languageDictionary` disponibles y cómo estructurar un archivo de idioma, consulta el [archivo de diccionario en inglés de Lock](https://github.com/auth0/lock/blob/master/src/i18n/en.js).

La opción `languageBaseUrl`, que acepta un valor de tipo string (una URL), sobrescribe la URL de origen del idioma para las traducciones proporcionadas por Auth0. De forma predeterminada, usa la URL de la CDN de Auth0 `https://cdn.auth0.com`, ya que allí se almacenan las traducciones de idioma proporcionadas. Si proporcionas otro valor, puedes usar tu propia fuente para las traducciones de idioma según sea necesario para tus aplicaciones. La fuente de idioma debe ser un archivo JavaScript.
