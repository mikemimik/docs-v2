description: Soporte de internacionalización en Lock.Android
title: Internacionalización de Lock.Android
---

De forma predeterminada, **Lock.Android** muestra todo el texto en inglés. Si deseas mostrar texto en otro idioma, puedes proporcionar un archivo `strings.xml` y definir los valores que se usarán para los distintos textos que Lock pueda mostrar.

Android consulta al dispositivo qué locale configuró el usuario e intenta obtener la lista de textos localizados para ese idioma. Para que esto funcione, el desarrollador debe agregar cada archivo de traducción a la aplicación siguiendo una convención especial de nombres de carpetas conforme a los estándares de Android. Puedes encontrar más información en la [documentación para desarrolladores de Android](https://developer.android.com/training/basics/supporting-devices/languages).

Algunos de los valores predeterminados que proporciona Lock incluyen:

```xml lines
<resources>
  <string name="com_auth0_lock_social_error_authentication">Error parsing Authentication data</string>
  <string name="com_auth0_lock_social_error_title">There was an error during authentication</string>
  ...
</resources>
```

Si proporcionas tu propio archivo `strings.xml`, estos valores se pueden ajustar de la siguiente manera:

```xml lines
<resources>
  <string name="com_auth0_lock_social_error_authentication">There was an authentication error!</string>
  <string name="com_auth0_lock_social_error_title">Social login error!!</string>
  ...
</resources>
```
