Puedes crear flujos de eventos que escuchen distintos tipos de eventos, como la creación de un nuevo perfil de usuario. En las secciones siguientes se detallan aspectos específicos de los tipos de eventos compatibles actualmente.

<div id="event-object">
  ## Objeto de evento
</div>

Los eventos se publican mediante un esquema que cumple con la [especificación CloudEvent](https://cloudevents.io/):

| Parámetro   | Tipo          | Obligatorio | Descripción                                                                                |
| ----------- | ------------- | ----------- | ------------------------------------------------------------------------------------------ |
| id          | string        | Sí          | Identifica el evento.                                                                      |
| source      | URI-reference | Sí          | Identifica el tenant que generó el evento, incluido su entorno.                            |
| specversion | string        | Sí          | La versión de la especificación de CloudEvents que utiliza el evento.                      |
| type        | string        | Sí          | El tipo de evento.                                                                         |
| time        | timestamp     | Sí          | Marca temporal de cuándo ocurrió el evento.                                                |
| dataschema  | URI           | Sí          | Identifica el esquema al que se ajustan los datos.                                         |
| data        | object        | Sí          | Información adicional relacionada con el evento. Este campo varía según el tipo de evento. |

```json lines expandable
{
  "id": "evt_<unique-id>",
  "source": "urn:auth0:<your-tenant>:users",
  "spec_version": "1.0",
  "type": "com.auth0.user.<event-type>",
  "time": "2025-01-29T14:36:25.794Z",
  "dataschema": "https://<your-tenant>.auth0.com/api/v2/schemas/user.<event-type>",
  "data": {
    "object": {
      "user_id": "<user-id>",
      "email": "<email>",
      "email_verified": <true/false>,
      "family_name": "<family-name>",
      "given_name": "<given-name>",
      "name": "<full-name>",
      "nickname": "<nickname>",
      "phone_number": "<phone-number>",
      "phone_verified": <true/false>,
      "user_metadata": {
        "hobby": "<user-hobby>"
      },
      "app_metadata": {
        "external_user_id": "<external-id>"
      },
      "identities": [
        {
          "connection": "<connection-name>",
          "user_id": "<identity-user-id>",
          "provider": "<identity-provider>",
          "isSocial": <true/false>
        }
      ],
      "created_at": "<user-created-time>",
      "updated_at": "<user-updated-time>",
      "picture": "<profile-picture-url>"
    }
  }
}
```

Para obtener más información sobre los Tipos de eventos, consulta el [Catálogo de eventos](https://auth0.com/docs/events/).
