<Card title="Antes de empezar">
  Si usas [autenticación Passwordless SMS](/docs/es/authenticate/passwordless/authentication-methods/use-sms-gateway-passwordless), [notificaciones de SMS y voz para MFA](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-sms-voice-notifications-mfa) o Phone as ID, debes configurar un proveedor telefónico con Unified Phone Experience.

  Si necesitas proveedores distintos para MFA, Passwordless y/o Phone as ID, debes configurar un [proveedor telefónico personalizado](/docs/es/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider) con Auth0 Actions.
</Card>

Unified Phone Experience centraliza toda la configuración de proveedores telefónicos en un solo lugar dentro de tu tenant. Esta centralización elimina la necesidad de configurar un proveedor telefónico varias veces para distintos factores de autenticación por teléfono.

Si quieres probar las capacidades de mensajería telefónica, debes configurar un proveedor telefónico a nivel del tenant.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 no informa al usuario final si un mensaje telefónico se envió correctamente con el proveedor a nivel del tenant. Para solucionar problemas, revisa los [logs del tenant](/docs/es/deploy-monitor/logs).
</Callout>

<div id="configure-mfa">
  ## Configurar MFA
</div>

Puedes usar Unified Phone Experience con los métodos de entrega y factores SMS, voz y SMS + voz. Para obtener más información sobre la <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuario que utiliza un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=MFA">MFA</Tooltip> con Auth0, consulta [Autenticación multifactor](/docs/es/secure/multi-factor-authentication).

1. En el Auth0 Dashboard, ve a [**Branding &gt; Phone Provider**](https://manage.auth0.com/#/phone/templates/phone/provider).
2. Activa el interruptor Phone Message Provider.

   <Frame>![Auth0 Dashboard > Branding > Phone Provider > Habilitar Phone Message Provider](/docs/images/cdy7uua7fh8z/1k93jvdxDq2FZLiRdvr5aR/d917431cd0265943c5118174a8673118/2025-02-24_16-43-14.png)</Frame>
3. Selecciona tu proveedor de mensajería telefónica.

   1. Para usar Twilio como tu proveedor telefónico, sigue [Configurar Twilio como proveedor de mensajería telefónica](/docs/es/customize/phone-messages/configure-phone-messaging-providers/configure-twilio-as-a-phone-messaging-provider).
   2. Para usar un proveedor personalizado, sigue [Configurar un proveedor telefónico personalizado](/docs/es/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider) para usar Auth0 Actions.
4. Selecciona tus métodos de entrega.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Los factores de MFA que quieras usar deben ser un subconjunto de los métodos de entrega que configures a nivel del tenant.
   </Callout>
5. **Guardar** tus cambios.
6. Cuando termines, ve a [Security &gt; Multi-factor Auth](https://manage.auth0.com/#/security/mfa).
7. Selecciona **Phone Message**.
8. Elige los factores de autenticación.
9. **Guardar** tus cambios.

<div id="configure-passwordless">
  ## Configurar Passwordless
</div>

Puedes usar Unified Phone Experience con una conexión SMS de <Tooltip tip="Passwordless: Forma de autenticación que no depende de una contraseña como primer factor." cta="Ver glosario" href="/docs/es/glossary?term=Passwordless">Passwordless</Tooltip>. Para obtener más información sobre la autenticación passwordless con Auth0, consulta [Conexiones passwordless](/docs/es/authenticate/passwordless).

1. En el Auth0 Dashboard, ve a [**Branding &gt; Phone Provider**](https://manage.auth0.com/#/phone/templates/phone/provider).
2. Activa el toggle Proveedor de mensajería telefónica.

   <Frame>![Auth0 Dashboard > Branding > Phone Provider > Enable Phone Message Provider](/docs/images/cdy7uua7fh8z/1k93jvdxDq2FZLiRdvr5aR/d917431cd0265943c5118174a8673118/2025-02-24_16-43-14.png)</Frame>
3. Selecciona tu proveedor de mensajería telefónica.

   1. Para usar Twilio como tu proveedor telefónico, sigue [Configurar Twilio como proveedor de mensajería telefónica](/docs/es/customize/phone-messages/configure-phone-messaging-providers/configure-twilio-as-a-phone-messaging-provider).
   2. Para usar un proveedor personalizado, sigue [Configurar un proveedor telefónico personalizado](/docs/es/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider) para usar Auth0 Actions.
4. Selecciona tu método de entrega.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Si no seleccionas **Text** como método de entrega configurado, tu integración con Passwordless no funcionará.
   </Callout>
5. **Guardar** tus cambios.
6. Cuando hayas terminado, ve a [**Authentication &gt; Passwordless &gt; SMS**](https://manage.auth0.com/#/connections/passwordless).
7. **Guardar** tus cambios.
8. Si usas Twilio:

   1. Ve a [**Applications &gt; Applications**](https://manage.auth0.com/#/applications).
   2. Selecciona las aplicaciones que quieras usar para tu conexión SMS de Twilio para Passwordless.
   3. En **Connections**, habilita la opción SMS (Twilio).
   4. Vuelve a [**Authentication &gt; Passwordless &gt; SMS**](https://manage.auth0.com/#/connections/passwordless).
   5. En la pestaña **Applications**, habilita todas las aplicaciones que quieras usar para Passwordless SMS.
   6. **Guardar** tus cambios.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Si tienes un caso de uso que requiere la experiencia heredada de [MFA](/docs/es/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-sms-voice-notifications-mfa) y [Passwordless](/docs/es/authenticate/passwordless/authentication-methods/use-sms-gateway-passwordless), ponte en contacto con [el equipo de soporte de Auth0](/docs/es/troubleshoot/customer-support).
</Callout>

<div id="test-your-phone-provider">
  ### Prueba tu proveedor de telefonía
</div>

Prueba el proveedor a nivel de tenant antes de usarlo en un entorno de producción:

1. Ve a [**Branding &gt; Phone Provider**](https://manage.auth0.com/#/phone/templates/phone/provider).
2. Selecciona **enviar mensaje de prueba** para probar el proveedor a nivel de tenant antes de usarlo en un entorno de producción.

<div id="configure-phone-templates">
  ## Configurar plantillas de teléfono
</div>

El proveedor telefónico a nivel del tenant usa la [sintaxis Liquid](https://github.com/Shopify/liquid/wiki/Liquid-for-Designers) para inyectar variables de contexto en el contenido de tus mensajes de teléfono. Para configurar tus plantillas de teléfono, sigue las instrucciones de [Personalizar plantillas de teléfono](/docs/es/customize/phone-messages/phone-templates/customize).

<div id="disable-phone-provider">
  ## Deshabilitar el proveedor telefónico
</div>

Si configuras Unified Phone Experience y luego la deshabilitas, las integraciones existentes con tu proveedor telefónico podrían dejar de funcionar, incluido el factor telefónico de MFA.
