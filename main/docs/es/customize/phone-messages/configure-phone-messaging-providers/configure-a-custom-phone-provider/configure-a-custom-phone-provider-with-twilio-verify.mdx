title: Configurar un proveedor telefónico personalizado con Twilio Verify
---

Puedes configurar un proveedor telefónico personalizado con [Twilio Verify](https://www.twilio.com/docs/verify), que ayuda a combatir el fraude y a proteger las cuentas de usuario. Esto configura Twilio Verify como tu proveedor telefónico personalizado.

<div id="prerequisites">
  ## Requisitos previos
</div>

Debes tener una cuenta de [Twilio](https://twilio.com/) con una opción válida de envío por SMS y/o por llamada telefónica.

<div id="set-up-twilio-verify-service">
  ## Configura el servicio Twilio Verify
</div>

Puedes crear un servicio Twilio Verify a través de la [consola de Twilio](https://www.twilio.com/console/verify/services) o de la [API de REST de Verify](https://www.twilio.com/docs/verify/api/service#create-a-verification-service). Necesitarás lo siguiente para conectar tu proveedor telefónico personalizado de Auth0 con Twilio Verify:

* TWILIO&#95;ACCOUNT&#95;SID
* TWILIO&#95;AUTH&#95;TOKEN
* TWILIO&#95;VERIFY&#95;SID

<div id="connect-custom-phone-provider-with-twilio-verify">
  ## Conecta un proveedor telefónico personalizado con Twilio Verify
</div>

1. En el Auth0 Dashboard, ve a [Branding &gt; Phone Provider](https://manage.auth0.com/dashboard/#/phone/templates/phone/provider). Esto te llevará a la página **Phone Message Provider**.
2. Selecciona **Custom** como **Phone Provider**.
3. En **Provider Configuration**, agrega el siguiente ejemplo de código para permitir que la API de Twilio envíe mensajes telefónicos al número de teléfono del usuario:

```javascript lines expandable
/**
* Manejador que se ejecuta al enviar una notificación telefónica
* @param {Event} event - Detalles sobre el usuario y el contexto en el que está iniciando sesión.
* @param {CustomPhoneProviderAPI} api - Métodos y utilidades para modificar el comportamiento del envío de una notificación telefónica.
*/
exports.onExecuteCustomPhoneProvider = async (event, api) => {
 const { TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_VERIFY_SID } = event.secrets;

  const client = require('twilio')(TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN);

  // asignar el valor de voz de auth0 a llamada
  const messageType = event.notification.delivery_method === 'voice' ? 'call' : 'sms';

  const { recipient, code } = event.notification;

  // agregar esto para corregir " 1333444999  "
  // "333 444 5555"
  const sanitizedNumber = recipient.replace(/\s/g, '').trim();

  await client.verify.v2.services(TWILIO_VERIFY_SID)
    .verifications.create({
      to: sanitizedNumber,
      channel: messageType,
      customCode: code
    })

};
```

3. Haz clic en el ícono de **Key** para abrir el menú **Secrets**. Agrega los siguientes valores de la configuración de tu servicio Twilio Verify:

* TWILIO&#95;ACCOUNT&#95;SID
* TWILIO&#95;AUTH&#95;TOKEN
* TWILIO&#95;VERIFY&#95;SID

4. Para cargar la biblioteca auxiliar de Twilio, haz clic en el ícono **Dependency**. Luego, haz clic en el botón **Add Dependency**. Cuando aparezca la ventana emergente **Add Dependency**, ingresa lo siguiente:

* **Nombre:** Twilio
* **Versión:** latest (debería completarse automáticamente)

Cuando hagas clic en **Create**, Auth0 buscará la biblioteca auxiliar de Twilio y cargará la versión más reciente.

5. Haz clic en **Save**. Cuando hagas clic en **Save,** la Action se guarda y se implementa automáticamente. Para probar la configuración del proveedor telefónico personalizado antes de usarla en un entorno de producción, haz clic en **Enviar mensaje de prueba**.
