<Warning>
  Para personalizar las plantillas telefónicas, debes [configurar un proveedor de telefonía externo](../configure-phone-messaging-providers).
</Warning>

<div id="customize-phone-templates">
  ## Personaliza las plantillas de teléfono
</div>

Para personalizar una plantilla telefónica:

1. Ve a [**Dashboard &gt; Branding &gt; Phone Templates**](https://manage.auth0.com/#/phone).

2. En el menú desplegable **Template**, selecciona la plantilla de teléfono que quieras actualizar.

3. Activa el control deslizante **Status** para habilitar la plantilla si todavía no lo está.

4. Actualiza el número de teléfono de **From** y/o la **plantilla de mensaje SMS**.

5. Selecciona **Save** para guardar los cambios, **Try** para probarlos o **Reset** para revertirlos.

<div id="from-phone-number">
  ### Número de teléfono de **From**
</div>

<Warning>
  Si usas [Twilio Messaging Services](https://www.twilio.com/docs/messaging/services) como origen de tus SMS, no puedes establecer un número de teléfono **From** personalizado. En su lugar, Twilio elige un número del grupo de remitentes de tu Messaging Service en función de tu configuración de Twilio.
</Warning>

El campo **From** establece el número de teléfono desde el que se envían los mensajes de texto. El número de teléfono **From** predeterminado es el que configuraste al configurar tu proveedor telefónico.

Para establecer un número de teléfono **From** personalizado para una plantilla telefónica, introduce el número de teléfono en este campo.

<div id="text-message-template">
  ### Plantilla de mensaje SMS
</div>

El campo **plantilla de mensaje SMS** define el contenido del mensaje de texto. Cada plantilla incluye de forma predeterminada un cuerpo de mensaje que puedes modificar o descartar por completo para escribir el tuyo.

Este campo admite sintaxis Liquid para generar contenido dinámicamente. Para obtener más información, consulta [Sintaxis Liquid compatible para las plantillas telefónicas](./supported-liquid-syntax).

<div id="test-updated-templates">
  ## Probar las plantillas actualizadas
</div>

Para probar una plantilla telefónica, haz clic en **Try**. Auth0 envía el mensaje telefónico a una aplicación predeterminada cuyo nombre corresponde al de tu tenant. Para probar plantillas para distintas aplicaciones, crea un usuario de ejemplo para recorrer los flujos de mensajes telefónicos.

Puedes activar manualmente correos electrónicos de verificación para aplicaciones y usuarios específicos mediante el endpoint [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) de la Management API.
