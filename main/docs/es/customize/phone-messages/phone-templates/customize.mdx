<Warning>
  Para personalizar las plantillas telefónicas, debes [configurar un proveedor telefónico externo](../configure-phone-messaging-providers).
</Warning>

<div id="customize-phone-templates">
  ## Personaliza las plantillas telefónicas
</div>

Para personalizar una plantilla telefónica:

1. Ve a [**Dashboard &gt; Branding &gt; Phone Templates**](https://manage.auth0.com/#/phone).

2. En el menú desplegable **Plantilla**, selecciona la plantilla telefónica que quieras actualizar.

3. Activa el control deslizante de **Estado** para habilitar la plantilla si aún no lo está.

4. Actualiza el número de teléfono de **From** y/o la **plantilla de mensaje SMS**.

5. Selecciona **Guardar** para guardar los cambios, **Probar** para probarlos o **Restablecer** para revertirlos.

<div id="from-phone-number">
  ### Número de teléfono de **From**
</div>

<Warning>
  Si usas [Twilio Messaging Services](https://www.twilio.com/docs/messaging/services) como origen de tus SMS, no puedes establecer un número de teléfono **From** personalizado. En su lugar, Twilio elige un número del grupo de remitentes de tu Messaging Service según tu configuración de Twilio.
</Warning>

El campo **From** establece el número de teléfono desde el que se envían los mensajes de texto. El número de teléfono **From** predeterminado es el que definiste al configurar tu proveedor telefónico.

Para establecer un número de teléfono **From** personalizado para una plantilla telefónica, introduce el número de teléfono en este campo.

<div id="text-message-template">
  ### Plantilla de mensaje SMS
</div>

El campo **Plantilla de mensaje SMS** establece el contenido del mensaje de texto. Cada plantilla incluye de forma predeterminada un cuerpo de mensaje que puedes modificar o descartar por completo para escribir el tuyo propio.

Este campo admite la sintaxis Liquid para generar contenido dinámicamente. Para obtener más información, consulta [Sintaxis Liquid admitida para las plantillas de teléfono](./supported-liquid-syntax).

<div id="test-updated-templates">
  ## Probar plantillas actualizadas
</div>

Para probar una plantilla telefónica, haz clic en **Try**. Auth0 envía el mensaje telefónico a una aplicación predeterminada cuyo nombre es el de tu tenant. Para probar plantillas para diferentes aplicaciones, crea un usuario de ejemplo para que recorra los flujos de mensajes telefónicos.

Puedes enviar manualmente emails de verificación para aplicaciones y usuarios específicos mediante el endpoint de la Management API [Enviar un email de verificación de dirección de email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email).
