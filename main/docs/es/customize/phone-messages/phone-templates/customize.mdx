<Warning>
  Para personalizar las plantillas de teléfono, debes [configurar un proveedor telefónico externo](../configure-phone-messaging-providers).
</Warning>

<div id="customize-phone-templates">
  ## Personaliza las plantillas de teléfono
</div>

Para personalizar una plantilla de teléfono:

1. Ve a [**Dashboard &gt; Branding &gt; Phone Templates**](https://manage.auth0.com/#/phone).

2. En el menú desplegable **Template**, selecciona la plantilla de teléfono que quieres actualizar.

3. Activa el control deslizante **Status** para habilitar la plantilla si todavía no lo está.

4. Actualiza el número de teléfono **From** y/o la **plantilla de mensaje SMS**.

5. Selecciona **Save** para guardar los cambios, **Try** para probarlos o **Reset** para revertirlos.

<div id="from-phone-number">
  ### Número de teléfono de **From**
</div>

<Warning>
  Si usas [Twilio Messaging Services](https://www.twilio.com/docs/messaging/services) como origen de tus SMS, no puedes establecer un número de teléfono de **From** personalizado. En su lugar, Twilio elige un número del grupo de remitentes de tu Messaging Service según tu configuración de Twilio.
</Warning>

El campo **From** establece el número de teléfono desde el que se envían los mensajes de texto. El número de teléfono de **From** predeterminado es el que configuraste cuando configuraste tu proveedor telefónico.

Para establecer un número de teléfono de **From** personalizado para una plantilla de teléfono, introduce el número de teléfono en este campo.

<div id="text-message-template">
  ### Plantilla de mensaje SMS
</div>

El campo **Plantilla de mensaje SMS** establece el contenido del mensaje de texto. Cada plantilla incluye un cuerpo de mensaje predeterminado que puedes modificar o descartar por completo para escribir el tuyo.

Este campo admite la sintaxis Liquid para generar contenido dinámico. Para obtener más información, consulta [Sintaxis Liquid compatible para plantillas de SMS](./supported-liquid-syntax).

<div id="test-updated-templates">
  ## Probar plantillas actualizadas
</div>

Para probar una plantilla de teléfono, haz clic en **Try**. Auth0 envía el mensaje de teléfono a una aplicación predeterminada con el nombre de tu tenant. Para probar plantillas para diferentes aplicaciones, crea un usuario de ejemplo para pasar por los flujos de mensajes de teléfono.

Puedes enviar correos de verificación manualmente para aplicaciones y usuarios específicos mediante el endpoint de la Management API [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email).
