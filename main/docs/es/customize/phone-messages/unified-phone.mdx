---
title: La experiencia unificada para telefonía de Auth0
description: Obtén información sobre la experiencia unificada para telefonía de Auth0 para configurar el proveedor telefónico.
---

Auth0 te permite configurar un proveedor telefónico para <Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuarios que utiliza un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=Multi-Factor+Authentication">Autenticación multifactor</Tooltip> (MFA) y <Tooltip tip="Autenticación multifactor (MFA): Proceso de autenticación de usuarios que utiliza un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/docs/es/glossary?term=Passwordless">Passwordless</Tooltip> a nivel del tenant con la experiencia unificada para telefonía. Esto elimina la necesidad de configurar un proveedor telefónico en varios lugares, sin perder flexibilidad al configurar los métodos de entrega.

Los tenants con proveedores telefónicos configurados pueden migrar a la experiencia unificada para telefonía y mantener las capacidades de mensajería telefónica en un solo lugar. Los tenants nuevos se inscriben automáticamente en la experiencia unificada para telefonía.
