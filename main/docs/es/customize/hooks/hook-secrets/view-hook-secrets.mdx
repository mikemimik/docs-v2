<Warning>
  La fecha de fin de vida útil (EOL) de Rules y Hooks será el **18 de noviembre de 2026**, y ya no están disponibles para los tenants nuevos creados a partir del **16 de octubre de 2023**. Los tenants existentes con Hooks activos conservarán el acceso al producto Hooks hasta su fin de vida útil.

  Te recomendamos encarecidamente que uses Actions para extender Auth0. Con Actions, tienes acceso a información de tipos detallada, documentación integrada y paquetes públicos de `npm`, y puedes conectar integraciones externas para mejorar tu experiencia general de extensibilidad. Para obtener más información sobre lo que ofrece Actions, lee [Cómo funcionan las Actions de Auth0](/docs/es/customize/actions/actions-overview).

  Para ayudarte con la migración, ofrecemos guías que te ayudarán a [migrar de Rules a Actions](/docs/es/customize/actions/migrate/migrate-from-rules-to-actions) y [migrar de Hooks a Actions](/docs/es/customize/actions/migrate/migrate-from-hooks-to-actions). También tenemos una página dedicada, [Move to Actions](https://auth0.com/extensibility/movetoactions), que destaca comparaciones de funcionalidades, [una demo de Actions](https://www.youtube.com/watch?v=UesFSY1klrI) y otros recursos para ayudarte durante la migración.

  Para obtener más información sobre la obsolescencia de Rules y Hooks, consulta nuestra entrada de blog: [Preparing for Rules and Hooks End of Life](https://auth0.com/blog/preparing-for-rules-and-hooks-end-of-life/).
</Warning>

<Warning>
  Como tenemos previsto eliminar las funciones de Rules y Hooks en 2026, debes crear nuevas Rules o Hooks solo en tu entorno de desarrollo y únicamente para probar la migración a Actions.

  Para aprender a migrar tus Rules a Actions, lee [Migrar de Rules a Actions](/docs/es/customize/actions/migrate/migrate-from-rules-to-actions). Para aprender a migrar tus Hooks a Actions, lee [Migrar de Hooks a Actions](/docs/es/customize/actions/migrate/migrate-from-hooks-to-actions).
</Warning>

Puedes ver los secretos de Hooks configurados para un Hook desde el Dashboard u obtener una lista de secretos de Hooks mediante la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

<div id="view-hook-secrets-using-the-dashboard">
  ## Ver los secretos de Hooks usando el Dashboard
</div>

1. Ve a [Auth0 Dashboard &gt; pipeline de autenticación &gt; Hooks](https://manage.auth0.com/#/) y haz clic en el icono del lápiz junto al Hook cuyos secretos quieres ver.
2. En el editor de Hooks, haz clic en el icono de la llave inglesa y luego en **Secrets**.

Se muestran todos los secretos configurados para el Hook seleccionado.

<div id="get-hook-secrets-using-the-management-api">
  ## Obtén los secretos de Hooks con la Management API
</div>

Haz una solicitud `GET` al [endpoint Get secretos de Hooks](https://auth0.com/docs/api/management/v2/#!/Hooks/get_secrets). Asegúrate de reemplazar los valores de marcador de posición `HOOK_ID` y `MGMT_API_ACCESS_TOKEN` por el ID de tu Hook y el <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=Access+Token">token de acceso</Tooltip> de la Management API, respectivamente.

  ```bash cURL
  curl --request GET \
    --url 'https://{yourDomain}/api/v2/hooks/HOOK_ID/secrets' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
  ```

| Valor                   | Descripción                                                                                                                                                                                                             |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `HOOK_ID`               | ID del Hook cuyos secretos deseas recuperar.                                                                                                                                                                            |
| `MGMT_API_ACCESS_TOKEN` | tokens de acceso de la Management API con el [scope](/docs/es/glossary?term=scope) `update:hooks`. Para obtener más información, consulta [Management API Access Tokens](https://auth0.com/docs/api/management/v2/tokens). |

<Warning>
  Los secretos de Hooks son de solo escritura. Al recuperar los secretos configurados para un Hook específico, los valores contendrán el siguiente texto de marcador de posición: `_VALUE_NOT_SHOWN`.
</Warning>
