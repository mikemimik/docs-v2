<Warning>
  La fecha de fin de vida útil (EOL) de Rules y Hooks será el **18 de noviembre de 2026**, y ya no están disponibles para los nuevos tenants creados a partir del **16 de octubre de 2023**. Los tenants existentes con Hooks activos conservarán el acceso al producto Hooks hasta su fin de vida útil.

  Te recomendamos encarecidamente que uses Actions para ampliar Auth0. Con Actions, tienes acceso a información de tipos más completa, documentación integrada y paquetes públicos de `npm`, y puedes conectar integraciones externas que mejoran tu experiencia general de extensibilidad. Para obtener más información sobre lo que ofrece Actions, consulta [Cómo funcionan las Actions de Auth0](/docs/es/customize/actions/actions-overview).

  Para ayudarte con la migración, ofrecemos guías que te ayudarán a [migrar de Rules a Actions](/docs/es/customize/actions/migrate/migrate-from-rules-to-actions) y [migrar de Hooks a Actions](/docs/es/customize/actions/migrate/migrate-from-hooks-to-actions). También tenemos una página específica, [Move to Actions](https://auth0.com/extensibility/movetoactions), que destaca comparaciones de funcionalidades, [una demostración de Actions](https://www.youtube.com/watch?v=UesFSY1klrI) y otros recursos para acompañarte en tu proceso de migración.

  Para obtener más información sobre la obsolescencia de Rules y Hooks, consulta nuestra entrada de blog: [Preparación para el fin de vida útil de Rules y Hooks](https://auth0.com/blog/preparing-for-rules-and-hooks-end-of-life/).
</Warning>

<Warning>
  Como tenemos previsto eliminar las funciones de Rules y Hooks en 2026, debes crear nuevas Rules o Hooks solo en tu entorno de desarrollo y únicamente para probar la migración a Actions.

  Para aprender a migrar tus Rules a Actions, consulta [Migrar de Rules a Actions](/docs/es/customize/actions/migrate/migrate-from-rules-to-actions). Para aprender a migrar tus Hooks a Actions, consulta [Migrar de Hooks a Actions](/docs/es/customize/actions/migrate/migrate-from-hooks-to-actions).
</Warning>

Cuando ya no necesites secretos de Hooks para un Hook determinado, puedes eliminarlos mediante el Dashboard o la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>.

Los secretos de Hooks también se pueden importar y exportar con la herramienta [Deploy Command-Line Interface (CLI)](/docs/es/deploy-monitor/deploy-cli-tool).

<div id="delete-hook-secrets-using-the-dashboard">
  ## Eliminar secretos de Hooks usando el Dashboard
</div>

1. Vaya a la página [Hooks](https://manage.auth0.com/#/hooks) del [Auth0 Dashboard](https://manage.auth0.com/#/) y haga clic en el icono de lápiz junto al Hook que desea editar.
2. En el editor de Hook, haga clic en el icono de llave inglesa y, a continuación, en **Secrets**.
3. Busque el secreto de Hook que desea eliminar, haga clic en el icono de la papelera y, a continuación, en **Yes**.

<div id="delete-hook-secrets-using-the-management-api">
  ## Eliminar secretos de Hook mediante la Management API
</div>

Haz una llamada `DELETE` al [endpoint Eliminar secretos de Hook](https://auth0.com/docs/api/management/v2/#!/Hooks/delete_secrets). Asegúrate de reemplazar los valores de marcador de posición `HOOK_ID`, `HOOK_SECRET_NAME` y `MGMT_API_ACCESS_TOKEN` por tu ID de Hook, los nombres de tus secretos de Hook y el <Tooltip tip="Token de acceso: credencial de Authorization, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/docs/es/glossary?term=access+token">token de acceso</Tooltip> de la Management API, respectivamente.

| Valor                   | Descripción                                                                                                                                                                                                               |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `HOOK_ID`               | ID del Hook cuyos secretos quieres eliminar.                                                                                                                                                                              |
| `MGMT_API_ACCESS_TOKEN` | Tokens de acceso para la Management API con el [scope](/docs/es/glossary?term=scope) `update:hooks`. Para obtener más información, consulta [Management API Access Tokens](https://auth0.com/docs/api/management/v2/tokens). |
| `HOOK_SECRET_NAME`      | Nombre o nombres de los secretos que quieres eliminar del Hook especificado. Este endpoint acepta un arreglo de nombres de secretos para eliminar.                                                                        |
