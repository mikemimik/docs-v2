Con el historial de versiones de Actions, puede comparar las versiones en borrador y desplegadas de Actions, y revertir a versiones anteriores cuando sea necesario.

<div id="deploy-actions">
  ## Desplegar Actions
</div>

Para poder vincular una Action a un flujo, primero debe desplegarla.

1. Vaya a [Auth0 Dashboard &gt; Actions &gt; Custom Actions](https://manage.auth0.com/#/actions/library).
2. Elija su Action.
3. Seleccione **Deploy**.

La Action seleccionada se desplegará y se agregará a la colección de Actions disponibles para los flujos. Además, se creará una nueva versión en borrador de la Action desplegada para futuras ediciones. Como los secretos y las dependencias están vinculados a las versiones, cualquier secreto o dependencia guardados estarán disponibles para la nueva versión en borrador.

<div id="revert-actions">
  ## Revertir Actions
</div>

Cuando sea necesario, puedes revertir a la última versión válida de una Action.

1. Ve a [Auth0 Dashboard &gt; Actions &gt; Custom Actions](https://manage.auth0.com/#/actions/library).
2. Elige tu Action.
3. Selecciona **Historial de versiones**.
4. Selecciona la versión a la que quieres revertir y haz clic en **Revertir**.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     En el historial de versiones, “Borrador” indica una versión que nunca se ha desplegado.
   </Callout>

Se creará y desplegará una nueva copia de la versión seleccionada. Además, se creará una nueva versión en borrador de la Action desplegada para futuras modificaciones. Como los secretos y las dependencias están vinculados a las versiones, cualquier secreto o dependencia guardados estarán disponibles para el nuevo borrador.
