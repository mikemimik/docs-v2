---
description: Describe cómo pasar metadatos de usuario o de la aplicación entre Actions de login y de Post Login.
title: Actions Transaction Metadata
---

Actions Transaction Metadata almacena, permite acceder y/o comparte metadatos personalizados dentro de una Action de [Post Login](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger) durante una transacción.

Anteriormente, cada Action funcionaba de forma independiente, lo que dificultaba pasar información entre ellas. Con Actions Transaction Metadata, ahora es posible:

* Compartir datos entre Actions, como respuestas de API o cálculos intermedios.
* Eliminar la necesidad de volver a obtener o recalcular la misma información en distintas Actions.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Antes de usar metadato de la transacción en desarrollo, te recomendamos revisar las limitaciones. Para obtener más información, consulta [Actions Limitations](/docs/es/customize/actions/limitations).
</Callout>

<div id="how-it-works">
  ## Cómo funciona
</div>

Utilice el [objeto de API de Post Login](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger/post-login-api-object) `api.transaction.setMetadata` para establecer el par clave/valor y almacenar metadatos de la transacción.

Utilice el [objeto Event de Post Login](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger/post-login-event-object) `event.transaction.metadata` para acceder al par clave/valor almacenado en la misma Action o en Actions posteriores del trigger Post Login dentro de una sola ejecución.

Los objetos API y Event aceptan los siguientes parámetros:

| Valor   | Tipo                          | Descripción                                                                                     |
| ------- | ----------------------------- | ----------------------------------------------------------------------------------------------- |
| `Key`   | `String`                      | La clave de la propiedad de metadatos que se va a establecer.                                   |
| `Value` | `String`, `Number`, `Boolean` | El valor de la propiedad de metadatos.<br />Si se establece en `null`, se elimina la propiedad. |

Para obtener más información sobre cómo escribir Actions, lea [Write Your First Action](/docs/es/customize/actions/write-your-first-action).

<div id="latency">
  ## Latencia
</div>

El uso de Actions Transaction Metadata podría añadir una latencia nominal. Esa latencia sería proporcional al tamaño del payload de los metadatos y cobraría relevancia cuando se produzcan suspensiones de una Action. Por ejemplo, activar <Tooltip tip="" cta="Ver glosario" href="/docs/es/glossary?term=MFA">MFA</Tooltip>, redirigir desde Actions o renderizar formularios podría generar problemas de latencia debido a la necesidad de recargar datos desde el almacenamiento.

Aun así, la latencia potencial debería ser menor que la de hacer solicitudes HTTP salientes redundantes para recuperar los datos que necesita la secuencia de Actions.

<div id="examples">
  ## Ejemplos
</div>

<div id="access-metadata-immediately">
  ### Accede de inmediato a los metadatos
</div>

Define pares clave-valor en los metadatos de la transacción y accede de inmediato a sus valores.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.transaction.setMetadata('hello', 'Auth0');

  console.log('Hello ', event.transaction?.metadata?.hello);
  /* Muestra "Hello Auth0" */
};
```

<div id="set-supported-values">
  ### Establecer valores admitidos
</div>

Establece valores de tipo `string`, `number` y `boolean`.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.transaction.setMetadata('string_value', 'Auth0');
  api.transaction.setMetadata('boolean_value', true);
  api.transaction.setMetadata('number_value', 12);

  console.log('string_value', event.transaction?.metadata?.string_value);
  /* Muestra "string_value Auth0" */
  console.log('boolean_value', event.transaction?.metadata?.boolean_value);
  /* Muestra "boolean_value true" */
  console.log('number_value', event.transaction?.metadata?.number_value);
  /* Muestra "number_value 12" */
};
```

<div id="serialize-values">
  ### Serializar valores
</div>

Serializa los valores como `strings` dentro de los [límites](/docs/es/customize/actions/limitations#transaction-metadata).

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  No puedes definir valores de tipo `object` o `array`, pero sí puedes serializarlos como `strings.`
</Callout>

```text lines expandable
exports.onExecutePostLogin = async (event, api) => {
  const serialized_object_value = JSON.stringify({
    string_value: 'Auth0',
    boolean_value: true,
    number_value: 12
  });

  api.transaction.setMetadata('serialized_object_value', serialized_object_value);

  const serialized_array_value = JSON.stringify([
    'Auth0',
    true,
    12
  ]);
  api.transaction.setMetadata('serialized_array_value', serialized_array_value);

  console.log('serialized_object_value',
    JSON.parse(event.transaction?.metadata?.serialized_object_value)
  );
  /* Muestra "serialized_object_value { string_value: 'Auth0', boolean_value: true, number_value: 12 }" */

  console.log('serialized_array_value',
    JSON.parse(event.transaction?.metadata?.serialized_array_value)
  );
  /* Muestra "serialized_array_value [ 'Auth0', true, 12 ]" */
};
```

<div id="share-values-between-actions">
  ### Compartir valores entre Actions
</div>

Comparta pares clave-valor entre Actions en la misma secuencia de ejecución.

**Action 1**

Establezca la clave `hello` con el valor `Auth0` usando el método `setMetadata` del objeto `api`.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.transaction.setMetadata('hello', 'Auth0');
};
```

**Action 2**

Registra el valor `Auth0` de la clave `hello` en los metadatos de la transacción y usa la propiedad `transaction.metadata` del objeto `event` para acceder a los valores asignados.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  console.log('Hello', event.transaction?.metadata?.hello);
  /* Muestra "Hello Auth0" */
};
```

<div id="update-metadata">
  ### Actualizar metadatos
</div>

Asigna un valor distinto a una clave existente para actualizar los metadatos.

**Action 1**

Asigna el par clave/valor de los metadatos de la transacción como `custom_tx_id` y `xyz123`.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.transaction.setMetadata('custom_tx_id', 'xyz123');
};
```

**Action 2**

Registra la clave `custom_tx_id` con el valor `xyz123`. Luego, cambia `custom_tx_id` a `abc456` para volver a registrarlo con el valor más reciente de `custom_tx_id` en los metadatos de la transacción.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  console.log('custom_tx_id', event.transaction?.metadata?.custom_tx_id);
  /* Muestra "custom_tx_id xyz123" */

  api.transaction.setMetadata('custom_tx_id', 'abc456');

  console.log('custom_tx_id', event.transaction?.metadata?.custom_tx_id);
  /* Muestra "custom_tx_id abc456" */
};
```

**Action 3**

Registra el valor `abc456` para `custom_tx_id`.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  console.log('custom_tx_id', event.transaction?.metadata?.custom_tx_id);
  /* Muestra "custom_tx_id abc456" */
};
```

<div id="remove-metadata">
  ### Eliminar metadatos
</div>

Elimina los valores de los metadatos de la transacción asignando `null` al valor de cada clave concreta.

**Action 1**

Establece `custom_tx_id` en los metadatos de la transacción.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.transaction.setMetadata('custom_tx_id', 'xyz123');
};
```

**Action 2**

Establece `custom_tx_id` como `null` y registra el valor `null` de `custom_tx_id`.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  console.log('custom_tx_id', event.transaction?.metadata?.custom_tx_id);
  /* Muestra "custom_tx_id xyz123" */

  api.transaction.setMetadata('custom_tx_id', null);

  console.log('custom_tx_id', event.transaction?.metadata?.custom_tx_id);
  /* Muestra "custom_tx_id undefined" */
};
```

**Action 3**

Registra el valor `null` de `custom_tx_id`.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  console.log('custom_tx_id', event.transaction?.metadata?.custom_tx_id);
  /* Muestra "custom_tx_id undefined" */
};
```

<div id="preserve-values-on-redirect-to-external-sites">
  ### Conserva los valores al redirigir a sitios externos
</div>

Conserva los metadatos de la transacción durante las [redirecciones](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger/redirect-with-actions). Los valores estarán disponibles cuando los usuarios continúen el flujo de autenticación.

**Action 1**

Establece un `custom_tx_id` en los metadatos de la transacción.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.transaction.setMetadata('custom_tx_id', 'xyz123');
};
```

**Action 2**

Redirige a un sitio externo enviando un token con el `custom_tx_id` de los metadatos de la transacción. Luego compara el valor de `custom_tx_id` de los metadatos de la transacción con el que se envió al sitio externo y se devolvió en el payload de otro token.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  const token = api.redirect.encodeToken({
    secret: event.secrets.REDIRECT_SECRET,
    expiresInSeconds: 60, 
    payload: {
      custom_tx_id: event.transaction?.metadata?.custom_tx_id,
      continue_uri: `https://${event.secrets.TENANT_DOMAIN}/continue`
    },
  });

  api.redirect.sendUserTo(event.secrets.REDIRECT_URL, {
    query: { session_token: token }
  });
};

exports.onContinuePostLogin = async (event, api) => {
  const payload = api.redirect.validateToken({
    secret: event.secrets.REDIRECT_SECRET
  });

  console.log('Does custon_tx_id match?',
    payload?.custom_tx_id === event.transaction?.metadata?.custom_tx_id
  );
  /* Muestra "Does custon_tx_id match? True" */
};
```

<div id="preserve-values-on-forms-rendering">
  ### Conservar valores al renderizar formularios
</div>

Al [Renderizar formularios con Actions](/docs/es/customize/forms/render), los valores de los metadatos de la transacción se conservarán y seguirán disponibles cuando los usuarios continúen el flujo de autenticación.

Para obtener más información sobre cómo usar formularios con Actions, lee [Renderizar formularios con Actions](/docs/es/customize/forms/render).

**Action 1**

Establece un `custom_tx_id` en los metadatos de la transacción.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.transaction.setMetadata('custom_tx_id', 'xyz123');
};
```

**Action 2**

Renderiza un Form. Luego registra el `custom_tx_id` conservado al continuar la ejecución de Actions.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.prompt.render(event.secrets.FORM_ID);
};

exports.onContinuePostLogin = async (event, api) => {
  console.log('custon_tx_id', event.transaction?.metadata?.custom_tx_id);
  /* Muestra "custom_tx_id xyz123" */
};
```

<div id="share-values-with-forms">
  ### Compartir valores con formularios
</div>

Puedes renderizar formularios con Actions y pasar al formulario valores de los metadatos de la transacción.

**Action 1**

Establece un `custom_tx_id` en los metadatos de la transacción.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.transaction.setMetadata('custom_tx_id', 'xyz123');
};
```

**Action 2**

Renderiza un Form y pasa el `custom_tx_id` de los metadatos de la transacción como parámetro `vars`. Luego, al continuar la ejecución de Actions, compara el valor de `custom_tx_id` de los metadatos de la transacción con el que se pasó al Form.

```text lines
exports.onExecutePostLogin = async (event, api) => {
  api.prompt.render(event.secrets.FORM_ID, {
    vars: {
      custom_tx_id: event.transaction?.metadata?.custom_tx_id
    }
  });
};

exports.onContinuePostLogin = async (event, api) => {
  console.log('Does custon_tx_id match?',
    event.prompt?.vars?.custom_tx_id === event.transaction?.metadata?.custom_tx_id
  );
  /* Muestra "Does custon_tx_id match? True" */
};
```
