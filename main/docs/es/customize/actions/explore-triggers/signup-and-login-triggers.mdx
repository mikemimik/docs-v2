Para obtener más información sobre los triggers de registro e inicio de sesión, consulta las siguientes secciones.

| Trigger                                                                                                                                       | Descripción                                                                                                                                 |
| --------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| [Trigger de Login](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)                                          | Los triggers de Login se ejecutan cuando un usuario inicia sesión en cualquier aplicación de un tenant de Auth0.                            |
| [Trigger de Registro previo de usuario](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger)     | El trigger de Registro previo de usuario se ejecuta antes de que se agregue un usuario a una conexión de base de datos o passwordless.      |
| [Trigger de registro posterior de usuario](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | El trigger de registro posterior de usuario se ejecuta después de que se agregue un usuario a una conexión de base de datos o passwordless. |
