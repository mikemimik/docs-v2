Para obtener más información sobre los triggers de registro e inicio de sesión, consulta las secciones siguientes.

| Trigger                                                                                                                                        | Descripción                                                                                                                                               |
| ---------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Trigger de inicio de sesión](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)                                | Los triggers de inicio de sesión se ejecutan cuando un usuario inicia sesión en cualquier aplicación de un tenant de Auth0.                               |
| [Trigger previo al registro de usuarios](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger)     | El trigger previo al registro de usuarios se ejecuta antes de que se agregue un usuario a una conexión de base de datos o una conexión Passwordless.      |
| [Trigger posterior al registro de usuarios](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | El trigger posterior al registro de usuarios se ejecuta después de que se agregue un usuario a una conexión de base de datos o una conexión Passwordless. |
