Para obtener más información sobre los triggers de registro y de inicio de sesión, consulta las secciones siguientes.

| Trigger                                                                                                                             | Descripción                                                                                                               |
| ----------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| [Login Trigger](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)                                   | Los triggers de Login se ejecutan cuando un usuario inicia sesión en cualquier aplicación de un tenant de Auth0.          |
| [Pre-user Registration Trigger](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger)   | El trigger Pre-user Registration se ejecuta antes de que se agregue un usuario a una conexión Database o passwordless.    |
| [Post-user Registration Trigger](/docs/es/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | El trigger Post-user Registration se ejecuta después de que se agregue un usuario a una conexión Database o passwordless. |
