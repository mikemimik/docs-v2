            * `sms` Se utilizó una conexión Passwordless SMS para autenticar al usuario como primer factor.
            * `email` Se utilizó una conexión Passwordless Email para autenticar al usuario como primer factor o para verificar el correo electrónico durante el restablecimiento de contraseña.
