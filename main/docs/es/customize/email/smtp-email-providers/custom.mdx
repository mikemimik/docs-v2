---
title: Configuración de Custom Email Provider con Actions
description: Escriba una Action para enviar mensajes a su proveedor de correo electrónico e implementar lógica personalizada para su caso de uso.
---

Para personalizar aún más la configuración de su proveedor de correo electrónico, puede configurar su tenant con una <Tooltip tip="Action: Una función de Node.js que se ejecuta en puntos definidos durante el runtime de Auth0." cta="Ver glosario" href="/docs/es/glossary?term=Action">Action</Tooltip>.

En lugar de usar nuestro proveedor de correo electrónico o nuestras integraciones con servidores SMTP, usted escribe el código de su Action para enviar mensajes a través del proveedor de correo electrónico o servidor SMTP que elija.

Esto le da más control sobre el proceso de entrega de correos electrónicos y le permite implementar lógica personalizada para su caso de uso, como reintentar envíos fallidos, cambiar destinatarios o crear lógica específica de la organización.
