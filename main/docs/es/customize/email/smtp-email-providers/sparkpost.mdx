<div id="prerequisites">
  ## Requisitos previos
</div>

* Debe tener una cuenta de [SparkPost](https://www.sparkpost.com/).

* Debe configurar su proveedor para que permita conexiones entrantes desde direcciones IP específicas de [Auth0](/docs/es/secure/security-guidance/data-security/allowlist).

<div id="configure-sparkpost">
  ## Configurar SparkPost
</div>

1. Inicia sesión en tu cuenta de [SparkPost](https://www.sparkpost.com/).
2. Ve a [Sending Domains](https://app.sparkpost.com/account/sending-domains) en SparkPost y agrega tu dominio personalizado. SparkPost solo permite enviar correos electrónicos desde dominios verificados.
3. Ve a [Account API Keys](https://app.sparkpost.com/account/credentials) en SparkPost y haz clic en **New API key**. Guarda este valor de clave y asegúrate de que tenga acceso `Transmissions: Read/Write`. Copia este valor de clave.
4. Ve a [Dashboard &gt; Branding &gt; proveedor de correo electrónico](https://manage.auth0.com/#/branding/email_provider) en Auth0.
5. Activa el interruptor **Use my own proveedor de correo electrónico**.
6. Haz clic en el logotipo de **SparkPost**.
7. Indica una dirección de correo electrónico en **From**, introduce la **API Key** de SparkPost que copiaste antes y selecciona tu **Region**. Si cambias a la región de la UE en una cuenta que ya estaba configurada para la región de EE. UU., debes reemplazar la **API Key** en Auth0 por una API Key de SparkPost para la UE.
8. Haz clic en **Save**.

<div id="send-a-test-email">
  ## Enviar un correo electrónico de prueba
</div>

Envía un correo electrónico de prueba con el botón **Send Test Email**. Si la configuración es correcta, recibirás un correo electrónico de confirmación. Si no lo recibes después de unos minutos, revisa tus [registros de Auth0](https://manage.auth0.com/#/logs) para identificar los códigos de error.

Si no has registrado un <Tooltip tip="Custom Domain: dominio de terceros con un nombre especializado o personalizado." cta="Ver glosario" href="/docs/es/glossary?term=custom+domain">dominio personalizado</Tooltip> con SparkPost, puedes enviar un máximo de cinco correos electrónicos de prueba desde el dominio `sparkpostbox.com`. Para obtener más información, consulta [Transmissions: The Sandbox Domain](https://developers.sparkpost.com/api/transmissions.html#header-the-sandbox-domain) de SparkPost.

Una vez configurado, SparkPost muestra información de entrega de todos los correos electrónicos enviados a tus usuarios en la página [Message Events](https://app.sparkpost.com/reports/message-events).
