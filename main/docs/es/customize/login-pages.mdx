[Auth0 Universal Login](/docs/es/authenticate/login/auth0-universal-login) proporciona la funcionalidad esencial de un <Tooltip tip="Servidor de autorización: Servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, tu servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/docs/es/glossary?term=authorization+server">servidor de autorización</Tooltip>: el flujo de login. Cuando un usuario necesita demostrar su identidad para acceder a tu aplicación, puedes redirigirlo a <Tooltip tip="Servidor de autorización: Servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, tu servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/docs/es/glossary?term=Universal+Login">Universal Login</Tooltip> y dejar que Auth0 se encargue del proceso de autenticación.

Universal Login es la principal solución de login alojado de Auth0. [Universal Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) ofrece herramientas de personalización fáciles de usar y una experiencia más simple y rápida para los usuarios finales. Como alternativa, Auth0 también admite [Classic Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/classic-experience), una experiencia de login alojado que utiliza controles de JavaScript para su personalización.

Para comparar ambas experiencias de login, consulta [Universal Login vs. Classic Login](/docs/es/authenticate/login/auth0-universal-login/universal-login-vs-classic-login).
