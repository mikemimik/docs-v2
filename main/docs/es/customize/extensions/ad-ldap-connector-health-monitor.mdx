---

Auth0 AD/LDAP Connector Health Monitor expone un endpoint de API que elijas para que puedas supervisar tus conectores AD/LDAP.

<div id="configure-the-extension">
  ## Configura la extensión
</div>

Para instalar y configurar esta extensión:

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona **Auth0 AD/LDAP Connector Health Monitor**. Se abre la ventana **Install Extension**.
2. Configura los siguientes parámetros y selecciona **Install**:

* **Auth0&#95;Domain**: El dominio de tu aplicación de Auth0, que puedes encontrar en [Application Settings de Auth0](https://manage.auth0.com/#/applications/\{yourClientId}/settings).
* **Auth0&#95;Global&#95;Client&#95;ID**: El <Tooltip tip="Client ID: Valor de identificación asignado a tu recurso registrado por Auth0." cta="Ver glosario" href="/docs/es/glossary?term=Client+ID">Client ID</Tooltip> global de tu aplicación de Auth0, que puedes encontrar en la sección **Global Application Information** de [Configuración avanzada del tenant](https://manage.auth0.com/#/tenant/advanced).
* **Auth0&#95;Global&#95;Client&#95;Secret**: El <Tooltip tip="Client Secret: Secreto utilizado por un cliente (aplicación) para autenticarse con el servidor de autorización; solo deben conocerlo el cliente y el servidor de autorización, y debe ser lo suficientemente aleatorio como para que no pueda adivinarse." cta="Ver glosario" href="/docs/es/glossary?term=Client+Secret">Client Secret</Tooltip> global de tu aplicación de Auth0, que puedes encontrar en la sección **Global Application Information** de [Configuración avanzada del tenant](https://manage.auth0.com/#/tenant/advanced).

<div id="use-the-extension">
  ## Usa la extensión
</div>

Para acceder a la URL que usarás con tu servicio de supervisión:

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y haz clic en **Installed Extensions**.

2. Selecciona **Auth0 AD/LDAP Connector Health Monitor**. Se abrirá una nueva ventana con un fragmento de JSON similar a este:

   ```json lines
   {
     "message": "Use this url: 'https://sandbox.it.auth0.com/api/run/auth0user/auth0-ldap-conector-health-monitor?connection={MY-LDAP-CONNECTOR}' for monitoring your AD/LDAP connector."
   }
   ```

3. Introduce el ID de tu conexión LDAP. Para obtenerlo, ve a  [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), haz clic en la conexión y copia el valor del campo **Identifier**.

Cuando termines, podrás usar la URL proporcionada con tu servicio de supervisión para supervisar tus conectores AD/LDAP.
