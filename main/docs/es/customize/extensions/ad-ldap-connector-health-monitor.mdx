---

Auth0 AD/LDAP Connector Health Monitor expone un endpoint de API que elijas para que puedas supervisar tus conectores AD/LDAP.

<div id="configure-the-extension">
  ## Configurar la extensión
</div>

Para instalar y configurar esta extensión:

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona **Auth0 AD/LDAP Connector Health Monitor**. Se abrirá la ventana **Install Extension**.
2. Establece los siguientes parámetros de configuración y selecciona **Install**:

* **Auth0&#95;Domain**: El dominio de tu aplicación de Auth0, que puedes encontrar en [Auth0 Application Settings](https://manage.auth0.com/#/applications/\{yourClientId}/settings).
* **Auth0&#95;Global&#95;Client&#95;ID**: El <Tooltip tip="ID de cliente: Valor de identificación otorgado por Auth0 a tu recurso registrado." cta="Ver glosario" href="/docs/es/glossary?term=Client+ID">ID de cliente</Tooltip> global de tu aplicación de Auth0, que puedes encontrar en la sección **Global Application Information** de [Tenant Advanced Settings](https://manage.auth0.com/#/tenant/advanced).
* **Auth0&#95;Global&#95;Client&#95;Secret**: El <Tooltip tip="Secreto de cliente: Secreto que usa un cliente (aplicación) para autenticarse con el servidor de autorización; solo deben conocerlo el cliente y el servidor de autorización, y debe ser lo suficientemente aleatorio como para que no se pueda adivinar." cta="Ver glosario" href="/docs/es/glossary?term=Client+Secret">secreto de cliente</Tooltip> global de tu aplicación de Auth0, que puedes encontrar en la sección **Global Application Information** de [Tenant Advanced Settings](https://manage.auth0.com/#/tenant/advanced).

<div id="use-the-extension">
  ## Usa la extensión
</div>

Para obtener la URL que usarás con tu servicio de monitoreo:

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y haz clic en **Installed Extensions**.

2. Selecciona **Auth0 AD/LDAP Connector Health Monitor**. Se abrirá una nueva ventana con un fragmento de JSON similar a este:

   ```json lines
   {
     "message": "Use this url: 'https://sandbox.it.auth0.com/api/run/auth0user/auth0-ldap-conector-health-monitor?connection={MY-LDAP-CONNECTOR}' for monitoring your AD/LDAP connector."
   }
   ```

3. Ingresa el ID de tu conexión LDAP. Para obtenerlo, ve a [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), haz clic en la conexión y copia el valor del campo **Identifier**.

Cuando hayas terminado, podrás usar la URL proporcionada con tu servicio de monitoreo para supervisar tus conectores AD/LDAP.
