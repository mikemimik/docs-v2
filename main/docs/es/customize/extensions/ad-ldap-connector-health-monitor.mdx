---

Auth0 AD/LDAP Connector Health Monitor expone un endpoint de API de tu elección para que puedas supervisar tus conectores AD/LDAP.

<div id="configure-the-extension">
  ## Configurar la extensión
</div>

Para instalar y configurar esta extensión:

1. Vaya a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y seleccione **Auth0 AD/LDAP Connector Health Monitor**. Se abrirá la ventana **Instalar extensión**.
2. Establezca los siguientes parámetros de configuración y seleccione **Instalar**:

* **Auth0&#95;Domain**: El dominio de su aplicación de Auth0, que puede encontrar en la [configuración de su aplicación de Auth0](https://manage.auth0.com/#/applications/\{yourClientId}/settings).
* **Auth0&#95;Global&#95;Client&#95;ID**: El <Tooltip tip="ID de cliente: Valor de identificación que Auth0 asigna a su recurso registrado." cta="Ver glosario" href="/docs/es/glossary?term=Client+ID">ID de cliente</Tooltip> global de su aplicación de Auth0, que puede encontrar en la sección **Información global de la aplicación** de la [configuración avanzada del tenant](https://manage.auth0.com/#/tenant/advanced).
* **Auth0&#95;Global&#95;Client&#95;Secret**: El <Tooltip tip="Secreto de cliente: Secreto que usa un cliente (aplicación) para autenticarse con el servidor de autorización; solo el cliente y el servidor de autorización deben conocerlo, y debe ser lo suficientemente aleatorio como para que no se pueda adivinar." cta="Ver glosario" href="/docs/es/glossary?term=Client+Secret">secreto de cliente</Tooltip> global de su aplicación de Auth0, que puede encontrar en la sección **Información global de la aplicación** de la [configuración avanzada del tenant](https://manage.auth0.com/#/tenant/advanced).

<div id="use-the-extension">
  ## Usa la extensión
</div>

Para acceder a la URL que usarás con tu servicio de monitoreo:

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y haz clic en **Installed Extensions**.

2. Selecciona **Auth0 AD/LDAP Connector Health Monitor**. Se abrirá una nueva ventana con un fragmento de JSON similar a este:

   ```json lines
   {
     "message": "Use this url: 'https://sandbox.it.auth0.com/api/run/auth0user/auth0-ldap-conector-health-monitor?connection={MY-LDAP-CONNECTOR}' for monitoring your AD/LDAP connector."
   }
   ```

3. Introduce el ID de tu conexión LDAP. Para obtenerlo, ve a [Auth0 Dashboard &gt; Authentication &gt; empresarial](https://manage.auth0.com/#/connections/enterprise), haz clic en la conexión y copia el valor del campo **Identifier**.

Una vez que termines, podrás usar la URL proporcionada con tu servicio de monitoreo para monitorear tus conectores AD/LDAP.
