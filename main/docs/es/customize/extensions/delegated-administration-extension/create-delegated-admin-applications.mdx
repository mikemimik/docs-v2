title: Crear aplicaciones de Delegated Admin
---

Usa la [extensión Delegated Admin](/docs/es/customize/extensions/delegated-administration-extension) para exponer la sección de **Users** del <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> a un grupo selecto de usuarios sin darles acceso al resto del Dashboard. Antes de agregar la extensión Delegated Admin, debes crear la aplicación Delegated Admin en Auth0.

* Ve a [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y selecciona **Create Application**.
* Ingresa un nombre descriptivo para tu aplicación (por ejemplo, **Users Dashboard**), selecciona el tipo de aplicación **Single-Page Web Application** y haz clic en **Create**.
* En la vista **Settings**, configura las **Allowed Callback URLs** y las **Allowed Logout URLs** según tu ubicación y haz clic en **Save Changes**.

**URL de devolución de llamada por región:**

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Reemplaza `{yourTenant}` por el nombre de tu tenant. Por ejemplo, si el nombre de tu tenant es `tenant_name_example`, entonces, para un tenant que se ejecuta en un entorno US-3, la URL de devolución de llamada correspondiente es: `https://tenant_name_example.us.webtask.run/auth0-delegated-admin/login`
</Callout>

| Ubicación   | Region | Enlace de inicio de sesión                                        |
| ----------- | ------ | ----------------------------------------------------------------- |
| Australia   | AU     | `https://{yourTenant}.au.webtask.io/auth0-delegated-admin/login`  |
| Europa      | EU     | `https://{yourTenant}.eu.webtask.io/auth0-delegated-admin/login`  |
| Europa      | EU-2   | `https://{yourTenant}.eu.webtask.run/auth0-delegated-admin/login` |
| Japón       | JP-1   | `https://{yourTenant}.jp.webtask.run/auth0-delegated-admin/login` |
| Reino Unido | UK     | `https://{yourTenant}.uk.webtask.run/auth0-delegated-admin/login` |
| EE. UU.     | US-1   | `https://{yourTenant}.us.webtask.io/auth0-delegated-admin/login`  |
| EE. UU.     | US-3   | `https://{yourTenant}.us.webtask.run/auth0-delegated-admin/login` |

A continuación, [instala la extensión Delegated Admin](/docs/es/customize/extensions/delegated-administration-extension/install-delegated-admin-extension).
