Puedes instalar la [extensión Delegated Admin](/docs/es/customize/extensions/delegated-administration-extension), que te permite poner la sección **Users** del <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> a disposición de un grupo selecto de usuarios sin darles acceso al resto del Dashboard.

<div id="prerequisite">
  ## Requisito previo
</div>

[Cree una aplicación de Dashboard para Delegated Admin](/docs/es/customize/extensions/delegated-administration-extension/create-delegated-admin-applications) en Auth0.

<div id="install-extension">
  ## Instalar extensión
</div>

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y filtra por **Delegated Admin**.
2. Selecciona **Dashboard de administración delegada** en la lista de extensiones disponibles. Se abrirá la ventana **Instalar extensión**.

   <Frame>![Dashboard - Extensions - Delegated Admin - Install](/docs/images/cdy7uua7fh8z/51vm5Sttau4SgnuLbUrKOK/c988e2a52c4b0fc1d091287f14edccda/dashboard-extensions-install_delegated-admin.png)</Frame>
3. Configura las siguientes variables de configuración y selecciona **Instalar**:

   | Variable                        | Descripción                                                                                                                                                                                                |
   | ------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **EXTENSION&#95;CLIENT&#95;ID** | **ID de cliente** de la aplicación con la que planeas usar esta extensión.                                                                                                                                 |
   | **TITLE**                       | Título personalizado que aparecerá en la parte superior de la página del Dashboard de administración delegada.                                                                                             |
   | **CUSTOM&#95;CSS**              | (*Opcional*) Enlace a un CSS personalizado que puedes usar para dar estilo a la apariencia de la página del Dashboard de administración delegada.                                                          |
   | **FAVICON&#95;PATH**            | (*Opcional*) Ruta al favicon personalizado.                                                                                                                                                                |
   | **AUTH0&#95;CUSTOM&#95;DOMAIN** | (*Opcional*) Si tienes configurado un nombre de dominio personalizado, introdúcelo aquí (por ejemplo, `login.example.com`). Esto cambiará el endpoint de autorización a `https://login.example.com/login`. |
   | **FEDERATED&#95;LOGOUT**        | (*Opcional*) Indica si se debe cerrar la sesión en la conexión cuando los usuarios cierran sesión.                                                                                                         |

   Configurar la variable `AUTH0_CUSTOM_DOMAIN` no afecta a la URL de la extensión; solo cambia el endpoint de autorización. Cuando se usa un dominio personalizado, los usuarios que inicien sesión en la extensión serán redirigidos a `https://AUTH0_CUSTOM_DOMAIN/login` en lugar de la URL predeterminada `https://tenant-name.us.auth0.com/login`.
   Si navegas a [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), verás que la extensión creó una aplicación adicional. Esta aplicación está autorizada para acceder a Management API, por lo que no debes modificarla.

<div id="add-callback-urls-to-application">
  ## Agregar URLs de devolución de llamada a la aplicación
</div>

1. Ve a [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y selecciona el nombre de la aplicación que planeas usar con esta extensión. Debe ser la aplicación con el ID de cliente que ingresaste anteriormente al configurar las variables de configuración del Dashboard de administración delegada.
2. Busca **Allowed Callback URLs** y agrega la URL de devolución de llamada correspondiente a tu región y al runtime de extensibilidad de tu tenant.

   | Ubicación | Nombre | URL de devolución de llamada permitida para Node.js 12             |
   | --------- | ------ | ------------------------------------------------------------------ |
   | USA       | US-1   | `https://{yourTenant}.us12.webtask.io/auth0-delegated-admin/login` |
   | USA       | US-3   | `https://{yourTenant}.us.webtask.run/auth0-delegated-admin/login`  |
   | Australia | AU     | `https://{yourTenant}.au12.webtask.io/auth0-delegated-admin/login` |
   | Europa    | EU     | `https://{yourTenant}.eu12.webtask.io/auth0-delegated-admin/login` |
   | Japón     | JP-1   | `https://{yourTenant}.jp.webtask.run/auth0-delegated-admin/login`  |

   | Ubicación | Nombre | URL de devolución de llamada permitida para Node.js 8            |
   | --------- | ------ | ---------------------------------------------------------------- |
   | USA       | US-1   | `https://YOUR_TENANT.us8.webtask.io/auth0-delegated-admin/login` |
   | Europa    | EU     | `https://YOUR_TENANT.eu8.webtask.io/auth0-delegated-admin/login` |
   | Australia | AU     | `https://YOUR_TENANT.au8.webtask.io/auth0-delegated-admin/login` |
3. Selecciona **Save Changes**.

<div id="use-extension">
  ## Usa la extensión
</div>

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona la vista **Installed Extensions**.
2. Selecciona **Delegated Administration Dashboard**. Se abrirá una nueva pestaña para mostrar el prompt de inicio de sesión.

   Como deshabilitamos el registro en la conexión de base de datos mientras la configurábamos, la pantalla de inicio de sesión no mostrará la opción **Registrarse**.

   Una vez que introduzcas credenciales válidas, se te redirigirá a tu página personalizada de **Delegated Administration Dashboard**, que mostrará en la parte superior el **Título** que hayas indicado y, si proporcionaste un archivo CSS personalizado, se aplicará ese estilo.
