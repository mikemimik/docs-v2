El <Tooltip tip="Inicio de sesión único (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=Single+Sign-on">inicio de sesión único</Tooltip> (SSO) Dashboard es una aplicación web diseñada para resolver un problema muy común. Las organizaciones de todos los tamaños mantienen distintas aplicaciones para gestionar diversas funciones empresariales, como contabilidad, RR. HH., desarrollo, soporte, etc. Recordar los nombres de usuario, las contraseñas y las URL de inicio de sesión de todas tus aplicaciones puede resultar engorroso. Con esta extensión, puedes simplificar la experiencia de autenticación habilitando el inicio de sesión único para tus usuarios en múltiples aplicaciones empresariales. Te permite crear una lista de todos los servicios en la nube en los que un usuario puede autenticarse mediante SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Un usuario no debería poder acceder, desde el Dashboard ni de ninguna otra forma, al proveedor de servicios sin tener los permisos adecuados (grupos/roles) para hacerlo. Lo ideal es que estos usuarios no vean en el Dashboard ningún proveedor de servicios al que no tengan acceso.
</Callout>

El SSO Dashboard admite dos tipos de usuarios:

* **Users** que pueden iniciar sesión en el Dashboard para seleccionar una aplicación en la que entrar con SSO.
* **Admins** que pueden configurar las aplicaciones visibles para los usuarios.

Para configurar esta extensión, haz lo siguiente:

1. [Crea una aplicación SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Instala la extensión SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Añade aplicaciones al SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Actualiza aplicaciones en el SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

También puedes [ver esta extensión SSO Dashboard en GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
