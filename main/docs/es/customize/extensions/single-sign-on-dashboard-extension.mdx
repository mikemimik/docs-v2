El Dashboard de <Tooltip tip="inicio de sesión único (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=Single+Sign-on">inicio de sesión único</Tooltip> (SSO) es una aplicación web diseñada para resolver un problema común para muchas personas. Las organizaciones de todos los tamaños mantienen una gran variedad de aplicaciones para cubrir distintas funciones empresariales, como contabilidad, RR. HH., desarrollo, soporte, etc. Recordar los nombres de usuario, las contraseñas y las URL de inicio de sesión de todas tus aplicaciones puede resultar engorroso. Con esta extensión, puedes simplificar la experiencia de autenticación habilitando el inicio de sesión único para tus usuarios en múltiples aplicaciones empresariales. Te permite crear una lista de todos los servicios en la nube en los que un usuario puede autenticarse mediante SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Un usuario no debería poder acceder, desde el Dashboard ni de ninguna otra forma, al proveedor de servicios sin tener los permisos adecuados (grupos/roles) para hacerlo. Idealmente, estos usuarios no deberían ver en el Dashboard ningún proveedor de servicios al que no tengan acceso.
</Callout>

El SSO Dashboard admite dos tipos de usuarios:

* **Users** que pueden iniciar sesión en el Dashboard para seleccionar una aplicación a la que acceder con SSO.
* **Admins** que pueden configurar las aplicaciones visibles para los usuarios.

Para configurar esta extensión, haz lo siguiente:

1. [Crear una aplicación de SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Instalar la extensión SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Agregar aplicaciones al SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Actualizar aplicaciones en el SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

También puedes [ver esta extensión SSO Dashboard en GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
