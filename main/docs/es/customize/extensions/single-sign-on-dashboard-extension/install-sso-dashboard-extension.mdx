<Card title="Antes de empezar">
  1. [Crea una aplicación de SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application) en Auth0.
  2. Copia el valor de **ID de cliente** de tu aplicación de SSO Dashboard.
</Card>

Usa la extensión SSO Dashboard de <Tooltip tip="Inicio de sesión único (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=Single+Sign-On">inicio de sesión único</Tooltip> (SSO) para gestionar el acceso de tus usuarios a múltiples aplicaciones empresariales. Para obtener más información, [consulta esta extensión en GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).

<div id="install-extension">
  ## Instalar la extensión
</div>

Puedes instalar la extensión SSO Dashboard desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>:

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona **SSO Dashboard** de la lista de extensiones disponibles. Se abrirá la ventana **Install Extension**:

   <Frame>![Dashboard - Extensions - SSO Dashboard - Install Extension](/docs/images/cdy7uua7fh8z/75heEyuJAN6FGSszXx06Hm/1cf10550c770beb56731285709442afb/SSO_Extension_-_EN.png)</Frame>
2. Configura las siguientes variables de configuración:

   | Variable              | Obligatorio | Descripción                                                                               |
   | --------------------- | ----------- | ----------------------------------------------------------------------------------------- |
   | `EXTENSION_CLIENT_ID` | Obligatorio | **ID de cliente** de la aplicación Auth0 que creaste para la extensión.                   |
   | `TITLE`               | Opcional    | Título personalizado que aparecerá en la parte superior de la página de SSO Dashboard.    |
   | `CUSTOM_CSS`          | Opcional    | Ruta a un archivo CSS personalizado para definir el estilo de tu página de SSO Dashboard. |
   | `FAVICON_PATH`        | Opcional    | Ruta a un favicon personalizado.                                                          |
   | `AUTH0_CUSTOM_DOMAIN` | Opcional    | [dominio personalizado](/docs/es/customize/custom-domains) configurado en Auth0.             |

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Configurar la variable `AUTH0_CUSTOM_DOMAIN` no afecta a la URL de la extensión; solo cambia la URL del endpoint de autorización que usa la extensión. Cuando se usa un dominio personalizado, los usuarios que inicien sesión en la extensión serán redirigidos a `https://{yourAuth0CustomDomain}/login` en lugar de la URL predeterminada `https://{yourTenant}.{yourRegion}.auth0.com/login`.
   </Callout>
3. Selecciona **Install**.

Si vas a [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications), verás que se creó automáticamente una aplicación llamada `auth0-sso-dashboard`. La extensión usa esta aplicación para autorizar el acceso a la <Tooltip tip="Management API: un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip>, y no debes modificarla.

<div id="use-extension">
  ## Usar la extensión
</div>

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona la vista **Installed Extensions**.
2. Selecciona **SSO Dashboard**. La primera vez que selecciones la extensión instalada, se te pedirá que le concedas los permisos necesarios. Una vez que aceptes, se te redirigirá a tu **SSO Dashboard** personalizado, que mostrará el **Título** que proporcionaste en la parte superior de la página. Si proporcionaste un archivo CSS personalizado, ese estilo también se aplicará.
3. Para iniciar sesión en el SSO Dashboard:

   1. Para **administradores,** usa:
      `https://{yourTenant}.YOUR_REGION_KEY.webtask.io/auth0-sso-dashboard/admins/login`
   2. Para **Users,** usa:
      `https://{yourTenant}.YOUR_REGION_KEY.webtask.io/auth0-sso-dashboard/login`

| Región | Dominio de la región |
| ------ | -------------------- |
| US-1   | us.webtask.io        |
| US-3   | us.webtask.run       |
| US-4   | us.webtask.run       |
| US-5   | us.webtask.run       |
| EU     | eu.webtask.io        |
| EU-2   | eu.webtask.run       |
| AU     | au.webtask.io        |
| JP-1   | jp.webtask.run       |
| UK-1   | uk.webtask.run       |
| CA-1   | ca.webtask.run       |
