title: Actualizar aplicaciones en el SSO Dashboard
---

Usa la extensión SSO Dashboard para administrar el <Tooltip tip="inicio de sesión único (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente sesión en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=SSO">SSO</Tooltip> de tus usuarios en múltiples aplicaciones empresariales.

Para obtener más información, consulta [Single Sign-On Dashboard Extension](/docs/es/customize/extensions/single-sign-on-dashboard-extension) o [ve esta extensión en GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).

<div id="prerequisites">
  ## Requisitos previos
</div>

* [Crea la aplicación SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application) en Auth0.
* [Instala la extensión SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).
* [Añade aplicaciones al SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard).

<div id="update-application">
  ## Actualizar aplicación
</div>

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona la vista **Installed Extensions**.
2. Selecciona **SSO Dashboard**.
   Si eres administrador, también puedes iniciar sesión en el **SSO Dashboard** con `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
3. Selecciona **Settings** en el menú desplegable que aparece debajo del nombre de tu tenant.

* Para especificar si los usuarios pueden ver la aplicación cuando está habilitada, selecciona **Publish** o **Unpublish**.
* Para actualizar la configuración de una aplicación, selecciona el icono de engranaje.
* Para eliminar una aplicación, selecciona su **X**. Aparecerá una ventana emergente de confirmación para confirmar la eliminación.
