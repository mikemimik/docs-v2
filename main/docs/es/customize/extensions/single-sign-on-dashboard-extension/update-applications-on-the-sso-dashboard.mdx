title: Actualizar aplicaciones en el SSO Dashboard
---

Usa la SSO Dashboard Extension para gestionar el inicio de sesión con <Tooltip tip="Single Sign-On (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=SSO">SSO</Tooltip> de tus usuarios en múltiples aplicaciones empresariales.

Para obtener más información, consulta [Single Sign-On Dashboard Extension](/docs/es/customize/extensions/single-sign-on-dashboard-extension) o [ve esta extensión en GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).

<div id="prerequisites">
  ## Prerrequisitos
</div>

* [Crear la aplicación del SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application) en Auth0.
* [Instalar la SSO Dashboard Extension](/docs/es/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).
* [Añadir aplicaciones al SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard).

<div id="update-application">
  ## Actualizar la aplicación
</div>

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona la vista **Installed Extensions**.
2. Selecciona **SSO Dashboard**.
   Si eres administrador, también puedes iniciar sesión en SSO Dashboard con `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
3. Selecciona **Settings** en el menú desplegable que aparece debajo del nombre de tu tenant.

* Para indicar si los usuarios pueden ver la aplicación cuando está habilitada, selecciona **Publish** o **Cancelar publicación**.
* Para actualizar la configuración de una aplicación, selecciona el icono de engranaje.
* Para eliminar una aplicación, selecciona la **X**. Aparecerá una ventana emergente para confirmar la eliminación.
