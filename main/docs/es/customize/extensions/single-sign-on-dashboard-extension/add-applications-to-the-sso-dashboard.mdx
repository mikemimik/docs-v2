title: Agregar aplicaciones a Single Sign-On Dashboard
---

Usa la extensión <Tooltip tip="Single Sign-On (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, lo inicia automáticamente en otras aplicaciones." cta="Ver glosario" href="/docs/es/glossary?term=SSO">SSO</Tooltip> Dashboard para gestionar el inicio de sesión único de tus usuarios en varias aplicaciones empresariales.

<div id="prerequisites">
  ## Requisitos previos
</div>

* [Cree la aplicación SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application).
* [Instale la extensión SSO Dashboard](/docs/es/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).

<div id="add-applications">
  ## Agregar aplicaciones
</div>

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona tu nueva extensión SSO Dashboard.
   Si eres administrador, también puedes iniciar sesión en el SSO Dashboard con `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
2. En la esquina superior derecha, selecciona **Settings** en el menú desplegable que aparece debajo del nombre de tu tenant.
3. Haz clic en **Create App** para agregar una nueva aplicación.

   Aparecerá el formulario **New Application**.
4. Completa los siguientes campos de la nueva aplicación:

   | Campo           | Descripción                                                                                                                                                                                                                                                                                               |
   | --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Type**        | Un menú desplegable donde seleccionas SAML, OpenID-Connect o WS-Federation, según el tipo de aplicación.                                                                                                                                                                                                  |
   | **Application** | Un menú desplegable donde seleccionas la aplicación que deseas agregar.                                                                                                                                                                                                                                   |
   | **Name**        | El nombre se completa automáticamente según la aplicación que seleccionaste. Puedes cambiarlo o usar el valor predeterminado.                                                                                                                                                                             |
   | **Logo**        | La URL del logotipo que deseas usar como ícono de la aplicación.                                                                                                                                                                                                                                          |
   | **Callback**    | Una de las **Allowed Callback URLs** de [Application Settings](https://manage.auth0.com/#/applications) de la aplicación.                                                                                                                                                                                 |
   | **Connection**  | (*Opcional*) El tipo de conexión. Puedes agregar o editar los tipos de conexión disponibles en la [sección Connections del management dashboard de Auth0](https://manage.auth0.com/#/connections/database). Si no se configura una conexión y el usuario no ha iniciado sesión, verá la Auth0 Login Page. |
   | **Custom URL**  | (*Opcional*) Casilla de verificación para usar una URL personalizada en lugar de la URL de Auth0. Si la marcas, aparecerá un campo donde podrás introducir tu URL personalizada.                                                                                                                          |
   | **Enabled**     | Casilla de verificación para que esta aplicación esté visible (publicada) para tus usuarios.                                                                                                                                                                                                              |
5. Selecciona **Create**. Tu nueva aplicación aparecerá en la página **Applications** del SSO Dashboard junto con cualquier otra aplicación que se haya creado.
6. Haz clic en una aplicación para probar la conexión.
