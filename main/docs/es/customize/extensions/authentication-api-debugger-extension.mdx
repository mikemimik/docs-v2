---

La extensión Authentication API Debugger te permite probar varios endpoints de Auth0 Authentication API.

<div id="install-the-extension">
  ## Instalar la extensión
</div>

Para instalar esta extensión:

1. En el [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), selecciona **Auth0 Authentication API Debugger**. Se abrirá la ventana **Install Extension**.
2. Selecciona **Install**.

<div id="authorize-the-extension">
  ## Autorizar la extensión
</div>

Como la extensión se comunicará con la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/docs/es/glossary?term=Management+API">Management API</Tooltip> en su nombre para obtener información sobre las aplicaciones que ha configurado en su <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/docs/es/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, deberá autorizar su acceso.

Para autorizar la extensión:

1. Si está siguiendo esta guía, ya debería encontrarse en la vista [Installed Extensions](https://manage.auth0.com/#/extensions) de [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions). Seleccione **Auth0 Authentication API Debugger** para abrir la extensión. Aparecerá un cuadro de diálogo de consentimiento para solicitar acceso a su cuenta.
2. Dé su consentimiento para permitir que la extensión acceda a su cuenta. Una vez que lo haga, la extensión se abrirá.

<div id="configure-the-extension">
  ## Configura la extensión
</div>

Después de instalar la extensión, debes aplicar la configuración básica. Para configurar la extensión:

1. Si sigues esta guía, ya deberías haber abierto la extensión. Selecciona la vista **Configuración**.

   <Frame>![Dashboard - Extensions - Authentication API Debugger - Configuration](/docs/images/cdy7uua7fh8z/5riI24ArqUAKRXhZfNWIQ5/ab7d6825288752132d14e54db1e563d0/configuration.png)</Frame>
2. Configura los siguientes parámetros y selecciona **Guardar**:

* **Dominio**: El dominio de tu tenant. Este campo es de solo lectura y se muestra únicamente con fines informativos.
* **Aplicación**: La aplicación para la que quieres iniciar flujos de autenticación. Puedes gestionar tus aplicaciones en [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications).
* **URL de devolución de llamada**: La URL de devolución de llamada de esta extensión. También debes agregar esta URL a **Allowed Callback URLs** en la configuración de la aplicación seleccionada.
* **Estado**: Información de estado opcional que se puede enviar con el flujo de autenticación.
* **Conexión**: El nombre de la conexión con la que quieres iniciar sesión. Puedes usar este parámetro para omitir la pantalla principal de inicio de sesión e ir directamente a la pantalla de inicio de sesión del <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/docs/es/glossary?term=Identity+Provider">Proveedor de identidad</Tooltip> correspondiente.

<div id="execute-the-flows">
  ## Ejecuta los flujos
</div>

Después de configurar la extensión, puedes ejecutar cualquiera de los flujos seleccionando la vista **OAuth2/OIDC**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Para algunos flujos, es posible que debas especificar opciones adicionales a las de la configuración básica. Estas opciones dependerán del flujo que se ejecute, por lo que te sugerimos consultar la [documentación de Auth0 Authentication API](/docs/es/api) para obtener más información sobre los parámetros aplicables a cada flujo.
</Callout>

<Frame>![Dashboard - Extensions - Authentication API Debugger - Configuración de OAuth OIDC](/docs/images/cdy7uua7fh8z/72iYPxZU9OiHkKkHtgPTjI/fc166f3ae74e595c8d916276e78edb99/oauth-oidc-settings.png)</Frame>

Una vez que ejecutes un flujo en particular, verás una pantalla que muestra la información devuelta por Auth0. La información exacta dependerá del flujo que se haya ejecutado. A continuación, se muestra un ejemplo de los resultados tras ejecutar un flujo normal de **Inicio de sesión OAuth2/OIDC**:

<Frame>![Dashboard - Extensions - Authentication API Debugger - Flujo ejecutado](/docs/images/cdy7uua7fh8z/13TRS8SQUd4OsfLYRWMoe6/3478778905ad253160022db1f1f6f31a/flow-executed.png)</Frame>

Para ejecutar otro flujo, selecciona la vista **Login** para volver a la pantalla de **Configuración**.
