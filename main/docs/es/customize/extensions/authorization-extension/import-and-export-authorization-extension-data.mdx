title: Importar y exportar datos de Authorization Extension
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 ofrece dos formas de implementar el control de acceso basado en roles (RBAC), que puedes usar en lugar del sistema interno de control de acceso de tu API o en combinación con él:

  * [Authorization Core](/docs/es/manage-users/access-control/rbac)
  * [Authorization Extension](/docs/es/customize/extensions/authorization-extension)

  El conjunto de funcionalidades de Authorization Core ofrece la misma funcionalidad que Authorization Extension, mejora el rendimiento y la escalabilidad, y proporciona un sistema RBAC más flexible que Authorization Extension.

  Actualmente, ambas implementan las funcionalidades clave de RBAC y te permiten restringir los scopes personalizados definidos para una API a aquellos que se hayan asignado al usuario como permisos.
</Callout>

Puedes importar datos nuevos desde un archivo JSON o exportar los datos de autorización existentes a un archivo JSON. Esto puede ser útil al mover entornos.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Los roles y los permisos están vinculados a aplicaciones específicas. Si exportas tu archivo JSON y lo importas en un entorno diferente, tendrás que cambiar el ID de cliente de esos registros.
</Callout>

1. Ve a la sección **Importar/Exportar** haciendo clic en **Configuración** en el menú desplegable, al que puedes acceder haciendo clic en el nombre de tu tenant en la esquina superior derecha del **Authorization Dashboard**.

   <Frame>![Dashboard - Authorization Extension Dashboard - Configuración](/docs/images/cdy7uua7fh8z/r8Ic6OYMcVfXDvIKePLrb/ed9c75536d20c12149e8b8e62a8edc44/click-configuration.png)</Frame>
2. Selecciona **Importar/Exportar**.

   <Frame>![Dashboard - Authorization Extension Dashboard - Configuración - Importar Exportar](/docs/images/cdy7uua7fh8z/2RAjzvNq2rg6bzMf3v6QeM/32e1677ba846398471173a3a90d95d83/import-export.png)</Frame>
