---
description: Auth0 Dashboard を使用して、アプリケーションで有効になっている接続を更新する方法を説明します。
title: アプリケーションの接続を更新する
---

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用して、アプリケーションで有効になっている接続を変更できます。

1. [Auth0 Dashboard &gt; アプリケーション](https://manage.auth0.com/#/applications) に移動し、表示するアプリケーションの名前をクリックします。

   <Frame>![Auth0 Dashboard アプリケーション一覧](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. **接続** タブをクリックし、そのアプリケーションに適した接続を有効または無効にします。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  [サードパーティアプリケーション](/docs/ja/get-started/applications/third-party-applications) は、ドメインレベルに昇格された接続、またはそのアプリケーションに明示的にマッピングされた接続を通じてユーザーを認証できます。詳しくは、[接続をドメインレベルに昇格する](/docs/ja/authenticate/identity-providers/promote-connections-to-domain-level) をご覧ください。
</Callout>
