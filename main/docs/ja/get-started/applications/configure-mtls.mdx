<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Highly Regulated Identity 機能を利用するには、Highly Regulated Identity アドオンを含む Enterprise プランが必要です。詳しくは、[Auth0 Pricing](https://auth0.com/pricing/)を参照してください。
</Callout>

[Auth0 Management API](https://auth0.com/docs/api/management/v2)または[Auth0 Dashboard](https://manage.auth0.com/)を使用して、mTLS 認証を設定できます。Auth0 における mTLS 認証の仕組みについて詳しくは、[Authenticate with mTLS](/docs/ja/get-started/authentication-and-authorization-flow/authenticate-with-mtls)をご覧ください。

| 読む内容...                                                                                                  | 学べる内容...                                                           |
| -------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| [Set up your customer edge](/docs/ja/get-started/applications/configure-mtls/set-up-the-customer-edge)      | クライアント証明書を検証し、リクエストを Auth0 のエッジネットワークに転送するための customer edge の設定方法。 |
| [Configure mTLS for a client](/docs/ja/get-started/applications/configure-mtls/configure-mtls-for-a-client) | Auth0 アプリケーションの mTLS 認証を設定する方法。                                    |
| [Configure mTLS for a tenant](/docs/ja/get-started/applications/configure-mtls/configure-mtls-for-a-tenant) | Auth0 テナントの mTLS 認証を設定する方法。                                        |
