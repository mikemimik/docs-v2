title: Android App Links のサポートを有効にする
---

[Android App Links](https://developer.android.com/training/app-links) を使うと、アプリケーションを特定の種類のリンクのデフォルトのハンドラーとして設定できます。たとえば、メール内の URL をクリックすると、そのリンクを指定したアプリケーションで開くことができます。このガイドでは、Auth0 Dashboard を使用して、Auth0 に登録されたアプリケーションで Android App Links のサポートを有効にする方法を説明します。

1. [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) に移動し、表示するアプリケーションの名前を選択します。

   <Frame>![Dashboard Applications List](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. **設定** ページの一番下までスクロールし、**Show Advanced Settings** を選択します。
3. **Device Settings** を選択し、Android アプリケーションの [App Package Name](https://developer.android.com/studio/build/application-id) と、アプリの署名証明書の SHA256 フィンガープリントを入力して、**変更を保存** を選択します。

   <Frame>![Dashboard Applications Application Settings Tab Advanced Settings Device Settings Tab](/docs/images/cdy7uua7fh8z/YKKPtWtswHn2FZGFSVWNw/086b32e41b3fd446ac8aa678be33648d/Device_Settings_-_EN.png)</Frame>

ターミナルで Java の keytool を使用してフィンガープリントを生成するには、次のコマンドを使用します。
`keytool -list -v -keystore my-release-key.keystore`

署名証明書の詳細については、Android の開発者向けドキュメント [Sign Your App](https://developer.android.com/studio/publish/app-signing.html) を参照してください。

<div id="test-link">
  ## リンクのテスト
</div>

1. ブラウザーで次の URL にアクセスします: `https://{yourDomain}/.well-known/assetlinks.json`
2. リンクが正しく設定されていれば、次の JSON が返されます (読みやすいように整形しています) 。

   ```json lines
   [{
     "relation": ["delegate_permission/common.handle_all_urls"],
     "target": {
       "namespace": "android_app",
       "package_name": "com.mycompany.app1",
       "sha256_cert_fingerprints":
       ["14:6D:E9:83:C5:73:06:50:D8:EE:B9:95:2F:34:FC:64:16:A0:83:42:E6:1D:BE:A8:8A:04:96:B2:3F:CF:44:E5"]
     }
   }]
   ```

アプリリンクのテストについて詳しくは、https://developer.android.com の [Verify Android App Links](https://developer.android.com/training/app-links/verify-applinks) を参照してください。
