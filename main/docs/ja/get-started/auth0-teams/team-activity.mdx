description: Team Activityでは、Team Ownerがチームメンバーによって生成されたイベントログを表示し、監査できます。
title: Team Activity
---

Team Activityでは、チームメンバーが自身のAuth0アカウントで行ったアクティビティに関連する監査ログを、Team Ownerが確認できます。

Team Activityレポートは、Teams Dashboard の **Reports** セクションにあります。

Team Activityでは、イベントを生成したメンバー、イベントタイプ、イベントが生成された日付の概要を確認できます。

**View Details** を選択すると、イベントの説明、イベントのステータス、イベントをトリガーしたユーザーのIPアドレス、ユーザーエージェントの詳細など、さらに詳しい情報を確認できます。

<div id="event-types">
  ## イベントタイプ
</div>

Team Activity のイベントタイプは以下のとおりです。

| イベントタイプ    | 説明                                   |
| ---------- | ------------------------------------ |
| チーム招待      | チームメンバー招待が作成された<br />チームメンバー招待が削除された |
| チームメンバー    | チームメンバーが削除された<br />チームメンバーのロールが更新された |
| セキュリティポリシー | セキュリティポリシーが更新された<br />ソーシャル接続が更新された  |
| チーム設定      | チーム設定が更新された                          |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 Dashboard と Teams Dashboard のユーザー認証、およびその他のイベントへの対応は、今後提供される予定です。
</Callout>

<div id="event-schema">
  ## イベントスキーマ
</div>

以下に Team Activity イベントのスキーマを示します。

```json lines
{  
    "id": "string", 
    "team_id": "string",
    "user_id": "string",
    "type": "string",
    "description": "string",
    "status": "string",
    "details": "object",
    "ip": "string",
    "hostname": "string",
    "user_agent": "string",
    "version": "string",
    "created_at": "date",
    "team_slug": "string",
    "user": "object"
}
```

| 名前            | 型        | 説明                                                   |
| ------------- | -------- | ---------------------------------------------------- |
| `id`          | string   | イベントログの一意の識別子。                                       |
| `team_id`     | string   | Team の一意の識別子。                                        |
| `user_id`     | string   | イベントを生成したユーザーの一意の識別子。                                |
| `type`        | string   | イベントタイプ。                                             |
| `description` | string   | イベントタイプの概要。                                          |
| `status`      | string   | イベントのステータス。指定可能な値は `Success` または `Failure` です。       |
| `details`     | object   | 記録されたイベントの詳細。                                        |
| `ip`          | string   | イベントログを生成したユーザーの IP アドレス。                            |
| `hostname`    | string   | イベントが実行された Auth0 App の FQDN。                         |
| `user_agent`  | string   | ウェブブラウザーおよびデバイス OS の種類とバージョン。                        |
| `version`     | string   | アクティビティログの生成時に使用された schema のバージョン。                   |
| `created_at`  | datetime | イベントログが生成された日時。                                      |
| `team_slug`   | string   | Team の一意のテキスト識別子。                                    |
| `user`        | object   | 名前、アバター URL、および Team のメンバーシップのステータス (boolean) を含みます。 |

<div id="audit-log-retention">
  ## 監査ログの保持期間
</div>

Team Activity の監査ログの保持期間は、プランによって異なります。詳細については、[料金](https://auth0.com/pricing) ページをご覧ください。
