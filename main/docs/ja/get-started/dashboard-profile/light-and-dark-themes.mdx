---
description: Auth0 Dashboard でライトテーマとダークテーマを切り替える方法
title: ライトモードとダークモード
---

Auth0 のお客様は、テナントの Auth0 Dashboard でライトモードまたはダークモードを使用できます。また、ローカルのシステム環境と同じ設定を使うように Auth0 Dashboard を設定することもできます。

Auth0 Dashboard のテーマを変更する方法は 2 つあります。

1. Auth0 Dashboard の右上にあるプロファイルのドロップダウンメニューを選択します。
2. メニューを使ってテーマを切り替えます。

<Frame>![ダークモードとライトモードの切り替えがどこにあるか](/docs/images/cdy7uua7fh8z/1BImxFvHXoULQVTkqCsQL0/a37f6cfdfce935fd20731aa68f8067de/2024-09-10_09-08-27.png)</Frame>

同じドロップダウンで **プロファイル** をクリックし、一番下までスクロールして、3 つのテーマから選択することもできます。
