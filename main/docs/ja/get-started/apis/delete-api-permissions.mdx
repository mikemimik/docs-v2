API から権限を削除するには、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を表示" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を表示" href="/docs/ja/glossary?term=Management+API">Management API</Tooltip> を使用します。

<div id="use-the-dashboard">
  ## Auth0 Dashboard を使用する
</div>

1. [Auth0 Dashboard &gt; アプリケーション &gt; APIs](https://manage.auth0.com/#/apis) に移動し、対象の API 名をクリックします。

   <Frame>![Auth0 Dashboard アプリケーション APIs 一覧](/docs/images/cdy7uua7fh8z/3rhmhghYZDSi6YWHRA5yMQ/c71340259481b0b6787d5f3887cfda0f/dashboard-apis-list.png)</Frame>
2. **権限** タブに移動し、削除する権限の横にあるごみ箱アイコンをクリックして確認します。

   <Frame>![Auth0 Dashboard API の権限を追加する API 権限を定義する画面](/docs/images/cdy7uua7fh8z/32Pb185OFs2mC6z2fmunEw/2708c52c2869d016066cc456dd00b6a2/dashboard-applications-apis-permissions.png)</Frame>

<div id="use-the-management-api">
  ## Management API を使用する
</div>

保持するすべての権限を含め、削除するすべての権限を除外した [Update Resource Server endpoint](https://auth0.com/docs/api/management/v2#!/Resource_Servers/patch_resource_servers_by_id) に対して PATCH リクエストを実行します。`API_ID,` `MGMT_API_ACCESS_TOKEN`、`PERMISSION_NAME`、`PERMISSION_DESC` の placeholder values は、それぞれ実際の API ID、Management API <Tooltip tip="アクセストークン: API へのアクセスに使用される、opaque 文字列または JWT 形式の Authorization 資格情報。" cta="用語集を見る" href="/docs/ja/glossary?term=Access+Token">アクセストークン</Tooltip>、権限名、権限の説明に置き換えてください。

  ```bash cURL
  curl --request PATCH \
    --url 'https://{yourDomain}/api/v2/resource-servers/API_ID' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
    --header 'cache-control: no-cache' \
    --header 'content-type: application/json' \
    --data '{ "scopes": [ { "value": "PERMISSION_NAME", "description": "PERMISSION_DESC" }, { "value": "PERMISSION_NAME", "description": "PERMISSION_DESC" } ] }'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/resource-servers/API_ID");
  var request = new RestRequest(Method.PATCH);
  request.AddHeader("content-type", "application/json");
  request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
  request.AddHeader("cache-control", "no-cache");
  request.AddParameter("application/json", "{ "scopes": [ { "value": "PERMISSION_NAME", "description": "PERMISSION_DESC" }, { "value": "PERMISSION_NAME", "description": "PERMISSION_DESC" } ] }", ParameterType.RequestBody);
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (

  	req, _ := http.NewRequest("PATCH", url, payload)

  	req.Header.Add("content-type", "application/json")
  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")
  	req.Header.Add("cache-control", "no-cache")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

</AuthCodeGroup>

| 値                       | 説明                                                                                                                         |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| `API_ID`                | 権限 を削除する対象の API の ID。                                                                                             |
| `MGMT_API_ACCESS_TOKEN` | スコープ `update:resource_servers` を持つ [Access Token for the Management API](https://auth0.com/docs/api/management/v2/tokens)。 |
| `PERMISSION_NAME`       | 指定した API に対して保持したい permission の名前。                                                                                         |
| `PERMISSION_DESC`       | 指定した API に対して保持したい permission のユーザー向け説明。                                                                                   |
