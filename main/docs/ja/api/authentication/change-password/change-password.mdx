<div id="endpoint">
  ## エンドポイント
</div>

`POST /dbconnections/change_password`

指定されたユーザーのメールアドレスと`connection`宛てに、パスワード変更用のメールを送信します。

必要に応じて、Organization ID を指定できます。これにより、[カスタマイズされたメールテンプレート](https://auth0.com/docs/customize/email/email-templates)で Organization 固有の変数を利用できるようになり、**Redirect To** URL に`organization_id`および`organization_name`パラメータが含まれます。

**注:** このエンドポイントはデータベース接続でのみ使用できます。

<div id="remarks">
  ### 注記
</div>

* ユーザーがパスワード変更リンクをクリックすると、新しいパスワードの設定を求めるページにリダイレクトされます。
* このエンドポイントは、レート制限に関する情報を示す 3 つの HTTP レスポンスヘッダーを返します。
  * `X-RateLimit-Limit`: 1 分あたりに許可されるリクエスト数。
  * `X-RateLimit-Remaining`: 利用可能なリクエスト数。リクエストがあるたびに、この数は 1 ずつ減少します。1 分経過するごとにリクエスト数が 1 つ戻されるため、この数はそのたびに 1 ずつ増加します。
  * `X-RateLimit-Reset`: レート制限 (`X-RateLimit-Limit`) がリセットされるまでの残り時間。値は [UTC エポック秒](https://en.wikipedia.org/wiki/Unix_time) です。

<div id="body-parameters">
  ## Body Parameters
</div>

<ParamField body="client_id" type="string" required>
  クライアントの `client_id` です。
</ParamField>

<ParamField body="email" type="string" required>
  ユーザーのメールアドレスです。
</ParamField>

<ParamField body="connection" type="string" required>
  クライアントに設定されているデータベース接続の名前です。
</ParamField>

<ParamField body="organization" type="string">
  ユーザーに関連付けられている Organization の `organization_id` です。
</ParamField>

<div id="response-messages">
  ## レスポンスメッセージ
</div>

| Status | Description                    |
| ------ | ------------------------------ |
| 200    | パスワード変更用のメールを送信しました。           |
| 400    | 不正なリクエストです。パラメータが不足しているか、無効です。 |
| 404    | ユーザーが見つかりません。                  |
