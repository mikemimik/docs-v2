<div id="endpoint">
  ## エンドポイント
</div>

`GET /mfa/authenticators`

アプリケーションに関連付けられている認証器の一覧を返します。

このエンドポイントにアクセスするには、Authorization ヘッダーにアクセストークンを指定し、次のクレームを含める必要があります。

* `scope`: `read:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="headers">
  ## ヘッダー
</div>

<ParamField header="Authorization" type="string" required>
  Bearer ACCESS&#95;TOKEN
</ParamField>

<div id="response-schema">
  ## レスポンススキーマ
</div>

<ResponseSchema>
  レスポンスは認証器オブジェクトの配列です。

  <ResponseField name="id" type="string" required>
    認証器の ID です。
  </ResponseField>

  <ResponseField name="authenticator_type" type="string" required>
    認証器の種類です。
  </ResponseField>

  <ResponseField name="active" type="boolean" required>
    認証器がアクティブかどうかを示します。
  </ResponseField>

  <ResponseField name="oob_channels" type="string">
    この認証器でサポートされる OOB チャネルです。
  </ResponseField>

  <ResponseField name="name" type="string">
    認証器の名前です。
  </ResponseField>
</ResponseSchema>

<div id="response-messages">
  ## レスポンスメッセージ
</div>

| 状態  | 説明      |
| --- | ------- |
| 200 | 認証器の一覧。 |
