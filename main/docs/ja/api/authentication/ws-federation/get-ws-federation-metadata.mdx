---
title: "WS-Federation メタデータを取得"
description: "Auth0テナントのWS-Federationメタデータを返します。"
---

<div id="endpoint">
  ## エンドポイント
</div>

`GET /wsfed/FederationMetadata/2007-06/FederationMetadata.xml`

このエンドポイントは、WS-Federation のメタデータを返します。

<div id="learn-more">
  ### 詳細
</div>

* [WS-Federation](https://auth0.com/docs/authenticate/protocols/ws-fed-protocol)

<div id="response-messages">
  ## レスポンスメッセージ
</div>

| Status | Description                    |
| ------ | ------------------------------ |
| 200    | WS-Federation メタデータを含む成功レスポンス。 |
