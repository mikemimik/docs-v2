title: User Details の表示
---

[Users](https://manage.auth0.com/#/users) ページには、アプリに関連付けられているユーザーの一覧が表示されます。特定のユーザーを開くには、[Name] 列にあるそのユーザーのプロフィール画像または名前をクリックします。User Details ページが開き、そのユーザーの情報が表示されます。

User Details ページには、次の 5 つのタブへのリンクがあります。

1. 詳細
2. デバイス
3. History
4. Locations
5. Raw JSON

<div id="user-details-details-tab">
  ## User Details: 詳細タブ
</div>

詳細タブには、ユーザーのプロファイルに含まれる情報を大まかに確認できる3つのセクションがあります。

* **ユーザー情報** には、ユーザーが使用した識別子、関連する接続、アクセス権など、ユーザーに関する概要がひと目でわかる形で表示されます。
* **メタデータ** には、`app_metadata` と `user_metadata` の情報が表示されます。これらの値は編集できます。
* **<Tooltip tip="IDプロバイダー (IdP): デジタルIDを保存および管理するサービス。" cta="用語集を表示" href="/docs/ja/glossary?term=Identity+Provider">IDプロバイダー</Tooltip> 属性** には、認証プロバイダーから取得した情報が表示されます。なお、IDプロバイダーの属性は読み取り専用です。
* **パスキー** には、ユーザーが作成した [パスキー](/docs/ja/authenticate/database-connections/passkeys) が表示されます。
* **<Tooltip tip="多要素認証 (MFA): SMS によるコードなど、ユーザー名とパスワードに加えて認証要素を使用するユーザー認証プロセス。" cta="用語集を表示" href="/docs/ja/glossary?term=Multi-Factor+Authentication">多要素認証</Tooltip>** には、ユーザーが登録済みの [MFA](/docs/ja/secure/multi-factor-authentication) 認証要素が表示されます。
  [複数のカスタムドメイン](/docs/ja/customize/custom-domains/multiple-custom-domains) を有効にすると、テナントの通知ドメインを選択する **Domain** オプションを使用できます。

<div id="user-details-devices-tab">
  ## User Details：デバイスタブ
</div>

デバイスタブには、ユーザーが認証を要求したデバイスが一覧表示されます。デバイスで認可を行うと、そのデバイスはユーザーのアカウントにリンクされます。

ユーザーのログイン情報は、そのデバイスに割り当てられた<Tooltip tip="リフレッシュトークン：ユーザーに再度ログインを求めることなく、新しいアクセストークンを取得するために使用されるトークンです。" cta="用語集を表示" href="/docs/ja/glossary?term=Refresh+Token">リフレッシュトークン</Tooltip>に関連付けられています。リフレッシュトークンを取り消すには、デバイスの横にある**リンク解除**をクリックします。

<div id="user-details-history-tab">
  ## User Details: History Tab
</div>

Historyタブには、過去2日間のユーザーアカウントのアクティビティログが表示されます。

ログには、次の情報が含まれます。

* 発生したイベント
* イベントが発生した日時
* イベントに関連付けられたアプリ
* 認証に使用されたアイデンティティプロバイダー
* イベントの発信元IPアドレス
* イベントが発生した場所

<div id="user-details-locations-tab">
  ## User Details: Locations タブ
</div>

Locations タブには、ユーザーがアプリにログインした場所を示すピンが表示された地図が表示されます。

<div id="user-details-raw-json-tab">
  ## User Details: Raw JSON タブ
</div>

Raw JSON タブには、ユーザーのプロファイルに含まれるすべての情報が JSON 形式で表示されるため、ユーザーに関する利用可能な情報をすばやく確認できます。
