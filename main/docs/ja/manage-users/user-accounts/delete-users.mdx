---
description: ユーザーを削除する方法を説明します。
title: ユーザーの削除
---

1. [Auth0 Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) に移動します。
2. 削除するユーザーの名前をクリックします。
3. **Details** タブでページの一番下までスクロールし、**Delete user** の横にある **Delete** をクリックします。<br />

   ポップアップが表示され、この操作は元に戻せないことと、ユーザーを削除するかどうかの確認が求められます。確認するには、**Yes, Delete It** ボタンをクリックします。これにより、ユーザーは完全に削除されます。
