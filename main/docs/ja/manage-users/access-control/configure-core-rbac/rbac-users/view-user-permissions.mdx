ユーザーに割り当てられている[権限](/docs/ja/manage-users/access-control/rbac)は、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 のメイン製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>で確認できます。この操作は、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 のメイン製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Management+API">Management API</Tooltip>を使用して行うこともできます。割り当てられた権限は、API Authorization Core 機能セットで使用されます。

<div id="prerequisites">
  ## 前提条件
</div>

ロールベースのアクセス制御 (RBAC) を正しく機能させるには、Auth0 Dashboard または Management API のいずれかを使用して、ご利用の API でこれを有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。違いについては、[Authorization Core と Authorization Extension の比較](/docs/ja/manage-users/access-control/authorization-core-vs-authorization-extension)を参照してください。

<div id="dashboard">
  ## Auth0 Dashboard
</div>

1. [Auth0 Dashboard &gt; Manage Users &gt; Users](https://manage.auth0.com/#/users) に移動し、詳細を表示するユーザーの名前をクリックします。
2. **Permissions** ビューをクリックします。

   <Frame>![このユーザーに割り当てられている権限の Auth0 Dashboard ユーザー詳細画面](/docs/images/cdy7uua7fh8z/4vShfxoUcSpCDnM5FnAbsR/de85942623430e3cedc097addbaa2f9d/User_Details_-_Permissions_-_English.png)</Frame>

各権限について、次の情報が表示されます。

| 列    | 説明                                            |
| ---- | --------------------------------------------- |
| 名前   | 権限定義で設定されている権限名。                              |
| 説明   | 権限定義で設定されている権限の説明。                            |
| API  | その権限が関連付けられている API の名前。                       |
| 割り当て | その権限がユーザーに直接割り当てられているか、ロール経由で割り当てられているかを示します。 |

<div id="management-api">
  ## Management API
</div>

[Get User Permissions エンドポイント](https://auth0.com/docs/api/management/v2#!/Users/get_permissions)に対して `GET` リクエストを送信します。プレースホルダー値 `USER_ID` と `MGMT_API_ACCESS_TOKEN` は、それぞれユーザー ID と Management API の<Tooltip tip="アクセストークン: API へのアクセスに使用される、不透明な文字列または JWT 形式の認可資格情報。" cta="用語集を見る" href="/docs/ja/glossary?term=access+token">アクセストークン</Tooltip>に必ず置き換えてください。

<AuthCodeGroup>
  ```bash cURL
  curl --request GET \
    --url 'https://{yourDomain}/api/v2/users/USER_ID/permissions' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/users/USER_ID/permissions");
  var request = new RestRequest(Method.GET);
  request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
  	"fmt"
  	"net/http"
  	"io/ioutil"
  )

  func main() {

  	url := "https://{yourDomain}/api/v2/users/USER_ID/permissions"

  	req, _ := http.NewRequest("GET", url, nil)

  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

  ```java Java
  HttpResponse<String> response = Unirest.get("https://{yourDomain}/api/v2/users/USER_ID/permissions")
    .header("authorization", "Bearer MGMT_API_ACCESS_TOKEN")
    .asString();
  ```

  ```javascript Node.JS
  var axios = require("axios").default;

  var options = {
    method: 'GET',
    url: 'https://{yourDomain}/api/v2/users/USER_ID/permissions',
    headers: {authorization: 'Bearer MGMT_API_ACCESS_TOKEN'}

  axios.request(options).then(function (response) {
    console.log(response.data);
  }).catch(function (error) {

  });
  ```

  ```php PHP
  $curl = curl_init();

  curl_setopt_array($curl, [
    CURLOPT_URL => "https://{yourDomain}/api/v2/users/USER_ID/permissions",
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_ENCODING => "",
    CURLOPT_MAXREDIRS => 10,
    CURLOPT_TIMEOUT => 30,
    CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
    CURLOPT_CUSTOMREQUEST => "GET",
    CURLOPT_HTTPHEADER => [
      "authorization: Bearer MGMT_API_ACCESS_TOKEN"
    ],
  ]);

  $response = curl_exec($curl);
  $err = curl_error($curl);

  curl_close($curl);

  if ($err) {
    echo "cURL Error #:" . $err;
  } else {
    echo $response;
  }
  ```

  ```python Python
  import http.client

  conn = http.client.HTTPSConnection("")

  headers = { 'authorization': "Bearer MGMT_API_ACCESS_TOKEN" }

  conn.request("GET", "/{yourDomain}/api/v2/users/USER_ID/permissions", headers=headers)

  res = conn.getresponse()
  data = res.read()

  print(data.decode("utf-8"))
  ```

  ```ruby Ruby
  require 'uri'
  require 'net/http'
  require 'openssl'

  url = URI("https://{yourDomain}/api/v2/users/USER_ID/permissions")

  http = Net::HTTP.new(url.host, url.port)
  http.use_ssl = true
  http.verify_mode = OpenSSL::SSL::VERIFY_NONE

  request = Net::HTTP::Get.new(url)
  request["authorization"] = 'Bearer MGMT_API_ACCESS_TOKEN'

  response = http.request(request)
  puts response.read_body
  ```
</AuthCodeGroup>

| 値                       | 説明                                                                                                  |
| ----------------------- | --------------------------------------------------------------------------------------------------- |
| `USER_ID`               | 権限を取得するユーザーの ID。                                                                                    |
| `MGMT_API_ACCESS_TOKEN` | スコープ `read:users` を持つ [Management API 用のアクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
