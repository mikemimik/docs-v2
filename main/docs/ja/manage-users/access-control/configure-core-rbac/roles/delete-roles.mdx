[ロール](/docs/ja/manage-users/access-control/rbac)は、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Management+API">Management API</Tooltip> を使用して削除できます。ロール は、API Authorization Core 機能セットで使用されます。

ロールベースのアクセス制御 (RBAC) を正しく機能させるには、Auth0 Dashboard または Management API のいずれかを使用して、API で RBAC を有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。比較については、[Authorization Core と Authorization Extension の比較](/docs/ja/manage-users/access-control/authorization-core-vs-authorization-extension)をご覧ください。

<div id="dashboard">
  ## Auth0 Dashboard
</div>

1. [Auth0 Dashboard &gt; User Management &gt; ロール](https://manage.auth0.com/#/roles) に移動し、確認するロールの名前をクリックします。
2. **Remove this Role** をクリックし、確認します。

<div id="management-api">
  ## Management API
</div>

[Delete Role endpoint](https://auth0.com/docs/api/management/v2#!/Roles/delete_roles_by_id) に `DELETE` リクエストを送信します。プレースホルダー値 `ROLE_ID` と `MGMT_API_ACCESS_TOKEN` は、それぞれロール ID と Management API の <Tooltip tip="アクセストークン: API へのアクセスに使用される、非透過文字列または JWT 形式の認可資格情報。" cta="用語集を見る" href="/docs/ja/glossary?term=access+token">アクセストークン</Tooltip> に必ず置き換えてください。

<AuthCodeGroup>
  ```bash cURL
  curl --request DELETE \
    --url 'https://{yourDomain}/api/v2/roles/ROLE_ID' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/roles/ROLE_ID");
  var request = new RestRequest(Method.DELETE);
  request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
  	"fmt"
  	"net/http"
  	"io/ioutil"
  )

  func main() {

  	url := "https://{yourDomain}/api/v2/roles/ROLE_ID"

  	req, _ := http.NewRequest("DELETE", url, nil)

  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

| 値                       | 説明                                                                                                   |
| ----------------------- | ---------------------------------------------------------------------------------------------------- |
| `ROLE_ID`               | 削除するロールの ID。                                                                                         |
| `MGMT_API_ACCESS_TOKEN` | スコープ `delete:roles` を持つ [Management API 用アクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
