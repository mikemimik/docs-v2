ロールに割り当てられた[権限](/docs/ja/manage-users/access-control/rbac)は、<Tooltip tip="Auth0 Dashboard: サービスを構成するための Auth0 の主要製品です。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Auth0 Dashboard: サービスを構成するための Auth0 の主要製品です。" cta="用語集を見る" href="/docs/ja/glossary?term=Management+API">Management API</Tooltip> を使用して削除できます。割り当てられた権限とロールは、API Authorization Core の機能セットで使用されます。

<div id="prerequisite">
  ## 前提条件
</div>

ロールベースのアクセス制御 (RBAC) を正しく機能させるには、Dashboard または Management API を使用して、API で RBAC を有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。違いについては、[Authorization Core と Authorization Extension の比較](/docs/ja/manage-users/access-control/authorization-core-vs-authorization-extension)を参照してください。

<div id="dashboard">
  ## Dashboard
</div>

1. [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/roles) に移動し、表示したいロール名をクリックします。
2. **Permissions** ビューをクリックし、削除したい権限の横にあるごみ箱アイコンをクリックして確認します。

<div id="management-api">
  ## Management API
</div>

[Delete Role Permissions エンドポイント](https://auth0.com/docs/api/management/v2#!/Roles/delete_role_permission_assignment) に `DELETE` リクエストを送信します。`ROLE_ID`、`MGMT_API_ACCESS_TOKEN`、`API_ID`、`PERMISSION_NAME` の各プレースホルダーは、それぞれロール ID、Management API <Tooltip tip="アクセストークン: API へのアクセスに使用される、不透明な文字列または JWT 形式の認証情報。" cta="用語集を見る" href="/docs/ja/glossary?term=access+token">access token</Tooltip>、API ID、権限名に必ず置き換えてください。

</AuthCodeGroup>

| 値                       | 説明                                                                                                   |
| ----------------------- | ---------------------------------------------------------------------------------------------------- |
| `ROLE_ID`               | 権限を削除する対象のロールの ID。                                                                                   |
| `MGMT_API_ACCESS_TOKEN` | スコープ `update:roles` を持つ [Management API のアクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
| `API_ID`                | 指定したロールから削除する権限に関連付けられている API の ID。                                                                  |
| `PERMISSION_NAME`       | 指定したロールから削除する権限の名前。                                                                                  |
