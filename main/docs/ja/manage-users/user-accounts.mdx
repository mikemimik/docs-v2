title: ユーザーアカウント
---

Auth0 のホスト型クラウドデータベースには、ユーザーに関するさまざまな情報が保存されており、それらにアクセスできます。この情報はユーザープロファイルを通じて参照でき、ユーザーはテナントごとにグループ化されます。ユーザー情報自体は、<Tooltip tip="アイデンティティプロバイダー（IdP）: デジタルアイデンティティを保存および管理するサービス。" cta="用語集を見る" href="/docs/ja/glossary?term=identity+providers">アイデンティティプロバイダー</Tooltip>、独自のデータベース、エンタープライズ接続 (Active Directory、<Tooltip tip="アイデンティティプロバイダー（IdP）: デジタルアイデンティティを保存および管理するサービス。" cta="用語集を見る" href="/docs/ja/glossary?term=SAML">SAML</Tooltip>) など、さまざまなソースから取得されます。
