アプリケーションに Machine-to-Machine アクセスを認可するには、特定の API に対して organization を使用できるようにする必要があります。そのためには、該当する API に対するアプリケーションのクライアントグラントを organization に関連付ける必要があります。関連付けが完了すると、アプリケーションは、クライアントグラントで定義された API とスコープに対するトークンをリクエストする際に、その organization を使用できます。アプリケーションが organization に対してアクセスする必要がある API ごとに、この関連付けを設定する必要があります。

アプリケーションの M2M アクセスは、[Auth0 Dashboard](https://manage.auth0.com/) または [Management API](https://auth0.com/docs/api/management/v2) を使用して認可できます。

<Tabs>
  <Tab title="Auth0 Dashboard">
    Auth0 Dashboard でアプリケーションのクライアントグラントを organization に関連付けるには、次の手順を実行します。

    1. **Organizations** に移動し、関連付ける organization を選択します。
    2. **Machine-to-Machine Access** タブを選択します。
    3. **Add Access.** をクリックします。
    4. organization に関連付けるアプリケーションをクリックします。
    5. API を選択します。
    6. **Save.** をクリックします。

  </Tab>
