ユーザー検索の結果をページ分けするには、リクエストで `page`、`per_page`、`include_totals` パラメータを使用します。

| パラメータ            | 説明                                                             |
| ---------------- | -------------------------------------------------------------- |
| `page`           | ページ番号です。0 から始まります。これが設定されていない場合は、レコード数にかかわらず最大 50 件のレコードを返します。 |
| `per_page`       | 1 ページあたりのユーザー数です。                                              |
| `include_totals` | `true` に設定すると、結果にクエリの概要が含まれます。                                 |

<div id="limitation">
  ## 制限
</div>

Auth0 では、取得できるユーザーの総数は最大 1000 件に制限されています (たとえば、1 ページあたり 100 ユーザーを 10 ページ分) 。`page` パラメータが設定されていない場合は、レコードが何件存在していても、返されるレコードは最大 50 件です。
