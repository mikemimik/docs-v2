title: ユーザーを検索する
---

ユーザー検索を使用すると、Auth0 の [Management API](https://auth0.com/docs/api/management/v2) を使ってユーザープロファイルの詳細を取得できます。検索結果は[表示](/docs/ja/manage-users/user-search/view-search-results-by-page)、[並べ替え](/docs/ja/manage-users/user-search/sort-search-results)、および[エクスポート](/docs/ja/manage-users/user-migration/bulk-user-exports)できます。

ユーザープロファイルのほとんどのフィールドは、[ID トークン](/docs/ja/secure/tokens/id-tokens)の一部としては返されず、Authentication API の [`/userinfo` エンドポイント](https://auth0.com/docs/api/authentication#get-user-info) からのレスポンスにも含まれません。

Auth0 でユーザーを検索する際は、ID、メールアドレス、またはその他の条件で検索するために複数のエンドポイントを使用できます:

<div id="search-result-terminology">
  ## 検索結果の用語
</div>

ユーザー検索結果を説明するために、次の用語を使用します。

* **結果整合性**: 検索結果には、直前に完了した書き込み操作がまだ反映されていない場合があります。ただし、少し時間をおいてリクエストを再実行すると、応答には最新のデータが返されます。
* **即時整合性**: 検索結果には、リクエストの直前に行われたものを含め、成功したすべての書き込み操作の結果が反映されます。
