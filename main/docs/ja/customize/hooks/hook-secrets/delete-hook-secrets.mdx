  ルールとHooksの非推奨化について詳しくは、ブログ記事 [Preparing for Rules and Hooks End of Life](https://auth0.com/blog/preparing-for-rules-and-hooks-end-of-life/) をご覧ください。

<Warning>
  2026 年にルールとHooksの機能を削除する予定であるため、新しいルールやHooksを作成する場合は、開発環境でのみ、かつ Actions への移行をテストする目的に限って作成してください。

  ルールを Actions に移行する方法については、[Migrate from Rules to Actions](/docs/ja/customize/actions/migrate/migrate-from-rules-to-actions) をご覧ください。Hooks を Actions に移行する方法については、[Migrate from Hooks to Actions](/docs/ja/customize/actions/migrate/migrate-from-hooks-to-actions) をご覧ください。
</Warning>

特定の Hook で Hook シークレットが不要になった場合は、Auth0 Dashboard または <Tooltip tip="Management API: お客様が管理タスクを実行するための製品です。" cta="用語集を見る" href="/docs/ja/glossary?term=Management+API">Management API</Tooltip> を使用して削除できます。

Hook シークレットは、[Deploy Command-Line Interface (CLI) tool](/docs/ja/deploy-monitor/deploy-cli-tool) を使用してインポートおよびエクスポートすることもできます。

<div id="delete-hook-secrets-using-the-dashboard">
  ## Auth0 Dashboard を使用して Hook シークレットを削除する
</div>

1. [Auth0 Dashboard](https://manage.auth0.com/#/) の [Hooks](https://manage.auth0.com/#/hooks) ページに移動し、編集する Hook の横にある鉛筆アイコンをクリックします。
2. Hook エディターでレンチアイコンをクリックし、**Secrets** をクリックします。
3. 削除する Hook Secret を見つけて、ごみ箱アイコンをクリックし、**Yes** をクリックします。

<div id="delete-hook-secrets-using-the-management-api">
  ## Management API を使用して Hook シークレットを削除する
</div>

[Delete Hook シークレット エンドポイント](https://auth0.com/docs/api/management/v2/#!/Hooks/delete_secrets)に `DELETE` リクエストを送信します。`HOOK_ID`、`HOOK_SECRET_NAME`、`MGMT_API_ACCESS_TOKEN` のプレースホルダー値は、それぞれご自身の Hook ID、Hook シークレット名、Management API の <Tooltip tip="Access Token: API へのアクセスに使用される、opaque 文字列または JWT 形式の Authorization 資格情報。" cta="用語集を見る" href="/docs/ja/glossary?term=access+token">access token</Tooltip> に必ず置き換えてください。

| 値                       | 説明                                                                                                                                                                             |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `HOOK_ID`               | シークレットを削除する対象の Hook の ID。                                                                                                                                                      |
| `MGMT_API_ACCESS_TOKEN` | [scope](/docs/ja/glossary?term=scope) `update:hooks` を持つ Management API のアクセストークン。詳しくは、[Management API Access Tokens](https://auth0.com/docs/api/management/v2/tokens) を参照してください。 |
| `HOOK_SECRET_NAME`      | 指定した Hook から削除するシークレット名。この endpoint では、削除するシークレット名の配列を受け付けます。                                                                                                                  |
