<Tooltip tip="シングルサインオン (SSO): ユーザーが 1 つのアプリケーションにログインすると、他のアプリケーションにも自動的にログインできるようにするサービスです。" cta="用語集を表示" href="/docs/ja/glossary?term=SSO">SSO</Tooltip> (シングルサインオン) 連携では、<Tooltip tip="シングルサインオン (SSO): ユーザーが 1 つのアプリケーションにログインすると、他のアプリケーションにも自動的にログインできるようにするサービスです。" cta="用語集を表示" href="/docs/ja/glossary?term=SAML">SAML</Tooltip> を介して Auth0 のアイデンティティを使用し、外部サービスにログインします。これにより、社内アプリケーションを持つ顧客は、Auth0 のアイデンティティを使ってサードパーティベンダーに SSO でアクセスできます。

[マーケットプレイスで利用可能な SSO 連携](https://marketplace.auth0.com/features/sso-integrations)では、Auth0 を外部ベンダーに接続するために必要な標準の SAML 設定値と、顧客固有の追加値に必要なフィールドの両方を定義します。これらの連携では、Auth0 はサービスプロバイダー (SP) 主導のシナリオで <Tooltip tip="アイデンティティプロバイダー (IdP): デジタル ID を保存および管理するサービスです。" cta="用語集を表示" href="/docs/ja/glossary?term=identity+provider">アイデンティティプロバイダー</Tooltip> として機能します。これらの連携向けに Auth0 を設定する方法については、[Auth0 を SAML IDプロバイダーとして設定する](/docs/ja/authenticate/single-sign-on/outbound-single-sign-on/configure-auth0-saml-identity-provider) を参照してください。

パートナーに提供していただくのは、この連携を機能させるためのデフォルトの SAML 設定です。これには、[Customize SAML Assertions: SAML Assertion Attributes](/docs/ja/authenticate/protocols/saml/saml-configuration/customize-saml-assertions) に記載されている属性の一部または全部について、値または顧客が要求するフィールドが含まれます。

連携を構築してテストするには、Auth0 で [テナントを作成](https://auth0.com/signup) し、上記のドキュメントに従って、ご利用のサービスで動作する SSO 連携を作成してください。

さらに、連携の構築にあたっては、当社の [SSO 連携](https://github.com/auth0-marketplace/TEMPLATE-sso) を活用できます。

<div id="submit-your-sso-integration">
  ## SSO 連携を提出する
</div>

パートナーとして連携を提出する準備ができたら、[partner portal](https://partners.marketplace.auth0.com/) でパートナーアカウントを作成し、次の手順を行ってください。

1. パートナープロファイルを完成させます。これには、ビジネスの簡単な説明、法務およびマーケティング向けのコンテンツ、会社のロゴ、利用規約、プライバシーポリシー、連絡先情報、サポート用メールアドレスが含まれます。
2. 連携を提出します。これには、連携コード、インストールガイド、名前や説明などの詳細が含まれます。必要に応じて、連携に 3 つのバリュープロポジションを追加できます。

**以下の設定とドキュメントが必要です。**

* 動作する SAML アドオン アプリケーションの設定に使用する JSON
* 連携を設定する顧客に表示するフィールドの一覧 (**Callback URL**、**<Tooltip tip="Audience: 発行されたトークンの audience の一意の識別子。トークン内では aud という名前で、その値には ID トークンの場合はアプリケーション（Client ID）の ID、アクセス トークンの場合は API（API 識別子）の ID が含まれます。" cta="用語集を見る" href="/docs/ja/glossary?term=Audience">Audience</Tooltip>**、**Signing Certificate** など)
* Markdown で記述されたインストールガイド
