Auth0 は、ニーズに応じてさまざまな Amazon の製品やサービスと統合できます。

<div id="product-integrations">
  ## 製品連携
</div>

以下の製品を Auth0 および AWS のサービスとあわせて利用する方法について、さらに詳しく確認できます。

* [CloudFront](/docs/ja/customize/custom-domains/self-managed-certificates/configure-aws-cloudfront-for-use-as-reverse-proxy): <Tooltip tip="カスタムドメイン: 専用名（バニティ名）を持つサードパーティドメイン。" cta="用語集を見る" href="/docs/ja/glossary?term=custom+domain">カスタムドメイン</Tooltip>のリバースプロキシとして使用します。
* [Simple Email Service (SES)](/docs/ja/customize/email/smtp-email-providers/configure-amazon-ses-as-external-smtp-email-provider): ユーザーへのメール送信を管理します。
* [EventBridge](https://marketplace.auth0.com/integrations/amazon-log-streaming): ログを EventBridge にストリーミングします。
* [Cognito](/docs/ja/customize/integrations/aws/amazon-cognito): アプリケーションのバックエンドとして使用します。
* AWS S3 と DynamoDB:

  * Auth0 は、AWS S3 および DynamoDB へのアクセス用に VPC ゲートウェイエンドポイントを提供しており、[Actions](/docs/ja/customize/actions) で必要になる場合があります。これには、[Gateway endpoints for Amazon S3](https://docs.aws.amazon.com/vpc/latest/privatelink/vpc-endpoints-s3.html) および [Gateway endpoints for Amazon DynamoDB](https://docs.aws.amazon.com/vpc/latest/privatelink/vpc-endpoints-ddb.html) で説明されている `aws:SourceVpc` ベースの IAM ポリシーが必要です。VPC 識別子の詳細については、[Auth0 Support](https://www.support.auth0.com) までお問い合わせください。

<div id="identity-and-access-management-iam-integrations">
  ## アイデンティティおよびアクセス管理 (IAM) の連携
</div>

AWS を Auth0 と連携させるための設定方法については、以下のハウツー記事をご覧ください。

* [カスタムオーソライザーを使用して AWS API Gateway のエンドポイントを保護する](/docs/ja/customize/integrations/aws/aws-api-gateway-custom-authorizers): Auth0 が発行した <Tooltip tip="Access Token: API へのアクセスに使用される、オペーク文字列または JWT の形式の認可資格情報。" cta="用語集を見る" href="/docs/ja/glossary?term=access+tokens">アクセストークン</Tooltip> を受け入れます。
* [AWS API とリソースで AWS セッションタグを使用する](/docs/ja/customize/integrations/aws/session-tags): リソースにタグを付け、RBAC のためにユーザーにキーと値のペアを割り当てます。
* [シングルサインオン用に AWS を設定する](/docs/ja/customize/integrations/aws/configure-amazon-web-services-for-sso): サポートされている任意の <Tooltip tip="Identity Provider (IdP): デジタルアイデンティティを保存および管理するサービス。" cta="用語集を見る" href="/docs/ja/glossary?term=identity+provider">アイデンティティプロバイダー</Tooltip> を使用して、ユーザーが AWS にログインできるようにします。
* [アプリに Amazon ログインを追加する](/docs/ja/customize/integrations/aws/configure-amazon-web-services-for-sso): ユーザーが自分の Amazon プロファイルを使ってアプリにログインできるようにします。
