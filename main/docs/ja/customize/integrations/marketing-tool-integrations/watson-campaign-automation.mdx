title: ユーザーデータを Watson Campaign Automation にエクスポートする
---

この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートしてから、Watson Campaign Automation のダッシュボードにインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Auth0 Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。この拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Watson Campaign Automation では CSV 形式のファイルをインポートできるため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性に対して **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

CSVファイルをWatson Campaign Automationにインポートするには、次の手順に従います。

1. Watson Campaign Automationにログインし、**Data &gt; Import New** に移動します。
2. **Select File** ステップで、表示されるフィールドに新しいデータベースの設定を入力します。
3. ローカルハードドライブからファイルをアップロードするオプションを選択し、**Browse** をクリックしてファイルを選択します。
4. ファイル形式をCSV (カンマ区切り値) に設定します。
5. **Next** をクリックして **Define Data Format** ステップに進みます。
6. データが正しく処理されたことを確認し、**Next** をクリックします。
7. **Map Fields** ステップで、インポートに含めるフィールドとそのタイプを選択します。
8. **Next** をクリックして **Edit Field Settings** ステップに進みます。
9. 必要に応じて、既定または新規フィールドの設定を編集します。
10. インポートを開始するには、**Next** をクリックします。

以上で完了です。Auth0ユーザーがWatson Campaign Automationに正常にインポートされました。
