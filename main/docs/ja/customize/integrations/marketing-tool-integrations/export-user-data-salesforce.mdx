title: Salesforce にユーザーデータをエクスポートする
---

この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートし、Data Import Wizard を使用して Salesforce にインポートする方法を説明します。システム要件については、Salesforce の記事 [Data Import Wizard](https://help.salesforce.com/articleView?id=data_import_wizard.htm) を参照してください。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Auth0 Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。extension ページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Salesforce では CSV 形式のファイルをインポートできるため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性について **User Field** と **Column Name** を指定します。たとえば、次のようになります。

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

アカウントの種類によっては、インポートを正常に完了するために追加の手順が必要になる場合があります。事前に、Salesforce の記事 [Prepare Your Data for Import](https://help.salesforce.com/articleView?id=import_prepare.htm) を確認してください。追加の手順については、[Salesforce: Import Data with the Data Import Wizard](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm) を参照してください。

開始する前に、レコードをインポートするために必要な [ユーザー権限](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm) が Salesforce アカウントに付与されていることを確認してください。

CSV ファイルを Salesforce にインポートするには、次の手順に従います。

1. [Salesforce](https://login.salesforce.com/) にログインします。
2. **設定** に移動し、**Data Import Wizard** を開きます。
3. **Launch Wizard** をクリックします。
4. インポート設定を行い、表示される項目を使用してアップロードする CSV ファイルを選択します。
5. **Next** をクリックします。
6. **Edit Field Mapping** ページで、CSV ファイルのデータ項目を Salesforce のデータ項目にマッピングします。続行するには **Next** をクリックします。
7. **Review** ページで情報を確認したら、**Start Import** をクリックします。
8. インポートのステータス更新は、**Data Import Wizard** ホームページの **Recent Import Jobs** タブで確認できます。

以上で完了です。Auth0 ユーザーが Salesforce に正常にインポートされました。
