title: ユーザーデータを MailChimp にエクスポートする
---

この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートしてから、[MailChimp ダッシュボード](https://login.mailchimp.com/) にインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Auth0 Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。MailChimp は CSV 形式でのファイルのインポートに対応しているため、`Tab Separated Value file (*.csv)` オプションを選択します。

詳細については、[MailChimp Knowledge Base: Format Guidelines for Your Import File](https://kb.mailchimp.com/lists/growth/format-guidelines-for-your-import-file) をご覧ください。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性について **User Field** と **Column Name** を指定します。MailChimp では、列名が `Email Address` のメールフィールドが必須となるため、必ず含めてください。たとえば、次のようになります。

| ユーザーフィールド     | 列名            |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

開始する前に、詳細については [MailChimp Knowledge Base: Import Subscribers to a List](https://kb.mailchimp.com/lists/growth/import-subscribers-to-a-list) をご覧ください。

MailChimp アカウントにログインし、**Lists** ページに移動します。Auth0 ユーザーのインポート先となるリストを選択します ([または新しいリストを作成します](https://kb.mailchimp.com/lists/growth/create-a-new-list)) 。

1. MailChimp の List ページで、**Add Contacts** メニューから **Import Contacts** をクリックします。
2. インポート元の連絡先として、`CSV or tab-delimited text file` オプションを選択します。
3. 前のセクションで Auth0 からエクスポートしたユーザーデータの CSV ファイルをアップロードします。MailChimp が次のページでユーザーデータを解析します。
4. 列名とフィールドタイプが正しいことを確認します。
5. **Next** ボタンをクリックします。
6. 選択内容を確認し、インポートカテゴリを設定します。
7. **Import** ボタンをクリックして、ユーザーのインポートを開始します。

以上です。Auth0 ユーザーの MailChimp へのインポートは正常に完了しました。
