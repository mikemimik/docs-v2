title: Oracle Eloqua にユーザーデータをエクスポートする
---

この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートし、連絡先アップロードウィザードを使って Eloqua にインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Auth0 Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Eloqua は CSV 形式でのファイルのインポートに対応しているため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性について **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

始める前に、詳細については [Oracle Eloqua Help Center: Uploading Contacts](https://docs.oracle.com/en/cloud/saas/marketing/eloqua-user/Help/Contacts/Tasks/UploadingContacts.htm) をご覧ください。

CSV ファイルを Eloqua にインポートするには、次の手順に従ってください。

1. Eloqua にログインし、**Audience &gt; Contacts &gt; Upload** に移動して **連絡先アップロードウィザード** を開きます。
2. **Pick Data Source** タブで、表示されるフィールドを使ってインポートを設定します。
3. **Cloud** ボタンをクリックし、アップロードするファイルを選択します。
4. **Review** タブでファイルの内容を確認したら、**Next Step** をクリックします。
5. **Map Fields** タブで、CSV ファイルのデータが連絡先フィールドに対応するようにフィールドの対応付けを設定します。続けるには、**Next Step** をクリックします。
6. ウィザードの最後の手順を完了し、**Finish** をクリックします。

これで完了です。Auth0 ユーザーの Eloqua へのインポートは正常に完了しました。
