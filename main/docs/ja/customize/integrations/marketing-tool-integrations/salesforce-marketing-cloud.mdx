title: ユーザーデータを Salesforce Marketing Cloud にエクスポートする
---

この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートし、Email Studio を使用して Salesforce Marketing Cloud にインポートする方法を説明します。

詳しくは、Salesforce の記事「[Get Started Checklist for Email Studio](https://help.salesforce.com/s/articleView?id=mktg.mc_es_get_started_checklist_for_email_studio.htm\&type=5)」をご覧ください。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルの作成
</div>

まず、Auth0 Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Salesforce Marketing Cloud は CSV 形式でのファイルインポートに対応しているため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性に対して **User Field** と **Column Name** を指定します。例:

| ユーザーフィールド     | 列名      |
| ------------- | ------- |
| `email`       | メールアドレス |
| `created_at`  | 作成日時    |
| `given_name`  | 名       |
| `family_name` | 姓       |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

始める前に、Salesforce アカウントにレコードをインポートするために必要なユーザー権限が付与されていることを確認してください。また、[サブスクライバーのインポートに関する Salesforce の記事](https://help.salesforce.com/s/articleView?id=mktg.mc_es_import_subscribers_from_file.htm\&type=5)と、[レコードをインポートするにはどのような権限が必要ですか](https://help.salesforce.com/s/articleView?id=xcloud.faq_import_general_permissions.htm\&type=5)の手順にも目を通しておくことをおすすめします。

CSV ファイルを Salesforce Marketing Cloud にインポートするには、次の手順に従います。

1. Salesforce Marketing Cloud にログインし、**Email Studio** を開きます。
2. **Subscribers &gt; Lists** に移動します。
3. インポート先のリストを選択し、**Import** を選択します。
4. インポートウィザードの案内に従って、**Upload Source** として CSV ファイルを選択し、**Data Format** として CSV を選択します。残りの設定を入力して、**Next** をクリックします。
5. **Map Attributes** ダイアログで、CSV ファイルのデータフィールドを対応する正しいデータフィールドにマッピングします。マッピングが完了したら、**Next** をクリックします。
6. **Confirmed Mappings** ダイアログでマッピングを確認したら、**Begin** をクリックしてインポートを開始します。

以上で完了です。Auth0 ユーザーは Salesforce Marketing Cloud に正常にインポートされました。
