Auth0 は、お使いのサイトを Auth0 アカウントと統合するための WordPress Plugin を提供しています。このプラグインを使うと、Enterprise、ソーシャルログイン、ユーザー名/パスワード、さらに <Tooltip tip="シングルサインオン（SSO）: ユーザーが1つのアプリケーションにログインすると、そのユーザーは他のアプリケーションにも自動的にログインされるサービスです。" cta="用語集を表示" href="/docs/ja/glossary?term=Single+Sign-on">シングルサインオン</Tooltip> (SSO) や <Tooltip tip="シングルサインオン（SSO）: ユーザーが1つのアプリケーションにログインすると、そのユーザーは他のアプリケーションにも自動的にログインされるサービスです。" cta="用語集を表示" href="/docs/ja/glossary?term=passwordless">パスワードレス</Tooltip>ログインを、すべてのインスタンスで利用できます。

ログイン機能は、Auth0 を利用した新しいフォームを通じて実装され、次のことが可能になります。

* Enterprise ディレクトリ (LDAP、AD、Google Workspace、Office365、および <Tooltip tip="Security Assertion Markup Language (SAML): Standardized protocol allowing two parties to exchange authentication information without a password." cta="用語集を表示" href="/docs/ja/glossary?term=SAML">SAML</Tooltip> Provider) との SSO
* 30 を超える [ソーシャルプロバイダー](/docs/ja/authenticate/identity-providers) との SSO
* WordPress のインストール環境と他のアプリケーション間での SSO
* ユーザー管理ダッシュボード
* オプションの <Tooltip tip="多要素認証（MFA）: SMS によるコードなど、ユーザー名とパスワードに加えた認証要素を使用するユーザー認証プロセスです。" cta="用語集を表示" href="/docs/ja/glossary?term=Multi-factor+Authentication">多要素認証</Tooltip>
* オプションのパスワードレス認証
* レポートと分析
