title: Twilio Verify でカスタム電話プロバイダーを設定する
---

[Twilio Verify](https://www.twilio.com/docs/verify) を使ってカスタム電話プロバイダーを設定できます。これにより、不正利用を防ぎ、ユーザーアカウントを保護できます。Twilio Verify がカスタム電話プロバイダーとして設定されます。

<div id="prerequisites">
  ## 前提条件
</div>

有効なSMSまたは電話での送信オプションを備えた[Twilio](https://twilio.com/)アカウントが必要です。

<div id="set-up-twilio-verify-service">
  ## Twilio Verify service を設定する
</div>

[Twilio Console](https://www.twilio.com/console/verify/services) または [Verify REST API](https://www.twilio.com/docs/verify/api/service#create-a-verification-service) で Twilio Verify service を作成できます。Auth0 の カスタム電話プロバイダー を Twilio Verify に接続するには、次のものが必要です。

* TWILIO&#95;ACCOUNT&#95;SID
* TWILIO&#95;AUTH&#95;TOKEN
* TWILIO&#95;VERIFY&#95;SID

<div id="connect-custom-phone-provider-with-twilio-verify">
  ## Twilio Verify でカスタム電話プロバイダーを接続する
</div>

1. Auth0 Dashboard で [Branding &gt; Phone Provider](https://manage.auth0.com/dashboard/#/phone/templates/phone/provider) に移動します。**Phone Message Provider** ページが表示されます。
2. **Phone Provider** として **Custom** を選択します。
3. **Provider Configuration** で、Twilio API がユーザーの電話番号に電話メッセージを送信できるよう、次のコードサンプルを追加します。

```javascript lines expandable
/**
* 電話通知の送信時に実行されるハンドラー
* @param {Event} event - ユーザーおよびログイン時のコンテキストに関する詳細情報。
* @param {CustomPhoneProviderAPI} api - 電話通知の送信動作を変更するためのメソッドとユーティリティ。
*/
exports.onExecuteCustomPhoneProvider = async (event, api) => {
 const { TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_VERIFY_SID } = event.secrets;

  const client = require('twilio')(TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN);

  // auth0 の voice 値を call にマッピング
  const messageType = event.notification.delivery_method === 'voice' ? 'call' : 'sms';

  const { recipient, code } = event.notification;

  // " 1333444999  " や "333 444 5555" を修正するために追加
  //
  const sanitizedNumber = recipient.replace(/\s/g, '').trim();

  await client.verify.v2.services(TWILIO_VERIFY_SID)
    .verifications.create({
      to: sanitizedNumber,
      channel: messageType,
      customCode: code
    })

};
```

3. **Key** アイコンをクリックして **Secrets** メニューを開きます。Twilio Verify service の設定から、次の値を追加します。

* TWILIO&#95;ACCOUNT&#95;SID
* TWILIO&#95;AUTH&#95;TOKEN
* TWILIO&#95;VERIFY&#95;SID

4. Twilio ヘルパーライブラリを読み込むには、**Dependency** アイコンをクリックします。次に、**Add Dependency** ボタンをクリックします。**Add Dependency** のポップアップウィンドウが表示されたら、次のように入力します。

* **Name:** Twilio
* **Version:** latest (自動入力されます) 

**Create** をクリックすると、Auth0 が Twilio ヘルパーライブラリを検索し、最新バージョンを読み込みます。

5. **Save** をクリックします。**Save** をクリックすると、Action は自動的に保存され、デプロイされます。本番環境で使用する前にカスタム電話プロバイダーの設定をテストするには、**Send Test Message** をクリックします。
