<Warning>
  電話テンプレートをカスタマイズするには、[外部の電話プロバイダーを設定](../configure-phone-messaging-providers)する必要があります。
</Warning>

<div id="customize-phone-templates">
  ## 電話テンプレート をカスタマイズする
</div>

電話テンプレート をカスタマイズするには、次の手順に従います。

1. [**Auth0 Dashboard &gt; Branding &gt; Phone Templates**](https://manage.auth0.com/#/phone) に移動します。

2. **Template** ドロップダウンから、更新する電話テンプレートを選択します。

3. **Status** スライダーを切り替えて、まだ有効になっていない場合はテンプレートを有効にします。

4. **From** の電話番号、**テキストメッセージテンプレート**、またはその両方を更新します。

5. **Save** を選択して変更を保存するか、**Try** を選択して変更をテストするか、**Reset** を選択して変更を元に戻します。

<div id="from-phone-number">
  ### 送信元の電話番号
</div>

<Warning>
  SMS の送信元として [Twilio Messaging Services](https://www.twilio.com/docs/messaging/services) を使用している場合、カスタムの **From** 電話番号は設定できません。代わりに、Twilio の設定に基づいて、Twilio が Messaging Service の送信者プールから番号を選択します。
</Warning>

**From** フィールドでは、テキストメッセージの送信元となる電話番号を設定します。デフォルトの **From** 電話番号は、phone provider の設定時に指定したものです。

電話テンプレートにカスタムの **From** 電話番号を設定するには、このフィールドに電話番号を入力します。

<div id="text-message-template">
  ### テキストメッセージテンプレート
</div>

**テキストメッセージテンプレート**フィールドでは、テキストメッセージの内容を設定します。各テンプレートには既定のメッセージ本文が用意されており、これを編集することも、すべて削除して独自の内容を作成することもできます。

このフィールドでは、動的に生成されるコンテンツに Liquid 構文を使用できます。詳しくは、[電話テンプレート でサポートされている Liquid 構文](./supported-liquid-syntax)をご覧ください。

<div id="test-updated-templates">
  ## 更新したテンプレートをテストする
</div>

電話メッセージのテンプレートをテストするには、**試す** をクリックします。Auth0 は、テナント名が付いたデフォルトのアプリに電話メッセージを送信します。別のアプリケーション向けのテンプレートをテストするには、電話メッセージのフローを試せるようにサンプルユーザーを作成します。

特定のアプリケーションとユーザーについては、Management API の [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) エンドポイントを使用して、確認メールを手動で送信できます。
