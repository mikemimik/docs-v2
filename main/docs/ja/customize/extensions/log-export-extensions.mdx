---
description: サードパーティサービスで利用できるログエクスポート拡張機能について説明します。
title: ログエクスポート拡張機能
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 では、すべてのログデータをサードパーティサービスに送信するには、[ログストリーミングサービス](/docs/ja/customize/log-streams)の利用をおすすめしています。
</Callout>

Auth0 のログを外部サービスにエクスポートできます。

* Application Insights
* AWS Cloudwatch
* Azure Blob Storage
* Logentries
* Loggly
* Logstash
* Mixpanel
* Papertrail
* Segment
* Splunk
* Sumo Logic
