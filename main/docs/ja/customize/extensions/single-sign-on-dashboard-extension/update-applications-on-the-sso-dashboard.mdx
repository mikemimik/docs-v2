title: SSO Dashboard のアプリケーションを更新する
---

SSO Dashboard 拡張機能を使用すると、複数のエンタープライズ アプリケーションでユーザー向けの <Tooltip tip="シングルサインオン (SSO): ユーザーが1つのアプリケーションにログインすると、そのユーザーは他のアプリケーションにも自動的にログインされるサービスです。" cta="用語集を見る" href="/docs/ja/glossary?term=SSO">SSO</Tooltip> ログインを管理できます。

詳細については、[Single Sign-On Dashboard Extension](/docs/ja/customize/extensions/single-sign-on-dashboard-extension) を参照するか、[GitHub でこの拡張機能を確認](https://github.com/auth0-extensions/auth0-sso-dashboard-extension) してください。

<div id="prerequisites">
  ## 前提条件
</div>

* Auth0 で [SSO Dashboard アプリケーションを作成](/docs/ja/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)。
* [SSO Dashboard 拡張機能をインストール](/docs/ja/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)。
* [SSO Dashboard にアプリケーションを追加](/docs/ja/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)。

<div id="update-application">
  ## アプリケーションを更新
</div>

1. [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) に移動し、**Installed Extensions** ビューを選択します。
2. **SSO Dashboard** を選択します。
   管理者の場合は、`https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login` を使用して **SSO Dashboard** にログインすることもできます。
3. テナント名の下にあるドロップダウンから **設定** を選択します。

* 有効にしたときにユーザーにアプリケーションを表示するかどうかを指定するには、**Publish** または **Unpublish** を選択します。
* アプリケーションの設定を更新するには、その歯車アイコンを選択します。
* アプリケーションを削除するには、その **X** を選択します。確認のポップアップが表示され、削除を確認できます。
