<div id="sign-json-web-token">
  ## JSON Web トークンに署名する
</div>

JSON Web トークンを生成します。

<Frame>![](/docs/images/cdy7uua7fh8z/3XQqUMKrmrc8fUkCsDhI2T/43f7f8d38673d8fa4f9dba7242e044d9/sign-json-web-token.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ    | 説明                                              |
| -------- | ----------------------------------------------- |
| ペイロード    | エンコードするデータです。OpenID 標準に従った形式にすることを推奨します。        |
| サブジェクト   | JWT の subject を示します。                            |
| 発行者      | JWT を発行した主体を示します。                               |
| audience | JWT の想定される受信者を示します。例: admin.your&#95;domain.com |
| 有効期限     | JWT を処理のために受け付けてはならなくなる時刻を示します。                 |

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ   | 型      | 説明                 |
| ------- | ------ | ------------------ |
| `token` | String | JSON Web トークンの文字列。 |

<div id="output-object-example">
  ### 出力オブジェクトの例
</div>

```json lines
{
  "token": "eyJhbGciOiJIUzI1N..."
}
```

<div id="decode-json-web-token">
  ## JSON Web トークンをデコード
</div>

指定されたJSON Web トークンをデコードします。

<Frame>![](/docs/images/cdy7uua7fh8z/7I25WVtppllW6qhC5sALvH/70143bbe0d96920e1d9dfabbb1d6aeff/decode-json-web-token.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ      | 説明                        |
| ---------- | ------------------------- |
| トークン (必須)  | デコード対象の JSON Web トークン文字列。 |

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ     | 型      | 説明                         |
| --------- | ------ | -------------------------- |
| `payload` | オブジェクト | デコード後の有効な JSON Web トークンの内容 |

<div id="output-object-example">
  ### 出力オブジェクトの例
</div>

```json lines
{
  "header": {
    "alg": "HS256",
    "typ": "JWT"
  },
  "payload": {
    "sub": "1234567890",
    "name": "John Doe",
    "iat": 1516239022
  },
  "signature": "SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c"
}
```

<div id="verify-json-web-token">
  ## Verify JSON Web トークン
</div>

JSON Web トークンのデータを検証し、改ざんされていないことを確認することで、その真正性を保証します。

<Frame>![](/docs/images/cdy7uua7fh8z/15MDJcfZqtC46h6mRPOGcz/55e0785db066a54f836d30199fa5f295/verify-json-web-token.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ      | 説明                         |
| ---------- | -------------------------- |
| Token (必須) | 検証するJSON Web トークン文字列。      |
| 発行者     | 検証するJWTの発行者。               |
| Audience   | 検証するJWTの想定受信者である audience。 |

<div id="output-object">
  ### 出力オブジェクト
</div>

| プロパティ     | 型      | 説明                                           |
| --------- | ------ | -------------------------------------------- |
| `valid`   | ブール値   | JWT の署名が有効かどうかに応じて、`true` または `false` を返します。 |
| `cause`   | 文字列    | `valid` プロパティが `false` の場合に、メッセージが表示されます。    |
| `payload` | オブジェクト | デコード済みで有効な JSON Web トークンの内容です。               |

<div id="output-object-example">
  ### 出力オブジェクトの例
</div>

```json lines
{
  "valid": true,
  "header": {
    "alg": "HS256",
    "typ": "JWT"
  },
  "payload": {
    "sub": "1234567890",
    "name": "Jane Doe",
    "iat": 1516239022
  },
  "signature": "SflKxwRJSMe..."
}
```

```json lines
{
  "valid": false,
  "cause": "INVALID_SIGNATURE"
}
```
