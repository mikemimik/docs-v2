この Twilio Actions の一覧では、Twilio をプロバイダーとして使用して SMS を送信したり、通話を発信したりできます。

<div id="configure-your-twilio-vault-connection">
  ## Twilio Vault Connection を設定する
</div>

Twilio フローの Actions で使用する Vault 接続の設定方法については、[Twilio Vault Connection](/docs/ja/customize/forms/vaults/twilio) を参照してください。

<div id="send-sms">
  ## SMSを送信
</div>

SMSを送信します。

<Frame>![](/docs/images/cdy7uua7fh8z/4lPqmcYGCJ2oRHM3GSta6w/10b0141d163e4be601c2786895486440/sendsmstwilio.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ        | 説明                                                     |
| ------------ | ------------------------------------------------------ |
| From (必須)    | SMSの送信者名。                                              |
| To (必須)      | [E.164形式](https://en.wikipedia.org/wiki/E.164)の宛先電話番号。 |
| Message (必須) | SMSの本文。                                                |

<div id="output-object">
  ### 出力オブジェクト
</div>

想定されるすべてのステータスコードは、[Twilio のドキュメント](https://www.twilio.com/docs/messaging/api/message-resource#message-status-values)で確認できます。

<div id="make-call">
  ## 発信
</div>

電話をかけます。

<Frame>![](/docs/images/cdy7uua7fh8z/4JaAbJIVuVUJY40lhQuVUN/1a1bd6143c8033c12e18fa49252bd36b/makecalltwilio.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ             | 説明                                                                                                         |
| ----------------- | ---------------------------------------------------------------------------------------------------------- |
| From (必須)         | Twilio の電話番号です。国番号を含める必要があります。例: +18005550175 (1 は米国の国番号です) 。                                              |
| To (必須)           | [E.164 形式](https://en.wikipedia.org/wiki/E.164)の電話番号です。                                                    |
| TwiML ペイロード (必須)  | 音声メッセージ用の TwiML 命令です。<br />詳しくは、[Twilio の TwiML ドキュメント](https://www.twilio.com/docs/voice/twiml)を参照してください。 |

<div id="output-object">
  ### 出力オブジェクト
</div>

想定されるすべてのステータスコードは、[Twilioのドキュメント](https://www.twilio.com/docs/voice/twiml)で確認できます。
