  ## Telegram の Vault 接続を設定する
</div>

[Telegram Vault Connection](/docs/ja/customize/forms/vaults/telegram) では、Telegram フローの Actions 用 Vault 接続を設定する方法を紹介します。

<div id="send-telegram-message">
  ## Telegramメッセージを送信
</div>

Telegramのチャットにメッセージを送信します

<Frame>![](/docs/images/cdy7uua7fh8z/kgSIPsWN5djsT1yQqg5Z8/fea0821d0e919c5101d01b84b968dd53/sendtelegrammessage.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ         | 説明                                                                                  |
| ------------- | ----------------------------------------------------------------------------------- |
| チャット ID (必須)  | Telegram のチャット ID。[Telegram’s Bot Father](https://core.telegram.org/bots) から取得できます。 |
| メッセージ (必須)    | Telegram メッセージの本文です。                                                                |

<div id="output-object">
  ### 出力オブジェクト
</div>

想定されるレスポンスとステータスコードは、[Telegram のドキュメント](https://core.telegram.org/method/messages.sendMessage)で確認できます。
