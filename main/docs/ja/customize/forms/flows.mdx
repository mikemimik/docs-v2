Flows を使用すると、Forms 内で実行するサーバー側ロジックを視覚的に構築し、収集した情報を処理したり、API を介してさまざまなサービスと連携したり、自動化をトリガーしたりできます。

<Frame>![Dashboard > Forms > Flows](/docs/images/cdy7uua7fh8z/6u6Huh5CbH3GEQJv3AH7cx/e49ea613b2c46a43c83c89ffecddbafe/Flows.png)</Frame>

Flows のアクションは、フォームで次のように使用できます。

* フォームの各ステップに対して、カスタムのサーバー側検証を作成する。
* カスタム HTTP リクエストを実行して、独自の API と連携する。
* すぐに使える連携機能を使用して、サードパーティと統合する。
* ユーザー入力やプロパティに基づいて、異なるビジネスロジックを実行する。
* <Tooltip tip="JSON Web Token (JWT): 2 者間でクレームを安全に表現するために使用される標準的な ID Token の形式（多くの場合、アクセストークンの形式としても使用されます）。" cta="用語集を見る" href="/docs/ja/glossary?term=JSON+Web+Tokens">JSON Web Tokens</Tooltip> を使用してデータを検証する。
