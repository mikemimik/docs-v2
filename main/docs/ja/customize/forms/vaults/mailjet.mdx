<div id="get-your-mailjet-keys">
  ## MailJet のキーを取得する
</div>

[MailJetダッシュボード](https://app.mailjet.com/account/api_keys)の Account ページで、API keys の項目を確認してください。

<div id="configure-your-mailjet-vault-connection">
  ## MailJet Vault 接続を設定する
</div>

MailJet フローの Actions で使用する MailJet Vault 接続を設定するには、必要なパラメーターを指定します。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| 項目         | 説明                               |
| ---------- | -------------------------------- |
| API Key    | MailJet API の認証に使用する API Key。    |
| Secret Key | MailJet API の認証に使用する Secret Key。 |
