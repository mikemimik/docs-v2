[Twilio のコンソールダッシュボード](https://console.twilio.com/)で確認できます。

<div id="configure-your-twilio-vault-connection">
  ## Twilio Vault 接続を設定する
</div>

Twilio Flow Actions の Twilio Vault 接続を設定するには、必要なパラメータを設定します。

<Frame>![Auth0 Dashboard > Actions > フォーム > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメータ       | 説明                             |
| ----------- | ------------------------------ |
| Account SID | Twilio APIの認証に使用するAccount SID。 |
| Auth token  | Twilio APIの認証に使用するAuth token。  |
