title: テスト用に Management API アクセストークンを取得する
---

<Warning>
  このアクセストークンの取得方法は、**テスト目的でのみ**使用してください。手動で長期間有効なトークンを取得してアプリケーションで使用すると、トークンが持つセキュリティ上の利点が失われます。
</Warning>

<Tooltip tip="Management API: お客様が管理タスクを実行できるようにする製品です。" cta="用語集を見る" href="/docs/ja/glossary?term=Management+API">Management API</Tooltip> のテストトークンを初めて取得するのは、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> で設定を完了したときです。新しいテナントを作成しない限り、これを再度行う必要はありません。すでにある別のトークンを再利用するのではなく、Management API へのアクセスを認可するためだけのテストトークンを作成することをおすすめします。

<div id="prerequisites">
  ## 前提条件
</div>

* [テスト用のMachine-to-Machineアプリケーションを作成する](/docs/ja/get-started/apis/create-m2m-app-test)

<div id="get-access-tokens-manually">
  ## アクセストークンを手動で取得する
</div>

1. Auth0 Dashboard で **Applications &gt; APIs &gt;** [Auth0 Management API](https://manage.auth0.com/#/apis/management/explorer) に移動します。**API Explorer** タブを選択し、**Token** セクションで自動生成されたトークンを確認します。
2. トークンの右側にある **Copy** アイコンを選択します。これで、このトークンを使って [Management API](https://auth0.com/docs/api/management/v2) に対する認可済みリクエストを実行できます。

   <Frame>![APIs - Management API アクセストークン - テスト - トークンをコピー](/docs/images/cdy7uua7fh8z/3Y4I9YJR77j73FCYDiegnC/44f32f76d27357bc39aad9ee7c4d5561/management-api-access-token.png)</Frame>
3. 必要に応じて有効期限を設定します。デフォルトでは、トークンの有効期限は 86400 秒 (24 時間) です。この期間を過ぎると、トークンは失効し、使用できなくなります。有効期限を変更するには、**Token Expiration (Seconds)** を更新し、**Update &amp; Regenerate Token** をクリックします。

<Warning>
  これらのトークンは**取り消すことができない**ため、有効期限を長く設定することは推奨されません。代わりに、有効期限は短くし、必要なときに毎回新しいトークンを発行することをお勧めします。有効期限は、[jwt.io](https://jwt.io/) を使ってトークンをデコードすることで確認できます。
</Warning>

<div id="use-access-tokens-for-testing">
  ## テストにアクセストークンを使用する
</div>

テスト用に、先ほど作成した <Tooltip tip="アクセストークン: API へのアクセスに使用される認可クレデンシャルで、オペーク文字列または JWT の形式を取ります。" cta="用語集を見る" href="/docs/ja/glossary?term=Access+Token">アクセストークン</Tooltip> を使って、[Management API Explorer](https://auth0.com/docs/api/management/v2) からエンドポイントを手動で呼び出すことができます。

1. [Management API Explorer](https://auth0.com/docs/api/management/v2#!) にアクセスし、**Set API Token** ボタンを選択します。
2. **API Token** フィールドに入力します。すると、アプリケーションに付与されたスコープなどの追加情報がポップアップに表示されます。
3. **Set Token** を選択してトークンを保存し、ポップアップを閉じます。Set API Token ボタンの表示が **Authorized** に変わったことを確認します。
4. テストを開始するには、呼び出したいエンドポイントに移動し、右側のフィールドに必要なパラメーターや任意のパラメーターを入力します。次に、**Test Endpoint** を選択します。
