キー内の各プロパティは、JWK 仕様 [RFC 7517 Section 4](https://tools.ietf.org/html/rfc7517#section-4) で定義されており、アルゴリズム固有のプロパティについては [RFC 7518](https://tools.ietf.org/html/rfc7518) で定義されています。

| Property name | Description                                                                                                                                                                                                                                                                                                                            |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `alg`         | キーで使用される具体的な暗号アルゴリズムです。これはオプションのパラメーターです。デフォルトでは、Auth0 は公開される JSON Web Key Set (JWKS) に、テナント レベルで定義された署名アルゴリズムを含めます。キーを RS256 のような単一のアルゴリズムではなく、複数のアルゴリズムで使用できるようにするには、Dashboard の Advanced Tenant settings で **Include Signing Algorithms in JSON Web Key Set** をオフにします。これにより `alg` パラメーターが削除され、JWKS の利用側で必要に応じて署名アルゴリズムを解釈する必要があります。 |
| `kty`         | キーで使用される暗号アルゴリズムの種類です。                                                                                                                                                                                                                                                                                                                 |
| `use`         | キーの用途を示します。`sig` は署名を表します。                                                                                                                                                                                                                                                                                                             |
| `x5c`         | x.509 証明書チェーンです。配列の最初のエントリーはトークンの検証に使用する証明書で、残りの証明書はその最初の証明書を検証するために使用できます。                                                                                                                                                                                                                                                            |
| `n`           | [RSA public key](https://tools.ietf.org/html/rfc7518#page-30) の法です。                                                                                                                                                                                                                                                                    |
| `e`           | [RSA public key](https://tools.ietf.org/html/rfc7518#page-30) の指数です。                                                                                                                                                                                                                                                                   |
| `kid`         | キーの一意の識別子です。                                                                                                                                                                                                                                                                                                                           |
| `x5t`         | x.509 証明書のサムプリントです (SHA-1 サムプリント) 。                                                                                                                                                                                                                                                                                                    |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 は HMAC と RSA のみをサポートしており、現在は Elliptic Curve encryption (ECDSA) をサポートしていません。
</Callout>
