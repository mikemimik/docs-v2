title: Auth0 MFA API
---

Auth0 では、[Universal Login](/docs/ja/authenticate/login/auth0-universal-login) を使用して、組み込みの <Tooltip tip="多要素認証 (MFA): SMS で送信されるコードなど、ユーザー名とパスワードに加えて認証要素を使用するユーザー認証プロセス。" cta="用語集を表示" href="/docs/ja/glossary?term=multi-factor+authentication">多要素認証</Tooltip> (MFA) の登録および認証フローを利用できます。次のような場合は、[MFA API](https://auth0.com/docs/api/authentication#multi-factor-authentication) を使用してください。

* [リソース所有者パスワードグラントを使用してユーザーを認証する](/docs/ja/secure/multi-factor-authentication/authenticate-using-ropg-flow-with-mfa)。
* [ユーザーが自身の認証要素を管理できる](/docs/ja/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api) インターフェースを構築する。

MFA API を使用するには、アプリケーションで MFA グラントタイプを有効にする必要があります。<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> で MFA グラントを有効にするには、[Dashboard &gt; Applications &gt; Advanced Settings &gt; Grant Types](https://manage.auth0.com/#/applications) に移動して **MFA** を選択します。詳しくは、[多要素認証を有効にする](/docs/ja/secure/multi-factor-authentication/enable-mfa) を参照してください。

<div id="limitations">
  ## 制限事項
</div>

MFA API は、SMS、音声通話、Auth0 Guardian アプリによるプッシュ通知、Email、OTP の認証要素に対応しています。現時点では、Duo または WebAuthn の認証要素の登録には対応していません。
