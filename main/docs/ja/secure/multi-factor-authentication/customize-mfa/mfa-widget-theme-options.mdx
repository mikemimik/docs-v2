title: MFA Widget のテーマオプション
---

Classic Login で <Tooltip tip="多要素認証（MFA）: SMS で送信されるコードなど、ユーザー名とパスワードに加えて認証要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/docs/ja/glossary?term=MFA">MFA</Tooltip> ページに独自の HTML を使用する場合は、Auth0 MFA Widget を利用します。この Widget には、次の制限があります。

* メールを使用した MFA には対応していません。
* ユーザーが複数の認証要素を登録している場合、使用する認証要素を選択できず、MFA Widget は最も安全性の高い認証要素でログインするよう求めます。
* <Tooltip tip="Universal Login: アプリケーションは、ユーザーの本人確認を行うために、Auth0 の Authorization Server でホストされている Universal Login にリダイレクトされます。" cta="用語集を見る" href="/docs/ja/glossary?term=Universal+Login">Universal Login</Tooltip> の [国際化](/docs/ja/customize/internationalization-and-localization/universal-login-internationalization) 機能は使用されません。

MFA Widget のテーマオプションは、`theme` プロパティ配下で名前空間化されています。

| Option         | Description                                                                            |
| -------------- | -------------------------------------------------------------------------------------- |
| `icon`         | ヘッダーで使用する画像の URL。デフォルトは Auth0 ロゴです。推奨される高さは 58 ピクセルです。                                 |
| `primaryColor` | MFA Widget のメインカラーを定義します。カスタム `icon` を使用する際に、色の組み合わせを調和させるのに役立ちます。デフォルトは `#ea5323` です。 |

<div id="examples">
  ## 例
</div>

```js lines
return new Auth0MFAWidget({

...

  theme: {
    icon: 'https://example.com/assets/logo.png',
    primaryColor: 'blue'
  },
  
...  
  
})
```

[auth0-guardian.js](https://github.com/auth0/auth0-guardian.js) を使用した [Guardian の基本ウィジェットの例](https://github.com/auth0/auth0-guardian.js/blob/master/example/basic_widget.html) をご覧ください。詳しくは [readme](https://github.com/auth0/auth0-guardian.js/tree/master/example) をご覧ください。
