title: デプロイとモニタリング
---

Auth0 は、パブリッククラウド、AWS 上の Private Cloud、または Private Cloud on Azure にデプロイできます。デプロイ用のツールやチェックリストを使うことで、作業をスムーズに進められます。モニタリング ツールやイベントログを使うと、システムの稼働状況やイベントを追跡できます。
