Lock は、メールおよび SMS の<Tooltip tip="パスワードレス: 第1の認証要素としてパスワードに依存しない認証方式。" cta="用語集を表示" href="/docs/ja/glossary?term=Passwordless">パスワードレス</Tooltip>接続を使用したパスワードレス認証に対応しています。パスワードレス認証を使用するには、Lock.Swift バージョン 2.14.0 以降が必要です。

Lock を表示するには、`UIViewController` に次のスニペットを追加します。

```swift lines
Lock
    .passwordless()
    .withOptions {
         $0.oidcConformant = true
    }
    // withConnections、withOptions、withStyle など。
    .onAuth { credentials in
      // Credentials オブジェクトを保存する
    }
    .present(from: self)
```

パスワードレスは1 つの接続でのみ使用でき、SMS よりもメール接続が優先されます。

<div id="passwordless-method">
  ### パスワードレス方式
</div>

Lock パスワードレス を使用する場合、デフォルトの `passwordlessMethod` は `.code` で、ログイン用のワンタイムパスコードがユーザーに送信されます。[Universal Links](/docs/ja/get-started/applications/enable-universal-links-support-in-apple-xcode) を使用する場合は、次の設定を追加できます。

```swift lines
.withOptions {
    $0.passwordlessMethod = .magicLink
}
```

<div id="activity-callback">
  ### Activity のコールバック
</div>

Lock パスワードレス を使用していて、ユーザーにユニバーサルリンクを送信するために `.magicLink` オプションを指定している場合は、`AppDelegate.swift` に次の内容を追加する必要があります。

```swift lines
func application(_ application: UIApplication, continue userActivity: NSUserActivity, restorationHandler: @escaping ([Any]?) -> Void) -> Bool {
    return Lock.continueAuth(using: userActivity)
}
```
