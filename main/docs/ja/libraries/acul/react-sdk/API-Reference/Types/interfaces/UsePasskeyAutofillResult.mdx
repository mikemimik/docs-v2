[`usePasskeyAutofill`](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/usePasskeyAutofill) の戻り値の型です。

```typescript
interface UsePasskeyAutofillResult {
  inputRef: RefObject<HTMLInputElement>;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  ユーザー名の入力要素に関連付ける React ref です。これをバインドすると、SDK が入力欄の `autocomplete` 属性を自動的に `"username webauthn"` に設定し、パスキーの自動入力に対応したブラウザーの Conditional UI を有効にします。マークアップですでに `autocomplete="username webauthn"` を指定している場合は、この ref のバインドは省略できます。
</ParamField>
