---
title: "UsernameValidationError"
---

バリデーション時に返される、単一のユーザー名バリデーションエラーを表します。

各エラーには、機械可読な`code`と人が読んで理解できる`message`があります。

```ts Example
export interface UsernameValidationError {
  code: string;
  message: string;
  field: string;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="code" type="string" required>
  失敗したバリデーションルールを一意に識別する識別子です。

  例: `"username-too-short"`, `"username-invalid-characters"`
</ParamField>

<ParamField body="message" type="string" required>
  エラーの内容を人が読める形で説明したものです。

  例: `"Username must be at least 3 characters long."`
</ParamField>

<ParamField body="field" type="string" required>
  バリデーションエラーに関連するフィールドで、通常は &quot;username&quot;、&quot;email&quot;、または &quot;phone&quot; です。
</ParamField>
