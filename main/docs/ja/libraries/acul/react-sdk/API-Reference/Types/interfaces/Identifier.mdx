---
title: "識別子"
---

サインアップフォームまたは認証フローで使用する、設定済みの識別子フィールド (例: email、phone、username) について説明します。

```ts Example
export interface Identifier {
  type: IdentifierType;
  required: boolean;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="type" type={<span><a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a></span>} required>
  `'email'`、`'phone'`、`'username'` などの 識別子 の種類です。
</ParamField>

<ParamField body="required" type="boolean" required>
  この 識別子 がアカウント登録時に必須かどうかを示します。
</ParamField>
