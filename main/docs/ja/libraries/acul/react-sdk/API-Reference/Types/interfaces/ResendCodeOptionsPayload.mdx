ResendCodeOptions
説明 Login メール確認画面の `resendCode` メソッドで使用するオプションです。
この操作は、ユーザーがメールへの新しい確認コードの送信をリクエストしたときに使用されます。
たとえば、最初のコードを受け取れなかった場合や、コードの有効期限が切れた場合です。

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```

  ## インデックス可能
