---
title: "UseErrorsResult"
---

[`useErrors`](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) が返す結果オブジェクトです。現在のエラー状態にアクセスでき、Auth0 Universal Login の画面で個々のエラーまたはすべてのエラーを閉じるためのメソッドを提供します。

```ts Example
export interface UseErrorsResult {
  errors: ErrorsResult;
  hasError: boolean;
  dismiss: (id: string) => void;
  dismissAll: () => void;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="errors" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorsResult">ErrorsResult</a>} required>
  メッセージや識別子を含む、現在のエラーの一覧です。
</ParamField>

<ParamField body="hasError" type="boolean" required>
  1 つ以上のエラーがある場合は `true` です。エラー UI を条件付きでレンダリングする際に役立ちます。
</ParamField>

<ParamField body="dismiss" type="(id: string) => void" required>
  `id` で指定した特定のエラーを削除します。

  <Expandable title="パラメーター">
    <ParamField body="id" type="string">
      削除するエラーの識別子です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="dismissAll" type="() => void" required>
  現在アクティブなエラーをすべて一度にクリアします。
</ParamField>
