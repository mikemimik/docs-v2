---
title: "ErrorType"
---

[`useErrors`](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) で管理されるエラーを分類する `string` のユニオン型です。`errors.byType()` でエラーを絞り込む際に使用され、各 [`ErrorItem`](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem) では `kind` プロパティとして保持されます。

```typescript
type ErrorType = "auth0" | "validation" | "configuration";
```

<div id="values">
  ## 値
</div>

<ParamField body="&#x22;auth0&#x22;" type="ErrorType">
  認証または認可の際に、Auth0 または独自のバックエンドから返されるエラー。
</ParamField>

<ParamField body="&#x22;validation&#x22;" type="ErrorType">
  リクエスト送信前の無効な入力など、クライアントサイドのフォームのバリデーションによるエラー。
</ParamField>

<ParamField body="&#x22;configuration&#x22;" type="ErrorType">
  必要なコンテキスト外でフックを呼び出した場合など、SDK の連携不備や設定ミスによって発生するエラー。[`UseErrorOptions`](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorOptions) で `includeDevErrors: true` が設定されている場合にのみ表示されます。
</ParamField>
