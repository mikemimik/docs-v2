---
title: "UsernameValidationResult"
---

1 つ以上のルールに基づいてユーザー名をバリデーションした結果です。

```ts Example
export interface UsernameValidationResult {
  isValid: boolean;
  errors: UsernameValidationError[];
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="isValid" type="boolean" required>
  ユーザー名がすべてのバリデーションルールを満たしているかどうかを示します。
</ParamField>

<ParamField body="errors" type={<span><a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError">UsernameValidationError[]</a></span>} required>
  [UsernameValidationError](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError) オブジェクトの配列で、失敗した個々のルールを表します (該当する場合) 。

  [UsernameValidationResult.isValid](#isvalid) が `true` の場合、この配列は空になります。
</ParamField>
