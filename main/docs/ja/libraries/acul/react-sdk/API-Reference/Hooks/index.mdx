<div id="methods">
  ## メソッド
</div>

* [useAuth0Themes](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes): Branding コンテキストから、設定をフラット化した現在のテーマオプションを取得する React hook。
* [useChangeLanguage](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage): 現在の ACUL 画面の表示言語を変更する関数を返す React hook。
* [useCurrentScreen](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen): 現在の画面のコンテキストと状態を取得する React hook。
* [useErrors](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors): ACUL (Advanced Customization of Universal Login) のエラーを読み取り、管理するための React hook。すべてのバリデーションエラーとサーバーサイドエラーを扱います。エラーは 3 種類に分類されます: - `server` — Auth0 または独自のバックエンドから返されるエラー。 - `client` — クライアントサイドのバリデーション (例: 無効なフォーム入力) によるエラー。 - `developer` — 不適切な連携や SDK の誤用によって発生するエラー。
* [useLoginIdentifiers](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useLoginIdentifiers): 認証フローまたはログインプロセスで現在使用されている、有効な identifier types (`'email'`、`'phone'`、`'username'` など) のリストを返します。
* [useMfaPolling](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling): Auth0 Advanced Customization of Universal Login (ACUL) 画面で、MFA プッシュの polling (例: プッシュ通知の承認待ち) を管理する React hook。
* [useResend](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useResend): この React hook は、ACUL 画面で「再送」アクション (例: 確認コードの再送) を管理します。
* [useSignupIdentifiers](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useSignupIdentifiers): 現在の画面のサインアップ識別子に基づいて、有効な識別子 (email、phone、またはユーザー名) のリストを、それぞれの `required` の状態とともに返します。
* [usePasswordValidation](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation): この React hook は、現在の Auth0 パスワードポリシーに照らしてパスワードを検証し、そのパスワードが各ルールを満たしているかどうかを示す構造化された結果を返します。
* [useUsernameValidation](/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useUsernameValidation): 現在の Auth0 ユーザー名ポリシーに照らしてユーザー名を検証するための React hook。
