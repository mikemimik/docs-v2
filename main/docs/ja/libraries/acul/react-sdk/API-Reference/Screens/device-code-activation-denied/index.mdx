`device-code-activation-denied` 画面は、デバイスのアクティベーション試行が拒否されたときに表示されます。この画面では、デバイスコードが承認されなかったため、アクティベーションフローを完了できなかったことをユーザーに通知します。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有のフックとメソッドのセットがあります。SDK は各画面で **partial import** と **root import** をサポートしています。

* partial import を使うと、特定のユースケースに必要なコードだけを取り込めます。
* root import を使うと、単一のバンドルからすべての画面を読み込めるため、想定されるあらゆる画面に 1 つのビルドで対応したい場合に便利です。

```jsx Import Example
// ルートインポート
import { useDeviceCodeActivationDenied } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useDeviceCodeActivationDenied,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティフック
  useChangeLanguage,
} from '@auth0/auth0-acul-react/device-code-activation-denied';

function DeviceCodeActivationDeniedScreen() {
  const { screen } = useDeviceCodeActivationDenied();
  return <p>Device activation was denied on {screen.name}</p>;
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

`device-code-activation-denied` 画面で、Auth0 のコンテキストデータへの読み取り専用アクセスを提供する、画面スコープのフックです。`@auth0/auth0-acul-react/device-code-activation-denied` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`device-code-activation-denied` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`device-code-activation-denied` 画面向けの `id`、`name`、`logoUrl` など、クライアント関連の設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、デバイスの有効化が organization スコープである場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フロー内の現在の prompt に関するデータが含まれます。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  このフックには、設定やコンテキストなど、`device-code-activation-denied` 画面に固有の詳細が含まれます。

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  このフックには、`id` や関連するメタデータなど、テナントに関連するデータが含まれます。

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  このフックは、アクティブな接続や現在のフローの状態など、`device-code-activation-denied` 画面におけるトランザクション固有のデータを提供します。

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  このフックは、URLパラメーターから事前入力されたデバイスコードなど、画面に渡される信頼できないデータを扱います。

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  このフックは、`username`、`email`、利用可能な認証方法など、現在アクティブなユーザーの詳細を提供します。

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/device-code-activation-denied';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useDeviceCodeActivationDenied" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/DeviceCodeActivationDeniedMembers">DeviceCodeActivationDeniedMembers</a>}>
  このフックは、`device-code-activation-denied` 画面で利用できるすべてのメソッドとコンテキストを返します。
</ParamField>

<div id="commonutility-hooks">
  ## 共通/ユーティリティ フック
</div>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  このフックは、ブランディングコンテキストからフラット化した設定を含む現在のテーマオプションを取得します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  このフックは、現在のACUL画面の表示言語を変更する関数を返します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  このフックは、現在の画面のコンテキストと状態を取得します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  このフックは、画面上のサーバー、クライアント、開発者向けのエラーを読み取り、管理します。
</ParamField>
