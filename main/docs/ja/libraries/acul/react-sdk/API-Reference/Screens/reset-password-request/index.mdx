`reset-password-request` 画面は、パスワードリセットフローの開始時に表示されます。ここでは、パスワードリセット用のリンクを受け取るために、ユーザーにユーザー名またはメールアドレスの入力を求めます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有のフックとメソッドのセットがあります。SDK は各画面で **部分インポート** と **ルートインポート** をサポートしています。

* 部分インポート を使用すると、特定のユースケースに必要なコードだけを含めることができます。
* ルートインポート を使用すると、単一のバンドルからすべての画面を読み込めるため、想定されるあらゆる画面に対応する統一ビルドが必要な場合に便利です。

```jsx Import Example
// ルートインポート
import { useResetPasswordRequest } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useResetPasswordRequest,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useLoginIdentifiers,
  // ユーティリティフック
  useChangeLanguage,
  // メソッド
  backToLogin,
  resetPassword,
} from '@auth0/auth0-acul-react/reset-password-request';

function ResetPasswordRequestScreen() {
  const { resetPassword } = useResetPasswordRequest();
  return (
    <button onClick={() => resetPassword({ username: 'user@example.com' })}>
      Send Reset Link
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

`reset-password-request` 画面で Auth0 のコンテキストデータへの読み取り専用アクセスを提供する、画面スコープのフックです。`@auth0/auth0-acul-react/reset-password-request` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`reset-password-request` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-request';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`reset-password-request` 画面に対する `id`、`name`、`logoUrl` などのクライアント関連の設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-request';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、パスワードリセットが組織スコープの場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-request';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フローにおける現在の prompt に関するデータが含まれます。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-request';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnResetPasswordRequest">ScreenMembersOnResetPasswordRequest</a></span>}>
  このフックには、設定やコンテキストを含む `reset-password-request` 画面固有の詳細が含まれます。

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/reset-password-request';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  このフックには、`id` や関連するメタデータなど、テナントに関するデータが含まれます。

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/reset-password-request';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembersOnResetPasswordRequest">TransactionMembersOnResetPasswordRequest</a></span>}>
  このフックは、アクティブな接続や現在のフローの状態など、`reset-password-request` 画面向けのトランザクション固有のデータを提供します。

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/reset-password-request';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  このフックは、事前入力されたメールアドレスや URLパラメーターから渡されたユーザー名など、画面に渡される信頼できないデータを扱います。

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/reset-password-request';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  このフックは、`username`、`email`、利用可能な認証方法など、現在アクティブなユーザーの詳細を提供します。

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/reset-password-request';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useResetPasswordRequest" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordRequestMembers">ResetPasswordRequestMembers</a>}>
  このフックは、`reset-password-request` 画面で利用できるすべてのメソッドとコンテキストを返します。
</ParamField>

<div id="methods">
  ## メソッド
</div>

<ParamField body="backToLogin" type="Promise<void>">
  このメソッドはユーザーをログイン画面に戻し、パスワードリセットフローを中止します。

  ```jsx Example
  import { useResetPasswordRequest } from '@auth0/auth0-acul-react/reset-password-request';

  function BackToLoginButton() {
    const { backToLogin } = useResetPasswordRequest();
    return (
      <button onClick={() => backToLogin()}>
        Back to Login
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="resetPassword" type="Promise<void>">
  このメソッドは、パスワードリセットを開始するために、ユーザー名またはメールアドレスを送信します。

  ```jsx Example
  import { useResetPasswordRequest } from '@auth0/auth0-acul-react/reset-password-request';

  function ResetPasswordForm() {
    const { resetPassword } = useResetPasswordRequest();
    return (
      <button onClick={() => resetPassword({ username: 'user@example.com' })}>
        Send Reset Link
      </button>
    );
  }
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [ResetPasswordRequestOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordRequestOptions)。
    </ParamField>

    <ParamField body="username" type="string" required>
      リセット対象のアカウントのユーザー名またはメールアドレス。
    </ParamField>

    <ParamField body="captcha?" type="string">
      テナントで CAPTCHA が有効になっている場合に必要な、CAPTCHA 応答トークン。
    </ParamField>
  </Expandable>
</ParamField>

<div id="commonutility-hooks">
  ## 共通/ユーティリティ フック
</div>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useLoginIdentifiers">useLoginIdentifiers</a>} type="Hooks">
  このフックは、現在の接続に設定されているログイン識別子の種類 (email、ユーザー名、phone) を返します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  このフックは、ブランディングコンテキストでフラット化された設定を含む現在のテーマオプションを取得します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  このフックは、現在のACUL画面の表示言語を変更するための関数を返します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  このフックは、現在の画面のコンテキストと状態を取得します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  このフックは、画面上のサーバー、クライアント、開発時のエラーを読み取り、管理します。
</ParamField>
