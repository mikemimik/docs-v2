`mfa-webauthn-roaming-challenge` 画面では、YubiKey などのローミング セキュリティ キーやその他の FIDO2 ハードウェア認証器を使用して、ユーザーに MFA チャレンジを完了するよう求めます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有の Hooksとメソッドがあります。SDK は、各画面で**partial import**と**root import**をサポートしています。

* partial import を使用すると、特定のユースケースに必要なコードだけを取り込めます。
* root import を使用すると、単一のバンドルからすべての画面を読み込めるため、想定されるあらゆる画面に対応する統一ビルドが必要な場合に便利です。

```jsx Import Example
// ルートインポート
import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useMfaWebAuthnRoamingChallenge,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティフック
  useChangeLanguage,
  // メソッド
  reportWebAuthnError,
  tryAnotherMethod,
  verify,
} from "@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge";

function MfaWebAuthnRoamingChallengeScreen() {
  const { verify } = useMfaWebAuthnRoamingChallenge();
  return (
    <button onClick={() => verify({ rememberDevice: true })}>
      Verify with Security Key
    </button>
  );
}
```

<div id="context-hooks">
  ## Context Hooks
</div>

`mfa-webauthn-roaming-challenge` 画面で Auth0 のコンテキストデータへの読み取り専用アクセスを提供する、画面スコープの Hooks です。`@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`mfa-webauthn-roaming-challenge` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`mfa-webauthn-roaming-challenge` 画面向けの `id`、`name`、`logoUrl` など、クライアントに関連する設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、MFA フローが組織スコープである場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>組織コンテキストがありません</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フロー内の現在の prompt に関するデータが含まれます。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaWebAuthnRoamingChallenge">ScreenMembersOnMfaWebAuthnRoamingChallenge</a></span>}>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>

<div id="methods">
  ## メソッド
</div>

<ParamField body="reportWebAuthnError" type="Promise<void>">
  このメソッドは、ローミング セキュリティ キーのチャレンジ中に発生した WebAuthn エラーを報告します。

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function HandleWebAuthnError({ error }) {
    const { reportWebAuthnError } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => reportWebAuthnError({ error })}>
        Report Error
      </button>
    );
  }
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaWebAuthnRoamingChallengeReportErrorOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaWebAuthnRoamingChallengeReportErrorOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      報告する WebAuthn API (`navigator.credentials.get()`) の error object。

      <ParamField body="name" type="string" required>
        エラー名 (例: `'NotAllowedError'`) 。
      </ParamField>

      <ParamField body="message" type="string" required>
        問題の内容を説明するエラーメッセージ。
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドは MFA方式の選択画面に移動し、ユーザーが別の認証要素を選択できるようにします。

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="verify" type="Promise<void>">
  このメソッドは、MFA チャレンジを完了するためにローミング セキュリティ キーの検証を開始します。

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function VerifyButton() {
    const { verify } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => verify({ rememberDevice: true })}>
        Verify with Security Key
      </button>
    );
  }
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [VerifySecurityKeyOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/VerifySecurityKeyOptions).
    </ParamField>

    <ParamField body="rememberDevice" type="boolean">
      任意。true の場合、今後の MFA チャレンジに備えてブラウザーを記憶することを試みます。
      `rememberBrowser` form フィールドに対応します。
    </ParamField>
  </Expandable>
</ParamField>
