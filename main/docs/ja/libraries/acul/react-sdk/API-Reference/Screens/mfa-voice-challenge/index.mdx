`mfa-voice-challenge` 画面では、MFA 検証を完了するために、音声通話で届いたワンタイムコードの入力をユーザーに求めます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有のフックとメソッドがあります。SDK は、各画面で **部分インポート** と **ルートインポート** の両方をサポートしています。

* 部分インポートを使うと、特定の用途に必要なコードだけを取り込めます。
* ルートインポートを使うと、1 つのバンドルからすべての画面を読み込めるため、考えられるすべての画面に対応する統一されたビルドにしたい場合に便利です。

```jsx Import Example
// rootインポート
import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useMfaVoiceChallenge,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティフック
  useChangeLanguage,
  useResend,
  // メソッド
  continueMethod,
  pickPhone,
  resendCode,
  switchToSms,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-voice-challenge";

function MfaVoiceChallengeScreen() {
  const { continueMethod } = useMfaVoiceChallenge();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## コンテキストフック
</div>

`mfa-voice-challenge` 画面で Auth0 のコンテキストデータへの読み取り専用アクセスを提供する画面スコープのフックです。`@auth0/auth0-acul-react/mfa-voice-challenge` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`mfa-voice-challenge` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`mfa-voice-challenge` 画面の `id`、`name`、`logoUrl` など、クライアントに関連する設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、MFA フローが組織にスコープ設定されている場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フロー内の現在の prompt に関するデータが含まれます。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaVoiceChallenge">ScreenMembersOnMfaVoiceChallenge</a></span>}>
  このフックには、設定やコンテキストを含む `mfa-voice-challenge` 画面固有の詳細が含まれます。

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  このフックには、`id` や関連するメタデータなど、テナントに関するデータが含まれます。

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  このフックは、現在の MFA フローの state など、`mfa-voice-challenge` 画面におけるトランザクション固有のデータを提供します。

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembersOnMfaVoiceChallenge">UntrustedDataMembersOnMfaVoiceChallenge</a></span>}>
  このフックは、URLパラメーターから事前入力された値など、画面に渡される信頼されていないデータを扱います。

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  このフックは、`username`、`email`、利用可能な認証方法など、現在のユーザーの詳細情報を提供します。

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-voice-challenge';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaVoiceChallenge" type={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaVoiceChallengeMembers">MfaVoiceChallengeMembers</a>}>
  このフックは、`mfa-voice-challenge` 画面で利用できるすべてのメソッドとコンテキストを返します。
</ParamField>

<div id="methods">
  ## メソッド
</div>

<ParamField body="continueMethod" type="Promise<void>">
  このメソッドは、MFAチャレンジを完了するために、音声通話で受け取ったワンタイムコードを送信します。

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function VerifyCodeForm() {
    const { continueMethod } = useMfaVoiceChallenge();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Verify Code
      </button>
    );
  }
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaVoiceChallengeContinueOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaVoiceChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      音声通話で受け取った確認コード。
    </ParamField>

    <ParamField body="rememberDevice" type="boolean">
      デバイスを30日間記憶し、今後のMFAチャレンジを省略するための省略可能なフラグ。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickPhone" type="Promise<void>">
  このメソッドは、電話番号の選択画面に移動します。

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function PickPhoneButton() {
    const { pickPhone } = useMfaVoiceChallenge();
    return (
      <button onClick={() => pickPhone()}>
        Use a Different Phone Number
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  このメソッドは、ユーザーの電話番号に音声通話のワンタイムコードを再送します。

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function ResendCodeButton() {
    const { resendCode } = useMfaVoiceChallenge();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  このメソッドは、MFAチャレンジの配信方法を音声通話からSMSに切り替えます。

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function SwitchToSmsButton() {
    const { switchToSms } = useMfaVoiceChallenge();
    return (
      <button onClick={() => switchToSms()}>
        Switch to SMS
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドは、ユーザーが別の認証要素を選べるように、MFA方式の選択画面に移動します。

  ```jsx Example
  import { useMfaVoiceChallenge } from '@auth0/auth0-acul-react/mfa-voice-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaVoiceChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>

<div id="commonutility-hooks">
  ## 共通/ユーティリティ フック
</div>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useResend">useResend</a>} type="Hooks">
  このフックは、コード再送機能を備えた画面で、再送のクールダウン状態とそのロジックを管理します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  このフックは、ブランディングコンテキストから、フラット化された設定を含む現在のテーマオプションを取得します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage">useChangeLanguage</a>} type="Hooks">
  このフックは、現在の ACUL 画面で表示言語を変更するための関数を返します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  このフックは、現在の画面のコンテキストと状態を取得します。
</ParamField>

<ParamField body={<a href="/docs/ja/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  このフックは、画面上のサーバー、クライアント、開発者のエラーを参照し、管理します。
</ParamField>
