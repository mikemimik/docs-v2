`mfa-webauthn-change-key-nickname` 画面では、ユーザーは登録済みの WebAuthn セキュリティキーの表示名を設定または更新できます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ固有のフックとメソッドのセットがあります。SDK は、各画面で **部分インポート** と **ルートインポート** の両方をサポートしています。

* 部分インポートを使うと、特定のユースケースに必要なコードだけを含めることができます。
* ルートインポートを使うと、単一のバンドルからすべての画面を読み込めるため、考えられるすべての画面に対応する統一ビルドを使用したい場合に便利です。

```jsx Import Example
// ルートインポート
import { useMfaWebAuthnChangeKeyNickname } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useMfaWebAuthnChangeKeyNickname,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティフック
  useChangeLanguage,
  // メソッド
  continueWithNewNickname,
} from "@auth0/auth0-acul-react/mfa-webauthn-change-key-nickname";

function MfaWebAuthnChangeKeyNicknameScreen() {
  const { continueWithNewNickname } = useMfaWebAuthnChangeKeyNickname();
  return (
    <button onClick={() => continueWithNewNickname({ nickname: 'My Security Key' })}>
      Save Nickname
    </button>
  );
}
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="continueWithNewNickname" type="Promise<void>">
  このメソッドは、登録済みのWebAuthnセキュリティキーの新しいニックネームを送信します。

  ```jsx Example
  import { useMfaWebAuthnChangeKeyNickname } from '@auth0/auth0-acul-react/mfa-webauthn-change-key-nickname';

  function SaveNicknameForm() {
    const { continueWithNewNickname } = useMfaWebAuthnChangeKeyNickname();
    return (
      <button onClick={() => continueWithNewNickname({ nickname: 'My Security Key' })}>
        ニックネームを保存
      </button>
    );
  }
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [MfaWebAuthnChangeKeyNicknameContinueOptions](/docs/ja/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaWebAuthnChangeKeyNicknameContinueOptions)。
    </ParamField>

    <ParamField body="nickname" type="string" required>
      WebAuthnセキュリティキーの新しいニックネーム。
    </ParamField>
  </Expandable>
</ParamField>
