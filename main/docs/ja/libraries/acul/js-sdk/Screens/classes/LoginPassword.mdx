LoginPassword クラスは、`login-password` 画面の機能を実装します。この画面では、ユーザーの識別子とパスワードを入力します。

<Frame>
  <img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/Em8VqaD4fPUx6FRevD8fh/a9133a19565d5d9ddc1454535a0a7a11/loginpassword.png" alt="" />
</Frame>

<div id="constructors">
  ## コンストラクター
</div>

LoginPassword 画面マネージャーのインスタンスを作成します：

```typescript Example
import LoginPassword from "@auth0/auth0-acul-js/login-password";
const loginPasswordManager = new LoginPassword();
loginPasswordManager.login({
  username: "testUser",
  password: "testPassword"
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中維持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedLogin" type="Promise<void>">
  このメソッドは、認証のためにユーザーをソーシャルまたはエンタープライズのアイデンティティプロバイダー (IdP) にリダイレクトします。

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.federatedLogin({
    connection: "google-oauth2"
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [FederatedLoginPayloadOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/FederatedLoginPayloadOptions)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      使用するソーシャル接続名。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションのエラー配列を取得します。存在しない場合は空の配列を返します。トランザクション コンテキスト内のエラーオブジェクトの配列です。
</ParamField>

<ParamField body="login" type="Promise<void>">
  このメソッドは、識別子 と指定したパスワードに基づいてユーザーを認証します。

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.login({
    username: "testUser",
    password: "testPassword"
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LoginPayloadOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LoginPayloadOptions)。
    </ParamField>

    <ParamField body="username" type="string" required>
      ユーザーの 識別子。
    </ParamField>

    <ParamField body="password" type="string" required>
      ユーザーのパスワード。
    </ParamField>

    <ParamField body="captcha?" type="string">
      captcha の code、または CAPTCHA provider からのレスポンスです。このプロパティは、Auth0 テナントで [ボット検出](/docs/ja/secure/attack-protection/bot-detection) が有効になっている場合に必須です。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  このメソッドは、データベース接続 (パスワード) とパスワードレス接続 (Email/SMS OTP) を切り替えます。

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.switchConnection({
    connection: "email"
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [SwitchConnectionOptionsLoginPassword](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/SwitchConnectionOptionsLoginPassword)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      切り替え先の接続名。パスワードレスには `'email'` または `'sms'` を使用し、パスワード認証には DB 接続名を使用します。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      `ulp-` で始まる追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>
