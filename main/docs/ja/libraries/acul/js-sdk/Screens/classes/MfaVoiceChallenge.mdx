MfaVoiceChallenge クラスは、`mfa-voice-challenge` 画面の機能を提供します。この画面は、ユーザーが本人確認のために音声通話で受け取ったコードを入力する必要がある場合に表示されます。

  ## コンストラクタ
</div>

MFA Voice Challenge画面マネージャーのインスタンスを作成します:

```typescript Example
import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

const mfaVoiceChallenge = new MfaVoiceChallenge();
await mfaVoiceChallenge.continue({ code: '123456' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  mfaVoiceChallenge.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  このメソッドは、MFA チャレンジを検証するために音声の確認コードを送信します。

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.continue({
    code: '123456',
    rememberDevice: true,
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [MfaVoiceChallengeContinueOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/MfaVoiceChallengeContinueOptions)。
    </ParamField>

    <ParamField body="code" type="string" required>
      音声通話で受信した確認コード。
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      `true` の場合、今後の MFA チャレンジに備えてブラウザーを記憶します。これは `screen.showRememberDevice` が `true` の場合にのみ適用されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。存在しない場合は空の配列を返します。
</ParamField>

<ParamField body="pickPhone" type="Promise<void>">
  このメソッドは、ユーザーが別の電話番号を選択できる画面に移動します。

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.pickPhone();
  ```
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  このメソッドは、確認コードを含む新しい音声通話をリクエストします。

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  このメソッドは、設定可能なタイムアウトとステータス管理を備えた再送機能を提供します。

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  const { startResend } = mfaVoiceChallenge.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    },
  });

  startResend();
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [StartResendOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions)。
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      再送のステータスが変化するたびに呼び出されるコールバックで、残り秒数と、現在再送が無効かどうかを受け取ります。
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      再送のタイムアウトが終了し、再送オプションが利用可能になったときに呼び出されるコールバック。
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      再送オプションが利用可能になるまでの秒数。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  このメソッドは、認証方法をSMSに切り替えます。

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.switchToSms();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドを使用すると、ユーザーは別のMFA方式を試すことができます。

  ```typescript Example
  import MfaVoiceChallenge from '@auth0/auth0-acul-js/mfa-voice-challenge';

  const mfaVoiceChallenge = new MfaVoiceChallenge();
  await mfaVoiceChallenge.tryAnotherMethod();
  ```
</ParamField>
