SignupPassword クラスは、`signup-password` 画面の機能を実装するクラスです。この画面では、ユーザーのパスワードを入力します。

  <img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/5h85r3sVhe8mVw7rDkDwTD/02c3521d9d6c4d490c1a64ae37ecca72/Signup_Password_with_Flexible_IDs.png" alt="Flexible IDs を使用した Signup Password" />
</Frame>

<div id="constructors">
  ## コンストラクター
</div>

SignupPassword画面マネージャーのインスタンスを作成します:

```typescript Example
import SignupPassword from '@auth0/auth0-acul-js/signup-password';
const signupPasswordManager = new SignupPassword();
signupPasswordManager.signup({
  email: 'test@example.com',
  password: 'P@$wOrd123!',
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  signupPasswordManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedSignup" type="Promise<void>">
  このメソッドを使用すると、Google や Facebook など、さまざまなソーシャル ID を使ってサインアップできます。

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  signupPasswordManager.federatedSignup({
    connection: 'google-oauth2',
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [FederatedSignupPasswordPayloadOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/FederatedSignupPasswordPayloadOptions)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      使用するソーシャル接続名です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーが存在しない場合は空の配列を返します。
</ParamField>

<ParamField body="signup" type="Promise<void>">
  このメソッドは、サインアップ用パスワードフォームの送信を処理します。

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  signupPasswordManager.signup({
    email: 'test@example.com',
    password: 'P@$wOrd123!',
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [SignupPasswordOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/SignupPasswordOptions)。
    </ParamField>

    <ParamField body="email?" type="string">
      ユーザーのメールアドレスです。
    </ParamField>

    <ParamField body="username?" type="string">
      ユーザーの識別子です。
    </ParamField>

    <ParamField body="phoneNumber?" type="string">
      ユーザーの電話番号です。
    </ParamField>

    <ParamField body="password" type="string" required>
      ユーザーのパスワードです。
    </ParamField>

    <ParamField body="captcha?" type="string">
      captcha プロバイダーから返される captcha コードまたはレスポンスです。Auth0 テナントで [ボット検出](/docs/ja/secure/attack-protection/bot-detection) が有効になっている場合、このプロパティは必須です。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  このメソッドは、signup-password フロー中に使用する接続を切り替えます。

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  signupPasswordManager.switchConnection({
    connection: 'Username-Password-Authentication',
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [SwitchConnectionOptionsSignupPassword](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/SwitchConnectionOptionsSignupPassword)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      signup-password 認証フロー中に切り替える接続識別子です。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="validatePassword" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/PasswordValidationResult">PasswordValidationResult</a></span>}>
  このメソッドは、パスワード文字列を現在のトランザクションのパスワードポリシーに照らして検証します。パスワードが有効かどうかと、その理由を示すオブジェクトを返します。

  ```typescript Example
  import SignupPassword from '@auth0/auth0-acul-js/signup-password';
  const signupPasswordManager = new SignupPassword();
  const result = signupPasswordManager.validatePassword('MyP@ssw0rd!');
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="password" type="string" required>
      検証するパスワード文字列です。
    </ParamField>
  </Expandable>
</ParamField>
