---
title: "Login"
description: "Universal Login の `login` 画面をカスタマイズするために使用できる、すべてのプロパティとメソッドについて説明します。"
---

Login クラスは、`login` 画面の機能を実装します。この画面では、ユーザーの識別子とパスワードを入力します。テナントの設定によっては、この識別子としてメールアドレス、電話番号、またはユーザー名を使用できます。

<div id="constructors">
  ## コンストラクター
</div>

Login画面マネージャーのインスタンスを作成します:

```typescript Example
import Login from "@auth0/auth0-acul-js/login";
const loginManager = new Login();
loginManager.login({
  username: "testUser",
  password: "testPassword"
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` プレフィックスが付いた追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedLogin" type="Promise<void>">
  このメソッドは、認証を行うためにユーザーをソーシャルまたはエンタープライズのアイデンティティプロバイダー (IdP) へリダイレクトします。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.federatedLogin({
    connection: "google-oauth2"
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [FederatedLoginPayloadOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/FederatedLoginPayloadOptions)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      使用するソーシャル接続名。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストから transaction のエラー配列を取得します。存在しない場合は空の配列を返します。transaction コンテキスト内の error object の配列です。
</ParamField>

<ParamField body="getLoginIdentifiers" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a>[] | null</span>}>
  このメソッドは、`login` 画面で有効な識別子タイプを返します。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.getLoginIdentifiers();
  ```
</ParamField>

<ParamField body="login" type="Promise<void>">
  このメソッドは、識別子と入力されたパスワードに基づいてユーザーを認証します。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.login({
    username: "testUser",
    password: "testPassword"
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LoginPayloadOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LoginPayloadOptions)。
    </ParamField>

    <ParamField body="username" type="string" required>
      ユーザーを識別する値です。
    </ParamField>

    <ParamField body="password" type="string" required>
      ユーザーのパスワードです。
    </ParamField>

    <ParamField body="captcha?" type="string">
      captcha コード、または CAPTCHA プロバイダーからのレスポンスです。Auth0 テナントで [ボット検出](/docs/ja/secure/attack-protection/bot-detection) が有効になっている場合、このプロパティは必須です。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  このメソッドはユーザーを国番号の選択リストにリダイレクトし、そこで電話番号の国番号プレフィックスを更新できます。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.pickCountryCode();
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [CustomOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/CustomOptions).
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>
