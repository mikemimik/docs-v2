ResetPasswordMfaEmailChallenge クラスは、`reset-password-mfa-email-challenge` 画面の機能を実装したクラスです。この画面では、パスワードリセットフロー中の MFA ステップの一環として、メールアドレスに送信された確認コードをユーザーに入力してもらいます。

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  resetPasswordMfaEmailChallengeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="Parameters">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語は session の間維持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continue" type="Promise<void>">
  このメソッドは、パスワードリセットフロー中の MFA チャレンジを続行するために、メールアドレスに送信された確認コードを送信します。

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  await resetPasswordMfaEmailChallengeManager.continue({
    code: '123456',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="Parameters">
    <ParamField body="options">
      [ContinueOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ContinueOptions)。
    </ParamField>

    <ParamField body="code" type="string" required>
      ユーザーのメールアドレスに送信された確認コード。
    </ParamField>

    <ParamField body="rememberDevice?" type="boolean">
      今後の MFA チャレンジのためにデバイスを記憶するかどうか。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクション エラーの配列を取得します。存在しない場合は空の配列を返します。
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  このメソッドは、ユーザーのメールアドレスに確認コードを再送します。

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  await resetPasswordMfaEmailChallengeManager.resendCode();
  ```
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  このメソッドは、`reset-password-mfa-email-challenge` 画面に対して、timeout management を含む resend functionality を提供します。

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  const { startResend } = resetPasswordMfaEmailChallengeManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // ユーザーが再送ボタンをクリックしたときに startResend を呼び出します
  startResend();
  ```

  **メソッドのパラメータ**

  <Expandable title="Parameters">
    <ParamField body="options">
      [StartResendOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions)。
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      ユーザーが code を再送できるようになるまでの待機秒数。
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      resend status が変化するたびに呼び出されるコールバックです。残り秒数と、resend functionality が現在無効かどうかを返します。
    </ParamField>

    <ParamField body="onTimeout?" type="void">
      timeout が終了し、再送ボタンが利用可能になったときに実行されるコールバック。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドを使用すると、ユーザーはチャレンジを完了するための別のMFA方式を選択できます。

  ```typescript Example
  import ResetPasswordMfaEmailChallenge from '@auth0/auth0-acul-js/reset-password-mfa-email-challenge';
  const resetPasswordMfaEmailChallengeManager = new ResetPasswordMfaEmailChallenge();
  await resetPasswordMfaEmailChallengeManager.tryAnotherMethod();
  ```
</ParamField>
