MfaEmailList クラスは、`mfa-email-list` 画面の機能を実装します。この画面では、ユーザーは MFA に使用する登録済みのメールアドレスを選択できます。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

MFA Email List画面マネージャーのインスタンスを作成します。

```typescript Example
import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

const mfaEmailList = new MfaEmailList();
await mfaEmailList.selectMfaEmail({
  index: 0
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';
  const mfaEmailList = new MfaEmailList();
  mfaEmailList.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語がセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストから transaction エラーの配列を取得します。エラーが存在しない場合は空の配列を返します。
</ParamField>

<ParamField body="goBack" type="Promise<void>">
  このメソッドは、前の画面に戻ります。

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

  const mfaEmailList = new MfaEmailList();
  await mfaEmailList.goBack();
  ```
</ParamField>

<ParamField body="selectMfaEmail" type="Promise<void>">
  このメソッドは、一覧から登録済みのメールアドレスを選択します。

  ```typescript Example
  import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

  const mfaEmailList = new MfaEmailList();
  await mfaEmailList.selectMfaEmail({
    index: 0
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [SelectMfaEmailOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/SelectMfaEmailOptions)。
    </ParamField>

    <ParamField body="index" type="number" required>
      選択するメールアドレスのインデックスです。
    </ParamField>
  </Expandable>
</ParamField>
