BruteForceProtectionUnblockFailure クラスは、`brute-force-protection-unblock-failure` 画面の機能を実装します。この画面は、ブルートフォース対策によってブロックされたアカウントのブロック解除を試みて失敗した場合に表示されます。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

BruteForceProtectionUnblockFailure 画面マネージャーのインスタンスを作成します。

```typescript Example
import BruteForceProtectionUnblockFailure from '@auth0/auth0-acul-js/brute-force-protection-unblock-failure';
const bruteForceProtectionUnblockFailureManager = new BruteForceProtectionUnblockFailure();
bruteForceProtectionUnblockFailureManager.changeLanguage({ language: 'en' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Loginページの表示言語を変更します。

  ```typescript Example
  import BruteForceProtectionUnblockFailure from '@auth0/auth0-acul-js/brute-force-protection-unblock-failure';
  const bruteForceProtectionUnblockFailureManager = new BruteForceProtectionUnblockFailure();
  bruteForceProtectionUnblockFailureManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のlocaleコード (たとえば、`'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッションの間保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからtransactionエラーのarrayを取得します。エラーが存在しない場合は、空の配列を返します。
</ParamField>
