MfaSmsList クラスは、`mfa-sms-list` 画面の機能を提供します。この画面は、ユーザーが SMS ベースの MFA 用に登録済みの電話番号を選択する必要がある場合に表示されます。

<div id="constructors">
  ## コンストラクター
</div>

MFA SMS List画面マネージャーのインスタンスを作成します:

```typescript Example
import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

const mfaSmsList = new MfaSmsList();
await mfaSmsList.selectPhoneNumber({ index: 0 });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="backAction" type="Promise<void>">
  このメソッドは前の画面に戻ります。

  ```typescript Example
  import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

  const mfaSmsList = new MfaSmsList();
  await mfaSmsList.backAction();
  ```
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

  const mfaSmsList = new MfaSmsList();
  mfaSmsList.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッド パラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      希望する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクション エラーの配列を取得します。存在しない場合は空の配列を返します。
</ParamField>

<ParamField body="selectPhoneNumber" type="Promise<void>">
  このメソッドは、登録済み電話番号の一覧から電話番号を選択します。

  ```typescript Example
  import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

  const mfaSmsList = new MfaSmsList();
  await mfaSmsList.selectPhoneNumber({ index: 0 });
  ```

  **メソッド パラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaSmsListOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/MfaSmsListOptions)。
    </ParamField>

    <ParamField body="index" type="number" required>
      選択する電話番号のインデックス。
    </ParamField>
  </Expandable>
</ParamField>
