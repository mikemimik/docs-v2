PhoneIdentifierEnrollment クラスは、`phone-identifier-enrollment` 画面の機能を提供します。この画面では、ユーザーは希望する確認コードの送信方法を選択して、電話番号を識別子として登録できます。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

PhoneIdentifierEnrollment画面マネージャーのインスタンスを作成します。

```typescript Example
import PhoneIdentifierEnrollment from '@auth0/auth0-acul-js/phone-identifier-enrollment';
const phoneIdentifierEnrollmentManager = new PhoneIdentifierEnrollment();
await phoneIdentifierEnrollmentManager.continuePhoneEnrollment({
  type: 'voice',
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import PhoneIdentifierEnrollment from '@auth0/auth0-acul-js/phone-identifier-enrollment';
  const phoneIdentifierEnrollmentManager = new PhoneIdentifierEnrollment();
  phoneIdentifierEnrollmentManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="Parameters">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語がセッションの間保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continuePhoneEnrollment" type="Promise<void>">
  このメソッドは、選択した配信方法で電話の登録を続行します。

  ```typescript Example
  import PhoneIdentifierEnrollment from '@auth0/auth0-acul-js/phone-identifier-enrollment';
  const phoneIdentifierEnrollmentManager = new PhoneIdentifierEnrollment();
  await phoneIdentifierEnrollmentManager.continuePhoneEnrollment({
    type: 'voice',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="Parameters">
    <ParamField body="options">
      [PhoneEnrollmentOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/PhoneEnrollmentOptions).
    </ParamField>

    <ParamField body="type" type="&#x22;text&#x22; | &#x22;voice&#x22;" required>
      確認コードの配信方法です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、context から トランザクション エラーの配列を取得します。エラーがない場合は空の配列を返します。
</ParamField>

<ParamField body="returnToPrevious" type="Promise<void>">
  このメソッドは、ユーザーを前の画面に戻します。

  ```typescript Example
  import PhoneIdentifierEnrollment from '@auth0/auth0-acul-js/phone-identifier-enrollment';
  const phoneIdentifierEnrollmentManager = new PhoneIdentifierEnrollment();
  await phoneIdentifierEnrollmentManager.returnToPrevious();
  ```
</ParamField>
