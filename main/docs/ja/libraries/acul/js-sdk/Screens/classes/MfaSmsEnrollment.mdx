MfaSmsEnrollment クラスは、`mfa-sms-enrollment` 画面の機能を提供します。この画面は、ユーザーが SMS ベースの MFA 用に電話番号を登録する必要がある場合に表示されます。

<div id="constructors">
  ## コンストラクター
</div>

MFA SMS登録画面マネージャーのインスタンスを作成します。

```typescript Example
import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

const mfaSmsEnrollment = new MfaSmsEnrollment();
await mfaSmsEnrollment.continueEnrollment({ phone: '1234567890' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  mfaSmsEnrollment.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      希望する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッションの間維持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueEnrollment" type="Promise<void>">
  このメソッドは、指定した電話番号で SMS の登録プロセスを続行します。

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.continueEnrollment({ phone: '1234567890' });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaSmsEnrollmentOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/MfaSmsEnrollmentOptions)。
    </ParamField>

    <ParamField body="captcha?" type="string">
      CAPTCHA プロバイダーからの CAPTCHA コードまたはレスポンスです。Auth0 テナントでボット検出が有効になっている場合は必須です。
    </ParamField>

    <ParamField body="phone?" type="string">
      ユーザーの電話番号です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションのエラー配列を取得します。存在しない場合は空の配列を返します。
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  このメソッドは、ユーザーが電話番号の国番号を選択できる画面に移動します。

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.pickCountryCode();
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  このメソッドでは、ユーザーは別のMFA方式を試すことができます。

  ```typescript Example
  import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

  const mfaSmsEnrollment = new MfaSmsEnrollment();
  await mfaSmsEnrollment.tryAnotherMethod();
  ```
</ParamField>
