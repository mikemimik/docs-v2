title: "ResetPasswordRequest"
description: "Universal Login の `reset-password-request` 画面をカスタマイズするために使用できる、すべてのプロパティとメソッドについて説明します。"
---

ResetPasswordRequest クラスは、`reset-password-request` 画面の機能を実装します。この画面では、パスワードリセットフローを開始するために、ユーザーに識別子の入力を求めます。

<div id="constructors">
  ## コンストラクター
</div>

ResetPasswordRequest画面マネージャーのインスタンスを作成します：

```typescript Example
import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
const resetPasswordRequestManager = new ResetPasswordRequest();
await resetPasswordRequestManager.resetPassword({ username: 'testuser' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="backToLogin" type="Promise<void>">
  このメソッドは、ユーザーをログイン画面に戻します。

  ```typescript Example
  import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
  const resetPasswordRequestManager = new ResetPasswordRequest();
  await resetPasswordRequestManager.backToLogin();
  ```
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
  const resetPasswordRequestManager = new ResetPasswordRequest();
  resetPasswordRequestManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語がセッション中維持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーがない場合は空の配列を返します。
</ParamField>

<ParamField body="getLoginIdentifiers" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a>[]</span>}>
  このメソッドは、`reset-password-request` 画面で設定されている有効な識別子タイプ (`username` や `email` など) を返します。

  ```typescript Example
  import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
  const resetPasswordRequestManager = new ResetPasswordRequest();
  resetPasswordRequestManager.getLoginIdentifiers();
  ```
</ParamField>

<ParamField body="resetPassword" type="Promise<void>">
  このメソッドは、ユーザーの識別子を送信してパスワードリセットフローを開始します。

  ```typescript Example
  import ResetPasswordRequest from '@auth0/auth0-acul-js/reset-password-request';
  const resetPasswordRequestManager = new ResetPasswordRequest();
  await resetPasswordRequestManager.resetPassword({ username: 'testuser' });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [ResetPasswordRequestOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResetPasswordRequestOptions)。
    </ParamField>

    <ParamField body="username" type="string" required>
      リセット対象のアカウントのユーザー名またはメールアドレスです。
    </ParamField>

    <ParamField body="captcha?" type="string">
      このテナントで CAPTCHA が有効な場合の CAPTCHA 応答トークンです。
    </ParamField>
  </Expandable>
</ParamField>
