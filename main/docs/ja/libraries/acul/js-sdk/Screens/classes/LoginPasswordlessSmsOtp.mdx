<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import LoginPasswordlessSmsOtp from '@auth0/auth0-acul-js/login-passwordless-sms-otp';
  const loginPasswordlessSmsOtpManager = new LoginPasswordlessSmsOtp();
  loginPasswordlessSmsOtpManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      希望する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` を接頭辞として持つ追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。存在しない場合は空の配列を返します。トランザクションのコンテキストに含まれるエラーオブジェクトの配列です。
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  このメソッドは、タイムアウト管理を備えた再送機能を提供します。`startResend()` メソッドを持つ [`ResendControl`](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/ResendControl) オブジェクトを返します。このメソッドは再送信をトリガーし、ユーザーが再度再送信できるようになるまでクールダウン期間を適用します。

  ```typescript Example
  import LoginPasswordlessSmsOtp from '@auth0/auth0-acul-js/login-passwordless-sms-otp';

  const loginPasswordlessSmsOtpManager = new LoginPasswordlessSmsOtp();
  const { startResend } = loginPasswordlessSmsOtpManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`再送信は ${remainingSeconds}s 後に利用可能です。無効状態: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('再送信が利用可能になりました');
    }
  });

  // ユーザーが再送信ボタンをクリックしたときに startResend を呼び出します
  startResend();
  ```

  **メソッドのパラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [StartResendOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions)。
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      ユーザーが再度コードを再送信できるようになるまでのクールダウン時間 (秒) 。
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      再送信の状態が変化するたびに呼び出されるコールバックです。残り秒数と、再送機能が現在無効かどうかを返します。
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      クールダウンが終了し、再送信が再び利用可能になったときに呼び出されるコールバックです。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  このメソッドは、データベース接続 (パスワード) とパスワードレス接続 (Email/SMS OTP) を切り替えます。

  ```typescript Example
  import LoginPasswordlessSmsOtp from '@auth0/auth0-acul-js/login-passwordless-sms-otp';
  const loginPasswordlessSmsOtpManager = new LoginPasswordlessSmsOtp();
  loginPasswordlessSmsOtpManager.switchConnection({
    connection: 'sms',
  });
  ```

  **メソッドのパラメータ**

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      `ulp-` を接頭辞として持つ追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
