BruteForceProtectionUnblockSuccess クラスは、`brute-force-protection-unblock-success` 画面の機能を実装します。この画面は、ブルートフォース対策を通じて送信されたアカウントのブロック解除リクエストが正常に完了した後に表示されます。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

BruteForceProtectionUnblockSuccess 画面マネージャーのインスタンスを作成します。

```typescript Example
import BruteForceProtectionUnblockSuccess from '@auth0/auth0-acul-js/brute-force-protection-unblock-success';
const bruteForceProtectionUnblockSuccessManager = new BruteForceProtectionUnblockSuccess();
bruteForceProtectionUnblockSuccessManager.changeLanguage({ language: 'en' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import BruteForceProtectionUnblockSuccess from '@auth0/auth0-acul-js/brute-force-protection-unblock-success';
  const bruteForceProtectionUnblockSuccessManager = new BruteForceProtectionUnblockSuccess();
  bruteForceProtectionUnblockSuccessManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッド パラメータ**

  <Expandable title="パラメータ">
    <ParamField body="options">
      [LanguageChangeOptions](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語の locale コード (例：`'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/ja/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストから transaction エラーの配列を取得します。エラーがない場合は空の配列を返します。
</ParamField>
