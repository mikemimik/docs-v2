---
title: "MfaPhoneEnrollmentContinueOptions"
---

MFA の電話番号登録を続行するためのオプションです。

`CustomOptions` を拡張し、phone と type に加えてカスタムオプションも渡せるようにします。

```ts Example
export interface ContinueOptions extends CustomOptions {
  phone: string;
  type: 'sms' | 'voice';
}
```
