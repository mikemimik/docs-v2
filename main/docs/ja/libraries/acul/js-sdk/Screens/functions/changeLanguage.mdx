title: "changeLanguage"
---

> **changeLanguage**(`options`: [`LanguageChangeOptions`](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)): `Promise`&lt;`void`&gt;

`change-language` アクションを使用して新しいロケール設定をサーバーに送信し、選択した言語でインターフェースを再レンダリングします。

<div id="parameters">
  ## パラメーター
</div>

<ParamField body="options" type="LanguageChangeOptions" required>
  言語変更の設定です。[`LanguageChangeOptions`](/docs/ja/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)を参照してください。

  <Expandable title="オプション">
    <ParamField body="language" type="string" required>
      対象言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。Auth0 テナントで有効になっているロケールのいずれかを指定する必要があります。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      言語設定を保持する範囲です。現在サポート対象なのは `"session"` のみです。
    </ParamField>
  </Expandable>
</ParamField>

<div id="returns">
  ## 戻り値
</div>

`Promise`&lt;`void`&gt;

ロケール設定が送信され、画面が選択した言語で再レンダリングされると、このPromiseは解決されます。
