**Auth0.Android** には、ブラウザーを使用して認可フローを処理するための `WebAuthProvider` クラスが含まれています。では、独自の実装や `AuthProvider` の Native 版を使用したい場合はどうすればよいのでしょうか？

<div id="the-authprovider-class">
  ## AuthProvider クラス
</div>

`AuthProvider` インターフェースを実装するクラスを作成し、そのメソッドをオーバーライドします。

Auth0 が提供している **Native** 実装を利用することもできます。現在利用できる[ネイティブソーシャル認証](/docs/ja/libraries/lock-android/lock-android-native-social-authentication)のプロバイダーは、Google と Facebook です。
