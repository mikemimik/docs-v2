<Tooltip tip="パスワードレス: 第一要素としてパスワードに依存しない認証方式です。" cta="用語集を見る" href="/docs/ja/glossary?term=Passwordless">パスワードレス</Tooltip> 接続では、ユーザーはパスワードを使わずにログインできます。代わりに、さまざまな認証方式を利用できます。Auth0 のパスワードレス接続では、次の認証方式がサポートされています。

| 手段                                                                                    | 説明                                                                                                                                   |
| ------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| [Email](/docs/ja/authenticate/passwordless/authentication-methods/email-otp)             | ユーザーはメールアドレスの入力を求められ、Auth0 はそのアドレスに 1 回限り有効なコードを送信します。ユーザーはそのコードをアプリケーションに入力します。                                                     |
| [Magic Link](/docs/ja/authenticate/passwordless/authentication-methods/email-magic-link) | ユーザーはメールアドレスの入力を求められ、Auth0 はそのアドレス宛てにリンク付きのメールを送信します。ユーザーがそのリンクをクリックすると、アプリケーションに直接ログインできます。                                         |
| [SMS](/docs/ja/authenticate/passwordless/authentication-methods/sms-otp)                 | ユーザーは電話番号の入力を求められ、Auth0 はその番号に 1 回限り有効なコードを送信します。デフォルトでは、Auth0 はコードの送信に Twilio を使用しますが、カスタム SMS ゲートウェイがある場合は、接続を変更してそちらを使用することもできます。 |
