title: パスワードレス接続の制限事項
---

使用する認証の種類は、アプリケーションで設定している <Tooltip tip="Universal Login: ユーザーの本人確認のために、アプリケーションが Auth0 の Authorization Server でホストされている Universal Login にリダイレクトされます。" cta="用語集を見る" href="/docs/ja/glossary?term=Universal+Login">Universal Login</Tooltip> のバージョンによって異なります。Universal Login を使用している場合は、[デバイスの生体認証を使用する WebAuthN によるパスワードレス認証](/docs/ja/authenticate/login/auth0-universal-login/passwordless-login/webauthn-device-biometrics)を設定するか、以下で説明する <Tooltip tip="Universal Login: ユーザーの本人確認のために、アプリケーションが Auth0 の Authorization Server でホストされている Universal Login にリダイレクトされます。" cta="用語集を見る" href="/docs/ja/glossary?term=passwordless">パスワードレス</Tooltip> 接続を使用できます。

次の方法で、メールまたは SMS によるパスワードレス認証を設定できます。

1. Universal Login + パスワードレス接続パラメーター
2. Classic Login + Lock
3. Classic Login + カスタム UI + Auth0.js

これらの実装の詳細については、[パスワードレス認証向けのメールまたは SMS を設定する](/docs/ja/authenticate/login/auth0-universal-login/passwordless-login/email-or-sms)を参照してください。

<div id="errors">
  ### エラー
</div>

埋め込みログインを使用している場合は、<Tooltip tip="カスタムドメイン: 独自の名前（バニティ名）を持つサードパーティドメイン。" cta="用語集を見る" href="/docs/ja/glossary?term=custom+domains">カスタムドメイン</Tooltip>の使用をおすすめします。詳しくは、[Custom Domains](/docs/ja/customize/custom-domains)をご覧ください。

<div id="considerations">
  ## 注意事項
</div>

* <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 のメインプロダクト。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>では、パスワードレスユーザーを作成できません。代わりに、[Management API](https://auth0.com/docs/api/management/v2#!/Users/post_users)を使って直接作成してください。
* メールのパスワードレス・マジックリンクを使用する場合、取得できるのは OTP のみで、<Tooltip tip="Refresh Token: ユーザーに再度ログインを求めることなく、新しいアクセストークンを取得するために使用されるトークン。" cta="用語集を見る" href="/docs/ja/glossary?term=Refresh+tokens">Refresh tokens</Tooltip>は取得できません。
