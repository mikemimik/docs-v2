description: データベース接続のログインにユーザー名フィールドを追加する方法を説明します。
title: データベース接続にユーザー名ログインを追加する
---

データベース接続では、ユーザーはメールアドレスではなくユーザー名でサインインできます。

<div id="require-username">
  ## ユーザー名を必須にする
</div>

1. [Auth0 Dashboard &gt; Authentication &gt; Database](https://manage.auth0.com/#/connections/database) に移動し、編集する接続名を選択します。
2. **Requires Username** を見つけ、トグルでユーザー名の必須設定を有効または無効にします。

   <Frame>![ユーザー名必須設定を有効または無効にする](/docs/images/cdy7uua7fh8z/1MlOrFmcSCOCRYKRxWz6xT/7a87dac16a31cbb6e608489c5843d6c7/2025-02-25_12-02-23.png)</Frame>

この設定がログイン画面にどう反映されるかを確認するには、**Try Connection** ビューを選択します。**Requires Username** を有効にすると、新規ユーザーはサインアップ時にユーザー名とメールアドレスの両方を入力する必要があります。

<Frame>![ユーザー名付きのログインフォーム](/docs/images/cdy7uua7fh8z/AiB37UZqyBPb8h55PZtzm/73e6fd0033f63f865c40f189db1620ef/2025-01-29_15-22-11.png)</Frame>

その後、ユーザーはユーザー名またはメールアドレスのいずれかでログインできます。**Requires Username** が無効な状態で登録したユーザーには ユーザー名 フィールドが保存されていないため、メールアドレスでログインする必要があります。

<div id="username-length">
  ### ユーザー名の長さ
</div>

ユーザー名のデフォルトの許容文字数は 1～15 文字です。ただし、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Management API: 顧客が管理タスクを実行できるようにする製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Management+API">Management API</Tooltip> v2 を使用すると、最小文字数と最大文字数を変更できます (最大 128 文字まで) 。

<div id="allowed-characters">
  ### 使用可能な文字
</div>

ユーザー名フィールドでは、次の文字を使用できます。

* 英数字 (アクセント記号は使用不可。自動的に小文字に変換されます) ;
* アットマーク (@)  (ただし、メールアドレスは使用できません) ;
* キャレット (^) ;
* ドル記号 ($) ;
* ピリオド (.) ;
* 感嘆符 (!) ;
* グレーブアクセント (&#96;) ;
* ハイフン (-) ;
* 番号記号 (#) ;
* プラス記号 (+) ;
* シングルクォート (&#39;) ;
* チルダ (~) ;
* アンダースコア (&#95;) ;

これ以外の文字や記号は使用できません。また、Auth0 はカスタムデータベースへの入力を検証したりサニタイズしたりしません。
