以下の手順に従って、SiteMinder を <Tooltip tip="Security Assertion Markup Language (SAML): パスワードなしで 2 者間の認証情報のやり取りを可能にする標準化されたプロトコル。" cta="用語集を見る" href="/docs/ja/glossary?term=SAML">SAML</Tooltip> <Tooltip tip="Security Assertion Markup Language (SAML): パスワードなしで 2 者間の認証情報のやり取りを可能にする標準化されたプロトコル。" cta="用語集を見る" href="/docs/ja/glossary?term=identity+provider">アイデンティティプロバイダー</Tooltip>として設定します。

<div id="prerequisite">
  ## 前提条件
</div>

[SiteMinder](https://www.siteminder.com/login/) のアカウントが必要です。

<div id="configure-siteminder-settings">
  ## SiteMinder 設定を構成する
</div>

SiteMinder を設定する際は、ほとんどのオプションで既定値を使用します。また、以下の設定手順では、次の Auth0 関連の値も必要になります。

* **EntityID:** `urn:auth0:{yourTenant}`
* **Assertion Consumer Service URL:** `https://{yourDomain}/login/callback`
* **ログアウト URL:** `https://{yourDomain}/logout`
* SAML リクエストには **HTTP-Redirect** バインディング
* SAML レスポンスには **HTTP-POST** バインディング

1. [SiteMinder](https://www.siteminder.com/login/) にログインします。
2. **SAML Service Provider Dialog** を開きます。
3. このサービスプロバイダーに適切な名前を設定します。
4. `NameIdentifier` を定義します。SiteMinder で認証されるユーザーに対してこれを生成する方法はいくつもあります。通常、この値は User Directory のユーザープロパティの 1 つ (`uid` など) にマッピングします。
5. サービスプロバイダーの一般的な SAML プロパティを設定します。

| Setting          | Description              |
| ---------------- | ------------------------ |
| **SP ID**        | `urn:auth0:{yourTenant}` |
| **SAML Version** | 2.0                      |
| **Skew Time:**   | 30 seconds               |

6. **Assertion Consumer Service URL** を設定します。これは、SiteMinder が SAML トークンを POST で返送する先の URL です。このサービスプロバイダー ({yourTenant}) では、SAML レスポンスに `HTTP-POST` バインディングのみをサポートしています。次を使用します。
   `https://{yourDomain}/login/callback`

7. **HTTP-Post** チェックボックスをオンにします。

8. トークンに含めて送信する追加のユーザープロパティを設定します。認証されたユーザーについてこのサービスプロバイダーと共有したいその他のプロパティを追加します。一般的な値は `name`、`lastname`、`email address` などです。このサービスプロバイダーは、前に定義した `NameIdentifier` をユーザーの一意の識別子として使用します。これらの属性は参照情報として扱われます。

9. **SLO Location URL:**`https://{yourDomain}/logout` を入力します。

10. (任意) このサービスプロバイダーはアサーションの暗号化をサポートしています。
    このオプションを使用するには、以下の手順でサービスプロバイダーの公開鍵証明書をダウンロードします。

11. Auth0 Dashboard で **Connections** をクリックし、次に **Enterprise** をクリックします。

12. SAMLP IDプロバイダーをクリックします。

13. 設定アイコン (鉛筆) をクリックします。

14. 表示されるウィンドウの 7 番目 (最後) の箇条書きに、.pem または .cer 形式の証明書をダウンロードするためのリンクがあります。

15. 必要な証明書をダウンロードし、SiteMinder の **Policy Server Keystore** に追加します。
