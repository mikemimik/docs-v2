description: Actions やユーザー管理などで Auth0 サービスを設定する際に参照する Auth0 接続の接続 ID を、Auth0 Dashboard で確認する方法を説明します。
title: 接続 ID または名前を確認する
---

一部のユーザー管理タスクでは、ユーザーソースを一意に識別する接続 ID が必要です。たとえば、ユーザーをインポートまたはエクスポートする場合や、<Tooltip tip="Identity Provider (IdP): デジタル ID を保存および管理するサービス。" cta="用語集を見る" href="/docs/ja/glossary?term=identity+providers">ID プロバイダー</Tooltip>に接続する場合は、接続 ID を指定する必要があります。

1. [Auth0 Dashboard](https://manage.auth0.com/#/) に移動し、**Authentication** を選択してから、ID が必要な接続タイプを選択します。
2. ID が必要な接続の名前を選択します。
3. **Settings** ビューで **Name** フィールドを確認します。このフィールドに接続 ID が表示されます。
