---

Auth0 Dashboard を使用して、アプリケーションのエンタープライズ接続をテストできます。

適切にテストするには、あらかじめ[エンタープライズ接続を設定](/docs/ja/authenticate/identity-providers/enterprise-identity-providers/enable-enterprise-connections)しておく必要があります。

1. [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise) に移動し、表示したい接続タイプを選択します。

   <Frame>![Dashboard - Connections - Enterprise](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. テストする接続の横にある **Try** の矢印を選択します。

   <Frame>![Dashboard - Connections - Enterprise - AD/LDAP - List](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/oidc-connection-try.png)</Frame>
3. ログインして、アプリへのアクセスを許可することに同意します。正しく設定されていれば、**It Works!** ページが表示されます。

   <Frame>![Dashboard - Connections - Database - Try - Success](/docs/images/cdy7uua7fh8z/5DWJWLf8zvZJH0rSlhK0cr/f33377edf6f82f17e11e08ec30d8d75a/connection-social-try-success.png)</Frame>
