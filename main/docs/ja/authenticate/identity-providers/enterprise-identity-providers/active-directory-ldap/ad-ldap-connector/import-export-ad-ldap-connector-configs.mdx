---
description: AD/LDAP Connectorの設定をConnector Admin Consoleからインポートまたはエクスポートする方法を説明します。
title: AD/LDAP Connector設定のインポートとエクスポート
---

AD/LDAP Connectorの設定は、エクスポートしたり、以前にエクスポートした設定をインポートしたりできます。これは、高可用性を実現するためにAD/LDAP Connectorを複数ノードにデプロイしているデプロイで便利です。1つのノードで設定とテストを行い、正常に動作する設定をエクスポートして、以降のすべてのノードにインポートできます。

1. ブラウザーで `http://localhost:8357` にアクセスし、AD/LDAP Connectorサーバー上の **Connector Admin Console** を起動します。
2. **Import / Export** タブをクリックします。
3. `%Program Files(x86)%\Auth0\AD LDAP Connector\` フォルダーから、`config.json` ファイル、`certs` フォルダー、および `lib\\profileMapper.js` ファイルを含む .zip ファイルをエクスポートします。
4. インポートするには、**Import / Export** タブでバックアップ用の .zip ファイルをアップロードします。
