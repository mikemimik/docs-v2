title: Okta Access Gateway
---

**Okta Access Gateway (OAG)** は、レガシーなオンプレミスアプリケーションにもモダンなアイデンティティを適用し、ハイブリッドクラウドを保護するために設計されたソリューションです。これにより、Auth0 を <Tooltip tip="アイデンティティプロバイダー（IdP）: デジタルアイデンティティを保存・管理するサービス。" cta="用語集を見る" href="/docs/ja/glossary?term=identity+provider">アイデンティティプロバイダー</Tooltip>として使用するよう Okta Access Gateway を簡単に設定でき、ユーザー認証や、オンプレミスアプリケーションへのアクセス時の <Tooltip tip="アイデンティティプロバイダー（IdP）: デジタルアイデンティティを保存・管理するサービス。" cta="用語集を見る" href="/docs/ja/glossary?term=Single+Sign-On">シングルサインオン</Tooltip> (SSO) を実現できます。設定の詳細については、[Okta Access Gateway の Web サイト](https://help.okta.com/oag/en-us/content/topics/access-gateway/ag-main.htm)を参照してください。

設定が正しく完了すると、OAG のアプリケーションを <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要な製品。" cta="用語集を見る" href="/docs/ja/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> 上で **SSO Integrations** として確認できます。
