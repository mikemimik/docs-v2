title: Auth0 でのログイン認証

description: Auth0 Universal Login または埋め込みログインを使用して、ユーザーのログイン認証を実装する方法について説明します。
---

Auth0 では、アプリケーションにログイン認証を実装する方法が 2 つあります。

* **<Tooltip tip="Universal Login: ユーザーの本人確認を行うため、アプリケーションは Auth0 の認可サーバーでホストされる Universal Login にリダイレクトされます。" cta="用語集を見る" href="/docs/ja/glossary?term=Universal+Login">Universal Login</Tooltip>**: ユーザーは Auth0 がホストするページを通じてアプリケーションにログインします。

* **埋め込みログイン**: ユーザーはお客様がホストするページを通じてアプリケーションにログインします。

ほとんどのユースケースでは Universal Login を推奨しますが、クライアント主導のバックチャネル認証を使用して、分離型の認証フローでユーザーを認証することもできます。
