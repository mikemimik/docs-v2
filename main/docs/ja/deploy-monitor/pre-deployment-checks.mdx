本番稼働を開始する前に、Auth0 の本番チェックを実行して、テナントが本番環境での利用に対応できていることを確認してください。

| 読む                                                                                | 学べること...                                                                                                        |
| --------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| [本番チェック](/docs/ja/deploy-monitor/pre-deployment-checks/how-to-run-production-checks) | 1 つ以上のアプリケーションに対して本番チェックを実行し、本番環境に対応できているかどうかを確認する方法。結果を確認すると、未対応の必須タスクや推奨タスクに加え、ベストプラクティスに準拠するために行える変更も把握できます。 |
| [デプロイ前テスト](/docs/ja/deploy-monitor/pre-deployment-checks/predeployment-tests)        | Auth0 をデプロイする前に、単体テストと結合テストを実行する方法。                                                                             |
| [ローンチ前のヒント](/docs/ja/deploy-monitor/pre-deployment-checks/pre-launch-tips)           | Auth0 を使い始める際に役立つ、お客様から寄せられたヒント。                                                                                |
