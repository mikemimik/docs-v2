title: ログデータの保存期間
---

Auth0 のログ保存期間は、契約プランによって異なります。

| プラン              | ログ保存期間 |
| ---------------- | ------ |
| Starter          | 1日     |
| B2C Essentials   | 5日     |
| B2C Professional | 10日    |
| B2B Essentials   | 5日     |
| B2B Professional | 10日    |
| Enterprise       | 30日    |

保存期間を超えてログを保持する必要がある場合は、[Auth0 Support](https://support.auth0.com) が外部のロギングサービスとの連携を支援します。この方法は、組織全体のログを集約する場合にも役立ちます。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 では、お使いのテナント向けにリアルタイムログを提供していません。イベントは到着後できる限り速やかにインデックス化するよう努めていますが、多少遅れる場合があります。
</Callout>

また、Auth0 のログストリーミングを使用してログデータをエクスポートすることもできます。詳しくは、Auth0 Marketplace の [Log Streaming](https://marketplace.auth0.com/features/log-streaming) を参照してください。
