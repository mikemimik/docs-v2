Private Cloud パッケージは、次のような場合に利用できるマネージドサービスです。

* 組織でマルチテナントの Public Cloud サービスを利用できない
* Public Cloud では対応が難しい地理的なデプロイ要件がある
* 1 秒あたりのリクエスト数 (RPS) の保証、または Public Cloud で提供可能な容量を上回るキャパシティが必要である

Auth0 Private Cloud は、次の方法でデプロイできます。

| オプション                                                                                          | 説明                                                                     |
| ---------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| [**Private Cloud on AWS**](/docs/ja/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | AWS 上で稼働する専用のマネージドクラウドサービスで、分離性、高パフォーマンス、開発用インスタンス、Geo-HA アドオンなどを提供します |
| [**Private Cloud on Azure**](/docs/ja/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Azure 上で稼働する専用クラウドサービスで、分離性、高パフォーマンス、開発用インスタンス、アドオンなどを提供します            |

また、Auth0 Private Cloud デプロイを補完する [アドオン機能](/docs/ja/deploy-monitor/deploy-private-cloud/private-cloud-add-on-features) も提供しています。

特定のサポート要件がある場合や、提供している Professional Services についてさらに詳しい情報が必要な場合は、[Auth0 Sales までお問い合わせください](https://auth0.com/contact-us)。
