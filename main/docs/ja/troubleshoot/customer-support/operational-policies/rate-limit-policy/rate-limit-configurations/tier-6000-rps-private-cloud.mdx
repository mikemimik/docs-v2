<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  環境内の Authentication API と MyAccount API の両方に対する持続的な総スループットは、通常、最大 500 RPS までサポート対象です。ただし、特有のワークロード、Authentication API と Management API の高負荷な併用、または Authentication API に対する ROPG などの特定の認証フローでは、より低い RPS でもパフォーマンスが低下する可能性があります。
</Callout>

以下に、Private Cloud Performance 6000 RPS (60x) および 6000 RPS (60x) Burst のサブスクリプションタイプにおけるレート制限を示します。

このため、リスク軽減のために、Private Cloud 環境ごとに 1 つのテナントをデプロイすることを推奨します。
