---
title: サービスレベル契約
description: Enterprise Public Cloud と Enterprise Private Cloud の両デプロイに対する Auth0 のサービスレベル契約（SLA）の主要な内容の概要。
---

Auth0 プラットフォームの一部の中核サービスにおける各月の平均可用性は、99.99% 以上とします。Auth0 がいずれかの暦月においてこの可用性のサービスレベルを満たさなかった場合、お客様はその事象の発生から 5 営業日以内にサービスレベルクレジットを請求できます。

Auth0 は、以下の表に記載のとおりクレジットを付与します。サブスクリプションの有効期限満了時に未使用のクレジットがある場合、当社はそのクレジットを、お客様が当社に支払うその他の料金または費用に充当します。そうしたその他の料金または費用がない場合は、クレジット額を返金します。

| Availability Level | Service Level Credit |
| ------------------ | -------------------- |
| 98.50% - 99.98%    | 月額サブスクリプション料金*の 5%   |
| 97.00% - 98.49%    | 月額サブスクリプション料金*の 10%  |
| &lt; 97.00%        | 月額サブスクリプション料金*の 20%  |

*クレジットは、サービス中断が発生した暦月に適用されます。

上記の可用性レベルを下回ったと思われる事象が発生した場合は、Auth0 テクニカルアカウントマネージャーにお問い合わせください。Basic または Silver Success Plan のお客様は、Auth0 の営業担当者またはアカウントエグゼクティブまでお問い合わせください。

Auth0 の [Private instance](/docs/ja/deploy-monitor/deploy-private-cloud) が必要な場合は、詳細について [Auth0 Sales](https://auth0.com/get-started?place=documentation%20post\&type=link\&text=contact%20auth0%20sales) にお問い合わせください。

[Auth0 Status Page](https://status.auth0.com/) にアクセスすると、Public Cloud 環境の現在および過去の稼働状況を確認できます。
